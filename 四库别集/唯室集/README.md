　　提要
　　唯室集四巻附録一巻　别集类三【宋】
　　臣等谨案唯室集四巻附録一巻宋陈长方撰长方有步里客谈已着録是集诗文散入永乐大典各韵下据胡百能行状原本凡十四巻又唐瑑原序称其家所刋凡二百篇今掇拾残缺仅得文五十二首诗三十九首勒为四巻而以他人所作铭状记序附録于后以备稽考虽较原书篇数祗及其半而菁华具在亦可以覩其大凡矣长方父侁与游酢杨时邹浩陈瓘等游故长方之学以程氏为宗朱子语録于同时学者多举其字惟于长方则称曰唯室先生盖颇引以为重也冯时可雨航襍録谓宋儒论人喜核而务深长方亦不免于是然如谓刘先主灭刘璋取蜀为行不义杀不辜故不能有天下谓张九龄与李林甫同辅政不能发其奸而去之以致天寳之乱虽核以事势均未必尽然要其理则不为不正至于绍兴六年应诏劄子谆谆以严师律备长江讲漕运为急又因朝廷罢赵鼎任张浚作里医一篇以为国家起痼疾必固元气补当持重攻当相机盖其意不主于和而亦不主于遽战富平淮西符离两败躁妄偾事若预睹之固与迂濶者异矣虽佚简残篇仅存什一要胜于虚谈高论徒供覆瓿者也乾隆四十六年九月恭校上

　　唯室集序
　　予尝谓古之圣贤未始爲文其见于语言者皆以明斯道也故曰辞达而已后世文胜质学者以文字相矜耀世亦以是重之故摛章绘句骈四俪六竞爲浮华之文而去道日逺矣流风之盛殆数千百年未易刮濯可胜叹哉亡友陈齐之初从师友学则有真得不事虚名潜心古道其读经则有春秋私记读史则有班范史论深造自得发其关键直覩堂奥圣人复起不易斯言矣不幸早殁故见于行事者泰山之一毫芒此善类所以深嗟而痛悼之也虽其平生应用之文亦不茍出率多有徳之言凡有所作又爲人持去所存者寡及其孤裒而集之仅得二百篇而已将椠木以广其传予故爲序之虽然此非其言之至也后之人能读其书逆其意以知其言则此亦不啻足矣又奚以多爲贵哉乾道戊子十一月朢日鲁国唐瑑谨序

　　帝学论
　　生民之巨福宗社之长计莫若人主知帝王之学三代而上尧舜禹汤得之故功髙万古当世被其赐三代而下孔孟得之而无其位故泽不旁流徒私淑于后人有志之士未尝不欲以献于其君或在下而无由自通或闻而不果信或信而不克用是以寥寥千百年间黔首不被圣人之泽也人主虽有尧舜禹汤之资无学以充之目夺于声色心荡于嗜好而又憸人谗夫以邪说逢迎逊顺薰渍陶染于其前后日复一日及其惑于先入善言无间而可乘则下民唯君之怨是岂知爱君之义哉是岂欲泽及黎庶哉臣防读故相司马光遗藁见其歴事三宗自爲谏官爲中执法进歴年图毎以三语爲献曰仁曰明曰武又言平生艰勤所得尽在于此光之心不愧于孟轲之爱君矣臣私自意人主心术之要无出光之三语及长得师乃知礼记大学一篇爲帝王学问之宗虽秦火之余简编紊散先后之次多失其旧然圣人之指昭昭可寻也大抵人主之学固在于博贯今古知书之多文字照人思侔造化然惟知帝王之学功徳巍巍而兼有是数者则爲多材多艺不知帝王之学徒挟数事以爲长则无益于国事矣大学言古之欲明明徳于天下者在于修身修身之道在于正心诚意欲心正意诚必先于格天下之物理使涣然氷释以极致其所知所知既明则心自正意自诚心正意诚则匪僻之心无自而生匪僻之言无自而入方寸之地昭昭然如鉴之明如水之澄由是而不已内之于进修则尧之所以钦明文思舜之所以濬哲文明温恭允塞禹之所以祗承于帝敷于四海汤伊尹之咸有一徳特名数不同尔皆此道也外之于应万事则烛见机防外之于揽人材则洞见邪正外之于辨疑似则能为权量其要特在致极其所知尔然而所谓知者非聪明才智之知非多能多艺之知在于熟察此心之正故大学又曰心有所恐惧则不得其正有所好乐则不得其正有所忿懥则不得其正有所忧患则不得其正去是四者而察此心之本体果爲如何于此了然无疑则大学所谓致知也自古圣人之徳未有不由此而进修也人主万几之暇能取大学之书防之以中庸澄神静虑不使匪僻之念萌于胷中日一览之以其疑义博访真儒则尧舜禹汤孔子孟子数圣人神交于千载之外天下将沐唐虞夏商之泽岂细事哉
　　孔子圣之时论
　　仲尼之道其犹天欤春夏秋冬发生敷荣揫敛肃杀人徒见万物枯荣之迹而已所谓化工之神初不可得而见也孟子去圣人百有余嵗而得圣人之传知圣人之奥故其发明仲尼之道不同于清和任特曰圣之时而已葢其仕止久速之不同人徒见仲尼出处之迹如是所谓仲尼之道初不可得而见也呜呼此仲尼之道所以大欤此孟子之学所以深知仲尼者欤请试论之有大人之事有圣人之亊大人者大而未化以道爲标的步趋言动视彼而行如持尺度而校长短如持权衡而酌轻重夫以尺度而论长短以权衡而量轻重宜其无所差也然而尺短寸长之间铢黍轻重之失一有少异则尺度权衡有时而不足恃其谬已千里矣圣人徳臻极致体与道合由仁义而中防通身爲度而声爲律故其动容周旋出处语黙无所待而明无所持而合不思而得不勉而中如天不言而四时自运行万物自生杀人徒见草木发生肃杀而不知化工之迹也此大人圣人之徳所以不同也伯夷伊尹柳下惠大而未化气禀于天者或有所偏故其性分之内如玉含质如金在镕所以胚浑琢磨之功有所未至其清其任其和虽各臻于圣人之一端使三子者易位迭处则或有时而未尽善矣仲尼以天纵之资造前圣之极事变千途百出于前吾随其宜而付之物各止其所事各中其防及其事遂功成仲尼之迹初不可得而见也齐景公夹谷之防徒知仲尼文儒礼义欲以莱人而刼鲁侯及其正色大数命士兵之则齐人惊畏其不可俄而度矣此仲尼之时一也衞灵公之南子凡洁身修已之士羞称其人子适衞南子请见而仲尼见之子路且以爲言则仲尼遇物而应各适其时不可俄而度此仲尼之时二也原思爲宰与粟九百子华使齐与釜与庾虽冉子且不知其意此仲尼之时三也于门人无所脱骖遇一哀于旧馆则脱骖而赙之子贡且疑其已重不知仲尼之用各适其时此仲尼之时四也圣人之徳见于亊者其梗槩如此圣人之心见于经者尚有得而言之在易之时各赞其大处泰处否则彚正彚征之不同居复居姤则朋来勿用之有异遯而尾厉干斯首凶凡易随时之义则亦仲尼之时也考之春秋赵盾之不出境则爲逆臣许止之不尝药则爲逆子里克之死不书讨贼之词郕伯之来待以成君之礼衞衎书复归而郑突不书小白书入齐而晋文不纪凡此春秋之时可得而见矣虽然孟子学圣而几于仲尼者也故其行事亦有类于圣人之时者均爲餽金或受于宋而齐则不受均爲发棠前则言之而后则不可均爲交币季子则报之而储子不报槩是三者则孟子之于圣人亦其殆庶几矣学者能知圣人之行事与其笔之于书知夫圣人之所谓时者其果爲如何知此则富贵穷达死生祸福之际其将不诡于圣人矣子思子曰君子而时中小人而无忌惮也其亦此之谓欤
　　蔺相如论
　　士之所患者才知之不足有才矣而或爲身之灾徳之累者有焉葢非才之难而用其才之难也秦自孝公而来爲封豕长蛇之毒以吞噬东方六国疲于奔命割地质子唯恐失秦之欢从约合而犀首解之齐楚和而张仪败之驱诸侯如羣羊缚懐王如稚子秦之威如是辅以昭王之暴是虎而角翼也飞以搏人何难之有蔺相如廷叱其羣臣而辱其君以击缻几于抚其背扼其防而夺之气矣赵赖之而复安相如之爲赵计则善而不自爲计也观此谓之勇者可也及其归爲上卿在廉颇之右廉颇自恃攻战之功而不平其轧已故扬言将辱之相如于是称疾不与朝防闻颇之车音望颇之羽旄必引车匿避逡巡畏缩若真有所畏者观此谓之怯者可也至舍人耻而愿辞则告以秦之不敢加兵于赵者徒以吾二人在也今两虎并鬭势不俱全故吾先国家之急而后其私是以致颇往谢而卒爲刎颈之交故后世之君子不多其能勇而多其能怯知秦之小赵久矣非有以服其口折其气则秦之暴未易胜也于是示之勇而秦堕相如之计中知颇之强也久矣非柔之以徳而服其心则颇之强不可屈也于是示之怯而颇游相如之术内此相如之或勇或怯皆相如之善用其才也后世之士欲用其才者如蔺相如可也
　　刘徳论
　　匹夫自立于乡党之间将以言行信于人必慎其举措谨其脩爲然后能有济也一有穿窬之嫌窃鈇之谤后虽输肝胆沥肺肠以吿于人人且疑之安能表表自见于世乎譬诸吴缣东绢方其洁素无染则见者莫不爱玩及其坠于泥涂则虽湔涤以西江之水人亦指之爲弃物矣此天下必然之理也又况举大号兴义师将合天下之力诛天下之贼者可不慎所注措哉刘徳自以中山靖王之孙而景帝之后也故﨑岖于吴魏之间思欲资英雄仗大义举兵东指而扫欃枪诛梼杌欲汉之宗庙复血食汉之遗民复有归其志非不善也其才非不足也奈何首攻刘璋而取蜀之土地人民兴师聚众而与曹氏程才较力以争其后先则其所以失天下仁人义士之心亦已多矣宜其卒无所就而成鼎峙之势也彼曹氏者所以大得罪于天下正在反覆谲诈移汉之神器而已然其始也犹以安汉爲名扫平盗贼尊奬王室假仁义以行其奸故能有济徳未及出一师下一令以慰民心副人望而取刘璋之恶先布于天下矣举兵之端反曹操之不如也后虽仗义东出彼仁人义士肯信其言以爲然乎不过谓其欲增土地而已惜哉徳以髀肉之复生恐功名之不就故仓卒爲此不复长虑却顾得不太左计乎既已尔矣而利兵秣马以治曹操之罪是何异于五十步而笑百步此仆所以发匹夫自立于乡党之说也孟子之称伯夷伊尹曰得百里之地而君之皆能以朝诸侯有天下又曰行一不义杀一不辜而得天下不爲惟其行一不义杀一不辜而得天下不爲故能以百里之褊小而至于朝诸侯有天下也圣人之所爲其初虽若钝而不利然而有不爲也爲必有济使徳能知此则安肯为取刘璋之事乎惜哉
