　　钦定四库全书
　　陆子余集巻一　　　　　　明　陆粲　撰注荀卿子序
　　陆子曰吾读荀卿子书髙其文辞而怪今之君子之好之何少也或曰荀卿者意广而为学濶疏议论大抵矫亢失中又时讥切孟子以故近世儒者或颇黜其书吾亦甚恨卿之不遭孟子也使夫得游従焉与相切磋而去其蔽则卿固可为醇儒哉然自孔氏没七十子之徒散亡既尽教益衰学士大夫争骛于权利而卿独脩先王之言卓然不阿时好彼所谓豪杰士也非耶战国之世儒者阨穷孟子既困于齐梁以终而卿亦老无所遇晩乃适楚楚既衰矣而黄歇当国未见其有可仕者也方天下日被秦虐吴之故墟去中国稍阔逺兵车不及卿将托焉以卒老欤非有意于干歇也卿没后李斯相秦以并天下斯虽尝游其门而卒叛去乃用异説以残民废先王之道盖其徒之不肖者也而或者谓卿之学有以使之夫学焉而失其真自孔氏之门不能无杂而独卿欤其师善医其弟子蓄蛊以杀人非师之罪也议者以斯故病卿亦良过矣卿之书三十二篇其礼乐二篇多戴记中语余篇自大略以下文辞不类往往杂见于家语诸书中将后来者勦焉以附益之乎余窃感卿以李斯故被污蔑因取其书粗为校定其章句又颇以已意训释之以俟夫读荀卿子者
　　濯缨亭笔记序
　　故绍兴郡学训导戴先生著书一编曰濯缨亭笔记余为绪正譌阙除其复重离为十巻华学士子潜取而刻之戴先生名冠字章甫吴之长洲人也少頴敏笃学始游乡校已刻意为古诗文博览无所不通而伉爽负气髙自许与不能诎折徇物八举不中以贡上礼部入试内廷奏名第一然例止得学官王三原自廵抚江南时则爱重先生及是方掌铨先生贻之书条刺十事皆经国大务语不及私三原为敛容降叹李长沙为学士亦竒其文皆不及荐也在绍兴久之与贵人语不相下弃官归年七十一终于家濒终犹歌吟不辍既而叹曰天梦梦乎世掝掝乎仳倠拥楹娵奢斥乎矫防驷驾随夷踣乎已乎已乎豪杰者废死乎闻者悲之先生早有志用世自兵农水利之説靡不论究既连蹇弗试益泄其感愤于文辞廉峭精确多所风切平生未尝一日废书不观得竒文奥义为抵掌自喜辄命笔识之是编所存仅什二三盖非其至者然其扶树教道绳枉黜邪之指亦略可睹矣君子曰夫士苟有以信于千载虽长陨沟壑不为辱也太史迁有言俶傥非常之人意有所郁结则退论书策以舒其愤思垂空文以自见若戴先生几是耶余少则知慕先生感风流之日遐惧遗文之泯坠爰叙列大校令后来者得考览焉先生尝作礼记集説辨疑未竟今掇其存者若干章附之编末他所纂述若诗文集尚数十巻藏其家
　　平越西刘氏族谱序
　　刘竒氏谱其族陆先生观之曰不亦善夫不亦善夫夫谱也者宗法之遗意也昔者先王因生以赐姓胙土而命氏以重本始辨系类明亲疎也小大之宗源流秩然防有殽杂故仁让行而天下密如也及其衰也犹有谱牒焉以识夫始终分合之故使无忽忘而已洎乎末世谱牒沦废族姓滋舛故士有服冕乘轩而其族混于甿莫知收恤者嗟乎先王之礼不行而民之散也久矣其孰能修而合之刘氏之先望于汝南自宣武君聚从髙帝戡乱以功延世赏其子义始奉诏徙平越人称西刘氏至竒五世矣乃为之谱竒可谓有志者也余也闻诸君子曰宗子之法立乃有世臣信哉斯言也今国家自通侯以及列校防弗世禄然于宗法犹未遑及是故奢丽同流而象贤济美之道阙焉夫变俗者庸众之所惊也复古者中士之所难也必也为之以渐则莫要于为谱矣谱立故分明分明故义达义达故仁让之道着而宗法可行也竒也为此将无意乎俾世禄之族胥慕而效之礼其有弗兴乎故序之以劝有礼也
　　怡老园燕集诗序
　　岁辛巳四月之朔少傅太原公张燕于怡老园之池亭门下士侍坐者凡八人是日风融日舒樽几洁修嚣尘俗虑若遁若避无自入焉时凭髙而望则秦余杭以西诸山间厠隠灭于城堞间若其合形效伎而来也酒半公取杜少陵句分韵命人为诗一章章次第成复移饮于清廕堂适他客有善歌者酒至则歌以送之谈咏间发竟日乃罢公曰乐矣今日之防也可无述乎于是颍川陈怡取诸诗聨为巻以授粲曰子盍有以复于公粲也闻之昔之居大位者患不能退即退矣有物焉役之则其身且不暇而何有于乐若夫养尊自髙倡焉而莫与和者又不论也公为三公年未及衰巻懐早退徜徉湖山乐亦甚矣而厚特达汲引后进恒若不逮四方之学者莫不思操几杖以従公游若今兹之防虽游从之常而诸生于是挹清饮醇固有不言之教无形而心成者矣可不谓乐乎昔裴晋公晩居洛阳与刘白辈穷昼夜饮午桥緑野之胜至于今人能道之然或者谓晋公盖覩时事之不满人意故托焉以自晦也虽乐而忧存矣今之时非晋公时比也公其何忧然古之大臣忧时体国固有身江湖心魏阙者而公岂遂恝然于此哉是故意兴感触形诸篇章者屡寄其忠爱之诚焉孰谓公之果无忧也夫公之忧忧以天下公所独也吾徒其何敢与知若夫乐则与众同矣虽然有独乐者焉非游従文字之谓也公且不能自言而吾徒又何足以知之粲也不敏敢卒书以请焉作怡老园燕集诗序
　　送同年戚秀夫知归安序
　　全椒戚秀夫以进士拜湖之归安令湖在浙西为名郡归安其上县也其地当吴江之阳震泽之隂升山在其东何山在其南卞山直其北武康长兴诸山属列环向蛇蜒起伏姢好秀丽其水则苕霅诸溪流衍灌溉为利甚博其土宜桑稼又多菱蒲茶苎之产木绵文绫衣被天下其民朴而务本愿而不争其士秀敏而好学王逸少谢安石顔鲁公诸君子之流风遗教犹有存者乎何其俗之美也临是邦者为政宜若易然而在今日犹病其难何哉自顷年来东南诸郡数罹水旱公私之力大抵困屈微独湖为然也役大而民劳赋繁而财窘长民者一身而百责萃焉信其有不易为者乎虽然吾闻古之君子将有为于天下其始必自夫难者试之进士为县仕之始也虽当其难亦有志者所不辞矣今之为县者治不一二年则諰諰然欲亟去之勇于取名而怯于为义巧于奉上而疎于得民前规后随以为当然使夫王泽不宣元元滋困非以是故与抑所谓有志者何世无之意其诚心经理如古循吏者或亦有人焉而世特未知也秀夫君子人也貌温而荘言慤而慎推是为政其行古道而不袭流俗之风决矣当斯人之憔悴在上者苟有意焉足以使其惠朝布而暮及又况民风土俗之厚若归安者乎其亦易治也秀夫往哉难不为沮易不为怠惟政之善不惟官之达则归安治矣
