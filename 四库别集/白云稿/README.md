　　提要

　　臣等谨案白云稿五卷明朱右撰右字伯贤临海人自号邹阳子元至正二十一年尝诣阙献河清颂不遇而归洪武三年召修元史六年修日厯除翰林院编修七年修洪武正韵寻迁晋府右长史卒于官明史文苑传附载赵埙传中所著白云稿本十卷今世所传仅存五巻杂文之后仅有琴操而无诗检勘诸本并同无可校补朱彛尊静志居诗话谓后五巻尝得内阁本一过眼恨未钞成足本则彛尊家所藏亦非完帙也右为文不矫语秦汉惟以唐宋为宗尝选韩栁欧阳曾王三苏为八先生文集八家之目实权舆于此其格律渊源悉出于是故所作类多修洁自好不为支蔓之词亦不为艰深之语虽谨守规程罕■〈月吾〉变化未免意言并尽而较诸野调芜词驰骋自喜终不知先民矩矱为何物者有上下床之别矣

　　●白云稿原序

　　古之人无时而不学而犹以为不足自其覊贯之始教于校室移于鄊党少长又去之四方博求贤人师之业成而入官又即其官以为学盖学焉以终其身而蕲至于圣人之道今学者几成而弃之儢儢然自以为足恶可与言哉临海朱伯贤好学而敏尝从吾甥叔夏游又从于林景和氏三年二氏之徒皆称之今年过余钱塘出其所著白云稿请于余余谓曰凡民之欲足而止惟学不可以足而止夫食饮以除渇饥而足衣以适体而足宫室以御寒暑风雨而足足则止矣学问异于是足则■〈施，弓代方〉不足则进足则损不足则益是故学常不足而明生焉欲常不足而惑生焉故古之人所为无时而不学余观伯贤生真所谓嗜学若不足者吾无以易之然犹为是渎告焉者亦以为惟伯贤然后可与言也至元五年五月一日永嘉李孝光序
　　白云稿者朱君伯贤之所作也伯贤天台人名右父母具在日望亲舍白云托于赋咏故因有所称焉伯贤幸临既坐定出一编授余余尽读之吊贾生瓌竒类贾生利泽碑震泽赋雄徤有西汉风读诗感兴当不在魏晋下至若读史论议虽当世宿学弗过也乃今得知君之学有所师皆合乎古之道凡古之道不得以行于今者亦足以传于后今信然数称誉于人曰吾欲以博学识字求若伯贤者盖鲜况其徳操风裁又孰与伯贤齿窃为有司惜之知者谓然不知者反相谓曰方今以明经取士为儒者先字且不暇识何暇工古文辞为朱君明经士也古之道其果能之乎余曰不然是非而等所知也昔者周制教民三物三年兴其贤与能者汉因之举孝亷举茂才至唐乃有明经进士宏词等选选寖多而儒道寖以湮没然而自古圣人贤士教人为为已之学何尝教人为学以为廪禄哉吾闻道穷则变物盛则衰譬犹汉之文章晋之字唐之诗此固其时也顾亦在乎人之去处何如耳余始居吴见伯贤郑宗鲁所宗鲁善伯贤温雅有持吾已存诸胸中矣是后伯贤复如建业从李季和游留岁余周覧故都名山大江之胜其所与接尽荐绅先生余益以竒之此二人者吾友也郑君死季和归老其家吾亦将隠矣又及与伯贤友盖亦有所自欤尚章协洽岁孟夏清河张天英序
　　余尝怪为古文者多用险语以文义句读异于时为工非有合于古道者也古之人为言辞少文致又时语不类故为训诰等文似难为解大约使通上下之情而已非故为其辞异于时也然其宣布号令君臣之等天伦之重性情之懿义理所在炳如日星含蓄万变无所不偹后之人虽剧于文辞欲着论其说者不尔过故其传久不衰而人宗师之下逮汉唐以至今日文之升降率与时等即其简策之存而传者读之岂故为其辞而为是异哉又尝怪业进士者多自称为时文言古文字异学不知古文又何乖于今之人也唐因隋法有明经进士自是取士者必设是科其间达人志士用以自见者亦甚众其为经义词赋果可尽传于人人耶又其人间有为史官秉笔为典彛载国家事盛衰传后世者其叙彛典明善恶果外于天人性命仁义道徳之说耶文古今诚不同不外是理理明辞达今与古不异也予为是说甚久每欲从事二者之学离其异而大同之独恨无才气不克自励今年来获与予友朱君伯贤共言之及得其所为文若干巻读尽数日其志传序记等书词义优洽叙事多理趣有两汉风其五经义皆正大严宻不剿取前人之成说不为时俗律拘且曰吾志是久矣通于古而达于今也子知我盍书为文集序焉至正甲申夏临川危素序
　　先辈称文章盛衰关乎世运愚窃以为未必然也彼见欧苏王曾诸子以文迭兴而适当宋祚之隆故因得以为说至若唐贞观开元之治岂下于庆厯元丰间哉然文习弊陋未闻有能掘起之者及韩桞軰出辞章始复于古而唐室之政日就衰乱何在其能关世运哉我朝政化之弘逺过前代延佑以降可谓极治者矣是以选举法行得人为盛然数十年中名公巨人往往由科目而进其能作为文章以上尊孔孟而下袭扬马者顾多在于穷贱之士吁是又不可诘矣传曰豪杰之民虽无文王犹兴岂文章之出尤无所待于人欤天台朱君伯贤清慎好学志专而才赡当羣士进取之日独能致力于空言之文非六经之道不传非史汉之书不取斤斤然惟古是法其所撰着诚可谓髙洁博茂有足喜者而不知朱君之志果何所为而为之是又为可叹矣然愚闻之君子修学以立已缵言以明道己立而道明岂复有待于外哉其视舎此而就彼者得之多寡亦既较然着矣况朱君之才天之所厚其能终困之乎近读其所谓白云稿见数公之序论甚详故独推其可喜而可叹者为朱君言之庻或以慰其志云至正十四年岁在甲午二月既望稾城俔中序
　　昔闻诸先子曰近世淛水之东以文章显名于时者恒欻然而起殆其山川风气使然哉翮尝服膺斯言顷识临海朱君伯贤氏于武林得其所著诗文而读之方讃其能不一二年复见其近作曰白云稿者则楙长迅骞倐瞠乎其不可及窃以为是不独山川风气之使然将亦其精颛所繇致不然何其腾进之骤若此也盖伯贤之先奕叶通贵而其祖父粹得儒先之传伯贤自其蚤年思缵世业心沈潜于经史壮则行四方从先生长者游既卓焉有所见闻复归而杜门力学夙夜以古人自期未尝暂弛伯贤从事于外日久所交友皆天下鸿硕儒士大夫而凡见之制述必与其人论长较胜直相颉颃不肯出其下繇其学务师古且尝经承作者讲画是以词气光焰流丽可观而其奔逸跌荡合散起伏之态象物成形有莫能名状其妙者故人称伯贤之文章为善于变化惟物之变化不常者莫如龙幽也而能显小也而能大潜见飞跃也而能神至其全体妙用曾不可得而测识云从龙者也其善变化亦然然则伯贤之所著号白云稿不以其变化之出于自然而无待乎藻绘以为餙诚有类夫白云之在天者乎今夫白云之英英自然成文孰不瞻望而玩悦之以伯贤氏之文章欻然而显名于时固宜矣而况乘山川风气之昌秀于淛水之东耶至正庚子岁三月既望上元杨翮序
　　鄊友朱君伯贤少颕拔不羣尚志气好学读书征师问道能自树以承其家业居临海金鳌山之阳山多白云常连接海气晨夕出没君覧其蓬茀悠扬欎畅巻舒若竒峰流水变态万状端俔莫测又蜿蜒若龙蛇鸾鹄缥缈神化俛仰嘘吸怡悦性情以成其文章也既而上会稽过金陵浮洞庭彭蠡吊古今名山大川灵文胜迹而白云之聚散又常在目睫别余去几几十载遭时多故遂遯栖于越古姚虞之区至正十九年予以师旅事过其地访得之既相见出示所著白云稿累十数巻且曰吾于金鳌白云不能一日忘也虽逺游四方而亲舎之望宁或嫌于狄公乎故当操觚命牍时白云之幽思遐情固已勃于胷臆发于颕端矣因之以名编可与余闻其语斐然兴怿然喻尽日读之识其气体浑厚有西京裁制而辞章严宻典雅不务险怪艰深以求古不为绮靡缋丽以徇时若此者宜其不利于有司而可驰骋于贾董班扬之疆场以垂美于后观也危太史序得其旨矣予不能释然忘情故又为之引说焉是岁庚子冬十有一月望日同郡刘仁本序
　　刘勰论文有云论说辞序则易统其首诏策章奏则书发其源赋颂歌赞则诗立其本铭诔箴祝则礼总其端纪传文檄则春秋为其根呜呼为此说者固知文本乎经而濓犹谓其有未尽焉何也易之彖象文言有韵者即诗之属周颂敷陈而不协音者非近于书欤书之禹贡顾命即序纪之宗礼之檀弓乐记非论说之极精者欤况春秋谨严诸经之体又无所不兼之欤错综而推则五经各备文之众法非可以一事而指名也盖苍然在上者天也天不能言而圣人代之经乃圣人所定实犹天然日月星辰之昭布山川草木之森列莫不系焉覆焉皆一气周流而融通之茍欲强索而分配非愚则惑矣夫经之所包广大如斯世之学文者其可不遵之以为法乎吾友朱先生伯贤以纯笃之资而留意于辞章先秦两汉以至近代诸文无不周覧用功之久灼见其是非之真复取近正无疵者聚而为书蝇头细字动至数十大册时出而讽咏之已而叹曰学文不本诸经其犹玩培塿之卑而忽嵩华之髙乎乃复致力于经功益倍于前时越数岁胷中浩然若有所得操觚书之凡阴阳盈虚之运民物伦品之理万彚屈伸之变皆随事而着源源乎罔知其所穷且其为体多而不冗简而有度神气流动而精魄苍劲诚可谓粲然藻火之章矣濓之有志为文不下于伯贤古今诸文章大家亦多究心乃游黄文献公门公诲之曰学文以六经为根本迁固二史为波澜二史姑迟迟盍先从事于经乎濓遂取而温绎之不知有寒暑昼夜今已四十春秋矣用心之苦虽与伯贤同而伯贤之所造诣濓固不能窥见其髣髴也然而太上立徳其次即立言立言甚非易也自孟子以来致力于是者非不多求其可与经并传者舂陵周元公一人而已元公之言曰文所以载道也轮辕餙而人弗庸徒餙也况虚车乎是则文者非道不立非道不充非道不行由其心与道一道与天一故出言无非经也元公岂尝拘拘学为文哉濓与伯贤又当共朂之可也伯贤以白云稿若干巻请予序濓故具论之使知伯贤之文盖以经为本而蹈袭近代以为美者其尚有所发也哉伯贤名右天台人著书甚多所谓书传发挥春秋传类编三史钩玄秦汉文衡深衣考邾子世家元史补遗皆不在集中金华宋濓序
