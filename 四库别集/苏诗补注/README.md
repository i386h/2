　　苏诗补注原序
　　成一代之文章必能立天下之大节立天下之大节非其气足以髙天下者未之能焉孔子曰临大节而不可夺君子人与孟子曰我善养吾浩然之气以直养而无害则塞乎天地之间盖存之于身谓之气见之于事谓之节节也气也合而言之道也以是成文刚而无馁故能参天地之化闗盛衰之运不然则雕虫篆刻童子之事耳乌足与论一代之文章哉故赠太师諡文忠苏轼忠言谠论立朝大节一时廷臣无出其右负其豪气志在行其所学放浪岭海文不少衰力斡造化元气淋漓穷理尽性贯通天人山川风云草木华实千彚万状可喜可愕有感于中一寓之于文雄视百代自作一家浑涵光芒至是而大成矣朕万防余暇防绎诗书他人之文或得或失多所取舍至于轼所着读之终日亹亹忘倦常寘左右以为矜式信可谓一代文章之宗也欤乃作赞曰
　　惟古文章言必已出缀词缉句文之蟊贼手扶云汉斡造化机气高天下乃克为之猗嗟若人冠冕百代忠言谠论不顾身害凛凛大节见于立朝放浪岭海侣于渔樵岁晚归来其文益伟波澜老成无所附丽昭晰无疑优游有余跨唐越汉自我师模贾马豪竒韩栁雅健前哲典刑未足多羡敬想髙风恨不同时掩巻三叹播以声诗

　　苏诗补注例畧
　　余于苏诗性有笃好向不满于王氏注为之駮正瑕璺零丁件系收弆箧中积乆渐成巻帙后读渭南集乃知有施注苏诗旧本苦不易购庚辰春与商丘宋山言并客辇下忽出新刻本见贻检阅终巻于鄙怀颇有未惬者因复补辑旧闻自忘芜陋将出以问世昔王原叔注杜诗既行世矣王宁祖则有改正王内翰注杜集薛梦符又有补注本黄长睿有较定本蔡兴宗有正异本杜田有补遗正谬本古人于笺疏之学各抒所得不肻雷同勦説如此非欲衒已长而攻人之短也【慎】不敏窃取此义
　　公诗自仁宗嘉祐已亥始见集中所谓南行集也牛口见月诗亦是年作注家顾系诸嘉祐元年按嘉祐元年为丙申而诗中有忽忆丙申年之句其背戾可知从来编年者或起辛丑或起壬寅南行集乃已亥庚子诗反置续集中殊失位置考宋史艺文志有南征集一巻征字乃行字之讹当时此巻本自单行今自郭纶及初发嘉州以下编次一准栾城集例虽独变序固不紊也苏诗宜编年固矣惟是先生升沉中外时地屡易篇什繁多必若部居州次一一不爽自非朝夕从游畴能定之施元之顾景繁生南渡时去先生之世未逺排纂尚有舛错如客位假寐一首凤翔所作而入倅杭时次韵曹九章一首黄州所作而入守湖州时姑举二段以见编年之难凡【慎】所辨正必先求之本诗及手书真迹又参以同时诸公文集洎宋元名家诗话题跋年经诗纬用以审定前后余家少藏书每从竹垞朱先生及马衎斋素村兄弟借阅援据考证实頼佽助焉
　　兹集旧有八注十注同时稍后者有唐子西赵防等注乾道末御制序刋行绍兴中有呉兴沈氏注【见呉兴备志经籍类中】漳州黄学臯补注【见王懋宣闽大记艺文类中】今皆不传传者惟王氏施氏两家耳施氏本又多残脱近从呉中借抄一本每首视新刻或多一二行乃知新刻复经増删大都掇拾王氏旧説失施氏面目矣今于施注原本所有而新刻所删者辄补録以存其旧漫不可辨者则缺之譬诸断碣残碑自成片段何取天呉紫凤颠倒裋褐哉
　　南宋时人有笺注先生诗句号东坡锦绣段随句撰事牵合殊无根蔕此与鲁訔黄鹤之注杜李歜之撰诗史同科固有识所姗笑若乃当代文献信而足征宁容阙畧赵叔平张退傅张天觉李诚之徐德占刘仲冯刘壮舆诸公宋史各有传非冺冺无闻者仁宗朝之制科范景仁之新乐王介甫之新法种谊之禽鬼章邢恕之搆宣仁王韶之启边衅何以一无援证元祐初年议回河七年议郊祀周思道等先后论蜀茶诠释亦复影响糢糊皆疎漏之大者余无论矣
