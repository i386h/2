　　召试博学鸿词授翰林院侍读王士祯选感旧山木二集所録闰章诗最多又取其五言近体八十二聫为摘句图见所撰池北偶谈中闰章尝语士祯门人洪升曰尔师诗如华严楼阁弹指即现吾诗如作室者瓴甓木石一一就平地筑起士祯亦记于居易録平心而论士祯诗自然髙妙固非闰章所及而末学沿其余波多成虚响以讲学譬之王所造如陆施所造如朱陆天分独髙自能超悟非拘守绳墨者所及朱则笃实操修由积学而渐进然陆学惟陆能为之杨简以下一传而为禅矣朱学数传尚有典型则虚悟实修之别也闰章所论或亦防有所讽寓规于颂欤其蠖斋诗话有曰山谷言近世少年不肯课治经史徒取助诗故致逺则泥此最为诗人鍼砭诗如其人不可不慎浮华者浪子呌号者粗人窘瘠者浅痴肥者俗风云月露铺张满眼识者见之直一叶空纸耳故曰君子以言有物观其持论其宗防可睹矣古文亦摹仿欧曾不失矩度然视其诗品则少亚魏禧为作集序乃置其诗而盛许其文非笃论也外集二巻一为砚林拾遗乃奉使广东时记所见端溪石品一为试院冰渊则厯年典试所作序文及条约今附存之又有别集四巻其二巻为蠖斋诗话二巻为矩斋襍记诗话瑕瑜叅半襍记则小説家言今析出别着録焉乾隆四十四年九月恭校上

　　学余堂集原序
　　士大夫之能诗古文者数百年以来于今为盛予最爱呉门汪户部宣城施愚山先生之文其他卓荦竒伟指不胜屈今天下文患才多二家独刬除一切浮腐之言而左规右矩与古人不失尺寸此其所以难能也嵗已未施先生自京师以书来邮其诗及传记诸作属予论定而序之予于先生旧为部民未尝一请见今定先生文尽吾意所欲道若吾于易堂诸子侪辈相晨夕者之所为葢先生学圣贤之道日久心虚平而气下方孜孜欲寡其过使人轻千里而告以善况区区文章之间乎予病废三十余年不敢懐一刺一启事于贵人独往好户部文欲有所商确先之以书而世不察也以为相訾议然颇寻绎是书纵横凌厉有求胜于人之气自恨生不学道不能自克其名胜之私以五十无闻之年蹈少年喜事之习今将破焚其版然而悔已不可追矣先生诗文原本道义无自矜厉之气其誉人也无过情之词绸缪往复亦未尝自过其情所称有道者之言非耶予内媿而白汗交下者葢往往而然然先生文意朴气静初读之若未尝有所惊动于人细寻绎之则意味深长详复而不厌文章瑰伟之士退然不敢践其籓篱间为议论若不得已而出而终不欲驰骋张皇自伤其体法葢才多之文先生非不能为而不为之而未尝不可以想见其才此则愚山氏之文而非他人之为文者也先生诗古节雅音得风人之性情海内士久服习而论定之无待予不知言者之厉已予既甲乙先生文以归因道所私见质先生若夫弁诸集首则何敢宁都易堂魏禧撰

　　璿玑玉衡赋【有序】
　　葢闻载物敷仁徳莫厚于配地膺图永祚功莫崇于敬天故上古圣皇首出夙推亶聪之哲鸿蒙初剖早竭仰观之思葢神明与造物同符而耳目与化工合撰然于穆之体以象而克彰钦若之心因器而悉察自羲轩在御分昼夜以清宁及姚唐授时定支干以甲子有虞制作久备玑衡肇详上以齐七政之运行烂星云于眉睫下以资庶彚之蕃息叙人物于雍熙自非圣人其孰知之我
　　皇上出震乘干登三迈五郊祀而盥荐必亲斋居而旦明防懈恩肥土域道格苍穹犹复广征俊良稍备采择以璿玑玉衡
　　命赋伏见敬
　　天勤民至意臣上惭
　　宸眷内赧管窥敢稽首顿首而赋之其词曰
　　维帝载之黙运冺声臭以难【名】既高下以定位纷散殊以流形嬗神功以阖辟妙旋转于不停固清宁之合撰乃天施而地生伊积气之上浮亶羣阳之至精风雨露雷司其教舒敛温肃推其序蕴之则防美利于不言见之则粲星辰以众着或随躔度以迅行或守次舎而若住或象九野而区分或拱辰枢而环聚亘终古其如斯夫谁得而知其故尔乃二曜周天五行列次象纬由之错综祥祲由以征异章亥不能施其步算保章无所容其巧智苟测景于自然良坐照其位置爰有天挺至圣志气如神仰观极目理瞩无垠三辰七曜既罗于胸臆迟速顺逆洞烛其纷纶乃殚精毕虑制器以当悬象缀珠刻玉造仪以绘紫氛玑为圆景而穹窿于焉曲肖衡为横管而日星于以烂陈由是命官则羲仲和仲分其任测晷则朝夕宾饯敬其时由玑以列其全则虽圆如欹盖而图之不出指掌由衡以测其详则虽繁于散珠而窥之炯若察眉斗杓东而勾芒御春识农事之攸始防垣耀而文武协翼知皇极之咸归其后言天之家有三而宣夜无传不足以穷蕴奥周髀之说虽晰而浑天仪立遂莫出其范围要以七政既齐而庶绩皆底非徒求之天象也王者父天母地兄日姊月呼吸可以上通精神防于孚格雨时若稽于洪范之书卿云载歌飏自虞姚之册日食修徳月食修刑圣王每用以省衮职风不鸣条雨不破块古帝本之以承天则他若泰阶平则上理丕奏奎井聚则贤才彚征天仓显而嵗丰黍稷欃枪晦而世偃甲兵皆有感而胥应信庶征之分明故三言近仁荧惑退舎而恐后以民为天管氏昌言于齐后董子对天人以陈王道春秋书灾异以警休咎既协极于维皇固知纯嘏之辐辏
　　粤江赋
　　壬辰暮春奉使西粤将下漓水泛苍梧时岭南甫定所在榛莽心悬万里势异三江行而不息遂浮涨海爰抒情托翰以粤江命篇事不期详聊书即目辞不竞侈略纪荒隅若曰元虚之清波鼓景纯之洪沫此则平子为之絶倒士龙需以覆瓿云尔
　　顿余辔兮桂岭鼓余枻兮漓江维此流之漰沛实分源于沅湘凿灵渠而曲注滙猺洞以沆瀁岂天地之钟祟抑河伯之流殃南连交阯北控汉阳西毗巫峡东浴扶桑神禹望渊而却步桑钦辍翰而徬徨险隘劣于容舠猋激腾于飞矢叠羊肠以为崖纷犬牙之错峙横矶截而若断积峡闭而还启瞻前后其不属洵目而骇耳于时青阳弭节朱明发辀瘴云酿厉炎风煽愁入舟若瓮披絺胜裘虽涟漪而乐饥曾不可以久留聊翘首以遐瞩托乘槎以写忧若夫夹岸层嵓诡形不若或上方而下圆或踵锐而顶缩此乍腾而旋倚彼奋飞而中落为杵臼之参差为刀剑之岝烟云之旖旎森虎豹之相搏云窦窅其洪深洞门閴其寥廓亶魑魅之窟宇江妃之所磅礴若夫石壁百仞的烁玲珑虫鸟绘绣风雨磨砻皎洁则玉屏失素渥丹则晨霞夺红盻盻若须眉之刻画累累悬钟鼓之肃雍俯坠而成饮猿仰跃而作飞熊日曜瑚之圃潭澄珠贝之宫湛缥青与结緑与云水相菁葱良绮丽而难忘亦愀怆而动容尔乃急溆危滩星罗电逐倍蒺藜之纠蔓驱地轴以反覆龙首乍昻鳌背潜伏雷劈声訇黄牛吼瀑【雷劈黄牛并滩名】贯舳舻于一线泞盘涡于千曲石硠硠而相豗水噭噭而自哭蝮蛇喷气以成雾鳄鱼张口以伺肉苟失路于秋毫将永逝而不复其草木则郁金馥烈冶葛蔓延菓镂人面树号钩縁地罗锦灿断续藤牵枫人椰子筯竹木棉桂縁冈以挺翠榕抱石以连蜷问桄榔于宻谷搴蒟蒌于崇山草吉利以祛毒竹轮囷而为船上林絶种本草遗编其鸟兽则豺狼叠迹兕狖交衢林潜犀象原卧麕鼯猩猩踯躅笑呼糖牛防舌飞虎餐肤锦鸡孔雀倒挂提壶怜彩翰之鵕防憎南飞之鹧鸪彼吉了与鹦鹉岂言语之与人殊何托体于岭表寄翩翾之防躯羡羽族之有口哀人生之郁纡尔其乘迅流之逸骇命兰楫之骀荡语未终而岸移瞬方迎而境往驾飞廉而奔駾骖虬螭而戃怳至于縁矶升舫濡裳引防濑急篙摧装倾水溅舟人屈蠖而呌号若哀鸿之悲弹方杲杲而日出郁炎炎以沸汤鼋负沙以出曝鲛跳波而惊翔忽淫淫而滂沛云惨惨以昼昏木防防兮肌粟水漎漎兮山奔迷帆樯其失影怅天地之何存叹隂霁之回互愆四序之寒暄访溪蛮兮岛夷问江亭兮水驿败瓦頺碑梯崩径坼或卉服而鸟言亦巢山而栖石乎若行蚁之上枯梨悄兮若蜗牛之庐絶壁于是有汉将南征戈船竹舸冯夷出鬭鲛人并坐马饮流而骇毒士解甲而就裸驰羽檄而逡廵仰跕鸢之前堕亦有宦游使者持节浮槎感梦山灵陨泪江花悼然犀之短照耿幽咽于哀笳霓旌折于穷岛鹢首沈于涨沙若乃鱼盐市贩之徒珍怪逺方之客采奇方于儋崖剖明珠于海舶巴姑笑而婵媛猺户闻而踵索忽抽刃以剽杀各鸟散而狼籍至于迁客羇人幽囚嫠妇衔悲岭峤寄愁林薮影畏含沙之虫心系啼猿之树朝露其寥萧夕月蔼其幽素然青燐于宿莽吊寃魂之麕聚或风嗥而雨啸涕阑干以相顾抚单影之不双嗟遇物之非故邈中土之人兮夫何为乎此路忽收泪以自疚兮将憔悴兮告谁尼父发波臣之叹张骞犯织女之机三闾吞恨于鱼腹伍胥流愤于鸱夷彼圣贤不忖乎死生复何有于险巇龙负舟而按剑蛟争璧而不迷审至人之髣佛悟哀乐之是非齐四海于勺水指十洲以为期
　　大樟赋
　　吉州官舍旁有废署亭馆既虚古樟独存荣枯相半不知几年顾影婆娑情感斯集为之赋曰
　　维豫章之都防兮依茂树以立名生七日而表异兮更累防以长生何乔柯之拂层霄兮寄婆娑于空庭洵魁垒而耆硕兮颀然如故国之老成旭日澹兮晻防灵飚发兮清泠惊絶巘之秀峙兮纷蒙茏兮峥嵘影扶摇兮千仞腹轮囷兮十抱体臃肿以辞绳墨兮节礧砢兮骨傲卜嵗年其莫知兮若根株之天造块独处兮旁无人风夜号兮雨啸岂不可崇栋而丹楹兮聊樗散以相羊烂五色兮绣错凝清液兮芬香涉萧摵而不槁兮栖绿叶兮霜苟孤荣之不陨虽憔悴兮何伤歘青阳兮载旋敷繁英兮峭蒨枝油油以云垂叶冉冉兮雾散弛巾带以命觞防炎歊而却扇岂日月之改度愕寒暑之顷变既连蜷兮善下复夭矫兮若防廻众羽以来巢兮羣长子孙而鸣啾啾即枭鸾其杂处兮亶道大而能周木寄生兮成拱芝丛秀兮菌聚宁頺形而瘁貌兮招万类以疣附怅亭馆之就墟兮岿阶基其如故彼息影而坐憇者閴焉如四时之代去悟盛衰之相推兮蕴憯怛而谁诉生偃蹇兮正直集百灵兮慿依驾飞龙而降云中兮森兰旌兮桂旗聆笙鹤之逸响想羽人之委蛇乐虬蟠以观化兮秉素节而不移鄙封爵兮秦松逺宫阙兮汉柞齐大椿兮春秋悲杨柳兮摇落即菀枯其代嬗兮若逍遥于广莫沧海竭而山摧兮愍邓林之殚残良托身兮得所歴斧斤而独完时开襟而矫首兮闲倚啸而盘桓终含贞兮抱朴审大道之所安
