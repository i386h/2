　　彛斋文编四巻　　　　别集类三【宋】
　　【臣】等谨案彛斋文编四巻宋赵孟坚撰孟坚字子固自号彛斋太祖十一世孙其先以安定郡王从髙宗南渡家于嘉禾之广陈镇而孟坚自作告墓文中又作广成盖俗语相沿初无定字至元嘉禾志载广陈镇在海盐县东北九十里则孟坚当为海盐人其或作嘉兴者误也孟坚以宗室子登寳庆三年进士好学工书喜藏名迹时人比之米芾至今遗墨流传人人能知其姓字惟其生平始末则诸书所纪往往不同如周宻齐东野语谓其终提辖左帑身后有严陵之命是孟坚殁于宋世而姚桐寿乐郊私语谓孟坚入元不乐仕进隠居避客从弟孟頫来访坐定问弁山笠泽佳否孟頫云佳孟坚曰弟奈山泽佳何既退使人濯其坐具云云则又似元初尚存者二説错互殊甚今案孟坚甲辰嵗朝把笔诗有四十五畨见除夕之句以干支逆数之当生于庆元已未距宋亡时凡七十八年孟頫仕元尚在其后孟坚必不能及见又考朱存理铁网珊瑚载孟坚梅竹谱巻有咸淳丁卯叶隆礼防称子固晚年工梅竹歩骤逃禅予自江右归将与之是正而子固死矣防出隆礼手迹其言可信是孟坚之卒于丁卯以前更为确凿亦足证桐夀之説为诞妄矣至其歴官次第他书不载而见于诗文自述者如为湖州椽入转运司幕知诸暨县以御史言罢归皆歴歴可考独不言其尝为朝官宋诗纪事乃谓其景定初迁翰林学士又不知何所据也其集宋史艺文志不着録惟见于明秘阁书目者四册世久失传今从永乐大典摭拾补缀厘为四卷大都清逺絶俗类其为人剰璧零珪风流未冺亦足与书画并传不朽云乾隆四十六年九月恭校上

　　彜斋文编巻一　　　　　宋　赵孟坚　撰赋
　　吐绶鸡赋
　　嗟尔禽豢华彩之殊于羽翰表均圆而珠匪瑳匪刓宻踈孰铺钟星精于蚌胎乃赋彩于尔躯固已不侪于埜鹜逈间于飞鳬矣頥颔之垂更禀异质五色相宣有藏斯宻乍展乍缩若虚若实彬然而彰以时而出此又造物者神其幻相不可致诘者也若乃药堦昼长蕙圃风暖茸茸草平细细沙软日轮卓午花隂将转饮啄既厌振刷防倦支体融怡翎翮敷阐百骸畅而自恣真閟形而莫掩撼首腾吻伸嗉迅膺乍露层碧中明错金婀娜离披绚烂深沈文理横陈宛丹符之叠篆縁饰缭绕讶翠羽之垂襟五云书防将飞诏百衲锦挂向当心众目诧视兮不敢廹近灵机自爱兮返顾沈吟怒目凝注绀髪森峙翠眉双妍凤翍独异嶃然斯张竟为妩媚初笋迸于犊角渐竹批于马耳弯拳前属精粲交丽故睨而视之曰吐绶以呈竒而惑之者谓非形而以气岂识方寸之间不伦不类可大可小有张有弛邪于是敛彼辉煌屏若退藏尽忘所有举止如常有隠君子见而叹曰是所谓明哲保身不显其光也夫吾闻鹿以走险麝为生香已之所累在有其长此禽虽禀文华不常宣露时然自娱巻舒有度其养晦也藏器其览辉也遇主可不取则为吾出处知格物以书绅因援毫而成赋
　　石菖蒲赋
　　天台石桥佛仙所居境联鴈峰厥生宜蒲劒挺分脊虎狞奋须维水清而石硬产劲质之伊殊较他山之蔓生盖烈士而腐儒岩壑之阿风日不到地脉所润澹泊自如又疑得道以逍遥不践名疆而利涂山僧埜夫鉴竒品异搜取来归得亦匪易临峻崖之十寻凛束身而下缒或续以长竿之袅或刺以纎钩之利自非好尚之孤髙孰肯钟情之及是乃渍以清泉借之文石玉峰之矗蜂房将乐之珍雪色齐安子供艾若指螺曲阳盆盛浪飜琼液壅沙屑金封苔澡碧于以滋生息于以安野逸搢绅先生绮纨公子膏梁厌饫舆马赫奕乍而见之神爽心怿欲顿忘乎肉食于是移置雕栏擎擡文几纩犀筯以镊尘雨牺樽而钟水夕凄清露之下昼防花隂之底其所以爱防者弥至然未阅月而根麋甫浃旬而色瘁者何哉顾乃过尤懈怠于臧仆疑被侵陵于虫蚁岂清虚富贵逈不同方钟鼎山林未容兼美徒使唐嘉緑埜之清风汉慕赤松之髙致彛斋处士从旁而嘻曰我知之矣请言所以余见隠徒其室方丈人迹罕至景宻防其南荣苔欲生而满地罗列峰峦奠陈罍洗翛踈数劒倚壁苍翠水仅濡而不盈根频殳而勿施每指一而问年动歴闰而逾纪静定纯一有终有始寒暖燥湿不迁不徙厥无竒方唯安汝止彼其爱之适以害之特未谙乎此理方梅霖骤至浸滛滂沛根联络于银丝主人为之欣喜不知力既下分叶枯而萎纷九节之连延乍丛然而森起亦犹饱豢于肥甘夀劣休粮之士其所以约水防根经年养力叶老沧浪亦炼形之术耳又况勤怠殊方炎凉易位不常厥居朝三暮四何异爪皮騐生防本观密有愧槖駞之子此言虽迩中有深防请书以资养生岂但曰蒲而已
　　五言古诗二十三首
　　上习庵陈先生
　　厥闻忠孝者所出同一源子于所亲孝移忠理必然亲贤亦子训忠节贵两全杀身能成仁扬名斯永延是亦孝所在岂但知色难嘻其叔世风本末嗟倒悬若曰能养耳食美衣华鲜茍求胡弗至侧媚謟笑肩彼其良自多喟叹逹者观向来欧阳公拓落夷陵年夫人告之公淡泊吾所安慎毋为险变守节当弥坚卓哉名母训纪厯伊谁传恭惟奉常君矢心直似立朝几抗论凛凛髙回天力苏愦愦徒若障狂澜川人知君守固当识君母贤想其勉励时谆勤诲膝边怡愉一领解心注靡变迁升而髙为天下而沉为渊但知慈训在此外浮云焉兹来从吏隠斵屋新数椽曲槛花葳防虚月婵娟雅意寓一赋懦立贪以亷我诵顺亲记有泪如迸泉我贫母亦老苒苒髪已宣家徒空四壁簟笥笑且便惟虞官箴旷勿以温饱牵泽深百川赴火明斯聚蝉投君以臭味君子毋弃捐
　　仲弟借书持要不谨防有损为之覆背葺还
　　昔闻勤读者三度絶韦编读书书已破书则在肺肝摘实咀其华发挥事业间伊惟事精妍整束悬牙籖缥囊手未触慨是龙泉言虎子畜溪鱼汝懵未凿颛胡防追元方纷纷探简编抱书手不靣壁防枯禅靴化例随水竟负一瓻还季也怒生瘿我惭家训顔愿季无防怒凤楼修则完嗟嗟仁义谏束湿何由痊颠倒幸尔存柿叶风掀掀糊老饱九炊纸捣翼拟蝉翻翻事兼两滑腻好且坚手鉴具在兹秋露无重妍旧书百回读训佩坡老僊请季事斯语慎勿髙阁悬因怜数匮藏徒疥小印存浮云忽白衣世事无不然眼底不自快封闭何取焉所贵得之心亦可忘蹄筌又怜囿大块华好竟饱鲜一饱羶聚蝇曷与千古传纎云飞四散皎月标中天
　　赠笔工呉升
　　兰台上狸毛山谷爱鸡距物胜因人成雅制传自古风流渡江初笔翰犹朴鲁曽窥上方制遗范典刑具寸簮束万頴赡足饱霜兎丰融沛行墨充实自妍富行间得茂宻夫岂窘尺度浇浮自趋薄羸劣丑毕露清快夸钩心节括号钗股纎纎铦甚锥秪便庸书伍杀锋出光芒苖枯旱无雨龌龊痴冻蝇安能劂石怒尔来邈东嘉法则自谁祖宛见昔制作齐力万毫努吾欲标诸人示兹明取与斵雕还反朴淳风招已去春苖异芦笋广袖谬织组谁能一羽力囬彼防防注百尔今已然岂但于笔故
　　题黄嵒夏氏晓山亭诗巻
　　元晖工画山最识山之趣每言笼朝云乃是竒絶处君家结幽亭两峯正当戸窣堵冠其巅云中双笔露宗老为扁名得自昌黎句相当巻书坐埙箎逓吟赋是时神宇泰白云未飞去滃然截山腰素练初横布须臾朝阳升景象几更互盘旋宛逰龙披离擘轻絮春澌苒而泮青烟惟一缕霏防辨冈峦歴落分岛屿露晞如膏沐苍翠尽还故松针与篁丛贯穿珠无数应接良不暇诗科亦既冨只忧红尘生分我淡思虑更须清夜来四更防月吐
　　诗谈
　　吾嗤彼云士努力事诗妍竟日搜枯肠抽黄对白间尔何无达观局促自防纒不见渊明陶有诗累百篇要以写吾心出语如流泉采菊见南山得句于悠然少陵动感慨忠义胆所宣有时心境夷亦复轻翩翩纎纎白云闲无心游日邉风石激而竒奔迸生云烟讵以天然态而事斧凿镌陶尔一觞酒警尔心地偏少焉明月上髙挂西山巅听我曵杖歌金石声撼天
　　万寿寺避暑读书
　　梵寺斋殿深屋古洞隂冷火云自空飞此地暑尽屏乍至心已清坐久体如氷官清务乃简居易事亦省是为臭味同玉碎金无矿萧闲日追随论辩当脩省展书相对读佳处撃节猛舆皁谓前兹文会无许盛觞咏直寄耳沈湎夫岂肯过从奚厌宻交道久弥敬无令数斯踈此语共知警
　　为仓使呉荆溪先生夀
　　人祝先生夀勲名而福禄伊惟小门生而廼异斯祝所祝在斯文其夀非已独尝闻侍席言櫽括因以复孔孟至皇朝文与道相属溯自熈丰后专门始分目欧苏以文雄周程理义熟从此判而二流各异躅伟哉水心叶同轨混列辐粲粲云锦章理义仍炳烛筠牕一传后人已沾膏馥正统的属任非公绍者孰鲰生耳闻此惊诧叹不足因捧介夀巵谱在歌诗轴愿言清节髙嵗寒等修竹长以文鸣道声金而振玉上而轧汉周俯命骚以仆门生严楷法剩将从纪録
　　漕使李宝文先生除宣城守
　　幕客崇酒觞渠渠饯行舟先生蘸甲饮不使涓滴留持盏顾谓言汝复能吟不尔不歌吾行而俟吾言求沃尔满钟酒畀尔笔力遒扶舆懵懵醉梦化蝴蝶游夜分起展烛落笔莼丝柔请歌先生行真与人不侔前者亦云归金帛囷载收今惟明月耳舴艋轻浮浮视彼掊敛徒推剥藂怨尤逮其去国日竞欲瓦砾投此乃恋寛防说着涕泗流国用既云裕民瘉亦云瘳而令分左符逺从牧山州投防戒忌器矧我无是谋弹乌或惊鸾盍亦谨厥筹兹焉皆汩汩去矣从休休伊惟小门生莫计攀前辀弥溪桃花水舟行愿悠悠行有赐还诏伫立螭坳头
　　拜范文正公祠
　　瘦石挿半天玲珑耸苍翠想从混沌初有此纷破碎断截不分离欹危不颠坠乃于其中间穿防石路细频劳谢屐行扪抚足幻怪下有泉溶溶根来从石背香山才品题名得因人贵瘦笻行清秋西风吹我袂跻攀幸未疲俯仰一叹喟下山拜范祠遗像俨如在胸中万甲兵曽使西夏畏岩瞻石不移可想鐡肝肺嘻其羣小人风林犬空吠于今孰存亡取舍良亦易但坚自力心嵗寒见节槩卓尔彼何人晞之则亦是
　　咏史赠王侍御
　　昔有阳城子髙卧中条山粤从丹诏聘职谏居朝端北垣七八年一词曽靡殚于中常戚戚事以秦越防诤臣论一篇见诮昌黎韩城日洽壶觞视之无汗顔一朝天子误鼎鼐庸雄奸皇皇朝埜人敢议不敢言城曰兹可矣不言吾岂安当廷壊其麻势若廻狂澜臣言不可夺臣肉所可剜孰谓雄猜主天聼亦为还一鸣必惊人朝阳之祥鸾一飞必冲天北溟之鹏抟啾啾燕雀辈终嵗寒檐间君今踵其武可想倚任专士习滋久未易一匕痊遗臭偃月奸殆今逾百年利欲锢人心深矣莫洗湔几新更化机仍澁胶柱洗泥何能清救应其源昔无起楼地今举田连阡而以求善治却行而欲前复有事诞衒奋臂张虚拳驱虎良民臧否已较然胶舟倚航海恻耸忧世肩夷甫在晋朝举世无几先卓哉辩奸论万世一老泉瞽言再拜献感愤涕泗涟昔者汉更生危言扶彼颠皎余明寸心不向流俗宣维君中流柱屹是障百川向来稍诡遇髙官懐防间耿介甘不售嵗寒松栢坚国无君子哉缄黙谁复言松江千顷浪涵月金蜿蜒于兹送君行商歌频扣舷明当重回首晩风杨栁邉伫希御史雨迅扫靖幅员
