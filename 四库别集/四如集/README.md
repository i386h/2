　　钦定四库全书　　　　　集部四
　　四如集　　　　　　　　别集类三【宋】
　　提要
　　【臣】等谨案四如集五巻宋黄仲元撰仲元字善甫号四如居士后改名渊字天叟莆阳人咸淳辛未进士官国子监簿宋亡隠居不仕宋濓序其集称仲元门人詹清子类次六经四书讲义为六巻仲元子梓又分记序墓铭字训为五巻其曾孙至又裒其遗文为十巻而请濓序之今此集止文四巻附录一巻盖即梓所辑之本至其所续辑之十巻则世巳失传濓序特后人于濓文集中摘出录入者耳集内各文俱有注释疑即梓所加又有咸淳甲戌余谦一至治三年傅定保诸序所作梦笔记一篇亦附在集末仲元宗洛闽之学议论多近醇正其文章以简质为主不事驰骋虽乏雄伟气象而骨力苍劲尚见真朴之遗在宋末诸儒中固犹为笃实者也乾隆四十二年八月防校上

　　四如集原序
　　濂成童时即读四如先生诸经説见立论精防凿凿过人逺甚未尝不慕艳之稍长厯求先生之事迹然后知其渊源之所自而凝道之有方也盖先生居莆阳唐御史滔之十二代孙世为儒家其父绩师事山潘公柄复斋陈公宓二公实考亭朱子勉斋黄公之髙弟子也先生夙承父训年十二试举子业郷校多占前列后二十九年始擢咸淳辛未进士第调监瑞安府比较务未上名公钜卿争聘之为讲师而丞相江公万里挽之尤力声誉翕然动于一时徳祐乙亥冬始之官与签枢陆公秀夫遇陆公一见极器之明年丙子杭都巳陷国事不可为陆公趣先生至行都充益王府撰述官兼处置使司干办公事改刑工部架阁以通直郎陞武学谕遂转朝奉郎国子监簿兼福建路招捕使司都叅议官先生皆辞不就宋旣改物归隐重山密林不与世接四方有受学者先生为敷绎濂洛闗闽之説而开导之其为文辞务以理胜不暇如他文士驰骋葩藻以为工而当时求者纷如也门人武夷詹清子类次六经四书讲义为六巻刋之先生之子元汀州路总管府知事梓又分记序墓铭字训之属为五巻而刋之先生年八十二而终遗文流落于荐绅者尚多其曾孙郷贡进士至又裒聚为十巻将刻诸梓介楚王府伴读陈子晟徴濂为叙以传濂闻濂洛中防考亭未兴艾轩林公光朝独倡道莆阳从者如归市红泉东井之学乃闻于天下网山林公亦之实绍艾轩之传网山之后乐轩陈公藻又起而继之乐轩家长乐网山居福清咸来讲学东井而风声所被气泽所薫人皆有邹鲁之行及乎山复斋之起推阐考亭经训东西角立化行而教敷文质尤彬彬可观先生产于艾轩之郷耳目之所濡染固巳先立乎其大者复因父师之教矻矻穷年遂探考亭之正绪则其所着皆六经之防防所宣皆天地之秘藏有未涯涘者奈何道未及行而宋祚巳移识者不能不为之惜先生之所蕴与不可传者固巳随物俱化若并其遗文同就泯灭不亦重可惜哉先生之子若孙孜孜以传世为务可谓不死者矣濂年耄而昏避求文者若避讐敌甚不得巳多令学子代之今以童年慕艳之乆勉循至等所请而躬叙之如此然先生之集自能行世亦不借区区之文而后传也先生黄氏讳仲元字善甫又改名渊字天叟号韵郷老人云









　　钦定四库全书
　　四如集巻一　　　　　　宋　黄仲元　撰夣笔记
　　丙戌岁夏五八日午漏三刻后学黄天叟渊课儿讲易罢坐而假寐夣一神人坐于龛【龛音堪】谛视之吾夫子燕居像也亟再拜稽首子曰女来前吾授女笔二渊敬受又拜子曰勿拜退懐其一窥其一围径三寸许渍墨浓甚袖而请业于夫子偶角丱来覔句【丱音贯】遂觉惊喜相半廼自占曰笔者述也述事而书之者也其斯文之祥乎吾夫子笔绝于获麟今一千八百年矣当时阙里弟子贤哲七十七人此笔奚不授孟子车【轲字】而后五季而上岂无大儒此笔奚不夣夫子尝谓文王既没文不在兹乎必如吾夫子而后以斯文为已任夣者夫人所有而圣人之夣与夫人不同吾夫子旧尝夣见周公于斯文闻而知之乎哉必如吾夫子而后精神心术与周公胥契于寤寐之间渊于斯文未闻万分一世古人今思执笔从二孺子后不可得安敢歩趋吾夫子虽然吾夫子所以授笔于渊者何春秋感麟而作作起于获麟则笔止于所起春秋经也续之僣或书帝正月或书阳正月皆叛经者然获麟以后之史犹可得修乎汉司马子长史记始黄帝终麟止而十二国世表起成周共和庚申以上传疑矣班孟坚专志西汉史而秦伯莽盗世经防寓书法廼以更始帝着纪宜削迁固而下无讥焉唐陈伯玉自汉武迄唐为后史记未及竟亦不传宋邵尧夫皇极经世书上稽唐尧受命甲辰为编年谱黜吕娥姁进孺子婴以土后魏附梁以宇文周附陈夫岂无见而书少帝恒山王书曹魏接献帝或者疑焉张敬夫经世纪年夏少康出处复国甚悉所假立惠帝子不得纪元闗渉世运稍大而最可议者山阳公尚存即絶汉而予蜀司马君实先约战国至秦二世如左氏体名为通志后作通鉴刘道原之力居多而是非予夺一出光手秦以前或即通志也举要厯稽古録歴年图俱助此书之不及其曾孙伋经纬离析图又惟与两汉西晋隋唐以一统然一统与正统异一统而正者若唐虞夏商周其次汉唐与宋若西晋若隋虽一而未正朱元晦纲目于始皇二十六年大书而不分注意谓前亡后系朝觐皆归即为正统当送一难左氏传终鲁悼之四年实周贞定王之五年也繇贞定王戊寅距威烈王丁丑六十载间于史阙如此大欠事吕伯恭以鲁史左氏国策史记说苑诸书修之曰大事记上接西狩下止征和【汉武】系楚义帝系髙后一失一得识者自识然于周鲁七国事迹疎畧此史之所以不可不修继周者百世可知或者吾夫子授笔之意其在兹乎虽然史未易修也乾坤未毁老身长在无忧衣食乱其心此夣决不负负独念家无屋壁之藏世多遗轶之书【轶与逸同】畜眼未见下笔恐未精妙惟吾夫子隂隲之俾昌斯文又昌我后之人渊遂次前语为夣笔记
　　渊旧读一书记夫子作孝经时簮笔拜北辰老忘所出何休序公羊传曰孔子有云吾志在春秋行在孝经此二经者夫子二笔与孝经当时授之子舆而春秋笔削子夏不能賛一辞朱元晦谓孝经曾氏门人所记若以为夫子自着可笑之尤吾乡郑渔仲又疑夫子所取鲁史之笔适至获麟时欲修之而未及修然非与后乎此复夣夫子记问此事记成之明日渊敬防
　　族祠思敬堂记
　　堂以祠名即古家庙或曰影堂东里族黄氏春秋享祀岁节序拜之所也堂以思敬名何祭之所思主乎敬也所以有斯堂者何堂即族伯通守府君讳时之旧防事某与弟仲固日新直公【即仲稼公】侄现祖与权【朱公】得之不欲分而有之愿移为堂而祠吾族祖所自出御史公讳滔以下若而人评事公讳陟以下大宗小宗继别继祢若而人【古者诸侯适子之弟为别子别子长子继别为大宗百世不迁别子之次子以其长子继巳谓之继祢为小宗五世则迁大宗一小宗四继髙祖继曾祖继祖继祢】上治旁治下治序以昭穆凡十三代【上治祖祢尊尊也下治子孙亲亲也旁治昆弟合族以食序以昭穆别之以礼义人道竭矣】亦曰天之生物一本也子孙孙子亲亲故尊祖尊祖故敬宗敬宗故收族不则何以奠世系聨族属接文献而相与维持礼法于永年哉虽然不敬恶乎可敬所以致爱也所以致慤也【致爱极其爱亲之心致慤极其敬亲之心】而所以致爱致慤者又存乎心惟有怵惕凄怆【怵惕惊动貎凄怆悲伤貎】之心【春有怵惕之心秋有凄怆之心】故祭之前期思其居处笑语思其所乐所嗜【居处笑语乐嗜往而不反非有实也岂形体所能交思之所至足以通之矣】以已之精明而与慌惚交然后祭之质明【质明而始行事】其立敬以诎其进敬以愉其荐敬以欲【身诎而为之变色愉而致其亲心欲而冀其享已彻而退色不絶于面则慎终如始矣】彻退敬齐之色陶陶遂遂【陶陶思亲心存乎内遂遂思亲心达乎外】夫如是则所思者敬而各敬尔思也若悠悠而来不蠲不享祭毕馂余泛泛而去此岂足以称斯堂之名之义仲元窃念花树韦家宗法畜眼所未及见【族人逺来为防族虽无事亦月为之韦家宗防法也】老苏族谱一引弟侄云仍读之油然而生孝弟【苏氏族谱谱苏氏族独所自出得详与尊谱明允作也】拜于斯飨于斯思之无射敬之无射先祖是听介尔景福岂不休哉或曰新斯堂也费焉须曰节缩祭田之嬴勾稽山林之入弟侄亲表间佽【音次】助焉或防乎防具刻牲碑此不书后来者墙屋之或当修噐具之或当庀【音譬】吾宗有显融者良奥者修之庀之犹今之年庶俾勿壊书之又不一书书已系之诗曰
