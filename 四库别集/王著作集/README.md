　　王著作集　　　　　　　别集类三【宋】
　　提要
　　【臣】等谨案王著作集八巻眀王观编辑其逺祖宋王苹遗文及他人荐劄像賛题跋之属苹字信伯福建通志称绍兴初平江守孙祐以徳行荐于朝召对赐进士出身除秘书省正字累官左朝奉郎陈振孙书録解题则作以赵忠简荐赐进士出身官至著作佐郎秦桧恶之防其族子坐法牵连文致夺官与通志所记不同然此集以著作为名则陈所言为是也陈氏着録作四巻寳祐中其曾孙思文刋于呉学卢钺为序此本为眀治中苹十世孙观所编一巻为传道支派圗二巻为劄子杂文十余篇三巻以下为像赞题跋及门人私志语録之类较陈氏所记巻数遽增一倍然遗文不过一巻余皆附録实则遗佚过半葢捃拾残剰而成已非旧本以其学出伊洛而能不附秦桧立身无愧于师门故録而存之不以残缺废焉乾隆四十六年十二月恭校上

　　王著作集原序
　　儒者处必有道而后出必有用也若王先生信伯者产于闽越之间防寓震泽之上从事河洛之门资禀精粹充养纯固当时若杨时尹焞朱震诸儒素所推让无意求达适宋髙宗临郡访求布衣时郡守孙公佑以先生学行荐召对便殿先生议论纯正谓尧舜禹汤文武周公孔子传授者一心也心者一人之心也广大悉备万善无垠人主茍能推之举而措之耳髙宗目为通儒勅进士出身除秘书省正字其后髙宗感后之言即诏令重脩神哲二宗实録先生实预其事遂使当时焞卞之奸诬谤潜释宣仁盛徳埀照万世朱墨之史行而邪正之説辨矣观髙宗勅词奬谕有曰凡笔削之去取兼载不遗核朱墨之异同咸得其当备一代之成书诏万世而埀宪盖匪特指一事为然也先王之达而在位虽不大任而行道之机于是可见矣呜呼若先生者处身优防守道纯正达则行退则藏所谓与顔渊同条而共贯者耶平生所受制勅若干道及闲居与尹焞诸儒诲诤经防今为世孙惟颙翻刻于梓使乡之后学皆得而观之知先生之为人坦然君子穷达一致如此若惟颙者医学既为今时所重而好儒又若此焉呜呼是可谓无忝尔祖聿脩厥徳者耶正徳九年三月吉日后学徐源序少始知学景行前脩闻邻邑福清有郑介公侠以忤王安石被谪仕竟不显有文集二册行于世吾乡言节谊者首称焉钺兹焉客吴始得著作王先生苹文集而读之乃知其先亦为福清人厥考始徙居吴中先生从学程门以王安石尚经义而废春秋守所学不就科举晚遭四七之际卒以道鸣而吾乡粹学之传又有人焉谨按国史绍兴四年伪齐挟金人入寇冬十月诏亲征上幸平江是时守臣孙佑疏先生学行以闻有防引见上殿初赐右迪功郎未防赐进士出身除秘书省正字葢尝抚巻三叹曰明哉髙宗之为君也其知急先务也能尊尚正学也良哉赵公鼎之为相也其知以道事君也不蔽贤也儒哉孙佑之为牧守也其知风化之本也能举善也夫惟上有传道统之君下有宅道揆之臣外而郡国皆能以道学相尚兹非人文之一泰欤故自王先生之既举也越明年六月召尹焞又越二年三月召胡安国凡程门同学之士以次显擢呜呼亦盛矣然尝疑之绍兴距今未逺也洛学源流世之学者皆能歴歴指数而著作之姓字则落落不挂齿颊间岂其言论风防不甚接于人耳目故耶今观著作出处本末则有信史忠言嘉猷则有奏篇师友渊源性理奥义则有语录非不足征也而付之湮晦岂非为人后裔者之责抑亦居于是邦者之羞也福清邑庠旧有先生文集而吴学独无有非一大欠缺欤曩王公遂守此邦始祠先生于学访其后曰思文者俾奉尝嵗时思文将以福清墨本刋于吴学嘱序之惟发挥旧徳世自有人第念生长永阳距福清才一舎且尝习春秋今复寓吴慕蔺最为亲切不可无一语以答闻孙之辱若夫先生之心惟求自得于己不希名于世文岂足以论先生哉且其独抱遗经毅不舎所学以从彼退然肥遯若将终身假设不际熙朝则韫玉藏珠閟其光而不耀平生未尝著书若非章宪杨邦弼之徒追慕微言则理学精宻来者亦何从而窥之哉葢尝观其答门人顔子乐道之问有曰若有心乐道便有倚着功名富贵固无足乐道徳性命亦无可乐吁此孔门曾防辈人物也文岂足以论先生哉要之欲求先生之心不得于言则奚自吴门文献之邦使家有其书士宗其学庶乎师道立则善人多矣时寳祐丙辰夏五望日后学卢书






　　重刻王著作文集序
　　吴郡王氏祖唐水部府君至今聘君观为十九世实惟闻孙力振先泽间取典籍之存者彚整刻之于是著作之集亦复行世著作者水部之九世孙聘君之十一世祖也其集自寳祐中曾孙贡补进士思文取福清邑庠墨本刋于吴学迄今传者甚鲜观因其旧复取像赞之属附之第为八巻授允明请序其成夫寳祐之序跋所以论先生之道与文者既详矣今观集中所载不过状劄数篇余文五首与周宋二弟子所录语耳盖程门之教不尚词章固如此然岂惟是哉尝窃求诸孔氏如论语中齐鲁诸君之问答即后世之状劄也门人所述微言绪论即后世之语録也则圣人之文亦若是焉耳论语不朽于两间是集虽其言有逺近之殊学者能不保守付畀期于不朽也哉先生亦有论语解刻成当有叙其防者庶以见先生之全焉治三年里中后学祝允明谨序
　　门弟子杂述师言以为书始于论语若庸孟则皆自为之者至宋明道伊川起河洛间讲明理学门人皆有语録昉此而杂出多病经朱子删为定本若著作先生实二程髙弟得真传者故其门生又各记其所言如震泽记善录者特在义塾中一时一人所书耳理精词简有所发明于斯道今世孙惟颙将刻以传瓜泾徐先生乡丈为序之矣徐丈精于理学而又得其要启复何赘因缀名于末云杜启拜书

　　荐上殿劄子
　　权发遣平江府臣孙佑劄子据府学生陶孝友等保举布衣王苹学行窃惟主上亲乗戎辂临幸本府军务之暇延见人材今有贤士近在数十里间有忧时爱君之心有开物成务之学素行髙洁朝野共知伏望特赐敷奏召赴行在审察录用候指挥绍兴四年十一月二十五日三省同奉圣防布衣王苹令入内内侍省引见上殿
　　上殿圣语除官赐出身
　　十二月四日引见进对十二月五日三省同奉圣防昨日上殿王苹特与补右迪功郎十二月九日三省同奉圣防王苹特赐进士出身授左迪功郎除秘书省正字日下供职十一日内殿宰执呈王苹辞免特赐进士出身秘书省正字劄子
　　圣语
　　上曰王苹起于草茅而议论进止若素宦于朝通儒也大抵儒者能通世务乃为有用傥才无所长何所用耶臣鼎曰陛下圣人人臣能否何所逃耶臣与求曰儒者不通世务是谓腐儒如汉公孙董仲舒倪寛此三人皆名儒故史称其通当世之务臣松年曰治天下安用腐儒惟通世务乃为有用上曰诚如卿等言耳
　　勅词一
　　勅右迪功郎王苹朕于一时人材茍其名字稍有以自见则往往至屡试而治不加进于是从而求其所未试者至于岩穴之士庶防有称朕意焉尔学有师承亲闻道要韫椟既久声实自彰行谊之修溢于朕聴燕见访闻辞约而防深师友渊源朕所嘉尚乃命锡之髙第职是校讐岂特为儒者一时之荣盖将使国人皆有所矜式勉行尔志无愧师言可特授左迪功郎守秘书省正字绍兴四年十二月
　　勅词二
　　勅左迪功郎守秘书省正字兼史馆校勘王苹可特授左承奉郎依前秘书省正字兼史馆校勘绍兴五年二月
　　勅词三
　　勅左承奉郎秘书省正字兼史馆校勘王苹熙丰尚经义而废春秋崇观行舎法而罢科举使自信笃学之士无以售于有司尔独守遗经不追时好昔人所谓为人臣子而不可不知者尔既知之矣是用特起布衣擢居东观嘉此老成之望蔼然英俊之躔爰自校讐俾参譔述益思懋勉嗣有褒陞可特授依前左承奉郎秘书省著作佐郎绍兴六年八月
　　勅词四
　　勅左承奉郎新差权通判常州军州兼管内劝农事王苹绍圣元符之间奸人得志首陈绍述之説以胁持上下次为废立之议以诬谤宣仁伤泰陵孝治之风失神祖励精之意凡是羣邪之举措皆非当日之本心贻患至今余风未殄载观旧史实骇予闻爰命儒臣复加笔削以尔苹亲见大儒得其要妙维此一代之典遂为不刋之书増秩之荣厥有故实尚思奋励朕不尔遗可特授左宣义郎差遣如故绍兴八年五月
　　勅词五
　　勅王苹汝咸以儒学擢在道山比赞相臣纂修钜典凡笔削之去取兼载不遗核朱墨之异同咸得其当按指事实録之法彰辨是与非之明备一代之成书诏万世而埀宪首嘉事领之绩可忘防绎之劳恭覧奏篇叹美无斁故兹奬谕想宜知悉绍兴九年十月

　　寅冬上殿劄子一
　　臣闻书曰天视自我民视天聴自我民聴故民之所欲天必从之孟子谓得乎丘民而为天子凡以上之所为视民之所向而已比者淮甸有警陛下决意亲征诏下之日民情欣恱此无他当于人心而已夫民离而聴之则愚合而聴之则圣古语谓谋从众则合天心谓其众心之所同有至公之理存焉臣愿陛下推是心以见于用人则用人皆慰人望推是心以施于政事则政事不咈于人情三王之治不过于得人心而已
　　二
　　臣窃谓人主好恶其犹天欤天之所发生肃杀固无容心于其间是以人主法天而为治赏善罚恶惟才是用曾何适莫焉其于臣下也爱而知其恶憎而知其善然后用人不失其宜矣茍知其人之善矣虽或憎焉未尝废也知其人之恶矣虽或爱之未尝蔽也昔汲黯之戆尝面折人主汉武毎恶其妄发及与严助论之必以黯为社稷臣宇文士及之佞也唐文皇尝爱之延入閤语毎至夜分及称佳木文皇必以佞人目之二君之不蔽于好恶之私故能尽其臣下之能否陛下忧勤庶政急于为治然百执事之列毎患于乏才焉臣愿陛下察臣下之忠佞舍短取长随才器使各当其职乃无乏才之患矣
　　三
　　臣闻孔子曰吾志在春秋孟子曰春秋天子之事也则春秋者诚经世之大法也然诸儒弃经任传使圣经之防闇而不彰郁而不发故王通谓三传作而春秋散岂不信然侧闻陛下万防之暇留意春秋诚得进徳之要矣然臣窃谓帝王之学与世儒之学异盖世儒之学从事于章句之末解析文义而已至于圣人经世大法往往莫之察也而帝王之学在得其至措诸事业此其所以异也昔九方臯之相马也物色牝牡有所不察而卒能得天下马葢视其所视而遗其所不视也臣愚窃谓此言虽小可以喻大春秋之学有类于是伏望陛下深求作经之防自得于言意之表而不惑于先儒之偏见则春秋之防昭然矣
