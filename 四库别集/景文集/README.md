　　【臣】等谨案景文集六十二巻宋宋祁撰祁字子京开封雍丘人天圣甲子与其兄庠同举进士累官翰林学士承防諡景文事迹具宋史晁公武读书志谓祁诗文多奇字证以苏轼诗渊源皆有考奇险或难句之语以今观之殆以祁撰唐书雕琢劖削务为艰澁故有是言实则所着诗文博奥典雅具有唐以前格残膏賸馥沾匄靡穷未可尽以诘屈斥也陈振孙书録解题又称祁自言年至六十见少时所作皆欲烧弃然考祁笔记有云年二十五即见奇于宰相夏公试礼部又见称于龙图刘公葢少作未尝不工晩年特为进境至于陆机之谢华启秀韩愈之陈言务去以为为文之要则其生平得力具可想见矣祁曽戒其子无妄编缀作集使后世嗤诋今据陈之强原序所称知在嘉定时安州守王允初等曽将二宋集合刋但序内仅称其言八十余万未经析及巻数祁本传称集百巻艺文志则称百五十巻又有濡削一巻刀笔集二十巻已与本传不符其余如范镇所作神道碑马端临通考则俱称百五十巻曽巩隆平集王偁东都事略陈振孙书録解题及焦竑经籍志则俱称百巻郑樵通志则止称七十八巻唐庚序则称二百巻因集已失传记载互殊莫详孰是陆游集又载祁诗有出麾小集西州猥藳蜀人任渊曽与黄庭坚陈无已二家同注今亦不传近人所辑北宋小集中有西州猥藳一种乃从成都文类灜奎律髓文翰类选诸书采辑而成非原本也兹就永乐大典所载荟萃裒次厘巻六十有二虽未必尽还旧观然名章钜制谅可得十之七八矣祁兄弟俱以文学名当时号大宋小宋其兄庠遗集已从永乐大典采缀成编今祁集亦于蠧蚀之余得以复见于世虽其文章足以自传实亦幸际

　　圜丘赋
　　若夫天地之区既奥而腴王者所以作京焉神明之隩匪攻而筑上帝所以定位焉我朝之拥归运也讥函镐保界之陋鄙周雒渟瀯之渊乃据梁之芒芒侦河之浑浑画邦畿之千里于以宅天子之尊然后翼翼干干作邦孚先禘其祖之所自出兮遂有事乎昊天占国南之七里得髙丘之崛然自干防之初辟保坤灵而不迁藏伟兆于遐叶震元符于兹年此烈祖所以裒神之对神宗所以旅物之蠲真考之所陟降丕后之所周旋蔼列圣以烝衎总万灵而宾筵翕降监之厚福焯巍巍而亡原则晋攷卜乎委粟汉肸蠁乎甘泉曽不得望我之末光絶炎况并驱而齐肩哉敢问圜丘之状也其何如矣广矣大矣畧可详矣上崔嵬以郁律兮外博敞而神丽遡朱鸟以髙蟠兮概瑶魁而衺峙休气回复乎其椒兮荣泉滋渗乎其趾魑魅不若泯伏于其逺兮神明肃然离卫乎其迩于是攘之辟之其菑其翳修之平之其坎其畷上三陔以积髙外四门而疏陛列道糊赪重营界紫无缩版以作劳不借罽而昭侈因天质之自然非人力之攸致崒兮似髙山之在周邦嶻焉若隆脽之亘汾澨及夫涓日肇祀于郊之宫陶匏尚质金石有容璧奠缛以苍苍兮鼎歊云而隆隆百神服食蔓衍乎坎间兮有司守燎粲烂乎壝中穆穆天子相维辟公咸盛气以彊力相升降兮穹崇披大紫之莫莫招翠黄之雍雍合萧芗于钦祡曳髙烟乎璇穹塞天渊以隤祉奋光明于无穷竣乎已事罔有不恭若乃自内出者无匹不行自外至者无王不止故我率乎祖而推本正乎位而升配使礼动乎上则神飨乐交乎下而人喜毕九州以献力罄一纯以尽意君子曰观天下之物无以称其徳所以因天事天取至诚为贵则斯丘也实国家集福之清场事神之寳畤国聴之所凭厚灵心之所翔会驻魄寳于飚歘贲万图之方志彼草楼列仙之馆像设王之庐豚蹄种祠之托鳞长九渊之居皆祠官之细祀族之余尚且落成者鼓吻而极叹乞灵者防笔而争书叛宣父以语怪溺丘明之好巫独圜丘岿而遗美宁儒者佁儗而未之思欤遂作颂曰屹圆坛赫昈昈大盘盘兮君之升帝是飨巩而安兮礼无违福不回委如山兮圣继圣万斯年长监观兮
　　陈州瑞麦赋【并表】
　　臣某言伏以厥田惟上界王国之右藩以谷俱来告我牟之纪瑞事昭邦縡美溢农书恭惟皇帝陛下茂拥蕃祺恪经善物弄汉田而劝啬揉神耒以训勤扈有九官务尽耕耘之法地杂五种深防水旱之虞屡勅攸司劝登宿麦至诚上达嘉气下翔发为两岐告成八政田祖有神而徯应守邦作绘以来图庖穗孔蕃天心为豫并诏儒馆交掞颂章良史必书无谢归禾之命升歌大备遂髙多稌之诗臣位属冗闲辞流澁讷隔从臣之品无预奏囊效游童之謡亦均嬉壤冒闻黼几集惧严渊谨夙夜斋戒撰成陈州瑞麦赋一首随表上进赋曰冠三辅之上者莫迩于陈接五谷之乏者孰先于麦当乗离之令序挺降麰之瑞殖盛气云郁混鳞隰之初霏密穗金繁动星田之霁色两岐旁秀六穗牙出厥华芁芁厥颖栗栗田畯奔告守臣骇观伻来以图悉上送官他谷弗书视麦禾之最重吾王攸助知稼穑之惟艰沬北爰采罔劭乎力农关中益种无闻于锡祉讵若天极归贶神明效异偕蕃椒之盈升配命禾而合穗迎层宙之休气冠中田之嘉谷绘我于瑞图辨我于凡菽防至尊之渥恵播新声于弦次上可以荐清庙之馨品下可以助外饔之食剂
　　皇帝后苑燕射赋【并表序　案仁宗本纪不载后苑燕射事此赋首云执徐统嵗当是天圣六年祁改国子监时上】
　　臣祁言伏闻今月九日幸后苑诏宗室赐宴射弓者气肃商旻神清汉籞展懿亲而合宴讲王射以侑欢常武是经良辰饰喜恭惟皇帝陛下靖嘉民极勤训道枢钧众志以惟精乂永图而无逸夹辅蕃屏立爱于周亲夏屋权舆示慈于镐饮乗九秋之盛集修六耦之帝容终日射侯不出正而命中嘉宾式燕实厥篚以奉将洽公姓之乐胥竦臣邻之咏叹多仪有焕在古防偕臣位属冗闲业専讲肄久陶景化窃耳大猷敢萌诵圣之心仰企强名之作感縁中发虽极抃于四支音以叩烦更増羞于五降谨夙夜斋戒撰成皇帝后苑燕射赋一首缮写随表上进
　　臣闻羲图肇画诸暌取星弧之象周文弥郁大乐着天驺之节谅以四方为志本男子之事一张作武震王者之威自秦制专私汉道云杂徒存燕射之旧礼罕闻明辟之亲御惟魏祖畋南皮之野一日射雉获六十三头史笔诧其妙文成避灵丘之区仰山发矢髙四百余丈琰刻旌其处是皆殊絶之诡遇本非揖让之多仪摅其善经徯我嘉会恭惟皇帝陛下载膺周之陈锡绍放唐之丕律髙拱岩廊之上丕冒环海之表百执之骏奔走无或旷官四方之大和防不知何力于是外朝夙退杪秋告丰庆重合之阳名申式遨之夏具披虎落之囿会麟定之亲始讲射侯之容将明洁觯之礼若其天威粹健帝表胖正雕弓栗笴俨若有容栖鹄画正砉然命中欢声囘商琯之肃余劲助霜旻之髙宗室奉觞以次称夀佥谓天挺其武所以辅文之缛事本于礼所以缀乐之谐盖陛下财成大猷冠卓千古是而弗纪有司之过也失言以文逺功由赋宣不有作者将何述焉其词曰执徐统嵗少皥司秋月着授衣之令日纪吹花之游僊盘露变神阙云浮田畯告茨粱之获乡童启豫助之讴上乃留瞩良辰载懐茂苑呼跸复道回舆帐殿望葱郁于麟棷披隂岑于虎圈跄鳯遥遡相乌细转凉翠下长杨之宫暝紫入棠梨之馆千门洞接九幄霞连周邦则伯叔之懿汉家则肺腑之贤相趋规地杂簉琼筵俨飞绥而交弭森委佩而凝鋗川醪溢泛骑炙纷传湛其露而在草需有云而上天先是司射谨官梓人庀具饰虎进侯张貍设步上干下干之交维三乏三正之并树排崇旌以虹拖画广堋而云布并夹并储决拾相旅我物孔扬我仪用光剡矢荐于夏箙雕弓出于扶桑彩五施而列晕歌九节而升堂万乗来御羣心乐康随月上弓与天张姬周则多材之主汤乙乃甚武之王有翼有严不戁不竦舍拔则拱应而中主皮划其已彻劲羽饮而犹动骇踆乌于日域震封狼于星塜彼艺有六徒得乎剡注参连我善兼五不専乎兴舞和颂掩前辟以擅名表上天之锡勇于是皇襟兑悦协气云蒸少府之赐钱流地庖人之割肉如陵示慈则折节加俎式宴则鼓瑟吹笙轩霞蕃后磐石宗英美发的之希代咸奉觞而造庭似雝泽之翔振鹭若辰居之环列星原夫射之为义是谓观礼外则审固而正躬内则雍容而绎志支左诎右之能项发口纵之异或合宴序宾或算多与祭公用隼而射墉观如墙而扬觯中则得为诸侯争有似乎君子然特人臣之善容未识我皇之射意若夫修五常为弓矫百度为矢总黎献以树的罄绵区而除位参天地之控引顺隂阳而张弛一发则英雄在彀虚抚则蛮夷防臂岂徒鹄为小鸟取难中之名驺曰仁兽乐庶官之备而已哉是日太师阕乐酒正彻觞辰旋回指归于未央
　　上苑牡丹赋【并序　案赋系天圣七年祁为国子监直讲时上】
