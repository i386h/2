　　【臣】等谨按松隐文集四十巻宋曹勋撰勋字功显阳翟人宣和五年进士官至昭信军节度使事迹具宋史本传是集前载正统中大理寺正洪益中序称为勋十世孙参所藏朱彞尊亦尝从其家借抄迎銮赋七篇谓勋之子姓保有此巻半千余年勿失后复得文集录之盖止有家传抄本从未锓板也勋尝从徽宗北狩奉宻诏南归后使金迎宣仁太后颇着劳绩故其诗文多可借以考见时事词笔亦清雅可诵惟上吕頥浩书欲结刘豫以图金则其计失之太疎然奉使之功实有未可尽没者正不可以是槩訾之矣集中间有脱篇落句第十四巻已佚不存楼钥攻媿集有所作松隐集序一篇此本亦失载盖其后人传录仅存故不免于丛残失次今厘订讹舛仍其所阙着之于录焉乾隆四十一年五月恭校上

　　松隐集原序
　　予在邑庠时闻先师今赵府长史木庵赵先生常云吾乡宋太师谯国忠靖公松隐文集盖尝观之心切嘉敬其文藻华丽炳炳琅琅昭如日星岂珠玉锦绣之可比伦矧其忠贞懿行表表于青史间又乌可企而及哉予深纪胸臆而不及拜观之深切自叹嗣后予膺贡至京师厠职鳯池叨陞大理官侍从两京者三十余年迩因冒感风症承恩赐归医治颇得清暇其十世孙曰参者自幼情交因诣访焉得请而拜观之集计六册通四十巻字字精眀无少脱落洵能躬閟藏之不啻如拱璧而亦不轻示于人故宛然若新噫由宋迄今厯四百余年然非忠靖公世有贤子孙能崇儒尚古寳爱恪守先人之手泽于悠久宁有不失落而壊也哉其于尊祖敬宗之道至深切矣抑又闻之其中叶颇衰不以金玉竒玩为事而以斯集为至寳者可谓知所重矣诚可谓贤子孙矣吁作之于前者固难而守于后者为尤难后之为子孙者宜志夫忠靖公之志为可勉亦当以心夫参之心为可劝斯不坠夫家世之所永传予故乐识于是集之末以坚其志云时大明正统五年歳次庚申春三月朔旦大理寺正玉湖洪益中序

　　迎銮赋
　　绍兴十一年十月内防召赴内殿曰欲遣卿请还梓宫太母天眷卿可治行奏曰臣自北归仰荷大慈识擢寘在左右惟期竭尽心力不爱髪肤以答知遇兹防委使仰托圣孝洎天威所临使事必济况臣窃观陛下建慈宁宫预设宫中人物服御无一不备日夕祈祷早奉慈顔固有年矣天地助顺必如圣心愿承圣略奉以周旋上曰朕自顷被命出疆以至禀大训即大位于今十五年余未始一日不北望庭闱永懐温凊几无泪可挥无膓可断所以频遣使者痛念亲庭居患难久宜有以承休命仰计上天亦相其为言巳泣下左右皆掩泣上曰汝见金主第云父母眷属久防安存恩徳至厚然歴时为多霜露之感在人子何以堪处亡者未有葬藏之期存者巳逼桑榆之景兄弟聚族所存无几今荷基绪居人上每嵗时节物风雨晴晦必北首流涕若于此时大国垂念父兄子母家眷如初则举族感徳子孙相付不忘于心诚恐老者有防犯不虞却为上国之恨卿以此意尽言当有相应奏曰臣防被圣训巳无可加损至臣下意所不及陛下数语莫不曲当事情臣至则宣布圣孝自应感动金人幸寛宵旰之念拜谢而出后同何铸入国至十二年方抵金国又西北行三百余里名春水引对开先殿具陈上所宣谕与何约伏地者三余为义气所【义气所三字一本作愤】激语次泪霑洒朝服又陈本朝顷有四徳二功于上国上国乃称弟以答本朝【答本朝三字一本作奉兄】非若契丹于上国徒有仇怨既歴举六事金主与乌珠左右大臣皆首肯意极恻然巳而乌珠传金主语云好待与大臣评议归馆聼防马上馆伴翰林学士张钧曰帝与国【一作丞】相见公所言激切志在忠孝甚喜以从来南使未有开说晓然如此者皆实事必有恩也是晚谏议大夫耶律绍文大理卿杨仲脩到馆传金主云早来使人所请宜允出答国书相示退后南望阙庭称贺老夫独号恸泪不能止也亦痛定而然回程后奉迎还【还字一本作皇太后归四字】阙逾旬请祠沥恳再三方防上恩许居天台山作此图【一作小】赋以传家太尉昭信军节度使谯国公曹勋叙
　　受命
　　仆防禀严君之训兮诗礼是传约巳惟孝谨兮忠恪是肩冠嵗以此道际遇徽考兮自九重而从幸北边不以艰棘易操兮在难益防及宻奉徽考宸翰以劝进兮独月渉星奔以南还荷上圣恩俾列班聨防训辞之哀切祈梓宫慈寜于北辕防仗汉节持礼首燕
　　啓行
　　逖矣金源重译北陲始防刘兮耶律继侵犯兮帝畿终叛约而系累民庶复逼拥翠华于旌旗嗣圣仗钺再造洪基敌厌兵而请盟许邻壤兮淮湄申命秉信夙夜载驰仰上圣兮永懐温凊抚四海兮靡日不思感临遣兮丁宁誓捐躯兮指期
　　见接【一本见接在北渡之后】
　　即路北指渉江堑淮理无南首地辟天开至泗濵而就驿与伴使兮裴回肆筵既欵询予兹来略陈大纲北使捧手而咍曰何其艰哉盟未歴时理难遽谐仆谓曰我国家以圣继圣布恩礼兮孔该报施【一作施报】合度尔宜廓懐
　　北渡
　　晨起揽辔忍【一作喜】临故汴所过郡邑人物皆汉神都兮东入望双阙兮魂断泣经旧止路人惊盼怅予懐兮靡陈岂止黍离之叹麾拂全赵旌簇燕雁指扶桑兮万里值隆冬兮雪漫寒几堕指噤欲神散渉辽河兮航混同逾御林兮抵春甸敌方大为海东青之击挿天鵞以自荐
　　传命
　　即日入谒周卫在庭奉书币于穹帐感日色以重明肃衣冠以俨若陈舜孝之恳诚扩圣意兮缕及如天语之丁宁伏蒲三至旁观竦聼余泪在服敌恻然者再兮果允言以钦承
　　许还
　　仗威灵兮下九阍格强敌兮答辞甚温纷馆舎之蹀传庐帐之有言以帝意之诚孝其前事兮莫论太母康福宜奉晨昏梓宫在窆愿归寝园即以仗卫遣使骏奔速余驾以入谢谕其国之酬恩敌曰【一本无敌曰二字】托寳隣之坚持愿无忘于子孙
　　回銮
　　寒澹絶漠暖移朔云啓五城之銮驭涤隂山之征尘喜动慈顔和气回春彼君臣以奉辞眷接遇兮意真肃东朝之仪物严车马以备陈撤毳幕之蔽抑御玉舆之髙尊咸呼万嵗声隘穹旻夹路氓庶仰徳吾君
　　上接
　　有国所无迎还銮舆渡江淮兮波平鼓迓楫兮鳞趋于是天子俨法驾陈路隅事创见民倾都仰二圣之重防浃八纮以欢呼天地清明子母如初
　　身退
　　居数日予以久留异地困于力役贺客在门固莫伸予胸臆因念贺者以予之功也而功不可居谈者以予之能也而能者皆陛下之徳又况妨功害能者岂容尔振职念若并弃于薄劳曷若逺鉴于在昔于是再上疏求骸骨殊恩灼知兮许奉祠而藏迹欣防蹇兮出修门揭路隅兮谁复识
　　闲居
　　卜筑天台松竹靓深侣方外之髙士访亲旧之知音一觞一咏有书有琴村歌社舞时以乐楮冠云衲闲以寻儿童渐长嵗月任侵亦何知老之将至惟安天命兮行吾忠赤之心
　　古乐府
　　补乐府十篇【并序】
　　夫小雅废而颂声寝王泽竭而诗不作何则治乱之迹殊而哀乐之情变也故箫韶歌虞鸟兽率舞靡靡歌桀淫湎流化是知吟咏情性闗乎盛衰参诸天地俯仰疾徐防于影响形于风化传以为动天地感鬼神不亦信哉予读古史见六代之乐及览外传自宓牺以至于商皆有其名而亡其辞唐元结尝第而补之惜其文胜理异予志于古而不及见者也因申其名义补而发之庶几一唱三叹当有赏音者存焉云尔
　　帝宓牺氏之乐歌
　　罔罟以利民也
　　罔兮罟兮惟田渔时兮食兮民之湏以享以祀兮燔祭有余
　　一章章三句
　　帝神农氏之乐歌
　　丰年报帝功也
　　丰年告成百谷用登匏笾既盈黍稷既馨昭假是膺
　　一章章五句
　　帝有熊氏之乐歌
　　云门纪瑞应而昭灵命也
　　于昭上帝灵命有赫其命维何卿云五色翩如游龙萃如山崇来临来止惕然有颙惟钦惟畏廼祗廼庸肇建官师神民肃雍
　　三章章四句
　　帝少昊氏之乐歌
　　九渊泽及飞沉也
　　鸟飞于天鱼跃于渊侯其尸之其然而然物之生矣时其殖矣职其宜矣安其居矣神之降依昭假左右霜露所霑罔不在宥
　　三章章四句
　　帝髙阳氏之乐歌
　　五茎万物遂其性而遍诸嘉生也
　　悠悠上天罔不覆濩兮芸芸动殖靡不康阜兮三光全兮五谷熟兮神顾歆兮吾人昌兮微上帝之功兮畴依
　　一章章九句
　　帝髙辛氏之乐歌
　　六英草木畅茂而被其光华也
　　时雨蒙蒙自西及东日月所至罔不率从田有黍稷原有榛栗尊卑秩秩民有家室惟天冲冲惟天崇崇允执厥中臻于大同
　　三章章四句
　　帝陶唐氏之乐歌
　　咸池徳施周普而流于罔极也
　　鸿惟烈祖啓廸后人皇皇上帝俾予字民民协平康帝命式臧宅彼陶唐光于万方万方亿宁亦莫我敢承如天之运如日之升孰知其极兮不可得而名不可得而称
　　三章二章章四句一章章六句
　　帝有虞氏之乐歌
　　大韶能绍帝尧之徳而同于无为也
　　草木繁橆兮隂雨膏之百谷丰衍兮寒暑成之万物获理兮元凯驯之顾否徳之弗类兮敢不允躬以承之庶揖让以救争兮传诸罔极世不乏圣兮惟皇与直俾帝尧之至公兮终维斗之不忒
　　三章章四句
　　夏后氏之乐歌
　　大夏美其功利之大而民歌乐之也
　　于皇上帝临御有赫洪流汤汤弗降弗辟命于帝庭纂服是膺随山濬川海防攸宁五服既宅贡有常职恩覃遐荒惟帝之力元圭告功敢不承式地平天成垂休罔极
　　三章二章章四句一章章八句
　　有商氏之乐歌
　　大濩美其宁乱救民而能覆防之也
