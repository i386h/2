　　钦定四库全书　　　　　集部四
　　提要
　　尊白堂集六卷　　　　别集类三【宋】
　　臣等谨案尊白堂集六卷宋虞俦撰俦字夀老寜国人隆兴初入太学举进士累官兵部侍郎奉祠卒其行事不见于宋史而志乗所载颇详始为绩溪令即以治行被荐迁监察御史搏击权贵朝廷肃然为浙东提刑徙知湖州值嵗祲推行荒政所全活甚众盖亦不徒以文学见长者集中有使北回上殿劄子是又尝衔命使金考金史交聘表泰和元年三月乙亥宋试刑部尚书虞俦泉州观察使张仲舒等来报谢即其事也俦慕白居易之为人以尊白名堂并以名集其读白乐天诗云大节更思公出处寥寥千载是吾师生平志趣可以想见故所作韵语类皆明白显畅不事藻饰其真朴之处颇近居易而粗率流易之处亦颇近居易盖心摹手追与之俱化长与短均似之也然如除日狱空春蚕行及劝农祷雨喜雨诸篇恺切慈祥词意恳到足以验其心劳抚字固不当仅求之吟咏间矣集中古文仅存制诰劄子二体已不免多所散佚而辞命温雅议论详明于当时废弛积言之尤切其意亦颇有所取者据陈贵谊原序集本二十四卷今従永乐大典中采掇裒次厘为诗四卷文二卷录而存之亦所谓布帛菽粟之文虽常而不可厌者欤乾隆四十六年三月恭校上

　　原序
　　绍定己丑秋八月武冈守朝请郎虞君衡遣介奉一编书谒余序其首启而视之曰尊白堂集乃其先人兵部侍郎讳俦字夀老所作也卷二十有二词制奏议之卷七诗之卷十五衍雅重根柢经术绪宻思清非苟作者独未谕尊白之意何谓也侍郎平日常曰韩魏公有醉白堂坡公有我似乐天之句此吾所不敢惟知尊之敬之故扁以为堂吁安得此长者之言哉宜其词章粹厚不迂有乐天之遗风也唐史臣賛乐天谓其最于诗今侍郎诗亦独到读之者知其为庄士雅人所谓畧无杜牧之所讥者顾乃扁而尊之可谓亷于取名矣歴事三朝出藩入从始终全徳使获尽展所蕴功业克就则赓醉白之歌又将与魏同志而惜其不尽着也武冈守继家学政和惠知其亲之善显而传之有足嘉者抑尝因是以观白公之为人位不登乎丞弼功不着乎鼎彛而百世之下殊勋伟业名徳文学之彦率愿尚友慨慕而不可得是其可尊者必有出于文章事业之外覧者遡其原焉可也朝请大夫权刑部尚书兼直学士院兼修玉牒官兼侍读长乐陈贵谊序

　　尊白堂集卷一　　　　　　宋　虞俦撰
　　五言古诗
　　暇日邀王天任诸公游南坡天任有诗因次韵
　　梅黄收宿雨山緑呈层巅髙轩防修竹朱槛俯流泉雅致轶天外俗物无眼前笑人声利场蛾眉竞争妍半生政坐拙州县空逥旋自视甘寂寞何心慕华鲜南坡吾莵裘杖履有厚縁暇日领佳客颇办沽酒钱愧无丝与竹后堂欠彭宣何以侑谈笑頼有兹山川送鸿烦逺目落鴈惊虚门径逥俗驾山泽非臞仙剧谈交玉麈醉吟飞彩牋榴花媚红皱李实荐翠圆相防恣谐谑小谨畧跽拳兹逰无可纪须君赋长篇
　　再和
　　我诗苦艰澁蹇歩太行巅笔干砚欲裂何地覔涌泉巧匠愧傍观袖手不能前敝帚徒自珍捧心岂成妍辟舍不获命属鞬试周旋君诗巧层出浥露晨花鲜前軰借波澜后生敢并縁平生程不识毁不直一钱五色烂摩尼八音和相宣挹袂庾开府拍肩扬盈川毎发必中的矢脱无空我盖三复之飘飘意欲仙此计太冷淡浪费五彩牋不如醉红裙且聴歌喉圆况当荷倾盖莫待栗过拳徐防骑鲸伯一斗挥百篇
　　再和
　　雨骤频垂脚山髙故隠巅南坡疑断路平地怪流泉猿鹤谁为伴鳬鸿亦堕前梅黄虽借润榴火漫呈妍人世隙驹过天形磨蚁旋云容愁暮合霞彩望朝鲜已负杖藜约宜修香火縁江山如得助风月敢论钱今日苦不乐此情谁与宣髙歌懐杜曲晚赋拟樊川屋破时悬溜弓惊不受弦好邀沽酒伴却羡泛舟仙竹色杯中緑花光笔下牋左持还有蟹右画更成圆自笑牛刀割难安鸡肋拳补天非我事聊尔继新篇
　　再和
　　君家有西园望我南山巅觞咏继曲水花石希平泉长沟流砌下逺岫列楹前春风一披拂百卉为渠妍迨兹夏日永杖履宜盘旋梅黄已过雨清润霁景鲜主人虽好事欲往无由縁似闻三径深日夜长苔钱行当载酒过髙情与君宣飞鸿渺天末浴鹭来平川良辰恐易失勿待月下前者坐中防俱是桂堂仙酒豪倒金壶诗健轻华牋琴防朱古碁枰寒玉圆君才尤中的霜鹘无空拳谆复记幽防舂容防大篇
　　和姜总管喜民间种麦
　　腰防刈晚禾荷锄种新麦田家竞作苦羔豚劳嵗夕翠浪惬初懐黄云期后获晴暖得天时肥瘠随地力賸欲观风謡未暇履阡陌寜唯饼饵香庶免鞭笞责拟踏青州麴酿酒邀佳客
　　病起据案无绪辄书二十五韵呈簿尉
　　祗役来异郷疟鬼巧伺便寒热四五行大小六七战初如蹈隂壑赤身卧氷霰继若煽红炉虐熖工煆链先生但坚壁髙卧观物变谓我则忘吾曰舍其犹传童子更进药先生强起咽羣鬼竞揶揄医工何瞑忽然汗四出心愧顔有腼湏臾返清凉轻健欲舞抃寜唯防枕簟便觉美肴饍气和疾自去福至祸亦转我生多蹇剥县尹剧卑贱王事有期程私室敢安宴日出事还生孤防无一援讼牒犹山积愚拙费裁剸催科正火急日夕竢诃谴督邮顾小忍斗米那可恋弹琴不复梦据案敢告倦仇梅二友生洒落邦之彦颇复相料理何尝语逰衍城南万荷花浣溪锦一片凉生雨初过香度风防扇碧筩亦易办白堕傥可荐诗以代折简辞兮非黄绢除日狱空唯欠租监系颇众因悉纵遣之期以开嵗五日毕来因记东坡先生倅杭除夜直都防囚系皆满题诗壁间輙用其韵
　　讼牒日已稀私幸庭无留胡不了租税顾为此拘囚催科迫星火对案亦包羞作苦念田家嵗莫得少休羔酒防邻里盍归与妇谋刻期亦人耳吾其反自修
　　病后有感作
　　今旦病思清把书坐晴檐体舒百息匀心逺万虑恬收视等庖郢妙观到羲炎顿忘饥寒苦但觉中邉甜习气尚稍稍自喜良沾沾故家东堂书髙架纷悬籖我时且儿子气欲俱并兼浇着胷次间政尔意不亷此意竟蹉跎衰迟侵防髯迩来又留滞郁郁三年淹是秋莭物髙露白原草霑斜河转空濶夜气方森严此日足可惜黄尘满征襜明年防当归赤米荐白盐繙经慰老眼甘作书蠧潜共嘲尚白指笑突不黔亦知屠沽儿羔豚方属餍
　　和张伯子韵
　　士流竞弹冠州牧纷劝驾器识在所先文艺乃其亚朅来同抱闗因得奉余暇软语杂嘲谑小酌见蕴借官禁不通风门键那容罅豹斑纵未窥鲸背几思跨君来怪已三我到嗟方乍前头万木春当户三星夜弃甲勿复来背城寜一借新诗扺骊珠归橐何妨诧
　　咏麦
　　平生笑盗儒口诵青青麦刻烛和新诗此夕知何夕六花来未晚三白防有获雨旸亦天事勤惰繄人力何妨纵牛羊况已遍陇陌青黄未交时补助岂无责我亦念归田聊作数旬客
　　和姜总管送别
　　富贵如浮云于我初何有转徙浙东西仅足糊其口去年在吴兴民物犹未阜赈荒事方殷奚暇诗与酒今年到东阳政成不待乆邂逅遇诗仙襟抱得细剖切磋愧子贡起予幸鲁叟琮琤苍玉佩闗键黄金钮拟学西子颦益愧邻女丑赓酬强追逐要是非敌手目短曹刘墙气轹鲍谢囿我如武成防苐可二三取山泽多蛇龙松栢无培塿缅懐南涧老斯人骨已朽人事真好乖又假苏台守遽辞山水邦暂缺云龙耦酒罢雨沾裾船开山入牖似鹤出樊笼如鱼脱窠臼分袂却闗情鼓棹犹囬首长篇肯赠我字字炳星斗愿保嵗寒心莫愧金石友
　　和寜国守沈察院九月严霜十五韵
　　九月十月交卦应西北干少昊犹执矩冥遽行权庆元敦牂嵗夏无雨旸愆早禾既有秋人斯望有年晚稻亦懐新頴实相接连不间东南畆寜分左右肩严霜忽早降节候无余暄黄熟乃无日青干空在田殆不如稊稗那敢咏好坚造物岂弃遗繄时失后先农夫亟欲诉有司不谓然课无责之有何异扼其咽使君恻兴念官租为渠蠲作诗寛彼忧至哉仁人言幸甚宣城民信乎有二天
　　冬至念母
　　儿行十里程母心千里逐昔者子曽子不忍离一宿今我独何为旅食乆辇毂来时月中秋坐待阳初复薄命难推挽阔歩窘邉幅慰藉烦友朋栖迟愧僮仆画饼唐生相説河严子卜世态方奔竞人情虑反覆夫岂不懐归所恶在欲速眉夀翠涛春洞庭霜橘熟亦足尽亲欢何必此防禄
　　秋日同官逰圣感寺晚饮范公亭分韵得须对二字
　　招提城东隅胜集僚友俱腐儒元自臞午饭厄斋盂不如计一醉况有髙阳徒小秋颇解事鸡鹜营庖厨我归与妇谋但问酒有无妇言有斗酒待子不时须
　　木末好风来山头凉月胐快哉城上亭登临吾亦再堂堂文正公精神凛犹在杯行且停饮四座俱起酹义气激余衷豁若挹沆瀣此夕饮也清且无俗子对
　　次早再出
　　晨起侯鸡鸣我车催发轫列炬犯星防衣襟防雾润将迎恐后期奔走追前进拭面愧缁尘垫巾怜雪鬓吾年已如此世态何当徇曷不归去来荣枯才一瞬
　　秋雨叹
　　少昊欲行秋骄阳犹防扈炎蒸不易涤颇费天公雨浸淫三日霖云气政旁午所虑早禾伤佳实卧场圃翻思向来旱南畆生尘土矫首望云霓一滴如膏乳及兹收敛际反以多为苦苍苍匪天欤恻怛羣物祖怨咨尚未免是非何所付防当返秋阳新晴照区宇禾实趂登塲不特盈我庾荷担赪肩归亦足慰田父黄鸡能渐肥红酒幸可酤持用答丰年造物傥复许
　　和张文潜有所叹韵
　　老骥困盐车四足若有绊鸴鸠抢榆枋控地不能逺我本田家子释耒事防宦少壮耻干请老大谁推挽抚字亦徒劳催科寜趣办扵心傥无愧此首何妨俛财由政事足轲也岂好辩苟得而富贵孰若甘贫贱
　　和陈倅留题奉圣
　　飞鸿渺天末目送类中散谪仙负诗狂直欲摘星汉白云千丈峯金碧美且焕忘言老瞿昙祗作空花观轩冕傥来耳世事亦其漫鸾凤在赤霄寜肯就绁绊惟余忧国心抚事輙悲怨饥馑有流离十室九不防上动黄屋念毎日不遑旰公岂爱登览防意寓篇翰山僧遽传似字字如珠贯劝分岂无术未必愧鞭筭升斗倘均恵沟壑庶少逭福星公恐是畴复虞嵗旱王事敢告劳行役春过半鬓丝禅榻畔纶巾时一岸溪山得胜赏歌舞废屡按请颂海沂康民謡兹可翫
　　和陈倅逰石门吊瞿硎先生
