　　提要
　　【臣】等谨案俨山集一百巻续集十巻明陆深撰深字子渊俨山其号也上海人治乙丑进士官至詹事府詹事兼翰林院学士事迹具明史文苑传是集为费寀徐阶文征明唐锦陆师道诸人所论定而其子国子生楫为编次而刋行之锦及师道并称尚有外集四十巻通此二集为一百五十巻此本不载外集皆其笔记杂著又自别行也史称深少与徐祯卿相切磨为文章有名书仿李邕赵孟頫赏鉴博雅为词臣冠徐阶称深以经济自许在翰林院在国子监数上书言事督学于晋防藩于楚旬宣于蜀则皆有功徳于其士民而惜其独以文章见费寀亦称其以剀切不谀忤宰臣左迁以后畧无感时愤俗之意而举其发教岩诗峡江道中诗证其无所怨尤今观其集虽篇章繁富而大扺根柢学问切近事理非徒鬭靡夸多当正嘉之间七子之派盛行而独以和平典雅为宗毅然不失其故步益亦可谓有守者矣乾隆四十三年九月恭校上

　　瑞麦赋
　　仆闲居田野多见瑞麦两岐三岐至五六岐彼九岐者得于传闻殆未之见云实有之感兹休祯作赋一篇有颂有美有风有刺义主劝戒附于古诗人之谲谏虽不足以希踪相如子云庶东京之流亚也示我同志靡得而布焉
　　天子正徳五祀孟月维夏知知子疡发下体更朔新愈有客唁焉登堂三揖乃掀髯吐论曰夫物有异产事有竒遭谈肉者不可与论味眯采者不可与即文濶哉希乎今兹之所覩也子足良苦抑未之知乎知知子蹷然而起危襟横几奉客下风曰唯唯愿客诏之客曰走故农家五谷是理爰自弱龄勤厥四体今年逾知命而丰穰凡几一畆三石稂莠吐米皆未若今嵗之为瑞也麦苗芃芃穟岐为二扬芒含颖复为三四多者五六将将覆地东邻一茎九岐尤异殆淳和之所薰蒸而上帝用以钖类也周书异畆汉歌两岐陋昔人之夸诩昭后皇之惠慈虽蒙白之翁负之老皆縁畆玩视相与嗟咨若走者齿发犹盛渉歴未广宜乎惊悸而伙頥也知知子仰屋太息索然久之曰否否客何谈之盭也夫縁物者贵质敷文者适用且夫麒鳯之希难以疗饥芝菌之祥难以充庖云锦之烂难以御寒蠙蚌之光难以续膏是以圣明抑难得之货壅不稽之言诚以重本而缓末弃无益而即有用也客幸目覩岐麦津津称瑞民瘼甚矣果谁之致独非客之所观见者乎试为客语徃嵗已已运厄元元夏耘被垄滛雨注天昼夕阅七飓风相牵海波怒而山立江潮喷以骏奔蛟龙舞于街衖冈阜沦为漭渊漂尸横野浮畜蔽川千里一壑万灶絶烟于是百年之完聚连邑之生全化为鱼鳖葬于鲸鳣者殆过半矣暨乎水退民失故居沧桑一变形胜都非朱门沈其阀阅碧瓦荡为丘墟鸟窥巢而不下狐访穴而重疑号哭振野提负沿途父弃其子妻别其夫相与转徙乞丐奔逐投依若流星之逼曙而败叶之辞枯也于是强有力者牢朽材于古岸塞行潦以腐薪依湿林为栋干缀败席为闼阍溷爨无别卧食不分什并为五弃仇讲邻相依为命枵腹连旬野无留菜树不遗根徼幸于万一茍活于旦昏尔乃积隂郁结隆冬盛寒层冰千尺竹栢枯干岂祝融之故都为防之停骖何暧暧之阳国顾风烈于塞垣民无夙具习不素安于是受冻而仆者又如干矣天子方轸念南服融照闾阎发徳音大王言贷常赋辟四门封简书于芝检勤使者于輶轩省太官之供调减司冦之坐论赈仓廪之储积蠲逋负之浩繁方将夺民命于沟壑续生气于游魂盖三五之罕有而二气所不能全之旷恩也良有司方忧经费之不足惧考课之殿后鸿泽持而不下限令疾于电走朝四莫三示一藏九使民破十家之产仅足以输一家费数畆之田未足以赋一畆笳鼓盈村扰及豚狗尔乃制为严刑迨及黄耉巨木囊头重金系肘臀无完肤指欲堕手谿壑之填未厌伸暴之门何有于是孑遗之民疮痍之末毙于敲扑困于征科者盖沦胥以尽渐不可久矣戾气醖酿蒸为疫疠方且乘阳发腾犷不可制今枕藉而病卧者比比皆是招医降巫若愦若醉是其冻馁蚀于胷肠刑罚惨其心志发双伏而并攻何方药之能治厥祸方萌殆未知其所至也使垄畆之植一本而十岐共蔕而百穗将安救之而客夸以为瑞哉且菽粟所以贵于珍鼎布缕所以加于黄者为其有用也今夫用者徒存而用用者已亡是谓隆虚而病实忘逺而娯细矣仆窃为客赘而不取也客闻而怃然曰噫嘻有是哉子之迂也信乎执一者未足与权二泥彼者不可与适此子徒鉴于已已之变为流而不止乎是殆滞于隂阳之迹而未深于斯理者也且夫于穆之化圜运不已剥终必复泰因于否吉凶互寻祸福相倚夫虫之蠕蠕也不屈不发地之窿窿也不伏不起数逆斯通气顺乃死是故九载之水或以成尧七年之旱终以啓禹子岂知夫凶俭之后继之以丰稔登进之渐承之以君子耶然而气机橐籥必有攸始兆先于物发逺伊迩钟鸣而陨霜础润而降雨走诚得于俯仰之余是以释近忧而崇逺喜也且夫麦备四气实首五谷诗人颂其于皇下氓赖以率育续嵗功于发春函潜气而多淑是休嘉之先露徯斯理于将复示帝心之仁爱啓方来之祉福谅有开而必继孰无征而廼获走且与子托丘壑以优游咏皇风之清穆是故有取于麦岐子何责之备而论之刻【叶】耶知知子不能难客乃蹑履而退曵杖而歌歌曰麦秀兮多岐覆垅兮累累彼其之子曾是兮弗思于是知知子返乎潜室沈思渊黙纬情愫于浑沦抽端绪于开辟推化之始终考休咎于遗册览春秋之所书测消长于三易道有殊而归同理既契而心戚然则客之言似亦未为失也将以厥明戒馆客循阡陌辨麦岐之疎数听邻党之损益问勤勚于三时吊疾苦于缓急于时风轻景融烟朗雾清谢雕舆却繁缨被大练策溪藤道以童子从以经生指三汀以东骛遵龙江而缓征瞻桑梓于原隰拜松楸于佳城睇海氛于极际俨波浪之奔轰感钓游之旧踪慨嵗月之不停蔼里墟之蔓延有孤物而屡更亦浮云之多态何难树而易倾怅久寄于异土心恋恋乎故京方徘徊以瞻眺謇彷徨而屏营顾见道左麦穗岐岐本同末异旁无附枝始防防以竞秀竟娈娈而莫携将神工之妙合复化钧之巧持或双昻以森矗或左右以纷披或越畎而希挺亦共房而骈垂薄长飇以洄洑照圆景而陆离等比翼于异类嗤连理而不为固物薄而禀厚腾众喙以增竒胡哲人之超轶随所如而见疑昔宣尼之瑞鲁匡人恠而围之比干之忠殷曰不祥而戮尸彼二圣且犹然般罹此又何辞抑轩轾之偶致将彼苍之有知于是歴览既倦羲驭未疲揽厥颖异采掇以归洵皇泽之渗漉拯黎民于阻饥托子墨以宣秘聊洋洋以陈词
　　南征赋
　　空同子阳明子同日去国作南征赋
　　亶肃肃以宵征兮悲徃路之未央惧中道之折轴兮思改辙又恶夫无良顾仆夫以先后兮喟河广之谁航瞻桑梓之翳翳兮孰云忍捐夫故乡方青春之骀荡兮何雨雪之纵横白日匿其耿光兮郁浮云以翻扬睇山川以无极兮陵谷杳乎其孰明【叶】祥狐嘷而风厉兮何有于哕哕之凤凰昔宣尼之遑遑兮固蒙笑于楚狂展直躬以事人兮卒三黜乎旧邦慨殷室之多贤兮王子剖而信芳茍璞玉之终在兮虽屡刖又何伤谋人之国兮焉有祸而弥藏覩巨盗之乘垣兮固将谒之以峻防谓余梦寐之颠倒兮岂敢幸其必当黑白之同体兮盖昔焉之所常惮婵娟之翘妬兮吾安忍刓夫清扬雧絺绤以御冬兮畴驾尤于寒凉狎逆鳞而批之固以不碎为庆【叶】也斥虎之使逝兮遭反噬未为殃也卬衷之洵安兮初未量乎得丧【叶】曩委羽于东海兮奚成功之可望矢贞心之不泯兮濒九死吾犹恇昔淑媛之见背兮竟结发之难忘怅恩情之中絶兮徃将洒扫乎室堂彼良农之俶载兮力刈乎莠与稂诞嘉谷之离离兮竟收功于千仓度中流以失楫兮岂俟共载而劻勷燕雀安于焚栋兮斯物知之不长服先哲之明训兮希旁烛之煌煌神龙之渊天兮谅所乘之允臧歩中夜以顾瞻兮睆牵牛与七襄永相望于咫尺兮庶精诚之可将仰天阍之九重兮冀羲驭之回光魂怦怦以上征兮謇徘徊而彷徨乱曰桂车兰轩服骐驎兮登高临深送征人兮怀芳握馨遗心亲兮瞻望弗及涕泗零兮
　　吊刘生赋
　　唐士刘蕡燕产也感激殷忧有蓟丘易水之余烈以直言废故有祠祀直言也在今昌平皇明正徳之六载中月南至史官陆深共承陵祀道出祠下顾瞻徘徊乃述赋焉
　　洵刘生之瓌伟兮已亮夫言出而祸随诞树虚而贾实兮纷振古之共悲测彼苍之倚伏兮嗟生独罹乎此时也当明庭而铺词兮既已谢乎无媒也茍忠信之自卬兮又奚必开金石而称竒愿韬巻以有徯兮恐嵗暮之难期凌冰雪以北度兮敬吊生之芳祠空苹藻于寒沍兮怅瞻遡之何迟吁防合其犹然兮羌昩已而前之塞河决于微穴兮当狂澜之既穨貔虎逸而负嵎兮欲徒手以径批临深渊而莫戒兮耸旁观之屡疑佩宣王之昭训兮亦有道而言危曰余既知夫隐衷兮冀身却而道垂谓将来之可恃兮竟陈言之谁施謇歴生之故墟兮睇西山之崔嵬寻采薇之旧踪兮路逶迤而多岐风号寒而木怒兮恍有遇于斯须情抑郁而欲语兮魂靡靡以难持傥决机于转圜兮曰独生之所私
　　后灔滪赋
　　昔苏公子瞻赋灔滪盖曰江防百川势易骄逞不先之以龃龉尽其快鋭为害斯大嗟乎是诚有之夫当国家丰亨豫大之时必有风靡波荡之俗使无正人法家出气力以扞之则末流有不可救者矣此公作赋之旨也嘉靖丁酉二月初予将出峡舟过瞿塘春水未生孤根欲露盘旋其下有感于心作后灔滪赋
　　道丛鳬之故国沿岷汶之长源睇灔滪之兀嵂绎先哲之名言测安危之倚伏乃始疑而今信然昔伯禹之导江兮命阳侯为驱先挟六龙与二虬兮劈雷斧之神铦劙地脉而中分兮挺孤高之一拳障犇轰之东骛兮回万折于澜翻时盈缩以浮沈兮拥百川而独尊或如象兮如马舟人睨而不敢上下相逝者之汩汩兮固昼夜之不舍惟上帝之黙佑兮恒斡运于区中示机缄于错纠兮物有穷而必通猗宣父之感麟兮既伐树又絶粮羌邹孟之仁义兮竟见沮于臧仓胡贤圣之迍邅兮夫岂兹堆之未汰派逺而弥昌兮象下流兮永赖抱遗经而先觉兮返乱治于否泰顾大化之茫茫兮俟万世于须臾俯中流之一柱兮盍前鉴而蚤图慨孔明之驰驱兮磊磈乎八阵犹未磨问草堂于瀼水兮吊江上之东坡予粲粲其白髪兮惧末路之蹉跎
　　宣悼赋
　　陆子羁旅两都三年之间四哭子女天永地厚控诉焉如作宣悼赋
　　岁靡靡以于迈兮嗟继续之未昌曰予邅回之可哀兮何岁三改而四殇背乡井以辽邈兮囊又告予以屡空谓彼苍为不仁兮先哲重之曰降祥将余戾之日积兮懵不知其内讼抑修短之防合兮长辛楚又奚用登高台以舒啸兮凄北风于襞缝涕既掩而复陨兮云坌兴而四涌岂余目之眵眯兮心恫而不舒俯危防以侘傺兮恍何物之满裾回曲房以偃息兮若有闻于欷歔顾父子之至情兮孰云排攘而获疎谓昬为旦兮谓日为年胡懽愉之易终兮而忧戚不可以延谅后皇之终惠兮恐日暮于虞渊反予袂以自持兮承慈顔于堂前
