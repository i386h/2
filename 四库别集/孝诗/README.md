　　宋

　　提要

　　（臣）等谨案孝诗一巻宋林同撰同字子真号空斋福清人祖瑑直宝章阁父公遇字养正荫补宁化尉以不忍违亲乞奉祠服阕调户曺不就自营精舎以居扁曰寒斋同其长子也与弟合字子常俱有隠操后元兵至福州抗节死之林希逸鬳斋续集中与其兄弟赠答诗文甚多又有同所作壬申酬倡集跋今其书已佚惟此集犹传前有淳佑庚戌刘克庄序皆摭古今孝事毎事为一五言絶句一首亦间有两事合咏为一者其变例也凡圣人之孝十首贤者之孝二百四十首仙佛之孝十首异域之孝十首物类之孝十首其间如唐李迥秀之类本为佞幸葢以一节取之至寒山子煮爷煮娘之类则爱无差等之谈録之不免于驳杂然大旨主于敦饬人伦感发天性未可以其词旨陈腐弃之况其人始以孝着终以忠闻虽零篇断什犹当珍而惜之是固不仅以文章论矣案宋史忠义传称林空斋失其名已为失考又误以空斋为同之子分一人为二人并云空斋举进士歴知县考克庄序谓同兄弟俱未脱白又尝曰年未四十慨然罢举是同固未尝举进士也又宋人刘麟瑞昭忠逸咏称林处士同是同固未尝任知县也且麟瑞诗内有血书矮壁存吾节气贯长虹任汝烹句又与史之啮指血书壁自誓不屈诸语相合然则死节者为同而非同之子也明甚宋史殊为失实又福建通志人物传称林同字子真公遇子元兵至不屈死隠逸传又称林仝字子贞与弟合并有隐操至元中郡府上其行谊诏征不就合字子常云云其说又异然同合之俱为公遇子已见原序同字子真合字子常已见鬳斋集俱确凿可信通志既以死难之林同为公遇子则隐逸之林仝不得又曰弟合字子常也且改同为仝改子真为子贞并属点画音声之变明为传写讹谬分为二人至曺溶学海类编亦载孝诗一巻题长乐林同季野著名同而地与字俱异又不知其何所据矣

　　孝诗

　　（宋）林同 撰

　　尧

　　道之大原出于天尧以是道传之舜尧舜之道孝悌而已矣
　　道原自天出尧以是相传曰孝悌而已人人具此天

　　舜

　　大孝终身慕父母予于大舜见之矣
　　孩提知所爱妻子具而衰大孝终身慕子于舜见之

　　禹

　　禹伤先人之功不成劳身焦思八年于外　启呱呱而泣予弗子惟荒度土功
　　伤心九载绩焦思八年中启亦呱呱泣惟思度土功

　　汤

　　葛伯不祀汤使亳众徃为之耕奉先思孝
　　奉先思孝也不祀岂其情以至使亳众而甘为葛耕

　　文王

　　朝于王季日三鸡初鸣至寝门外问安否何如日中至及暮又至亦如之
　　问安寝门外旦旦俟鸡鸣以至日中昃居常尽此诚

　　武王

　　文王有疾不说冠带而养文王再饭亦再饭疾间曰女何梦梦帝与九龄
　　经言事父孝所以事天明疾间加再饭梦中与九龄

　　周公

　　我之弗辟我无以告我先王辟马郑音避谓避居东都　东山周公东征
　　我避故云辟东行是谓征先王犹可告岂为伐而刑

　　孔子

　　子曰所求乎子以事父未能也
　　事亲良不易战战复兢兢学得如夫子犹言丘未能

　　颜子

　　谓子路曰去国则哭于墓而后行返其国不哭展墓而入
　　返必展墓入去当哭墓行谁知颜氏子事死乃如生

　　孟子

　　幼被慈母三迁之教长师孔子之孙子思
　　如何年稍长汲汲子思师知是为儿日三迁感母慈

　　右圣人之孝十首

　　契

　　契为司徒教以人伦父子有亲
　　天性生虽具人伦教可无徃哉亲父子汝契作司徒

　　元凯

　　左传季文子曰孝敬忠信为吉徳髙阳高辛氏有才子谓之八元八凯
　　当知元与凯不出孝而忠能举十六相斯为舜大功

　　孝已

　　武丁之子庄子云人莫不欲其子之孝孝已忧而曾参悲
　　子则欲其孝人皆有是心如何武丁圣却使子忧深

　　微子

　　微子之命曰尔惟恪慎克孝微子去之子曰殷有三仁焉
　　岂计身封宋祗愁祀絶殷不惟称克孝亦自有三仁

　　泰伯

　　季歴有圣子知太王欲立之逃奔子贡曰太伯端委治呉仲雍断髪文身
　　为怜季有子逆探父传贤断髪文身事诬哉司马迁

　　康叔

　　元恶大憝矧惟不孝不友子弗祗服厥父事大伤厥考心乃其刑兹无赦
　　勿云服父事不敬亦无伤元恶与大憝邦刑自有常

　　君陈

　　惟尔令德孝恭惟孝友于兄弟克施有政命汝尹兹东郊敬哉
　　为尔有令德存心孝且恭推之施有政徃矣尹兹东

　　伯竒

　　尹吉甫子履霜操云无衣兮儿寒无食兮儿饥儿罪兮当笞逐儿兮奚为
　　无食又无衣逐儿安所之不辞履霜苦犹谓罪当笞

　　晋文侯

　　平王锡文侯秬鬯圭瓒曰汝克昭乃显祖追孝于前文人若汝予嘉
　　嘉汝故锡汝非将玉展亲克昭乃显祖追孝尔文人

　　卫武公

　　年九十犹日警于国曰必朝夕以交戒我　又孝子之身终
　　孝子身终者非终父母身武公年九十戒我尚谆谆

　　齐孝公

　　伐鲁展喜犒师曰岂其嗣世而弃命若先君何君必不然齐侯乃还
　　岂其伐与国独不命先君想得还师去难堪此语闻

　　齐侯无野

　　晋人欲质其母对曰若以匹敌则亦晋君之母也又汉太公事
　　我母亦君母吾翁即若翁分羮何太忍请质可堪从

　　申生

　　姬泣曰贼由太子人谓太子子辞姬必有罪曰君老矣吾又不乐乃缢死
　　姬泣贼由子子辞必罪姬为忧君不乐寜死莫君知

　　晋文公

　　寺人披伐蒲公曰君父之命不校校者吾雠也踰垣而走披斩其祛
　　伐蒲君父命校者是吾雠披直寺人耳鉏麑信有谋

　　秦康公

　　晋文公甥渭阳诗云我见舅氏如母存焉又曰我送舅氏悠悠我思
　　懐哉送舅氏曰至渭之阳念母如存意悠悠寄二章

　　旣醉君子

　　孝子不匮永锡尔类其类维何室家之壸厘尔女士从以孙子
　　若为锡尔类不独善其身既念化家室还思遗子孙

　　四牡使臣

　　有功见知则说矣四牡騑騑岂不懐归王事靡盬不遑将父母
　　懐归将父母四牡苦騑騑是我心中事君从何处知

　　南陔孝子

　　南陔孝子相戒以养也晋束晳补亡
　　为养还相戒南陔昔有诗都縁这个意焉用补亡为

　　白华孝子

　　白华孝子之洁白也
　　亷耻有微缺须知孝未纯白华何所似所似皎然身

　　北山大夫

　　役使不均已劳于王事不得养其父母焉王事靡盬忧我父母
　　非止不得养如忧父母何能忍不我念使我独贤劳

　　蓼莪孝子

　　民人劳苦不得终养哀哀父母生我劬劳欲报之德昊天罔极
　　莫遂终养志难酬罔极恩哀哀父母句膓断不堪闻

　　妹土小子

　　其艺黍稷奔走事厥考厥长肇牵车牛逺服贾用孝养厥父母
　　勤耕艺黍稷逺贾服车牛奔走何为者心乎孝养忧

　　老莱子

　　年已七十父母犹存着斑斓之衣为婴儿戏于亲前
　　七十已中寿人生似此稀絶怜老莱子犹自作儿嬉

　　栁下惠

　　三为鲁士师曰何必去父母之邦
　　忍于父母国去此欲何之所以栁下季三为鲁士师

　　子思

　　孔子喟然而叹请曰意子孙将忝祖乎伋毎以不克负荷为大恐子忻然
　　如何方喟叹乃尔复忻然念汝祖无忝知予道可传

　　闵子

　　子曰孝哉闵子骞人不间于其父母昆弟之言又曾子母投杼事
　　参寜杀人者三至尚踰垣叹息子闵子孝哉无间言

　　有子

　　其为人也孝弟而好作乱者未有也凉州易乱课人诵孝经盖勲谓取怨
　　其为人也孝好乱未之闻曾以是取怨无人语葢勲

　　曾子

　　有疾召门弟子曰启予足启予手而今而后吾知免夫小子
　　全归有名训珍重恐伤肤小子启手足吾今知免夫

　　子路

　　见孔子曰今由也累茵而坐列鼎而食愿欲食藜藿为亲负米不可得矣
　　食藜殊列鼎负米异重茵不待今朝养伤哉昔日贫

　　子游

　　问孝子曰今之孝者是谓能养至于犬马皆能有养不敬何以别乎
　　犬马何以别斯言似太深深言为养者可不敬于心

　　子夏

　　既除丧而见孔子与之琴弹之而不成声作而曰哀未忘也
　　为有心中事琴音自不和礼虽弗敢过哀若未忘何

　　子贡

　　曰赐也愿息于事亲　又曰大哉死乎君子息焉又大块息我以死
　　未须论大块以死息乎人所愿学赐也姑为息事亲

　　林放

　　问礼之本子曰礼与其奢也寜俭丧与其易也宁戚
　　故宁戚以俭罔或易而奢致得千载下人称问礼家

　　樊迟

　　问辨惑子曰一朝之忿忘其身以及其亲非惑欤
　　当知一朝忿必不忍于人岂止身忘却还应及乃亲

　　阙党子弟

　　阙党之子弟罔不分均有亲者取多孝悌以化之也
　　旣渐夫子教罔有不分均何复间多取齐言彼有亲

乐正子春

　　侍亲疾复加一饭则脱然愈复损一饭则脱然愈
　　向非亲在念念念毎精专未必饭加损能令疾脱然

　　公明宣

　　学于曾子曰宣见夫子之亲在叱咤不至犬马宣说之学而未能也
　　子于亲在日叱咤不闻声既曰见而说应无学未能

　　孔子党人

　　孔子曰吾党之直者子为父隠直在其中矣
　　忍以证羊故而云我直躬那知为父隠直故在其中

　　孟庄子

　　孟庄子之孝其不改父之臣与父之政是难能也
　　无改父之道真堪以孝称若论庄子孝此直是难能

　　孟懿子

　　问孝子曰无违樊迟御子告曰生事之以礼死葬之以礼祭之以礼
　　不复问无违祗应懿子知尚愁人不省以礼告樊迟

　　孟武伯

　　问孝子曰父母惟其疾之忧
　　父母之爱子心乎惟疾忧可能不自爱而忍使亲愁

　　管仲

　　曰吾尝三战三走鲍叔不以我为怯知我有老母也
　　战败不羞走谁为知我深叔知我有母亮我走时心

　　晏子

　　曰使吾父之党无不乐生者使吾母之党无不足衣食者
　　母家令足食父族使安生睦党可谓孝能之惟晏婴

　　晋君子

　　下从征役不得养其父母父母何怙父母何尝悠悠苍天曷其有常
　　何怙复何尝衘悲问彼苍苍苍不可问重使我心伤

　　颍考叔

　　留羮感郑庄公母子如初见左传茅焦諌始皇即虚左迎母见史记
　　茅焦甘伏质考叔请留羮大隧如初乐咸阳虚左迎

　　魏孝子

　　役于大国陟彼屺兮瞻望母兮狄仁杰登山望云曰吾亲舍其下
　　望母嗟陟屺思亲徒望云可怜魏孝子不比狄参军

　　卫七子

　　凯风美七子能尽孝道以慰母心小弁小人之诗也何以言之曰怨
　　几谏劳不怨明明载鲁论未应小弁作得并凯风言

　　石祁子
