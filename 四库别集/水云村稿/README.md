　　臣等谨案水云村稾十五巻元刘壎撰壎字起潜南丰人水云村其所居地名因以自号生于宋嘉熙庚子咸淳庚午本郡试第一入元以荐为延平教授延祐六年卒年八十所着隐居通议颇淹博可取巳别着于録其文集旧有二本一曰水云村冺稿乃明洪武间孙瑛所手抄大抵杂采隐居通议中语缀辑成编其文亦皆与此本相一即此本乃其裔孙凝収拾遗佚别加排次者所収各体较为详备惟原目二十巻而今本祗存十五巻其十六巻以下已阙当由传抄者失之特按原目此五巻所载本靑词祝文无关体要之作则其存佚亦无足重轻也壎才力雄放尤长于四六绮丽之制集中所载诸啓劄大都皆在宋世所作考隐居通议自述其得意之笔如代吴浚谢建阃表吊吴浚文代赵必岊谢庙堂啓通丁应奎啓今皆不见于集则其散佚者亦自不少然即所存者观之下笔铸词亦复颇见精采壎尝自言赵必岊称其防以散文为四六正是片叚议论非若世俗抽黄对白而血脉不贯者虽不无夸诩之词而生平得力所在其甘苦固有防自道者矣至其他古文则入元后所作为多灏瀚流转亦殊有淸隽之气而间以俳句绮语搀杂其间颇乖典则不免稍逊一筹特宋末元初遗闻轶事时见一二亦足以资考据云乾隆四十五年四月恭校上

　　汉髙帝庙赋【有引】
　　丰西山水隈曲处多汉帝庙庙像多古雅里俗岁时致祭乞灵以为常按汉史帝生于丰起于沛战于河南都于秦雍师行所至多循江北暮年以来出师仅至于蕲则大江以西帝之马足固未尝辱其后庙祀相望无亦仁义公恕四百年黙有以深入人心者故世祀而弗忍防与吁三代而降政多以谲而不正庻防无愧者髙帝尔盛徳必百世祀固宜抑帝也仁明英武度越后王风起云飞威扬海内千秋万歳后黍离麦秀其遂栖神深山之孤庙受野老只鸡豚蹏之祭耶则千古英雄之志有触目而可悲者矣乃引而为之赋曰
　　陟山曲之坡陁兮有古松之蟠屈积楮镪之纷披兮见庙貌之突兀蛛虫络兮穨檐龙蛇绘兮败壁旒冕俨乎帝王顾劔佩之侍侧始欲谂其何神汔莫覩夫扁额则见瓦砾飘风阶除落照樵牧三五偃蹇言笑遂前询夫故老曰此汉髙帝之庙也于赫髙皇神武天毓韬光振采美石韫玉巳乃起丰沛王巴蜀战河南入函谷眇西江之僻左未尝辱夫马足顾遗祠之何多徧山隈与水曲将非汉德深厚汉泽悠长皇天后土鍳约法之三章除苛解娆宜万代蒸尝之不防嗟夫芒砀二界瑞采成龙泗上一亭天开帝宫挥靑萍于酒后夺威斗于月中时则霆迅电掣雾滃风从山石为泐草木失容集士马兮云黑列旗帜兮天红声撼河朔势赫川东羣狡众黠束手鞠躬开炎祀之四百后三十世而勿穷其威灵可谓甚雄矣哉方其定都咸秦控引伊洛函闗巩固陇蜀盘错观阙苕嶤黼扆煇烁万方球珙一统正朔宴未央朝长乐千官舞蹈六师罗络大常奏乐上公执爵祝帝夀之无疆谓与天而相若日月防何山川寂寥长陵无树烟雨萧萧彼寒食松楸之馁竟无人兮为浇则廼栖神古庙慰我黎兆乗彼白云去来缥缈葢星弁牢落而受农夫蚕妇之拜奠象貌凄凉而飨斗酒豚蹏之祭醮何昔盛而今衰俛仰悲歌不觉为之失笑也秋淸兮木雕屋老兮瓦飘石炉烬兮烟断土像穨兮榻揺日色薄兮岚雾冥鸱鸮鸣兮风怒号忆翦嬴而蹶项气葢世而凌霄噫而今果安在哉令人魂断而意消或曰有隆有替从古然矣未烦博采陈编泛引前事姑以近于汉者言之则前有始皇二世后有备权操懿莫不奋威逞防角力鬬智或豪举而席巻或交争而鼎峙极意气于一快骋诡秘以万计苐求子孙帝王之业不悟世变轮回之数始焉肩摩毂击云扰波沸已而声销景没缩首防志雄心敛于片木霸气沈于九地葢石火之晔乎易灭也槿华之飘乎易坠也其横骛絶出耸动华夏生平之夸豪竟不多史官数叶之纸信自古莫不皆然徒以増志伤情于一喟繇是观之威力气势排山倒海穷贪极暴讥讪诋诃然而莫防当数十年寒暑之鍜链日月之消磨虽合古今帝王之杰特无奈此元化之宻移何琼宫瑶台荡为浮烟飞埃矣恩浅惠薄岂复有遗庙之嵯峨其仰视髙帝弥足增感彼英灵有觉兮宁不心恧而神瘥乎天道人事终有止汔傥盛极而不反则必递迁而周匝惟道德之迓续延洪胜智勇之震詟凌压使当百僚朝防之赫奕而坐念千秋庙祀之毁残知兴废之靡常敛威力而勿殚委厯数于代谢置生灵于安闲虽不幸运去物改祚移鼎迁抑感恩怀德终讴思而永叹荒祠野祭正以着人心之腾欢而仁深泽厚廼于是乎可观则覩斯庙也且将庄敬思慕之不暇尚忍讥其祼荐之酸寒乎重凄慨而兴怀爰作赋以矢志请无非其深文期昭垂于永世
　　夀文堂赋
　　大元革命初兵祸频仍士学都废往往易节独刘水村翁确守素业如平时故扁其读书之室曰夀文喜其文之犹夀也
　　两仪肇判三才以分爰有精粹融而成文葢钟秀于海岳宜交瑞于乾坤其在天也星宿罗络燿灵煇焕雾湛烟霏之炳映云光霞采之绚烂其在地也机杼纵横锦绣川原峯峦之丹碧峙丽江湖之漪縠沦涟其在人也精神志气经天纬地大则纲常建立礼乐粲贲宫庙朝市之位官班防爵之制下至州邑以及闾里莫不仪物黼黻等威堂陛有尊有卑有都有鄙有钜有细有纲有纪施诸政刑秩然条理郁郁彬彬立厥统体次则制诏坦明号令传宣出丝纶而章日月走风雷而动山川悍将流涕而向化癃老扶杖而争先又其次则奇人秀士防餐失寐雕镂才情讨论经史或赓歌以诵太平或哀吟而慨兴废或争先圣之教于衰微或著书以俟百世其尤下者塲屋声华音调委靡猎科名钩富贵是区区者最不足齿然苟为之则亦斯文之一利也颛辛以前存而勿论葢自放勲登极之岁暨乎德祐移鼎之年以厯计凡三千六百三十有三矣文脉未始一日而不传噫河洛聿啓世变运殊元化粹精乆矣发舒天矜其凿欲夀以愚尔廼刋落其华燿蹙廹其纡徐翳黯其昭明芜薉其芳腴举数年之所尝发挥至是摧剥而焦枯于是泊然反素归于太虚意者犹六月之息将复开斯文之初乎盘桓康食几十载而余诸君其思非天夀吾文而何如或者夙修日寡新习改化典籍寇雠仁义土苴彼有怀跖心而术孔业立可家崇恺而位王谢息我天黥解我天劓若夫沈痼书癖夭阏文气惟吾中年之朋侣屈指东南而有几歳推月移磨灭谁继则飞走同归卉木接趾斯白日之行尸而黄泉之愚鬼也文且弗灵夀乃在是譬犹游方之士志求登僊周行万里出入霞烟神气瓠落足力茧穿觉其防失之已多乃块守乎泉石之间遗形骸而精固収视听而神全翛然羽化夀也后天呜乎天不忍斯文之防是以开夀域而演真诠予悟而喜揭扁于楣登斯堂者为予绎之
　　观雨赋【有引】
　　某歳夏旱惔焚不可奈一日歩江皋纳晩凉乃忽风云交作雷雨踵至予喜谛观因叹化机转旋在呼吸间一何易也将纪以诗客曰诗不如赋赋亦古诗之流且昔人未有因观雨而作赋者盍试为焉予曰诺作观雨赋
　　鹑火序次祝融涨氛垤鹳沈声畴龟露文暑浸肌兮若烝气塞膺兮如醺予意愦愦散防江村冀受晩凉消此黄昏则见水涸沙壅炎熇顽洞睨千林以招风嘅纤埃之不动予益愦愦而眊瞢矣盘薄顷之西望髙山微云冉冉出如穟烟蔽亏斜日荡漾澂天俄如萦缕广袤敷布又如车轮径南以去而南山之南亦见奇峰腾骧蜿蜒如虬如龙又径东以之北忽薲末之生风俄而风威渐劲云气弥黒浓淡砌叠如冈峦之横出倒景晃燿映天地而碧色尔乃排空如城倐忽流铃海光揺闪卷水南溟隂风雄号牛马悲鸣撼枯木扬飞尘彼有浴川登岸如鳬雁乱羣拽衣挟巾狂走兀唐似挥刃之欲逼亟投家而防窜予方徐登野亭冯阑倚楹观霮防与汹涌何瞚息而晦冥乃见疏林低枝声战败屋其着地者有如飞镞风益健而疾呼云挟雨于欲垂列缺挥霍霹雳轰飞俄而风力寖息西南转值乎云薄则网宻如线至若云屯正厚建瓴决堰又如水帘与飞练是雨也横嶂沿堤循山谿方直或似乎屏障弯环又类乎罣罳惟骤雨乃始若是葢浮云有以幻之瞻彼逺山白雾吐洒飘风霏薄与雨上下奇呈秀变忽化天巧所到虽妙笔而不防畵雨沾衣单氷凝露寒于愦愦以顿清忽翛翛而欲僊则咏曰此天下之奇观也余尝喜登峯以观雪月喜卧石以观云雾尤喜伊何临江观雨尝试思之风雨云雷交战并集神怪掀舞魑魅为泣众骇防而无人色我乃浩乎太虚以独立正于此观天地之变又何嫌乎好奇之癖心志激壮精神飞翩劔槊舞钲鼓喧如见乎赤壁走操淝水歼坚唐兵破夹寨之日周师防髙平之年山开石泐河倾海翻将非奇观之尤杰然者耶嗟夫天地焦枯金石俱靡陂泽欲竭黍苗已悴斯人旦暮已矣而昊旻至仁忍遂涸弃骄阳化霖旱虐一洗然则神工之转旋其易特反掌耳海宇之大黎庻之繁一亭毒于苍防宁俾暍死于是间葢穷则变通大易之义岂惟造化之仁将亦理之所必至吾因观雨而悟夫天道隂阳之机世道升降之际则怃然而起曰如斯而已乎一反掌之转移亦甚易也
　　养生赋
　　至元庚寅歳秋暑炽甚予患痢疾良苦既小瘳作养生赋以自警
　　维炎歊之侵薄兮幸苟免夫陨坠委瘦骨于一席兮亘中宵而不寐展转乏力吐纳短气葢五脏六腑之郁积实九窍百骸之腴腻穷日之力抉而去之故虽有存焉者寡矣情思眇绵支体废捐惟百念之顿清见一真之湛然则反而思之曰羣生赋形危浅脆弱外而皮肤内而筋膜非金非石易朘易削肺肝有似乎羽轻肠胃何殊乎纸薄偶一息之不续不崇朝臭腐而销铄嗟夫幻躯若是其不足恃耶而以此当天地之迁变亦甚岌岌而不觉雪风堕指霜硝裂齿百川凝氷隂毒入髓其酷暑也太空流火山木焦燎汚沱沸波六合烘窰吁可畏己乃以危浅脆弱之质敌此毒烈惨酷之气孰胜孰负可惊可悸矧夫思虑纠结忧乐摧折噭呼损肝劳苦伤骨或疾鞭其神魂或颠荡其气血乃若醇醪沈酣冶容斵防极口腹之无厌竭精髓于一畅是殆逃生于毒鸩血肉之中寄命于猛虎齿牙之上也寒暑迭冲情欲夹攻曾人生所息之防何乃欲以应夫无穷将见如电光之闪忽曷不为老氏之守中天地生化父母保毓葢至贵者人身难得者夀年安忍以其至贵难得者弃之空縁乎予既疾而获痊抚弱质而凛然愿清浄以养生庻永夀而或僊
