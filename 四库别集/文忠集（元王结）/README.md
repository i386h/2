　　古意
　　幽人捐筋力自斸十亩荒手持蟠桃核种之南山阳一年萌蘖生三年枝干长迤逦三十年美实期见尝客从都邑来衣马多辉光舜华欲分赠朝露含幽香笑我太逺计悠悠何所望
　　咏史
　　公孙乗丧乱窃号防巴蜀援也惟故人千里来不速陛防森以严賔仪繁且缛畧去乡曲情夸耀帝王服伏波盖代英豪气谢羁束一笑井底蛙区区事边幅驾言复归来玉趾真劳辱东方有美人展也刘文叔
　　孟氏命世贤抱道思经邦将隆尧舜业千里游齐梁终老无遇合迟迟尚徬徨鲁平何足云谗夫有臧仓燕石哂荆璧鱼目笑夜光良宝竟弃捐千载为悲伤
　　战国事富强好士争相先翩翩四公子賔客盛三千呉起既适楚乐毅亦归燕鸡鸣狗盗贱峩冠列賔筵孟荀绍孔业赫赫盖世贤伊谁共天职高论徒森然收彼腹背羽六翮反弃捐乱亡谅不救烱戒垂千年
　　汉祖握干符锐意除暴秦求贤冠台鼎萧曹唯故人叔孙亦世儒坐见礼乐新纷纷刀笔吏济济俱簮绅两生抱遗学独与世殊伦躬耕汶水曲索寞谁相亲
　　汉庭富衣冠流风重周旋君防往还人名位相后先宪府领举劾中台执机权门阑隘轮蹄交口称才贤亦有同门友穷居守太过门不一顾寜望相荐延富贵易交驩贫贱相弃捐翕翕夸毗子区区亦堪怜

　　平津未遇时牧豕东海壖白首抱麟经英名动菑川王明扬侧陋台席冠羣贤未覩经邦业龌龊徒备员迎合天子意首鼠持两端既弃仲舒才寜顾汲黯言居然刀笔吏谬以儒术縁阿谀负重任千载有余愆自古寜一死俯仰兴长叹
　　严光隠桐庐逺迹混樵牧鹤书屡搜访物色遍岩谷安车入洛阳天子来同宿狂奴饶故态伸脚加帝腹长揖金銮殿超然返初服世祖不爲屈子陵岂为渎公卿亦何人视我同草木
　　古意
　　穹宇孰纲维率土同照临端拱逰穆清重明烛幽深寛裕谅天覆发育惟天心仁义奠民极阳开复阖隂天人永相符至理无古今应兮系所感善败常相寻驾言适郊原四牡行骎骎鸿鴈号空陂鹰隼集北林长歌漫呜呜殷忧亦钦钦后防威鳯来如玉式如金浑浑开尘编渺驱烦襟
　　猖狂无盐丑自献投齐君绰约东家施俯首羞效颦蹇修称良妁采采华其身千金不一诺未易充下陈庭下春华敷门前秋草新幽兰生深林含香徒自珍忧心良悄悄薄幸空纷纷愿言金玉操歳晚无缁磷
