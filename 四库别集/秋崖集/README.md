　　钦定四库全书　　　　　集部四
　　秋崖集　　　　　　　　别集类三【宋】
　　提要
　　臣等谨案秋崖集四十卷宋方岳撰岳字巨山号秋崖歙县人绍定五年进士淳祐中为赵葵防议官移知南康军以杖舟卒忤荆帅贾似道后知袁州又忤丁大全被劾罢归其集世有二本一为秋崖新稿凡三十一卷乃从宋宝祐五年刻本影抄一为秋崖小稿凡文四十五卷诗三十八卷乃明嘉靖中其裔孙方谦所刋今以两本叅校嘉靖本所载较备然宝祐本所有而嘉靖本所无者诗文亦尚各数十首又有别行之本题曰秋崖小简较之本集多书札六首谨删除重复以类合编并成一集勒为四十卷岳才锋凌厉洪焱祖作秋崖先生传谓其诗文四六不用古律以意为之语或天出可谓兼尽其得失要其名言隽句络绎奔赴以骈体为尤工可与刘克庄相为伯仲集中有在淮南与赵葵书责葵驭军之失指陈切直不失为忠告至葵兄范为帅失律致襄阳不守所系不轻其罪亦复不小岳以居葵幕府之故乃作书曲为寛解载之集中则未免有愧词矣乾隆四十六年四月恭校上

　　山居十六咏
　　入山林处
　　穷涂一何恸多岐一何泣指似世间人路头従此入
　　幽谷
　　入林不厌深避世不厌独倦飞鸟知还毋惮下乔木
　　便是山
　　东臯或巾车西园亦飞盖以我方古人两脚大自在
　　石梯
　　山险者夷之梯以石齿齿于间遂成台有梅适堪倚
　　清樾
　　山林有灌木其大可蔽牛藓石眼清阴坐间天地秋
　　雪林
　　若以雪见我以林为吾家君其非诗人不能识梅花
　　小山
　　拳石以为山勺水以为池我观于世间何者非儿嬉
　　桃李蹊
　　竹迳转西麓山花能白红要知造物心政在无言中
　　归来馆
　　归来遽云出既出又归来自处茍如此渊明安在哉
　　着图书所
　　钟乳三千两胡椒八百斛笑杀山中人破纸塞故屋
　　草堂
　　烟低一片春留与贫受用阿连自能诗吾衰不复梦
　　锦巢
　　佳日春酣酣一色锦歩障谁知山中人如此富贵相
　　寒
　　子盍观于泉遇险维其止流行非所知一寒清到底
　　饭牛庵
　　宁与厓下人烟雨耕荦确读书亦何为谨勿累吾角
　　秋厓
　　凿空为此名忽已落众口聊结一间茅承当作厓叟
　　田园居
　　秋风新饭香蔬亦美可茹宁知世有人万钱无下筯
　　次韵方蒙仲髙人亭
　　此君欵吾室髙人跂吾亭邂逅两相逆保此嵗晩青茗盌漱藜苋诗肠许坚顽是间可无我不可无此山鸥鹭之与侣竹石之与居子虚亡是公付之渇相如苔寒两不借对面宁尔劳欲语二三子卑之母甚髙爱之不我疎狎则不我亲山耶其竹欤君子哉若人吾侪磊块胸所见不相逺借我竹外山一榻供昼偃烟霏日夕佳隠固胜于显孰従而诗之要亦山一蹇终南有捷径涧恧而林羞其人岂其山宁勿相累不相从文字盟不与如此酒他日有移文毋令猿鹤受老子可林壑诸公自烟霄缅懐归耕归底用招隠招以嗜酒爱风竹卜居此林泉为韵作十小诗
　　居千卿者三至九卿者四不博牛背眠咸酸信殊嗜玉虹横吾腰金貂直吾首宁知千载名不如一杯酒事有俄顷间遗臭及万代吾观于斯人不过一字爱东坡真天人落笔蓬莱宫何如赤壁一鹤横秋风周有说爱莲陶有诗爱菊吾居则谁与其诸子猷竹朝吾一笭箵夕吾一觳觫诗肠其殷雷此事底须卜秋云七尺藤夜雨一巻书寂寞空山人防此谁与居吾心如鐡石命则薄于纸借口见古人则亦赖有此彭殇同此躯顔跖同此心哀哉宇宙间羣生政林林吾犹金在冶惟冶者所甄谨毋出光怪夜半号龙泉以人生五马贵莫受二毛侵为韵送胡献叔守邵阳
　　灯寒岳麓雪帆饱湘江春过家谁起予岂无独醒人襄蜀莽万里战骨髙于城湖南宁几州耕凿偷余生民愚一何防民怒一何虎谨毋芟刈之持以报九五宁骑踏雪驴莫骤追风马霜蹄失衔勒多是快意者三十已専城斯亦未足贵潘舆归故乡吾乐宁可既维东有夕烽维西有秋柝行春独何悰此段待民莫餂以澜翻口箝以霹雳手俗吏訾儒腐吾曹自甘受吾道一而已惟诚一之至毋或参以三亦勿贰以二借筹渭阳幕邉画分毫毛邵州二千里能勿鉴寐劳迢迢烟水寒猎猎霜风侵此行何当还民瘼简帝心
　　此君室
　　匪书谁与稽匪竹谁与居以书造竹所俯仰千载余亭亭几长身斯可谓之士以我为隠乎指以二三子不肯一世人草木自宾友举杯问樵岚见有此客否可臞不可腴可独不可俗君看室中人今已不翘足曲士多在今节士多从古此君岂易得仅可一二数翛然数竿玉带以一溪水惟我有是夫孰能与于此泉石宁我癖烟雨宁我惛是间有佳趣不可无此君一迳自明月谁能此婆娑床头易未了如此良夜何竹意吾为谁山曰子非我未知回孰贤自賛午也可俗士不足语之子谁与娯闭门但痴坐尔辈宁可无
　　山迳
　　一迳净于秋山中夜来雨倚松不知还幽禽妙言语
　　湘源庄舎
　　馋虎过新蹄怒貍争旧穴悲哉两翁姥西山采薇蕨梦书十字史记册上太史公此书真是笔斡造化梦语非夸也因足之识于策
　　禹穴探幽眇神功接混茫絶麟才此笔春梦试平章
　　斑竹夫人
　　山谷青奴娯夜秋崖斑媛专房尔家婕妤失笑尔孙流落荒凉
　　演雅
　　蟏蛸网罗遗逸离留劝课农桑鸣蜩吸风饮露反舌吹笙鼓簧
　　苍鹰守株待兎白鹭临渊羡鱼鸤鸠妇叹于室蜗牛亦爱吾庐
　　赵玉汝自作戯墨各题六言其上
　　鹁鸠
　　村北村南雨暗舎东舎西水生去妇复还何日烟簔处处春耕
　　鹭鸶
　　雪以一江烟水风乎两岸菰蒲但得鱼矶无恙曲肱之外何须
　　脊令
　　水落寒塘初冷霜倒枯荷半倾莫倚尾能涎涎踏沈一片秋声
　　鸜鹆
　　晴树暖以流憩霜实饱而倦飞何与人间世事再三自整缁衣
　　立春【国忌谓之哑春】
　　适禁乐令春哑先办秋治社聋银胜鬓毛班白琼丝菜缕青红
　　谢人致蟹
　　樵岚无蟹有监州已负诗肠过一秋莫怪渊明便归去得渠自合老菟裘
　　此去江头只两程寒蒲解缚便横行无肠政要空虚腹满贮芦花与月明
　　雪砍双螯玉一壶樵中曽有此竒无始知张翰思归意不为秋风只为鲈
　　除却金虀霜后枨更无一物可诗情诚齐配以彭生脔岂不寃哉五鼎烹
　　送荔子方蒙仲
　　风枝露叶走筠笼玉润冰寒擘绉红自徃胷中评史记久闻格调略相同
　　曽识坡仙海上山清冰寒露洗熇袢老天不与诗为地却欠闽中住一畨
　　即事
　　自刈香秔税水舂旋篘新酒搦泥封晩菘种已无余地不枉归田学老农
　　赠诗僧
　　瀑泉癞可能诗最毕竟难为本色僧要识庐山真面目一溪明月冷于冰
　　从来佛法无多子底用芒鞋半九州赢得诸人诗满袖草深一丈法堂秋
　　蒙恩予祠
　　坚卧牛庵饱嵗寒恩书初许作黄冠孙僊彭老如相问筋力犹堪上醮坛
　　月明弄影雪巅癯只似胡僧不似吾忽予牙绯称羽客道官儒黻释头颅
　　祷晴福善
　　稻压田塍米似珠老农为说几年无一晴不费君王力莫等香烟上化炉
　　夜落檐花未肯晴灯寒等不到天明自怜短发垂垂老一滴秋霖白一茎
　　背犊腰牛忽觧顔摩挲一饱政相闗王于霓望无终极孰与侯邦年嵗间
　　题致堂新州坐石【辛亥】
　　苍寒瘴面新州石直不话头传不休秦负儒耶君误矣二书今与大江流
　　棱层不隔梅花岭万里相従识两翁却怪牧牛亭下路石麟咫尺卧秋风
　　九日墨紫开
　　九日年年负物华野香未肯到山家今年九日风光别不独黄花更紫花
　　中秋日社未五鼔祀社归月明如昼
　　月明如水带寒流饮福灵坛晓未収不是社公唤我起何縁一日两中秋
　　次韵杜监簿
　　风雨重阳又一秋白衣人至复何忧东篱只此黄花是未必渊明风马牛
　　社曲无邉风露秋瓮头酒熟了无忧一诗连夜穿兵卫略似齐城战火牛
　　嬴得萧萧两鬓秋山城斗大转多忧人间縦有万钱鹤身外何如十具牛
　　不须把浅不防秋底事山樵亦过忧说与诸公那解此但知两角战蜗牛
　　明日诗至过相称谓次韵还之
　　卷还一幅玉壶秋无实之名我所忧不似蒙庄解齐物以吾为马以为牛
　　次韵程弟
　　一船月自不嫌贫伴我荒湾理钓缗归去莫横深夜怕惊溪友与山宾
　　三径风烟手自开春猿秋鹤几徘徊松根石铫烹鱼眼老子行山又一回
　　防援茅茨隠者家底须日厯记年华山花山草无终极秋后寂寥春自葩
　　眼中转觉可人稀得似孤云自在飞花落花开俱有意略于闲处见几防
　　草堂四壁一瓢空举世无人与我同黄犊山南又山北犂春犹有古人风
　　柴门虽设不曽开俗面向人三寸埃却是前溪双白鹭门闗不住又飞来
　　垒麴成台糟筑丘衰迟无复少年游君看烟草离离外不问人间第一流
　　莫为盘飱烹不鸣借公松石聴溪声貂蝉只是麒麟楦寄语诸公卿自卿
　　鱼鸟相忘不世情浮生于我一丝轻夜凉月色光如水莫遣微云滓太清
　　此腹宁须文字香漫淹虀瓮半青黄午一枕睡世上何人最得忙
　　书维扬张君梅卷
　　平生何逊扬州梦到得扬州月自寒谁遣雪蓬春半树诗情更作故人看
　　月明临得霜桥影只恁横斜自一竒唤起江南归梦逺竹村烟水有茅茨
　　雪中迓新宪
　　踏碎琼瑶千里寒新年才此试吟鞍不须更说逢迎恶只作防桥诗兴看
　　雪后草亭
　　梅花面目冷于冰亦笑山翁草作亭一夜雪寒重整过碧琉璃瓦水晶钉
　　雪后梅邉
　　老来筋力倦登山契濶梅花几日间莫与梅花筋力倦且推一雪阻跻攀
　　其二
　　与君只隔竹笆篱踏雪过从自一竒开口便遭花问当老夫媿面了无辞
　　其三
　　半身苍藓雪槎枒直到顶头才数花看尽玉林山上下还须渠是当行家
　　其四
　　花中淡竚只梅花未见梅中蕚緑华一色琼瑶裙织翠尚嫌红蜡蔕村些
　　其五
　　玉防五出是天姿六出时时更絶竒夜半同参雪花去春风元自不曽知
　　其六
　　小小茅亭面面寒玉花能薄雪花干今年不是开迟着等得羣花比并看
　　其七
　　一枝宻宻一枝踈一树亭亭一树枯月是毛锥烟是纸为予写作百梅图
　　其八
　　野迳茅茨竹作墙嵗寒曽亦几平章髙人风味天然别不在横斜不在香
　　其九
　　寒雀羣飞最上头啄残香玉翠云裘老坡诗里么麽凤待与梅花渠是不
　　其十
　　春风且莫一齐开留与山翁逐日来只恐难将冰雪面伴渠桃李并尘埃
　　观梅
　　苍烟乔木野人家数掩山篱一径斜此老不知何面目只将空手对梅花
　　其二
　　卧疴半月不曽来等得来时渐次开世外交情无久近嵗寒然后始知梅
　　其三
　　几日商量雪不成竹邉寒得瘦棱层秋崖老子缩如猬撚断吟髭寸寸冰
　　其四
　　片片随风隔坞飘野香相逐过溪桥今年便觉春风早雪不成绵到地消
　　其五
