　　钦定四库全书　　　　　集部四
　　提要
　　相山集三十巻　　　别集类三【宋】
　　臣等谨案相山集三十卷宋王之道撰之道字彦猷庐州人宣和六年与兄之义弟之深同登进士第调歴阳丞南渡后累官湖南转运判官以朝奉大夫致仕后以其子蔺官枢宻使追赠太师宋史为蔺立传而不及之道故其事迹不详惟尢袤所撰神道碑尚在永乐大典中可以考见大畧之道尝自号相山居士其集即以为名宋史艺文志作二十五巻书録解题作二十六巻寳祐濡须志及濡须续志俱作四十巻尤袤碑文作三十巻彼此乖互不合今原集既亡无可复证然袤碑乃据其子家状所书似当得其实也初金兵南伐庐州盗起之道率乡人据险共保城赖以全干畧颇有足称又其登第对防时即极言燕云用兵之非以切直抑置下列及绍兴和议初成之道方通判滁州移吏部侍郎魏矼司谏曽统书力陈辱国非便寻又上疏论之并以前书缴进大忤秦桧意谪监南雄盐税坐是沦废者二十年今原疏虽佚不存而所与矼统二书具在集中其所论九不可和之说慷慨激烈足与胡铨封事相匹气节尢不可及其他论事诸劄子亦多明白晓畅可以见诸施行韵语虽非所长而抒冩性情具有真朴之致盖有体有用之言固不徒以文章工拙论矣谨就永乐大典各韵中搜辑编次仍可得三十巻疑明初纂修诸臣重其为人全部收入故虽偶有脱遗而仍去原数不逺欤乾隆四十六年九月恭校上

　　钦定四库全书
　　相山集卷一　　　　　　宋　王之道　撰五言古诗
　　和吕叔恭题和州香林汤【是时趋郡苦滛雨独坐精舍中贾邵周出示和倅吕叔恭诗顿忘逆旅之态輙继其韵】
　　新诗等絺绣文采照座右尤怜继絶唱峥嵘竞髙秀词源眉山长句法杜陵旧相从今老矣岂独歌邂逅香林在何许湓沸发岩窦其流渐清温脉助草木茂嵗旱添苗槁一溉十倍收追防记畴昔我仆苦痛疚别来隔兵火荆榛挂猿温温自然汤蠲疴古非谬我闻泉所发江海远相透其温本硫黄气味自同臭大哉无碍浴天地不轻授严霜与烈日冬夏特异人生困凌薄往往少中夀飘飘澡雪力狴犴忽防宥春寒不喜出块坐拨残槱晦隂又连朔夜雨屡通昼谁能慰穷愁邵州有贤守开箧出大轴令我和其后此意重虚辱勉彊忘固陋吕侯卿相家博学靡不究防当似前人嘉猷日告后君防华清宫诲滛似招冦当年从千官车马所犇辏渔阳一炬火曾不半钱售争如寄僧房泠泠灔苔甃常遂尘土濯永免脂粉潄
　　和富公权送行诗韵
　　公权提刑年兄与之道有四十年雁塔题名之旧备员湖北始得见君子聨王事相从二年所以眷予之意何翅骨肉今之道罢官以归于分擕燕饯之余特出长篇宠光行色令人恋恋不能自巳輙勉彊次韵
　　叨尘四十年静计所得禄十分才二三清饿似饥鹄中昨守义阳一时苦穷促忽闻栢防长表荐彻聪目不惟免罪戾英防许追逐接武宪漕间寒庖见粱肉从兄逮两载种种荷推毂容我数百大矣空洞腹为怜昔同升相防日不足芳时木芙蓉唱酬动成轴烧香每开奁惠茗屡倾簏青青阅冬夏髙视松栢独顾我如碔砆讵敢比良玉而兄不我鄙歩骤俾继躅是行集霰霜威日增肃解维在朝夕离绪可堪触恋恋重掺袪移庖过吾屋紫嚢出巨编还我海陵録全家被燕衎日尽继以烛何人鲙红鲜想见鳊项缩腹腴愧偏劝佐此一杯緑实筐出越罗币帛更加束酒阑儿女曹起舞应歌曲殷勤药石诲美疢肯滋毒新诗追古作髙义激頽俗临岐沐宠贶固陋敢虚辱但惭水底蛙勉趂丝与竹正尔无足道聊用资报复谦谦古君子厚徳卑自牧请歌伐木篇敬谢酾藇速
　　有荐胡仁叔为歴阳令者仁叔以诗送知已暇日杨徳润録以示予因次其韵
　　四海皆兄弟三界是道塲果且无彼我安用分封疆胡公厯阳来气熖何可当威严走邸吏不露太守章锦衣荣防稽暮雨成髙唐观望改侪鸡羣鹤昻昻斯民苦兵火有意哀劳伤岂无九转丹为助千金方古来记循吏不称茍道将杀弟自取酷防人人所防子潜去少室渊明恋柴桑出处虽异事一梦归黄粱昔日骨已朽伤哉今也亡国歌栢舟曹风赋鹈梁秋来河伯喜未覩若与洋大小自有见一笑还两忘我生分每越泉石成膏肓弃遗等麟角空虚漫龟肠常欲埋车轮除奸继张纲召此人气和致彼天时康语之有暇矣未窥夫子墙理所不可及马腹徒鞭长讵知王者师异日韩张良愿言三顾庐卧龙起南阳
　　和张咏老
　　三公与刺史秩等二千石人生不知足犇驰竟朝夕谁能话名理尊酒时促席贤愚大相逺丛玕映丛荻多谢张南巢诗筒肯予觅琅琅若明珠其来自灵壁僧房聊解鞍当暑得偃息挽回清微风笔有万钧力
　　彦嘉通守当今之俊伟人也余心向慕之常以未获识君为恨玆因孔纯老遂有邂逅之适其何幸如此将归纯老置酒饯别仆实预坐酒数行纯老丐诗于彦嘉以记坐上笑谈之乐顷刻扫数百言挥毫落纸云烟飞动永叔所谓盈前尽珠玑一一难简汰者殆谓是邪索鄙句不敢以固陋辞輙和北庄二诗奉呈
　　春寒不肯去谷雨犹裌衣溪流灔池塘逺防緑生漪识君恨不早语离我心悲行行欲何往丞郡金房西顚风送飞雨日日吼东北檐声决江河倾泻不成滴朝来喜新霁晨光透云隙垂杨翠幄开逺岫青葢立孔君公事暇侯子归期逼开樽且精设论文萃清集顾我老无能分甘口挂壁行矣不可挽思君浩无极
　　送无为倅张南仲归吉州
　　吾友李粹老少也防胶庠六年得一第自谓贵且昌衽席不知戒飞黄中道僵三阅大渊献嵗月亦已长嗟嗟死非所至今未能忘忠言类良药苦口不见尝那知兵火余获见同舍郎来丞濡湏郡两嵗依余光邂逅偶及此为之泪淋浪髙义借激薄美政称扶伤康沂富邦国何独歌王祥江城四月尽麦熟梅子黄归欤佐吾君谈笑跻羲皇遥应不可留船头鼔其镗
　　次韵因上人晚菊简魏定父
　　荆棘困松栢风雨乱昏晓干戈几时息年来费天讨孤芳如谢安端恐非小草此意谁人知伊余相自保不学桃与李芬芳竞春时天公岂多情清露为君迟荣悴休问命冷暖当自知青荧夜窗灯相对哦新诗离索自可笑衰迟谁见怜亭亭揺落中一枝小窗前正色有家法细防清且妍浮觞未云晚松醪吾所便经年混蒿艾蕴藏应待时清香宻渍沉微风助纷披宋玉昧秋意乃作儿女悲君子有所养因君观解頥道人如青松孤髙不成丛爱此黄金花萧然有淳风新诗皎秋月警我野性慵佳意不在酒此身今醉翁介夀诚有取无问人所宜君防菊花潭事见桃源诗东坡百世士敢比流俗知相逢勿相失不醉欲何时黄花如故人别乆喜复面浇泥一杯酒安用较深浅清霜开浮云黄落信风卷岩岩幽素姿秋来不多见
　　鉴止亭
　　扬波欲澄浊孰若波暂停止波浊自澄万象无逃形世人竞驰骛方寸不得宁投闲幸一来先生常独醒灯下读魏彦成诸公诗因次其韵呈曾子修诸君
　　距春尚旬月春信已回栁时来芝与菌往往出粪朽流俗不自重困穷变操守贵贱等人耳未易遽薄厚桃李何敢轻所报必瑶玖一笑置勿论防有君子酒
　　见山堂
　　淮西提干章序臣即其官舎逺亭之西为屋数楹命名曰见山堂盖繁昌诸山一举头则隠然见于檐外也
　　浮云开西风落木下夕阳谁人步东篱采菊哦秋光悠然若有思此意何可量南山适当前翠色摩穹苍我心不期见我眼偶相望咄哉逐兽夫泰山空在旁
　　西灵鹫
　　我闻西灵鹫幽竒冠龙舒山神知我来夜雨自涤除篮舁犯霏防敢惮石径纡清风偃林柯征衣尽霑濡樵人向我言君行欲何如松杉已在望尚有七里余﨑岖雨过水蹊云上天衢飞楼及涌殿峥嵘映浮屠幽鸟若相喜好音弄笙竽为言一宿觉何妨更踟蹰
　　憩灵鹫道院
　　黎明发龙门亭午憩西院冒雨且不辞宁能避泥溅是行良不恶此处殊可恋从今山中人何独识吾靣
　　追和老杜种莴苣诗【并序】
　　春晚雨过行蔬畦见园夫栽莴苣谓予曰苣有二种其色青而味苦者曰牛耳其色白而味甘者曰银条今方得银条一束于隣翁将栽以作种予顾园畦才十数菘已过韭且老矣惟牛耳青青可爱而园夫以苦鄙之伤时恶直丑正不见容于君子因和老杜种莴苣云
　　我有数亩园荒秽索锄理干戈何许久生意困屠燬道也其兴与时哉亦甚矣去年失短吕培植乏驱使今年得老杨衰病更临起晚菘巳芜没早韭就茅靡菁菁何所有莴苣独牛耳纵苦可自珍供盘况潇洒银条直且甘风味胡伯始其视李太尉小人与君子明明如古鉴不虑纎尘止廓廓若太清强欲浮云滓老氏有遗训重去彼取此方今向太平臯防正端委台省登硕徳要荒窜诬毁击冦岂无人蕃将来黒齿破齐在不日逆以诛正己停观劳还役杕杜颙采芑万邦遇有道贫贱古人耻文王来吕夷汉祖致园绮异时免沟坑愿附织贝篚
　　种田庵
　　古人即耒耜自有中田庐防君种田庵名异实则如涤场归去来宫功不容纾
　　亷余堂
　　髙堂面山水境与动静兼惟动故能乐惟静故能亷乐也知有余亷也笑无餍
　　次韵魏定文
　　半年去城府寤寐溪东扉有时得君书想像抉石猊岂知纪渻子养气如养鸡应响与疾视一洗三旬非我生坐穷困哆侈伤南箕鼷鼠亦何物焉用千钧机封侯絣澼絖药等手不龟所用有异耳此言寓非巵梅蕾良可人如椒缀南枝芬芳想不晩太皥方执防
　　次韵沈次韩见赠
　　我狂众所易頼有君见容十年卧岩壑百念付一空新诗似流泉涓涓江汉供人言鸡羣鹤我目霜中松相距无两舍防见何溟蒙谁能写离索正借诗思工
　　次韵许志仁
　　我技类族庖用刀常月更君才妙天下逰刄得我惊前年获相值因忆吴中行一别忽两载依然旧交情新诗璧夜光厥价轻连城相思不可见把玩我愿盈笔札擅南州文章动西京行矣秋风髙更听賔鸿声
　　盆池
　　剜苔得古鉴湛湛清不浑大哉壶中天勿与俗子言谁能时汲井注兹老瓦盆欲观从容鯈无羡横海鲲
　　秋晩采菊和徐伯逺
　　少皥司肃杀于渠独留情一枝黄金花照眼增双明香清露溥溥秀色云英英防当饮潭水使我病骨轻
　　题徳章山郭养源墨竹
　　胷次富盘错笔端妙苍翠霜枝岂人为烟叶有生意与可骨已朽此法谁复嗣伟哉寻丈姿绰有万尺势
　　追和东坡雨中睡熟至晩
　　春睡酣带酒晩食美当肉细观东坡诗万事贵精熟一隂七十日雨意犹未足山云开复合檐溜断还续东隣娇痴儿索饭恣啼哭惊廻梦中蝶重把君诗读
　　防河道中
　　防河村店西沙水清练练我行独带月滉漾掣金线凝寒逼征衣霜华重如霰至此懐僧弥老泪几欲【防河光州境也建炎初家弟甞尉浮光故有是句】
　　和龚圣任即事
　　新诗似霜月谛玩清且好欲见势未能思君令人老启蛰在朝夕雷雨发枯槁愿言从杖屦髙吟备搜讨
　　月台
　　低迷委径栁氛氲被池莲亭前秋雨余明月来婵娟髙台甃苍玉清风夜周旋举杯属姮娥醉舞还蹁蹮
　　小桥
　　芳榭绕流水春余半篙緑登临赏幽花兹亦我所欲小桥果何往长堤正相属朝防鱼逰晩听鸥鹭浴
