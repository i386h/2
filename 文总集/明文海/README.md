　　明文海　　　　　　　总集类
　　提要
　　【臣】等谨案明文海四百八十二巻
　　国朝黄宗羲编宗羲有易学象数论已着録宗羲于康熈乙夘以前尝选明文案二百巻既复得昆山徐氏所藏明人文集因更辑成是编分体二十有八毎体之中又各为子目赋之目至十有六书之目至二十有七序之目至五记之目至十有七传之目至二十墓文之目至十有三分类殊为繁碎又颇错互不伦如议己别立一门而奏疏内复出此体既立诸体文一门而却巧瘗笔放雀诸篇复别为一类而止目为文尤为无谓他若书序传记诸门或析学校书院为二或叙文苑于儒林之上或列论文论诗于讲学议礼议乐论史之前编次糅杂颇为后人所讥考阎若璩潜邱劄记辨此书体例为必非黄先生所编乃其子主一所为若璩常游宗羲之门其説当为可据盖晚年未定之本也明代文章自何李盛行天下相率为沿袭剽窃之学逮嘉隆以后其益甚宗羲之意在于扫除摹拟空所倚傍以情至为宗又欲使一代典章人物俱借以考见大凡故虽游戯小説家言亦为兼收并采不免失之泛滥然其搜罗极富所阅明人集几至二千余家如桑悦南都北都二赋朱彛尊着日下旧闻时搜讨未见而宗羲得之以冠兹选其他散失零落頼此以传者尚复不少亦可谓一代文章之渊薮考明人著作者当必以是编为极备矣其书巻帙繁重传钞者希此本犹其原藁四百八十一及八十二巻内文十二篇有録无书无可核补今亦并仍之云乾隆四十四年二月恭校上
