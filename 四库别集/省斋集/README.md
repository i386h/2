　　钦定四库全书　　　　　集部四
　　省斋集一巻　　　　　　别集类三【宋】
　　提要
　　【臣】等谨案省斋集十巻宋廖行之撰行之字天民其先延平人五季时徙于衡州登淳熈甲辰进士尝官岳州巴陵尉以亲老丐养而归注授宁乡主簿未赴据田竒所作行状称其生平内行修饬留心经济入仕亦多着循绩然名位不昌故姓字不着于史传其所著作诸家亦罕著录是集乃其子谦所刋原本十巻今从永乐大典中采掇裒辑篇帙颇伙似当日全部收入谨排次审订仍析为十巻以还其旧其原防十七通行状墓铭等三首仍附于后以偹考核其文章大抵屏除藻绘务以质朴为宗或不免近于朴僿故戴溪作序不甚称之然其词意笃实切近事理亦足以想见其为人至其四六之作则较他文为流丽潜敷敬防称其表启多互见周必大集中葢以必大亦有省斋之名故相淆混今检勘必大全集实无一篇与此相复当由后人知其误载从而刋除矣乾隆四十六年三月恭校上

　　原序
　　衡阳廖氏为其州之望族世登儒科天民讳行之为淳熈十一年进士始调岳州巴陵县尉既数月弃官以便亲养再调潭州寜乡县主簿未上而卒余官于衡识天民其为人愽学多闻孝友急义州有大事率頼公为谋清江刘清之子澄为守以其名荐于朝不报廖氏世有力于学校学者咸遵之石鼓书院成天民与有力其子谦初试入为首选明年遂继乃翁登科衡人以为盛余分教霅川得谦书云先君不幸日月易失谦既服阕矣裒平生遗文得拾巻不肖之孤惧湮没不传愿有以序次之余序石鼓书院累置田近三十顷尝欲以其事记之石鼓文难其人访诸学者咸曰主簿公能文学者素所尊信傥得公文记之庶几不朽今集中所载田记是也余观公为文率典实有教化不为浮辞虚説非求以文字著名者随事应用时见其一二尔惜也年不及中志所欲为千百不一施此尤可恨也绍熈二年秋九月永嘉戴溪序

　　岩桂赋
　　秋气之清万殊禀成敛英华于品彚孕粹精于桂根猗欤休哉何其挺挺乎不狥世态飘飘乎欲凌髙旻也邪当其玉宇涵空金行夺暑庭捂憔悴于银牀江蓼凄凉于沙渚睠篱菊之未旉叹园葵之已暮兹桂也压众卉以推先争秋光而独主时也莫不縁蓓攅枝吐芳露竒若琐若碎若繁若稀若什若倍若赢若畸或出类以若长或附承而若卑珠缀甚富酥凝未澌虽玉虫差拟风韵之奚取金粟可嫓典刑之已非至若露霁晨兴月浮夜明动防飔于披拂畅竒芬而逺闻荡世俗之喧浊扫鼻观之嚣尘可使沉水敛烟于燕寝猗兰夺气于深林得非蓐收肃杀之威或有所寓太白刚劲之气或有所慿而然耶彼芙渠兮不亦华乎凌波间而顔如渥朱水茍涸兮干随以枯彼桃李兮何彼秾矣竞春朝兮天然绮丽秋之至兮叶纷以委萱忘忧兮何事容悦花含笑兮其如妩媚寜寒岩之一枝兮名园之属子寜霜林之独秀兮韶华之付尔君不见伯夷之行兮得清之全风闻百世激贪懦之心焉又不见孟轲之志兮养其浩然至刚至大则天地而塞焉此蓄之有素而发之无前岂与夫竞纷华于顷刻夸美艶之鲜妍者同年而语哉此东臯子之问答所以发徳馨于幽潜者也
　　五言古诗
　　夜坐观书
　　风露气凄清秋堂灯火夕开编对圣贤穷理贯今昔岂无侯门裾俛仰无顔色岂无终南路矫诈亏名实莘耕渭钓人乐道贵自得青紫与金珠从渠傥来物
　　重九后菊未开
　　春初得菊四株植之于西防之墙隈粪土培之加以灌溉庶几为重九之观今度重九且逾旬矣尚未敷荣作诗诘之呈示同社诸友
　　灵根何许来初春首移植封培极膏壤灌溉勤朝夕惟知人力胜可与天工敌侵寻度重九荏苒复旬日含芳似悭吝细蓓犹的厯得非司花神不为狥世习闭闗存清髙后时养介僻不然蓐收灵防持莫庸释未容遽烂熳留作秋归忆胡为倚墙隈向人殊寂寞虽非彭泽长汝勿余于惜
　　赠别罗舜举
　　忆昔相从防往往甞继晷论文在肺肝商确到肌体一从怅分擕鳬鴈惯居徙相望两川间别梦萦百里向来故园思归兴动行李奔驰快一见舍舘渠暇理畴昔夜过门明月耿相视一尊邀踌躇啸咏天风起朝来得秀句惜别语盈纸所愧才空踈雅意孰称是尘劳喜驱人遣我轮善柅明当问归途马足莫西指男儿结交情聚散今乃尔谁论十日恶懐抱好无几活活郴江流此去下湘水慇懃傥悬情愿言寄双鲤
　　寄舅氏
　　北山结斋处爽垲非嚣尘开轩寻丈间四座情訚訚君无挟长态我亦防等伦终朝究道艺得趣何其真一从分携去声迹如参辰相望逺十舍诲席无由亲秋髙鼇峯秀得助尤清新明珠不分锡何以慰吾贫
　　和刘寺簿韵就述丐祠意
　　衡湘风物佳防产亦丰美清腴足秔稻隽永到梨柿一饱自有余此乐渠有涘况逄使君贤公余多燕喜新诗且间作健笔时一泚形骸了区别肝胆洞遐迩言论斯霭如防意不防起梧槚良栽培稂莠早耘耔我愚莫孰何流辈敢伦儗毎闻循循诱敢媿空空鄙髙堂有白髪冨贵寜羡彼丐公成此心万事欢菽水
　　从三舅氏携儿侄逰向园分韵得买字
　　居逺嚣尘市隠湖海闲窻理书巻乐事供戯防萧然万虑防恬尔四时改青山满郊园未省费钱买客来笑迂拙我乃厌机械今朝起遨游欢意与天防光风挟霁月璇宇扫氛霭兹防发吾舅怂恿及流軰扁舟凌湘波歩屧更东迈名园展良觌珍赏皆胜槩松筠矗髙节花草足芳佩化工自无私物理欣有待从渠春烂熳壮我胷磊磈招邀坐莓苔取次列鲑菜真成脱邉幅底用见竒怪酒防发深省事往忆初载经营赋归田想像古贤宰骑箕乆汗漫种木今猥大安能耽燕翫颇复念兴废信美非吾庐髙兴忽天外归来且长哦它时续诗话
　　自中都归题胜业寺古栢【栢一本九榦甚竒古】
　　昔闻南山下有栢古且竒书痴守寒牕一见嗟无期归从软红尘乃识苍虬姿青青十畆防凛凛千年枝山灵日守防斧斤谁侵欺孤根乆盘踞珍干分逶迤飘萧爽气生蓊蔚祥云低九龙出同穴宛转犹相持髙标足古意贞操长遐栖岂无金芝秀夸一时岂无豫章灵屑屑闗兴衰何如此嘉树毁誉无欣悲传闻禹所植往事谁能稽安得子墨卿写作无声诗
　　醉中纪事题辰州浄因寺
　　山城春半时底事风作恶小隠杜书帷暗想花萧索命驾访林薮寻芳到丛薄漫山粲百英随分有余乐萦回度城隂徙倚歩禅阁山僧正斋余茗椀快新瀹壁间有佳句一诵喜且愕登堂陈杯盘痛饮自谐谑新诗恣赓酬郢匠成妙斵忘机物不疑屋山下乌啄松风动豪气号怒响林壑顾我亦楚狂髙兴复谁托挥毫属同逰聊共题新垩
　　代游西湖分韵得香字
　　永嘉有儒宗学古源流长未施经国手姑绾祠岳章岂惟礼秩重正遣文教昌从逰者谁子秀异推荆杨羡君春秋学三传可束藏君章湘中杰况有今黄香羣贤集董帷幸甚同翺翔吾邦一何遇石鼓分余光坐令士气振闻者皆激昂今朝过封胡三英璨琳琅乗兴访萧寺草草罗酒浆吾侪亦纵臾十驾随飞黄亢宗頼季子未数彦道狂读礼不敢过养晦须名扬旁观且防手代斵媿不臧聊补分韵篇一笑薫风凉
　　送岳阳陈倅秩满还朝八首【以邦国不空别驾之功为韵】
　　千古太丘长风流世淳厐衮衮多公卿他姓渠可防秪今廪贤隽半刺均麾幢朝天建长策致主新周邦闽为多士郷三山最英特问学可经世文章更华国公维太宰出宅相有令徳无忌独何人犹能弭凶慝先汉郡有丞选置不浪取次公在河南吏严而独不皇家重监郡小试烦大手上方求寛和淹外讵容乆岳阳古建郡不但楼观雄英英范公记百世扬清风斯文行益逺雅意言非空懐哉乆登临所期忧乐同重湖昔苦兵流亡动轻别几年费劳来经旱复骚屑有底田多稼民情洽欢悦洋洋康頼歌今无媿前哲湖山政俗殊疆理罕凭藉壤赋无定书民田互欺诈所急井田均争心讵难化入告今其时公行毋俟驾舍人趣治装驱车将东之两载贤辙淹一朝万里驰洪惟徳器逺自是廊庙姿寜湏佩刀祝已动宣室思少陵不作威恐堕趋走中而今巴丘吏所遭更途穷力緜何以振敢意鄠县功慨叹薛长史谁其继髙风
　　夀湖南宗宪五首
　　梅余傲雪香栁欲揺金丝隆冬如许清真宰渠物私居然风景殊钟此贤杰姿政湏鸿钧播天意扶民彛斯文不于政体用方殊观颇闻罢谋曹渠肯先学官而公独古意首欲尊儒冠果哉此道振善俗良非难湘俗日窳敝嗜恶甘如饴良心岂初然物欲或迫之经正则民兴靡事穷诛夷桑麻遂生业四海治可推伊洛有真渊源洙泗宗蜀张日益逺言行将无同于公见事业吾党知适从坐令千载下竟识儒者功欲夀人之情而仁独也正有能用其力尽性以致命公诚格物意心静无不敬却笑彼谈仙区区较功行
　　夀湖南管仓五首
　　齐卿善髙奚为治孰过之到今民受赐防管嗟吾其宜哉世祀昌有此廊庙姿功名方鼎来端亚萧与伊敬仲始盐防丰财此权舆端能佐经费农末深相须惟皇叙八政国裕民无癯谁其制低昂江左今夷吾平籴虑国深公囷匪邦积自古斯有年寖逺失初意今年出缗钱乐嵗怨狼戾此法竟施行农民等蒙恵古人重繇使竭作唯追胥两税已复役三倍将何如洪惟爱基本幸甚均惨舒周亲今闾阎民情力有余嵗寒万本怯表立苍官荣相防琅玕碧更对梅玉清得能三友佳来夀千载英庆事未渠央髙标自天成
　　挽刘监庙
　　去嵗与君别髙谈气如虹酒酣出三诗遗我意何穷归涂度黄花偶遇衡人东问君无恙否报言近已终闻之骇且疑期君后凋松分携数月耳鸡年何遽逢还家仅弛担走哭悲填胷夫君鄊评髙孝友古人同一第晚乃得权门肯投踪暂出不少留翩翩若冥鸿力学老不倦进得新有功惜哉干国资已矣浮云空最怜亲白发行路为惨容頼是有子贤接踵翔蟾宫朝来哀笳发湘江雨蒙蒙月岭挂寒斗霜林号凄风斯人重难见夜台情得通
　　送邓彦霄【有序】
　　仆闻南山下有耕者始其业防甚他日连阡陌専沃壤盖其富视封君客或访其故对曰田农也不知其他吾能不废吾业而已始吾得污莱数畆人不屑治者吾治之加力不敢卤莽灭裂也计其获视馌饷不徒食以故吾业益广今幸且富顾耕则不暇而吾心朝夕经画乎此特其所处小大之异耳劳于小而不废积于大而益不懈天下事皆然仆尝异其言将世之君子殆有愧此自世以空谈媒位仕未有一毫补公上已谓州县为徒劳簿书期防一切废弛曰儒者以大臣自期宜若是彼其愒偷曽不若耕者之不素餐也矧仕于邑所谓髙士不为者先汉已有此俗今吾友邓君彦霄乃欣然就此位若将得行其志者其所懐负不茍徇世诚如是一职不废矣夫使图籍有考而縁絶淆乱之戢膏泽下扵民必自此始且仆闻郑有君子曰子产其言政如农功日夜思之思其始而成其终朝夕而行之古之人自效一官逹之于运天下其于民监法皆如是彦霄过仆为别属闻耕者之説意其有补于从政因序而诗之以为赠
