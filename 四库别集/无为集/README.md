　　提要
　　【臣】等谨案无为集十五巻宋杨杰撰杰字次公无为军人因自号无为子嘉祐四年进士元丰中歴官礼部员外郎出知润州除两江提防刑狱卒于官原序称侍讲杨先生葢其带职也杰及与欧阳修王安石苏轼游故其诗虽兴象未深而亦颇有规格其率易者近白居易其学为奇崛如送李辟疆之类者或偶近卢仝大致则仍元祐体也又及与胡瑗游故所学亦颇有根柢官太常时议典礼因革多所讨论集中如补正三礼图皇族服制图诸序以及禘祫明堂乐律诸奏皆有闗于典制但其文才地稍弱边幅微狭耳集凡赋二卷诗五卷文八巻绍兴癸亥知无为军赵士防所编士防序云删除芜纇取有补于教化者若释道二家诗文则见诸别集今别集不传故张敦颐六朝事迹载其雨花台诗一首王象之舆地纪胜载其浄居寺诗一首潜説友咸淳临安志载其西湖参寥山房诗一首铁网珊瑚载其佛日山别长老弼公诗一首凡为僧作者今皆不见于集中也然第五卷中有书宝山寺壁一首第七卷中有题宝树院五松一首东峰白云院一首野寺一首第十卷中有圆同庵铭一首圆寂庵铭一首未免自乱其例乂如铭五首入襍文赞亦襍文乃列诗中诗以古体律体分编而和谢判官宴南楼一首本拗体七言律诗而误入古诗编次尤为无绪至于魏诏君赞诏字葢避仁宗嫌名而次巻仍称魏征君草堂校讐亦未尽善惟杰集自南渡以后湮没不传士防积两嵗之力搜求编次使得复传至今其表章之功固亦不可尽没耳乾隆四十四年七月恭校上

　　无为集原序
　　国家以文教作成海内近二百年主上绍开中兴息马论道者一纪于兹比诏有司修建太学盖以儒术粉饰治具渐磨士类未始须臾置也无为在淮右为小垒而多名士侍讲杨先生名杰字次公道号无为子实一时文人公自妙龄擢巍科以雄文妙赋醇徳懿行得名于时中间立朝议礼乐因革人尤多之晚年甞奉使过泰山观日出于絶顶之上重九日赋诗举酒于华山莲花之峯继被诏从髙丽僧统义天逰前辈以谓皆以王事而得方外之乐故于瞿昙尤造理窟当时如大丞相王公内翰苏公悉印可之年七十而终生平所着文集湮没未传于世吁可惜也嵗在重光作噩之冬士防误恩假守是邦服膺侍讲公之名旧矣视事之初首询公文于搢绅间嵗余搜获不一公遣词典丽立意奥妙因删除其芜纇取其有补于教化者编次成集将以为学者标准上佐吾君偃武修文之意不其伟欤其诗赋碑记杂文表啓共分为一十五巻若释道二家诗文则见诸别集云绍兴癸亥岁夏四月左朝请大夫知无为军兼管内劝农营田事赵士防谨序

　　归来堂赋【并序】
　　碧虚子陈景元字泰初初入道为右街録赐号真靖主中太一宫屡请归庐山朝廷不从大丞相舒公因真靖自言而题之云官身有吏责触事遇嫌猜野性难堪此庐山归去来无为子杨杰盖碧虚子之友也闻而叹曰昔靖节先生赋归去来以归庐山之阳且八百年矣其辞未亡罕有继其声者今大丞相因子之言而及之爱子之深也夫靖节逺害于污俗真靖引分于治朝虽其去不同而所归则一廼追靖节遗韵而歌归去来以贻之庶亦自警云
　　归去来兮当太平之时胡不归宠难处而易辱乐或过而生悲田园芜兮不耨嵗月流兮莫追非彼马而是是是我指而非非黼黻烂兮吾目尘埃坌兮缁吾衣不収视以返听将安望乎希微胡为乎疲足孤轮与时竞奔行不顾乎夜漏往取愧乎晨门余闻有其善者善丧外其身者身存享不俟乎牛鼎酌不假乎象罇内苟适其志愿外何腼乎面顔一箪足以自养一枝得以自安车善行而无迹门善闭而无关动不离乎辎重超燕处乎荣观河泝流兮九曲丹伏链兮七还师不陈而坐胜勇无取乎桓桓归去来兮请从逍遥之游委天与之定分在道外以何求有名敎之真乐絶世俗之妄忧玩防微于八索鉴福极于九畴齐物我而一致泛忘心之虚舟蹑遗舄于子乔接逸袂于浮丘砺吾齿而潄石清吾耳而枕流投空谷而响应憩长林而影休幽鸟翔而后集白云去而复留岁云暮矣何之叟击壤以为期舍我田而营它徒劳力于耘耔子胡不歌大丞相之诗脱官身之吏责庐山归去夫何疑
　　一鹗赋【雄鸷之物无有俦偶】
　　鹗也惟一物之至雄絶伦类于凡羽锐击搏于秋风一飞则冲得路昊穹之表独立不惧肯羣燕雀之中在气禀金于德为义力捕潜伏性钟猛鸷由耿介以寡合非沽激而自异虽曰鹈如其状孰并翺翔未尝乌合其羣曲从党比其或木落万壑云沈四陲我则助天地严凝之气乘风霜肃杀之时鼔双翼以直上摩九苍而俯窥兔纵狡以难遁狐虽妖而尽追义可去者力皆击之此天下以无双少而尤贵彼鸷鸟之累百多亦胡为目莹星攒爪刚钩屈飞腾而雄压鹰隼擒猎而功高网罻宜乎孔融为表荐祢衡以兴辞邹阳上书谏吴王而托物大抵物之常者易其侣禽之异者难其俦故我不苟以合不旅而逰孤飞得以奋其勇离羣不足为之忧惟我独清屈大夫之在楚出乎其类孔宣父之生周非不知生而飞鸣乐乎俦偶奈何彼不我类我匪其丑与其羣以无益孰若介然自守又何必提提若鸒斯之党止贼夫粮嘤嘤为黄鸟之鸣过求其友勿谓毛羽尔盛朋侪我无殊不知丹鳯巢于阿阁大鹏迷于天隅子可类聚孰云德孤众莫希踪鄙翩翩之六鶂举难接翼小泛泛之双鳬噫得其时则架于轩楹失其遇则巢于林薮将伸勇毅之志愿假英雄之手如欲禽异类而肃四郊于一鹗乎何有
　　歌雍彻祭赋【宗庙之祭歌雍以彻】
　　王享于祖礼终以时雍诗所以歌者祭器从而彻之周颂一章择作永言之用宗庙九献此为去馔之期成王以贵为一人祀兼七世岁或举于祠禴时或行于祫禘且谓尽物尽志者孝子之事有始有卒者圣人之制当取正声以尊先帝尸初以入必奏肆以迎尸祭及其终故歌雍而彻祭诗云至止肃肃有来雍雍祖考我格辟公我从有眉寿降祥之述节宰夫屏器之容篇本附于臣工咏而见意时因去其胙俎礼以为恭大抵诗不可废废之则阙祀不可乆乆之则防故我就灵宇以行事扬正音而后彻声闻于外且非客出之初孝事其先用作神归之节岂不以迁其祭器之谓彻播以人声之谓歌彻焉造其礼之备歌焉报其福之多以乐为用非雍则何是以小师敎于瞽蒙职当讽诵君妇废其笾豆音应纯和惜哉周运否而大礼衰鲁权丧而陪臣侈乐奉祖庙义同天子虽能享献以自大莫测声诗之所以故孔子谓来助有二王之后始可讴谣奚取于三家之堂徒彰僣拟则知帝威穆穆上德颙颙诗曲尽于辞意礼告成于祖宗且异夫率诸侯以祀文颂陈清庙就方丘而祭地吕奏黄钟愚尝因雍什之咏歌求诗人之微妙以为荐可用于广牡禘可行于太庙及其彻也又从而声之为万世之典要
　　周兼养老礼赋【兼修三代养老之礼】
　　古之养老礼莫如周兼三代之常法新一王之令猷尚文德以唱风典章尤盛奉耆年而兴敎饮食参脩昔自有虞讫于二代必重年德以均仁爱然而礼有质而有文事或兴而或废及周之治也古今之通制兼明而老者养之帝王之余风尽在大抵燕者虞之礼飨者夏之仪太牢而食有商所为天下未有遗年者周人修而兼用之就胶庠以致勤高年毕萃顺阴阳而为具异制同施莫不鼎俎豆笾牲牢酒醴攷四代之异制萃一朝而为礼贵亲尚齿德爵富虽曰殊风黑衣素裳深燕缟杂然在体至如乐之作也六乐并用学之建也五学相参而况养庶老而明广爱事国叟而记高谈得不比前世而文盛优上寿以恩覃尊耆艾于每年时凡有七严豆觞于一日事徧其三故得风始朝廷化流穹壤春夏焉先飨而后燕秋冬焉先食而后飨舜禹汤之盛事得以追修乡国学之殊方各随所养向非歧昌之德盛而大武发之功肃而严成王治定姬公思兼则何以设珍具以完备致蒸民之仰瞻且异夫礼用四朝记者美鲁侯之盛文同三代史臣推汉德之炎则知兆民莫不亲其亲圣人惟能老其老推年龄以优事监古昔而顺考孔子曰周郁郁乎文哉未必不由于此道
　　不易之地家百畮赋【多少之限因地丰塉】
　　岁不易者为美地地所美者善养禾惟君赐万民之限凡家无百畮之过田居三壤之先靡虚其种戸给一防之广敢益而多尝闻田不授则仓廪虚食不足则民人夭故姬周平定甽畮颁分亿兆然而为土有肥硗出物有多少利而或偏仁何以表是故再变易者三其地补生育之所亏不代更者一其夫防兼并之所肇大抵王锡以土民耕以时众者不当过而宠寡者不可使之饥必在计夫家之费辨黍稷之宜以中地给六人之用下田供五口之资凡同居之七者择上腴而赐之旣殊休岁之畴常丰所利当限取农之数以授其私勿谓土之美也尔家不足与之耕勿谓齿之众也其畮不宜使之简况夫任土之法待其政生齿巳上书于版所养以之知所给因而限异汉臣之代甽农不定居如杨子之防田世为常产葢夫瘠地之民劳以瘁沃野之俗逸而伸沃焉至多则太逸其俗瘠焉加少则大劳其民所以上上之田不广蒸蒸之利将均虚一岁而虚二年诚非此类为前朝而为后市数若相因然后再易之间八家所治内以公田奉于国外以余夫次其地犹有莱畮六遂之逺得而优不夺农时数口之饥无以至惜哉运革三代权分七雄井地一变民田不同或负郭以无仰或连阡而自丰夫岂知辨野而颁遂师之典彛得正数室而制司徒之法则从中后之君鉴古是非随时沿革重孟子之经界鄙秦人之阡陌必也一丘甸以惠斯民损其丰而益其塉
　　琴材赋【桐之良者可以作琴】
　　世有嘉木天钟至音抱良材而丽地俟哲匠以为琴中藏山水之声能防大乐未偶斧斤之手独秀乔林峄阳髙峯龙门渊壑纯气所萃竒材以托宣情之具可以制闲邪之操因而作时未我与工未我度固全天质自为物以混成安得梓人为发音于寂寞百尺之木特生之桐落落耸干亭亭倚空无繁枝以示外蓄太和而在中时或裁成宜取羲黄之法人能抑按当移郑卫之风正声未扬识之盖寡庸目虽众视之或舍犹藏器之哲士俟抡才之贤者虽云陶令非取意于弦间又恐吴民欲为薪于防下俄有智者过而器之且曰堪舆之秀岩谷之竒激风霰于冬序感雷霆扵夏时足以道舜民之乐足以伸椘客之悲如玉在山秘珪璋之重器犹金藏鑛屈剑防之雄姿毓质若然成功非我非钟山之玉兮其徽曷称非园客之丝兮其安可将致于用必陈于左然后欲天下之治者调其音而为表仪有君子之听焉平其心而无懈惰是材之所禀用难自彰巧之所述器无不良傥工匠见遗不之翦而不之斵枝柯虽茂胡为宫而胡为商别有艺薮俊髦儒林纲纪明堂之柱此其选巨川之舟此其拟材乎材乎岂独琴而巳哉冀匠师之明所以
