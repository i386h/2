　　奸邪大臣背恩附党疏 

　　南京刑科给事中臣张永明谨奏为奸邪大臣背恩附党乞奋干断以昭国法以正人心事臣窃惟人臣事君之道犹事天也矢心对越精一无贰内无私虑外无私交日惟恪恭以勤乃职犹恐智能浅劣无所禆补以仰答恩造若复怀奸肆欺互相朋比以崇长其阴私岂惟臣道之缺实上乖天常下乱人纪罪莫大焉者也兹者郭勋滔天之恶逾于四凶仰赖圣明昭鉴穷治其罪欢呼之声遐迩倾动莫不举手加额庆明威之震曜如雷霆也天下幸甚然武人犷悍凶悖由来久矣若夫缙绅之臣习服圣贤之训而犹不闻君臣之义顾有阳効恭顺阴肆矫诬为私党之附此尤非臣所敢知也窃照礼部尚书严嵩受皇上殊眷崇阶峻秩恩宠逾涯苟圗报称虽捐躯恐不逮也乃与郭勋阴相交纳大肆奸欺宻请造以为欢缔声势以为固至如互分边帅之金而相悦以利偏克军士之资而相蒙以私大蠧兵政以生戎心勲戚家奴代造第宅嵩常被论而自辩孙澐非郭勋之私人矣澐今首称嵩复何解耶无日不造勋第情好尤宻纵妻赴饮其家路人所知嵩之附勋又表表彰灼者也臣闻之见无礼于君者若鹰鹯之逐鸟雀也无礼如勋者不能逐之己为罪矣况亲昵之乎又闻人臣之义无私交苟非公义泛有所交且不可况悖逆之臣而可与之交纳乎在小臣且不敢况受恩深重如嵩者而可二三其心乎再照严嵩近又具疏自劾是不惟假此以掩罪而又援此以卖直肺肝如见其将谁欺夫严嵩与勋其居甚宻其迹甚亲纵不轨之谋隐秘难知若其咆哮恣横骜逆贪饕其谁不切齿且亦尝经论列矣然嵩不言于虐焰方炽之时而顾言之于踪迹既败之后附其势以为利又尾其败而因以为功此尤见其险诈阴贼反复无恒之甚也易曰开国承家小人勿用盖自有载籍以来未闻小人而能成功者也宋臣司马光有言徳胜才谓之君子才胜徳谓之小人然从古小人易售其欺而君子弗究于用何哉盖憸险之夫习为欺罔多便利快顺之可喜而端方重厚之臣上下阴有赖焉者多以其悃愊无华而忽之耳才徳之辩君子小人之进退惟皇上深留睿察天下幸甚臣再维嵩与勋比附之故有二焉一则其嗜利蔑义之臭味相同一则其窃位固宠之声势相倚夫嗜利之心重则効忠之念轻徇私之党众则奉公之臣孤二者之势若持衡然亟反之以归于正其机甚微而世道登降之由于此焉决今圣君当极正臣子殚心翼戴之昌遇嵩乃敢怙终罔悛阴相朋比以自厚其私圗是岂所以报主恩明臣节哉大臣如此小臣何式文臣如此武臣何忌风之渐靡臣不知将何底止也乞将严嵩速赐罢黜为人臣忘公徇私者之戒庶有以破私邪之党明臣子之义国法大彰士风聿变而太平之治可拭目覩矣臣待罪南科已歴数月每愤诸臣深负上恩无以赞成睿志今元凶幸已伏辜而邪党犹得隐匿其奸以溷圣明若畏避权贵不敢指斥为循黙保身之计则不忠不直臣罪不容于死矣又何诸臣之愤为哉仰恃圣明敷沥冒陈臣无任惶惧战越之至縁系奸邪大臣背恩附党乞奋干断以昭国法以正人心事理具本专差办事吏杜文阳赍捧谨具奏闻 

　　贪污匪人久玷清班疏 

　　南京刑科给事中臣张永明谨奏为贪污匪人久玷清班大壊风教事臣窃惟臣道之多阙由士习之弗端而士习之弗端则师道之不立实启之古云养士莫大乎太学盖羣天下之髦士而萃之责以陶铸且为四方风教之表倡任莫大焉者也伏照南京国子监祭酒龚某素行庸劣士林不齿叨任成均污滥尤甚穷饕餮之欲蔑亷耻之心馔米膳银巧立名以侵克拨差拨歴惟视货以偏私监生罚旷者与未背监规者因而扣支似有名矣不知无旷与将拨者类多不给米肉何耶项元等四名出银四百八十两假起号舍之名以文饰似矣不知机户武曙与同乡郑金之超拨又何耶瞿塘戏坐师席亵嫚甚矣纳绫绢各四十疋而苟容孙继科冒引回籍与周重同罪矣馈银三十两而独免受堵禧金首饰三副纵其集班之时嬉舞谩骂于讲堂徒诡言彼方穉年而不加诃禁此六馆诸生之私指笑议者也勒王慎初银四十两致其拨出之后狂呌忿詈于私衙徒托言彼有心疾而不敢诘问此又逺近之所骇闻者也家僮容其乞差闲有家之义何居门子纵其通贿通政司之状可据邹来朋金盂银段之贽累数百金周干鱼果杂物之馈亦且数扛凡此者盖又不止为今日取悦之地而他日夤缘覆露之阶且将于是托始也兼之赋性躁妄处事乖违亡家狸而徧逮旁舍荷校列国学之门失库银而私拷东厢威胁陨无辜之命大体之隳所存几何官常之越肆然无忌况司府公事之私属素闻闽省夫价过闗之多索显播官衢其它卑鄙之行盖有臣耻言之而某不耻为之者矣夫朝廷养贤之地不闻弦诵之声而惟肆虐于刑威不闻礼义之训而惟征取乎货贿不闻诸生之衎衎乐业而惟致其羣然以诽议则污名器蠧士风岂特师道之不立已哉再思某常魁多士矣今为太学师循其资阶将或任台衡赞化理夫其魁多士也则后进者企而慕焉其率慆淫之风而煽导之也易为太学师则模不模范不范士习日以棼溃而入彀率者无真材将或任台衡而赞化理则徇私灭公训邪播恶其为害愈漫滥不可救此臣不敢不为陛下言之也夫言人之恶屡列章牍臣自失敦厚之体然使奸慝日滋无所禁戢臣又何以勉供微职以仰助皇上精明之治哉伏乞早赐干断将龚某罢黜别求名儒以充斯任庶菁莪朴棫之化所助为不小矣干渎天威无任陨越俟命之至缘系贪污匪人久玷清班大坏风教事理为此具本专差办事吏呉学赍捧谨具奏闻 

　　乞黜不职抚臣疏 

　　南京刑科给事中臣张永明谨奏为乞黜不职抚臣以慎固边防事臣惟国家各边重镇既设将官复设巡抚都御史盖欲文武兼济以仁义教化之用而成杀伐果敢之功其为任至重大也况大同一镇内拱都城外制塞北而又反侧甫平军情未固视之他镇更为重大为巡抚者必其忠诚足以化服顽暴徳望足以兴起敬畏才术足以建立事功而又军情毕照处置有方军机谙练设施中节夫然后反侧帖安狙诈不作使军威振而安攘之业成也臣访得巡抚大同都察院右佥都御史赵某人品素卑才识尤劣贪滥之声显著立身之节久隳节经给事中曾钧周采御史王献芝符验先后论劾皆有指实臣不敢复掇拾以渎圣听姑即其在大同不职之状为陛下陈之大同之兵骄惰失驭操练旷弛总兵官周尚文到任首罢私门占役之卒众服其公遂皆还操听命为赵某者当同心振作庶几人怀自新而一军之政举矣顾乃一筹莫展阘茸柔邪返忌周尚文声名出已之右巧言中伤揺惑众志见其操则诡曰操胡不已也见其役则又诡曰役胡不罢也不恤事体之宜专以煦煦之私市恩嫁怨周尚文愤其所为自恃素守亷介乃又执赵某之短以号于众曰贪黩而隳官守掊克而负军士者明神殛之夫使一军之士不闻忠义勇敢之训而习窥其倾危贪鄙之私今幸庙谟宏逺北鄙无患万一有警臣知赵锦之偾事必有甚于龙大有者臣愚以为论之于衂成罪稔之日孰若去之于事变未极之时犹冀有善其后者之为幸也臣尝谓边方巡抚才苟称任则虽素行可议皆当弃短取长以观其后独如某者本末皆无可观负乘足以致寇此诚首当罢黜以励臣工者也乞勅该部访查如果臣言不谬乞将赵锦罢黜速推素谙军事才望大臣一员往代其任仍乞天语警谕周尚文务在益敦晩节毋废前功或有疎虞必罪不宥庶几用舍适宜人知奋励边务其修举矣缘系乞黜不职抚臣以慎固边防事理为此具本专差办事吏郑隆赍捧谨具奏闻 

　　东宫礼仪疏 

　　南京礼科署科事南京刑科给事中臣张永明谨奏为东宫礼仪事臣闻汉臣贾谊谓太子为天下之本而惓惓以预养为言伏惟皇上圣明天纵启佑中兴父尧子舜一徳相承固有不待于预教者然易曰童蒙吉又曰蒙以养正圣功也夫当纯一未漓之初而能辅之以圣贤礼义之教则真性融而情窦不妄见闻正而左右嬖近无所惑也所谓自贻哲命其端在此臣愚以为就傅讲学之义实东宫今日之急务矣谨陈所见五事惟圣明采择焉缘系东宫礼仪事理为此开坐具本专差办事吏郑隆赍捧谨具奏闻 
　　一曰适冠服之宜以待年臣按大明集礼皇太子冠礼云皇太子加元服参用周文王成王冠礼之年近则十二逺则十五其冠服用九旒冕若东宫出阁讲学皆年八歳夫八歳则是未加元服也今东宫出阁未及冠期臣愚窃欲乞令礼官议用便服以从安适俟年十二以上可行冠礼庶讲学不至过时而于礼意似亦未失也伏乞裁择 
　　二曰隆君父之礼以始教臣闻文王之为世子朝于王季日三盖尊无二上皇太子虽曰天下之本而于皇上则臣子也臣愚以为出阁讲学之后宜习宫庭朝参之礼至于临见羣臣参决机务等事一切教之避让使知君父犹天之尊不敢有少抗也则孝敬之心兴而君道之大本立矣伏乞裁择 
　　三曰省仪文之繁以勤学臣闻帝王之学与韦布不同经纶之业与章句有异况皇太子初出讲学若仪文太繁拘迫太骤岂能使之乐学而不倦哉臣愚乞命礼官详定礼文务从简便宜令讲读儒臣与宾礼耆耄更番陪侍随事纳忠不必尽拘讲筵之礼则精神意气日进日倍而圣学有实功矣伏乞裁择 
　　四曰宾耄老之臣以让齿臣按大明集礼大本堂受贺圗三师宾客谕徳拜位在左西向皇太子立位在右东向其赞礼引礼者皆南北相向此国初所定盖犹冲年之礼也其后遇大朝贺皇太子常服升座三师宾客谕徳常服入就位北向立皇太子起立南向赞礼唱鞠躬四拜皇太子受前二拜答后二拜礼毕而出臣闻古之元子皆齿于学所以教让也皇祖制礼意本诸此臣愚以为东宫官属莫非人臣似不可以当答拜之礼欲乞命礼官会同吏部集议延聘徳望素着致仕大臣年八十或七十以上者数人赋之廪饩决不复令授官任事但取其体貌老成仪度庄重足为皇太子宾礼使之入侍左右盖致仕乃事外之臣年老无官守之寄优之拜礼稍似相宜伏乞裁择
