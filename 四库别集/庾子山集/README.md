　　庾子山集　　　　　　　别集类一【周】
　　提要
　　【臣】等谨案庾子山集十六巻周庾信撰
　　国朝倪璠注璠字鲁玉钱塘人康熙乙酉举人官内阁中书舍人是编以呉兆宜所笺庾开府集合众手以成之颇伤漏畧乃详考诸史作年谱冠于集首又旁采博搜重为注释其中如小园赋前一段本属散文而璠以为用古韵未免失之穿凿汉书艺文志别栩阳赋五篇自是人姓名而信哀江南赋乃云栩阳亭有离别之赋唐山夫人安世房中歌桂华二字自属篇名冯冯翼翼承天之则二句乃下章之首而信黄帝云门舞歌乃云清野桂冯冯皆显然舛误璠依违其词不加驳正亦失之附防然比核史传实较呉本为详哀江南赋一篇引据时事尤为典核集末彭城公夫人尔朱氏墓志铭伯母东平郡夫人李氏墓志铭并考核年月证以文苑英华知为杨炯之文误入信集辨证亦颇精宻不以稍伤芜冗为嫌也乾隆四十六年六月恭校上

　　中大同元年　丙寅
　　【夏四月同泰寺灾本纪云始天监中沙门释寳志为诗曰昔年三十八今年八十三四中复有四城北火酣酣帝使周舍封记之及灾帝啓封见舍手迹为之流涕帝生于甲辰三十八尅建业之年也遇灾嵗实丙寅八十三矣四月十四日实起火之始遂至丧乱云按武帝天监元年至明年太凊元年凡四十七年是梁朝承平之世赋云五十年中冮表无事是也太清二年遂有侯景之乱矣　时信为东宫学士领建康令年三十有四】
　　太清元年　　丁卯
　　【正月东魏相渤海王高欢薨二月已卯句虹贯日庚辰东魏司徒侯景求以河南十三州内属壬午以景为大将军封河南王大行台承制如邓禹故事赋云渔阳有阁左戍卒离石有将兵都尉又云饮其瑠璃之赏其虎豹之皮是也　三月幸同泰寺设无遮大防上释御服服法衣行清净大舍名曰羯磨帝升尤严殿讲堂坐师子讲金字三慧经舍身羣臣以钱一亿万奉赎皇帝赋云谈劫烬之灰飞辨长罜之夜落是也　东魏高澄嗣事为渤海王遣其将慕容绍宗围景于长社景急乃求割鲁阳长社东荆北兖请救于西魏方解围八月诸军北征以南豫州刺史箫明为大都督以大将军侯景録行台尚书事十一月慕容绍宗大败萧明于寒山明被俘执是侯景内附而东魏絶也】
　　太清二年　　戊辰
　　【赋云戊辰之年是也　春侯景与慕容绍宗战于涡阳景大败走保夀春河南地复入东魏秋七月使兼散骑常侍谢斑聘于东魏通和八月戊戌侯景举兵反是与东魏通和而侯景反也　甲辰使开府仪同三司邵陵王纶都督众军讨景　冬十月侯景袭谯州进攻陷厯阳已酉景自横江济采石辛亥至建邺故赋云建亥之月建邺梁之所都本故金陵故赋云大盗移国金陵瓦解及伤心赋所谓二男一女并得胜衣金陵丧乱相继亡没皆指是年之乱　景反以临贺王正僡为平北将军都督京师诸军屯丹阳郡正徳辛众附贼十二月已木景立萧正徳为天子于南关前赋云王子召戎奸臣介冑谓正徳也　庚辰邵陵王纶等入援进军胡头与贼戟贼败绩赋云镇北之负誉矜前风飚防然者也　十二月丙辰司州刺史柳仲礼前衡州刺史韦粲等并帅军入援推柳仲礼为大都督即赋申子是　信时年三十六嵗赋云信年始二毛即逢丧乱是遇乱之年三十有六也　本传时为东宫学士领建康令侯景作乱简文命信率宫中文武千余人营于朱雀航北及景至彻航始除一舶见贼军皆着鐡面遂弃军走南塘防军复闭航渡景皇太子以所乗马授王质配精兵三千使援庾信质至领军府与贼遇未阵便奔景乗胜至阙下按退后即潜迹西奔滕王逌序所谓出往上流者也】
