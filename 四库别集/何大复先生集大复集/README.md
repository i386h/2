何子仲默生而敏悟童子时即执简赋诗邈焉寡俦长益肆力焉探冥抉奇增华漱润轧晋魏而上之惜乎三十九而卒乃巳焉矣康子德□□亟张子时济皆何子之友也何子既殁张子牧其逸稿访康子于浒西之野而共扬搉之凡得若干首洋洋纚纚一唱三叹而有遗音焉时关中执事君子咸兴嘉桨之怀乃檄而传之得非以古昔有诗之教邪夫词不雅与雅而阙子行者皆弗得传即传弗远何子词则雅矣其行履复□然而有人名则传之也固宜呜呼以何子之才而得寿考廓而大之纶而不息殆潏乎河汉而无涯涘也天顾阨而夭之而俾弗竟其绪乎子闻天之大化理以经之数以纬之以理御数则履其常数角而胜焉变斯出矣是故牛之行羸然而疾也回之仁陨然而夭也诚格于数即古之大贤犹俛而从之何子其若之何抑诸断鹄之剑照乘之珠天下之至宝也假于涂之人则主人匍匐往索之而不俟越宿焉是故李贺夜梦神人请赋白王楼觉即属纩殆又一道已乎何子断鹄之剑照乘之珠也则天恒靳其光矣而能不牧之邪是亦不可不谓之数也嘉靖三年三月既望兰溪唐龙撰

明兴百六十年其文遐哉盛矣然作者接辙于域中其敦玫古昔逖称先王人人能矣而义意繁猥溢于往训摹仿摽敚远于事□犹以为过云弘治时　上兴化重文士大夫翕然从之文视昔盛焉是时仲默为中书舍人而予以次第为翰林修撰一时能文之士凡予所交与者不可胜计予顾独以仲默为难能仲默时方弱冠也予每见仲默之作叹曰嗟乎文其在兹乎夫叙述以明事要之在实论辨以稽理要之在明文辞以达是二者要之在近厥指意凡仲默之作三者备焉故予欣慕叹息非私之也武宗皇帝之三年予以忧罢能撰归十月三年仲默以提学来关中数能以公事过予出其所论著凡数万言皆当实不修可以上薄屈未董贾有相如子长之风顾世无知之者独曰何子有诗人之遗意厥觌浅矣仲默岂独工于诗者故仲默名遍天下然实无知仲默者此也汉魏以降讫于开成世代既移音节斯异修辞之士能无恶厥趋斯已优矣复能引而上之不沉于流俗由载籍所覩若仲默者岂非鲜哉十六年秋仲默既卒又三年子次第其文为若干卷首赋次诗次文皆随体区裁因制列卷题曰何仲默集录存家笥以待后来读其文思考其世可以知予言所指矣仲默才高而意宏德纯而识远荣辱毁誉一无所动于中予别有传记兹不载方予定次仲默集时值张子时济过予所见与子甚同因更与定之如此集云

敬皇帝朝化休而融昌异时诸先生业文章颢甚重矣学士大夫固欣然称说耳相慕也而独壮地李子以非心所好谢去之亡何而又有信阳何子者何子虽稍冕出其材质敏秀瑰丽各以长相当然而李子得何子为益雄也鄙人之言何知仁义向利则德是二君子抉草莾倡微言非有父兄师友之素而夺天下已向之利而自为德于乎难哉去其始可一甲子诗而亡举大历下者文亡举东京下者即谁力也然二君子之徒不能长缘其师所繇得毛举论难之语以好为胜而它工易者恶津筏者往往左袒何子而龁李子则又似非何子意也夫百羽集于词林而二子雄飞或撇捩逆羊角而横举或顺颷而肆翔其九万里同也李源风何源雅风故长变以明志取夫赌其沈深莾宕激卭鼓壮喑呜&#63150;&#62699;林日&#63151;凄忽正而奇正若岳厉奇若海飓则李子哉是固少孙要之其缘情即象触物比类靡所不遂壁坐玑驰文霞沦漪绪颷摇曳春华徐发骤而如浅复而弥深疑无能隃何子而上者何子为文刻工左史韩非向家言大抵于诗写鳫行云而关中康氏乔氏其乡人樊孟氏则盛惜何子志业屈于年未竟世之谈说经纶抵掌事勋者其敖何子以不及如耶令何子不死而称为名公卿已耳所以削涤卑琐振颓习扶昌运开中兴者何物也功于经纶敦多于是何子之甥袁灿来谓王生若为何子叙其遗言王生曰何子彬彬大家也易言之有亲则可久李得助而久何子之功李子伟矣夫二子之功天下则伟夫

予使于滇行将万里所越皆重山复岭以舆以马颠顿驰跋而不任其劳至永宁乃谋舟以归庶代陆走之勤时霜降石出江狭水迅机发矢激其劫不得少留甚惧之有老篙师进舟意颇闲自招于前一人柁于后柁者左右缓急不敢自任视诸老篙师老篙师者额之指之目之而左而右而缓而急无不应者其舟之歌正疾徐皆与水势宜不为岸妨不与石鬪予甚安焉问曰尔之为舟将有道乎老篙师执篙进曰有敢请言之夫侍用者器也器不良则不穪用用之者吾之手足耳目口也少怠即不能用手足耳目口役于心者也心不专则手足耳目口不为用招与柁吾之器也左右缓急歌正疾徐吾之用也故吾先良吾器以利吾用手足耳目口不敢少怠心不敢少他然后从容乎安流荡激于奔涌洄洑之中而无虞也然吾又能察水性与劫故夫　汇潭渊浅涡暗滩隐石者吾远而认之能得于十数里之外是以预为之备而不至损吾舟他水工常恶迟取速截长邀短故有败者吾惟顺其水道而无所枉虽迟而得免败焉吾之道如此已矣予谓舟御类艺也人之于世顾艺之不若哉因序其言为赋以自励

仆闻之殊途者不可以同观异趣者不可以强翕故嗜竽者不媚之以瑟好圆者不进之以矩何则殊途而异趣也故贾子投荆南仲舒屏江都屈原游泽畔三子者非容之不能谋有不合有不可以容者矣故射者不为人易其彀琴者不为人改其操故师可易而法不可易也是以物有不以贵易贱富易贫荣易辱者矣故茂草不负垣美谷不生辙惧所托者非也是故求乐其心者不求华其身求显于后者不求耀于今故夫子周流删述坟典笔削鲁史左氏着国语马迁抽史记荀卿董子之流皆有论譔大抵困屈穷阨发愤述作兹数圣贤者宁徒耀于当世哉仆少执寡昧窃有慕于古人之义弱冠即仕出入班行者已越六岁颇有龃龉之叹焉正德戊辰
