　　钦定四库全书　　　　集部四
　　提要
　　紫微集三十六巻　　　别集类三【宋】
　　臣等谨案紫微集三十六巻宋张嵲撰嵲字巨山襄阳人宣和三年上舍中第绍兴九年除司勲员外郎累迁敷文阁待制知衢州终于提举江州太平兴国宫事迹具宋史文苑传嵲为陈与义之表侄少时尝从受学故刘克庄后村诗话谓其诗句法与简斋相似而于五言古诗尤极赏其语意高简意味深逺又克荘所摘七言絶句如故园坟树想青葱诸篇尤能以标格见长而集中似此类者尚多大抵絶句清和婉约较胜与义其他虽未能遽相方驾而气体高朗颇足以自名一家至古文典雅沈实亦尚有北宋诸家矩矱所上奏议如论和战守论攻取等篇史皆采入于当时事势尤条析详明惟绍兴复古诗一章贡谀秦桧深玷生平考朱子语録有云金人败盟时秦桧大恐顾朝士问计张巨山微诵曰德无常师主善为师善无常主恊于克一桧因留与语巨山为之画防桧喜即命作奏藁仓卒不仔细起头两句以德无常师为伊尹告成汤陈力就列为孔子之语寻擢巨山为中书舍人有无名子作诗嘲之云成汤为太甲宣圣作周任云云是嵲本因附桧得进陈振孙书録解题亦载此事且称桧旋疑嵲贰已未几亦罢然则此诗之作乃借以修好于桧者故本传谓诗进后将复召用殆由桧意解而然特作史者不能得其情耳今其代桧奏藁虽已不存而是诗尚传留供千秋之嗤防亦足以昭炯戒矣宋史艺文志载紫微集三十巻书録解题则作张巨山集亦三十巻自明以来久无传本今据永乐大典所録裒辑排比诸体咸备当已尠所缺遗以其篇帙较富析为三十六巻仍依宋史题作紫微集复其旧目焉乾隆四十六年九月恭校上

　　紫微集巻一　　　　　　　宋　张嵲　撰赋
　　梅花赋
　　潘骑省之植花氛氲河县桓司马之种柳依依汉南未若兹梅之峻洁逺自托于层岩格侔蕙茝志傲氷霜青枝瘦而依防乱蘂繁而向阳风披逸态月射孤光挺出尘之絶艳吐超世之竒香若乃逺壑氷消疎林雪后沙村逈而日晚石涧浅而寒溜临山径之欹危出茅檐之左右或芬敷而盛发或伶俜而欲瘦或含葩而未吐或喷蘂而竞秀其髙者如举其低者如坠其疎者如刻其密者如缀其素者如愁其绛者如醉倾日而煦者如笑迎风而靡者如愧覩节物之芳华乱乡愁于晚嵗怀故园之春色惟兹花之颇类嗟物是而人非闵节同而时异矧昔壮而今老复前丰而后悴攀柔条而未折嗅青蘂而堕泪悲憔悴于荒山惜赏心之莫防如志士之陆沉亦何为乎此世羌欲去而还止步落英而徙倚眷孤芳之信姱谅瑶华之莫比故知国香服媚非独燕姞之幽兰珍树珑未下唐昌之玉蘂
　　中秋赋
　　陈王之客元瑜从军吴防季重出宰朝歌寂寥观防賔从顿寡时当仲秋积雨新霁烟消广厦步兰路而逶迤美西园之清夜怅别逺而防稀感时迁而物化乃设醴以命宾跻凌风之累榭中置以后酒酣耳热顾公干而谓之曰今兹野有鸣鹤天絶归云丹枫微脱白鴈初賔望舒曜夜而逈彻六合清朗而无尘兹逰乐乎盍为寡人赋之公干乃执简而言曰臣生居东国少遭丧乱幸值明时簉迹宾馆臣闻豳籥迎寒逺载于周典乗查上泝近传于汉臣臞仙则采露于栢杪吴士则观潮于水濵信白藏之令月盖西陆之佳辰也于时天宇正碧凉夜方永曕东峯之朣胧见素辉之初引斜汉淡而茫茫列宿疎而烱烱白露而晶荧回潭澄而逺镜翫老蚌之含胎惊栖禽之不定俄而随珠委曜赵璧舒光羣峯变缟旷野凝霜映林岫以玲珑射榱甍而焜煌散珠树之清樾莹美人之明珰于是停妙舞息繁吹聆清籁于林表听胡笳于空外望桂影而凝情睇澄辉而伫思怅氷轮之既满谅无逺而不烛照思妇之髙楼鉴幽人于空谷光出塞而漫漫影流波而煜煜乃有负羽少年怀才滞史对絶景而恨结悲故乡而心死况乎操琴多雍门之意琵琶有出塞之声征衣捣而未寄戍卒去而将行朋友散兮山海隔音书絶兮雁无凭于斯夕也望千里之素魄缄万恨于幽情莫不言将发而气塞语未终而涕零者矣于是徳祖闻之伤怀慷慨泣下歔欷含毫奋笔而为明月之诗曰秋气髙兮桂魄圆风泠泠兮露团团嵗将暮兮兰菊芳思佳人兮天一方照兰殿兮映绮罗呈激楚兮发阳阿人生宴乐兮孰知其他公干乃跪而言曰夫古人以宴安为酖毒沉湎为至戒今流连长夜丝竹繁防载号载呶蜩螗沸何吾子之赋独无尽规之义乎乃歌曰酒既酣兮夜既阑管停吹兮丝罢弹长袖拂面为君欢愿公子兮保嵗寒辅明主兮寰宇安翫清辉兮复年欢陈王于是敛袵谢客三复至言永以为则
　　夀赋【并序】
　　窃以某官以仲冬初五日降诞仲冬者一阳复之时也五者大中之位也至阳所以伏羣隂彼夷狄者中国之隂也大中所以距颇邪颇邪者小人之谓也天将以公攘戎狄而退小人其兆既存矣谨按洪范惟皇之极敛时五福用敷锡厥庶民由是观之自天子至庶人大中既建则五福从之其为福莫大于此者谨成古赋一首以申其义且致善祝之意云尔嵗强圉赤奋若兮仲冬皇极实生伟人兮以佐王国禀正阳之贞性兮资大中之妙质宜博蹇而修姱兮好是正直挺濯濯之瑰姿兮角犀丰盈伟视瞻之英发兮进退齐平事纷至而毕应兮刃新发硎豁逹而不忌兮一毫莫撄抗逸气而无前兮云梦可吞进羣士而下之兮言色惟温视冦氛犹一决兮抗志不羣誓一扫而平之兮归报天阍宜公宜侯兮夹辅大君铭功鼎钟兮画像麒麟原建子之令月兮一阳来复考皇极于元日兮大中斯得惟一阳复而中建兮羣隂伏而颇僻销中国强而君子进兮可验其由愿公攘戎狄兮勿速勿亟去小人兮勿二勿惑修徳政而缮甲兵兮戎丑自怀进亷直而退贪佞兮君子大来戎狄攘而小人去考休征兮五福是具膺公衮兮践槐序间两社兮为王室辅原厥初之畀付兮谅非偶然虽公徳之既懋兮尚祈勉旃秉徳不忘兮奉以周旋繁祉优裕兮安乐延年下吏献言永矢弗谖
　　续囚山赋【并序】
　　余居山二年汲汲鲜况读柳先生囚山赋因叹曰旡桎梏而独居于人之地真囚也或曰柳先生得罪于时放逐湘南欲去不可故云今子非得罪居此其去也顾不在子耶余曰子厚之所为囚者人也其不可去者罪也余之所谓囚者天也其不可去者势也客未解作赋以晓之因命之曰续囚山赋其辞曰环麋庸其防里兮中间栉比乎山丘如疾风之过钜海兮翠涛竞起而漂浮窘束偪仄纷以回互兮气壅结而不流产毒孕怪兮土风殊异乎中州陂陁绵亘若圜土兮草树郁丛棘之相樛仰天俯地了不见其隆寛兮怀壹郁而増忧嗟余生之不辰兮偶托迹而淹留何怀安而不去兮乃独堪此牢愁岂不乐通都之是处兮与羣俊而遨逰顾世路方险絶兮羊肠莫之与侔风波荡潏兮行客沉舟谗口听听兮又将彼戈矛余虽厌是而欲往兮复慑詟而更休明方夷而未融兮时方剥而未贲忍尤而攘垢兮甘作山之孤囚
　　憎雨赋
　　皇天不容于昭明兮白日不容于显融恶堪舆之清朗兮好云霓之晻霭反明以为暗兮变昼而作夜莽惛惛而不可讯兮谁为此者岂造物之固然兮将羣隂之所为茍大钧之若是兮吾将排阊阖而问之建列宿以为旌兮召丰隆以为御令祝融使先驱兮厯元防而问故命蓐収司刑兮臯陶为理囚屏翳于雷渊兮制飞电而诛虹蜺闻羲御之将驾兮吾将怀椒糈而要之皇天反其明兮后土无所汚望舒承夜兮昬与明其代序巻此志而弗抒兮吾犹穷困乎此时也将防行以问涂兮又何以为此态也
