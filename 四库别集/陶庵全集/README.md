　　提要
　　【臣】等谨案陶庵全集二十二巻明黄淳耀撰淳耀初名金耀字蕴生陶庵其号也嘉定人崇祯癸未进士未授官归福王立不赴选家居讲学乙酉南都破
　　大兵徇嘉定淳耀与其弟渊耀入僧舍自经死乾隆四十一年
　　赐諡忠节事迹具明史儒林传淳耀湛深经术刻意学古所作科举之文精雅纯粹一扫明季剽摹谲怪之习天下皆传诵之而平日讲求正道孳孳不倦尤能以躬行实践为务毅然不为荣利所挠夺如吾师自监诸录皆其早年所订论学之语趋向极其醇正而平易可近絶无党同伐异之风足以见其所得之邃文章和平温厚矩矱先民诗亦浑雅天成絶无懦响于王李钟谭余派去之惟恐若凂其立志之坚确如此卒之致命成仁垂芳百世卓然不愧其生平可以知立言之有本矣集为其门人陆元辅所裒辑见于明史者十五巻今此本为文七巻文补遗一巻诗八巻诗补遗一巻吾师录一巻自监录四巻共二十二巻乃后人续加増辑以行者也乾隆四十五年四月恭校上

　　陶庵全集原序
　　黄陶庵先生死忠之五年其门人陆翼王收其遗文得所论著百余篇属予爲之序呜呼陶庵之文止于此而已乎当其城陷引决投笔絶命搤吭而死翼王访求搜购于流离煨烬之中遗编断烂什不一存此可爲流涕叹息者也陶庵深沈好书于学无所不闚居常独坐一室不交当世迁固以下诸史朱黄钩贯略皆上口其于攷据得失训诂异同在诸儒不能通其条要陶庵顿五指而数之首尾通渉铢两歴然虽起古人靣与之讐问莫能难也其爲人清刚简贵言规行矩蚤有得于濂雒之传尝谓人曰吾比来爲文初无所长然皆折衷大道称心而立言质之于古騐之于今其不合于理者亦已少矣此其一生读书之大略也当庄烈皇帝初年海内方乡古学一二通人儒者将以表章六经修明先王之道爲务廼曲学诡行则又起而乗之依光扬声互相题拂剽取一切坚僻之辞以欺当时而误流俗论者不察乃比其始事者同类而訾之噫此亦不思之甚矣世之降也先王之教化既熄法度既亡人奋其私智家尚其私学纰缪杂糅蟠戾于天下虽有高世之君子欲整齐而分别之其道无繇惟夫忠孝大节皆出于醇正博洽之儒其似是而非者不一见焉然后天下后世了然知异学之当诛而大雅之可尚以观我陶庵非其人耶陶庵爲诸生二十年与其弟伟恭其徒侯几道云俱其友夏启霖軰昼夜讲性命之学晩而后遇不肯就官城破之日师友兄弟同日并命今其书虽不全使读之者忾然想见其爲人益足以徴于今而信于后无疑矣翼王以五年之力掇辑散亡其功于斯道不细固不专爲陶庵已也吾故表而出之俾后之人知所习焉己丑秋九月太仓社弟呉伟业题于梅村旧学庵
　　予自束髪受书即读陶庵先生之文见其精深纯粹髙者可以羽翼经传下者可以淩轹韩欧心窃慕之以爲是何如人而其文之超轶絶伦如此及闻先生从容就义慨然太息谓先生于死生之际不苟如此志与日月争光而行与天地同久宜其发爲文章精纯超絶协金石而中宫商也既又思之自变故以来平日谈忠孝讲仁义之徒临利害而防其所守者何可胜数而先生独毅然不变人之所隐忍徘徊而不能决者先生视之若渴饮饥食之不容已夏葛冬裘之一定而不待拟议也此岂可以强至而卒办哉盖其所积者厚而所养者深矣及待罪先生之乡见其遗老访问先生之平生则羣以爲先生平日孝友忠信取与不苟泊然于富贵而发愤于正学孜孜矻矻惟以圣贤爲己任而世俗之尘埃不足以入之呜呼此其所以能临利害而不变也欤自世教之衰士不知以防隅自饬谓正学爲迂濶谓功利爲不可已遇小利害则攘臂而起蝇聚蚁逐无所复顾平日之志气如此欲其临大节而不可夺岂不难哉由是发爲文章不入于卑陋则病于杂驳虽欲弥缝润色自附前贤如窭人之装爲富贵非其所有张皇支吾百病俱见无怪也故予以爲先生之文本乎行先生之行所以能卓荦于临变者本于平日之养士苟能以先生之养爲养自然险夷如一履变则爲岁寒之松柏处盛则爲髙冈之鸣凤皆是物也区区文章之焜耀何足道耶不然虽文如先生犹不足贵也而况乎其必不能哉因先生之门人裒集遗文以传而爲之推论其本俾世之学先生者知所取则焉当湖陆陇其撰
　　君子之学一于诚而已以之治心而心正以之决事而事无可疑察乎防防祸福之萌信诸进退出处死生之际孔子曰笃信好学守死善道夫惟诚立乎中斯毅然有不可夺之节防难不失其正顺道而死盖虽圭璧析于前而不顾刀锯鼎镬惩于后而视之若无物也齐之虞人招以旌不往孔子取之孟氏以爲枉已未有能直人者则圣人之所守可知己接淅而去齐不税冕而去鲁是岂肯应公山不狃佛肸之召者故曰可以止则止可以处而处孔子也顾后世躁进若扬雄之徒每援圣人以自文其过其进也不以礼其禄也非其道防防祸福之不明进退出处死生之未能信善道之谓何无他诚未立于中宜所守之易夺矣嘉定黄先生讳淳耀字蕴生别字陶庵平居讲圣贤之学躬行而不倦崇祯十六年秋赐进士出身未授官归越二年殉难以死同里门人陆元辅辑其诗若干巻雕刻行之又搜其遗文仅四十余首藏之笥元辅请彝尊序受而读之其言和以舒其析理也审以辨其援据经史博而不诬所谓脩辞立其诚者非与于是先生之殁三十年矣诵其文恍若觌其容而聆其謦欬信夫有道之言之入人深也呜呼以先生大节如彼其学业文章又如此宜其于人少可而多怪今观集中论学书絶去儒者党同伐异之习是尤恒人之所难能也讲学莫盛于宋然汴京临安之陷道学诸臣以身殉国者不数见至于明死节义则有若方公孝孺死阉祸则有若高公攀龙而山隂刘公宗周漳浦黄公道周与先生后先自靖咸以道学兼忠节即宋儒有未逮焉而元辅以兵戈俶扰之余能集其师之遗文俾无失坠亦可谓笃信之君子已秀水朱彝尊锡鬯譔

　　上座师王登水先生启
　　伏念某海壖贱士林草鄙生抗髙标于媚学之塲弹古调于无人之野书忘寝食思起班扬贾董而与游学论精防将求濓洛关闽所未发至若帖经墨义耻爲绘句絺章风变永嘉力追正始功非武事高语廓清坐是浮沉乡校者廿年因之蹭蹬棘闱者五举虽年未臻于强仕人方滥数爲时髦顾名已宿于文塲已亦自疑其晼晚不谓雕虫末技荐诸冰雪聪明叨居摸索之中得骋风云之气兹盖恭遇老师台下诚能体国公以生明龚黄髙汉吏之称燕许擅玉堂之笔方且五雀六燕平操人物之权衡遂令纎利小材尽入文章之渊府苶然骫骳荷此甄收感乃铭心谢宜重茧然而赍粮千里方嗟赵壹之空囊偕计公车复迫郄诜之献赋是敢敬修牋记参承之礼用以稍纾髙山仰止之诚虽其迹渉踽凉似永叔之不登阶序或者心存永久如任安之独在门阑谨沥下情以尘清覧有违此语是负师思
　　上座师王登水先生书
　　某蹇浅下材自十有七岁而入胶庠今二十有一年矣生平厌薄陈言独好泛观古人之书盖尝求义理于六艺求事迹于二十一史求万物之情状于骚赋诗歌求载道之噐于汉唐宋数十家之文章编刬规橅涵揉櫽括放而之于诗若文之间有一言之合道一篇之追古则欣然以喜至于忘食若今之制举业固未尝屑屑以求工然亦以爲绎圣经尊王制无所苟而已矣近则深惟臧氏三不朽之防尝谓古之立言传世者非其有得于心则莫能爲也夫既有得于心矣虽有言可也如迁固荀扬韩欧之属是也夫既有得于心而有言者矣虽无言可也如鄙宗之叔度是也某之所见如此则视俗学弥不足好比来敓华践实玩思性理将求所以悟明其心而刚大其气以庶几于古之因文见道者寻绎久之亦复超然有见于文字语言之外始知近代河东余干新防姚江诸君子之理学门庭或殊而归趋则一世儒舍性命而谈事功舍事功而谈文章是以事功日陋文章日卑而诐淫邪遁之害浸浔及于政事而不可救盖天下之俇攘数十年于兹矣某也粗有识知上受罔极之恩于孔孟其敢贸贸焉以文人自居以富贵利达之学术自陷也哉重念有亲在堂不敢絶意进取是以前此虽经屡刖今年复随众入闱属有天幸得出于执事之门下恭惟执事慈祥温惠之风清刚防洁之守仁思义色洽被逺迩精鍳妙裁出于诚一闱中所得率多竒杰卓诡之士而某也误厠其间虽其才不及韩愈苏轼万分之一而执事固今之陆宣公欧阳永叔也世有出于宣公永叔之门而不竭蹙奔走叩首函丈者欤惟是榜放后即入南都台旌已还青阳是时督学有亲供之召留江隂者旬日家间复苦京报人乞索无厌乃归家匄贷以遣之而某窭贫人也计偕在即束装愈难因思执事所以甄收某者固将望其有所树立不必责以区区之一见是以不揆狂斐輙敢自述其爲学爲文之本末与其所以不得至前之故熏沐拜书仰尘清覧伏惟执事悯其至诚也教且诲焉而不督其罪戾幸甚幸甚
　　寄弟伟恭书【癸未】
　　日夜盼家信惟得七月中一书心悬旌如也知弟疟疾此不足爲苦想当旋复矣闻学台岁试在九月中名次前后不知何如然此直呼卢耳大得失不足介意况小者乎吾廷试传胪时见鼎甲三君先上不得与者皆啧啧称叹以爲登僊甚者至闭目摇头不欲观盖羡之之极也吾此时叹息无限昔人谓状元三年一个何足多慕此至言也天地间自有数千年一个者数百年一个者数十年一个者今人必不肯爲数千年一个的人而必欲爲三年一个的人已是可笑况数月一个又何足言乎我近来意味甚襍皆因终日尘中打滚自然多走失处又胸中横着一个矜字眼见他人品骨不如我议论不如我意思识见皆不如我便不免有轻蔑时俗之意坐此学力不进然在宠辱塲中壁立如铁则中庸所谓爵禄可辞也白刃可蹈也吾自信无憾推而言之天下国家可均也惟中庸不可能则过此以往并心一向犹冀天不絶我圣人不弃我我终有一立脚去处不徒然而已弟勿忧我爲俗人所牵俗念所染但我当自忧处正多耳客中无可与语时时仰屋而叹买得唐诗数册小鉴一部诵诗至精防入妙处读史至得失分际处窗虚月白风急天髙自叹自嗟自解自防真恨千载上人不从吾游也目前诸公赴馆选如渴我掩门独坐既无得理而又悬念吾父吾弟且思昔人三防在浅土便汲汲求人以今吾家二防未举爲人子孙者亦安得旷年于外乎兵寇交讧倐忽不常今趁道路略通时且归料理一番如可终不谒选竟作隐局否则打叠身子干净然后入世鼠肝虫臂随所赋予可也不知此念是否大约出月初十前后可以戒行至迟则腊月二十后必到家矣半月中惟有召对一着爲意外覉留之计恐亦未必然矣世间事真不可做十分勘破可做者只有已分内事勉之勉之致意同好诸君厚积德而深养晦乃今日事也
　　答归元恭书
