　　钦定四库全书　　　　　集部四
　　止斋文集　　　　　　　别集类三【宋】
　　提要
　　【臣】等谨按止斋文集五十一卷宋陈傅良撰傅良有春秋后传已着録此集凡诗九巻文四十二巻其门人陈叔逺所编前后各有叔逺序所取断自乾道丁亥讫于嘉泰癸亥凡乾道以前之少作尽削不存其去取特为精审末为附録一卷为楼钥所作神道碑蔡幼学所作墓志叶适所作行状而又有杂文八篇缀于其后不知谁所续入据宏治乙丑王瓒序称泽州张璡欲掇拾遗逸以为外集其璡重刋所附入欤自周行已传程子之学永嘉遂自为一而傅良及叶适尤其巨擘本传称永嘉郑伯熊薛季宣皆以学行闻伯熊于古人经制治法讨论尤精傅良皆师事之而得季宣之学为多及入太学与广汉张栻东莱吕祖谦友善祖谦为言本朝文献相承而主敬集义之功得于栻为多然传良之学终以通知成败谙练掌故为长不専于坐谈心性故本传又称傅良为学自三代秦汉以下靡不研究一事一物必稽于实而后已盖记其实也当宁宗即位之初朱子以赵汝愚荐内召既汝愚与韩侂胄忤内批与朱子在外宫观傅良为中书舍人持不肯下其于朱子亦不薄然叶绍翁四朝闻见録称考亭先生晚注毛诗尽去序文以彤管为淫奔之具以城阙为偷期之所止斋陈氏得其说而病之谓以千七百年女史之彤管与三代之学校以为淫奔之具偷期之所窃有所未安独藏其说不与考亭先生辨考亭微知其然尝移书求其诗说止斋答以公近与陆子静互辨无极又与陈同甫争论王霸矣且某未尝注诗所以说诗者不过与平人为举子讲义今皆毁弃之矣盖不欲滋朱之辨也云云则传良虽与讲学者游而不渉植党之私曲相附和亦不渉争名之见显立异同在宋儒之中可称笃实故集中多切于实用之文而密栗坚峭自然髙雅亦无南渡末流冗沓腐滥之气盖有本之言固迥不同矣乾隆四十二年五月恭校上

　　止斋集原序
　　先生禀抱天頴硏尽学力据六经奥防执九经百家之辔俾环向以趍于一披剔文义躏借众纠究明帝王经世宏模而放于秦汉以下治乱兴衰之故独掲源要不牵多岐由是彰往考来黙察当世丕平之机深抱大业至于化裁推行不动声色使人囬心而向道者其纲领条目靡不该具盖尝忘寝废食审玩孰复庶几对越天地百世以俟后圣而不惑也虽言论未孚几进輙沮而志念回皇与物委蛇左推右挽旁接广诱其任重道逺终老未尝一日敢忘于斯焉呜呼盛矣邹鲁之统绪河洛之承续千载以来不知其能几见也执经户外方屦阗集片言落笔传诵震响塲屋相师而绍兴之文丕变则肇于隆兴之癸未屏居梅潭危坐覃思超诣絶轶学成道尊则防于乾道之丁亥博交遍騐洞碍融窒对防初第恳荩独到则盛于乾道之壬辰官太学倅闽府诋劾却扫勤十寒暑防绎文献宏纲具举则备于淳熙之丁未起守桂监持节湖南防涤拊摩民信有古百年之思郁乎湘山则騐于绍兴之庚戌召对光宗骤遇奨用侍立代言賛翊储邸次第蕴画庶将发挥则着于绍熙之癸丑宛转极諌徬徨乞身龙飞急召十旬乃罢爰抒旧志着于训传疾疢渐臻梁木竟殒则终于嘉泰之癸亥叔逺夙蒙挈防俾窥津涯蒿干鷃羽惕负大赐适当新义川涨群文猬兴畴昔愬析而致孤条之靡郁砥途之失榛乎是用敬辑遗稿寄诸琬琰傥开后哲庶弗沦坠矧韦布慕影响徧传或混幼作或杂真赝诡题丛帙诞弥遐陬轮耀掩汚理合厘别故今裒次防自梅潭丁亥之后凡为歌辞古律诗内外制奏状劄子表啓书简序记杂著祭文墓志行状总五十一卷即先生燕坐之斋以为集名若成书则有读书谱二卷春秋后传十二卷左氏章指三十卷周礼进説三卷进读艺祖皇帝实録一卷未脱稿则有诗训义周汉以来兵制皇朝大事记皇朝百官公卿拜罢谱皇朝财赋兵防秩官志藁别自为编附识其目庸熄淆乱先生名傅良字君举世系歴官具见于神道碑墓志铭行状云嘉定戊辰门人承直郎太学博士曹叔逺谨序
　　儒者之所难曰徳曰功曰言而已三者克具斯为儒者之盛逺而有以恢鲁邹所传之绪近而有以昭阐濓洛未启之机尚论其世有足征者此止斋陈文节公所以不可及也公少以斯文为己任强学笃志弗得弗舎其从郑景望薛季宣也以克己兢业为要其友张钦夫吕伯恭也则交致夫持敬集义之功涵养于致知之际躬行于自得之余刚毅乐易孝友恭肃融澈混成莫测涯涘是能立徳者矣其教人也诱掖不倦四方景从士习丕变随才有造其仕于外也事无细钜一裁以义劝善革奸缩用溥利卓越之绩传不絶书其在朝也正色谠论直前极陈扶翊大政匡持君徳欲进忠贤则黄冕仲朱元晦之迁改不与书行欲摧权奸则率逢原张子仁陈源之除目缴驳论奏茍禆社稷奋不顾身是能立功者矣读书有谱六经有论建隆有编毛诗有解诂春秋有后传左氏有章指周礼有进説制诰有集皇宋有大事记进读有艺祖实録周汉以来有兵制著书明道简册充栋是能立言者矣功在人徳言在己三者相须以为用者也瓉幸生公之乡屡尝诵读遗文而私淑之茫乎如隂阳之阖辟也浩乎如河海之防潏也灿乎如日星之炳耀也油乎如风云之流行也雄伟而不放精深而不晦驰轶而不迫起伏敛纵愈出愈驶引古质今涤冗为新错综万务体悉人情而归宿于至理盖不独绳矩之具而精粗隠显皆可以适天下之用自有文字以来学士大夫竦企倾动固其时乎非有本者其孰能之公淹贯六经包括百氏洞彻天人之奥而于歴代经制大法与夫当世制度沿革失得之故稽验钩索委曲该洽此岂泛然雕饬以骛于虚言者也公文散载于羣书遐陬寡陋未见有统彚为全帙者瓉近于秘阁録出公集五十二卷则向所尝诵读者百无一二存焉盖曹公所编止自梅潭丁亥之嵗而他作不入也治乙丑侍御史同年泽州张君伯纯往巡浙中因论鄊哲而于公尤致向慕瓉遂出示公集伯纯喜曰璡求公文乆矣而莫获见之是行未广也请得梓之以传且欲彚拾散逸以为外集伯纯积学善文风裁峻整在公若有宿契焉者宜其乐公之制作行于时也濒行属瓉序于集后瓉何人乃敢序公文哉况楼蔡水心诸公纪述备矣聊摭立徳立功立言之实以寓泰山乔岳企仰之意然公名髙招忌行方取诋君眷未几而辄沮道术畧伸而复斥不肯少贬以从流俗而所谓立功者未能充满其志而国运防以衰下矣可胜叹哉天佑斯文而使公之制作复行于时固世教之所闗也后学王瓉谨序

　　止斋集卷一　　　　　　宋　陈傅良　撰歌辞
　　暮之春六章章五句【先生有堂曰暮春御翰扁榜】
　　暮之春兮物维其嘉干际兮坤涯母将雏兮彼实者华鱼在藻兮燕子还于故家今者不乐兮云何
　　暮之春兮风日与柔桑女兮南畴相尔夏畦兮悲秋斵冰兮长夜无裘今者不乐兮何求
　　暮之春兮雍雍熈熈尧裳兮舜衣五弦之琴兮一防曽不知结绳与秉钺兮何时瞻言千载兮忽焉其逺而山有龟防兮水有沂天未防斯文兮在兹二三子兮皇皇欲何之鼓瑟兮为谁舍此兮吾将安归
　　止斋兮年年室环堵兮两山有川鸥鹭巢檐兮圆荷田田岂无芳草兮杜鹃世防孔子兮独抱乎韦编
　　云汉之章兮光烛天朝宗兮列仙胡归晚兮病骨拳然与客歌商兮忘食与眠防尔何如兮吾未知其孰贤
　　西庙招辞【并序】
　　淳熈十有三年二月知瑞安县事公非先生之曽孙刘龟从再作西庙以祠宣和县令王公经国于其落也邑人陈某为作迎神一章使工歌之经国字公济其守御事许忠简公尝识之故不叙
　　君昔兮胡不去斯君今兮胡不来斯文栋兮雕楣岘山之麓兮江圻防酒牲兮洁肥皓遗老兮须眉坎击鼔兮吹篪君勿来兮迟迟山蜿蜿兮榕隂萎蕤江水清夷兮无蛟与螭君之来兮可以娯嬉君不共安兮谁与共危鸣玉兮衮衣彼亦君兮一时莾中州兮为夷匪冡则庙兮牛羊累累尚我民兮有知子子孙孙兮永依揆之兮以是非君舍此兮安之田有畔兮泽有陂桑稻兮诞弥懐不报兮忸怩日早暮兮夜何其犹容与兮谁湏
　　古诗
　　落叶一首别诸友客新昌
　　落叶一二数遥夜忽己午百年能防何望道犹未睹清扬得邂逅简易从险阻明朝买扁舟万议付风雨新诗话别恨为我谢酸楚苍苍嵗后竹白首见肺腑要有不别时尸居一环堵
　　读范文正公神道碑有感佚事
　　武侯不可致元徳造其庐公在衰绖中乃上时政书维时君臣定事与草昧殊出处千载同岂必名迹如行伍防大将寒饥得名儒推毂天下士百年用其余生平慕河汾未许王魏俱慇懃八司马意独何区区自古朋党论消复莽无期谁令羣疑亡韩富及有为惜哉公不见功名止西陲
　　送国子监丞顔几圣提举江东分韵得动字
　　长安夜未艾客枕厌羣动平明视天阙剑佩鸣珌琫学省虽冷官上意向周孔六飞自天下多士溘云滃饔飱燕苹鹿道义发醯蠓方将属耆英髙举出埃塕离觞春骀荡归路月膧胧勲业当及时千年付南董
　　懐寜国守郑少卿景望
　　一日一度过短墙三日风雨挠我觞天球河图在东序保障茧丝分晋阳我欲从之苕水逺搔首不见月满廊笑看儿女罗列拜老人星下烧夜香
　　送宋国博叅议江东分韵得夜字前年上幸学一时同舍今去畧尽为之怅然
