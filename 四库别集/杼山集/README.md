　　钦定四库全书　　　　集部二
　　杼山集　　　　　　　别集类一【唐】
　　提要
　　【臣】等谨案杼山集十巻唐僧皎然撰案唐书艺文志皎然字清昼湖州人谢灵运十世孙居杼山顔真卿为刺史集文士撰韵海镜源预其论著贞元中取其集藏集贤御书院刺史于頔为序此集巻数与唐志合頔序亦存葢犹　本前有赞寜所为传葢自髙僧传録入末有集外诗则毛晋所补缉也皎然及贯休齐已皆以诗名今观所作弱于齐已而雅于贯休在中唐作者之间可厠末席集末附载杂文数篇则聊以备体非其所长矣别本附刋杼山诗式一巻按唐志昼公诗式诗评皆载文史类中不附本集今亦析出别着録焉乾隆四十一年五月恭校上

　　杼山集原序
　　诗自风雅道息二百余年而骚人作其旨愁思其文婉丽亡楚之变风欤至西汉李陵苏武始全为五言诗体源于风流于骚故多忧伤离逺之情梁昭明所撰文选録古诗十九首亡其名氏观其辞盖东汉之世亦苏李之流也洎建安中王仲宣曹子建皷其风晋世陆士衡潘安仁扬其波王曹以气胜潘陆以文尚气胜者魏祖兴武功于二京已覆文尚者晋武亡帝图于刘渊肇乱观其人文兴亡之迹人焉廋哉人焉廋哉宋髙祖平桓定江表文帝继业五十年间江左宁谧魏晋文章欝然复兴康乐矦谢灵运独歩江南俯视潘陆其文炳而丽其气逸而畅驱风雷于江山变晴昏于洲渚烟云以之惨淡景气为其澄霁信江表之文英五言之丽则者也迨于齐世宣城守谢晖亦得其辞调涵于气格不侔康乐矣梁陈已降虽作者不絶而五言之道不胜其情矣有唐呉兴开士释皎然字清昼即康乐之十世孙得诗人之奥旨传乃祖之菁华江南词人莫不楷范极于缘情绮靡故辞多芳泽师古兴制故律尚清壮其或发明理则深契真如又不可得而思议也贞元壬申嵗余分刺呉兴之明年集贤殿御书院有命征其文集余遂采而编之得诗笔五百四十六首分为十巻纳于延阁书府上人以余尝着诗述论前代之诗遂托余以集序辞不获已畧志其变上人之植性清和禀质端懿中秘空寂外开方便妙言説于文字了心境于定惠又释门之慈航智炬也余游方之内者何足以扣关谢氏世为诗人岂佛书所为习气云尔于頔撰






　　钦定四库全书
　　杼山集巻一　　　　　唐　释皎然　撰
　　五言奉酬于中丞使君郡斋卧病见示一首
　　宿昔祖师教了空无不可枯槁未死身理心寄行坐仁公施春令和风来泽我生成一草木大道无负荷论入空王室明月开心胸性起妙不染心行寂无踪若非禅中侣君为雷次宗比闻朝端名今贻郡斋作真思凝瑶瑟髙情属云鹤抉得骊龙珠光彩曜掌握若作诗中友君为谢康乐盘薄西山气贮在君子衿澄澹秋水影用为字人心群物如鳬鹥游翺爱清深格居第一品髙步凌前躅精义究天人四坐听不足伊昔栁太守曾赏汀洲苹如何五百年重见江南春公每省往事咏歌懐昔辰以兹得髙卧任物化自淳还因访禅隠知有雪山人
　　附郡斋卧疾赠昼上人【上人早名皎然晚字画】
　　湖州刺史于頔
　　夙陪翰墨徒深论穷文格丽则风骚后然公我词客晚依方外友极理探精赜曶合南北宗书公我禅伯光明性不染故我行贞白随顺令得解故我言芳泽霅水漾清浔呉山横碧岑含珠复蕴玉价重双南金的皪曜竒彩凄凄流雅音商声发楚调调切谐瑶琴呉山为我髙霅水为我深万景徒有象孤云本无心众木岂无声椅桐有清响众耳岂不聆钟期有真赏髙洁古人操素懐夙所仰觌君水雪姿祛我淫滞想常吟栁恽诗苕浦乆相思逮此逺为郡苹洲芳草衰逢师年腊长值我病容羸共话无生理聊用契心期
　　五言赠李中丞一首
　　深沉阃外畧奕世当荣寄地裂大将封家传介珪瑞至今漳河俗犹受仁人赐公初镇惟邢决胜无精兵重围逼大敌六月守孤城政用仁恕立恩繇赏罚明遂令麾下士感徳不顾生于时闻王师诸将兵颇黩天子狩南汉烟尘满凾谷纯臣独耿介下士多反覆明公仗忠节一言感万夫物性如蒺防化作春兰敷见説金被烁终期玉有瑜移官万里道君子情何如伊昔避事心乃是方袍客顿了空王旨仍髙致君防安知七十年一朝值宗伯言如及清风醒然开我懐宴息与游乐不将衣褐乖海底取明月鲸波不可渡上有巨蟒吞下有毒龙防一与吾师言乃于中心悟咄哉防防子胡为自尘汚五言杼山禅居寄赠东溪呉处士冯一首
　　青云何润泽下有贤人隠路入菱湖深迹与黄鹤近野风歌白芷山月揺清轸诗祖呉叔庠致君名不尽身当青山秀【诗曰家住青山下青山有呉筠故宅后改为呉筠山】文体多郢声澄澈湘水碧泬寥楚山青时人格不同至今罕知名昔贤敦师友此道君独行既得庐霍趣乃髙雷逺情别时春风多扫尽雪山雪为君中夜起孤坐石上月悠然遗尘想邈矣逹性説故人不在兹幽桂惜未结
　　五言妙喜寺髙房期灵澈上人不至重招之一首
　　晨起峰顶心懐人望空碧扫雪开寺门洒水浄僧席言笑形外阻风仪想中觌驰心惊叶动倾耳闻泉滴岂虑咆虎逢乍疑崩湍隔前期或不顾知尔隳常格如今谁山下秋霖步淅沥吾亦聊自得行禅荷轻防松声畅幽情山意导遐迹举目无世人题诗足竒石贫山何所有特此邀来客
　　五言奉和薛员外谊赠汤评事衡反招隠之迹兼见寄十二韵
　　喜友称髙儒旷懐美无度近为东田诱遂耽西山趣庭有介隠心得无云泉误府公中司贵频贻咫尺素郡佐仙省髙亦赠琼瑶句诮兹长往志纡彼独游步禅子方外期梦想山中路艰难亲稼穑晨夕苦烟雾曷若孟尝门日荣国士遇铿锵聆绮瑟攀折迩琼树幽践随鹿麋乆期怨蟾兔情同不系舟有迹道所恶
　　五言荅黎士曹黎生前适越后之楚
　　楚木纷如麻髙松自孤直愿得苦寒枝与君比顔色故乡渺天末羇旅沧江隅委质在忠信苦心无变渝何繇表名义赠君金辘轳何以美知才投我悬黎珠遽为千里别南风思越絶爱君随海鸥倚棹宿沙月不栖恶木上肯蹈巴蛇穴【台人表朝逆命故君不践其土】一上萧然峰拟踪幽人辙晨兴独西望郢水期泝沿夜到洞庭月秋经云梦天黎生知吾道此地不澘然欲寄楚人住学拏渔子船奈何北风至搅我窓中游子动归思江蓠亦绵绵箧中封禅书欲献无繇縁岂乏晨风翼翻飞到日边
　　五言荅豆卢次方
　　吾爱道交论为髙贵世名昔称柴桑令今闻豆卢生彼生清淮气独钟文中彩近作公防诗如逢何栁在吾用古人耳采君四坐珍贤士胜朝晖温温无冬春朝晖烁我肌贤士清我神微尔与云鹄幽懐何繇申别来秋风至独坐楚山碧髙月当清防禅心正寂厯增波徒相骇人逺情不隔有书遗琼什以代貂襜褕风教凌越絶声名掩呉趋悬壁安可酧徙倚还踟蹰
　　五言荅苏州韦应物郎中
　　诗教殆沦缺庸音互相倾忽观风骚韵防我夙昔情荡漾学海资欝为诗人英格将寒松髙气与秋江清何必邺中作可为千载程受辞分虎竹万里临江城到日扫烦政况今休黩兵应怜禅家子林下寂无营迹隳世上华心得道中精脱畧文字累免为外物撄书衣流埃积砚石駮藓生恨未识君子空传手中琼安可诱我性始愿愆素诚为无鸑鷟音继公云和笙吟之向禅薮反媿幽松声
　　五言荅郑方囘
　　独禅外念入中夜不成定顾我顦顇容泽君阳春咏词贞思且逸琼彩何晖映如聆云和音况覩声名盛琴语掩为【去声】闻山心声宜听是时寒光澈万境澄以浄髙秋日月清中气天地正逺情偶兹夕道用增寥夐思君处虚空一操不可更时美城北徐家承谷口郑轩车未有辙蒿兰且同径庄生诫近名夫子罕言命是以耕楚田旷然殊独行萎蕤鸾鳯彩特逹珪璋性通隠嘉黄绮髙儒重荀孟世汚【平声】我未起道蹇吾犹病逸翮思防防潜鳞乐游泳宗师许学外恨不逢孔圣説诗迷頽靡偶俗伤趋竞此道谁共诠因君情欲罄
　　五言荅俞校书冬夜
　　夜闲禅用精空界亦清迥子真仙曹吏好我如宗炳一宿觌幽胜形清烦虑屏新声殊激楚丽句同歌郢遗此感予懐沉吟忘夕永月彩散瑶碧示君禅中境真思在杳防浮念寄形影遥得四明心【荅四明】何须蹈岑岭诗情聊作用空性惟寂静若许林下期看君辞簿领
　　五言妙喜寺逹公禅斋寄李司直公孙房都曹徳裕従事方舟顔武康士骋四十二韵
　　我祖传六经精义思朝彻【庄子曰守之九日而后能外生外生而后能朝彻】方舟颇周览逸书亦备阅墨家伤刻薄【墨流刻薄而不仁可以理身不可以济世】儒氏知优劣弱植庻可雕苦心未尝辍中年慕仙术永愿传其诀嵗驻若木景日飡琼禾屑婵娟羡门子斯语岂徒设天上生白榆葳蕤信好折实可反柔顔花堪养髪求之性分外业弃金亦竭药化成白云形雕辞素穴【素穴山名】一闻西天防初禅已无热【第四禅中有天名无热无三灾之患故以为名今初禅即得葢诗人美已证超出常】涓子非我宗然公有真诀却寻邱壑趣始与缨绂别野饭敌膏梁山楹代藻棁与君北岩侣游寓日常昳静对春谷泉晴披阳林雪境清觉神王道胜知机灭诣寂长杳防忘归暂采撷物生岂有心丽容俟予别桂子何蓂苓琪葩亦皎洁此木生意髙亦与众芳列赏墨识屡换省躬悟弥切防尚若不亏足以全吾节北风吹蕙带萧寥闻蜻蛚宿昔庐峰期流芳已再歇不有清屏鉴使我商絶愿寄千里心月髙不可掇优游邦之直逺矣踵前烈立俗忘毁誉遇物遗巧拙真气独翛然轩裳讵能绁都曹风韵整纲纪信明决于交必倾写立行岂矜伐政与清渭同分流自澄澈裴侯资亮直中诚岂徒説古人比明义清士愿交结温温躬珪彩终始声不缺顔生烱介士有志不可越仗义冒险难持操去淄涅世论髙二贤【即鲁公犹子也】贤贤继前哲四子遭明盛裒然皆秀杰理名虽殊迹悟道宁异辙爱尔竹栢姿为予寒不折
　　五言遥酬袁使君髙春暮行县过报徳寺见懐
　　江春行求瘼偶与真境期见説三陵下前朝闻佛祠停舟仰丽刹绣组发香墀咫尺空界色天人花落时盛游限羸疾悚踈瞻旌旗峰翠羡闲步松声入遥思素髙淮阳理况负东山姿迨此一登览深情见新诗
　　五言冬日遥和卢使君幼平綦母居士游法华寺髙顶临湖亭
　　仁坊标絶境防守蹑髙踪天晓才分刹风传欲尽钟城中归路逺湖上碧山重水照千花界云开七叶峰寒空艾绶满空翠白纶浓逸韵知难继佳游恨不逢仍闻抚禅石为我乆従容
　　五言秋日遥和卢使君游何山寺宿上人房论涅槃经义
　　江郡当秋景期将道者同迹髙怜竹寺夜静赏莲宫古磬清霜下寒山晓月中诗情縁境发法性寄筌空翻译推南本何人继谢公
　　五言酬秦山人系题赠
　　出斋步杉影手自开禅扉花满不汚地云多従触衣思山石藓浄欵客露葵肥果得宗居士论心到极微五言奉酬袁使君髙寺院新亭对雨【其亭即使君所创】
　　兹亭迹素浅胜事并随公法界飘香雨禅窓洒竹风浮烟披夕景髙鹤下秋空防寂四山乆宁期此防同五言奉酬顔使君真卿见过郭中寺寺无山水之赏故予述其意以荅焉
　　州西栁家寺禅舍隠人间证性轻观水栖心不买山履声知客贵云影悟身闲彦防前贤事方今可得攀五言酬乌程杨明府华雨后小亭对月见呈
