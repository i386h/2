　　御制读髙启威爱论
　　髙启着鳬藻集首章即威爱论读之卒篇叹其知一而未知其二也夫兵者非常用之事以仲康之贤讨羲和之乱承命者犹恐其威之不立而功之不成启所云好生恶死人之至情有威则怯者勇无威则勇者怯其言皆是也至谓使国君而知此则国可以治天子而知此则天下可得而理则失之甚矣夫治国平天下非若用兵偶行之事也御众以寛寛则得众犹恐吾仁之不能广被焉孰谓徒作其威使下无所容而可以治国平天下哉且自古亡国者多矣失之威者其亡速失之寛者其亡缓致亡虽同然缓速之间斯有别矣吾非谓当失之寛盖辟启谓宜威克之言示有君人之责者当体天心仁民爱物虽辟以止辟仍存哀矜恻怛得情勿喜之意非不用威而不可徒恃其威威者猛之类爱者寛之施寛以济猛威爱并行或者其庶几耳余故谓启知其一而未知其二论者不可不察也

　　书简
　　【臣】等谨案鳬藻集五巻明髙啓撰啓有诗集别着録唐时为古文者主于矫俗体故成家者蔚为钜制不成家者则流于僻澁宋时为古文者主于宗先正故欧苏王曾而后沿及于元成家者不能尽辟门户不成家者亦具有典型啓在明初其诗才藻富健工于摹古为一代巨擘而古文则不甚著名然生于元末距宋未逺犹有前辈轨度非洪宣以后渐流为肤廓冗防号台阁体者所及是集不知谁所编以其诗集例之殆亦啓所自定末有魏夫人宋氏墓志铭魏夫人者苏州知府魏观母也按明史本传啓坐为观作上梁文见法则为其末年之作葢平生古文尽于此集矣初无刻本周忱为苏州巡抚时始得抄本于郡人周立立之姑即啓妇也正统九年监察御史钱塘郑士昻又得本于忱因命教授张素校刋之而忱为之序此本为雍正戊申桐乡金檀所刻即因郑本而正其譌多所校正檀即注啓诗集者故并刻是集成一家完书云乾隆四十三年九月恭校上

　　威爱论
　　书曰威克厥爱允济爱克厥威允罔功或以爱诚有以结扵人者则趋事赴功有不期然而然何以威为哉予曰不然此御军之要也盖爱胜则姑息威胜则严眀侯知其然故扵誓师之际深警之若此欲其恐惧而用命也呜呼以仲康之贤讨羲和之沉乱其必济可知矣然犹恐其威之不立而功之不成而况后世之众人欲从事扵强敌者哉夫三代之兵也出扵民居则习其政教而知义出则闻其节制而知法皆有尊君死上之心赴公战如报私仇者抑且有谓孥戮之刑弗朂之戒焉近世之聚而为兵者非田野之惰夫则乡里之恶少亡命行剽椎埋鼓铸之流也政教不习而节制不闻茍无威以临之则其桀傲狠戻悻悻自肆者可胜道哉今之人家有骄子非其子之性骄也爱之而致其骄也教之而不从役之而不动扵是有悖逆干犯之患矣若小过则训之大过则杖之子其有骄乎将之御三军者固无异扵是然将之扵三军又非若父子之有天性之亲而不可一日离者则爱之其可过扵威乎况战者以驱之扵死也好生恶死人之至情非得尊君死上之人则视白刃之交扵前流矢之集扵左其不震慴辟易颠倒而奔走者防希矣故兵法曰畏敌者不畏我畏我者不畏敌何以使其能畏我也杀之而已矣盖非嗜杀而自残也恐其畏敌而先奔敌或乗而覆之是举军而弃之扵敌其自残不已多乎故爱其子者贼其子杀其军者全其军设使两军勇怯相若一乐其将之寛一畏其将之严卒然遇扵原野之间援桴鼔之则严者莫不奋戈而争前而寛者或有一二遁矣何则彼恃爱之而不杀也故有威则怯者勇无威则勇者怯且立威者非欲其若杨素之求人之过而杀之也亦曰令之严而罚之果不为煦煦姑息之计耳古之豪杰所以能使士卒畏之若鬼神之不可犯纳之扵死而不避投之扵险而无所辞百战百胜功立扵当时而名存于后世者用此道也或又曰然则威可以无爱矣乎曰何可以无爱也専爱则亵亵则怠専威则急急则怨怨与怠其败一也故爱而恐其至于怠也则摄之以威而作其气威而恐其至于怨也则济之以爱而收其心爱非威恩不加威非爱势不固威爱之道所以兼施并行而不可偏废者也虽然岂特为将之事哉使国君而知此则国可以治天子而知此天下可得而理矣
　　四臣论
　　古之所以能国者有四臣焉何谓四臣曰社稷之臣腹心之臣谏诤之臣执法之臣也何谓社稷之臣忠荩孚扵上下威望加扵内外敌国闻之而不敢谋奸宄畏之而不敢发正色立朝招之不来而麾之不去若汉汲黯吴张昭唐郭子仪是也何谓腹心之臣识足以达天下之机畧足以济天下之业从容帷幄谋成而羣臣不知计定而将军不闻若汉良平魏荀彧秦王猛是也何谓谏诤之臣匡君之非而纳君扵善不阿顺以取容不迎合以求恱正言不廽触犯忌讳雷霆发扵上而不惊鼎镬具扵前而不顾若唐魏徴褚遂良张九龄是也何谓执法之臣直道而行不惮权贵逢奸必举遇恶必击使豺狼狐狸屏息而不敢动若汉王章盖寛饶唐宋璟是也盖社稷之臣以忠腹心之臣以智谏诤之臣以直执法之臣以刚此四臣者国之不可以一日无者也夫以匹夫之取友尚有能死义者能忠谋者能责善者能御侮者而况扵国君乎而况扵天子乎故国无社稷之臣则无与抗大难无腹心之臣则无与图大功无谏诤之臣则无与格大过无执法之臣则无与除大奸无与抗大难必危无与图大功必败无与格大过必昏无与除大奸必弱故古之兴者未尝无四臣而亡者未尝有四臣也呜呼四臣者岂真不易得耶君无优养作起之术尔故所以待社稷之臣者当尊以礼髙爵而重禄之使危言不能中细故不能疎则彼必以社稷之忧为已忧社稷之辱为已辱毅然以身徇节而不变而大难可抗矣待腹心之臣者当推以诚畧去苛礼示之坦然食则同器坐则促席所言无不用所欲无不与则彼必竭思虑之精效胜负之计而大功可图矣待谏诤之臣者则当纳以寛凡有所论奏停舆以受之赐帛以旌之虽激切不怒虽指斥不罪则彼必务尽直心政事之阙日闻聪眀之道益广而大过可格矣待执法之臣当假以威不以私爱挠其权不以谴辱挫其气使强者不敢伤防者不敢毁则彼必竦涌风生刺举无避以尊朝廷之势而大奸可除矣若或弃忠而擅智恶直而害刚平居而上唱下和相聚自贤势孤而不知机去而不察政失而不闻威削而不悟及一旦临变茫然而无所救岂不可哀也哉诗曰如彼泉流无沦胥以败予恐后世之君无四臣而致其败也

　　逰天平山记
　　至正二十二年九月九日积霖既霁灏气澄肃予与同志之友以登髙之盟不可寒也廼治馔载醪相与诣天平山而逰焉山距城西南水行三十里至则舍舟就舆平林浅坞间道旁竹石蒙翳有泉伏不见作泠泠琴筑声予欣然停舆听乆之而去至白云寺谒魏公祠憇逺公庵然后由其麓攀杙以上山多恠石若卧若立若搏若噬蟠拏撑拄不可名状复有泉出乱石间曰白云泉线脉萦络下坠扵沼举瓢酌尝味极甘冷泉上有亭名与泉同草木秀润可防可息过此则峰廻磴盘十歩一折委曲而上至扵龙门两崖并峙若合而通窄险深黒过者侧足又其上有石屋二大可坐十人小可坐六七人皆石穴空洞广石覆之如屋既入则懔然若将压者遂相引以去至此盖始及山之半矣乃复离朋散伍竞逐幽胜登者止者哦者啸者惫而喘者恐而咷者怡然若有乐者怅然俛仰感慨若有悲者虽所遇不同然莫不皆有得也予居前益上觉石益恠径益狭山之景益竒而人之力亦益以惫矣顾后者不予继廼独褰裳奋武穷山之髙而止焉其上始平旷坦石为地拂石以坐则见山之云浮浮天之风飂飂太湖之水乎其悠悠予超乎若举泊乎若休然后知山之不负扵兹逰也既而欲下失其故路树隐石蔽愈索愈迷遂困扵荒茅丛篠之间时日欲暮大风忽来洞谷谽呀鸟兽鸣吼予心恐俯下疾呼有樵者闻之遂相导以出至白云亭复与同逰者防众莫不尤予好竒之过而余亦笑其恇怯颓败不能得兹山之絶胜也于是采菊泛酒乐饮将半予起言扵众曰今天下板荡十年之间诸侯不能保其国大夫士之不能保其家奔走离散扵四方者多矣而我与诸君防在上者之力得安扵田里抚佳节之来临登名山以眺望举觞一醉岂易得哉然恐盛衰之不常离合之难保也请书之扵石眀年将复来使得有所考焉众曰诺遂书以为记
　　生白室记
　　庄周氏之言曰瞻彼阕者虚室生白谓人能遗耳目去心意而任夫性则道集至虚之宅而纯白生焉四眀陈君徳眀恱其说乃以生白名所寓之室介友人求予记之予尝读周之书观是説者虽仲尼所以告顔子盖寓言耳其义虽美然未能尽合乎圣人也陈君岂将学者邪周之道盖欲放心自得之塲以与物寘所谓逰方之外者也君今筮仕昌朝出赞宥府简牍填委实待剸裁而目欲无所视耳欲无所听而心欲无所思能乎不能也君既不能为之则吾亦不能言之矣请言其可能者以记君室可乎夫心之体本虚有不虗者物之窒也物非能窒之也诱于物而为之累也故圣人教人目不能使无视能勿视扵邪耳不能使无听能勿听于淫心不能使无思能勿思扵妄而已尔茍三者之用皆出扵理而不私则虽日与物接其外蔽交而中之虚自若也吾虚既存然后光眀洞彻昭然而不昧者发焉烛至幽而不遗察至隠而能着此则眀而诚诚则眀之道也又岂务扵虚寂而无为扵世者之事哉君好学善辨尝燕休是室之中尚能虚心而观以审其取舎之防也夫
　　蜀山书舎记
　　蜀山书舎者友人徐君幼文肄学之所也幼文尝自吴兴以书抵予曰吾山在城东若干里吾屋在山若干楹吾书在屋若干巻山虽小而甚美屋虽朴而粗完书虽不多而足以备阅吾将扵是卒业焉子幸为我记之予惟古之君子所取以成其学者无常物所居以致其学者无常地也故弁裳之扵容珩瑀之扵歩豆笾之扵陈琴瑟之于乐弓矢车马之扵服度量权衡之扵用凡接扵物皆学也岂専扵六籍之内哉往于田入于市处于戸庭览于山川立于宗庙朝廷逰于庠序军旅凡履之地皆学也岂限扵一室之间哉后世讲学之道既废而人之不能然也有志者始各占山水之胜筑庐聚书而读之虽其以学之者异乎古然凡事之理与夫羣圣贤修已治人之要实皆不出扵书况安僻阻之区絶纷嚣之役得一肆其力扵是则其至扵成就岂不反有易者哉今幼文以方壮之齿有可用之材而不急进取益务扵学以求其所未至岂非有志之士哉而予也北郭之野有土东里之第有书皆先人之遗也遭时多艰茀秽扵榛芜残壊扵尘蠧伥伥焉日事奔走而不知返则其荒陋宜有愧扵幼文矣尚能为是记乎然而书此而不辞者盖姑复幼文之请亦因以自厉焉
