　　剡源集　　　　　　　别集类四【元】
　　提要
　　【臣】等谨案剡源集三十巻元戴表元撰表元字帅初一字曽伯庆元奉化人宋咸淳中登进士乙科除建康教授迁临安又迁行户部掌故国子主簿皆以兵乱不就元大徳中以荐信州教授调婺州移疾归再以修撰博士荐不起终于家事迹具元史儒学传表元所着剡源集明初上于史馆宋濓曽序而刻之凡二十八巻其板乆佚此本乃嘉靖间四明周仪得其旧目广为搜辑厘为三十巻表元后裔洵复梓行之王士祯居易録称海寜刻剡源集四巻乃黄宗羲所选録非完书也表元少从王应麟舒岳祥游学问渊源具有授受顾嗣立元诗选小传称宋季文章气萎薾而词骫骳帅初慨然以振起斯文为已任其学博而肆其文清深雅洁化朽腐为神竒间事摹画而隅角不露尤自秘重不妄许与至元大徳间东南之士以文章大家名重一时帅初一人而已又引宋濓之言曰濓尝学文于黄文献公公于宋季词章之士乐道之而不已者惟剡源集先生为然云云于元人之中推之独至今观其诗文信嗣立所论不诬也乾隆四十六年六月恭校上

　　剡源文集原序
　　濂尝学文于黄文献公公于宋季词章之士乐道之而弗已者惟剡源戴先生为然濂因日购先生之文絶不能以多致防有诏纂脩元史命濂总裁其事事有阙遗者遂以上闻遣使访于郡国窃以谓先生著作有闗于胜国宜多乃属使者入鄞徧求之鄞先生乡国庶几有得之者曾未防何果以剡源文集二十八巻来上濂始获而览焉因而作曰辞章至于宋季其敝甚矣公卿大夫视应用为急俳偕以为体偶俪以为竒腼然自负其名髙稍上之则穿凿经义櫽括声律孳孳为华世取宠之具又稍上之摽掠前修语録佐以方言累十百而弗休且曰我将以眀道奚文之为又稍上之骋宏慱则精粗杂揉而略绳墨慕古奥则删去语助之辞而不可以句顾欲矫敝而其敝尤滋私自念词章在世如日月之丽乎天虽疾风暴雨动作无时将不能蔽蚀其精眀独怪夫当时之士奚为乏一人障其狂澜耶复念豪杰之士何代云无区区所见孤陋故鲜能知之非诚然也及览先生之作新而不刋清而不露如青峦出云姿态横逸而连翩弗断如通川萦纡十歩九折而无直泻怒奔之失呜呼此非近于所谓豪杰之士耶盖先生七嵗即知攻文咸淳中入太学以三舎法升内舎生既而试礼部第十人登进士乙科调教授建康府又迁临安教授行户部掌故皆不就防宋亡为元执政者荐之起为信州教授先生年已六十一矣寻迁婺州以疾辞后六年终于家初先生既擢第悯宋季词章之陋即濯然自异乆之四方人士争相师法故至元大徳间东南文章大家皆归先生无异词先生之没仅六十年巳罕有知其名若字者殊可哀也濂在史局既命彚入儒学传中及司业成均复将録其剡源集者归以示诸人而先生之乡有夏君阅者来为国子学正方与先生之孙资先谋刻于梓夏君遂以题词为请且谓知先生之深者惟黄文献公公既不可作子幸无让于是忘其僣逾而为序之如此呜呼丰城之劒荆山之玉纵埋没泉壤为已乆神光上贯于霄汉者终弗能掩也其先生之文之谓乎先生讳表元字帅初一字曾伯庆元奉化州人洪武四年秋八月望日金华后学宋濓谨序












　　剡源文集自序
　　先生姓戴氏名表元字帅初一字曾伯其世谱可知者六代祖居奉化县小方门三传而徙坊墎又再传而徙剡源之榆林先生生淳祐甲辰五嵗知读书六嵗知为诗七嵗知习古文十五始学词赋十七试郡校连优补守六经谕即厌去逰杭作书言时政激摩公卿大人无所避杭学每嵗贡士得三百员试礼部中者十人入太学谓之类申二十六嵗巳巳用类申入太学眀年庚午试中太学秋举嵗终挍外舎生试优升内舎辛未春试南省中第十名五月对防中乙科赐进士及第授廸功郎升学教授癸酉冬起升及乙亥春以故归旧庐改杭学教授辞不就既而以恩转文林郎都督掾行户部掌故国子主簿防兵变走避邻郡及丁丑嵗兵定归鄞至是三十四嵗矣家素贫燬刼之余衣食殆絶乃始専意读书授徒卖文以活老稚鄞居度亦不可乆遂买榆林之地而庐焉如是垂三十年执政者知而怜之荐授一儒学官因起教授信州噫老矣大徳丙午归自信州时体气益衰而婚嫁渐已毕即以家事属诸子使自力业以治养具然性好山水每杖防东逰西眺不十里近才数百歩不求甚劳意倦輙止忘懐委分自号曰剡源先生因以名其集或称质野翁充安老人云

　　仁夀殿记【代阮侯】
　　庆元路奉化县学仁夀殿成县尹丁济以书谂于同知总管府事阮麟翁曰愿有所述麟翁曰尝学诗而窃闻诗之说曰天下之生未有无本者也万物本乎天为之子者本乎父母而民本乎君是三者仁之至也而有报物莫夀乎天天之夀不待物愿之而夀也然而孰不愿之若人子之夀其父母则茍可以愿者无不至矣此报之道也昔者周之始兴其诗有七月周之极盛其诗有鳬鹥既醉周至于文武成康而仁其民至矣其民如天如父母以懐之懐之不巳则咏歌之咏歌之不已则相与持酒醪饮食具乐舞设祝嘏以夀之此人之情亦报之道也然独至于鹿鸣四牡皇皇者华棠棣伐木之燕乐天保之归美则疑若君私仁其臣臣私夀其君而民无所与乆而知之则鹿鸣诸诗之所燕乐者即周之亲贤中外诸臣所托命以仁其民者也天保之归美则诸臣采诸其民而献之即其平时持酒醪饮食具乐舞设祝嘏而欲以寿其君者也噫乎休哉洪惟皇元继宋御宇奄有诸夏櫜弓偃钺而天下戢其威蠲征缓狱而天下颂其平惟兹海邦逺在数千里外慈仁所加无间轩陛故诏书每下斥卤之氓岩穴之叟投缗植耒驩喜出听诚可谓千载一时太平混合之嘉防而麟翁及济适于此时分受国邑何荣如之始麟翁佐治宣城济实为附属迨麟翁之来复相际接诚嘉其亷明肃给之材而出之岂弟乐易庶防乎知古诸侯之职而无愧于诸侯之所称说今又因斯民休养之余力不烦寸符尺棰而斯宫鼎成以彰尊君报本之意麟翁安敢不发扬而褒侈之乎若夫作为一代声歌以追继风雅使天子仁声令闻无疆无极则学馆诸生必有美其事者麟翁愿窃有俟焉至元壬辰秋七月望
　　奉化州学兴筑记
　　古之齐民一名为儒则其人所以自待与官府所以待之皆异至于学校虽有常居而发政出师养老习射献馘聴狱之类无不在焉然方其盛也有优礼以乞其一言俗之既衰乃或欲毁之而杜其议政俗之益衰上下始専守夫子遗言以为法故有庙以严鬼神而有学以眀礼乐吾奉化犹为县也庙学栋宇防为兵废襄贲丁公济来为尹兴之县既升为州相距不十年而垣籓不修卫防旷空荆芜被之蹊隧生焉某郡王公某来为守悒然叹曰兹非吾职乎即与同寮议兴之计其役赋板防均丈仞章逢乐输胥徒驩从不累旬百堵齐立于是增绘象施蔽帷鼓箧之堂嵩呼之殿风雩之亭童衿之舎缺完仆兴罅补茅塞闯防有禁观眺有节偃憇有适瞻展有敬重扄穹屏修衢清浸于于相仍云行星辉噫乎美哉州之耆老遂相与燕乐而谋勒文以颂公之贤着公之恵余自龆龀实尝从父兄居游庠序间见魁儒钜公无虑百数皆修衣冠隆阀阅嵗时燕毛序坐谈古今乆近文献亦或雌黄当世人物孩穉軰立听不倦归必充然有得以余之愚至今犹能缕缕记忆本末如昨日也每课试县大夫亲命题第赏格慈爱教督如子弟及以事相见酬答如客于时风淳气厚上下情义周洽与前所称古俗殊为未逺顾孪然眉顔亦复苍皓犹幸及身为贤侯之氓狎见庙学之兴而咏歌徳化之成良可自庆因为摭实纪载如右而并缀所闻见一二以励吾党亦务谨重修饬以称官府见厚之意云大徳五年嵗在辛丑冬十一月望
