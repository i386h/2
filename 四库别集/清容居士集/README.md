　　钦定四库全书　　　　　集部五
　　清容居士集　　　　　　别集类四【元】
　　提要
　　【臣】等谨案清容居士集五十卷元袁桷撰桷有延祐四明志已着録其集据苏天爵行状及元史本传俱称五十卷此本卷数相符盖犹旧本也桷少从戴表元王应麟舒岳祥诸遗老逰学问渊源具有所自其在朝践厯清华再入集贤八登翰苑凡朝廷制册勋臣碑版多出其手故其文章博硕闳丽有盛世之音尤练习掌故长于考据集中如南郊十议明堂郊天异制议祭天无间嵗议郊不当立从祀议郊非辛日议诸篇皆成宗初所上其援引经训原原本本非空谈聚讼者所能当时以其精博并采用之其诗格俊迈髙华造语亦多工链卓然能自成一家盖桷本旧家文献之遗又当大徳延祐间为元治极盛之际故其著作宏富气象光昌蔚为承平雅颂之声文采风流遂为虞杨范掲等先路之其承前啓后称一代文章之钜公良无愧色矣乾隆四十六年十一月恭校上

　　清容居士集卷一
　　元　袁桷　撰
　　赃
　　导游赋
　　陈郡袁桷家于甬之隅足不逾阙而四方交游徃复之义亦能举其槩闻吴兴子昂赵子清名雅言为作导游赋道予所蕴亦以见夫思其人而不可见之意词曰
　　尝悲乎好游之士阅终岁而莫闻非秉志而媚私亦饰行以矫羣方其命车于王良假骖于造父过屠缯之沛吊弦歌之鲁许身结志慷慨伛偻抗铜盘以矢辞握幽兰而结赋炫华堂之冰雪眇修途之霜露葢将修断金之利虽防墙而莫我敢侮息肩于门晞髪于庐不一歳月迨见夫蝇声而歌倚楹而吁虽千里之异殊何老壮之卒逾也世有索居肥遯墼门塞窦尊委俚之闻友汗青之朽是犹谈智于尺寸测日于戸牖猛虎嚵嚼幽猱悲啾独行无俦谨揉其辀粲彼一士与时偕藏具大块之清淑服忠厚以自昌余将徃而未得倚呉山以相望振衣中庭万籁寂颀形广颡防兮翩翩坠余以言曰懐辛之椒人或议其妖充梁之木人或恶其独夫既自以为直而亦岂不以为曲阮生自持几于目语削木已漫假手于祖匪达人之旷羇终察察乎亷隅余冁然而笑喟然而思拂以占莫测其疑抑搔首而欷故髣髴以致期遂申之以辞连城之璧佩以容兮环龙之劒喜其锋兮伊若人兮物不可同兮我欲从兮乐有终兮江山兮复长方舟兮可航辞未竟童子拥篲而前曰岂所谓千载之人耶吾当从子而西矣
　　云林赋
　　若知云乎昭昭然服光以广微隆然廓然又似夫五色万物而沉几来弗以趾去弗以驰其始也盘辟夹辅轮轮囷囷孰合其处所其终也寸芒以兴导轰轰启铃铃汨沮洳襄丘陵若慕而伫若泣而赴愀兮哀不知所诉蜿蝉眇绵含英吐鲜逌倾葢而忘返粲接席以争先诚果乐乎莫与之宣形成乎无形名生于有名囿于家林允为仪刑奉蓍占之曰居髙而卑佩矩带防升于木无徃不复不泄不辱言归其谷伊南维湖将以为居斋三日祝而莅诸荟兮中峰式瞻其端为章为祥为轮为冠风以润之雨以需之挟阴而乘阳而舒朝飞夕跻莫穷其栖有翼者鹄遵于其涯曰将有行自东徂西绥而迎之维墉之崇施而流之维泉之通以沐以容是谓云林之宫
　　云石赋
　　泗流回渊禹朝步兮秉阴发竒灵鼇负兮玉质墨章浮帝都兮廉辨以清协磬濩兮坚刚洞虗积元感兮磅礴太素屹空嵌兮静居泊如日仪以湛兮虗一阐图握九赞兮八方宣明二仪窍兮抱形介贞合窈眇兮絪絪缊缊道之奥兮肇称云吾将以观众妙兮
　　淳赋
　　粤若防古仁风熙熙击壤灌阴以佚以嬉巷无夭瘥物无疠疵祥麟应囿灵负池乐以导和礼以黜衺奫奫其春浑浑其天守冲若虗不夸不镌佩矩帯防养于深渊拟诸朴兮文理具宣拟诸象兮充然混完见不以迹视不以形斟酌太和肧胎粹精居之莫疑神维壹清若有若亡讫莫可名二仪絪缊肇分化醇人行乎中孰全其真注兹酒淡兮在尊御史氏尹敏学潜珍服习典谟葆厥灵根斋居静仪人莫可闻攷诸庖羲其名曰淳
　　玉杖赋
　　有璞于此纯纶氤氲瑶光之精为灵为文谨椟以藏不泄孔温穹窿中干夷静合坤渊渊其容深深其神爰有善木含章扬芬饰端以仪徃来无垠郁峩寿宫有粲者云维季翼翼聿懐彼濆跪而祝诸以奉番君曰履道坦而元吉者欤任重而不希其力者欤视下而无危嬉戏如儿审时而行得艮以防抑心自强厎夫康庄其形不孤其智若圆化而神之曷知其迁以相以持是谓期颐
　　桐柏观赋
　　孙兴公天台赋运意髙简忘象以求斯得之凝和卢君居是山以桐柏观图授余曰子为我縯绎其景不揆瓦石援笔为之赋曰
　　厚坤载徳台山効灵蜿蜒扶舆天孙之精僊书易之以桐柏敞神界为金庭轇轕三光磅礴太空睇旸谷之倒景卒莫能拟诸形容也睠彼华顶羣山启踪倡九牧以执玉巍冠仰其颙颙香林玉霄夹骖以从华林不雕卧龙嵸巃玉泉御右法莲殿戎东峙翠微西蹲紫霄巧面势以规辅扄沉沉乎泬寥维妙有之珍馆匪攀援之可求除三彭之内荡守中戒以默脩影息于邓林机忘于海鸥上池饮之以洞视金膏服之而冥搜不然者良不能以乆留也粤昔子晋韬精帝京弄参差于缑山抟扶揺以遐临琼防俨其负双阙屹乎龙蹲正右弼以赞化千龄髣乎氤氲故兹山也四面削成中心如掌真风合于冥廓梵炁通乎肸蠁三井之泉千尺之潭臯羽警其瀼露髯羊匿于阴岚飞梁凝冰侧足涉褰翠藤流霜触缨扪縁梭溪交流以织文忽防蝀之钩连望飞瀑之悬絶日五色而成妍夫乐与饵过客所止夸诞逐魂践形者耻长生之药眇茫于神山执期之候诙谲于五利却衰防顽简易理得其亦有所益乎菌蠢春秀筼筜冬萌五叶之松三脊之香丹光腾于金鼎冰泉彻于铜罂服食之而不坠身轻忽焉以复丁企列僊之遗踪葛翁防名于太极斗晶荧于枢纽金光杂流于瑶席纳雷谷之砰磕穷幽渊之罔象丹凤昔焉以来仪凡羽厥今以避让谅守贞以制气匪狡狯以增诳法不孤立道不虗行继之者白云先生受素书于轩辕袭帝胄于典午憺寡言而无营思坐忘以遐举承唐皇之明问敷清净之旧章署漆书于宝幢迎黄云于玉堂厥后宪章希夷范围灵寳诵蘂珠之文过期颐以终考若是者诚何异于蓬岛或有言曰楼台荣观方士侈崇夷考是山不绮而丰不幽而通廓兮其有容閟兮其讫逢味兴公之抉竒讯择木之写真若増其嵚崟尽其窈深身不羽而凌遡俄曾城以瑶林乃若丹霞伏光铁柯舒英集众妙以凝睇髣三真之传经拟金刚于铁磨警黑业于鸿钟何钱李之二王莫能悟其禄终猿抱子以望月虎伏威而讽风灵璈清以集鸾哀急而惊鸿爰自有而即无眇壹气之鸿蒙悼南国之蒸歊髣天都之肃冽彼夏虫与火鼠徒自煎以内热申之以辞曰
　　帝子下兮云中质通明兮粹冲览炎洲兮垢纷节容与兮兹峰蓬瀛杳兮无极浪滔天兮不可以屐境荒灭兮亡益曰此留兮吾将以内息神为凥兮气为舆朝紫霞兮暮清都黜视听兮道靡渝歴千劫兮复故初云凝素兮中坛铃流露兮日暾三灾屏兮嘉祉繁语夫世人兮归休乎静观
　　椿堂赋【为宜春彭有年道士作】
　　昔尝疑夫漆园之叟学幻而逃虗询虫鱼之疏考风土之书莫之敢图余游上京有客北穷夫金山西歴夫贺兰南逾雕题东超三韩冁然而言芒芒禹迹子所履者诚然乎冬裘夏葛寒暑之运环也今也日临中街霜飞冰霾毳集于踵貂挟于懐戴斗接淅承露凝桮子安视而靡猜骇闻忽见知奚穷乎齐谐吾先枚夫九州者而言之屡熟之稻南州是专十丈之莲华峰所传栝化石以锡贡樟生庭而为州梅梁分以镇妖牂牁肇乎维舟杞犬吠而质变葠人形以剂投华平雒常瑞于皇图蓂荚屈轶盛于尧世木沉产子以成国树古精而为魅松分秦王之封柏擅蜀相之祠若是则复何疑焉在昔大帝穷源昆仑见北溟之鹏视之若雪即之如云得其遗羽丈寻不能以论是齐物者非寓言也耶维老彭之遗裔握六气以自驯肇兹斋居其名为椿二仪絪缊周流无垠则以自仪道合至淳吸兮若鞴旋兮若轮肃兮似秋熙兮如春行乎六合而不踬则所谓八千者吾犹以为歳年之稊米也客揖而去遂为之赋
　　仁寿堂赋
　　合江老人杜氏行年八十有二其乡人尊之以仁寿因名其堂子辉卿今为蜀省掾来京师求赋赋曰
　　合江老人生于三卯歳之中秀睂疎明鲐背隠隆家事断遗闻防容口欲宣而弥哽情屡发而莫逢防稽生揖而进曰生居东南袭轻饫甘耳不属于钲鼓目不接于戈锬合江之事相传于游谈愿削牍书椠俾鸿笔以有参老人改容正襟徃佀可寻梦若初觉病犹在身方其誓师斗城灭亲割情盟重于白马命轻于浮尘黄风飏沙千戍屏营拥若波涌排若算经危堞隳而礮激阘防护而矢承躄者瞽者各有攸职熏窒旁穴旦不遑夕坏馆门毁甃石树列对峙填负旁阸卓飞泉于云根起清流之千尺目不能承睫气不得纳息黑祲障空燐烟蔽江横厉排荡抶击舂撞拟效顺思投降提奬之主怒莫敢当忽风敛而川止活噞喁于鼎汤噫嘻吁此其琐谫者耳华阳之徃躅肇于文翁悉于太冲纤侈日滋繇是铄景福荡醇风鬼啸髙居神泣巨峰栈阁如砥万骑突驰火连摩围氛翳峨睂筑京观以专戮甚槀街之陈尸顺舟而下者十不得一缘涧歴堑视京垓之豪厘﨑岖古渝前彭后余保衣冠之弱绪犹兴文以舒徐逮夫六合防同弹丸震惊发涣号以澡雪缉政俗以治平则夫两州之土民无遗镞之害歌帝徳以底寜防稽生仰而愀俯而笑在易之象复存于硕果物极者斯通盛过者斯祸维今老人熙熙其天休休其心事至不忮智藏惟深式是表仪化而慆淫国有耄叟乡有善人仁寿锡名爰则其淳感古乐今兹其为击壤之尧民
　　乐水图赋
　　东坡先生脱逺游之履衣紫丝之衣与黄冠师跪坐息机临清泠之渊憩巨樾之阴意消言忘其得在心飞泉琮琤寒源杳深悲尘寰之隘险抱竒璞而莫陈将以善夫济世之极乐至理以自新维兹水之无倪非智者莫能以鉴彼河伯之好夸终望洋而浩叹翳西海之沈羽竟其说以恠诞小大不同其病则通骇于所闻者忽所见诚哀乎举世之盲聋余尝披神禹之地图考先生之辙迹而诵之方其发轫于瞿塘千里一泻鉴燕安之酖媒悟孤孽之非祸虽盛气之飘忽已足穷物理之化迨其中年倦趋分符东上慨河流之奔驶城不没者三板先生胼胝以先激而返之有以遏其性之狂悍伊文登之贝宫空百幻以自献实精诚之冥通异燃犀之获谴至于羣妖肆憎百慑相乗先生怡然而行试正命于巨浸俨安流之无惊耿耿明夜乘桴南归海若息波冯夷効竒混六凿于一噫信天地之不私匪深于水者其能若斯耶玩王孙之游艺弄五采于素丝聴之而无声即之而若遗若然者欲以求先生之所乐无异画饼之充饥
