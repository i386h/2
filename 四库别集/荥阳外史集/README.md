　　提要 

　　臣等谨案荥阳外史集一百巻明郑真撰真字千之鄞县人成化四明郡志称其研穷六籍尤长于春秋吴澄尝策以治道十二事皆经史之隽永真答之无凝滞洪武四年乡试第一授临淮县教谕升广信府教授真与兄驹弟鳯并以文学擅名真尤以古文着初与金华宋濂声价相埓尝与濂共作裴中着存堂记真文先成濂为之阁笔后濂致位通显黼黻庙廊真偃蹇卑栖以学官没世故声华閴寂传述者稀今观所録不能与濂并骛词坛而义有根柢词有轨度与濂实可肩随未可以名位之升沈定文章之优劣也原集百巻明代已佚其三十巻今所存者亦多残缺失次讹不胜乙或至于不可句读殆世不甚传故莫为是正其不全佚者幸耳今推验文句各为校定其必不可通者则仍原本録之庶不失阙疑之义焉

　　聴竹轩记 

　　姑苏陈君仲良来居鳯阳之临淮萧散夷旷有林野之趣构一室内列图画书史及古彛器等物外设崇栏环以竒石植竹数十本虚心劲节整比森密大者若幽人贞士特立而不可屈挠小者若门人弟子恭肃而不可狎玩仲良葛中野服宴坐其中披阅今古或时至夜分一气孔神万虑不作而凉颷时来竹之声鏦鏦铮铮珑珑玲玲如钧天广乐交奏并作宫商调而律吕谐也如蓬莱列仙湘江帝子乗龙驷鳯佩玉鸣而珫璜击也由是境与心融心与意会怡然涣然有不知其所以然者乃扁其室曰聴竹轩云夫大钧播物坱圠无垠因所植而形生焉因所触而声生焉声之发也入乎耳警乎心徐而察之其声何如哉万化本乎一理而万理会于一心竹之动而静以聴之生物之情于是见矣托兴之士所谓声与静相宜者葢在于此岂但清风故人想象于彷佛间也耶呜呼若仲良者其观物察理者欤虽然至人遗物独与道俱仲良学道者也一斋坐忘形冺迹化彼吹万不同有若天籁地籁者固已相忘于无何有之乡矣竹声云乎哉予家东海上万竹林立耳存目注清思如洗时异事殊芜废不治久矣及来濠梁与仲良所居为近每促膝过従旧梦为醒庶几朝夕问竹之平安以及其人聴竹之声以妙悟于无声不亦可乎哉 

　　梅雪山房记 

　　寿州蒙城县主簿林君孔嘉士宾广西人也年三十余神气清莹襟怀疏爽早以才调见知名公巨卿得为广西行省行人进退周折必以规矩而应对従容举措闲雅行李使命之重不违其时尝度庾岭时固阴冱寒见梅已着花而大雪盈尺山川林壑逺近一色徘徊凝伫喟然叹曰天下之物有如梅雪之清而白者乎猥以不才服劳于公清白之操慎终如始其殆庶几哉于是扁所居曰梅雪山房寝处笑语朝斯夕斯既调官寿阳仍以旧扁掲之介乡友亳县簿叶君伯兴求文以记夫士之托兴于物者非私所嗜而溺所好也用以比徳焉尔桃李之卉之华而夭艶无取霜雹积阴所致而变异或臻士君子所贵重者梅与雪而已夫梅不徒清也岁寒之孤芬有以坚所守也雪不徒白也阳春之寡和有以励所学也昔之名哲有若何逊宋璟林逋苏武袁安之流或以梅或以雪抒幽发兴极其所至以砥砺名节天下后世固有闻风兴起者矣士宾南海之英名在宦籍而芳香自珍光洁有耀澹澹乎静且幽而莸秽不得入也凛凛乎凝且重而尘氛不得杂也呜呼其非托物之君子哉且夫俗尚不同人品或异知愚贤不肖不能以相一也然而人之生也眇然一身万物皆备反身而诚乐莫大焉梅雪云乎哉而世之重功名崇道徳者犹以调商鼎之梅立程门之雪为当时美谈不识士宾尝用意于斯否耶予忝教临淮于今有年异日或得以公事道过寿阳造其所居闻所闻而见所见斯可乎记之 

　　乾坤一草亭记 

　　姑苏呉庠孟周谪于濠水之上相地一区空阔高爽斩木诛茅构为一亭檐牙轩翥阶级静深虚其牗以资凭览崇其径以通出入外不设藩篱内不施帷幛琴樽书史庋置左右孟周游息其中仰高俯深干端坤倪开豁呈露心有所悟物亦随之因题曰乾坤一草亭云人有讶之者曰乾坤至大也草亭至微也以草亭而系诸乾坤毋乃管窥蠡测不知量者耶孟周曰此以迹言者也乾坤之道大不能外小莫能违乾坤诚大矣吾従而观之则乾坤皆在吾草亭之内吾不知其孰为大孰为小也寓形宇内与乾坤上下相为流通而万事万变皆得以傲睨而従容焉谓之乾坤一草亭不亦可哉予闻而解之曰孟周其学圣门者欤夫干天之道也坤地之道也干言徤而坤言顺也乾坤之大周髀不能测章亥不能步也草亭之在乾坤何啻沧海一粟哉孟周朝夕其中观夫二仪之运四时之行日月星辰之耀风雨露雷之变山岳之峙河海之流其理之所以然必有合于心之所同然者矣且人与天地参者也天地之心即吾之心天地一太极吾心亦一太极也天地无有限量吾心亦无限量也囿乎方寸之中而极乎六合之内静而与乾坤同其体也动而与乾坤同其用也大而裁成辅相小而人伦日用何莫非乾坤之所为哉是故由乾坤而视草亭草亭实乾坤之一物由草亭而视乾坤则乾坤亦草亭一物尔物我两忘显微无间非知道之君子其孰能与于斯且夫乾坤莫外于易居室之宜取诸大壮乾坤之理已存于中矣然而一亭之微不如万间之盛简陋俭朴非有雕刻粉饰之富而君子有取于孟周者岂非以天地万物之理萃其心欤昔临川呉文正公作草屋数间而题其牗曰抱膝梁父吟浩歌出师表程公雪麓扁之曰草庐葢以孔明期之也公既登金门上玉堂著书立言以道徳文章自任学者称为草庐先生其名实之允称矣孟周其迪光前烈也哉孟周谢曰先生之望我至矣谫焉末学其能底于是也耶予嘉其言让而有礼也记诸亭上而去 

　　梅堂记 

　　东南多竒葩异卉求其铁心石肠凌厉于风霜雪月间莫梅若也是以君子比徳焉书称若作和羮用汝作盐梅其取譬精矣后世名臣贞士若何逊宋玉林逋之流模写而形容之非夸大其词以炫耀流俗也葢其心意所适有至理存焉乃若堂构基图祖孙相继手植所存恭敬弗怠亦足以观世泽之深厚永久矣鳯阳府寿州儒学正潘先生怀玉世为三山阀阅葢宋太师河南郑王美之后由郑王而下峩豸台端相继号四叶中丞传及数世曰征曰衢者在祥符间聮登进士第有司号其乡曰二难里曰同荣又传及十有八世讳继贤字彦能号梅堂处士者则怀玉先大父也处士生于咸淳末年入元以先世为宋显官杜门求志不嗜仕进尝筑堂三水上而对麏石山控引形胜种梅百本巾屦逍遥吟咏自适今去处士数十年根深本固而梅堂故无恙缙绅之士过其下者为之徘徊顾瞻而不忍去想夫晚节岁寒万木僵仆正人心永保贞固之时也而梅作其花皜素呈露固有以见阳春之泽无有间断宁息矣天日卓午鸟语风微传杯索笑芳馨静幽天地造化之心于是乎在矣梅乎梅乎处士其有得于兹乎怀玉以处士嫡孙之贤封而殖之升堂肃躬顾盼流睐灵游燕娯锡兹祉福其可量哉唐李徳裕自着平泉记以为坏一草一木非吾子孙况祖考神明道徳之所在者乎然则保有先泽与其家相为无穷在怀玉有不容辞矣予家居四明去闽中为逺每従缙绅论东南文献故家于处士毎想其风范而以生晚不及亲炙为恨今越在淮海与怀玉同在斯文安得抠衣上堂心领神会于暗香疎影间耶是为记 

　　竹林庵记 

　　永嘉金氏代为儒族其始祖宁甫氏在宋时自闽之赤岸来居平阳金洲以积善有名所居东南厥地爽垲野水分流林麓掩映遥山迤逦起伏如龙虎如象如龟如鳖如琴如几如珠如印皆在逺近数十里间以曾杨之学揆之信为风气所聚由是増土为垄潴水为沼环以松栢杂以荷芰清颷徐来香气■〈芬上羽下〉勃仰高俯深孔乐斯土以窆以竁是为始祖之藏尔来二百余年矣元至正末所在构兵郡将缮修城郭士卒樵采不禁合抱之木为所伐者八九金氏有谷汶父者聚族谋曰斤斧之祸乃至此乎众心为动相与出资财赂之得还故物遂命族人伯贤督工构屋三楹列为两庑外置庖湢旁储祭器深广高下悉如其度别置田若干畆以供祭祀岁时殷荐家聚族于斯乡缙绅大老叹羡称慕以金氏不乏人焉以其地多竹扁之曰竹林庵云洪武十三年夏四月谷汶之子寿以俊选为国学生试艺高等授鳯阳府光州固始县主簿以其家君命来求为记予惟礼莫重于亲亲尊祖所以重本亲亲之至也谷汶其知所本者哉周礼家人凡祭为尸汉制定为上陵食举之仪下至公侯士大夫家皆得有事于墓墓也者体魄之所藏也墓之有祭尚矣然非有栋宇之制其何以肃位奠之诚哉且墓必树木种木者期之于百年之前用之于百年之后谷汶念其始祖之手植在我后人培而植之而不忍用之也然而事变之来势莫能遏幸而购得于军旅摧挫之余择其礼之宜事之可者行之亦足以慰始祖之神矣矧夫华构翚飞林泉美观灵游宴喜者哉想夫霜露之交少长咸在自谷汶一身观之有伯叔诸父焉有子姓羣従焉樽累酒醴之馨俎豆菓殽之荐洁盥兴俯其中仪式精神血气之交肸蠁幽冥之格固有如见其笑貌闻其容声者矣享事既彻饮福荐胙或有倡言于众者曰祖宗余庆庶其在此诗书礼义之遗堂构菑播之责其保有弗坠矣至于患难贫贱有背礼废恩者乎彛伦日用有见利忘义者乎闻者惕然惧耸然悟黾勉用力不辱其先矣此固谷汶之志也岂徒谓万竹之林立以为一日游观之乐哉昔者诗人作淇澳之诗美卫武公先师子曽子赞以为盛徳至善民不能忘金氏以竹林名庵慨思先徳信乎其不能忘矣子子孙孙弗替引之其不在兹耶予忝同淛产兴教淮山间金洲之胜为向慕兴起异日幸得一至其处咏歌猗猗之间亦云幸矣姑为之记以遗寿使刻之山中以传之寿清修俊迈种学绩文见用于时如木向荣金氏始祖之泽殆未艾也
