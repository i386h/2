其一
送陈咏之归镇阳 【镇阳汉东垣县赵地在恒山之阳唐曰镇州今为真定府河北西路兵马都总管按蔡大学墓表陈沂字咏之大学子壻天眷中官承德】
东垣步秋水几曲冷玻瓈 【庄子有秋水篇云秋水时至刘希声剡溪几曲知名处郑獬万顷玻瓈烂不收东坡二十四桥亦何有换此十顷玻瓈风又相望六十里共饮玻瓈江又归路万顷青玻瓈山谷青玻瓈盆插千岑皆以玻瓈喻水也】 沙鸥一点晴雪知我老无机 【列子海上之人有好沤鸟者每旦从之游沤鸟百住而不止也其父曰汝取来吾玩之明日之海上沤鸟舞而不下也东坡岂惟见惯沙鸥熟惠崇鹭诗栖烟一点明老杜晴雪落长松太白摇拽双白鸥忘机从尔游苏广文汀畔数鸥闲不起只应知我忘机司空图有无机诗云无机免误身】 共约经营五亩 【毛诗灵台经之营之不日成之孟子五亩之宅树之以桑五亩谓宅也东坡五亩渐成终老计】 卧看西山烟雨窗户舞涟漪 【东坡西山烟雨卷疏帘王勃珠帘暮卷西山雨毛诗河水清且涟漪刘梦得诗寒藻舞沦漪涟漪沦漪皆水纹】 雅志易华发 【谢安镇新安尽室而行欲经畧粗定自海道东迁俄遇疾自以雅志不遂深自慨失东坡多情应笑我早生华发】 岁晚羡君归 【东坡回首吾家山岁晚将焉归又羡君欲归去奈此未报恩】
月边梅湖底石入新诗 【梅横月边石出湖底皆可收拾入诗笔也】 飘然东晋奇韵此道赏音稀 【晋司马氏始都长安自元帝度江号东晋当时王谢诸人祖尚玄虚不以世故婴心其光风胜韵映照千古适丁乱世人多非之故言赏叹其风声者几希也东坡东晋风流王谢家文选采南皮之奇韵山谷寂寥吾道传万世忍向时人觅赏音】 我有一峯明秀 【明秀湖山名公家所蓄其峯甚妙故公之诗曲必言及之】 尚恋三升春酒辜负绿蓑衣 【唐王绩字无功武德初待诏门下省日给酒三升或问待诏何乐答曰良酝可恋侍中陈叔达闻之命日给酒一斗时号斗酒学士毛诗为此春酒以介眉寿张志和青蒻笠绿蓑衣斜风细雨不须归魏野江上负烟蓑史李陵云陵实孤恩汉亦负德】 为写倦游兴说与水云知 【史司马相如传长卿故倦游言倦于游宦也东坡凭君说与春风知公作词欲令镇州水云知此心也】
其二
曹侯浩然人品高秀玉立而冠其问学文章落尽贵骄之气蔼然在寒士右惜乎流离顿挫无以见于事业身闲胜日独对名酒悠然得意引满径醉醉中出豪爽语往往冰雪逼人翰墨淋漓殆与海岳并驱争先虽其平生风味可以想见然流离顿挫之助乃不为不多东坡先生云士践忧患焉知非福浩然有焉老子于此所谓兴复不浅者闻其风而悦之念方问舍于萧闲阴求老伴若加以数年得相从乎林影水光之闲信足了此一生犹恐君之嫌俗客也作水调歌曲以访之 【曹浩字浩然宋驸马曹寔之子武惠王彬之六世孙也山谷诗小序周茂叔人品高秀史陈平如冠玉唐崔玙与子澹举止秀峙时谓玉而冠者老杜赵生玉立高歌起山谷云萧元礼恂恂退让从容问学文章史贵不期骄而骄自至南史寒士不逊易发于事业卫玠遇有胜日亲友时请一言渊明遇有名酒无夕不饮汉书引满举白刘伶一石径醉桓温豪爽有风槩法帖亹亹逼人选诗粲粲翰墨场退之醉墨淋浪不整齐海岳取其高深也米元章自号海岳翁东坡醉翁此句与子美并驱争先又坐念故友之风味岂复可见孟子不为不多庾亮老子于此兴复不浅庄子闻其风而悦之魏志求田问舍言无足采公作圃于镇阳号萧闲圃又公始寓汴都其第有萧闲堂因自号萧闲老人真诰童男得道者居萧闲堂东坡阴求我辈人规作林泉伴又旧人眼看尽老伴余几个论语予曰加我数年五十以学易皇甫曾澄澄倒林影东坡水光接天毕卓拍浮酒舡中便足了一生梅圣俞且复与妇饮颇胜俗客对】
云闲贵公子 【晋史云闲陆士龙太白闻道青云贵公子】 玉骨秀横秋 【老杜秋水为神玉为骨东坡玉骨那愁瘴雾孔德璋北山移文霜气横秋】 十年流落冰雪香叆紫貂裘 【言宋京不守浩然流寓于朔漠冰雪之地而衣裘故弊也东坡念君官舍冰雪冷石敏若启顽颜钝颊有穷乡冰雪之容叆音爱云状坡词香蔼雕盘史苏子游说不遇黑貂之裘敝说文貂鼠也出丁零国魏书有貂鼷子皮毛柔软可为裘今云紫貂其色紫者尤贵又有白貂银黄貂愈艰得】 灯火春城咫尺 【言京师元宵将近也东坡马行灯火记当年又九衢灯火杂梦寐韩翃春城无处不飞花东坡欲见隔咫尺左注八寸曰咫】 晓梦梅花消息 【类说赵师雄迁罗浮遇一美人淡妆素服时已昏暮与之饮但觉风寒相袭久之东方日出起视在梅花树下许丹房纸帐梅花醉~梦闲陈去非杏花消息雨声中】 茧纸写银钩 【山谷银钩玉唾明茧纸茅亭客话王羲之书兰亭诗叙用绵茧纸鼠须笔北户录宋亳闲有一种纸长丈余就船钞之谓之茧纸或云直用蚕茧造之今人闲往往有古纸存焉法书苑索靖书绝代名曰银钩虿尾又云宛若银钩飘若惊鸿墨薮云点欲坚真如铁钩欲活而有力如银】 老矣黄尘眼如对白苹洲 【论语子曰吾老矣不能用也王介甫黄尘入眼见山愁老杜且归来黄土污衣眼易眯柳恽守湖州诗汀洲采白苹日暮江南春后人名其处曰白苹洲山谷眼青见白苹洲】
世闲物唯有酒可忘忧 【乐天应是世闲神圣物谓酒也晋顾荣曰唯酒可忘忧但无如作病何耳】 萧闲一段归计佳处着君侯 【公有萧闲圃在真定法帖羲之果此一段奇事张旭帖永嘉路远难遂归计东坡故乡归去千里佳处辄迟留葢宽饶传许伯曰君侯可不戒哉】 翠竹江村月上 【老杜白沙翠竹江村路相送柴门月色新东坡翠竹江村绕白沙欧词翠竹岭头明月上】 但要纶巾鹤氅来往亦风流 【晋谢万着白纶巾鹤氅裘见简文东坡纶巾鹤氅惊笑吴妇老杜与子成二老来往亦风流】 醉墨蔷薇露洒徧酒家楼 【山谷速直蔷薇露一斗与君洒墨三千首香后谱周显德五年昆明国献蔷薇露十五瓶云得自西域以洒衣衣弊香不灭谈苑金陵宫人采蔷薇染帛经夜露色辄鲜明又岭南以蔷薇水染衣皆黄唐张旭与贺知章相善每醉后索笔书酒楼杨大年江南堤柳拂人头李白题诗徧酒楼】

闰八月望夕有作 【闰八月即九月故词中皆言月与菊】
空凉万家月摇荡菊花期 【退之客堂喜空凉永叔万家明月放灯天老杜晚来高兴尽摇荡菊花期】 飘飘六合清气欲唤紫鸾骑 【汉武览相如大人赋飘飘有凌云之气庄子疏四方上下为六合礼疏二气始分其清气为天浊气为地坡词红鸾骖乘青鸾驭介甫月中仍有女乘鸾骖鸾乘凤仙人之事】 京洛花浮酒市 【京洛洛阳葢指汴都也言菊花浮动于酒肆闲选诗京洛多风尘酒谱酒市有旗始自于宋】 初把两螯风味橙子半青时 【毕卓左手持蟹螯坡赋味霜余之两螯山谷不比二螯风味东坡朱甘绿橘半青时又正是橙黄橘绿时老杜三寸黄甘犹自青】 莫话旧年梦聊赋倦游诗 【公于东国遇秋忆昔京师花酒之事恍然如梦不欲话之但赋诗以排闷也旧词十年前事浑似梦初惊】
玉盘高金靥小笑相窥 【太白小时不识月呼作白玉盘张正见裁金作小靥介甫露金靥篱边笑金靥金钿以比菊】 市朝声利场里谁肯畧忘机 【山谷安肯声利场牵黄臂老苍太白陶然共忘机】 庾老南楼佳兴 【晋庾亮在武昌佐史殷浩之徒乘秋夜共登南楼俄而不觉亮至诸人将避之亮徐曰诸君少住老子于此兴复不浅便踞胡床谈咏竟夜】 陶令东篱高咏千古赏音稀 【晋陶潜为彭泽令九月于东篱采菊盈把忽白衣人至乃刺史王弘送酒便饮至醉其诗曰采菊东篱下悠然见南山永叔高咏且陶情言陶庾风流千载而下无知音者】 手捻冷香碎和月卷玻瓈 【贺方回笑捻粉香归洞户永叔野径冷香黄菊瘦东坡劝君且吸杯中渌又引南海之玻瓈妃子外传妃子持玻瓈七宝杯事始古有酒令名卷白波卷玻瓈言饮之爵也玻瓈盏出南海】

丙辰九日从猎涿水道中 【涿州有涿水即古涿鹿黄帝与蚩尤战处】
星河淡城阙疏柳转清流 【宫天晓行柳堤闲柳拂清波也沈约城阙郁盘桓曹崧御柳垂着水东坡淡烟疏柳媚晴汉】 黄云南卷千骑晓猎冷貂裘 【黄云黄尘如云涿水在都南故云南卷冷貂裘言早寒山谷不使黄云没马头东坡锦帽貂裘千骑卷平冈又遥知千骑出清晓】 我欲幽寻节物只有西风黄菊香似故园秋 【故园指汴都言涿之与汴节候物态虽殊其菊花香色则同也山谷物物有幽寻东坡不忿东君专节物寇平仲故园应坠叶昨夜又西风】 俛仰十年事华屋几山邱 【兰亭帖俛仰之闲以为陈迹旧词十年前事浑似梦初惊曹植生存华屋处零落归山邱言倏忽十年闲贵人几许沦逝也】
倦游客一樽酒便忘忧 【韦庄马足倦游客苏武我有一樽酒】 拟穷醉眼何处还有一层楼 【老杜朝廷醉眼中王之涣鹳雀楼诗欲穷千里目更上一层楼】 不用悲凉今昔 【羲之后之视今亦犹今之视昔悲夫东坡感时怀旧一悲凉】 好在西山寒碧金屑酒光浮 【东坡故山知好在又青天落寒碧辋川有金屑泉初寮金屑烂光彩此谓屑菊如碎金投酒杯中然晋代以金屑酒赐臣下自尽恐非佳意】 老境玩清世甘作醉乡侯 【曲礼疏耋至也言至老之境山谷白发侵老境东方朔滑稽玩世公尝自称玩世酒狂故有此句皮日休他时谒帝言何事请赠刘伶作醉侯东坡不妨仍带醉乡侯】

仆以戊申之秋始识吾季沾兄于燕市稠人中轩昂简贵使人神竦既而过之未尝□弥日忘归至于一邱一壑心通神解殆不容声自是朝夕与之期邻里与之游者葢十有二年己未五月复别于燕之传舍及其得官汴梁仆已去彼怅然之情日日往来乎心也 【季沾姓范文正公希文之四世孙家许昌聚图书万余卷知名当世公心友之王戎曰昨于稠人中见嵇绍昂昂若野鹤之在鸡羣永叔其志独轩昂文选情骇神竦晋明谓谢琨曰或以卿方元规何如答曰一邱一壑自谓过之庄子解心释神又云仲尼见温伯雪子而不言曰若斯人者目击而道存亦不可以容声矣易憧憧往来朋从尔思】
