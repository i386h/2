　　《文选》、又称《昭明文选》。是中国现存的最早一部诗文总集、由南朝梁武帝的长子萧统组织文人共同编选。萧统死后谥「昭明」、所以他主编的这部文选称作《昭明文选》。本版本是宋代淳熙年间尤袤所刻之李善注本，并以清代胡克家之文选《考异》加注释。
