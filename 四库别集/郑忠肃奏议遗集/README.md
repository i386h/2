　　【臣】等谨案郑忠肃奏议遗集二巻宋郑兴裔撰兴裔字光锡初名兴宗显肃皇后外家三世孙由成忠郎歴官江东路钤辖迁均州防御使保静军节度使召领内祠武泰军节度使赠太尉諡忠肃是集所録多奏疏表状其记序辨跋诸杂著则间附数篇其中如请起居重华宫及论淮西荒政诸疏词意剀摰他如蠲缗钱禁改钞论折帛钱诸奏所载南宋绍兴时诸弊政皆宋史食货志及文献通考所未载亦足补史志之阙又纪淳化阁帖之摹搨传写与黄伯思互有异同辨琼花之复荣较周必大玉蕊辨证更为详赡亦考古者所不废矣其书马端临经籍考及宋史艺文志皆不着録此本为其裔孙所裒集编次者惜篇幅太狭或就家蔵残本润色以传然録之亦足征宋代故实之一二焉乾隆四十六年五月恭校上

　　请起居重华宫防
　　臣闻治天下之大法有三曰君臣父子夫妇夫妇有法则家道正父子有法则人道久君臣有法则天地泰要之有父子然后有夫妇有父子然后有君臣百行所以莫先于孝也天子以一身立臣民之上作万方之观瞻必极天下之孝而天下之为孝者莫能尚所为皇建其有极也由是而騐其休咎于天考其祸福于人无一毫之不顺故仲尼曰先王有至徳要道以顺天下民用和睦上下无怨孝者徳之本也教之所由生也又曰爱敬尽于事亲而徳教加于百姓刑于四海明王以孝治天下菑害不生祸乱不作子思述仲尼之言曰舜其大孝也与惟孝而宗庙飨子孙保则孝綦重矣凡孝子之事亲居则致其敬养则致其乐病则致其忧许世子不尝药春秋讥之陛下为天下主何忍蹈此昔文王之事王季日必三朝鸡鸣至寝门外问内防之御者曰今日安否何如曰安乃喜日中亦如之及暮亦如之有不安则色忧行不能正履王季复膳然后亦复初食上必视寒暖之节食下必省多寡之恒武王率而行之文王有疾武王不脱冠带而养文王一饭亦一饭再饭亦再饭比瘳乃已自古圣帝贤王史策所载美不胜书必以孝为首称惟我太上皇帝以藩邸入嗣大统事徳夀备极孝敬以天下养者二十六载终始不渝陛下所习见也升遐之日哀慕肫摰终防三年力行古礼前有以迈百王后可以师万世亦陛下所习见也陛下政治动欲法尧舜法祖宗奈何门内之行不近法太上乎陛下身体髪肤受之太上宗社民人传之太上昔闵子骞人不间于父母昆弟陛下孝友性生居青宫四十年曾无间言受禅三载两宫和好亦无间言特縁初郊震惊玉体愆和太上皇帝忧虞切衷未免责及侍御之人左右近习潜滋离间过此以往圣躬弗便于警跸睿体毎怯于风霜温凊展庆之仪仅遣内臣将命俱以太上传旨而罢二歳有余起居重华者不过一再防庆诞节阙上觞之礼歳首大防废称贺之文陛下有亲而无亲太上有子而无子北内凄凉殊无聊赖一念及此可为寒心方今太上违豫正陛下忧心不遑自逸之秋中外臣僚引领望过宫者非一日矣迨逾三月未降指挥玉津之游不惮命驾人情危震不知所措窃观小民之家居恒父子嘻嘻一旦疾痛疴痒侍奉汤药不离左右非是则隣里乡党指而摘之其有孝行卓著者朝廷特有旌异之典凡以教孝也太上皇帝为天子之父陛下处万乘之尊固不宜下拟闾阎然父子天性自天子达于庻人上下同揆今南北内暌隔末由晨昏侍奉问安尝药又不以时何以示天下后世书曰祗载见瞽瞍防防斋栗以瞽瞍之为父而定省罔懈若乃太上皇帝仁圣慈爱今古罕觏辛亥十一月闻陛下感疴中心惊皇轻舆径至青城殿省视值驾少寐惟恐震惊戒左右勿言俟既寤亲诣榻前抚摩数四殷勤慰谕语云父母惟其疾之忧太上之于陛下何如者而忍忘之今太上不怿思欲与陛下相见者屡矣陛下临朝与臣僚论及过宫未尝不心动迨入后庭即深居不出知必有止而尼焉者伏按内侍陈源绍兴间获罪太上皇帝窜逐陛下复俾给事禁近羣情扰攘佥曰两宫之隙源实搆之圣徳之亏源实致之陛下至亲父子若非谗人投间奚至此极伏望俯鉴舆请斥源軰以谢中外即日夙驾诣重华宫尽问安尝药之节释朝野忧懑之情宗社幸甚臣民幸甚臣世受国恩埀暮之年不忍见此危疑之迹谨昧死上陈干冐威顔无任悚切待罪之至
　　请罢建康行宫防
　　臣闻天生民而立之君使司牧之勿使失性不闻其戕民以树怨也易曰天地之大徳曰生圣人之大宝曰位何以守位曰人何以聚人曰财故民为邦本本固邦宁食乃民天有备无患孔子曰节用而爱人使民以时春秋书新作南门于告籴之后葢言劳民伤财时绌举赢也是以三王之世知人情莫不欲富故厚之而不困知人情莫不欲安故扶之而不危知人情莫不欲逸故节其力而不尽禹汤文武享国长久者人有余力民有余财也秦始皇造阿房之宫而天怒人怨旋致灭亡汉武帝好神仙兴土木之功而民疲财殚夫秦皇汉武以全盛之势而力役繁兴犹有民不堪命之嗟况今偏安一隅四郊多事正人主宵衣旰食之时而臣子枕戈待旦之日也迩者建康留守诸臣廼请治行宫备廵幸臣愚无似窃以为兴工动众有大不便于今日者夫建立行宫规制宏敞采备物料经营土木山川防履之劳水陆舟车之运军匠不足继以平民望风承旨为害何穷语云一人从役举家失业即使琼宫璇室岿然挿汉而道殣相望陛下其忍见之乎哀鸿四布陛下其忍闻之乎一不便也古之经国者曰国无九年之蓄曰不足无六年之蓄曰急无三年之蓄曰国非其国方今军需告急飞挽不时使命频仍刍粟不继天地之生财有限小民之膏血几何势无取不尽而用不竭之理是故善理财者量入以为出不为不经之用不为无益之费太上皇帝肇造中兴建都临安诚以邦家多难实逼处此事从草率一旦禅位陛下宜方励大有为之业即使府库之中金钱贯朽仓廪之内粟米山积而散之孔易敛之维艰尚宜撙节以留有余况处国非其国之时而复为土木以耗之一旦有急其何以应二不便也且陛下纉承大业二载于兹敌人数数入扰江淮之间调发不常民有菜色汉陈蕃有言曰今世有三空田野空朝廷空仓庾空陛下正须休息而安全之养其力无使乏厚其生无使贫鸠流移而安集之庶其有济若使工作方兴强敌乘而袭我疲贫之民望风解散陛下谁与为守虽有行宫将安用之三不便也礼曰四郊多垒卿大夫之辱君父之雠不共戴天留守诸臣罔思报雠雪耻而徒陛下以廵幸不顾君国之利害不念生灵之涂炭欺君病民莫此为甚夫霍去病中材之将犹以单于未殄不治第宅国家靖康之祸二帝防尘而不返两河防没而不复今日割地明日献币忘雠敌之辱图堂构之安此秦皇汉武之事非禹汤文武所宜有窃恐陛下不为也如曰立志恢复临建康以集流离抚三军以图大举则绍兴丙辰太上皇帝行营具在亦何烦更造为也伏望敇下留司即罢其役国计民生幸甚抑臣更有请者符离之溃中外臣民防气罪由都统非人非张浚李显忠之咎陛下知浚之忠而倚任不终俾浚中道捐躯以显忠之戮力王室而窜谪筠州彼忌功误国之人建康都统职任如故岂赏罚之公乎军门行司类皆碌碌庸懦不足与任大事窃恐有悮国家匪浅更乞陛下大奋乾纲早赐斥逐择诸臣之刚正勇毅者简任之俾统辖江淮军马专司留守之寄则将帅得人敌人闻而震慴必不敢称兵南下然后明诏中外待勤王之师四集为祖宗収复陵寝为二帝昭雪垢耻为中原吊民伐罪斯计出万全不然车驾临幸建康无庸也臣有感于中无所顾避自知干冐宸严罪不容逭伏念臣自祖父以来累世防恩以臣一介愚防恭荷太上皇帝皇帝陛下两朝知遇虽捐糜踵顶亦所不惜倘缄默茍容不为一陈利是重负朝廷也率布庸妄上渎睿听不任悚仄待罪之至
　　宋史载乾道初建康留司请治行宫备廵幸兴裔奏劳人费财乞罢其役且言都统及军门行司皆非其人过阙入见询守令臧否兴裔条析以对帝曰卿识时务习吏事行当用卿
　　论淮西荒政防
　　臣伏见陛下以今夏阙雨焦劳特甚起臣于闲散之中抚辑淮西以臣菲才叨任牧守救荒之政臣之责也日夜疚心思有以仰副陛下爱民之意凡蠲下户租赋计赈粜赈济等事条具闻奏谬当圣心亟为报行一路之民见臣以荒政奏闻其应如响莫不以手加额愿国祚万年无疆虽饥寒廹身不肯轻去乡里亦有未经陈请之先流离迁徙钦奉俞防辄皆中道复还傥非陛下爱民至切事干荒政计办早定报下速达则流亡之民已不可遏近者闇于事体之人扬言谓荒政举行失于太早行之太鋭彼但见目前流移稍定便谓可以髙枕万一朝廷误听州县或致解体饥民岂不失望臣虽至愚亦岂瞢然全不体国自始及今施行荒政除桩管常平米数外凡乞度牒截留和籴之属皆非过当难从之请而饥民狠狈多在冬春之交今荒政虽有端绪臣惴惴忧惕尚恐不获善后为之寝不能安食不能甘思今常平之法虽有其名而无其实窃欲复汉耿夀昌遗意即日另具奏请若遽畏立异者之口毎事中废将来必孤陛下委任之意人臣事君意趣不同安辑饥羸乃慰悦君父之急者若预揣君上之心恶闻凶荒饥馑之説以歉为丰此容悦佞臣之所为臣窃鄙之臣一面将本路应有米斛斟量轻重赈救贫乏之民钤束官吏必无妄费伏望陛下详臣所奏遍下诸郡官吏使晓然知陛下爱养元元之意有加无已毋致迎合怠慢则实惠所及一路有更生之幸臣歴世受恩与常人不同茍有所当言者不敢顾避缄黙以负优遇惟陛下留神省察周益公集载郑忠肃公再使金还迁潭州观察使世父荣国公薨请闲致追报之义起守淮西提举荒政活饥民为多
