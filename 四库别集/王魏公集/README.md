　　提要

　　臣等谨案王魏公集七巻宋王安礼撰安礼字和甫临川人安石之弟也登嘉佑六年进士第厯官翰林学士知开封府尚书左丞迁资政殿学士知太原府事迹具宋史本传安石兄弟三人惟安国数以正议见绌其文集亦湮没不传安礼位稍通显史称其以经济自任而阔畧细谨故其生平一以知湖润两州与倡女共饮论罢一以贪论罢屡踬屡起盖亦跅弛于法度之外者然其知制诰时因彗星见极言执政大臣不察上惠养元元之意用力殚于沟瘠取利究于园夫其语皆以讥刺新法则于大体尚能持正固未可以一节概贬之矣其集本二十卷见于宋史艺文志陈振孙书録解题者并同明叶盛菉竹堂书目亦载有王魏公集六册是明初尚有传本厥后诸家书目皆不着録葢自明中叶以后已佚不存今从永乐大典散见各韵者裒辑彚编厘为七巻其中内外制草颇典重可观叙事之文亦具有法度如沈季长元绛诸志铭尤足补史传之阙以视安石虽规模稍隘而核其体格固亦约畧相似也安礼封魏公史所不载惟田昼所撰王和甫家传有云累勲至上柱国爵魏郡开国公食邑三千户食实封五百户葢宋每遇郊恩輙赐羣臣勲封名号冗滥故史不尽载观安礼所修灵台秘苑今着録子部中者书前有安礼署衔一条题上骑都尉剧县开国男而本传亦未之及则其为史所畧者固已多矣
