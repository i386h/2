　　提要
　　中庵集二十卷　　　　别集类四【元】
　　【臣】等谨案中庵集二十卷元刘敏中撰敏中字端甫章邱人由中书掾歴官至翰林学士承防卒追封齐国公谥文简事迹具元史本传史载敏中中庵集二十五卷文渊阁书目作五册不着卷数梁维枢内阁书目不载其名则是时官书已佚明人藏书之家惟叶盛菉竹堂书目仅着于録亦无卷数黄虞稷千顷堂书目虽有其名而独作三十五卷与史不符盖虞稷所列诸书乃徧徴各家书目为之多未亲见其本故卷数多讹存佚不确未可尽援为据也蘓天爵元文类中仅载其贺正旦表忠献王庙碑二首其他作则不槩见今从永乐大典所载搜罗裒辑以类编次尚可得二十卷则所佚者不过十之二三矣其诗文率平正通达无钩章棘句之习在元人中亦元明善马祖常之亚本传称其文理明辞备韩性原序亦谓其不藻缋而华不琢镂而工户枢门键庭旅陛列进乎古人之作固不诬也史称敏中为御史时劾权臣僧格为集贤学士时上书陈十事其文今皆不见集中殆已散佚集中有星变奏议皇庆改元奏议则为本传所未及盖史佚之其金石之文如巴延庙碑哈喇哈斯沙扎该丹达尔耀珠布哈尔李唐诸神道碑记大智全寺碑罔极寺碑皆承诏撰述之作今考元史哈喇哈斯传即用敏中所撰墓碑然不载其在宗正时从世皇北廵猝遇乱突出破敌事又不载其在中书省时每退食延见四方賔使访以物情得失吏治否臧人材显晦年谷丰歉采可行行之数语又度地置两仓句两字讹为内字沙扎该传亦用敏中碑而其子丹达尔谕降襄阳取汉口破婺贼功不在沙扎该下而沙扎该传末乃仅附其子昻阿喇名无一语及丹达尔事尤为舛漏盖元史仓猝成书疎脱实多不但重复割裂如顾炎武所讥则是集之存并可以订史传之讹异不徒贵其文章矣乾隆四十六年九月恭校上

　　原序
　　声音与政通而文词者声之寓也不观夫大化之流行哉一元所寓众彚有得色激者为霆散者为风骞者植者舒者吸者柔且缦者若金奏若万鼓振而余乌乌声万不同其生物一也春出而发之三时奉而成之嵗功以植元化以行然其机之发矢跃而轮运目不得瞬尚可抟而执之哉于是神瞽度律均钟则其自然而寓之于数赞阳出滞修絜万物始乎宇宙之间而伶伦之俦因之而不泯可谓盛矣若昔剏业之君体元出治千载之下洋洋乎如倡而复叹之不足非有寓扵文辞之间哉国家肇造区夏世祖皇帝昭显文治若嵗之阳协风自应当是时觚简奏技仗内者皆天下士所为文词皆足以荅钧天之球而谱云和之奏后世叹息瞻仰以为不可及其乗风云而流声实者固已众矣中庵刘公以文学受简知致身通显朝廷典册钜公铭诔所着为多而集蔵扵家学者愿见而不可得总管魏公公子壻也将刋以传扵代余观公之文不缋而华不琢镂而工不屈折条榦而扶疎茂好户枢门键庭旅陛列进乎古人之作矣其所纪载足以裨太史之阙传之后学传诵玩绎得以审中和之声而窥圣人徳化之盛教思无穷非其他别集所可拟也魏公俾性为之叙辞不获谨诵所闻附扵篇端年月日韩性序

　　励志赋
　　余读书既冠殊拙于生理其于求田问舍之事辽然不知也然给养之资日以艰窘恐以贫乏累其志气乃作励志赋以自勉其词曰
　　吁咄哉何天之生我也悟然而有知既不能粹我以超然而智兮又不能愚我以朦然而痴自亲生之膝下兮固亦知爱敬之日孜元气假我以五常兮父母鞠我以恩慈逮总角之嬉戏兮导我以文字为学之规既庭训之只受兮亦学礼而兴诗信圣之不我欺兮日是诵而是师知仁义之为美兮以忠信而自持纫秋兰以为佩兮服薜芷与江蓠审先民之行为兮独愤然而永思既踊跃以欣慕兮亦慷慨而伤悲彼皋陶之在上兮既帝载之克熙释耒耜以天下自任兮永徽烈乎商伊辞傅岩以即相兮岂时运之或竒也哉夫何三闾之憔悴兮沈溺乎江之湄贾谊倐其轩举兮忽长沙之弃遗嗟小人之纷扰兮莫不移世以阿时既靡靡以自安兮亦睊睊以相危恃其志之已得兮惟侈纵之日滋苟一时之荣耀兮实万世之嗤也兹既通塞之异轨兮何清浊之不可力而推也蹇余生之不辰兮抱寸诚以孳孶企古人之悠邈兮喟遑遑其欲追拟修名之屹立兮以圣道而自期春与秋其代序兮日月忽其若驰翔世途以徘徊兮抚斯文以嗟咨时既与我龃龉兮事与我而相违学澶澷而未精兮形瘵弱其神疲顾生资之迫狭兮虑给养之莫支余既不能规规于田舍兮又不能求龙断而登之望朱门而将进兮无愉色与妍辞愿得羽翼如云之我身兮余将薄六合而逺飞又欲浮灵槎于沧海兮泛昆仑以游嬉乗天风以越阊阖兮访王母于瑶池饵金砂以炼质兮潄琼液以生肥跨鸾鹤以为驭兮襜云霞以为衣亦知兹懐之不可必兮竟郁郁以安归已焉哉盖士于世也穷达否泰讵能极兮姑俯仰而随宜幸孔门之可以倚依兮依顔曾以栖迟从众人之相易兮保良心而靡移听天命以升沈兮余终不能若儿女子之所为也
　　东山赋
　　东山者实长白山也自歴城观之故言东山时吾友人智仲敬寓迹于兹山之左故作东山赋以叙相忆之意云耳
　　歴城之东直望百余里见云峰之嶕嶤实曰长白傲然今古横絶于层霄崖崩壁削岚飞烟薄兮变气候于昏朝麋鹿游而猿鹤哀鸣兮木无风而萧萧念佳人兮此留度期防兮何遥想夫君兮安适抑余懐之罔极或登髙而望逺亦沈吟而太息穷仙台之胜境访黉堂之遗迹慨然长歌而悲啸忽风惊而云析山泉鸣兮笔雨寒山光动兮酒波碧发壮懐于尘世飞妙墨于石壁仰苍穹兮何言抚丹心兮谁识骐骥之骏足戢鸿鹄之劲翮亦必能扫情而涤思听其时而自得望夫君兮安至忧纷纷兮心醉知危楼之几登越河桥兮再四目苍波之东注恨白日之西坠每当渇君之徳望君之顔兮空徘徊把酒挹夫东山之余翠
　　奉试俊民甸四方赋
　　运启明主时多俊民列庻位以咸若甸四方而有因伟兹间出之贤优加委任治尔攸同之域尽入陶钧书曰民罔胥匡也赖君宠绥君不徧爱也须臣共理然英特之士弗得而用则治化之功曷从而起载惟此故任三俊以为先肇自于今甸四方而有以以尔瑚琏之秀楩楠之良抱经世之才者乃稷乃契明抚字之术者如龚如黄上果纯于任使世可拱于平康蔼蔼盈庭觏尔云集之士皇皇奠枕安于斗制之方夫何泽周人物之繁教被海隅之四外张国势之隆盛内格邦风之纯粹良由妙于选贤使之在位有文学有徳行以次而升自南北自东西不劳而治惟贤臣乃能圣主之业而君子所以为邦家之基方国必欲甸也俊民其何舍之以见周任仁人国果闻于昌炽殷获良弼邦自底于安绥是知民区甚逺也难徧于照临事几至微也孰知其纪统必求治效之克即可不人材而与共不然何以疏详方朔云得士则乃昌传质班生谓惟贤而是用已而公道无壅俊民必登忠殚兮力竭害除兮利兴翕若四方之作乂截然庻绩之丕凝如是下彰报国之劳咸伸素愿上享得人之逸益介休徴大凡当世之安皆本真贤之进舜帝以是而大辟于仕路唐尧以是而克明于登俊方今网罗天下之贤才而甸夫四方功业逺超于尧舜
　　腊粥赋【原注戯为智仲敬作】
　　东里先生腊辰入学为诸生设粥以应故事方其为粥也戒灶厨之神洁锜釜之器屏黜异馔扫除邪味意以清而取竒物以多而为贵于是勑老庖役台舆鸡未鸣而先起咸受命而奔趋积薪若山贮水成湖大杓巨盎后列前敷淅长腰之玉粒泻小豆之红珠破霜粟之金丸屑山药之琼肤纷众材之毕集俾效用于一区方相接其未久迹难混而犹踈逮夫坎离既济之后庚癸相合之初翕和气之囘春【一作云蒸】破腊寒于须臾乍蚯蚓之一鸣忽秋虫之满庐俄而怒风逺来万窍号呼撼千丈之长松响十里之楸梧少焉则惊湍拍堤急溜鸣渠渐威声之衰息方讼语之嗫嚅复若泣以若叱亦或吹而或欨忽蓬蓬然告熟恍融蜡以凝酥品色香之殊絶赏至味之芳腴笑仓惶于公孙陋咄嗟于齐奴彼黄冠与缁衣方击钟以鸣鱼虽大同而小异讵可企吾之清虚于时朝日升阳和舒乌丸炯兮映窻兽煤烘兮满炉则有龄稚儿青衿小儒若骥之子或鳯之雏咸鞠躬而来思矜步武之襜如揖先生而登堂各齿列乎坐隅比粥令之一行皆攫椀以拏盂俄匕鸣而颊响惊齐噪于千夫类昆阳之战酣卷百万以如无先生熟视抚掌助声势而奋须谓吾党之能事快雄俊之起予顾童子其何知已皷腹而奔衢矣先生于是余勇未歇食思方骋取谷气之精华喜渐入于佳境促孟光之举案方缓带而笑领鄙勺饮之孱懦空数杯于俄顷已而玉池之水生甘诗腹之雷不鸣和畅五神驱除三彭杜邪疾于他日养浩气于平生彼东坡之玉糁与吴下之莼羮较功用于锱铢盖未许其同盟况乎周顗之炙孝仪之鲭皆潜伏于酖毒亦何恤乎羶腥将奔避之不暇尚安敢婴吾锋而抗吾衡也先生乃旷然心平融然气和舒然而伸悠然而歌歌曰以不杀而仁兮吾何忧既易得而易足兮吾何求谅此味之天成兮吾何以酬虽然无用山泽之癯兮犹可以为肉食之谋也
　　骚
　　思兰辞二首并序
　　大易称同心之言其臭如兰屈原著离骚以香草譬君子有滋兰纫兰之语则兰之见重于世久矣吾党张君智甫隠居好古取兰江南植之家园月养嵗滋九畹可待且命其所居之室曰楚香食饮起居必与兰对得君子同徳之义若是兰与人偕重矣然余徒知重其所重而兰则未见也乃拟楚词作思兰吟二首贻智甫俟他日醉楚香堂歌以为兰慰其辞曰
　　兰生兮猗猗其芳兮菲菲山空兮无人水溅溅兮石累累楚魂不归兮夫谁与期俨髙洁兮自淑谅无待兮奚悲胡寸根逺播弗瘁而且宜兮曰美人礼我而不我疑同臭协徳兮尚庻几爱之兮不见怅余思兮眷眷兰生兮修修其芳兮幽幽山空兮无人树林隂兮猿猱愁楚魂不归兮夫谁与俦俨髙洁兮自淑谅无待兮奚尤胡寸根逺播弗瘁而且售兮曰美人礼我而我息我游同臭协徳兮庻久留爱之兮弗得怅余思兮恻恻
　　五言古诗
　　送胡学士紫山
　　去古日已逺正路多迷踪众説挠六经啧啧纷异同繁音笑雅颂冶容衒青红风云与月露一字夸余工譬如膏肓疾不受砭剂功自非名世贤反正难为功寥寥几何时得我中庸公公姿何神秀自是间气钟蔚蔚天上麟矫矫人中龙斯文入掌握造化盘心胷吐辞烂星斗落笔驱雷风传写圣人意秘奥咸开通竒伟出韩栁正大续轲雄垂绅玉堂上文风变区中贱子生陋壤志学不知宗北骛驾南辕失计为雕虫居然守迂僻无由去颛蒙公来幸一见谈笑及从容快意徒仰羡抠衣遽难从明朝驿亭外凴髙望飞鸿
