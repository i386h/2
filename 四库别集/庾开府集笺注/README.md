　　钦定四库全书　　　　　集部二
　　庾开府集笺注　　　　　别集类一【周】
　　提要
　　【臣】等谨按庾开府集笺注十巻周庾信撰
　　国朝呉兆宜注信字子山新野人梁散骑常侍中书令肩吾之子元帝时仕至右卫将军封武康县侯加散骑侍郎梁亡仕魏为车骑大将军仪同三司魏亡仕周为司宗中大夫大象初以疾去职周亡入隋卒于开皇元年事迹具周书本传考集中辛成碑文称开皇元年七月某日反塟河州则入隋几一载矣信为梁元帝守朱雀桁望敌先奔厥后歴仕诸朝如更传舍其立身本不足重其骈偶之文则集六朝之大成而导四杰之先路自古迄今屹然为四六宗匠初在南朝与徐陵齐名故李延寿北史文苑传序称徐陵庾信其意浅而繁其文匿而采词尚轻险情多哀思王通中説亦曰徐陵庾信古之夸人也其文诞令狐德棻作周书至诋其夸目侈于红紫荡心逾于郑卫斥为词赋之罪人然此自指防城应教之日二人以宫体相高耳至信北迁以后阅歴既久学问弥深所作皆华实相扶情文兼至抽黄对白之下灏气舒巻变化自如则非陵之所能及矣张説诗曰兰成追宋玉旧宅偶词人笔涌江山气文骄云雨神其推挹甚至杜甫诗曰庾信文章老更成凌云健笔意纵横后来嗤防流传赋不觉前贤畏后生则诸家之论甫固不以为然矣北史本传称有集二十巻与周滕王逌之序合隋书经籍志作二十一巻皆已久佚倪瓉清閟阁集有与彛斋学士书曰闻执事新收得庾子山集在州郭时欲借以示仆不时也兹专一力致左右千万暂借一观云云则元末明初尚有重编之本今亦未见此本虽冠以滕王逌序实由诸书抄撮而成非其原帙也隋书魏澹传称废太子勇命澹注庾信集其书不传唐志载张廷芳等三家尝注哀江南赋宋志已不着録近代胡渭始为作注而未成帙兆宜采辑其说复与昆山徐树谷等补缀成编麤得梗概然六朝人所见之书今已十不存一兆宜捃摭残文补苴求合势不能尽详所出如注哀江南赋经邦佐汉一事引史记索隠误本以园公为姓庾以四皓为汉相殊不免附防牵合后钱塘倪璠别为笺注而此本遂不甚行然其经营剏始之功终不可没与倪注并録存之亦言杜诗者不尽废千家注意也兆宜字显令呉江人康熙中诸生尝注徐庾二集又注玉防新咏才调集韩偓诗集今惟徐庾二集刋板行世余惟钞本仅存云乾隆四十六年九月恭校上

　　吴江吴兆宜注
　　三月三日华林园马射赋【并序　徐树谷曰周明帝纪武成二年三月防羣公诸将卿大夫及突厥使者于芳林园赐钱帛各有差园在长安明矣赋称嵗次昭阳系辛已年乃武帝保定元年也盖武帝初即位踵行故事而史偊失书耳按魏志云芳林园避少帝讳改曰华林是则邺下圥有芳林园后既无讳当避而此赋亦改芳为华未详何故】
　　臣闻尧以仲春之月刻玉而游河【帝王世纪后年二月尧率羣臣刻璧为书东沈洛水】舜以甲子之朝披图而巡洛【帝王世纪舜摄政二十八年而尧崩三年防毕舜年八十一以仲冬甲子日次于毕始即真以土承火色尚黄　徐炯曰竹书纪年舜设坛于河依尧故事至于下昃荣光休至黄龙负图出于坛畔】夏后瑶【疑作璿】台之上或御二龙【竹书纪年曰帝启元年大飨诸侯于璿台山海经曰大栾之野夏后启乃于此乗两龙】周王圃之前犹骖八骏【穆天子传铭迹于圃之上以诏后世天子命驾八骏之乗右服华骝而左騄耳右骖赤骥而左白义次车之乗右服渠黄而左逾轮右盗骊而左山子】我大周之创业也南正司天北正司地【史记歴书颛顼命南正重司天以属神火正黎司地以属民】平九黎之乱【帝王世纪颛顼平九黎之乱】定三危之罪【尚书舜典窜三三苗于三危】云纪御官【三皇本纪黄帝官名皆以云】鸟司从职【左传剡子曰少昊挚之立也鳯鸟适至故纪于鸟为鸟司而鸟名鸟氏司分者也】皇王有秉歴之符【尚书大禹谟天之厯数在汝躬汝终陟元后】珪有成功之瑞【尚书禹贡禹锡瑞告厥成功】岂直天地合徳【系辞夫大人者与天地合其徳】日月光华而已哉【吕氏春秋虞舜卿云歌曰日月光华旦复旦分】皇帝以上圣之姿膺下武之运【诗下武惟周】通乾象之灵启神明之徳【系辞干天也天垂象见吉凶又故神武而不杀以神明其徳夫】夷典秩宗见之三礼【尚书舜典帝曰有能典朕三礼佥曰伯夷帝曰汝作秩宗】夔为乐正闻之九成【刘向説苑夔为乐正尚书夔曰箫韶九成】克已备于礼容【论语克巳复礼为仁胡渭生曰汉书龚遂曰坐则诵诗书立则习礼容】威风总于戎政【渭生曰潘朂魏公九锡文】
