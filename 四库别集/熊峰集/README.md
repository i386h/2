　　【臣】等谨案熊峯集十巻明石珤撰珤字邦彦藁城人成化丁未进士官至文渊阁大学士諡文隠改諡文介事迹具明史本传珤出李东阳之门东阳毎称后进可托以柄斯文者惟珤一人皇甫汸尝删定其集为四巻嵗久板佚

　　送杨提学应寜
　　蚤闻伯起誉亦欲抠吾衣长风振郊壑云龙忽分飞仰止徒髙山而势不能依文章绚圭璧日得窥余辉臯鸟倐一举渊鱼遂其肥道合志乃同此意识者稀所以渊明琴不在与徽京国再相逢蹲豸有余威一展畴昔愿清论聆霏霏洒我尘埃襟登之清风矶虽非暮春姿忽尔吟咏归何怪公门人器识俱可希公门有青骢行色动朝晖安能一束刍聊以驻騑騑晨星本易散流云亦易防遡彼西北风兹懐良不违
　　旭日春耕为阎大防赋
　　东方白已发垄畮明浩浩不辞荷耒倦穑事贵及早道逢同心人劳问亦草草共言年谷登百嵗可相保
　　秋日遣兴
　　旭日戒僮夫驾言适东野市朝垢心境广莫快一写况值目疾除秋色尚可假岂徒望封龙岩壑翠堪把近观百谷繁一一含大化我虽宦游人自是悠悠者忽逢邻叟来相欢坐桑下
　　寄兄旧作
　　寒空变春云奔骥逐迅景君看宫前杨忽已露碧影幽居西牕下感事发深省谁言有为者譬掘九仭井盛顔易消歇此道独未领空奏郢人吟懐抱殊耿耿吾兄绣衣使仗钺指西境清诗赏风流正论崇骨鲠寄言愿相朂共此百嵗永
　　鼓不絶
　　邻巷相见语兹农薄收成何縁累旬日箫鼓未絶鸣东家男已长西家女将行羔雁来问吉鸡豚往通名置宴召賔侣责逋急官征斛粟易百钱入坐恐未盈花顔照四席典客稽考明布帛身不着妻子徒力耕违俗惧乖义先王近人情歌管沸两耳中心悁且惊但见蚁杯緑不知鲂尾赪凭君聴箫鼓中有劳者声
　　怨歌行
　　珠人本中国万里向海陬结网沈洪波深入鼋鼍湫瑚由寸积骊龙不易求百死冒霜雾累年被淹留举手得夜光宝树照斗牛三献入汉庭声名动九州日往嵗亦徂忽生瑕纇忧长恐弃道侧勤苦不能酬岂惜寒与饥惧贻重宝羞中夜起长叹泪下如泉流
　　春日即事
　　林塘鸣春鸠忽忽过谷雨繁红多辞柯緑甲半出土鱼欣荷钱青鸟悦石发古凌云思髙台对日作独舞为生良支离被命孰伛偻悠然东风来兀坐聴杜宇
　　饮鲍云瑞荘
　　城栁初尽处薄霭开空明潦降古洲出千里关寸情防岸带芳草逺树啼残莺钓艇日往还或有羊裘生下马对尊斚故人相与倾坐久出歌舞柔风濯长缨落霞酣野色吟蛩战秋声长恐西日下衢路立屏营
　　夜渡滹沱
　　西日光已微跨马出村境唧唧林鸟鸣仰见河汉影南巡古堤麓地峻行屡警踏软知近沙野濶夜逾静谁将笙歌耳复聴秋虫哽万事倐春梦对此山河永惊波啮防岸跳梁老鱼猛星斗揺长练薄吹搜鬓頴鼓枻歌沧浪懐人忆千顷遥望城西门熠熠烟火冷
　　秋夜
　　凉夜始多梦升沈顷相寻蟠蟀入牀下萧序已载阴褰裳出庭戸河汉清且深华月挂林杪髙鸿送哀音百嵗岂不多半为风雨侵嗟彼蜗角名烦此孤鹤心流水激古洲髙山出逺岑子期果难逢独卧枕素琴
　　蚤起
　　城阴苦寒沍积雪冬未消日夕人马驰流尘乱琼瑶抱志谅犹洁凌风讵能飘徒然漾空月皎皎射牕绡野客官大都鸡鸣晨朝卧闻车毂声轧哑上干霄起披黒貂裘啓明正揺揺竦踊不能寐曳佩趋午桥
　　寄罗汝实
　　寒斋坐无事对此春昼雨双鸿自南来流韵落庭庑起寻故人书别绪那可数春草南园緑秋月西林吐歌诗毎达夜笑噱曽过午云龙忽乖张感物益凄楚何时倒清尊共醉恒川浒
　　雨中寄白文明
　　开书复开书云影照顔色东风昨夜至渠栁柔无力虽有龙泉劔袖短时拂拭时人竞弯弧古道恶直圣王崇旧儒不忍弃一得漫随东观班岂有圗南翼经书虽满案抱病长忧恻遥见孤云飞伫立而叹息
　　新妇词
　　丑妇羞见姑终难不上堂鸡鸣理箴帨对镜着衣裳薰沐再至三稍觉有容光长跪问起居顾影尚徬徨大人总家政百务惟徳将举手能绩麻出门时采桑辛勤无日夜欢喜动姑嫜入与诸姑游诸妇日翺翔爱璧不见瑕度材或惜长邦家本一致竭力安可忘
　　章锦衣园饯克温
　　惜别驻郊坰名园及璀璨朱荣悬弱葼清樾防修榦载展夙昔欢颇兴萍蓬叹觥繁乱回波坐逸何泮涣妙舞出京洛清歌彻云汉探幽入虎谷蹑蹬耸飞翰几欲凌长风六极游汗漫主人爱真景废榭临防岸岂惟示朴淳正欲知忧患吾侪睹落叶朝聚夕又散当时盛红顔几许发已换安知垂汗青试聴歌石烂愿言保终始庆云旦复旦
　　杂诗
　　妙舞恒赴节善歌多应艺精契至理轮扁非言传情矫性始凿礼烦本日颠所以大观人游目无为先岂不笑与嚬造适意悠然投足得规矩率真外蹄筌庶几达圣境并可延天年
　　雨坐述懐
　　结发事竒谋年徂志弥逺虽蒙簮绂荣抱病日长偃兴到偶繙阅吟哦不终巻古来忠与邪往往激褊浅长城能自壊元恶畴克殄所以乐道人但欲脱轩冕境大意自舒身安虑长鲜此理本分明不用更辗转
　　杂诗
　　大朴已雕散頽波漫末涂牋诰日纷纶君臣互为谀直忽见絶白璧多掩瑜但闻都俞声无复咈与吁往往抗顔色犹称迈唐虞唐虞不可见烈士始全躯王掾名痴絶槐里号狂愚安能起臯益重述邦家谟
　　春出西郭门
　　芳郊艶春阳百卉待葱蒨念我多悲人如何及游宴闻鸦堕清泪睹草増眷恋岂独感蓼莪正自哀衣线仰天恩防极衔恸髪先变愧彼九臯禽和鸣彻霄汉
　　拟古
　　阅世贵知忌肮脏始多諐趾发物已迕意行悔相连苦抱叔夜僻谁辨箕季贤坦枉固有分直丝寜改岂不悟裸挝贾祸几席前鸿鹄各负志輮方岂成圎出门欲学步足乱首忽颠何如执坚贞嵗晏循固然王道本寛衍锲薄世谓良鞟革大始裂棰轭齐已僵虽剔毫末疵不顾根本伤譬彼几上琴轸急寜久张中行任智败季孙以敛亡覆车鉴不逺胡为复周章庙堂宜有枢开阖道始昌凭谁用杨绾暂为辍丝簧
　　熊耳峯
　　封龙山势壮嶻嶪见熊耳髙峙层霄间下聴几千里威深貔虎避意拔雷电起乃知天降灵自与凡石异浮岚出晴丹淑气凝夜紫彷佛悬崖旁呵防有神鬼我思汉三老幽谷访遗屣人去事亦非荒池但秋水地灵应未散烈士还继轨遥遥白云中何人隠髙几
　　恭睹临雍释奠敬述以诗
　　璧水晨漭瀁石坛花始垂云开大成殿桧栢郁葳防盛王重孔道祼献皆亲持箫韶切云汉八佾盈堦墀礼成进诸生讲席命尔师敷陈唐虞理论説羲文辞坐久进珍茗雨露必旁施三氏既荣耀九卿亦华滋东瞻学士班文采何陆离各敛补衣手黙黙犹论思斯文日中天志士正扬眉
　　送王司徒分韵得五字
　　在昔称髙士箕山及巢父虽垂千载名吾道固未取君臣义自重岂得薄圭组惟是勲业成几防贵先睹二疏辞汉荣季鹰懐吴土司徒志亦然百世殆同矩辛勤三十年出身事清主行者必有归安得长伛偻明山有茅亭亭南有圃象观有巻舒理玩月三五始息负重肩时与荷蓧伍睡足江南春无人报风雨
　　夜至张秋
　　日暮出荆门夜行复十里半帆乗顺风一篙得春水篝灯危樯末余映彻洲沚逺逺闻犬声渐喜近井市缆人相和歌饥渇被驱使恨无浊酒劳毕力共王事

　　过沛望歌风台
　　三王久不作淳风变縦横兵戈入战国耆宿填秦坑天赐赤帝子龙泉定升平岂独体貌异圣度畴与京功成归故里慷慨道平生意气自雄俊歌词亦峥嵘至今千载下闻者心胆明是宜冠庶物直与商周并帝子乗龙升故台未全平台下春草緑时鸟间关鸣猛士安在哉四海复永清仰见云飞扬遡风有余情
　　送王尚书徳辉还余姚五首
　　杳杳稽山青茫茫舜江白意倦行且休身存嵗寜迫唐虞世已逺孔孟道未塞长啸凌秋空黄花正盈陌
　　其二
　　吾行何所求青山映华发板舆傍春花清波溅仙韈功名一炊黍人世几明月歴览千万秋百巧未如拙
　　其三
　　春花岂胜秋新人不如故纷纷红紫羣酣艶良未悟千金买马首百防捕毚兔一曲雍门歌夕阳在髙树
　　其四
　　塞北多归雁江南有去人尘寰回鹤梦神剑敛龙鳞童冠同三月莺花又一春羊裘滩底在吾欲问吾津
　　其五
