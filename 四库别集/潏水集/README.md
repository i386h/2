　　钦定四库全书　　　　集部三
　　潏水集十六巻　　　　　别集类二【宋】
　　提要
　　【臣】等谨案潏水集十六巻宋李复撰复字履中先世家开封祥符以其父官关右遂为长安人登元丰二年进士歴官熈河转运使终于中大夫集贤殿修撰其事迹不见于宋史洪迈容斋随笔载其于蔡京邢恕谋用战舰一事上疏排诋甚为切直而恨史之不能详尽朱子语録亦曰闽人李复【案复非闽人此句或传冩之误】及识横渠先生绍圣间为西边使者博记能文今信州有潏水集者即其文也其间有论孟子养气谓动必由理故仰不愧于天俯不怍于人无忧无惧其气岂不充乎舍是则明有人非幽有鬼责自歉于中气为之丧矣此语虽踈却得其大防近世诸儒之论多似过髙流于老庄而不知不若此説之为得也今观是集如谓扬雄不知道谓井田兵制不可遽言复古皆确然中理其他持论亦皆醇正不止朱子所称一条又久居兵间娴习戎事故所上奏议大都侃侃建白深中时亦不止洪迈所称二疏至其考证今古贯穿博洽于易象算术五行律吕之学无不剖晰精微具有本末尤非空谈者所可及在宋儒之中可谓有用者矣集本四十巻乾道间尝刻于饶郡即朱子所谓信州本也后散佚无存谈宋文者多不能举其名氏今从永乐大典裒辑编缀厘为一十六巻着之于録既以发潜徳之光且以补史传之阙畧焉乾隆四十六年九月恭校上

　　潏水集巻一　　　　　　　宋　李复　撰奏疏
　　论治道
　　臣闻圣人御天下也必以道而道者南面之术也其所谓道者岂但漠然而无所事哉其用至防其功至周皆隠于纲纪法度礼乐徳政之间使四海安然而无事至千万世而无弊天下由之不知其所以然而然也古之致治者惟尧能之孔子称之曰焕乎其有文章乃纲纪法度礼乐徳政之谓也巍乎其有成功乃天下安然无弊之效也荡荡乎民无能名乃不知其所以然而然也夫惟如是所谓道也后世无可稽焉前世自唐末至五季天下糜烂大壊有识者伤之至于不忍言国家之艺祖太宗潜而未跃熟稔昔者祸乱之所由起自膺天命凡立一法欲絶一蠧凡举一政欲去一弊小大逺近皆有纲纪法度维持不以喜而妄予不以怒而妄罚使居官者修其职安其分而不敢妄作不敢茍悦无狂易徼幸之心百姓守其业乐其生无横扰困苦之患累圣循之迄今一百六十年天下晏然自三代而下未有若本朝平定之久也恭惟陛下即位以来延见臣下必访治道四方闻之皆曰今唐尧在上矣臣愿陛下思祖宗修立纲纪法度维持天下之意不取目前之虗美而求经久之逺虑使上下各安其分守职业具举朝廷清明民物繁富弊无由而起蠧无从而生天下不知其所以然岂非配天广大之业哉治道莫盛于此臣疎逺愚憃不知治体狂妄献説惟赦之幸甚
　　议礼
　　臣闻圣王治世莫重于礼事不由于礼无巨细皆不可行三代之礼至周而备今考诸载籍所传者十无二三前世江都开元皆尝纂缀旧文而行之当时折衷执于古者泥而不通顺其时者陋而无法学礼者有所不取后世无可稽焉传曰治定制礼国朝承平一百六十年髙出唐虞岂三代可拟一代礼典今犹未讲至使好礼之士有家自为礼者荀况云礼莫大于圣王是惟圣人乃能制礼惟王者乃能行礼记曰非天子不议礼不制度不考文臣愿诏有司上自郊庙社稷下至三祀一祀与夫冠昬防賔军辨其等威裁其文物不僭不偪据于古而不泥宜于今而不陋着为一代之典其士庻所当行则颁之郡县使通知焉事有制度灿然可观四夷百蛮向风取则为治世甚盛之举岂胜幸甚取进止
　　议乐
　　臣闻治定制礼功成作乐此王者甚盛之举天下熙洽人心悦豫发为和声因其人声之和而播之八音又形容其成功之象也三王不相沿乐岂茍为异哉治世成功各不同也记曰大乐与天地同和乐岂易知乎三代之乐亡已久矣唐贞观中命祖孝孙张文收考定雅正粗而未备后累经防乱其器与书今皆不传载籍所言虽皆以黄钟为本上生下生隔八相生及其律管径寸短长但糟粕耳有能遗其旧説脱然识其声别其音者未之闻也夫黄钟律之始也半之清声也倍之缓声也三分其一而损益之此相生之声也十二变而复黄钟之总乃旋相为宫之法也万物动皆有声若造乐精防之妙凡闻其声则知是何音合何律是为正音是为变音是为清是为浊如此方为知音可以议乐矣近者陛下有诏选官定乐又愽求前代之器夫前代之器各一时之用若得汉唐之器乃汉唐之乐也若得魏晋之器乃魏晋之乐也但欲求为多见则可矣遽欲用为今日本朝之乐恐未然也晋之荀勖取牛铎为黄钟出于独见果合于古乎乐之作欲动天地感鬼神自汉以还未之闻也朝廷昔尝定乐矣陛下以为未尽美善亦不能形容祖宗之功业而又本朝运膺火徳独徴音未明此固当重为考定也今闻众议又只依往昔糟粕而制器此安足以副陛下所降之诏意夫知音者闻之于耳得之于心自不能传之言遇其应于心方可黙契徴音火南方之音也火性炎上音当象之乃欲就其下而抑之恐非也臣愿诏天下广求天性自能知音者敦遣令赴议乐所多方以试之是诚不谬共为讲论庻几其可矣若徒以旧説尺寸长短广狭重轻而制器此工匠皆能为之矣何足以为乐乎臣愚见如此惟陛下择之
　　论虗名实弊
　　臣闻古先哲王之举事也常艰于其始而深虑于其终始虽可为终不可继则不为盖虑得其虗名而受其实弊天下四海虽甚大亦犹庻民之一家以一家之事推之乃天下之事尔臣尝观旧史见前世不能深思逺虑轻动生患者其事甚众不敢逺引以凟圣聪以臣今亲见所谓庻民之家者论之臣居有邻人承其父业负郭有美田十余顷衣食富足不能力穑笃治为人所怵喜多田之豪名罄索家资又营逺山之瘠田数十顷欲人称其田之多也逺田无所得常以负郭资之嵗久因逺而困此求虗名而受实弊也陛下承祖宗积累之休无幽不烛无逺不察其多事轻动以求利者不能昧圣聪必乆已察之矣固不待蝼蚁之防献其愚臣惓惓之诚更兾终谨之幸甚
　　论取士
　　臣恭覩神宗皇帝悯士弊于俗学之久慨然作新造之以经术发明圣人之遗言使讲求义理之所归庻知乎修身行已上以事君内以事亲涖官接物弗畔于道而今之学者曽不思此平日惟是编类义题传集海语又大小经题目有数公试私课久已重叠印行传写其义甚多无不诵念公然剽窃以应有司之试终身之学止于如此甚者至于所专之经句读不知音切不识或误中选入仕平生所学皆无可用非惟乡闾无一善可称虽有甚不齿者亦更不问朝廷建学立师设馆给食而偷惰茍且若是安能副上教养之意哉欲责其移孝资忠临民应务之效必不能也古者乡举里选非但取其浮文必皆考其素行臣欲乞立法取士以博学行义为先试言为次抑亦絶其干托奔竞之私察其器识材术之异庻几所养可取所取可用圣朝有得士之实取进止
　　乞置塲
　　臣窃见回纥于阗卢甘等国人尝赍蕃货以中国交易为利来称入贡出熙河路朝廷察知其情故限之以年依到本路先后之次发遣赴阙而来者不已守待发遣有留滞在本路十余年者其所赍蕃货防入诸路多是禁物民间私相交易逺商物货厚利尽归于牙侩臣累次详问所赍物货上者有至十余万缗下者亦不减五七万且逺人懐久客之情平民陷冒禁之法利赢无极尽归于牙侩往来无已每遇发遣徒扰州县今湟州新复正要措置使商贾奔辏不惟通其有无诚亦厚其根本之势臣今欲乞于湟州依雄州火山军等处例置立塲于湟州别置蕃市以居来者更不发遣赴阙使利归于公货通于下亦可少补经费取进止
　　乞置弓箭手堡
　　臣窃见极边弓箭手就土山削成峻壁盘开细径凿穴以居谓之崖巉凡一巉所聚少者百余家多至三四百家其情非不乐居城邑就堡鄣意务便于耕牧又宻迩敌境相与保险以防抄掠之患然本欲防患贼防若至患不可逃盖各潜于穴不能相救贼前阨其门寻气窻灌浸熏燎束手就祸向者元祐间夏贼防犯泾原大被其毒边民皆自知其如此顾恋生业无以为计臣欲乞下逐路经畧司委官相度就弓箭手耕牧近便处团并指挥建置堡子分擘地步先为土堋渐次修完使之就居备列战格开掘壕堑修立门桥一堡之内正丁家丁不减二三千人于其暇时教以战守之法逐指挥人员将校统领分定城上地分毎季轮那城寨官前去防检地分廵检常切往来照管遇有探报无起遣之扰虽有攻围必以死守逺近声势相接互相应援近年新边城寨相去近者四五十里缓急卒不相应此垒若成血脉方通人旅行役不及城寨可就安泊不待官兵戍守而藩篱宻完诚一举而两利取进止
　　乞罢造战车
　　臣准枢密院劄子奉圣防令本司制造战车三百辆专令臣催促制造应副本路出入攻战使用臣尝览载籍古者师行固尝用车诗云公车千乘公徒三万又曰元戎十乘以先启行及周制自步百为亩积而至于通十为成成出革车一乘其他班班亦可见古之用兵与今不同古者兵不妄动征战有礼不为诡遇皆有法度多在平原易野故车可以行敌人不敢轻犯此以车为利也今之用兵尽在极边戎狄乘势而来虽鸷鸟飞翥不如是之迅防下寨驻军各以保险为利得其机便或有负败各逃散不相顾其往车不及期居而保险车不能登归则敌多袭逐争先奔趋不暇回顾车安能收非若古昔于中国为用臣在兵马间亲冒矢石前后十余次观之屡矣有至粮糗衣服器械不能为用者又况于车乎闻此议出于许彦圭彦圭因姚麟之子师闵而得见麟遂献説于麟恳告求以此为进身之谋麟遂怜而上其説朝廷以麟边人必熟于边事意遂然之不知彦圭剧为轻妄麟立私恩不思误朝廷之事是亦容易昔唐之房琯尝用车战大败于陈涛斜十万义军无有脱者当时止在畿邑平地且如此况今欲用于峻坂沟谷之间乎又战车比常车濶六七寸运不合辙牵拽不行今配买木植物料差顾工匠大为费扰既成艰于牵拽昨东来者牵拽不行致兵夫典卖衣物自赁牛具终日方进五七里遂致兵夫逃亡战车弃于道路大为诸路之患其糜费不知其几千万矣茍望一官之进上防朝廷下害百姓此而不诛何以惩后臣今乞便行罢造如别路已有造者乞不牵拽前来免致徒费人力如朝廷未以臣言为然乞赐博访谨具奏闻伏勅防
