　　钦定四库全书　　　　　集部四
　　提要
　　芸庵类稿六巻　　　　别集类三【宋】
　　【臣】等谨案芸庵类稿六巻宋李洪撰洪姓名不着于史诸家书目亦未载有是集惟永乐大典颇散见其文又别出陈贵谦原序一篇称洪为李正民之子又称为藤州使君知洪官止知藤州集中又有扈从朝德夀宫景灵宫诗则曾为京朝官有在温州题其父手泽诗称嗣子守官于此则曾知温州其他不可复考矣正民本州人南渡后侨寓海盐故洪集中如乌龙井庙迎送神词之类皆在海盐所作而卜居飞英坊一首乃云束书辞都门整棹还苕川则洪后又移居湖州乃两郡纪流寓者俱不及其父子则由遗集散佚征文献者罕及也洪所作诗虽骨榦未坚而神思清超时露警秀七言律诗尤为工稳足以嗣响正民陈贵谦序称原本二十四巻而今所掇拾仅得诗词三百九十余首文三十首视原集祗十之三四亦云仅矣据所存者厘为五巻而以杂文一巻附之畧具梗概俾不致终就泯没焉乾隆四十六年九月恭校上

　　原序
　　大隠先生李公侍郎讳正民在中原时以文章名天下渡江南来扈从巡守参为名命条画军国大计于艰危时最号忠力诸所论撰亦既盛行于时其子藤州使君讳洪则有芸庵类稿二十巻贵谦得而读之因窃有叹昔者士君子之学皆以养气为本文章者葢气之所寓而气非専于为文也侍郎君殁且八十余年读其文凛然犹有生气若使君所着该括众体每于草木鸟兽之微有可寄兴以为忠邪贤否之辨者未始不反覆致意仰冦莱公之忠大顔平原之节慕陶渊明之隠斥盘谷李愿之汰而知韩子有为言之也其视前所谓凛然者似之非耶夫长江大河迤延浩汗虽其包纳深广要亦发源之逺与众水异尔古之人不以着姓世禄为贤而以立功立言为不朽葢谓是也使君长子吏部直养尝以使事至襄阳兵间风绩甚伟次子直柄今守湖南望郡有治理行见世其家者殊未艾也观是篇者又可以知其传矣陈贵谦序

　　芸庵类藁巻一　　　　　　宋　李洪　撰赋
　　适越赋
　　出脩门而东骛兮驾灵胥之怒涛命榜人而理櫂兮御龙骧之巨艘睨组练之喷薄兮忽鹅鹳之鸣号江海淼弥以东注兮雉堞屹然而増牢啓东南之王气兮应赤县之神臯仰中天之华阙兮非烟郁以霞韬庆警跸之清夷兮见弓矢之载櫜届西陵以爰憩兮灌木蔚乎蓬蒿丛祠鵶噪而日暝兮古戍鬻海以烹熬越山千嶂以相矗兮防港萦迂而一篙临钱清之汐浸兮拖废埭之轻艚览禹穴之夐古兮嗟防稽之栖劳悼壮图于种蠡兮镌谄辞于斯髙眺蓬莱之清胜兮懐兰亭之儁髦史迁之防孰继蕺山之扇谁操少陵壮节之英游兮太白俊逸之诗豪泛鉴湖之佳致兮羡外监之嬉遨读幼妇之刻辞兮遗貌摽夫孝曹吊贞魂于千载兮奠桂醑之单醪夜宿上虞舜风所陶民淳尚存于濬井鸟驯犹象于耘三江既泛其汹涌万壑亦饫于崂嶆姚江萦带而并海兮四眀勃郁以周遭山峭防而益竒溪洄洑而滔滔云献状而絶态松逞姿而猬毛猿清吟而送客竹蓊荟而解苞雨淙淙而鸣瀑风飕飕而势饕予既防搜而遐瞩舒啸以抽毫
　　苦益菜赋【并序】
　　李子闲居自暇从老圃灌畦毓蔬日忘其劳客有以山中苦益菜为饷葅芼调聏嘉其小苦防甘咀嚼隽永似乎忠臣医国之言也赋以自广瞻清晨之始旦独植杖而灌园遡望舒之落月逆扶桑之朝暾零露漙兮薤本败叶委兮葵根纷冐防而甲拆擢翠羽以庶蕃顾不熟而为馑岂饭芼于盘飱时乃享庾郎之三九擅诸葛之五利想千里之莼羮撷五溪之春荠诳杞菊于天随嗜芥芹于羊鼻人苋发园官之讥马兰寓玉川之意客有防葰茂于山椒濯沧浪之水裔羞我鼎铏芼之姜桂其苦口若批【音必结反】鳞之切諌其愈疾若上池之良剂苌宏染指而化碧贾谊濡唇而流涕故国医鍼在肓之疾忠言趣蹑足之计既谦受而为益信隽永之有味彼啜羮之少恩不如放麑之多仁紾臂而得食有类轑釡之无亲肆甘言之怂慂美佞舌之嗫嚅鼻莫辨于薰莸忧寜忘于萱蘓矧兹菜之耿介宜弃置于路隅然而嵗寒后凋者松栢所独主圣臣直者社稷之福鉴至眀者察妍蚩于顾盻水既止者视毫髪与眉目从谏有转圜之易去佞无防山之酷汉文除挟书之禁尧庭设诽谤之木嘻其甚哉此野人炙背之献异圣哲啓心之沃徒采薇于西山慕白驹于空谷固不足以当宣室前席之咨而备广厦细旃之渎
　　哀岩桂赋【并序】
　　版曹防前有岩桂焉根干凋瘁枝叶萎薾秋八月山中之桂烂然吐芬芳芗袭人兹桂也封殖失地坐视羣葩竞秀不能荣擢其华争妍一时客有白事长贰退而问于李子仆对曰木犹人也士之赋材禀教懐文抱质不犹此木之受气毓形乎种学绩文澡身浴徳不犹此木之芬芳郁烈乎挺姿嵗寒老死岩穴不犹此木之托植山谷乎弹冠结绶愿登王畿不犹此木之徙幽崖而入帝乡乎逮其争春于蒲桞之侧承泽于文昌之台见辱皂取诮舆儓日滋月徃遂为枯荄不亦可哀乎客又何怪客跽而请子其赋之仆曰唯唯伊金风之浩荡兮将摧残于众芳何岩桂之挺特兮独禀粹于颢商着家声于荒逺兮擅蟠根于后皇亶毓灵于蟾窟兮廼移植于帝鄊擢自拱把斸彼岩隈小山丛生而寓兴墨客一枝而寄懐竟怅望于空谷思委质于瑶阶时也少昊囘御万寳向成蕴通犀之内徳蔼文客之擅名露漙漙兮僊掌借润风嫋嫋兮天籁发清或秀拔夫差之茂苑或香动东瓯之粤城鞠衣沮艳薝卜惭馨惧过时而薄采羞自以含情则有骚人逋客头陀炼士或栖山庱之寂寥或饮沆瀣而度世既迢遥于间旷亦冺絶于势利乃吸坠露以嗅其菁英展裓衣而收其荂蘂或佩服以配秋兰之纫或薫沐以陋蔷薇之水非其标致不凡安得赏玩若此至于结根灵囿托体连昌晞鳯盖之朝日被龙顔之末光白榆厯厯于天上江梅垂垂于玉堂依尧蓂而并砌嗤汉栢之为梁今则立文昌之禁省昡金炉之夕熏地近左戸时当小春无连珠聚星之吟客有鞭筭心计之重臣惟国用之所制匪词人之拟伦埋没于喧嚣倥偬之地低囘于霜霰风雪之辰纎柯困于敲榜蒸粟化于缁尘俗眼屡白捧心漫颦莫不对春雨而泣望秋日以伤神然其深根固蒂而负嵗寒之期改柯易叶而无摇落之变傥遇邹律之吹嘘郭驼之善幻濯锦成桤木之园博望致蒲萄之舘沐之以西湖之清波植之以孤山之絶岸尚能偃蹇媻姗于灵鹫之崖蓊荟萧森于广寒之殿毋入门而见嫉岂深谷之徒怨也于是子墨客卿起而为山中招隠之辞曰秋桂丛生托岩阿兮团栾葰茂禀太和兮囘谿深壑水波兮秋风披拂兰菊芳兮郁烈袅娜芬都房兮一粟粒中万斛芗兮山中之人玉云顔兮薜荔女萝防讥讪兮归欤归欤保嵗寒兮
　　双鹤赋【并序】
　　侍讲舎人胡邦衡清节忠言冠冕当代具疏诛奸眀若蓍厌服天下若汉汲大夫近古社稷臣炳炳千古名烈莫二方犯雷霆之怒囘薄岭海南极穷髪二十六年处之泊焉与造物者逰晚际上皇更化趣使来朝辙环万里入天官冠蓬省司言动掌纶制今天子优礼旧徳以祭酒侍经幄日进嘉谟隂功隠利膏泽天下多矣直庐静深庭有一鹤缁襟俛啄戞然长鸣命俦寥廓俄有胎禽裳羽衣将翺将翔轩然来仪麾之不去遂玩双鹤昔赵清献入蜀以一鹤一自随李文正公居洛亦名六客鹤其一焉惟公清节可嫓清献而谟猷剀切视文正无愧将晋秉国钧为社稷镇功成身退从赤松逰虽骖仙人之骐骥朝圃而夕瀛洲为不难也赋曰敞直庐之靓防兮有独舞之胎仙服深衣之古雅兮视倒景以留连神清臞而貌闲旷兮若俛啄于芝田时戞然以长鸣兮由命侣而翩翩嘉此九臯之逸翮兮擅凌虚而得仙斯真人之骐骥兮着法相于浮丘之篇公方爱而玩之兮诵蘂珠以延年舞琴心之三叠兮挹洪崖以拍肩愍兹世之垢顽兮遂谒帝于钧天冠切云而佩陆离兮袭蓠芷而纫蕙兰建十絶之桂旂兮锵玉銮而中周旋闯九阍之蠖濩兮厉貔虎之当闗羣犬狺狺以吠日兮骇嗾獒之防防彼众口之铄金兮竞儿柔而婢顔抚长剑而拂旬始兮斩防貐而落女媊挟日月以陵清都兮谗夫媢予之昌言排阊阖而蹑泰阶兮怅幽愤之莫蠲下大荒而隘宇宙兮絶瀛海以南辕乱湘波而愍英皇兮从虞巡而弗还投雄词以悼灵均兮痛汨罗之沉寃上九嶷以望苍梧兮闻韶音而忘餐渉穷髪而歴儋耳兮虽九夷而亦安躬践蛇以茹蛊兮岂能侵灵府之浩然慨嵗月之何永兮再终星纪之二躔视蝇营与狗苟兮皆万坟之压颠帝秉箓以赐环兮入承眀而侍细旃非向之骑麒麟而控仙骥兮支造物而逰广寒又安得兹羽鹤也感在隂而俪焉亶中孚之信及豚鱼兮羌孰得而究研叹凤鸟之不至兮圣若宣尼而有言也广大而作赋兮白乃愤发于尘编也感鸜鹆之入贡兮愈自伤于迍邅也因鹦鹉以娯宾兮衡亦罹夫巧谝也维兹双鹤之伟观兮托君子而称贤也乱曰洪范五福贵眉夀兮诗人作歌惟黄耉兮懿兹胎禽寥廓偶兮翩然来仪如宾旧兮千年毳翎天地久兮清献之清文正友兮公致太平垂不朽兮万古仰之如泰斗兮
　　殇幼赋【并序】
　　余稚子直均未字而夭余与妻盛哭之哀而不成声呜呼吾儿逾朞而能行聪悟而敏慧甫三嵗已能捉笔模字效兄姊诵书不忘生之嵗月时干全甲戊庚皆居寅合于日之亥不幸遇疾余有五林之役归自外疾革视之形神离矣呜呼痛哉昔羊祜五嵗探桑木中金环白乐天七月能识之无古皆谓再来嗟乎人禀一真之性幻防幻生汨于生死大苦海中自非上根顿悟通其夙命者孰能超出轮回非幻不防而脱离生死究竟真谛也哉余视其母悲伤弗忍忘作殇幼之词极于性命生死之说塞余悲焉嗟人生于万化兮汨真识以嬗形禀五纬之纯粹兮托二仪以毓灵视幻防于瞿昙兮齐夀夭于殇彭哀穉子之韶美兮甫弱嵗之三龄既颖异而翘秀兮暨学语而和平诧庭兰之茁茂兮齐玉雪之敷荣余薄祜而钟罚兮痛夭阏于孺婴诉九阍之窱兮闯北辰之列星彼穹穹与昊昊兮临下民而俯听杳不闻于哀吁兮气澌尽而冥冥割衷肠之九囘兮懮菀懑而填膺防衣篝之故迹兮时误称于遗名愍号咷于慈母兮伤怛于姊兄惝怳而疑其死兮杳不闻于履声惚然而谓之生兮讶泯絶而无徴沕神理之茫昧兮戢精爽之依凭风飘飘而振帏兮雨淋淋而霣楹草夤缘而依砌兮雀踯躅而悲鸣恍春晖之芳芷兮忽霜霰而先零视崦嵫之頽景兮怅营魂之遐征竁黄垆而閟质兮无彷佛于平生谇曰泣既尽而继血兮髪变白以垂领了浊昏而丧眀兮涕横集而委绠哀稚子之无年兮即厚夜而徂永悲童乌而草兮踵叔子之探环懐东门之旷达兮岂情钟而爱捐觉大梦以蘧然兮衔永疚于终天悟龟毛于真诠兮将返初于泰始又安得羽化而飞仙

　　春事忽已阑年华何驶急客子困穷年卧若蛇蚓蛰泥深没屐齿市桥灯熠熠残缸玉虫缀檐雨如泣归期傥有徴缓我百忧集余生本畸人三年楚囚絷经商逐毫末索米慁畿邑君门深九重无籍岂敢入涸辙望斗升然桂炊玉粒妻孥寄田庐诛茒冬未葺朅来几晦漏屋想沾湿但喜一犁足犉犊起耕耨贵及时归欤具籉笠
