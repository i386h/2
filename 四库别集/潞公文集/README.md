　　潞公集　　　　　　　　别集类二【宋】
　　提要
　　【臣】等谨按潞公集四十巻宋文彦博撰彦博事迹具宋史本传是集凡赋颂二巻诗六巻论一巻表启一巻序一巻碑记墓志一巻杂文一巻自十四巻以后则皆奏议劄子之文核其巻数与陈振孙书録解题同惟尚缺补选一巻考叶梦得序称兵兴以来世家大族多奔走迁徙于是公之集藏于家者散亡无余其少子维申【按维中乃文及甫之字】稍讨求迨辑犹得二百八十六篇以类编次为畧集二十巻是叶氏所序者也非原本陈氏所着録者又非叶氏所序本今所传者又较陈氏之本其一巻也彦博不以诗名而风格秀逸情文相生王士祯称其婉丽浓妩絶似西昆尝掇其佳句载之池北偶谈其文章不事雕饰而议论通逹卓然经济之言奏劄下多注年月亦可与正史相叅考叶梦得序称其未尝有意于为文而因事辄见操笔立成简质重厚经纬错出譬之贲鼔镛钟音莭疏缓杂然并奏于堂上不害与嘒嘒箫韶舞百兽而谐八风也斯言允矣乾隆四十一年五月恭校上

　　圣驾幸大学赋【并序】
　　国家以寰宇昭泰仍歳登平务恢儒风以章示黎献皇帝乃偹法驾幸于太学诏诸儒博士讲论前典亲临听焉臣获逢休吉之期恭闻伟盛之事舞蹈不足形于赋咏诚不能述宣上徳褒讃形容姑第樵夫之谈以恊击辕之韵尔词曰
　　炎宋受命之四叶皇上御极之三年九有咸若六合晏然黎庶济于夀域文教烛乎冰天朝无阙政野无遗贤九叙可歌而不紊百官承式以惟防刑罚几措宿防不愆荒憬清夷而偃革狙犷慓詟而慕羶轶漠逾沙趣藁街者有万受缨请吏伏魏观者且千碧砮文钺之琛充仭乎储邸黒章肉角之兽驯扰乎郊阡嘉祥麕集异瑞蝉聨天降甘露地出醴泉语鸿烈则超图而溢牒较盛时则絶后而光前宜乎优防当宁拱黙承干尚乃愓严而驭朽思嘉谟而渉渊以为治国之道学校为先故周氏东胶往诰之所显商人右学来叶之以传上既行则民胥効也君所令则臣必从焉虽平昔之已务在于今之益宣于是命有司涓良日祲仪盛蒇法驾乃出天威穆穆国容皇皇采章焕烂和鸾铿锵太史恊乐以前太丙弭节而徐翔严羽衞歴康庄在浚之都于国之阳神移斗运乃至于上庠宸心防巩天歩髙骧歴阶逾阃覩奥窥堂晞将圣兮有穆如悉数兮相当然后趣讲室明典章缨緌匝序巾巻充廊钜儒硕生奉帙而在列礼官博士掌婣而诏王展东面之殊礼法西周之旧章禹听弥审尧聪益详夫子善言由兹而不昩虞舜好问于是乎有光【时宣诸直讲臣讲鲁论】于时闾闬相欢民灵胥喜扶老携弱自遐及迩莫不连踵而憬集驾肩而戾止悉云委于桥门尽覩观于璧水粤有华颠胡老童牙胄子含经味道之流方领髙冠之士咸相与而言曰赫胥逺而大庭尚矣无得称焉不可详已自夫五帝而降三王之始书契可以传闻宪章可以追拟悉皆恢雍泮之基盛胶庠之阯四术四教因是而兴三行三徳所由而起是故淳化丕隆至徳逾羙为万世之所宗弥亿载而罕比卯金之后盖不足纪当涂窘蹙典午沦弛南取岛夷之讥北贻强敌之耻干戈于是日寻俎豆以之中圮咸不永于卜世但胥循于覆轨逮乎有唐皇维诞张五室之仪兮絶而复嗣四郊之制兮抑而复扬置序于术建塾于乡家知礼让民用和康所以卜年久而享国长号治古而振懿纲朱石之际声闰位覆亡则曽不暇给学校则诚非拟议噫大道不可以终否斯文不可以久坠于是天命方有特眷之隆世运遂有日新之意维我太祖扫除僣伪俾万方无涂炭之劳百姓有息肩之地三后继明百祥遝至展云岱之鸿仪绍元封之故事盛徳大业固无与二今我皇上克奉先烈夤绍庆基体元则大累洽重熙将使儒风寖盛文教日滋故乘舆亲视于学俾亿丑预觇其仪足鄙元鼎之间屡有甘泉之幸堪讥延熹之际惟尚濯龙之祠夫然则三代之风必能缓歩而越矣两汉之盛岂可并日而论之伟乎轨迹夷易文物葳蕤信千载而一时
　　金苔赋【并序】
　　王嘉拾遗记曰晋惠帝初有祖梁国献金苔其色如金聚之如卵投水中则蔓延于波上光生照日上于宫中凿池以置之常观焉外人莫得见也惟进御嫔防多被其赐与置于盘中则照耀满室故宫中亦呼为夜明苔仆颇异之因为斯赋追其羙词曰
　　伊两仪之髙厚育万彚兮纷纭嘉金苔之有异穷古史而未闻出于祖梁之国献诸典午之君其为状也色掩捐山光分化鹊非沃壤之可植向华池而可托精气所为众口难铄色焜朝日宁同沈郎之钱根覆轻漪岂羙陈王之阁如赐郭釡难藏陆槖萦流荇而细细缭舒荷而漠漠烁堂上之江蓠沮洲中之杜若风团而或谓能铸浪飐而多虞自跃洊雷临睨疑抵燕池之鼃出震误观常歌汉液之鹤所谓丽水非云满堂东篱之菊兮瞻我而失色北堂之萱兮对我而不芳葳防频盛蔓延【上声】弥长缉作水衣海上濯吉光之服梳为石髪舟中见黄头之郎伊诡异之余状非名言之罄量宜乎首出庶彚超兰掩蕙持药友以荃交使苹舆而藻虽薙氏之务去兮不我芟夷纵骚人之善咏兮莫吾拟议噫昔产遐陬常依异类既作贡于大国遂见珍于中地唯皇居之必蓄依圆海而是置自逺成羙以少为贵非世俗之或覩岂凡品之能譬其或长乐清闲承光秘邃翠华黄屋宵逰夕憇春霄宫之鳯脑曷骋辉光咸阳库之螭鳞皆从摈弃置吾于隆栋之下则午夜而昕升我于文石之前则重昏而燧伟乎哉苔之为状也亦以异苔之为用也亦以至然不能方屈轶効灵蓍指邪触佞兮清君之侧钩深索隐兮决人之疑此无所取又将焉为徒能随波澜而上下与蒲稗而因依辉煌禁籞赐锡壶闱悦目兮媪宦侈心兮嫔妃南风之荡兮岂率循于四徳正度之荒兮遂旷弛于万机是致永熙永嘉之荡析刘耀刘聪之倾逼玉辇播迁金行否塞蟠龙踞虎虽别王于偏方封豕长虵恣吞噬于中国万户蒿榛千门荆棘金苔亦防于敌人深可为之太息

　　省试诸侯春入贡赋【天下侯国春入方物】
　　圣启洪绪君临溥天侯国之辨方有要王春之入贡昭宣列爵正仪谨奉藩而立制建宣恊序致任土以居先稽芳载于礼经仰徽名于帝者诸侯述职以无旷太史奉时而可假以谓惟王建国我则叙五等于域中与物为春我则任九贡于天下徒观夫爵分显父位列元侯当是时缇管顺煦和之羙皇祗布发生之休震方之善气潜逹长乐之洪仪聿脩帝容执瑁以端拱臣节奉璋而告猷旅币羣方咸奉舜班之瑞充庭万品皆分禹别之州但见云布封疆绮分邦域故我当歳首以入用致坤珍而罔忒巽风和令导传胪之九賔迟日当阳丽执玉之万国岂不以辨九土者当贡首四序者上春葢将偹物宜于时育助邦光之日新纳江沱之锡磬浮泗水之濵齿革羽毛以偕至球琳琅玕而毕陈三品良金于以向履端而执贽五都竒货于以当献歳而効珍莫不名物森罗衣缨杂袭虽厚篚以斯偹在庶邦而允集此春色霈于皇泽当邹律之均温彼贡也错于地财异楚茅之不入彼来宗者夏之礼制献功者秋之典章曷若谨歳贡以偹物庆春祺之载阳谅脩时而贡职乃辨物以居方偹于蕃于宣之仪皇皇辑瑞当载生载育之候济济来王故圣人灼叙声明光昭文物懿公侯之隆盛克贡赋之繁蔚属后王布和之辰献国珍而罔咈
　　省试青圭礼东方赋【修举春祀崇尚圭荐】
　　青惟五色之首圭乃六噐之俦朝日之郊是荐迎春之礼聿脩结緑鸿辉既肃陈于震位出蓝羙质将奉仰于神休原夫谷旦前诹祲威具举太史先春而必告宗伯厄职而攸叙青圭之秘寳爰资苍帝之明灵可伫谓物生有象我则鋭其质以协宜谓歳起于东我则傃其方而得所无瑕可贵有羙惟珍配其色表盛徳之在木正其位彰与物以为春异雕楮之竒文琢工奚取侔剪桐之秀彩寅位斯陈讵止温其宁专瑟彼成形自表于瑰器致用必先于毖祀色斯举矣俯玉按以相宣礼以行之背金方而致羙岂不以标十徳者圭为贵列四序者春是崇将顺时而展礼故辨方而在东防蒨成姿禀粹自辉山之石防恭致荐逆厘从觧冻之风然则为器之用不同礼神之功可尚名参有邸之义色异因方之状缥华中秘生成之气候斯迎绀防外融温厚之方隅相向懿哉享兹生物之主异彼迎秋于西神感而登礼必荅时和而迓衡不迷用可重焉且异舜班之瑞文为贵也还殊禹锡之圭故得鸿覆降康太庭锡羡祈民福以昭格授人时而于变葢有以圭方防翠之美欵青郊而可荐
　　焚雉头裘赋【珍异之服焚去无取】
　　晋武帝以徳继惟睿功齐乃神焚雉裘而崇俭负鳯扆以临民化彼元元必被先王之服燔兹楚楚葢除希世之珍原夫圣泽遐敷皇风广被当百度之攸叙见万邦之从乂诸侯述职既悉贡于瑰竒扩俗賔王亦咸输于珍异伊彼程防当兹盛时庶为臣之羙矣必竭节以事之由是制此雉裘将充乎任土之贡献诸龙陛爰陈乎执帛之仪徒观其丽彩鲜明烂光彬郁弥缝皆自于翰制作遂成于珍服异王恭之鹤但取清竒殊李兊之貂裘惟彰温燠帝乃念兹至巧命以俱焚虑滛靡之下漓薄俗恐奢华之上惑明君俄委燎原之势遂同有齿之文红熖初腾漠漠而渐成余烬青烟欲断依依而尚霭微芬然后珍恠罔求奸邪悉去六合咸归于俭徳万化永安于鸿绪虽斯裘之甚羙焉能衣之以奉天且厥用之至繁岂可被之而当宁若然则圣政敷于九有帝徳合于三无阐易俗移风之道遵还淳返朴之途虽文帝之罢露台尤难并矣纵武皇之焚甲帐未可同乎则知徳不广无以化蚩蚩之氓俭不崇无以成荡荡之主故焚裘之可羙在去奢而有取既着羙于一时遂流芳于千古者也
　　汾隂出寳鼎赋【皇汉之道神鼎斯出】
　　至徳昭彰炎灵道昌当汾隂而展礼见寳鼎以呈祥有感则通表承干之穆穆为时而出彰负扆之皇皇昔孝武以运继东周位崇西汉居尊克务于兢业临下弥勤于宵旰崇诸厚徳俾逺迩之悦随祀彼方丘冀神人之幽賛礼斯盛矣神惟享之允降穣穣之福靡愆抑抑之仪由是睢丘之畔汾水之湄仰窥乎天见黄云之缭绕俯察于地得寳鼎以瑰竒莫不焕金景之荣光蔚龙文之丽非惟启于至圣抑亦见乎有道宁虞覆餗实天地之殊祥不假铭功乃邦家之盛寳伟夫万邦悉庆百辟咸欣非铸鼎以象物葢至诚之感神瑞启明时岂类宗周之寳天祚皇徳实为巨汉之珍是何泽及坤灵祥昭金鼎固莫量于轻重又难偕乎竒梃载于瑞典非迁洛以堪同获彼灵祠岂于汤之足并不然则又安得并芝房而荐庙配白雉以陈诗于以显皇化之广运于以昭圣政之无私岳脩贡而川効灵徒虚语尔鼎取新而革去故何莫由斯羙夫祀事聿脩休祥有秩非劳九牧之贡自显三材之质所以标瑞谍而纪祥经弥万代而首出
