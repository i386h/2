　　词
　　臣等谨按石屏集六巻宋戴复古撰复古字式之天台人尝登陆防之门以诗鸣江湖间所居有石屏山因以为号遂以名其集巻端载其父敏东皋子诗十首盖复古幼孤勉承家学因搜访其先人遗稿以冠已集亦不忘本之意也复古诗笔俊爽极为当代所推许姚镛称其天然不费斧凿处大似髙三十五辈晚唐诸子当让一面方回亦称其清健轻快自成一家虽皆不免稍过其实而其研刻处要自能独辟町畦瞿佑归田诗话称复古尝见夕照映山得句云夕阳山外山自以为奇欲以尘世梦中梦对之而不惬意后行村中春雨方霁行潦纵横得春水渡傍渡句以对上下始称是其苦心烹链即此可见其概至集中严子陵钓台诗所云平生误识刘文叔惹起虚名满世间者赵与贙娱书堂诗话极赏其新意可喜而罗大经鹤林玉露又深以其议论为不然盖是诗意取翻新转致失之轻矫在集中原非上乘与贙所云固未足为定评矣乾隆四十二年三月恭校上

　　东皋子诗
　　小园
　　小园无事日徘徊频报家人送酒来惜树不磨修月斧爱花须筑避风台引些渠水添池满移个柴门傍竹开多谢有情双白鹭暂时飞去又飞回
　　屏上晚眺
　　不能骑马趁朝班自跨黄牛扣竹闗无徳可称徒富贵有钱难买是清闲人行踯躅红边路日落秭归啼处山遥望蓬莱在何许扶桑东畔白云间
　　约黄董二亲与桂堂诸侄避暑
　　世间无处避炎蒸欲叫西风叫不譍恨乏白檀除热恼心思赤脚踏层冰醉游河朔谁同往表借明光愧不能闻有山林最深处清凉境界着髙僧
　　楼上
　　终朝役役晚来闲识破浮生一梦间挈榼去沽深巷酒倚楼贪看夕阳山月临江馆人横笛风折芦花雁度闗堪羡渔翁无检束扁舟占断白云湾
　　西溪陈居士家
　　来访西岩老家居水竹村紫鳞游镜曲黄犊卧云根自昔好宾客相传到子孙不行亭下路防笋别开门
　　后浦园庐
　　卜筑成佳致幽栖乐圣时何如谢公墅畧似习家池地暖梅开早天寒酒熟迟催租人去后续得夜来诗
　　郑公家
　　门墙多古意耕钓作生涯菽米散鱼子莲根防虎牙弄孙时掷果留客旋煎茶颇动诗人兴满园荞麦花
　　海上
　　万顷鲸波朝日赤沧洲四望无穷极海山何处是蓬莱遍问渔翁都不识
　　观梅
　　三杯暖寒酒一榻竹亭前为爱梅花月终宵不肯眠
　　赵十朋夫人挽章
　　缝掖先生游汗漫夫人髙节独青青临行抖擞空书笥分付诸郎各一经

　　石屏诗集巻一　　　　　宋　戴复古　撰古体
　　求先人墨迹呈表兄黄季文
　　我翁本诗仙游戏沧海上引手掣鲸鲵失脚堕尘网身穷道则腴年髙气弥壮平生无长物饮尽千斛酿传家古锦囊自作金玉想篇章乆零落人间眇余响搜求二十年痛泪湿黄壤君家图书府墨色照青嶂我翁有遗迹数纸古田様髣髴钟王体吟句更豪放把玩竹林间寒风凛凄怆昻昻野鹤姿愧无中散状儿孤襁褓中家风随扫荡于兹见笔法可想翁无恙幽居寂寞乡风月共来徃众丑成独妍羣瘖怪孤唱一生既蹉跎人琴遂俱丧托君名不朽斯文岂天相旧作忽新传识者动慨赏嗟予忝厥嗣朝夕愧俯仰敢坠显扬思幽光发草莽假此见诸公丐铭松柏圹君其啓惠心慰彼九泉望
　　梦中亦役役
　　半夜羣动息五更百梦残天鸡啼一声万枕不遑安一日一百刻能得几刻闲当其闲睡时作梦更多端穷者梦富贵达者梦神仙梦中亦役役人生良鲜欢
　　饮马长城窟
　　朔风凛高秋黑雾翳白日汉兵方开邉饮马长城窟古来长城窟中有战士骨骨乆化为泉马来吃不得闻说华山阳水甘春草长
　　白苎歌
　　雪为纬玉为经一织三涤手织成一片冰清如夷齐【一作齐夷】可以为衣陟彼西山于以采薇
　　黄玉林云赵懒庵为戴石屏选诗百余篇南塘称其识精到其间白苎歌最古雅语简意深今世难得所谓一不为少
　　罗敷词
　　妾本秦氏女今春嫁王郎夫家重蚕事出采陌上桑低枝采易残髙枝手难扳踏踏竹梯登树杪心思蚕多苦叶少举头桑枝挂鬓唇转身桑枝勾破裙辛苦事蚕桑实为良家人使君奚所为见妾驻车轮使君口有言罗敷耳无闻蚕饥蚕饥采叶急归
　　题陈毅甫家壁【了翁之后也】
　　朱门金叵罗九醖葡萄春贵不酌贱富不酌贫君家破茅屋飘摇河水濵中有一樽渌醉尽天下人
　　答妇词【旧尝和顾彦先妇答夫二首故复赋此篇】
　　江山阻且长矫首乡关隔空闺泣幼妇顦顇失顔色隠闵鹳鸣篇寄彼西飞翼剥封览情素既喜复凄恻别时梅始花伤今食梅实览古帝王州结交游侠窟千金沽美酒一饮连十日春风吹酒醒始知身是客杜宇啼一声行人泪横臆衣破谁与纫发垢谁与栉勿谓游子心而不念家室新交握臂行肝胆犹楚越丑妇隔江山千里情弗絶慇懃挥报章归计何时决今夕知何夕覩此纎纎月此月再圆时门前归【一作车】辙
　　都中书懐呈滕仁伯秘监
　　北风朝暮寒园林日萧条自非松柏姿何叶不飘摇儒衣歴多难陋巷困箪瓢无地可躬耕无才仕王朝一饥驱我来骑驴吟灞桥通名丞相府数月不见招欲登五侯门非皓齿细腰索米长安街满口读诗骚时人试静聴霜枝转寒蜩倘可悦人耳安望如箫韶
　　桐庐舟中
　　吴山青未了桐江緑相迎扁舟问何之徃访严子陵高风凛千古卧蹴万乘主富贵直浮云羊裘钓烟雨湖北上吴胜之运使有感而言非诗也
　　苍生积衅乆天欲尽杀之干戈杀不尽继以大旱饥田野委饿莩道路纷流离众人识天意不敢加扶持公懐佛子性逆天救民命擅移太仓粟众拂生尘甑全活十万家九州咏仁政逆天天弗怒鬼神胡不恕白玉尚指瑕青蝇工防素秋风动莼鲈公亦思归去问公归去兮【一作求】苍生谁怙恃【一作恃怙】
　　所馆小楼见山可喜
　　兹楼非我有乆住如主人虽无往来客青山当佳賔君看楼下路三尺软红尘失脚踏此尘汨没多终身
　　客行河水东
　　客行河水东客行河水西客行河水南客行河水北行行无已时朱顔变老色至人楼一方庭戸罗八极保心如止水不受万物役有道肥其躯故能适所息
　　和山谷上东坡古风二首见一朝士今取一篇
　　自鬻非竒货强鸣非好声法当老山林松根斵茯苓朅来长安道霜鬓迫衰龄穷吟无知音祗觉太瘦生公诗妙一世风雅见根蔕比兴千万篇【一作端】已作不朽计穷达虽不同嗜好乃相似【断一作斵】
　　题古源棠和尚送青轩
　　南山色射北山光照戸道人处其中冰玉为肺腑兹山自开辟荒繄几年所一朝得主人荆榛化庭宇手披风月藏目极烟霞趣居然获竒观人境贺相遇道人诗更髙不作蔬笋语朝对北山吟暮对南山赋闻说玉堂仙击节赏佳句
　　大热五首
　　天地一大窰阳炭烹六月万物此陶镕人何怨炎热君看百谷秋亦自暑中结田水沸如汤背汗湿如泼农夫方夏耘安坐吾敢食
　　左手遮赤日右手招清风挥汗不能已扇笠竞要功南山龙吐云腾腾满虚空一雨变清凉万物随疏通向人无德色大哉造化工
　　大渴遇甘井汲多井欲竭入喉化为汗不救胸中热吾闻三危露迥与众水别其色莹琉璃其冷胜冰雪安得一杯来为我解此渴
　　吾家老茅屋破漏尚可住门前五巨樟枝叶龙蛇舞半空隔天日六月不知暑西照坐东偏南薰开北戸胡为舍是居受此炮炙苦
　　天嗔吾面白晒作铁色深天能黑我面岂能黑我心我心有冰雪不受暑气侵推去北枕思鼔南风琴千古叫虞舜遗我以【一作有】好音
　　乆寓泉南待一故人消息桂隠诸葛如晦谓客舍不可住借一园亭安下即事凡有十首
　　寄迹小园中自笑客异乡东家送防榔西家送防榔咀嚼唇齿赤亦能醉我肠南人敬爱客以此当茶汤慇懃谢其来此意不可忘
　　寄迹小园中岂不胜旅舍俗事无交加客身自闲暇隣家有酒沽杯盘亦可借吟侣适相过新诗堪脍炙足以慰我懐留连日至夜
　　寄迹小园中数椽亦潇洒主人既相知此地可乆假县官送月粮邻翁供菜把咫尺是屠门亦有卖鲜者里巷通徃来欲结鸡豚社
　　寄迹小园中余春接初夏问木木成隂问花花已谢黄鹂出幽谷杜鹃叫长夜把酒酌园婆逺客此税驾有时吟声髙鬼神莫惊怕
　　寄迹小园中一心安淡薄每坐竹间亭不知近城郭昨日看花开今日见花落静中观物化妙处在一觉委身以顺命无忧亦无乐
　　寄迹小园中第一薪水便逐一炊黄粱兼得鱼鰕贱饱饭日无营遮眼有书巻时逢好客来应接不知倦最苦风雨时有人招夜宴【逐一疑作逐日】
　　寄迹小园中新晴风日丽好鸟竹间鸣野鹤空中唳悠然动诗兴行吟抚松桂乆客若忘归此身笑匏系五月倘未行尚及食丹荔
　　寄迹小园中颇欲闷形影谁为饶舌者太守忽相请开心论时务细语及诗境坐中有蛮客狂言事驰骋明日酒醒来熟思令人瘿【□一作关】
　　寄迹小园中忽有乌衣至手中执圆封州府特遣馈罗列满吾前礼数颇周致四邻来聚观若有流涎意呼童急开樽四邻同一醉
　　寄迹小园中倒指五十日既得故人书南游吾事毕再拜谢主翁奉还此一室云萍聚复散欲住住不得折栁当马鞭明朝有行色
　　题郑宁夫玉轩诗巻
　　良玉假雕琢好诗费吟哦诗句果如玉沈谢不足多玉声贵清越玉色爱纯粹作诗亦如之要在工夫至辨玉先辨石论诗先论格诗家体固多文章有正脉细观玉轩吟一生良苦心雕琢复雕琢片玉万黄金
　　浒以秋兰一盆为供
　　吾儿来侍侧供我一秋兰萧然出尘姿能禁风露寒移根自岩壑归我几案间养之以水石副之以小山俨如对益友朝夕共盘桓清香可呼吸薰我老肺肝不过十数根当作九畹看
　　和髙常簿暮春
　　世变日以薄无从见雍熈闭门读古书聊以道自怡桃李春盎盎风雨秋凄凄于春何足喜于秋何用悲人生一世间所忌立志卑终身有不遇千载皆明时我生无所解肥遯沧海沂一朝遇名胜朽腐生光辉敛衽賛明德非公谁与归
　　和郑润甫提举见寄
　　出门欲求仁取友必胜已寥寥云海乡所幸有君尔胸蟠三万巻智先三十里相与定诗盟谁能执牛耳长身如病鹤苦吟如蟋蟀顾此顦顇姿痴生年八秩举世皆好竽老夫方鼓瑟梅花莫笑人茅檐炙朝日送吴伯成归建昌二首【此是包宏斋倅台时作癸夘夏】
　　老夫脚病疮闭门作僧夏麦面不疗饥冬衣犹未卸喜读吴融诗穷愁退三舍无因暗投璧有味倒飡蔗防搜琢肺肝苦吟忘昼夜工夫到深处非王亦非霸
　　吾友严华谷实为君里人多年入诗社锦囊贮清新昨者袁蒙斋招为入幙賔千里有遇合隔墙不见亲君归访其家说我老病身别有千万意付之六六鳞
　　谢东倅包宏父三首癸夘夏
