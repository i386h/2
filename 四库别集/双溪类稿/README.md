　　双溪类槀　　　　　　　　别集类三【宋】
　　提要
　　【臣】等谨案双溪类槀二十七巻宋王炎撰炎字晦叔婺源人乾道五年进士官至军器少监与淳熙中观文殿大学士王炎名姓偶同非一人也所着有读易笔记尚书小传礼记论语孝经老子解春秋衍义象数稽疑禹贡辨考工记乡饮酒仪诸经考疑编年通纪纪年提要天对解韩栁辨证伤寒论总题曰双溪类槀今已无传惟诗文集仅存仍以双溪类槀为名世所行者凡二本一本为康熙中其族孙琪所刋凡十二巻一即此本乃明万厯丙申尚宝司丞王麟得沈一贯家旧本为校正开雕者也凡赋乐府一巻诗词九巻文十七巻炎初与朱子相契朱子集中和炎寄弟诗有只今心事同千里静对箪瓢独喟然之句炎亦多与朱子往还之作交谊颇笃其诗文博雅精深亦具有根柢程敏志辑新安文献志所采最多而此外议论纯粹引据典确者尚不可悉数盖学有本原则词无鄙诞较以语録为诗文者固有蹈空征实之别矣乾隆四十六年七月恭校上

　　双溪类槀序
　　余为丞尚玺少保四明沈公雅好双溪集惜其秘于金匮石室中世人徒耳而嗜也图欲表之余因请得拜受退而鸠工梓焉竟则弁其首曰王氏自始兴文献公导以来散居江左衣冠与六朝相始终其间岂乏作者惟文宪公俭集传不朽夫深山大泽有龙有蛇呵风云而陈琰琬三尺赫蹏之纸直纡其胸臆以夸诩藻绘世亦卒无以摩其垒而攻其瑕如右丞之句昌龄之诗也者可谓后先竞爽矣嗣是诸王欲求文宪公集而读之发已藏于名山大川寥寥不可得怀耿慨者徒赋夕阳野草尔有宋肇兴诸儒軰出三槐半山后先柄国则始兴哉固不少逊要之断霞零露销歇有时如我双溪公者辞不粉饰而道劲摛章政不峻急而恳恻流惠性不脂韦而严毅立朝论不诡随而和平定是其起家也登乾道进士而敡厯中外也始为临湘令精于吏治下有所滥见以为棘心故厘假手之士而风清罢无职之禄而幸塞上有所索见以为剥肌故杜前任之旧靡羡余拒和籴之令靡侵蠧至策襄阳为国之门户而恃江陵为唇齿鄂岳为根抵也非封疆之臣石画耶苐非凡所见谁报可尔比入中都官博士以夀皇在殡而开讲防闲为非礼朝议凛凛于公矣庆元四年为实録检讨寻入着庭为佐郎由是有东都纪年之典焉呜嘑方大奸之戮也世皆以为无忧公谓霍氏之防汉宣尝收权矣其微附贵要者汉宣不能尽察也太平之诛明皇尝收权矣其交通近习者明皇不能尽禁也弊在元帝肃宗忧方大耳知言哉宋事如左劵矣所谓庙廊臣者非耶大扺公学传于南轩而自得之趣则公之所自许故遗文不藏名山大川而以金匮石室为山川其藏固矣讵能负之而趋抵今学士得捧而读焉信非文宪之寥寥者比乃知江左风流靳禳侯印为未易夺者其视公之印可真天壤哉公世居婺源武水之曲双溪合流因以为号邑有端风祠祀先哲也双溪名在祀典中楚楚精神驾文螭而肹蠁是为不朽之盛典亦为衣冠之世族也集今行于世赋乐府一卷诗词九卷文十七巻沈公有定论矣余姑论其道术徽猷以为后学仰止云敬序之以复沈公也时万厯丙申春三月谷旦赐进士第尚寳司司丞海阳见庵王麟谨序

　　水西风光赋【并序】
　　隆兴癸未冬龙图閤待制吕公来守新安览其山川而乐之后八年卜居于城西南隅榜其楼曰水西风光门下客王炎赋之附于众作之末其词曰
　　古歙为郡山水竒秀飞仙徃徃窟宅扵其中号居江左十州之右至扵水西之胜槩又称甲扵一州挟以披云之孤峯绕以漾沙之寒流昔翰林之诗伯骑鲸鱼而来逰至今墨客骚人无不剧谈其髙风而渔翁樵老亦能寻访其遗迹人与景而俱胜首尾三百余年而不可再得那知神崧夫子之卜居而气象反髙乎畴昔岂非人物为主山川为賔故风光虽满扵天下然大半落莫而无闻苟有恃而获传厯千载而不朽有白傅而知香滩因叔子而知岘首浣花寄声扵工部醉吟之篇辋川托迹扵右丞丹青之手况如我公三朝廊庙之夀俊六世轩裳之故家何紫乎文石鸣玉乎金华一旦丐闲引身而退傲睨扵烟尘之表袖手扵功名之外遂令水西之声称自于今而益大飞槛渠渠压城之隅予虽未获寓目扵几席之末缅想其临髙眺逺凡可以恱耳目而娱心志者一皆领略而无遗公旧为伯扵此邦留去思扵千里父老儿童见公而喜故虽山灵与溪神争欲回巧而献技揽波光空翠扵防蒙杳霭之间阅云烟沙鸟扵旦莫晦明之际飘然如坐蓬莱方丈而下视人间之世憇曲几以欠伸抚修栏而倚徙彼谪仙放浪而一来非可同日议也无已则犹有太公望之事乎处则钓渭水之璜出则佐六州之王方圣天子之仄席登耆英扵庙堂胸中之策十未用其五六恐未可留意扵山水之风光
　　竹赋【并序】
　　小人之情得意则颉颃自髙少不得意则摧折不能自守君子反是竹之操甚有似夫君子者感之作赋以自箴其词曰
　　晦叔读书南斋之上门巷僻左交防日稀环以桑麻之场带以瓜芋之区路折西南万竹苍然下縁乎曲涧之湄其清可以延风月其髙可以扰云霓乎鸣苍玉之佩摇摇乎舒翠鳯之旗森森乎甲胄之襍峙而切切乎矛防之参差其偃蹇挫折者如忠臣节士赴患难而不辞其婵娟萧爽者如慈孙孝子侍父祖而不违其挺拔雄劲者气毅色严又如侠客与勇夫其孤髙介特者格清貌古又如骚人与臞儒予虽朝夕吟啸扵其下曽无以名其美而状其竒然泛观宇宙之中万物均函育扵一气而有刚柔坚脆之不齐荣者必悴盛者必衰实繁者易剥色丽者早萎惟松柏之有心及竹箭之有筠足以阅寒暑而贯四时春日载阳山川含滋零露兮宵润恵风兮晓披或萼或条或茁或荑含英吐华夭夭怡怡竹扵是时清而不腴冬日祁寒天地积威雪惨兮冰坚风号兮日凄川原千里木脱草枯香尽芳歇扫迹无遗竹扵是时秀而不癯今而后见其含徳之有常特操之不移此吾所以无羡渭川之千畆有取淇奥之猗猗愿定交扵金石邀嵗寒以为期否泰兮消长剥复兮乗除秉吾心之坚一视此君乎庻防
　　雉赋【并序】
　　之野外见一雉为野人所得悼其轻敌不戒而毙焉因作赋以自警其词曰
　　绛冠兮緑跗翠衿兮锦衣耿介兮自守雄鸣兮雌随平原兮浅草徐啄兮卑飞何野人之机巧乃以媒而致之方气骄而贾勇敢直前而不疑事常败扵轻敌碎身首扵须臾肉不登扵鼎爼毛或用扵旌旗子玉无礼而殒命赵括狂勇而防躯酆舒怙俊而遭杀却至称伐而见尸苟谋身之有累皆灾害之所罹翠有羽而就罗鸡以尾而为牺鴈不鸣而先烹鹦巧言而长羇信渉世之孔艰虽后悔而何追惟羽族之至灵独丹穴之鹓雏噰噰兮音声锵锵兮羽仪摩千仭而髙翔岂矰缴之敢施兹吉人之能事扵鄙夫兮何诛
　　夏日郊行赋
　　四月维夏先晦三日子王子散策扵东臯之上尔汝老农问劳既毕遂曵履乎委巷之间反而息肩乎乔木之侧喟然叹曰以吾流目之所见物化之亟如此哉麦穟穟而饼饵桑隂隂而茧丝麻斾斾而没人秧芃芃而布畦蜩解甲而效鸣鸎迁乔而载飞雀绕屋而孳尾乌俯巢而哺雏是孰斡其机缄至扵日运而不齐也耶吾闻二气嘘吸一瞬不留向之温然者俄而为今日之夏今之燠然者又且为后日之秋所以使人慷慨増懐踊跃自激感嵗月之遄迈畏功名之湮没故志士之竞时殆不容乎寝食如予之椎钝迟鲁则姑苟安而休息者也人方逐逐我独徐徐人方役役我独于于谅愚智之悬隔宜动静之殊涂其在武水之曲有先人之敝庐一裘一葛日饭一盂非扫轨以谢物且杜门而读书付荣枯扵自然庶顺命而无违抑古人有言曰优哉防哉聊以卒嵗是其获我私者乎
　　喜雨赋【并序】
　　丙申夏四月武昌阖郡不雨越五月三日崇阳宰吴侯以诚祷雨遂优渥按春秋僖公三年春正月不雨夏四月不雨六月雨说者曰书不雨者闵雨也书雨者喜雨也喜雨者有志扵民也今吴侯祷雨以诚不崇朝而雨随之可谓有志扵民矣作喜雨赋其词曰
　　嵗行扵涒滩之辰斗建扵实沈之次环鄂渚之七县值亢阳之为沴风自南而且霾日将北而如燬云霓郁而不兴旱魃骄而孰御策余马扵郊牧喧龙骨之咿嚘滮池潴其既竭陂堰支其断流视衍沃而坼况髙田之未耰苗已悴而半槁惧西畴之不收几张頥而待哺举蹙頞而増忧粤桃溪之令君念民命之惟谷兹土旷而农惰虽儋石而无蓄繄告籴之少艰米翔踊而如玉傥既月而旱甚民必仆扵沟渎讲雩祀之故事虚心斋而竭诚谒梵王之莲宫羞苹蘩之洁清欵仙伯之灵岩释舆马而草行隂曀曀以随盖飔悠悠而拥旌柱础湿其有汗钟皷鞳其无声纷举手以加额曰雨兆其已形日蒇仪肃我庭宇驾龙骖螭攸屇攸止云将前驱雨师踵至越三日而为霖罄一同而均利稼芃芃而骤苏浍汨汨而如注免为殍扵凶年占筑场扵乐嵗沸舆颂而载涂歌令君之隂赐其歌曰霡霂兮涵濡汚邪兮满车其饟兮有黍稲稉兮可炊岂弟之泽民肥不臞我字我抚公留勿归均此大恵公归勿徐吴侯闻而哂之曰父老之言何美之溢也向也民忧而忧此吏责也今也民喜而喜吏不敢以为徳也然旱而祷祷而雨如重负之获释也酌我醑醴旅我殽核合僚友而一笑吾敢忘夫帝力
　　中隐赋【并序】
　　三衢留侯以中隐自处范石湖为书二大字以表章之侯有中隐对自叙甚详夫若隐若显内心泰然不为圭组所累侯之志髙矣顾进为扵世功大名白民受其赐侯未可辞此责也乃赋之曰
　　子留子道髙而位下丰蓄而啬施发挥所有可以雷霆一世而畏人之知沈酣自得扵理义之府盘薄不鹜扵功名之涂廼以中隐扁其室庐圑蒲曲几诗编酒壶凡世间尤物一切无有盖淡乎如逋客之舍幽子之居吾见其鼻间栩栩刍豢顔色殆将贵吾身扵隋侯之珠外物扵秋蝉之翼此所谓与道逶迤一龙一虵不露文彩如玉如石者耶夫孤竹二子采薇西山揭日月扵百世之上可望而不可攀槁项黄馘山泽下士徒恱其风未喻斯防友麋鹿扵闲旷之墟玩鱼鸟扵泬寥之涘惟陆沈而无用徒牢闗而固拒故荷蒉与耦耕均圣门所不许万夫一贤天地间气海涵岳负以育徳防刅迎解以为智阳潜于泉则草木无色云出于山则膏泽千里古之抱此道者屣脱富贵而无累蝉蜕尘埃而不缁然燕坐一室而经营四海未尝泊然与世而相遗投耒耜而执玉释版筑而乗车万物賔従受其宠绥萧条陋巷之布衣诘屈乘田之小吏非为我而不为盖欲为而不遂先生弹冠而出曵履而趋岂无意扵斯人而何以隐为漆园傲睨达近乎诡金马玩世和失之鄙凖绳以中又安取此有如乐天可谓贤矣不膏盲扵邱壑之下不柴栅扵绅脩之间招释子扵匡庐之阜呼酒徒扵香林之滩得丧两忘去就俱闲无乃欲眇今世而尚友与斯人而比肩虽然进退有正以义为主可否无固因时为度元和之际犹莫尽其所懐长庆之后可飘然而引去时既异而辙殊何必追其故歩曲阜之野麟出而非岐山之阳鳯鸣而宜可隐而见其中躁可见而隐其徳孤吾欲与白傅异议廼申以招隐之诗其诗曰白银双阙髙巍巍佩印如斗生光辉药石之妙国可医缩手未试众迟之前旒侧席九天上青云发轫遄其驱
