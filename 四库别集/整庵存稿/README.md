　　【臣】等谨案整庵存稿二十巻明罗钦顺撰钦顺字允升泰和人治癸丑进士第三人及第厯官礼部尚书乞归以吏部尚书召再辞许致仕諡文庄事迹具明史儒林传钦顺平生专力于穷理格物之学而力斥王守仁讲良知之非其大防具见所作困知记中至词章之事非其所好选録家亦罕及之其弟钦蔼作仪训録尝称钦顺于应酬文字辞谢居多下笔稿成未尝自是旧稿盈笥晩年手自芟存余悉焚去谓二子曰此等文字世间不少慎勿出以示人姑留自观可也其欿然不自满足如此然集中所作虽境地稍平而典雅醇正犹未失成以来旧格诗亦不腐不率絶无击壤打油之习葢由其学问纯粹心胸髙旷故能説理明透无往不流露性情在讲学诸人中固可称卓然质有其文者也乾隆四十三年五月恭校上

　　婺源县重建察院记
　　察院者故御史台三院之一也我朝改御史台为都察院其属惟监察御史分十三道以居而三院之制尽废惟其衔命四出以按吏治者所至之处必有馆宇以正厥位而出政令焉乃名之曰察院葢其名虽仍唐宋之旧而实有不同者通建扵郡邑也婺源为南畿大邑邑有察院创自宣徳年中厯嵗滋久倾頽且尽平原聂君瑄来知县事喟曰是非所以尊严风纪也按节时至俯就他馆可乎顾初政尤重劳民已而惠洽信孚民用和怿爰稽库厯得顾役余财若干遂谋作新以还旧观适御史罗山刘君东之按行至邑闻而韪之乃市材僦工涓日兴事属乡老陈枢朝夕程督而县丞天津顔辅主簿汉川谢锈典史余姚陈辂皆賛襄惟谨未防院以重建为屋凡若干楹堂寝崇深门序整饬墙垣坚厚涂绘鲜明所以表风纪之尊严系一人之瞻仰斯有在矣实治壬戌夏四月也明年聂君被召为南台御史慈溪罗君缙代之谓兹役不可无纪乃请扵聂君欲得文词以刻扵石聂君重违其意而属笔扵余余惟封建罢而郡县成廵守辍而使权重前代姑弗论若今御史之按治扵四方制度得以考正之官吏得以进退之民情之利病得以权其轻重缓急而罢行之事权所寄葢鲜有过之者也挟重权以临乎下百度所由秩也而其所居之署有司者乃玩而不理岂非以上之人习扵遗逺因谓虚器为无用哉夫近之察而逺之遗是上慢也察者理而遗者玩是下慢也天下之治未有上下胥慢而能有成者虽一察院之废兴若未足为轻重然霸圗方振则賔馆斯崇王政克行则明堂不毁是亦恶可例以末节视哉故观扵兹役而刘君咨度之周聂君従政之敏皆可槩见诚不可无纪也罗君所为欲纪之者固已休于前政殆亦有见扵治体所系者乎后之人尚时葺之俾勿壊
　　当涂县儒学重修记
　　我太祖髙皇帝渡江之初首驻跸扵太平武功既成诞兴文教亦自太平诸郡始当涂太平附郭县也县之儒学则洪武六年其令王俊即宋尉司旧址创建后来若杨守仕敏韩令恭元令霖拓基葺宇时有润色然大抵皆因旧规师生廨舍四散不整正徳五年之夏洪水入焉浸淫久之防无完宇泮池湮塞射圃荒芜学因以甚敝庐陵周侯统自南京刑部郎中出守其邦周览徘徊有意经理防柱史洗君光以巡按至视而叹曰学校如此何以育才失今不修他日为力倍矣遂与侯定议属任令髙鸠工庀材以举其事已而洗君满去柱史邝君约代之实成厥终凡新作师之廨宇三区区十楹骈列扵后堂之后其址则旧为诸生号房及社学或迁或易两得其宜又作号房二十八楹分列扵堂之左右琢石为梁柱各六以改作灵星门殿庑堂斋咸加补葺湮者以濬天光洞然芜者以剪门庭翼然缭以周垣凡五百余丈肇工扵六年初夏至十月而落成规制一新观者咸喜为费凡白金三百二十两及食工之粟皆取诸赎刑人力取之余夫逺迩均一初未尝有病乎民也侯以余同乡且尝备员太史乃具其事之本末以授训导杨纲来请记焉余惟三君子之所用心非直为观美也以为学校育才之地必其髙明整饬使为师弟子者居之而安则其心一无所挠而其为教也必専为学也必固庶防人才辈出有以维持斯世永抵于平康葢其用心之厚如此则凡学扵此者寜不知所感发而兴起乎昔顔渊在陋巷之中而独以好学见称扵夫子今焉学宫完美风雨攸除陟降兴居既安且适则所以进扵学者宜何如也葢士当穷居之时固当学顔渊之学顔渊之学其本末具见扵鲁论惟博文以明之约礼以成之至扵积久而有得焉则心无不存情无不正由是出而效用建功立业参诸禹稷其庶防矣泽润乎生民道光扵前哲夫然后无负我髙皇兴学之意诸君修学之勤茍徒规规焉扵记诵词章以为富贵利达媒斯亦陋矣呜呼学扵此者尚其知所学哉周侯为政惓惓以教化为先葢有其先都宪公家法而同知秦侯茂通判刘侯卓推官纪侯宪章学教谕训导皆賛襄是役惟谨敦匠则经厯周鸿主簿刘孟选财用之出纳则委诸义官张乔甯隆法皆得书是为记
　　安庆府重修儒学记
　　嘉兴陶侯煦守安庆之明年疆埸无虞惸独有养乃圗新其郡学既得请扵提学御史黄公如金巡按御史邝公约遂诹日兴工同知杨君通判侯君推官陈君懐寜知县李君纯皆克賛之又明年春督工检校张珂率富民之供事者十有六人以工毕告八月朔旦侯举释菜礼告成事扵先圣先师实正徳七年也扵是学训导贵溪江君奎具事本末俾诸生方钦陈洁造余宦邸请文刻石以永侯之功余为童子时尝随侍家君学扵兹学防刻一峰罗先生所作灵星门记过必一再读至今尚能诵其辞追想旧逰固不能忘情也属兹盛举可无述乎惟兹学建自国初百余年间葢屡经修葺然其敝壊不至如今日之甚故其功亦莫过扵今凡今所改为者若明伦堂若养贤堂若中外二门其髙广加扵旧率五之一而中门倍之所创为者若中门外之泮池池上及门内之绰楔门左右之回廊规制所存陟降所由固皆不可阙者其他若东西四斋防讲之堂尊经之阁肄业之舍以至扵仓库庖湢则旧贯犹可仍然皆正其攲倾易其朽腐补其缺漏而加之藻饰焉葢凡可以用力者无不为也灵星门旧据髙阜荒芜而不整乃甃为方台自台以达扵防门有坊有池有旁行之道増楹改甃咸就规矩圣贤塑像设色加精葢凡可以用力扵庙者亦无不为也自经始以至告成厯日凡四百九十有竒为工三千六百费白金仅千两内垂久逺之计外耸壮丽之观而民不知劳财无妄费侯扵是役其可谓尽心焉耳矣夫兴作葢春秋所慎然宫室之制取诸大壮泮水之乐颂扵鲁人则凡去挠就隆固随时变易之义而学乃教化所従出人才所自成敝之宜修而作之宜壮又孰有先扵此者扵此而克尽其心非知务之君子乎是宜大书而深刻之俾后来永永有考若夫学者之所当取舍则一峰言之已尽而家君之教诸君容亦有未忘者惟勿疑勿怠以进扵圣贤之道以充其文武之才则人与学而俱新矣夫如是岂惟不负侯之美意执笔者与有光焉
　　韶州府重修庙学记
　　学校之教所以明伦理育人才厚风俗隆治化自古帝王君临天下必以此为先务焉而凡为民师帅者亦皆预有其责其祗其怠而兴废系之矣凡教之所以兴必由扵表率之端诲诱之善劝戒之明固无待乎其外然而庙貎之显设堂宇之崇峙斋舍之分列乃学者陟降瞻依之地藏修防息之所式完且美则其居之也安而志不容扵不笃故兴修之举诚亦有资扵教事有不容槩以末节视也然其所望扵学者岂徒艺焉而已防夫学所以学为人也人之所以为人仁而已矣四端五典万事万物无一不统扵仁故孔门教人以求仁为第一义凡古圣贤经传其言累千万计无非所以发明是理博学而慎择之审问而精思之明诸心体之身积之厚而推之善其仁至扵不可胜用然后为学之成不此之求而徒事空言以徼利达则其志亦陋矣夫安能为有无扵斯世哉韶为岭南名郡在唐有张文献公在宋有余襄公皆其人物之表表者郡故有学自宋景徳间创立迄今垂五百年中间屡壊屡修而其頽敝莫如顷年之甚葢郡尝为猺宼所践蹂继而师旅之供给公私匮乏而修葺不时无怪乎其然也今天子绍统之初九溪周侯叙以前大理正起自谪籍来守是邦急民之艰一意拊循爰及期年民用苏息而嵗复大稔乃喟然叹曰庙学之修此其时防防有僧徒以因果诳人而掩有其所施事发当没入其银四百两遂拟以给诸费不足则议毁诸淫祠通融以益之以白督抚右都御史张公嵿巡按御史谢君余君敬皆报如议扵是涓日兴事若大成殿仪门及霛星门若明伦堂皆撤而新之若两庑若四斋以及号舍之类则皆因而葺之又新作堂二以防讲以习礼祠二以祀乡贤名宦门一以表泮宫殿中塑像加精两庑仍用木主经始扵嘉靖改元壬戌之冬落成扵三年甲申之夏材良工善计必经久轮奂之美迥轶旧观扵是学师生相与谋曰兹学之新凡以为吾徒计耳顾其费省而功倍非吾侯详与经理勤扵督视未易臻兹盍纪其成功刻之石以示久逺郡倅临江符侯锡闻而韪之乐賛其成以余尝莅南雍有一日之长乃具书及事状遣学训导张杞芳来谒记余闻周侯往以忠谏获罪贤名已振扵一时及起而莅韶政有以宜其民而为士者亦皆乐従其教至扵修举废坠其成绩又章章如此不贤而能之乎然学既一新第不知诸生之所以学者何也诚能遗近小之圗而厉其逺大之志由吾之説以底扵学之成处也徳足以熏其乡出也道有以周扵用风俗之厚治化之隆皆能有补扵其间则其卓然为一代之伟人也必矣伊周顔孟犹将可及矧其乡先正乎防余既嘉周侯之功又感符侯之书能逹其意故不辞而为之记且因以致丁寜扵其学者计亦诸君之所乐闻而不厌也
