　　提要

　　臣等谨案温恭毅集三十巻明温纯撰纯字希文三原人嘉靖乙丑进士官至左都御史赠少保谥恭毅事迹具明史本传纯在隆万之际正色立朝初忤张居正罢官再起又与中使争矿税卒以忤沈一贯致仕可谓毅然自立不负君国虽阨于羣小无一日安于其位而日乆论定究称名臣其奏疏皆切中情事字句或失之太质而明白晓畅易于观览葢期于指陈利弊初不以文字为工其它序记铭传诸体则多雅饬可诵诗凡八巻大抵沿溯七子之派而稍失之麤尺牍五巻亦多闗时政末一巻为理学六十一则皆论学语録大旨以程朱为本不宗姚江而亦不甚斥姚江葢纯一生惟以国是为已任所争者不在此也言以人重是集之谓欤

　　修实政图治安疏（隆庆戊辰年任戸科给事中）

　　臣窃惟天下事不患不可为而患不必为故任天下事不得曰不可为也而使其事必可为治且安者乃为善治我皇上嗣登宝极葢三年于今矣视天下为治且安耶视诸臣任皇上事者以为可为而必于为耶抑以为不可为而不必于为耶夫病者求三年之艾而不得必为之蓄今天下病矣而艾犹不蓄是可惑也已臣观今日之所最急而付之无可柰何者有三一人材缺而无计能振一财用匮而无计能理一武备弛而无计能整岂真无计哉有之而不为为之而不力也臣为此惧恐因循之风日甚一日将来之弊竟若之何是以忘其迂陋披素衷为皇上画计幸留神采纳焉臣闻天生一代人材自足一代之用夫何迩来一遇员缺动称乏材即推补多用试之可则已不可则别举而试之又复不可又复试竟未有得人如古昔盛时者此其故何也臣以为责成泛资格拘又储养之无素也葢唐虞三代典礼典乐司马司冦始终俱一人掌之而今之时则今日司民牧明日改兵衔今日计钱粮明日理刑名甚至以一人之身而各司庶务罔不厯焉傥以钱榖壊即长于刑名弗计矣傥以吏治壊即长于文学弗计矣汉县令可入为太傅今即有卓茂之贤谁其超迁之乎即超迁谁不骇疑而指议之乎是以循阶而进虽左右先后亦不敢越次且当其会推也一人曰可众唯唯而退袭公举之虚名诿是非于铨部又或明知其人可用而曰资格未及待其资而老将至矣国家之树人犹树木有摧折无长养材未有能成者今天下方面有司如故而举刺衙门日多有一人焉或目为跖或目为夷倘其时不相值则以跖目者谁是之而以夷目者谁非之乎即今各省之方面两京之有司不足以应其刺矣至于各州县佐贰首领尽天下无虑数千百亡不视为虐民者上既以虐民视彼安得不虐民乎禄不足以养廉资不可以逺到而以亷望之已难矣况亷未必即显不亷未必即罢人孰肯为亷哉臣愿皇上特勅吏部通行内外九卿抚按衙门或年终或考满将各属分别所长开送该部责其一不责其二凡钱榖甲兵刑名吏治文学各照所长终其身用之宜原任者不妨増秩乆任堪别用者不妨就长升调以后部寺藩臬等衙门不得视为传舍随级递转使人得各精一业以求他日实用每遇会推大臣合无预行在京会推衙门各举所知一二人揭送该部公同拟用拔其尤不计其资亦为官择人之要也至于举刺或専责抚按或分别职掌州县佐贰首领贤之尤者另疏荐举不贤之尤者不时问遣庶激劝之中未必无长养之意行之数年而人不思奋者臣不信也臣闻国无九年之蓄曰不足无六年之蓄曰急无三年之蓄曰国非其国今内而太仓无一年之蓄外而府州县无数月之蓄一遇水旱不惟不能赈且额办岁办日望其取盈焉假如不幸而年不再登敌人又侵恐追征之令难复如故而民之贫者逃逃者盗臣且不忧门庭之冦而忧萧墙之冦矣然天地间财物产于一岁者自足一岁之用产于一世者自足一世之用乃匮乏至此臣以为年例増武职滥屯田废也葢国初各边年例不过四五十万而兵非不盛食非不足也何今之为将领者惮清勾而乐召募为士卒者假逃亡而顶新军夫以召募之力为清勾何所不可以扣除逃亡者为召募何所不宜乃不此之图而日请月益漫无已时计今岁所出已不下三百万矣是国初六年之用仅足今一年之费万有警急増赋则病民不増则病国司国计者计将安出国初武职非军功不授应袭严比试之法何今之武职不尽出军功比试之法又习为故套以故乳臭之子皆可领印绶豢养之流不堪任坚鋭以朝廷有用之财养市井无赖之徒至所望以摧锋陷阵者又不在是焉是所养非其所用所用非其所养有识者付之长叹而已国初各边充实堡塞星罗粒米狼籍屯政盐法相须并举不烦内帑而饱歌腾槽之休犹可想见今军民日困于掳掠土地日见其荒芜米珠薪桂之谣所在而是说者归咎于盐法改折是矣而不知屯田既废欲盐法不改折不可得也近差都御史龎尚鹏清理屯盐诚计之得者然以九边而付之一人令其清理之可也欲望且耕且守如赵充国成金城之功鹏一人能之乎故得一王之诰则辽束之屯田兴矣得一张守中则宻云之屯田兴矣是在督抚兵备加之意而已臣愿皇上特勅该部将各边岁出之数总计若干各省直岁入之数总计若干如其相当宜图积余之策如所出多于所入尤宜早为奏请会官议处以求万世之利其武职应袭必严比试之法凡不由军功授者尽行罢革是除冗官即所以除冗食也至于都御史龎尚鹏应否乆任督抚兵备应否加以兼理屯田职衔任事兵备如张守中者应否改调要边镇以为诸道之倡务使督抚指挥于上兵备实行于下如有仍前因循玩视者许总理都御史与屯田御史指名参究行之数年而财不足用者臣不信也臣闻天子有道守在四夷今各边惟蓟辽二镇颇在得人其余宣大山陜所恃无策闽广恵潮急利旦夕倘一隅失守则肢体不完岂得腹心无恙且堂堂天朝顾不能振武威以扫彼氛孽哉臣特患赏罚不严任用不専又边腹异视焉耳葢朝廷之赏罚行而后将帅服将帅之赏罚行而后士卒服故穰苴得以诛庄贾亚夫得以抗汉文今福建廵抚涂泽民冐叙功勲欺诳朝廷广东副使陈纪推故避难乆误推补原任总兵汤克宽玩冦殃民乆不正刑是用一人不为我用罪一人不为我罪朝廷之赏罚何在耶至于总兵将领则屈膝重臣与士卒同其卑分守信地与参游同其权平居无一金之赏则无恩可懐临时无必诛之刑则无威可畏世特无穰苴亚夫耳有之而亦若是臣知其必无功也古之善用兵者莫如诸葛亮终其身不迁尚无全功今督抚之臣一蹶即罢既罢又起恐今日所起者多前日所罢今日所罢者未必非后日所起矣是以为督抚者一心以筹边又一心以图归宁以一蹶而去不欲乆居而待罪如宣大总督陈其学素系时望亦通书当路欲求归计他尚何望哉穷边之地莫非疆域穷边之民莫非赤子而今则腹里之不职者调边方矣边方之不职者调极边矣即铨补边方有司亦往往以举贡之下者充焉疮痍遗民岂能抚摩累卵城郭岂能保守方今恵潮之民朝应役于公堂夕执杆于贼党吾有赤子不令其爱我卫我而令其与我敌焉有司者安可辞其责哉臣愿皇上特勅该部将汤克宽亟正其刑陈纪禠夺其职涂泽民既经论劾必难措布别选才望素着如侍郎吴百朋者代之令其刻期赴任仍申明各边将领之权总兵得以生杀偏禆偏禆得以生杀士卒凡赏劳钱粮苟得其人即令便宜行事可也各边督抚要在慎择于初万一有蹶果系可用之才而罪非极刑宁降级罚俸以责后效不轻易改迁以滋劳扰总督陈其学清谨雅望臣所夙钦挥霍雄才原非所长亟当议处以重鎻钥母贻他日临敌易将之虞边方有司改调不职之例即应停止博选才望布置郡邑候有保守之功照依军功升赏以酬其劳如是而武备不修者臣不信也夫此三者皆今日之急务臣亦据所知以陈其梗槩耳然尤有大可虑者焉昔先臣王守仁有曰忠义降而为气节气节降而为客气臣又曰浑厚降而为调停调停降而为因循客气与因循合而天下始有不可为之事不可与之人矣今之以客气待人以因循自处者何其多也故论人才则以爱憎为毁誉议政事则执已见而拂众论或阳为唯诺而阴肆诽谤或身在一堂而心如千里又其甚者则以君子攻小人以小人攻君子又以君子而攻君子至于自处则任事者少避事者多持两可以为善处喜和同而陷孤直以此临事祗见其掣肘而不可为以此与人祗见其暌隔而不相入矣此臣之所大虑也伏乞皇上申勅吏部通行内外大小臣工各洗夙习无事为同心一德之孚有事切同舟共济之念在已者不必尽是在人者不必尽非与其攻人不如自攻是非必归于一定议论毋涉于雷同至于任事者不以求全之毁而去避事者不以媚时之态而留又当事者留意焉耳如是则国是可定实政可举而万世治安之效在是矣又何患天下事不可为哉
