　　提要
　　【臣】等谨案秋涧集一百卷元王恽撰恽字仲谋汲县人世祖时官翰林学士承防事迹具元史本传恽文章自谓学于元好问故其波澜意度皆不失前人矩矱诗篇笔力坚浑亦能嗣响遗山史称恽有材干集中闗系政治诸作尤为疏达详明了如指掌其集凡诗文七十七巻又承华事略二巻乃裕宗在东宫时所撰以进者裕宗甚喜其言令诸皇孙传观焉中堂事纪三巻载中统元年九月在燕京随中书省官赴开平议事至明年九月复回燕京之事于时政缀録极详可补史阙乌台笔补十巻乃为监察御史时所辑御史台故事玉堂嘉话八巻则至元戊子所作乃追记在翰林日所闻见者凡文章得失典制沿革皆彚而録之甚称精核所谓辽金不当为载记之説尤为平允后托克托等以宋辽金三史并修实本其説其明于史法亦非杨维桢诸人固执偏见所及也乾隆四十六年十月恭校上

　　中统神武颂【并序】
　　维二年春王正月逆璮悖负天恩扞我大刑裒奸株顽啸凶东土于是命将致讨天戈一麾不五月而克清大憝兹盖皇帝陛下寅奉天心布昭神武睿智足临有征无战故也昔肃宗中兴唐室而元结献颂用歌大业垂示无极臣恽忝属太史亲覩盛事颂声不作咎将曷归敢缀缉所闻谨撰中统神武颂一首凡千有一百二十四字虽辞理未应庶防神武仁圣之徳铿鍧炳燿播宣金石振一代之徽烈也万冀微诚仰蒙天听臣恽惶惧昧死拜手稽首而献颂曰
　　皇帝践阼粤维三年万灵清跸廼狩于燕曰寒而寒曰燠而燠百度修明朝廷清肃东截海表西亘庸濮悉臣悉贡罔不率服时维逆璮祸心久纒再世鸱张颛征擅权奄殿全齐厉阶泗涟煮盐涸海铲铜夷山地伙物众蘖芽其间懵恃奸嚚海波可田背胁疽根怒而自鞭时徼邉功外示忠惓警我宋急骑通甘泉帝以徳御听之诚然再命不庭秽彰溥天浪骇鲸奔拟腾百川投袂东来盗我西偏天意有在聚而歼旃皇帝神武不震不愆燎我神鼎烹此小鲜命将兴师策出万全方叔召虎爪牙利宣爰命宗王相臣是丞颛彼封拜虎符发兵赫赫禁旅犀甲熊旌相臣曰璧维周之祯气盖一世徐方震惊皇帝曰嘻殱厥渠魁胁从罔惩既誓我师鋭莫与争贼谓我怯虿蜂来乗一战而退旷懐自矜谁以有待孤城自婴皇帝曰嘻璮不时下我民是恫武略神机运之渊衷相与谋谟丞相天泽乃拜稽首贼出下筹狶突于苙不死何求缭以巨堙毕克于秋帝曰元老壮哉谋猷其往视师代予以休马首既东零雨其蒙精彩一变气吐而虹万杵一动月晕而墉山防外援水防内通髙垒深沟俟其自毙日压贼营云頽死气贼势既窘魄褫心悸鼠防其牙兽奔厥角斧偃螳濳锋推猬缩七月壬戌环陴内讧贼伺我隙悉甲来攻万死幸脱突我梯冲相史相赵出竒无穷既败而壁众犹啸凶迨其癸酉有星蓬蓬騞若矢石妖声堕空因之出降羣凶率同彼狡狂卒不知所图衔窭鼠窜自投于湖宛转不死可知其愚童子执戈缚之以趋比枭逆首磔裂其躯并取同恶牵挛就诛林林部曲面缚俟死皇帝曰噫皆朕赤子若蠢何知一从先防解甲投戈纵归田里东际海壖西逾胶济杀气一空欢声四起既定厯城东趣广固比傅城下降幡内竖凡厥士女无或畏惧取彼逆属取其府库市肆农耕按堵如故累累妇妾婉婉弱龄三木囊头来献阙庭既令市朝咸即厥刑尽颁所获大犒将旅稛载驩呼各归禁伍劳我元臣其安尔所视我民政衮阙则补肆赦东方皇恩斯溥録我阵殁防我孤苦旧染惟新各安尔土皇帝曰噫厯城横燎凶干戈所厯复其租敛恵赐谷帛拯其昏垫阴伏阳开干旋坤奠海岱重清一出圣筭奄奄北风詟而逺窜威憺蛮陬东夷来献盖代称觞万口交庆再拜稽首皇帝神圣天锡勇智万邦以正亿万斯年受天明命皇帝孝友敦序九族有爵有土煌煌车服终嵗燕赐克谐克睦帯砺山河与天齐福皇帝睿哲别白忠直天赉虎臣左辅右弼孰敢予侮为天立极皇帝圣慈约法好生奚取八辟三章更轻大狱之起寜失不经如璮之诛万死何容罪止一族余无所从汉制党锢唐兴罗织滥及无辜惨莫之惕天地大徳曰生曰植致讨已来帷天是则吾皇之心尧仁舜徳海宇熙熙春台夀域羣臣归美度越前古唐平淮西汉灭呉楚以今方之其烈斯钜规规孝景计诛臣错岌岌宪宗皇咨于度兵纒四载吴防于愬惟此齐功五月而成断以睿筭畀之戎衡皇帝执谦弗居弗处宗庙之灵将士之助敢告成功于考于祖祀事载【阙】牲牢钟虡树羽崇牙应田县鼔肃雝和鸣先祖是听神灵曰都嗣皇之英持盈之徳守成之能保我丕祚万福是膺夀等南山不骞不崩小臣眷眷职司述称属辞比事献芹之诚亲覩盛事拭目太平渊嘿无言咎将孰胜冐昧狂斐触犯雷霆敢摭舆议用垂颂声
　　赋
　　蛾眉砚赋【并序】
　　余家所役二砚研磨有年殆与瓦砾不异书生作计朝夕笔研间所用若是欲善其事而不能也宜矣乙卯春偶得是砚于贩夫之手质名于谱盖歙石蛾眉之种也噫岂天诱愚衷将为文字之祥邪抑亦有数自王氏而发潜徳之光耶遂笔而赋焉其辞曰
　　嘿斋主人行年廿有九日从事于翰墨将逰心于道囿家畜二砚形模老丑研磨嵗深埃尘瓦缶叹材石之莫遇偶见授于野叟发此秘藏殆天所诱归予之月卯插其斗从吾囊之羞澁喜溪光之照牖其质圎以古其色緑而黝鬼掷滈璧镜蚀秦纽太初之面目已露元气之肧胎未剖涵玉徳之余润鉴金声于一扣其阅人多矣防墨翻于醉防也肌理细腻丰而不蔀顾溪痕之隐然旋摩挲而无有雪浪翻江映日光走其莹滑温纯之徳诚砚中之琼玖也若夫苍文上横眉妩初就浓抹逺山濯如春栁其横云却月之状又似开乎成都之画手质名于谱兹歙石蛾眉之右也予想夫龙山之英歙溪之浏醖醸防年种此竒秀信神物之有归不圗为嘿斋座中之友也当其书幌昼明笔耕而耨一嘘然云蒸雾凑试墨花于春融觉松风之泻窦纵未洒蔷薇之春露听此亦足以醒张颠之宿酒也相期百年以介尔夀砚兮砚兮汝名虽异傥浮于寔兮终有愧于屋漏故东坡有云澁不拒笔滑不褪墨兹乃砚之佳焉又何必鸜其目而鳯其咮也耶
　　茹野菊赋【并序】
　　丙寅春三月予以司明告病避忌食物无可供嚼啮者家人辈采菊芽为蔬考之圗经味甘平疗风热于头目极顺适也遂笔而赋之其辞曰
　　蹇予投劾而来归兮守先人之故庐日闭闗而却扫兮殆幽潜之不殊视外物其敝屣兮将游心于圣书何庖防之逺戒兮防釡中之生鱼嗟肉食其无相兮喜盘蔬之有余洁我晨羞出于不虞拥案未前露香满裾童子哑然而告予曰兹野菊之方苗适采撷于林墟考之圗经食有益于头颅也若夫春芽翠茁芳茎碧腴汤沃苦尽鼎调气濡甘涌舌本味欺园蔬拉晚菘以为友役春蒲而作奴贮空腹而不厌恐落英之不如彼寒花之可爱寄髙情于一壶纵傲睨乎秋霜逾佳晨而日踈此方腾借春阳中拆而敷收微生于全体供野人之所需吾乃沥芳馨而三咽觉眼花之扫除其气平夷其乐舒徐岂一杯之野馥等悦人之豢刍虽草木之诳口亦因兹而起予世有啖糠覈而瓠肥厌梁肉而黒癯岂非适于口而心苦濡以膏而体劬吾知甘菲薄以自乐能充恱其体肤既而隠几防坐心明志虚佳境翛然恍与梦俱以神为马以凥为舁既发轫于菊潭或弭节于山隅访渊眀于栗里寻天台之隠居然后鼓东南之归枻追散人于江湖觉而赋之犹足以续离骚而吊三闾也
　　未央瓦砚赋【赠终南李錬师和甫】
　　斗直寅日维酉栩栩蝶化梦入灵囿觉而扣门有来叟投我天砚形圜色黝鬼掷滈璧镜蚀秦纽文章郁嵂蛟蟠虺走以长乐无极而言乃汉殿烬余之瓦首也当其炎运天开禹卑秦陋未央屹建长乐丕构体紫极之圎方壮神丽于九有俾子孙无复加于后也若夫万叠云浮跨凌天宇鳞影参差鳌背掀举琼缀檐牙香团桂树倒景激日驭之光流润浥金茎之露耸万之衣冠荫千秋之歌舞砺太华而帯河渭谓斯瓦之贞固天落妖彗一扫而去咄尔微质归复其土何千嵗而不化偶野人之与遇今归乎我又系厥数摩挲汉月碧花灿吐似喜余之主也试蛟兮海运泻松风于牕户滑不墨褪澁不笔拒然复其天则御阳【周宣王宫名】之耳孙较其古乃铜雀之鼻祖彼鸜其眼而罗其纹咮则凤而形则【阙】用或【阙】　　奚足取是不若兹瓦之【阙】　而【阙】
　　屋廕生聚帡幪霜雪蔽亏风雨是则用
　　得其所付大任于汝也岂其棐几呈材辞客与伍处虽失宜其功尚溥鍜我铦锋借汝涵湑乞英灵于班马鉴兴亡于今古大书特书犹足以旌朱游而削张禹也道士顾笑吾子其为我赋之
　　三峰晴雪赋【并序】
　　乙亥嵗冬十一月余以守吏将事自蒲抵华仓皇待命于尘坌间望三峰晴雪焕烂髙洁皜不可尚已顾而自怜岂胜惭秽适与左山商公遇于祠南柏行间归作是赋以寄其词曰
　　华于众岳峻极莫之京兮三峰絶观松雪益其清兮倒景旁射银阙烂峥嵘兮仙掌翻空灏爽下青防兮岂玊井莲芳仙游拥霓旌兮巨灵变现咤昆丘之琼兮金天移仗敞瑶华之宫兮冷飇吹而不灭海月照还空兮轩豁呈露何物外之雄兮又如螭蟠之老子与希夷之钜公朗出天表饮太泉而游封中兮安得杖仙人之九节登神丘之崇兮问真源于白帝追二公之踪兮胡一行而作吏抢尘土之容兮对清都而不克往世故驱以东兮掉尘鞅而北骛刺梗满吾胸兮望岳神而三叹归梦挂岩松兮纵灵移之罔诮独不愧于躬兮归而赋之吾言固夸犹足以为执热者之清风也
　　熙春阮赋【并序】
　　晖上人得隆徳故閤余材斵而为阮因以熙春目之亦文殊之义也至元戊寅春同尹端甫刘御史叔谦赵太博彦伯坐心逺轩师为鼓緑水悲风二曲清越悲壮坐客感叹兴亡有怆然于懐者师请予赋之其辞曰
　　天新雨风日妍心与闲防境趣静便拉旷懐之逸士过雪庵之老禅敛袂危坐添炉注笺于是出桂阮调素控朏月而当抱抚筠柱之横骈师曰兹熈春之断余偶丝之而清圎者也遂鼓一再行清越悲壮如泉咽水底风行樾间翛翛兮欲断而复作淙淙兮似潄而还湲铿尔擿铜丸之韵凄兮聆髙树之蝉又如游士骋説劒歌去燕何辞气之从横激太清而沈羽渊客乃呀然而惊惨然而悲信乎物之感人有不期然而然者想老徽之般乐荡侈魄以无前厌华阳之湫隘敞楼居而致仙海鼇首抃金鹏翼骞郁鼎莲之香雾锁月殿之婵娟荫千秋之歌舞夀艮岳而不骞俄龙移而鼎去继金亡而阁迁殆露槃之声折怅秋风而涕澘兴废一瞬两朝百年委繁华于野草锁莽苑之荒烟咄此木之何幸遇幽人之达观纳须弥于芥子寓余哀于一弹犹足以舞鱼龙于海曲洗亡之愁顔也呜呼噫嘻向之使人凛而悲者物讵能动于人哉人之哀乐妄为物之所牵不知太虚寥寥既寂而唯此理之嘿识何繁华废兴之足叹师曰子其为我赋之吾欲秋月当户松风满轩写仲容之别操合清商而洞宣使云阴蔽空古鹤下蹮走商延于濮上掩新声于师涓也
