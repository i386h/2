　　提要

　　臣等谨案文简集五十八巻明孙承恩撰承恩字贞父南直隶华亭人正徳卒未进士官至礼部尚书兼翰林学士掌詹事府谥文简是集为其门人杨豫孙董宜阳朱大韶所编七巻以前为疏表讲章皆进呈之作八巻以后为赋诗词曲二十七巻以后为杂文陆树声为之序承恩于嘉靖之初以庶子充经筵讲官今集中所载正始箴鉴古韵语及讲章即是时所作及官宗伯时斋宫设醮承恩独不肯黄冠遂乞致仕较之严嵩诸人青词自媚者人品卓乎不同其文章亦醇正恬雅有明初作者之遗树声序有曰国初之文淳厚浑噩彬彬焉质有其文迨闗西信阳两君子出追宗秦汉薄魏晋而下海内艺学之士咸愿执鞭弭从之标品位置率人人自诡先秦两汉以希方轨虽体尚一新国初淳龎浑厚之气或少漓焉公生长宪孝朝博稽宏览邃诣渊蓄故出之撰述类皆深厚尔雅纡徐委折论者谓公平生立言类其为人云云亦平允之论也

　　●文简集原序

　　毅斋先生孙文简公平生所著诗文属公门人校次成集者总若干巻公殁子汉阳守克弘出以授梓问序于余余不佞何能为役往不佞释褐公与闗中马文简公实贰宗伯两先生皆馆阁宗巨不佞辱公里闬后进数因之以幸承教两先生两先生所为推毂不佞已不佞入读中秘书职史氏获从子大夫后入侍交戟之内退而诠论作者大都国初之文醇厚浑噩维时鼎命方新宇夏寜廓一二文儒应兴运而出以当制作之任鸿裁瑰撰遵尚体要列圣绍休太龢淳龎之气萃宣人文炳蔚融鬯一时儁喆逓兴登馆阁以黼黻丝纶藻缋帝謩者彬彬焉质有其文迨闗西信阳两君子出追宗秦汉薄魏晋而下之海内艺学之士咸愿执鞭弭从之标品位置率人人自诡先秦两汉以希方轨虽体尚一新概之国初淳龎浑厚之气或少漓焉公生长宪孝朝夙挺英喆少壮登朝偶亨嘉之会洊厯华贯直承明著作之庭博稽闳览邃诣渊蓄故出之撰述类皆深厚尔雅纡徐委密不钩棘而匠意闳深不琱镌而藻思芊缛不拟躐而动引绳墨舂容典致卓乎大雅譬之冠冕佩玉委蛇中度铏豋在列苍然古色要之体格取裁一轨于先正而出入驰骤在庐陵步武间论者谓公平生立言类其为人公直项不茍词色淳靖廉退终始一节独以忠实结主知当公宫詹时疏请建储者三出阁讲读者二上亷公忠悃方向意公会公引年去不踰年召还公朝论谓公旦夕且拜麻未几公坚意乞归矣昔宋陈恭公入相学士张安道草制麻独叙其首请建储谓功在纳忠向令公不去即少须臾入正台揆功业可逆睹公竟髙引使论世者惜公不究用于时乃余考按公立朝前后章奏恳恳诚至得老成爱君忧国之体即所施措虽未万一试其经纶大指亦较然着矣余不佞追惟始进窃冀幸承教于公晚复承命序公之集因论次公生平并识其大者若此万厯癸未夏五月赐进士出身资政大夫礼部尚书兼翰林院学士经筵讲官国史总裁前吏部右侍郎掌詹事府事陆树声序

　　○箴

　　正始箴

　　翰林院编修臣孙承恩谨奏为端本始以隆治道事恭惟皇帝陛下承太皇太后大行皇帝简命付托入继大统神器有归生民有主社稷幸甚天下幸甚臣惟天位维艰不可不谨然尤莫当谨于初政之日初政者天下臣民拭目望治之始天意鉴观以为去留治忽安危之所系也是以古人忠告于其君者莫不致意于是昨者陛下甫正宸极念承传之重明发诏旨大新庶政谦冲退托图治拳拳臣谨伏读则任贤纳谏爱民屏奸収人心慎爵赏振纪纲肃刑威凡所以端本始者陛下固已得之矣夫自古帝王之兴未有不获天人之归而能永保治安者窃观陛下当毓徳名藩圣明之誉已洽逺近及承命将至万姓踊跃争先快覩而至止之晨灵雨时降登极之际天清日熙及明诏之降父老感激垂涕谓不图亲见圣明天子与此太平景象天休震动人心欢腾非偶然也陛下真可谓极天与人归之至矣天人既同治安可必则陛下所以上答下承者当何如哉夫始者终之对谨始所以善终善终必先谨始未有谨始而不能善终有罔终者则由始谨之未至也亦未有善终而不本于谨始少怠于初斯终之所以不善也陛下兹可谓得端本始者矣因所已得而益探其本自身心以及政治益求所以端本始者斯则所以善终也固结天人培植国祚亿万载无疆之休端在于此臣以陋劣备员词垣幸逢圣明媿乏禆补然区区诚恳不胜愿望之私欲以刍言上赞万一特竭庸钝作十一箴一曰正始二曰修身三曰敬天四曰法祖五曰勤政六曰讲学七曰慎微八曰持志九曰谨好尚十曰简近习十一曰溥仁恩纂葺旧言麤成条贯意存忠爱僣越不辞谨随本献上臣无任惶悚战惧之至
　　正始
　　皇皇大君其道不易克艰所先慎始之义始义云何若弩之牙必审必慎乃罔攸差惟昔杨朱见途而泣可东可西惟所意适嗣徳在初伊告其君作事谋始易亦有云毫厘之差千里之谬发轫弗臧莫可药救惟古圣哲谋逺虑深上承天意下副人心天人咸归绸缪固结治以渐弘盛大之业茍简微见百虑一讹即使改为其损已多于惟我皇当阳出震中外臣民翘首观听新服厥命徳亦惟新恭已端本追踪古人惟善惟恶厥有贰辙出此入彼更靡他说念兹伊始兢业自持尚观治平式隆丕基
　　修身
　　天下之本实惟君身身修之化至疾至神修身之功正心诚意由身而推家齐国治正心律已罔有弗恭意诚及物幽明感通不诚不信不正曷令颠迷昏狂咎恶滋甚仰惟圣哲夜寐夙兴如对上帝如承神明制事以义制心以礼内外夹持罔或过举声律身度左矩右规一言一动宪天之彝身焉既修是曰端本动无不化推罔不凖于昭史册巍巍皇皇万世人主戴其休光彼昏彼狂遗讥罔极孝子慈孙不可湔涤敬告陛下监念在兹恶可为戒善可为师治化日洽声光日着天徳无私作配尧禹
　　敬天
　　在昔伊尹敬告厥辟曰天无亲所辅以徳天虽巍然视下则卑人君动息天必应之彼昏不知谓天不吾计泻天弃天无所不至天乃震怒降以祸灾弃之若遗不可复回惟天爱君岂欲如是天之于君父子之义父固爱子教望实深子固絶之父宁不嗔是故明君天命是勅兢兢业业不敢怠忽敬天云何匪徒虚文勉尽君道循理修身以是格天天乃用慰区区虚文天不容伪况此初服始承天休天命所视或短或修钦若圣皇名藩毓徳仁孝敬承天所眷锡自此以往益敬益虔用祈永命享祚万年
　　法祖
　　书监成宪诗由旧章是曰君道靡愆靡忘圣帝明君孰非可法法必于祖益近而切耳目所逮气体所承追思企仰寤寐墙羮矧惟臣民安此徳泽思复见之鼓舞悦怿于穆孝宗逺承三王十八年在位其道弥昌不敢逰畋不敢纵逸商宗寅恭周王畏抑如汉文恭俭纯诚则殊如宋宗仁厚明断有余厚泽深仁深培固结岂惟则然有谟有烈尊亲爱戴率土媚兹培植国命万世赖之圣子神孙宜是则效绍述不忘斯谓大孝恭惟圣皇干蛊惟新振起饬治如涉大津践修前猷功光圣祖庶几鸿名嫓美万古
　　勤政
　　昔益告舜罔逸罔淫奭也告武夙夜罔不勤舜武圣人犹以是训则凡人君其可不慎旦陈无逸谆切丁宁废兴治忽监戒孔明天位惟艰万机沓至逰豫宴逸政乃废坠天运惟健永位高明君子自强法天之行未明而衣当旰忘食汲汲孜孜靡间顷刻况复初政蛊壊之余兴化善治易辙改图事之当为未易数缕匪倍其勤曷以克举逸则日纵改悟莫从讳过拒谏乃罔克终帝其念兹愿自振励上师古人励精图治斯须宴乐虑妨万几毫髪纵恣终身之非慎终如初与勤相习庶绩用熈治道以毕
　　讲学
　　君心昏明政由治忽欲明无昏惟学为急难明易昏惟人心之常开以义理警以存亡心犹刃焉学乃砥砺心犹鉴焉学去尘翳思古明辟率是孜孜日就月将功期缉熈说告武丁学于古训汉唐令主亦克好问惟帝为学胡可弗勤培养圣质启发圣心然学之方当尽其道末而词章则匪其要词章技艺丧志无裨精一之传圣学之轨朝有经筵讲学之地立厥成规不以寒暑废爰咨爰择侍从儒绅降畧陛级俾从容以陈或稽之经或参之史义理大原兴亡大旨法善戒恶圣心昭明以达为政治罔不成
　　慎微
　　天下之事必有其几几之所伏至隐至微众人蚩蚩昧厥先见懵焉弗觉纵恣怠玩岂知至隐至显者存孰谓至微至着之原彼惟不觉乃迄祸害弗虑将长弗虑将大一星弗灭势必燎原蚁穴弗治遂成滔天滔天既成溃堤莫塞燎原既盛竭泽奚益大易有训履霜坚冰絶过未着防祸未形未着而絶乃至无过未形而防卒以无祸故曰圣哲知彰知微众人所忽圣哲所虞一念必谨中心恒惕如有祸患近在袵席帝实圣哲明照弗遗而况初服祸福之基敬哉念哉无隐无显杜渐防微噬脐可免
　　持志
　　人欲有为贵持其志志焉弗持或进或退是故明哲发强毅坚见之既灼守之弗迁志用正直不杂佞士志本抑畏不夺侈肆志在听纳触讳靡尤志在问学罔有辍休志在天徳不杂人伪志存法古不安近治志明如日万古常新志仁如天永冐万年彼中才主执徳弗固始非弗臧卒则改度亦有昏虐长恶遂非志存不善莫可转移仰惟吾皇赫赫初政坤转干旋百度咸正阴霾清廓阳徳焕升旁烛无疆九幽昭明仰惟自今乆暂如一如水必东无有改易道日以广治日以纯万万世下仰瞻圣神
　　谨好尚
　　惟人之心众欲交病端虽甚微卒罔克禁矧惟人君富贵崇高易纵难遏六马弗调况兹憸壬善诱巧伺投间抵隙以售其计惟是上智烛理斯明屏絶嗜好湛然中扄亦有嗜好道徳仁义亦好善言亦好直士纷华可喜视之尘泥渊然澹然泯焉莫窥道用以周徳用以植虽百憸壬无可自入茍或悖是多欲一萌憸壬抵掌竞为承迎心本易淫复此沉溺颠倒祸败乃罔纪极钦若陛下正始之基羣小喁喁潜窥隐微治忽之分如反复手操持维难若御大冦惟帝念哉寡欲养心岂惟逺佞百福是临
　　简近习
　　君徳修否实系近习气体移养近习之执与吉人居习于其正黙化潜孚弗觉其进与邪人居习于其非潜引黙夺与化弗知渐染熏陶莫是切近鲍鱼芝兰古人所慎近朱者赤近墨者玄齐言楚语以地而迁惟昔盛时特谨于是陪仆■〈执上目下〉御罔非吉士旦夕承弼纳之矩规欲为不善其孰与为细而起居大及为政乃罔不钦亦罔不令恭惟陛下即位之初奔趋盈庭莫辨贤愚尝以是求谀佞正直正直必进谀佞必斥进一正直众正乐辅斥一谀佞众佞自阻前后左右丞疑交修有益无损圣徳日休
　　溥仁恩
　　于惟天道有冬有春人君法天有义与仁冬主肃杀春主发育贞下起元剥则有复霜雪之后雨露以濡阳和昭融一气而嘘刑诛之余乃有庆赏洪恩汪濊照泽生养元贞迭运天道以行仁义迭施君道以成于惟圣皇配天出治振纲整纪风飞雷厉赫赫肃肃恭行天诛元恶巨奸咸服其辜天威孔昭天恩未沛人知其威未见其惠况兹余孽战惧靡宁圣徳包荒许以更新旷荡之仁当以时出以释危疑以安反侧明明我后讵独是遗曰有所待以渐而施小臣僭冒仰赞万一仁昭义行永奠邦国
