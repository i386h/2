　　提要
　　
　　臣等谨案说学斋稿四巻明危素撰素有草庐年谱已着録据千顷堂书目其文集本五十巻明代已散佚不存此本乃嘉靖三十八年归有光从吴氏得素手稿传抄其文不分巻帙但于纸尾记所作年岁皆在元时所作有光跋称共一百三十六篇此本乃止一百三十三篇又王懋竑白田杂着有是集跋称赋三赞二铭二颂三记五十有一序七十有六共一百三十八首以有光跋为传写之误然据懋竑所列实止一百三十七首数亦不符殆旧无刋板好事者递相传録故篇数参差不能画一实则一本也素晚节不终为世谬笑其人本不足称而文章则欧虞黄栁之后屹为大宗懋竑跋称其文演迤澄泓视之若平易而实不可几及非熙甫莫知其深其珍重钞传葢非漫然矣

　　三节堂赋（乙酉）
　　
　　稽受氏于髙堂兮奠偰辇之攸居际神元之兴运兮肆强暴而攘除伟令子之俶傥兮从世皇以平呉匪徒戮力于战陈兮陈征讨之吁谟信直道以迕时兮转漕挽于畨禺属羣盗之交炽兮集草间而啸呼阨险隘以弄兵兮值陆梁之凶渠矢赤心而奋斗兮凛英烈而莫渝俄势穷以被执兮必临难而捐躯惨海雾之阴翳兮嘂山鬼而乗狐嗟良配之嫠居兮厯三纪而有余抚青镜以含悲兮収涕泗于衣裾哀孤惸于幼稚兮念生我之劳劬悯无父之何怙兮赖慈母之携扶撄疾疢以莫瘳兮吾岂爱于肌肤刲股肉以和药兮欣沈痼之获苏世惊叹乎忠义兮作人纪之良橅表三节以咏歌兮峙堂构于东湖葆遗器以椟蔵兮纹缇袭乎诗书惟延佑之后皇兮思搜罗于文儒爰趾美于髙科兮羌簪笏之满拯黎庶于泥涂虽余文之骫骳兮揭潜徳以张铺信天道之孔昭兮式原本于厥初
　　
　　别友赋送葛子熙（癸酉）
　　
　　余生楚南之穷邦兮嗟不逮乎古之人执简策以永歌兮视前行之逡廵怅风气之日凋兮众猱杂乎滓秽独凉凉而奚适兮虞诽言之交至能服仁而蹈义兮虽颠沛庸何伤顾掺践之或怠兮悲丝染之玄黄思骖云螭以髙驰兮下览观乎四海带玉璐而佩寳剑兮划长啸乎天之外值美人于丹丘兮云抱朴之来孙朝吸上池之沆瀣兮夕掇玄圃之芳荪猎六籍之精华兮悲黄农之不吾待并騑騑之良马兮欲导车轮而言迈有疑义而与析兮且交修予之慝岂私情之昵比兮将以缉乎道徳懐夫子之固穷兮虽制义而自安顾甘旨之或缺兮宁孝子之所欢念一门之孤惸兮而衣食之莫具汨奔走于泥涂兮岁冉冉焉迟莫辍彭泽之桂棹兮爰东憇乎钱唐抚句践之故墟兮吊台观之榛荒扬髙帆以度海兮仍羣仙于云坞凌榑桑而遐瞩兮浴赤曦之暠暠盻梅梁而太息兮企圣贤于诸冯何三千歳之寥寥兮大运若是其污隆翳娭遨之足乐兮伤驰驱之劳勚扇朱光于炎宇兮安得激清风而来至旋轻辀于汝濵兮将同采乎苋藿玩曽峯于翠云兮（金溪有翠云峯）射麋鹿以为乐乱曰子何为行兮呉越之乡兮韶音之锵锵兮矫首以望虞皇兮邈矣夏禹海芒芒兮哀民生兮曷时而康兮
　　
　　存存斋赋（为李孟豳尚书赋　庚寅）
　　
　　繄太极之混沦兮为生化之本根维二气之阖辟兮畅万物之滋蕃羌易道之广大兮备吾身于乾坤极知崇以效天兮循礼卑而法地体成性之存存兮由道义之不匮信弛张之靡忒兮亦柔刚之相济伟邹孟之崛起兮当战国之兵争悼民心之迷缪兮揭皇极而载明谓君子之善存兮异羽毛之顽冥彼嵬琐之纷纷兮遂梏亡于外诱湛明镜之当台兮竟昬蒙于尘垢蔚嘉谷之罗生兮胡杂糅于稂莠懐咎繇之苖裔兮蚤奋迹于滕邦炯氷壶之交暎兮粹白璧之无双玩玄理于简册兮驱长才于海江爰校书于史馆兮持宪度于台阁善和爕于羣情兮或誾誾而谔谔沛大旱之甘霖兮苏八纮之毒蠚退休沐而斋居兮抚易象而观天亶存存之不息兮契洙泗之心传虽首路之邈悠兮傒白首而不迁忆旋辀于虎林兮郄倾盖于京阙瞻云霄之轩翥兮莫策厉于驽劣葆贞白于岁晏兮追徃行于前哲乐几筵之洁清兮陈左图而右书殖卉木之芬芳兮启韦编而载愉托中情于藻翰兮着懿戒于坐隅
　　
　　望番禺（有序　庚寅）
　　
　　广东道肃政亷访使钦察核军民达噜噶齐托欢察尔在广州多不法事江南行御史台遣监察御史镏振徃按之振受赇以钦察言非实钦察忿死振亦恐惧得疾还至龙兴驿舎白日见钦察于前因噤而死未几行台又遣监察御史杜某访其事得今衢州路总管薛超吾为江西行中书省令史时所赋诗遂合诸御史上章核振后世有（阙）年临川危素闻而哀之作望番禺望番禺兮海气沉沉青天高高兮白日为阴悲风满山兮夜闻噫喑豺虎呀哮兮麒麟遯潜望番禺兮尽伤予心
　　
　　经厯律图赞（庚寅）
　　
　　猗昔上古狉狉兟兟帝战涿鹿遂生乱民唐虞命契敷教人伦乃设庠序尊尊亲亲世变风移彛典弗循乃置刑威法令式陈楚有君子哀此顽嚚表经附律提要谆谆流行区宇明徳懐新安得率土返朴还淳各守厥职葆其天真上下四方休然阳春刑措不用而多凤麟我赞斯图爰诏无垠
　　
　　异石铭（有序　庚寅）
　　
　　寿武库副使髙昌额森布哈君以异石示予其大三寸色正碧上有白文有神人戴笠跨驴白气上贯中双龙摩空而飞君谓其大父甘州守库哩拜托欢公诣仁祖山佛寺得此石道中故牍而蔵之甚久盖灵气所锺睹斯异物乃为之铭铭曰
　　
　　太和絪缊二气摩荡山川储英竒石异状赋质正碧中含素文晔乎神人其行逡廵蜿蜿惊龙乗云飞跃干端坤倪胶轕回薄赫尔精气腾为列星善椟而蔵何百千龄
　　
　　范文正公读书台铭（有序　庚寅）
　　
　　宋资政殿学士户部侍郎累赠太师开府仪同三司楚国文正范公幼孤贫从其母适长山朱氏朱氏宰安乡公读书于太平兴国观既擢进士第入叅大政功名烜赫着在竹帛后人指其遗迹而咏歌懐思者无已会宋中书舎人庐陵刘公才邵八世孙珍侨寓是邑乃作祠宇割良田以供祭祀俾素述文刻诸台下乃为之铭其词曰
　　
　　允显范公东南之英出将入相晔乎功名睠兹安乡为酆属邑出自北门林阜孔岌公有遗迹峩峩髙台世变事移过者兴哀仲璧氏刘衣冠之裔侨居此乡仰止异世伐木甚良爰树髙堂爼豆载陈灵其来享陟彼崇丘凄其延伫江流滔滔岁不我与猗公之仕载逢其辰孰使荆扬才杰沈沦邈哉风旨弥乆弥新篆铭贞石公有鬼神
　　
　　赐帛颂（有序　庚寅）
　　
　　皇帝御天下十有八年凡三赐老人帛于是抚州民张天佑与被宠命初赐时天佑年八十有（阙）再赐时八十有（阙）三赐时八十有（阙）最后之赐则九十有二天佑少好道术善保啬其精神大隐市廛无求于世蔵书甚富以朂其子孙其所以至于上寿者盖异于常人矣国家自祖宗以来子惠元元然于尊礼髙年尤为备至此先王之所以治其天下其民化其俗易成者也于是史臣素作颂以归美于上其词曰
　　
　　皇有万国莫不尊亲绍天明命抚之以仁民吾同胞海涵天覆仁政之行敬老慈幼筐篚肆颁皇有恩言惠泽斯沛孝理实敦维抚为郡江湖之表山谷僻深民多寿考微臣天佑生于宋时束帛所蒙明诏飙驰养老乞言三王之世临雍拜老亟称汉帝乃如斯人道术孔多蔵书满家守一处和安车聘贤尚法古昔宜秉赤心上禆圣徳化行俗美维皇之明昭垂罔极史作颂声
　　
　　蕲春县兴学颂（有序　庚寅）
　　
　　蕲春县长额森布哈君既新庙学郡学教授卫子炎为记君上计京师稍为余言治邑之事君始至官修农桑水利之政平狱讼恤灾伤无所不用其情其兴学之绩为最大盖蕲春之学在郡城东北之鲜原素无恒产儒之在籍者廿有八家多贫不能自振于是学舎倾圯弦诵寂寥君始至伏谒庙下大惧无以称国家崇学右文之意乃出禀给为之倡尹边守诚丞孙克昭主簿苏世荣教谕鲁志暨胥吏富民咸相其役因其旧而修之者大成之殿明新之堂撤而新之者门庑斋舎创成之者棂星大成二门若夫祀所龛室帐帟祭器无有弗备学之闲田悉以种桑岁集之以补赡学之费民知令之所向相劝从事于诗书而敦行孝弟君之于政知所先后如是乃为之颂使刻之石其词曰
　　
　　我行江滨爰采其蕲田多污莱埜有嚬呻天监下土哀此罢民畀我贤令惟皇之仁学校之行先王善政迨我世皇着在甲令蕲春有学以祀玄圣栋桡宇倾曷以广敬贤令聿至顾瞻裴回乃鸠羣工罔敢弗来祇奉明命佥言允谐有严爼豆民徳孔懐雍雍童冠济济弦诵修行明经聨翩充贡繄此长才小试其用伐石刻辞舆人之颂
　　
　　兴学颂（有序　辛巳）
　　
　　至正元年四月大都路都逹噜噶齐喀喇公伯嘉努请作左右南三警廵院大兴宛平二县孔子庙并建学舎都府亟达诸礼部部上之中书省中书从其言各以室庐之在官者为之又言学设教谕而无廪食非所以尊敬师道中书亦从之呜呼古之立学所以作成人材备国家之任使非细故也三院之所辖五方之人咸聚焉虽有长才秀民犹必待教而后明况于蚩蚩之氓乎二县者在畿内教之尤弗可以阙然也我朝建邦设都百年于此国学京学规制已修顾于五庙学未遑有作而作于公尹京之日则公之于政可谓知本以能成昔人未成之志何其盛欤抑亦遭逢圣人埀意斯道故公之言得以取信于朝廷而无或沮止之者五学之师弟子其亦思无负公之志哉公字九德在官守职若活海贾事皆可书今独取其为政之知本者为颂之其词曰
　　
　　维昔火帝即燕设都山髙川回神明相扶乃作庙学于京于国以祀圣师以讲道徳人才勃兴邦家之光小为杗桷大为栋梁三院警廵以及畿县置师无学有弗闻见繄康里公实莅神京治政以平豪猾震惊曰兹教化为予之责请于宰臣报可甚亟爰奏瑟琴升歌雅南笾豆在筵有赫其临匪公言行际时之盛化被国都风俗以正肃肃冠裾诵诗读书凡百多士毋昧厥初
