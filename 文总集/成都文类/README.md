　　提要
　　【臣】等谨案成都文类五十巻诸家着録皆称宋袁説友编説友字起岩建安人隆兴元年进士嘉泰中官至同知枢宻院参知政事是编前有説友序葢其庆元五年为四川安抚使时所作然巻首别有题名一页称迪功郎监永康军崇徳庙扈仲荣迪功郎新差充利州州学教授杨汝明从事郎广安军军学教授费士威从亊郎前成都府府学教授何惪固文林郎山南西道节度掌书记宋徳之文林郎前利州东路安抚司干办公亊赵震宣教郎新奏辟知绵州魏城县主管劝农公事徐景望奉议郎新云安军使兼知防州云安县主管劝农公事借绯程遇孙编集而不列説友之名説友序中亦但云爰属僚士摭诸方防裒诸碑识而无自为裁定之语然则此集之编出説友之意此集之成则出八人之手当时旧本题识本明后人以序出説友遂并此书而归之非其实也所録凡赋一巻诗歌十四巻文三十五巻上起西汉下迄孝宗淳熙间凡一千篇有竒分为十有一门各以文体相从故曰文类每类之中又各有子目颇伤繁碎然昭明文选已创是例宋人编杜甫苏轼诗亦徃徃如斯当时风尚使然不足怪也以杨慎全蜀蓺文志校之所载不免于挂漏然此集剏始难工而慎书踵事而增则易于为力固不能一例视之且使先无此书则逸篇遗什慎必有不能尽考者此书搜辑之功固亦未可尽没矣乾隆四十五年十一月恭校上

　　成都文类序
　　天地之秘蔵发而为名山大川山川之秀灵敛而为文章华藻二者相为颉颃而光明焉也两京三都之赋摹写天地绘绣山川绚道徳掞天庭润金石谐韶濩与乾坤造化周流盛大于宇宙之间千百万世下而知有两京三都者以此文也然则天地山川而可无此文哉而可以不传此文哉益古大都防也有江山之雄有文物之盛竒观絶景仙防神迹一草一木一丘一壑名公才士骚人墨客窥竒吐芳声流文畅散落人间何可一二数也凡此者予来三年亦既略睹矣或曰两京三都以赋而传使无传焉斯文泯矣然则繇汉以来其文以益而作者今独无传可乎有益都斯有此文此文传益都亦传矣爰属寮士摭诸方策裒诸碑识流传之所脍炙友士之所见闻大篇雄章英词绮语折法度极耀其以益而文者悉登载而彚辑焉断自汉以下迄于淳熙其文篇凡一千有竒类为十一目厘为五十巻益之文兹备矣嗟乎后世之士岂无浮沅湘由巴蜀【阙】　　如司马子长者乎岂无上瞿唐过防梓赋雪锦如杜少陵者乎又岂无自西蜀歴荆楚栖迟山水间如田防岩者乎倘复得如二三公者而访斯益擥斯文焉则知清宁阖辟至大至广之内而有所谓蜀蜀六十州亘五千里之内而有所谓益益都无量江汉炳灵之内而有所谓文者其不在此书乎是书也而有传焉庶几乎无负于益庆元五年二月望日宝文阁学士通议大夫四川安抚制度使兼知成都军府事建安袁説友谨序

　　蜀都赋　　　　　　　　　汉扬雄
　　蜀都之地古曰梁州禹治其江亭皋弥望欝乎青防丹青玲珑石防水螭扵近则有瑕英菌芝玉石江珠扵逺则有银鈆钖碧马犀象僰西有塩泉鐡冶橘林铜陵其傍则有期牛兕旄金马碧鸡其竹则宗生族攅俊茂丰美夹江縁山寻卒而起其深则有猵獭沉鱓水豹蛟蛇其都门二九四百余闾两江珥其市九桥带其流苴竹浮流龟鼈碛石风胎雨鷇众物骇目百华投春隆隐芳芬螭烛若挥锦布绣望芒芒兮无幅其布则筩中黄润一端数金雕镂扣器百技千工上乃使有伊之徒调夫五味甘甜之和勺药之江东鲐鲍陇西牛羊五肉七菜朦猒腥臊若其吉日嘉防期扵送春之阴迎夏之阳置酒扵荣川之闲宅设坐于华都之髙堂延帷扬幕接帐连岗郄公之徒相与如乎阳臣诏罗车百乘期防投宿观者方防行船竞逐也
　　右篇得之艺文类聚疑非全文自郄公之徒以下字义若不相聨属按左思蜀都赋注云扬雄赋曰若其渔弋郤公之徒相与如乎巨野罗车百乘观者万隄意臣野讹为臣诏万隄讹为方防云
　　蜀都赋　　　　　　　　　晋左思
　　有西蜀公子者言于东吴王孙曰盖闻天以日月为纲地以四海为纪九土星分万国错跱崤函有帝皇之宅河洛为王者之里吾子岂亦曽闻蜀都之事欤请为左右扬防而陈之夫蜀都者盖兆基扵上世开国扵中古廓灵闗以为门包玉垒而为宇带二江之双流抗峨嵋之重阻水陆所凑兼六合而交会焉丰蔚所盛茂八区而庵蔼焉扵前则跨蹑犍防枕倚交趾经途所亘五千余里山阜相属含谿懐谷冈峦纠纷触石吐云欝葐蒀以翠防崫巍巍以峨峨干青霄而秀出舒丹气以为霞龙池瀑濆其隈漏江洑流溃其阿汨若汤谷之扬涛沛若蒙汜之涌波扵是乎卭竹缘岭菌桂临崖旁挺龙目侧生荔枝布绿叶之萋萋结朱实之离离迎隆冬而不雕常而猗猗孔翠羣翔犀象竞驰白雉朝雊猩猩夜啼金马骋光而絶影碧鸡倐忽而曜仪火井沈荧扵幽泉髙焰飞煽扵天垂其间则有虎魄丹青江珠瑕英金砂银铄符采彪炳晖丽灼烁扵后则却背华容北指昆仑缘以剑阁阻以石门流汉汤汤惊浪雷奔望之天逈即之云昏水物殊品鳞介异族或藏蛟螭或隠碧玉嘉鱼出扵丙穴良木攅扵褒谷其树则有木兰梫桂杞櫹椅桐防枒楔枞楩楠幽蔼扵谷底松柏蓊欝扵山峰擢脩榦竦长条扇飞云拂轻霄羲和假道扵峻岐阳乌回翼乎髙标巢居栖翔聿兼邓林穴宅竒兽窠宿异禽熊罴咆其阳雕鹗鴪其阴猨狖腾希而竞防虎豹长啸而永吟扵东则左緜巴中百濮所充外负铜梁扵宕渠内函要害扵膏腴其中则有巴菽巴防灵夀桃枝樊以蒩圃滨以盐池蛦山栖鼋龟水处潜龙蟠扵沮泽应鸣鼔而兴雨丹沙赩炽出其坂蜜房郁毓被其阜山圗采而得道赤斧服而不朽若乃刚悍生其方风謡尚其武奋之则賨旅翫之则渝舞鋭气剽扵中叶蹻容世于乐府扵西则右挟岷山涌渎发川陪以白狼夷歌成章坰野草昧林麓黝儵交让所植蹲鸱所伏百药灌丛寒卉冬馥异类众伙于何不育其中则有青珠黄环碧砮芒硝或丰緑荑或蕃丹椒蘪芜布濩扵中阿风连莚蔓于兰臯红葩紫饰柯叶渐苞敷蘂葳防落英飘飖神农是尝卢跗是料芳追气邪味蠲疠痟其封域之内则有原隰坟衍通望弥博演以潜沫浸以緜洛沟洫脉散疆里绮错黍稷油油稉稻莫莫指渠口以为云门洒滮池而为淕泽虽星毕之滂沲尚未齐其膏液尔乃邑居隠赈夹江傍山栋宇相望桑梓接连家有塩泉之井户有橘柚之园其园则有林檎枇杷橙柿梬楟榹桃函列梅李罗生百果甲宅异色同荣朱樱春熟素柰夏成若乃大火流凉风厉白露凝防霜结紫梨津润樼栗罅发蒲桃乱溃石榴竞裂甘至自零芬芳酷烈其圃则有蒟蒻茱茰畴芋区甘蔗辛姜阳蓲阴敷日往菲防月来扶防任土所丽众献而储其沃瀛则有攅蒋丛蒲绿菱红莲杂以蕴藻糅以苹蘩揔茎柅柅裛叶蓁蓁蕡实时味王公羞焉其中则有鸿俦鹄侣防鹭鹈鹕晨鳬旦至雁衔芦木落南翔冰泮北徂云飞水宿哢吭清渠其深则有白鼋命鼈獭上祭鳣鲔鳟鲂鮷鳢鲨鲿差鳞次色锦质报章跃涛戏瀬中流相忘扵是乎金城石郭兼市中区既丽且崇实号成都辟二九之通门画方轨之广涂营新宫于爽垲拟承明而起庐结阳城之延阁飞观榭乎云中开髙轩以临山列绮窓而瞰江内则议殿爵堂武义虎威宣化之闼崇礼之闱华阙双邈重门洞开金铺交映玉题相辉外则轨躅八逹里闬对出比屋连甍千庑万室亦有甲第当衢向术坛宇显敞髙门纳驷庭扣钟磬堂抚琴瑟匪葛匪姜畴能是恤亚以少城接乎其西市鄽所防万商之渊列隧百重罗肆巨千贿货山积纎丽星繁都人士女袨服靓妆贾贸墆鬻舛错縦横异物诡谲竒于八方布有橦华麫有桄榔卭竹传节于大夏之邑蒟醤流味于畨禺之乡舆辇杂沓冠带混并累毂叠迹叛衍相倾諠哗鼎沸则哤聒宇宙嚻尘张天则埃曜灵阛阓之里伎巧之家百室离房机杼相和贝锦斐成濯色江波黄润比筒籯金所过侈侈隆富卓郑埒名公擅山川货殖私庭藏镪巨万鈲槻兼呈亦以财雄翕习边城三蜀之豪时来时徃飬交都邑结俦附党剧谈戯论扼腕抵掌出则连骑归从百两若其旧俗终冬始春吉日良辰置酒髙堂以御嘉宾金罍中坐肴核四陈觞以清醥鲜以紫鳞羽爵旣竞丝竹乃发巴姬弹弦汉女击节起西音于促柱歌江上之厉纡长防而屡舞翩跹跹以裔裔合尊促席引满相罸乐饮今夕一醉累月若夫王孙之属郤公之伦従禽于外巷无居人并乘骥子俱服鱼文黄异校结驷缤纷西逾金隄东越玉津朔别期晦匪日匪旬蹵蹈防笼渉猎寥廓鹰犬倐眒罻罗络幕毛羣陆离羽族纷泊翕响挥霍中网林薄屠麖麋剪旄尘带文虵跨雕虎志未骋时欲晚追轻翼赴絶逺出彭门之阙驰九折之坂经三峡之峥嵘蹑五屼之蹇产防食鐡之兽射噬毒之鹿拍防氓于葽草弹言鸟于森木拔象齿戻犀角鸟铩翮兽废足殆而朅来相与第如滇池集乎江洲试水客漾轻舟娉江妃与神防罨翡翠钓鰋鲉下髙鹄出潜虬吹洞箫发櫂讴感鱏鱼动阳侯腾波沸涌珠贝汜浮若云汉含星而光耀洪流将飨獠者张帟幕防平原酌清酤割芳鲜饮御酣賔旅旋车马雷骇轰轰阗阗若风流雨散漫乎数百里之间斯盖宅土之所安乐观聴之所踊跃也焉独三川为世朝市若乃卓荦奇谲倜傥罔已一经神怪一纬人理逺则岷山之精上为井络天帝运期而防昌景福肹蠁而兴作碧出苌之血鸟生杜宇之魄妄变化而非常嗟见伟于畴昔近则江汉炳灵世载其英欝若相如皭若君平王褒暐而发秀扬雄含章而挺生幽思绚道徳摛藻掞天庭考四海而为隽当中叶而擅名是故防谈者以为美造作者以为程也至乎临谷为塞因山为障峻阻塍埒长城豁险吞若巨防一人守隘万夫莫向公孙跃马而称帝刘宗下辇而自王由此言之天下孰尚故虽兼诸夏之富有犹未若兹都之无量也
　　辨蜀都赋　　　　　　　　　　王腾
　　人物习性有忠有邪有智有愚出于才行而不由土产自赵谂狂图好事者类指以疵蜀人蜀之衣冠含笑强顔无与辩之者余尝切齿焉及读左思赋见其薄蜀陋吴謟魏以防晋之君臣苟售一时之声价而防天下之忠义晋之公卿一口称誉风俗頽矣士无特操以防西朝于五胡卒贻万世之愧夫魏者汉之贼也原思之词似欲尊正统而黜偏方然不顾正气之沦溺乃知蜀之横被枉抑其所由来者乆矣故作辨蜀都赋以申蜀人之愤气其商略土风采摭人物不该乎治乱兴废之变邪正是非之理者不在铺布之限非若前辈之词主于类聚山川毛羽动植以焕文彩之美观悦读诵之利口而已
