王惺所先生集题辞 

此惺所王先生绪论也万历癸巳余与惺所同里选覩潜心大业湛研古道步趋濂洛超然闻见形骸之表相得甚驩余总角受书时闻先人遗训顾问津洙泗未涉其源而西面望洋庶几遇以旦暮及见惺所爽然自失儗为玄筏且以谓名理尽在是矣后十二年而主士高等出宰邢台余以使事过其邑邑人亟称令贤比于中牟单父私心谓方释经生业何以通达国体使民歌颂若是暨班荆道故妙契独觧能发前人所未发因手一编示余余才力蹇浅罔所当于持论然惺所辄以余言椒兰其味不之弃□惺所家弘农自月川西川二先生倡明大道而云浦孟先生相与羽翼而发明之三先生所为统一圣真模楷后进皆以实跋为本惺所克承厥绪一洗近世空寂支离之陋习自青衿时名已闻于海内于书无所不窥宇内事无所不究乡国天下士无所不友虽不肖如宗亦得从印可今读其集所谓道学不明往往薄躬行为无奇上者溺情训诂借口翼道不者以文人援玄虚要眇之谭自列于儒林而求闻于后此吾道之蠹则统承往圣启佑后学独赖躬行斯道之人耳旨哉盖自慱约以后其言不数数见也惺所于性命之学非惟知之实允蹈之故未尝自标门户视洒扫应对即为神化簿书米盐靡韭身心荑稗土苴糟粕煨烬极之引楯万族幎覆六幕不离匕箸坐而策之皆是物也记曰文以君子之辞实以君子之德此可以论惺所矣三门砥柱障百川而东惺所生于里灵灏锺毓欲吐于胎白□鬼其精神学力殆亦似之然师友乃渐不可谓无自故即事赋言墨传意肖纷纶满家不假雕刻皆以意识为宗俾天下晓然望斯道之帜躬行有淂于圣门殆庶几乎今天下异学繁兴譬诸鼎据混一为难若夫正人心息邪说惺所固有无言为言者予拭目为惺所竣矣 赐进士第翰林院检讨记注起居编纂六曹章奏眉山年弟王毓宗拜譔 

叙惺所王先生语录 

昔吾夫子忧学之不讲此后世学者所权舆也然其为之则不厌矣言信行谨慎乐相继以忘年矣颜之不惰曾之力行皆从实处树基区区口耳云乎哉□綗可以达天日新用以新命□熙敬止不出仁敬孝慈信纔□实地便属虚夸如捕风镂尘尽力奇巧竟无寸功故不讲不明何以入窍归宗不修而道涂听说亦终日指燕指越坐犹拘于斗室者耳孔门先求仁而圣贤问荅无一语涉于恍惚幻眇旨莫奥于克复请事只在视听言动之际非向实地修证非夫子所谓道所谓学也今人溺于词章之靡丽世网之牵缠者诚不足论即知向学者类以立会为名高讲学为般数群居聚谭缔众树交　　已数言道破矧树交中又生无限岐径心非尚絅功不日新学斯蠹矣众且诋之曰彼讲学者若是不□与于不仁之甚哉夫子又将何以为忧也我国初理学聿兴月川曹先生应奎聚而出好古力行实衍邹鲁濂洛之传嗣是西川尤先生云浦孟先生继起率以践屐笃实开导后人山斗洛阳惺所北面云浦而超悟更多实心实事躬修匪懈以力追古人不佞思藉高标风河汝乃今应期而出将为吾道顺风一呼挽世教之颓靡余波余暴奚止被于河汝者兹幸得惺所名言若干篇字字而会之根极理要抽关启钥而恳恳惓惓务崇实行真学问中入门究竟苐一义疚恶俱新声色浑化何莫非此充拓易不云乎履信思顺跖实而行冥念而合天人性命一贯之学也惺所游虎围而归引鼎新会以固同志今倡道宇内引掖斯世广于一乡雅有俱立俱达之思不佞为吾道庆而尤顾同志者以实心为之母徒曰弟子不能学则真修矣 苏岭山人陈大道撰

常惺惺所稿序 

学无实践譬之画脂镂氷即费日损力无益也文不载道譬之春华海枣即舒锦凌云无用也故有心学者斯有心得有心淂者斯有心声庸可诬哉常惺惺所稿者吾师惺所王先生所为文及师友门人问答之语也稿何以惺惺所名盖吾师自从尤孟两先生讲学后即题其读书处曰惺惺所而自号曰惺惺主人盖取常惺惺之法为不违仁之功其学问淂力处端在于是故吾侪录所□稿亦以是名焉□从先生游有年矣见先生之从事于惺惺法者亦有年矣惟有常惺惺法故不去伪而诚不撤蔽而明不戒肆而敬不警惰而勤不胜私而公于一切事务伦常根于性兮关于职兮者先生名因其固然若迫于不淂不然者然曾无一毫安排布置于其间而亦无一毫轶越渗漏于其中皆所谓惺惺之淂力处也故笔舌所到罔非从惺惺中流出如树木之有根底而非徒以枝叶竞秀如流泒之有源委而非徒以波涛见奇谓之曰以心学为心得以心得为心声亶其然乎吾侪诚由先生之心声以会先生之心学心淂则是编不啻觉路之金绳迷川之宝筏矣若以语言文字求之则世之为语言文字者岂少哉予尝妄拟先生之学曰主人翁在家照管一家事自是照管得着又妄拟先生之文曰自家屋里人说自家屋里话自是说得着是言虽俚而 训所见如此敢以质之同志者 雒阳张孔训顷首书

余少读书白云山中即闻召南有王惺所氏性地空灵辞葩璀璨每诵所为历试诸卷私心窃向往之万历癸巳奉 命选诸饩廪士肄业六馆余与惺所同为肖岩邹老师所拔因习公所为人盖粹然笃行君子也道本天人学说性命所云善言德行非耶私心逾归依之巳入长安并张见室年□同寓栢林寺中昕夕琢磨提诲谆恳遡□马之源抉伊雒之秘余亦恍然口有淂焉已为余先容谒云浦孟老师座下于时春风气嘘霁月澄清余庶几淂问津涯此公立人达人之雅念而大道为公之极思也嗣是余荐贤书而公寻成进士京国握手兰茫香浮可柰宦海南北仕蹃升沉从此心旌徒悬而踪迹离索屈指十余春秋矣今年各空谷寂寥跫然足声则公使者俨然持常惺惺所稿见示也余受而卒业名理名言开诱后学一臣热肠究绎大旨总之于洙泗源头蚤寻活水是尼父志仁之说也实践真修不矜口吻是尼父躬行之说也遗筌蹄为无用砭虚幻为空谭实实落落其于常惺惺之本体岂非缉熙不昧而明明德于天下不称止于至善也乎白鹿鹅湖两家藩篱可使立剖德性问学诸说争讼可使涣觧甚矣公之有功于后学深也余山径迷失蓬心蓊欝视公绝尘宁止瞠乎其后顾一念不忍自弃之心终不敢外于大冶朴樕数言仅以标愫幸公怜而教之 天启三年闰十月吉日刑部山东清吏司主事汝上年万夏禹英林宗甫顿首拜序 

常惺惺所稿序 

余家去分陕仅一衣带水分陕有吾师王惺所先生者当弱冠时即有志圣贤之学及闻云浦孟先生倡道雒涧负笈往依焉孟先生既为先生易旧字曰幼真而为之说且手题其读书之舍曰常惺惺所自是以来由诸生而贡选而登第宦游以至归老随在所居斋室悉摹之以寓观省以故同门张君敏孟辈辑所著作而付之梨枣因即命曰常惺惺所稿云稿凡十卷一二为序为记三则奏疏四则书札五六七八为状志为表传为赞为祭文为杂着九则日录则问荅则散言而诸体诗终其卷焉先后眉山王公光化陈公及雒阳汝上张夏两公各为之序顷者余始得而毕读之因喟然曰凡先生所为学本求仁功先主敬以羽翼往圣而启发后人者稿中具载之诸序颇能发明之余不必复袭其说苐以世之口实讲学者不曰刻脂镂氷则曰尘饭土羔□火谓其空谈毫无实用也窃观先生之宰邢襄所为谕士谕俗诸约先贤村霍路村诸咏则岂厌薄乎民事者于郎署所上议处禁卫严禁异端诸疏则岂不达时务者至参究凌应登张差二事皆一时所箝口结舌而先生言之不少却顾又岂观望首鼠而两端者至里居之日不为阖门养重而靡私问不为绝迹公府而靡私谒从口所发皆味言从身所应皆矩行使儇薄者归于厚而荡肆者化其僻缙绅士大夫既雅慕好先生即市侩里胥以至田夫野父亦能举先生以相赞述此于车□丸物范俗视宦游之日更深也有实用乎无实用乎当世以正学鸣者亟称新建王公余居恒以无论当日所以车□丸物范俗者即缚汀漳之山寇窆田州之士夷擒江藩之叛王已先生生而觧空谈无用之嘲矣虽然即汀漳田州江藩诸勋迹自新建视之亦枝叶耳余不揣犹欲以见于邢襄及郎署者为先生增泰华之高益河海之深且欲为先生鸣其不平则赘疣矣则骈拇矣非先生意矣 古虞门人李虞夔首拜撰
