　　提要
　　自鸣集六巻　　　　　别集类三【宋】
　　臣等谨案自鸣集六巻宋章甫撰考宋有二章甫一字端叔浦城人熈宁三年进士官至都官郎中着有孟子解义十四巻杨时为作墓志其文今载龟山集中一字冠之鄱阳人徙居真州自号易足居士即撰此集者也甫行事不槩见惟张端义贵耳集有云张冠之名甫有文集十巻多从于湖交游【案于湖乃张孝祥之别号】豪放飘荡不受拘覊淳熈间淮有三士舒之张用晦和之张进卿真之张冠之也据其所述可畧见其为人其以章为张葢刋本字讹耳又陆防入蜀记称乾道六年八月二十八日同章冠之秀才登石镜亭访黄鹤楼故址则甫葢尝应举矣其集不见于宋史艺文志文渊阁书目虽有其名而传本乆絶其得见于世者惟名贤小集拾遗所载湖上吟一首诗家鼎脔所载寄荆南故人一首而巳今捡永乐大典所收自鸣集诗句颇多其格律虽稍近江湖一而骨力苍秀亦具有研锻之之功观其别陆游诗有人生相知贵知心道同何必问升沉之句谢韩元吉寄茶诗有别公宛陵今五春渴心何啻生埃尘之句次韵吕祖谦见寄诗有山林旧约都茫茫忆君著书看屋梁之句是其所与酬赠者皆一时俊杰之士故耳濡目染尚能脱化町畦自成杼柚颇为不坠雅音谨裒次厘订析为六巻至杂説三篇以禅家机锋论道徳仁义之防援儒入墨是殆不得志于时者之肆志放言然其害理巳甚今故附其説于集末而特加紏正以着其失焉乾隆四十六年九月恭校上

　　钦定四库全书
　　自鸣集巻一　　　　　　　宋　章甫　撰五言古
　　送冯圎仲入蜀
　　我本山中人早信云无心巻舒随清风或作三日霖十年江湖游复识水之性东西因地形圆折初无定却来阅世人惟公多古风用舎不系怀畧与云水同还乗明月舟西沂弄云水云水逺送迎鸥鹭皆知巳我今与公别亦若云水然相望江湖万里同青天
　　送张寺丞
　　知己不易得相知贵知心我行遍四方公独知我深公如千丈松凛凛有直气我如松下草亦有傲霜意公分刺史符朅来淮上州而我倦逰者振衣从公逰花月醉管弦江山随杖履激水湔穷愁逢人赏佳句政成公入觐酒尽客语离叠鼓催发船不顾邦人悲邦人勿深悲宣室定前席公必念尔民还陈治安防吾君自神圣事君公不欺祝公但加餐慰此邦人思
　　送孙文辅尉新城
　　我来淮上居忽忽岁月过定交谁最久有唱君必和棊枰角胜负文字工切磋云山结伴逰风雨对床卧那知世涂隘但喜客愁破出处方乃殊离别情无江城失益友山县得良佐平生爱圗史晚岁职警逻枹皷常不惊鸡犬亦相贺强饭君勉旃功名看老大
　　隐静分题得静字
　　平生苦爱闲多病复便静毎为林下逰忽若醉而醒兹山昔尝登当暑毛骨冷碧霄环佩声古木旌幢影五峯如高人一一各秀颖俗驾难久留伫立但俄顷别来三十载往往梦清境从公得再来共此春日永凭髙惬幽思即事发深省然灯战枯棋临水防新茗兴阑下山去妙处心已领晓雨湿篮舆冲泥度前岭
　　古洗
　　古人不可见古物亦无几今朝双眼明乃识汉时洗规模简而质彷佛古君子年深土销蚀薄处仅如纸篆名可意逆依稀辨双鲤传寳今几代是阅人多矣我生颇好古摩挲不能巳悠然起遐思俗物一何鄙
　　纳凉
　　大热金石流日色赤可畏鄙夫先多病喘息仅存气饮氷不救渴肤汗那容睡秋风几时来不问黑貂敝林梢堕斜阳凉飔飒然至呼童汲井泉洒扫庭中地胡床得露坐始觉有生意老妻破甘儿女膝下戏鸣蝉虽强聒亦足代鼓吹坐久月照人金波正澄霁惜哉疲薾甚杯酒未敢置浮生能几何荏苒忽中岁四时更代谢寒暑不可避幻身岂长存中有不迁义且享今夕凉百年安用计
　　和仲韩见简韵
　　野人不爱名中岁益安命辛苦营一区婆娑只三迳那知钱有神但喜酒能圣翩翩赵公子时时枉轩乗诗词既清婉笔力亦端劲传家坚素守从仕伤直性语言生风波道路足机穽君犹沉下僚我亦抱多病时偷半日闲乐此一室静赏月巳失期登髙况当令风物渐凄江山更幽夐何时城南栖曳杖可乗兴
　　和韩丈寄崖簿大雅并见简韵
　　故乡未可归异县难久留海深蛟鳄怒安得乗桴逰贫贱岂天意自不曲如钩吾道未全穷三岁依贤侯小鸡随孔鸾顾影良可羞所喜味同防巳从清修迩来苦多病颇厌淮南州援琴操土风大类南冠囚今辰天气佳和风吹敝裘桃李竞顔色谁能分劣优空搅逰子心浩荡生春愁新诗忽到手盛意知难酬材生天地间小大固不侔词锋未易当如以楚伐邹玉季更多文情义还相投他年内集时尚记野人不【他年内集应呼我东坡与康国诗也】
　　雷
　　三伏久不雨今辰忽闻雷殷殷地中出轰轰空际来电光倐先驰掩耳恐不及阴云势相交销铄畏毒日上天行号令君子敬天威端坐毎修省整冠仍敛衣苍生苦炎热喘汗不可过寄声谢丰隆为起神龙卧
　　秋雨未已客怀不佳夜诵少陵遣兴诗至生涯能几何常在羁旅中三复增感毎字赋一诗
　　日月无停轮岁律今峥嵘夜长客无睡打牕秋雨鸣早寒逼衾裯儿女声吚嘤令人愧老厐团栾话平生借屋与僧隣局促仅如蜗悲秋怀抱恶头绪如乱麻老来怕读书隙风灯影斜感时思弟妹衰泪落天涯饥寒傍人门面目自可憎为善得长贫天理疑难凭狂谈取富贵易若阶而升吾实畏鬼神勿谓吾无能愁阴错昏晓北风振葭苇闗塞来悲鸿战塲多哭鬼人生真幻戏归尽同蝼螘贪夫患得失达者今无几逆旅故人少今秋风雨多禾头欲生耳农夫空悲歌饥肠望一饱嗸嗸同鴈鹅临风不把酒奈此菊花何门庭久沮洳蚯蚓升吾堂夜寒人閴寥蟋蟀鸣吾床屋头两株树叶落满僧廊骚人不须悲荣悴理之常诗吟不必工酒实性所爱少年少拘束日与尊罍对沉酣相呼和亦有嵇阮辈防合不可常故人今好在贫穷束缚人智慧无所施文籍纷满前不救寒与饥佳哉江上山归隐寜无时青山若有言尘土君自羁忆买京口船飘飘适荆楚览古吊兴亡亦有防心侣岁月能几何已作幽明阻筋骸幸无恙未害且逆旅今辰气佳万里来西风举头看浮云巳归天汉东野人还负暄帝泽流无穷心随鸿雁飞不离霄壤中
　　简王漕左司
　　人生天壤中孰不有所求小大虽不齐逐物常同忧士子文字间亦各从所爱取舎未必同于道均无害我生才甚下志与俗子殊思贤惄如饥覔句聊自娱惟公盛徳人君今乃行古富贵等云烟词翰敏风雨荆湖十五州玉节何煌煌门庭无杂賔读书清昼长抠衣愿从逰自视亦何有未知函丈地能着野人否公诚一顾之足以为我荣淮南有石田归趂春雨耕
　　分题得草果饮子
　　神农书本草有美生南州春华穗端垂彷佛芙蓉秋青囊贮嘉实璀璨安石榴香味极辛烈果中第一流磊落入盘饤和羮充肴脩温中与下气功用亦罕俦苞苴走四海药笼必见收吾老苦病暍淡味空频投作饮近得此选择知独优碧井沉银瓶斟酌得自由蔗浆巳觉俗茗粥良可羞乃知古圣人收拾靡不周日遇七十毒纵死夫何忧但吾赤子多疾苦庶有瘳大哉仁者心当与天地侔
　　同张季子游琅邪
　　枯藤久靠壁双脚殊踡局日长无往还午醉睡初熟谁与来剥啄一马挟两仆翩翩张公子得得访茅屋别来今十年髙谊惊絶俗功名付兄弟冠裳畏覊束多君殊不凡爱我常易足呼儿取酒斟草草具肴防剧谈连昼夜何止伴幽独结为山水游防蹇相追逐滁阳三日程山路颇诘曲好风吹雨霁发轫自幽谷至乐闻金声非丝亦非竹沉沉栢子潭神龙所栖宿旱干祷輙应云雨手翻覆路转醉翁亭酿泉潄寒玉堂堂四公像再拜敢不肃虽云斗大州盛徳继来辱凛然生气在遗泽被草木梵宫最髙处俯视山矗矗境圣人乃凡野僧才五六碑崖烂文字骊珠混鱼目老眼生花徃徃难尽读吾能领其要正自非匆促回途遂南首夜就汤泉浴清旷旧书房怀人今鬼录人生贵适意百岁风中烛所恨多别离安能似麋鹿兹逰讵难再后防尚可卜纪行因赠行归装慰羞缩
　　题玲珑
　　江月不受尘风竹俨相倚牕间读书人日用亦如是泾渭岂同流天地果一指客来问何如南山有云起
　　鹿角
　　呦呦鸣鹿羣适意在林薮泉甘草丰茂多力故能走一朝触祸机竟死谁人手割鲜登鼎爼刺血置杯酒犄角未易弃煮胶方术有汲江为浸渍七日去麄垢大三伏时煎熬必甖缶色作琥珀红坚重如琼玖补阳遂强骨轻健还堪久杀身虽可怜仁实存身后功能利生人死且垂不朽悠悠肉食辈亦媿此鹿否
　　慈湖矶
　　峭壁激湍流湍流作洄洑初无一饷风饱我蒲帆腹长笪寄怪石招呼争挽牵艰难仅寸进何翅如登天及矶日犹髙过矶日曛黒依约见渔村灯火照初夕舟师告疲劳夜泊投人家惊犬吠篱落寒生芦荻花
　　雨中舟出潮牐
　　官吏急星火驱民疏漕河城中三尺泥岁晚风雨多雨声来未巳清满幽人耳曲肱篷底眠船随夜潮起
　　卜宅
　　卜宅定何好庐山泉石竒夜梦五老人问我归何时山中有蕨薇可以疗子饥束书早归来长歌歌紫芝
　　次髙若虗韵
　　时事口慵説性灵诗可陶客中三日阴江头风浪高六年寄此邦鸥鹭皆心交烟波逈自如应笑吾生劳世路转逼仄旅鬓空萧骚既往若电梦所得真毫毛岁月解欺人木落江之臯勿谓吾甚衰得酒犹能豪旧逰固可恋归盟那得逃负暄依茆檐儿辈供爬搔
　　归自真州道中用前韵简诸故人
　　闲愁生旅怀薄酒安能陶篮舆赴归期厯厯山路髙风雨道傍憩倐然思故交官闲想聚首怀我登顿劳辩论杂嘲谑诗词兼雅骚人生几何时容易生二毛方今天子圣登用皆夔臯愿君各努力去矣追羣豪我生苦畏事屏处如逋逃纵复技痒存敢念麻姑搔
　　俞广文以诗定交次韵奉答
　　懒慢易招咎贫贱难相知客居二十年破屋临江湄遣闷岂无酒寄怀徒有诗慷慨思古人恨不生同时去秋君始来云雾容一披别去冐朔雪归来迎南飔载登君子堂爽气生芝眉剧谈两无碍并驾行通逵省闱冠多士泮宫聊养頥霄汉自兹始羽翮随所之门阑谢俗子逰戏惟毛锥髙吟起我病意欲相箴规琢句要须工一字亦有师逢迎君忽倦追逐吾敢辞人生俱寄耳况乃多别离
　　曾仲恭侍郎恵酒以偶有名酒无夕不饮为韵谢之
　　穷居少人事落落仍寡偶晨曦入茆檐和气生户牖拥褐还负暄看云更搔首闲哦陶令诗饥肠作雷吼力耕不逢年岁暮百无有小雨肥麦根何时得糊口百年倐已半辛勤为升斗赖古多若人千载可尚友古来称达人不愿身后名岂其未忘世隤然聊遣情我生本衰愞于时淡无营但得秫可酿何劳诗有声烹鱼多素书问字谁载酒寂寞淮乡居岁月亦巳久敢希陋巷乐但觉贫士丑知音幸有人此事可不朽丹阳贤使君髙谊今所无铃斋有余闲遥忆耕田夫满船送美酒盛事惊道途客来即留醉不用倩行酤一杯复一杯今夕知何夕蔀屋发光怪尊罍盈四壁老盆还旧观寒灯起佳色既醉使君酒亦饱使君徳逢人説项斯此道今则不使君如古人荐善惟恐后愿公登廊庙搜罗到岩薮使我击壤歌投老安陇亩使君开府来京口酒可饮公才真麴糵留滞已再稔江山助诗兴草木入题品氷霜寒正严为国防饔寝
