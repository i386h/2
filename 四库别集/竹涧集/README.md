　　病中偶述

　　闭戸东风又一春竹阴庭院静无尘花枝开好能撩我燕子来频不避人与酒絶交偏寂寞作诗为计有吟呻病余筋力难为礼拄杖人前自亦嗔

　　早春闻鸟

　　庭树朝来好鸟鸣似闻相唤不知名神机妙处应须玩昨夜春雷送一声

　　喜雨

　　亢阳当午天欲赤霹雳堕地坤隅坼楼头飒然风雨来席卷万瓦飞黄埃初如丝纷忽如注河翻海立蛟龙怒一洗燀热清八极顿令枯槁回颜色东园青可菹南亩足我糈虫鸟相和鸣花竹秀以舒吁嗟物我将无同炎凉燥湿苦乐之感相流通向来困惫一时起渴心灌溉生欢喜乃知茫茫大造非小儿阴阳推夺故应尔雨余更爱暮天碧月华满院凉于水

　　题仲兄书斋

　　先生志在当世者蔵修暂结青山社芳园半亩屋数椽月戸风檐共潇洒闭门却扫门更深抱膝时作南阳吟缃帙堆窓自披覧古琴挂壁谁知音爱花不与花争地红白盈阶鬬明媚拂天修竹起秋声卷露芭蕉滴晴翠坐我终日忘我疲况是兄弟相怡怡清谈小笑恣幽赏不数对床风雨时笑谈未了林月出欲去还留兴逾逸及此清光夜未央题诗素壁传他日

　　樗林散人为严处士赋

　　散木岂无用散人何可覊树来广莫野看到逍遥时朝市甘抛弃烟霞足护持高风千载上谁嗣考盘诗

　　甲子除夜

　　岁除那可守聊坐小堂寒椒酒春先到梅花兴未阑三旬明忽满几刻夜将残泰运回天地清愁谩自寛

　　游虎丘寺

　　红尘乆碍登山屐緑水今来载酒船塔顶风凄催客饮佛前灯暗照僧禅烟霞满目聊延伫名姓逢人莫浪传寺石不湏重试劔倚天千尺在吟邉

　　晓舟漫兴

　　溽暑逾三伏新凉生五更卧便衾布薄起觉帽纱轻树拥孤村秀霞横极浦明官舟两月路今日渴心清

　　访王主政秉衡不值仍小候

　　下马欵门门正关主人辰出酉未还三年见我儿童喜一鸟啼春院落闲竹所径来元不俗剡溪乘兴未容悭斜阳满地看辉尽更待花稍月一弯

　　送董内翰文玉归娶

　　乡闱识我同升日翰苑逢君及第时青镜朱颜年独妙蟾宫仙侣喜相期班衣着锦光生国玊树临风润照楣此去西湖梅正发好慿征客寄南枝

　　湘东曾生随兄山海闗俄归省亲赠此

　　闗海连宵雨湘江万里云离情与归思春絮正纷纷

　　经筵侍班

　　便殿春和辟讲筵叨陪法从五云邉迎风缥缈炉烟细映日徘徊寳扇圎鸡舌含香承旨近雉膏推食荷恩偏缉熈圣学微臣愿供奉朝朝力未愆

　　小斋遣兴

　　窓畔然香枕畔书小斋无事日长初冥心似欲迷庄蝶刮目犹堪校鲁鱼役役浮生聊自遣悠悠终古竟何如重门在告经旬闭一任傍人笑索居

　　送鲁内翰振之使安南

　　妙选出瀛洲乾坤此胜游节辞金殿晓诏下珥江秋日月开王政文章炳帝猷过门江汉路戏彩一停舟

　　赋秋海棠次马中舍文明韵

　　自是东风第一流不随桃李弄春柔驻颜谁与传丹诀倾国天留艶素秋清极傅霜肌不粟睡来侵月影还幽沉香亭畔如曾见莫道杨妃得与俦

　　送陈经府廷盛之河间

　　少年文采动时髦老去功名信所遭千里不须嗟骥枥一官聊复试牛刀行经易水秋风冷吟对瀛台海月高两地相望俱是客缅思乡社共松醪

　　丁卯春苦雪次陶掌科世和韵

　　顽阴春不退急雪日争飞竟夕明通牖乘风暗入帏不成宜麦瑞空假却蝗威漫拥袁安戸偏沾东郭衣燮调吾道拙斟唱此心违挟纩何当喜重裘幸免讥耽诗应益瘦多病故难肥拟订梅花约山阴访钓矶

　　书懐次李内翰宗易韵

　　两月东风冻未除手皴非乏豹为祛江乡旧社思浮蚁宾馆流年愧食鱼梦逐卢生成底事方抄陆氏有新书瀛洲仙侣劳相念几度诗来伏枕余

　　岸栁二首

　　岸栁緑丝丝春风不断吹昨来谁此别留得折残枝
　　湖上緑烟丝迎风不自持别离无意管解鬬好腰肢

　　安山闸与薛掌科子纯陈侍御廷禄酌栁下

　　栁岸追凉偶共来一罇分坐緑阴隈歌声欲彻云中去望眼真从天际回客路光阴三日阻诗家花鸟二公裁安山此集成佳会莫讶河枯闸未开

　　南旺夜酌呈薛掌科

　　月色波光满画船一罇相对且留连与君万里归来日却忆今宵作去年

　　夜抵沛县懐沽头章水部以道

　　孤舟尽日栁邉撑尚隔烟波十里程篷底一灯聴雨落枕邉三鼓盼天明笑谈可待还当别身世无慿忽自惊欲假诗筒报消息曲江幽梦恐先成

　　晓发清河

　　破晓发孤艇春和风不惊推篷烟树湿荡浆浪花明野色入淮逈河流带济清江湖万里去独抱古人情

　　京口即事

　　行尽东风爱日长京江风日似钱塘望知酒斾心先醉吟对花枝句亦香浪里金山妨短屐云邉鐡瓮倚危樯长安到此三千里更拟扬舲过楚湘

　　过玉山书懐

　　一别长安路五千几回云里望青天山从絶顶朝驱马水向高源夜放船蒲艾渐香临午日风霜饱厯任流年本来带得烟霞疾歌入沧浪意惘然

　　湖上叟

　　夕阳湖上叟短褐意悠哉放犊看儿戏打鱼携妇来茅檐和土覆竹牖傍云开万里懐乡客行歌首重回

　　南昌道中

　　客子去何适鸠声断复啼阴晴五月半天地大江西水远湘流合山重岭峤迷向来弧矢意不敢惮航梯

　　客夜独酌

　　为客有长夜出门无故人多情千里月送此一杯春大梦难醒世虚舟不定身颓然倚篷底犹自忆垂纶

　　夏夜

　　夏夜苦暄浊散步青山冈顾瞻閴无人明月流清光系舟杨栁根移床松桂傍黙黙坐复起仰天夜未央咄此万里身顾影徒彷徨水蛙聒靡厌草虫鸣暗伤蛙虫讵足恠蛇虺未可量银河淡欲没露下凉沾裳且复归我舟濯足临沧浪

　　题袁州分司壁

　　槐阴栢影几屏间倒映池盆石假山深院无尘人罕到苍苔有迹鸟初还薫风幸解吾民愠客子聊乘此日闲向晩诗成谁和者粉墙题处墨斑斑

　　萍乡马上

　　山衘斜日澹余晖匹马迢遥厯翠微雨过不知三伏在风来那得一尘飞舂残水碓香粳熟灌足溪田老芋肥物色满前吟未己又逢横笛牧童归

　　泊湘东

　　残暑侵陵困未苏短篷闲倚欲长吁不闻更漏怜村僻渐转星河见夜徂万里潇湘梧叶落百年天地野航孤思莼剩有秋风兴未卜归途是坦途

　　湘潭庆霞寺

　　野寺扪萝入荒凉岁月深偶寻逃暑处一洗入禅心共坐僧闲语眠听鸟细吟出门仍理棹落日半川阴

　　长沙秋日

　　六月长沙早见秋凉风吹雨暮飕飕不忧卑湿能为患稍喜炎蒸欲解雠贾傅庙荒侵薜荔屈原沙冷呌钩辀南来未觉江湖逺对此令人动别愁

　　长沙分司竹桂橘三植皆南州高品感而有作时虞逆瑾见害三首

　　两竿苍玊何年植潇洒当轩伴寂寥终日看来吟不足有时倚处醉微销影垂月下湘君佩声咽风前秦女箫彷佛故园池上见不知身在五湖遥
　　摇落方同楚客伤峥嵘一树好秋光谁从月窟移仙种独把天香殿众芳玉露濯枝丛浅碧金丹点蕋破轻黄广寒记得曾攀折屈指于今己十霜
　　青青江橘映秋堂浥露含风亦自香行处寻常供独玩霜前三百莫轻尝风流雅称离骚颂品味珍収本草方寄语湘灵须爱謢待充包贡入明光

　　山行柬刘侍御克温

　　两日山行罢咏诗不知诗句剩相随晓鸿摩月悲鸣处秋叶吟风欲堕时万里羊肠谙客况百年犬齿敢吾私聮镳幸有刘公干倡和时时妙色丝

　　龙阳道中

　　山城百里道多半在岧嶤岫湿含云气崖香长药苗野田荒作路古木仆为桥日暮西风急吟蝉抱叶飘

　　早行

　　早行乗月晩乗风残暑无烦苦若翁万里邉闗羸马底一秋光景乱蝉中长安北望云何极衡岳南来鴈不通独有寸心堪百炼乾坤随处是帡幪

　　澧州次李佥宪士修韵二首

　　下马西风鬂有沙偶同尊酒话通家春风二十一回梦几见锦官城外花
　　每嗟云树隔江天目断西山旧雪巅忽漫相逄又成别秋灯挑尽雨连绵

　　登澧州喜晴柬刘侍御

　　十日秋阴乍放晴又随騘马出孤城林霏滴翠时沾湿花气浮香远送迎软陌正宜聮辔语好山都作觅诗行村田处处逄収秫不用衘杯意己倾

　　石门道中

　　烟雾冥蒙匝四山紫骝駄我画图间不妨终岁长为客李蜀人予幼随宦其地纔到吟诗便属闲沉海鱼书无梦想穿云鸟道有跻攀折冲久佩宣尼训敢讶行邉渐入蛮

　　石门县

　　路尽石为门门中緑树村山峦当城郭风俗自乾坤水接桃源泒人通洞獠言晩厨供石鲫为倒菊花樽

　　慈利县【古崇山地】

　　百里慈阳地巉岩半不毛天垂南裔尽山揷北辰高种粟惟便火耕崖秪惯刀驩兠墓犹在掩鼻过秋蒿

　　九溪道中二首

　　秋思正凄迷千山入九溪路危频下马午过不闻鸡秃树疑人立荒禽作恠啼孤烟生远垒隐隐动昏鼙

　　又

　　青云万迭山山下水潺潺诘屈路难出崔嵬星可攀劳哉千里骥入此九溪蛮坦坦中原地长驱何日还

　　茅花岭

　　羣山絶顶茅花岭俯瞰蛮溪势转严下马几经难冐险排云直上最高尖茫茫白眼空人世霭霭青天作帽檐安得画归摩诘句满囊携去未伤廉

　　野牛闗

　　穷闗过野牛举目使人愁挂树猿通臂伤人蛇两头青天一线影翠壁四时秋决决寒泉洞遥通江汉流

　　宿九溪安福所

　　青山官舍雨踈踈扫地焚香客到初秋意萧条一年暮土风荒陋九夷居扶持病骨多将药爱惜清宵祗读书忽忆故园兄弟乐不知茰菊近何如

　　大庸山中大雾
