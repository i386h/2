　　提要
　　【臣】等谨案勿轩集八卷宋熊禾撰禾初名鉌字去非号勿轩又号退斋建阳人咸淳十年改名禾登进士第授汀州司戸防军宋亡隠居不仕筑洪原书院教授生徒着有四书标题易经讲义等书以阐明正学平生诗文甚富没后十亡八九其族孙孟秉类次成帙厘为八卷明天顺中禾六世孙斌为博罗主簿始刋行之其后板复散佚近时张伯行重刻其集有文无诗颇为缺畧此犹孟秉所编之旧夲也禾文章平正质实不以藻采见长而根柢六经自见夲色固非浮谈无根者所可防及原本前有元许衡序称禾晚修三礼通解将脱藁竟以疾卒嗣孙澍家藏遗稿传诸二世孙斌授梓以行求予序之云云攷至元为世祖年号而禾卒于仁宗皇庆元年自至元讫皇庆相距三十余年何以先称其疾卒且斌为明天顺时人尤非衡所及见其为依托显然葢其后人伪撰此文借名炫俗不知禾亦通儒固不必假衡以为重今特加删削庻不使其乱真焉乾隆四十三年六月恭校上

　　防交信録序
　　余读叠山谢公交信録序未尝不掩巻太息嗟夫同富贵易交也独患难之际生死慑其前利害怵其后当此时而不动心不易节则可以言交矣三代而降朋友道缺吾于东汉党锢诸贤取节焉惜不一变而至道也宋道学大明伊洛考亭之集盛矣一时借誉饰虚之人稍经炉鞴灰烬烟灭惟同门同志之士不以穷达皆能信其道守其学不变依然孔氏家法也流风所渐江左诸贤持节秉义九死不衰一时交防气谊皆班班可纪宇宙间三纲五常之道尚有所系而不坠者谓非道学之效不可也皇极在上师道立而教化明友谊敦而风俗美其效又当何如哉余在鳌峯山中与二三同志有求仁约方将尚友天下士邹孟子距夫子百余岁而已有无有乎尔之叹者无其人也韩宣子因过鲁见易象春秋遽曰周礼尽在是者以其有书也君归江东与我同志其以是告之梅花祖道晴日暄美山中诸友各有赠诗余前日夀君之章意已至矣族长麟斋翁又申言之輙序其所深望者如此我日斯迈而月斯征载歌此为续后序
　　孝经大义序
　　孔门之学惟曾氏得其宗曽氏之书有二曰大学曰孝经经传章句大略亦相似学以大学为本行以孝经为先自天子至于庶人一也尧典一篇大学孝经之祖也自克明峻德以至亲睦九族极而百姓之昭明万邦之于变大学之序也孝之为道葢已具于亲睦九族之中矣何也一本故也自是舜以克孝而徽五典禹以致孝而叙彞伦伊尹述成汤之德一则曰立爱惟亲二则曰奉先思孝当时人纪之修孰大乎是文武周公率是而行上而宗庙之飨下而子孙之保宗支庶蕃道化流衍且二千余年推其效必至于四海之内人皆亲其亲长其长一鳞毛一芽甲之微无不得所而后为孝之极致呜呼二帝三王之教可谓大矣孝经一书即其遗法也世入春秋皇纲纽解孔子伤之三复昔者明王孝治之言思之深望之切矣诚使天子公卿躬行于上凡礼乐刑政之具壹是以孝为本则斯道也固天性之自然人心之固有一转移间王道顾不易易乎惜也徒托之空言而仅见于门人记録之书也书存而道可奉虽不能行诸一时犹可诏诸来世今此经之可攷者不过汉艺文志而巳而其篇次则顔注古文二十二章孔壁所藏本也今文一十八章汉河间王所得顔芝本而刘向之所叅校者也要之出于汉儒傅防皆非曽氏门人所记旧文矣唐宗开元敕议意非不美而司马贞浅学陋识并以闺门一章去之卒启宗无礼无度之祸而其所制序文至以礼为外饰之所资仁义为后来之渐有不知所谓因心之孝者果何所因而又何自而萌乎学之不讲德之不修一至于此桓桓朱子特起南夏平生精力用功于易四书为多至此书则仅成刋误一编注释大义犹有所未及噫人子不可斯须忘孝则此为天子至庶人一日不可无之书章句巳明而文义犹阙顾非一大欠事乎葢尝有志彚集诸家传注以明一经而未果一日余友人新安胡庭芳挈其髙弟畨阳董真卿访余云谷山中手擕父书有孝经大义者取而阅之则其家君深山先生董君季亨父所辑也其书为初学设故其词皆明白易晓熟玩之则其间义趣精深又有非浅见謏闻所能窥者輙为刋之鳌峯书塾以广其传此岂惟学者修身齐家之要而有国有天下者亦岂能外是而他有化民成俗之道哉噫文公一用之于滕而四方草偃风动拓跋帝再用之于魏至使邻国君臣耸动愧悔而不自巳生于其心发于其政今攷二君行事皆班班有三代之风而况不止为滕魏者乎嗟夫此经之废葢千五百余年矣悠悠天壤人极未坠岂无以二帝三王之心为心者仁人心也学所以求仁而孝则行仁之本也语曰如有王者必世而后仁愚何幸身亲见之
　　史纂通要序
　　尧迄今年六十二甲辰矣自是而上年代不可攷意混茫初开洪水未有泄距洪荒之世未甚久也邵康节以元会运世推皇帝王伯自始开物至尧甲辰岁数甚阔大抵三皇之世邈矣其不可徴矣五伯权利又君子所不道有天下者行帝道而帝行王道而王二者而已外此无他道也葢尝纂皇王大纪一编托始帝尧元载下逮威烈王二十二年以为古今一部通史适吾友胡庭芳挟史纂通要来闽甚有契焉益相与讨论间亦蒙取节不废其言葢以人心道心为千万世治乱兴亡之机此河南程子所谓三代而上纯是天理三代而下纯是人欲之意而武夷朱子与永康陈氏所以反覆论辨而不置者也可谓确论矣予又为之踌躇却顾而有感于濓溪周子之言通书不云乎纯心要矣用贤急焉噫人主一心攻之者众或以声色或以货利投吾之欲千条万端大抵亲贤臣逺小人则阳明胜而天理用事此其所以治而兴也亲小人逺贤臣则隂浊肆而人欲用事此其所以乱而亡也尧舜禹汤文武之为君必有臯益伊莱姬吕之为臣皆以辅弼之职居师保之位其从容启沃赞导弥缝所以纳君当道而格其非心其功端不细矣后世人主正心之学不讲大臣格心之道不明其患葢本于无相业无师道上之人徃徃好臣其所教而尊德乐道之意未有其在下者又道失正传学昧素业至于事君亦苟焉富贵利达而已矣茫茫宇宙至以天下之大而无可相以天子之尊而无可师千五百年冺冺棼棼乱多治寡厥有由矣嗟夫天生斯民果何日而蒙至治之泽乎輙赞数语以广庭芳未尽之意庭芳复谓予曰为政在人取人以身修身以道修道以仁人主一心无取舍之极则焉知其贤而用之然则天理人欲消长之几正君子小人进退之候周子固不曰心纯则贤才辅贤才辅则天下治乎此愚之所以决然于人心道心为千万世治乱兴亡之机者非虚言也輙次其语使览是编者知所本而庭芳之言为不可易云
　　翰墨全书序
　　文公尝言制诰是君谀其臣表笺是臣谀其君然则近世士大夫以啓劄相尚无乃交相谀者乎书坊之书徧行天下凡平日交际应用之书例以启劄名其亦文体之变乎省轩刘君应季为此编命曰翰墨全书凡儒者操翰墨之文皆具非但启劄而已也其所选之文大畧变俗归雅返浇从厚去浮华尚质实多是先哲大家数而时贤之作亦在所不遗斯亦可谓之全书矣葢尝因是而论之文之体莫善于书诗君之于臣诰命而已即后世书疏之体也纪述之体如尧典禹贡等作后世纪志碑记叙事之文始于此问答之体如微子君奭等篇后世论辨徃复之文始于此若后世诗词一类则自虞夏赓歌而下备见于三百篇之风雅颂舍是之外亦未见有能易者至制诰笺表启劄胥为骈俪而后文始尽变矣甚者纪事实録之文亦为四六之体吟咏性情且尚对偶之工至于末流连篇累牍虽百千万言而辞不足果何日而可复返于雅厚质实之归乎且刘君此编自冠婚以至防祭近自人伦日用逺而至于天地万物凡可以寓之文者莫不毕备其亦异乎世之所谓启劄者矣其间俚俗之言异怪之説虽不能悉去亦必为之订其谬悮而究其指归刘君之用志亦可尚矣刘君力学善文与余讲学武夷洪源山中者十有二年所造甚深此特其防艺之末耳平磵伯氏为刋是书君之可传于后世者固不止是也輙书编端以谂观者
　　熊竹谷文集防
　　右从伯父竹谷先生熊公所作公平生所为文极多其杂著有敬思斋小稿则居竹谷时作也直方斋小稿则居平山时作也此二巻题曰直方斋小藁三葢兵后壁藏仅有此耳其所谓存十一于千百者矣公生平精力于三礼通义春秋约説中兴三朝通略用功最久又有大学中庸绪言经集传采小编史学提纲等编悉燬于丙子兵难三朝通畧葢欲续从祖左史公九朝通畧以成一家言庸斋赵公在史馆时尝以上闻有防下本郡抄写且荐公由布衣入馆同豫史事公力辞不就公少以礼记决科于礼学尤精博尝谓国家设科当以仪礼不当以礼记其着通义一书葢本文公先生义例而春秋约説亦本文公所论大防此二书于学者葢不无小补也丞相克斋防公左史存斋牟公皆欲以公经学荐闻公又贻书辞不就葢公平生潜心问学不求知闻蚤受学于节斋蔡先生与进斋徐公觉轩蔡公敬斋詹公思斋翁公为同门友所造既深自任之意弥笃后登西山真先生静斋刘先生之门尤见器重壮岁即弃科举一意精研经史之学惜乎书稿甫脱而公竟抱志以没可胜惜哉呜呼公之学进虽不得显于其时退犹足以善于其乡公奉先极其孝敬待兄弟极其友爱至于族婣乡党情义尤笃岁收不上三百石捐其半创敬思斋以训后进立孝永庄以赈饥贫后居平山甥馆岁必一归故居归必防集族中长稚宴饮馈遗渠渠欵欵各尽其懽见后进有向学者必谆谆诲诱不倦葢忠厚恺悌人也最谨于礼节冠昏防祭必用古礼旧居竹谷与云谷相望一时学行声称甚着逺方士友登其门必竭力馆谷之虽屡空不顾也禾年六岁就傅于敬思斋已防公器许今年入无闻壮志销落重惟我族自始祖尚书郭门公于唐中和间繇豫章入建诸书之泽四百年矣前辈凋零后进渺然未有甚于斯时者也公之学其谁能传之公之志其孰能知之癸巳秋从弟敬归自江东来相存问聚首旬日重整旧编相与感慨者良久重念甲戌侍族叔父复斋先生归自行都因论公遗事怃然兴念求所以嗣成公志者规模未就而复斋又没矣时事日殊年运而徃力不逮念又未知成就二父之志乎否也并书此以识
