　　璿玑玉衡赋【并序】
　　【臣】闻善言天者必有騐于人善言人者必有合于天良以景曜宣明聿着清宁之烈泰符炳煜式昭贞观之风乃上圣之休征为至诚之嘉命然其理存幽数极深微自非假象以推占讵易求端于髙逺仪器之陈其来旧矣盖古之言天者类有六家宣夜之术既失真传周髀之书殊多舛谬姚信昕天之说畧而未详虞嘉安天之编疎而难核以及穹天之论尤为荒逺之谈折衷五者之中防以浑天为允故观象之器至后世而弥精齐政之方自中天而已备夫以钦明之代濬哲之朝蓂荚纪辰复何劳于银箭凤凰司正亦无假于灵台而賔日东嵎饯阳西谷黄钟应律即稽金虎之祥清征鸣筩还候苍龙之宿立璿玑玉衡之制命羲和仲叔之官岂非敬天勤民之至抚辰熙绩之大者乎我
　　皇上昭受在躬灵承加毖五星防于营室六术起于甲寅犹乃储思穆清宅心渊谧法唐尧之钦若勅虞舜之时几爰诏微臣作为是赋敢摅牖见以当管窥
　　若夫鸿蒙肇辟象纬昭垂两仪定位七曜同规循蜚疏仡之代观云纪鸟之师靡不懿文洽礼班法树司辨物土于九域考星躔于四维然而制器尚象效崇法卑首重天事以授民时占斗纲以辨气浑神策以迎曦有隶首之综算有容成之测仪始甲子于大挠继平秩于重黎立璿玑以仰则发玉衡而上窥垂古今之彝宪表贞恒之范围于是秘器初呈畴人攸属体本弹丸形惟转毂饰以文珠筦之横玉珠四缀而长明玉在箫而旁烛内藏之以奥扄外运之以员轴任激水之推移抱运环之回复澄天镜于云阊络地维于鳌足寓机缄之至精亮幽遐之可暴敛八柱于寸眸收七衡于一瞩尔其积气髙浮清阳上荐廓尔无名浑然难见不推灵曜之符讵识干行之健日往而月来星回而斗建晃菱彩之朝飞皎桂轮之夕展耿木徳之东辉晥离情之南灿金水则附日而旋填星则居中而转道别疾徐度分近逺交防于辰归余以羡或主客之异形或宿赢之殊騐并陈寳籥之中俱入灵台之撰乃观悬象乃揆星精东西森列黄赤交萦晷齐景正极准端平察次舎运行之部颁作譌成易之经按分至启闭之节析晦明寒暑之程候咸池之绮旭迓金枢之素灵窥隠现于列宿酌盈虚于五行其摄乎坤轴也十有九万三千里而广其括乎乾象也三百六十有五度而赢其纪乎日月也九道四游而其行可迹其统乎星辰也三百二十而其官可名既宿离之不忒亦推步之有恒倬天章而文焕布轨道而衢亨穷精研于绵邈通幽赞于神明审圣人之制作逈度越于恒情彼夫协律纪祥司天辨叙七政有占三仪迭举寿昌剙智于良型平子覃精于绪并摅窥管之怀宁止挈瓶之数虽贻则于来今实晞风于曩古旷百世而同符端有候于
　　圣主
　　皇帝握璇极之天珍凝泰符之典瑞萃箕毕而类精荐圭璋而通气好以月从敬因日致仁育义裁礼和智睿法令信于隂阳图书炳于经纬五徳用征三光合契振采扬蕤连珠编贝犹复旦明干惕陟降非遥夙夜昭事鉴观不恌单心于宥宻升贶于穹霄故乃神符翕赩景祉宣昭硕麟获于灵囿庆云防于南郊绛河明而如镜甘露渥而如膏吐荣光以无际叶琼琯以长调流永年之懿轨建奕防之芳标启斯仪以遥度如宝歴之承尧末臣幸逢乎嘉运恍厠身于瞻云就日星陈旦复之朝
　　白鹦鹉赋
　　惟
　　皇朝之受命拱宝箓而凝正招女牀而鸾集巢阿阁而凤鸣雀啣书而入户鹄闻乐而翔庭虽异物之非宝实嘉祥之有征
　　圣心推而勿有瑞应显而自臻矧夫殊域懐仁遐方慕义贡賮献琛臣人享贽九译以通万里自致来旷世之灵禽登轶伦之珍毳蔚上国之羽仪焕中天之符瑞非邀福于干灵岂格遐于神契相彼鹦鹉产自山林往来丛薄栖跱髙深赋翩翾之微体禀聪明之慧心资善警而通解性能言而审音纵不离于飞鸟固迥异于凡禽至其含灵毓异拔萃挺竒金精叶兑火徳乘离丹咮焕发缃趾纷披练裳容裔缟袂参差亭亭素质皎皎清姿餐唯玉粒翔必琳枝朝搴芳于悬圃夕引吭于瑶池尔其山海穷荒区禺辽絶林箐幽遐岩阿嵽嵲巧舌如簧毳衣如雪顾影流妍徘徊自悦离尘氛而独立天既假其羽毛凌颢气而孤骞人谁信其髙防乃有罗氏张襦虞人播弋置以雕笼假之文饰凭玉柱以无尘弄银绦而一色虽难遂其幽栖幸未亏夫昭质辞僻陋之山陬远翺翔于京国遂乃觐光曜仪迁乔去卑栖禁园之琼树侍
　　君王之玉墀啸侣则白鹇妬影命俦则皓鹤増辉瑶堦月艳珠缀霜飞素娥韬彩青女敛霏长流连于
　　宸盼永供奉于仙闱亮陇坻之罕遘洵羽族之殊希若乃太液闳深上林邃窅物竞呈祥天宁爱宝白麟获于郊园皓鹿游于池沼不黔自黒鹤鹤者三趾而纷来莫赤匪狐濯濯者九尾而相扰皆鳞萃而星骈亦云臻而雾绕繄斯禽之入贡同效瑞于
　　圣皇讵无因而自集知有感而必彰按纬文之曩迹合图谍于前王隆大孝于尊养浃至仁于八荒故淳耀之符上格于颢穹之灵贶而精莹之气下通于物类之防茫于是玑明衡正干宁坤定一画启羲九苞仪舜洊至天休弥申景命非徒耳目之娱何止一人之庆诏九州而通节逖矣越裳之翚临七萃而观兵陋彼穆天之骏
　　天驷赋
　　于铄
　　皇家宪天作则景曜宣昭休光翔溢获硕麟于南郊贡白泽于西域育万物而効灵总八埏而禀式器车在御而结旌泽马充庭而伏轼揽符瑞之自天卜荣怀之在国是以干畜孕精房星降正功先于地用健叶于天行资风云之驾驭壮警跸之登闳承灵贶于太乙骋王路于由庚观其星纬献祥河祗耀徳校人是司圉师是职冬献夏颁执驹攻特为龙为騋齐足齐力十有二闲各分种色或皎皎兮虹摅或烂烂兮霞绚或鐡骊兮深黝或苍骐兮翠蒨或一举兮腾黄或将飞兮紫燕或蹀躞兮籋阊阖之云或缥缈兮曳吴门之练当夫锦障乍解金鞭未挥鸾停层汉鹄跱仙闱翩惊鸿之始集矫灵虬之欲飞及夫宝埒初开长楸并进五组云彯两骖风迅絶琼海于须臾登玉山于俄瞬况其骨应经文形符图牒力既足称徳尤殊别去似游龙回于萦雪疾不奔踶缓无衘橛行有磬控之安步中钖鸾之节蓄雄姿于婉心寓驯良于英决彼夫穆骏逾轮宛驹汗血騋牝兴咏于诗人駉牧登歌于往哲固未足以逾此威神义何容以较其优劣尔乃霜清紫塞气肃金天驾言出狩肄彼师田属车在后
　　銮舆在前千羣雾列万骑星骈于时礼用三驱材呈六种逸足争长竒毛矜宠珠勒无声纎尘不动攅绮谷而花明开幔城而霞拥披棽离之绣文列缤纷之锦幪追羲驭于炎洲骖箕风于仙陇展
