　　陆士龙集　　　　　　　别集类一【晋】
　　提要
　　【臣】等谨案陆士龙集十巻晋陆云撰云与兄机齐名时称二陆史谓其文章不及机而持论过之今观集中诸启其执辞谏诤陈义鲠切诚近于古之遗直至其文藻丽密词防深雅与机亦宜上下平吴二俊要亦未易优劣也隋书经籍志载云集十二巻又称梁十巻録一巻是当时所传之本已有异同新唐书艺文志但作十巻则所谓十二巻者已不复见至南宋时十巻之本又渐湮没庆元间信安徐氏瞻始得之于秘书省与机集并刋以行然今亦未见宋刻世所行者惟此本考史称云所着文词凡三百四十九篇此仅録二百余篇似非足本葢宋以前相传旧集乆已亡佚此特裒合散亡重加编缉故叙次颇丛杂如答兄平原诗二首其行矣怨路长一首乃机赠云之作故冯惟讷诗纪収入机诗内而此本误作云答机诗又緑房含青实四语及逍遥近南畔二语皆自艺文类聚芙蕖部啸部摘出佚其全篇故诗纪以为失题系之巻末但注见艺文某部此乃直标曰啸殆明人不学者所编又出诗纪之后矣特是云之原集既不可见惟借此以传什一故悉仍其旧録之始以存其梗概焉乾隆四十二年十月恭校上

　　逸民赋【并序】
　　富与贵人之所欲也而古之逸民或轻天下细万物而欲専一丘之欢擅一壑之美岂不以身胜于宇宙而心恬于纷华者哉故天地不易其乐万物不干其心然后可以妙有生之极享无疆之休也乃为赋云
　　世有逸民兮栖迟于一丘委天刑之外心兮淡浩然其何求陋此世之险隘兮又安足以盘游杖短防而遂往兮乃枕石而潄流载营抱魄懐元服一傲物思宁妙世自逸静芬响于永言灭絶景于无质相荒土而卜居度山河而考室曽丘蓊荟穹谷重深樷木振颖葛藟垂隂潜鱼泳沚嘤鸟来吟仍蔬圃于芝薄兮即兰堂于芳林靡炎颷以赴节兮挥天籁而兴音假乐器于神造兮咏幽人于鸣琴挹迥源于别沼兮飱秋菊于髙岑防玉泉以濯发兮临濬谷而投簪寂然尸居俨焉山立遵渚龙见在林凤戢遁绵野以宅心望空岩而凯入明发悟歌有懐在昔賔濮水之清渊兮仪磻溪之一壑毒万物之諠哗兮聊渔钓于此泽尔乃薄言容与式宴盘桓朝挹芳露夕玩幽兰区外而放志兮眷天路而怡顔望灵岳之清景兮想佳人于云端悲沧浪之浊波兮咏芳池之清澜鄙终南之辱节兮韪伯阳之考槃眄清霄以寄傲兮泝凌风而頽叹微载晏何思何欲漂若行云之浮泊若穷林之木咨有得之必丧兮葢怙宠之召辱彼贪夫之死权兮固遗生以要禄竦战兢而履氷兮只肃懐以临谷亮据鼎之无栗兮在颠沛其必渥是故夫形瑰者徴咎体壮者为牺虽明文而龙藻兮终俛首而受羁立修名于祸始兮登全生于戾阶资朝华之促节兮抱千载之长懐挤考终于逺期兮颠灵根而自摧殉有丧之假乐彼无身其孰哀美达人之览兮邈藏器于无为物有自遗道无不可万殊有同齐物无寡并家于国等朝于野荣在世而贵身兮神居形而忘我钦妙古之达言兮信懐庄而恱贾曾旣明于天爵兮何惙悲于人祸陋国风之遑恤同明哲于大雅乱曰乗白驹兮皎皎游穹谷兮蔼蔼寻峻路兮峥嵘临芳水兮悠裔槃丘园兮暇豫翳翠叶兮重葢瞻洪崖兮清煇纷容与兮云际欲凌霄兮从之恨穹天兮未泰咏欢友兮清唱和尔音兮此世
　　逸民箴
　　余昔为逸民赋大将军掾何道彦大府之俊才也作反逸民赋盛称官人以美宠禄之华靡伟名位之大寳斐然其可观也夫名者实之賔位者物之寄穷髙有必颠之溢美有大恶之尤可不慎哉故为逸民箴以戒反正焉
　　浩浩太素制为两仪经始君臣朝有俊弼野有逸民各有攸届而后品物有伦在昔后帝齐物达观赏不假爵教不示劝号諡莫设而生黎淳晏降迄中古黄象可观而唐文有焕乃雕乃藻大朴既散朴之既散万百熙心形为宠放神为利淫有翘者车命彼在林是故懐玉丧寳而被褐解襟恬亡智生与世或竞匪智无鉴匪心伊镜芒芒禹迹鞠为涂径惟是每懐偏彼反正正反于宠既尸干禄相协厥居而丰其屋禄之既尸刑为尔司屋之既丰丧家于宫故非据之灾戒之在凶人皆知存之为存而莫知得之为丧荣犹振颖坠若頽荒咎自専宠福在循墙是故保其安者常危而忘其存者不亡无休尔荣身寔亲名无谓尔崇神期好冲戒彼覆餗冐此栋隆慎微如显乃保身以终自古在昔逸民有作相彼宇宙方之委舄夫岂无不休而好是冲漠是故名利之灾至人攸恪谓予未信无宁监于桑霍天明既畏神道无亲善在求已庆由积仁无念尔本聿修厥淳执盈如虚乃反天真逸民司正敢告官人
　　嵗暮赋【并序】
　　余只役京邑载离永久永宁二年春忝宠北郡其夏又转大将军右司马于邺都自去故乡荏苒六年惟姑与姊仍见背弃衔痛万里哀思伤毒而日月逝速嵗聿云暮感万物之既改瞻天地而伤懐乃作赋以言情焉夫何干行之变通兮昏明迭而载路羡飞辔之逺御兮腾六龙于天步时赴节而渐流兮气移数而改度挥促节于短日兮振修防于长夜运悠忽其既周兮嵗冉冉而告暮变棘心之柔风兮滋丰草之湛露晖邈以峻服兮黄裳皓而振素于是颛顼御时防统官天庙既底日月贞观沦重阳于潜户兮严积隂于司寒日回天以灭景兮颷冲渊而无澜坚氷涸于川底兮白雪陨于云端普区宇之瘁景兮频万物之衰顔时凛戾其可悲兮气萧索而伤心凄风怆其鸣条兮落叶翻而洒林兽藏丘而絶迹兮鸟攀木而栖音山振枯于曾岭兮民懐惨于重襟寒与暑其代谢兮年冉冉其将老丰顔而朝荣兮髪粲其夕皓感芳华之志学兮悲时暮而难考逺图逝而辞懐兮密思集而盈抱羡厚徳之溥载兮嘉丰化之大造恨盛来之苦晏兮悲衰至之常蚤指晞露而怵心兮衍死生于靡草防时来之嘉运兮游上京而凯入委乗辂于紫宫兮剖金虎而底邑凭台光之发晖兮荷宠灵而来集望故畴之逈辽兮泝南风而頽泣长叹息而永懐兮感逝物而伤悲哀年嵗之攸往兮伊行人之思归结隆思于朝日兮缀永念于汜晖表寸隂而贞吝兮盻盈尺其若遗嗟我行之久永兮何归途之芒芒遵渚兮【一作于】盻川攸逝兮江湘处孝敬于神丘兮结只慕于惟桑瞻山川而物存兮思六亲而人亡问仁姑而背世兮及伯姊而沦丧寻余踪于空宇兮想絶景于遗堂悲山林之杳蔼兮痛华构之丘荒靖深情以遐慕兮思纒绵而懐楚涕垂颐以交頽兮哀凌心而洞骇神寻路而窘逝兮形嚬蹙乎其所心悠悠其若悬兮音既絶而复举悲人生之有终兮何天造而罔极仰悲谷之方中兮顾悬车而日昃百年迅于分嘘兮千嵗疾于一息咏大椿之万祀兮同蟪蛄于未识嵗难停而易逝兮情艰多而泰寡年有来而弃子兮时无算而非我祗生心于日顺兮虽呼翕其难假摄防生于逆旅兮欲淹留其焉可彼鉴寐之有时兮亦始卒之固然舒逺懐于千载兮怅同感乎中山鉴通人之炯戒兮惧晏平之达言啓贞心以自责兮览遗籍而问道亮爽鸠之既徂兮故营丘之有绍在吾侪之陋心兮岂取乐于东表茍长生而自得兮将奚待而有夭考大徳于天地兮知斯言之益矫
　　愁霖赋【并序】
　　永宁三年夏六月邺都大霖旬有奇日稼穑沈湮生民愁瘁时文雅之士焕然并作同僚见命乃作赋曰在朱明之季月兮反极阳于重隂兴介丘之肤寸兮坠崩云而洪沈谷风扇而攸遂兮苦雨播而成淫天泱漭以懐惨兮民嚬蹙而愁霖于是天地发挥隂阳交烈万物混而同波兮黄浩其无质雷凭虚以振庭兮电凌牖而辉室霤鼎沸以骏奔兮潦风驱而竞疾岂南山之暴济兮将冥海之蹔溢隠隠填填若降自天髙岸涣其无崕兮平原荡而为渊遵渚回于凌河兮黍稷仆于中田匮多稼于亿廪兮虚夙敬于祈年外薄郊甸内荒都城隂无晞景霤无辍声纎波靡于前途兮微津隔于峻庭纷云扰而雾塞兮漫天頽而地盈于是愁音比屋叹发屡省阳堂乏晖朗室无景望曽云之万仞兮想白日之寸胫感虚无而思深兮对寂漠而言靖毒甚雨之未晞兮悲夏日之方永瞻大辰以頽息兮仰天衢而引领愁情沈疾明发哀吟永言有懐感物伤心结南枝之旧思兮咏庄舄之遗音羡弁彼之归飞兮寄予思乎江隂天末以流目兮涕潺湲而沾襟何人生之倐忽痛存亡之无期方千嵗于天壤兮吾固已陋夫灵龟矧百年之促节兮又莫登乎期颐哀戚容之易感兮悲懽顔之难怡考伤懐于众苦兮愁岂霖之足悲云昙昙而叠结兮雨淫淫而未散晞朱阳于崇朝兮悲此日之屡晏劾丰隆于岳阳兮执赤松于神馆命云师以藏用兮绁乗龙于河汉照蒙泛之清晖兮炳扶桑之始旦考幽明于人神兮妙万物以达观
　　喜霁赋【并序】
　　余既作愁霖赋雨亦霁昔魏之文士又作喜霁赋聊厠作者之末而作是赋焉
　　毒霖雨之掩时兮情懐愤而无怿肃有祷于人谋兮反极隂于天律靖屏翳之洪隧兮戢太山之触石凌风絶而谧宁兮归云反而挥霍改望舒之离毕兮曜六龙于紫阁扬天步之剡剡兮播灵辉之赫奕于是朱明自皓凯风来南复火正之旧司兮黜后土于重隂夷中原之多潦兮反髙岸于嵩岑萎禾竦而振颖兮偃木竖而成林嘉大田之未坠兮幸神祗之有歆尔乃俯顺习坎仰炽重离兼明畅而天地恱兮羣生恱而万物齐鱼凌渊以増跃兮鸟望林而朝隮戢流波于桂水兮起芳尘于沈泥朱光播于瓮牖兮素景衍乎中闺天监作照幽明毕觌普厥有懽覃及四国翕万情而咸喜兮虽无获而自得灾未及害和斯有祥翼翼黍稷油油稻粱望有年于自古兮晞隆周之万箱原思恱于蓬戸兮孤竹欣于首阳隂阳交泰万物方遒炎神送暑素灵迎秋四时逝而代谢兮大火忽其西流年冉冉其易頽兮时靡靡而难留嗟沈哀之愁思兮瞻日月而増忧感年华之行暮兮思乗烟而逺游命海若以量津兮吾欲往乎瀛洲临仪天之大川兮凌懐山之洪波瞻増城之峻极兮仰蓬莱之嵬峩望王母于弱水兮咏白云之清歌虽嘉命之未锡兮将轻举于流沙振仙车之鸣鸾兮吐玉衡之八和托芝葢之后乗兮飱琼林之朝华修无穷以容与兮岂万载之足多
　　登台赋【并序】
　　永宁中参大府之佐于邺都以时事巡行邺宫三台登髙有感因以言崇替乃作赋云
