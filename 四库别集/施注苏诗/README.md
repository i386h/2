　　钦定四库全书　　　　　　集部三
　　施注苏诗　　　　　　　　别集类二【宋】
　　提要
　　【臣】等谨按苏诗四十二巻宋施元之注附东坡年谱一巻王注正譌一巻补遗二巻则邵长蘅等续编也元之字徳初吴兴人陆游作是书序伹称其官曰司諌其始末则无考矣其同注者为呉郡顾禧游序所谓助以顾君景繁之赅洽也元之子宿又为补缀书録解题所谓其子宿从而推广且为年谱以传于世也呉兴掌故但言宿推广为年谱不言补注与书録解题不同今考书中实有宿注则吴兴掌故漏矣嘉泰中宿官余姚尝以是书刋板竟縁是遭论罢故传本颇稀世所行者惟王十朋分类注本康熈已夘宋荦官江苏巡抚始得残本于蔵书家巳佚其巻一巻二巻五巻六巻八巻九巻二十三巻二十六巻三十五巻三十六巻三十九巻四十荦属长洲邵长蘅补其阙巻长蘅撰王注正譌一巻又订定王宗稷年谱一巻冠于集首其注则仅补八巻以病未能卒业更倩髙邮李必恒绩成三十五巻三十六巻三十九巻四十巻荦又摭拾遗诗为施氏所未收者得四百余首别属钱塘冯景注之重为刋板乾隆初又
　　诏内府刋为巾箱本取擕既便遂衣被弥宏元之原本注在各句之下长蘅病其间隔乃彚注于篇末又于原注多所刋削或失其旧后查慎行作苏诗补注颇斥其非亦如长蘅之诋王注然数百年沉晦之笈实由荦与长蘅复见于世遂得以上邀
　　乙夜之观且剞枣梨夀诸不朽其功亦何可尽没欤乾隆四十三年八月恭校上
　　总纂官【臣】纪昀【臣】陆锡熊【臣】孙士毅
　　总　校　官【臣】陆　费　墀
