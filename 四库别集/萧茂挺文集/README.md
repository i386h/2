　　钦定四库全书　　　　　集部二
　　萧茂挺文集　　　　　　别集类一【唐】
　　提要
　　【臣】等谨案萧茂挺文集一巻唐萧頴士撰頴士颍川人茂挺其字也梁鄱阳王之裔世系俱载其赠韦司业书中开元二十三年举进士对策第一天寳初官秘书正字以搜括遗书淹久不报劾免寻召为集贤校理忤李林甫调广陵防军韦述荐为史馆待制又忤林甫免林甫死调河南府防军安禄山反頴士走山南源洧辟掌书记后为扬州功曹防军复弃官去遂客死于汝南事迹具唐书本传頴士尝作伐樱桃赋以刺林甫唐书本传讥其褊而晁公武读书志则称其每俯临于萧墙奸回得而窥伺句为知防先见唐书贬之为非今考颖士当禄山宠盛之时尝与栁并策其必反既而言验乃诣河南采访使郭纳言献策守御纳言不能用禄山别将攻南阳山南节度使源洧欲遯頴士力持之乃坚意拒贼永王璘尝召之不赴而与宰相崔圆书请先防江淮之乱既而刘展果叛其才畧志节皆过于人不但如晁氏之所云文章根柢固不仅在学问之博奥也頴士与李华齐名而頴士尤当代所重李邕负一代宿望而进芝草表假手頴士则其推挹可知唐志载頴士游梁新集三巻文集十巻宋志仅载文集十巻而游梁新集已佚此本前有曹溶名字二印葢其所藏仅赋九篇表五篇牋一篇序五篇书五篇史称其与崔圆书今集中不载书録解题所云栁并序今亦佚之乃后人抄撮文苑英华唐文粹诸书而成非复十巻之旧矣然残膏賸馥犹足沾溉正不必以不完为歉也乾隆四十四年三月恭校上

　　至日圆丘祀昊天上帝赋
　　政教之始莫重乎郊祀郊祀之先莫尊乎昊天是以前王垂之于典训后帝奉之以周旋其事大矣其仪盛焉日之至也所以明气之至丘之圆也亦以象天之圆于是致斋于宫合乐于律羣有司肃肃以儆戒百执事干干而庄栗牲用骍犊以贵诚酌用酒以明质岂但爱人而尊祖盖欲报天而主日天子乃乗玉辂驾苍虬搢方珽服大裘率九仪之卿士从五等之诸侯旌旗露巻冠盖云浮展国容于御路行大礼乎郊丘百役既备司仪辨位剑佩纷纭以陆离钟鼓铿訇而沸渭君明其义臣敬其事执鸾刀以啓毛奠苍璧以为贽爵一献而上下胥悦乐六成而神祗麏至后乃取血膋陈玉币寘于薪樵之上燔于嘉坛之际飞燎烟于太清合萧光于上帝是以神降我福人懐我惠时罔凶荒物无疵疠致洪化于仁寿岂不由肃敬于大祭客有旅逰函关钦兹至道观祀事于国典仰明灵于有昊敢陈舆颂式播造颂曰日南至兮既望祀太乙兮圆丘上万斯年兮承天贶
　　爱而不见赋
　　嗟乎或爱之而不见者有之矣何必周秦异代夷夏殊轨阻严城之九重限方舟之一水茍时事之多怨故人遐而室迩关山起于足下堂上逺乎千里耸专注之目成遽将逝而复止诗人所以思婉娈而搔首赋城隅之有俟吁不得其已也惟夙昔之良防梦佳期于北方叙渤澥之三山吸流霞之景光含芳词以况予云惠好之不忘愿报义于永日陪游宴于帝乡广莫忽而号怒鲸波汹而腾张俄惊魂以辍寐问穷髪之茫茫将揭厉以复从骇风涛之匪量思投躯以靡吝抚遗体以兢惶晨切切以凄凄夕屏营以彷徨追前欢之甫迩叹此恨之攸长于是收神返虑澄澹静黙防然就寝兀若无识冀良宵之复遇希旧游之可即徒有赖兮且未克忧深沉兮萃胸臆风兮兮思君子兮何极
　　滞舟赋
　　摄提嵗拂衣海岳应调函洛咏佩服之臯兰美絷维之塲藿徴良图以趋事窘中道以摧落昔谬价于当年今后来之不若众飞下以抵余矩枘而规凿悲介直之不可媒想云林以自托眷眷离忧行行独愁邅我车而北上朅吾道以东游爱迟迟之暮春登泛泛之轻舟过岩邑以信次防荥波之下流于时丙丁守位恢台肇节朱云四腾瑶草半歇景冲冲而炽旱风翳翳以敲热赫中潬之平沙渗通用而殆竭则有危樯巨舸长艥广艘龙翼锦轴雀颅方艚材木兰兮竹箭纫齿革与羽毛顿修笮于回塘骈曲岸以戢篙于是迅拏轻防河舠汉艇乗时泝洄赴利驰骋混渔商而沸杂期数日而俄顷事也时哉咸适其才于嗟大舰安得而来借如三江五湖之漫礐石飘沙之汩淈望赤岸以天低临清波而景没峻舻冲涛以直透髙防云巻而上彗朝发乎荆衡夕止乎扬越晷刻千里之外一何去留之倏忽彼斗筲钟鬲之余捷迳趋时之末曾压溺以不暇亦何知于嵗月材防则致逺而自覆量大则俟时而可贵茍或喻于穷通又奚分于器类运之来也贾长渊髙视于三台谋不用焉梅子真近辞于一尉吾将敛防以饮气覩维舟而歔欷
　　登临河城赋【并序】
　　亡舅孝亷元君才髙位下一命屈临河尉寻遭风瘵有加无瘳忧悒迄逾一纪故不复仕而风标俊杰文史清隽则君所着别传详矣舅于予有教授之恩只词片字皆资训诱既而射策桂林校书芸阁首为知已名称舅氏之力也天寳元年秋八月奉使求遗书于人间越来月届于临河之旧邑览物增懐然有赋羊昙是日独吟零落之篇周翼终身宁亡吐哺之爱词曰
　　登孤城兮见河水之漫漫城有隍兮水有澜歘翻覆兮无端俯崇墉兮心酸心断絶兮河水之干借如韩伯懐恩羊昙念昔追北渚之曩饯叹西剡之忽觌曾一顾而不忘况仁深与宻戚也惟佩觿之弱嵗荷哲舅之矜怜枉月旦之殊品超等夷而独偏过虽小而必诫善无防而不悛备润身之黼闻染翰之蹄筌岂期文嗣作者价参时贤谬昆墟而比玉滥蓬岛而懐铅匪舅徳其焉尔谅师资乎在焉痛才髙而位下悲道悠而运促甫一命于兹城蹇无媒兮窘束傃层飚而坠羽凌永路而倾轴悼晋竖之行深哀秦良之莫赎昔自公而暇豫陪作赋于兹楼懐一纪以如昨怆今晨而独游俯萧条之邑里对零落之徂秋旧馆凄其在目长川逝而不留徒临风而挥涕孰知夫四望可以销忧者也
　　庭莎赋【并序】
　　天寳十载予以史臣推择待诏阙下僻直多忤连嵗不偶未选叙求参河南府军事府尹裴公以予浮名枉顾遇焉而尹之外姻或绾纪纲之局怙势矜权求府僚降礼于已予清慎自守不能附防爰逝我陈嫌怒遂搆凡同官多贵逰后戚酒食之防丝竹之娱无间旬朔予人质鄙野雅不之好常愿鸥鸟为俦江海是处往嵗久游剡中将遂终焉朝防迫召故不获展着白鹇赋以寄斯意至是鬱悒弥用增想防堦之下蹊有莎草故参军宋之问徙于伊川而植焉结根五纪緜羃庭际广累万歩髙树十余间以杂果隂蔽其上俗吏往来必凌践之叹其禀山野之姿而托非其所以就窘迫因而赋曰厌公门之窘束玩纎草于兹亭奚卑弱之斯极岂露之愆灵尚含和以顺时随春夏之凄清轩房洞启广堦脩直槐杨蔽亏桃李对植横层隂之宻缀繁英之翕赩既髙低以异姿亦浓淡而殊色胥徒牒诉杂沓乎其侧游尘浮烟防翳而不息虽萧飒以自得亦喧卑而见逼宜夫坐莽浪之野带江湖之涘托根山阿揺颖绿水芊绵靃靡连亘乎十数里何推迁而连防缪产莳于庭隅忧好尚之倾夺见芟夷于难除既无心于宠辱又奚议于亲疎承滴沥之甘润蔽衣衿之曳娄虽为幸于斯日谅禀性之云殊闻哲王之布泽逮萧苇而霑铺茍一类而失所犹纳隍之在予矧皇穹之播气陶庻彚于灵枢曷兹卉之攸托惨终年而莫舒吾将徴宰物之至理聿归问于虚者焉
　　莲蘂散赋【并序】
　　予同生继夭戚所萃已未嵗夏六月旅寄韦城忧伤感疾肿生于左脇之下弥旬不愈楚痛备至友生于逖张南容在大梁闻之以言于方牧李公公予之旧知也俯垂惊嗟逺致是散题曰莲叶合之以苏用附肿上又覆以油帛以羃之其瘳如洗一夕复故感恩叹异于以赋焉
　　彼散维黄曰莲之蘂有轻其质如雪伊洒君子赉焉厥疾遄已揆艰疾之永戚矧覊孤之逺情谅积悲而成爰彼肿而斯婴遘徂夏之赫曦蹇忧虞于此城堆以壅蓄介于腰腹如烟斯焮如虿斯触靡宵靡昼莫获偃伏亦既浃辰寘予于毒惴然其恐兮如集于木幸于张之乆要干至贵而为言感知已于名公降逾涯之厚恩旅信宿以问至致良散以斯存于是滫以苏膏幕以油帛兹焉涂附未始竟夕有瘳如神兮厥痛斯涤彼挂防而奔驷曾莫速乎灵迹虽兼金而制锦岂厥价而能敌异哉讨竒篇于绿帙搜秘巻于青囊奚要术之备列独无闻于此方茍佳名之是徴乃菡萏之余芳原夫托根清泚敷蘤馨香宜蠲秽而荡邪救吾人之疾于以用之终然允臧恺悌君子德音不忘昔禽蛇之见拯尚有答于隋哙矧圆首之为贵聿称灵于覆载惭力防而施重惧殒坠于酬戴莲之蘂兮永以为佩
