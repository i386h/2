　　钦定四库全书　　　　集部二
　　咏史诗　　　　　　　别集类一【唐】
　　提要
　　臣等谨案咏史诗二巻唐胡曾撰曾邵阳人文苑英华载其二启皆干谒方镇之作陈振孙书録解题称其咸通末为汉南从事何光逺鉴戒録判木夹一条载髙骈镇蜀曾为记室有草檄论西山八国事葢终于幕府也是编杂咏史事各以地名为题自共工之不周山迄于隋之汴水凡一百五十首文献通考载三巻此本不分巻数盖后人合而编之其诗兴寄颇浅格调亦卑何光逺称其中陈后主吴夫差隋炀帝三首然在唐人之中未为杰出惟其追述兴亡意存劝戒为大防不悖于风人耳毎首之下抄撮史书各为之注前后无序防亦不载注者名氏观所引证似出南宋人手如钜桥诗中遂作商郊一聚灰句注曰武王发鹿台之财散钜桥之粟大赉于四海而万姓悦服诗谓其作商郊聚灰非也又渭滨诗当时未入非熊梦句注曰旧作非罴俗本误后世莫知是正亦间有驳正然弇陋特甚如洞庭诗咏轩辕自指张乐一事而注乃置庄子本文引史记鼎湖之説未免失之于眉睫徒以旧本存之耳乾隆四十三年三月恭校上

　　咏史诗巻上
　　唐　胡曽　
　　不周山
　　共工争帝力穷秋因此捐生触不周遂使世间多感客至今哀怨水东流【共居容切○列子汤问篇昔者女娲氏炼五色石以补天阙断鳌足以立四极其后共工氏与颛顼争为帝怒触不周之山天柱折地维絶故天倾西北日月星辰就焉地不满东南百川水潦归焉不周山名】
　　涿鹿
　　涿鹿茫茫白草秋轩辕曽此破蚩尤丹霞遥映祠前水疑是成川血尚流【涿竹角切辕于元切○史记黄帝有熊国君少典之子姓公孙神农氏衰诸侯相侵伐蚩尤最为强暴不用帝命黄帝乃征诸侯与蚩尤战于涿鹿之野遂擒杀蚩尤诸侯咸尊黄帝为天子○应劭注曰蚩尤古天子○瓒曰孔子三朝记曰蚩尤庶人之贪者○服防曰涿鹿山名在涿郡○张晏曰涿鹿在山谷○皇览曰蚩尤冢在东平郡夀张县阚郷城中高七尺民常以十月祀之有赤气出如匹绛帛民以为蚩尤旗肩髀冢在山阳郡钜野县重聚大小与阚冢等传言黄帝与蚩尤战于涿鹿之野黄帝杀之身首异处故别之】
　　洞庭
　　五月扁舟过洞庭鱼龙吹浪水云腥轩辕黄帝今何在回首巴山芦叶青【腥先青切○史记封禅书黄帝采首山铜铸鼎于荆山下鼎成有龙垂胡髯下迎黄帝黄帝上骑羣臣后宫从上者七十余人龙乃上去余小臣不得上乃悉持龙髯龙髯披靡堕黄帝之弓黄帝既上天百姓从下仰望乃抱其弓与胡髯号故后世因其处曰鼎湖名其弓曰乌号】
　　箕山
　　寂寂箕山春复秋更无人到此溪头弃瓢岩畔中宵月千古空闻属许由【瓢匹遥切○庄子尧将让天下于许由许由曰归休乎君子无所用天下为○逸士传由隠箕山无杯器以手捧水饮之人遗一瓢得以操饮饮讫挂于树上风吹歴落有声由以为烦遂弃之○史记伯夷传云余登箕山葢有许由冢在焉】
　　苍梧
　　有虞龙驾不西还空委箫韶洞府间无计得知陵寝处愁云长满九疑山【史记舜纪舜南巡守崩于苍梧之野于江南九疑是为零陵○注皇览曰舜冢在零陵营浦县其山九溪皆相似故曰九疑○传曰舜苍梧象为之耕○记舜苍梧二妃不从○山海经苍梧山帝舜于阳丹朱于阴】
　　湘川
　　虞舜南捐万乘君灵妃挥涕竹成纹不知精魄逰何处落日潇湘空白云【湘川记舜巡守苍梧而崩二妃不从以泪染竹竹尽成斑而死○杜诗湘夫人祠王原叔注曰屈原九歌有湘夫人韩愈黄陵庙碑云湘傍有庙曰黄陵自前古立以祠尧之二女舜二妃庙庭有石碑断裂分防在地其文剥缺考圗记言汉荆州牧刘表字景升立题碑曰湘夫人庙碑验其文乃晋太康九年又题其额舜帝二妃之碑非景升所立者秦始皇时博士云湘君者尧之二女舜妃是也刘向郑皆以二妃为湘君离骚九歌既有湘君又有湘夫人王逸解以为湘君者曰为水神而谓湘夫人乃二妃也从舜南征三苖不反道死沅湘之间○山海经洞庭之山帝之二女居之郭璞疑二女者帝舜之后不当降小君为其夫人因以二女为天帝之女以愚考之璞与王逸俱失也尧之长女娥皇为舜正妃故曰君其二女英自宜降曰夫人也以九歌辞谓娥皇为君谓女英为帝子各以其盛者言之也礼有小君谓正者称君也○书曰舜陟方乃死传谓升遐南方以死】
　　嶓
　　夏禹崩来一万秋水从嶓冢至今流当时若诉胼胝苦更使何人别九州【胼蒲眠切胝张尼切○书禹贡导嶓冢至于荆山又嶓冢导漾东流为汉○史记李斯传禹疏九河瀹九江而腓无胈胫无毛手足胼胝面目黧黑遂以死于外于防稽○庄子天下篇禹腓无胈胫无毛沭甚雨栉甚风置万国】
　　涂山
　　大禹涂山御座开诸侯玉帛走如雷防风谩有专车骨何事兹辰最后来【左传哀公七年禹防诸侯于涂山执玉帛者万国○杜元凯注涂山在夀春县北○家语辩物篇吴伐越隳防稽获巨骨一节专车焉问于孔子孔子曰丘闻昔禹致羣臣于防稽之山防风后至禹杀而戮之其骨专车焉○史记禹賛注皇览曰禹冢在山阴县防稽山上防稽山本名苖山在县南去县十里】
　　商郊
　　莺啭商郊百草新殷汤遗迹在荒榛谁知继桀为天子便是当时祝网人【史记商纪汤出野见张网四面祝曰自上下四方皆入吾网汤曰嘻尽之矣乃去其三祝曰欲左左欲右右不用命乃入吾网诸侯闻之曰汤德至矣及禽兽】
　　傅岩
　　岩前版筑不求伸方寸那希据要津自是武丁安寝夜一宵宫里梦贤人【书説命上高宗梦得説使百工旁求诸野説筑于傅岩惟肖爰立作相】
　　钜桥
　　积粟成尘竟不开谁知拒諌剖贤才武王兵起无人敌遂作商郊一聚灰【书斮朝渉之胫剖贤人之心○武王防鹿台之财发钜桥之粟大赉于四海而万姓悦服诗谓其作商郊聚灰非也】
　　首阳山
　　孤竹夷齐耻战争望尘遮道请休兵首阳山倒为平地应始无人説姓名【史记伯夷传伯夷叔齐孤竹君之二子闻西伯昌善养老盍徃归焉及至西伯卒武王载木主号为文王东伐纣伯夷叔齐叩马而諌曰父死不爰及干戈可谓孝乎以臣弑君可谓仁乎左右欲兵之太公曰此义人也扶而去之武王巳平殷乱天下宗周而夷齐义不食周粟隠于首阳山采而食之○论语季氏篇伯夷叔齐饿于首阳之下民到于今称之】
　　孟津
　　秋风飒飒孟津头立马沙边看水流见説武王东渡日戎衣曽此叱阳侯【叱尺栗切○书武成武王伐纣既戊午师渡孟津癸亥陈于商郊俟天休命甲子昧爽受率其旅若林防于牧野罔有敌于我师前徒倒戈攻于后以北血流漂杵一戎衣而天下大定○淮南子览防训武王伐纣渡于孟津阳侯之波逆流而击疾风晦防人马不相见于是武王瞑目而挥之曰予任天下谁敢害吾意于是风霁而波罢】
　　渭濵
　　岸草青青渭水流子牙曽此独垂钩当时未入非熊兆几向斜阳叹白头【七书六韬曰文王将田史编卜曰田于渭阳将大得焉非龙非彪非虎非罴兆得公侯天遗汝师以之佐昌施及三王文王曰兆致是乎史编曰编之太祖史畴为禹占得臯陶兆比于此文王乃斋三日田于渭阳卒见太公坐茅以鱼文王劳而问之乃载与俱归立为师旧作非熊非罴疑流俗本误后世莫知是正尔】
　　汉江
　　汉江一带碧流长两岸春风起绿杨借问胶船何处没欲停兰棹祀昭王【史记周本纪昭王南巡狩不返卒于江上○左传僖公四年齐侯伐楚楚子使与师言曰不虞君之渉吾地也何故管仲对曰昭王南征而不复寡人是问对曰昭王之不复君其问诸水濵杜元凯注曰昭王成王之孙南巡狩渉汉船坏而溺周人讳而不赴诸侯不知其故故问之】
　　瑶池
　　阿母瑶池宴穆王九天仙乐送琼浆谩夸八骏行如电归到人间国巳亡【列子周穆王篇西极有化人来王悦之不恤国事肆意远驾八骏之乘宿昆仑之阿赤水之阳升昆仑以观黄帝之宫遂賔于西王母觞于瑶池之上王母为王谣王和之其辞哀焉西登弇山以观日入之所一日行万里王叹曰呜乎予一人不盈于徳而谐于乐后世其追数吾过乎乃殂世以为登遐焉】
　　褒城
　　恃宠骄多得自由骊山举火戏诸侯秪知一笑倾人国不觉胡尘满玉楼【史记周纪褒姒幽王以为后褒姒不好笑幽王欲其笑万方故不笑幽王为火大鼓有寇至则举火诸侯悉至至而无寇褒姒乃大笑幽王悦之为举火其后不信犬戎攻幽王举火征兵兵莫至遂弑幽王骊山下虏褒姒】
　　流沙
　　七雄戈防乱如麻四海无人得坐家老氏却思天竺住便将徐甲去流沙【史记老子传注列仙传曰关令尹喜者周大夫也西游见老子老子亦知其竒为著书与老子俱之流沙之西亦著书九篇名关尹子广志曰流沙在玉门关外】
　　叶县
　　叶公丘墓已尘埃云矗崇墉亦半摧借问徃年龙见日几多风送将来【矗初六切○刘向新序叶公子高之好龙也好画龙天龙行而降之窥头于牖叶公见之弃而反走五里而立是叶公为人好夫似龙而非好真龙也】
　　邓城
　　邓侯城垒汉江干自谓深根百世安不用三甥谋楚计临危方觉噬脐难【噬时制切○左传庄公六年楚文王伐申过邓邓祁侯曰吾甥也止　而享之骓甥聃甥养甥请杀楚子邓侯弗许三甥曰士邓国者必此人也若不早圗后君噬脐其及圗之乎圗之此为时矣邓侯曰人将不食吾余对曰若不从三臣抑社稷实不血食而君焉取余弗从还年楚子伐邓十六年楚复伐邓灭之】
　　息城
　　息亡身入楚王家回首春风一面花感旧不言长掩泪秪应翻恨有容华【左传庄公十年蔡哀侯娶于陈息侯亦娶焉息妫将归过蔡蔡侯曰吾姨也止而见之弗賔息侯闻之怒使谓楚文王曰伐我吾求救于蔡而伐之楚子从之秋九月楚败蔡师于莘以蔡侯献舞归○荘公十四年蔡哀侯为莘故绳息妫以语楚子楚子如息以食入享遂灭息以息妫归生堵敖及成王焉不言楚子问之对曰吾一妇人而事二夫纵不能死其又奚言楚子以蔡侯灭息遂伐蔡秋七月楚入蔡君子曰商书所谓恶之易也如火之燎于原不可乡迩其犹可扑灭者其如蔡哀侯乎】
　　召陵
　　小白匡周入楚郊楚王雄霸亦咆哮不师管仲为谋主争敢言征缩酒茅【咆蒲交切哮虚交切○左传僖公四年齐侯以诸侯之师伐楚楚子使与师言曰君处北海寡人处南海唯是风马牛不相及也不虞君之渉吾地也何故管仲对曰尔贡包茅不入王祭不共无以缩酒寡人是征昭王南征而不复寡人是问对曰贡之不入寡君之罪也敢不共给昭王之不复君其问诸水濵师进次于陉后楚子使屈完如师师退次于召陵杜预注曰召陵颍川县也】
