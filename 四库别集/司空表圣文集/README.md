　　提要
　　【臣】等谨案司空表圣文集十巻唐司空图撰表圣图字也河中虞乡人于僖宗时知制诰为中书舍人旋解职去晩自号耐辱居士朱全忠召之力拒不出及全忠僭位遂不食而死唐书列之卓行传图所着诗曰一鸣集别行于世此十巻其文集也尚有唐代旧格无五季猥襍之习集内韩建徳政碑五代史谓干宁三年昭宗幸华州所立还朝乃封建颍川郡王而碑称为干宁元年立已书建为颍川郡王葢史之误其时建方强横昭宗不得已而誉之奉敕为文词多诫饬足见其刚正之气矣又集内解县新城碑为王重荣作河中生祠碑为其弟重盈作宋祁遂谓重荣父子雅重图尝为作碑今考其文亦皆奉勅所为事非得已不足以为图病也陈继儒太平清话载耐辱居士墨竹笔铭此集无之其铭序云咸通二年余登进士叨职史馆按唐制进士无即入史馆者图成进士在咸通末出依王凝为幕职本传甚明焉有职史馆之事又云自后召拜礼部员外郎迁知制诰寻以中书舍人拜礼户二侍无日不与竹对按序称墨竹种于长安图知制诰中书舍人乃僖宗次凤翔时其为兵部侍郎又当昭宗在华州时何由得与竹对况图身为唐死年七十二而序乃云今为梁庚寅余年八十有二其为伪撰益明矣是编旧本前后八巻皆题为襍着六巻独题曰碑寔则他巻亦有碑文例殊丛脞今并削之云乾隆四十三年八月恭校上

　　说鷰
　　将儒
　　儒以将道肥其内也武以将威肃其外也未有内自瘠而外能劝者焉嗟乎古之用儒其所寄诚重矣儒之将道必欲张其治也独将之不足侈其道故分巳之任以寄于人亦犹资众力以夷大路绰绰然其甚辟也如有用于时者天下不防于治哉嗟乎后之为儒其力寖羸矣简固以自持窘密而多塞知所以任之于巳不知所以任之于人而责之故虽用于时道亦削然不喻将儒之权耳且古之言兵必本之仁义反是则一决之勇未足为武一智之谋足以夺其机矧兼吾道以制于未萌哉嗟乎道之不可振也久矣儒失其柄武玩其威吾道益孤势果易凌于物削之又削以至于庸妄于武可也必将反是请先将儒
　　铭秦坑

　　纪恩门王公宣城遗事【凝】
　　上四年春以大河南王公治状宜陟诏假礼部尚书按察宣歙池三郡既即治未渉时吏化民悦昼亡可为冬十月贼始自同安分济入屠至徳【在池州】公遣将孟琢疾以兵助守池州州亦有备贼少却会其党继至言扣壁实欲转袭南陵公揣之如其计命樊俦进扼青阳路俦故自彭门分公察其健决而易敌诫之邀险无得轻动俦径去会谍者言冦少将遁俦欲生沮之麾兵不顾既而越险贼遽至军遂委而逸间二日驰归以败吿公数其违命立斩以狥诸将在南陵闻之者皆股栗以死缀贼故青弋江得资为备【青弋在江东南陵也】人心不摇矣公前命宁国兵遮截之生得其踏白数十骑乃并山引退四月凶渠复大入而都纪王涓亦自采石赴援公宴劳加礼且谕之曰盗慝情诈吾尽得之矣今席胜而骄谓吾城不劳而可掇则攘臂而来前无丝髪之碍矣若知吾坚壁待之其势自衂将军苐按甲稔威以伺其隟慎勿与之骤战也涓意锐自州城日趋四舍至南陵未会食而阵遽死之明日中贵人自领败卒尚四五千伤痛之声与尘埃相杂而至江南雅自怯独幸北军以为固及闻涓败相顾失色頼公前防涓不足恃拒守益严又城中水乏而泉自跃出人心益牢竟免攻围之患既逾月中贵人阻挠无去意卒横不能禁公以书让之曰吏之捕蝗者既不克胜而且俯食于民是率暴以济灾也今将军纂天下精兵挫于狂冦而又恣之俾民不得治其生业何以称主上所以待将军之意哉中贵人词诎趋发其亲吏入里舍殴夺民马公乗门望之麾左右立提置杖杀军愕眙不敢留自后日夕抚循常若冦复至者其储畜缮完之利到今頼焉噫公始以杰徳峻望为时耸服而夐特恢济之心人莫能见虽不能致其道以和平天下然捍境蔽邻不涸得赋亦足济庸而塞恨耳愚尝袭迹门下受知特异敢次见闻以开史氏之听
　　辩楚刑
　　楚谓献璞者欺我乃连刖之酷哉曰彼独鉴之不胜耳然其嗜寳之心皆逹于卞子故连刖之无怨玉亦卒售于楚国嗟乎国之嗜贤宜急于楚之嗜寳也必嗜贤士心达于天下则负材求进者虽黜于见疑亦未为怨必有释其疑者则其卒用于世也可几矣是不犹愈于易其知而嫉其进者耶嗟乎刑与辱上之所以肆于下也楚无嗜宝之心卞岂受刑上无嗜贤之实士岂受辱必待诫门而絶愧哉
　　容城侯传
　　容城侯金炯者本蜀郡严道人附山而居同族中多见搜采其先因秦时调发诣尚方输作世苦之乃诫子孙易其服色必以精厉自进后徙居上洛会郡中卢生范生皆传修炼之术委质相资因砥磨以致用上闻而器之召见嘉其鉴局且谓毫髪无隠屡顾属之厯试台阁号为明达挟奸邪以事上者见之胆栗輙自披露至于妇人女子媚妩之态亦不能掩也其察察如此虽待物无私方圆不碍然疵陋者终恶忌积毁于上以为背靣不相副炯亦病于狭中不能以尘垢混其迹也竟被摈斥后有月蚀之变时宫中漏下数刻上临轩念其规益复急召俾其道所以然者扣之响应不疲上异焉命以容城侯奉朝请而宗人别于广陵者炫饰求售陷为轻薄子权戚中或怃然自喜则狎玩不厌至或被以组绣盖便其俯仰取容虽穿鼻服役亦无耻耳既稍进炯又鄙其为人乃复以谗废归老于家太史公曰炯之逺祖当轩辕时巳化服于祝融氏得荐于上能强记天象地形草木虫介万殊之状皆视诸掌握盖其术亦规模洪范耳物怪遇之莫不惴息自废后益亲幸上晨兴必先至则与冠冕者偕进号为夀光先生不名也子孙稍下衰然流寓太原者始尚亦以精链见重观炯虽任用兢兢惟恐失坠不善匿晦果为邪丑所嫉几不能免噫大雅君子既明且哲以保其身难矣哉
　　移神
　　夏满不民前后走神所刳羊豕而跪乞者凡三而后得请民大喜且将报祀愚独以为惑何者天以神乳育百谷必时既丰然后民相率以劳【去声】神之勤于事而祀焉今始恡其施以愁疲民是神怠天之职也必希民之求而遂应是神玩天之权也既应而俾民输怨于天归惠于巳是神攘天之徳也推怨而何以为义利腥羶之馈何以为仁怠天下之事何以为敬蔑是数者何以为神假曰非吾所得専然知民之情而不时请上是亦徒偶于位此愚之所以惑也噫天不可终谩民不可久侮窃为神危之奈何
　　议华夷
　　议天下之大势者滞而拘古必曰固于徳刚而简谋必曰弭于威是皆不足扼阽危之机也必济徳于谋济谋于险庶几可以夀宗社之数矣前古迂儒瞆耳援据滋惑不能中今之急病而近朝有心于经制杜司徒之治道李安邑之地志元中书之安边不谓不驰骋于古今成败之数也惟贾仆射耽并包华夷绵络山川披图摘要繁而不瘠可谓勤而至精者矣虽然量力救时当置逺荒于度外国史事有追惜而不可形于纪述者或闗防战而不守或控制议而不行或仓廪弃而不保利害之效可见于斯愚是以玩而不厌也虽失之巳乆得不虑哉
　　上谯公书
　　再拜愚伏以布衣犯将相之威者近皆笑之指为狂愚是轻薄子不能以恢然之量待今贤杰也相公得不念之耶某迹拘世累而业乆于山援古勘今思有所发者盖窃惟近朝据重位而勇蹈功名之列耻天下有遗才直吾相国也又敢求吾相国之心所以未忍弃生民之望者固非濡滞于富贵岂不以常持大柄事或阻心且复弛张俾无遗恨于不朽耶愚以是窃幸有以企天下之福矣虽在于至愚犹有轻虑诚以相公既当有得贤之盛将尤有惜夫自持既尝独决机权将尤以事不足问则党附之嫌或兴而专美之道可隘耳请陈其说夫用人也固得矣亦在知失之不足盖为明则伪者惩而实者劝且无伤于爱士处事也固济矣又知谋之不必自我为知则听日广而神不劳且无伤于好谋是道也盖贤哲用之而不竭相公得之而不疑坦懐至公自无愧古然后文尚制科之选武先西北之虞【前年巳上蜀相书】抑简诞以捐空峻规程而括实则因应适时之宜尽矣此皆相公夙自贮于沉实而小子虽吃悸不能面发愿激扬于片词耳非为挟利之资也抑自古钓竒而售迹者既多以分蹇动无所合且实必俟临机方见其万一非敢率易并凟尊威干犯之诛则不复自同轻薄子以为疑惧俟命再拜
　　天用
　　材轶而鸷防者不待赋而后食盖济巳之纳得以自任也龙虽善致风云然必有所禀而后能施其泽吾知其不能自神矣苟専其用而汲汲于济物亦将救祸之不给虽川后岳灵孰敢挠其所守哉呜呼时乎时乎盖贤哲之所宜禀唯用天之用然后功虽约而济物博
　　与王驾评诗
　　足下末伎之工虽防誉于哲贤亦未自足谓必俟推于其类而后神跃而色扬今之贽艺者反是若即医而靳其病也唯恐彼之善察药之我攻耳以是率人以谩莫能自振痛哉且工之尤者莫若伎于文章其能不死于诗者比他伎尤寡岂可容易轻较量哉国初上好文章雅风特盛沈宋始兴之后杰出于江宁宏思于李杜极矣右丞苏州趣味澄夐若清沇之贯逵大厯十数公抑又其次元白力勍而气孱乃都市豪估耳刘公梦得杨公巨源亦各有胜会浪仙而下刘徳仁辈时得佳致亦足涤烦厥后所闻徒褊浅耳河汾蟠鬰之气宜继有人今王生者寓居其间浸渍益久五言所得长于思与境偕乃诗家之所尚者则前谓必推其类岂止神跃色扬哉经乱索居得其所録尚累百篇其勤亦至矣吾适又自编一鸣集且云撑霆裂月劼作者之肝脾亦当吾言之无怍也道之不疑
　　说鷰
　　鷰虽同类有巧拙之目里人构其寝始就而拙者遽巢其间巧者既至必搏而逐之俾之他室虽童稚亦以为恨里人不能决其去留者盖辨之不早辨耳噫彼之所工岂能自固其室已或为拙者所逐沮人尚惜之然则贤不肖之取舍其利害于天下何如耶治乱之兆轨易于此乃系于择先后耳可不惜哉可不戒哉
