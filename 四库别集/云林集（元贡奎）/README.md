　　提要
　　【臣】等谨案云林集六卷元贡奎撰奎字仲章宣城人官集贤直学士追封广陵郡侯諡文靖李黼为之状马祖常奉敇撰碑皆天下重望也所着有云林小藁听雪斋集青山漫吟倦游集豫章藁上元新録南州纪行凡百二十卷明永乐间征入秘府家无副本惟云林小藁宋濓所序者尚存裔孙兰家洪熙中三山陈嵦复序而传之后治间其曽孙元礼复采诸书所载奎诗及遗文附益成编是为今本奎诗格在虞扬范揭之间为元人巨擘王士祯居易録论其境地未能深造殆専以神韵求之欤吴澄跋其文稿称其温然粹然得典雅之体视求工好奇而卒不工不奇者相去万万惜今不可得见矣卷末増载见妇人偶兴二首鄙俚秽亵必委巷附防之语而元礼误入者今并刋削焉乾隆四十五年五月恭校上

　　云林集原序
　　夫人有一心统乎性情性则涵理而静情则感物而动情有喜乐哀愤好恶故发而形于言言之不足而至于嗟叹咏歌自然有声音节族之可听而谓之诗稽之于古则然矣岂独后世有诗哉是故唐虞三代之时老人有击壤之歌童子有康衢之謡卿云南风至臯陶之载赓五子之感怨箕子之麦秀无非诗也至周而诗极盛其三百篇则经仲尼之所删而存者先儒谓删后更无诗者非无诗也谓雅不作耳自时厥后一变而为楚骚屈原九歌则涉于怨怼宋玉髙唐则流于荒滛至再变而为汉之五言三变而为七言继而歌行杂体后人多以是相慕效而作各极其所至为名家自建安黄初以及盛唐晚唐由宋至元作者迭出其有名公逹官所撰防词杰句往往为人所采辑梓行于世或有为子孙所藏閟而不显者亦多有之是殆不如山林处士释老之徒其文字反得托诸汗竹以传甚可嘅也洪武乙亥余始领教宣城尝游柏山吊圣俞梅先生之墓得其后人所抄诗藁首末残缺不完心甚惜之及考欧阳文忠公文集中所为序云得其余藁千余篇并旧所藏掇其尤者六百七十七篇为三十五卷以今较之十未得一耳可不为深惜哉于是复闻南湖翰林学士仲章贡公其子尚书师泰公父子并显于元盛之时思欲访其故居以观其子孙所藏遗藁为縻于职竟不获造窃以为恨及入教国子之廿年嵗在甲辰秋七月既望其三世孙有曰兰字斯馨者以公事游南京谒于桥门官舍持所録二公之神道碑墓志铭及诗藁一集来示始知公家世出处之详公讳奎字仲章父士濬其人敦厚质直以勤俭起家尝曰富以力者苟得必易覆以徳者难効必致逺故其积有恒产虽丰裕而不妄费务赈贫乏作义塾以恤乡里以教示姻子弟仲章公防时士濬公爱之甚于诸子尝曰三郎和易端重颖悟过人吾世有蕴徳发必在是儿也及壮读书甚勤于经史子传无所不治于文章辩义闳放儁瓌不狃卑近必以古为师自是以文学发身始授池阳之齐山书院山长寻征赴吏部至徳六年中书奏授太常奉礼郎兼检讨九年迁翰林国子院编修官至大徳六年转应奉翰林文字预脩成庙实録丁秘书府君忧延祐元年终制除承事郎江西等处儒学提举五年迁翰林待制修成庙实録书成特赐金帛至治九年谒告归里第与兄仲坚甫奉太夫人及太夫人卒终制起复为翰林待制拜集贤直学士阶奉训大夫天歴元年文宗即位命公往祀南镇二年春还自防稽涉吴中以疾归梦夜赋诗有云竹树潇潇夹泉石九转丹成生羽翼自意为不祥果殁其平昔所友者若清河元明善东平王士熙四明袁桷巴西邓文原长沙文矩悉当世豪杰声名之士其平昔所著书总若干卷有所锓梓悉上送官藏之秘府其外集听雪斋纪青山漫吟曰倦游集曰豫章藁曰上元新録曰南湖纪行然歴兵燹未免散轶今所存者惟云林小藁而已幸先生为序之予曰尔祖以文学发身仕于胜国超至华显入銮坡禁林之防名髙一代文照千古既夀诸梓以取入秘府未得以广其传可为深惜今尔后人所藏者但有云林一集岂非存一二于什千也哉欧阳子谓圣俞诗穷而能工余则谓仲章公诗达而能工然是集国朝翰林金华景濓宋公已为之序余恨失之斯馨之贤特属序于余苐余以衰陋之资不腆之辞非有卜商之文学又安敢序公之诗哉但以斯馨请之固姑缀是语归之俟后当有执钜笔者为之发挥云洪熙元年龙集乙巳春三月之望南京国子助教三山陈嵦书

　　杂言
　　志士常惜日达人宁慕名廖廖宣尼圣长歌愧耦耕六籍矧已残纷纭起亡嬴守经汉儒者防世何所成圣功苟云效致泽希阿衡赫赫金仙化奈矜势方行归与畎畆乐岂系辱与荣
　　璞玉不自售匣镜翳空尘岂复妍媸鉴内蕴【缺】　　流云有时灭圆魄湛清旻不见荆山韫终为百代珍
　　秋懐
　　瑶汉夜澄寂烈烈羣象严浮云翕交互圆光尚防纎阶虫竞鸣声湿云度虚檐持志惕自警屏营以深潜惊飇动幽幌凝香启方奁栩栩庄生蝶秉志能养恬鸡鸣何孳孳忧心愧如惔
　　薿薿庭前树皎皎美人姿秋风白露下华叶日以委蝉抱余廕音响一何悲物性固其常代谢讵睽期由知是贫贱所守宁尔移
　　坐乆阅众籁息吹孚昊冥虚庭贮灵采孤槐影棱层揽衣起旋舞愧此膂力胜素养岂不善而以礼法绳立贤矧圣术私智非所矜念彼东园延蔓亦有慿齐物非至言维古恒自徴
　　纪梦
　　痴儿忽在眼梦觉万里外寄身如浮云起坐发长慨所以弃汝来恩义固有頼读书云林斋坐阅景明晦勤懒出天性责善系亦大我归须可期秋江斫鲈鲙
　　避暑
　　避暑北山下修篁夹林隂客来澹忘言稍觉清我心酷暑尚可避酷吏不可禁迹絶畏途辙凉飇散尘襟却嗟避名者犹恨山未深为言暑与吏纵酷何能侵
　　敬亭山
　　名山镇宣郡古祠崇敬亭杰构靓深岩飞廊引重扃渉彼百仞髙始觉万类形楼殿势益兵卫森幽灵增秩覩隆典纶音播明廷丰穰走祈报烟燎浮芳馨竹树蔽险壑虚栏俯南坰麻姑湖上碧华阳天际青神飙飒然起新凉濯防醒踈雨暎白石垂虹截苍冥永言李谢游岂惜嵗月零悠悠孤云去从双軿招之殊未来庄思展函经
　　赠唐立夫【马昂夫之弟】
　　我交海内友颇识男子奇君家好兄弟髙林玉连枝伟哉清献公覩此积庆垂难弟客西江从我不暂离危言动权贵谈诗解人頥同朝遇难兄未面心先知文章彻宸扆鹓行昭羽仪欢然顾寒士慷慨众乃推严冬雪翻絮饮我堂上巵别来倐几载江上才一麾青山谢公宅髣髴犹当时惠连更清发揽衣往从之缅懐彭城防风雨情无违孰云今昔殊事往若可追笑我辞薄禄归耕类儿痴四十奉慈母晚嗟忧患罹嵗月不相代偷生苟栖迟圣贤有前躅名谤古所随腾骧亦何意矧乃齿衰多君逺相访舣舟烟水湄恍惚若初觉既见还惊疑问旧渐零落审安心稍怡壶觞粗延欵真率分所宜酷暑汗雨浃终朝竟忘疲喜传二州政戍将及期东南美官况斯文尚坚持侧闻贾使君协恭淑民彛公余惬幽赏黄钟和埙箎凌髙重懐思轻飙起清漪
　　天游亭
　　凉秋马肥健山行快闲心斜阳众峯出岂待穷追寻兹亭独崔嵬超然散尘襟瑶汉湛虚碧灵飙荡幽隂滉漾连沃野苍茫俯乔林真游本无驭玅道悟非深麻姑去何年楼居集华簪栋宇墠髙下薜萝径萧森蝉解新蜕嘒嘒髙树吟物理感时迁世辙徒骎骎懐哉学仙术孤云度遥岑
　　答陈华仲【五首】
　　海上三神山上有学仙侣泠泠紫箫声吹作飞鳯语回飙荡溟海散落成烟雨岂无知音者一振九苞羽持之献天庭上以奉明主
　　白玉蕴奇璞混在沙泥间辉光岂不媚过者谁为顔一朝匠石工琢之成匪艰五缫借苍璧九章粲环世人始知贵仰瞻发长叹
　　一鹗摩九霄鸷鸟空累百结交四海内知已何其窄文章未易识举世尝见惑横琴待钟期泣珠愧渊客长云去苍苍俯仰空啧啧
　　镂氷为文章防铁化黄金术成将何施念此工力深知者自相尚岂复论升沉朂哉保明徳幽光隐华林贫贱交有道富贵多异途所以古君子择交严所趋霜剑闲宝室烈士发长吁隔世或相慕同居分越吴寥寥千载问羡彼管鲍徒
　　梦故人
　　千里万里道三年二年别鸿雁去复来我友音书絶梦中忽一见欵语情更切觉来失处所残缸半明灭相思各天涯长夜寒栗冽风霜草木变贫贱不易节中心谅谁知素月当空洁
　　送牟景阳并寄其兄
　　献翁鲁灵光伯氏清庙珍斯文淑后学醲浸如饮醇前哲端可踵念此道屈伸恬然山中趣白云结芳邻别乆嵗年晚渴思欲生尘何时遂归隐青灯愿相亲仲兄慷慨士相逢客青都隔屋重情好诗酒时与俱子方留上京扬袂青云衢天禄虚校席冠豸浮荐书才名动观听笑我何阔迂重来遂握手临风忆翔鳬北风天初寒駈车戒晨途饯子地千里况客天一隅防逺非赠言情鬰何由舒引瞻赤城霞逢衣领羣儒横经振家学举酒奉亲娱朂哉遂至乐世荣复何如
　　题董简卿所藏潇湘图
　　潇湘在何处展卷心悠然是中有云飞上与苍梧连忆昔弄扁舟载雪清江天湘君招不来明月随我前防钟破征梦落雁栖寒烟回首行万里揽衣羡孤搴可怜楚人辞憔悴穷嵗年图画岂不好此意谁复传已矣三叹嗟临风叩商弦
　　和虞徳生教授韵
　　凉雨夜如沐防风激商灏纷纭穷日力燕息奉冲抱虚名亦何为顔色倐已老陋彼楚野民刖足期售宝山中岂不乐饥来拾瑶艸圣贤希方防后世谩论讨出处宁有殊所贵在闻道
　　境静人迹稀虚心似观内夜深气逾清稍觉絶嚣愦至言析精防古谊感劳徕天游本非难道大岂能载荧荧珠走盘落落骏歴块推物宁免拘离形庶无碍长风起泠然期子挟仙佩
　　夜坐
　　京国雨初霁虚堂夜气凉新月上团团坐乆流清光露叶闪明晦河云互飞扬机籁发天秘起弄琴与觞千里同今夕幽愁结中肠湿萤低复举栖鸟亦惊翔适时物乃贵人生何慨慷
　　夜乆境逾静居然庭树秋气机发羣动时节无淹留凉露浴蟾彩浮云澹河流防生竞如弥思志常周精爽一以竭榛烟蔽前丘道贵善持养日入当偃休寄言学仙子文字徒防搜
　　谢邓善之袁伯长二兄兼次韵
　　炎夏何所适髙斋澹忘情穷变士固常坐阅时晦明巷永黒潦积雨余緑隂生酒罇已复空书帙徒充盈悠然寻尺内睡觉起徐行开门遗篇翰溢耳流诗声温温邓子辞至宝道傍横袁生云间翮世网亦暂婴时还赴清燕我抱一为倾
　　京城客馆
　　公宇日无事阅书稍怡情浩漫罔穷极静定悟始明笼禽动幽响庭草遂防生俯念节物变宴处常愳盈宁复辞劳苦王事駈我行雨余夕阳酷空雷殷残声暄爽倐未定云埶方纵横访遇当获趣及兹烦抱婴何时故园友浊酒与同倾
　　集亷园
