　　钦定四库全书　　　　集部二
　　宗集　　　　　　　别集类一【唐】
　　提要
　　【臣】等谨案宗集三巻附録元纲论一巻内丹九章经一卷唐吴筠撰筠字贞节华隂人隠于南阳天寳中召至京师请为道士居嵩山复求还茅山东逰会稽往来天台剡中与李白孔巢父酬唱大歴中卒弟子私諡曰宗先生新旧唐书皆载隠逸传此本为浙江鲍氏知不足斋所钞末有原跋云收入道藏中世无别本然文献通考云吴筠宗先生集十卷前有权徳舆序列于别集诸人之次则当时非无传本此跋题戊申嵗不着年号疑作于通考后也巻首权徳舆序称太原王顔类遗文为三十巻后又有吴尊师□亦徳舆撰乃言文集二十巻均与文献通考称十巻者不合考徳舆序称四百五十篇而此本合诗赋论仅一百十九篇则非完书矣又旧书筠本传云鲁中儒士也新书本传云华州华隂人徳舆序称华隂人而传又云鲁儒士序称受正一法于冯尊师上距陶宏景五传传又云受正一法于潘体元乃冯之师亦相乖刺考旧书李白传称天寳初客逰会稽与道士吴筠隠于剡中而传乃言禄山将乱求还茅山既而中原大乱江淮多盗乃东逰会稽与诗人李白孔巢父诗篇酬和不知天寳乱后白已因永王璘事流夜郎矣安能与筠同隠此传殆出于依托序又称筠卒于大歴十三年卒后二十五歳乃序此集其年为贞元十九年徳舆于贞元十七年知礼部贡举明年真拜侍郎故是年作序系衔云礼部侍郎其文与史合而金丹九章经后又载筠自序一篇题元和戊戌年作戊戌乃元和十三年距所谓先生化去之年又隔四十年后且云元和中逰淮西遇王师讨蔡贼吴元济避乱东岳遇李谪仙授以内丹九章经殆似呓语然则此序与传同一伪撰矣据新旧唐书皆有元纲三篇语则巻末所附元纲论三篇自属筠作至内丹九章经核之以序伪妄显然以流传已久姑并録之而辨其抵牾如右乾隆四十六年四月恭校上

　　宗集原序
　　道之为物无不由也无不贯也而况本于览发为至言言而蕴道犹三辰之丽天百卉之丽地平夷章大恬澹温粹飘飘然轶八纮而泝三古与造物者为徒其不至者遗言则华涉理则泥虽辩丽可嘉采真之士不与也宗先生吴君其知言者欤先生讳筠字贞节华隂人也年十五笃志于道与同术者隠于南阳倚帝山闳览古先遐蹈物表芝耕云卧声利不入天宝初纁鹤板征至京师用希夷启沃脗合圣请度为道士宅居于嵩阳丘乃就冯尊师齐整受正一之法初梁贞白陶君以此道授升王君王君授体潘君潘君授冯君自陶君至于先生凡五代矣皆以隂功救物为王者师十三年召入大同殿寻又诏居翰林明皇在宥天下顺风祈向乃献纲三篇优诏嘉纳志在遐举累章乞还以禽鱼自况薮泽为乐得请未几盗污三川羽衣虚舟泛然东下栖匡庐登防稽浮浙江息天柱隠机埋照顺吾灵有时放言以畅天理且以园公歌咏于紫芝景怡恱于白云故属词之中尤工比兴观其自古王化诗与大雅吟步虚词游仙杂感之作或遐想古理以哀世道或磅礴万象用防环枢稽性命之纪达人事之变大率以啬神挫锐为本至于奇采逸响琅琅然若戛云璈而凌倒景昆阆松乔森然在目近古游方外而言六义者先生实主盟焉至若总论谷神之妙则有纲哀蓬心蒿目之逺于道也则有神仙可学论疏瀹澡雪使无落吾事则有洗心赋岩居赋脩胸中之诚而休乎天君则有心目论契形神颂其他操章寓书賛美叙别非道不言言而可行泊然以微妙卓尔而昭旷合为四百五十篇博大真人之言尽在是矣大厯十三歳歳直鹑首止于宣城道观焚香返眞于虚室之中门弟子有邵冀者率吁其徒宁神于天柱西麓从其命也冀徧得先生之道如槁木止水刳心遗形太原王顔尝恱先生之风自先生化去二十五歳顔为御史丞类斯遗文为三十篇拜章上献藏在秘府厥后冀得其本以授予请序引其迳庭庻传永乆别有辩析世惑之论不列于此编至若挺神竒祛物怪告链脱之地合肸蠁之符皆备刻于金石者之说今徒采获斯文以序崖略且俾后学知道者必知言云权徳舆序

　　岩栖赋
　　感圣之垂训悟已亲而名疎言可放而从黙身应巻而勿舒爱鹪鹩之巢林在一枝而有余性所恱而难违托兹山以结庐果栖迟而我惬即逍遥之灵墟观其缭崇峦横峻谷激泌泉罗森木后巍嶷以萦纡前参差而耸伏追隂壑之夏凉偃阳崖之冬燠美劲节于松筠翫幽芳于兰菊虚籁清耳闲云莹目因海鹤以警夜任晨鸡以知旭虑静于无扰神恬于寡欲于是歌考槃于诗人讽嘉遯于大易逺浮俗之艰险消毁誉之损益蹈方外之坦途信可免于兢惕既即隂以息影由不行而灭迹虽区中之末计实世表之长防人所弃而已收故处约而恒适览无见以收视聴无声以返黙和匪专于防酒乐奚必于丝桐焚清香以链气启玉检而击防期遣滞于昭旷庻延真于感通筌太虚之有象覈妙用之非空朝天甚简采药多暇形犹资于吐纳意已屏于将迓知道无废兴而物有存谢故挹生本而常生体化宗而不化萧萧絶尘谁与为隣迹逺而朋从益旷机忘而鸟兽可驯韵靡叶于当时心常依于古人仰巢由浩浩之逸轨咏羲农黙黙之化淳师黄老之奥友乔松之道真慙无功之逮物良独善于吾身只所幸其自得敢韬精于隠沦
　　竹赋
　　惟坤灵之播育何备物之实繁伟兹竹之标挺得造化之清源契道合虚表贞示节叶森散以翠错茎鲜脩而琼洁尔其和风流畅万彚昭陈杨葩晔其密苑荑栁蔼于通津不郁馥以启曜但葳蕤而有筠亦未之为异也至如杀气凌厉凝霜萧瑟览平楚之苍茫窥众林之如失冒冰霜之涸沍逾青荧以郁密若是则殊可重焉故诗曰如苞书称厥贡猗兹淇园美彼云梦昔在轩后肇官隂阳俾伶伦于嶰谷伐脩竹之珍篁裁六律以协气调八风而顺常然后成竽籥以备乐其声滥而弥长笙镛以间鸟兽跄跄谅自然之纯粹曷羣材之可方若乃渭川千畆山阳数林会稽方润于碧玉罗浮比色于黄金上黕黕以云翳下泠泠而风吟祛赫曦之燠景纳凄清之凉隂王子所以啸咏嵇生为之幽寻召嘉宾及令友畅醁醴与鸣琴美游盘之逸趣清寂寞之逺心若乃夹沧江倚丹崿蓄水雾之沈沈摇岩烟之漠漠湘妃有挥涕之感楚谣兴防露之作或挺鼔吹之嵚崟或垂天门之磅礴皆鸾凤之所翔集孔翠之所栖托岂独婵姢于广漠之壤亦有璀璨于蓬莱之峯结实珠粒敷花紫茸拂皓粉以飞雪摧绀茎以韵钟固列仙之攸翫匪吾人之所从也亦有化雉吴国成龙葛陂容人筼筜育虫桃枝一笋明其嗣三节获乎婴儿萦灯纂以感孝茂窓櫺以表竒篲冡坛以尘灭环石牀以防滋皆灵变之谲怪良难得而备知尔其众彚非一则有篥簩防笾筒箖箊菰棰之肃矗龙钟云母之扶疎箬箭浮色以缥焕菡綷文而绣摅方志之所遗载山经之所阙书者安可得而详矣靡不劲坚其性葱蒨厥色不规而圆不揉而直故髙皇制冠以宝位孝文剖符以表职博望侯传于大夏之外穆天子封于池之侧推此类以弥广匪斯文之可极
