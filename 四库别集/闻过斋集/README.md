　　提要
　　【臣】等谨案闻过斋集八卷元呉海撰海字朝宗闽县人至正末遭逢兵乱絶意仕进明洪武初守臣欲荐诸朝力辞得免既而徴诣史局亦力辞不赴事迹具明史隠逸传是集为其门人王偁所编初海与永福王翰善元亡之后海以翰尝仕元劝以死节而自抚其遗孤教之成立即偁是也史称其文严整典雅一归诸理又载海尝言杨墨释老圣道之贼管商申韩治道之贼稗官野乗史道之贼支词艳説文章之贼上之人宜勅通经大臣会诸儒定其品目颁之天下民间非此不得辄藏坊肆不得辄鬻云云虽持论少侠非古人兼资博攷之义然其宗防之正亦于此可见矣其题曰闻过斋者海平生虚懐乐善有规过者欣然立改尝以闻过名其斋偁因以名其集云乾隆四十一年五月恭校上

　　闻过斋集原序
　　文以道轻重自昔有是言也由古讫今其文之行后者莫不皆然舍道而论文则其人品之髙下何由而见哉夫言者心之声而文又言之精者也心明乎道故其言皆可法昧乎道者反是是故理明而辞达文之上也理胜乎辞文之次也辞胜乎理不足谓之文矣况磔裂靡陋以趋时好者乎今观闽中吴先生闻过斋集叙事严整议论正大雄健宏奥离合变化一归于理非通经博古之精而信之笃养之有素何能致是韩子有云根之茂者其实邃膏之沃者其光晔仁义之人其言蔼如也信然哉先生平生刚直终身隠约未尝求知于人然非其人则亦莫能知也如尚书宣城贡公翰林学士晋安林公皆擅文名当世不易下人惟于先生者深加敬畏则其人品可知矣矧文章如良金美玉自有定价何待后世如扬子云者出而后知耶先生没迨今逾十五年是集藏于门人王偁家近吾友胡伯寜氏同知盐运于闽见而喜之将于其同志建寜郡守芮君志文率好义之士为锓诸梓以广其传间以公事道过乡郡出以示余请序自顾老病昏塞安敢谓知言哉鄙野之辞不足轩轾姑书此以志景仰之私云岁在辛巳春正既望将仕佐郎前温州府儒学教授永嘉徐起书

　　吴氏世谱序
　　按吴本姬姓泰伯之后以国氏后子孙散居天下其在闽有繇光州来者唐光啓中有曰英从王氏入闽相传始祖也海昔为儿童闻先君言防时见吾父尝阅巨帙问曰此族谱也自先世宗族及后代子孙悉录于是吾父不幸没于易代之际母仓猝弃产擕防入古灵山中依外家以居谱牒遗失吾年方九岁及冠乃归又言吾支故不蕃吾曽祖举进士初授寜国府知录而卒曽祖生祖及叔祖而叔祖无子先祖生吾父兄弟四人长及季又无嗣次仅得一子而吾父乃生我及叔寜知后世何如也又言吾先世家郡城东后徙居下渡侯官吴帝俞自横棣来继石鼈之宗至则谒先君讲兄弟之礼曰兄派河口河口派出侯官吾偪族也帝俞生与先君同年而月日差后侯官兵不及耉老犹存其言必当有受先君又言吾前后询吾谱殆三十余年而竟莫能得其诸族分已乆而吾所记特近宜其不能合也苟终吾身而不得其作于汝乎先君即世海追承先志嗣而录之恐于是弗作则来者益无所考昔谷梁子有言信以传信疑以传疑吾亦由是而已吾父吾所见之世也寜国以下吾所闻于吾父也此传信也其他则传疑也吾不敢以疑为信故是谱断自寜国以下抑吾于此重有感焉古者宗法行于天下宗族有所统一人心有所聨属故孝弟隆而习俗美先王之治易易然是法既废人各以意自私其亲恩不广而义不终至有视周亲若涂人者然则今日所頼惟简牒之存使知木本水原之义是谱所以作也嗟夫夫人孰不念祖宗哉亦孰不愿其子若孙世世亲睦哉而祖宗之望有不若予乎苟推予之心以及祖宗则逺者犹近疏者犹亲服斩而情不斩属尽而爱无穷夫恩固赖于相成而道实原于自致子焉自致其孝无怨乎父之不慈父焉自致其慈无疾乎子之不孝兄焉自致其友无责乎弟之不恭弟焉自致其恭无恤乎兄之不友致于已而不望于人则其道易成也是则海命谱之意非亶为纪名系设也呜呼子孙念敬之哉呜呼子孙念敬之哉
　　一谱首为图具世次而派别之以名系世盖畧以考派别则不紊无后者直疏其下曰絶【谓无子而不置后者】有官者疏曰某官【从后授】迁居者曰迁某所
　　一首既为图以系世次次为谱亦以派别乃详记名字行次娶某氏歴官某生子几人某甲子生年若干卒葬某处某人为志若迁居者备述其由
　　一谱后述先世家训文字畧者及墓志若先世著述文字多者别为集不录于此【先世家训及述子孙保藏之以传后人】
　　一子孙名次从水木火土金行为一世五行相生循环无穷
　　一子孙行次五行从名次五行男阳女隂世次易考【如名从水则行次男壬一女癸一名从木则行次男甲一女乙一之类】每世从一起数则不相紊
　　一后世子孙有弃父母出家为僧为道者不录【谓不系世次也】
　　一后世有无子不立宗人而以壻与外孙为继者不录直疏其下曰絶【谓其自絶于祖宗也】
　　一丧事不得用浮屠道士营修科典不惟于死者无益而生者重有损
　　一葬事随力厚薄不得用俗礼焚化大不孝后虽有悔终不能及
　　一浯州府君尝谓海曰吾行四方乐邹鲁土风之厚甚欲徙居其地万一不能汝能承吾志乎
　　厚本录序
　　人之大伦有五而父子夫妇兄弟莫先焉此人道之始也盖三者居于内而君臣朋友际于外内者由恩而起义外者因义以生恩恩厚则义无不隆矣世学不明风俗益薄人之道将不立于天下予甚惧焉暇日采摭传记小説凡古今孝子顺孙节妇列女兄弟之相友姒娣之相宜者着为一编必求其行事卓卓而不尚乎多将使夫妇女童稚里巷小人皆向诵而习之养心以成德猝有变故祸难当有所守又附以感应祸福禽兽微物各为一卷意谓人道固所当尽有不能者观于报应之间亦可以劝矣而又不察则禽兽微物固犹有人心者其可不自愧乎以人道本于此故名曰厚本录若夫君臣之事则有史册在朋友之义则学者所讲此不录
　　自试方题
　　乙亥嵗予自武林疾归逾年弗瘳以至于困前后阅医二十余辈最后医来但诊脉即辞去皆云不可治始甚忧即不可如何则安以俟之頼先世藏书颇多医方亦数十种稍读之以意度其可用者试之每一试而一疾卒减遂屡试之疾以次而除毕是嵗脱然悉去体矣自后遍阅诸方遇家人有疾请医弗効者又屡试之颇皆得愈因录前后所试及意度可者裒为集以备遗忘而便检寻不幸有疾又将以自试而非敢以试人也
