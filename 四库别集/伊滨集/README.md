　　伊滨集二十四卷
　　【臣】等谨案伊滨集二十四卷元王沂撰沂字思鲁先世云中人徙于真定父元父官至承事郎监黄池税务马祖常石田集有所作元父墓碣铭叙其家世甚详而沂始末不槩见今以集中所自述与他书参考之尚可得其大略据马祖常碣铭称与沂同榜则当为延祐初进士据集中送李县令序则尝为临淮县尹据义应候庙记称延祐四年佐郡伊阳考地理志伊阳在嵩州则尝为嵩州同知又诗中有纶巾羽服卧伊滨之句则集名伊滨亦即起于此时据祀南镇北岳诸记则至顺三年尝为国史院编修官据送翟生序及胡节母诗序诸篇则元统三年尝在国子学为博士据送余阙序称元统初佐考试见阙对策云云则尝入试院同考而余阙实为所得士据祀西镇记御书防诸篇则至元六年尝为翰林待制并尝待诏宣文阁又宋辽金三史成于至正五年而书前列诸臣修史有总裁官中大夫礼部尚书王沂之名则是时已位至列卿其后迁转遂不可考疑即致仕以去然集中壬寅纪异诗有壬寅仲春天雨电南平城中昼惊愕自从兵革十年来澒洞风尘亘沙漠之句又邻冦逼境仓皇南渡诗有隣邑举烽燧长驱冦南平中宵始闻警挈家遂逺行之句又有寓吉林塘避桃林兵警诗壬寅为至正二十二年正中原盗起之时距沂登第已五十载尚转侧兵戈间计其年亦当过七十矣沂歴跻舘阁多居文字之职庙堂著作多出其手与传若金许有壬周伯琦陈旅等俱相唱和故所作诗文舂容和雅犹有先正轨度惜其名不甚着集亦絶尠流传选元诗者并不能举其名氏今从永乐太典中裒掇编次厘为二十四卷庶梗概尚具不致遂就湮没焉乾隆四十六年九月恭校上

　　函谷关赋
　　客有逺适于西土兮聊游盘乎关东窥河流之奔放兮眄髙掌之遗踪衍平臯之如席兮豁函谷之谾谾观天险之固防兮据五达而隆崇廼喟然而叹曰遐乎邈然姬徳之衰也有真人焉致虚克极兮守静曰笃游物之初兮而立于独缀明月以为佩兮制青霞而为服厌中区之溷浊兮朅逺游以观复乗素蜺之脩幡兮綷云盖而虹旗驷青虬之连卷兮先云浮之陆离矞狂奔而警跸兮走招摇与黔雷焱腾踊而防飒兮曵光耀之葳蕤诏陵阳以疾驱兮吾将往乎南娭巨灵下袭而奔突兮飏紫气之逶迤歴函关而弭节兮与尹喜乎相期着微言之元妙兮垂金薤之瑰竒曰无名天地之始兮有名万物之母听之而不闻兮视之而不覩唯恍唯惚兮以阅众甫喜然而自失兮防夫子谁发吾之醯鸡何天地之大全兮吾茫洋而不知廼谷神于元牝兮载营魄而髙驰相与过太乙之所居兮歴昆仑之嵗魁驱松乔与偓佺兮役羡门乎三危轶列缺之倒景兮飡沆瀣而充饥越八纮而朅度兮渉流沙而徘徊风枞枞而扶轪兮云来迎而襂纚矍欃枪与旬始兮格泽歘其扬辉羾阊阖而谒帝兮帝亦欣然而来只命西王母以上夀兮玉女鼓瑟而舞云师乗虚亡以独存兮超寥廓而忘归悲人寰而下视兮扰羣蚁之纷移伊兹关之形势兮为强暴之所恃若乃嬴秦虎视列国犬猘开关延敌鸟散麋沸又若羣雄云扰汉祖龙骧项羽枭冽纵火擅强烽燧之勤井干之固战马屯云猛士如雨既已势穷力竭变灭无余河深华髙草生故墟圣朝以四海为家八荒为居普天所覆咸沾咸濡不烽不吏匪柝匪符示王者之无外廓四防之通途吾将观紫气之遥遥俟青牛之仙车传清浄之言树雍熙之舆陋文景之迫隘蹑羲轩之元符
　　温泉赋
　　客有游于临汝之山徘徊温泉之上喟然叹曰山之有泉犹人之气也吹之则寒呵之则温盖其性之自然而无待于外也彼狃于灵砂暮石酷烈之气惑于烛龙荧惑潜照之怪葢昧乎物理之大而囿于耳目之隘然泉之为汤者在在有之而骊山汝水独以名著讵以乘舆后宫之所临幸欤吾是以嘉斯泉之遇也当其盛时绮疏璇题玉珰璧英鱼龙飞动人目睛及其势徂事变光沉响息翳之以荒烟古木罥之以泽葵藂刺薪翁渔子之所灌沐木魅山精之所啸集于是骚人墨客凝思寂听登髙逺望叹息成咏以为亡国之遗迹吾又以哀斯泉之不幸吾尝歌乆视之遗诗想舆衞之雄丽烟霏霭兮千官风雨忽其万骑三思之鬼蜮昌宗之狐媚涸斯泉之流而不足以湔其耻俄而白日西匿狞风振沙野羽啸其丛薄寒鸱鸣兮枯槎灌莽杳而无际心惨恻其无涯吾亦按辔而去恐山川之神笑其言之夸也
　　放山鸡赋
　　乆栖迟于山泽厌羽族之珍竒嘉兹禽之自得固殊智于羣飞抱耿介之贞性表瑰丽之妍姿焕绣膺兮丹颈炫披霓之仙衣﨑幽闲之容止逺人境之喧卑栖遐深以是适照沧浪而自怡匪华轩之是乗岂雕笼之能羁聊优游兮饮啄与忧患兮长辞何巧夫之艰险设云罗以见縻旅燕雀而为掳伤羽毛之低摧讵文采之贾祸将狷洁而殃罹敢微躯之自保念畴侣之暌违适野叟之见售动余心之长悲始防布之甚急旋释仪而纵归讵羊公之责报抑简子之是希无求友而遭卖无啄食而触机偶六翮之未铩聊逺举而遐嬉嗟微以喻大聊抒意而成诗
　　勤耕堂赋
　　冨州邹福名堂曰勤耕莆田陈众仲既铭而福请余不已乃为之赋曰浮西江以遐逝择瘠土而归耕躬劬劳而不贰庶善心之由萌时辍耒以太息感先民之所程苟向义之或违谅余心之未寜好风东来兮膏雨宵零升阳润土兮氷涣川盈嘉植阿那兮羽鸣嘤嘤川原髙卑兮锦机红青夜饭牛以悲歌晨拥犂而戴星忘骈胝之作苦幸袯襫之蔽形绵绵秧针衣被清泠荀令置膝兮武子过庭虽娟秀之可爱卒未知其所成既幽栗乎旱魃又明惧乎螽螟谨嵗功之云始分百忧之怦怦狞飇霣叶兮蟋蟀宵鸣黄云之弥望鎌挃挃而风生舍盖避堂祖疎辞荣顾耒耜之在壁犹盘盂之有铭托陋质于昭世旅譲耕之遗氓获同颖之嘉禾数盈亏之阶蓂从君子之好脩羌羽仪于上京躬劳瘁以逾纪亦含芬其吐馨讵桑梓之悠旷愧蒸尝之弗营惟垂颖而顾本指南云而效诚嵗冉冉其云暮渺悁悁之余情収畴昔之逸豫修初服以南征
　　雪竹轩赋
　　大梁王希孟榜轩雪竹师鲁道人为赋曰江空嵗暮僵立万木有伟青士强项不屈初秀挺以立玉倐铮鏦而响筑虚室生白若逃空谷柙镇帷以悬犀裘堆膝而裁鹿拥残炉而敧防甘手以肥粟清声逺响倐断还续起而视之羣仙鹤驭羽衣缟服云情风纛空旷出没岂居家之此君而箨龙之诸族也邪呜呼噫嘻坚贞则有立洁白斯无欲虚心所以持身敷泽所以成物岂惟徜徉以卒嵗葢将继君子之遐躅
　　啸台赋
　　阮嗣宗酒酣登台箕踞甚逸眼空四海神游八极既骋望以慷慨乃长啸而自适声清厉而弥长音磞硠而络绎或徘徊而萦云或激跃而裂石或逺响之澎湃或余哀之剽急云变色而凝飞鸟悲鸣而举翼清飇振而若厉流泉咽而如泣研羣声之至清含自然之钟律涤埃之溷烦聊超然而独立忽仙去兮不返遗兹台之岌岌呜呼噫嘻斯人之不可得也士固有大言而欺世虚名而无实然味言而察其志循名而观其识则真伪之分奚啻白黒也当其恸穷途叹广武岂昏冥之是逃抑遗世而不顾俾其遘辰得位以奋厥志则将伏南阳之轨躅驰东山之极挚惜其懐瑾握瑜乆幽而不试乐邱壑之栖迟衔杯酒以自肆贞而不谅遯以傲世过故台而一吊动志士之歔欷
　　望思台赋
　　出湖城而西骛兮眺泉鸠于中涂吊汉武之肆暴兮悯戾园之无辜忽孤墉之如块兮曰归来望思之遗墟吁赵虏之谗贼兮乱廼国犹摴蒱武雄畧其焉在兮曾不寤夫覆车初犬台之召见兮冠歩摇而垂输固将窜巧以逞志兮帝已湔拭其逃逋知衔毒以肆噬兮寜畏夫宗灭而身屠间茂陵之猜忌兮实惑神而信巫亲文成而溺五利兮耗生民于怪迂挟蛊道以祝诅兮疑左右之是图抑天道之好还兮聊假手乎憸夫彼长平与阳石兮加显戮乎其孥探隐伏以或慊兮诈夜祠以沾濡瘗桐木于宫闱兮卒嫁祸于帝储哀浅谋之狂傅兮爰昔鉴于扶苏盗父兵以戮充兮冀忿忿之或摅独寃结以谁语兮将何适而容躯懿三老与千秋兮销恶运兮无余焚横桥其已晚兮徒兴哀于既徂筑新宫以永思兮魂茕茕其焉如缅先王之储贰兮唯端良之与居峙博望以延士兮杂卫翼以壬谀哀持身之不谨兮卒失势以就诛帝复信譌而杀兮不能慎夫厥初搆祸败之弗救兮征蚩尤以为符兴余悲于千古兮聊盘桓以踌躇陟古馗之谽谺兮盼頽基之蓁芜歌青蝇以一吊兮遂慷慨以成书
　　不寐赋
　　何杪秋之遥夜兮块独处兮心怦感嵗月之徂迈兮蹇邅廻而无成怅反侧而不寐兮纷所忧其相仍聊披襟以容与兮聿发愤而扬清澹苍苍之天宇兮流孤月之荧荧横明河兮素练万籁寂兮无声练余心于元妙离尘垢之营营歌逺游以激烈兮飡沆瀣之宵零存亭毒之正灵兮息扰扰之外慕履孔徳之窕冥兮超遗俗而不顾吾以元气为车兮以防风为御遨翔于太清兮孰云桎梏之未固夫何美人之遥遥兮乗翠云而容与哀吾生之孔艰兮淹踌躇而弗能去及行迷之未逺兮还吾返乎故宇闻至言以斟酌兮幸年嵗之未暮稽同人之先号鉴北叟之后悟聊肆志以从容复何忧而何惧
　　看云赋【为吴闲间作】
　　太虚清明极灵无声风雨霜露性命咸贞日余惟星月遡亏盈昭回在上独云多形或鱼鳞之错错或车葢之軿輧泰山崇朝六合同气目不容瞬倐若掣帜已霑溉乎穜稑复覆防乎弱穉马駞辒辌是知朔方氛祲正祥占台书常何质无而神有而隐显钜细之叵量也元隐子于是延伫而睇之重歌曰我观于天云有物兮轮囷扶舆生忽兮变无化有亦其神兮离有入无反乎真兮九州百物象其精兮一气之嘘龙或凭兮蒸烰歘霱油然昭倬又将焉名兮
　　胡桃木赋
　　吾庐之阳峙嘉植兮受命于天物莫比兮修榦巍峩钜叶旎兮屹如承天中立不倚兮鸱枭不集矧毒螫兮逺彼邪恶斯明哲兮累累佳实荐之珍兮膏以自泽徳润身兮厥材之良剖牺尊兮不青不黄而自文兮林林天壤寓奚异兮或断沟中万乗器兮使然者谁抑偶置兮曷若斯木徳之备兮明告君子以为类兮
　　四言古诗
　　短歌行
　　南山采樵林木未凋北厓多薪何岧嶤白云在巅惊风日飘豺虎相逐猿猴可招长啸倐往睨彼柯条匪材不经中则乃遥荆棘钩衣瀑流喧嚣还顾旧乡心旌摇摇商山采芝巢由避尧絶唱千载清风寥寥五十负薪行歌车尘彼独何心我思古人
　　和溪钓者图
　　流泉幽幽钓丝浮浮风涛既逺云木相缪方之舟之竹竿恒持鲸鲵弗顾鰋鲤何为今焉在溪伊昔望海彼美钓徒志尚千载洪冈之东其谁与同欸乃天际悠然清风
　　五言古诗
　　古诗【一以送克哷景兖入梁其二以谢张子文应举访余伊上兼简汴中诸名胜】
　　山童照伊白日瘦埋云黄雍容车骑都长卿薄游梁文采丽宫锦邹枚敛其光我飘山林牢三见天雨霜公如左右手易亲复难忘交深出苦语惬于冬饮汤夷门二三友问我行与藏长吟归田赋乱以囚山章
