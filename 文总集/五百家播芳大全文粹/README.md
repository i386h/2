　　提要
　　【臣】等谨案五百家播芳大全文粹一百十巻宋魏齐贤叶棻同编齐贤字仲贤自署钜鹿人棻字子寔自署南阳人考宋南渡以后钜鹿南阳皆金地殆以魏氏本出南阳偶题郡望非其真籍也是编皆録宋代之文骈体居十之六七凡表牋制诰简疏赋颂记序铭跋无不毕备题曰五百家而巻首所列姓氏实五百二十家网罗可云极富中间多采宦途应酬之作取充巻数不能一一精纯又仿文选之例于作者止书其字人逺年湮亦往往难以考见疑为书肆刋本原无鉴裁故买菜求益不免失之冗滥朱彛尊尝跋此书惜无人为之删繁举要则亦病其冗杂矣然渣滓虽多精华亦寓宋人专集不传于今者实頼是书畧存梗概亦钟嵘所谓披沙拣金往往见寳者故彛尊虽恨其芜终赏其博也又彛尊所见徐烱家宋刻本称二百卷今抄本止一百十巻寻检首尾似无阙佚殆彛尊记忆未审或偶然笔悮欤首载绍兴庚戌南徐许开序开字仲启以中奉大夫提举武夷冲祐观着有志隠类藁见赵希弁读书附志云乾隆四十五年十月恭校上

　　五百家播芳大全文粹序
　　昔吾从搢绅先生逰而得其论寳之说夫所谓寳者焜燿瑰竒铿锵温润世之美瑞国之重镇周传汉佩虞求罕辞易连城照十二乘曾未得其髣髴也然必聚于沆之巨浸蕴于不可测识之深渊元圃之邃昆山之巅元气钟之造物蓄之然后芒寒色正烁烁乎牛斗之间岂蹄涔之所能【缺】培塿之所能有哉今夫墨庄有天下之书虽三阁四库之储道家蓬莱之所蔵者相捋也故家有蔵书之富钜鹿魏君仲贤南阳叶君子实实徜徉其间储蔵之丰奚啻挿架三万轴而已一日合并且欲集本朝名公襍着之文以恵同志于是各出所有辟饷以居之巨篇奥帖竒书秘字充衍其中以我圣朝之文章冠前古而二君八窗玲珑旁搜逺绍类以成帙凡世用之文靡所不备灿灿珠玑烟烟瑚琏如渊海之蔵如府库之羡吾昔所闻缙绅先生所谓寳者此焉当之矣盖将披袭以氷蚕茧之光荐之以漆仍几之安寕不贵哉余往者守官【缺】阳于书市经从为欵二君走书言其大防属余叙之余不得辞绍熙改元庚戌八月南徐许开仲启序
