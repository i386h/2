　　钦定四库全书　　　　集部二
　　云台编　　　　　　　别集类一【唐】
　　提要
　　臣等谨案云台编三巻唐郑谷撰谷字守愚宜春人光啓三年进士为右拾遗干宁中仕至都官郎中谷父尝为永州刺史与司空图同院图见谷即竒之谓当为一代风骚主诗名盛于唐末人多传讽称为郑都官与张乔许棠诸人同号十哲其事迹颇见计有功唐诗记事中新唐书艺文志载谷所着有云台编三巻宜阳集三巻乃随时裒订分帙各行者今宜阳集已佚不复见惟此编尚存凡上中下三巻所録诗约三百首其名云台编者据自序称干宁初上幸三峯朝谒多暇寓止云台道舍因以所记编而成之葢昭宗以李茂贞犯阙行幸华州时也谷以鹧鸪诗得名至有郑鹧鸪之称而其诗格调卑下实非集中上乗方回灜奎律髓又称谷诗多用僧字凡四十余处谷自有句云诗无僧字格还卑其僻好所在殊不可解要亦其诗中一病而其他佳作尚多往往于风调之中独饶骨力汰其肤浅撷其菁华固亦足为晩唐之巨孹矣乾隆四十四年三月恭校上

　　云台编原序
　　谷勤苦于风雅者自骑竹之年则有赋咏虽属对声律未畅而不无防讽同年丈人故川守李公朋同官丈人马博士戴尝抚顶叹勉谓他日必垂名及冠则编轴盈笥求试春闱歴干于大匠故少师相国太原公深推奬之故薛许昌能李建州频不以晚軰见待预于唱和之流而忝所得为多游举塲凡十六年著述近千余首自可者无几登第之后孜孜忘倦甚于始学也丧乱奔离散坠略尽干寜初上幸三峰朝谒多暇寓止云台道舍因以所记或得章句缀于牋毫或得于故侯屋壁或闻于江左近儒或秖首一聨或不知落句遂拾坠补遗编成三百首分为上中下三巻目之为云台编所不能自负初心非敢矜于作者唐都官郎中郑谷守愚序

　　别同志
　　所立共寒苦平生同与游相看临逺水独自上孤舟天澹沧浪晚风悲兰杜秋前程吟此景为子上高楼
　　送进士卢棨东归
　　防岸草萋萋离觞我独擕流年俱老大失意又东西晓楚山云满春吴水树低到家梅雨歇犹有子规啼从叔郎中【諴】辍自秋曹分符安陆属羣盗狂炽流毒江境竟以援兵不来城池失守例削今任却叙省衔退居荆汉之间颇得琴樽之乐因有寄献
　　华省称前任何慙削一挥沧洲失孤垒白髪出重围苦节翻多难空山自喜归悠悠清汉上渔者日相依
　　送徐端公南归
　　青衿离白社朱绶始言归此去应多羡初心尽不违江帆和日落越鸟近乡飞一路春风里杨花雪满衣
　　送祠部曹郎中邺出守洋州
　　为儒欣出守上路亦戎装旧制诗多讽分忧俗必康开懐江稻熟寄信露橙香郡阁清吟夜寒星识望郎
　　送进士许彬
　　泗上未休兵壶关事可惊流年催我老逺道念君行残雪临晴水寒梅发故城何当食新稻嵗稔又时平
　　次韵和王驾校书结绶见寄之什
　　直应归谏署方肯别山村勤苦常同业孤单共感恩醉披仙鹤氅吟扣野僧门梦见君高趣天凉自灌园
　　秘阁伴直
　　秘阁锁书深墙南列晚岑吏人同野鹿庭木似山林浅井寒芜入回廊叠藓侵闲看薛稷鹤共起五湖心
　　送太学顔明经及第东归
　　平楚干戈后田园失耦耕艰难登一第离乱省诸兄树没春江涨人繁野渡晴闲来思学馆犹梦雪牕明
　　送进士赵能卿下第南归
　　不归何慰亲归去旧风尘洒涙慙关吏无言对越人逺帆花月夜微岸水天春莫便随渔钓平生已苦辛
　　送人游边
　　春亦怯边游此行风正秋别离闻夜雨道路向云州碛树藏城近沙河漾日流将军方破虏莫惜献良筹
　　送人之九江谒郡侯苗员外【绅】
　　泽国寻知已南浮不偶游湓城分楚塞庐岳对江州晓饭临孤屿春帆入乱流双旌相望处月白庾公楼
　　送许棠先軰之官泾县
　　白头新作尉县在故山中高第能卑宦前贤尚此风芜湖春荡漾梅雨昼溟蒙佐理人安后篇章莫废功送司封从叔员外【徼】赴华州裴尚书【均】辟
　　如何抛锦帐莲府对莲峰旧有云霞约暂留鹓鹭踪敷溪秋雪岸树谷夕阳钟尽入新吟境归朝兴莫慵
　　送京参翁先軰归闽中
　　解印东归去人情此际多名高五七字道胜两重科宿馆明寒烧吟船兀夜波家山春更好越鸟在庭柯
　　赠别
　　南游曽共游相别倍相留行色回灯晓离声满竹秋稳眠彭蠡浪好醉岳阳楼明日逢佳景为君成白头
　　南游
　　凄凉懐古意湘浦吊灵均故国经新嵗扁舟寄病身山城多晓瘴泽国少晴春渐逺无相识青梅独向人
　　巴賨旅寓寄朝中从叔
　　惊秋思浩然信美向巴天独倚临江树初闻落日蝉哀荣悲徃事漂泊念多年未便甘休去吾宗尽见怜
　　寄司勲张员外学士
　　平昔偏知我司勲张外郎昨来闻俶扰忧甚欲颠狂烟暝搔愁鬓春隂頼酒乡江楼倚不得横笛数声长
　　寄边上从事
　　男儿懐壮节河外事嫖姚高垒观诸寨全师防大朝浅山寒放马乱火夜防苗下第春愁甚劳君逺见招
　　寄左省韦起居【宇】
　　风神何蕴借张绪正当年端简炉香里濡毫洞案边餙装无雨备著述减春眠旦夕应弥入银台晓候宣
　　寄赠蓝田韦少府先軰
　　王畿第一县县尉是词人馆殿非初意图书是旧贫破氷泉窦响赛雪庙松春自此升通籍清华日近身
　　寄懐元秀上人
　　悠悠干禄利草草废渔樵身世堪惆怅风骚颇寂寥高秋期野歩积雨放趋朝得句如相忆莎斋且见招
　　赠圆昉公【昉蜀僧僖宗幸蜀昉坚免紫衣】
　　天阶让紫衣冷格鹤犹卑道胜嫌名出身闲觉老迟晚香延宿火寒磬度高枝长説长松寺他年与我期
　　寄题方干处士
　　山雪照湖水漾舟湖畔归松篁调逺籁台榭发清辉野岫分闲径渔家并掩扉暮年诗力在新句更幽微
　　寄献湖州从叔员外
　　顾渚山边郡溪将罨画通逺看城郭里全在水云中西阁归何晚东吴兴未穷茶香紫笋露洲逈白苹风歌缓眉低翠杯明蜡翦红政成寻徃事辍棹问渔翁
　　访姨兄渭口别墅
　　枯桑河上村寥落旧田园少小曽来此悲凉不可言访邻多指冢问路半移原久歉家僮散初晴野荠繁客帆悬极浦渔网晒危轩苦澁诗盈箧荒唐酒满樽高枝霜果在幽渚暝禽喧逺霭笼樵响微烟起烧痕哀荣孤族分感激外兄恩三宿忘归去圭峰恰对门
　　放朝偶作
　　寒极放朝天欣闻半夜宣时安逢宻雪日晏得高眠拥褐同休假吟诗贺有年坐来幽兴在松亚小窻前
　　顺动后蓝田偶作【时丙辰初夏月】
　　小谏升中谏三年侍玉除且言无所补浩叹欲何如宫阙飞灰烬嫔嫱落里闾蓝峰秋更碧沾洒望銮舆
　　府中寓止寄赵大谏
　　老作含香客贫无僦舍钱神州容寄迹大尹是同年宻迩都忘倦乖慵益见怜雪风花月好中夜便招延
　　峡中寓止二首
　　荆州未解围小县结茅茨强对官人笑甘为野鹤欺江春铺网濶市晚鬻蔬迟子美犹如此翻然不敢悲传闻殊不定銮辂几时还俗易无常性江清见老顔夜船归草市春歩上茶山寨将来相问儿童竞啓关顔惠詹事即孤侄舅氏谪官黔巫舟中相遇怆然有寄
　　犹子在天末念渠懐渭阳巴山偶防遇江浦共悲凉谪宦君何逺穷游我自强瘴村三月暮雨熟野梅黄
　　访题进士孙秦延福南街居
　　多病久离索相寻聊解顔短墙通御水疎树出南山嵗月何难老园林未得还无门共荣逹孤坐却如闲
　　访题进士张乔延兴门外所居
　　平生苦节同旦夕防原东掩巻斜阳里看山万木中星霜吟欲老江海业全空近日文塲内因君起古风
　　寄南浦谪官
　　多才翻得罪天末穷忧白首为迁客青山绕万州醉欹梅障晓歌厌竹枝秋望阙懐乡泪荆江水共流
　　李夷遇侍御久滞水乡因抒寄懐
　　簪豸年何乆悬帆兴甚长江流爱吴越诗格愈齐梁竹寺晴吟逺兰洲晚泊香高闲徒自任华省待为郎
　　寄膳部李郎中【昌符】
　　鄠郊陪野歩早嵗偶因诗自后吟新句长愁减旧知静灯微落烬寒砚旋生澌夜夜捜苦那能鬓不衰
　　寄前水部贾员外【嵩】
　　谢病别文昌仙舟向越乡贵为金马客雅称水曹郎白鹭同孤洁清波共渺茫相如词赋外骚雅趣何长
　　寄棊客
　　松窻楸局稳相顾思皆凝几局赌山果一先饶海僧覆图闻夜雨下子对秋灯何日无覊束期君向杜陵
　　闻进士许彬罢举归睦州怅然懐寄
　　桐庐归旧庐垂老复樵渔吾子虽言命乡人懒读书烟舟撑晩浦雨屐翦春蔬异代名方振哀吟莫废初
　　长安夜坐寄懐湖外嵇处士
　　万里念江海浩然天地秋风高羣木落夜乆数星流钟絶分宫漏萤微隔御沟遥思洞庭上苇露滴渔舟
　　赠文士王雄
　　知已竟何人夫君尚苦辛图书长在手文学老于身公道天难废贞姿世任嗔小斋松菊静愿卜子为邻
　　赠富平李宰
　　夫君清且贫琴鹤最相亲简肃诸曹事安闲一境人陵山云里拜渠路雨中巡易得连宵醉千缸石冻春
　　赠尚顔上人
　　相寻喜可知放锡便论诗酷爱山兼水唯应我与师风雷吟不觉猿鹤老为期近軰推栖白其如趣向卑
　　赠油口苗居士
　　嵗晏乐园林维摩契道心江云寒不散庭雪夜方深酒共游人饮诗怜稚子吟四郊多垒日勉我舍朝簪梁烛处士辞金陵相国杜公归旧山因以寄赠
　　相庭留不得江野有苔矶两浙寻山徧孤舟载鹤归世间书读尽云外客来稀谏署捜贤急应难惜布衣
　　哭建州李员外【频】
　　令终归故里末嵗道如初旧友谁为志清风岂易书雨坟生野蕨乡奠钓江鱼独夜吟还泣前年伴直庐哭进士李洞二首【李生酷爱贾浪仙诗长江在东蜀境内浪仙冢于此处】
　　所惜絶吟声不悲君不荣李端终薄宦贾岛得高名旅新坟小遗孤逺俗轻犹疑随计晩昨夜草虫鸣自闻东蜀病唯我独关情若近长江死想君胜在生瘴蒸丹旐湿灯隔素帷清冢树僧栽后新蝉一两声南康郡牧陆【肱】郎中辟许【棠】先軰为郡从事因有寄赠
