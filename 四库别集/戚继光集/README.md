王明府园亭

秋日[二首]

苍苍晚木群猿集，历历睛沙一鸟还。

其二
清溪谁与濯尘缨，水气常昏海上城。
北塞晓烟嘶戍马，西风寒浪吸长鲸。
旗翻紫电皇威远，剑跃青虹阃令明。
骄子莫询周鼎重，郁葱佳气满神京。

述竹间妇

高楼临通津，朝暮阅行人。

过文登营

乙卯元日登太平楼

抚松亭
