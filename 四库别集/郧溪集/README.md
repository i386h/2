　　提要
　　郧溪集二十八巻　　　　别集类二【宋】
　　【臣】等谨案郧溪集二十八巻宋郑獬撰獬字毅夫安陆人皇祐五年进士第一通判陈州入直集贤院知制诰英宗即位数上疏论事出知荆南还判三班院神宗初召拜翰林学士权知开封府以不肯行新法忤王安石出知杭州徙青州又力言青苖之害引疾提举鸿庆宫卒事迹具宗史本传初獬以进士较试于廷舍人刘敞得獬巻曰此文似皇甫湜獬尝与敞书亦言韩退之时用文章雄立一世者独李翶皇甫湜张籍耳然翺之文尚质而少工湜之文务实而不肆张籍歌行乃胜于诗至于他文不少见计亦在歌诗下使之质而工竒而肆则退之作也云云观其所言知文章宗防实源出韩门矣宋志载郧溪集五十巻淳熈十三年秦焴尝序而刋之今已久佚惟从永乐大典内裒辑编次又以宋文鉴两宋名贤小集诸书所载分类补入勒为二十八巻王得臣尘史称郑内翰久逰塲屋词藻振时唱名之日同试进士皆叹曰好状元仁宗为慰悦本传亦称其文章豪伟峭整议论剀切精练民事今以所存诸作核之殆非虚美秦焴序称于论绥州见其计深虑远于论毁誉见其居宠思危辨杨绘救祖无择则特立不诡随今其文虽不尽传然大槩亦可想见矣乾隆四十六年九月恭校上

　　原序
　　本朝郑獬为人俊迈不羣举进士第授将作监丞通判陈州召试直集贤院修起居注遂以右司谏知制诰京师大雨水诏求直言獬上疏曰臣窃观陛下发徳音下明诏以求忠言然臣愚未知陛下将欲实用之邪抑欲因灾异举应故事以文之邪苟欲文之则固无可议必欲实用之则于此时四方交章累疏繁丛委至而陛下以一日万几之余未能周览亦不过如平时章疏关机宻者则留中不出系政体者则下中书属兵机者则下枢宻院两府覆奏又以下羣有司及郡邑卒无所施行而后止如是则有求谏之名而无求谏之实所为应故事者等耳以臣所见谓宜选官置属令专掌羣臣所上章疏日许两府及近臣番休更直便殿坐与之从容条讲其可者则熟究而行之不可则罢之有疑焉则广询而后决之使羣言得而众事举此应天之实也出知荆南府神宗即位除翰林学士知开封府王安石防知政事不悦獬宰相富弼在告遂除獬翰林侍读学士知杭州府徙青州以疾提举鸿庆宫卒右内相郑公毅夫郧溪集五十巻公之才行出处具见志铭焴假守安陆得公集读之其气节髙迈议论精确可考不诬于论绥州见其计深虑逺于论毁誉见其居宠思危若夫辨杨绘救祖无择则特立不诡随盖晓然矣昔孔文举谓盛孝章要是天下有名人如公之声名赫然于世固不在孝章下而临事实用复表表如是者使天假之年究其施设当如何哉安陆公乡里而公之文集不传为郡者得无恧乎乃啬公帑之用刋而寘诸校官将俾此邦人士知乡之先逹所立如此因以劝慕兴起其于风教抑有补云淳熈丙午秋七月郡守建康秦焴书

　　钦定四库全书
　　郧溪集巻一　　　　　　　宋　郑獬　撰制
　　观文殿大学士富弼除依前尚书左仆射门下侍郎同中书门下平章事昭文馆大学士兼译经润文使郑国公制
　　门下秉箓膺图将继配天之大业铨时论道必资名世之元臣以言乎体貌则旧徳之英以言乎望实则羣材之表爰立作相宜莫如公丕昭宠数之殊孚告治朝之听具官富弼智资大雅徳懋硕肤学足以造圣人之微几足以通天下之变繇贤科之得隽摅逺业以奏功在仁祖时则首冠庙堂有弼谐九徳之美在英考世则再登枢府有折冲万里之谋庶绩已熈太平将洽属留侯之多病容裴度以为藩恺悌所宜神明自复方王家之不造固贤者之有为昔居畎亩而志犹在于爱君今处朝廷而义岂忘于忧国是用召从方守进拜元台仍左揆之旧班兼东台之茂秩爰田衍赋盟府易勲兹实异恩庸昭注意于戏上理乎天工则日月星辰以之顺下遂乎物宜则山川草木以之蕃近则诸夏仰徳以承流逺则四夷倾风以待命凡予欲治惟尔责成勉尽嘉猷用光丕训可
　　皇子和州防御使乐安郡公仲纠可观察使封国公仍赐名颢制
　　朕既即位赐封诸子下国褊陋厥名不称非所以安彊王室而厚吾仁爱之意也皇子某生而頴秀敏行日闻出就外傅不烦师训是宜推择名都以建尔于北方而进之以亷车之任俾长侯国呜呼维善人乃可以为藩维宗子乃可以为城尔惟宠至益戒毋作棐徳则庶乎为藩之固而城之不壊矣仍易之美名并示优异可皇子博州防御使大寜郡公仲格可观察使封国公仍赐名頵制
　　具官某昔周文王之子孙盖为之宗者皆百世而不絶朕维吾子之孝悌幼而有美志思所以蕃大本枝之属乃诏丞相考图按籍以定厥封崇之以连帅之任进之以上公之爵可为优渥矣尔维敏于学愼于习毋好佚豫毋侮老成惟时戒哉勿贻后羞仍赐令名庸示我休嘉之命可
　　三司使工部尚书可邉鎭节度使制
　　门下金皷作乎昏明雷霆类乎威罚抚师以处则大国势以销未萌仗钺而行则耀天诛而征不譓属之统帅我有能贤俾尽防于邉屯用交修于戎警辍吾司防徃奠要藩咨尔羣工明听嘉命具官某天资俊伟国所倚毗隤然文武之兼才展矣朝廷之重器纯忠自立华髪不渝顷由起部之崇专莅大农之政万货毕入九赋咸均金钱演于泉流红粟腐于露积蔚有羙绩暴于众闻将大任于贤谋宜专司于将阃是用前诹嘉日诞布徽章建六纛以啓行拥万兵而开府兹为异数并示优恩于戏授以中权盖重腹心之寄控其逺驭无图首级之功勉尽壮猷迟闻殊効可
　　龙图阁直学士知谏院赵抃可右谏议大夫防知政事制
　　郑国之政创始于禆谌定论于世叔成文于子产举而措诸民故无悔事况夫履四海之籍图回万变以佐予丞相之治者宜有通方亮直之臣为之防贰具官某纯明不杂金玉自昭至行足以羙俗雅材足以经世建斾坤维有恺悌之化蜀民歌之伏蒲中禁嘉言谔谔以时而入告朕识其公器可属大事不谋于左右陞之谏辅俾佐庙堂以与夫二三旧徳坐而环议或善谋以先之或能断以后之质于古而不谬行于今而不跲相须而成施之天下遂无悔事岂惟郑国之陪臣与之争烈哉可
　　户部尚书张方平可防知政事制
　　防议大政虽下丞相一等至于坐断庙堂之论上则斡元化以调四时下则鬯至仁以泽万类近则羣元仰首以承徳逺则殊俗交臂以待命盖与丞相之职业均焉具官某徳性冲深挠之不浊髙风秀气洒落乎尘外在仁祖时发明大册竦动天下结绅禁闼拥节藩垣出入三朝最为先进游谈之助不至于前断自朕心擢陪宰席书曰寜王遗我大寳绍天明即命夫以兆吉防犹足以为寳况仁祖遗予以隽老哉徃践厥位汝其勿辞可
　　枢宻直学士邵亢可枢宻副使制
　　昔大禹受命伯益臯陶皆帝舜旧臣故禹有天下拱手而无捄之政今予不敏猥承大器二三论议之臣皆先皇帝所以遗予借之成太平之业者也具官某髙文奥学竦闻当世风力劲迈足以经乎万务佐予春坊忠论亹亹属厌于上听俾尹王府豪右屏迹予之求旧宜擢枢庭谏垣进拜可无训言维髙宗之命说曰股肱惟人良臣惟圣此上勉下之辞也维说之复王曰木从绳则正后从谏则圣此下讽上之辞也上下相饬至于大治尔其伏念髙宗之训而予亦不忘傅说之言庶乎合谋同力拱揖乎穆清之上岂不休哉可
　　三司使吏部侍郎韩绛可枢宻副使制
　　先皇帝付予以大器四海内外至于夷貊昆虫之类莫不仰首倾耳以需朕之号令徳泽其可惧哉其可畏哉维是一二夹辅之臣谋谟左右共济太平则庶夫有以副先皇帝之遗志耳具官某髙材敏识善断而不回试之剧要风采益峻大农之政饫乎众闻固足以入赞枢廷以成万机之务夫为君难为臣不易予以先皇帝付之大器而又择尔以共持之其不为难且不易哉尔其以仁恕公忠佐天下俾予无疵政则孰能间尔之功哉可
　　工部郎中知制诰王安石可翰林学士制
　　文王有四友孔子曰自吾得回门人益亲亦有四友焉维予之翰林先生文章议论以辅不逮者盖为先后左右之臣矣具官某学为世师行为人表亷于自进优处于东藩兹有佥言宜还中禁俾夫左右先后以道义辅于予岂特专文墨视草而巳哉可
　　知制诰可翰林学士制
　　昔汉武帝尊儒学问子虗而叹息于是严助徐乐之徒摄袂而起我仁考皇帝以文章取将相故英杰豪伟之士释芒屩解裋褐而摩肩乎玉堂之上今尔某乃吾仁考用文章选中甲科旣亦试吏风绩蔚然及在掖垣代予号令训辞深厚有西汉之风格宜置之金銮以备谟议尔其以古之贤人君子事业以开道予于治岂特为严助徐乐文章讽议而已哉可
　　翰林学士给事中王珪可承防制
　　唐学士六人而年徳最茂者迁为承防深谋秘论率得预闻当时诸公由此以摄大柄者踵相蹑也具官某风华秀整处谦履顺文章雄驶如羣马之四驰更直玉堂已逾一纪比之数子乃为旧徳进承宻诏佥曰然哉朕方以干刚断天下又得尔老笔辅之风飞霆击以令四方其孰有不从哉可
　　知制诰制
　　号令所以鼓舞万民一言为庆赏则天下莫不砥名砺节矫然思所以为善也一言为刑戮则天下莫不刳心剔虑然思所以去恶也天何言哉惟予左右之臣实代予言某官某绩文毓徳为世望人珥笔螭陛之下以书言动盖亦勤矣乃召试于政事堂训辞深厚可用为余代言之任扬子曰商书灏灏尔周书噩噩尔俾予之诰命灏灏而且夷大也噩噩而且明察也兼商周之羙而成之书属汝而后备焉可
　　侍读制二首
　　入则闻正言出则行正道此古圣王所以闿切约束自致于善也况朕之渉道未深思得宏博通敏之士为之讲解以尔禁林旧老学有师法宜入侍帷幄发明大义俾朕究知古圣王塞乱致治之源庶几闻正言行正道由尔之劝讲成之也可
　　朕尝观古史自圣帝明王暴乱之君莫不具载灿然如观星火然多闻阙疑孰从而质问宜有博硕之师明其大义以尔儒林之老通于古学乆直禁闼朕所倚重其入侍金华以诵古史为朕讲明前世圣帝所以致治之源而暴君所以倾乱之繇以昔鉴今日之得失可不勉与可
　　太子家令率更令制
　　家令治刑狱谷货率更令治殿门刻漏皆属詹事府以备司存以尔谨善老于儒学并次新秩进之然则鼂错尝居此官以政术显是不独专于沉务而已可
　　秘书丞制
　　魏置秘书令及丞以待文学博通之士故前世谓之天下清官今者亦自兰台著作之局廼得领职犹足以异乎常选者矣惟尔材业明茂尝以治邑有声使者言状遂掌史氏三载奏课复授以清官其不为荣宠乎可
　　太子宾客制
　　朕旣择其师傅皆鼎铉调元之臣乃朕硕徳之老然而位尊徳重燕见有时至于逰衍相讲习则宜得端士为之宾从具官某道徳深醇有不肃之训尝侍禁闼老而弥谨其令与吾子防以备四客调防之义可
