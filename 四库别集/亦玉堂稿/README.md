　　●本传 

　　沈公讳鲤字仲化号龙江商丘人也十九举于乡夜有奔女拒弗纳中嘉靖乙丑进士改庶吉士授翰林院检讨累晋侍读学士吏部左侍郎拜礼部尚书太子太保公方直恪慎权珰求一剌之投不可得而积诚悟主婉转効谋一事与上可否甞至三四不得忧郁形于食寝得则不敢以为功秦王谊澏请进封其弟奉国中尉谊■〈氵〈巛上歹下〉〉谊漶为郡王将军上报可公力争之上曰此特恩不为例也公曰特恩亦私恩也不为例亦一例也自古帝王未有不欲恩施无穷者顾人情之不可餍而朝廷之泽必有所竭故不得不限之以制平其锡予而示有各足之分上日有法在吾不受私下日有法在不敢言私画一相守尺寸不踰而后天下之愿欲以定若法制一隳请乞纷至臣不知所以善其后矣缮宗藩要例宗藩名封进之庆成王请封其次妃戚畹郑承宪乞恤典翊圣夫人金氏欲以侄承夫爵皆援祖宗法反复力格不行羣臣屡请皇长子正储位不报乃下诏封贵妃为皇贵妃公请并封恭妃恭妃者皇长子所由出也上无以夺光山产麒麟□取观之公言人主好尚其端甚微而传播于四海九州岛伤盛德甚大此物一进则天下之进奇异者踵至矣麟固祥物而毙也则已不祥不祥之物宜桃茢以祓除之而奈何敢达于至尊之前封还诏旨其不依徇取容类如此请复建文年号修景帝实录正文体毁淫祠定庙享礼制上皆嘉纳焉万历戊子以疾乞归辛丑起公少保兼太子太保东合大学士五辞而后拜命上倦于勤久不御朝公特请面见曰人臣不得仰觐天颜是地天不交也其时差中使开矿榷税所在流毒天下多以变告公上章曰探榷之役陛下本以权宜济国用而内臣不能仰承德意滥用羣小虎噬狼吞穷搜远猎矿非取诸山泽税非得之商旅率皆科派闾左以足额激为祸变纷纷未巳今日欲弭乱形必先收入心欲收人心必先罢矿税而撤中使昔唐泾原之变百姓操白梃逐官吏曰复能税间架除陌钱耶臣愚诚恐至于此极也公念时艰列谨天戒恤民穷等十事上之虑不省览书天启圣聪牌焚香祷祀谮者言公诅呪上密遣小内使取观之乃知公献纳之诚首辅沈一贯与公多异议公作协恭约示之一贯不能用奸书之狱起蔓及公公不辩而上亦不疑自张江陵相尝于私宅票拟遂为故事公曰国政决于私室非朝廷之体而机事不密又安知不有意外之虞臣不敢也朝臣捐俸钱助工作公曰吾知养廉不知逢君之欲闻者愧焉上每以疆场捷闻推恩宰辅公皆固辞曰此师武臣万死一生之所致臣不敢贪天之赐攘人之功公居政府四年以小心持大体老谋远虑不图近小之益汲引天下贤士大夫惟恐失之有嫌怨于公者则过而辄忘恢恢有并包之度每晋秩惕然以盛满为恐七乞骸骨乃得致位还里辟自适园与二三故人修文雅社崇尚俭质立社仓设义塾鳏寡癈疾无告之人尤极周恤而使之必得其所公无子有代求好女盛装资奉公公峻却之病将革犹进保泰之谟皆切中时事年八十余薨上震悼辍朝赠太师谥文端着亦玉堂文集若干卷

　　●序 

　　沈文端公旧有亦玉堂稿十卷续稿八卷新城王大司马刻之鼎革时化为煨烬予拾其残阙是正其□鱼合而汇为十卷与同志授之梓而序其端曰天地万物一生理之所旁礴也然而天地万物之生理实旁礴于吾心顾圣人之道不明而学者中于私利此心渐消渐亡常至于蠧天地贼万物者何也盖不胜其蔽焉而无学以袪之也袪其蔽吾心者则吾心之生理亦还旁礴于天地万物文端公相神庙时盖已百年之久其间立名之士澌灭于飘风逝水者不知凡几而公于今父老犹能颂其泽学士相与慕其义此岂能袭而致之哉兹考其遗集有言必尽有诚必竭上忠??貇于君父而下视天下之恫瘝若切已肌肤之痛虽不必能尽行其志而原其本心盖古良臣不过是矣即其处家国党闾之间蔼然一恻怛之所流行周徧而惟恐有伤其诸所谓一心之生理有以旁礴于上下民物而无间欤谓非学焉而袪其私利之蔽不至此虽生不及同公时无由一问其所以为功然而读公之书犹可以见公之心因公之心犹可以自得其心也公随事立言不刻意纂组以立文章名而天下亦未有知公之文者其文则真以朴醇以法也如此彼世之操觚为专家业者抑又何如也哉康熙庚午夏四月刘榛题

　　○请复建文年号立景泰实录疏 

　　窃惟人君有位号则有纪年有政令则有实录此春秋不易之法自古及今无有以兴亡隆替而因革予夺其间者我朝自太祖开基列圣相承金匮石室之藏具在乃建文以革除而槩称洪武景泰以分附而并系英宗则皆为我朝阙兴矣所据司业王祖嫡有循史职修缺典之奏臣等请先述其畧而后及所以当正之故为皇上陈之我太祖高皇帝在位懿文太子先薨至洪武三十一年大孙嗣位改元建文在位四年成祖文皇帝靖难践位乃削去建文年号仍以洪武纪年及重修高庙实录遂将建文实历附作洪武虚数此则革除之太畧也正统十四年北□入犯英宗睿皇帝躬驭六师自行天讨至士木北狩景皇帝时为郕王奉皇太后命监国旋正大位次年八月英庙回銮后七年复辟是为天顺元年其时奸臣石亨辈奏将景皇帝废为郕王加谥曰戾成化元年修英庙实录遂以景泰事迹附于正统之后天顺之前注曰郕戾附至成化十一年宪宗纯皇帝追体英宗木意始敕廷臣复景帝位号上恭仁康定尊谥改修陵寝景泰七年事迹未及厘正此则附录之大畧也因循以至于今非谓其不可而遂已之也夫位号既复则实录自当改正许其大不许其细是未体夫英庙之心而已矣伏读成祖登极诏书不过以建文四年为洪武三十五年然犹称为少主未闻降削位号是在成祖亲亲之心亦必有不忍绝者而一时宣力归命诸臣或务张功伐或苟存形迹遂赞成革除之事其亦未达夫成祖之心耳夫成祖奉天靖难再造邦家正使年号不除何损万一而曲为掩讳若此且天下后世各有耳目安可尽泯稗官野史各有纪载安可尽革此不但无益于事适足示人以疑故议复革除者非为建文为成祖也议更附录者非为景帝为英宗也兹皆所不必讳者也方今圣明在上修遗举坠以弘先德正在此时臣等不敢为更张之议惟将英庙实录中间七年事迹名曰恭仁康定景皇帝实录无相混淆如斯而已至于建文位号诏书中原无降削之文今亦无凭议复亦惟于高庙实录之末摘出四年事迹复称建文年号如斯而已夫以我太祖甫定天下即首命儒臣纂修元史又追谥其主为顺帝渊哉圣心至公至厚矣成祖英宗之心同符太祖而建文景泰又非胜国之君之比也元主可谥何忍没其号于建文元史且修何可阙□鎶于景泰且景帝之位号既可复则建文年号亦可复建文之死事诸臣且蒙我皇上之恤录而况其君乎况其为太祖之嫡胤乎慰祖宗在天□灵伸寰海久抑之意彰微显阐幽之烈寓兴灭继绝□仁斯举也实圣德圣政之第一事也独可称昭代之信史而已耶惟巨典湮于累朝而一旦顿复公论关乎万世而传信无穷皆出自人主独断非臣等所敢擅拟伏候圣明裁定 

　　○乞停取麒麟疏 

　　该文书房口传圣旨闻河南产有麒麟抚按官如何不奏着礼部传与他速进来看钦此臣等仰惟我皇上之意不过以麟为异物世所未见欲取而观之以知夫麟之所以为麟者之状如何非有贵奇好异之意也为臣子者承君之命奔走不遑岂暇有言以及他事惟麟产河南光山县而臣鲤河南人也知之颇详不及今预言其状或奏进之后无所可观则臣为缄默取容而陛下有误举之悔臣等虽九死不足以赎罪故敢据实为皇上言之盖麟之生本自于牛故其形犹在乎麟与牛之间初生虽有鳞甲不甚显著非如图画所传出于好事者之手而楚楚可观也踰日而毙至于今又复数月窃意其形已枯槁必不如其初之犹有麟之状也陛下方轸念元元整饬吏治日勤思于三皇五帝之业如将不及忽涣发大号布于下国以征一概槁之麟无乃骇人观听乎夫人主之有所好尚举动以昭示于天下也其端甚微而及其树之风声传播于四海九州岛之外使天下皆耳而目之则其究有不可胜言者故一物之微而责以不奏则四方之奏奇异者且纷至焉督以速进则四方之进奇异者又且纷至焉此寻常之所谓无妨而登三迈五如我皇上则一举一动有不可不慎焉者也夫物固有以祥称者迨其毙也则已为不祥之物不祥之物所宜用桃茢以祓除之而奈何敢达于至尊之前臣等窃意抚按官之不敢奏进及臣之不敢遽有征取亦惟以慎重其事而非敢拒违也惟陛下俯鉴臣愚特赐停止幸甚
