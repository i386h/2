　　提要
　　【臣】等谨案文毅集十六巻明觧缙撰缙字大绅吉水人洪武戊辰进士永乐初官翰林学士出为江西参议改交阯为汉王髙煦所譛下狱死事迹具明史本传缙所着有白云稿东山集太平奏疏等书殁后多散佚天顺初金城黄谏始辑其遗文为三十巻后亦渐湮嘉靖中同邑罗洪先复与缙従孙相辑成十巻千顷堂书目又载有似罗隠集一巻学士集二十巻今并未见此本十六巻则康熙戊戌其十世孙悦所补辑也缙才气放逸下笔不能自休当时有才子之目迄今委巷流传其少年夙慧诸事率多鄙诞不经故李东阳懐麓堂诗话谓其诗无全稿真伪相半葢出于后人窜乱者为多然其中佳句间存亦复不减作者至其奏议如大庖西封事白李善长寃诸篇俱明白凯切黄汝亨狂言纪畧诋其文议繁偶使当贾长沙直是奴苛矣又案大庖西封事中有云陛下好观韵府杂书抄辑秽芜畧无文彩若喜其便于检阅愿集一二儒英随事类别勒成一经云云其后成祖修永乐大典缙实为总栽官果用分韵编类之法勒为巨帙一切遗文坠简竟赖以传于今日以待

　　文毅集原序
　　大江之西山川之灵秀最多者由得天地盛大流行之气为多也山川之灵秀磅礴而郁积必钟而为人蕴乎道徳发乎文章着为事业以炳燿天下而埀万世者非得天地之气为尤多者乎惟其得是气之多实为雄伟俊杰宏博硕大之才以埀丕隆之运而昌其所得盛大流行之气其宏辞钜笔足以铺张神藻纂叙鸿业铿乎有声熠乎有光弥满宇宙超轶前古使天下知朝廷之大远夷知中国之尊后世知今日之盛其为用甚博者不本于是气之所为乎自有元运隳光岳之气分至国朝始混一西江山川灵秀所钟而吉水解先生大绅出焉文章始复大振故在当时天下之广人物之众以及四夷之远千百世之悠久皆知有先生者又非本诸文章之发见乎古人之文见于经者是也岂雕琢而为之哉其出言吐辞于论事之顷皆成文章葢以太朴始散淳厐大雅之气在人犹多故也汉唐以来治虽不如古而述作乃或有古之风者抑幸而际夫天地之气漓而复合也以天地之气混合而为一未有盛于今日如先生岂非得盛大流行之气于混合之日而西江山川孕灵毓秀之所生者乎谏在童稚始知人事时即知先生名先生先谪河州河州去敝邑才二百里谏生也晚恒以未识先生面为平生恨每得断纸防墨以为珍玩日临写之如接先生于几席之上暨得所作上高庙书六十余事及五七言诗歌读之其论切时务其言格君心其忠犯人主其学之赡才识之高如布帛菽粟充溢庾藏视金玉虽贵锦绣虽美而卒有益于人国之用莫若是也其诗如古体诸篇使入太白集中孰别其为近时之作其言非雕琢其意卓然有见而非泛其气象严峻凛然而有不可犯之色也岂非一代雄伟俊杰宏博硕大之才也与及来京师访其全集于先生仲子中书舍人正亮家谓先生之于文有求輙应下笔滔滔不待思索虽千万言顷刻立就诸作流播天下遗稿不复尽拾今存万无一二于是遍求大夫士家及闻人口诵而录之共得古今诗稿百五十五首文杂体共七首为之校编成集分为六巻乃积所得俸资托宗友用和誊刻五七言絶句五十首因被命修史日趋馆阁不暇为南京国子监吴祭酒与俭闻之复寄诗一集及御医胡秉常处又得序记十余首所未载者仍为并入又增六巻自喜所得之富葢有待以完也今年春皇上光复宝祚谏叨进侍读职朝回无事欲毕前工无防使交南命又下矣兹功不果就重为慨然秋九月使自南回过广西龙川太平至横州贵县皆先生经行之地于致仕萧教谕鳯岐诸君处共得杂文六十余篇诗百首梧州又得广西何布政孟焕所寄诗数首曩尝以类广而未考其作之先后兹于舟中按地里岁月远近而次第之共成三十巻先生虽终交广防议而学士则天下人所共称而易知者也因名曰解学士先生集先生于洪武戊辰自吉来京师时中江西乡试第一隐然名重公卿间暨登第即防宠眷授中书庻吉士日侍左右谘访政事补禆甚多尝进封事切中时宜皆见采纳时先生年方十九毅然以天下自任虽老成练达莫之或先比之贾谊尚少而才亦优故高庙之待遇又远过文帝虽百绛灌之毁而终莫易帝意此高庙洪度如天地又岂汉文之所及哉奈何龙去鼎湖而竟遭权臣之害逺谪西陲文庙正大统首召为翰林侍讲寻陞学士凡朝廷诏勅及大制作大经典皆出先生手与凡大政事大议论有关国家生民之重者先生皆与之建储之议羽翼之功尤多其知之深信之笃任之専一时文臣未有及者虽在位不久而道行当时功施国家泽被生民声名之在天下虽遐方异域以及千百世之远无不知之初非待文而后显也诚以先生得之天地钟于山川本诸道德着为事业以炳耀天下而埀后世者必于是焉发之则先生之可传者未必不在是也呜呼西江山川所钟前乎公而有欧阳修文天祥数百年后复于先生见焉使先生久于其位其匡君泽民不在二公下使在徳祐景炎之日其精忠大节凛然不可夺又未必不与文山相颉颃也观平日之直言正论如所谓事有不穷之变必有不易之主与全躯不学禇渊生嚼舌甘为杲卿死之语则先生之心可见矣余虽未识先生而于先生之文见先生之心读者尚于此求之明天顺元年冬十月望日赐进士及第奉议大夫尚宝司兼翰林院侍讲金城黄谏序古今人论文曰理为主曰以载道曰昌其气大要不出孟子知言养气而已古之人动静语黙无非文者岂操觚翰而后为文也哉后世之文愈趋而愈下非文之下也文与行岐而二之也今翰林解公大绅自生七年以来至于今其文若干巻一时耆俊皆属而序论之余幸同朝读之徒不释手而莫措一辞其间深自咎其言未至也因公之行以序论公之文庻防不至于谀亦不至于诬也闻公自幼能言惇敏超絶郡守令至其家或抱置膝上应声成文皆错愕惊叹稍长小心静宻识量过人知读书为文皆不劳而习既敏而益勤父母惟其病之忧也时时劝之少息为文口占操笔皆立就未尝起草其简才数语已足其多至万言不穷命一题一心不厌倦而思益精到崇正学而辟异端不为支离诞谩之说不为艰难辛苦之态随事著作洞视万世而耻观摹古人求其归则未尝异于古人也此钟山川河汉之灵南方极盛之会天资之明大非人所可企而及矣甫十九江西以第一人贡于京师士大夫素闻其盛名相期会连袵结驷往见公应答举止如平常不以为异既登高第擢巍科上及东宫竒其才眷注之隆前此未有即入中书知制诰兼翰林内外制皆掌之谒告时刻上輙问安在尝与闲语屡上封事上輙自持入其论思献纳亲故莫闻而中正才敏两制皆自以为莫及也廷辨直言闻者肃然其自处犯而不校居常恂恂尤善开导人心奬劝后学休休乐善爱人之心一息不间浩然之气发心由理所以年甚少而巍然有硕大之望又尝自诵曰处其心常在熈春丽日之间则天下无可怒之人咸以为名言公之文所以汪洋大肆而无龃龉其间正直闳深而无偏陂之失也兼说万有贯通而时出之浚其源于六经要其归于周孔虽不求工文与行如影响之出于形声也浑然天成卓尔出类集义养气孳焉未已余何足以知公之万一哉公名缙大绅其字也其族自唐家吉已七百余年每举进士必父子兄弟聨叠以为常至宋有讳元者破乌珠累功至封王元初从祖讳诚连三世国公其讳真我者以名进士死节赠中书平章其祖也以名进士勅修宋书讳观者其从祖也其父元末辞参知政事为世大儒学者称筠涧先生公生名家道授有绪姿淳頴秀望之玉立天子爱惜其徳操材量与其楷书至亲为之持研行草精妙絶伦善议论古事皆如目覩初与伯兄纶俱登第履厯华要逺方异域皆能诵说其名而天子毎临羣臣诵以为荣且朂励诸王贵人皆注视之余非欲评公之文与行事如此以见公之为文天资学问充积发见不能自己而至于如此初非若操觚翰而学为文者之所能至也明礼部侍郎襄阳任亨泰序
　　始予游东山经三麓而后跻浮黎浮黎者东山之巅也据是返顾培塿层叠环以百数当是时苐谓高厚必资积累理固然耳后数年友人相期于衡百里而近望其高与东山似即而仰攀百折萦纡穷日乃至祝融之峰自峰顶下视旁无倚附縁麓一谷横四十里葢防坤维入汉纪特立而博敷诸山非其伦也又数年入匡庐其它不异东山独五老峭峙得衡一体于是悟曰物固有至钜者乎彼名岳者山之至钜不由积累而高厚者也匡以五老亦得名岳譬之形貎两人戴立岂不矗然而不可为钜偏长拘行无以语圣贤之大亦犹是也夫圣贤至矣彼略毁誉轻利害视天下事无不可为而不可罔以世俗之见故卒应王佐而称豪杰若是者亦谓钜人春雨解公防之方高皇挥斥英雄濯拭宇宙此何时也而公未弱冠天眷独隆成祖之初契符鱼水观其应制寓讽封事犯顔有郑公之正乳儿朝贵敝屣爵位有沅湘之竒忤权蹈危投荒厉节有太白之迈保储忘身徙家戍邉有柬之之烈是果积累得之否乎即使未优于圣域亦当不失为豪杰何则其才固自殊也世之知公者浅类以词翰赏之至论平生莫定题品伏闻仁庙尝曰人言解缙狂缙非狂士呜呼非日月之明哉公亦有言宁为有瑕玉莫作无瑕石斯固其自况矣公防祸既酷藁附外氏洪先再过其下欷歔叹息不忍輙去会柱史遵化古松段君来江省吊古采言檄县礼葬将刻其遗文以传公天才逸发援笔万言不事属藳而又经籍没故多散亡从孙桐缉录凡十巻求加诠次稍剔其伪不尽删繁亦以见公不屑屑尺寸间所谓不由积累一验也呜呼览斯集者其辨人中之岳安在无徒高卑之校也哉嘉靖壬戌春邑后学罗洪先撰
