　　传録纪事
　　【臣】等谨案唐文粹一百巻宋姚铉编陈善扪虱新话以为徐铉者悮也铉字宝臣庐州人自署郡望故曰吴兴太平兴国中第进士官至两浙转运使是编文赋惟取古体而四六之文不録诗歌亦惟取古体而五七言近体不録考阮阅诗话总载铉于淳化中侍宴赋赏花钓鱼七言律诗赐金百两时以比夺袍赐花故事又江少虞事实类苑载铉诗有疎钟天竺晓一雁海门秋句亦颇清逺则铉非不究心于声律者葢诗文俪偶皆莫盛于唐盛极而衰流为俗体亦莫杂于唐铉欲力挽其未流故其体例如是于欧梅未出以前毅然矫五代之与穆修栁开相应者实自铉始其中如杜审言卧病人事絶一首较集本少后四句则铉亦有所删削又如岑文本请勤政改过疏之类皆文苑英华所不载其搜罗亦云广博王得臣麈史乃讥其未见张登集殊失之苛惟文中芟韩愈平淮西碑而仍录段文昌作未免有心立异诗中如陆蒙江湖散人歌皎然古意诗之类一概收之亦未免过求朴野稍失别裁然论唐文者终以是书为总滙不以一二小疵掩其全美也乾隆四十六年正月恭校上
