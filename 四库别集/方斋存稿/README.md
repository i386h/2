　　提要
　　【臣】等谨案方斋存稿十巻明林文俊撰文俊字汝英号方斋莆田人正徳辛未进士官至南京吏部右侍郎諡文修事迹具明史本传湛若水所撰神道碑载所着有方斋存稿世无刋本此本乃其家藏旧抄疏表序文襍着九巻诗一巻史称其文章醇雅隽永今观其诗亦舂容恬适不事雕琢
　　国朝朱彞尊辑明诗综乃独不载之当由未见此本非置之不録也又近人郑王臣辑莆风清籁集所选文俊诗尚有彭城夜泊七律一首送黄主簿赴蕲水七絶二首为是集所未收未知王臣何自得之王臣即莆田人于文俊为乡里或墨迹流传据以载入欤今世传明南监板二十一史即文俊所校刋窜改舛讹颇为后人訾议然文俊为祭酒已在雕板将竣之日陈骙馆阁续録所谓经进不经修者未可以是并訾其诗文也乾隆四十四年三月恭校上

　　进二十史表
　　南京国子监祭酒臣林文俊谨以校刋二十史进呈圣览者前闻具在垂圣王稽古之资遗籍复完徴昭代右文之治载蠲令日辄冒宸严冀少禆于燕闲或可广于流布臣诚惶诚恐顿首顿首惟明君务学为急而信史载事尤详顾兹歴代之全书贮于成均之两库上自邃古下讫朝元二十家笔削之功千百年行事之实博大纎悉之毕具体裁义例之攸存但歴时既久而踳驳滋多故补缺虽勤而舛讹亦甚渐致前功之弃深贻后学之忧幸文运之诞兴际圣人之在上好学夙成乎天性崇文聿采乎人言遂俾愚臣并辱隆委因知盛事有待熈朝南宫发四库之书仍令博求于士类起部备三品之用兼容借给于司徒盖欲昭一代之人文奚暇惜千金之重费先是祭酒臣张邦竒司业臣江汝璧实共领之既而臣文俊司业臣张星踵成而博士臣程煌助教臣刘世龙臣聂曼臣王昆臣许尧学正臣黄良弼臣邹鲁学录臣张杰虽各委任不一亦皆校勘惟均功绩三年深慙驽骀之钝书尘乙览犹恐鱼豕之讹兹盖伏遇皇帝陛下圣智天成文明日丽志存康济学有缉熈谓载道固存乎经而考古必资于史泛观前代博究诸家见残编断简之失真恐往事遗言之莫考乃以校修之责特委文学之臣晋魏隋唐等书残者则补史汉辽金之板腐者重刋岂徒为秘阁珍蔵垂于不朽抑将使寒乡晩出睹之争先吾道重光斯文有托臣等材虽谫薄学本空疎忝职贤闗每恨无阶以报塞滥竽史事但知殚志于校雠移牒鸠工抡材分局职兼六馆书购多方茍中间有一字之疑必往复尽诸臣之见虽勤劬于夙夜漫淹歴于嵗时幸而毕工兹为完璧装盈缃帙随书各识其标题别以牙籖开巻不烦于检阅仰呈天陛庻遍儒林伏愿念建事本乎多闻命讲官时以进读凡其善之当法恶之当惩与夫治所由兴危所由兆务究是非于既往用资法戒于将来车同轨书同文绵一统舆图之盛徳可大业可久垂千年汗简之光臣下情无任瞻天仰圣欣跃屏营之至谨以二十史随表上进以闻
　　谢重幸学表
　　国子监祭酒臣林文俊等嘉靖十二年三月十三日钦防圣驾临幸大学谨奉表称谢者臣等诚懽诚忭稽首顿首上言伏以典礼更新百代仰大君之有作辟雍临幸四方占文运之方隆载举旷仪聿光旧典懽均朝野庆协天人兹盖伏遇皇帝陛下圣由天纵徳与日新彛伦叙而五典以惇礼乐兴而百度咸正况兹首善尤在留神默契圣心大祛俗见谓孔子称王本非礼宜正先师之名陋元人塑像为不经特遵皇祖之训别祠启圣聿明父子之伦配食先贤尽去公侯之号礼无僭渎道益尊崇顾兹典之鼎新烦至尊以贲重当薄海同文之日适践阼一纪之初乃涓令辰爰脩盛事六龙捧御逺戾止于胶庠万乘执圭亲释奠于先圣崇儒重道稽古诹经喜溢章缝光流简执豆笾以奔走乃有三氏之子孙圜桥门而聴观奚止六馆之英俊瞻威严于咫尺幸法语之丁宁建皇极以敷言斯其至矣观人文以成化是之谓乎臣等职忝师儒躬逢盛美幸睹大人之虎变愿先多士以骏奔赐坐而讲易书觉叨荣之已过教人必先忠孝誓图报于将来伏愿配乎天配乎地皇朝绵一统之图作之君作之师天子介万年之夀臣等无任瞻天仰圣激切屏营之至谨奉表称谢以闻
　　经筵讲章
　　孟子曰为政不难不得罪于巨室巨室之所慕一国慕之一国之所慕天下慕之故沛然徳教溢乎四海这是孟子离娄上篇说为政者当脩身以服人心的意思巨室是世臣大家慕是心悦服孟子説道人君为政于天下其道不难在不得罪于巨室而已盖巨室之心难以力服而国人素所取信茍君心未脩所行的事多乖戾则国人怨怒必自世臣大家始所以人君为政先要正心诚意使反之于身无声色货利之累而达之于政得刑赏举措之宜则巨室见之自然悦服而无怨怒这便是不得罪于巨室巨室既慕则一国之人视巨室为向背的亦皆悦服而慕之一国既慕则天下之人视一国为向背的又将悦服而慕之人君为政能使自近及逺无一人不服如此则推无不准动无不化而吾徳教之行无所留碍自沛然充溢四海而天下之民无不被其泽者其于为政也何难之有臣考战国之时王政不纲巨室擅权当时诸侯不务脩徳以服其心而欲以力胜之反以取败孟子有见于此故为此说以救之可谓探本之论矣况人君一身天下之本茍其身正则礼乐征伐出自朝廷四海之内无不用命何忧乎巨室如其身不正则众叛亲离匹夫匹妇或能胜予岂特巨室为可畏哉此人君所以贵脩身也恭惟皇上以天纵之资绍列圣之统即位以来励精图治日御经筵以熈圣学日览章奏以决大政洪范有序敬一有箴脩身之功可以匹休尧舜而迈迹汤武矣尤愿法天广运慎终如始使徳教之施广被于四海运祚之隆永垂于万年臣等犬马之诚无任惓惓颙望
　　孟子曰养心莫善于寡欲其为人也寡欲虽有不存焉者寡矣其为人也多欲虽有存焉者寡矣这是孟子悯世之人溺于私欲而丧其本心故发此论所以示人以养心之要也欲是嗜欲存字不存字都指此心说孟子説道凡人有生便有欲如目之于色耳之于声口之于味鼻之于臭四肢之于安佚皆是也是虽人所不能无然多而不节便流于邪僻而丧失其本心故欲养此心其要莫善于寡欲何则欲者心之累也茍其人能寡欲矣则此心不至外驰自然收敛在这腔子里足为万变酬酢之主其有不存者寡矣茍其人多欲则昏昧放逸身虽在这里此心却与物欲俱驰其有存者亦寡矣臣尝论之人之一心只是个天理而已若私欲则生于形气之私原非性中所有但是二者尝交战于胸中迭为消长如长了一分人欲便消了一分天理彼愈长则此愈消甚可畏也是以圣贤教人时时要存养省察克治使欲心不留则此心莹然浑是天理清明纯一之体不亏而应事揆物自不失其当然之则孟子提出寡欲二字于心学上极为亲切为人君者尤不可以不知也何也人君尊居九重富有四海求无不得欲无不遂尤易纵欲自肆且如声色货利游畋土木征伐之类茍其心溺于此其为害岂可胜计哉是故舜圣人也其臣臯陶戒之曰无教逸欲有邦武王亦圣人也其臣师尚父戒之曰义胜欲者从欲胜义者凶夫二臣者岂不知舜与武王必不至于纵欲哉正以心之难操欲之可畏所以防之于豫而杜之于防也其虑逺矣仰惟皇上聪明睿知得古帝王心学之正传以敬天法祖为心以求治保氏为务心不耽声色之娱目不接玩好之具虽诗书所称何以过之然临御有年而治效犹若未臻比者星变地震水旱蝗虫之灾尤不能不廑圣虑也伏愿详味孟子此章之言深究前代得失之故清心寡欲务学亲贤内外交脩跻于一徳则治本既端治具毕举唐虞三代之盛不难致矣臣犬马之心无任惓惓
　　元首明哉股肱良哉庶事康哉这是虞书益稷篇内史臣记臯陶赓歌告帝舜的说话元首在上指君说股肱在下指臣说良是善康是安臯陶因帝舜作歌责难于臣于是赓歌以责难于帝説道君为臣之元首臣为君之股肱势虽结于上下义则一体相须诚使为元首者智足以知人明足以照物贤者必用而不贤者必退有功必赏而有罪必罚譬之日月当空无所障碍而万象洞然毕照人君之明如此则为之股肱者皆将奉公守法莫敢偷惰以隳职业而布列朝廷者皆蔼蔼之吉人矣股肱有不良乎股肱既良则能为君分理庶务以礼则备以乐则和以教则行以养则遂凡大小政事莫有废堕不举之患矣庶事有不安乎由是观之君明则臣良而天下之事可不劳而理也臯陶此歌虽若归美于帝之词而责难之意寓焉其忠爱至矣臣尝因是而考有虞之时若舜之明四目达四聪元首何其明哉其臣如禹臯陶稷垂伯夷防龙皆一时之贤济济相让股肱何其良哉明良相逢都俞吁咈于一堂之上志同道合所以成有虞之治而后世鲜俪也仰惟皇上聪明天纵刚徤日新君临天下一纪于兹无一念不在乎敬天勤民无一时而忘乎亲贤讲学道冠百王而不自以为圣智周万物而不自以为明尊礼耆旧信任忠良在廷文武羣工咸思竭股肱之力以祗承休徳明良喜起之歌于变时雍之化复见于圣明之朝矣臣等何幸躬逢其盛
　　六三徳一曰正直二曰刚克三曰柔克平康正直彊弗友刚克燮友柔克沉潜刚克高明柔克这是周书洪范篇箕子演三徳之畴以告武王的言语六三徳是大禹所叙九畴中第六畴正是无邪直是无曲克字解作治字箕子说道人君建极于上既足以为臣民取法然无三徳之用却不能使人都归于极故大禹叙畴皇极之后即继以三徳三徳之目第一曰正直是以正直之徳治天下第二曰刚克是以刚徳治天下第三曰柔克是以柔徳治天下盖正直者无为而治圣人之本心刚柔者赖之以辅治圣人不得己而用之故一曰正直二曰刚克三曰柔克平康是天下有道之世彊弗友是彊梗弗顺的人燮友是和柔委顺的人沉潜是沉深潜退的高明是高亢明爽的箕子又说三徳之用各有所宜人君如遇平康之世人皆会极归极遵义遵道这是好时则惟持正秉直建极以先天下而己不必事乎矫拂如遇那彊梗弗顺可以由极而不欲者这是习俗偏于刚的人则必董之以威使他有所畏而归于极是以刚而克刚如遇那和柔委顺欲由乎极而不能者这是习俗偏于柔的人则必予之以福使他有所慕而归于极是以柔而克柔又有一等气禀柔弱沉深潜退不及乎极者却要激励奬进使他企而及之是以刚而克柔有一等气禀刚方高亢明爽而过乎极者却要防闲裁抑使他俯而就之是以柔而克刚夫人君三徳之用抚世酬物因时制宜如此则天下之人虽有溺于习俗囿于气禀者至此悉归于皇极矣三徳乂用其効不既大哉臣按洪范三徳正直之用一刚柔之用四盖人君之徳有刚有柔犹天道有隂有阳隂以敛之阳以舒之而后嵗功成万物遂人君之治天下何以异于此哉歴观古昔圣王所以能成天下之治用此道也汉唐以来人主不知三徳之用故或偏于刚如汉宣帝之残忍或偏于柔如汉元帝之懦弱失君道矣治不古若何足怪哉仰惟皇上秉干刚之徳普离照之明即位以来刑罚不遗于奸究徳泽遍及于惸嫠刚柔相济寛猛适中其于所谓三徳盖已允蹈之尤愿终始此徳终始此治纳斯世于皇极绵宗社于万年天下臣民不胜至愿
