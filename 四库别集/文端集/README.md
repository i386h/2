　　文端集原序
　　余自束髪学为诗今自顺治已亥年以迨于康熙壬申约略凡三十四年存其诗若干首为二十五巻自幼至老多好言山林农圃耕凿之事即与人赠荅往来游歴之所至亦不能离乎此迨年五十以后山林之思益迫引退之思愈急每不惮其言之重复而恒苦出于不自觉殆欧阳子所谓年益加老病益加衰而其心渐迫其言愈多者欤余自弱冠即抱此志每见才隽之士著作非不多当其言廊庙则志躭轩冕言山林则志躭邱壑一巻之中忽而慕夔龙忽而慕巢许乍浓而乍淡倐近而倐逺情随境迁心与物移令人读之而茫然不知其志之所在夫诗以言志虽中更出处进退而无中变其志之事洵如此则其诗可知矣则其人可知矣余诗谫鄙固多重复而自少至老止言其志之所在而无暇计论工拙聊可以免于读其诗不知其志之所在云尔敢云望古人堂奥哉既又辑癸酉以后诗以年月为次序为笃素堂诗若干卷康熙甲申年三月望日双溪英自序
