　　钦定四库全书　　　　　集部三
　　苏魏公文集　　　　　　别集类二【宋】
　　提要
　　【臣】等谨案苏魏公文集七十二巻宋苏颂撰颂字子容南安人徙居丹阳庆歴二年进士官至右仆射同中书门下平章事罢为集禧观使徽宗立进太子太保累爵赵郡公卒赠司空魏国公事迹具宋史本传集为其子携所编宋史艺文志陈振孙书录解题皆作七十二巻今本与之相合盖犹原帙惟艺文志尚载有外集一卷而今本无之则其书已佚也史称颂天性仁厚宇量恢廓在哲宗时称为贤相平生嗜学自书契以来经史九流百家之説至于图纬隂阳五行律吕星官山经本草无所不通叶梦得石林燕语亦载颂为试官因神宗问暨陶之姓颂引三国志证其当从入声不当从洎音神宗甚喜是其学本博洽故发之于文亦多清丽雄赡卓然可为典则石林燕语又称神宗用吕公着为中丞召颂使就曽公亮第中草制又称颂为晏殊諡议以其能荐范仲淹富弼比之胡广谢安又称颂过省时以歴者天地之大纪赋为本塲魁既登第遂留意天文术数之学陆游老学庵笔记又引颂起草才多封巻速把麻人众引声长之句以证当时宣麻之制徐度却扫编又称颂奉使契丹文彦博留守北京与之宴问魏收逋峭难为之语何谓颂言梁上小柱名取曲折之义因即席作诗以献今检是集凡诸家所举各篇悉在其中足知完本尚存无所阙佚而颂文翰之羙单词只句脍炙人口即此亦可见其槩矣乾隆四十六年六月恭校上

　　古律诗
　　进南郊大礼庆成诗状
　　右臣伏以礼行于郊由古谓之大事祭得其道先圣着于格言皇帝陛下受命皇穹继昭仁考爰初纂嗣不忘丕承大报圜丘恭展三年之礼祗见宗庙遍祀八世之灵至于朝享道宫肆赦端阙奉事长乐尊上册文惟是亲行之仪莫非盛德之事上意专慤熙事备成宜有咏歌被之金石臣幸尘词馆恭覩礼文思所以摅发精微形容万一愧无鸿笔藻润之美徒有芜音累句之慙出于至诚不敢自蔽谨夙夜斋祓课成大礼庆成诗五首缮冩随状上进干凟宸严臣无任惶惧屏营之至谨奏
　　大礼庆成诗五首
　　惟宋五世将郊泰祗若前典钦崇道尊其日吉午天时薄寒帝出斋寝朝神紫垣降辂就次被龙搢山礼荐苍玉乐歌太安酌以琼醴羞以灵飱飙御来格上诚益防思我基命系于帝轩真祖膺符锡类贻孙皇继其志受祉弥繁炎炎景厯亿万斯年【右朝享景灵宫】
　　天命有宋系轩纂唐祖武宗文积厚流光奕奕清庙八祏同堂五祀备飨百顺储祥皇帝孝思明发不忘躬执圭瓒再荐嘉觞公卿陟降鼎俎芬芳广乐九变羣神迪甞昭我先烈缉熙万方永思继述成不敢康始朝于庙诗歌嗣王烈圣在天保佑我皇【右荐享于太庙】
　　惟皇继天宅位三禩灵德及逺嘉生茂遂仲冬月良一阳气至上将告成礼严肆类自庙徂郊万灵拱衞天收宿雰日丽佳气月逾既望再登午陛对越苍祗配以太帝奠玉酌匏升烟燎币志尽防恭备成熙事玉舆旋衡欢声沸渭皇专乃诚下防厥惠【右有事于南郊】
　　帝陟圜丘天降之祜嘉与士民均承福祚还御端闱躬布皇度鸡星验占鹄书胪句庆浃天渊恩霏雨露涤垢弃瑕蠲租薄赋三泉靡封五刑几措太和充塞羣心率吁自古无赦孰除诖误我朝本仁实清弊斁赦匪其私仁则有裕万民以怀邦基巩固【右赐赦宣德门】
　　大孝尊亲治本人伦自家之政所以刑民天正元日帝郊毕禋将崇母仪爰即嘉辰正朝乾辟羽衞星陈金石既作公卿荐臻帝步西序躬咨尔臣授以册宝极其礼文羣官在位靡不欢欣谓天子孝感于人神祝天子年后于灵椿皇心匪懈圣度惟新【右发册文德殿】
　　元日鸿庆宫朝拜二十韵【皇祐三年任南京留守推官】
　　上国房心次仙都太紫微璧门开仿雉金阙立如翚昔圣躬三祀乘时动六飞奠圭涡曲已鸣跸涣阳归孚号兴王泽尊名象日畿经营原庙制防巩上天威祖系崇轩鼎宗祊奉汉衣豫游恩甚盛谒欵礼无违却顾和鸾迹咸嗟歳序非璿源流永永尧德冠巍巍我后遵鸿烈声明极范围乾坤扶寳厯日月拱琳扉朝防元正始班联内着依叙官风宪肃就列羽仪辉北面鸾分党东隅斗转机寒销冰玉砌风绕大龙旂瑞色花藏緑欢声雾散霏未竣真馆集遥仰帝阳晞至治尤遐邈先猷重发挥邦人徒望幸延首向端闱
　　恭和御制赏花钓鱼
　　四照仙花次第开上林重覩属车来雨晴驰道纡天步风引浓香到寿杯瑞燕入檐时拂掠恩鱼窥钓更徘徊栢梁絶唱容赓缀独愧非才亦与陪
　　恭和御制上元观灯【元丰五年正月毕仲衍押伴高丽赐宴楼下赐诗】
　　宝杯莲烛宫台万户千门五夜开楼下舞韶清吹发云间鸣跸翠华来九賔宴集占风使四近班陞构夏材自愧周南独留滞十三年隔侍臣杯
　　恭和御制赐太师致仕文彦博五言六韵诗一首
　　晓日清明霁华林物色新诏颁慈惠宴恩厚老成人衮冕三川客杯觞四府臣献酬同饱德恺乐尽含醇帝惜营丘去民思尹氏均宸章一褒贲道不愧如仁
　　恭和御制赐太师致仕文彦博七言四韵诗一首
　　出处逢时道不殊当年功立老闲居来膺冕绂三公命归奉云章五色书社稷元臣名已遂山河雄镇宠仍疏琼林诏饯衣冠盛岂止都门百两车
　　谢赐御书诗五言十二韵
　　圣学天攸纵宸文日又新因摛镂牙管分赐挈囊臣玉几繙经暇瑶宫锡宴辰清躬忘旰食睿藻出丝纶才赏唐词客人称贺季真开编掇风什洒翰落霜筠楷法前无古书评妙入神良慙孤陋甚获对宠光频得异登牀诮藏过韫匵珍护持同始卒荷戴等穹旻职幸联簪笔忧常念负薪惟勤黄巻业仰答大君仁
　　进皇帝初郊大礼庆成诗状
　　右臣伏覩今月十四日皇帝躬祀南郊礼毕者三岁一郊汉唐以来为定制二仪并侑祖宗之世皆亲行盖王者谨于承天而经典重于飨帝国之大事兹实为先伏惟皇帝陛下禀尧舜之聪明监夏商之忠敬明德恤祀靡神而不宗精意曰禋无文而咸秩爰卜天正之吉躬修郊见之仪前诏攸司聿遵先典内则禀训于文母外则询谋于迩臣稽皇祐旧章既两行于宗祀案熙宁故事当合祭于圜丘尊泰元富媪之灵徧六宗羣神之祀义兼圣孝礼尽情文瑄玉嘉牲合二精于志物黄钟太蔟备六乐于神祗烟燎配黎而四施休气欝葱而竟夕膺万寿而褆百福丛庆于三宫畅九垓而泝八埏覃恩于庶物臣亲逢熙旦叨领近司从警跸之奔趋望庙丘之陟降赞盛德之事虽莫究于仪形听舆人之言辄载赓于歌咏顾慙鄙俚难逭谴诃谨夙夜斋戒赋成皇帝初郊大礼庆成七言二十韵律诗一首缮冩上进冒渎冕旒臣无任惶惧屏营之至谨奏
　　皇帝初郊大礼庆成诗
　　六帝贻谋圣继明两宫崇祀上躬行熙坛初展严禋报观盥三终孝享成仰法祖宗尊本始致隆高厚罄纯精告防并荐圆方玉在涤交充茧栗牲吉土兆南殊汉畤景圭迎至协周正羽旄备物甘泉仗觚陛层垓委粟营斋寝累朝思志意真廷谷旦奉粢盛八神警跸敺氛沴九貊梯航入贡菁过庙徂郊纡步武撤茵停盖肃公卿服裘尚质遵前制配册修辞正重名篆鼎割烹工奏夏午阶登降斗旋衡先时非雾滋鳞隰竟夕祥光烛缦城钟虡四厢庭乐毕樵蒸千石燎烟轻龙回雕辇升峣阙鹤负防书下采楹海外鸡星占泽霈楼前鼍鼓震霆惊风云气应灵台候雷雨仁深犴圄清有昊感通钦辅德聿修寅畏念持盈旁求遗逸搜岩穴寛舍租逋惠氓天锡泰元神防瑞民歌华黍歳丰声老臣扈从知何补敢次舆言颂太平
　　恭谢庆成诗十韵【嘉祐元年九月代刘相作时二府皆有诗议者云皆不逮刘公者公闻之曰某久不事笔砚此亲倩苏校勘所作也人嘉其不掩善】
　　圣孝当天世继明两全能飨礼交行路朝躬欵三年报斋寝先端七日诚穹厚博临通蠁祖宗严祀极专精冕章从质更新制钟虡铿纯复旧声【先是命礼官重定衮冕制度务从简质仍诏用旧乐】舞佾前陈徽烈茂燎烟升逹上仪成不祈厚福归诸已务达厖防及庶氓还御端闱孚大号载颁元厯庆鸿名欢謡沸渭盈都辇协气旁流浃海瀛百辟旅庭纷岳抃四方延首效葵倾帝心防巩常无怠亿万斯年享治平

　　古诗
　　送郑无忌南归
　　我家本南闽处世若羇寓侍亲从宦逰东西迁者屡因循歴年载未识南中路间歳至都城始与里儒遇尝闻闽中竒英隽所丛聚山川降灵气禀肖时魁梧往在祥符中黄李称独步当时盛文物在廷充振鹭二子簉其间轩轩并镳驱得志何所如康庄跃骐时文变新律高辞无累句人传幼妇碑朝奏客卿赋竒器擅珪璧至音有韶濩当宁喜同时士林争景附渊澄或无鱼木秀乃生蠧嗟哉士之贤入朝尝见恶叔才亦稍迁秘书观四库命穷时不偶伶仃独翁孺名虽鼎甲贵官止尘劳汚蹉跎三十年困踬卒颠仆伊予生后时举颈徒企慕郑子乃其甥肖貎何颖悟读书博而详伟词亦能铸朅来至京国首与予欵晤吾尝念古昔人贤犹爱树况遇贤之出何必縁雅素从游遂无间往来靡常度近遭试秋官报罢不知故飘飘叹穷途归橐不能具当路少知已腹心谁可布我嗟近世人龊龊何足数见善不能举徒然用烋噢士有处忧困岂尝一防顾劝君自求已何用频龃龉还家乐诗书养心益坚固无因一伤手廼欲改前措泽豹未成文尚能隐深雾齐禽戢其翼久亦防傅羽【去】况子气方刚丁辰年未暮勉勉信予言勤勤蹈前趣更想渭阳贤怀道当有遻王忱希范甯羊昙宗谢傅武子称衞玠逢时廼腾骛深源赏康伯传经有训注子其慕斯人戒哉惇向素还来取高第擢俊膺帝吁更发翁孺名后来益裕作歌送子行予言岂虚谕
　　送王秀才出京
　　都城大雪酒价高我送故人倾浊醪酒阑置杯欲话别不觉中心生郁陶丈夫聚散岂为念所叹人事多徒劳忆昔我年未弱冠西来观艺游神臯金门献赋嗟失路因循四载何所遭京华碌碌少知已但闻嘲弄声嗸嗸今年诏书恤西事大开言路烝羣髦因思平生负气槩临事始可观持操长缨羁虏岂无计防虑未献心徒忉感君壮志正凌厉术业沈究钤与韬上书论画兵家事直请西去亲鞬櫜又言若将众十旅可为手足除纷搔逾时待报不闻命西秦便欲从游遨束书防劔瞥然去却顾名利轻如毛我谓方今正急士帝言屡下搜英豪公卿悉意奉诏防推贤好善如干旄劝君失意且韬养有能必见时称褒中朝列爵本驭众进任将相皆儒曹一时未遇何须叹吾辈岂久居蓬蒿
