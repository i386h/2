　　钦定四库全书
　　升庵集巻一　　　　　　　明　杨慎　撰鳯赋
　　黄帝晨坐于扈阁降观于荣阿有鸟来巢命音交和乃召天老而问焉天老对曰西申之国丹穴之山爰有神鸟名为鳯焉羽族三百六十以应周天天帝命此鸟以爲羽族先其为状也鸿前而麟后蛇颈而鱼尾鹳颡而鸳腮龙文而龟身燕颔而鸡咮鹤植而鹿化首若防青戴仁也婴若白垩抱义也背若赤丹负礼也胷若石墨蕴智也足下赨黄履信也厥像有六厥苞有九六像维何头像天足像地日像在目月像在背翼像曰风尾像曰纬九苞维何口苞命眼合度耳聪逹舌屈伸色光彩冠短朱距兊钩音激扬腹文户天为生食爰在积石其树名曰琼枝琦玗琅玕以为实琅玕之腴是曰珠一人三头逓卧逓起以伺琅玕与琦玗子栖必朝阳髙冈之桐啄必紫脱苍筤之粒其饮有仪其食有质其和鸣也小音金大音鼓音如箫自歌翰如干自舞昬鸣曰固常旦鸣曰发明昼鸣曰保章举鸣曰上翔集鸣曰归昌节节足足喈喈邕邕其音英英凄凄锵锵延颈奋翼五彩备明举随八风气应时雨徊欝岛翔圃过昆仑轶底柱风穴暮宿弱水濯羽五彩晶英九文鲜明乃先天宫之畜南方鹑火之精览七德律五音通天祉应地灵翥即文始来即嘉成惟鳯爲能天下有道得鳯象之一则鳯过而睨之得鳯象之二则鳯翔而企之得鳯象之三则鳯集而翽之得鳯象之四则鳯春秋防之得鳯象之五则鳯朝夕比之没身居之矣于是时也鳯翩蹮帝竹而不惊宿留帝梧而不去吸昆丘之琅霜吞嵰山之紫露驯于宫隣爲帝司晨扈于轸车以节路驱爰率其属朝嘲夜防露翥风翔千百爲羣朝于明堂故曰鳯忘其翼鸷鸟忘其爪而可扰神龙丧其角鸷兽忘其距而可御咸若鸟兽而柔缓畴若草木而葳防苍麟白驺可系而羁鸟巢鹊梁可俯而窥是以万方来朝四灵咸依也乃若刳胎剔神龙不逰覆巢防卵鳯不翔留四灵之瑞交违则万物之庚不由矣帝于是始作咸池之乐张于洞庭之野有焱氏承而颂之曰鳯皇秋秋其音若箫其翼若干有鳯有皇乐帝之心于是风后从而赓之曰舟张辟雍鸧鸧相从八风回回鳯皇喈喈力牧载而和之曰敕尔瞽帅尔众工奏尔光诵肃肃邕邕无怠无凶帝曰昭乎防天老之言也緑图侍侧拜手而賛于帝曰臣闻缥青者曰鸾黅黄者曰鹓饶紫者曰鸑多白者曰鹄得鳯体而分者也西有鳻雀东有谏珂北有定甲南有锦驼鵕防飞光而暎天绶鸡曵彩而骈罗窃鳯形而似者也且羽嘉生蜚翼蜚翼生鳯皇鳯皇生鸾鸟鸾鸟生庶鸟庶鸟之变乃产妖鸟是曰羽虫之孽徳下衰则得其分徳又下衰则得其似徳极下衰则得其孽臣恐千百世之后淡见而甘闻诬天而矫神必鳯而弗能将歌舞羽孽而珍贶咎徴天无口孰分其然否神无喙孰析其真伪諴孚几何矫枉寔多鳯之瑞世仅见孽世则那帝曰远乎防緑图之见也遂乃命伶伦断嶰谷之竹吹律写鳯之音素女调三舌之簧炙笙象鳯之翼史皇作斧扆绘鳯之羽胡曹黹衣裳画鳯之文又采飞龙氏之篆陈帝鸿氏之砚书之右书曰巾几铭圗之左图为帝命验
　　伊兰赋
　　江阳有花名赛兰香不足于艶而有余于香戴之防紒经旬犹馨意古者纫佩之用颒浴之具必此物也西域有伊兰以为佛供即此汉书所谓伊蒲之供也从而为之赋曰
　　英英有兰猗猗其美諡以伊兰实自卭始维兰之品粤繁有藂曰蕙曰荪错遌丰茸曰芷曰茝名殊物同形如蒲萱盆盝是荐謡俗攸珍乃兰之赝可佩可纫服之媚人昔号国香今兹曷湮宜汤宜浴陈除新沃昔闻其语今兹则先正兰谓识之艰山谷致疑晦翁屡叹懿若卉之挺生兮何珵美而琦絶既葳防以晞旸兮亦扆岑而带岊友射干而偕生兮朋荔挺而俱发潄英之朝泽兮应复至之阳月秉棻蒕之旉荣兮擅芳菲之酷烈开以景风之淑辰兮贯乎星回之火节潄沆瀣之芳润兮全朱明之炎徳匪直十歩之有芳兮曽验经旬之未歇乃有娥媌掩嫮靡曼嫈眳步步移艶笑笑倾城子夫兴于鬒发昭仪宠于体馨曵歩揺之馥郁暎角犀之丰盈若兰机回文之锦季兰琴緑绮之声爰色授而防与且目而心萦倭堕梁家之髻浮渲韦娘之缨咸纎指兮争掇并巧粲兮相迎都梁蟾蜍兮阏尔而减价虎蒲龙枣兮瞠乎其亡菁超旃檀于雪域压迷迭于云清星芒当昼而弗隠金粟未秋而先成埓瑶华兮玉蕋叶緑叶兮紫茎堪纳凉于圃思御风兮蓬瀛讵数秋红之蕳子岂顾晩翠之长卿爰感子兮体物遂锡子以嘉名重为系曰肇允嘉卉兮昉自炎皇媺人之佩兮王者之香深谷逶迤兮无人自芳宣尼息鄹兮屈平累湘晨月秋风兮属国之堂洪波霜晩兮谪仙之乡纫遗佩捐兮庸亦何伤体物浏亮兮聊以相羊
　　鴈来红赋
　　此治甲子余十七岁时作近得其稿録于此
　　蜀城之花与玉蝉而同房汉宫之菊配黄鹄以分裳兹纎茎兮独异阳鸟而敷芳盈盈渥赭奕奕鲜妆承景火旻之下委质金神之乡吴台草长红心不同时节楚岸枫愁赤叶逺谢煇煌迎风动彩露生光映夫容兮濒枉渚杂桂蕋兮侍君堂残蝶留连而惊艶胡蜂踯躅而疑香矜陨霜之不杀笑何草之不黄脱梧楸兮索索销兰蕙兮扬扬琅玕摧兮好树箫籁响兮长廊栁呈疎兮绮陌荷委盖兮银塘矫兮红蘤烂然朱芳赏生兮稀有物珎兮非常误停车之杜牧诧剪防之隋皇江淹多才拟抽毫而赋咏班姬含怨亦却扇而徬徨乱曰春花红兮杜宇春草碧兮苌兹防生兮秋頴乃借荣于春工根伴兮寒螀名托兮霄鸿彼游龙兮杕杜亦诗人兮为风采佳名于饰羽聊以贻夫雕虫
　　乐清秋赋
　　皇天平则成四时兮窃独乐此新秋无宋玉之悲懐兮匪江淹之离愁澹吾虑以抚景兮遁炎威于金祛赫曦之焰焰兮追凉飑之飕飕屏羽扇而箧藏兮御纨素之轻柔听琅玕之朝坠兮玩金波之夕流桂连蜷于山阿兮兰猗靡于岩陬既葳防其可懐兮又芬蒀以绸缪嘉华黍与膏稷兮获万寳于西畴虽四壁之徒立兮欣人足而我优謇不乐而胡为兮攅百虑以挐忧命一觞而孤斟兮奚吴歈兮秦讴慕漆园之斥鷃兮畅逍遥以优防赋印段之蟋蟀兮庶蹶蹶而休休
　　防扇赋
　　江文通有防扇赋首尾衡决讹舛复多夏日枕疴稍为补正
　　青阳谢兮朱明临度槐景兮际梅霖日车亭午风柯不吟纎絺在御轻羽重寻或蒲葵兮纨素又紫绀兮緑沉空青生峨眉之阳雌黄出嶓冡之阴金膏诒河伯之渚碧髄挺青蛉之岑竹染湘妃之泪纸捣蔡伦之砧合为防扇翳君瑶琴瑶琴兮青琴知音兮赏音解明星兮繠佩衔半月兮兰襟堕马罢梁家之髻含矉捧西子之心销独愁兮片玉恡一笑兮千金怨蕴隆之赫赫忘逝景之骎骎愿鼓幽兰兮白雪情寄山髙兮水深
　　药市赋
　　蓬丘仙伯刀圭而通神奈园老宿品千二而称珍何如蓉城之日重开药市之春若夫岐挚汤液和缓瀡滫采嘉林之嘉草掇芳洲之芳羞屏风水发书带山抽人参三桠来自高句丽之国桃枝九折出于嶲昆明之陬椒聊下虚星之舍果蓏荐织女之楼逢茅龙之卫叔遇木羊之葛由检本草之圣録极山海之防搜别有天棘日精泽葵海藻火林之蔚炎柯雪津之擢阳草欵冬叩氷而吐葩葶苈迎露而先槁青纶碧组苍龙就食于甲辰素茵皜蔯白兎腾仙于辛卯帝休鬼□荡邪而蠲忧神锦仙盐驻顔而难老又不若剸犀截角搴翠删毛澄禺山之水碧樊桐之金膏鲛人泣珠而酬价狼防齅金而解櫜神鹿两头饮上池之水灵鸟四翼归金匮之韬莫不攀丹危而陟翠险霣仙的而下神嚣渉淼淼驰苹苹刈翘翘伐丁丁香草比于君子嘉卉方于女贞藤名简子花号长卿五芝色配乎五岳六饮芳冠乎六清乃延巫彭招长桑飞绛雪捣霜巴戎赤斧梓谷王昌蓬州涂定辞武张开光希夷出崇龛之落真多栖葛璝之乡黄足之鸟呼季玉而并复朱顶之鹤名令威而骈翔负局磨镜之翁箍桶讲易之叟扶龟床于行阿防鸠杖于林薮严遵之芳乳平分王烈之香膏共剖是月也江波锦落火井星还枫瑟瑟而鸣籁菊英英而缀斑访名迹于古昔啓药市于重关被开眀广都之野带杜宇天隳之山哀危蟪于寸阴矜局鼃于井谷布濩天府之区蔓衍坤维之域八万四千之人烟五十四州之謡俗国无夜户之虞人有春台之燠郤公程郑之伦骥子鱼文之服乍发剂以针龙奄飞符而擒蜮黄耇背鲐苍恖髫蜗行歌击壤羣嬉聚沙成行鴈鴈学语鸦鸦毛姬饵叶鳯子藏花色辉九璧之锦光漾百潭之霞采思妇之巻耳折放臣之疎麻息风枝之不静起露蓼之含嗟崔公之镜云逺华陀之嚢已遐既富矣韩康之肆宁蕲乎博望之槎扬雄梦肠顿醒滞思于雕篆相如消渴歘防沉疴于齿牙岂知西颢之盘露北方之河车哉有林闾翁孺者起而为系曰黄能兮休入寝朱罴兮罢窥屏永遁兮二竖载殱兮九婴分菊潭兮灔灔宴蓉城兮鸿安丘继而作曰安期神楼散扁鹊陷氷丸银筒昔谁蕴金瓶今子殚谢惊飍兮驻流电访大药兮浴还丹邀籛偓兮雀跃共乔松兮鲵桓主人闻之欣然称蔅修容丽级献寿升筵敬缉光诵奉以周旋
　　戎旅赋
　　恭承恩谴兮于役滇越捐佩江皋兮解绅云阙三陟崔崔兮九折不日不月兮遂届穷发抚孤旅而悁脰兮揜众困而心怅圭籥之骎遄兮逾四稔而迄今父母孔逺兮懿亲而北南类连逵而分衢兮似同波而殊浔慈乌忻于共巢兮恒鸟悲乎异林彼纎羽之防族兮亦命侣而跢跦何生人之含灵兮乃羣而弗如咏清人之介驷兮感放士之鸣鴸姬公畏于熠燿兮尼父唶夫蟪蛄屈托乘于螭豹兮庄寄径于鼪鼯在圣哲而固然兮揽古人而重欨哀吾生之罹邮兮背中土而播荒粤戴盆而伏嵁兮望崦嵫之末光神怳悢而蜚飏兮形脔巻而怆嚢睇孙水之浩渺兮瞻灵关之峻极聆猩猩之夜啼兮履狒狒之朝迹寻终古之攸居兮问祝融之昔宅胥靡登而不惧兮魑魅过而奚栗堀堁飑扬兮含沙影流喟兹徂春兮忽焉杪秋月令殊于九州兮瘴卉华而嵗周若有人兮好我擕防酒兮思柔采防榔兮缀扶留赠相兮结忘忧寒鸡兮为脯露江鱼兮为脩滇歌兮爨舞白日逝兮景浮独持觞而懐逺兮杂叹歗其向陬遂还轸而休室兮隠零雨乎寂夜引簟枕而假寐兮遥梦归乎亲舍家人嘻以欵语兮闾里纷其来讶众鸡鸣而惊余兮晨光昒乎东射怅梦欢而觉悲兮泪承睫而交下假灵氛以歴占兮援龟颂兮余谢曰明庭其布徳兮子行归乎肆赦系曰莫靡荒服自中古兮日月之表烛不普兮章亥步穷禹罔睹兮兰津开道行商苦兮碧鸡望祭使者阻兮余亦何为恒此土兮金跃不祥顺勿忤兮乐天知命去何怃兮
