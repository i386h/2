　　提要
　　【臣】等谨案宋景濓未刻集二巻明宋濓撰濓集重刻于嘉靖中行世已乆此本乃
　　国朝顺治乙未濓裔孙实颖得文徴明家所藏原刻旧本以示金坛蒋超超择其中今本所未载者得三十八篇编为此集以补其遗今以嘉靖中韩叔阳所刻重勘其中跋何道夫所着宣抚郑公墓铭等十一篇皆今本所已载超盖检之未审其余二十七篇则实属佚文推究当日之意盖或以元代功臣诸颂及志铭诸篇大抵作于前朝至明不免有所讳或以尊崇二氏不免过当嫌于躭溺异学而隠之观杨士竒东里集倪谦文僖集凡文章为二氏而作者并别为巻帙附缀末简不散入各体之中则正徳嘉靖以前士大夫之持论可大畧覩矣然古来操觚之士如韩愈之于髙闲文畅持论始终谨严固其正也至顔真卿书多寳塔碑撰麻姑坛记于平生大节要亦无亏厯代文集此类颇有固未可以一端论也观是集者知其持论之失但以文章取之可矣乾隆四十五年六月恭校上

　　颂
　　皇太子受玉册颂
　　至正十五年春三月丁亥朔越八日甲午皇帝命太尉汪家努持节授皇太子玉册及冕服九旒在廷臣邻莫不骏奔东朝恭行大礼越翼日乙未复诣大明殿俯伏拜舞举觞上千万歳寿百僚具欣无间小大佥曰我太祖圣武皇帝茂昭寳训预建冡嫡嗣服继綂累叶上承灵贶防敢或违今我皇帝春秋方即定元良以绍隆丕构其神谟睿断与雷霆同其英烈山岳同其雄高河海同其幽深求之前古夐然无比呜呼盛哉然以万国至众主器至重豫定则治否则乱治乱所闗不翅一反掌之易布在方策葢可见已奈何为人上者不是之思嗣乱相寻有不忍言惟我皇帝圣神文武与天同运而不以四海自私惟皇太子刚果温文与圣合徳而足膺主鬯之寄明两作离照耀无穷对越天地之耿光丕承祖宗之休烈是可以无媿者矣易曰主器莫若长子记曰一人元良万国以正其是之谓乎臣濓遐方贱士躬闻大庆不敢喑无声歌谨遵诗人美盛徳之义而作颂曰
　　皇朝御防图治靡寜邦本既建万国以正仰瞻紫微中居帝座前星煌煌厥象斯着圣人法天夙夜弗违爰升上嗣以弼大基春日载阳洊雷出震龟协筮从降兹大训乃勑近臣持节而驰锡以玉册冕服九旒【叶音厘】帝曰休哉神器甚重继体之思朕敢不汝宜敬事勿懈益恭服滋宠嘉祗谒太宫勿谓天高征应甚速勿谓宫深千里在目若军若国汝抚汝监流祚无穷朕心乃安皇储翼翼拜手稽首臣敢不朂以主鬯负百僚萃止冠弁之峨舞蹈不足继以咏歌四海之命悬于元冡少阳位隆不震不动何威弗宣何气弗清太白敛芒天髙日明微臣作颂流播盛徳勒之贞珉埀示千亿
　　皇太子入学颂
　　至正九年某月日皇帝开端本堂命皇太子肄业其中致圣功也前期一日司经设帝坐于堂中南面朱几在其南芗俎又在朱几南俎前有筵复设皇太子位于坐西东面太子太傅位于坐东西面谕徳赞善二人文学一人位在太傅之东皆西面北上厥明司经以裕皇所御圣典寘朱几上公卿子从学者分东西序立堂南北面以中为上正字一人司经一人对立两序端以劾不如仪者赞者皇太子就位立谕徳以下亦如之皇帝遣使者来致命在位者皆北面立受以次出堂门外望北阙再拜者二复初位赞者又皇太子至芗俎南北面立升筵再拜就筵北跪焫芗于炉者三少退再拜复位立如初凡拜若跪公卿子咸从司经以圣典还阁几与俎皆彻皇太子即位坐谕徳以下皆坐公卿子别设几席于谕徳位西皆再拜赞者引齿最长者一人执贽自门左入即席跪奠贽于几复位如初又再拜拜已执斚以升执尊及脯醢者从诣谕徳西东面立注酒于斚即席跪行一献礼从者荐脯醢赞善文学亦如之少退执事者勺酒授谕徳以酢献者复位如初皆再拜赞者退司经设授读位正字一人执策启皇太子受经一人诣谕徳请授经至位东西相向坐授受已复位如初惟皇帝丕式先王成宪敷佑四方旁求俊彦用廸皇太子以学惟敬惟慎如此其至实万世无疆之休敢拜手稽首而献颂曰
　　于赫皇帝日照月临飇驰霆轰克威克明国本既定【叶音庭】万方靡弗承乃命宫师乃辟宫学侯谘侯度侯究侯宅于彼有觉宫学峨峨龙楼之阿奕奕其楹侐侐其廷既穆且贞皇帝曰噫莽莽九围此实其基其基何居有书有诗羽籥干戈亦以其时尔尚朂哉毋越我彛我心我縻毋隳我维我善我师上窥黄虞百度惟熈皇储曰都敢不敬恭以养其防以徼其功以逹于家邦【叶布洪切】旭日煌煌出自东方鸾车彭彭青旂阳阳佩玉锵锵跻彼新堂载诵载或舞以飏帝命防败荒维幅员既长维天命靡常匪德曷臧兢兢皇皇寤寐弗忘于赫皇帝诒谋寜止皇储则之学又烝止百禄盈止庶绩凝止庶绩凝止万世期无已
　　国朝名臣颂【有序】
　　帝王之兴必有不世出之人豪以自赴云龙风虎之防易所谓圣人作而万物覩者是已我皇元受天明命抚安方夏天戈所指万方毕从是故一鼓而诸部服再鼓而夏人纳欵三鼓而完顔氏请降四鼓而南宋平东西极日之出入防不洽被政教共惟帝臣虽睿谋雄断动无不胜亦頼熊罴之士不贰心之臣有以诞宣天威故功成治定若是之神速也自今观之防阵攻城无战弗克则有若鲁国忠武王之伦面折廷诤丕赞皇猷则有若耶律文正王之属斟酌百王恢文教则有若许文正公之流其他智士谋臣袂接肩摩殆未可以一二数稽之于书在虞则有四岳九官十二牧以亮天功在商则有六臣保乂有殷礼陟配天在周则有五臣为辅以克脩和我有夏以皇朝嫓之未足多让夫以明良相逢如此之盛治功告成又如此之懿意谓有以宣着鸿猷勒之琬琰昭示无极逮兹百年而颂声不作非甚阙典欤汉唐之臣本无足议陆机吕温尚浓墨大书为之颂赞况逢今日昌熈之运安可默默濓窃不自揆輙取行事着明熟于耳目者自鲁国忠武王而下凡二十二人人各为颂虽立言不文亦颇谓能美盛德之形容使读之者知列圣之勤劳诸臣之忠荩王业之成匪一朝夕其于治道似不为无所助也苐以金匮石室之藏远在天上有非遐方陋儒所得窥故其所颂止此而已非敢有所略也
　　鲁国忠武王穆呼哩
　　鄂诺之河白气如虹王生其中虎首虬须为天下雄光辅帝极宪天惟聪如鹰之扬如飚之发如云之从右执大斧左櫜彤弓铁垒层层一劈而崩遹骏有声诸部用平相彼完顔逞于滛凶我伐用张旗鼓有容僵尸百余里浍河为红太行以南敛手就降【叶音红】帝録其功锡茆土之封丹书铁劵与国始终传世于无穷
　　淮忠武王巴延
　　淮王桓桓凝峻寡言一言之间如雷破山及履上台四国尔瞻誓师南征大乱汝戡三军飞渡目无江南前扼其吭宋胆自寒老枭既夜遁直捣临安俘厥君臣大敷帝仁皡皡熈熈市肆不移宋鼎已易而民弗知崇德报功王谦让弗居此天子德威臣何能为古有曹彬于王见之
　　楚国武定公阿尔哈雅
　　于维楚公既勇且武手挺白矛能舂猛虎建谋于廷谓宋疆可平王师出征公实在行陷阵攻城风驰霆轰一驾而襄鄂下再驾而吴楚寜山獠洞猫被毳荷氊亦归王明士出其门咸为国桢能黼黻皇灵呜呼楚公葢世之雄
　　广平贞宪王王昔
　　倬彼武忠驱猋驾云来奔真龙无谟不宣无战不从有孙烈烈劲气直节雷霆之下孰不殒防王不震不慑宗藩摄难变生肘腋帝命出师三战三防鹤驭上宾众言沸腾王静以镇之神器弗倾玉带寳衣用旌尔能社稷之臣繄王之伦
　　河南武定王阿珠
　　河南之先世有大勋列阵蟾河强部禠魂三炬夜爇回鹘乃奔乃蹴汴京以洗妖氛西南夷不庭即勒鋭兵以讨以征金鼓震天跃入其城叚智兴及生河南沉几有智挺英毅之气瞠目一叱万人皆废顾彼江南一弹丸之地恭行天诛羽书日驰卒献馘京师九土茫茫来享来王开元祚于无疆
　　洪阳忠武王伊彻察喇
　　【阙三字】将一气之传有如洪阳不负其先威容言言望之如神亲【阙三字】宿卫帝宸白斥大奸防去祸根金山之隂蚁附蠭屯王鞠其军以绥远人以镇北门以靖大藩宠赉加厚铸印如斗大有烈光千载弗亡
　　勾容武毅王托克托呼
　　西北絶域有山峨峨襟带二河其人鸷猛如【阙】负戈况我勾容人中之杰来辅真元孰得而遏从大军北伐应变如神有坚必防成杭海之烈譬如太阿百錬不折无媿臣节
　　耶律文正王楚材
　　惟楚有材晋实用之逹人先知曰千里驹堂堂中书执政之枢相我太宗拓开鸿基拱立龙庭上陈帝谟三灵协和万象昭苏舒吾阳和脱彼翦屠人文褰开民献争趍于变时雍上登黄虞厥功何如请视鼎彞
　　杨忠肃公惟中
　　西域小邦以险自婴帝命杨公徃宣威灵雷霆震惊风云流行孰敢不来庭秉钺而南椎结卉裳惧不能兵恒山蘖牙盘防大明公仗节而呼巢穴用倾鼎湖遗弓乾纲解结公以两手上扶日月不欹不昃小大毕逹海外有帝业
　　汪义武王世显
　　金运之熄亳社已屋谁栅石门廵陴恸哭时维汪王义不负国国亡三秋将焉寘力神元龙兴六合有赫遂乘刚风来朝帝侧出其所蕴以效劳绩乃截嘉陵乃践大安乃擒劲帅撤蜀之藩王之出矣提防而前气吞三军王之旋矣雅歌投壶右琴左尊既武且文懿哉元勲
　　张献武王范
　　眞人开天时乘六龙麾斥八极羣雄云从剑气上冲日星晦防宋人不恭假息海邦帝命张王汝师汝将汝防樊襄汝渡大江汝揭义旗以受其降王既受命横槊上马鸷击隼翔有夫甚武直奋大刀众莫敢当王迎刺之应手断肮军气益扬大声震天敌有手若亡遂籍其土疆遗爝未息厥势犹彊帝壮王之威复命征征【叶音张】寳剑名甲锡自尚方一麾而殒海波镜平【叶皮良反】崖山苍苍武功洸洸
　　刘文正公秉忠
　　贤者在世视时诎信挂瓢武安絶世离羣翻然而起拏风跃云乃垒乃城乃阙乃庭乃建滦京滦京之封龙冈欝冲王气所钟伊刘公是庸作皇邑土中敷文教万邦【叶容】驰车书来同维公之功
　　史忠武王天泽
　　谔谔忠武有力如虎斩将搴旗疾行如飞统诸侯大兵前无坚城马不留影士不留行枭李擒葛氛防肃清凶鸷既锄残惸亦寜春生秋杀以奠邦经以严邦刑维忠武有勇大敌不恐惟忠武有慈服即弗诛维忠武有节始终弗湼纪功旂常为邦之光
　　亷文正王希宪
　　天启景运挺生人豪豹畧龙韬呼吸风涛英英亷公才雄气骞力捧濳龙上升九天诞彰皇威闗陇晏然广厦细旃嘉猷日宣出镇辽霫强宗震叠移节荆湖民气遂苏连征大猾正气烈烈式扬发则莫我敢遏大星煌煌西流于堂天下之伤
　　窦文正公默
　　有倬窦公訚訚而驯雍雍而惇炳炳而文大衣埀绅似不能言及摧权奸执将排山纲常之陈皦若白日照耀天下至今有赫大道既明旁艺亦精九箴所及以死为生其学之醇其志之忠宜耄矣之嗟发自帝
