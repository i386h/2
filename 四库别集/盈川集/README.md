　　钦定四库全书　　　　　集部二
　　盈川集　　　　　　　　别集类一【唐】
　　提要
　　【臣】等谨案盈川集十巻唐杨炯传唐书文苑传称其文集本三十巻云今多亡逸志仅着録二十巻云今多亡逸是宋代已非完本然其本今亦不传此乃明万厯中龙防童佩从诸书裒集诠次成编并以本传及赠答之文评论之语别为附録一巻皇甫汸为之序凡赋八首诗三十四首襍文三十九首文苑英华载其彭城公夫人尒朱氏墓铭志铭一首伯母东平郡夫人李氏墓志铭一首列庾信文后明人因误编入信集中此本収尒朱氏志一篇而李氏志仍不传则搜罗尚有所遗也旧唐书本传防称其盂兰盆赋然炯之丽制不止此篇刘昫殆以为奏御之作故特加纪録欤传又传其驳太常博士苏知防冕服议一篇引援经义排斥防谈炯文之有根柢者知其词章瑰丽由于贯穿典籍不止涉猎浮华而新唐书本传删之不载盖犹本纪不载诏令之意是宋祁之偏见非定评也又新旧唐书并称炯为政严酷则非循吏可槩见童佩序称盈川废县在瀫水北其地龙邱去郡四十余里今址岿然独存炯令盈川无何卒县寻罢氏尸其地至今春秋不辍是则因其文艺而更粉饰其治绩亦非公论矣乾隆四十一年五月恭校上

　　浑天赋【并序】
　　显庆五年烱时年十一待制文舘上元三年始以应制举补校书郎朝夕灵台之下备见铜浑之仪寻返初服卧疾丘园二十年而一徙官斯亦拙之効也代之言天体者未知浑盖孰是代之言天命者以为祸福由人故作浑天赋以辩之其辞曰客有为宣夜之学喟然而言曰旁望万里之横山而皆青翠俯察千仭之深谷而皆黝黒苍苍在上非其正色逺而望之无所至极日月载于元气所以或中或昃星辰浮于大空所以有行有息故知天常安而不动地极深而不测可以作观象之凖绳可以作谭天之楷式有称周髀之术者冁然而笑曰阳动而隂静天廻而地防天如倚盖地若浮舟出于卯入于酉而生昼夜交于奎合于角而有春秋天则西北既倾而三光北转地则东南不足而万水东流比于圎首前临胷者后不能覆背方于执炬南称明者北可以言幽此天与而不取恶遑遑而更求太史公有睟其容乃盱衡而告曰楚既失之齐亦未为得也言宣夜者星辰不可以阔狭有常言盖天者刻漏不可以春秋各半周三径一逺近乖于辰极东井南箕曲直殊于河汉明入于地葛稚川所以有辞应于天桓君山由其发难假苏秦之不死既莫能知其説傥首之重生亦不能成其筭也二客尝亦知浑天之事欤请为左右扬防而陈之原夫杳杳冥冥天地之精混混沌沌隂阳之本何太虚之无碍俾造化之多端南溟玉室之宫爰皇是宅西极金台之镇上帝攸安地则方如棊局天则圆如弹丸天之运也一北而物生一南而物死地之平也影短而多暑影长而多寒太隂当日之冲也成其薄蚀众星傅月之光也因其波澜乾坤阖辟天地成矣动静有常隂阳行矣方以类聚物以羣分吉防生矣在天成象在地成形变化见矣部之以三门张之以八纪其周天也三百六十五度其去地也九万一千余里日居月诸天行地止载之以气浮之以水生之育之长之畜之亭之毒之盖之覆之天聦明也圣人得之天垂象也圣人则之其道也不言而信其神也不怒而威騐之以衡轴考之以枢机三十五宫有羣生之系命一十二次当下土之封畿中衡外衡毎不召而自至黄道赤道亦殊涂而同归表里见伏圣人于是乎发挥分至啓闭圣人于是乎范围可以穷理而尽性可以极深而研防天有北斗杓携龙角魁枕参首天有北辰众星环拱天帝威神尊之以耀魄配之以勾陈有四辅之上相有三公之近臣华盖岩岩俯临于帝座离宫奕奕旁絶于天津列长垣之百堵啓阊阖之重闉文昌拜于大将大理囚于贵人泰阶平而君臣穆招揺指而天下春东宫则析木之津夀星之野箕为敖客房为驷马天皇对于摄提皇极临于宦者左角右角两曜之所廵行隂间阳间五星之所次舎后宫掌于燕息太子承于冢社宗人宗正内外敦叙于家邦市楼市垣货殖毕陈于天下北宫则灵龟濳匿螣蛇伏藏匏宛然而独处织女终朝而七襄登渐台而顾歩御辇道而徜徉闻雷霆之隠隠听枹皷之硠硠南斗主爵禄东壁主文章湏女主布币牵牛主闗梁羽林之军所以除暴乱垒壁之阵所以备非常西宫则天潢咸池五车三柱奎为封豕参为白虎胃为天仓娄为众聚旄头之北宰制其胡虏天毕之隂蓄泄其雷雨大陵积尸之肃杀参旗九斿之部伍樵苏之地出入于园苑万亿之资填积于仓庾南宫则黄龙赋象朱鸟成形五帝之座三光之庭伤成于诛成于锧祸成于井徳成于衡执法者廷尉之曹大臣之象少防者储君之位处士之星天弧直而狼顾军市晓而鸡鸣三川之交鹑火通其耀七泽之国翼轸寓其精南河北河象阙于是乎増峻左辖右辖边荒于是乎自寜乃有金之散气水之精液法渭水之横桥像昆明之刻石嵗时占其水旱沧溟应其潮汐织女之室汉家之使可寻饮牛之津海上之人易觌日也者众阳之长人君之尊天鸡晓唱灵乌昼踆扶桑临于大海若木照于昆仑太平太象所以司其出入南至北至所以节其寒温龙山衔烛不能议其光景夸父弃防无以方其骏奔月也者羣隂之纪上天之使异姓之王后妃之事方诸对而明水浃重晕匝而边风驶裁盈蚌蛤则虏骑先侵适鬭麒麟则暗虎濳值五星者木为重华火为荧惑镇居戊已斯为土徳太白主西辰星主北俯察人事仰观天则比参右肩之黄如奎火星之黒五材所以致用七政于焉不忒同舎而有四方分天而利中国赤角犯我城黄角天之争五星同色天下偃兵趋前舎为盈退后舎为缩盈则侯王不寜缩则军旅不复或向而或背或迟而或速金火犯之而甚忧嵗镇居之而有福观众星之部署歴七曜之驱驰定天下之文所以通其变见天下之赜所以象其宜然后播之以风雨威之以霜霰或吐雾而蒸云或撃雷而鞭电一旬而太平感肤寸而天下遍白日为之昼昏恒星为之不见尔乃重明合璧五纬连珠青气夜朗黄云旦扶握天镜授河圗若曰赐之以福此明王圣帝之休符至如恠云祅氛冬雷夏雪日晕长虹星芒伏鼈隂有余而地动阳不足而天裂若曰惧之以灾此昏主乱君之妖孽昔者颛顼之命重黎司天而司地陶唐之分仲叔宅西而宅东其后宋有子韦郑有裨灶魏有石氏齐有甘公唐都之推星王朔之候气周文之视日呉范之占风有以见天地之情状识隂阳之变通诗云谓天盖高语曰惟天为大至髙而无上至大而无外四时行焉万物生焉羣神莫尊于上帝法象莫大于皇天灵心不测神理难诠日何为兮右转天何为兮左旋盘古何神兮立天地巨灵何圣兮造山川螟何细兮师旷清耳而不闻离朱拭目而无见鹏何壮兮抟扶摇而翔九万运海水而击三千与蛇兮异其短长之质椿与菌兮殊其小大之年钟何鸣兮应霜气剑何仗兮动星躔列子何方兮御风而有待师门何术兮验火而登仙鲁阳麾戈兮转于西日陶侃折翼兮登于上女何寃兮化精卫帝何耻兮为杜防争疆理者有零陵之石闻弦歌者有盖山之泉若恠神之不语夫何述于此篇以天乙之武也焦土而烂石以唐尧之徳也襄陵而懐山以顔回之贤也贫居于陋巷以孔丘之圣也情希于执鞭冯唐入于郎署也两君而未识扬雄在于天禄也三代而不迁桓谭思周于圗谍也忽焉不乐张衡术穷于天地也退而归田我无为而人自化吾不知其所以然而然
