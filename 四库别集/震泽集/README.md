　　钦定四库全书　　　　　集部六
　　震泽集　　　　　　　　别集类五【明】
　　提要
　　【臣】等谨案震泽集三十六巻明王鏊撰鏊有史余已着録鏊以制义名一代虽乡塾童稚才能诵读八比即无不知有王守溪者然其古文亦湛深经术典雅遒洁有唐宋遗风盖有明盛时虽为时文者亦必研索六籍泛览百氏以培其根柢而穷其波澜鏊困顿名场老乃得遇其泽于古者已深故时文工而古文亦工也明史鏊本传称鏊上言欲仿前代制科如博学宏词之类以收异才六年一举尤异者授以清要之职有官者加秩数年之后士类濯磨必以通经学古为高脱去謏闻之陋时不能用又称鏊取士尚经术险诡者一切屏去正间文体为一变则鏊之所学可知矣集中尊号议昭穆对大防与张璁桂蕚相合故霍韬为其集序极相推挹至比于孔门之游夏未免明党之私然其谓鏊早学于苏晩学于韩折衷于程朱则固公论也其河源考一篇能不信笃什所言似为有见而襍引佛典道书以驳昆仑之説则考证殊为踈舛此由明代幅员至嘉峪关而止轺车不到之地徒执故籍以揣摩之其影响糢糊固亦不足怪矣乾隆四十六年四月恭校上

　　震泽集序
　　文之传三人品一也学力二也才格三也以人品者人为世重者也以学力者学为世师者也以才格者文由才竒者也世之文由才传者吾见之矣合才与学者我未之见也合才与学盖有之矣合人品与学与才我殆未之见也故曰文之传也人品也学力也才格也三者具焉上也二具焉次也一焉又次也守谿先生早年词气如风樯驾涛如逸骥驰野如银河注溟如长虹横汉如列缺划云如驶飓之啸六合可谓雄矣晩年脱枝落英尚淡崇质大羮不和大烹不割酒不麯大音不古钟石磬俚耳不谐盖雄而古者也故曰公文可传也才格也一也先生早学于苏晩学于韩折衷于程朱是故观怒惧箴知公于心学力矣观慈湖记知公于异端辩矣观昭穆对观尊亲议知公于天伦明矣观教太子知公慎天下大本矣观讲学观亲政观边事议知公康济之懐矣观厯代官制知公考古矣通之识矣观诸序记观诸柬书观诸碑铭观诸诗歌知公黜浮崇古之大畧矣是可传矣故曰公文可传也学力也二也公于夀宁侯旧也夀宁侯戚密椒官大臣无耻者趋焉公自夀宁显后不通字姓君子曰公之介也武皇帝时瑾权横赫无耻者趋焉公决去人曰去则忤瑾或怒不测公曰吾不得吾职则去祸福奚虞焉天下乃知有大臣之烈故公之出也人曰瑞麟威鳯之防也公仕也人曰景星卿云之见也公归也人曰雪冬柏松之秀也公没也人曰夏鼎周彛之藏也公也无忝名臣也已故曰公文可传也人品也三也又曰才格也学力也人品也胥赖以成者也三善有一焉犹可名世况于公具焉或曰公在孔门如何谓厓子曰公若及孔门宜列防夏之间性善之对则式孟氏人心道心之论宜式程朱南海霍韬渭先书

　　吊阖庐赋
　　昔阖庐之覇呉兮卒托体乎兹丘慨往迹之日湮兮曽不可乎复求峯峦纷以环合兮浮屠台殿郁以相缪【叶】忽平冈之坼裂兮劔池沦而深黒俯莫测其所竆兮仰不见乎白日两崖嵚崟而鬭囓兮又巉岩而斗絶信天造之险兮为神怪之窟穴将举首而闯其浅深兮先魂惊而瘆栗彼秦始之雄哮兮力驱石而填海将破山而求之兮藐不知其所在宜元之之不信兮谓徃牒之我绐嵗正徳之协洽兮劔池忽焉其枯涸何昔日之沦兮今山径之峣崅伊水旱之常数兮非予心之所度石谽谺而双敞兮类墓门之頽駮试徃造乎其间兮将举武而旋却始沮洳以忽明兮谅欲退而不可俨凑题之可扪兮森怪石之欲堕岂漆镫之将灭兮若有俟乎王果唉呉王之物化兮求彷佛其不见想澒池之既尘兮何有玊鳬与金雁彼槃郢与扁诸兮疑此语之为幻何千年之神閟兮忽一朝而呈露也旋黯然而复闭兮殆神物之呵防也前骊山之强项兮后防稽之妖珈锢南山其犹有隙兮信多藏之为害独空空兮以保全兮因兹丘而増慨
　　双松赋寄同年谢少傅
　　郁双松之并峙兮一不知其封殖之嵗年干轮囷而磊砢兮文章错而斑斒苍虬拏空以腾上兮惊螭镢地以屈蟠万木凌冬而僵死兮独青青其不刋似端人义士之立朝兮正色俨然曽不可乎犯干彼幽人之何知兮日坐对而盘桓朝予仪其挺直兮夕予挹其清寒固非蓝田之弃丞兮抑岂华阳之真官愿千秋以相保兮歴氷霜于嵗残使明堂大厦之弃遗兮非予心之所患
　　去思赋【并序】
　　三山林侯利瞻知呉郡甫二年擢分省滇南阖郡之民咸戚嗟若谓侯甚宜于民而遽敓以去予谓循吏之难久矣自顷权奸柄政政以贿成一时在位者因竞为掊克以入私门不惟国法有不及虽士之清议亦不复行曰彼以缓祸也而莫之罪焉于是相师成风虽平日稍知慕义者亦且肆行而莫顾曰吾以缓祸也于戏纲纪其壊乎廉耻其堕乎于斯时也有能洁然自守终始不洿者几人此林侯之政所以尤难而重呉民之思也予乃推愿防者之意为之词
　　猗嗟林侯兮胡不为我少防世方竞以贪惏兮侯独端靖而好脩莅呉民曽防时兮惠政日以旁流念徃嵗之大祲兮分转死于沟渠赖我侯之敷政兮不竞絿而于于哺我兮被我育我民兮如母我耕于野兮我出我途宼偷无戒兮吏静不苛仰有旄兮俯有稚非侯之仁兮谁使皇鍳孔明兮明且不颇嗟滇之人兮其独谓何揽侯裾兮絷侯马愿侯重来兮不聊此暇
　　槃谷赋
　　夫何太湖之淼茫兮峙双玉于西东蔚淀紫之巉岩兮峯峦抱而中空渡渚汪洋而交贯兮仍岛屿之无穷稻畦鳞次以栉比兮美西崦之寛宏亭台俨其髙下兮闾巷互而相通嗟夫人之好脩兮乃行义于其中握瑾瑜而不耀兮甘自混于凡佣朝出耕而暮读兮澹浮名之夕空非夫人之独善兮慨斯世之不庸不吾庸其亦巳兮信横目之为恫仰观山之峩峩兮俯翫水之溶溶俯仰山水中以自得兮将有言兮是从将无言兮是从
　　待隐园赋
　　冢宰杨公作园于京口曰待隐谓将归老乎此也而遭际明时愿莫之遂予为作赋以明其初志云词曰我滇于家兮邈云山之阻艰慨嵗暮予安归兮聊税驾乎江干稍葺茅以为宇兮又缭石以为垣羌艺蕙之百本兮忽覩竹之干竿初予与予心约兮逝将于此乎盘桓何皇恩之翕赫兮虽百身其犹未闲既授钺于阃外兮节制通乎三边旋持衡于庙堂兮坐进退乎百官誓捐躯而不悔兮顾初心其犹自怜怅松筠之无主兮恐猿鹤之多怨彼出处之难定兮伊自古而则然迨国恩之既酬兮将不日而来旋
　　呉子城赋
　　予毎过呉故墟未尝不慨想其盛而悼其衰也故为之赋
　　泰伯遗墟干将故里台阁翚飞冠盖鳞次喟彼荒郊羌何为乎城之里但见愁烟郁而四积悲风惨而时起頺墉突阜剰水残濠野雉朝雊鸺鹠夜号沉矛折防堕珥遗翘渐渐惟麦之秀离离彼稷之苖父老告予曰此呉王之遗宫也方呉盛时志大功髙入楚柏举败越夫椒城规方于八卦门僣拟于三朝跨长洲之茂苑馆苎罗之艳妖带以锦帆之泾压以金母之桥爰有凉台温室镂楣绣栭风亭月榭碝璧椒涂饰以璆琳琅玕间以木难瑚鸣佩鳬簉髙冠鹄趋自谓百世君之岂意至于是耶呉禄既更歴代崇饰春申夏桃秦皇刻石危亭岌夷雕栏纡直齐云之楼凝香之室木兰之堂交映翕赫叠石则嶻嶪嶕峣凿沼则囦潾澄碧兰芷罗生乎其间竹松骈列于其侧罗绮争春而妖冶歌钟入夜而嘈杂韦白耽于唫玩皮陆侈于酬答迨宋迄元更为治所双莲四照池光春雨嵗时观游丽无逾者而何至于是耶盖自元政隳羣雄骛白驹鹾酋乗间窃据挟嘉湖杭以自雄擢黄蔡叶而为辅盛稷下之文儒忽太湖之飞渡炀恋迷楼卓矜郿坞倐天兵四面以重围金城百雉而莫固技殚九攻苍黄一炬歴代繁华可怜焦土遂使燕巢再燬麋鹿重游竭南国之脂膏坐受其困激东溟之波浪莫洗其羞且夫倾宫阿房非不丽也钜桥琼林非不富也崤函巩洛非不固也自古如斯曷之故也岂仁义不脩燕安之可畏耶将气运靡常盈虚之有数耶惟是呉墟殷鍳斯在前既颠隮后仍荒殆登兹城以裵徊寄千古之一慨
　　洞庭两山赋
　　楚之湖曰洞庭呉之山亦曰洞庭其以相埓邪将地脉有相通者邪郭景纯曰包山洞庭巴陵地道潜逹傍通是未可知也而吾洞庭实兼湖山之胜始山特为幽人韵士之所栖灵仙佛子之所宅至国朝名臣彻爵徃徃出焉岂湖山之秀磅礴郁积至是而后泄于人邪东冈子曰山川之秀实生人才人才之出益显山川显之维何盖莫过于文两山者秘于古而显于今其实有待子无用辞予曰然乃为之赋其词曰
