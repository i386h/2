　　竹隐畸士集二十巻　　别集类二【宋】
　　【臣】等谨案竹隐畸士集二十巻宋赵鼎臣撰鼎臣字承之卫城人自号苇溪翁元祐间进士绍圣中登宏词科宣和中以右文殿修撰知邓州召为太府卿其集见于宋史艺文志者四十巻陈振孙书録解题云其孙纲立刋于复州本百二十巻刋至四十巻而代去遂止是在当时板行者已非完本刘克庄后村诗话又云竹隐集十一巻多其旧作暮年诗无枣本所纪巻目多寡颇不合疑克庄所称十一巻者乃専指其诗而言也鼎臣宋史无传其家世无可考独集中有缴进其父元祐末所上河议奏状一首今考宋史河渠志元祐绍圣间水官建议回河独转运使赵偁不以为然力主北流之议疏凡数上言皆切直与鼎臣奏状相合是鼎臣即偁之子渊源有自其后尝往来大名真定间与苏轼王安石诸人交好相与酬和故所作具有门迳能力追古人刘克庄称其诗谓材气飘逸记问精博警句巧对殆天造地设畧不防人喉舌费人心目其推挹甚至今克庄所摘诸句已多佚其全篇而即所存诸诗观之工巧流丽其才实未易及克庄之言故非溢美至其杂文刻意研练古雅可观亦非俭陋者所能望其项背惜原集久经失传谨就永康大典各韵中搜采彚辑勒成二十巻诸体具备蔚然可观虽未能齐轨苏黄然比于唐庚晁补之诸人则不啻骖之有靳矣乾隆四十六年九月恭校上

　　竹隐畸士集巻一　　　　宋　赵鼎【臣】　撰赋
　　邺都赋【并序】
　　仲尼有言质胜文则野文胜质则史扬子云亦曰事胜辞则伉辞胜事则赋盖赋者古诗之流也其感物造端主文而辨事因事以陈辞则近于史故子夏叙诗而系以国史不其然乎虽然文不害辞则辞不害志以意逆志其要归止于礼义者诗人之赋也两汉而下词人之赋始为丽淫竞相祖述至左太冲则讥之以谓卢橘非上林所植海若非西京所出辞不称事指为诟病然观其论魏也举禅代则以谓虞舜比踪述风化则以谓羲熊踵武非尧誉桀诞谩滋甚夫辨物或失其方记事之小疵拟人不以其伦立言之大蔽昔有独夫既殄天下同归于周明王不作海内莫强于秦然犹伯夷抗登山之志仲连懐蹈海之义相与耻而非之况乎助卫君之奸国褒吴楚之僣号以古揆今壹何相去之邈也方且笑昔人之未工忘已事之已拙欲使览者信之过矣尝因暇日读相台志盛言山川之美宫室之富爱其博而讥其杂于是感三都因相台志又颇裁其伪体削其醲词略前载之已详补后来之未备总折衷以有宋之制命曰邺都赋云尔
　　赵子与客游于三台之上以望邺都之城客曰乌虖壮哉豪隽所宅市朝之区三啓覇邑四成帝都子亦尝闻旧史之传观相台之志乎赵子曰未也相台志何言哉客曰魏之初基厥惟冀野中奠邺而居相始冠冕于诸夏世虎踞以龙蟠亘千龄而髙跨其左则浩浩洪河犇涛涌波汇以清渊之聚为黄泽之涡东崓嵚岑而突兀西陵屹嶫而陂陀其右则太行隆虑玉泉天平擎据穹壌蔽亏日星隐若叠障巍如置屏斩晋脊以中断连代襟而外萦其阳则宜师之沟司马之泊交荡羑之浸润互淇泉之脉络矗大岯以中峙障惊澜而犹却猗緑竹以弥望丰邦储而利博其隂则近控漳滏逺连常碣水东逝以委带山西倾而坠玦广迳方轨平原四逹豳扼吭以形茹赵咋喉而气夺若乃厥赋上上土厚水深穜稑蔽野桑麻耀林顷必万秉畆皆百金错轮毂于涂中资菽粟与缣纴有葛屦之遗化故虽富而不淫尔其雅俗推重尚气矜节三晋之豪四君之侠顾眄刄挺喑呜眦裂驱市人而一战可挞秦而系越故其物伙财阜兵雄势张择要塞以胥宇据膏腴而造邦思重闭以固国务崇宫而濬隍笑前帝之阨廹图后王之炜煌悭瑶台而吝琼室讵数阿房与未央盖其号也前则铜爵金虎显扬太武后则凉风清都鸳鸯鹦鹉鬬鸡戏马之榭赤桥紫陌之籞耸飞阁之千雉联危谯于百歩玉壁珠帘银楹金柱不可亿记而指数岂直千门与万户伟鳯阳之特建饰翔离之双鸟何帝子之懐彼亦矜能而护巧物太盛以弗禁假飞柯而匠夭怅灵质之难驻忽遡波而孤矫伊仙都之肈造恢华林之旧规倾地产于土木殚心计于般倕嗟燕翼之败度反歆羡于穷黎谅习侈以玩富岂艰难之与知故其制创之广博缔构之穹隆藻饰之瓌丽观眺之防通升而瞰之则翼翼眈眈云储雾涵泰华仅形于培塿辰极才映于楹櫩类竦身于天表迷不悟于北南仰而瞻之则炳炳欝欝烟霏日出但闻丝竹之音环佩之声似蓬宫之仿佛于是孟德季龙之际慕容渤海之俦鹿既获于嬴氏羊方絷于宗周则燕则誉以遨以游莫不宰割区宇分裂华夏邀乙发以计功揖髙光而第课正南面以居尊袭冕旒而朝羣下于时鞮女畏力旄倪戴仁来享来贡或臣或宾彼孙马之旅拒潜窜迹于江濵如黒子之廑着亦何与于藩邻若乃泛览九原周流四坰求故老之謡咏访先民之典型过羑里而太息悲伯昌之絷囹圣人发愤以储思演防言而作经则天人之学所由兴也邦有滛祀西门发之野有巨浸史公决之或蠡或贼岑君遏之生歌死哭齐民悦之则守令之贤有善称也周鼎既震尉迟抗钺隋歴方季尧生挺节墓藏太尉之骨衣溅侍中之血力殉庙社气凌氷雪凛芒采以映世虽身摧而志洁则忠谊之士蔼令名也吐实为秋掞华为春雕刻造化鑪锤鬼神唱则应刘徐陈和则邢魏子升俨思王以独歩若众星之拱北辰则文章之美擅英声也尔其洞宂漫汗仙灵屈竒絶伎之所兴起肥遁之所幽栖酒千里以噀火丹一丸而疗饥孙登放浪于谲诡佛图巧幻而瑰琦森逸驾与隽轨差难得而备知若仆者徒捃摭于旧志譬犹万钧之一厘故言其地则四境莫及论其人则一邦有余古在前以不足今居后而岂如首相亶甲起踬奋跲当涂继兴遂城遂堞三髙厉吻以旁噬二石磨牙而外猎率皆禆壊补敝披图案牒据之则防去之则怯得之则为妇失之则为妾緜奕姓以传祚常鋭锋而不慴夫岂代无僻主哉亦云形势之所挟也是以文命敷土厥载惟冀美哉山河武侯所伟逺兆朕于听音迩萌芽于望气故张宾慷慨以揆防崔光恳激而纳説元氏弗康卜雒之阳曽二帝而乃蹶信人谋之不臧逮夫李唐中叶有震且业彼庆绪之孤童奋螗肱而拒辙由借险以凴固故王师之不渉以是观之人无贤愚而地有厚薄兵无利钝而势有强弱诸葛家蜀而破宣尼用鲁而削故仆以为冀不常重于天下而钜邺独雄于河朔也岂不谓然哉于是赵子慨然而叹有间而荅曰异乎吾子者之撰夫登髙能赋者非以耀文所以辨义也陈诗观风者非以娱目所以验俗也故陟景山者见中兴之美馆雒汭者思平成之功未闻商颂有固圉之谈夏谚啓崇墉之论也今客诵糟粕拾腐余掉三寸之舌美六尺之躯指金汤为仁寿之域谓干戈为礼义之涂长王追貊主盟裔俘遂欲覇燕赵而帝齐魏兄二京而父三都是犹防鸠豕虱择蹄而系苇自以谓有王侯之乐安土之娱曾不知风摧火燎率虀粉而遽焦枯也褊居晋鄙狭隘沮洳子迁姬系灭迹失据虽无忌之杰黠亦何延于天祚炎风不竞典午委驭故使阿瞒与贺六得防张而防扈况值羯胡与鲜卑岂正余分而闰序当建安之考室兾日月以齐光奚三马之不戒俾一槽之遽亡彼季龙之侈忲摐地防以天崩曾僵胔之未腐遂流水以飘零凡此皆莫绍其终无传于始三光之所不纬五行之所不纪危朝菌之待暮徒倐生而忽死而吾子则美之岂其惑于旧贯欤胡取舍之不韪也自昔属辞之士缀翰之徒不务明五典之常道九畴之盛符乃惟夸国炫域度城计郛较广狭以诽訾无异商防与贾区盖尚东都者以伊洛为帝王之里主西京者谓崤函为天子之居甘泉逺防于西海上林左引于苍梧将誉美而章恶信欲近而返踈左生﨑岖遵迷遂诬拏三国以等竞角蜗牛之所庐不独陵轹岷蜀诋諆句吴而已又将超秦越汉污唐渍虞尊盗臣以拟圣谓羲熊之可逾客又实之岂不过欤仆病木讷而倦于谈虽然请为客粗陈其靡者子独不闻皇宋之光宅清汳之规摹乎背澶面谯挈郑提曹非宛非邺不瀍不崤邦畿千里坦然四郊环万国以面内类百川之海朝厥初生民纉唐之绪纷纶后辟昜代以五咸罔堪于顾天乃眷命于艺祖当此之时陵谷昜处天地否闭野有舆尸室无噍类国靡一定之民朝乏委质之士冀媮食以终夕敢燕居而卒岁帝用不蠲既薙既薅扫欃枪于一皷荡浇豷于崇朝援赤子而出涂炭谒大命于旻霄蚩蚩遗黎不徒去乱即治诛蓬刈蒿乃复慈母游乐郊安太平之后笑悔既徃之先咷既而人获更生时有逺虑成周饰阙以望幸西镐清宫而请御臣怀良敬之防士献班张之赋咸荣古以陋今脗雷同而景附天子穆然载思而未俞也乃规干矩坤与神合契灵谋睿谟尽屈羣议即故国以营基懋皇图于亿祀盖谓洞庭梦薮曷若以四夷为守未央章华孰若以六合为家方且陟岱勒华裒神翕河建道德以为营卫讵论丘垤与污沱至于体国经野之法宫室苑囿之制寝不逾庙菲不废礼姑以备一王之轨仪同百姓之欲利固无伪采与滛泰畴克夸诩而奢丽若乃风俗之纯懿政教之缉熈人物之磊砢货殖之陆离既立谈而未判且非创业之枢机徒尚口以哗众亦鄙人之不为也皇皇百年显显七世尧父舜子神传圣继无增尺帛之奉不益十家之费晏然磐石而覆盂若天维而地置遂使豫里辍险雍郊弛防江靖建业气清南阳洞外阖以不闭咸变雍而乐康于维此邦陶醇化醲川润岳峻旁薄虚空蒸粹炳灵降为英雄作我国栋时惟魏公勲在社稷行铭鼎钟非后牧与伊稷畴并芳而比崇兹乃相之所以隆也客遂畧之鲸鲵陆梁峻宇雕墙葅醢庻类黥灼一方斯乃邺之所以亡也客顾乐之意者非相人之志盛德之事缙绅之所宜谈国史之所可记也客又不闻嬀氏陶渔三年成都商盘五迁邑无奠居安有择地而化被简民而信孚者哉将吾子未之思乎于是客乃词殚辩屈不悸自栗泚颡却避懵然若失
　　尧山赋【并序】
　　尧山在县北十里不甚髙大四望皆平川而居中岿然特起按图经尧尝登此以望洪水后人思之立庙其上故以名山因以山名县元祐已已冬余以童子侍亲过之追感遗迹为赋以叙之
　　当季冬之严凝兮驰燕路之骎骎揽征辔而踌躇兮过尧山之嵚岑按图牒之所传兮得遗迹而追寻曰尧尝登于兹山兮望洪水之湍滛嗟皇天于艰难兮在圣人而忧患物何生而不乐兮独水行之漫漫曽九载之莫息兮舞鱼龙于波澜惟圣心之焦劳兮念元元之不食举鳏民于侧陋兮登咎防与禹益虽羣后之奏功兮匪自安而用逸登兹山而四顾兮忧盖深于已溺余诚无心于黄屋兮哀赤子之将鱼茍一身之可济国兮固何有于捐躯赖文命之底绩兮卒大安而晏如嗟殷季之怠荒兮侈天位而自康峻瑶台于深宫兮民或亡于盖藏犬马饫夫梁肉兮士或未充于糟糠惟口腹以纵汰兮孰防民之否臧既上下之覆隔兮终沦胥于防亡仰尧帝之孔仁兮轸父母之至慈流惠泽而到今兮民愈乆而慕之即山巅之崔嵬兮尚经营其神祠俨衣冠与环佩兮犹髣髴乎天日之姿瞻遗像以兴叹兮故老对余而嗟咨怅徘徊而不忍去兮夫孰知余之所思
