　　游仙诗四章，为祖紫云作，其一（贞一斋诗文稿卷下，宛委别藏本）
　　五陵有佳士，乃在西山颠。昼餐金光草，夜读青苔篇。俯视世间荣，于我若浮烟。野鹤出云表，潜鱼媚深渊。至性不可羁，谁能识其全？惟应羡门子，相与长周旋。（1）
　　　其二（贞一斋诗文稿卷下，宛委别藏本）
　　西山多灵药，服食颜色好。在昔有洪崖，高举云得道。曹卢继芳躅，天地长不老。后来紫云翁，辟世一何早！连云种参杞，带月拾瑶草。贵贱俱营营，终然恨枯槁。栖迟匪迷涂，破褐自怀宝。去去不可留，期之在蓬岛。（2）

　　戊辰年腊月，大雪弥旬，古樟黄竹皆冻死。百岁翁云：有生以来所未见者。追忆儿时乐事，赋诗三十二韵，寄呈仁叔兄，并示云章弟、立竹侄（贞一斋诗文稿卷下，宛委别藏本）
　　忆昔童艹时，见雪喜无寐。清晨走相唤，兄弟剧嬉戏。击竹坠琼玖，攀檐泻饙饎。模形常透怀，印迹屦亡屣。玉虬巧追琢，龟手那肯置？冰盘脱明镜，蜜盌空美饵。盥栉固已忘，笔砚亦胥弃。父师恒检责，掩耳宁畏避。充然有余乐，岂惜岁月驶？中年客京华，飞雪大如翅。天街跨鞍马，顷刻须鬓异。归来拥貂裘，酒炙交饷馈。公卿共谈笑，盈尺诧其瑞。虽无少年欢，颇惬四方志。朅来五十六，老态日夕至。凭陵怯隆冬，疏懒便午睡。弥旬值冻雪，尽日响寒吹。虚明眩窗户，凛冽逼衣被。炽炭曾不温，醇醪漠难醉。万室被纨素，千山失苍翠。狡兔方迷离，饥乌亦憔悴。豫樟变枯槎，黄竹絶生意。适闻百岁翁，咄咄书怪事。荆扬古炎洲，杀气何以致？或言兆丰稔，蝥蠈深入地。又云弭残瘴，自此蕃庶类。胡为鲜欢笑？孤坐成叹喟。翻思同队时，此乐讵可觊？朱颜与绿髪，去去不再值。深心幸闻道，俯仰复无愧。踪浪囿元化，浮生眇如寄。聊为冻雪吟，庶用勖（3）贤知。（4）
　　道传先生才学重当世，与予游将二十年。往岁别于都中，思慕良切。比提学江西，复时［得］聚首。○1殆将满考，乃访予玉隆，宿留数日。临别，赋长句见赠。次韵奉酬，聊叙今昔以写怀耳（贞一斋诗文稿卷下，宛委别藏本）
　　忆昔事北游，山川饱经行。挥鞭过河朔，振袂趋神京。维时柳夫子，倾盖相逢迎。成均有鸿儒，安能复遐征？箴规靡晨夕，孔道日以明。义胜尘虑遣，理融心神清。粤从分别来，耿耿驰中诚。易水霜月白，燕山云树青。朅来振鸣铎，坐使聋聩惊。翼轸吴楚交，秀气锺元精。自古足文献，化机浩无停。欧曾出奇正，王晏何踪横。往者不可作，汗漫骑长鲸。予生百年后，企望心靡宁。公来实增重，末俗儒化成。贪污变廉耻，譣诐还忠贞。信知德有征，孰谓风无形？圣皇急求贤，为霖洗炎蒸。召还匪朝夕，羽翮横青冥。挥手提天纲，举足顿地纮。吾闻柳柳州，千古更炳灵。抵掌载言笑，吾孙得吾軿。家声亦烜赫，何心尚怦怦！夫子不自满，犹言有常经。用舍非所牵，澹然自和平。俯仰无愧怍，在己无德馨。（4）再拜服斯训，文华愿收声。当为松栢姿，勿作桃李英。眷言从兹游，蔬笋供南烹。歑忘鸳鸿侣，乃与鸥鹭盟。临期重携手，欲别终含情。高歌伐木诗，侧耳嘤其鸣。（5）

　　放鲮鲤（贞一斋诗文稿卷下，宛委别藏本）
　　大钧播万物，赋形有媸妍。羽毛族林薮，鳞介繁江川。嗟尔穴居类，蓄缩如鳗蜒。修鳞与利觜，触地无刚坚。壑饮混泥滓，蚁食多腥膻。于人既无害，在物苟自全。鹿豕每犯稼，虞罗所宜先。蛇虺恒伤人，掊击理则然。尔肉不登俎，尔名古无传。野人亦何为，畚锸必及泉。得之诧珍异，笼槛置我前。且盐芼姜桂，甘美逾割鲜。性复去风湿，力能已拘挛。感子献芹意，肉食吾无偏。风痹非所患，藜藿良独便。南烹已云恶，韩苏着诗篇。蜜唧常呕吐，鼃黾曾弃捐。河豚美而毒，智者宁垂涎？况此更微贱，忍以汤火煎？逍遥古名山，土壤大（5）且卷。使尔遂生成，族类当绵延。长者活鱼报，还生忉利天。放尔匪缴福，夭阏诚吾怜。（6）
　　送蔡体仁三首，其一（贞一斋诗文稿卷下，宛委别藏本）
　　秋高万木落，夜永候虫鸣。披衣待明发，骋望空前庭。悠然感所思，乃在天北溟。一别四五载，关山何峥嵘！绿酒薄清赏，素琴歇芳声。忽有佳客至，聊足慰吾情。（9）
　　　其二（贞一斋诗文稿卷下，宛委别藏本）
　　佳客江海士，儒衣章甫冠。文工画更精，风雨生毫端。不作脂粉媚，不为儒生酸。孤骞出云表，远访千年丹。丹成井不改，仙去随飞翰。握手叙契阔，碧树生初寒。（9）

　　发山中（贞一斋诗文稿卷下，宛委别藏本）
　　榖旦发名山，驾言趋上京。车马如云集，把酒送我行。须臾陟东岭，回盼仙人城。晚圃秋正浓，露华缀金英。胡为舍此去？乃与尘俗萦。人生有行役，岂必皆蝇营？威鳯高其翔，千载相和鸣。勿作儿女别，慨慷舒长缨。（13）
　　芗溪卧病（贞一斋诗文稿卷下，宛委别藏本）
　　辞家至芗溪，信宿不能去。六气冯吾躯，右足重如注。医师试针砭，转覚气血聚。呻吟数更漏，永夜杳难曙。令佐贤主人，情亲靡新故。盘餐交馈饷，药石云致助。怀土非吾心，岂为桑下顾？安得乘飞车，一去千里路。（13）
　　玉山县（贞一斋诗文稿卷下，宛委别藏本）
　　肩舆出城闉，初日半天赤。暮宿玉山县，风雨秋寂历。连峰结长阴，一水界空碧。翻思壮游地，白髪非宿昔。去去欲何为？感叹不能息。何当营丹砂？吾师契金石。（13）
　　庙山九日（贞一斋诗文稿卷下，宛委别藏本）
　　畴昔使钱塘，夜宿庙山驿。维时九日至，丰年尽欢怿。亭长醉如泥，推情靡呵责。迩来二十年，驻此感今昔。黄花复重九，采采耀颜色。良由没巨浸，鱼鳖为鲜食。壮健多流亡，老羸转沟洫。厨传恒萧然，酒炙那复得？清晨驱车去，城闉更寥阒。当道多相知，欲语泪沾臆。东南千万斛，岁漕输上国。今兹民力竭，何以继供亿？圣明仁如天，闻此应怆恻。谁当绘为图，献纳通宸极？（14）
　　寒疾（贞一斋诗文稿卷下，宛委别藏本）
　　足疾良未愈，又与寒疾婴。黄昏抱冰雪，二气疑交兵。齿牙自相击，颤棹儿童惊。须臾内热攻，袵席风火生。汗流甚出浴，渴饮如长鲸。呕泄加痛楚，永夜同严更。（14）投老复行役，何以舒吾情？寄语故山友，慎勿辞力耕。（15）
　　题柴南山云岛隠居图（贞一斋诗文稿卷下，宛委别藏本）
