　　提要
　　【臣】等谨案豫章文集十七巻宋罗从彦撰从彦字仲素沙县人以累举恩授惠州博罗县主簿绍兴初卒淳祐间追諡文质事迹具宋史道学传是编为至正三年延平进士曹道振所编以宋儒称从彦为豫章先生因以名集道振原序称郡人许源堂刻其遗集五巻近得邑人呉绍宗稿厘为一十三巻附录三巻外集一巻年谱一巻凡一十八巻此本乃明代重刻前有成化八年张泰序后有嘉靖甲寅谢鸾跋凡遵尧录八巻集二程及杨龟山语录一巻杂著二巻诗一巻附录三巻外集一巻以年谱别置于前不入巻数故题为十七巻然第一巻虽列经解之目而其文久佚有录无书实止十六巻而已乾隆四十六年十一月恭校上

　　豫章文集序
　　泰尝读孟子称舜大孝章至延平李氏注有曰昔罗仲素语此云秖为天下无不是底父母了翁闻而善之曰唯如此而后天下之为父子者定徐而思之已知先生立言埀训之绪余矣初未尝获覩所谓豫章文集也暨筮仕知沙阳明年春二月适今提学宪副丰城防公按节考校之暇手以是集授泰曰是廼豫章罗先生遗文前进士曹道振编次校正梓行于世久矣正统戊辰燬于兵燹殆尽其幸存者仅见此本亟图锓梓以广其传可也泰对曰谨受命自是退食之余披诵累阅月于是益有以知先生渊源之所自葢先生初受学于山之门闻山讲干九四爻义曰伊川说得甚善即鬻田裹粮适洛中求教于伊川竟不外山之说既而南归益肆力于圣贤之学晚就特科授博罗县主簿居罗浮山中静坐三年以观天地万物之理超然自得而不滞于言语文字之末故山之门从防者众求其潜思力行任重诣极先生一人而已今观集録彪分胪列大而君臣父子之伦小而事事物物之故以至于道徳性命之奥靡不该悉则是书不可以不传于世明矣或曰传其书不若明其道求诸言不若求诸心是固然矣然而圣贤之道具于心圣贤之心形于言不得于言而能得其心者鲜矣不得其心而能得其道者未之有也学者茍能即其书以玩其言则心与道可得而言矣夫六经四书皆道之所存也尧舜禹汤文武之所以建极保民孔曽思孟之所以埀世立教周程张朱之所以著书立言举不外乎是先生上承伊洛山之统下启延平晦庵之传斯文一脉万世是宗而可不因其言以求其心传其书以明其道也邪是用重锓诸梓以广其传与四方君子共之凡有志者获覩先生是书口诵心惟而力行焉生乎百世之下而有以传先生之道于百世之上穷则淑诸人达则善斯世则于风化之万一庶几或有小补云成化八年龙集壬辰十二月甲子日赐进士文林郎知沙县事后学岭南张泰拜手谨序

　　遵尧录序
　　尧舜三代之君不作也久矣自获麟以来讫五代千五百余年惟汉唐颇有足称道汉大纲正唐万目举然皆杂以霸道而已有宋龙兴一祖开基三宗绍述其精神之运心术之动见于纪纲法度者沛乎大醇皆足以追配前王之盛故其规模亦无所愧焉在太平兴国初太宗尝谓宰相曰朕嗣守基业边防事大万机至重当悉依先朝旧规无得改易仁庙见东封西祀及修五清宫等过侈曰如此之事朕当戒之若二圣者其知所以绍述者邪故终太宗之世无复改张终仁宗之世一于恭俭至熙宁元丰中不然管心鞅法甲倡乙和功利之说杂然并陈宣和之末遂召金人犯阙之变盖其源流非一日也今皇帝受禅遭时之难悯生民之重困也发徳音下明诏悉刬熙丰法一以遵祖宗故事为言四方企踵以望太平矣议者犹谓金陵之焰势未能熄天下皆其徒是抱薪而救之者也臣惧其然也窃语诸心曰昔唐呉兢作贞观政要录本朝石介亦有圣政录岂茍然哉因采祖宗故事四圣所行可以闿今传后者以事相比类纂录之歴三季而书成名曰圣宋遵尧录其间事之至当而理之可久者则衍而新之善在可久而意或未明者则释以发之以今准古有少不合者作辨微以着其事又自章圣以来得宰相李沆等及先儒程颢共十人择其言行之可考者附于其后若乃创始开基之事庙谟雄断仁心仁闻则于其君见之袭太平之基业守格法行故事竭尽公忠则于其臣见之爰及熙丰之卒归于道分七巻添别录一巻合四万余言欲进之黼座力未暇及而秋毫之间已爽忽矣然事固有始暌而终合失之于前而得之于后者古人有之若周成王楚文王秦穆公是也不久朝廷清明金人宾伏且当有以来天下之言辄纪岁月以俟采择靖康丙午十月日延平臣罗从彦序
　　遵尧录一
　　太祖
　　国初劒南交广各僭大号荆湖江表止通贡奉西夏北辽皆未宾伏太祖垂意诸将命李汉超屯闗南马仁瑀守瀛州韩令坤镇常山贺惟忠守易州何继筠镇棣州以拒北敌又以郭进控西山武守琪戍晋州李谦溥守隰州李继勲镇昭义以御太原赵賛屯延州姚内斌守庆州董遵诲屯环州王彦升守原州冯继业镇灵武以备西戎其家族在京师者抚之甚厚郡中管搉之利悉与之恣其图回贸易免所过征税许令召募骁勇以为爪牙凡军中许便宜从事毎来朝必召对命坐赐以饮食锡赉殊异以遣之由是边臣皆富于财得以养募死力使为间谍洞知蕃夷情状毎外敌入冦必预为之备设伏掩击多致克捷二十年间无西北之忧以至命将出师吊民伐罪平西蜀复湖湘下岭表克江南兵力雄盛武功盖世良由得猛士以守边推赤心以御下之所致也
　　太祖以李汉超为闗南巡检使捍北敌与兵二千而已然以齐州赋敛最多乃以为齐州防御使悉与一州之赋俾之养士而汉超武人所为多不法久之闗南百姓诣闗讼汉超贷民钱多不还及掠其女以为妾帝召百姓入见便殿以酒食慰劳之徐问曰自汉超在闗南契丹入冦者几曰无也帝曰往时契丹入防边将不能御河北之民岁遭劫掠汝于此时能保其赀财妇女乎今汉超所取孰与契丹之多又问讼者曰汝家几女所嫁何人百姓具以对帝曰然则所嫁皆村夫也若汉超者吾之贵臣也以爱汝女则取之得之必不使失所与其嫁村夫孰若处汉超家之富贵也于是百姓感说而去帝使人语汉超曰汝须要钱何不告我而取于民乎乃赐以银百两曰汝自还之使其感汝也汉超感泣誓以死报
　　太祖以郭进为西山巡检有告其隂通河东刘继元将有异志者帝大怒以其诬告忠臣命缚其人予进使自处置进得而不杀谓曰尔能为我取继元一城一寨不止免尔死当请赏尔一官岁余其人诱其一城
　　来降进具其事送之于朝请赏帝曰尔诬害我忠良此才可贳死尔赏不可得命以其人还进进复请曰使臣失信则不能用人矣于是赏以一官
　　太祖以贺惟忠知易州及捍边有功迁正使开寳二年又加本州刺史兼易定祈等州都巡检使惟忠在易州十余年缮治亭障抚士卒得其死力毎乘寒用兵所向必克威名震于北敌
　　太祖以李谦溥为隰州刺史在州十年幷人不敢犯其境开寳三年移齐州团练使后边将失律复以谦溥为晋隰防边巡检边民喜之
