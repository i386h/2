　　钦定四库全书　　　　集部三
　　忠肃集　　　　　　　别集类二【宋】
　　提要
　　臣等谨案忠肃集三卷宋傅察撰察字公晦孟州济源人年十八登进士第蔡京闻其名将妻以女拒弗荅调青州司法防军厯迁吏部员外郎宣和七年充接伴金国贺正旦使时金人已起兵而宋人未之知察至玉田县韩城镇使人不来居数日金数十骑驰入馆彊之东北行遇金大帅斡哩雅布迫令下拜察抗辨不屈死之赠徽猷阁待制乾道中追諡忠肃事迹具宋史忠义传集为其孙直焕章阁伯夀所编周必大为之序察为尧俞从孙学有原本史称其文温丽有体裁特年三十七即遘国难以死未尝有大著作以发挥其才气故集中所存大抵酬赠篇什及表笺启劄骈丽之作为多然其深厚尔雅亦自足资讽诵且当宣和未造士大夫名节扫地之余一旦猝遭事变独能奋身碎首致命成仁卓然自遂其志其忠义之气尤有不可磨灭者诗文虽属无多固当与李若水之忠愍集并垂不朽矣乾隆四十五年三月恭校上

　　忠肃集原序
　　二帝三王时人才多出于国子盖其见闻积习作成有素非如秀民必俟族党州乡賔兴之然后用也观舜命防典乐教胄子曰直而温寛而栗刚而无虐简而无傲及命契爲司徒教民则敬敷五教在寛而已周官大司乐以乐徳教国子曰中和祗庸孝友及司徒以三物教万民则置礼乐于六徳六行之后其视成材详略次第固有别矣夫子不云乎兴于诗立于礼成于乐学至于乐则义精仁熟和顺于道徳而性成焉故以之事亲必孝事君必忠临大节必不可夺文其余力也晋唐以来各学与监置祭酒若司业皆冠以国子亦古之遗意欤本朝世臣巨室前后相望在仁宗时有若献简傅公讳尧俞未冠以进士起家平居号称长者及事四朝屡歴言路忘身殉国有不可夺之节遭时遇主致位二府生多美誉没保令名遂爲大家其从孙忠肃公讳察年十有八复践世科宣和末以吏部郎假宗正廷劳金国贺正使行及境上防幽燕交兵或劝其归公不可猝遇斡哩雅布强公使拜公又不可竟握节死之诏赠徽猷阁待制乾道中追赐今諡其诸子皆以问学才猷翺翔仕途至孙伯寿文采益髙方以直焕章阁按刑畿部兴念前烈既编定献简公草堂集又裒公遗稿成三卷将传布四方属必大序其首惟公文务体要辞约而理尽甚类献简诗尤温纯该贯间次韵愈多而愈工惜乎享年才三十有七名宦未逹髙文大册无自而发其渊源则可考矣窃尝论之献简幸生太平无事时止于正言不讳是以爲宋良臣公则不幸将命艰难之际仗节死难遂在忠臣之目要之忠孝大节易地则皆然特所遇不同耳故爲推本帝王之教以及本朝之盛使学士大夫知公世济其美不陨其名者如此周必大序

　　诗
　　用亷夫韵招诸友登清防亭
　　束带缚吏事发狂中愤兴世情多面朋噂防恶背憎缅懐数君子皎皎莹壶氷披云见顔色洗翠露锋棱忘形到坦率髙论破炎蒸蔼然嘉誉在顾我独无称文塲屡交镝师老畏侵凌况复时雨过绿苗覆千塍便当共临眺此兴那可胜孤亭枕席佳下有寒潭澄仰看霞绮散坐待月华升嘲咏纵欢谑明朝期再登
　　亷夫有防当归去来之句复次前韵
　　少小尚奇伟长年病莫兴折腰仕五斗讵敢论好憎居然畏罪罟局如履春氷每懐鹰隼翮韝上防锋棱追随缙绅后糠籺杂肴烝泮宫天下士借甚得儒称巍巍众峰上跛足岂易凌朝廷重荐延防茅余空塍公孙富经术非子谁能胜坐见万顷波甖缶汲逾澄沛然下膏泽蓊若朝云升勿云叹留滞青史他年登
　　次韵亷夫登清防亭二首
　　蜗舍不容膝烦歊局寝兴岂惟倦束带絺绤犹云憎清防跨城角窣兀疑凝氷深池暮黯黯逺岫朝棱棱往嵗与申子賔朋日云蒸遂令栋宇陋顾盻流名称先生今诗伯气欲青云凌锵然出佳句椒兰布畦塍邀我避暑逰意重愧难胜何当酌沆瀣神明自清澄嗟彼浮名者此亭岂易升踟蹰恐后时摄衣请先登
　　大火气方炽新金伏未兴茅屋苦卑陋蕴毒空自憎流汗常被体涤烦屡饮氷那堪青蝇軰鼓趐作威棱薰风烁炎日沸鼎助薪蒸每思谋爽垲地隘旧无称高亭枕城隈粉堞势腾凌清水绕其足茂林瞰田塍此间有异趣跻攀弗能胜佳招适我愿坐觉神魂澄何当卜邻舍与子朝夕升飘然防世累长啸效孙登
　　次韵祝亷夫登南楼
　　膏雨润四野草木苏中干驾言共临眺凌云陟层盘华宇荫曦日溽暑随风防清谈无俗好茶果有余欢拨置簿领烦憇此簟席安悠然得佳趣可以寄簮冠耳目去局促心体自广胖何如敛手版触热事上官晓来纎霭霁矫首欣遐观一鸟没天外千林杳云端西山有爽气浅绿浮防岏翻疑图画里未觉云梦寛萧萧林间蝉已如咽暮寒直恐霜露至归欤可忘餐
　　同七兄寄二李
　　昔我在淇水二陆皆少年伯也气轩昻丹鹤唳青田季方文章士摛华桃李妍力学乃未已有如火始然相见一倾葢便诵胶漆坚笑语玊麈折登临蜡屐穿时时出秀句洒若沈疴痊尔来逾十嵗自当冠貂蝉胡爲茅屋下寂寞尚草方今明堂构收拾栋与椽安得刷羽翮吹送上天此志傥不遂何用空言传不如李翰林沉醉自称仙不如陶彭泽归来谢尘縁但解琴中趣何妨瓮傍眠富贵草头露名誉空中烟子意竟何如余心方寂然愿求真实语莫作猾头禅
　　夏日登清防亭
　　永日留烦暑斯楼得屡登向来知异境高处若无凭逺岫生云直轻风蹙浪层试承金掌露更接玉壶氷抵掌賔朋盛题诗气象増望穷遗憾在我欲醉时升
　　清防亭分韵得空字
　　缥缈千层外萦纡一径通飞甍明落日巨栋挂斜虹赏眺资公退招邀得调同九天开浩荡万象入牢笼水色连晴碧云容借晚空清乗庾公兴快挹楚王风弁侧金罍倒楼昏画角终明朝车马客陡觉厌尘红
　　大暑同蒋明亚叟熈载夜登清防亭
　　土润伏新金阳歊困亭午曦御驻征骖朱光彻厚土鸟雀噪堦除蛙蚓鸣草莽簿领抗尘容茅檐庇环堵蜂虿乱防床蚊蟁隠雷鼓防拂徒尔爲青蝇敢予侮避喧空百虑对食但三吐举箑未揺风挥汗已成雨那能整冠巾谁复对賔主岂惟生花坐觉气如缕南方有高楼兀然蔽华宇抠衣倦跻攀策杖愁伛偻俄顷叩云端飘萧开洞府新凉防尔来旧暑脱然愈清池涌波澜踈星穿牖户夜气袭氷霜月明粲珪组乌鹊倦还飞逰鱼清可数菓走金盘清谈挥玉麈羣籁正嚣嚣幽梦方栩栩何当食琼浆一夕生毛羽
　　夜登清防亭云阴掩翳东北有雨少顷月出皎然复次前韵
　　融融暑未防悄悄夜方午命驾聊叩门寻幽得兹土俯瞩但苍茫前瞻何卤莽万籁正虗无羣生各安堵蝉噪作笙箫蛙鸣当钟鼓鳬鴈正参差诙谐惧狎侮岂惟冠葢倾已觉肝胆吐坐久隠奔雷天邉垂宻雨踈木列賔朋众峰自臣主萤火落明星炉烟袅青缕那能胜酒杯所欣瞻徳宇高歌屡激昻失喜或俛偻萧防濯尘襟清虗湛灵府快如仙爪爬洒若头风愈湏防出金盘照曜开玉户映浪杂琼瑰穿帘间璧组斯逰岂前谋后防难逆数欢余岸纶巾语约挥松麈可能伴鸡栖亦复随蝶栩广寒防可升试爲刷翎羽
　　次晁之道韵
　　翰林诗百篇发兴自斗酒同逰八仙中七者顔已厚迩来数百年斯人那复有风流城东晁四海蔚称首之子乃其豪异代得宻友满腹贮诗书妙意寓醪酎政使誉日新位卑未爲丑亭亭涧底松不如园中栁芝菌冠众芳其出木枯朽呌阍问大钧得致君侧否
　　送承祖赴定州幕府
　　中山控北陲强弩号百万单于衔国恩方且事逃遁谋帅得真儒不烦一防顿风流本江左亹亹有余论幕府极妙选参谋不容逊借甚丞相孙凌云气豪徤所懐社稷计尝愿窃有献蹭蹬三十年羇旅颇日困因人作逺逰实亦乖素愿谈笑青油幕聊复可排闷嗟余谬从学尘根苦迟钝一官聨事久矜式毎日劝但觉有不如未见有进寸公亦不吾厌叩撃愧烦溷念当爲逺别怅然亦多恨时方苦炎蒸行矣蕲强饭
　　伯时往河东戱与亷夫分韵道相思之意三首
　　日月剧跳丸暂别亦已久青山缭层巅道里阻且右今晨登南楼防此同志友思公聊北望竚立但搔首秋风八月天霁色明朝牖轩车来何时迟尔一杯酒
　　朝氛霁西山晚日明髣髴遥怜太行山陟巘可畏王侯眇小夫中盘凌云气端如车公贤谈笑出真味胡爲久崎岖使我重歔欷王事有程期来归惧其未
　　之子何磊落本是青云才委翅蓬蒿下黧面垂防防卜邻与我逰肝胆向人开酒酣遗世累谑浪絶嫌猜西风换烦燠四野无纎埃方当共临眺公兮胡不来
　　题毛彦谟容膝轩
　　夫子负异才端若防稽笴平生懐刚膓于世少许可文词类激昻议论耻媕婀高名三十年坐此常坎坷同时英俊流往往歩青琐归来乐箪瓢琳宫得安坐小轩仅容膝陋巷生堀堁唯余豪侠气屡折每不挫却眡利名塲终身系缰锁生虽华屋居死或翳蓬颗固知宇宙间所有终非我但当清吾心余事等么麽
　　次七兄韵
　　蜗舍席爲门清风无所扇闲视才半寻起行空百转对食不下喉流汗常被面矧复鳬鹜行摘尾抱大卷余方有公事汝軰宜罢遣頼此伯仲贤纵谈逞雄辨时时杂博奕岂不贤逰燕呼奴取其来据榻聊一战初如两军交整阵咸伺便俄顷锋刃集解纷在一箭或喜而抚掌或骇而色变或傍睨而审或当局而剥剥伐木声离离众星现胜负既已分反奁若收电却寻文字观触物本无恋覔句屡吟哦诵诗乍逰衍百谪勿相闻从渠书我殿
　　晚登清防亭次祝教授韵
　　雉堞亭亭倚翠防髙楼袅袅弄清晖共邀嘉客扶笻杖骤喜凉风振羽衣过雨山川颜色好投林鸟雀噪声稀月揺翠浪明珠碎云满金波破镜飞雄辨琅琅如落屑新诗句句似连玑自怜性僻躭幽寂却恨匆匆便赋归
　　送朱熈载改官还乡
　　陶唐名臣有朱虎揖逊防夷尊文祖后来杰出汉云孙折槛当时谁敢侮煌煌圣代见斯人凛凛遗风振千古乗牛不数鲁家侠衣锦何妨买臣窭十年牢落困胶庠五斗栖迟尚环堵唯余豪气自不除倾葢向人肝胆吐忽然结束事征鞍轩轩喜色浮眉宇谓余倚门亲久待况复枫宸拜明主朝廷公卿重荐延骞腾一鹗生毛羽高义如君世所稀早据要路思禆补
　　芝堂
　　度负土忘晨夕空肠未忍进一溢伯防隆冬不衣帛野火燎原天雨泽当时庐侧产灵芝盛事至今光史册王家兄弟皆英豪借借声华若连璧高文奥学乃余事一门躬行如万石孝思他日彻苍旻血泪垂頥洒枯栢谁云草木但有生始信幽明本无隔煌煌三秀发坤祥小大高低紫与白搢绅骇叹絶前闻老稚提携溢阡陌争言图写献修门坐见恩荣来络绎公独掩耳更悲啼吾恐货名于窀穸岂如采摘付儿童大胜籯金满千镒方今孝治轶唐虞八行求贤当侧席公虽韫匵畏人知定有新诗振奇迹
　　雪九首
　　都城十日雪庭户浩已盈呼儿试轻扫留伴小明天公亦戯人奇幻惊倐忽朝来檐霤间一一缀明月晚风扫连氛客被憎夜峭明朝赴看烛更把昆山照高堂连夜赏蛾眉映珠栊谁怜闭门客衣穿履亦空黑云埋日车厚雪镇坤轴行路勿匆匆包藏恐深谷忽惊陋巷间踏作琼瑶迹坐闻郢中歌妙句压元白春雷忽隠隠春雪复纷纷农夫戒及泽荷锸已成云撒盐乍可拟刻楮竟难成妍词覻天巧应复召羣生盈尺兆嘉祥三登贺时防我虽无寸田一饭犹足頼
