　　钦定四库全书　　　　　集部五
　　杨仲宏集　　　　　　　别集类四【元】
　　提要
　　【臣】等谨案杨仲宏集八巻元杨载撰载字仲宏浦城人后徙杭州初以布衣荐授翰林国史院编修官调海船万户府照磨防仁宗复行科举之制遂登延祐二年进士授饶州路同知浮梁州事终于宁国路总管府推官事迹具元史儒学传焦竑国史经籍志载杨载仲宏集四巻此本八巻不知何人所分元代诗人世推虞杨范掲史称其文章一以气为主而于诗尤有法度自其诗出一洗宋季之陋云云葢宋代诗派凡数变西昆伤于雕琢一变而为元祐之朴雅元祐伤于平易一变而为江西之生新南渡以后江西宗派盛极而衰江湖诸人欲变之而力不胜于是仄径旁行相率而为琐屑寒陋宋诗于是扫地矣载生于诗道壊之后穷极而变乃复其始风规雅赡雍雍有元祐之遗音史之所称固非溢美故清思不及范梈秀韵不及掲傒斯权竒飞动尤不及虞集而四家并称终无怍色葢以此也都穆南濠诗话曰杨孟载诗律精其追和李义山无题五首词意兼到真义山之勍敌瞿宗吉归田诗话曰杨仲宏以宗阳宫翫月诗得名然他作如风雨五更鸡乱呌江湖千里雁相呼挟书万里朝明主仗剑三年别故鄊间夜雨销银烛城上春雪压防旗空桑説法黄龙听贝叶繙经白马防沉雄典实先叔祖每称之长篇如古墙行梅梁歌亦皆为时所称夫人瞿氏余祖姑也尝以仲宏亲笔草稿数纸授予字画端谨而前后防窜防尽盖不苟作如是则载于是事亦以苦吟得之者矣陶宗仪辍耕录曰虞伯生先生杨仲宏先生同在京日杨先生每言伯生不能作诗虞先生载酒请问作诗之法杨先生酒既酣尽为倾倒虞先生遂超悟其理云云竟谓载诗在虞集上则非其实也乾隆四十二年五月恭校上

　　杨仲集原序
　　大德间余始得浦城杨君仲诗读之恨不识其为人及至京师与余定交商论雅道则未尝不与挺掌而说也皇庆初仲与余同为史官会时有纂述事每同舍下直已而犹相与囘翔畱署或至见月月尽继烛相语刻苦澹泊寒暑不易者唯余一二人耳故其后余以御史府用筦南宪架阁适海上仲复登乙卯进士第为浮梁别驾余迁江西仲亦改宣城理官相违十数年相距数千里迹虽如是而心固犹数晨夕也而仲竟未任宣城以卒乌乎恸哉余尝观于风骚以降汉魏下至六朝弊矣唐初陈子昻辈乘一时元气之会卓然起而振之开元大厯之音由是丕变至晚宋又极矣今天下同文而治平盛大之音称者絶少于斯际也方有望于仲也天又不年假之岂非命耶葢仲之天禀旷达气象宏朗开口论议直视千古每大众广席占纸命辞敖睨横放尽意所止众方拘拘已独坦坦众方纡余已独驰骏马之长坂而无畱行故当时好之者虽多而知之者絶少要一代之杰作也仲有子尚幼其残藁流落未有能为辑次者友人杜君伯原自武夷命仆曰将就其平生所得诗刻诸山中此诚知仲者而杜君猥谓罄仲海内之交相好又莫余若也俾为序之用掇其梗槩着于篇端致和元年六月一日临江范梈序










　　钦定四库全书
　　杨仲集巻一　　　　　　元　杨载　撰雪轩
　　北风海上来大雪何壮哉上下九万里洗净无纎埃君家十二楼轩窗洞然开吹笙击鸣鼓呼賔与衔杯名言落四座大笑声如雷举头望长空高兴惊冥鸿仙人五六辈飞下白云中粲粲明珠袍相从万玉童问君何所事未就丹鼎功翩然却携手共入蓬莱宫
　　次方韶父游九锁山韵
　　平生一枝笻畏踏市朝路出门见青山如鹘追狡兔兹晨正炎暑赤日那肯振衣得泠风超若忘世故冥冥九锁云未睹心已慕想当挟良朋尽取竒絶处旷哉天壤间寓物岂为痼归来思旧丘我服宁改素无官正尔佳有子亦堪据谁欤事虚名白首不知误
　　桶底图
　　巨鼇奋其首戴山出海中神人择所处共搆金银宫凭髙开户牖屈曲互相通女仙七十二顔色如花红一一执乐器奏曲殊未终世传桃源图共徴彭泽翁此本出异士雕刻尤精工天地极广大为地当不同我愿学仙道积久乃有功链形去滓秽五色浮虚空一朝乘风去浩然入无穷
　　次景逺学士立春日韵【四首】
　　大雪入歳其兆为有年漉漉始弥旬父老怨苦寒平旦上马去重裘忽如单况彼牛衣子束緼嘘防赖有固穷节犹念及古先
　　又
　　古来有道士声利不置怀顾于草木微而爱松若梅兹辰阅庭户适逢数花开芳意未渠央是足衔吾杯
　　又
　　人事重名实趋向尽百端丈夫负雄气动欲追古贤于意少不惬自放江海边登髙望八极云气生我前万事何足问所湏惟酒钱
　　又
　　﨑岖既山曲迢递亦水濵既见蓬苇士辄复忘为勤得之眉睫间我遇固防神所善但楼护岂暇从若人
　　送淳安徐敎谕之任
　　握手出南城眷眷从此辞苍茫寒江色日莫浮云滋大舟挟双舻轻若四马驰淳安众才彦竝迟得师资击鼓会高堂帷帐春风吹执经列座下质问无留疑平生五车书少出胸中竒歘然听佳誉庶用怿我思
　　赠季尊师
　　季君神人也防晦居山中还丹九转成顔色如儿童致虚感元气语黙与天通频歳伤水旱下民食不充曰雨须有雨曰风必有风上帝闻其贤行世多隂功防名紫虚府进位称仙公骑龙上天去此乐无终穷
　　太古雪为杨友宜赋
　　往时天大风能吹华山裂坠此一卷石或云太古雪粤自开辟初云气所凝结已经六万歳变化同嶻嶪望之色正白表里共澄澈尘土不能涴光采燿日月愿君勿爱此持以献金阙囘中避暑时持用消毒热
　　胡氏刺虎图
　　宛有烈妇人胡姓嫁刘氏从夫枣阳戍车宿水之澨夜半猛虎至衔夫入菑翳夫人自惊起不暇生为计直前执虎足尽力与牵制呼儿抽佩刀刃剚虎立毙夫命复不救复讐古所贵呜呼流俗弊人义日以废妇不失其夫臣子竞为利患难图苟免闻此得无媿吾故作此诗于以告万世
　　溪山积雪图
　　大雪归溪山长林昼冥窅□桥跨絶岸逺望行客少买鱼酤新酒独酌竟昬晓似此非他人高士戴安道偕虞伯生魏雄卿魏池燕集分韵得阁字
　　侯家有贵客置酒燕池阁溽暑方盛极风雨俄大作栘杨相簸荡蒲荷纷偃薄乱呌失蛙黾囘翔迷观鹤街流污秽尽窦涌波涛恶喧声剧降杀神气欣洗濯趣欢争酬酢献笑杂歌咢益弛劔佩严未厌觥筹错调燮维理得防漉宜施博和泽布原野湿烟散城郭嘉蔬已蒨蒨良苗常濯濯吾徒家食余豫有丰稔乐
　　题温日观墨葡萄
　　日观作葡萄电扫无踪迹谁于故纸上防缀辨墨笔西天十万里来往仅瞬息败□固不小遗下履一只
　　题金坛宋氏自怡轩
　　白云生何许萌芽自山石厥初仅毫末湏臾大充斥蔓延塞虚空日月为昬匿或时为霖雨庶类蒙润泽或随飘风散起灭竟无迹相縁千万变发色黑青赤畸人饬栋宇逺俗喜幽閴开轩纳高旷抚翫自怡怿人情慕富贵驱逐不暂息转盼失故常在已何所得上下数千载往事犹厯厯于焉服圣训无用苦沉溺昔年陶隠居常作帝王客挂冠神武门作诗写胸臆超然处物外可谓贞且白何以企若人鞠躬修道德
　　磵松为丁师善作
　　磵松髙百尺磊落多大节连根抱危石古色黑如铁隆冬方沍寒山中积霜雪萝茑悉枯老苍崖犹冻裂此时唯老榦峭直不摧折国匠求美材瞻望叹竒絶险阻不可致既作复中辍我愿驱六丁操斧隳嵑得此栋梁材持以献天阙
　　刘将军诗
　　交趾小蛮夷去国将万里土产无异物其人状如鬼湿热生瘴气疾者无不死天兵虽南征弃之良有以往年鄂州省绥靖失其理交驰赤白囊来告犯边鄙遣人觇虚实在廷孰可使矫矫刘将军一旦备行李有才兼文武不但善弧矢深入险恶地限敌才一水介者数百人视之若蝼蚁移文至其国诘问事终始指摘中利害文辞更深美报书礼甚恭敝邑何敢尔彊吏争怨隙搆煽乃为此贿赂却勿受足以振纲纪威声闻逺方一边祸为弭国家方全盛武备不可弛如此将帅才宜歌宿卫士
　　古木诗
　　大木生何许乃在泰山隈泰山崖石裂怒木响如雷浸润长膏液根株日以培纵厯千百载霜雪不能摧长安天子诏欲筑九层台台上搆宫殿青云共徘徊匠氏走海内博求栋梁材万夫治道路挥斧重林开大器当大用小器易剪裁有如豪侈士踪迹困尘埃适可厚自养毋为兴叹哀
　　东山诗为叶子实赋
　　东山髙几许上有浮空云叶君居其间好道夙所闻穷年絶人事矻坐攷遗文松关昼常揜石田朝自芸葆真去汨昬养息袭氤氲轩冕有遗防髙举谢世纷悠悠阻林莽长与麋鹿羣繇来巢许臣不事尧舜君
　　题恒斋
　　精金有恒质入火受百链君子有恒性宁逐外物变既不慕富贵又不厌贫贱素履往无咎夷险随所践张侯老世故此义颇深见勉之复勉之继之在无倦
　　次仲实见寄韵
　　平居忆故旧怀抱辄不佳兴来望浮云怅恨独举杯竒文孰能赏髙言孰能偕兹辰辱使至喜有意问来讽诵剧倾倒百过巻且开得知贤者心出处与作排山岳纵可移厥髙固崔嵬敝庐迩喧市卑湿苦病骸常念川路脩杖屦安得陪抚事意疎散毋论与俗乖犹恐战不胜常处涧谷隈斯言弗果践临风重徘徊
　　题文丞相书梅堂
　　大厦就倾覆难以一木支惟公抱忠义挺然出天姿死既得所处自顾乃不疑恻怆大江南名与日月垂我行见遗墨再拜堕涕洟名堂有深意亦唯歳寒枝可知平昔心慷慨非一时峩峩着栋宇昭昭示民知勿使风雨败永慰千古思
　　塞上曲
　　沙塞何窅窅树短百草长大河屈曲流不复辨四方驱车日将夕黑云隠长冈人马俱饥疲解鞍饮寒塘张坐逐平地击火烧乌羊桐酪过醇酎摇艳盈杯觞既醉謌呜呜顿蹋如惊狂月从天外来耿耿流素光悲风动寥廓拂面吹胡霜白鴈中夜飞参差自成行一箭落霜羽挟弓负豪彊中情无留滞千载能鹰扬
　　邑中南峰陈翁义庄
　　骨肉分枝叶由来同一身荣枯不相待此富或彼贫君子尚周急况复在亲亲昧兹损益理有腼独何人南峰自善士义勇超常伦燋心视末俗变化何由淳孰利佳田园壑舟固有神割之惠宗族为世成吾仁仰止文正公遗法实所循垂名倘不朽当与斯人均
　　偶题
　　丛生园中草蔚蔚媚幽姿乘时各敷荣皓色被緑枝似有微香发秋风辄吹之入人襟袖中亦复可怜兹诘朝步荒畦茎叶稍离披飘零即已矣霜露不尔私荣悴既有时小大何异而幸爱七尺躯天道不可诬
　　梦读退之诗颇竒诡已觉记其大防作此篇
　　自闻蓬莱山大林夹长峦上生瘿藤萝下生荆榛菅仰不见天日藏蓄雾雨寒仙圣常所处沮洳亦少干虽云夸凤凰出入伤羽翰哀音起空洞令人鼻准酸我欲诉上帝为施铁凿钻琢落石侧裂化为千丈磐琼楼与玉殿玲珑相通寛日月交户牖光射青琅玕而我从吾师来游乐且般呼吸养元气华腴生肺肝万劫永不死如循环无端窃为仙圣虑炳然心若丹上帝如许我致此亦何难
