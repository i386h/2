　　方舟集二十四卷　　　别集类三【宋】
　　臣等谨案方舟集二十四卷宋李石撰石有方舟易学已着録宋史不为石立传其集亦不见于艺文志惟书録解题载方舟集五十卷后集二十卷自明已来絶无传本今从永乐大典采掇编次犹可得十之六七考邓椿画继称其出主石室【案此指由太学博士黜为成都学官时】就学者如云闽越之士万里而来刻石题诸生名几千人蜀学之盛古今鲜俪李心传建炎以来朝野杂记称石在太学时适右学生芝草学官称贺石独以为兵兆由是坐斥赵雄其乡人骤贵石不与通书及石罢官值雄秉政遂不复起是石亦学问气节之士资川志又称其好学能属文少从苏符尚书游而集中亦有为苏峤所作苏文忠集御序跋知其文字渊源出于苏氏故其文以闳肆见长虽间失之险僻而大致自为古雅诸体诗纵横跌宕亦与眉山门径为近也谨以类排比编为诗五卷词一卷文十一卷又浙江采进遗书中有石所撰易十例畧互体例象统左氏卦例诗如例左氏君子例圣语例诗补遗诸篇皆题门人刘伯龙编而帙首一行乃标曰方舟先生集勘騐永乐大典所録经说诸篇与浙江本无异而其前冠以方舟集字亦与浙江本同盖本附入集中后全集散亡仅存此经说今仍别为六卷附之于后以还其旧焉乾隆四十六年九月恭校上

　　栀子赋
　　若夫栀子花者薝卜花也天之生物物各有时时各有职职主一时时主一色威福之柄如君如臣如主如客不可紊也榴花妥檐与鹤顶鬭鲜芙蓉濯水与蜀锦争魄此非天运之叙乎祝融用威朱鸟奋翼火轮曳空炎炎赫赫异哉诧乎一花纎微东皇刻一气浩大天皇织而乃与较瑞花虚名为六出乎纷纷披披雾霰争驰凛凛栗栗圭璧盈尺与阳俱升终嵗取食可以实百果可以瑞麰麦皎皎乎浩浩乎攘汾隂之琼仙倾夀阳之粉额谬老李之须鬓夺蟾桂之标格六气旋轸六月一息六而六之变化羲画凡物待之曰此天数也物各有得汝自怙权与天争力上帝照临汝不无责言毕梦一道士曰某即风霆之精云雨之泽也炎黄苗裔皆吾子孙爰自典谟既逺有熊是宅涨风雷之浩渺攀星虹于倐忽巫咸之九神姬旦之土圭所不能测曰戊曰已吾之降圣也继南离以述志植九土而立极西皥之杀物然洪波之滔天然吾之役使任为妃匹也佐禹治水我即黄魔翊汉帝储我即黄公传授兵略帏幄帝师赤松羽化者薝卜之为栀子乎栀子之为薝卜乎嵗晚后凋屈意樛阴有不改柯易叶者犹足以侣青青而嫓松栢也
　　四君子汤赋
　　药固多变味因以殊一出圣人之制四名君子之徒执徳而居岂但反隅之举分方而治足供调鼎之须原夫神农既已着经黄帝因而治已七十毒也我固知味四百病也谁适可使力调造化谓君者为羣善之君心在扶持如子者乃通称之子肩股具体齿龄可期乔松下兮负仙人之质百草魁兮荷国老之姿权衡自有轻重臭味何敢差池左右前后小人属餍于此元亨利贞天下咸乐从之且以子桑裹饭兮以不贫为忧顔回饮瓢兮以独乐为喜酥酪鴈行兮防起死之懿膏粱穉子兮有更生之美将军防劒自割肉以就烹宰相隔墙悟饮醇于妙理大抵气之运也或湿或燥人所感也有炎有凉得不按之以脉医之以方伏羲致无妄之喜宣尼有未达之尝最调和之得所纵瞑以何伤此治人之强弱况医国之存亡遂令筯下安危因吐哺而归汉鼎中强弱用滋味以干汤且以六官兮有国医六职兮为尚药由中及外为益不少自迩至逺其利甚博方剂之书若繁而简经济之用盖详而约防咀之诚既至刀圭之仙可作化人之国继周满之遨游圣夀无疆迈武王之安乐迹夫造化小儿固有百种膏肓童子岂无一薫常不死者此固有道羡久生者盍尊所闻以标本而为主以佐使之从君五福夀为先兹为皇极十全次为上永驭人羣异哉已然者治于未然有备者愈于无备藏室之抱一圣门之絶四大哉四君子汤乎此吾医国之本意
　　五言古诗
　　九里松
　　荷花湖上船松影沙外渡松行又九里凉藓引深步石桥枕溪斜中有入山路意行忘屈曲拄杖扣层户入门已欣然但恐所见阻更须上髙楼霁影防薄雾急风层巅起雾破山缺处青林一防明炯炯白鸟度独坐不须禅山水得妙悟
　　山图
　　丹砂山中期交棃海上约﨑岖九折坂准拟一丸药至今山仙翁已跨紫霄鹤鹤去当归来尘世忆如昨天使自天降招去游五岳洞府日月长流水花片落我来似天意一官不足缚道气迄未充何以痊瘴恶云霞閟灵蘂雾雨罩仙萼臭稗杂菁英凡草混荃若未免七十毒谁敢试口嚼上焉忧国病下者念民瘼安得如昔人一举翔寥落
　　同邓使君赏双头牡丹
　　兼金必贵双珏玉不独只嫣然彼丛生佳处亦求匹沙寒水成练雪洞山立壁春回小洛阳淑气自融液主人似维摩拈花许入室不碍清浄眼粉黛青赤聨香飒轻吹并影旭朝日有如猗兰心相同不相敌偶来饮此酒对花情有激试哦常棣章反经道如一
　　黎墨
　　少休大夫封应梦上公禄向来岩壑姿髙节嫓青竹低徊文绣地自谓迁进速迩者为客卿翰苑文史足炎炎虽可鄙炳炳有足録一朝污浊流钱劵满吾蜀【黎州嵗应副印钱引墨三千余斤】童山要进取万石压车轴通侯宠富民都尉侈搜粟虽防须臾荣不盖平生辱社栎乃天全不及不才木春秋戒求名吾诗纪山麓
　　王显之求名其二子曰忠厚曰忠恕
　　有子固可贺无子谁为羞尧圣传世念厉丑似已忧有子与无子人事天为谋王家乌衣巷叶叶追前修乃翁万卷书一一手校雠厚恕命尔名万金美田畴种徳日复日藨蓘期深秋字表名所出逺意孔与周陶公责端宣晋人警师仇君家膝上郎有如昔徐侯富贵二雏语岂有卑防休
　　西堂前老松状如偃盖或云即摩顶松也松产茯苓为人窃去松今不存因栽数小松补其处书此诗壁上
　　阿师从何来最是两肩苦一头载梵书一头荷慈母至今江上松东西随仰俯此松岂有情摩顶记佛语松根白衣僊日夜瑞光吐【或曰松产茯苓有瑞光见如菩萨像】翩然紫霄上蛰龙起雷雨我来悟前縁父老指环堵风花即几席经草随步武重栽渺生意合抱阶尺土他年十八公此地记初祖
　　木芙蓉
　　荣枯一气内造化本自公春花与秋蔕孰云有不同试防木芙蕖雨露滋芳丛嫣然生百巧秀色倾房栊婢子有不惬偃蹇霜枝红溪栁疎其西池荷亚其东等作一物防曽是烦春工开阖寓至理嵗事号秋风惟有桃李场感叹无言中
　　人日集蟇頥视堰
　　大江拆两股小江分一支秋水七八月老蟇喜持頥往往自夸大未觉海若嗤灌溉虽小恵湮溺已可危父老指水迹栖苴挂林枝髙田混坑谷漠漠沙风吹城堞不数步长堤漫委蛇我非河洛想颇为梁益悲神禹岂不到吾蜀几为池人日有故事敢后壶浆期野叟餽鱼兔鼔舞亦致词一笑谢父老何由救民饥但得麦早熟眼底无疮痍老守为尔醉灯火江之湄老蟇亦放光跳月天南垂
　　小桃
　　方春花气苏谁忍犯霜晓漫山千树桃缩颈聴春鸟独此开数株此桃也不小晨霞碎林霏绛雪压枝袅梅魂正零落月影夜悄悄陈新接物态俯仰映池沼蓬莱有大桃嵗月较多少自从偷儿得弱水隔浩渺且与倾緑尊乍见喜围绕尚有芥子微须弥寄毫杪
　　送张道士游西山至道观
　　雪打牕外竹共卧牕间床起来忍饥冻对坐唯炉香写之三尺琴绕屋声琅琅相从无此好要俶西山装西山接飞鸿路入云水乡六月余积雪冬木含春芳羣仙竞来往御风骑鳯凰我昔曽见之瞳子了且光张子长鬛师沆瀣入肺肠为取桥下履仍传枕中方此时觅刀圭念我双鬓苍
　　送浩侄成都学官
　　忆昔官博士所得英俊多斥去典蜀学蜀士烦搜罗二井转辘轳犹能挹余波袍子白纷纷有如镜重磨爱汝似二父此地曽经过分职有十师圣门严四科傥非一万卷难取三百禾我有十咏诗考古烦吟哦鼔钟乐髙文羽翼崇雄轲似闻礼殿栢久矣寻斧柯古物天为惜蒸薪鬼所呵堂堂公议地嵗月穷羲娥忍此恣横説后来敢谁何我集四库书琬琰藏洛河此外有石经参酌正舛讹熟读懋汝学师友相切磋汝有屋三间竹墅连松坡日夜望汝成门户髙嵯峩我贫自有道一竿钓渔蓑后生问老子守死山之阿
　　病中谢三山问讯
　　我昔择师友所至多朋俦道义自然合如以胶漆投一官既已得势利寻戈矛向者市道交日亲成日雠仅余山林士共结方外游一语不及俗自是无悔尤况乃生死事同参无异谋何暇及汝曹惴惴性命忧此间地步窄椰子包山丘三山妙僧杰道眼寛处求怜我本逐客饥冻妻孥羞有如呼云鹤拍拍青泥沟傥非香火縁讵肯相恤收阿璋栴檀林手挽银河流便欲洗俗辈欢喜解我愁阿兴两随我五湖盟白鸥对师思亡友中书烂银钩阿鉴殊可人出世依大侯禅学瑄演辈圭峯家法优人人蹵龙象一笑风马牛死鼠岂鳯食徒然吓鸺鹠行藏汝自用吾生去来休
　　大水寓武信二首
　　前时布谷鸣早种不入土翻然变隂沴两旬作淫雨半夜水囓堤一决不及御贫民无灯火下牀已没股晓登树杪呼出没见屋膂舟车或幸灾桂玉岂易煮天数苟有定民患谁适与我穷一席地此忧岂暇汝为汝愁天公闭户不欲语
　　民无穴居智不如彼狐狸奔流决户入尚懐囊槖资仓卒老穉计共此髙树栖茫茫无舟筏渺渺失陵陂号呼性命急千钧悬一丝引手幸免鱼十五为流尸古今民父母唯有恺悌诗吾诗本不作聊以箴厥堤
　　齐人画礼器
　　齐永明十年刺史刘惔画殿壁器服如三礼图席益模本于石经堂【惔或作悛或云刘悛所画之壁今亡矣】
　　漆器侈初俗长袖喧都城如何古防殿天开垂日星竒竒与怪怪惝怳不识名我尝阅此画肇自齐永明人间务备物未易穷丹青忆昨因郊丘盛礼严天庭器服各异数如此系大成仰瞻万乘圣逺想三代英虮虱容着足鸳鹭参结綪西归访江梅钧天梦初醒心焉感此画眼角欲零所见恐未伙蜀都问君平
　　送吴道明推官并问赵荘叔何公礼兄弟
　　送客涪水上我家珠水湄我适归未得此别那不思髙才府从事芙蕖映清池堂堂万卷书乐此簿领卑公退得绕膝戏防笑脱頥山路宿雨干篱落黄橘垂归鞍三百里处处吟酒旗赵子与南阳衡门各栖迟下马恐问及説与黄梅期尚念鸿鴈行冷落秋风吹
　　送黄安国同年【同行先上峡】
　　沙水夜仍白江月晓未收聨船共一岸撼枕闻开头五湖翻然归鳬鴈芦花秋低徊弄氷雪笑面防有由有如此江波大小同一浮纎花不自惜风卷随轻鸥谁能徇残啄五羖肥防牛我欲知行津上峡询耡耰
　　李漕和诗次韵
　　斯文百官富未易众目窥共此一鸟过谁其补三遗粗能述前辈抗説不敢卑一朝得印可宠若华衮辞神仙防金手为脱瓦砾姿微吟氷雪镂落墨星虹垂老成今几人縻以嵗月迟未用五字工防此两鬓丝圣贤无尽藏一一生物滋丐公且藏舌尚结他时知
