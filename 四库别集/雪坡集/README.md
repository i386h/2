　　雪坡文集　　　　　　　别集类三【宋】
　　提要
　　【臣】等谨按雪坡文集五十卷宋姚勉撰勉字述之一字成一髙安人宝祐元年以词赋擢等廷对万言策第一除校书郎兼太子舍人宋史无传是集艺文志亦失载此本为其从子龙起所编凡奏对牋策七巻讲义二巻赋一卷诗十一卷杂文二十九巻勉受业于乐雷发诗法颇有渊源虽微涉粗豪然落落有气文亦媕雅可观无宋末语录之俚语外间传本颇稀讹缺特甚今以永乐大典所载各为校补其永乐大典不及载者则仍其旧集首有文及翁序称其磊落有竒节官仅校黄本书备青宫寀又有方逢辰序亦称为瑞之奇士观其所上封事奏劄以及廷对诸篇论时政之谬辨宰相之奸皆侃侃不阿惟二十二巻载贺丞相贾秋壑一启题下注庚申五月十六日考宋史理宗开庆元年十二月贾似道奏鄂州围解景定元年正月诏奬贾似道功四月诏赴阙庚申即景定元年启盖作于是时与其攻丁大全封事若出两手殊为白璧微瑕然启末多进规之语犹曲终奏雅之意固视刘克庄之谀颂求荣差有间矣乾隆四十三年三月恭校上

　　雪坡集原序
　　寳祐元年嵗在癸丑上临轩赐进士第予与姚成一适相后先聮镳入期集所一见倾盖懽若生平握手论心知其慷慨有大志越眀年予逰清江碧嶂间距瑞阳三舎竹舆山行入境问俗知其倜傥有义气尔后渭北江东末由再晤四方传诵累防囊封愤世嫉邪排奸指佞又知磊磊落落有竒节夫以成一之志与气节奋乎百世上下而官仅校黄本书备青宫寀年仅四十有六遽修文白玉楼骑鲸白云乡去岂不可悲也夫不幸之幸成一有从子龙起得陛天子之学有一子元夫已受天子之命振家声而接文脉不在兹乎龙起彚编成一文藁五十巻予读之悲不自胜乌乎此持成一之文而已其志与气节固自有不恃生而存不随死而止者防龙川陈同父亦癸丑伦魁也尝伏阙三上书孝庙览之惊异俾执政召问当从何处下手晚得一第未及大用而殁又尝自作长短句四巻酒酣浩歌一章辄自叹曰平生经济之懐略已陈矣抑亦可悲也夫时东莱吕成公迁居金华同父数造焉成公深期之曰未可以为世不能用虎帅以聴谁敢犯子予于成公何能为役而所以深期成一与成公同抚遗编而欷歔悼吾党之零落敢以期成一者期其子若侄景定五年夏六月甲子古涪文及翁序
　　姚成一瑞之竒气也未可专以文章论也予癸丑夏自吴幙入馆时成一初第见其文如长江大河一防千里每与友朋相语必曰姚成一之文章不易及也越数年相防于馆中见其持身之介立论之壮负气之英且屡挫而不衰又知成一之所以为成一者同馆之士其知成一者莫如予而知予者亦莫如成一也未防咸以罪去越三年予起家承乏于瑞则成一已下世矣其族子龙起刋其平生所为文属予序予曰成一之操守议论气槩欲为天下国家兴事立业者也岂文章而已乎天夺之早而仅以文章传后惜也景定癸亥秋八月四日蛟方逄辰序
