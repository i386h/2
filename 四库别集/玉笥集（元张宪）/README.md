　　钦定四库全书　　　　集部五
　　玉笥集　　　　　　　别集类四【元】
　　提要
　　【臣】等谨案玉笥集十卷元张宪撰宪字思廉山隂人家玉笥山因以为号少负才不羁晩为张士诚所招署太尉府叅谋稍迁枢宻院都事元亡后变姓名寄食僧寺以殁明史文苑传附载陶宗仪传末然二人出处不同非气类也是集卷首有同时杨维桢周砥戴良及成化初安成刘釪四序又孙大雅玉笥生传一篇杨基玉笥生一篇后一篇其平生事状尚畧具其梗概宪早歳入元都所作红骝马歌酬海一沤诸篇皆在集中竒气郁勃颇有志于功名后从淮张之招非其本愿故其枕上感兴诗云拓疆良在念择木讵忘腼嘉猷固久抱忠愤欲谁展葢初同王粲之依刘晚类韦庄之仕蜀亦自知所托非人而贫贱衔恩不能自拔读其词可以知其志矣宪学诗干杨维桢维桢许其独能古乐府今集中乐府相操凡五卷皆颇得维桢之体其他感时懐古诸作类多磊落肮脏豪气坌涌诗末间附评语葢亦维桢所防定云乾隆四十三年二月恭校上

　　玉笥集原序
　　予尝于岳武穆王精忠录阅先辈张思亷所作诗歌其辞气抑扬感激所以为王自处而表其忠烈者曲尽无遗意其为人必慷慨瓌特以忠义自许非可以一诗人目之也惜未观其全集以究其蕴乃者常山丞前侍御黄君玉辉发其尊府佥宪嘿庵先生所藏思廉玉笥集一帙将绣梓以传托同年佥宪高公汝贤属予序因得遍阅之其咏史非徒详其事实且寓褒贬鉴戒之意其琴操乐府五言及近体诸作通畅俊爽森严壮丽如入羣玉之府琳琅琨瑶粲然照目如阅武库之兵戈矛殳防杂然陈列诚非苟作者于是益重其人焉盖思亷当元季扰攘之秋尝仕为帅府参谋与僚寀议不合罢去志不获伸才不克售伤时感物而泄其悲愤于诗者如是使其生当熙皥之世所以咏歌泰平之化当何如哉嗟乎圣人删诗三百篇其美其刺存乎劝戒下至楚骚比物兴言虽过乎怨其感人亦至矣汉魏而唐作者不一独杜子美之诗谓之诗史以其忠君爱国之诚恳也思亷其殆欲追逐古作也与考其所交若铁崖杨君伯雨张君辈皆一时能言之士其所得固有自哉夫诗运于精神心术之微勤一世以尽心而不得以诗鸣者多矣间有鸣者不幸不见收录于贤人君子往往湮没无闻者何可胜数独思亷之诗晦于前而显于今吾又幸其有遭乎佥宪先生昔以邃学宿德提学于吾江右造就士类甚众玉辉能世其业立朝佐邑皆有声兹又承其先志不没人善以嘉惠后学可谓济美者矣故为序之不辞思亷名宪会稽山隂人其居有玉笥山自号玉笥山人因以名其集云成化五年春三月旣望赐进士出身中宪大夫浙江等处提刑按察司副使奉勅提督学校安成刘釬序



　　钦定四库全书
　　玉笥集卷一　　　　　　　元　张宪　撰咏史
　　黄金台
　　黄金台高且坚三十六层凌紫烟上不贮妓女下不列管弦凤箫鸾曲匪求偶铜盘露掌非希仙高城不筑太师坞黄屋肯饰呉王椽但欲强兵富国来英贤上雪父母仇下洗黎庶寃黄金万斤光烛天曰邹曰剧相后先乐生徃兮重燕权燕鼎复兮齐鼎迁变雌冀兮为雄燕呜呼火牛触燕师衂乐生去兮燕土蹙黄金台荒狐伏昭王地下精灵哭
　　屠隠行
　　聂屠隐轵井里朱屠隠大梁市轵井匕首尺有咫白日杀人报知己梁市金椎四十斤捄赵功成耀青史两屠义勇勇不同金椎匕首分雌雄丈夫一死泰山重匕首胡为成小用丈夫一死鸿毛轻聂屠之死不若朱屠生
　　荆卿叹
　　白虹贯赤日易水生凄风燕人尽悲愤相送冀城中筑声何惨慽歌意哀无穷岂不念即徃立揕嬴政胷徘徊有所待匪畏秦帝雄自恨剑术疎未易了君事秣马膏吾车我友乆不至奈何遽相促苦未知人意丈夫重然诺防臂死不辞但怜樊将军九泉终见疑咄彼死灰儿曷足同等夷
　　安期生
　　吾闻安期生卖药阜亭东时人不识其始末见者但呼千嵗翁嗟嗟秦皇帝仗剑扫六合富贵亦已极犹欲事吐纳琅琊台上三月留三昼三夜语不休黄金白璧赐千万阜亭弃置无人收空留赤玉舄报以后防期蓬莱渺茫不知处童男丱女无归时鲍鱼风腥不忍嗅金棺玉匣万古埋僵尸悲无穷可奈何岂如安期者海上骑青骡黄金化丹砂赤枣大如瓜遨游宇宙间天地以为家
　　胜广
　　秦帝扫关东六王家业空典坟俱燬灭兵器尽销镕胜广起田畆一呼天下同虽然不成事亦足号英雄
　　秦鹿行
　　望夷宫中养秦鹿百二山河春草緑穿花尚作呦呦鸣宁识外人须尔肉李斯父子牵黄犬上蔡东门志何浅血汚云阳腰领红狡纵肥能几脔阉高貎软足心路称马献君君不悟羣臣相视莫敢非只恐出言丞相怒丞相怒秦祚移函谷不守秦鹿驰高材疾足争逐之项王叱咤起乌骓日千里逐之不得不肯止人疲马困乌江死沛公隠芒砀手剑三尺长网罗一举围咸阳扼其角刳其肠食肉寝皮传后王秦鹿死走狗烹后人不用悲韩彭帝王神器匪力争炎炎火德多洪福前有高皇后文叔回首平灵莽卓生汉业亦同蕉下鹿
　　沧海君
　　渐离筑中铅作水荆卿臂落图穷匕鲍鱼风起鸾车腥不待明年祖龙死赤帝子沧海君剑三尺椎百斤千金胡为轻报秦英雄不进圯桥履亦作秦皇刀下鬼
　　鸿门防
　　云成龙气成虎椎鼓撞钟宴真主披帷壮士髪指冠侧盾当筵请公舞白髪老臣心独苦玉玦三看君不语五星东井夜联珠天狗搀抢落如雨鸿沟咫尺接鸿门千里神骓一夜奔君不见龙泉影里重瞳瞽玉斗声中五体分
　　垓下歌
　　力拔山兮举世称雄頥指诸侯兮孰与君王一战不遂兮胡为自伤江东虽小兮胜负何常努力君王兮亟渡江贱妾请死兮先就剑芒
　　淮隂侯
　　勇畧震人主奇功葢天下持此求令终全身古来寡淮隂将之杰用智如炙輠不忍衣食恩甘为辕下马重兵在掌握茅土复求假不悟齐巫言终然三族赭跳梁驹覂驾覆锤金跃冶旣匪跋扈雄大权宜早舍忠逆无定见身败惨裂瓦所以赤松游浮荣真土苴
　　旦舂词
　　子为王母为囚赭我衣髠我头终日舂兮不死不得休东望邯郸兮漳水流使谁报汝兮终无由忆君王兮解安刘不能逺虑兮为妾谋彼四老兮君何求
　　朱虚侯行酒歌
　　长乐宫中女天子盛设賔筵懽戚里百官侍坐莫敢违诸吕諠阗笑声起御史中丞不纠仪叔孙制作成虚礼朱虚奉勅起行觞手提三尺昆吾钢田歌声振野鸡防颈血光寒汉道昌
　　悲吴王
　　悲吴王悲吴王后此五十年乱起东南方天下旣一家骨月母自伤胡为铸金币聚糇粮招亡命修甲兵几杖不复朝西京博局之恨竟不置反相之语奚独忘悲吴王白髪种种心何长旣不能入武关扼咸阳又不能据武库食敖仓淹纒梁楚真自败鏦死东越非天亡君不见下密老人首生角哀哉七王何不觉
　　广川王愁莫愁曲
　　愁莫愁死即休铅灌口刀灼眸托灰煮尸剑击头愁莫愁永巷春深作系囚愁莫愁老妪哭使者来穷诏狱徙上庸伏显戮雪我无辜人十六
　　卖卜翁
　　三月长安道雨过街树绿下有卖卜翁危坐营握粟举口讲仁义旋式推祸福不学贾大夫忧来方赋鵩
　　巫蛊使
　　禅襹歩揺冠曲裾纱縠衣伟哉燕赵士借问此为谁谒帝登大堂利口兴祸阶能令亲父子恩爱一朝乖血溅长安城尸横泉鸠里虽族佞臣家不益储君死望思思不归至今天下悲请听三老议儿罪只当笞
　　董君行
　　长安三月春如画卖珠儿骑公主马青年廿九桃花顔
　　挟弹垂鞭花影下蝉翼【阙】　　　　　　　帻红丝缰黄金百斤寿佳客彩段千端贻后房长门【阙】　行宫立董君贵宠倾堂邑掖庭拨赐买筵钱御驾亲临百官集煮豹屠麟考鼓钟宰人蔽膝如花红东厢未下膳夫拜【阙】　　传谒主翁主翁极尊荣上殿不呼名北宫夜鬭狗平乐昼调鹰入侍【阙】姑寝出从人主行生同欢乐死同茔君不见三斩罪名加弄子陛防先生饥欲死
　　霍将军
　　荧惑守【阙】星防鸮呌庭树车骑声慿【阙】　　来相捕赫赫奕【阙】豪君触【阙】王怒孱子不早诛悍妻尤可恶祸萌在骖乗恶贯起投附茂陵休上书徙薪非不悟何事麒麟楼犹图博陆侯
　　月支王头杯歌
　　朔风冻合诺眞水东方寒日曈昽起白马初腥径路刀一尺留犂搅金匕穹庐侧坐呼韩邪谷蠡抚掌贤王歌割牲饮血定约束酒器高擎天【阙】窝顶平额深如白玉月角日庭糟透骨刃痕截防伏犀根疑是晋阳刳智伯金莲凿落玻瓈杯银棱阖叶车渠魁葡萄未【阙】小马潼桃花先染真珠醅老上单于遗手泽谁问戎王包马革藁街不镂郅支头将谓汉家轻首馘陈汤血战未封侯张猛韩昌何足责
　　上元夫人词
　　七月七日夜王母降汉宫上元何夫人仪卫畧与同头作三角髻年可二十余身着青霜袍坐拥紫霞车九云夜光冠六山火玉佩敛袂登殿阶进向王母拜王母坐止之呼与共良防翩翩三青鸟飞来緑窻歇口衔七蟠桃甘脆胜氷雪刘郎非仙器内欲未防絶怀核欲种之开花待桃结一桃九千年徒尔縻歳月上仙哑然笑忽逐防云灭
　　托孤行
　　河间有异女其气上成云望气往求之乃得拳夫人夫人双握固见帝即披伸生子钩弋宫立名尧母门一朝忽被谴割爱非无恩子少母且壮乱阶人所闻君王顾大计牵去不容存拓跋踵遗范胡姬埋祸根司马受寄托弱儿终被吞乃知宣室画仅有霍将军
　　燕燕吟
　　燕燕在昭阳昭阳胜未央黄金涂地限白玉防椒房燕燕情无已专宠恩无比啄死汉皇孙占断张公子【名放富平侯】哀哀许美人戚戚曹宫使绿绨方防赫蹏书【赫蹏薄小纸也】天道好还须记取
　　折槛行
　　师傅不言事国柄归五侯贤哉槐里令抗疏谒宸旒愿请上方剑先断佞臣头狂直不畏死甘心地下游大呼攀殿槛壮气惊千秋御史将云去将军拜不休君王意亦解折槛不容修表表直臣节汉廷无与俦
　　东门行
　　东都门外今古稀东宫二傅同日归百官祖道设供帐勑赐黄金作酒赀归来日日防亲友尽卖赐金买醇酒白头刚傅【萧望之也】空劳劳一杯鸩羽不就狱愽得君王祠少牢
　　董舍人【董贤哀帝宠之】
　　莲箭铜壶催日晷舍人走报天顔喜日骖鸾驾共裀慿夜直龙牀同卧起双阙罘罳立冢茔洞口重殿营轩楹珠襦玉匣赐秘器锦阑绨槛涂丹青金剪落衣龙袖防黄肠刳柏御棺成合门贵宠专天眷内筵屡上麒麟殿元舅徒争城邑封天子已师尧舜禅古来丧邦由女色籍闳韩邓多男德司马何为位独尊曼柔便辟能倾国【王莽】白髪大师躬拜尘【孔光】病兑中郎坚避姻【中郎望之子萧咸也贤父恭尝为子求咸女咸避】臝尸未入圜扉里汉业潜移王巨君【巨君王贺女为元帝后】
　　圣女謡
　　阴为阳雄天道【阙】元城郭东君莫语绣衣御史活千人【王贺】火土相乘生圣女六百四十又五年汉家大业归齐田梦中谁报满轮月尤物祸人天使然【阙】　诸子多女德丙殿目成意相得画堂良夜太孙生兄弟五人俱食邑日月无光黄雾昏谏书终莫夺天恩兵权国柄递相袭四世母仪培祸根黄耉龙钟容色槁铜壁鬼符夸寿考黒貂徒守旧家风【太后令官属黒貂用汉腊】白玉已亏传国寳君不见持诏筹行藁棷析关逾墙走不休君如不信西王母试看门枢亦白头
