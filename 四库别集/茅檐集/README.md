　　茅檐集　　　　　　　别集类五【明】
　　提要
　　【臣】等谨案茅檐集八巻明魏学洢撰学洢字子敬嘉善人大中长子也大中忤阉被逮学洢微服变姓名匿定兴鹿善继家万计营救不可得柩归之后竟以毁卒世称忠臣孝子萃于一门事迹具明史大中传诸书所载亦大概相近然学洢尚有老母而为无益之死或颇疑其过中今观集中与潘茂庄书曰追比方始洢将就浙狱矣又辞里中父老书曰目今公差来捉且夕将死家门倾覆无可复言然则大中没后所谓坐受杨镐熊廷弼贿三千三百两者所司仍追呼于家学洢积忧积瘁于前积痛于后又重以阉党之威虐数者交廹乃无生理非真徒以一瞑不视蹈灭性之戒故学洢之孝在于大中被祸之日竭力殚心蹈危履险出万死以冀一生今诵其与人书至性恻怛足以感天地而动神而钱士升等作序惟欲以陨身殉父称之遂讳其追逮之事浅之乎知学洢矣其集一刋于钱棻棻大中门人也再刋于其弟学濓是为今本学濓颓其家声论者不能以大中之故曲为寛假以此益见学洢之不朽由所自立不以父防矣乾隆四十六年六月恭校上

　　摅懐赋【并序】
　　余頫览流俗百忧缤纷窃不自知其何心也间于记诵之暇紃察平生一篑方施规局略覩抚衷俞热托径俞寒余殆有私锢焉夫鑪冶既分资格斯剖狂者恒乐獧士鲜懽盖或区之非烝民能自为也昔幽通抽绪于北地思蔚起于南阳大都借葩艳于前人摅素懐之独往胷臆自吐须发可遗余窃防慕焉爰缀斯篇用鸣厥志虽非楚声亦姑自广云尔辞曰
　　嗟余懐之坱圠兮勔袭迹于前修神斾斾以应县兮恐厥疹之弗瘳蹇独行以亢回兮轶雰而彯輶纕酷烈为佩兮餐沆瀣为羞驾迅猋其淜滂兮曳氛旄其哹哹佽蜚为余犍芝兮造父为余持辀防我马毋契需兮登迥路之悠悠既晃朗以遐征兮下临睨乎时俗相樷井之諰諰兮曾不可乎偕服竞脕顔以蛊媚兮宵容与而昕伏集众垢于厉乡兮寄生涯于诋防党人既以为常径兮恬不恧夫龌龊余不忍觏此恒态兮挽青天而恸哭聊鼓荡以贡愤兮纷吾离此羣訽既詈余之眐眐兮庚若怜余之寡就出门閴无人兮入蹇产而伤独伴纗蓩以相饟兮孰胡绳之为淑宜苹蘅之节离兮被回飚之谌苏世而特立兮又何病乎块鞠忽中情之仓兄兮欸堦前之秋霜气沆砀而曾髙兮天防懔其不阳遒披离此芳蔼兮顇萚萧瑟而霣黄恫芙蓉之方茁兮曾不及其芬芳岂鷶之先鸣兮抑遌时之独后也羣生各有攸当兮何良辰之不又也诞流睇以懰栗兮志欿憾而难平伤庸夫之渨涹兮羣嗌喔而謍謍彼裮被之曼着兮余乃諓諓与相迎必骫靡以追曲兮宁处廓而无友生缅哲人之榘矱兮畴蠠没而飏声缔扈珵以勿替兮惟兹瑶瑾与琇莹莹緜邈其难亏兮瑾翕赩而可带荃既淫溶溶其靡岸兮逝将萉焉以蝉蜕漫湎湎弗裁兮惧一朝之狼狈独不相夫迷涂兮洵沕潏以滂沛徒褰裳而无航兮固余心之所害意蛩蛩其弗怿兮防儃佪而自怜不可讯之厚地兮不可谋之旻天事沉洄而好还兮何独长此迍邅遡愍风其蓬龙兮埃坲坲以湍防荃瞒瞒而上下兮胥滔汤而弗迁余将与此终古兮曷禁涕泗之涟涟昔鲍焦之自沈兮偕务光而不反聴燕市之悲歌兮白日惨焉其晼晚吊湘累于汨罗兮伤幽兰之九畹步兵值穷路以号哭兮岂好言之可挽彼焦先之枕露兮袭冷风之防防文诩之击几兮恋周道之阻修桃椎家乎世外兮伯伦葬乎醩丘若浔阳之兀敖兮永嬉嬉而自由悲者莫可与晤语兮岂乐者独有好仇疾溷浊而莫遯兮吾聊与之媮游虽歌与哭其异径兮余知其相对而长愁恨髙风之邈邈兮挟昊穹以俱浮余独丛此坎毒兮当涉世之末流安得霜翰之凌飞兮奄忽越此九州览九州之旷漭兮余惘惘焉谁届或趋余以返辔兮勿踽凉以自隘如蝉翼之孑孑兮畴頔步而可施众徾徾其唼佞兮相态以狡辞被竒服之摄叶兮衽旁挂而不可防辟之九戎而索赖兮贾用丧夫珍防既蒶蕴其霉黧兮秪槁悴而见欺超回轪其犹莫兮戎乃巩巩而自私荃苟不识予之妙婧兮曷羣喙而呰之更嫮饰以自暴兮长沉菀其何为绎行言之娓娓兮幸来者之可谏先民各缤纷其成章兮何独乐此谣讪聊洋洋而为客兮侣鸾鹥于雏鷰杂筐簏与鸡骇兮盛蘮蒘以琼玹偕陇亷以同升兮何闾娵之可唁伊至人其藏器兮翫时故之相嬗苟余鼎其勿抎兮孰沈防而可贱观周行之凌防兮窃又爱此春华芳菲菲其葰茂兮敖冰雪而扬葩睼乱流之浇钻兮志闷瞀而不嘉既纷员其波逝兮庐中垢而为家余将矫矫其独殿兮肯丧心而党邪禹功未施于臝国兮厥躬靡焉先自化上青天而无阶兮墯江流而骤下两骖释鞿羁以驰骋兮孰云纤离之可驾强綦蹊以利跂兮尚恐恐或未舎苟防羮而吹□兮眇不测其所防尝上观乎西山兮頫东陵与为隣展生介焉中处兮澹相羊以訚訚右髙歌之嘅嘅兮左鸠党而剚人汨淢靸余趋驾兮怵余波之相亲宁槁死而萎歼兮谁崄涂而可驯思和光之绪言兮诞错愕其惊神余既不克刓厥志兮又轁轲以多牾不速速而挤臧兮将譨譨而取殬怊茫茫其懭悢兮顾余帏焉瞿瞿懰竀俗以拟步兮俨鵕防之冲雾匪中心之墆翳兮瞥天而无路将穷岑嵓之嵚崟兮罙幽壑之谽谺既碕礒以狋兮复庨窌而聱牙纷頵砡其巃嵸兮攀窞而咤嗟夐睄窕以无人兮殆猿蜼之所家顝嵺廓其奋晋兮瞰坑谷且焉跱防闻猛夫之票举兮侣蟼螀而友豭腹岩之窏洝兮膺碅磳之怪沙岂展狎夫浍防兮谅坦夷于彼涂余既不难徦遰兮馺余陟乎高冈篾缕以为梯兮凌游雯以为杭路曲坲以倾昃兮石岳岳以相望仰重巘其崭岩兮睇故道而渺茫壁既峭而难驻兮力又竭而欲僵盼樛木其若頫兮徒连蜷而弗将勉陟巅以偃息兮恍魂存而魄亡世路既絶兮志遄归而不可抚长松以逈眺兮身飘飘其欲堕冰皑皑而凝霜兮云霏霏而在下鸟磔磔其惊飞兮雾蒙蒙而覆壄树蓊郁以茇骫兮刜羣芜而聊舎窃独乐此异壤兮将往游乎僊者忽凄风之骚屑兮逢豺狼之防防虎豹骇而怒号兮挺敦脄而锯龂争駓駓其相逐兮险沦磒夫厥身余窃覩此咆哮兮殊无异乎今人中俇俇其怫郁兮邅吾道以长迈下峭陁之嶕兮弄惊涛之滂湃鼔荪桡与桂櫂兮馚馡馡此香馤搴白苹于江臯兮剪杜若而为带波顄淡以汨活兮鸿防涌而扬磕纷防蟺其相纠兮兰舟荡而濊濊既永去此流俗兮载骋观乎四荒泝昆仑之淫翳兮测尾闾之坱茫抨阳侯为余先导兮御冯夷与忀徉云澒溶而无文兮天澎濞而欲狂龙卭脟圏以瀺穴兮忽苹萦砀突而益将怪鲛为之屏息兮长鲸窅其固藏怒彪休以喧豗兮意骄骋而不可降奔湍拍其相防兮愁大块之不长曼余目以游防兮窥液之所积淼浮岚之出没兮疑风雨之窟宅齐喷沬与颉颃兮挟飞澜而漰砉水浏浏以驶兮石峉峉其俞瘠此方意别有天地兮余永辞乎局蹐纚余檝以息征兮对芦华而为客枕顽石之岖崯兮聴潮声之傍媮江山之浩荡兮忘大寓之迫迮倚秋风以长啸兮闻木叶之萧萧黝云涌而上屯兮神惨惨而弗骄懐美人于天际兮隔千里之波涛鸟栖栖其慕类兮兽踽踽而寡曹日悠扬以就暮兮送去濑之滔滔牉独羇此穷域兮伤魂梦之徧劳余将释结絓以宴娭兮又曷长困乎水臯量八裔之迢递兮絙流峙而靡控鹪明怯夫修罠兮应龙廪乎衡緵慙蜚鸿之茇茇兮戾防以遐羾纷纬繣其弗可治兮长悇憛而增痛憎母之勃屑兮疾鸱鸮之憙哢祗居居其不合兮冀弋获焉靡中泛曾波之溶溢兮穿魁堆之幽洞词纚纚而永日兮觉乃悟其为梦神瞀乱以驰骛兮庸织路而遘穷行不程古人之茂则兮将伥伥其焉从勿登高山兮勿涉流水之浺瀜余初有成説兮展紃紃而可崇曰孰董义而苦特兮畴佩仁而苦同蒸蒙之昧明兮曾纤繋乎厥躬鳯皇翀于蓬陼兮蔑鸠寄乎苕丛爰翺翔其各适兮觉区宇之蒙蒙眡彼党之回遹兮工喔咿与儒儿防独修此好顔兮蹇瘏瘁而忸怩余生耦彼佌佌兮黾练要而自覗苟株绳墨以弗跌兮孰黕防而污之览稠木之飃芳兮亦各膺夫茂时无贱桃华之秾郁兮亦无妒乔松之后衰薫风不可数被兮孰枳涂而长治宾时俗其过苛兮恶在葛天之可居聊以媮适兮勿怦怦焉踌躇剪防防之蒿径兮葺硁硁之故庐登逴逴之药房兮望蠡蠡之芷闾及余年之屈桥兮滋莞防以扶疎防竢时而充佩兮垂长褏之虚徐仰余陞皇之赫戏兮俛余御夫防魖彼謰謱之为虿兮余偕行而借防于埃垢兮庇景于华胥忧永路之坦坦兮不愁嶰壑之沮洳愿正防而孟晋兮长无负此居诸乱曰流光迅徂汨前驱兮律崛遄征勿踟蹰兮十琢九置丧良防兮目长足短成凡夫兮髙薄秋旻阚闉阇兮哫訾防斯羣鵱鷜兮众不可户诛兮刻自掫兮神洒洒兮将与天俱兮
　　离思赋【并序】
　　乙卯春季家大人北之文安时序如流忽尔觱烈离思耿耿爰造斯篇情而不文亦聊以自摅也辞曰
　　忆春阳之恢台兮擥垂杨之青青将余亲于江臯兮忳余行其竛竮数辟咡余頫诏兮逝将去此而遐征各其弗怿兮中仓兄而难平陟巉巗以周览兮覩大江之滔滔道曼曼其靡极兮忾磳之怒涛恍齐鲁之在望兮迷防防之颾颾蓱岛汨其浮没兮冲风动夫布袍意徘徊而罢眺兮步徙倚乎彼勉商畧江山之竒胜兮志奄奄其不豪载宿宿以俶装兮归者先解维椉潮水之迅驶兮抚中心而不夷愿及行之未逺兮更壹见而致辞计淹留其终别兮强自割而遂驰道晋陵之殷辚兮逾姑苏之漫烂出新诗于轻箑兮唫数章而永叹涕霪霪其若霰兮神溶溶而无畔冯惆怅以独返兮癙忧集而悺悺既伤儿之懐亲兮重伤亲之念儿戒归櫂令遄发兮还三复而眡之匪无则言以自壮兮写中曲乎两眉孰有恨而非别兮孰有别而不思佯婆娑其媮乐兮各不言而相知羌萎约以自废兮怜风日之凄其既驾桧檝兮临迥道之三千归期勿可道兮行逺緜乎嵗年肇发棹于吴防兮辖将弛乎幽燕遘女魃之儇舞兮路迍邅而不前桃李摽落以遒尽兮榴火歕而复然顷艾虎其萦懐兮又闻江南之采莲踆乌不可与俪偕兮艸木芸芸其屡迁伤征途之踯躅兮忽涕泗之流防石圻沸焦兮望钱塘而驻斾蹈赤坂之赫威兮蹇傍徨以赖防欻悐悐其若撼兮懐余亲而长嘅去衡泌之肃泠兮服顽沙之飏荃不堪此蕴隆兮矧羁旅之狼狈臆怫鬰以缭悷兮恒假寐而独悕惚忻忻其承顔兮或髣髴乎既归盱余目而莫觌兮羌厌浥其渍帏每戃慌以竟日兮留梦境而弗违雁翩翩南翺翔兮寄儿以新稾憎世路之防曀兮鸣吾鼎之为寳鳣鲸裁于蝼蚁兮虎豹吼而当道鳯凰羾乎天门兮览九苞其犹未保挟蓬矢无肃心兮徒伤谋生之艸艸儿愿及年之方将兮懐夜光而粥之薫椒桂与木兰兮辑羽翠之陆离苟善贾其可兮将良弓以为箕出门遘夫文穷兮羣众兆而见嗤送裘马于五陵兮什敝貂而自私天实搆此西山之局兮讵贲获之可为忳愁约以侘傺兮志逴逴其俞亢块独处此穷檐兮倚斜阳而四望感严威之觱发兮俄飚涌以跳逷女机杼而士唫兮幸家人之无恙日坎坎其育鞠兮独有行乎四方俱昕夕而匪所睼兮知永懐夫故乡跂游云以揽涕兮霑余袂其浪浪乱曰人生孰无别离兮恃来者之可期防不诵夫甫田兮曷怛怛其相思
