　　钦定四库全书　　　　　集部四
　　提要
　　崧庵集六巻　　　　　别集类三【宋】
　　【臣】等谨案崧庵集六巻宋李处权撰处权宋史无传其集诸家亦不著录惟方囘瀛奎律髓内录所作送二十兄还镇江诗一首而注其后云处权字巽伯洛阳人邯郸公淑之后有崧庵集宣和间与陈叔易朱希真以诗名南渡后尝领三衢云云其履贯畧可攷证而亦不言其距李淑为防世今其集传本已絶独永乐大典中有之且寻检各韵处权自序及其从弟处全原序邵骥原防一一具在所纪生平踪迹差为详备惟世系仍未明晰今
　　案建康志有李处全小传称处全淑之曾孙本丰县人后迁溧阳官至朝请大夫而王明清挥麈余话亦称大理少卿李传正为淑孙即处全之父据此则处权实淑曾孙而家扵溧阳瀛奎律髓所称洛阳当是刻本传讹以溧为洛耳淑家富典籍其邯郸图书志晁公武每引以为据又精研声律所作诗苑类格今尚散见诸书中处权承其世学更能标新领异别出以清隽之思扵诗道颇为深造处全序称其齿益髙心益苦句法益老与少作不类是其覃思吟咏老而愈工虽原帙散佚东京与南渡以后所作互相糅杂不复能以年岁辨析而综其大概五言清脱浏亮畧似张耒七言爽健伉浪可拟陈与义在当时实一作手久经湮没幸而复存亦论宋诗者所宜甄录也谨采掇排比以体区别厘为六巻仍以原序防分系前后俾将来有以考见焉乾隆四十六年六月恭校上

　　原序
　　少陵句法出审言豫章句法出亚夫有唐若我宋诗人莫可居二老先盖其祖考源流所从来逺世父莱阳府君少承家学尤刻苦于诗建中靖国间送张芸叟帅中山三十韵有孝治追羊枣嘉言问冻棃之句浮休抚几嗟赏以为发语浑厚托意防婉可为后学模范自是名重京洛为郎拜州汩汩旧辙晚节益不偶至于废放流离度陇蜀入涪万逮还故官北归下瞿塘灔滪泛大江泝淮汴多厯竒崄益昌其诗崧庵兄世父仲子也人物髙秀山立玉举世父爱之早付以衣鉢在政宣时与陈叔易朱希真皆以诗名中更兵火奔迸遇物托兴亦未尝一日废绍兴甲子而后齿益髙家益贫用心益苦句法益老始与少作不类然暮年颇艰出故不多见噫诗果能穷人抑穷而后工耶审如是则诗固人所忌也绍兴己巳余与兄合并于武林者逾月将行饮饯浙江上临分哽怆不忍别且道家世斯文之属后六年兄殁于荆州盖不获见矣诸子流落湖外遗编断藁多为喜吟咏者持去心常慊焉陈君安民惠伯调官中都因过余溧阳暇日语长沙佳士有刘氏伯仲庐陵人自其先世喜与贤士大夫逰兄建炎末避地客其家其调防甚至旧所藏诗文颇多或者乆假而不归每恨不获锓版与吾人共之因裒辑少日所记与十数年来摭拾得之于亲旧间者四百余篇寄惠伯使归诸刘氏以满其好事乐善之意兄尝自叙作诗本末独幸偶存较之今所亡则甚伙而先后亦失次矣异时复有得焉当为别集处全行年五十而学益落虽呕出肺肝望古人地位终不能到一灯弗焰愧负识别慨然雪涕书所欲言不文其辞也淳熈六年嵗在己亥八月望日弟处全谨题




　　自序
　　处权未冠始学诗于先君一日谓处权曰三百篇诗之祖也六义存焉至六月序篇卒之以小雅尽废则四夷交侵中国防矣诗之道岂小补哉小子识之处权再拜服膺而退既冠归洛则又学诗于乡先生段处厚约亦一日谓处权曰昌黎公作荐士诗序诗之髙下得失至矣尽矣不可以加矣学者当于此求外此非吾所知也处权敬佩书绅于今不忘处权固陋既不敢坠先君之训又不敢负段丈之教虽羇旅鞍马疾病呻吟拳拳之思终在是也昔人所谓目瞑知深搜吻动觉苦思与夫钩章棘句月锻季链者自谓有焉五十年间作古赋五古诗三百律诗一千二百杂文二百长短句一百平生之力尽于此矣不自知其不可也异时阳翟陈叔易恬尝语孙正夫端曰恬归自岷峩行六千里复见中州知旧其气不衰文益进者李巽伯而已汪内相藻彦章程紫防俱致道皆极力推称口之不置故一时名流胜士翕然相与衢守张巨山嵲尝遗处权书云近朱希真报巽伯寄示梦归古赋文采不减两汉字画逺追晋宋当今第一流也公有此客顾不佳哉处权繇是稍得自信而其志益确不知老之已至身世皆寓于其中而未始语人亦难与世俗道也暇日拾掇次第粗成编缀名之曰崧庵集后日有能敦义爱半夜走缄书俾作者序而传之耶或谓尚白用覆酱瓿耶则不敢知也绍兴二十有四年嵗在癸酉二月旦日崧庵惰夫李处权序








　　钦定四库全书
　　崧庵集巻一　　　　　　宋　李处权　撰赋
　　乐郊赋【并引】
　　春雨既霁放歩于野见飞者鸣者哺于巢者防泳于渊者戯走于陆者与夫行歌而醉于路者凡生物之遂植者盖俯仰之间无不欣欣而乐也于是感焉予亦忘其老之将至身世之所遭也而况不陨获于贫贱既行乎其素矣则何往而不自得邪作乐郊赋其辞曰
　　感余生之几何兮冒霜雪以盈颠邈吾庐之安在兮寄幽梦于涧驶嵗月之不留兮梗行路之且邅起临风之浩叹兮嗟欲往以无缘徒搔首以延伫兮复顾影而自怜眷于越之上游兮适寓止之所便税予驾于萧寺兮占南向之明轩敞函丈之几席兮纳万顷之风烟属幽人之晨兴兮畅晞发而潄泉徐拂尘而启户兮蔼春物以相鲜纷宿云之解駮兮散朝阳于一川挺嘉木以敷隂兮嘤其鸣而始迁念人生之行乐兮宜及时而勉旃饫笋蕨于既饱兮破月团而自煎幅巾藜杖适野在田或狎鸥而借草或观鱼而临渊或流憇以骋望或防心以忘言于时逺山兮连连近水兮溅溅青秧兮芊芊白露兮娟娟麦初芒兮燕乳桑已葚兮蚕眠喜膏润之及私兮庶遂逢于有年田父野老醉着道邉扶擕不去堕帻頽然盖不减拾遗之杜曲右丞之辋川也予亦兴尽而返欣此地偏披襟脱帽断简絶编樽陶情而有酒兮琴寓意以无弦惟受性之拙疎兮乏趋时之周旋委穷达于造物兮问行藏于彼天凛嵗寒之孤操兮固久久而弥坚拟登髙之能赋兮托余思以终篇方俯仰而自得兮尚无媿于昔贤
　　梦归赋【并引】
　　予洛人也既更衰乱奔窜潜伏烟尘阻絶身世相吊遑遑乎羇旅之忧忽忽乎嵗月之久也迟暮良苦一夕梦归故里顾经于目而歴于耳者憯不胜乎其悲也作梦归赋虽文不足采而援古知今卒反乎正庶不谬述作之旨云其辞曰
　　梦故国之圮城兮陟曲隅之欹亭俯伊洛之蜿蜒兮仰嵩少之峥嵘痛一炬之焦土兮彻云汉以宵赪吊瓦砾之塞路兮失厦屋之连甍蓊草木之丛灌兮骇鸟兽之悲鸣访吾庐之无处兮嘅遗址之已平动心目之惋伤兮惨神沮而骨惊究此都之宏达兮匪郡国之可配也挺龙门以防其面兮伏邙阜之峙其背也亘飞梁于波上兮矫双阙于云外也鸣天鸡之一声兮非烟拥乎冠盖也思当年之行乐兮信承平之嘉防也想涧瀍之始卜兮奠九鼎而初定也宅取夫天地之中兮气得乎隂阳之正也奉巍峩以再迁兮转岌嶪之连乗也揭神行而百灵防兮极方怒之万牛并也世三十而年七百兮姬公导其庆也谅轻重之在徳兮胡楚子之可问也皇哉我朝兮遹徳事之追前圣也伟恢复之鸿业兮眷上帝之所命也陋肃而不算兮抗宣光之与并也席成基以永固兮膺寳运之方竞也观兹鼎之复出兮奄旧宅而果盛也
　　梅花赋
　　予衰病索居经嵗不出芳物过眼真成一梦偶作梅花小赋以寄所懐其辞曰
　　伊造物之发生兮无一木之不卉顾四时之不齐兮曜白黄与丹紫懿江梅之秀出兮俨亭亭而絶比既婵娟以暎岫兮复窈窕以临水类忘言之贞士兮肖独洁之君子许兰茝之仅似兮睨桃李之可鄙占嘉月与好风兮泛幽香而未已寄美人之一枝兮想横斜于镜里抗翠袖于天寒兮抚修竹而孤倚怅衰翁之揜闗兮抱防疴之在体起三嗅而三叹兮魂忽荡而心欲死
　　悼亡赋【并引】
　　予几七十而鳏倡无和也问无应也视人世之苦无甚于兹者一日抑泪茹悲作悼亡赋虽不足以増伉俪之重庶使览者知予不敢于其厚者薄也其辞曰
　　噫予虽老而鳏兮坐成窘独之叟眷伊人之具美兮视同列而未有追维琴瑟之义兮谓相从于白首忽一日而不见兮惨四顾而无偶始来嫔于我家兮倐五十之改嵗惟内政之克修兮躬苹蘩于时祀施慈恵于仆妾兮靡失色于娣姒既甘苦而食淡兮实于君而有愧何沉疴之不解兮岂厄防之偶值楚切切而在牀兮亘日夜以相继愀皮骨之仅存兮情恻恻而不忍视讫命尽而数絶兮奄悠然而长逝痛割裂于心膂兮陨滂沱之涕泗譬令孽妇之久生兮曷若假夫君以永世信尤物之易毁兮审竒花之早落抱雌如吴门之剑兮返魂无齐人之药顾晨夕之傍偟兮类失侣之孤鹤揽明月之入牕兮惊风动乎帘幙淡星河之欲晓兮清露零于丛薄悄空闺之岑寂兮想音容于冥漠矧早衰之多难兮决非久于世者叹幽明之异路兮复何时而见也哀断弦之莫续兮怅贝书之不复把懐平生之好合兮竟缱绻而难舍觊魂梦之可接兮睇长松于广野虽涸流以濡翰兮浩予悲之莫写
　　五言古诗
　　谢士特饷樱桃
　　今年客闽中春山亦花鸟时果次第熟风露枝袅袅含桃品居最朱实若自表髙人肯见分厚味未觉少绛囊敢鬭荔木蜜虚名枣多食顔自驻此语闻本草翩翩出谷莺啄取定先饱故园当此时行树映池沼散火耀隅隈灼目看未了翳叶色逾深累颗核更小园人挈筠笼银甲摘清晓匀圆入雕俎拟借麻姑爪择佳间留玩引杖日数绕烟尘坐流落魄断首空矫
　　赠刘至道
　　性静乃可琴徳常乃可医二事人所难而子能兼之好古身则润活人功不赀未媿孙思邈何慙钟子期俗眼多重轻世态工瑕疵自视无所嫌安行复何疑子有贤舅氏闻名四方驰勉矣归问津沛然有余师
　　赠彭表臣才臣
　　我交半天下颇接金石友相逢便论心已别更回首三年厄兵火水陆东南走风餐近波浪露宿薄林薮庐陵信名邦昔贤犹见取水南遇彭子门巷似无帚胷中有泾渭如镜不受垢呼儿具杯盘物物皆可口新青撷嘉蔬大白酾旨酒使我忘身世不复问升斗茫茫四海内二子可与久我昔当少年摆落谢纷纠老来龃龉事在十常八九尚倚笔锋劲恃此或不朽飘零欲何归故国生藜莠无乃造物意穷途一假手丹枫作初寒客袂不掩肘冥冥甲子雨山逺似无有试问夜归舟网中得鱼否
　　同吕倅小饮
　　一日不作诗已觉神不王一日不饮酒更苦舌本彊梁王自堕马贾傅恨无状寥寥千载后有懐意谁谅夫君出世家伟一魁三相人物方辈出事业雅相尚笑我守丘壑踪迹更漫浪尚余杯中物足以助豪放相逢倾五斗老子未多让
　　送嘉仲兄赴永康宰
　　我兄如琼林皎皎尘滓外不从门地初便许人物最裁诗陶谢流射防晁董辈欬唾落人间一一珠无纇羣居款名理澡雪赖箴诲坐我琅玕林清风长十倍谁令困小官寸进而尺退前年宰荥泽与古能吏配兹入山水国牛刀钦益淬穏借一帆风开涛上鲸背独怜藐然孤厌逐儿童队梦想在东南浮云寄心防定不俟及归奉明光对犹当念竒穷寂寞将把耒
　　右司公致书显上人并遗楮衾诗以赞之并简都公
