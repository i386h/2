　　渔墅类稿八巻　　　　别集类三【宋】
　　【臣】等谨案渔墅类稿八巻宋陈元晋撰元晋宋史无传惟江西通志载其字明父崇仁人登嘉定四年进士初授雩都尉迁知福州融州累官邕管安抚使尝建渔墅书院因以名集然考赵汸东山存稿有虞集行状称集之祖觧组过临川寓公陈元晋之夫人为其女弟因迎以归则元晋亦蜀人侨居崇仁通志尚考之未详也焦竑经籍志载有元晋渔墅类稿十巻诸家悉不着録今检永乐大典中尚存杂文八十余首各体诗一百一十余首谨以类编缉厘为八巻江西志称元晋嗜学好义为徳于乡人者甚多厯官所至俱着政绩今观集中如乞差甲首催科劄子则极论当时赋役之上曽知院书则力陈上流防江之策且谓天下非事功难立之为忧而人心不睦之可畏又谓邉遽戒严则号召郡国不教之卒坐糜粟于长江以南谓之防江警报逺则防遣解弛又复置之度外自开禧以来同一痼病其于南宋废弛聚讼之象指陈痛切可谓深中膏肓又上魏了翁启有云善类之势不振付之乍佞乍贤正论之脉仅存听其自鸣自息以奔趋为防径以软熟为圆机习成脂韦病入骨髓皆愤世嫉俗之言则知其生平必以伉直不谐于时者读其遗文犹可以见其人也乾隆四十六年九月恭校上

　　渔墅类稿巻一　　　　　宋　陈元晋　撰劄子
　　乞差甲首催科劄子
　　臣窃观江西差役之中产之家以戸长破家者相踵葢里正号为大役而易戸长号为小役而难名大而实易则以盗贼烟火之事不常有名小而实难则以催科督廹之扰顷刻不得免也易者任之大家难者任之中产甚者狭乡之中有不赢百钱之税而应充者有不满两三年而再及者其人丁少则耕殖荒于犇命之勤其气势轻则追呼不能以令豪强之族展限代输费用亡艺比较笞棰怨嗟流闻葢执役既毕其资产亦与之俱罄矣不急捄之更十数年氷消火燎岂复有可全之下戸哉臣愚尝思所以捄之之术惟曩者浙东提举袁说友已行之策有当推行于江之西者其法攅聚一乡之税数上者二十戸下者三十戸随输送之相等居止之附近以类相从各为一籍分为甲首责以拘催若其本名合输之数如限纳足则从甲而乙递相授受至籍尾而止其承籍之家惟恐舎去之不亟则必以时输纳故期限易了而胥吏无所诛求赋租易集而官府免于比较无争役之讼无执役之扰官民两便未有易此臣愚欲望睿慈降付都省参详行下江西提举司照臣所请施行若其委蛇曲折窒民奸求民便者俾之从长剂量第向来浙东不无小则以逃亡死絶产去税存之戸无所追索官司不为从实销豁一例籍之人不免代输胥吏常以此为邀赂之地合委监司检察若有此等许令越诉官吏并以违制论惩一足以戒百矣是法茍行一道中产之民得以生长如天之仁实非小惠
　　乞换雩都武尉劄子
　　县之有尉与令等故初授而铨量终更而批书视丞簿为加详葢一县狱讼之大者争讼则有定验盗贼则有追捕杀伤则有检证皆倚辨于尉惟是赣之诸邑讼事尤繁故尉之职尤不容以轻授其间十县有二员并置文尉者赣县瑞金防昌是也有一文一武者宁都兴国是也独员文尉者石城信丰龙南安逺是也雩都号壮县民顽地广争讼盗贼杀伤之事视他邑数倍乃独置一尉又以注右选之士近攷数任来尉雩都者多出于军班之内与武人之门荫徃徃于逺方民情利病全不谙晓一切受成吏手谨书纸尾而已甚者不识姓字托书判于他人唯知恃麄豪以虐使其下肆掊扰以求厌其欲邑之滞讼不得而决逋冦不得而获寃狱不得而直徃徃繇此欲乞睿慈将雩都县尉窠缺检照守臣费所请抚州乐安县体例改作员缺注授选人却于见任县尉某人改本等差遣庶几邑尉得一文臣恊赞邑事狱讼不至失平实为荒逺之幸
　　代奏事劄子
　　臣一违轩陛两玷麾符积媿素餐蔑闻报効防恩奏事再觐清光日夜以思惟当罄竭千一之愚庶几补报涓埃之末敢冒昧以所闻为陛下言之臣闻人心者事功之地也故天下非事功难立之为忧而人心不睦之为可畏茍其睦也虽艰难多事之际而无小无大气势聨合自无支吾不及之患否则虽处安平无事之际而或前或却意向乖张已有龃龉难安之形周盛时有昆夷玁狁之侵而恃以为攘却之本者今天保以上之诗是也夷攷其诗乃皆咏写其君臣和乐之情而曽不及兵戎守备之事葢情意之浃洽血脉之流通则始之图揆终之酬酢内外上下如手足心膂之运不相期而相应顾何事之不可为哉故晋不失备而加以睦识者知其能以国霸夫差善用其民子西不以为患而惟患二三子之不相睦天下犹一家也亦安有一家之内人自为谋心各有主彼此之不相能而能御隣里之窥觎杜盗贼之侵凌者耶夫议论一于上而后下之所视以趋向者定心力恊于内而后外之所以捍御者坚窃迹近事任事论事意见病于背驰谋国忧国言论几于乖剌主战者难合于和而在边者勿谐于庭是未可以言睦也趋向何由而定哉制阃与饷司之意异而进退观望之莫决列将视中权之令轻而号召调发之不行前邀后击之不相援而城自为守大军民兵之两相忌而人自为战是不可以言睦也捍御何由而坚哉夫人心不能以两用不用之公则用之私始于议论之不一而终于党与之角立始于心力之不恊而终于衅隙之横生则如师旷所谓不心竞而力争私欲侈而公议废矣虽陛下竭宵衣旰食之勤是自为焦劳而已大臣励鞠躬尽瘁之操是自为勉强而已非所以服奸雄之心也非所以耸遐荒之聴也事功悠悠亦复奚怪急解而更张之犹有及乎臣愚区区窃谓谋论异同翕受无间而折衷以是非之公则议论不患其不一托重阃寄情实必孚而警厉以赏罚之公则心力不患其不协使上下之心一主于公求以相济而不求以相胜庶几可以言睦而应敌之方可得而议矣家国一体也公私一理也未有国之不安而能有其家者也亦未有公之不济而能保其私者也吕范言于孙防曰今日之事譬如涉海一事不劳俱受其败此所以为范计非但将军也今之充满于班行奔走于边圉者谁独无吕范之心哉亦在陛下与二三大臣意气之所感动而已是岂以不睦为患哉惟陛下畱意
　　代奏劄
　　臣闻先朝范镇之言曰欲备契丹莫若寛河东河北之民备灵夏莫若寛关陜之民备云南莫若寛湖岭川峡之民备四海莫若寛天下之民镇之此言葢至和间所奏请也是时中国之势全安而无隙西北之国帖息而听命而镇首以寛民力为言葢防养元气于四支强壮之日是则忠臣医国之至计也矧夫外之酬应方艰而内之败缺不掩使非有以养其气力之浑厚结其心志之忠爱其何以杜窥觎而销间隙支缓急而应事变哉人之常情逐其利而不暇计其害故贪利于邉陲者多失备于奸宄葢卢循之事是已有所泄则有所忘故备重于西北者多兆变于东南葢黄巢之事是已故圣人当多事之际非能事事而逆为之备也亦恃其有以得斯民之心而已非能人人而有所予也亦惟有以寛斯民之力而已其力既纾则其心有所系矣其心有所系则其所以为备者至矣自昔竭民之力失民之心其蠧有二曰盗臣曰聚敛之臣盗臣志于奉已聚敛之臣志于奉国奉国者非奉已之比然为根本之害则一仰惟陛下仁心发于天性仁闻孚于寰海视匹夫之失所如一已之隠忧戎兵跳梁兵以应动臣知圣心葢有甚不得已焉者收辑流散抚摩疮痍恩诏丁宁徳意劬密寛三边之民为三边之备者亦既闗圣虑矣然臣观内而郡邑之间嵗事荐丰糓价甚贱而田野焦然愁恨叹息犹闻于耳大义感动人宜奋发而羽檄一闻骈首聚议若旦夕蹙之于无聊赖者此其故何耶安平无事取极锱铢军兴旁縁更求趣办诸州桩积稍归于上有所支遣必以不足病其民和籴频年民无余粟一或缓急其何恃以为之备开奸人告讦之门而科罚错出幸富家罣误之失而占籍株连弓兵络绎于郊原计臣恣横于井邑求为辨事无复惠心甚至取苗斛概量之赢覈田畆隠占之数常赋足而复输号曰备助军须名曰和买其实科抑或以奉国而取能名或以奉已而厌私欲借使博施济众之心日勤于上而下之奉令承教者不满人意如此欲天下之寛难矣夫斯民尊君亲上之诚常生于养生无憾之际使其廹蹙不自安则无聊之心生矣后茧丝而先保障则沈灶产鼃而民无叛志不然间架陌钱之除奸人将得以借口海内乂安固未必遽有是事识者虑微要不可谓无是理使镇生此时其不为之寒心耶臣愚欲望陛下濬发徳音明戒郡县继自今日凡旁縁之苛敛与虐取之巧名一切掊克之说禁勿复行又申严赃吏之法俾无得借公以济其私庶几仁意流行人心固结导迎善气消遏乱原皆自此始实非小补惟陛下裁察幸甚

　　贺皇帝表
　　天意右仁克相慈闱之夀圣心广孝于昭寳册之徽朝野欢呼神人闿怿恭惟皇帝陛下家邦勤俭宫庙肃雝尽人伦之常迈防防之虞舜全天性之乐配翼翼之周文钦承长乐之懽祗奉昕庭之册起敬起爱美欲报以奚殚得夀得名庆用垂于罔极臣心存恋阙迹阻班庭扬伟绩而铺闳休拟请封于东岱绥眉夀而介繁祉知永燕于西昆
　　贺皇太后表二首
　　位正中闱诞敷显册尊居慈极昭嗣徽音凡囿载持惟时欢忭恭惟皇太后陛下宅心渊静秉徳齐庄大明行黄道之天功髙佑辟内治本青蒲之地忧切进贤嘉相阀之钟龢俪宸居而作配贻谋有永受福无疆臣欣覩鸿仪莫陪虎拜介以繁祉但扬夀母之休祝之多男愿衍文孙之庆
　　六月徂征诞扬我武三城遄复不失故疆喜溢两宫欢腾四表恭惟皇太后陛下功齐太极徳并坤元怡怡愉愉子孝极亲欢之尽雝雝肃肃家齐卜天下之平奏此肤公福于王母某隃驱牡辔阻簉鸳行颂天子之万年欣际平淮之运奉太后于亿载愿赓长乐之诗

　　长乐頥神有开燕喜昕庭晋册増衍鸿名八表均欢六宫溢喜恭惟皇太后陛下补天功大配地徳深物仰广生显仁心之持载年登希有侈富履之康宁方续迓于湛祺爰推崇于熈号奉觞称庆龙章聨防翟之辉镂玉传徽虹气粲盘螭之饰诞扬盛典允属明时臣逖阻观风莫陪展寀既多受祉敢扬文母之休于万斯年请上尧封之祝
　　贺皇帝册后表
　　风自火出示端本于家人月并日明表俪尊于宸极宗祧有庆防县均欢窃稽古之盛时有思齐之具美任氏姒氏全母慈妇顺之规周南召南成家齐国治之效掩五三而独出幸千一之亲逢恭惟皇帝陛下盛徳巍巍小心翼翼问安侍膳躬行曽闵之难养志承顔思得涂莘之助恪遵慈训式嗣徽音播显册于明廷下孚民聴奉夀巵于长乐上格亲欢绵延百世之休根本四方之化臣服勤驰隰阻迹班庭刑于妻而御家邦愿宣美教厘尔士而从孙子拟颂鸿禧
　　贺皇后受册表
　　位正坤仪有衍长秋之庆庭扬涣号聿新诸夏之瞻朝野均欢宫闱底豫恭惟皇后殿下祥钟沙麓徳迈河洲夙夜相成绰着贤妃之誉寤寐思服允符淑女之求知家道之攸宜底亲心之克顺式涓谷旦丕称椒涂祏室祎防相礼聿严于宗祀寝门笄总承顔永燕于慈闱某防被光华遭逢休庆风天下而正夫妇但宣化于关睢言后妃之多子孙请读诗于螽羽
