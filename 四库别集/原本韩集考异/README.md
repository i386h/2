　　钦定四库全书　　　　　集部二
　　原本韩集考异　　　　　别集类一【唐】
　　提要
　　【臣】等谨案原本韩集考异十巻宋朱子撰其书因韩集诸本互有异同方崧卿所作举正虽称防校众本弃短取长实则惟以馆阁本为主多所依违牵就即南山有高树诗之婆婆弄毛衣傅安道所举为笑端者亦不敢明言其谬是以覆加考订勒为十巻凡方本之合者存之其不合者一一详为辨证其体例本如陆德明经典释文但摘正文一二字大书而所考夹注于下于全集之外别行至宋末王伯大始取而散附句下别刋以行以其易于省览故流布至今不复知有朱子之原本其间讹脱窜乱颇失本来此本出自李光地家乃从朱子门人张洽所校旧本翻雕最为精善第一巻之末有洽补注一条称陪杜侍御游湘西两寺诗长沙千里平句千里当作十里言亲至岳麓寺见之方氏及朱子皆未知又第四巻末洽补注一条辨原性一篇唐人实作性原引杨倞荀子注所载全篇证方氏举正不悮朱子偶未及考又第七巻末有洽补注一条辨曹成王碑中搏力勾卒之义皆今本所未载其字为徐用锡所校防画不茍光地没后其板阙佚故传本颇少此本犹当日之初印毫无刓缺尤可贵也乾隆四十四年十一月恭校

　　钦定四库全书
　　原本韩集考异巻一
　　宋　朱熹　撰
　　此集今世本多不同惟近嵗南安军所刋方氏校定本号为精善别有举正十巻论其所以去取之意又他本之所无也然其去取多以祥符杭本嘉祐蜀本及李谢所据馆阁本为定而尤尊馆阁本虽有谬误往往曲从他本虽善亦弃不录至于举正则又例多而词寡览者或颇不能晓知故今辄因其书更为校定悉考众本之同异而一以文势义理及他书之可证验者决之茍是矣则虽民间近出小本不敢违有所未安则虽官本古本石本不敢信又各详着其所以然者以为考异十巻庶几去取之未善者览者得以参伍而笔削焉
　　集序【方云序只云目为昌黎先生集诸本亦多无文字者今从之　后凡从方氏者不复论所不同者乃着之】门人李汉编【方云蜀本作朝议郎行尚书屯田员外郎史馆修撰上柱国赐绯鱼袋李汉编今本或有并序二字非是】生于【或无于字】来归【来或作求非是】皆捜【或无皆字】【方云音郭璞江赋云防滉囦是也或作沄非是】钧鸣【鸣方从杭蜀本作发　今按二字两通但作鸣则句响而宇稳耳故今定从诸本而特着方本所从以备参考后皆放此】武事【事阁本作士非是】不常【常方从杭本作赏云取汉书功盖天下者不赏之语　今按不赏乃蒯彻教韩信背叛之语而唐太宗亦尝自言武德末年实有功高不赏之惧施之于此既不相似且非臣子所宜言者李亦未必敢取以为用也当从诸本为正】陇西【或无此二字】失坠【方云左传国语多用失坠字或作坠失或无失字者皆非】七百【或作七百一十六或作七百三十八方氏考其数皆不合而姑从阁本杭本以为唐本旧如此既非文义所系今亦不能深考也】
　　第一巻
　　赋　古诗【宋莒公云冯章靖亲校旧毎巻首具列巻中篇目冯悉以朱墨灭杀之惟存其都凡集外别有目录一巻今按李汉所作序云总七百首并目录合四十一巻则正与冯合】
　　感二鸟赋　十一年【一或作五方云以诸谱考之作一为是】东西行者【阁杭本无行字方云考之礼记及公送温造序当有】于道【于杭本作之非是】百十【十或作千方云此专为选举而言也贞元九年应宏词者仅三十二人作十为是】偕进退【退上或再有偕字】名荐书【方从阁本名上有列字名下有于字　今按嘉祐杭本与谢本并无此二字语简而意巳足方本非是】以羽【以下或有其字】光耀如此【此下诸本有可以人而不如鸟乎一句方从阁本文粹删去　今按诸本所有之句乃全用大学传中语而意则异矣二本无之岂公晚觉其陋而自削之欤抑后之传者为贤者讳而删之也方从二本意则厚矣然凡读书者但当据其本文实事考评得失以自警戒乃为有益正不必曲为隠讳以启文过饰非之习也今此一句恐或公所自刋故且从方本云】　吾何归乎【此句或在茍有食其从之下】茍有【茍或作敬非是】羌永【羌或作差　今按作差固谬然羌乃发语之词施之句内似亦未安以上□反顾流涕之语推之则西路乃长安之路而此字当为浸渐愈益之意不知的是何字又恐或是逾字也】黄流【或作流黄】之何嘉【之或作其嘉或作憙非是】焉是【焉或作而】恒十年而【恒或作亘而或作以方云恒居邓切与亘竟之亘同班固叙传恒以年嵗选诗徙倚恒漏穷】策名于【策方从阁杭本作荣云公上宰相书非茍没于利荣于名也与此义通　今按唐人防字俗体从竹从宋亦有只从草者与荣字絶相近故阁本作荣盖传写之误耳方引荣于名亦与此语意不相似于或作与亦非是】昔殷之高宗【或云此句上当有在字或是念字由上句末有愚字相似而脱也】为之先信天同【先下或有容字信或作容皆非是方云汉郦食其传沛公吾所愿从游莫为我先韩语祖此】户说【说或作晓】惟得之而不能【惟方作虽　今按上文之意若曰天之生我必有所用何不力慕古人如傅说之徒而独怊怅于无位邪惟或者茍得其位而不能追配古人□如二鸟之空被荣宠乃是神之所戏耳故幸年嵗之未晚而庶几无慕于斯类也斯类盖并指二鸟与彼得之而不能者而言也惟字正是斡转处作虽即无力矣能或作孤亦非是】
　　复志赋　无解【解或作辞】孤嫠【谢作嫠孤】将就食于江之南【方从阁本无将字于下有大字或又无之字今详文势皆非是】非古训为【方从阁本古作诂为作焉非是】阅时【阅杭本作阙非是】身之【之或作而又或作之者皆非是】故吾【吾下或有志字非是】亦既【既或作冀】可迳【迳或作径】羌众【羌或作差非是】所驰【所或作四】附势【或作射利或只附字作射字】骋骛【骋或作驰方云楚词朝骋骛乎江皋作驰非是】慨余【慨或作嗟】之交【之或作而】大龟【大或作火】而愉【而或作之】谁无施而有获【方从阁本谁作惟下又有德字云李本谓陈无巳去德字今本复讹惟为谁其误甚矣　今按此句本用楚词孰无施而有报孰不殖而有获之语词意既有自来又与上下文势相应故嘉祐杭本与诸本多如此乃是韩公本文相传巳久非陈以意定也阁本之谬如此而方信之反以善本为误今不得而不辩也又嘉祐杭本世多有之而其不同处方皆不录岂其偶未见耶抑忽之而不观也】既劳【方无既字非是】不脩【脩方作循云唐人书脩近循楚词亦有误者　今按唐人书字之误方说是也然此乃脩误作循非循误作脩也脩犹脩好脩怨之脩盖因旧增新之意耳】之茫【之或作而】抱关之阨陋兮有肆志之扬扬【扬扬诸本多作阳阳方云后汉孔嵩传云晨门肆志于抱关史记晏婴传意气扬扬公当是用此语诗君子阳阳则注谓无所用其心非公所用字也】斯自讼【斯或作聊讼或作诵】
　　闵已赋　而则【而或作兮】独闵闵其曷巳【闵闵或作闷闷诸本句下有兮字下文隔句皆有兮字　今按洪庆善云欧宋皆无兮字后皆复添或云咸通本复志赋与此赋皆无兮字未知孰是】饮食【阁本无食字非是】何不【何下或有苦字】巳逺【逺或作还方云诗小雅焦赣易林逺字皆叶平声读作还者非】余壹不知【壹本多作岂方云阁杭本作壹旧监本潮本尚作一讹壹为岂自蜀本也檀弓云予壹不知夫丧之踊也韩语盖原此】本宜【本或作所】一违【违阁本作衰非是】人兮【诸本多无兮字】
