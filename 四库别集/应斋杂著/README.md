　　应斋襍着六巻　　　　别集类三【宋】
　　臣等谨案应斋襍着六巻宋赵善括撰善括宋史无传惟宗室世系表载有此名为太宗第四子商王元份六代孙考集中有赵运干墓志铭称同姓兄可大艺祖七世孙与宗室表世次相合盖即其人然表亦不详其爵里今案集中迓黄枢宻知隆兴府启有一防为氓应脩桑梓之敬语则籍隆兴祭汉阳赵守文有仙源同盟雁塔俱第语则尝登进士得常熟宰谢政府启有奋身科第千载之遇语则释褐为常熟令贺赵枢宻启有赘员别驾叨禄京畿语又有长沙倅署磬沼诗则由县令倅郡与湖北张安抚启有叨把一麾深惭共理语又有鄂州上殿三劄子则由郡倅知鄂州谢宰执启有方怀沟壑之虞遽増幕府之数语谢湖南帅启有四十年低徊州县三万里奔走尘埃语则任满闲居复充幕职又有次计幕诸友韵诗谢岳漕启则终于岳州漕佐也是集宋志不载其原本巻帙不可考今以永乐大典所载裒为六巻宋人奏议多浮文妨要动至万言往往晦蚀其本意善括所上诸劄率简洁切当得论事之要如陈纷更之纠赏罚之失皆深中时而永乐中修厯代名臣奏议乃不载其一字未明何故诗词多与洪迈章甫唱和而与辛弃疾酬赠尤多其词气骏迈亦复相似观其金陵有感诗有谢安王导亦可罪至今遂使南北分句其不满于湖山歌舞文恬武嬉意趣盖与弃疾等固宜其相契也乾隆四十六年二月恭校上

　　应斋杂著巻一　　　　　宋　赵善括　撰奏议
　　建明奏议
　　臣闻古之人臣建功立业者皆有一定之谋其始为之甚难故其终成之常易如汉之韩信蜀之诸葛亮皆以功业许其君而亲以身成之言出于口终其身而不变者其谋一定也近世则不然轻言而遽改忽行而骤废始虑不审终之汗漫而无所就不可不察也仰惟陛下焦劳忧勤十七年于兹然而大功大业尚闗圣虑以臣策之意者今之人臣所以事君者未至于古耶请畧举数端言之如两淮安抚旧为两司往者以议臣之策为一司矣继又分而为二荆鄂都统旧为两军往者以议臣之策为一军矣继又分而为二如发运司则骤兴而骤废如宣抚司则再置而再罢如市舶司则既并之复置之如铸钱司则既分之复合之如福建聴民鬻盐乆矣忽夺而归之于官继又委之于民如广南就邕州市马乆矣忽改而置之于南诏继又归之于邕凡此皆所谓轻言而遽改忽行而骤废者也陛下以虚受人闻善则从如汉高皇之刻印销印以明圣人之无我可谓至矣然议臣之策纷纷易变臣窃谓古之人臣以功业许其君者殆不如此臣愚欲望圣慈特降睿防应自今人臣有所建明其事大体重者并令两省杂议某以为可某以为否中书籍记而上之陛下徐观其事之成败取其议之可否以定其言之是非而大明黜陟之法用是以察天下之真材助成圣治实建功立业之本也取进止
　　严赏罚奏议
　　臣闻赏当功则臣下劝罚当罪则奸邪沮赏罚公于上事功立于下此必然之理也陛下即位以来勤综核之政行必信之法十有六年海内乂安百度时叙圣徳休盛非臣愚所能胜识然求之事功终未见卓然有以大慰天下之望其故何哉岂非劝沮之道或有未尽耶臣愚不揆昧死妄论其故陛下试少垂聴焉自昔士大夫出而为世用上之人有以激厉其委靡之气而崇奬其功名之心者舍赏罚莫行焉然赏罚之行必有次第传曰罚不阿近举不遗逺此明君驭世之至权也是故赏先行于下则在上者咸知劝罚先用于近则居逺者无不服比年以来臣所闻见窃有疑焉何者一命而仕州县以知効官惟其位卑其势逺故建议立言也大吏輙以为强聒拾掇细微文致以罪者有之矣兴利除害也豪民以不利于己増辞饰非讼陷以罪者有之矣大吏按于上豪民讼于下追召囚锢靡辱不至棰楚之下何求不得往往先从废罢继坐赃私终身不得齿于仕版其间有材术素优绩効昭著上乏提撕旁无介绍虽有尺寸之赏而文吏吹毛求疵陆沉下僚赍志以没者盖不可胜数也若夫官崇势近身贵名显外持麾节内列班荣虽纵欲以害民怙恃而贪利罪恶昭著无从上逹偶然风闻名挂白简亦不过镌一秩罢一任而已曽不逾年旋得美官其间又有巧为谀说公肆诞谩聴其言如布锦摛绣责其効如繋风捕影动以虚文霑被醲赏陛下夤用刑赏章别公罪岂容谬戾如此臣敢妄论其万一者冯衍有言冨贵易为善贫贱难为工官尊位重则营助者众法治之诚难官小力轻人得而挤之矣由是观之赏或醲于贵近罚或偏于贱逺劝沮疑有未尽非职此之故欤臣愚欲望圣慈特降睿防宣谕执政大臣继自今官有崇卑赏无厚薄势有逺近刑无重轻加惠孤寒微贱之质勿庇冨贵权豪之罪如是行之而士气不振事功不着未之有也狂斐之言冒黩天威罪当万死惟陛下裁察
　　宗室补试厘务奏议
　　臣仰惟陛下即位以来亲睦宗族备尽教飬作成之道遂有名冠多士学中词科持节把麾政事举者不一而足虽祖宗之时亦所未有可谓盛矣然宗学有官从学者盖鲜武臣苟禄效职者无闻可不参稽旧法而为之制乎书传所载国子谓王之子与夫公卿大夫之子也今国子监止试公卿之子而宗子独不得预焉宗室武臣近世咸知自爱遵守礼法出官之后例得添差不得干预职事虽有材术无路可显臣愚欲望圣慈特降睿防应宗室仕于朝为厘务官者其子弟并曽得解人并许赴国子监试如有升补依宗学舍法施行宗室武臣添差州县差遣不厘务之人许令知州监司精加选择如委有材力可备任使保明备申朝廷许令厘务如有功过依庶官罪赏施行庶几文武全材卓然表着各殚智力共赴事功非特宗子之幸抑所以上合祖宗立法之本意
　　差役奏议
　　臣窃见比年士大夫多好轻议旧法随即变更往往不能无弊试以差役一事言之限田之法出于祖宗之旧其制乆矣比年有减半之法盖所以抑强扶弱也又复为品官身后再减半之法其弊致使士大夫之后不得义居其利害曲折有不可不详辨而熟究者谓如一品官限田原格一百顷今减半为五十顷子孙又减半为二十五顷其有子孙多者通不得过二十五顷之数其法非不善也殊不思一品官之后岂无荫补三五人京官朝官仕宦者乎以七品官一员计之合破二十顷以五员计之则百顷矣又如三品官限田元格八十顷今减半为四十顷子孙又减半为二十顷其有子孙多者通不过二十顷之数其法非不善也殊不思三品之后岂无荫补三四选人京官仕宦者乎以八品一员计之合得十顷以五员计之则五十顷矣彼为子孙者义居而通以父祖之官立戸则共占限田之数少析居而各以自已之官之立户则通占限田之数多占数少者不免差役占数多者皆得免差是使士大夫之后皆不得义居也此岂所以厚风俗乎盖尝以此法考之史传之间惟秦商鞅立法家有两男而不分异者倍其赋儒者之所耻言也以堂堂之治世而使士大夫之后析居者免役义居者差役不几于阴用鞅之法乎臣愚欲望圣慈时降睿防应自今官戸用父祖官品立户者其限田皆仍旧数至用曽高立戸者依减半法庶几不至析居以窥避差役此亦朝廷厚风俗之本意也
　　免临安丁役奏议
　　臣窃见临安府车驾驻跸五十余年列壁连营建宫赐第卜筑之地本皆民产一从分占乆免推排其间物力合科色役与夫夏秋二税无可催理者有司失于申请未曽蠲免钱塘仁和两县嵗有定额无繇销豁遂将和买役钱逐科増添抑配乡村贫下人户谓如绍兴二十一年已前物力及三十四贯四百文方敷和买绢一疋物力及三十贯已上毎一贯方敷役钱一十九文今来物力止及一十九贯五百文即敷和买绢一疋物力止十贯已上毎一贯即敷役钱二十五文其数相去不翅倍蓰夏税秋苗名数虽存人戸无有乡司保长逐嵗陪偿若不巧取于民何所从出况临安之民亲依日月之光近濡雨露之泽其被恩优渥宜若非他郡比也而产去税存科配不均之患岂应有之臣愚欲望圣慈特降睿防行下临安府将见今应有营寨官府宫寺所占地段并拨赐田产合纳夏秋二税并和买物钱覈实保明委无溢数欲断自宸尽行除豁所有两县后来増添抑配乡村之数并合蠲减庶使畿甸千里咸戴上恩各安其业不胜幸甚
　　乞免临安府丁钱
　　两淛税丁之重至有生子不举长不褁头者丁谓为相苏秀获免胡则在朝衢婺遂蠲天子驻跸临安五方之人萃聚繁衍法当嘉恵而优恤之已降圣防自淳熙四年为始权免三年去冬限满圣意勤隠不忍举催再下展年之令徳至渥也而九县之民犹若闵闵然有望于上者惧日后再有起催之患尔六年蠲免一旦复行诚亦难矣圣恩寛大国用无阙何惜毫末以慰畿甸之心是宜断自宸特下明诏将临安管下诸县丁钱自今后永与蠲免更不催理将来恢复中原銮舆还京亦使此邦之民子子孙孙感戴上恩无有穷极不胜幸甚
　　公荐举奏议
　　臣窃谓时俗诞谩士心迎合其来久矣以汉宣帝责实之政冝其天下无复懐虚诈之心者而王成伪増戸口至八万余乃首防劳来之赏以唐太宗英武之资冝其天下无敢献谀悦之言者而封伦乃谓圣武戡难太宗不怒不能正诡随之罪由是观之人君端居九重坐制万方真伪胶扰贤否混淆将使革诞谩为实效变迎合为忠言盖亦难矣陛下励精圗治宵旰防劳寤寐贤才勤恤民隠真尧舜之用心汉唐之君盖不足道迩者申明严令濬发徳音俾荐举人材必先实迹陈献利害务在可行诚当今之急务也愚臣不佞窃谓陛下责实以求士而荐举者未必皆实效虚己以兼聼而奏对者未必皆嘉言辜负陛下求治之意臣不胜愤懑今夫州县之官抗尘走俗服役下僚不求闻逹而当路大臣其有特逹见知奏名于上者几何人哉莫不引吭声哀摇尾乞怜短轴长牋恃亲挟贵铺张其履歴夸大其艺能撰造事端妄陈政绩亷耻已防荐举方行所谓实迹果安在哉献言之臣忧国爱君不顾得失力辨是非酌古治之本原拂人主之胜心其有一言悟合事功立成者几何人哉莫不观时之为窥上所好视其文如布锦摛绣责其效如捕风系影以至毛举细故道聼陈言所觊投契上心侥幸身计人材碌碌议论平平冝乎陛下当馈而兴叹也臣愚欲望圣慈特降睿防应今后荐举之士必先博询素行精察治功委有材力方合荐不得私受启书自陈实迹稍有虚妄并坐谬举之罪委获实材乞加举贤之赏奏对之臣必先分别是非指摘可否民间之疾苦朝政之阙遗咸得尽言有犯无隠不得窥徇时俗迎合上心其文切中时病乞旌褒嘉之赏如此则责实之令行求言之路广诈伪少息忠谠备闻共济事功以图恢复臣不胜至愿
