　　提要

　　臣等谨案幔亭诗集十五巻明徐熥撰熥字惟和闽县人万厯戊子举人熥负才淹蹇肆力诗章圭臬唐人而不为决裂饾饤之习巻首有张献翼序称其调匪偏长体必兼擅力返古则尽涤时趋所以推许者甚至又谢肇淛谓其才情声调足以伯仲髙季迪微憾古体不及朱彛尊静志居诗话亦谓其七言絶原本王江宁多情至语详阅是集固非尽出标榜当明季诗道冗杂之时亦可谓蝉蜕秽浊者矣闽中诗人自林鸿王蒙诸人以后即推郑继之为冠熥生平喜称继之而卒年仅三十九与继之正同亦一异也集为其弟■〈火勃〉所镌■〈火勃〉字兴公亦以淹贯见称所撰有笔精榕阴新检诸书俱别着録云

　　●幔亭集原序

　　诗岂易言哉求其成一家言良不易易况备诸体称大家乎盖工近体者或弱扵古风长七言者或短扵五字即王孟二氏盛唐名家一以清婉称近体一以风骨雄古风且各擅所长况其它乎予友徐惟和氏永宁令名父之子也授玄肖诸童乌属草方之文考少即抵掌而谈秦汉奋力以挽风骚非晋魏之音絶口不谈非六籍之书屏目不视盖病乎世之决裂以为体饾饤以为词故音非朱弦词非黄绢宁弃去不屑就也弱冠尝循例计偕登台展骥都人钦迟朝列虚左诗歌本之古选兴寄备乎开元彬彬然名家矣每停桡见过輙下榻信宿凢三视予诗初见之已无杜徳机矣再见之其善者机耶别去三易寒暑乃出幔亭集二帙予执简叹曰其衡气机矣夫当其兴寄扵山川发愤于岁月周爰扵驱驰宣情于羇旅由乐府而逮五七言古由近体而至五七言絶调匪偏长体必兼善力追古则尽涤时趋可谓头头是道重重发光矣又一字之工闻竒而倾耳片言之巧覩缛而跃心纸价腾涌扵都门箫谱浸淫扵禁掖雕章绮合藻思罗开行且上书鳯鸣解褐鸿渐探其襟抱将扶摇宇内岂区区抢榆所可控而笑哉闽中一时诸子昆弟咸追述大雅取裁风人作者响臻同好景附真足驰骋海内而惟和则独步当时矣予岁方迟暮笔砚都烧徒有辍翰毁草乌能述匠心之源究作者之概乎因不爱其敝帚而答其请之意长洲张献翼序

　　幔亭集巻一

　　（明）徐熥 撰

　　○古乐府

　　飞龙篇

　　海上三神山中为僊人居金楼髙崔嵬玉堂结绮疏朝乗飞龙去翱翔登鼎湖骋风周八极飘飘凌紫虚安期同遨逰八公相为徒看日扶桑巅采药昆仑区桑田有变迁人世多荣枯安得乞灵药蜕此凡浊躯一朝生毛羽白日升天衢

　　秋风辞

　　秋风摇落兮草木变衰白云缭绕兮鸿鴈声悲兰为佩兮芙蓉为裳彼君子兮天一方鼓桂棹兮度横塘明星烂兮露为霜歌声沸兮夜未央誓与君兮永不忘人生欢会兮安可常

　　悲愁歌

　　我之生兮在深宫嗟薄命兮事逺戎乌孙老兮语不通征尘起兮多悲风望故土兮心忡忡羡黄鹄兮泪无穷

　　战城南

　　战城南死城北枯骨累累弃路侧为我谓乌逰魂不归臭腐将焉用饲汝安足悲阴云漠漠寒雨潇潇黄昏古城下新鬼声号咷阳九厄战城南空褁革七尺微躯我不惜老死蓬蒿竟何益愿为忠臣忠臣诚可为茫茫穹庐掩骼为谁

　　落叶哀蝉曲

　　西风兮吹襟蝉声兮满林古砌静而叶落凄然感乎予心忆美人于闺房兮方敛袂而沉吟

　　野田黄雀行

　　朝来歩野田田中有黄雀翩翩飞且鸣饮啄亦自若何期侠少年遽尔加矰缴置之樊笼中絷以黄金络微物有至性安能事束缚悲鸣念故羣所志在林薄感此宗庙牺叹彼华亭鹤何如辞网罗髙翔在寥廓

　　饮马长城窟行

　　饮马长城窟水深没马腹朝发祈连山暮宿太原谷男儿事格鬬安能讳杀戮昨日新战败死者什五六残血吹腥风尸骸路相续身死自不悲所痛在骨肉嗟哉杞梁妻哀哀城下哭

　　铜雀妓

　　日暮西陵望但见蒿与莱悲风起四野歌舞有余哀漳河流不返繐帐空尘埃荒台久寂寞雄图安在哉

　　羽林郎

　　昔有羽林郎持戟出未央挥鞭过酒肆招摇都市傍酒家女二八颜色耀春芳黄金为条脱白玉为耳珰越罗裁中裙齐纨制下裳服饰既璀璨膏沐尤馨香不意冶逰子停骖登我堂银鞍耀红日韎韐何辉煌提壶索美酒岸帻据胡床金刀脍鲜鲤烈火炮羔羊酒尽出门去婵媛结衷肠已去复再返解赠紫罗囊君心一何苦君情一何长妇人贵逺别持此非所当还君紫罗囊妾非青楼倡

　　车遥遥

　　车轮何遥遥出门不顾返轮岂不能回君心自难挽听此辚辚声一声一以逺看此两行辙能令欢爱断如何驭者手不肯须臾缓敢望君心回但愿车轮烂

　　门有车马客行

　　门有车马客来从故乡陌乍见虽可欢道旧反不怿试问旧亲宾存亡异畴昔别时诸儿童于今皆半百今时诸壮夫多非故相识旧人居新坟新人居旧宅里社既荒凉市朝多变易累累丘垄间郁郁生松栢离别纔几时光阴如此廹静言思故乡含情泪沾臆

　　白马篇

　　谁家冶逰子捐躯事戎行白马黄金羁驱驰赴朔方兠鍪既焜耀宝剑生光芒良弓悬韎韐激矢藏缥囊仰天射鳬雁俯身射豺狼膂力已絶世捷黠谁能当边风方惨冽壮士临风僵跃马入重围生擒左贤王丈夫思报国安肯思故乡裹尸诚所愿毋然悲死伤

　　君马黄

　　君马黄臣马赤君马驰驱臣马辟易君马登天闲臣马伏槽枥二马同行一马太息

　　临髙台

　　临髙台望故乡下有流水流汤汤羣草委腓万木黄翩翩南雁来回翔揽衣起彷徨美人不见兮断我衷肠

　　怨歌行

　　当时匣中铜于今翳尘土当时箧中丝于今无寸缕磨镜休使光缲丝休使长镜光惊貌换丝长嫌绪乱对镜与治丝两者俱肠断

　　江南弄

　　氷肌皓魄红罗襦郁金裙带悬流苏莲舟荡漾临澄湖临澄湖连臂歌欢不来将奈何

　　龙笛曲

　　于阗玉管流哀音余音寥亮水龙吟调繁指弱忧思深忧思深涕汍澜怀逺道催清颜

　　采莲曲

　　兰舟容与过横塘步摇惊起双鸳鸯罗裙带湿生浓香生浓香红汗滴坐中流奏龙笛

　　逰女曲

　　罗衣半褪飘金缕约束纎腰五杂组日暮采芳临洛浦临洛浦鲜明珠望交甫心踟蹰

　　朝云曲

　　朝云日日阳台上晻暧迷茫开绮帐襄王一去劳相望劳相望梦难醒湘水緑湘山青

　　凤笙曲

　　纎指按节玉参差樱唇缓吸声迟迟吹侬无限心中悲心中悲望逺情何时吹双凤鸣

　　春日白纻曲

　　辛夷半落桃花开金羁宝瑟上春台佳人倦倚阑干曲两两黄莺飞相逐

　　夏日白纻曲

　　緑池风来菡萏香临流照影见妍妆隔花忽听郎歌调自取齐纨障巧笑

　　秋日白纻曲

　　庭槐初落秋气深月中玉腕捣清碪东家倡妇恣行乐酒阑更说罗衣薄

　　冬日白纻曲

　　玉肌起粟寒相侵金壶美酒对君斟麝兰香气罗衾满夜长转觉氍毹暖

　　乌栖曲

　　髙楼四面栽杨栁白马骊珂蹔回首玉壶美酒任君沽妾身十五已当垆
　　微灭银灯隔重幔宛转娇声恣啼唤不辞今夕尽风流只愁明日见君羞
　　帐中弱态含娇妩一夜为云更为雨汝南鸡啼天未明与君还尽片时情
　　月衔半岭灯将灭罗衫掩泪送君别入门独自剔残灯寻思无限夜来情

　　塞上曲

　　将军出北平匈奴争遁走皷振天地昏令出风雷吼边霜裂旌旗塞月寒刁斗系俘受降城屯兵井陉口归来见至尊自献单于首

　　白头吟

　　貌如雨中花节似霜中栢貌改节不移别君意凄恻【一解】妾心如女萝君情如飞絮萝生空纒绵絮飞何处去【二解】嫁君当红颜辞君当白首请君看镜中双鬓何如旧【三解】死树难再荣死灰难再热寄语新欢人莫遣头如雪【四解】含悲下中堂此情空脉脉君心能再回白头应再黒【五解】

　　捣衣篇

　　皎月入房帷凉风吹衣襟逰子逺行役少妇鸣清碪良夜既以永怀人益以深孤鸿唳天际促织吟堂阴余亦别家久涕泣不能禁

　　妾薄命

　　南国有佳人容华世莫比嫁与东邻儿愿言结生死良人轻别离匆匆辞故里朝醉阊阖门暮宿长安市新爱移平生恩情付流水欲促逺人归裁书寄双鲤遗之同心结兼以合欢被人匪金石坚容颜何足恃天涯君不归河清难再俟

　　有所思

　　徙倚向花丛罗衣湿清露乌鹊隠茂林寒蝉抱髙树焚香理素琴聊以抒心愫美人期不来中宵泪如注月色满花阴花阴空竚立何以赠所驩悲歌聊当泣啾啾阶蛩吟喔喔邻鸡急幽会杳无期空令长太息

　　○五言古诗

　　拟古

　　把袂登河梁秋风惨行色去去天一涯此心长恻恻酒殽纷错陈哽咽不能食思君有梦魂从君无羽翼浮云方西驰羲轮忽已匿微款俱未伸相持空太息会面未有期功名当努力
　　落日不回照颓波无西流人生处世间倐忽如蜉蝣百年伤局促其忧乃千秋一朝时命尽悲风起松楸累累丘土中谁辨王与侯贵亦不必喜贱亦不必忧圣贤多微言达者耼与周斗酒相娱乐虚名安足谋
　　闺房何凄清思君意如捣追恨当别时光景太草草愁款积胸臆恍惚不能道别君未须臾依然结怀抱安得回君车与君再倾倒
　　徙倚髙楼上西北多浮云浮云望不极妾泪徒纷纷丈夫思报国仗剑逺从军十年辽水上生死不相闻
　　洛阳有异卉灼灼生路傍颜色如朝霞百步闻馨香五陵多逰冶观者如堵墙竟日欢不足秉烛飞羽觞凉秋降白露气序回清商嗟哉此异卉倐忽无辉光容华不足恃欢爱靡故常因之感薄俗意绪重惨伤

　　感遇

　　山居絶尘鞅众虑寡所营长啸谷声应抚琴熏风生云流古洞湿雨过寒潭清年光自代谢草木随枯荣且尽杯中酒遑图身后名
　　出门尽秋色策杖登寒山云来幽径迷风急衣裳单林深豺虎嘷洞古虬龙蟠路逺人烟絶日昃不得餐蒿藜蔽田野中心自悲酸嚱吁望絶巘始知行路难不如归林卧长啸赋猗兰

　　咏荆轲

　　荆卿一剑客击筑游燕都许身太子丹生死誓不渝文夫报知已何惜此微躯悲歌度易水慷慨自驱车匕首中铜柱非由剑术疎报秦虽不济千载仰雄图

　　薤露

　　日出何杲杲薤露不长保人生在世间岂能长寿考穰穰陌上尘离离坟畔草殇子与彭籛胡然较迟早瞻彼薤露晞感叹伤怀抱

　　咏怀
