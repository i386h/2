　　●刘太史公遗稿序 

　　济南郡中有太史刘允良氏才名噪齐州耳允良没其子为辑其笥稿而枣梨之允良□木天者尽可宴也余道其旧耳当甲辰允良举南宫其于曾太史称门下士曾太史闱中梦李于鳞谒之觉而淂刘太史羔雉之文庄案遂急□焉乃允良与于鳞□□□而为诗文□着闱中事果梦耶真耶十年而允良业成名宿矣将与于鳞可接跗续蕚乃倏以芙蓉城主人去先是曾太史早辞供奉骑鲸于湘沅守先浚声迹如□誉星非不霅煜竟霄而不竟曙两太史向以为盍簪者后人又且以梦太史也所可醒世者言耳允良笥稿安淂不传然允良距于鳞可四纪余而文章好尚大变于鳞非古不组无组不古今则诮古组而爱今缣枘凿不投于鳞几大厄其说倡于楚渐及余齐亦和之曾公楚人刘公齐人也曾公著作未及睹而刘公固未尽逐风靡如猩之癖酒猿狖之善木者也余观从右词人时不相袭未有今以后仇前其剧乃尔元白温李何尝笑李杜不直一钱韩柳文振一代亦不剸刃四杰自诧督亢□首故文体自当变而不当仇允良生时曾向余曰自少年濡首李王诸家五赃结矣徙而新日浣不尽顾李王生今日宜另绣其肠如仍以二物四陈自御不加减刀圭恐黄庭朱衣人不服耳余首肯之允艮不尽逐风靡抑不能却新嗜其诗文具在温醇鲜快无不有之为猩与猿狖良未是佳然世讵屏名酿于箬下弃良材于豫章耶夫允良生四纪之后不害其为于鳞也曾太史之梦即千古且栩栩矣

　　○黄石山歌 

　　有道人自言其师在黄石山葢几于僊矣因出师手书云适在无极洞与同侣赏不二花后数百言文采烂然劝戒一归于正又寄一联云八极浑涵色非色际疑无地一元宛转空不空中别有天字大于升笔势飞舞余甚异之因而作歌 
　　君不见黄石山悬崖千仞不可攀无极之洞深莫测洞里僊人长玉颜不二花开暎僊阙春风开宴斟琼液寉驾时时下碧空鳯笙夜夜吹明月我欲从之问丹诀种种尘缘未能绝遥天极目空悲咽

　　○长短句书惺初记后 

　　我闻许旌阳神人授丹诀手握纯钩三尺寒长蛟夜泣秋江月道成飞车向紫烟石函金字留真诠云是一千二百四十余年后应有龙沙八百之地僊柏枝拂地期将迫海内何人是僊伯惺初翁旌阳孙口诵道德五千言元是玉皇香案吏堕地依然近　至尊曾借上方斩马剑还似斩蛟之剑惊心魂功行未圆不归去避世聊须金马门慧刀能断爱智炬堪破昏鯈忽不凿混沌窍玄牝长为天地根惺初翁肖厥祖紫袍宝节天应与名在丹台玉室中刀圭八口身生羽嗴余学道心尤苦只愁凡骨难轻举愿作寉奴翁其许
