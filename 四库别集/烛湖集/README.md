　　钦定四库全书　　　　　集部四
　　烛湖集二十巻附编二巻　别集类三【宋】
　　提要
　　臣等谨案烛湖集二十巻附编二巻宋孙应时撰应时字季和自号烛湖居士余姚人登淳煕乙未进士初尉黄岩迁海陵丞再迁遂安今改知常熟县以仓粟流欠贬秩移判邵武军未上而卒考杨简作应时圹志及张淏防稽续志均称其绍煕初尝应蜀帅邱崈辟预料吴曦逆谋白崈以别将领其军后曦以叛诛其言果騐时应时已殁三省奏官其子祖开盖亦智畧之士又史弥逺受业于应时集中与弥逺诸书皆深相规戒迨弥逺柄国超然自逺无所假借甘沦一倅而终其人品尢不可及矣宋史艺文志载烛湖集十巻据应时诗中自序盖尝应刘克庄之求手编其藁为五十巻集末有其侄祖祐跋称洓水司马述先以十巻付梓后付以问思録五十条通鉴摘义三十条总名之曰经史说又附雪斋父子倡和诗及雪斋行状墓铭扬简所撰圹记防稽续志小传子祖开补官省劄等篇是十巻为祖祐所编非其旧本也年逺散佚久无传本故厉鹗作宋诗纪事仅于吴礼部诗话王应麟困学纪问黄宗羲姚江逸诗内采掇数篇寥寥不备兹从永乐大典所载排纂成编惟经史说残缺特甚仅存一篇其余则约畧篇数殆已十得八九以巻帙繁重分二十巻仍附编其父介及其兄应求应符诗并録应时父子志传行状子祖开补官省劄诸篇为上下二巻应求字伯起尝登乡荐应符字仲潜所着有歴代帝王纂要二巻初学须知五巻载于陈振孙书録解题今并未见云乾隆四十六年九月恭校上

　　原序
　　国朝人文之盛列圣涵濡逮熈丰元祐间伊洛诸先生出焉其于发挥洙泗防言可谓至矣髙宗宏济大业首崇经术天下举知儒道之贵孝宗稽古好学敬事元老天下益知师道之尊声应气求师儒辈出若南轩张公象山陆公晦庵朱公东莱吕公皆以斯文自任烛湖孙先生早承学于象山晦庵之门天分既髙学力尤至穷理尽性深探阃域四方之士翕然景从凡经指授随其才品有以自立淳熈甲辰史忠定王延致先生讲道东湖今丞相鲁国公昆弟实从之游述获在执经之列丙午嵗先人宰海陵先生适丞是邑尤得朝夕侍左右遐思榘矱尚有遗文诸家藏得已防粹者十有二巻敬锓梓以恵后学述在海陵时尝升堂拜雪斋老先生见手编家庭唱酬集父子兄弟自为师友读之使人起敬是嵗雪斋有赏月诗先人与述两兄亦相与赓韵感念畴昔阅四十载唱酬集有遇雪鬻田二诗乃先生龆齓所作因并赏月诗撮十题刋于附録之末既以表先生防年识趣之伟又以见雪斋源流之可敬也述不肖安敢效李汉彚昌黎文而为之序姑序次刋集嵗月纪于巻首云宝庆丁亥长至日门人洓水司马述谨书












　　钦定四库全书
　　烛湖集巻一　　　　　　宋　孙应时　撰表
　　慈福太后加上尊号贺皇帝表
　　圣治重光侈璇宫之锡美慈闱备福新宝册之扬徽奉缛礼于中天伟闳休之冠古【中贺】恭惟皇帝陛下祗承舜孝遹广文声履揖逊之昌图辑圣眀之钜典于皇帝祉盛三宫燕喜之时仰止坤元大八秩夀康之庆肆防谷旦登衍鸿名极荣养于东朝永延洪于奕世臣身縻外阃迹逺修门扬厉铺张莫预千官之末议咏歌舞蹈徒切四表之欢心
　　贺夀皇圣帝表
　　嗣圣总师祗广舜心之孝重闱奉册丕昭文母之尊焕缛典于丛霄施闳休于区宇【中贺】恭惟至尊夀皇圣帝陛下仁深与子道尽事亲置酒未央踵十年之荣庆问安长乐新八秩之夀康兹搜举于上仪益光华于叠矩动龙楼之喜色沸鼇极之欢声臣身使蜀门心驰魏阙亲逢盛事亶千古之一时嘉与逺民共三呼于万嵗
　　又
　　今古一时莫盛慈皇之事母国家多庆诞闻圣子之生孙喜集丛霄荣均广宇【中贺】恭惟至尊夀皇圣帝陛下循尧尽道命禹执中浃宇宙以归仁备天人之荐祉龙楼就养已欣付托之无忧甲观发祥更卜继承之有永神民呼舞宗社奠安臣身使蜀门心存魏阙丕惟盛事逺超载籍之五三益愿慈顔坐阅来昆之千亿
　　皇孙生贺皇帝表
　　圭继治赫帝业之流光神剑告祥挺孙枝之毓秀福全五世喜浃四方【中贺】恭惟皇帝陛下逺纉禹功率行舜孝重亲恱豫躬大安长乐之朝世嫡蕃昌新甲观画堂之庆亶天人之嘉会轶宇宙之前闻遥知三日之亲临想见五云之佳气有室大竞无疆惟休臣逺去鸳行阻陪虎拜流传盛事实欣千古之一时鼓舞斯民同祝三宫之万夀
　　代史魏公贺皇孙出閤表
　　帝祉鸿蒙奕叶畀皇家之庆孙谋燕翼异宫开世嫡之尊盛事难逢函生胥悦【中贺】恭惟皇帝陛下率履尧道绍新周邦雍雍慈孝之传蛰蛰繁昌之绪含饴弄膝前知映日之精神赐服班朝喜见成人之礼乐爰疏恩于甲观俾开府于禁垣光昭万世之规丕对三宫之训臣身虽谢事心敢忘君欣闻国家典礼之行敬为宗社安荣之贺
　　防庆节贺表
　　宅位倦勤髙蹈清都之表大徳得夀増多神防之元仰庆旦之虹流溢欢心之鼇抃【中贺】恭惟至尊夀皇圣帝陛下仁如天覆道与时偕法度循尧底继承之尽善厯数命禹知付托之无忧龙楼临五日之朝鳯纪届千秋之节合三宫而式燕骈众福以来同臣早滥周行新防蜀寄想望未央之前殿虽觉岷江万里之遥窃与青城之老成同上嵩岳三呼之祝
　　又
　　祥开虹渚适当盈月之期庆溢龙楼丕衍后天之算三宫式燕四表同欢【中贺】恭惟至尊夀皇圣帝陛下徳配两仪道超万古循尧继治巍巍其有成功命禹执中恢恢无复余事履大安之至养奉长乐之慈顔方将纪八千嵗之春秋为亿万年之父母属临庆节益茂宏休臣滥使遐方欣逢盛际玉巵在望莫陪汉殿之羣臣芹献不防但効华封之三祝
　　又
　　生商运启日月防析木之津戴舜心同麟凤舞箫韶之奏当玉巵之称夀想璇极之増懽【中贺】恭惟至尊夀皇圣帝陛下道与时行仁如天大倦法宫之事诏游姑射以神凝集四世之全荣冠百王之景铄流虹绕电属天人嘉防之期就日望云均臣子爱君之意臣缀班奎阁将指坤维尚想鸳行闻九霄之警跸敢忘虎拜歌万夀之声诗
　　贺光宗皇帝登极代司马通判俨进文正公奏劄表
　　葵藿无情犹知向日蚁蜂有悃能不忘君矧滥齿于衣冠而久防于禄秩拊躬抃蹈际运休眀三圣相传信千古未闻之盛六龙在御当万物咸睹之时加以言路宏开封囊交上虽惭应诏之奇策窃贡承家之旧闻内激愚衷死有余罪【中谢】伏念臣曽叔祖臣光见推一代受任累朝毎懐入告之嘉猷自信生平之力学盖尝以曰仁曰眀曰武为君人之徳以任官信赏必罚为政治之规自仁庙英庙之畴庸逮神宗哲宗之访落始终六説恳切一辞然九重不谓之迂防而四海咸称其忠荩巍巍底乂往往由兹曩者臣兄伋以州佐之防官逢夀皇之初政用誊奏藁进彻宸聪曲荷并容俯嘉狂狷恭惟皇帝陛下祗承内禅遹济中兴若稽本朝之宪章追想当时之耉老曽是先臣之献替久尘乙夜之览观臣实无知懐不自已敢縁故事特寓防衷忝谓世家久已厚乾坤之雨露庶几后裔犹能増海岳之涓尘惟无忽于常谈斯立观于实效所有先臣光前件劄子四道谨缮冩随表昧死以闻
　　重眀节贺表
　　为天下君赫乾坤之眷命祝圣人夀均臣子之至情瞻盛旦之虹流溢懽心之鼇抃【中贺】恭惟皇帝陛下严恭寅畏缉熈光眀成禹万世之功兴舜九招之乐三宫四世方永保于欢荣百姓羣黎悉陶成于仁夀于皇嘉节滋受宏休臣逺迹周行分符蜀土望天顔之玉晬阻奉宸欢占嵗事之金穰并为国庆
　　又
　　地辟天开丕赫离眀之照虹流电绕亶临震夙之期沸率土之欢声永皇穹之休命【中贺】恭惟皇帝陛下道传精一功格平成视膳问安仰尽重闱之养垂衣拱手俯观万物之宁星眀南极之躔神授泰元之防允兹庆旦丕拥宏休臣叨被眀恩逺将使指难趋汉殿进都防万年之觞嘉与蜀民致封人三祝之意
　　又
　　身縻万里忝西清学士之班节过千秋庆南极老人之见仰致后天之祝俯殚报上之勤臣闻作善而降之祥大徳必得其夀于皇上圣出应昌期恭惟皇帝陛下天锡九畴躬行一道总章衢室不忘尧舜求贤之心长乐大安逺过汉唐事亲之孝是増神防以永帝龄臣逺去鹓行阻陪虎拜金茎沆瀣惟愿奉万年之觞玉殿罘罳徒想听九霄之跸
　　代邱制帅谢赐厯日表
　　寅正得天亶在躬之厯数昕朝颁朔通薄海之车书爰敇庶邦以兴嗣嵗【中谢】恭惟皇帝陛下继舜一道叙禹九畴察治象于璇玑凝圣功于玉烛析因夷隩庸敬授于人时东西朔南方大同于声教臣夙浮江汉来使岷嶓初布诏条恭逢驿赐自天而下仰星辰日月之旁罗与物为春庆草木山川之咸若
　　牋
　　贺慈福宫牋
　　福盛东朝极两宫之敬养庆钟甲观全五世之欢荣【中贺】恭惟夀圣皇太后殿下徳启姒任名光马邓圣而继圣萱庭素释于忧勤孙又生孙椒匊坐观于蕃衍坤仪所镇帝祉滋多臣荷国厚恩逢时嘉庆含饴弄膝想知霄极之怡顔奉册称觞尚见云礽之列侍
　　贺夀成皇后牋
　　思齐之助浃燕喜于三宫长发其祥见蝉聨之五世画堂毓秀慈阃増辉【中贺】恭惟夀成皇后殿下俪日竝眀配天不老媚自周姜之妇格有王家施于洛诵之孙生防帝祉荣超古昔福萃宗祧臣夙荷国恩逺违朝迹丛霄在望惟祈万夀之无疆盛事难逢敢后四方之来贺
　　贺皇后牋
　　六宫治内重光彤史之传五世全荣俶奏画堂之庆福扶庙社喜浃方维【中贺】恭惟皇后殿下莘女笃周涂山兴夏徽音是嗣恵宗公以御于家邦景命有开厘女士而从以孙子厥钟世嫡其对天休臣叨际昌辰欣闻盛事祝厘朱邸更符榴子之多垂裕皇基用卜椒条之逺
