　　国朝朱鹤龄撰鹤龄所着尚书埤传诸书俱别着録此集乃所着诗文凡赋一卷诸体诗五巻杂文九巻末附传家质言十三则鹤龄始专力于词赋自顾炎武勗以本原之学乃研思经义于汉唐注疏皆能爬梳抉摘独出心裁故所为文章悉能典雅醇实不堕剽窃摩拟之习其邶卫三国禹贡三江震泽太湖嶓冢汉源诸辨多有禆于考证生平尝笺注杜甫李义山诗其作韵语颇出入二家之间而寄兴清逺亦复自饶神趣至其与钱谦益同郡方谦益笺注杜诗时尝馆于其家乃集中无一语推重之所作书元裕之集后一篇称裕之举金进士歴官左司员外郎及金亡不仕隠居秀容诗文无一语指斥者裕之于元既足践其土口茹其毛即无訽詈之理非独免咎亦谊所当然乃今之讪辞诋语曾不少避若欲掩其失身之事以诳国人者非徒誖也其愚亦甚云云其言盖隠刺谦益而发尤可谓能知大义者矣乾隆四十二年十月恭校上

　　广志赋
　　繄我生之耿介兮励修名而不颇搴瑶华以抗节兮耨仁义为菑畬游坟典以渔猎兮惧或中乎虞罗戢形影于蜗庐兮循绳墨以贞度彼诡行之昌披兮余不敢以改歩忧歳月之逾迈兮追姱修于古路何流俗之险诐兮好与恶之迷方珍鱼目以什袭兮谓明月其不光絷威鳯以在笯兮驱罢驽使服襄嘈瓦釡以聒听兮毁钟镛而弗鍧巧占风于要路兮工树栅于康庄羣听听以嘷吠兮索兰佩而儜簸秕穅以眯目兮转粪壤而自芳诚嫭妬之难竝兮畴防径而能良蹇余情之信练兮秉昭质而难易屑越桂以为粮兮编胡绳以为席被荰若之清芬兮茹玉英之膏液块独寳此幽贞兮匪是时之攸识惜曜灵之不居兮老将至而为客菊英坠于中庭兮奄离披而独立彼独立其何伤兮哀众芳之芜没意纬防而不制兮神惝怳而难持愁沈沈而芚结兮歩踽踽而躨跜怀纡郁而无语兮思繁乱以长咨迷路而无所适兮索琼茅以占之将逺游而散虑兮得吉繇【宙】于庖羲洵区中之偪仄兮利攸往乎八荒简元辰而奋翼兮载云旂以髙翔驾苍虬之婉婉兮鸣玉鸾之将将风伯俨其翼卫兮冯夷导夫津梁朝发轫于三危兮夕緤马于扶木濯余足于天池兮晞余髪于旸谷恨鼇极之不正兮就朱阳之赫曦吊帝子于苍梧兮遭炎歊而去之将问涂于昆崘兮凌弱水而径度畏北极之层氷兮谢防之我遻【忤】羌弭节于地首兮廕木禾之扶疎循芝田以纵辔兮过瑶圃而回车憎王母之白首戴胜而穴处兮嗤穆满之骋骅驑騄耳于长衢乗灵风以来往兮睨下土而心恻纷总总其腥秽兮类青蝇之集棘郁余怀其未申兮望清都而伏谒瞻绛节之繁防兮听天钧之将阕叫帝阍而不闻兮邈天皇之难觌向左以陈词兮剖沈疑于宿昔何履仁而凋年何肆暴而终令何坎每集于翘材何丰隆多享于破镜牛哀之化虎兮磨牙殒其兄命虬精之化人兮厥勲烈于南郡何流变之殊形兮理叅差而莫定虞忠汉而暴烈日兮终不逢夫雨师苌叛苻而絶汲道兮竟滂沱而相之何顺逆之较然兮乃倒应而错施桂必伐而膏必煎兮盍并绝乎斯类也菉葹荣而枳棘茂兮余不忍见此态也庄畏牺而贾忌鵩兮固前哲之所喟也彼维皇之降监兮宜祐善而殄滛苟回惑其若兹兮信天道之难谌左曰吁汝言大愚品物迁流干于太虚潣潣墨墨实为洪鑪道混成而条别兮任万肇之荣枯譬春葩之着树兮随风举而辞株或下沈于粪溷兮或飞坠于金铺总一气之搏捖兮谁斟酌以差殊是故否兮不可豫泰兮不可期荣兮不可据辱兮不可辞福兮不可觊祸兮不可闚子亦访塞翁之倚伏师老氏之守雌何事怀褊心之愤懑长伊郁而不怡承嘉告以廓落兮释端忧之蹇产溯大钧之陶物兮非薫莸之必辨审性命之靡求兮悟悔尤之双遣苟精诚之必通兮虽圣哲其可勉能铲迹而含和兮登道崖其未晩修初服以悦康兮奚徬徨而适逺辞阊阖之迢遥兮整飚驭乎吾将反临旧丘之庵蔼兮遵夷涂以游衍归班生之故庐兮息扬子之亭眷天路之缥缈兮神迭荡而未寜葺诗书之靓圃兮践蕙若之芳庭永端居以摄志兮栖寂寞而存真旷览于大道兮乐自然以无营庻灵根之内保兮絶情累于尘缨
　　【徐介白曰出入幽通思之间而稍变其机杼时出新意怆怳徬徨此亦楚骚之苗裔也】
　　游灵岩山赋
　　郁西山之崩奔萃扶舆之清淑巩吴防以作标翼帝图而建福秦余飞动以崔嵬邓尉蜿蜒而起伏天平拥崒之竒穹窿擅曲纡之目玉遮之横列如屏龙坞之踞湖孑矗是皆迥福地于人寰锡嘉名于仙箓未若灵岩之奥区为神秀之所族也爰自浮玉俶基天目分陇元气所扶冲融澒洞苞峦孕巘控越奠吴俯临巨浸上薄清都斡地轴之纲维蕴物华之明儁睨儿列之羣峰卓灵标之千仞开丹嶂于烟霄挂青霓于风磴轧游氛以上千变万象于朝暝势延亘以贡竒气孤清而含润尔其戴【阙】　　诡状非一或偃如兽蹲或森如人立或奋如涛奔或齴如斧剨张势竞形空翠滴歴弥天之云触焉斯出又有琪枝防宻灌木森沈叶舒帷以聚碧花散锦以成茵蔚春荣之的皪敷夏彩之缤纷啭深林之绮翼戏曲沼之文鳞激松杉之逸籁飞涧瀑之清音其右则重湖浩淼惊涛震荡水豹狂呼灵胥贾壮估舶分风渔舠出浪抗洞庭之双峰砥百川而作障挹朝爽于西来招灵威以极望其左则原田每每沟塍绮织近畛遥阡莱芜大辟嵌窦通流桔橰引泽休文之菑耜可陈【沈约赋陈东菑之故耜】元亮之耘杖堪植其阳则防通九陌市走三衢储胥广衍瓌货敷腴士女填溢以敖以娱其隂则林臯转幽竹苞松茂曲渚【平】绪分枝阜环凑谢墅庾村逸人逺就洵地脉之储精而土膏之发秀也若夫山开晋代寺建梁年招提烁日宝刹凌烟嶤阁竦乎霒靉香台镇乎防岏岧亭因岩而构址露井即以疏泉虹梁披离以切汉月榭杳窕以干天曲槛周遭以緜亘鸣铃窸窣以髙悬幡幢夐绎而四竪金碧晃耀以相鲜于是防振兴公【孙绰赋振金策之铃铃】经翻谢客【谢灵运在庐山有翻经台】开士骈肩净侣叠迹建曼陀之莲台展灵鹫之法席合能秀之宗风融支许之辨析殷【隠】法鼔之轰阗铿晨钟之叩击宣三乗之奥窔释五惑之烦慽葢自智积之西臻防黍以千百洎天监之示圗徴象教于廊壁虽名标志林事纪僧册未及今之齐大化于吴防普神光于缁白也尔乃循岫曲歴松门抚陈迹按旧闻求洞穴践村砚池谁为防笔石鼔何以徴兵访馆娃之故址吊响屧之幽魂悲琴台之寂寂濯香水之泠泠慨属镂之既赐俄宫阙之成荆实夫差之自烬岂越甲之能鸣嗟乎噫哉方吴之盛香径铺棻璚台对整花明桂寝之玉人月照梧宫之金井迄今惟朝霞之绛气横空夕烧之荒烟接岭彼甲楯雄圗甬东覇骨俱巳荡灭而销沈而况乎舞袖之绮罗与妆台之红粉览兴亡之一瞬悟生死之同防物无新而不故人畴寄而不归譬石火之飚飞如薤露之朝晞知荣名之难宝信逆旅之非欺曷若依绀宇以息钳廕只园以屏伪探六度以观生防三旛以发智饭香积而世味蠲游化城而害马去廵兹山以陟降兮澹余虑之纷营溯髙深于亭毒兮等今古于埃尘怅沈忧之促算兮希閴静以延龄愿税驾于东林兮武宗雷之髙躅仰佛日于靓墟兮皈化人于灵谷庻游防之可谢兮登薄躬于沕穆
　　【潘江如曰具体天台更加妍整五色迭宣八音繁防使人怡赏不给】
　　宫人入道赋
　　夫何女冠之婀娜兮被羽服而体轻夙承恩以入御兮侍更衣于掖庭忽离垢而遐举兮岂徒怨夫深宫永漏之凄清粤自吴鷰焚巢楚猿亡树镜埋仁寿之庭弓倚未央之柱攀乌号而莫逮山陵泣湘浦而长辞月御悟物理之无常荣悲韶顔之难久驻羞同戚姬侍儿更作扶风之妇愿随王母使者时陪戴胜之旁于是离椒寝即云房捐珠佩理霞装谢昭阳之队游紫府之郷屏长门之扇簟裁羽客之衣裳洗鈆华之腻粉爇太极之神香尔乃金阙玉台神栖三岛璚枝琪草景蹑十洲药圃之芝英不落仙岩之桂叶长留乳炼金壶餐紫霞而内照丹调石髓咽液以防搜若夫朝瞻羣帝夕礼上真碧落磬敲缥缈珠函之字坛鼓动琳琅玉检之文绛节霓旌髣髴朝元启醮龙车凤吹依稀太乙祠神羽帔却防垂手舞法歌惟按歩虚声爰有八洞灵妃九天女时来往于空虚每殷勤而延伫许飞琼要世外之逢迎蕚緑华结云中之伴侣鹤飞缑岭听子晋之瑶笙霞散幔亭防神君之法鼔寜复忆夫青蛾永巷之悲纨扇秋风之怨御沟流一叶之词弱线结重生之愿行随玉辇曲翻红药之栏帚奉金门愁满菱花之靣也哉遂歌曰谁信瑶台别有春焚香时拜魏夫人欲将蓬岛三生梦扫尽沧桑万古尘又歌曰灯照九防天宇濶云开三素日华鲜蜀王巳化啼鹃魄岂得蛾睂恋翠钿
　　【徐昭法曰悱恻之音归于正则貎悴而神不伤】
　　秋闺赋
　　维四序之平分兮凛素秋之凄景谁骀荡而不悲兮矧空闺之孤冷若夫月行白道霜下青腰授衣气冽落木天髙斑骓未返芳草俄雕玉露弥空湿前除于永漏金波斜照逗疎幌于清宵云送千山之皜素水兴七泽之波涛鷰去空梁展鱼笺而不乐鸿来北渚抚玉枕而无憀于时摵叶鸣梢哀蝉去翳看罗带之渐长觉瘦腰之转细茱萸幔里风撩鷰尾之髾玉镜台前光冷鳯头之髻芳辰谢兮倚宝瑟而谁弹沈思增兮对金樽而掩涕綦履遥兮惜蛩砌之苔生房栊寂兮叹蛛丝之网缀所以霏霏银汉难传锦字之情皛皛玉钩不断髙楼之思尔乃京洛之英邹鲁之子景汉伯随师七州葛稚川负笈千里绛帐未归金风忽起能不目睇桂枝魂牵槐市又有商轺逺道估舶河梁载离寒暑几隔星霜上水之篙时防百丈镂金之椀谁与双行河鼔星槎愁结三洲之曲洞庭木叶梦廻七宝之牀乃若代北沙昏榆中候晩出紫塞以遐征伫青骢而不反寒催刀尺忆襻袴之短长望阻帛书昧关山之近逺罗帏悄悄占华蕚于银缸虬箭沈沈冷麝熏于玉簟以至弃妇长辞故雄永去离鸾之曲孤弹别鹤之悲莫诉空庭销蕙艸之芳夜织断鸳机之素秋风厉兮团扇捐秋月清兮流萤度秋声凛兮罗防单秋气肃兮鈆华暮当斯时也莫不愁黯黯而难胜泪浪浪而欲注夫候偏惊于徂谢感独切于变衰壮夫为之砭骨骚客以是兴悲登山兮伤白道临水兮惭青谿逺行兮悼覉旅送归兮怅别离彼四端之窈矫皆千古所欷歔曾未若中轻寒于玉肤引凉飔于绣闼惊一叶于空闱感双星于令节结永恋于征轮寄防情于皓月穷时变之萧森萃人生之骚屑抽思纬之纒緜殚情条之飞越乃为乱曰繄大化之回斡兮独春蠢而秋愁非金商之肃杀兮岂知冶日之和柔彼佳人之冷艶兮固难遣楼头之蟾兎与林下之飕飗诚物感之自然兮吾独何恨乎蓐収
