　　钦定四库全书　　　　　集部二
　　温飞卿诗集笺注　　　　别集类一【唐】
　　提要
　　【臣】等谨案温飞卿诗集笺注九巻明曾益撰
　　国朝顾予咸补缉其子嗣立又重订之凡注中不署名者益原注署补字者予咸注署嗣立案者则所续注也益字予谦山隂人其书成于天啓中予咸字小阮长洲人顺治丁亥进士官至吏部考工司员外郎嗣立字侠君康熙壬辰进士由庶吉士改补中书舍人曾注譌颇多如汉皇迎春词乃咏汉成帝时事而以汉皇为髙祖邯郸郭公词为北齐乐府旧题郭公者傀儡戏也旧本讹词为祠遂引东京郭子仪祠以附防祠字之讹嗣立悉为是正考据颇为详核然多引白居易李贺李商隠诗为注虽李善注洛神赋逺逰履字引繁钦定情诗为证古人本有此例然必谓夜宴謡裂管字用白居易翕然声作如管裂句晓仙謡下视九州字用贺遥望齐州九防烟句生禖屏风歌银鸭字用商隠睡鸭香鑪换夕薰句似乎不然是亦一短也唐艺文志载庭筠握兰集三巻金荃集十巻诗集五巻汉南真稿十巻宋志亦同陈振孙书録解题作飞卿集七巻又陆防渭南集有温庭筠集防称其父所藏旧本以华清宫诗为首中有早行诗后得蜀本则早行诗已佚文献通考则云温庭筠金荃集七巻别集一巻是宋刻已非一本矣曾本合为四巻名曰八义集以作赋之事名其诗颇为杜撰嗣立此注称从所见宋刻分诗集七巻别集一巻以还其旧疑即通考所载之本又称采文苑英华万首絶句所録为集外诗一巻较曾本差为完备然总之非唐本之旧也乾隆四十六年六月恭校上
