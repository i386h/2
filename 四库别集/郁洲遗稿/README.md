　　○奏疏 

　　乞致仕疏 

　　臣梁储谨奏为自劾不职乞恩休致事臣才微徳薄素多疾病今年踰六十病日益増而久叨重位瘝旷无补其为可去无可疑者顾旧岁二三月及十一月间曽三次自劾求退俱未蒙圣恩俞允今岁正月间又同大学士杨廷和等以灾异之故具疏乞休亦未蒙俞允之旨臣伏自惟念尤有不胜其惶惧者请更以臣宜去之义重为皇上陈之臣闻君父之恩无穷而大臣之分有限为大臣者若昧其止足之戒冥行而不自顾则折足覆餗之凶咎随之矣况今内阁三臣之中惟臣年最老病最多才最短辅导最无状欲求因灾异策免以警戒于有位宜无有先于臣者皇上纵存念旧物未遽捐弃然其如臣之私愿未遂何哉如天下之公论未协何哉臣之此情实尤恳切伏望皇上特垂睿察即日容臣以礼致仕别选贤能以充任使庶几尽修徳弭灾之实以回天意而臣于林下亦得以终被无疆之泽矣臣无任待罪祈恩激切恳请之至为此具本亲赍谨具奏闻伏候勅旨正徳九年二月初四日奉圣旨卿职司宻勿劳勚著称正方倚任才识老成岂可引年輙求休致所辞不允吏部知道 

　　问安疏 

　　臣梁储等谨题二十九日早司礼监太监萧敬等节该传奉圣旨朕躬偶尔违和暂免朝参数日臣等闻命下情不胜瞻恋伏望圣明慎起居节饮食頥养天和茂膺洪福以慰臣民之望谨具题问安正徳十年四月三十日 

　　请停工疏 

　　臣梁储等谨题前日钦蒙发下工部一本为修理殿宇添盖房屋等事大意以为见今营建干清坤宁二宫工程浩大财力匮乏将内官监所奏修理太素殿及天鹅房船坞等项房屋工程停止臣等已拟旨进呈未蒙批出中心惶惑不知所为窃惟陛下宫寝未建中外臣工日夕不安工部职掌工作已经会计查处派征差官分投前去各处采取大木烧造砖瓦但値民穷财尽之时猝难办集非惟工部以为忧臣等皆切忧之恨无方略以佐经费今大工方兴若将前项不急之工一并整理则合用钱粮将何取给取之官则官无蓄积取之民则民已困敝况此役一兴大工未免为其妨误夫以可缓之役而妨当急之务深为未便该部计无所出利害切身事势窘廹言非得已伏乞圣明干断速赐批行将前项殿宇房屋工程暂且停止待后大工就绪民力稍舒之日该衙门另行具奏定夺臣等不胜恳悃之至谨题请旨正徳十年闰四月日 

　　劝止临幸疏 

　　臣梁储等谨题日者窃闻圣驾自西安门出外经宿而回不知临幸何所臣等初闻未敢遽信既而道路相传众口藉藉使臣等心志忧惶神魂飞越辗转思惟莫知所处窃惟天子出入必偹法驾必传警跸卫士环列百官扈从所以严至尊之分而防意外之虞也且如南郊大祀不过一宿虎贲之旅鹰扬之将周旋左右而直庐拱卫官军万余警柝之声夜以达旦至于皇城各门又令勋戚重臣守把祖宗之法至为详偹今圣驾之出不知环卫者何人居守者何官文武羣臣茫不与闻若徒无故轻身而出率意而徃扰扰尘埃中万一车马惊蹷之虞奸盗窃发之变出于意料之所不及未知何以偹之虽然天神协相决无是事而臣等私忧过计实切寒心夫千金之子尚不肯垂堂而坐陛下一身乃宗庙社稷之主纵不为身惜独不为宗庙社稷计乎仰惟圣性髙明天资英迈洞烛天下之事机已非一日必不轻易举动窃恐左右羣小贡谀希宠之徒倡引事端蛊惑聪明陛下偶未深思而遽从之上累圣徳下骇人心凡此导引之人其罪殆不容诛但事在秘宻非臣所知不敢妄有指议伏望陛下念祖宗付托之重体臣民瞻戴之情自今以后端拱穆清以保威重节宣劳逸以頥天和严内外出入之防正堂陛尊卑之分戒非时之宴逰屏无益之玩好仍乞查究导引出外之人置之于法以彰刚断之徳以解臣民之疑宗社幸甚天下幸甚臣等偹员辅导平时既不能调护圣躬弼亮治理兹有所闻若复忌避不言则欺君负国之罪死不足赎傥以为所闻不的干冒天威眀示黜罚不敢辞避臣等不胜忠爱恳悃惶愳廹切之至谨具题知正徳十年六月十六日 

　　议邉务疏 

　　臣梁储等谨题为邉务事近因陜西地方节报边宼声息特命大监张忠监督军务帯领参将宋赟湛臣部下官军三千员名从北路去都督张洪挂印充总兵官帯领京营官军五百员名辽东官军三千员名从南路去都御吏陈天祥提督军务各调度陜西将官杀贼臣等仰见皇上悯念闗陜军民重被荼毒故欲大彰天讨以靖地方古帝王安夏攘夷伐罪救民之心不是过也窃惟陜西自去冬北敌入套今年由夏徂秋不次入境宼剠各该总制镇廵等官不能设策防御致今八月以来拥众深入腹里平凉鳯翔地方又侵犯延绥米脂绥徳等处大肆掳剠我军曽无斩获之功上廑圣虑逺劳王师固非得已但敌人故智入套则西寇闗陜过河则东窥宣大今在套已及一年入掠不止一次饱其所欲满载而归腹里地方孳畜已空无可复掠此后水冷草枯似难再入冬深冻坚必将蹈氷过河为大同宣府偏头闗等处之患及照陜西守臣初未尝请兵该部原拟设主将一员帯领京营官军五百员名限十日内起程星驰前去调度征剿本欲救陜西目前之急已蒙圣明俞允深合事宜今张洪受命月余机会已过若复统领重兵纡回数千里以至陜西冬令已深河氷已合腹里恐无可寻之贼徒劳士马徒费刍粮何益于事至于亦卜刺余宼逼近洮岷则又逃难残兵守臣自足当之且陜西邉徼之贼四肢之疾也宣大门庭之宼腹心之疾也况近日朶颜夷人屡犯我邉与小王子结亲其奸黠未可窥测臣等私忧过虑不在陜西而在宣大窃谓宣府及辽东官军正宜养威蓄锐以防意外之虞以剿门庭之宼今乃逺调西陲为后时之举徼难成之功恐非至计倘或宣府以东烽火有警士马单弱又将征调何地之兵以应之乎伏望圣明干断合无令张忠张洪俱从宣府大同前去一面差人驰徃延绥探访如果敌兵尚在陜西地方抢掠镇廵官飞报速即前进督同各镇将官会兵征剿若潜伏套内势将过河前项军马且在宣大驻札休息听调待其过河各官应否改命提督宣大山西或敌众北遁归巢相应班师俱听兵部议奏定夺惟复照依兵部原拟先令张洪帯领京营官军五百从山西石州地方过河张忠亦先帯令奏讨团营官军五百员名与陈天祥俱徃赴延绥相机行事其辽东并宋赟等部下官军俱暂在宣府操候本处有警可以调用若陜西果有紧急声息方纔催调前去庶免徒劳人马亦不虚费供给况陜西延宁二镇兵马俱素称勇锐而大同逰竒等兵又见在彼处得人调度足可成功何至逺劳辽东之兵以示弱于敌人乎臣等又思得宣大二镇及山西偏头闗等处节被敌人宼扰地方凋残士马疲敝而宣府军威屡挫失事最多比之他镇尤为狼狈切近京师与宻云古北口一帯唇齿相连尤宜保护伏愿皇上申敕三镇守臣及时修饬墙堑操练军马禁剥削之弊以养兵力时粮赏之颁以作士气一应防邉御敌事宜预为区处贼来有以待之使地方不至重贻患害以贻九重北顾之忧臣等偹员辅导知而不言是为欺君畧陈所见如此伏惟圣明裁择谨题请旨正徳十年十月初九日 

　　请明敕遣中官疏 

　　臣梁储等谨题为传奉事该兵部掲帖开称正徳十年十一月十三日该司礼监太监温祥传奉圣旨司设监太监刘允着他前去四川等处公干写勅与他该衙门知道钦此钦遵传奉到部合具掲帖赴翰林院请写勅书等因照得内外官员钦奉勅内必湏偹开所行事务今太监刘允前去四川未知干理何事该部掲帖内不见开载臣等无凭撰写仰惟圣明在上事遵成法兹欲勅遣近臣于数千里之外公干必是事情重大伏乞明谕臣等知之庶使所撰勅文有所凭据谨题请旨正徳十年十一月十七日 

　　修书籍疏 

　　臣梁储等谨题为修理书籍事照得内阁并东阁所藏书籍年岁既久残缺颇多必湏专委官员用心管理方可次第修补今推得见在诰勅房办事中书舍人胡頥序班刘伟俱堪委用合无令胡頥仍旧职刘伟改典籍与同原管主事李继先专一管理前项书籍臣等仍督令逐一查对要见某书原数若干见在完全者若干残缺应修补者若干待查明之日再行奏请裁处縁系修理书籍事理未敢擅便谨题请旨正徳十年十一月日次日奉圣旨是吏部知道 

　　留逺人疏 

　　臣梁储等谨题暂留逺人教习以便审译事据提督四夷馆太常寺卿沈冬魁等呈该回回馆教习主簿王祥等呈窃照本馆专一译写回回字凡遇海中诸国如占城暹罗等处进贡来文亦附本馆帯译但各国土语土字与回回不同审译之际全凭通事讲说及至降勅回赐等项俱用回回字今次有暹罗国王差人来京进贡金叶表文无人识认节次审译不便及查得近年八百大甸等处夷字失传该内阁具题暂留差来头目蓝者哥在馆教习成效合无比照蓝者哥事例于暹罗国来夷人内选留一二名在馆并选各馆官下世业子弟数名送馆令其教习待有成之日将本夷照例送回本王等因实为便益据此臣等看得习译夷字以通朝贡系是重事今暹罗夷字委的缺人教习相应处置合无着礼部行令大通事并主簿王祥等将本国差来通晓夷字人再加审译暂留一二在馆教习待教有成效奏请照例送回庶日后审译不致差误縁系暂留逺人教习事理未敢擅便谨题请旨
