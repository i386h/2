　　●序 

　　柘轩先生文集序 

　　杭郡为宋故都前元盛时衣冠文物有宋遗风予生也晚前辈诸公恨不及识之惟柘轩凌先生与予衡斋叔祖为同年友至正间予侍叔祖于家塾先生来过见予所作课业亟加誉焉予视先生为尊行而先生待以忘年父每以所作求正先生不吝于教诱掖奖劝之功居多自与先生别五十年矣宣德初自山后名还北京先生曾孙暹来见求为尊德堂制记盖先生在日所蓄前代典籍甚富暹父敬与收藏无遗于所居作堂崇奉之可谓知所尊者矣今予告老赐归则先生所作若文若诗及诸杂着悉汇次成集矣暹以予受知先生虑传写有所舛讹乞为校正且俾为之序予不敢辞因记先生存日所同游者若大章徐公仲谊朱公光弼张公景行莫公景元常公复初杨公等或以古文或以词章或以经学檀名惟先生兼工诸作而又愽学多闻自经史及诸子百家以至稗官小说靡不观览搜其秘而穷其奥采其华而摭其实故其见于制作诸体咸傋不以一善成名岂区区浅学所能窥其涯涘哉矧又有贤孙曾能宝惜而珍藏之以贻后世知吾郡之有名儒之若是也岂非盛德之所致欤而如予者亦得以姓名附于卷者又何其幸之至欤宣德五年岁在庚戌夏四月朔旦前国子助教兼修国史奉议大夫国府右长史郡人后学瞿佑序 

　　柘轩诗集钱唐贡士凌公彦翀所作也其诗诸体悉备庄敬而不亵和乐而不淫虽无刻苦丽密之工而平易典则皆则发秋情之自然有足观者夫古今体格之高下与音响之盛衰虽云由乎其人然尒随时以变赓歌之后一变而为雅颂之音其高不可尚矣再变而为蘓李曹刘之五言漠魏去古未远也故其言猷足以羽翼乎虞周之盛晋宋以来沈佺期宋之问之流其风韵则趋而下矣李唐再变而有五七言律诗绝句歌行之难其格度虽遭屡变之余而音响之盛如李白杜子美浑涵汪洋法兼今右盖卓然承乎千载之上者也他如王维高适岑参而下诸人之作皆号称盛唐之音尒沨沨乎其盛矣自是而后虽高下各因其时而欲追逐李杜且不可淂况虞周哉呜呼沧海竭而江河不能为之自盈日月明而山川不能为之自晦治平之世而欲其音之不盛可乎凌公涵濡　国家道化日久而培植为有其本故其发而为诗宜其为治世之音也欤使其得以揖让进退乎李杜高岑之间宜不必在弟子之列也夫古今之诗虽其盛衰高下之不齐然皆足以脍炙当时而传诵于后世者若公之作吾知其传于后也审矣诗总若于篇某孙宗载集于成帙凡若干卷以予有亲交之好间属予序之予唯不识凌公尝闻诸孔子曰有德者必有言观公之诗知其为有德之君子也是为序
