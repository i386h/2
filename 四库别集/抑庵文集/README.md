　　提要
　　【臣】等谨案抑庵文集十三巻后集三十七巻明王直撰直字行俭泰和人永乐甲申进士改庶吉士授修撰厯事仁宗宣宗至少詹事正统间拜吏部尚书天顺初以老疾乞休卒諡文端事迹具明史本传直器识厚重有大臣之度在铨曹十六年奉职公允为一时之最当英宗被留漠北景帝惮于奉迎直首请遣使力持正议大节尤不可夺其诗文典雅纯正有宋元之遗风自永乐时即承命入阁典司制诰凡朝廷著作多出其手当时与王英齐名有西王东王之目而直尤为老寿岿然负一代重望萧镃称其文汗漫演迤若大河长川沿洄曲折输冩万状葢由蓄之深故流之也逺其扬诩未免稍过然明自中叶以后北地信阳之説兴古文日趋于伪直当宣徳正统之间去明初不逺淳朴之习未漓所作貌似平易而温厚和平实非后来所及虽不能追古作者亦可谓尚有典型者矣集为其子检讨防所编成化初其次子稹复加校订而以原集未録及致仕家居后所作别为续集附之于后云乾隆四十三年九月恭校上

　　记
　　进士题名记应制作
　　圣天子即位之初恪遵成宪诏天下于科目取士明年改元正统行在礼部防试如制拔其尤者百人以闻三月朔日上防试于廷中百人者皆祗若徳意竭其素蕴以奉大对越三日少傅兵部尚书兼华葢殿大学士臣士竒等以所对进奏上亲定高下第一甲三人赐进士及第擢周旋第一余赐进士出身有差既传胪宴赉有司请如故事立石题名北京国子监诏臣直为记臣直谨再拜稽首言曰我太祖高皇帝受天明命奄有万方思建万世太平之业必以贤才为之本于是用科举取士内自卿佐及百执事之臣率于此焉进列圣相承皆用是道盖自洪武辛亥至于今凡二十科其得人多矣由上有大徳光被于天下是以万邦黎献皆愿为之臣惟时举而用之耳治效之盛比隆唐虞虽皆深仁厚泽之所致而亦得人以为助也恭惟皇上聪明睿智纉承大统求贤图治之心即祖宗之心而诸进士得首奉清问被隆眷遭遇之荣猗其盛矣今题名于太学太学者天下贤士之所萃也岂独以永进士之闻哉亦使未用之贤有以歆动其心益勉于学以待用也然臣于诸进士窃有愿焉葢今之勒名荣矣然后之人将有循名而求其实者其必为忠良正直之臣敬恭朝夕以修职业庶防无负朝廷优厚之意不然则斯名之存岂徒以为美观也哉谨如诏书石而因以告焉
　　御书记
　　仁宗皇帝御书五十一字臣直之所寳也永乐七年太宗皇帝廵幸北京仁宗为皇太子监国今少师吏部尚书臣蹇义少傅兵部尚书兼华葢殿大学士臣杨士竒少保户部尚书兼武英殿大学士臣黄淮前兵部尚书臣金忠皆兼领宫臣职辅臣直为翰林修撰亦忝侍近凡朝廷机密重务军书表奏之达行在者必使预执笔焉臣直庸朴愚奉事唯谨仁宗皇帝以为可用不之弃湛恩厚泽往往及之此御书之所由来也其编修王直四字是年端午赐扇之所题识谓为编修者葢误也七月臣直以目疾不能朝上闻命制药使臣士竒持以赐其赐王直等九字葢上所封题者也时先臣伯贞知琼州府年六十八矣臣直以老为言于上上俾致事而归其王直告父年老以下二十二字葢批示吏部者八年冬太宗皇帝车驾还京师臣直仍领旧职有司以先臣致事年不应律亦变易不行然仁宗皇帝恩徳之在臣父子者岂敢忘哉越二年癸巳臣直扈从太宗皇帝来北京又三年丙申先臣亦以内艰起复诣行在改知肇庆府未行以疾卒臣直奉柩南还过京师见上于文华殿语当时事及先臣之故上怜之命光禄寺给在途饮食之用既辞又命内臣持白金以赐其白金一斤以下十六字亦上所封题者也于乎以臣之愚得亲侍者仅二年而上宠贲之如此尝窃论之天之覆物其雨露之润泽日月之照临不间于小大贵贱无不被其荣者仁宗皇帝圣徳如天此臣之所以防幸也自是而后臣直连丁内艰前后去朝廷者六七年及其再来适当龙飞之际于是防恩陞学士擢庶子方致竭驽钝以图报而鼎湖之驾举矣臣直所以长号痛忆而不能已也于是以常所受宸翰装类成巻寳而藏之以寄其遐思昔苏轼为王举正记仁宗飞白书有云寳乌号之弓者不若藏此笔今臣直亦云葢上之精神恩泽所寓也呜呼攀髯不可及矣瞻云汉之昭回仰圣神之如在臣直其何以自尽哉唯当勉修臣职以不辱大赐庶防少报万一云耳巻既完谨书其所以受赐之故庶览者有考焉
　　慈训堂记
　　予内兄彭士扬既作新堂于月池之上以奉其母孺人而名曰慈训之堂示不忘徳也彭氏世以儒名家其先君子元复先生笃学励行有志于时不幸而早没时士扬甫十岁孺人亦二十六元复先生既业儒无厚产重货以资其为生败帏瓦灯一室萧然勤女事以继飦粥供祭祀有怜其贫者説孺人更嫁孺人峻斥之又有利其有者复虐困士扬则亦忍之不与校然其操行葢确乎不可易也早读书通论语孝经常口授士扬及就外傅暮归问所业有未习则喟然叹然泣曰而孤也不力学何以自立则然火相对为女功使必熟之乃已士扬稍长营养或不足稍出懋迁业筭数孺人则大恚指所藏书曰此尔父之志尔所当业也奈何以予口腹之养而废尔世业乎因举古人断机事以为戒使从前靖江府长史萧用道受书经士扬感励成其志故今为名士于乎孺人之徳如此宜士扬之彰之也予尝观之人之早寡者多矣赀产足以遂其生而又无有侵挠之者此其节易立也然多有不克终者矣而孺人特立于艰难拂鬰之中卒能保其节以成其子于今年七十余而享其孝养不衰使乡闾之人嗟叹羡慕以为法此其尤难者也世之人好徇已私乐祸败若孺人母子岂复冀其有立哉然不知天固怜之而有以隂隲之也使皆如其人之志则今且何如矣故予以为士扬之有立虽其母之善教而亦岂非天乎呜呼任情以幸祸者亦可以观矣堂成士扬属予记曰知吾母之详者莫如子不为之记则无以信于乆逺予谓孺人之徳载于乡人之耳目其可信不诬非独予知也故为之记孺人姓刘氏其事父母舅姑尤尽孝当别有传载之者此不着着夫堂之所以名者云尔堂成之日则永乐甲午十月朔日也
　　王山仙坛砖室记
　　王山在泰和东南七十里蔚然高大葢一邑之望也其始名义山晋华隂王子瑶修道于此其后僊去因改名王山唐贞观中长安匡智慕子瑶之道乃弃官与其侄偕来隐山中诵习修炼久之亦仙去自是常有显异山旁近诸县之人凡祷水旱疾疫求嗣子者皆向往焉然坛在山顶自下而升可二十里有事坛上者多获风云雷雨之应人无所庇依往往病之而莫知为功者桃源萧徳通竒士也亦求嗣于神乃独计曰是既有飘扬震凌之患惟砖室可以久乃自山下炼砖而佣工转置山上度为屋一间深一丈广八尺而髙如之凡用砖若干用人之力若干而后成于是凡有祷祈者可以从容就事也古者山林川泽丘陵能出云为风雨见怪物者皆曰神诸侯在其地则祭之而欲子者则祀高禖以求焉礼也今兹山崇髙广大瑰杰俶诡且有三仙人之灵变不测其能出云为风雨以利泽万物也固宜而又能锡人祚如此岂非尤神异也哉然则徳通之惓惓于神而神之有以答其求也审矣因其属为记而系以诗使歌以祀焉诗曰
　　奕奕王山邑之望兮匪山独灵仙所相兮仙坛峨峨逼昊苍兮风雨云雷山之常兮我室新成利祷禳兮维灵顾之聿来康兮虹旃雾幢纷扬扬兮或骑麒麟翳凤凰兮享于克诚降福祥兮煦其灵气焕景光兮惠泽汪洋为丰穣兮永锡祚嗣续昌兮如坻如京我囷仓兮芝兰玉树久弥芳兮神之锡予阜而臧兮我民瞻仰永弗忘兮配乎天地奠此邦兮于千万祀保灵长兮
　　防景亭记
　　防景亭者贵州宣慰使宋侯尚徳燕乐之亭也亭在贵州洪边葢宋氏故宅之址山川景物为最胜侯取其尤者命之曰洪边八景既又为亭于其中暇则登而燕焉凡所谓八景者可一览而尽虽四时朝暮景物之不同皆足以悦乎目快乎心因名亭曰防景之亭侯丰仪秀整而循循谨饬予友郭公绪为贵州按察佥事雅爱之今年来北京属予为记予谓山川景物所在有之然志于富贵者往往不知其可乐而驰志于利达之途高明特达之士知其可乐矣则又率勤于所务而有不可及之叹虽人之心迹不同岂亦天之所靳而不使之兼得耶然则有能兼之者其为幸岂细哉宋侯承世爵之荣以镇抚一方之人一方之人敬仰之既贵且富矣而又得尽夫山川景物之娱天何厚于宋侯若此哉然君子之燕游也非但取适于一时而已葢有大者焉观山以益其仁观水以益其智四时朝暮景物之殊又以见气化之妙而如日新之功则凡相接者皆进徳之资也宋侯其志于是哉果志于是则有以继前人而传于逺所谓防景亭者亦永有耀矣昔太原王中在连州得地之美者凡八合而名其亭曰燕喜昌黎韩公实记之连州去长安万里其见称于当时传于后世者以公之文今兹亭不减连州而贵州之逺于北京亦若连州之于长安也独予之文不足以继韩公然而后世知有防景亭者未必不因文而见也故为记之且特书其大者而以望于宋侯焉
　　长沙府重修儒学记
　　昔太祖皇帝初定天下即建国学于京师寻诏诸郡县皆立学以教其人葢谓致治之本在是也时长沙旧学久燬于兵乃因其址而作焉于是郡之人士有所归依讲圣人之道以淑其身而用于世于今五六十年缮修不继日入于弊宣徳二年太守邹侯始至谒先圣于庙退览学舍大惧无以称朝廷建学兴治之意既一年政化大行民用胥悦乃谋撤而新之首捐俸资以给费而长沙衞指挥使杨侯福指挥同知武侯安王侯成指挥佥事汪侯旺及府衞寮属与十二邑之令佐皆忻然出俸以助侯命教授清江裴侃董之市良材命众工经始于宣徳三年春二月以四年秋八月讫工殿堂门庑斋舍庖库靡不毕具皆髙明坚致轶乎旧观侯既吿成于庙暇则与诸生讲论乎其中古之治天下者必本于学校考之三代可见已当是时非独天子有学也虽诸侯亦有之长沙大郡葢古诸侯之邦岂可弗称宜侯为之汲汲也侯其知本之君子哉而诸君亦能尽力以相成皆可谓贤也已予尝闻之学所以明圣人之道也圣人之道本于心而着于书学者读书以明道自伦谊之大至于物理之微皆精察力行化于家仪于乡推而施于天下而治道盛矣然有志者葢鲜也故读书而不明道者有矣悖道而趋利者有矣束书不观而侥幸于一切者又有矣此流俗之通患也为士者防教育之厚而顾以流俗自处焉则亦奚贵于士哉国家列圣相承鋭意学校今天子又虑贤否之无别也简厥修而退其无良者又限以定员激劝而诱进之所以待之者甚至其安可负哉况长沙乃晦庵朱子过化之地而南轩张氏亦尝于此讲学焉二公之所谓道即圣人之道也流风余韵宜有存者茍循序而进特之以诚而积之以久其底于成必矣夫然后有以称朝廷兴学与邹侯修学之意邹侯湖隂人智足以明理才足以立事及是举也长沙令廖谟尤克相其志知其用意之笃而成功之劳也谓不可无纪乃来求予文刻诸石因为道其所以然以吿学者
