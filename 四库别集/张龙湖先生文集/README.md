　　○彭思眷 

　　杨子云言□宁言败俗故立言者必衷诸道夫六经道之所汇也立言而不知宗经则终末由庶几于道近代以文章鸣者众矣而律以道之离合盖廑□力有存者此无他经术疏也吾里自宋元迄明人才迭出类由经学显而张龙湖先生尤以命世经儒回文章之澜于既倒者也先生幼孤窘崎岖负米之余抱守六籍单厥心于微言大义其于道也盖用功深而培养邃故其所为文章根柢盘深枝条峻茂无论巨篇杂着罔不综理道之要揭忠孝之恒树劝戒之义而纲纪于修身治世之程者夫胜国之文至中叶而衰征兆矣猥庸者囿于疲薾黯黮之习而聪明奇杰之士又率为荡谲嵬琐之态以相矜诩非有魁儒巨公倡明经学以救其敝而力挽其流则靡靡之音狂惑竞煽者不待至于隆万之际矣先生于时以身为之导其总持艺苑呼往发蒙必折诸圣历主文柄多崇奖经术士如归震川薛方山诸先辈皆出其门而震川被指授尤频数故生平知己之感独于先生三致意焉然则先生之文非惟自寿于世又能使经学逓广其传起衰之功宁逊前哲哉且先生之足垂不朽者非徒立言而已余闻之故老先生未达时慨然欲行所学以大其设施及冠南宫入翰林以至典南铨陡政府翻颦蹙不乐当食辄欷歔初与张桂龃龉有托以归迨后世庙为方士权辅所蔽屡効补牍而迄无所济熏莸同器竟郁悒困悴以至于疾疾革矣于时事犹惓惓焉凡此孤忠亮节不为小人之匈匈也而辍行洵无媿于古大臣正己格主之道者而媢嫉之臣轶其事于身后当时纪载家亦莫敢昌言称述故世苐传其自劾及筹边定会典数事其匡正于寮友之间启沃于辅扆之侧者仅仅流传于乡里故老之口而已可胜叹哉然先生之可垂不朽者固与江河俱流也今读其遗文浑厚深醇古雅森峭有一或诡于道者乎则先生穷经之效更可因其言以想见其为人也已是集也余先曾叔祖治中公为先生壻于其没后镂板以传岁久漫阙翻本杂淆甲辰秋儿子新校士两浙爰授以家藏原本及亲友所藏诸本俾于吴越藏书家勘对讹误挍正篇次重梓行世以还先治中公之旧且使通经学古者知所劝焉雍正四年丙午秋八月上浣同里后学石陇彭思眷鹤田氏识于莱浦山房 

　　○雷礼 

　　旧序 

　　明兴自设立内阁以来代有名哲以文章功业相宣者在楚则有李文正文正为茶陵人由天顺甲申入翰林至弘治乙卯入相孝宗以底成绩说者谓其关天运锺地灵不徒以文章致身也非偶然矣及文正物故未逾四载龙湖翁又崛起于茶陵举南宫第一值驾南狩廷试不果躬逢皇上龙飞江汉应运中兴首赐进士出身选为庶吉士第一俾读中秘书以钧陶之非贞元之会默有所期而然欤何天时人事若是其相成不爽也翁自辛巳入翰林既与文正之甲申同运而志孝经济不安于萎儒掇华袭馨以窃章句凡军旅刑狱钱谷水利之数与夫边隘夷险地图修阻户口登耗无不讲求其法而欲通之以托于世既而列宫采掌艺苑柄两京铨政校文武士铢铢两两不淆毫厘论人才正若邪忠若佞义色直言凛莫敢犯至所崇奖名节士又如恐不及故海内缙绅相与属望于翁者久而孚矣晚年简自当宁于己酉之春命典机密距文正不逾一甲子而公亦慨然以天下为己任与一二元老奉法顺流斡旋元气必欲朝政清明善类布列以培丕丕之基视文正与刘谢一心共赞弘治之绩先后相炳耀也而天运地灵不于是乎有明征哉然文正功虽着于弘治至正德初季劫盖岌岌矣翁尝谓其有遗憾今皇上以上圣之资而干刚独运同符孝宗殆非正德时事比使假之岁月久于其位充翁之言必不以文正自限细旃广厦从容启沃自有以深中圣心而熙鸿号于无穷乃不克竟志以没则翁之自憾宁不甚于憾文正耶是何生之不偶而久道之化不胥赖以成耶尝观邵二泉叙文正怀麓堂藁谓其泽被海内如大将御戎不闻号令是不专以其文而诵其文则知文正所持以不朽者固自在也翁虽不可作矣然片言只字流布于人者类系国家大体而方正严毅巍然有壁立千仞之气则翁之自期待而未竟者不可槩其一二耶礼忝翁知爱尝欲勒成一代大典委责于礼而不获执笔随翁后盖有余憾焉因与翁婿治中彭宣拾遗藁序而存之使求翁于不朽者观其大焉可也嘉靖癸丑季春吉旦后学丰城雷礼撰 

　　○薛应旗 

　　旧序 

　　夫道以文显而亦以文而晦文以人传亦以人而冺古今之为文者多矣上下千百年间号称名家者亦无虑数十辈而其遂传不冺者非其人品卓越必其言几于道者也然而讥评指摘者则亦既有所不免矣其诸于道无所发明人不足为世轻重铅椠甫释而影响罔闻竭一坐之精力而曾不足以与春花朝露比荣者又岂可以一二数哉吾师龙湖张先生挺生南楚径趋高峻旗初以文受知于场屋既以吏事事先生于南部见先生直躬正气侃然不回而好善嫉恶无少假借盖信道之笃而果于自任者也故其为文皆根诸性情理道而光明俊伟一洗菁藻浮华之习此其志盖欲上窥邃古而不屑以后世之文自命矣既先生登内阁而余为南官司属先生数以文示而虚心咨访谓当无隐旗谓先生上下今古博极羣籍固于书无所不读而左国戴管恒有所嗜惟是文之风容色泽时或有于四家之书大都相类先生乃抓髯长笑曰而谓吾文尚有蹊径耶规矩者方圆之所就也律吕者声音之所谐也成器以利天下导和以宣化机固有物焉以行乎其间矣行将甘苦俱忘疾徐融释陶钧造物大音希声再当示子未几而先生疾革不起余亦以视学去兹余再至京师先生馆甥彭君宣上追李汉故事刻先生之文以传属余序诸末简余惟全楚山川古称贤薮迨我明兴益宏以肆在文庙则有杨文定夏忠靖在孝庙则李文正刘忠宣皆以文章饰政昭著中外其在于今与先生同时者则有石首蒲圻沔阳黄冈随州各以才名艺林竞驰湘汉载笔摛辞咸有纪述兹取先生之文熟复之正伦轨物经世范俗罔非载道之言而沉涵挥霍将兼体用而益大之虽天不憗遗而未酬之志盖有足征实于前四公有耿光焉而同时诸公固未能或之先也其为显道传世之文也又何疑乎嘉靖甲寅冬十二月门人武进薛应旗谨序 

　　○陈栢 

　　旧后序 

　　栢自髫年即闻龙湖先生者翩翩起湘中矣后游京师留滞于辟雍相与朝夕而私心遂向往之己酉先生由南宰入典机密毅然以治安为己任而宇内欣欣然想见其太平明年校文礼闱先生素厌文体与时俗相卑弱雅意复古独以欧阳文忠自许而因子瞻器余也居未数月而病且已矣栢峕学政屯部待奉命往营葬焉与其子壻治中彭君欲收遗文刻之今甲寅刻成因属余序其后且曰是子之志也海内知己非子谁宜序焉序曰夫文章之道不独时与政关之而其人可类知也三代而下条词之士众矣曷尝不效法坟典拟格于风雅哉然徒艺文尔已惟禹皋诸谟世称绝德烝民之什千载诵焉盖精一执中之学相与都俞而物则之懿山甫辈所克举也是故其根之而为德建之而为功敷之而为言其致一也虽其时太和之气使之善鸣而诸人者所陈说咏歌于世道有裨焉故得之破壁断简至今训为经也明兴承胡元之季学士之文多伏于草野惟我成祖加意表章乃稍稍着见然委靡俳俪之习至李文正始洗濯之文正余楚产也值成化弘治之时海内又安得从容中秘尽肆力于词翰故所传甚广卒能以文名也迨于今而龙湖先生出焉先生生于衡岳之麓天性早慧少负时望会今上中兴即应期以起几三十年乃入而论道凡所历翰院宫坊礼闱铨部其培养君德节宣化理图度政务铺张礼乐进退人才莫非至文至所陈说咏歌者则绪论也况其风节凛凛深系天下之望而一时人士析六经之疑考历代之遗表当世之典靡不于先生折衷焉凡有所作属兴而就辄伸笔满纸不为婉眤细语崒乎如祝融浩乎如洞庭不尚藻饰比拟而终不可及观其自考一疏烈烈皆忠孝语而格心之妙隐然于天保之颂谓非其有得于精一物则之学者不能夫岂词章之士所可及哉余独憾夫秉政未几世莫能究其用倘假之年优游经幄多所献替必能如夔皋山甫诸人都俞讽咏庶几哉称虞周矣然先生之志虽锐意启沃而其气与节卒不肯少贬以徇时好即令久于其位或亦莫之能究也然其所未究者其绪论犹幸有兹集焉百世之下诵而读之当自知其所不朽也嘉靖甲寅夏五月朔赐进士出身兵部职方司主事汉阳门人陈栢撰
