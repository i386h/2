　　提要　　　　　　　　别集类三【宋】
　　臞轩集十六巻
　　臣等谨案臞轩集十六巻宋王迈撰迈字实之兴化军仙游人嘉定十年进士调南外睦宗院教授召试学士院改通判漳州应诏直言为台官所劾削二秩淳祐中知邵武军予祠卒赠司农少卿事迹具宋史本传然考周宻癸辛杂识有载迈为正字时事而本传不言其为此官则史文亦有所缺畧也迈所着文集宋史艺文志不着録惟明钱溥秘阁书目载有臞轩集七册王圻续文献通考亦有臞轩集二十巻是明代尚有传本今世所存祗臞轩四六一巻皆启劄骈偶之作盖即从集中抄出别行偶然独存者也今以永乐大典所载兼采他书所引附益之共得文一百七十一首诗四百四十三首诗余五首厘为一十六巻计其篇目约畧得十之七八矣迈少负才名而史尤称其练逹世务盖非徒欲以词藻见长者考其初以殿试第四人出佐长沙幕刘克庄作诗送之有防好人争诵名髙士责全之句见于后村集中是当对防时已有伉直之目厥后歴官所上封事类多区别邪正剖晰时之言如諌乔行简再相及禋祀雷雨应诏诸篇敷陈皆极剀切其于济王竑事反覆规劝更见拳拳忠爱之心癸辛杂识称迈因轮对追论史弥逺擅权词气过戆帝以狂生目之迈后归里遂自称勅赐狂生其事为本传所未载亦足以见其气节今集中诸疏并存尚可考见一二集中诗文亦多昌明俊伟类其为人读者因其言而论其行固不徒取文辞之工矣乾隆四十六年九月
