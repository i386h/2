　　欧阳行周文集　　　　　别集类一【唐】
　　提要
　　【臣】等谨案欧阳行周文集十巻唐欧阳詹撰詹字行周泉州人举进士官至四门助教事迹具新唐书文萟传其集有大中六年李贻孙序称韩侍郎愈李校书观洎君并数百嵗杰出今观詹之文与李观相上下去愈逺甚葢此三人同年举进士皆出陆防之门并有名声其优劣未经论定故贻孙之言如此然詹之文实有古格在当时纂组排偶者上韩愈为欧阳生哀辞称许詹甚至亦非过情也王士祯池北偶谈尝摘其自明诚论尹喜自明诚而长生公孙宏自明诚而为卿张子房自明诚而辅刘公孙鞅自明诚而佐嬴诸句以为离经畔道其説信然然宋儒未出以前学者论多驳杂难以尽纠亦存而不论可矣愈称闽举进士自詹始王应麟谓黄璞闽川名士传詹之前已有薛令之林藻攷之登科记璞所传诚然愈言为误葢古事可资典籍近事惟据见闻沈括沈辽吕祖谦皆一代名流而宋人或误称其世系则愈之失攷固亦不足怪矣乾隆四十三年六月恭校上

　　欧阳行周文集原序
　　欧阳君生于闽之里防为儿孩时即不与众童亲狎行止多自处年十许嵗里中无爱者每见河滨山畔有片景可采心独娯之常执巻一编忘归于其间逮风月清晖或暮而尚留窅不能释不自知所由葢其性所多也未甚识文字随人而问章句忽有一言契于心移日自得长吟髙啸不知其所止也父母不识其志每尝谓里人曰此男子未知其指何如要恐不为汩没之饥氓也未知为吉凶邪乡人有览事多而熟于闻见者皆贺之曰此若家之寳也奈何虑之过欤自此遂日日知书伏圣人之教慕恺悌之化达君臣父子之节忠孝之际唯恐不及操笔属词其言秀而多思率人所未言者君道之容易由是振发于乡里之间建中贞元时文词崛兴遂大振耀欧闽之乡不知有他人也防故相常衮来为福之观察使有文章髙名又性颇嗜诱进后生推拔于寒素中唯恐不及至之日比君为芝英每有一作屡加赏进游娯燕飨必召同席君加以谦德动不逾节常公之知日又加深矣君之声渐腾于江淮已达于京师矣时人谓常公能识真寻而陆相贽知贡举搜罗天下文章得士之盛前无伦比故君名在榜中常与君同道而相上下者有韩侍郎愈李校书观洎君并数百嵗杰出人到于今伏之君之文新无所袭才未尝困精于理故言多周详切于情故叙事重复宜其司当代文柄以变风雅一命而卒天其絶邪君于贻孙言旧故之分于外氏为一家故其属文之内名为予伯舅所著者有南阳孝子传有韩城县尉防壁记有与郑居方书皆可徴于集故予冲幼之嵗即拜君于外家之门太和中予为福建团练副使日其子槚自南安抵福州进君之旧文共十编首尾凡若干首泣拜请序予诺其命矣而词竟未就槚微有文又早死大中六年予又为观察使令访其裔因获其孙曰澥不可使欧阳氏之文遂絶其所传也为题其序亦以卒后嗣之愿君名詹行周其字云福建等州都团练观察处置等使正议大夫使持节都督福州诸军事福州刺史兼御史中丞上柱国赐紫金鱼袋李贻孙撰

　　欧阳行周文集巻一　　　唐　欧阳詹　撰赋
　　出门赋
　　岀门辞家也人有志而斯逞予纷然而逺游别天性之至慈去人情之好仇严训诫予以勿久指蒲栁以伤秋弱室咨予以遄归目女萝而起愁心眷眷以纒绵泪浪浪而共流惕怀安以败名曾何可以少留于是驱忠信以为车执艺业以为贽越三江逾五岭望尧旌而求试庶亦呈功取爵建德扬名获甘旨而报勤光昼锦以回衡如弧斯张如鸟斯征射百步而期中飞三年而必鸣飃天寒峥嵘嵗晚鹊聨翩以不定蓬悠扬而自转逮前程之尚遥顾所离而日逺事纷挐以争防情交戾而不和退藩篱则弱羽恋于云路激龙门则纤鳞限乎尺波身违日日之晨昏恋凄凄而莫遣亲益年年之羸老思摇摇而若何慜灵輙于困穷举冀缺于垄亩一仁声之永大一孝徳之兹久伊锡类以拯穷岂今无而昔有尔乃循否泰以俟命黙风尘以怆艰苟疏溟以纳流愿覆篑以成山路实多岐丝无定色任黄之濡染信疆理之南北管因媒而解缚越自遇而升车虞先荣而后悴姜始巻而终舒伤哉数子之税驾吾未知其所如
　　石韫玉赋【清润积中光华外发】
　　荆山之石兮玉在其中和氏未异兮追师不攻内抱贞明蓄珪璋而自异外封硗确与砮砺而攸同纷尔千峯块然一石石居山而有纇玉处胎而无迹昭彰奇彩象鸾镜之犹埋特达英姿状蠙珠之未拆齐草木之偕贱叠泥沙而共积瓌材则韫精气时扬结白虹于林薄浮清气于岩冈多见已形空知六瑞之贵罕穷未朕谁分十仞之光混瘗嵌岩沈蒙翳荟同夫有智怀其有以若无侔彼不争守厥屯而俟泰明其内晦其外将藏器以待知不干物以招害原夫石则称坚而可转玉则受琢而凝清日遇良工一则有顺而无固时惟哲后一则无胫而前呈我唐文武建元成康绍获王母之玉琯致淮夷之琛賮向华池而効色从温树以流润伊抱璞之未闻亦梯山之自进佳粱糅粃黄金在沙必簸糠而飏砾冀取实以除华雕琢傥行辉章希发愿同三献之纳庶免再来之刖
　　回鸾赋
　　夫何防一人兮将凝帖乎万方神其精而杰其质兮赫赫魏魏以昻昻应千年之寳歴承八圣之重光道为纪德为纲仁为宅义为防化悠悠而荡荡风习习以洋洋沐雨露以蕃昌烛日月以偕康癸亥之嵗大皥司政乃作幸于西顺上帝之令将行曰相彼元元以哲后为父母视淫君犹刍狗予其在德则庶民皆予之子也伊重闗击柝虞谁而守乎予其不淑则骨肉实予雠敌也虽金城汤池于予何有乎四门大开七寝停惊飏凛凓以风清寂澄凝而月静于时厥有顽民从愚至逆假鸿恩以出入弄神器于间隙于是天忿地怒人惨神恻积愤气以交冲叠寃心以潜逼灾变流演妖氛充塞山河列以长晦日月在而无色明则士庶幽则灵只豚鱼有识草木无知企喁喁以嗷嗷望我后之来仪如孺子之忆慈亲焉如涸鳞之念长津焉如枯苗之待膏雨焉如笼鸟之仰林莽焉既而文物无荒声明有素木叶犹飞金风未暮圣泽西浃天顔东顾廻旂旗整銮辂雨师启途风伯前驱丰隆布令烈缺行诛神功莫仇天力谁虞栉缤纷于瀴溟骈骆驿乎虚无洗地轴拂天衢殱有罪福无辜雰霾扫荡于寰区尘埃涤濯乎皇都元凶不戮而沦痛品物未觐而遥苏尔其灵物既光乗舆乃从云车烟驭春心日容郁霏霏以葳蕤辉熠熠以严颙祥风以淫淫瑞色靉靉而溶溶蒙笼焉虹霓之萦仪鳯髣髴焉江雾之送游龙若夷若夏乃愚乃贤振振骈骈殷殷阗阗巷如流以汤汤野若草而芊芊云浮巨岳水集洪川至喜翻悲含泪而前曰自沐化冥冥绵绵如载于天如饮于泉卒嵗永年皆调自然异日殷忧方昭厥由归欤归欤人待其居是日也皇帝乃辟金门升紫宫宣睿旨奬天衷熈乎若微雷淑气畅昆虫芬乎若韶光丽景发青葱下蟠厚地上洽穹扶桑而西虞泉而东百福交通万彚大同白日三舍以逡巡瞪不征以融融南山万岁以争呼颔若动而崇崇至矣哉之德也深乎大乎滂乎沛乎尔其汪濊乎可谓上合天经下叶神灵旁统神明中获人情故能不守有与之守不争有与之争此一举也足见天地之心足辨人神之意谅无党以无偏唯夫道德之比者也
　　将归赋
　　忆求名于薄艺曾十稔以别离才还乡之半龄又三年于路歧红顔匪长白日如驰苒苒皆尽悠悠为谁亲有父母情有闺闱居唯苦饥行加相思加相思兮宁苦饥辞家千里心与偕归南省之兰北山之薇一芳一菲何是何非归去来兮秋露霑衣
　　王者宜日中赋【以题为韵】
　　杲杲者日中则重光烛生生于有晦暖物物以无疆人在下君体阳故法之象之宜诸帝王亭然止六合居中赫矣洞九霄临下取其正谅无邪僻之徒仿其明讵有幽隂之者瞻端严则体率慕光昭以心冩想照弥乎八纮庶化覃于九野观夫髙舂始上虞泉未移面方舆而再朗防圆昊以重规有隼之墉匿寸隂而影尽无禽之井透百尺以光披含灵亹亹处植离离宂有向阳之戸柯无不煦之枝伊体元在已有国于斯形厥功以居大固其义之攸宜况日则类王王实况日道符罔二理契惟一当其食昃用贬膳以去县仰以髙明直弃俦而背匹是以如之致唐尧之盛假之在周易之丰暮非乏照朝亦有融或背篱东之蛰或遗山右之蒙掩彼不言黜履邪而有蔽放兹取则贵无偏以处中正不正通难通可以励垂衣之圣政可以激御逺之神聪者也我皇祗若髙穹保兹洪祚顺三辰之耿耀稽八卦之明谕觌兢庄霑煦妪美盛徳之形容遂屏营而作赋
　　瑾瑜匿瑕赋【物无终美舍短从长】
　　玉之美者其曰瑾瑜虽特达之自有岂玼瑕之则无菲食其端葑采其下苟当无而可用谅在人而罔舍况服饰所珍礼容攸假五德人尚居然总之百寳物雄又其尤者只如夷吾委质曲逆从王一则措其所短一则举其所长伊十仞之可贵讵一眚而为伤是故异比荆山竒同郑市纵青蝇之下防有白虹以旁起琢中良工佩宜君子尔若恶其细而弃其大我则扬其表而掩其里矧乃珪或致磨璧当可指终詶九年之积不损连城之美朂矣乎韫独明之见宰万物之工覩其材而辨其器履其始而知其终建莫大之勲与攸攸既异收稀代之寳将琐琐宁同砮璞自充散材徒郁苟无分寸之痕翳罔有寻尺之盘屈莹乏光华绂乖音律攻之有旷乎日力斵之不益于人物空知有玷与无疣岂不道疏而理拂至刚也必时时而外缺至清也乍浑浑而罔容考瑾瑜之含匿亦厥义之云从不然者黄已疲奚复骋乎千里轮囷则病焉得用于九重瑜之体全者则稀瑾之无瑕者亦罕惟追师之鉴选纳尺长而寸短
　　徴君洪涯子图赋【云际长枕以表贞节】
　　矫矫徴君居幽行闻朗咏尧年之日栖迟姑射之云英英时杰好竒艺絶规穷图绘之能写得隐沦之哲岂不以怀材习技我蕴跨俗之工履道全真彼有过人之节者也观夫杖藜载酒面石依松尽是山中之意全移物外之踪入室终窥知裂缯而画出升堂始睇疑在野而相逢实黙如言如行实止萝纎纎以垂防草青青而借履洋乎令闻昭晣得其所由俨夫仪形髣髴知其所似原夫贤达作范丹青立程将模前而示后必体物而归诚服惟身表容实心旌对冰雪之顔覩兰蕙之缨暗识伯夷之防遥怜虞仲之贞知身已识看画如生矜且复庄若北辰之有识贪之与欲同在日之无情形似植以亭亭衣如风而曳曳临诸瑶席之上想彼云林之际万物方秀千峯初霁神飘飘而自逺身悠悠而不系我之心矣惟贤允臧披图画于是日得夫君乎此堂乃知君之于德也大画之于工也长画非君无以展其妙君非画莫得扬其光物有相假不其昭彰揆人事之美恶论功庸之伙少伊画也可以称智者之兼惟君也可以作真人之表者也
