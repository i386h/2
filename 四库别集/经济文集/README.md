　　上中书丞相书【为右司省掾时办济宁省官作】

　　盖闻赏罚者国家之大柄是非者人心之至公赏罚无章虽尧舜不能以为治况后世乎是非混殽虽智者不能以有为况庸人乎我国家自太祖皇帝肇基朔方世祖皇帝奄有区夏皆未始有尺地专城之藉一旦乘斯民云霓之望乃能挥霍八极席卷中原一举而蹙残金再举而踬南宋一时将相大臣又能丕阐皇猷聿宣至化遂使海内之人争効臣妾豪杰之士奔走慕义翕然如云龙风虎之相从者此无他赏罚明而是非公故也承平以来百年于兹礼乐教化日益不明纪纲法度日益废弛上下之间玩岁愒日率以为常恬不为怪一旦盗贼猝起茫若无措总兵者惟事虚声秉钧者务存姑息其失律丧师者未闻显戮一人玩兵养冦者未闻明诛一将是以不数年间使中原云扰海内鼎沸山东河北莽为丘墟千里王畿举皆骚动而终未见尺寸之效者此无他赏罚不明而是非不公故也近者天子以两丞相为元老旧臣命总万机今日之事政宜开诚布公振作纲纪明示赏罚号令天下使六军之众鼓舞踊跃忠义之气振彩扬精一新天下之耳目以尽扫前日之宿弊庶几盗贼可平四海可定中兴之事业可计日而待也今巨冦在迩败军满前其雷霆之威邈乎无闻而姑息之风愈胜前日但闻今日取一招明日送刑部切为阁下惑焉今姑以一节近事言之如哈喇巴图尔等所犯最为容易处置者乃至迁延到今未有定论中外之人谓圣君贤相作事犹且若此安得不觖望乎传曰赏不踰时罚必当罪斯言是矣何则盖哈喇巴图尔官虽知院其职则总兵也右丞阿克塔齐叅政贾惟贞名虽省官其职则供给也在律主将所犯冝处重刑供给所招应得杖流今哈喇巴图尔虽有特敕免死之文不过圣上寛仁少延其死耳然法者祖宗之所授天下之名器也虽天子不能以自私是故舜为天子皋陶为士瞽瞍杀人彼为舜者惟知窃负而逃耳汉薄昭太后之母弟也昭有罪文帝终不忍贷之以死焉夫虞舜大圣人也汉文贤主也而其所为若是诚以恩不足以掩义私不可以废公古者制勑一下事有弗顺于人而碍于理者在百官有司犹且三复还奏不即奉诏未闻宰相秉国之钧职当绳愆纠缪而徒以承颜顺旨为事者也必不得已其总兵官既已特勅免死则分省官正应坐以本罪如此则轻重纔适其均后世犹惧为公道之玷安可使冠履倒置轻重失伦以启将来纷然之议乎况济宁密迩贼垒窃恐此声一闻将必使贼人有窥测朝廷之浅深以懈惰诸军之心者诚非国家之福也丞相聪明絶人识见髙逺则此三人之罪宜公是非以厌天下之望不宜依违两端以滋人心之惑此所谓关治道之大者宰相之事也若夫规规于簿书之末琐琐于钱榖之务而昧夫经国之要不图为长久之计者此非仆之所敢知也仆一介草茅滥叨掾属顾惟职在簿书叨司法守固不敢偷容取合以卖法误国亦不敢畏避诡随以陷阁下于不义此不忠之甚者非曰弗能寔所弗忍伏惟阁下亮之进退惟命干冐威严不胜战栗之至

　　上中书总兵书【为刑部主事时军前作】

　　夫用兵之要贵于察敌情量彼我审缓急之宜而用之不务出此虽有管葛之智韩白之能仆未见其可也请姑以目前之势论之今贼人之形出没难知我军覊縻动为所制此政兵家所谓形人而我无形则我专而敌分我专为一敌分为十是以十攻其一也则我众而敌寡今我势反分贼反专我形易知敌形难测此所以彼势日张我力日屈而疲于奔命者也然贼前日攻陷济南徳陵之时使能举全齐之众乘破竹之势谕拙速之机捣我无备鼓噪长驱则胜败存亡未可知也彼乃计不出此方且图为固蒂自全之虑此贼之失计我军得志之时也释此不虑且复征调逺兵近救目前之急此贼前日之失计未必不为今日之得计也今日贼必度我援兵俱驻西南其直沽之兵虽设守备尚且巢垒未坚类皆无足畏者止谓真定以北涿州以南必有备御而又以努都尔噶太尉伊苏知院等军俱系败衂之兵志丧气馁虽散屯于御河单家桥等处其锋已挫皆无足恃也况又有陵州长芦青州之冦以缀其后不敢动摇此贼所以深知我之虚实我之形迹而但取京师东南一角水陆并进首尾相衔有直向长安之意其志岂小小哉所以东西二路多设疑兵多张声势以挠我救援之师此寔批亢捣虚之意也何则盖贼兵之进必分为三路其一路畏直沽分院在后虑有腹背受敌之患必先发海舶数百艘往来海口并永平等处以为东道之疑兵缀我直沽之军马则我军之左臂不敢动矣其一路畏今哈喇布哈副枢及平章之军马有横冲邀遏之患故今但出霸州之背直抵后卫右卫交横侵掠以为西道之疑兵缀我霸州涿州之军马则我军之右臂不敢动矣布置既定其大冦必取水路沿靖海而下以出直沽之右而达河西务以及通州取陆路自东安州而上以出青润之背而望京邑日使驱迫之民喧阗于前巨冦之众鼓噪于后比及诸道勤王之师至而京师旋募之兵出则大事去矣愚前所谓宜审缓急之势而用之者此也且朝廷但知平章之兵一出将大有所倚仗谓其必能舍缓就急随机应敌遂尔不遑逺畧傥或因循如此岂不深为悮事也哉况各处居民惊惶四散曽无定志王畿州县悉皆空城有司不才谁肯死守且邦家之本在于百姓王畿州县所以藩屏京师者也今百姓之流离如此州县之不完如此有司之不守又如此此诚危急存亡之秋不可不慎也仆窃为今日计其目有六一者平章冝綂本部军马于固安霸州上下权且深沟髙垒多设疑兵于青润后卫右卫左右昼夜广张旗帜烽火以疑贼心贼虽能深入我境然其心不能不懐疑惧我军当乘其窠穴未定急发精骑攻之少挫其鋭使贼无必向之志急调副枢哈喇布哈军马直趍青州捣贼不意则进有擒敌必克之势退无牵连掣肘之患矣二者调发诸道援兵地逺一时必不能到又丞相台哈布哈军马控制大河以北京师西南诸郡不为不重若大营一前则彰徳卫辉必为曹濮之冦所有是山东全齐之地不独失之于前而河北诸郡又复失之于后且海滨距河东西绵亘二千余里举齐鲁卫之地以抗燕赵岂不危哉宜急咨达朝廷大发府库并各投下所有珎玩财货就大都内外募选骁勇之士计旬日可得十余万再别选精鋭骑兵计见在可得五万使马步错进作三运而行一运姑存护京师二运姑存备非常三运急引当前敌仍分为二道且徐徐而发视缓急如何以图大举一道出自涿州向固安霸州以断贼之左肩一道出自通州向河西靖海以断贼之右肩其东西则连引直沽分院与平章副枢军马相为犄角则贼之前锋挫衂已甚而京师无虞矣三者冝急催请太尉伊苏知院收集各枝军马除拨军守御单家桥等处以捍陵州之贼其余精兵急令各官綂领直趍青州以助副枢之兵出贼腹背截贼归路使贼进无所前退无所据则已骄之贼兵不虞我师之冲击将如猛虎跳跌必有自毙之势兵法乘劳此其时也四者近郊之攻既宁则始可以举我全胜之师议讨济南益都之贼盖兵贵我为主而敌为客今主客之势既分便当行逺取近攻之策分调我之大军自陵州长芦青州河间而下以攻贼之北面次调左丞察罕特穆尔军马自黄河而下以攻贼之南面又次调国珎王瑄顾稷等军马自莒宻穆陵关而上以捣贼之东面又次调丞相台哈布哈杨孟等军马自济宁东昌而进以冲贼之西面分布既定速令各军分拣精鋭之兵立为二一居阵前一居营后在内则东起靖海西至东昌以备曹灞在外则南连淮海北接济宁以狥河北各军所驻之营星罗碁布使烽堠相连声势相接每于要害之地加谨廵视御众以寛练兵则严居阵前者则选精鋭以从庶能一以当十其余羸弱专事农作近畿州县守令之不才者以及丞尉佐贰之不堪任使者不拘于资格以用之使所在董率无或因循怠惰修城浚池与民死守耕桑课农毋失其时如此则粮无转输之劳军无匮乏之患邦本既宁藩屏亦固彼三齐之孱冦直犹釡中之鱼苟延残息耳夫何虞哉近日乃欲调左丞察罕特穆尔副枢哈喇布哈等兵来屯涿州捍御京师谓之耀兵则不武谓之勤王则不力窃观特穆尔布哈等僄疾有余而持重不足况逺来之兵不习水土人懐乡井各无奋心于事何济又百姓但闻逺兵一至驿骚惊动鸡犬不宁皆骇于耳目之所见闻传曰大军所在荆棘生焉宁可不虑欤五者朝廷急宜侧身修行以回天心振作纲纪以荅人望专并总兵以一军权彻去遥制以责成效汰冗官以给军食节浮费滥赏以助国用庶几天怒回于上人怨销于下士气鼔舞而人思自奋矣六者冝作急住罢京东南分司农衙门并入有司其一切槩有已种未垦系官田地或召诸色人民或募流移百姓或验口数或验丁力随宜取便拨付佃种已熟之地比依常例权且増倍征收未垦之地照依常例止令二十取一仍传谕此等百姓随宜就便团结非惟少固民心抑且有助国用即日三春将尽陇无人耕土无宿种失此不救则时不再来兴丧安危在兹一举凡此数事皆今日之急务所惧者姑见是害优游不断若能决而行之则不必上烦睿驾亲征而天下之势已可以逆料之矣横槊之暇特赐观览甚之转而上闻则愚者千虑必有一得阁下其毋以为嫌而畧之且阁下平日智术出于众人名声扬于中外而又博通前览练达时宜而况适丁此难负兹重寄亟宜虚已授人广揽英豪恢弘逺畧毋为嫌疑所惮毋为小节所拘识时观变鋭意决为将见指日可以建不世之功树不犮之业播休烈于歌声勒骏勲于金石岂独俾方叔召虎专美于前哉某日者叅侍掾属过蒙知奬今兹实当从事敢不剖心悉虑以答万一第愧病在枕席未获跪诵昨辱宠命倍切惊惶夫尊爵重禄所以网罗俊彦优礼有功无功冐授是谓贪荣切恐军士闻之必有离心解体者甚可惧也仆虽不才寔所不敢今将所赐札付一道谨同此纳上非敢为是矫情以干誉也亦宜也伏望钧慈俯垂矜亮宗社幸甚生民幸甚
