　　【臣】等谨案湛园集八巻
　　国朝姜宸英撰宸英号西溟慈谿人康熙丁丑进士官翰林院编修初自编其文为湛园未定稿秦松龄韩菼皆为之序此本为黄叔琳所重编凡古文八巻又湛园札记二巻宸英少习古文年七十始得第绩学勤苦用力颇深集中有与洪虞邻书论选两浙十家古文事谓两浙自洪永以来三百余年不过王子充宋景濂方希哲王阳明三四人其余谢方石茅鹿门徐文长等尚具体而未醇不应浙东西一水之间一时至十人之多不欲以身厠九人之列盖能不涉标榜之习以求一时之名者故其文闳肆雅健往往有北宋人意亦有以也是集前二巻皆应酬之作去取之间未必得宸英本意然梗槩亦略具于斯矣集末札记二巻据郑羽逵所作宸英小传本自单行今亦别着于録不入是集焉乾隆四十六年六月恭校上

　　湛园集序
　　余友慈谿姜子西溟负耿介之气介洁自将与世寡偶独好观古人书每比次其行事而论其是非成败得失必准诸理辞尽而意沛若有余其他所为文字俱不苟随人髙下近世作者未有能或过之者也西溟少精举子业屡踬有司愈不喜诡随弋获前年己有以其名上闻者会格于例旋报罢故余尝谓西溟嗜古近癖而不能与时文定其荣辱之数名达九重而不能与流軰争其一日之遇西溟闻之殊不以介怀也日就余言古今文字有一定之的虽铢毫分寸不可逾越若学者则务与年俱进与时俱变耳终其身无得止法也以是益发愤欲尽屏人事并力以从事此道会奉有纂修之命治装北上裒其前后着为一集而中所芟汰者不下十之三四集成将挈之以行余视其才力雄富而一规于法拟古作者分量恢恢有余地然犹自署为未定藁即其志可知矣无锡秦松龄序
　　湛园集序
　　余识西溟先生三十余年矣固未能尽知海内之贤豪黙数所及见以为不可及无如先生者而所如之穷穷且久久益自疆益不售亦无如先生盖三十年间人事之变化多矣姑勿论贵逰子弟挟其声势气力弋取功名意满以去即穷老失志覊孤侘傺不平之士屡踬乆困塲屋中晩乃终得一当以不负其豪于平生者比比也而先生乃独如故夫穷亦何病然至斯极矣先生负气至髙不肯浮湛俛仰岂亦有嫉而挤之者与廼輙相左如此可异也方徴博学鸿儒时廷臣得举所知余亟欲以先生荐院长叶文敏公约同署名会公宣入禁中待之两月及余独呈吏部己不及期矣睢州汤先生后主试浙中叹息语同事暗中好摸索勿误失姜君竟亦不能得也自是之后每榜发诸公无不以失先生为恨至乃名彻
　　圣主之知而防防之中卒或尼之与夫诵南山之句傲逆旅之中者其不遇更奇而深足悲也然先生一不以介意益肆力于诗古文辞挟其髙洁轶尘之骨韵而出入斟酌于古大家一句字之未安不轻出也久之自定其古文若干首犹名之曰未定稿问序于余余窃喟夫恃才睥睨意轻一切者多也衙官北面大儿老兵讵长风流徒资儇薄丁敬礼有云后世谁相知定吾文者王文宪尝出所作属任彦升改定曰后世谁知子定吾文文章于道未为不尊知之寸心俟诸来者岂易言定乎先生盖孤诣入微而用心益细也其意直追古作者上下惟恐有毫厘缺漏未满之处其取精可谓奢而亦己贪矣造物之所予不能两有而于才名尤靳焉成此亏彼其穷故宜即使遇亦未必不穷古之取上第为朝官而以文章自名者其颠顿狼狈岂少哉嗟夫穷达何足言正悔不读书耳如先生者足以乐而防老矣长洲韩菼序

　　【臣】闻巡狩着于虞周有五载十二嵗之别肆觐徧于方岳垂陈诗纳市价之文虽时有不同而举无非事要皆法天行之不息求民瘼于永宁为神圣惓惓之至意也然或千乗万骑之纷驰必致诸司供设之滋扰钦惟我皇上德光上下治洽平成文物声名丕观谟烈之盛干戈挞伐无殊缔造之奇至化翔被于无垠休徴协应于有象惟是淮扬下流为河漕要地徃者
　　廵逰至止上厪

　　躬亲阅视以今年二月癸夘吉辰恭奉
　　皇太后銮舆南幸兼以省观民俗上悦
　　慈心同黄姚东狩之期为青阳布令之始舟皆鱼贯岸絶骑行乃裹糇粮悉先储峙驿递无顿置之苦役夫丰扉屦之资是以所过郡邑桑柘之行列不乱市肆之阛阓寂如篱落声讙与櫂讴而偕沸旄倪目送随帆影以俱长然后
　　驻跸河干沿流上下
　　命司空以授方畧约河伯以就疏排经画必出于万全无容苟且而贻事后之悔隄防悉期于可久勿恤多费而烦再举之劳既喜行地有时犹恐民天未足
　　诏截漕粮先后二十余万石赈贷各被灾州县银米无数虑如此其周也徳如此其厚也于是吴越父老望幸益勤缙绅大夫连章上请
　　皇上俯念而怜之乃不辞逺渉曲狥其意
　　躬奉
　　皇太后渡江而南厯览山川只奉
　　色笑顾以沿途所见闾阎生聚视昔少殊谓有司抚字之未周则奨廉惩贪以肃官方蠲税停征以蘓农困裁征关之浮额旅集千航罢煮海之増输烟开万灶寛罣误赦灾以启人为善之途广生徒校骑射以辟人功名之路而且敬修前代之陵寝优赉百嵗之耆氓诏防朝宣温文轶于西汉封章夕发旰食过乎周文褒贤德以降
　　宸章尽是丽日辉云之翰墨对江山而发

　　圣学迈隆于千古
　　神谋卓絶于百王而又本之于至诚推之于大孝所以銮舆未届而童叟欢迎

　　恩膏于三月之中河淮既已底定齐鲁悉被沾濡自古帝王过化存神之妙诚莫盛于今日者也【臣】向因编纂之余谬应射之选恭荷
　　皇上念【臣】老生食俸有日防之俦軰置以殊科自愧衰庸报称无地谨制七言律体八章上祝
　　皇躬之仁夀敢窃比于扈从如亲觌乎光辉执巻多惭濡毫増悚
　　恭制荡平沙漠恺歌序
　　【臣】闻王者之师有征无战其不得己而用兵非为生民除患害万世贻治安则勿举也我
　　皇上文武圣神之徳度越前古自
　　御极以来削平三蘗内消藩镇尾大之虞开郡海疆外控扶桑万里之国
　　神枢阖辟出天潜地
　　庙算指画雷动颷驰固己遐至迩安中外禔福埀卜世无疆之祉矣厄鲁特噶尔丹者向防
　　国家覆庇厚恩不知感激反肆其爪距屡惩不悛皇上于是恻然思为民除之以为防不可长不一劳则不永逸也乃上稽于天内防于心遂下
　　亲征之诏令所司储峙既周分师三路
　　帝车中运祃牙之日祥风来应军行出境属国羣瞻自初发迄于
　　回銮首尾才八十日已絶大漠渡克勒河朔徧躏贼境至于拕诺山逼巴顔乌喇贼方惊我师从天而下喙奔不暇而
　　皇上所遣西路大师已分师逆击大破贼于昭木多地方矣夫亲屈万乗之尊逺临絶辔之野劳身损膳以为民攘患至仁也大计一定不挠众阻不避艰险平行大漠若从枕席上过师黄钺一麾所向披靡三月之间顿成扫荡至勇也若至选将授律决胜帷幄因机制变鬼神莫知此非天下之至明极圣不足以与于此【臣】草茅下贱防

　　顾问及【臣】名字至今糜俸一十五年念
　　君恩深厚未知图报之地值肤功告成还师饮至普天率土无不额手称庆喜遇升平若夫作为文章以敷扬盛烈【臣】忝列词馆亦与有职焉谨案周礼大司马王师有功以恺乐献于社大司乐王师大献则奏恺乐恺乐者即后世铙歌鼓吹之所本也汉世谓之恺歌【臣】愚不敢妄拟前人之作自同雅奏辄制成恺歌十章虽辞义浅薄惧不足以发皇

　　东祀草序
　　丙子秋
　　皇上北征凯旋以祭告武功之成遣使四出翰林院侍读学士溧阳史先生奉
　　命祀少昊帝尧陵寝先师孔子阙里当是时先生之不出京师十有六年矣自释褐中秘校书内殿起居在钩陈豹尾之间枚马之赋颂苏李之应制其余游览登涉未数数然也及乎揽辔出国门云山入目竒思横流数衔
　　命就道之日至讫事还朝共得诗六十四首其只谒陵庙则肃穆峻整慿吊山川则俯仰悲壮酒塲文防之激昂而趣逸思亲懐逺之凄怆而情深体擅众美不主故常然先生之诗实非有变于前也独其天然自得之趣根抵于性灵藻绣于学植至是而始畅所欲言耳张燕公王右丞岂曽学为山林枮槁之习哉乃燕公居岳州所作音调凄婉偹骚人之情思右丞望春兴庆陪宴从防与其辋水云溪竹洲花坞之逸兴何以异哉盖古之达人君子以泉石烟霞为性情之穷之显无徃不存虽然使山林枯槁之軰终日含毫以求肖其所为萧散闲逺者而已不胜其寒窘之态矣况于清庙明堂煌煌巨制其可矫而为之与今
　　圣人有道区宇寜谧二三儒臣出奉简书入资启沃以发为咏歌蔼然治世之音天下将有想慕其遭际之盛如在皇古邈不可即者顾某以老生浅学犹得逡巡其后辱先生复命之序出其荒芜之辞以窃窥见夫作者之意防何其幸也
　　史彚序
　　太史公司马迁变春秋编年之例作为史记后史家皆仿其体而因史记成书者又数十家或发明义例或诠释音韵或训诂名物皆得以其名自列于后世江都前孝廉元长先生之史彚亦是类也孝廉分史记所载为十集立治原正学将畧诸门文与事各以类从使读者知太史公所作其关切于治道人心如是其不苟可谓司马氏之功臣然迁遭难困辱自序其著书之防歴引文王孔子以及屈原左邱明孙子吕不韦韩非之徒而曰此人皆意有所郁结不得通其道退而论书防以舒其愤思垂空文以自见孝防遇时变迁凿坏肥遯者三十年所寓意惟在此书此与太史公著书之防何异而孝防之苦心尤有难与流俗人言者读者当自得于文辞之外也孝亷有孙曰绳武字存恕惧手泽不传徧求于有力者谋梓其书以与余有文字之知故不逺数千里来请为序史公之后惟外孙杨恽有闻誉孝防遂有象贤如吾存恕者此可谓幸已
