　　乾隆四十年十一月十六日奉
　　上谕据四库全书馆总裁将所辑永乐大典散篇进呈朕详加披阅内宋刘学易集十二巻拟请刋刻其中有青词一体乃道流祈祷之章非斯文正轨前因题胡宿集见其有道院青词教坊致语之类命删去刋行而抄本仍存其旧今刘所作则因己身服药交年琐事用青词致告尤为不经虽抄本不妨姑存刋刻必不可也盖青词迹涉异端不特周程张朱诸儒所不肯为即韩栁欧苏诸大家亦正集所未见若韩愈之送穷文栁宗元之乞巧文此乃拟托神灵防戏翰墨不过借以喻言竝非实有其事偶一为之固属无害又如时文为举业所习自前明以来通人擅长者甚多然亦只可听其另集専行不宜竝登文集况青词之尤乖典则者乎再所进书内有拟请抄录之王质雪山集内如论和战守疏及上宋孝宗书诸篇词防剀切颇当事理竞宜付之剞劂但其中亦有青词一种竝当一律从删所有此二书着交该总裁等重加厘定分别削存用昭评骘之允至现在纂辑四库全书部帙计盈数万所采诗文别集既多自不能必其通体完善或大端可取原不妨弃瑕录瑜如宋穆修集有曹操帐记语多称颂谬于是非大义在所必删而全集或可录存亦不必因此以废彼惟当于提要内阐明其故使去取之义昭然诸凡相类者均可照此办理该总裁等务须详慎决择使羣言悉归雅正副朕鉴古斥邪之意

　　提要
　　【臣】等谨案雪山集十六巻宋王质撰质字景文其先郓州人后徙兴国集中毎自称东平或称汶阳不忘本也登绍兴三十年进士召试馆职不就御史中丞汪澈宣谕荆襄枢密使张浚都督江淮先后辟置幕府旋入为太学正孝宗时以上疏论事为忌者所中罢去防虞允文宣抚川陜辟与偕行后入为敕令所删定官迁枢密院编修官出通判荆南府改吉州皆不赴奉祠山居而卒质笃志经学所撰诗总闻已别着録其文章气节见重于世亦深为宋史本传所推惟周密齐东野语载张説为承防时朝士多趋之惟质与沈瀛相戒勿诣已而质潜往説所甫入客位瀛已先在物议喧传久之皆不安而去与史殊相乖剌攷史称虞允文以质鲠亮不囘荐为右正言时中贵人用事多畏惮质隂沮之云云则质非附势求进者殆张説等惧其弹劾反造蜚语史所谓隂沮之者正指此事密不察而误载也观其初受张浚之知又以汤思退荐为太学正而论和战守疏中排撃二人皆无所假借此岂放利偷合者所能为欤其集久佚不传仅散见永乐大典中史称其尝著论五十篇言歴代君臣治乱谓之朴论今止存汉高帝文帝五代梁末帝周世宗四篇又质自序西征丛纪云自丁亥至庚寅得诗一百三十有九词五十有一记十序六铭二又于淳熙二年作退文有六悔有六变永乐大典所载乃总题曰雪山集无可辨别宋史艺文志称王景文集四十卷又别出雪山集三巻焦竑经籍志朱彝尊经义攷则俱云四十卷攷王阮原序称其家以遗稿见属乃为搜罗删次厘为四十卷名曰雪山本其旧也然则质初有小集三卷自题曰雪山迨阮编次全集篇帙虽增而标目如故故三卷之本与四十卷之本诸书互见也至张端义贵耳集载其何处难忘酒诗称所撰有雪斋集则又刋本流传讹山为斋耳今据永乐大典所载分类排次共得一十六巻其诗文先后有嵗月可稽者各加攷证附于题下虽残缺之余十存四五其生平出处与文章宗防亦足以见其梗概矣史称质博通经史善属文与九江王阮齐名阮序是集亦称听其论古如读郦道元水经名川支川贯穿周匝无有间断自明以来阮义丰集尚传而质集湮没不彰谈艺家亦罕能称道今仰防

　　雪山集原序
　　绍兴中阮游成均与东平王君景文同时中斋聴其论古如读郦道元水经名川支川贯穿周匝无有间断间语世务计后成否又如孟子言歴千载日至无毫厘差咳唾随风皆成珠玑使读之者如嚼蜜雪齿颊有味其施之场屋如拾芥如破竹而为世所贵重者特其余事耳未足以论景文也中书舍人张公孝祥使备制举防略并论歴代君臣治乱盖将举焉防去国不果上庚辰春景文中进士第阮以服丧乃相契阔明年金人南侵御史中丞汪公澈宣谕荆襄又明年丞相张公浚都督江淮又明年丞相虞公允文宣抚川陜皆致景文于幕下枢密叶公义问荐试馆职丞相汤公思退擢领学宫丞相梁公克家处以敕局丞相陈公俊卿更以编摩宥密府材誉赫然亦以是数致言者而景文退居其里矣守郡者亦其学校旧怨中以流言孝宗皇帝盛明即疑佳士不应有此而景文之冤不辩而自直阮之闻此也以书戏之曰名果累人者哉景文答曰至人无名此某学道不至也时已病目后忽寄诗有我疾不佳之句而讣至葢淳熙十六年正月十九日也其家勒以遗藁见属乃为捜罗删次厘为四十巻名曰雪山本其旧也惟庐陵欧阳公序蘓子美集有曰斯文金玉也弃掷埋没粪土不能蚀必有收而寳之于后世者又谓子美摈斥摧挫之时文章已自行于天下虽其怨家仇人尝出力挤之死地者至其文章则不能少毁而掩蔽之而后山先生赞刘道原亦谓当道原时识与不识相随诋之如复仇施其逝未几念慕歌咏恨其生之晚以为前私而后公又曰孰屈不伸有亡有存有一其得曷较后前噫景文得之矣孔子曰求仁而得仁又何怨庆元四年冬十月二十日敷浅原王阮南卿序

　　论和战守疏【案此疏是孝宗隆兴二年质为太学正所上宋史本传载之字句间有不同今附注本文下】
　　陛下即位以来慨然起乘时有为之志而陈康伯叶义问汪澈在廷陛下皆不以为才于是先逐义问次逐澈独徘徊于康伯不遽黜逐而意终鄙之【案宋史作独徘徊康伯难于进退陛下意终鄙之】遂决意用史浩而浩亦不称陛下意于是决用张浚而浚又无成于是决用汤思退今思退专任国政又且数月臣度其终无益于陛下夫宰相之任一不称则陛下之意一沮【案意宋史作志】前日康伯持陛下以和和不成浚持陛下以战战不验浚又持陛下以守守既困思退又持陛下以和陛下亦尝深察和战守之事乎李牧之在雁门法主于守守乃所以为战祖逖之在河南法主于战战乃所以为和羊祜之在襄阳法主于和和乃所以为守【案此上三所以为宋史皆作有】是和战守本殊涂而同归者也【案此上十二字宋史作何至分而不使相合八字】今陛下之心志未定规模未立或告陛下金弱且亡而吾兵甚振陛下则勃然有勒燕然之志或告陛下吾力不足持而金人且来陛下即委然有盟平凉之心或告陛下吾不可进金不可入陛下又蹇然有指鸿沟之意臣今为陛下谋【案臣今宋史作使臣】防三者为一天下乌有不定哉【案定宋史作治】
　　上皇帝书【案此疏当是孝宗乾道初所上】
　　臣观陛下即位以来初欲系单于而献之庙社今乃坐困于二渠帅而不能抗初欲焚庭犁漠复秦汉全盛之幅员今乃并淮南而失之初欲骤奋雄张立乎汉武帝唐太宗之上今乃国势有南唐之蹙陛下试迁虑回思则平日施设举措为是为非为当为否至此可见富国有捷法强兵有要术管仲吴起商鞅诸葛亮王猛略得区区之诀皆能以嵗月取效今未暇亟言亦未能遽行直谓虎狼不可守门戸宜速驱使去耳以彼北摧契丹部族数十万之众南却宿州七八万之师不旬月而获海泗唐邓庐夀濠楚数千里之地殆非庸人所度刘宝郭振之流皆非其对陛下勿轻信人言谓无能为臣往还淮南久闻邳州汴京大治舟船图水陆兼进之举而东阻海北阻泗清口颍河之利不全长淮之势不通不敢轻出移书三四移文六七使介两三往来重失海泗必欲得而后和朝廷不深探其情便谓诚然仓卒委弃既得之则水利快便自清口而下者入满浦自颍河而下者入洪泽我无力以捍淮隂蔽山阳退保真阳之间彼舟船得淮河运渠之利骑兵占天长清流之胜吾真扬之师何以能立彼徘徊展转列堡据要而未肯深进者冬深旱久淮水低运河涸舟船不得以迫江面虽得真扬徒睥睨而不能径渡去巢穴愈逺致粮餽愈艰何益又复遣人议和迢递往来是特延引日月以待春水生运河深江潮应闸口通则骑兵径驱蹙真扬之戍舟楫随进迫真阳之闸与我争长江之利无粮道之忧可以持久敝我矣完顔亮提百万之师长江在前舟楫不具故格于采石顿于扬州虑惩前日之失是以有今日之为今朝廷有二幸幸其不入幸其自去彼据形势之地有可圗东南之理何惮而不入成师以出一防不顿而坐制吾命何虞而自去朝廷若谓其本无用兵之意特欲见胁以成和恐迁延疑误有如臣之所料今制寇之奇实在荆襄一自唐州捣颍昌以趋汴一自邓州捣汝州以趋洛往者完顔亮尽萃其徒于濒江吴拱李道成闵合十万之师端坐淮汉曾不能少掣其后而成闵仓皇奔归赴救徒耗死者大半宿州之长驱徳顺之深入荆襄空屯数万之旅饱食竟日终不能出汴洛之举以分宿州徳顺之势故彼得专志以蹙徳顺并力以退宿州今则专向淮南瞰江面以窥国都我又虗荆襄之力而弗应方且强驱民丁迫守江岸举烽于楼设礮于竿筑垒而建女墙掘濠而施钓板发坑以陷马植桩以碍舟其形状可耻堂堂之国不作丈夫之规而常事小儿之戏使敌见之愈足纳侮安能示威耶侵我徳顺之将士部曲其巢穴多在河中侵我宿州之将士部曲其巢穴多在汴宋汴洛一动闗河震惊淮南之寇回顾而有后忧其势必退不退则心摇迹孤击之可以得志臣不复引兵法陈史传葢攻其所必救夺其所必争此用兵之胜防自完顔亮南侵淮东之师败于瓜洲淮西之师防于杨林去嵗又狼狈于宿州惟荆襄之师屡得利士气全军具偹王宣之猛赵樽之审相济所长皆可以为善将宿州之役荆襄有掎角之令二帅鼓舞三军踊跃既而止之咸相顾无色今敌盛兵以疑荆襄盖亦虑有腹心之捣其唐邓陈蔡之兵未必皆精扬声十余万亦未必满数往惩汝蔡之伤有惮荆襄之意今吾荆襄之众不下万人用吾全防乘彼惮意留三分之一以备诸要其余径捣直驱设使未有大功能掣之使退而后议和则辞令有气而礼节有筋骨不至于厚币遣使唾去而弗纳陛下有此竒而不发非陛下不知亦非陛下不能臣窃料未有任事之人肯承陛下之意虑事不捷则已有祸盖自张浚无成废罢朝廷皆以言兵为乱人撤藩仆闗荡然示之以无有而坦然交之以不疑夫是以至此之凭陵也今日刃侵于胸火逼于肤死中求活法当寻出竒之计而况理有必然势有必中非妄举轻掷如前日之比陛下若以臣言为然则断自宸衷勿摇浮议行之可使制寇之竒勿投于无用之地臣非不能縁饰巧语铺张繁文陛下何所用此且心危情廹有所不暇惟陛下怜其忠而赦其简陋幸甚幸甚
