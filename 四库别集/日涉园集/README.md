　　钦定四库全书　　　　　集部三
　　提要
　　日涉园集十巻　　　　别集类二【宋】
　　臣等谨案日涉园集十巻宋李彭撰彭字商老南康军建昌人陈振孙书録解题以为公择之从孙王明清挥麈録谓李定仲求以不得预苏舜钦赛神防兴大狱彭即其孙也二説未知孰是宋史不为立传其行履已不可考赵彦卫云麓漫钞载吕居仁江西诗图録自黄庭坚以下二十五人彭名在第十五居韩驹之亚则彭本文章之士故事迹不见于史也其集书録解题作十巻世久无传今检永乐大典所载彭诗颇多抄撮编次共得七百二十余首诸体咸备谨校定讹谬仍厘为十巻以还其旧集中所与酬倡者如苏轼张耒刘羲仲等皆一代胜流故其诗具有轨度无南宋人粗犷之态吕居仁称其诗文富赡宏博非后生容易可到刘克荘后村诗话亦称其博览强记而独惜其诗体拘狭少变化今观所作克庄所论为近之然邉幅未宏而捶链精研时多警防颇见磨淬之功在江西中与谢逸洪朋诸人足相颉顽终非江湖末近所能及也乾隆四十六年九月恭校上

　　钦定四库全书
　　日涉园集巻一　　　　　　宋　李彭　撰五言古诗
　　次正平上座韵赠子充
　　吾宗大宛种未入十二闲膺门沬流赭气已无燕山何时黄金羁鞍辔长楸闲皎皎自空谷待君当御还
　　戏赠兼简李翘叟
　　荆州有醉客踏雪至我庐缺月寒皎皎挂在东南隅索我五字句慙非竞病徒卖菜欲求益故态发狂奴寄声李夫人聊为一笑娱
　　赠子充
　　阿充克家才昭代真孝秀白髪在板舆色养侍昏昼来营甘脆具底事成宿留到骨似我穷黄钟满君脰落防哉生魄步屧月成彀幽园语更仆莫遣双眉皱觅句洗愁兵河倾转箕斗
　　赠无着
　　萧萧玊涧翁韵调实瓌朗怀人修文去音尘生逸想岂唯招重客我庐亦时枉徽轸发妙言幽园复清旷飘忽下岩隈嵗月驶两桨寒蝉久停号新禽哢佳响去者念履綦方来惊蕙帐伊予厌嚣尘絶壑期自放从君割半峰披云同矫掌
　　送人游吴
　　朔风号木末迫此嵗时晏驱车不停轨客子老昏宦平生蒋诩径求羊毎娱玩持觞送落日抵掌河宿烂胡为向吴防雅意访亲串亲串者谁欤昻昻金闺彦嘘枯由涵照要津借推挽念子赴修涂发轫在昧旦姢姢所娇儿嶷嶷赴门限闺人密缝裳遐征苦多绽结束志千里二事忘一眄莫遣扊扅歌愆期发长叹
　　宿归宗赠轼老
　　平生丘壑心只园多幽趣杖藜过悬崖投足澄百虑涓涓一滴泉湛湛三危露渊源自曹溪甘腴胜牛乳诸峰罗孙曽一一尊鼻祖法筵碧眼人家世本鄠杜紫枢与黄閤去天才尺五不为世网婴粲可叶心素绳牀作禅定大音响韶濩应梦先补陀祝髪得玊斧我来心眼明意气谐律吕金地雨曼殊天香绕巾屦明当整归鞍天末蜚鸿度
　　奉赠正平上座
　　若人韵凝逺天资复敦厖萧然出尘表幽寺聴逄逄剑峰极天净悬瀑声舂撞心猿久调御业白莹明缸残河横晓鉢霁雾起寒窓禅余覔佳句鼎鼐笔能扛我来孟冬月望履尘心降世纷剧羊负是身如帝江愿借三峡水一滋陇头泷永办桑门服犹堪追老厐
　　和连雨独饮
　　力田胜巧宦燕息自超然懒觑竹素书俯仰丘壑间时时过只园亦复舞胎仙偶然得欢趣举觞望青天六凿自无竞歘见象帝先羲皇隔晨炊坐见淳风还磨螘不少留鼎鼎遂百年但得防心侣相对两忘言
　　次秦处度赠归宗老示中上座韵
　　夕霏催节往槁叶径辞风力疾招秦叔望山扶老笻泄云为卿霭牵余来鹫峰中有厐眉老常持半圣弓一席跨牛地诛茅期此同虚怀包朗月意表得诗翁逺志是小草慎在出处中公莫厌苦相嵗晏长相从
　　嗣首座以老夫诗西岭障斜日为韵作五章见寄次韵荅之
　　寻春舍南北呼酒玊东西絶胜刘校尉太乙持青藜天边艺兰人风光今防畦胡为寂灭海妙语纾虹霓清霜熟柿栗防面一俄顷蕨拳渐紫苞鸟语度诸岭仰头占昏中弧星光炳炳何当聴谈令我发深省我家二季子风标荀令上优孟何足求慎勿作楚相毛顔独摧頺儿能缚篱障摩挲茧栗姿我老那复壮争端起儒墨春蚓兼秋蛇笔研真浅事専门自名家但有香积饭不妨薰毗耶温风媚髙栁招客整还斜欧峯压半天堦除转晴日落硙多雪芽香草皆碧苾嗣公释门老眈眈虎而翼我欲往叩之藤萝迷石室
　　再和嗣首座五诗寄舒老微老
　　在昔董仲舒屈身相胶西穷经着繁露窥园废扶藜而我异于是清晨来灌畦雨余聊自适微吟看断蜺黄生牛医儿襟量包万顷称讥等梦事缓带耳不领公议如丹青千载焕且炳古人何必逺薄躬吾屡省堂堂一舒翁道价融壑上烟霞断尘縁那复知苦相防将超诸缘夺我事理障莫言霜满颐矍铄一何壮嗣公丹凤羽髙笑破羣蛙汲引后来秀义侠如朱家出语掩顔谢逰戏达摩耶卧读春风句飞鸿澹墨斜牧庵担板翁锐气如平日明璧不可瑕崇兰有余苾何尝疲津梁苦铩垂天翼索价无乃髙孤峰问少室
　　送杲上人复往荆南
　　荆州老居士曽着进贤冠似欲専一壑其谁知孔鸾苍苔门巷秋不许造席客世无王粲流倒屣端不惜上人宣城彦俊气横九州学道贫彻骨与世初无求马驹大道场属者失宗匠欲铭窣堵波见此真汉相行乞千里余朔风枯叶颠晓踏涧下雪暝宿天边烟赤手泛沧溟笑此太草草不为丞相嗔乃得照乗宝归来寂无闻翠琰开险艰精卫既填海愚公果移山力贫办行李复作南郡去游子多苦顔开士淡无虑霜风木飕飕便当赋式防日涉望烟霭扶笻迟来归
　　奉赠归宗才首座
　　五着匡山屐三息紫霄阴龚闻耶舍塔势吞天姥岑俯窥雕鹗背上聆钟梵音未航萧梁苇峻风先少林灵骨化大璐钜名偕南金松门欲牵叶苦雨毎见临才公三衢秀古貌又古心老胡即渠是宴坐蔵幽深何用登紫霄自足披烦襟我过避秦客茗椀来相寻真成虎溪笑稍寛梁甫吟衲子叹雌伏道师非陆沉世多石窟病须下鼓山针
　　送宁上人归净慈
　　之子逺行迈眷言欵松门云物起离色况乃闻清猿颇云吴防去饱防南山禅南山燕颔郎气敌千人军要当理断履过我艺兰荪
　　寄珍首座
　　诗宁浅而静不贵深而芜观公秋怀作峻洁仍纡余譬之列华馔陆海盈中厨食蟹贵抱黄食鱼先腹腴多入出或少我昔闻先儒年来病蹇拙意广材复疎之子禅定起焚香满晴虚坐看氤氲处緜緜诗思俱应作牛腰束能寄草堂无
　　题胡九龄画夏景牛
　　阴岑不造天憩此老觳觫辍耕垄上来投闲慰心曲养牲奉严禋盖用茧栗犊莫起陆沈悲焉知果非福
　　送敦诗游临川
　　吕郎客秦淮诗思翻秋涛徐郎憩峰句压天柱髙相此两公子信是千人豪叔也往从之似欲诉不遭探囊得长句如取绥山桃疗饥非蒸砂御寒必绨袍一簪未着身所得不补劳复为饥所驱未许学卧陶穷军敌孤客如在皋兰鏖防成刘文叔须髪变二毛搘笻蹑草鞋进退似桔槔引领长江西沙头唤鱼舠金溪有谢侯胷多略与韬天资类文靖貌实中郎曹叔也往从之攻愁解鞬櫜毋轻歌骊驹不复吟畔牢待公稇载归为我擕蒲萄来赏傲霜菊一尊浇老饕
　　和季敌戏书
　　短草被南陌晴丝飏幽轩心静境云寂黙居宣妙言林深鸟乌乐花繁蜂蝶喧是中即真意何须只树园
　　元夕髙卧
　　伊昔宅闗辅门闾萃冠盖停杯邀明月意气殊蔼蔼春风堕江城搜狝顔鬓改粪除二畆地羊裘力薪采矫首望舒圆何由赏心在烟村杂箫皷丛祠响竽籁谁能伴儿嬉颇复偿睡债初惊釡鸣雷遽作涛海非闗骨相屯长闲荷真宰吾生计已决无劳问蓍蔡
　　宿西林寺有先特进及先学士诗
　　渡江送行人我行亦清兴草木春路賖斜斜复整整独造此宝坊深穏庭宇静复阁衬烟霞疎林泄钟磬残僧澁对人短童工汲井杖屦若能神迥知兹游胜楹间盘硬语恠底光炯炯追还曽髙风熟复发深省来归十阳秋足不践斯境猿鹤情未忘拾松来煮茗
　　何生用韵见寄复荅之兼示小何
　　绕屋阴扶踈空堦藓文皱懐人殊未来安得倾盖旧好鸟政相呼吟蛩俄应夜梦嵇阮辈醉乡须少留莫踏肥马尘我自贫马痩
　　水部有耳孙湖海胷中浩小陆复何人相逢嵗亦早并游苏黄门我曹不容扫梧桐生朝阳要须闻鸣鸟云中遣冯唐防鞍未渠老
　　次韵荅禹功兼简周国振时禹功一夕泛舟归郡不及话离
　　攀萝委松宿潄石看云行松疎暝色漫云度夕阴生若人有髙躅领畧遗世情聊为公府步冉冉复盈盈苍苔映槿墉商飇薄庭树方期着屐来遽作张防去择交如择金披沙歘相遇慈明畏无双伟节慙最怒公抱青田姿俛啄家鸡羣徽之拄颊外遐瞩西山云昔我藻鉴士【泗外老仙宜州柱史】拱木蔽孤坟讵意王子敬晚乃知羊欣
　　相望两牛鸣顿觉千里别未别如怀氷别来三伏热赖有清辉吟当窓时玩阅音尘在我前悠然情话悦飃飃顾曲郎恬愉仍静深相逢何必早但贵相知心临博斯识道闻韶真赏音端须理三径嵗华多浸淫王生以诗见贻次韵荅之生视脊中环中实仲父
　　渠渠太火门诸郎盖当世大匠无弃材知人亦未易阿戎两云孙骎骎遣人畏难兄江南秀美比尝托意歘为修文郎不作活国士令弟老数奇禅寂甘穷顇时于翰墨场往往见髙致扬州故时语奕奕好风气独识史君顔周礼秉早嵗背城防借一问学晚尤粹相逢且班荆十事欲九异何须张女弹端为建康醉请抛髙门竿莫反鲁麟袂花药聘君湖暄风揽春思人生行乐耳余事栩栩寐崇兰有国香宁以不服废虽无双南金可报緑绮惠此岂小物哉吾言未渠弃
　　杜钦之次渊明荅厐防军韵见送用韵荅之
　　征南逺韵度诣絶皆妙言银台陈谏疏耻作赋兎园手板聊拄颊爽气来新萹临窓无谢守僧珍自超然我游赤壁下为君且延缘周礼尽在鲁惊嗟类韩宣雁响横河游子念故山行当理烟艇相期醉余年
　　送润上人归宛陵
　　我昔游宣城稚节等雏凤稍知风物奇未解入嘲弄歘尔鬓负霜嵗月杂机综唯有谢公亭颇复到清梦上人宛陵秀禅余喜吟讽胷中淡无畦妙觉自能种偶随欧峰云悠然入茅栋谈间无俗调不应亭午供谁云佳少年意气极庄重别我归故山索我赠言送邻邦古法窟老衲妙擒纵伽黎宴湖海名字走梁宋吾其从之游射隼期必中勿学汉阴人终年徒抱瓮
　　蒲塘道中寄怀归宗逈中法阐法仁三僧
　　牕鸡唤晨炊毂声转水辙出遨得防稽辨严未明发朔气犯襟裾露华袭毛髪仿像滹沱河幽事期可悦旅雁起寒塘疎林耿明月挽彼嵗寒姿然萁共明灭岂惟苏奚奴政可烘履袜黎明诣萧寺饷糜随盥栉风廊谑谈僧蒲团寻老衲堂堂大圜照此渠亲记莂缅怀汤恵休妙理颇诣絶超然宴法窟胷次包禹穴阐师逺游孙卓卓霜下杰真成一角麟笑我缘盎鼈寂子延陵秀晖映敌氷雪渥洼真龙媒作驹已汗血刮膜借金箆嘉言霏玉屑徳星毎难聚顾兎复易缺紫霄上防岏石镜悬莹澈幽期不嫌频难遣芳嵗歇
　　次韵谢朓观朝雨
　　凉飇起屋角防云天际来摵摵荷上雨冉冉自阳台轩牕积余润竹树絶纤埃懒放事幽屏柴扉为谁开肥遯病使然黄门岂负哉可怜南巷翁擕鱼栁贯腮饷我共杯盏眷焉久徘徊凭虚耳热后褰裳望蓬莱
