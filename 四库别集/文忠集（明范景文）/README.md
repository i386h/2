　　【臣】等谨案文忠集十二巻明范景文撰景文字梦章吴桥人万厯癸丑进士授东昌推官厯官东阁大学士工部尚书李自成陷京师景文至演象所拜辞阙墓赴双塔寺旁古井死顺治九年
　　赐諡文忠给地七十亩建祠致祭事迹具明史本传所着有味铉堂疏稿思仁堂存稿玉静阁存稿且园存稿澜园存稿餐氷斋诗稿诸目其子毓秀及其甥王孙锡等合编以为此集景文为庄烈帝所知入阁未五十日而都城沦破卒能从容蹈义自得所安大节炳然争光日月至生平厯官所至亦多引绳切墨持正不阿史称其在文选时值魏忠贤魏广微中外用事景文同乡不一诣其门亦不附东林孤立行意而已是其丰裁峻厉而不肯矫激以取名在明季尤为难得今观集中摄铨副铨诸稿所载奏议大抵剀切详明切中时弊而抚豫出镇等稿所载诸疏于兴利除害之方规画不遗余力虽遭时艰棘弗获尽展其谋猷而干济之才实可借觇崖畧固不独以义烈见重矣乾隆四十三年六月恭校上

　　奏疏【摄铨稿】
　　起废缘繇疏
　　题为开读事文选司案呈到部伏读恩诏一欵一建言废弃并矿税诖误诸臣已奉遗诏酌量起用其有事关国本抗言得罪降斥谪戍永锢殁身者吏部作速查开职名分别奏请召用卹录钦此夫废籍诸臣放弃数十年自分已断沟中无复云霄之望矣明纶一颁岩穴生色亦诸臣殊遭熙朝盛事也臣部仰奉明诏一朝尽用此百余人岂非至愿而敢有后言但诸臣穷岩沈晦姓名常不时闻黄髪老成存亡亦难具晰倘以既朽之骨复防启事大典为之不光或以未当之举启疑宸衷德意因而中格则臣之所大惧也用是再三踌蹰遵明防除事关国本抗言得罪者查开职名不论存殁另本奏闻外而建言诖误诸臣灼知最确者先行起补其余次第采访不出一两月而咨诹当无不真不出数十疏而前后亦不相远况施为有序员缺渐通铨法自不得不尔也惟祈圣明疏上即下用如流水人解积薪则浩荡之恩于焉可广而师济之美于焉立见矣
　　考选科道职名疏
　　题为开读事万厯四十八年七月二十三日伏读大行皇帝遗诏内开两次考选并散馆科道俱令授职钦此又查本年二月内本部题为目前急宜疏通候命诸臣为第一义万乞即赐俞允事奉圣防是钦此本年七月该本部文选司题为从古无八年久候之臣等因奉圣防考选散馆各官已遵遗诏补用吏部知道钦此夫以穷年累月所仰望而不得者一朝立为疏通以千言万语所吁请而不得者一防最为明白诸臣业已连茹而升臣部无庸补牍以请矣但周朝瑞等职名履厯前疏未备谨再开具上览伏祈圣鉴
　　卹用建储被废诸臣缘繇疏
　　题为开读事文选司案呈到部为照伏读诏书内一欵建言废弃并矿税诖误诸臣已奉遗诏酌量起用其有事关国本抗言得罪降黜谪戍永锢殁身者吏部作速查开职名分别奏请召用卹録钦此夫建言诸臣而中有事关国本者或叩清阍或忤时宰或引义而廷诤或因人而株连皆成先帝之令德厪宗社之远虑者也虽宸衷有主未忍言夹日之功而犯颜不避何可没囘天之力当其慷慨发言计不反顾时岂知有今日哉而一斥长甘九死不悔其忠悃良可念也先帝神明不测用意渊深隂采其言以巩万年之庆阳弃其身以贻一生之名今皇上笃念旧人并其身而显庸之即殁而不忘优卹焉固已生辉于岩穴贲彩于泉壤矣亦诸臣千载一时也臣部仰遵明诏除旧辅王家屏之功存宻勿不敢概叙外其余疏名上览伏祈勅下臣部再加查访存者不次起用殁者移咨礼部优卹其有一时开载未备者再行奏闻未敢擅便谨题请防
　　起原任户部主事鹿善继疏
　　为计曹念在急公举朝望切宥过伏祈圣慈免降以惜人才以光新政事文选清吏司案呈为照原任户部河南清吏司主事鹿善继公忠不贰足称社稷之臣沈毅有为卓为名教之主持节可坚百折筹饷特见一斑相应请复原职案呈到部为照鹿善继之被谪也以留籍金花而其留籍金花也以需饷急于燃眉措饷穷于露肘聊出此不得已之计耳夫不惜一官以收溃散之军心以存残破之疆土以宣上德意而垂万世之令名有臣若此正宜破格奬拔用示风励而反防谴降人其谓之何先帝神明天纵岂见不及此毋亦以蠲济之名不出自上权宜之法径行自下姑假此以警后来而重朝廷乎臣部遵防拟降朝廷之权已伸而令已遵矣疏上而不报可益见原无成心姑留以待后庸或者别有深意是未可知也日见皇上慨发内帑二百万犒边薄海内外无不仰颂明明后举动非常夫在内者尚可出之而宏大赉在外者何不可蠲之以济急需若鹿善继者想亦圣明之所欲急宥也伏祈勅下免其降级照旧复职边士闻之必将曰国家未尝忘我而断不解体廷臣见之必将曰国家不肯负人而莫不尽心非特为一人之官而已也
　　陞原任御史刘光复疏
　　题为戅臣效忠可嘉明主录用难缓伏祈起陞以宏先德事文选清吏司案呈到部为照刘光复既以言获罪矣彼其赤忠欲洒臣取其心犯颜不避臣取其气而孝慈之语奉扬休美亦举朝所共欲言者不闻有大罪也或但罪其髙声越次耳然覆盆五载已寓磨励之微权而释系一朝弥见浩荡之德意若竟其所为必能殚竭忠猷报答圣明一以教臣下之忠一以明圣主之孝一以广先帝之仁一举而三善备焉伏祈勅下起陞光禄寺寺丞在本官感恩而图报在诸臣闻风而作忠亦风励一大机括也
　　鸿胪正卿改用甲科疏
　　为临御方新朝仪当肃鸿胪正卿改用甲科以复祖制事文选清吏司案呈照得鸿胪寺正卿自王用贤察处以后员缺已四年矣先后建言诸臣皆议宜用甲科此非无据也查得万厯元年十一月鸿胪寺缺正卿该本部覆准正卿仍用进士其别途出身者歴俸年深但陞服俸官至左少卿而止此载在职掌班班可考者尝有屠义英以嘉靖丙辰进士由太仆寺少卿改鸿胪正卿此即先帝初年事也后失其制槩用别途固以其礼度之素娴耳然家起赀郎以厠九列之班已自不韵即仪习绵蕞以示百僚之瞻必且不光虽资格不可局人而典章自宜还旧今南鸿胪卿余启元现用甲科何独于北而反不然合行亟请案呈到部为照圣主御极中外具瞻汉官之仪复见于今矣然肃朝常于九重宣法仪于百辟则鸿胪实司其事而可复以他途混乎若用甲科则扬眉于济跄之列而该寺重树仪于殿陛之间而朝廷尊所关亦自不细也既经该司备呈前来相应催请祈勅下臣部除该寺左右少卿仍用别途聴其礼熟资深止加服俸正卿一缺务择中外清望甲科以充斯任庶仪观整肃典礼昭明亦维新中一事也未敢擅便谨题请防
　　救吏科给事中周朝瑞免降疏
　　题为愚忠可鉴圣度宜宏祈恩免降以作敢言之气事文选清吏司案呈到部臣等伏读圣谕内阁朕昨览文书见有吏科给事中周朝瑞条议仰献慎初三要内停止金花银两朕即位之初言路方开矿税已撤如何擅行凟扰此项银两原系旧制进内以备万夀圣节等并典礼等项及各宫赏赐武职军官俸粮之用难以停减周朝瑞这厮要誉妄言本当拿问朕哀思皇考皇妣梓宫在殡姑从轻降一级调外任用卿可传示该部这银两还遵照旧规行特谕卿知钦此随该内阁大学士方掲帖奉圣防览卿所奏朕已悉知周朝瑞这厮不谙事体逞臆妄言擅行议减内库金花阻絶国用好生可恶周朝瑞已有防了卿不必申救该部知道钦此钦遵窃念周朝瑞忤触宸严夫何敢言臣部仰遵明防又何敢言但周朝瑞职居言路遭际明时未忍自负以负主上慷慨发舒故不觉虑之过语之戅耳至所条奏留停金花以寛民力而光上德酌祖制而裕国储亦忠于皇上之职分也乃皇上罪以要誉妄言夫主圣臣直名在皇上下亦何誉之可要若因此而罪之则适成其名耳夫当容言之时而有拒言之事恐未免为圣德累也日见皇上用人行政下如流水举朝相庆皆谓衮职不阙补救无事臣工此时止宜将顺德意彼目覩大涣王居不殖货利已明知上意所向矣正所谓善将顺者而何深罪乎雷霆一震已惕不测之威露雨旋施弥彰有容之度知圣德如天谅必出此也伏祈圣慈免其降谪照旧供职使下有犯颜敢諌之士足见盛世之有人使上有刻印销印之举愈觇圣心之无我其所裨益岂渺小者哉
　　奏疏【副铨稿】
　　总催大僚考选疏
　　国家正当全盛朝宁渐见空虚恳祈圣明立允防推考选以备任使以勷治理事文选司案呈到部为照大僚国之股肱无股肱何所依毗言官国之耳目无耳目何由聪明故有凭有翼有孝有德盛世所以兴隆也衰世廷无人焉有人不用自沦于无耳今部院见止五六员科道见止十数员可谓无人之极矣启事频烦不防报可岂以臣部补牍皆为人求官耶即为人求官亦以酬庸而为官求人乃以集事皇上何疑臣下之深也积疑成厌既有人不用而积厌成暌究且欲用无人是以臣所患者官无其人耳人无官又何患焉独不见夫避难规便者以无官为幸乎悠悠之口皆谓皇上吝惜爵禄然此爵禄在人主借以神其鼓舞在臣下借以効其功名使刓吝不予上失鼓舞之权下无功名之地士亦安往而不贫贱者而人主将与谁共治天下乎诸疏现在御前惟皇上留意立赐俞允师济之盛即在目前劻勷之効可收异日矣
　　催大僚疏
　　为九列多缺时事可虑恳祈圣明速赐简用以保治安事文选清吏司案呈到部为照国家稽古建官政分六曹有长以提领挈纲有贰以分猷佐理意至深远也无事之时得人以勷郅隆多艰之时得人以挽否运而缺人废事从来久矣皇上试观近日各部院之官缺耶不缺耶近日各部院之事废耶不废耶综覈之朝事聴其废弛师济之世人任其空虚所悮者谁之政务所摧者谁之才贤则皇上自为社稷计者左也今刑部缺尚书臣部户部兵部缺左侍郎而工部都察院则三堂全缺南京户部礼部兵部俱缺尚书都察院缺左都防推各官诸疏现在御前惟祈圣上立赐俞允施行
