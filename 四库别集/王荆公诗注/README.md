　　提要
　　【臣】防谨案王荆公诗注宋李壁撰考宋史及诸刋本壁或从玉作璧然壁为李焘第三子其兄曰垕曰塾其弟曰名皆从土则作璧悮也壁字季章号雁湖居士初以防入官后登进士寜宗朝累迁礼部尚书叅知政事兼同知枢宻院事諡文懿事迹具宋史本传是书乃其謪居临川时所作刘克荘后村诗话甞讥其注归肠一夜绕钟山句引韩诗不引吴志注世论安以虫疑冰句引荘子不引卢鸿一唐彦谦语指为疎漏然大致捃摭搜采具有根据疑则阙之非穿凿附防者比原本流传絶少故近代藏书家俱不着録海盐张宗松得元人椠本始为校刋集中古今体诗以世行临川集较之増多七十二首其所佚者附録巻末考叶绍翁四朝闻见録称开禧初韩平原欲兴兵遣张嗣古觇敌张还大拂韩防复遣壁壁还与张异词阶是进政府云云是壁附和权奸以致丧师辱国实堕其家声其人殊不足重笺释之功足禆后学固与安石之诗均不以人废云乾隆四十四年七月恭校

　　王荆公诗注题辞
　　王荆公诗五十巻鴈湖先生李壁季章笺注予十年前购得华山马氏所藏元刻本间取通行临川集勘之篇目既多寡不同题字亦增损互异乃叹是书之善不独援据该洽可号王氏功臣也史称季章嗜学如饥渴羣经百氏搜抉靡遗今鴈湖集既不存其他着録亦尽逸惟是书见称艺林而流布絶少因重锓之以广其传俾嗜古者得窥先生之蕴涵识临川之意匠而并可正俗本之纰缪殆如景星鳯凰争先覩之为快已乾隆辛酉上已后五日武原张宗松题于清绮斋






　　畧例
　　荆公诗注五十巻见晁氏读书志与是书巻帙相符马氏通考称十五巻盖传冩者偶倒其文耳陈氏云参政眉山李壁季章譔谪居临川时所作助之者曾极景建魏鹤山作序
　　王逸注骚李善注选皆分疏于句下未有讥其横隔者后人注杜始变其例并注于全诗之后是书体例不一或分注或总注或用圈隔之或牵连书之悉遵其旧不敢妄为搀易
　　注家多穿凿附防之病又有稠叠蔓引之病李氏捃摭搜采务求来歴或字义稍有异同则云公别有所本或云未详出处岂有改窜旧文伪造故事以疑悮后学者乎至于彼此复出李氏亦或不免然一经删削便失昔人面目不若仍之为愈
　　改壮月为牡丹疑木刋为水利勘书所以贵阙疑也然元刋不逮宋椠也是翁屡言之是书间有显然讹脱处悉取四库书校正惟释典道藏揷架所无未敢增易一字
　　薛梦符之补注杜集黄学臯之补注苏集类皆自为一书是书补注或缀巻末或附诗后今悉仍其旧惟低一字以别之
　　原本尚有刘须溪评防品藻甲乙容有未当且杂乱注中观者目今并芟之使李注孤行更觉心目开朗
　　诗集之前例有年谱杜韩苏诸家皆然荆公独无或以为请予曰李氏于题下或标年月或引时事姓名必着官爵必书其于公生平出处兄弟朋友亲属过从离合之迹亦畧可考焉予闻见浅尠未敢为续貂之举止彷杜韩苏三家例附宋史本传于巻首知人论世请竢后贤
　　诗中先后序次与世俗所行临川集本亦无大异但俗本遗漏甚多古诗如八巻中鳯凰山第二首十二巻中扬雄第一首十五巻中寓言第四首第六首第七首第八首第十首第十三首二十巻中澶州一首寄平甫弟衢州道中一首二十一巻寄慎伯筠以下十三首河北民以下二首律诗如二十五巻中次韵张子野秋中乆雨晚晴以下五首三十六巻中次韵王禹玉平戎庆捷一首三十七巻得孙正之诗因寄呈曾子固至末凡三十首四十巻中春怨一首宫词一首四十六巻中杂咏第六首四十八巻中晩春以下七首子贡一首通计古今体诗七十二首皆此本所有而俗本所无也惟五古中车螯一首及信陵坊有笼山乐官一首五律中致仕邵少卿挽词二首则俗本载之而此本反缺因彷草堂诗笺逸诗拾遗之例附録于末又七律中偶成二首李氏逸去其一而附注于寄曾子固诗下或疑中四句重用之故然题下及目録内仍作二首岂李氏原未尝删而刋书者误去之与不敢妄增亦列之于补遗中
　　通行临川集本诗题毎有增损字句亦多异同或如鲁鱼亥豕亟当改正或如别风淮雨无嫌并存遽数之不能终也意欲仿朱子之于韩文别为考异一巻卒卒未暇姑俟诸异日云
　　李氏之注王诗犹施氏之注苏诗任氏之注黄陈二家诗也山谷后山诗注尚有前明雕板东坡诗注则宋漫堂先生获宋椠本刋行于吴中今遂家有其书矣独是书絶无仅有近代储藏之家若绛云若传是俱不列其目华山马氏至晩嵗始得之故道古楼书目亦未之载衎斋没后复随云烟飘荡流转数姓归予插架因念吾乡姚叔祥有言藏书家但知秘惜为藏不知传布为藏遂缮冩雕印公诸同好览者勿易视之
　　宋漫堂先生所得宋椠施注原阙一十二巻其存者又多残脱虽经时贤补缀究非昔人完书识者不无憾焉是书首尾完好不为蟫蚀防齧殆有神物防持惟巻端失去魏鹤山序一篇第三十巻第五十巻失去两末页访求数年无从钞録与昔之刻山谷诗注者失去鄱阳许尹一序又失去题下注脚二页髣髴同符亦一异也俟访得之当即补入
　　开雕于庚申腊月蒇事于辛酉夏五纵臾剞劂鸠工而饬材者倪子东铭【大成】也往来商析疑而辩譌者许子初筠【昂霄】也至于讐校之劳余独任之而某书某篇代余检觅则塾师朱子学顔【以发】暨【訚谔】两儿与有力焉例得附书

　　元丰行示徳逢【徳逢姓杨与公隣曲○按王直方杂记徳逢号湖隂先生丹阳陈辅浙西佳士也每嵗清明过金陵上塜事毕则至蒋山过湖隂先生之居清谈终日嵗率以为常元丰辛酉癸亥频嵗访之不遇题一絶于门云北山松粉未飘花白下风轻麦脚斜身似旧时王谢燕一年一度到君家湖隂归见其诗吟赏乆之曾称于舒王闻之笑曰此正戏君为寻常百姓耳湖隂亦大笑】
　　四山翛翛映赤日田背坼如兆出【诗予尾翛翛此借用○退之诗或如坼兆】湖隂先生坐草室看踏沟车望秋实雷蟠电掣云滔滔夜半载输亭臯旱禾秀发埋牛凥【子虚赋云亭臯千里师古曰为亭于臯隰之地○埋牛凥言乆旱得雨禾皆怒长其髙可没牛凥也】豆死更苏肥荚毛倒持龙骨挂屋敖【月令孟夏之月靡草死龙骨所以车水既得则无用故挂之屋敖】买酒浇客追前劳三年五谷贱如水今见西成复如此元丰圣人与天通千秋万嵗与此同先生在野固不穷击壤至老歌元丰【论衡曰尧时百姓无事有五十之民撃壤于涂观者曰大哉尧之徳也撃壤者曰吾日出而作日入而息凿井而饮耕田而食尧何力于我也】
　　后元丰行【介甫熈宁七年罢政作此歌正居钟山时或谓公欲以彻神考之听冀复相此缪论也】
　　歌元丰十日五日一雨风【盐铁论周公之时风不鸣条不破块五日一风十日一】麦行千里不见土连山没云皆种黍水秧绵绵复多稌龙骨长干挂梁梠【秧禾秧也载芟绵绵其麃○尔雅释宫楣谓之梁檐谓之摘注屋梠也説文云楣齐谓之担楚谓之梠樀丁狄反梠音吕】鲥鱼出网蔽洲渚荻笋肥甘胜牛乳百钱可得酒斗许虽非社日长闻鼓吴儿踏歌女起舞【张説刘禹皆有踏歌行神仙蓝采和常醉踏歌云云】但道快乐无所苦老翁堑水西南流杨桞中间杙小舟乗兴欹眠过白下逢人欢笑得无愁【尔雅云樴谓之注云橜也説文樴之弋反弋也从木戠声杙逸即反从木弋声橜巨月反弋也从木厥声○汉地志云防柯系船杙也○华阳国志云楚顷襄王时遣庄蹻伐夜郎军至且阑防船于岸而步战既灭夜郎以且阑有防船防柯处乃改其名为防柯杙音弋○白下见题半山寺壁注】
　　夜梦与和甫别如赴北京时和甫作诗觉而有作因寄纯甫【和甫名安礼行第三十六纯甫最幼名安上行第三十七】
　　水菽中嵗乐鼎茵暮年悲【檀弓孔子曰啜菽饮水尽其欢斯谓之孝庄子寓言曾子再仕而心再化曰吾及亲三釡而心乐后仕三千钟不洎吾心悲】同胞苦零落防合尚凄其况乃梦乖濶伤怀而赋诗诗言道路寒乃似北征时叔兮今安否季也来何迟中夜遂不眠展转涕流离【礼记文王世子今日安否何如】老我孤主恩结草以为期【沈括笔谈欧公有奉使囘寄刘原甫云老我倦鞍马谁能事吟嘲王荆公赠弟和父诗云老我孤主恩结草以为期言老我则语有情上下句皆惜老之意若作我老与老我虽同而语无情诗意遂頽惰此文章佳语独可心喻○宣公十五年魏颗败秦师于辅氏见老人结草以亢杜回杜囘踬而颠】冀叔善事国有知无不为【左氏荀息曰公家之利知无不为忠也】千里永相望昧昧我思之幸唯季优游嵗晩相擕持于焉可晤语水木有茅茨畹兰伫归憇绕屋正华滋【纯甫晩以管勾江宁府集禧观家居○水木湛清华○今人以古语采椽土堦对茅茨遂以茨为屋其实茨亦草也又按韦忠传吾茨檐贱士本无宦情然太史迁传茅茨不翦注则云屋盖曰茅茨茨以茅覆屋据此则茨又为屋盖也两存之○白诗清和四月初树木正华滋】
