　　清惠集序
　　当世庙初旌别大僚御屛所书氷清玉洁者二人一为尚书刘清惠公麟一为大理卿陈公恪陈归安人世称矩齐先生是已刘豫章人先从安仁徙建业慕吾湖山水清逺复卜居吾邑东山之坦上因号坦翁云不佞鳯翔少时闻诸先达公杜门絶轨台使监司守令之车辙盈门不得一见惟佳时胜日与高士吴甘泉孙太初诸公扁舟徃来菰芦间人望之若神仙已复睹公家所藏神楼图知公居万山中欲搆一楼而不可得文太史征明绘是图以赠一时名公歌咏盈幅已复读公所撰岘山浮碧亭碑记文最尔雅其所传尺牍书法人间重如玑璧及索公全集大都散逸即陈大理公子方伯公应和有所摭次亦以垂槖故梓适锡山筠塘陈公来守吾郡询及郡中典刑慨然有当于清惠公遂征其集幸伊曾孙陛藏若干巻文则防鬯简直出入秦汉诗则骎骎韦杜葢公之标格髙入云霄胸中无一毫芥带故所发皆盎然天趣读之足消鄙吝恍若濯魄氷壶澄心玉井者想公之在朝耿耿大节如忤逆珰瑾首倡节慎库等事足垂不朽守越甫五十日而百姓歌思尸祝之王学士华为去思记比之刘宠至今稽山有二刘祠公殁且五十余年而其刚介绝尘之姿孤髙轶世之品凛凛犹有生气筠塘陈公固公臭味公之集若有待于今日语云丰城之剑合于延津信然哉信然哉万歴丙午秋七月既望朱鳯翔序

　　诗
　　四言古诗
　　述懐五咏
　　今我乐矣叙旧以贤瞻彼眀徳不以厥年生事匪多尺宅寸田暮色惟瞑返照复鲜君子保终夕厉愓然载我酒践彼初筵悠悠我心无懐葛天
　　其二
　　今我乐矣尘劳靡夕俯仰褒然乾坤归宅冠盖各天妻孥在侧知已辰星道途扫迹空谷足音使我太息返照入林牛羊是则
　　其三
　　今我乐矣自春徂秋髙人用息志士廼愁终风急雨恐撼吾楼楼有眞人邈不可求求而得之泛我轻舟三山何许垂老一逰石川野航曰予销忧龙骧虎变挟小九州
　　其四
　　今我乐矣爰即我谋陂塘雨足上下其流百年垂尽寄兴孤舟华胄玉台浮逰石头羣隂蔽空百神在楼此夕何夕梦逰九州初随枕蝶终执沙鸥兴倦而觉再梦再忧空谷多鬼其声啾啾
　　其五
　　今我乐矣渐解尘缘藏修还谧始于今年八十有六十一师籖同生之士艾耄涸然我孙森森玉立满前仰忆宗祧瓞绵绵升髙有屐临流有船邀我良友矢我新篇退休于室守真尚不忮于人不求于天诏尔后人勿忘勿愆
　　五言古诗
　　山中奉答孙太初
　　望古复望古驾舟行逺空浮云日夕尽天光上下同濳鳞入渊防归鸟正西东亭亭夫容花凄凄风露中羣情各有适至理将在躬日月去已多忧伤来无穷临流一濯足顾影见双蓬歌我水上心答彼巢居翁
　　答太初和韦苏州
　　道存甘拙守时艰慙幸生在昔缪有寄所至皆名城感彼民物忧违我泉石情一朝印章解归来读且耕年登衣食足心闲文字精侧身鸥鹭中尘网何由撄
　　动静语
　　客言动静机相禅苦不了无私动亦得无欲静逾妙动静能几时去去勿复道
　　携琴古
　　疎疎子贱子千载一单父斯民不忍欺厥治登太古问渠何所修得贤以自辅谈笑黄虞间岂复问齐鲁孔铎本夙闻舜弦时一鼔赤子尔何知所欲在慈母吾施既春阳物化若时雨至今梧桐枝金徽照寰宇
　　秋社次苕溪呉大叅韵
　　举世重豪华何人爱幽独达哉晋处士逺心寄松菊物理本寻常道胜乃芬郁前哲流其风吾生踵其躅贳酒箬水头援琴岘山曲青钱挂髙笻紫团出深竹籊籊渭之濵猗猗淇之澳徐徐栁外嬉欵欵花边肃道义常交儆声利久不辱长生桃辉辉内光烛觞咏集浮洲形骸归寓屋逍遥本今夕分防在明旭
　　社韵
　　天壤冇胜事其逢或千春颙昂复藻雅天兴何彬彬铺观同生志天实亨厥屯文明郁然起大半充词臣即有未遇者清修常苦辛颇厌东呉富不道西越贫厐然自宜俗安问桃花津社中诸长老玉立髙嶙峋引翼并眉夀几杖集今晨鸣丝杂挥毫晤语恊眞醇一防义已正再防心弥亲相观各有得退食如见賔翩翩穴凤翥光华动星辰乐善不吾弃垂死善吾身天目无茂草太湖有扬尘精英一以聚千里成荆榛斯言茍不轻请问啇岩人一朝食君食为霖如有神嗟嗟委巷乐逺逰不获伸安得如椽笔大刻垂苍珉燕鴈年年社铭名日日新我徳如广川毋为物所湮
　　读书台
　　暮色常如斯孤眞保终吉爱此嘉树林倚杖数晨夕有时髙鸟下驯扰不吾避好音醒逺心文羽耀眞憇昏鸦结侣归烧灯犹未食耽幽有余兴减食非所惜逾八又五年吾生过其歴无事此堧中古期或可必举觞祝吾树愿树五千尺何必庄家椿得气各有适
　　和文内史陆春官
　　清时两文伯表里呉门居三生契晤后八袠衰頽初挥毫引星汉飞佩入云墟佳境本已寂芳尘未全疎衮褒信未度鼎勒良有余酒醒对蕉緑叶尽还一书
　　寄文衡山
　　戏马非芳春征鸿逆新暑去住本无心隐显不成侣令徳自有隣予美胡独处东野昔为隂渭城今欲雨假寐始难任沉忧终不去夜发呉门书晨集清溪路中多鹤髪人待子聨膝语一诵虎林偈何人不延竚
　　书逸老诗册
　　嵗晩田园归遂逺防龙軰弁岭若天绅亭亭日相对紫翠中有精静应得其槩朝舒五色霞飡之益五内感彼造物者遗我平生爱气序日已迁儿女日已大尚齿登君堂耉老自成防宿约引壶觞晨兴理冠带椽笔洒云根楼船荡湖霭客操如意钩击我瑚碎青莲白苹洲银刀紫花瀬何如安期生鹤背不可载
　　神楼
　　天生楼上翁百年浑是病遭时或振歩颠沛干吾正中嵗幸投簪有物如造命择栖苦不早乞湖何必镜呉兴逸老成春秋迭觞咏从此作行窝东南称独盛契哉文内史绘楼孤且夐谓我居其中懐葛失其静从此不复下得酒歌明圣问余何所得楼中有真性
　　次陈鲁南韵
　　民社闗身地淹留廊庙心能将制锦意和曲比鸣琴鸾凤有文翮低囬枳棘林食我中园竹贻我希世音覧徳起翺翔来归阿阁深春光照城郭桃李成繁隂雨余清树枝团团明月岑懐君妙生殖古道重相寻
　　柬呉甘泉处士
　　闻园临广壑蔓草带蒹葭归来理荒秽绕屋皆桑麻防起惜芳时抱瓮清涧涯露下百卉长羣蔬亦惟嘉陶翁南山豆邵子东门一身在力食外物何由加
　　赠沈竹里
　　我爱竹里子身世一秋蝉吟风兴自足吸露心充然下有谷食者生荷主人怜亦有草食者依依恋平阡岂不竞肥腯何由更髙骞嗟余感君谊抚已思齐贤平生志寡欲老为众物牵卓哉无欲地咫尺不可攀闻君重文防切磋如有传为余辨斯惑拜领春风前
　　神楼漫咏二首
　　青嶂隐澄潭钟灵自太古结楼翠防间云中启朱户风帘半在钩月射人髙歩安期不可期霑濡惜多露
　　其二
　　七十已无友百嵗谁知心独徃日欲暮悲歌天地隂卜居谢人境结楼深复深残躯幸有托安得挥黄金
　　寄甘泉
　　为圃何汲汲所乐在烟霞露下剪青紫或取黄金芽朝朝厌芳脆时时易鱼虾举觞清涧曲一醉当山花今晨共谁酌明夕去君家及尔各强健言笑欢无涯幽求茍如此曰以忘纷华
　　桂竹歌
　　桂香何冉冉不以春吹嘘竹枝何青青不以寒巻舒所以学道者爱惜如璠璵轩当其下琴书与之俱感彼赋命偏忘此生计疎修途既坎短景又虚老色朝无赖秋声夜有余荆南王子灿于世竟何如
　　招石川入社
　　结知自台省僻境亦穷追君年睿而富我骨衰以迟拙劣世遗矣昏塞君忘之暌离本多岐会晤幸有时佳名超九棘古谊冠诸耆三呉得白石握手登黄羲米舟书与画良足解我頥请与此心誓金兰欢不移
　　与李半溪
　　翀翀五湖水水浒多遗老官罢各来归优防遵故道惊尘方蔽空无一挂懐抱永谢车马喧但知衣食好社饮已再三叹想苦不早
　　其二
　　古今重八袠何幸懐君年余也湖心水君如水上船大风扬其波千尺不敢先俯仰社及社唱酬篇复篇凤声云日表蝉号风露前耕桑本一曲杖履得相牵多闻已谐约直谅不违天饮酒或取适放懐空万縁此谊幸不孤瑚琏同一川烈烈青州守吐气成云烟前年入我室百斛甘我泉自云招南村慷慨沽十千今晨践斯约逸兴挥蛮笺月斜褁足去天地俱茫然临别敢一博一安还一迁吕望及衞武何如与籛东施与西施异代知谁贤
　　七言古诗
　　挽王封君
　　一心斋古髙云多潇洒衣冠在耆耋许身孝义日月悬传家节操珪璋洁三槐防郁敷广堂裁培雨露初瓞槐生直干当清霄槐有深根下盘结如旦如曽自本源生封死諡推前烈秋风瑟瑟吹白杨秋草芊芊带孤碣鹤驭飘飖去不还杜鹃啼尽寒江雪
　　覆陈鲁南
　　我居深山真木石一年易过如一日人生难得白髪时且许于今发已白发白有方可再黑感君惠我不死术百年总与一日同世间岂有夀多百青山白云天外天一日一醉与君别
　　竹鹿歌与张石川
　　毎过城市常嘘嘻此竹此堂懽不疑仁和酒肆貂可解苏晋禅床可支年年两醉若天造一日一社无人知朶頥解醲夙所鄙雕虫绮丽晩不宜老人至乐在无事良朋适兴相追随彼此生儿颇不恶两家门户粗支持几人负米可糊口几人弄笔粗解頥名山秀水取便得抹月披风饱即嬉仰学蜗涎力已倦俯逐蝇利心不覊与君择胜论中曲切切偲偲艾复耆儦儦俟俟媒和侣瑟瑟亭亭连理枝同生交儆聚其畔朝燕夕楚将焉为富贵豪华等蜂蚁营营扰扰真堪悲握手桑榆日西下放舟大壑山之陲洼尊轮囷亦何有劝酬莫计黄金巵有时酒渇得甘露蝉吸湑湑清入脾酌露夀君君莫讶金茎仙掌青天低颂君直拟淇园物青青如箦常猗猗懐葛之民渺何许有斐君子眞吾师盟神黙祝九十五清歌抑戒酬吾师
　　七袠咏
