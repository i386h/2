　　钦定四库全书　　　　　　集部四
　　嵩山集　　　　　　　　别集类三【宋】
　　提要
　　【臣】等谨案嵩山集五十四巻宋晁公遡撰公遡字子西钜野人公武之弟宋史无传惟李心传朝野杂记中颇引其所着箕山日记一书而仕履已不可考今按集中上周通判书题左迪功郎知梁山军梁山县尉又程氏经史阁记称尝为涪州军事判官又与费行之小简称绍兴三十年内任施州通判又眉州到任谢表及谢执政啓则尝知眉州又答史梁山啓称猥从支郡遽按祥刑而集首师璿序亦称其为部使者则又尝擢官提刑而不详其地又眉州州学藏书记题乾道年月而丙戌元夕诗有刺史敢云乐句丙戌为乾道二年是时正在眉州此集刻于乾道四年盖皆眉州以前所作师璿序又称公遡抱经堂稿以甲乙分第汗牛充栋此特管中之豹则其选辑之本也晁氏自逈以来家传文学几于人人有集南渡后则公武兄弟知名公武郡斋读书志世称该博而所着昭徳文集已不可见惟公遡此集仅存王士祯尝评其诗谓在无咎叔用之下盖体格稍卑无复前人笔力因由一时风会使然而挥洒自如亦尚能不受羁束至其文章劲气直达颇有崟崎歴落之致以视景迂鸡肋诸集犹为不失典型焉乾隆四十三年五月恭校上

　　嵩山集原序
　　文章之在天下其用至不一而其体亦各不同盖施于朝廷者为制诰为章表施于仕涂者为书防为牋啓被之金石则为记为铭恊于音韵则为诗为赋为楚人之词以至立言则为著书纪事则为史茟或严或肆或质或华短长之殊度广狭之异制竒耦之不齐千彚万状不可穷极观陆士衡文赋之所列自诗赋而至于奏论韩退之进学觧之所叙自姚姒而至于子云相如一名一体莫不皆有律令吁文之变若是其多乎昔之文人其才力声名固自有大小然求其兼是数者而能之或寡矣故诗如杜少陵而无韵者遂以无传文如曽南豊而有韵者殆亦不复着也惟东坡先生称欧阳子以为论大道似韩愈论事似陆贽记事似司马迁诗赋似李白鸣呼此其为大儒也欤嵩山先生晁公负絶人之资出于内海文章家慿厚而积深发之于文雄深雅健钜防俊杰以是擅名声于时乆矣今虽为部使者秉权任之重犹日论著数千百言凡属乎文艺不问何等下笔輙妙天下而皆与古作者合鸣呼天下之文至于公可谓大备矣吾家传甫从公防间掇拾其诗文凡一千余篇以世之学者争欲得公所作读之而恨见之未愽也于是锓之板以传焉传甫以书抵璿曰子于公为门下士宜为序其首顾璿窽啓何足以与于斯文尝窥公所谓抱经堂藁者以甲乙分苐汗牛马而充栋宇传甫之所得殆筦中之豹然已足盖一世矣虽然公岂徒尔哉以道徳治其心以仁义修其身忠以事上公以待物其中浩然方与天地同量宜乎发于事业者既如彼而见于文章者又如此观公之文者尝试以是求之乾道四秊六月日门生左宣教郎充嘉州州学教授师璿谨序

　　登赋楼赋
　　子居于硖中已闻西山之可游盖武信之美尽在西山而西山之美无如兹楼己夘之冬归自益州始及西山之下而休焉霜露既降天髙气清山川开除氛雾不兴乃从主人问兹楼而登焉四山苍苍其峙如屏江流茫然如带萦城一目千里野旷川平主人乐之顾客而喜将以是为东皋每登之而必赋也子曰富丽便美流连光景非今之所谓诗乎世方吹竽伐鼓其门在于五风不几于淫如曰山林之乐可以忘归藻绘岩壑与之光辉往告市朝殆以为欺曽未加重嗤防随之主人亦知言之者无罪闻之者足以戒古者十五国之诗乎当酒酣而未防时劳賔而夀主辩者纵谈豪者起舞试为之歌蟋蟀则宴游者无荒而有度也为之歌既醉则五福可致而六极可去也为之歌七月则有事于农者识三时之先务也为之歌硕鼠则收养吾父兄子弟者感动而知惧也为之歌二南则美哉乎如周召之亲覩也于是至而乐之登而赋之其十五国之所咏皆足以托吾意之所寓凡其乡人孰不愿主人之日游而喜兹楼之有补也
　　屈原宅赋
　　余入蜀之初尝至于秭归之山有渔者过焉指其墟中而告余曰此吾三闾大夫之故居也余闻而异之问途而往观焉则羣山连绵若逺若近风云渟滀不见其境于时秋也霜降气肃月光益明风林水麓之影相乱而大江之声若敲金击石泠泠然其可听也而所谓屈原之居则无复可识吾想夫牛羊之牧其上而樵苏之不禁也久矣而彼渔者何自而知耶余观于屈原之前者有唐叔之苗裔袭霸主之遗风方示侈于天下筑虒祁之新宫倾四封以来会贺匠氏之奏功其玉帛之容焜燿于下而环佩之音铿锵于中固已为诸侯之雄也自后彊君桀主日益侈矣东西五里南北千步采玉砂以莹础布金椎以隠路者秦之骊山阿房也璧门鳯阙上栖金爵缭周墙以百里而终南泰华之气上下而交错者汉之长杨五柞也嘉木崇冈蔽亏杳冥而珍台闲馆间见层出于幽深者唐之玉华九成也方其作而未毁固极侈以増丽五都之豪杰足留而目注者彷徨而不已然而千载之后皆漫灭而不记又况屈原之宅哉自沈沙之告终凡几易于星纪观陵谷之迁变想丘陇其已毁而后之人犹于荒榛野蔓之间求髣髴于田里而谓屈原之在是也噫吁悲哉独何为然岂五方之异俗惟楚人之为贤秦晋汉唐之址已泯絶而无有至于此而独传考厥俗之所托实祝融之世臣能遗迹于不朽矧郢中之旧京然今也平原旷野上下禾黍九嶷云梦之间水波烟云之容轮囷浩荡而弥漫于九土其章华之馆兰台之宫亦不知其处所矣予于是瞻怅久之然流涕而后知名节之可尊而富贵之为不足恃也渔者闻而笑之曰子真知吾三闾大夫者欤观此荒芜寻常之地岂昔者所以被放逐而不忍去者也闻其始也渔父语之而不从其终也宋玉招之而不来卒自于鱼腹邈神游于九陔曩云懐乎故都今何不少留而徘徊也余曰封狐雄虺象蚁壶蜂层冰积雪流金铄石之域当凛而夏宜燠而冬生于四方为物之凶然吾知其为异可前备而不逢惟楚国之众士实同质而异心吾不量其有毒故见放于江濵然则彼可畏欤此可畏欤虽渔父之见告使扬其波焉如诵招防之哀唱亦小智而大愚所以赴江流而不悔其何爱于庐
　　神女庙赋
　　汉武帝既封泰山之五年临朝而叹曰朕念元元之民未防休德周览中土以施惠泽而南方以逺故独弗及也朕甚悯焉是冬诏发佽飞羽林之士简车骑之众盛清道之仪天子御雕玉之舆服龙文而驾鱼目系蒲梢而骖蹑云至于盛唐望九疑登天柱薄枞阳而出休于琅琊天子大恱作盛唐枞阳之诗命协律都尉延年歌之以觞羣臣酒未半天子戚然不怿时东方朔枚皋侍因进曰陛下不怿臣敢请罪帝曰朕适望琅瑘之上忽然云兴其气甚异因感髙唐之事闻楚阳台之山下有神女旦为朝云暮为行雨朕心慕之异时诸方士尝言仙者非求人主人主求之乃可致今巡游天下冀一覩列仙之属而莫获焉殆朕之德不如楚王能有所遇也是以不怿朔跪曰楚王诸侯耳有臣宋玉善为防词感动神女见梦于王臣尝笑之玉安知神女若臣者乃知之上意乃解命谒者给朔札使为之赋朔即献辞曰径西那之绵邈兮积阆风之崇基缭玉墉以千里兮右翠水而左瑶池昆仑层峙以嵯峨兮弱渊周流而逶迤中龟台之清都兮垲洪敞而甚治粲丹房与室兮罨浮云而上齐谅丰隆列缺之矜工兮斧雷霆而斵之疏悬黎以代础兮莹结绿而饰墀虞渊倒景而下射兮光反激以交驰萃飞仙之游遨兮饵若英而咀琼枝戛丛霄之灵璈兮歌白云而忘归状愉乐而不可殚兮非羽轮其莫窥帝九灵之少女兮其名曰瑶姬受素书于紫清兮含洞隂之华滋习玉瑛而厌处兮乗回飚以长辞狂章大翳为之奉辔兮防苍虬而驾白蜺驭八景之玉辖兮曳纷纶之云旗载灵气而轻举兮朅鸾戾而鸿飞涉巨溟之层波兮予将蹑乎南箕聆梦泽之雄爽兮漭天水之相围介青丘而澶漫兮奄髙唐以冥迷忽意乐而延伫兮弭绛节以徘徊胠丹笈授夫神禹兮靖九土而安柔只下民懐斯遗烈兮即石化而为祠象琼光之华阙兮骞辛夷以为楣矫藻栋以埀虹兮列药房而张薜帷羣峯连巻而十二兮烂云屏而扬辉俨玉立而正中兮貌渥饰而具宜沐兰泽而含若芳兮被袿裳而绣衣炯黼妆之丰丽兮澹聨娟之脩眉肃容华之拱侍兮纷环佩之陆离神武蹲而抱关兮夕夹陛以文貍元猨悲吟以度曲兮女娲倚歌而舞冯夷三足乌往来为之使兮讯东华之灵妃迹逍遥乎中区兮亮素节之靡移鼻祖祝融之裔子兮窃息妫以荆尸蛊文夫人于前兮后又夺郧阳子之妻豹舄毙掩而攘内兮蠭目暴芉而防訾黒要挟夏而与居兮于菟盗防而遂孳世朋淫而上蒸兮尝见刺于湘累横下臣繄宋玉兮揆暴戾之不可防称先王尝与灵游兮荐枕席而嬖私今胡为而复遇兮意托讽于防词启后世之瞽惑兮诳魄化而为芝曰媚而服焉兮则与梦期宜圣鉴之孔昭兮独超悟于昨非厉皇荒德而慢神兮祸源起乎龙嫠悼卫蒯之失国兮艾貑实发其乱机屏宓妃而却玉女兮幸后王之三思
　　暑赋
　　盛夏之月风缄土囊郁伏不兴阳亢滋骄铄石流金草木焦燃气蒸山冥客有过其主人而告之曰吁可畏哉暑也吾闻朝歌令质从魏公子浮清泉沈李寒氷嗟人夀之无几须富贵其何时南皮之乐仆甚慕之试与主人观乎崤函伊阙东西二京平乐毕圭长杨上林髙台虽毁故基犹存菀多灌丛交映纷披渭濵雒脽弥皋被隄风来鸣枝影为参差散于平中清清泠泠凉丘爽垲不烦不蒸则主人亦欲游此耶试又倘佯四方彭蠡洞庭太湖松江龙渊金隄汨罗潇湘澒洞浩弥周数千里蛟龙郁怒水涌波起山谷震动草木尽靡观者凛然髪立胆悸则主人亦欲游此耶试又薄游东陲四明会稽衡庐九疑玉霄赤城髙唐峨大木干章崇巘百常含隂翳阳不见日光深林窅防六月飞霜则主人亦欲游此耶主人笑曰佚者不视劳崇者不观庳终身戚戚何时乐为今夫大农之廛栉比钩聨谷稚草壮尽日出田携畚荷锸长后幼先此人适逢线溜滥觞如骥赴泉齐鲁之交平原广野黍稷盈畴百里一舍脩途曼曼日赫如赭车殆人罢汗浃牛马举首而望不见尺瓦此人思得一丘息肩税驾止咸阳游邯郸郑卫溱洧五都之间迎夏之阳羣女条桑柔柯沃叶宻不蔽日炎光下射肤泽沾溢此人登菅蒯之庐意犹自得从军出戍奉车挟毂象弭百钧甲裳三属荷戈而趋足蹈平陆毒阳中体炎沙燀目此人幸得休徒止舍以为至足绳枢之子宅不盈亩外逼阛阓纵步无所行者接武如穴中鼠衔窭龃龉偪仄环堵中置锜釜炀灶之烟烦寃勃郁冲牖袭户此人困于烦嚣不得动作一见天宇犹以为乐今吾偃仰一室息交谢客散髪弛带萧然终日客犹喔咿不满于色岂不过矣哉客无乃贪佚忘劳挟崇弃庳视我之居若绊若縻僒不得驰我实不然自适其适以休以息其乐易给故自居此室而暑不我疾也且夫与客清谈相对危坐隠几吾心湛然清若止水屏嚣尘其已除视纷华而不起尚何有于暑哉而当世之士恶寒附炎焦烂不止弗内省于厥躬徒归咎于一气亦甚矣客起而谢曰仆小人也实不及此
