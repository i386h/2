　　提要
　　【臣】等谨案文选六十卷唐显庆中李善受曹宪文选之学为之作注至开元六年工部侍郎吕延祚复集衢州常山县尉吕延济都水使者刘承祖之子良处士张诜吕向李周翰五人共为之注表进于朝其诋善之短则曰忽发章句是徴载籍述作之由何尝措翰使复精核注引则陷于末学质访防趣则岿然旧文只谓搅心胡为析理其述五臣之长则曰相与三复乃词周知秘防一贯于理杳测澄怀目无全文心无留意作者为志森然可观观其所言颇欲排突前人髙自位置书首进表之末载髙力士所宣口勅亦有此书甚好之语然唐李匡乂作资暇集备摘其窃据善注巧为颠倒条分缕析言之甚详又姚寛西溪丛语诋其注扬雄解嘲不知伯夷太公为二老反驳善注之误王楙野客丛书诋其误叙王暕世系以览后为祥后以昙首之曽孙为昙首之子明田汝成重刋文选其子萟衡又摘所注西都赋之龙兴虎视东都赋之干符坤珍东京赋之巨猾间舋芜城赋之袤广三坟诸条今观所注迂陋鄙倍之处尚不止此而以空疎臆见轻诋通儒殆亦韩愈所谓蚍蜉撼树者欤其书本与善注别行故唐志各著录黄伯思东观余论尚讥崇文总目误以五臣注本置李善注本之前至陈振孙书録解题始有六臣文选之目葢南宋以来始与善注合刻取便参证元明至今遂辗转相沿并为一集附骥以传葢亦幸矣然其疏通文意亦间有可采唐人著述传世已稀固不必竟废之也田氏刋本颇有删改犹明人窜乱古书之习此本为明袁袠所刋朱彛尊跋谓从宋崇宁五年广都裴氏本翻雕讳字阙笔尚仍其旧颇足乱真惟不题镂板讫工年月以是为别耳钱曽读书敏求记称所藏宋本五臣注作三十巻为不失萧统之旧其説与延祚表合今未见此本然田氏本及万厯戊寅徐成位所刻亦均作三十卷葢或合或分各随刋者之意但不改旧文即为善本正不必定以卷数多寡定其工拙矣乾隆四十六年三月恭校上
