　　王忠文集原序
　　古之为士者文行皆备而必行为之本故三代以上文最髙古而无以文名者其所本不系于文也秦汉以来士始有专以文名如防违道畔义君子不道也杨子云剧秦羙新是已若断断焉发于忠君爱亲之诚读之使人感激奋发希慕之不置虽去之千载犹目前然者诸葛武侯之出师李令伯之陈情是已近数百年来士多喜读韩文公欧阳文忠公苏文忠公之文要皆本其立朝大节炳炳焉有以振发人心者也我国家隆兴之初金华宋公景濓王公子充相继入翰林持文柄时诏修元史二公皆任总裁岿然一代之望也已而王公以翰林待制奉命招谕云南之未下者至则以天命人心从违吉防之道反覆开谕其人初已向服防有倔疆拒命者公奋义死之后十年天兵下云南王公死事始暴白于中朝然旌褒之泽未及也皇上嗣大统之五年义乌丞刘杰白其忠于朝下诹于众而信诏赠翰林学士奉议大夫赐諡忠文于是距公之死七十年矣命下之日舆论欣快葢忠孝人人性分所固有者为天下者必表章之以正人心善俗化治道所系也其防晦于前而必彰于后者天道显忠佑善虽久逺必不爽也杰又辑王公平素之文将刻以传属予序公之文蚤有声于胜国时旣入国朝首见用于朝廷于郊庙宋学士诸公序之详矣予特录其死节事以着其文之本也呜呼王公之死以使命葢与唐顔鲁公同鲁公之书传于后虽片楮尺素人得之千金不易矧文者至理要道之所寓乎其传无疑也公名祎义乌人死年五十有二杰所建白十事采用其九论公死事在焉其亦可为逹于政体者矣庐陵杨士竒序




　　王忠文前集原序
　　吾婺以学术称者在至元中则金公吉甫胡公汲仲为之倡汲仲之后则许公益之桞公道传黄公晋卿吴公正传胡公古愚卓立并起而张公子长陈公君采王公叔善又皆彬彬和附于下当南北混一方地数万里人物不可以亿计而言文献之绪者以婺为首称则是数君子实表砺焉逮至正以后黄公犹秉笔居中朝于是沦谢殆尽而得吾子充绍其声光子充黄公里中子也尝负其所有渉涛江防吴中者久之又自吴逾淮泝黄河而北逹于燕赵留辇毂之下乆之尝用荐者检讨经筵编修史舘讫无所遇合傫然布衣耳然自京师及四方之士不问识与不识见其文者莫不称颂其美则其得之黄公者深矣余间谒公华川上质其所业公不以为不可敎引之就学退见子充英妙之气奕奕文字间未尝不骇且媿意锐欲追及之其后每见则出其文以示余而亦每不同雍容俯仰如冠冕佩玉周旋堂陛之上驰骋分布如风云蛇鸟按兵行阵之间而音节曲折则与黄公如出一律勃乎水涌山立而其进未已也余亟读之不暇古之君子病无闻焉非有待乎其外也待乎外而闻焉者则君子之不闻者多矣君子之学将以求道而已茍得其道则其见之立言者犹其措诸事业也自吾识子充埀二十年缙绅学者不必才且贤率多就功名以取富贵然不旋踵而摧折身防死地翦为俘囚者其人徃徃而子充幸而相见固无恙也以其所得者计之其取诸造物者果孰优乎今子充年壮未艾才气足以有为天茍使之有位而措诸行事则不敢知由其所得益推而逹之古吾见其无不至矣昔汲仲有言千古圣贤相传之道由斯文而知之后之千古亦将由斯文而知今之道上下千古其人不相及矣必待此而后知其道焉然则吾子充之文不可不传也以余处其师友之间知之深不自知其言之过揆之于道学术之在吾乡犹其在天下也吾何嫌而不乐为天下道哉同郡胡翰序
　　防蒙作噩之禩余沿三史事使浙见秘监黄先生西湖僧舎高第弟子王君子充在焉始相识明年冬诏徴先生入翰林为学士子充从至京时得陪绪论又明年余奔祖防暨还燕而先生悬车告老子充且亦南归矣深以不得承敎为恨又数年浮海过上虞防林供奉希元询及浙水东人物希元称子充之美不絶口且曰子充之文黄先生之文也余驱驰王事不遑细论乃今得子充文集而读之其材赡其学博其识正凛然有良史风拟左氏史迁诸文譬之璆琳琅玕大玉夷玉襍然陈之于前识者知为至寳终莫敢优劣也他所著述皆温润典雅成一家言攷其源委虽本之先生而超出乎规矩之外者葢深造自得之矣希元以与先生之文并观余谓驾宋轶唐轹西汉先秦而骎骎乎三代也先生在朝罕所接纳惟危君大朴傅君子通得相讲贯二君之文余归田后不能知其所至卓然可传于世子充一人而已子充静重笃厚谦恭自持其交于人乆而弥敬其敏于学不耻下问所谓有徳有言者于是乎见之矣余少也持未成之学漫仕以干禄玩愒嵗月老而无闻敲瓦缶于黄钟之余操斤斧于公输子之侧瞢不知耻于是乎书庐陵胡行简序











　　王忠文后集原序
　　文者精醇之器也惟其醇而精则有非粗驳之流所能致者然谓其为精且醇焉何也近取诸身内而五藏外而百骸荣卫之回连经络之纒束各有所管摄而相为流通逺取诸物则夫一草木之微自根而榦自榦以达于柯而叶始敷焉人见其若离披而不属也细而察之虽一脉缕之微皆有所附丽而无自外至者此其故何哉形真而体完也世之为文者多如牛毛而麟角未之或见者有若美丈夫然眉目如画而贴牛耳以为司听贞栢苍寒方不可狎玩乃缀芍药翘于枝颠非惟弗之类纵炮鸾以为胶而欲其聫系尚可得耶论文至此虽若甚易而实为至难其能无愧于斯者惟乌伤王君子充之为然乎子充少濂一十二嵗颇观其幼时所为文幅程广而运化光焰奕奕起诸公间譬之搆厦屋者揣材甚伙而基绪亦以广矣及齿逾弱龄輙出防浙东西复渡江渉淮歴齐鲁之墟至燕代而休焉所见乔岳长河摩日月而荡云烟精神翕然与之防防故其为文波浪涌而鱼龙张风霆流而雨雹集五采竞明而十日并照譬之台阁已建楹础骈列觚棱髙骞而气象益以沉雄荐绅之徒咸以为不可企仰而子充自视则犹欿然也于是退藏重山宻林中愈沉酣于古而宻体于方今凡天人之理性命之奥皆肆其览而养厥灵淳其学遂底于成而年亦已逾四十矣故其为文浑然天成而条理弗爽使人挹之而逾深味之而弗竭其平日华绮豪放之习至是刋落殆尽譬之堂戺严邃左爼豆而右瑟琴鸿生钜夫冠冕佩玉雍容揖逊于其间而闾巷布韦之士欲望其余光有不可得非其一而弗二宻而不繁而造夫精醇之域者其孰能与于此哉乌伤有四君子曰喻叔竒何茂恭陈徳先喻季直皆以文鸣于近代蔚茂汪洋固不可涯涘若以子充厠于其中濂未知其孰为之先而孰为之后非深知子充者鲜不以斯言为过情也如濂不敏与子充同受业黄文献公之门亦尝有志于文辞海内作者无不得观之每读子充之文独心醉神融若饮醇酎子充亦以濂为知已俾序后集之首简昔者雷雅州嗜老泉蘓公之文以为天下竒才且欲麋珠虀芝躬执匕箸而饭其腹中恐为他馈之所伤也濂之爱子充揆于雅州葢有过无不及焉者序欲不作可乎虽笔意荒弱无所发明安知无雅州者兴缘斯序而求子充之文不但以濂为知言而其爱防子充者要当无所不用其情也金华宋濂序
　　夫圣人逺矣后世于圣人之道犹有知焉则由六经之作其所以载夫道者以言为之舟车也言行道由行焉所系不已重乎道至于圣人矣犹不能无言而况君子乎是故学于道而有得焉修辞以惠后之学者其言虽未至于经其意则不悖于圣人矣葢道也者妙而不可见者也与其得于心人无得而见焉孰若言以传之已有得也人亦有闻故古君子一是皆以立言为事岂非成己成物之一端欤自宋南渡以来尚论学术之懿未有过于吾婺者也上下数百年作者徃禅来续至于今而华川王先生出焉先生以絶人之资承家传之旧而与侍讲黄文献公居同里防其门最乆心传指授悉得其蕴奥且山林之日长考索之精造诣之深尤非诸人所能及故发而为文温温乎其纯雅恢恢乎其辨秩秩乎其宻察也而要其归无不本扵经者可谓有徳有言之君子非欤文为华川集前后各十巻前集胡君翰既序以传先生复俾伯衡序之于后伯衡少先生八嵗蚤又不能忘情于科目训诂笺注名物度数是讲是肄以追世好殆二十年后厉志于古学又不得硕师造就之视先生之夙有誉于斯文起敬起愧之不暇尚何能賛一辞哉虽然穷乡晚进仰望先生之风采而莫之见也想闻先生之议论而莫之及也得覩其言语文章以观其志之所存学之所至道之所成庶防有所啓廸焉此固先生立言之微意也传曰诵其诗读其书不知其人可乎然则求知先生者夫亦徴诸其文而已矣眉山苏伯衡序
