　　提要

　　臣等谨案蚓窍集十巻明管时敏撰时敏初名讷以字行华亭人九岁能诗长师事刘俨杨维桢而友袁凯洪武九年征拜楚王府纪善从王之国后进左长史事楚昭王桢二十五年乞致仕归里桢请命于朝留居武昌禄之终身筑室黄屯山命曰全庵而名其集曰蚓窍盖取韩愈石鼎聨句语也集为桢所刋行其诗舂和雅淡风格多近唐音不染元未雕绘之习张弼序董纪集厯数松江诗人而独谓时敏之清丽优柔足与袁凯方驾盖不诬也时敏又有秋香百咏还乡纪行诸编在集外别行见周子治所作全庵纪中今俱未见又时敏晚而生子楚王名之曰延枝亦有集行世今并不可覩矣

　　●蚓窍集原序

　　云间管公字时敏竹间其别号也公蚤歳读书三泖之上钟山水之秀为文儒尝师事廉夫杨先生执经座下为髙弟故其心术之正学问之博文章德行之精醇用于文明盛世而功名事业炳乎其有耀也壮年时仕为楚府纪善扈从来武昌邂逅旅邸一见知为佳士读其诗风格髙古一一可追古之作者其五言乐府有汉魏体五七言律诗多出盛唐舂容雅澹譬之黄锺大吕之音孰不为之洗耳者乎别去二十年余忝膺教命来是邦复得朝夕见读公篇什愈老愈健尤见笔力所至而制作之富倍于昔时如碧海冲融天光万顷汪洋莫能窥其涯涘如华峰千仞翠屏倚空莫能陟其层顚如淸庙之瑟朱弦疏越一倡三叹有遗音何其快人意欤仆以衰迈之年得闻治世之音启余多矣公晚年由纪善升秩长史其忠诚爱君之心一于诗歌发之而学问德业其所从来者逺矣集中诗凡数百篇尝名其集曰蚓窍盖取韩子石鼎聨句之语而名之也以余观之公之作铿鍧炳耀流出肺肝钟山水之秀而鸣治世之音是奚足以名其集欤以是名其集者公之谦德也辱不鄙征言序其篇首讽咏再三不能自已于是乎书洪武三十一年正月望日楚府教授庐陵吴勤序

　　●蚓窍集原序

　　唐人有言文章实小技于道未为尊此非知言者也夫道之显者谓之文故道以文而传而文所以为载道之器也道不违乎人人之为文而违乎道岂足以言文哉况乎声音之道与政通于其淸浊髙下疾徐促缓而道之升降政之美恶俗之隆污系焉盖审音足以知政闻乐足以知德皇明受天命一海内光岳之气浑融纯全灏灏噩噩之文沨沨乎其音播之宇宙时则楚有旧学之臣曰云间管公时敏辅翼匡赞开陈启沃凡二十五年而贤王仁厚明恕寅畏恭俭之德闻于天下公自纪善超左长史属余来备员陪位得日聆其绪言余论间出示所自作诗一编古制近体通若干首舂容乎其意度铿鍧乎其节奏追琢乎其文章其言丽以则其思深以逺其义葩而正温柔敦厚不迫不切方诸古人亦未多让顾自谦曰蚓窍集夫古之人修于身者不必施于政事施于政矣无待见于言语然德之发于口者为言盖和顺积中而英华发外自有不容已者公立身行事实践其言故发为文辞和平中正其可传于来世也较然矣而公之所可以埀不朽者则不在是也贤王侈公之德欲寿其传命刻诸板余不敏僣序其朔云永乐元年歳次癸未春三月望日奉议大夫楚府右长史山阴胡粹中序

　　○四言古诗

　　素堂

　　湘水之涘衡山之阳有美一人于焉构堂维榱维桷匪雕匪斲不尚其华爰复其朴堂之翼翼人之谆谆匪谋自适爰以养亲于时兴止于时甘旨朝夕弗违惟亲之喜委顺在天穷达是宜居易而行心毋外驰奢则蛊情俭以养德用戒后昆服之无斁

　　来德堂为中都国学分教林敬伯赋

　　木之有本其干乃强水之有源其流乃长吉士好修德日以茂德为本源克昌厥后猗欤林氏奕奕名门维祖维祢曰子曰孙匪德之来曷裕其昆斯堂翼翼以示将来保而勿射维德是积我言尔铭于堂之壁

　　毘陵薛黉自号渊泉求铭其室

　　五行在天惟一生水水之有本源原于始潴之斯止孰曰匪渊道之斯达孰曰匪泉盈科后进弗停一息维静维深厥本乃立弗澄则浊罔覩山岳弗挠则淸毫髪乃明挠之澄之淸浊辨焉匪水若兹惟人亦然心而弗澄百欲是兴身而弗挠一私靡扰昔在圣贤观厥川海以近取譬道无不在允尔泓父逝者若斯我告尔铭永言服之

　　○五言古诗

　　白马篇

　　白马行且嘶玉勒锦障泥扬鞭羽林郎宿卫通金闺一朝赴邉急转战玉关西长戈挥白日髙旌巻彩霓驱山塡火井累石起云梯烽举星芒乱弓开月影底分符降赤勒飞箭定渠黎功许韩彭并名与卫霍齐主恩谅可报国难讵难挤孰甘局乡邑门巷闭蒿藜

　　逺游篇

　　御风日万里逍遥登逺游竦身青冥表瞬息四顾周目力隘八极壮志局九州岛泰山小封垤沧海眇浮沤俯视澒洞间万彚等蜉蝣浩歌挟洪崖长啸访浮邱回飊欵阊阖谒帝十二楼酌我紫霞觞遗我靑鳯裘授以长生诀歴刼三千秋从兹谢流俗大块同悠悠

　　少小从行伍二首

　　少小从行伍而今作饮徒博钱寻剧孟买剑问风胡结伴军中戏嗔人醉后扶黄金三万镒誓娶陆家姑
　　少小从行伍千金办武装日骑官廏马夜宿酒家娼（阙）已宁论死临危肯畏强蛮夷知姓字不敢犯封疆

　　铜爵妓

　　铜爵髙台上西陵古墓边君恩徒自重妾貌竟谁怜羞掩歌时扇愁登舞后筵惟余漳浦月三五夜空圆

　　二隐图

　　溪上放舟回山中负薪暇一笑偶相逢于焉遂淸话宁知有秦晋况复论王霸回首总忘机靑山夕阳下

　　星夕

　　牛渚淸且浅鹊桥西复东黄姑仙驾动天女锦机空别泪零清露离魂惨惊风如何一水隔一会一匆匆

　　监生王廷采分教中都国学植槐于庭扁曰槐轩求题

　　虚星孕良木植彼文苑中教雨既沾溉枝叶何靑葱常刑庶无犯守护克有终歳久已云茂郁然若云从诸生俨成列弦诵怡清风伊人抱良材树德匪尔同当知进修者亦由滋培功

　　秋夜

　　不寐耿长夜揽衣徒歔欷阴虫吟近砌凉飊动疎帏素抱殊未释淸欢念多违岂惟南飞鹊兹夕无所依

　　写懐

　　朝回憇公馆残阳坐来暝疎雨过髙林华月出东岭流光晨复晡朱颜岂吾永故园归何时悠然发深省

　　病后喜雨谢同寅纪善贝季翔见过

　　夜月忽离毕今晨雨滂沱秋声落庭树水气没池荷岂云三伏中感此凉意多神淸体自适顿然释烦痾明当谒公馆仰谢数经过

　　晓起

　　直馆晓晴初官淸逺尘俗怡然对简编庶矣忘荣辱遥峰拥归云髙城澹微旭落尽冻靑花江南雨新足

　　听松楼为国学林敬伯赋

　　凉飙度中林长松发淸响震荡出岩峦萧骚纳天壤初疑繁弦作复讶惊涛上倐尔若崩奔悠然亦平广大音本无声至理奚有象复动阳始生姤静阴已徃达人居髙明会兹适眞养心静听转聪虑屏神逾爽岂彼廛市间啾啾局尘坱好招陶隐居山中一淸赏

　　和陆伴读闿过梅根

　　江行惬素懐遐览匪游冶开篷望九华隐隐白云下素波明逺川靑天入平野烟树澹欲无风泉断还泻因知渼陂舟差胜习池马斯游乐未央我辈胡为者

　　过芜湖望天门山

　　扬帆下芜湖梁山在吾目大江从西来天门势如束川日波上明野树烟中绿良朋有佳制每媿貂难续顾兹欲忘言何以慰心曲

　　和吴教授晚泊大信阻风

　　楼船遡流上髙帆阻长风大江浩无际一气何鸿蒙回望天门山峩峩相长雄洪涛走其下屹立西与东不有浮海叹岂无济川功大鹏度寥廓斥鷃守卑丛物理固有定世事徒匆匆张灯启华筵一笑杯酒中

　　上慈母矶

　　危石立江中扬帆泝流上潮回慈母矶木落和尚港野阔风景明秋凈天宇广西望淮山靑持杯惬吾赏

　　游寳林寺

　　金碧绚楼台禅关向此开水分双涧出云拥万峰来古佛遗香火残僧话刦灰黄尘吾老矣何幸一徘徊

　　独坐

　　独坐茅堂上长吟不下床乱书堆几席疎雨过陂塘野燕冲帘入汀花度水香应须存晚计次第学畊桑

　　答友人雨中见寄

　　雨声连晓夕牢落闭幽扉屋漏频移榻江寒更觅衣野乌饥自下水鸟湿还飞尔亦忘机者终同理钓矶

　　过田家

　　偶过吴家墅残阳半掩扉逺汀低树宻小径杂花稀新酒浮篘酽鲜鳞坠钓肥年来任疎懒一醉即忘归

　　郊居二首

　　移家虽偪侧只此足幽居茅屋三间小春田廿亩余人来论稼穑自学把犁锄东舍催租急烹鸡酌小胥
　　春城俯平野草色静柴门鱼蟹舟中市桑麻屋后村遣儿招社饮为客置盘飱老计能如此陶然付一尊

　　赠洪山龙门海禅师侍者眞净

　　祝髪名眞净从今净已眞诵经当作佛持律定过人江月空中夜山花物外春龙门多寳藏种种是奇珍

　　送宗师还玉泉寺

　　省师何处去古寺玉泉清山笠披云过江船趂月行拈香参祖塔乞食到王城此去传心印应知道可成

　　和季翔梦与海禅师看牡丹之作（师时以事在京栽松）

　　道者栽松去何因看牡丹相思春独苦孤梦夜初残子已空中见师应定里观归时当记取饶舌问丰干
　　其二
　　太息东岩叟艰危命若丝当知成佛日还有受魔时短发萧萧落孤心耿耿持可堪江树晚千里动遐思

　　梦游洪山寺

　　不到东岩阁重来又一年无縁能离俗有梦只栖禅客坐茶香里僧行竹影边夕阳天未晚幽思满归韀

　　从征古州蛮回途纪驿二十三首

　　晨发渠阳郡归心一惘然离家今五月为客已中年暮雨行人里春泥去马前洪江明日到沽酒上官船
　　右发靖州
　　行行过若水古驿在江皋自是登舟险终无策马劳山云含宿雨滩石漏奔涛此地如能隐尘烦或我逃
　　右洪江
　　夜过安江驿停舟不敢行乱山藏月色暗石鼓滩声千里从军事中宵感客情不眠聊伏枕天白更孤征
　　右安江
　　早行今几里又说次潼安古驿烟中入前山雨里看江回逢断洑水落见危滩浊酒虽非嗜聊持破晓寒
　　右潼安
　　江口何年驿过逢欲雨天居人张夜火使客唤春船云起崇山外江通溆浦前鸬鹚滩已过今夕解衣眠
　　右江口
　　归舟春水疾举棹即辰溪岸转山疑动江分路欲迷乾坤千里客风雨五更鸡自愧如萍迹飘飘未定栖
　　右辰溪
　　武陵溪上驿因纪武陵名树列千旗势江喧万鼓声生苗今向化落草久无兵好载铜槽酒沿山看晚晴
　　右武溪
　　王程不敢缓四日下辰阳古木将军庙春波使者航城依山势险江纳雨声长莫上观澜阁伤心在异乡
　　右怡溶
　　今日行兹驿如何是北溶云涛千数里烟树万余重江黑留春棹林虚动夜舂驿官应被酒出迓甚疎慵
　　右北溶
　　天设非常险舟航岂易过大风掀逆浪暗石起盘涡行路难如此埀堂意若何榜人殊可恠不住唱蛮歌
