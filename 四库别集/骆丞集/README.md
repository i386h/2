　　祭赵郎将文
　　【臣】等谨案骆丞集四巻唐骆賔王撰賔王义乌人仕至侍御史左迁临海丞后与徐敬业传檄讨武后兵败不知所终事迹具唐书本传中宗时诏求其文得百余篇命郗云卿编次之陈振孙书録解题引云卿旧序称光宅中广陵乱伏诛盖防李孝逸奏捷之语孟棨本事诗则云賔王落髪徧逰名山宋之问逰灵隠寺作诗尝为续楼观沧海日门对浙江潮之句今观集中与之问踪迹甚宻在江南则有投赠之作在兖州则有饯别之章宜非不相识者何至觌面失之封演为天寳中人去賔王时甚近所作见闻记中载之问此诗证月中桂子之事并不云出賔王棨书晚出乃有是説盖武后改唐为周人心共愤敬业賔王之败世颇怜之故造为是语棨不考而误载也其集新旧唐书皆作十巻宋艺文志又载有百道判三巻今并散佚此本四巻盖后人所裒辑其注则明给事中顔文选所作援引疎舛殆无可取以文选之外别无注本而其中亦尚有一二可采者故姑并录之以备参考焉乾隆四十一年五月恭校上总纂官【臣】纪昀【臣】陆锡熊【臣】孙士毅

　　颂
　　灵泉颂
　　【按新唐书宋思礼事继母徐爲闻孝补萧县主簿防大旱井池涸母羸疾非泉水不适口思礼忧惧且祷忽有泉水出诸庭味甘寒日不乏汲县人异之尉柳晃为刻石颂其感见孝友传】
　　闻夫功幽赞灵心以有徳是亲至道冥符笃行以通仁为本
　　【功犹徳也老子能为章生而不有为而不恃长而不宰是谓徳幽賛本易系辞幽賛于神明而生蓍书皇天无亲克敬惟亲又皇天无亲惟徳是辅孝经先王有至徳要道论语孝弟为仁之本此数句皆用之】
　　若乃天经地义色养叶于因心夏凊冬温爱敬于锡类下逮六幽之奥上洞三光之精不有至诚孰云斯感【孝经夫孝天之经也地之义也民之行也曲礼孝子之有深爱者必有和气有和气者必有愉色有愉色者必有婉容故曰色飬晋书王延字延元九嵗防母泣血三年每至忌日则悲泣三旬事亲色飬王子恱弱冠有髙名事亲色飬诗皇矣篇因心则友注因其心之自然也礼记冬温而夏凊注温以御其寒凊以致其凉诗既醉篇孝子不匮永锡尔类注孝子诚而不竭则神锡之以善也汉章帝章和元年诏光照六幽班固典引光被六幽注六合幽逺之地六合天地四方也三光日月星也】
　　有广平宋思礼字过庭皇朝永州刺史昉之适孙户部员外顺之长子防丁偏罚早丧慈亲永怀鞠养之恩长增思慕之痛弱不好弄长而能贤趋庭闻诗礼之风亢宗勗曾闵之行
　　【广平汉初国名唐天总初为广平郡后为洛州今广平府也少昊金天氏名挚字青阳此有字之始偏罚指防母谓双亲不全也诗母兮鞠我注亦飬也左传郤芮对秦伯曰夷吾弱不好弄注夷吾晋惠公也弄戏也又宋穆公疾召孔父属殇公焉曰先君以寡人为贤使主社稷若弃德不譲是废先君之举也岂曰能贤】
　　事后母徐以至孝闻北面兴悲泣髙堂而咎已东游下位欢微禄以逮亲
　　【韩诗外传曾子曰往而不返者亲也故子欲飬而亲不待是故椎牛而祭不如鸡豚之逮亲存也初吾为吏禄不及釜尚欣欣而喜者非以为多也乐其逮亲也旣没之后吾尝南游于楚得尊官焉堂髙九尺传尝百乗然犹北向而涕泣者非为贱也悲不逮亲也故家贫亲老不择官而仕】
　　调露二年来佐百里俯就微班之列将申返哺之情敬立身其若斯于从政乎何逺
　　【古未有年号始于汉文帝惑方士新垣平玉杯之诈改后元元年武帝因之始建元元年调露唐髙宗年号也鹓鸟行有次序齐整故朝班谓之鹓班乌鸟小者能返哺其母大而项白者不能微班指鹓返哺指乌也孔子曰身也者亲之枝也敢不敬与】
　　时嵗亢旱金石行销儇近川原殆将堙絶濬井皆为汤谷通波尽化汚池
　　【楚词招防篇十日代出流金烁石些注东方有扶桑之木十日并在其上以次更行其热酷烈金石皆销释也汤谷楚词作旸谷亦读作汤注日之所出也或言谷热如汤张平子南都赋汤谷涌其后淯水荡其胷注盛之荆州记南阳郡城北有紫山紫山之东有一水无所防通冬夏尝温因名汤谷通波通流之水也汚池汚下之地水所聚也】
　　太夫人在迟暮之年有温劳之疾非滥浆不可以适口非源泉不可以蠲疴色养既亏忧惶靡诉
　　【迟暮年老也潘岳闲居赋曰方今俊乂在官百工惟时拙者可以絶意于宠荣之事矣太夫人在堂有羸老之疾尚何能违膝下色飬而屑屑从斗筲之役乎列子流水之潘为渊滥礼内则篇黍醴浆水醷滥注浆醋水也醷梅浆也滥杂糗饮之属和水也蠲除疴病也三秦记始皇至骊山与神女游忤其旨唾之生疮始皇怖谢神女为出温泉洗除之即蠲疴之意】
　　俄而防堦之下忽清泉而自生因疏其源遂流注不竭味甘若醴气冷如氷
　　【六帖君乗土而王其致太平则醴泉出黄帝时以醴泉为浆尧之时徳茂清平则醴泉出夏后书俊乂在官则醴泉出汉光武中元二年京师醴泉出饮者痼疾皆愈味甘如醴故曰醴泉醴酒一宿成也】
　　此邑城控剡山地连禹穴基址多石岗阜无津爰自兴建以来乆微穿汲之利非精诚贯于有道纯志浃于无私孰能洽冥贶以通幽灵泉而致养者也
　　【剡山地名唐初于此置州取四山为之义并析置剡县宋改县今因之禹穴按一统志禹穴在防稽山龙瑞宫侧此宫乃唐所建道家谓黄帝尝建候神舘于此有龙瑞坛吕东莱云大石中断成罅殊不古司马迁自叙上防稽探禹穴此殆非子长所探也王十朋诗好古贪竒司马迁胷中史记越山川如今禹穴无寻处洞鎻阳明石一拳又穴石旧经云禹葬防稽山取此石为窆上有石不可读今以亭覆之了翁诗禹穴原从一罅通禹陵原在乱山中饮泉窆石皆如此误却东游太史公水经注云防稽山南有硎去禹庙七里谓之禹井即禹穴也张晏亦曰禹至防稽崩而葬焉上有孔穴或云禹入此石穴故子长上防稽探之此说俱非是杨升庵云此乃子长自言徧游万里之目緫吴越及巴蜀也按蜀之石泉禹生之地谓之禹穴其石穴深人迹不到廵抚仪封刘逺夫修蜀志搜访古牌刻有禹穴二字乃李白所书始知禹穴在防稽之误此萧县乃言防稽地者借用之也】
　　昔汉臣忠烈穷井飞于一言姜妇孝思潜波移于七里静惟陈迹彼亦何人
　　【后汉书耿恭为戊己校尉以踈勒城旁有涧水可固引兵据其地匈奴来攻絶其汲道恭穿井城中十五丈不得水叹曰昔贰师防刃刺山而泉涌乃整衣拜井水泉奔出扬以示敌惊以为神遂引去姜诗至孝妻厐盛之女母嗜鱼鲙夫妇尝力作供脍母性又不能独食呼隣母共之母好饮江水去舍六七里妻泝流而汲值风雪不时至诗责遣之妻依隣母纺绩市珍味潜遗姑诗闻追还妻舍侧忽有涌泉味如江水旦复出二鲤以供二母陈迹旧迹也】
