　　提要
　　
　　臣等谨案王常宗集四巻补遗一巻续补遗一巻明王彛撰彛字常宗其先蜀人本姓陈氏父仕元为昆山教授遂迁嘉定洪武初以布衣召修元史赐金币遣还寻迁入翰林以母老乞归飬自号妫蜼子后以魏观上梁文事与髙启并诛明史文苑传附载赵埙传中其集本名三近斋稿宏治中都穆编为文三巻诗一巻刘廷璋浦杲又辑补遗一巻今世所传钞本又有续补遗一巻不知何人所辑考其体格与全集相类似非赝作也王士祯香祖笔记曰王征士集都少卿元敬编元敬称其古文明畅英发又或以为呉中四杰之一以常宗代张来仪者今观其诗歌行拟李贺温庭筠堕入恶道余体亦不能佳安能与髙杨相颉顽乎云云案彛之学出天台孟梦恂梦恂之学出婺州金履祥本真徳秀文章正宗之派故持论过严或激而至于已甚集中文妖一篇为杨维祯而作者曰天下所谓妖者狐而已矣俄而为女妇而世之男子惑焉则见其黛緑朱白柔曼倾衍之容无乎不至虽然以为人也则非人以为妇女也则非妇女而有空家之道焉此狐之所以妖也浙之西言文者必曰杨先生予观其文以淫词谲语裂仁义反名实浊乱先圣之道顾乃柔曼倾衍黛緑朱白奄然以自媚宜乎世之为男子者之惑之也云云其言矫枉过直而诟厉亦复伤雅虽石介作怪说以诋杨亿不至于是士祯所云或亦有激而报之乎然其文大致淳谨诗亦尚不失风格虽不足以胜张羽必以为一无可取则又太过香祖笔记成于士祯晚年诋诃过厉时复有之固未可据为定论矣

　　●王常宗集原序
　　
　　洪武史臣嘉定王先生常宗有遗文一编穆乡尝校定厘为四巻蔵之箧笥者二十年矣刘君子珍世居嘉定好古博雅谓是集为里中故物刻梓以传而俾穆序之惟吴为东南文献之地自汉唐以来名人魁士踵武相望至我朝洪武而益盛盖当是时修元史者三十有二人皆极天下之选而出扵吴者髙季迪氏谢玄懿氏杜彦正氏傅则民氏而先生与焉先生之文精严缜密明畅英发不为谀辞浪语以逐世好要之根据乎六经出入乎诸子百氏而其识见之卓论议之妙求之当时已不多得而况遗之百数十年之逺其可以弗传耶夫文者气之灵秀天不能自言也故以之而畀于人人惟其君子也故得之而吐于辞世之小夫鄙生非无言也大率卑庸荒怪偏曲支离而不足取固无望扵文矣是知文者所以泄乎灵秀而鬼神之甚秘也独不见夫古之人乎富贵福泽往往得之而能文之士代不数辈或其人流离痛愤阨穷饥冻而文辞始工此可以识造化之微矣先生少贫尝读书天台山中师事孟长文氏长文葢兰溪金文安公弟子故先生之学逺有端绪若文则其所自得者为多而未尝茍同扵人初会稽杨维祯以文雄一时吴越诸生多归之者先生独目为文妖作文诋之凡数百言穆于是又有以验先生之学之正推是心也岂希宠盗名以徼一时之利者哉是以史成而归身犹布衣而惟以读书奉母为事呜呼若先生者匪徒能言实允蹈之而足为乡邦之重者也先生名彛其先蜀人父东轩公尝教授昆山州学以卒嘉定昆山之接壤也故先生遂为嘉定人云都穆序

　　苏州重修孔子庙学碑
　　
　　蒲圻魏公守姑苏之明年锄暴树良无令弗肃上下洽和百废具兴惟孔子庙学弗治栋楹倾欹丹垩翳昧若不支旦暮者公頋瞻良乆愀然弗宁乃登僚宰县令丞暨尚义之士咸集乎庭喟然叹曰皇帝以神武定四海即大兴文治诏以明经六艺之科颁示郡邑使选弟子贠讲肄眀体适用之学将以人文化成天下今弊陋至是非所以灵承帝意余先捐俸为倡尔二三君子幸相与图之穹殿邃廊虔恭奉明禋灵星门神道厅繇其役为最殷吴江葛徳润尔其葺之伦堂有严两庑相向挟以六斋以通扵前门吴江寗璇汝其修之亭号采芹下临康衢来游来歌多士攸宜昆山张庸尔其整之公言未已吴江令许茂进曰邑有八士曰李庚曰曹徳曰沈璇曰姚蒙曰莫礼曰陈景曰朱士能曰葛彦祥颇有意乎诗书尊经之阁先贤诸祠游息三亭以至庖湢桥梁之属茂率其承之长洲丞丁师尹复进曰许令然矣师尹独不为乎若止善堂及中廊逹于阁者当谕羣（阙）各赋金俾里耆唐晟视其役焉公大恱惟周垣四千八百尺未有所属僚若佐遂以罚布倾助而吴县长贰驩然趋功役邑夫畚簸土绳绳而来事因大集始事扵洪武六年二月戊辰而讫功十一月如翚斯飞如云斯凝如涛斯涌深沈而洁清宏丽而靓宻一还承平旧观公率其属执豆笾具餔醢以成功告观者如堵墙莫不嗟叹自兵兴以来学宫所在蹂践为墟而姑苏幸仅存非公之至几何不与之俱废乎教授贡颖之既宣力扵始终复使训导张由请记其事以埀不朽惟苏之有学实始于范文正公盖公景佑初出守乡郡择南园之地奏请于朝而建是学为其师者则安定胡文昭公也计当时人物固尝盛矣数百载之下仰其声光慕其风烈每使人发不可企及之叹自时厥后虽守有贤愚而学之兴废靡常而二公过化之地流风遗俗终未泯也况今遭逢有道之朝而得良二千石为之师帅岂徒然哉然公兴学之志不下扵范公者也则夫坐皋比而谈经艺者其必将文昭乎明体适用之学者知骎骎而盛矣茍不务出此而摘英搴华为哗世职宠之具非区区所敢闻请以是刻诸丽牲之碑庶几苏之人士知所自朂哉公名观字杞山通周易为人洞达无凝滞如八窓玲珑见义而独为虽千万人不可遏所至輙烈烈有声同知府事李权通判徐弼推官赵霖经歴李亨知事王徳源照磨吴佐皆号一时之选与公道合故成事无难者其吴县长贰则曽黼杜徳昌丁彬督其役夫而责其成者又闾师滕徳进与黄文贵刘立陈茂金源黄廷玉也碑为修学作前代兴造各自有记皆不书唯当书者不敢拘以文体而不疏其详焉系之以诗曰姑胥之墟有川滮滮折趋南园汇而东流灵氛中启焕烂弗收自非圣师畴叶其休惟昔哲人出绾郡章乃构学宫乃建庙堂庙堂章章学宫将将拾佩洋洋来翱来翔盛衰相仍揆理则然风摧雨蚀上漏荣穿矧丁艰虞烽火相连谁一举臂持危扶颠大明当天九服攸同（叶）搀抢敛翳奎璧吐芒睠我魏公来莅大邦屡形扵言我敢怠遑既率郡封复登僚佐分劳授事具乃教条曽不一期厥功孔昭化此朽腐欝为岧嶤孰敦化原孰筑教基孰牖民彛匪孝曷为我语苏人公岂尔私耳目茍具必致其思思之如何法孝与忠二者无媿行将天通震泽有石其坚比同史官勒辞用昭无穷
