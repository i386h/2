　　提要

　　（臣）等谨案环谷集八巻元汪克寛撰克寛有经礼补逸已着録其平生以聚徒讲学为业本不留意于文章谈萟之家亦未有以文章称克寛者然其学以朱子为宗故其文皆持论谨严敷词明达无支离迂怪之习诗仅存十余首虽亦濂洛风雅之泒而其中七言古诗数首造语新警乃颇近温庭筠李贺之格较诸演语録以成篇方言俚字无不可以入集者亦殊胜之在其乡人之中不失为陈栎胡炳文之亚文士之文以词胜而防其害理词胜而不至害理则其词可传道学之文以理胜而病其不文理胜而不至不文则其理亦可传固不必以一格绳古人矣

　　夫子之墙赋

　　敏学主人与博古先生游于尼山之麓曲阜之墟造孔林之阙里瞻玄圣之攸居楩楠连云而荟鬰桧栢叅天而扶疎跸亭嵯峩而倚空杏坛甃甓而荒芜列横序之层构峙巨殿之中岿屹崇门之突兀缭周垣之廽纡主人喟然而叹曰端木子所谓数仞之墙其在兹乎吾子衣蹁跹之逢掖冠崔嵬之章甫涂抹丹铅摹冩今古行式渊骞言称求路盍为我抽思逞辞饬章绘句铺张弘丽之规缘餙高广之度极词人之炫耀亟援笔而为赋先生曰嘻夫子之墙岂今之所谓墙哉粤自二龙绕室五星降庭萃大块之清淑会元气之晶英缵圣神之华胄集条理之大成拓天下之广居开异代之文明夫是以覆泰宇以为櫩立人极以为柱存至诚而为基凝至道以为土文章为之华餙德业为之培累贯一理以为桢崇万善而为堵仁义中正乃其板载之方诗书礼乐乃其畚插之具不思不勉初何事于经营弥高弥坚又奚烦于削镂悠乆无疆乃斯墙巩固之迹博厚配地乃斯墙延袤之数葢非有冯冯之声俾人得而闻又非有甍甍之形俾人得而睹子舆守约而独诣复圣瞻前而无方卫赣及肩之浅室仲由驻足而升堂彼州仇之何人固无惑乎不足以窥圣域之渺茫主人展然而笑曰繄尼父之诲人若太虚之时雨示后进之表仪固无行而不与夫岂强蔽于垣墉岌岌乎城堞之扞御使升高而无阶漫颙颙而延竚先生曰非也夫物理之不齐亶巨纎之异宜凡藏蓄之广博必高深而难窥倘其中之狭隘外卑薄以奚疑茍处下而视高曾何异乎斥鷃笑大鹏于藩篱主人于是与客蹑高■〈票〉眺清赏辟重扃披宿莽歴石级而歩趋睇宫庭之峻广扣玲珑之绮疏拜蒙倛之遗像览碑碣之籕文聆丝竹之絶响爰从容而咏归挹清风之萧爽

　　无逸图赋

　　粤有好古书生博雅搜竒阅几上之册絶编中之韦览商周之逸迹睹汉唐之懿仪渉猎前昔究竟无遗秘阁主人见而问之曰子亦尝闻唐人之无逸图乎演徃圣之箴规易明君之玩娯维良相之伟制实盛代之弘谟也盍抽子秘思摅子远谋探训诂之赜阐方策之腴摭芳逞妍而为余赋诸书生曰唯唯繄唐中叶实惟明皇式遏冦攘克正纪纲嗣睿而帝旣安且平虎皮包刃鹄板收英爰得宋璟股肱之良敷陈治道讨寻宪章璟乃思曰繄人君之履位如临深而捧盈宜处逸而思囏宜旰食而遑宁于是写周书无逸之图为唐皇克己之铭不餙不文不丹不青惟取周公旦之数语所以戒成王之守成者也念圣人之远虑惧嗣王之暇逸献无逸之嘉谟陈艰难于稼穑于以戒夫怠荒于以勉夫兢业考歴代之隆规敷祖考之弘绩呜呼七叹感慨乎今王之心不敢五言鉴观乎前哲之德于观逸于游田再四为之丁宁生则逸生则逸反复为之戒饬陈享国之乆近论得寿之隆啬是皆以昏为惩而以善为则也璟之图之必精致而详极明皇观之宜亦感省而修慝矧夫九龄进金鉴之録藴古遗大寳之箴贤臣交纳乎忠言贤君亦谨于操心故回奢为俭戒荒为勤毁殿庭之绮绣却锦坊之织纴是宜开元之盛治遂与贞观而比伦也夫何贤辅旣徃侈心遽萌肆情欲于玩好易山水于扆屏霓裳羽衣稔宴安之鸩毒盘盂觞豆忘古道之典刑弗致萧墙之警卒罹马嵬之行葢无逸之戒明皇不能愼终以如始是以乐极而患生方今圣皇宵衣图治笃缉熙之学继纂成之志有皋夔以论思有伊吕以献替固不假宋璟之为图而可以轶成周之盛治也秘阁主人展然而笑曰子之述是葢将致吾君于唐虞跻斯世于雍熙异于图混一而赋大都陈謟颂而翊佞谀者矣吾为尔歌之以发其余歌曰公旦辅周厪奬劝兮无逸数策埀寳训兮宋璟制图条不紊兮惜哉唐皇功亏九仭兮猗欤我皇勤劳克谨兮保惠黎民延亿兆而有永兮

　　○宣文阁赋【有序】

　　皇帝九年制作宣文阁于大明殿之西北皇上万几之暇御阁阅经史以左右儒臣为经筵官日侍讲读兹阁深列紫御杰出青霄朝野传诵瞻望踊跃布衣微臣欣幸睿圣崇文致治之隆旷古莫及敢竭蚁忱赞扬之私拜手稽首而献赋曰
　　鳌极立兮时雍四海一兮书同作神京于燕蓟贯北辰乎天中懿圣皇之御极焕离明以当空复至元之盛治绳神武而丕隆淳风沕穆膏泽涵洪文焰烜赫乎扶桑崦嵫之域仁声馺沓乎濮铅祝栗之封旣而干符昭陈神珍聿锡文德诞敷皇图广辟建杰阁乎中霄屹大明之西北揭宣文之嘉名示弘谟于万亿尔乃凖水臬恢鸿基公输献巧匠石殚思陶人运其埏埴玉工效其雕几豫章松栢剪千寻之夭矫金璧银镂致万里之瓌奇斧斤振铿鍧之韵砻斵极精致之宜四方子来经之营之羌不日而告成俨崔嵬而峗巍观夫岧嶤嵯峨崛岉瓌谲构复桷而重櫩邈希世而特出丽不踰奢俭不及质璇题翠甓崱缯绫而龙鳞蛟吻踶甍奂翚飞而跂翼绮疏镂列钱之玲珑藻井粲菱荷之绘餙玉铛璧英瀖护而零乱隋珠明月照耀而恍惚琼戸耀孚尹之璀错金铺响连环之复叠浮梲■〈山畾〉■〈山果〉而星悬鬬拱衡纵而斗折枝撑■〈木义〉枒而斜据芝栭攒罗而戢孴曲折要绍而环句层栌■〈石累〉佹而岌嶪交龙纒楣横飞云之梁栋双蜺蟠础耸擎天之柱石朱楣结曲而萦廽铅砌晶萦而炜燡雘垣流丹以周缭翠栢樛枝而行立于是圣皇驾玉辂张龙旗展乎国容辉乎皇仪望舒陪夫左驭屏翳导夫先驰鸣和鸾之铿锵服衮龙之陆离御斯阁以问道阐经幄之弘规帝幕高悬天颜孔怡列儒绅而进读对黼扆之严威舒缃帙之蟫蠧扩六籍之精微咏仁咀义聆天语于羲昊言温气和陈古道于皋夔殚词臣之忠荩恢圣学之缉熙内府颁竒珍于翠釡上方泻甘醴于玉巵和气春融起天庭之黄色文星环拱映帝座之清辉想夫圣躬听讲之余临眺倚徒接羲娥之耿光纳穹堮于无际东望则延春之阁崔嵬峥嵘横絶天半窅邈觚棱西瞰玉德秘殿■〈日高〉曜月朗渺太液之轻漪涵天光而滉瀁北瞻则万歳之山■〈山集〉嶫崷崪厓洞谽谺草木苍郁南视象魏岝■〈山上额下〉朱阙巉嵓嫓黄道之启途列阊阖之九闗猗欤休哉卓彼斯阁旷古曷偶奠亿代之基图辟八纮之戸牖萃鸿硕以讲劘稽典坟于蝌蚪于以广睿览之明极乎有截之垓埏廓圣德之聪逹乎无垠之宇宙思昔石渠天禄徒以贮简编之糟粕麒麟凌烟秪以图辅翼之英雄曷若圣皇之制作宣人文于万代致文治于无穷彼弘文馆聚四部之奥博白虎观辨五经之异同又乌足与圣世而比隆也哉乃舞蹈而歌之曰于穆圣皇亶聪明兮云汉昭回昭万方兮嘉惠兆人臻羲皇兮杰阁中天讨宪章兮鹓鹭章逢侍帝旁兮载稽经训研籀仓兮治功弥隆追虞唐兮人文宣朗贻亿万世而无疆兮

　　皇极赋

　　繄鸿蒙之未凿兮閟瞢闇之昏浑迨太极之肇判兮廓旁魄而昆仑崇与卑之旣陈兮繄人文其朗宣孰主张是兮总九野之元元伟睿智之首出兮奠中区而建极履大寳之巍巍兮秉元圣之休德侔大学于虞后兮至仁嫓乎唐国作庶类之标凖兮靡不于斯而作则实无偏而无陂兮恒特立乎中央羌至极而莫踰兮厥表正乎四方东西南北各止乎是兮俱辐辏而瞻望犹北辰之居其所兮烂众星之环拱屋中高而四下兮揭干栋之独耸任斯道而弗颇兮邦总总而归徃慨余怀古先兮屡翘首而企踵禹浚川而疏洛兮睹神龟之负书背文自一而至九兮亶后皇之锡畴稽皇极之位于何而存兮俨五数之中居统八类而为纲兮挈裘领于襟裾歴夏曁商兮斯理湮汨嗟周王之有道兮缵禹旧服释累囚而访箕兮演大法于简策陈名言于不朽兮皦中天之日月道不虚行兮在乎其人武得之而致理兮绵八百而永年五行顺兮五事察八政厚兮五纪协斯皇极之所以立三德以乂兮稽疑以明庶征允念兮福极于以劝惩斯皇极之所以行噫后世之昏迷兮彛伦攸斁天不畀乎此畴兮垂徃轨于初古举世瞀瞀而伥伥兮象纬黯其失度嗟大中之是训兮袭舛讹于训诂彼优游而姑息兮胡弗遵于王路宜孝元与代宗兮竟莫延于歴数惟蝌蚪之未昧兮尚斯文之可征仰圣皇之御极兮践五位而文明由圣道之正直兮粤作则于八纮述敷言以为世训兮贻亿代之媺声顾鲰生之何幸兮将观光于上京叩帝阍而献颂兮庆四海之咸宁歌曰我皇在上亶明圣兮建其有极德至盛兮参赞化育天地并兮四方取则无不敬兮寤寐禹箕为龟镜兮亿万斯年延福庆兮

　　紫微垣赋

　　璧月皎兮朗明银浦烂兮晶荧轻飙恬兮肃清纎云收兮窅冥若有客兮游木天陟仙瀛瞻太虚仰圆灵顾谓翰林主人曰吾观紫微之垣天皇之庭高高而在上遥遥而莫登愿先生搜典籍之赜隐阐象纬之纵横摅藻丽之锦心赋铿锵之金声某也将屏息而竢倾耳而听主人曰唯唯葢闻紫宫巍巍天皇是处大帝之座尊居其所抱以勾陈之六珠佐以天极之四辅灿灿其后者椒房之后妃烨烨其前者青宫之储副尚书大理炳焕而森罗柱史女御辉光而旁午运转于百余万里之外在中而弗移周回于七十二度之间常现而可覩繄荣卫之昭布俨七八之相联环乎皇极之居斯谓紫宫之垣左枢右枢夹乎离南之穹门上丞下丞摄乎坎北之重阍上宰少宰以对待少尉特立而聿存上辅少辅炜东西之次列上卫少卫分左右而屏藩是垣也覆以华葢之轮囷植以天柱之突兀黄金为城堑以云汉之津白玉作京辟以阊阖之阙叠银砾之层层树白榆之歴歴匪筑以干戈河图之倚杵匪甃以五色女娲之炼石传舎庖丁之宻迩内阶天府之相逼瑶宫玉台岧嶤乎其中广寒清虚暎照乎其侧故至尊之覆位代紫宫以居中师保耿台躔之拱侍臣邻炯郎星之列从奎壁炫文章之府执法肃御史之风由是天帝之垣环卫太一于高空万国羣黎林林总总以仰时雍之化岂非微星之万一千五百二十旋绕而无穷若太微象明堂之房天市嫓权衡之宗虽皆帝座之所寓羣星之所丛或邻翼轸之墟或贯房心之冲曾未若紫宫环卫于北辰之扛毂五纬连珠之所拱向列宿轑辐之所会同也客闻而歌之曰我皇圣明兮握符御极贤才并翰兮光华赫奕紫辰倚空兮金墉嶻嶪万国归心兮黔黎载德又歌曰丕图弘开兮景星耀辉帝垣昭晰兮中天巍巍至和坱圠兮玉烛献竒枢轴旋转兮万寿无期歌毕月挂觚棱露寒秘阁璇杓低昻玉绳回薄立清寒于掖垣仰紫微于寥廓
