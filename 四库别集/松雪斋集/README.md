　　松雪斋集　　　　　　　别集类四【元】
　　提要
　　【臣】等谨案松雪斋集十巻外集一巻元赵孟頫撰孟頫字子昻宋太祖之后以秀王伯圭赐第湖州故为湖州人年十四以父荫入仕宋亡家居防程钜夫访遗逸于江南以孟頫入见即授兵部郎中累官翰林学士承防卒追封魏国公諡文敏事迹具元史本传杨载作孟頫行状称所着有松雪斋诗集不详巻数明万厯间江元禧所编松雪斋集寥寥数篇实非足本惟焦竑经籍志载孟頫集十巻与此本目次相合而史所称琴原乐原得律吕不传之妙者检勘均在其中外集杂文十九首亦他本所未载葢全帙也孟頫以宋朝皇族改节仕元颇不谐于物论观其和姚子敬韵诗有同学故人今已稀重嗟出处寸心违句是晚年亦不免于自悔然论其才蓺则风流文采冠絶当时不但翰墨为元代第一即其文章亦揖让于虞杨范掲之间不甚出其后也集前有戴表元序见剡源文集中末题大徳戊戌嵗葢孟頫自汾州知州谒告归里时裒集所作请表元序之者表元不妄许与而此序推挹甚至其有所以取之矣后人编录全集仍録此序以为冠非无意也乾隆四十六年十月恭校上

　　松雪斋集序
　　吴兴赵子昂与余交十五年凡五见每见必以诗文相振激子昂才极高气极爽余跂之不能及然而未尝不为余尽也最后又见于杭始大出其平生之作曰松雪斋诗文集者若干卷属余评之余惟人之各以其材自致于世必能相及也而后相知必相知也而后能相为言余于子昂不相及而何以知何以言乎子昂曰虽然必言之余曰必言之则就吾二人之今所歴者请以杭喻淛东西之山水莫美于杭虽儿童妇女未尝至杭者知其美也使之言杭亦不敢不以为美也而不如吾二人之能言何者吾二人尝身歴而知之而彼未尝至也他日试以其説问居杭之人则言之不能以皆一彼所取于杭者异也今人之于诗之于文未尝身歴而知之而言之同者亦未之有也子昂未弱冠时出语巳惊其里中儒先稍长大而四方万里重购以求其文车马所至填门倾郭得片纸只字人人心惬意满而去此非可以声色致也而子昂岂谓其皆知我哉故古之相知者必若韩孟欧梅同声壹迹绸缪倾吐而后为遇而后世乃欲望此于道途解后之间则又过矣余评子昂古赋凌厉顿迅在楚汉之间古诗沈涵鲍谢自余诸作犹傲睨高适李翺云子昂自知之以为如何大徳戊戌仲春既望剡源戴表元叙

　　吴兴赋
　　猗与休哉吴兴之为郡也苍峰北峙羣山西迤龙腾兽舞云蒸霞起造太空自古始双谿夹流繇天目而来者三百里曲折委蛇演漾涟漪束为碕湾滙为湖陂渟皎澈百尺无泥贯乎城中缭于诸毗东注具区漭漭以天为隄不然诚末知所以受之观夫山川暎发照朗日月清气焉钟冲和攸集星列乎斗野势雄乎楚越神禹之所底定泰伯之所奄宅自汉而下徃徃开国洎晋城之揽秀据实沿流千雉面势作邑是故歴代慎牧必抡大才选有识前有王谢周虞后有何栁顔苏风流互映治行同符皆所以宣上徳意俾民懽娱况乎土地之所生风气之所宜人无外求用之有余其东则涂泥膏腴畆钟之田宿麦再收秔稻所便玉粒长腰照筥及箱转输旁郡常无凶年其南则伏虎之山金盖之麓浮图标其巅兰若栖其足鼓钟相闻飞甍华屋衡山絶水鲁史所録盘纡犬牙陂泽相属蒹葭菰芦鸿头荷华菱苕鳬茨萑蒲轩于四望弗极乌可胜数其中则有鲂鲤鲦鲿鍼头白小鲈鳜脍余鼋鼍鼈有蛟龙焉长鱼如人歕浪生风一举百钧渔师来同罔罟笭箵罩汕是工鸣榔鼔枻隠然商宫巨细不遗噞噞喁喁日亦无穷其西则重冈复岭川原是来其北则黄龙瑶阜之洞玲珑长夀之坞县水百仞既高且阻砑嵚崟崴磊碅磳怪石万数旅乎如林其高陵则有杨梅枣栗樝梨木瓜橘柚夏孕枇杷冬华槐檀松柏椅桐梓漆之属文竿绿竹篠簜杂遝味登俎豆材中宫室下逮薪槱无求不得其平陆则有桑麻如云郁郁纷纷嘉蔬含液不蓄长新陆伐雉兔水弋凫雁舟极之利率十过半衣食滋植容容衎衎既乐且庶匪教伊慢于是有搢绅先生明先圣之道以道之建学校立庠序服逢掖戴章甫济济多士日跻于古乃择元日用量币尊酒陈簠簋选能者秉周礼赞者在前献者在后雍容俯仰周旋节奏成礼而退神人和右当是之时家有诗书之声户习廉耻之道辟雍取法列郡观效诚不朽之盛事巳或者难曰自古论著之士曷尝不识人物纪风俗哉夫人才者济时之具而风俗者为治之质也今子徒捃摭细碎推比货食高谈不切炫燿自饰莫大于斯二者顾乃畧而弗録虽文夺组绣声谐金石窃为子不取也仆应之曰否子独不闻夫子之言乎十室之邑必有忠信今年且千载地且千里人物之富胡可殚纪史册毕书可无赘矣若乃风俗之隆污在为政者之所移易又弗可得而定着也夫吴虽介在江左尝被至徳之风矣且吾闻之风行而草偃日中而表正上行下效置邮传命辟若季子为守言游为令以仁义为化礼乐为政镇以不贪之寳喻以不言之信即刑可使不用俗可使益盛方将还敦朴于上古考休祥于庶徴今美则美矣又可遂以为定乎于是难者唯唯逡巡而失意
　　求友赋答袁养直
　　思古人之不可见兮心郁结而不舒登高丘而逺望兮独叹慨乎増歔波洋洋其泛滥兮欲济而无航膏吾车而孤徃兮山郁乎其苍苍四顾阒其寂寥兮思蹇产而不铄采众芳以自娱兮聊假日以愉乐搴长洲之芙蓉兮擥大泽之兰茝撷江蓠之秀颖兮结秋兰以为佩世俗方尚同兮余独异乎今之人谓兰茝为不芳兮蛾睂枉之以善婬众不可以户説兮歴年歳而觏闵变心以从俗兮而吾又不忍徯朝阳之鸣凤兮企空谷之白驹何驽骀之众多兮鷃雀纷其瞿瞿宁饥渴以需时兮耻属餍乎腥腐古道久其弗继兮吾惕焉为此惧何美人之好修兮独与余其同心懐余以厚徳兮遗余以好音世固有同居而弗察兮何千里而能同乱淛江之滔滔兮涉逺道以相从觌清扬之信美兮懐盛年而莫当子濯足于东海兮晞发乎扶桑吸沆瀣以为饮兮餐朝霞以为粮披列星之粲烂兮抱明月之夜光驾青云而高驰兮鸣和鸾之锵锵扰苍龙使挟辀兮服白虎以驱不祥愿发轫其勿亟兮聊弭节而抑志指黄昏以为期兮夫孰逺而弗至乱曰鹿鸣呦呦犹求友兮曾谓斯人不如物之知兮独行无俦吾将与子为类兮
　　纨扇赋
　　炎暑时至阳乌怒飞金石为流白汗沾衣吹纤条延爽南扉玉枕徒设桃笙安施旁皇踯躅不知所为于是裂轻防兮似雪制圆扇兮如月光揺懐袖凉生毛髪起遐想于青苹引清飔于天末萧然襟带凄其絺葛醒人肌骨袢歊如脱须臾或离中肠为热殆造物者欲解民之愠假人力以为之不然岂天时之可夺也复有题诗欣赏因书竒絶障轻尘以寄恨扬仁风而言别或画乘鸾之女或误成蝇之笔白羽防褷而自愧蒲葵比方而知劣及乎商气应厥民夷玉露降兮百草金风生兮桂枝罗衣重拂秋兰复菲孤萤冷照寒螀暗啼弃捐箧笥绸缪网丝班姬形中道之怨江淹赋零落之诗嗟夫用舍有时出处有宜惟人亦尔于物奚疑彼狐貉之御冬岂当暑而亦悲苟行藏之任道愿竢时乎安之伊圣贤其不可见兮之二人又何知
　　修竹赋
　　猗猗修竹不卉不蔓非草非木操挺特以高世姿萧洒以拔俗叶深翠羽干森碧玉孤生太山之阿千畆渭川之曲来清飇于逺岑娱佳人于空谷观夫临曲槛俯清溪色侵云汉影动涟漪苍云夏集緑雾朝霏萧萧雨沐袅袅风披露鹤长歗秋蝉独嘶金石间作笙竽谁吹若乃良夜明月穷冬积雪扫石上之隂听林间之折意参太古声沉寥泬耳目为之开涤神情以之怡恱盖其嫓秀碧梧托友青松蒲栁惭弱桃李羞容歌籊籊于卫女咏淇澳于国风故子猷吟啸于其下仲宣息宴乎其中七贤同调六逸齐踪良有以也又况鸣嶰谷之凤化葛陂之龙者哉至于虚其心实其节贯四时而不改柯易叶则吾以是观君子之徳
　　赤兔鹘赋
　　皇庆元年上赐大都留守臣伯特穆尔白兔鹘一翎翮皓洁白雪同皎至延祐元年凡三笼而毛羽变赤光采艶奕异于寻常搏击勇鸷加于畴昔虽老于五坊者亦不知其所以然岂若君子进于徳以润身者耶公既写其形质集贤侍读学士臣赵孟頫为之赋其辞曰
　　猗鸷禽兮出金门赐贵臣兮示殊恩色霜雪兮耀前轩解绦镟兮纵平原狐兔骇兮不及奔歴年歳兮毳如璊粲晨霞兮炫朝暾刚气胜兮肆飞鶱徳愈进兮义愈敦炯双眸兮睨乾坤

　　古风十首
　　诗亡春秋作仲尼盖苦心空言恐难托指事着以深大义炳如日万古仰照临凤鸟久不至楚狂乃知音愁来不得语起坐弹吾琴
　　周衰有战国纷纷极荆蓁黄金聘辩士驷马迎从人朝为刻骨讐莫作歃血亲终然智力屈奉身俱入秦相如赋大人出语颇竒怪飘然凌云意过耳诚一快夸言入无际自觉尘俗隘安知翻成劝何用名为戒自有天地来蓬莱几清浅人生空皓首举世谁得见瑶台在何许渺渺烟波逺方舟不可渡使我空展转絶代有佳人被服绮与纨蛾睂秀联姢吐词馥若兰清歌啓皓齿瑶琴发哀弹一弹再三叹听者涕汍澜借问谁家子为言本邯郸
　　秋风吹庭树故叶纷以队明月耿夜长草虫促经纬四序苦不淹少壮何足贵展转复展转寤辟不能寐昔为闺中秀今作市门鄙岂无膏与沐甘心得憔悴
　　山深多悲风日暮愁我心云降寒雨松柏自哀吟人生百年后奄然閟重隂念此每不乐天路何由寻仙人偓佺辈消揺在青岑奈何不尔思委命重黄金
　　海山有竒树粲粲瑚枝由来植物性无胫可推移如何石氏子树之白玉墀洛阳经几战金谷久荒夷空令千载下闻者为伤悲
　　烈风号枯条落叶满周道原野何萧索川流亦浩浩离居日以逺懐思令人老功名防有时生世苦不早顾瞻靡所骋忧心惄如捣
　　浮云何方来不知竟安之飘飘随风去汗漫以为期自昔功名士徃徃事驱驰白驹空谷中谁能加絷维皎皎难见容翻受世妍媸虚名诚无益不见斗与箕
　　有所思
　　思与君别来几见芙蓉花盈盈隔秋水若在天一涯欲涉不得去茫茫足烟雾汀洲多芳草何心采蘅杜青鸟翺云间锦书何时还君心虽匪石祗恐雕朱顔朱顔不可仗那能不惆怅何如双翡翠飞去兰苕上
　　和子俊感秋五首
　　秋至倐廿日天宇豁然清凉飚旦夕至林木发商声良辰不我与慨然心自惊古人久巳逝念之动中情苦心婴世患只博身后名今我将无同庶复得此生
　　明月照北林翩翩栖鸟翻虚室当静夜幸絶人事喧念子巳独寐无人相与言吾生性坦率与世无竞奔空懐丘壑志耿耿固长存何由持此意徃与严郑论
　　披衣步中庭仰视河汉白寓形天地内聊复度朝夕仲尼谅非愚皇皇不暖席履运有异同喟然慨今昔丘园岂云逺终当期屏迹日月驰骛去鸟影飞过隙归休曷不早胡为受形役
　　晴云暧高宇轻飚弄防凉天气政尔佳抚巳徒自伤今日非昨日荏苒叹流光几见春鸟鸣巳复啼寒螀一时良亦短万世则为长
