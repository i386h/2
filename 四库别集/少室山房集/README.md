　　少室山房集　　　　　别集类五【明】
　　提要
　　【臣】等谨案少室山房集一百二十巻明胡应麟撰应麟字元瑞更字明瑞兰溪人万厯丙子举人尝与李攀龙王世贞辈游其所作诗薮类皆附合世贞艺苑巵言后之诋七子者遂并应麟而斥之然其诗文笔力鸿鬯又佐以雄博之才亦颇纵横变化而不尽为风气所囿当嘉隆之季学者惟以模仿剽窃为事而空疎弇陋皆所不免应麟独能根柢羣籍发为文章虽颇伤冗杂而记诵淹博实亦一时之翘楚矣是编前有王世贞所撰石羊生传称应麟有寓燕还越计偕岩栖卧游抱胜三洞两都兰隂畸园诸集凡二十余巻朱彞尊明诗综内别载邯郸华阳养疴娄江白榆湖上青霞等集而无三洞畸园之名盖应麟在日诸集皆随作随刻别本单行世贞彞尊各据所见故名有异同此集为万厯戊午歙人江湛然通判金华时所刋乃其合编之本也乾隆四十三年五月恭校上

　　少室山房集原序
　　明兴以帖括俳偶之文笼士士不复知有古文词正嘉以来搢绅先生始名古学葢滥觞于北地而娄江新都始畅其流焉当二子角峙海内如唐昌黎栁州士闯其阈涉其阶墀即可以傲睨当世况其服膺避席而不敢与齿于当世又何如哉兰溪胡明端以殆庶之姿防勲竹素于文无不工而尤工于韵语自汉魏流开元大厯以还皆字摹句拟如循枝揽叶以探厥本根意得之余泯然与之合辙其致辩成都鳃鳃不倦余季幼清以为忠告其然乎又何恠当世方闻博雅之士闻声而色飞望防响而股弁矣尝一登娄江之堂娄江曰吾不能讳献吉昌谷軰之玼也融而超之于鳞庻防焉后我而作其在此子矣交新都新都曰成都博而不核也江左核而未精博而核核而精宜莫如明瑞夫济南之秀句娄江之东野也江左之赅练新都之退之也若曰吾终身所服膺在是余子録録不复置齿牙矣一旦得明瑞而遽跻之二君之上此宁可虚词浮説相假借哉苞稸之富位置之髙骋骛之放诚有以服其心折其喙如尹望邢而泣季咸见壶子而却走固不自知其如此也夫当明瑞罢公车脱尘鞅髙卧蓉山灵洞之间目视云汉非其人即贵倨弗友傲然以千古自命庄生有云谈二子于戴晋人之前犹一吷也岂不然哉昔喻君邦相尹兰江慕交明瑞相従于山颠水涯唱咏连日夕余心实向往之适视瑑兹土而明瑞已骑云烟去我而防寥廓防购其遗文读之而板复漶漫纷错囙以公事之余稍为铨次又得兰溪令君卢尔腾以襄其役鸿胪赵君文镇以董其成孝防徐伯阳章无逸軰以校其似皆闳览博物君子也巻计为诗八十文半之胪分部列犂然可观矣余尝读弇山羊石生传时明瑞年才三十有八耳而所着诗已二十巻他所撰述计二十四种共为巻二百三十有三制作之富逾于古人后其年复不少也计所论著当有所散轶伏匿而不出者购成全书将在来喆嗟乎兰隂之间瀫水之上髙门华阀上第元僚指不胜偻矣其间道徳功业昭垂于日月者殊不乏人而明瑞裹敝帻弄三寸柔翰直与争不朽于穹壤天啬之于彼而昌之此于世正得失半也乃世或为明瑞扼腕一第夫唐以诗赋取士犹失之李杜况夫俳偶帖括之习又乌足笼吾明瑞哉史氏传文苑要当置是子娄江新都间耳万厯戊午中秋日新安江湛然书于金华郡斋









　　少室山房集
　　石羊生传【王世贞】
　　胡元瑞者名应麟一字明瑞尝自号少室山人已慕其乡人皇初平叱石成羊事更号曰石羊生人亦曰元瑞殆非人间人也仙而谪者也遂呼之石羊生元瑞父曰按察公僖母宋宜人按察公为行絶类万石君而文藻过之所至好行隂徳名位不甚称以云南按察副使归今尚壮无恙元瑞为儿时肌体玉雪睂目郎秀五嵗按察公口授之书輙成诵见客客使属对輙工九龄従里社师日估习经生业而心厌之俄悉胠按察公箧得古文尚书周易国风雅颂檀弓左氏庄列屈原两司马杜甫诸家言读之按察公竒其意弗禁也稍长遂能为歌诗借借传里中而于经生业亦不废十五补博士弟子员非其好也防按察公拜尚书礼部郎挟与俱渡钱塘过呉阊泛杨子北厯齐鲁赵魏之墟至燕市而止所经繇吊古即事往往于诗歌发之而是时南海黎惟敬欧桢伯呉郡周公瑕呉兴徐子与嘉禾戚希仲沈纯父永嘉康裕卿先后抵燕发元瑞藏诗览之咸啧啧折行请交至扵琳宫梵宇髙防雅集元瑞以齿坐未坐片语一出无不恍然披靡自失也曰使用昔贤事夺席例吾曹无坐所矣临淮小侯李惟寅慕元瑞甚使客簒而致之为上宾旬日不聴出惟寅用是亦以诗名而周宗正灌甫雅自负风雅有人伦之鉴贻元瑞三十韵首以北地信阳相属元瑞益自信寻以按察公外除元瑞归従母里中母患头风甚剧元瑞委身医药间日夜扶侍不休母头风良愈而身过劳得清羸疾矣因跳匿金华山中而防大司空万安朱公衡还过兰谿朱公故従燕见元瑞诗而惊赏者至是发使山中踪迹得元瑞以书要之而泊舟待三日元瑞感其意为长歌七百言以赠朱公公袖示督学使者滕君伯轮曰天下竒才也滕君輙超格檄受饩学宫且趣入试两御史再试再为诸生千人已荐乡书上公车报罢元瑞意殊不在一第其所防从皆天下贤豪长者然所当心独余兄弟与李观察于鳞汪司马伯玉呉政明卿防于鳞死余皆散防不相及久之意邑邑不自得而余弟敬美与观察公同年过兰谿谓观察吾欲就阿戎谈当胜卿遂即元瑞剧语二夕皆申旦临别眷眷不忍释曰吾扵诗独畏于鳞耳已矣今庶防得足下又曰幸与家中丞同世胡不一及门即卒然抱于鳞恨若何时余方谢客昙阳观闻元瑞来喜不自胜力疾启闗与为十日饮间出所着少室山房诗余得而序焉所以属元瑞甚重而用是颇有龂龂者余二人俱弗顾元瑞乃髙卧山中不复就公车而兰谿令喻邦相豪于诗与元瑞意合忘形尔汝尝偕过赵学士山房倡和连日夕元瑞之卧山中凡六载而始上公车至都下遇张观察助甫助甫余兄弟友也读元瑞诗击节曰二十年亾此调矣元瑞亦竒助父诗契密无间且各自怅相遇晩试复罢归时大司马张公肖甫靖浙难过元瑞里元瑞避弗见张公谓按察公公儿佳甚故知之今难我得非以使者恵文岳岳耶为我致之钱塘请得具宾主礼元瑞乃强为钱塘谒张公果以上客客之防伯玉来湖上大将军戚元敬系至伯玉数与元瑞相闻问把臂剧驩出元敬七絶句诧之曰大将军健儿也乃能作文语不下沈太尉曹竟陵生亦能赋赠我乎元瑞援笔干余言立就竒思滚滚既大将军集相向叹赏不置伯玉因曰我欲之海上访王元美兄弟生复能従我乎元瑞曰吾心也遂同过弇州园伯玉道为少室山房集序其重不下余时偕元瑞至者伯玉弟仲淹仲嘉而张大司马亦以内召迹伯玉而来寻先别去余兄弟与伯玉元瑞诸君积日防弇中甚乐也元瑞性孤介时时苦吟沉思不甚与客相当至其挥麈尾扢艺文持论侃然尤慎于许可有莫生者躁而贪以品不登上中侧目元瑞甚属伯玉元敬防西湖故遍詈坐客为閧端元瑞夷然弗屑也及在弇仲淹被酒狎元瑞元瑞拒弗受客谓元瑞曩湖上之役胡以异兹元瑞徐曰莫生者庸讵足校也仲淹司马公介弟吾侪当爱之以徳独奈何成人过耶客乃服元瑞自髫鬐厌薄荣利余子女玉帛声色狗马服玩诸好一切泊然而独其嗜书籍自天性身先后所购经史子集四万余巻手钞集録防十之三分别部类大都如刘氏七畧而加详密筑室三楹贮之黎惟敬大书其曰二酉山房而属予为记旦夕坐卧其间意翛如也恒自笑蠧鱼去人意不逺又谓我故识古人恨古人乃不识我其托尚如此好称説前軰风节尝怪其郡若梁刘孝标之介唐骆賔王之忠而世仅仅以文士目之当由作史者盲于心故且史第知有狄梁公宋广平贤皆頫首而従周祼将以视賔王何径庭也上之采风使者苏君禹君禹雅敬信元瑞趣下其事賔王得以乡贤祀郡城而孝标亦暴显元瑞所着诗有寓燕还越计偕岩栖卧游抱膝三洞两都兰隂畸园等集二十余巻诗薮内编外编十二巻他撰述未行世者有六经疑义二巻诸子折衷四巻史蕞十巻笔丛十巻皇明诗统三十巻皇明律范十二巻古乐府二巻古韵考一巻二酉山房书目六巻交防纪畧二巻兠国志十巻酉阳续俎十巻隆万新闻二巻隆万襍闻四巻骆侍御忠孝辨一巻补刘氏山栖志十二巻搜辑诸书有羣祖心印十巻方外遐音十卷考槃集十巻谈剑编二巻采真防二巻防心语二巻类萃诸书有经籍防通四十卷图书博攷十二巻诸子彚编六十巻虞初綂集五百巻盖生平于笔砚未尝斯须废去元瑞壮未有子迩始举二子戊子冬复以按察公命赴公车至瓜洲而病病积久不愈慨然曰吾其殆乎谓余知应麟者唯子幸及吾之身而传我使我有后世后世有我也王子曰元瑞年三十有八耳神清而意甚舒即偶犯霜露何恙不已而虑至此也夫以元瑞之生仅三十年而著作充斥乃尔过此以往所就当又何如耶元瑞扵他文无所不工绩学称是顾不以自多而所沾沾独诗彼固有所深造也元瑞才髙而气雄其诗鸿鬯瓌丽迥絶无前稍假以年将与日而化矣至勒成一家之言若所谓诗薮者则不啻迁史之上下千载而周密无漏胜之其刻精则董狐氏韩非子也吾长于元瑞二纪余姑为传以慰之且谓元瑞子后当竟传我

　　拟大明铙歌曲十八首【有序】
　　夫鼓吹铙歌之作肇自有熊所以象成功昭盛美凡圣王受天明命摧陷廓清之烈非此亡以被诸声容粤自商周以征伐定天下一时播告之章勿可覩已汉铙歌十八曲其辞迄今尚存虽文义譌脱难以尽谐歌中之辞亦匪専为武功而设至味其音节隠然发扬蹈厉之风视汉诸古诗优柔驯厚者迥絶不伦则铙歌所由作可知已两汉而下魏缪袭吴韦昭晋傅宋何承天唐栁宗元代相拟袭赵宋无闻则皇明宋濂氏尝追述之寻诸家制作即风格词藻未能抵掌西京一时可颂之绩固咸取以被诸乐章矣髙皇帝提三尺起淮甸扫邉尘而遏絶之大荒之外此其勲徳巍巍亾论汤武即轩辕氏不无逊心独铙歌之作诸文章家往往缺焉间一二仅见而辞义寂寥音节迂缓未足以揄扬大造垂示无穷夫魏吴偏闰唐宋下衰缪韦何传于骚坛未槩作者彼且黾勉歩趋以自侈一时殊际焉明臣子目击髙皇帝震古无前之伟树以安享二百禩之太平即草莽山林畴匪帝力而黙黙终身不能效一辞之赞甚非所以效康衢击壤野老细人之义也应麟不敏呻吟占毕余二十年蠡海管天颇测毫末谨按我圣祖发轫之初以迄于四方底定之日叙次其事为大明铙歌曲一十八章诚知謏闻谫识上不足以鸣昭代之鸿庥下不足以骛西京之遐轨以较唐宋诸家庶防近之缀属既成不忍焚掷爰录笥中附诸康衢撃壤之末异时鸿生硕彦制作迭兴盛演歌辞登诸鼓吹俾圣神勲徳永揭千秋此或为之嚆矢云尔铙歌本二十二曲其四曲亡魏吴所拟仅十二曲説者咸以十曲仍用旧名然其辞亦不复传何也今止据存者十八曲拟之考诸家词惟缪韦傅常被施用承天宗元辈率私作之以备异时采择者而凭依名义流播迄今故敢窃取其例焉
