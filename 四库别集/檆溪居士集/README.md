　　钦定四库全书　　　　集部四
　　提要
　　檆溪居士集十二巻　　别集类三【宋】
　　臣等谨案檆溪居士集十二巻宋刘才邵撰才邵字美中庐陵人檆溪居士其自号也大观二年上舍释褐宣和二年又中宏词科累迁校书郎以养亲归家居十年绍兴初起为秘书丞再掌制诰官至工部侍郎权吏部尚书加显谟阁直学士事迹具宋史本传是集之名亦见本传之中然艺文志乃阙而不载据周必大序原本盖二十二巻也自明以来传本甚稀厉鹗宋诗纪事从诗话补遗中録其夜度娘歌一首今检勘原集乃相思曲中之四句误作全诗知旧本久亡故无从而考正矣谨就永乐大典所载裒辑编次厘为诗三巻内外制四巻襍文五巻其间如必大原序所举清江引大堤曲诸诗皆不复存然约畧巻帙似尚得十之六七其诗源出苏氏故才气颇为纵横其余亦多驯雅而制诰诸作尤有体裁其他所纪朝廷典故与宋史徃徃异同如地理志谓南渡有淮平无盱眙而集中有向子固知盱眙军转官制职官志载政和七年易观察留后为承宣使而集中赐董先辞免新除承宣使恩命不允诏有顷因留务之职易以使名之语知承宣使之名乃始于绍兴不始于政和又如选举志不载绍兴二十六年戒谕科举事张纲传不载纲参知政事陈康伯传不载其居馆职之类皆可据以订讹补缺惟所作秦桧制词语多溢量至称其道义接邱轲之传勲名真伊吕之佐尤为谬妄史称其于权臣用事之时能雍容逊避以保名节颇着微词其指此类欤是则白璧之瑕矣乾隆四十六年九月恭校上

　　原序
　　登文章之箓固难矣诗于其中抑又难哉刘梦得曰心之精微发而为文文之神妙咏而为诗司空表圣亦云文之难而诗尤难又尝喻以饮食不可无盐梅而其美常在酸咸之外之二説前辈有取焉古者造士四术教子于过庭皆以诗为首本朝苏氏自编东坡前后集亦先列诗篇其所从来逺矣今檆溪居士刘公殆有得于斯欤予少时闻公赋咏一出輙手抄而口诵之味清江引则欲进乎技而凝于神也歌出塞行则如视旗影而聆鼓声也读大堤曲则又如望归舟对斜月而听情人思妇之语切切也【案此二篇大典不载】其他摹写物象美今怀古登临比兴酬赠祖饯皆凌厉乎先贤度越乎流辈盖得于天者气和而心平勉于已者学富而功深故于所谓至难者既优为之则其制诰有体议论有源铭志能叙事偈颂多达理固余事也藻饰王度冠冕诸儒领袖乡党有以也夫公讳才邵字美中世居庐陵年二十四大观三年释褐甲科宣和初复中词学第二两登馆阁一掌外制再直翰苑仕至工部侍郎兼权吏部尚书丐归优进显谟阁直学士奉祠防于家年七十二其孙承议郎恪类公集为二十二巻授其子国学生千龄至是以序为请故为追述见闻而题其端檆溪公自号也嘉泰元年二月旦日周必大撰
　　古今文章至我宋集大成矣盖自奎宿宣精列圣制作于是焕乎之文日月光华云汉昭回天经地纬衣被万物河岳炳灵鸿硕挺出在仁宗时则有若六一先生主斯文之夏盟在神宗时则有若东坡先生传六一之大宗在哲宗时则有若山谷先生续国风雅颂之絶视汉之迁固卿云唐之李杜韩栁盖奄有而包举之矣中更羣小崇奸绌正目为僻学禁而锢之盖斯文至此而一厄也惟我庐陵有泸溪之王檆溪之刘两先生身作金城以郛此道自王公游太学刘公继至独犯大禁挟六一坡谷之书以入昼则庋藏夜则繙阅每伺同舎生息烛酣寝必起坐吹灯纵观三书迨暇或哦诗歌或续古文每一篇出流布辇毂脍炙荐绅纸价为髙嗟乎若两先生当妖禽羣啾而发紫鸾之鸣折杨骤歌而奏清庙之瑟鹬冠妖服之竞丽而觌黄收纯衣之制其有大勳劳于斯文其伟乎哉予生十有七年始得进拜泸溪而师焉而问焉其所以告予者太学犯禁之説也后十年又得进拜檆溪而师焉而问焉其所以告予者亦太学犯禁之説也今两先生逺矣予亦老而归休矣檆溪之曾孙千龄一日访予于南溪之上出其祖之书曰檆溪集者示予请书其后予披而读之见丞相益国周公序其篇首凢檆溪防新领异之诗登峯造极之文既攡张发挥不遗余矣予尚何言哉独书两先生所以告予者于篇末俾后学者有闻焉泸溪又云是时书肆畏罪坡谷二书皆毁其板独一贵戚家刻板印焉率黄金斤易坡文十盖其禁愈急其文愈贵也今家有此书人习此学有知当时斯文之难得如此者乎是小人之厄斯文乃所以昌斯文也然厄斯文者本皆泯然与草木共尽而斯文之传与日月争光然则斯文病不厄耳厄奚病哉古者圣贤君子之所守于是可得而知矣顾吾道之是非何如而已时之好恶足为之动耶六一坡谷其知之矣至于吾州之两先生独首犯时之大禁力学众人之所不敢学所谓豪杰特立之士者不在斯人欤不在斯人欤檆溪讳才邵字美中泸溪讳庭珪字民瞻皆擢进士第檆溪再中宏词科终官工部侍郎兼权吏部尚书赠显谟阁直学士泸溪晚为国子监簿终官直敷文阁奉祠云嘉泰三年十月丙午寳谟阁直学士通议大夫致仕杨万里谨序

　　檆溪居士集巻一　　　　宋　刘才邵　撰古体诗
　　飞龙引
　　忆昔游洛川投宿龙门中逺意托清夣缑山访遗踪道逢王子晋琼轮驾飞龙幢盖纷飒沓导从千玉童行行蹑清尘遂到紫翠峯整冠叩仙阍获礼氷雪容琼觞酌霞醴劳苦问所从授以台书玉笈启灵封劝我勤修炼他时防方蓬再拜领真诀永永藏丹衷俯仰十载余光景如飘风常恐年嵗晏难收梨枣功愿言坚此心真缘傥潜通相与乘飞霞谒帝青琳宫
　　登景德寺塔
　　耽耽梵释宫凭髙剏灵址中有窣堵波排突青冥里梯磴回转蟠屈龙蛇尾飞步出紫氛眼界浄如洗望极天南端沓若孤烟起何当结蜃楼亭亭聊对峙云生空山顶飞过朱阑底慿栏瞰重城楼堞杂云水市人何憧憧閙若穿垤蚁壮观自何年五季运初圯干戈暗淮海裒敛到骨髓谁知黑云都金帛浩填委聊复推其余办此咄嗟尔【原注寺中殿塔多伪呉时黑云都将校所建梁间姓名尚存】因知破悭法骁悍亦风靡百年能几时两曜无停晷吊古悲陵谷山荒树如荠双江虽依旧去浪何曾止从来旷达士遐想尘外轨安得凌风翰一举九万里
　　次韵陈乆道秋雨书事二首
　　寻常秋色髙风力未遽恶应缘杀气盛惨淡连沙漠【原注传闻诸将并力北征】林枯日向疎溪断水初落闲云亦何心酿雨添寂寞赖有金兰契新诗慰萧索盛意难虚辱终愧笔锋弱元知鸾鹤音异彼藩篱雀欲泛东篱英聊庆西畴获区区恋阙心对酒飜不乐天祸有时悔妖氛防清廓【案史称髙宗即位才邵以亲老归侍闲居十年诗中有云区区恋阙心当为归飬时所作又第四句原注云传闻诸将并力北征疑即指建炎四年岳飞韩世忠等屡败金师而言附识于此】
　　萧萧凉气增发兴殊不恶长烟拂秋晕小雨滑溪涸【案此诗乃叠前韵前一首第四句漠字韵此首用涸字韵而句意皆稳适不可移易似非传写之误或作诗时偶失检耳谨附订于此】夜长虫语乏池冷莲衣落崇兰媚深静晚菊甘寂寞寒姿晓更鲜芳气初不索譬彼幽闲女静饰保纎弱又如耿介士逸志卑燕雀无心竞纷华何处生陨获知君深悟此髙歌写真乐悠然脱尘鞅寸田本恢廓
　　子夜四时歌
　　栁烟晨气酽花露春容湿翠幕卷宫罗翩翩双燕入无心补粟眉闲对妆台立谁调紫鸾笙晴响散空碧湘簟涌双纹氷山象羣玉神飚送荷香冉冉度圆绿雪腕荡兰舟齐唱横塘曲暗忆采莲侣横翠长眉蹙月痕抹宫帘凉风满僊腋金闺歌吹深不觉东方白谁知长门宫偏怪蟾漏澁脉脉对晓兰风中抱香泣氷花生曲池云叶暗髙阁初筵动云韶寒气不待却翠茵承妙舞拂衣燕低掠飜愁囘雪妬不遣褰珠箔
　　次韵王文成谢人恵莼菜
　　颇闻富鲑菜径欲作不速忍以辛甘故而烦主人肃蒙分紫虬髯芳滑盐豉足中原困苦旱干死食藿蠋何从覔嘉蔬得以供口腹幸益殷孚羮一解炎蒸毒物随人意重为恵同鼎餗荒哉膏粱子丰馔夸填谷
　　谢王君寳恵菊花
　　常怜避俗翁寄傲谢簪绂三径竹虽存苦乏杯中物采采黄金英篱邉空竚立赖有好事人陶然得酣适吾家颇种秫压滓夜槽泣其如移菊晚青蘂不堪摘念谁分秋香泛此花瓷白髙情歘相投遣送劳一力风期宛然同未易分今昔两株千万朶冷艳光相射银盘何烂烂承露带青碧月下更奇絶碎影寒气逼桃红吐鲜妍醉晕不容拭嘉名配黄花姚魏真匹敌开窓对二妙清意争先入却思陪语笑坐叹溪山隔幸有真契存满目共秋色
　　青原台次前人韵
　　伯仲聨华萼登临拥旌幢从行真凤雏不数襄阳厐圣人念逺民暂辍临此邦共惊后防手胡为调众腔公独体上意勤民起膏肓余闲养诗勇万钧一指扛谢家内集盛文彩今可双江白曳缟练山青逼轩窓缅怀风幡语闻者拱手降独此双泉老机锋暗相撞曹溪一滴水倾出白玉缸至今正脉通洪流咽三江逺水注江腹浮云荡山胷悠悠动遐想矗矗标竒峰祖意透今古风期协霜钟诗中无碍辨未信景能供黄堂正多暇森防闲卫兵月波复荡漾遥接银河倾天髙万物肃星润三阶平使君乐嵗丰髙台延客登从来中秋夕难得满意晴江山自髙下歌管揺空明参差桂影动清湿露华凝新诗出妙思心境还双清皎如白玉壶满中贮氷凌坐乆羣动息风生两腋轻觞咏有余乐何物更闗身不减庾楼兴风流自古情
　　次韵傅安道同陈元功诸公游石狮岩
　　中书英烈气凛然照人寒济美世有人盛德启其原幸逢贤别乘论文得交欢逸辩倾逺海潮波歘飞奔烛理无滞碍白日破幽昏谁云前辈逺岂独文采存新诗引人目当食不暇餐为説游岩乐飞步恣腾鶱却立视万象森如列庭间飘然乘风兴星榆不难攀聊复陈大概彼美难尽言少年陈将军奋迹起江村共知宰社手美好岂常贫深怀恩未报竭诚奉至尊驱兵猎余冦烈火焚蜂屯亦复事招纳投弓悯穷猿坐令归业民井灶罗丘樊是日出近郊旌旗郁幡幡同游不知倦落照横松门士卒共甘苦无人讥华元老守懒出游何从涤尘烦况阻陪归骑幽寻欵禅闗铃斋但兀坐有酒空盈樽常能歌妙句为激壮士肝解劳须痛饮杯行勿留残
