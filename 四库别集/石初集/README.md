　　美人昔燕赵

　　美人昔燕赵岁乆江汉间轩窓粲明霞被服罗绮纨頩怒恒自持含意讵敢攀一朝强暴陵相从即欢颜倾身作歌舞豫恐恩意阑新声与娇态取媚巧百端主家隔风尘庭宇深且完岂无昔共处永夜青灯寒魂梦各所依寜复相徃还沉沉九秋霜履屦思蒯菅娟娟中天月照影留空山盛时槩无虞末路良独难弃置勿重陈使人摧肺肝

　　濯濯江汉女

　　濯濯江汉女幽贞出良家十年守空帏夫婿天之涯良夜秉明烛悉心务丝麻忆昨秋风起河洛昏尘沙伤彼朱丝弦逸响趋淫哇朝为御沟栁莫作陌上花世态固应尔委置勿复嗟所悲在新春雨露凝朝华感此深闭门渊氷虑弥加蔵玉万仞冈守护盘龙蛇投珠千顷波光焰惊鱼鰕夫壻会有期岂问岁月賖寸心如皦日终始宜无瑕

　　燕山万里雪

　　燕山万里雪不作黄河氷玄徳専所司曷为失其慿赤云窃乘间遂逐炎风兴连延土山焦反更繁青蝇毛炰列饔饩臭腐日以升禋祀古有严孰与存芳馨昨梦观圜丘玉辂苍龙乘正气六合周瑞花反严凝坐见昆冈火瞬息投淄渑愿言哀江南尽洗毒雾蒸

　　潮生浙江白

　　潮生浙江白积水何其多悠悠子胥魂千古终不磨东南溢潢潦污渎纷旁罗独此障百川昼夜激海波力排坤轴转正气通天河从星趣好雨浴日阳之阿苍苍紫微垣弁冕锵鸣珂宜蒙湛露恩领袖湔甲戈云阵声万雷银山势嵳峩吹嘘蛟龙翔奋起驱鲸鼍功推五行首朝宗谅无他

　　种瓜南山下

　　种瓜南山下实晚霜落之冒寒拾瓜仁生意或在兹収蔵箧笥中恒恐鼠蠧窥羣儿顾窃笑痴絶将奚为昨闻北里富晨昏厌甘肥祸机隠骄溢瞬息靡子遗嗟我食无肉藜藿每晏炊俟时启包裹瓜地审所宜殷勤谢春雨庶以忘朝饥

　　十五纪懐

　　止酒已多日当秋病连绵一窓此寄傲晏起仍早眠岂不念佳节负此明月圎叩门客语我蚤作星尚悬白气出东北横贯西南天夜凉步中庭所见还依然力疾起语客丧乱今十年谓天盖无情埀象在目前谓天其有知吉语皆虚传未遑悉吴越况敢谈幽燕须臾报军出明日攻富田焚掠先近郊斩艾无愚贤咫尺成俘囚妻子畴能全戒客姑罢休世事从推迁客去深掩耳闭关涕澘焉

　　大风发屋雨雹交集

　　大风西北来屋瓦去如走丽谯双阙壮关楗决枢纽苍苍泮宫栢枝叶互纷纠移时折偃蹇巧力无措手西禅鐡浮圗摧仆如拉朽临江战楼飞夹巷坊额剖恍疑会羣龙奋发交怒吼泰山恐崩裂寜复计培塿老夫闭门卧雨雹散窓牖仓皇正衣冠力疾坐良乆天威俄咫尺性命亦何有黙悟贵存心渊氷慎吾守

　　岁莫柬张梅间

　　忆昔慕逺游朝华沐春露焉知丧乱后林杪秋叶蠧间闗西州门寒日收踽步尚余平生友示我广平赋耿耿河汉辉幽幽氷雪度终惭媚求悦寜悔贞取妬朗吟发孤音明月落庭户世事殊未涯岁年忽云莫有美君子心永言金石固

　　除夜书懐

　　一榻陋容膝老身兀枯株淹留恒鲜悰况兹岁云徂儿女虽茍完丧乱多异居同也乆乡曲奔走一饭驱鳯也妻女累近亦反故庐拙谋康最少甥馆横溪隅阿巽孀七载忍弃吴氏孤靓家食指繁绩纴勤养姑市廛病株守环偶锥刀趋顾我此相守出入孙符俱孩乌颇机警戱弄聊与娱止酒动经年宿病无由祛晚餐率减半夜枕气始舒宴逰多却避礼节那能拘亲友既少存存者亦暗踈骎骎齿髪敝勉勉筋力余抚事百感集来岁知何如存亡且未测艰苦焉足虞骨肉姑眼前开懐尽斯须地炉熟春茶明烛散欝纡忽忆孤山梦风雪随吟驴梅花千载心皎若沧海珠服之配明月阅时终无渝

　　和萧仲仪

　　十年树桑柘结屋山涧旁辛苦蚕丝成将备公子裳缁尘一朝起冠盖纷劻勷飘零落绮纨翕忽归渺茫鲜鲜皮卉服摇曵凌空翔感此桑下心临风时自伤

　　招隠

　　妖氛被东南丑类日响应承平七十载仓卒骇观听堂堂忠节邦奕奕守与命流血动成川分奔马蹄竞羣凶首被赤心额双悬镜赤以欺愚庸镜以眩昏瞑张皇礼像设诳托西方圣烧蜡动千斤相承势弥盛喧呼短兵接迅若斤斧运杀人先剖心刳腹抉肝肾攟摭穷窖蔵备网罗机穽官曹列廐牧民屋委灰烬郡治歌舞场朋从悉枭獍颇闻连骑出近郭更蹂躏计程倐徃还日以严示信距城此虽逺畴敢保安静闻鸡心即危过午魂始定颠连混寃亲跋涉无老病晨兴不遑夕永日饥餐并岂复虞道途所悲室悬磬妻孥且目下侥幸须臾命近传某泽中逆料絶窥侦儿啼误失声掩捕一朝尽回头顾童穉反复深砭订事会固叵量寜能忘戒训因思少壮日膏泽溥涵泳冠盖先送迎农桑杂歌咏闾阎饫粱肉厮养翘车乗飞潜偕动植一一遂天性焉知厄遭遇埀老百忧并圣朝果何负奸宄妄依凭得非湛恩隆窃禄隠谗佞贪冒逓相蒙浸淫成此衅中天扬日月光彩烂相映云雾偶晦冥何能滋疾疢众心一此理磨濯光愈莹化机黙回旋民伍奋豪俊川东事大竒气与北风劲划开聚石圗卷入背水阵翩然破竹势节解自迎刃郡侯起収集转盻复归正誓将扫余孽侧耳四方靖麦秋告登场秔稻亦已竟死者可胜哀存者聊自庆萧条旧邑里庶免忧釡甑书生守衡茆才具非济胜中宵投袂起踊跃歌解愠四顾扫曀霾如疾脱危证亲知稍慰藉松菊尚三径雨露极昭苏草茅微报称忠谋虑或过深计众多屏薫莸绚疑似玉石互缁磷吏弊袭贪残罕由慈惠进吾侯新涉难庶务职其柄抚定在哀矜朂哉慎行政

　　清明
