鸳鸯湖棹歌　　（清）张燕昌 撰 

　　张燕昌，海盐人。字文渔，号芑堂。清嘉庆时以优贡举孝廉方正，受知于阮元。平生力学好古，曾师事丁敬、张庚，尤嗜金石，搜罗宏富，曾自摹古文字为《金石契》，又曾登宁波天一阁，摹石鼓文，筑石鼓亭勒石于家。著有《石鼓文释存》一卷、《飞帛录》二卷、《金粟笺说》一卷。《金石契》一书流行极广，其书画篆刻，为世所珍。以上见《嘉兴文献》及和《鸳鸯湖棹歌》自注。 

　　乾隆甲午三十九年之冬，临川李公视学两浙，以《鸳鸯湖棹歌》课嘉禾士子。盖欲采里巷之谣谚，觇民情于歌咏，而因以谱太平之风景也。燕昌生长鸳鸯湖畔，扁舟来往，寻鸥问鹭，樵歌渔唱，时领略于烟波缥渺间。昔杨铁崖云：道扬讽谕，古人教广，有兴国显家之道焉。不揆梼昧，仿朱谭二公倡和之作，续成百首，非敢踵美前贤，亦祗操音下里云尔！ 

　　东南一带稻鱼乡， 
　　男务田农女务桑。 
　　侬谱太平新乐府， 
　　棹歌声里渡鸳鸯。 

　　波纹如縠叠轻罗， 
　　风波不作烟波多。 
　　惯拨瓜皮小艇子， 
　　浪花圆处疾子梭。 

　　春城士女晓粧妍， 
　　插鬓红薇浥露鲜。 
　　唱入饧箫深巷里， 
　　卖花天气看秋千。 

　　换衣亭外念郎衣， 
　　落帆亭畔望郎归， 
　　花落阶前郎不至， 
　　竹林新长燕来肥。 

　　花开如锦柳如丝， 
　　学绣村中学绣时， 
　　绣得鸳鸯常比翼， 
　　双双棲在合欢枝。 

　　注：常绣村在郡城西。 

　　绿萝水阁白蘋湖， 
　　小妇当窗簾不钩， 
　　三丈日高春梦醒， 
　　街前唤卖马兰头。 

　　注：绿萝庄在鸳鸯湖南。 

　　新晴滟滟晓云开， 
　　铃语钟声尽日催。 
　　莫道侬舟归独晚， 
　　烧香三塔寺前来。 

　　贤娼巷里春风温， 
　　苏小坟前夜月昏。 
　　惟有离离青草色， 
　　年年寒食最销魂。 

　　三月吴蚕已二眠， 
　　蚕房辛苦春秋天。 
　　楝花开后人如醉， 
　　才放蚕忙又种田。 

　　岸花片片落柔枝， 
　　岸柳依依拂钓丝。 
　　绿涨深时宜阿子， 
　　红桥围处是孩儿。 

　　樱桃熟时乳燕飞， 
　　豆荚堆盘笋蕨肥。 
　　榴花红映交叶绿， 
　　与郎吃粽送寒衣。 

　　溪上犹传旧姓钱， 
　　溪边是处起炊烟。 
　　连天小雨桑阴密， 
　　夜夜青蛙叫水田。 

　　高贴红笺蚕月韯， 
　　柳丝处处插茅檐。 
　　与郎禁忌休来往， 
　　怪杀人家闭户严。 

　　熟梅时节雨连绵， 
　　石首鱼腥正上筵。 
　　一路冰鲜齐入市， 
　　海滨新到贩鱼船。 

　　注：石首鱼俗称黄鱼，嘉兴贩黄鱼者从宁波航海到乍浦，鱼上盖冰以保鲜。 

　　头蚕五月住丝车， 
　　织得新罗夺锦霞。 
　　妾欲为郎悬艾虎， 
　　郎须为妾插榴花。 

　　梅风竹雨入筠簾， 
　　甪里街头屐齿黏。 
　　记送郎行行不得， 
　　履鞋桥上履鞋尖。 

　　青丝笼带看蚕娘， 
　　两岸人家齐采桑。 
　　缫得新丝何处卖， 
　　东新行贩北新行。 

　　注：新行镇在嘉兴，新行里在秀水王江泾。 

　　嘉绫满幅细花纹， 
　　一丝一缕见辛勤。 
　　制就鸳鸯相映绿， 
　　绣衣坊外遇郎君。 

　　注：嘉兴产绫，细花纹，称嘉绫。绣衣坊一在郡城，一在海盐资圣寺左。 

　　湖柳丝丝醮碧波， 
　　金陀坊外绿阴多。 
　　落花何处寻西子， 
　　范蠡湖中五色螺。 

　　注：金陀坊、范蠡湖都在嘉兴市内，范蠡湖产五色螺，其中有西施妆台。 

　　妾家却住滮湖东， 
　　爱趁晴江五两风。 
　　正是相思无限处， 
　　子城隔断女墙中。 

　　注：滮湖在郡城南，即今南湖。子城郡府治围墙。 

　　处处名花列肆？， 
　　侬家西市市湖边。 
　　凉风水阁浓香透， 
　　一带篷窗茉莉船。 

　　通越门边鸟夜棲， 
　　马嘶隐隐杂鸣鸡。 
　　五更听得邻船语， 
　　一道斜封到水西。 

　　注：水西驿名。 

　　子夜低翻一曲歌， 
　　湘簾处处隐纤蛾。 
　　人家好似船联泊， 
　　隔岸红楼跨水多。 

　　罢钓归来月正昏， 
　　橹声矣乃市声喧。 
　　乘风直到湖心处， 
　　放鹤湖边又一村。 

　　州倅东厅锁碧池， 
　　柳丝婀娜夕阳迟。 
　　闲来受煞清宵景， 
　　高唱侬家花影词。 

　　忧欢石边草离离， 
　　为郎忧欢无尽时。 
　　可憐石有忧欢意， 
　　侬有忧欢石不知。 

　　注：忧欢石在嘉兴。 

　　麒麟塘外藕溪前， 
　　种剩荷花已百年。 
　　闻道名人觞咏地， 
　　可能十丈藕如船。 

　　注：麒麟塘藕溪俱在郡城东，藕溪明正统初许松泉在此种藕，花开时开，名人多游咏于此。 

　　五色东陵文贝光， 
　　甜瓜味好赛琼浆。 
　　输它玉乳犁还嫩， 
　　不及金鹅蛋更香。 

　　注：东陵堰多甜瓜，其小者名金鹅蛋。 

　　苹花香送木兰桡， 
　　一叶湖心趁晚潮。 
　　网得双鱼怜比目， 
　　移船载放放生桥。 

　　镜香亭子好风过， 
　　翠盖红衣映水多。 
　　水底鸳鸯眠正稳， 
　　教郎低唱采莲歌。 

　　注：镜香亭在嘉兴，旧时亭边多栽莲花。 

　　春风亭下百花林， 
　　遗泽甘棠岁月深。 
　　最是当年陆蒙老， 
　　凉蝉好句幕中吟。 

　　丹枫乌桕护柴门， 
　　仿佛江南黄叶村。 
　　门外南湖菱最美， 
　　胜它风味鸭馄饨。 

　　注：明诗僧智舷筑黄叶菴于郊西。南湖产圆角菱，俗名馄饨。 

　　汾湖水接鹤湖长， 
　　芡子莲心次第香。 
　　怪底无肠是紫蟹， 
　　霜天切莫劝郎尝。 

　　注：汾湖，鹤湖俱在嘉善。 

　　古寺金铭映水流， 
　　宵深海月接天秋。 
　　清光好抵湖天阁， 
　　郎自迷离烟雨楼。 

　　注：金铭寺在郡城，湖天海月楼名。 

　　槐花落尽桂花开， 
　　人向南宫祝福回。 
　　年少风流齐目断， 
　　等闲折得一枝来。 

　　杨柳弯头布谷啼， 
　　黄云叆叆遍锄犁。 
　　连宵了雨三朝雾， 
　　不到秋分稻秀齐。 

　　家住秦塘第几湾， 
　　空明不隔一重山。 
　　劝郎两浆双溪过， 
　　莫向七星桥下还。 

　　注：秦塘在秀水，双溪七星桥在嘉兴。 

　　写山若水富云烟， 
　　慧业文人旧有缘。 
　　卷轴缥缃皆锦秀， 
　　题韯金粟藏经笺。 

　　注：李日华有鸟山楼，项墨林有若水轩，皆种鉴赏家，海盐金粟山藏经数千卷，好事者剥取背纸以供名人书画、名曰"金票笺"。 

　　清宛堂空花石存， 
　　题名犹说赵王孙。 
　　我来太仆祠前拜， 
　　翠墨登登手自扪。 

　　注：舞蛟石旧在范长康清宛堂前，今在小烟雨楼、石上篆文"舞蛟"两字相传是赵松雪手笔。 

　　载酒寻花傍水扬， 
　　秋泾秋叶舞斜阳。 
　　郎心莫向分湖去， 
　　妾意终怜合路长。 

　　注：秋泾在秀水，分湖即汾湖，丰属吴江，故名合路，贝陆游日记。 

　　竹荘花发满沙汀， 
　　桂楫兰桡几度经。 
　　犹记武塘传十詠， 
　　藕花香列水心亭。 

　　梦回鸳枕小锣声， 
　　报道绸船趁晓行。 
　　尽日秀湖城内外， 
　　归来凉月伴残更。 

　　注：盛泽绸船泊郡城南门外。 

　　蟋蟀声中白露寒， 
　　洲边鹤唳晓星阑。 
　　侬家只爱金笼听， 
　　不向裴公岛上看。 

　　注：郡城卖蟋蟀者都聚市于真如寺前。 

　　僧庐到处挂胡芦， 
　　秋老霜高共断壶。 
　　一自高人亲制后， 
　　操来风韻胜厅觚。 

　　芦花白处雁衔霜， 
　　爱棹扁舟过练塘。 
　　最是吴郎多逸趣， 
　　客中滋味饭檀香。 

　　池边乳鸭半双栖， 
　　远树参差望里低。 
　　为爱枇杷多结子， 
　　园东载酒列园西。 

　　祥符塘上芰荷香， 
　　十里茜溪一带长。 
　　不解平川平似镜， 
　　游人惯说是斜塘。 

　　注：祥符塘、茜溪、平川俱在嘉善，平川又名斜塘。 

　　郎来问柳妾寻梅， 
　　镜水花容岁月催。 
　　漫道闻湖梅与柳， 
　　亭空税暑好迟回。 

　　注：税署亭名，问柳寻梅皆桥名。 

　　魏塘精舍渺烟汀， 
　　花气薰窗润画棂， 
　　楼头得月长相望， 
　　燕乐堂边高卧亭。 

　　梅花庵外竹千竿， 
　　梅花庵里写琅玕， 
　　梅花真迹传竹嫩， 
　　高节清风白昼寒。 

　　云东逸史泛仙槎， 
　　琴鹤相随帆影斜， 
　　又见沧江虹贯月， 
　　米家书画属姚家。 

　　菖浦泾上泊鸣桡， 
　　共酌菖浦酒兴豪， 
　　醉后回塘看竟渡， 
　　龙舟飞过会龙桥。 

　　春晴连日好摊灰， 
　　处处烧盐尽作堆， 
　　南北东西团似雪， 
　　鲍郎浦上望郎来。 

　　横山山畔吊逋翁， 
　　吹落松花上钓筒， 
　　家住东南山水窟， 
　　双峰兰若翠微中。 

　　闻琴台畔仰山高， 
　　闻琴桥下荡轻舠， 
　　千载知音遗迹在， 
　　水仙一曲隔云涛。 

　　壁立山头夕照衔， 
　　望夫千载立巉岩， 
　　爱它一片心如石， 
　　望断江南叶叶帆。 

　　两湖湖水碧如油， 
　　禅悦钟声冷素秋， 
　　仿佛六桥明月夜， 
　　游人争说小杭州。 

　　好是冬初天气清， 
　　山容秀丽水容明， 
　　与郎同上鹰窠顶， 
　　乌兔今宵尚有情。 

　　白洋一带渚烟秋， 
　　日暮传来渔唱幽， 
　　海月年年圆又缺， 
　　劝郎莫漫去登楼。 

　　注： 白洋河在海盐东门外海上。 

　　花开金粟寺前浓， 
　　朝莫山僧响梵钟， 
　　石篆不知归底处， 
　　独桑战鼓白云封。 

　　澉浦诸峰接邵湾， 
　　永安湖面绿回环， 
　　随郎前后登高去， 
　　大步山连小步山。
