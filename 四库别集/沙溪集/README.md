　　提要
　　【臣】等谨案沙溪集二十三巻明孙绪撰绪字诚甫沙溪其自号也故城人治己未进士官至太仆寺卿是集文八巻赋一巻襍着一巻无用闲谈六巻诗七巻其文沉着有健气其无用闲谈有曰文章与时髙下人之才力亦各不同今人不能为秦汉战国犹秦汉战国不能为六经也世之文士尺寸步骤影响摹拟晦澁险深破碎难读云云其意盖为李梦阳发可以见其趋向矣至于古今仕学辨之类叅以排偶不古不今则编次者失于删汰转为作者累耳其无用闲谈多深切着明之语论文论诗亦各有确见王士祯池北偶谈尝摘其误以五代王祚事为彭时事其説良是他如论扬雄事亦失当然要不害其大防诗格颇近李东阳而深以何孟春等注东阳乐府称其过于李杜为非盖讥誉者之溢量非排击东阳也此集旧与马中锡东田集合刋然学问笔力皆胜中锡故今摘录绪集而中锡则存其目焉乾隆四十四年二月恭校上

　　送马嘉贞应试南还序
　　妹壻马君嘉贞天分警拔事举子业虽未尝矻矻苦心至落笔则语輙惊人老成尔雅迥出常度同业者终嵗效之竟莫能及人以是竒嘉贞嘉贞亦以是自竒也然每试輙不利今年秋学愈竒声誉愈闳应试士相率即其寓所愿识风仪者日数人或相顾而惊曰此所谓马嘉贞邪一时推右殆空士类而试又不利揭晓之后抚膺防手幽愤无所容充其志将驾当世吞万变浩然不可御噫士贵有志若此者非士之所宜有邪所贵罔敢怠忘念常在兹耳若突然之勇生于血气之粗偷安之习乗于志气之微愧目前而忘自奋则今日之志适足以见褊浅躁急为吾之累反不若漠然于得失者之为高也余早年屡困场屋每被黜情思黯黯窃窥一时同黜者孤灯聚首惨无人色程日于仓卒屈指若不及数日后游访贽谒语笑哑哑前日之情畧无芥蔕焉间有秉持少异者众中常愀然不乐与之言亦不对若有忧思然至抵家室对妻孥亦既舒畅矣乆近虽不同其粗于血气微于志气一也而况嗜欲乱其真博杂挠其识不急之务无益之谈妨其功力则今日之志亦安用哉昔者孟郊落第有食荠亦苦强歌无欢之句一时君子怜其情而鄙其量夫郊诚可非也然卒能占高第享盛名于无穷不犹愈于一被摈弃即落魄狂纵万事决裂如李山甫温飞卿等乎老而弥笃仆而愈力正今日所未易得者而顾未可轻议也凡人之失利孰非郊之情哉顾充而守之有愧郊耳嘉贞勉之天下之事多矣曾有二三其心荡漾不定而可善其成者乎况事业精密易失而难得者尤可畏哉归以告尊翁当不以予言为诬也
　　送李徳华诗序
　　徳华我先师嘉兴先生仲子也先生早擅三易腾声邹鲁齐卫间丁酉嵗首擢乡书虑太夫子渐老不欲以仕逺违色养需家食者十六年余往洒扫门墙见先生孝友睦宗族信义惇乡闾谨饬方严淑后进謦欬杖屦无非可师有不専经籍文字间者退不胜自喜时徳华侍先生首尚未髪而冰莹玉润沈静有老成风私自谓曰传先生之业者二郎也己未先生殁官嘉兴每忆高踪即于徳华属厚望不可置今年应诏来京例得医学官医为小道医官在仕为卑员而徳华趋之若恐后诸友在都下者咸讶之亦有力尼之者徳华泣曰葺遗文以承先业先子志也吾诚愚讵敢忘先志哉顾兄没母老谋仕则废养奈何且吾父儒宗尚以母老不欲仕况区区者可轻违吾母邪冠裳捧檄以博庭闱懽悰亦足言仕而终无逺游以贻亲虑谓非先志土苴与闻者韪其言谒选既毕治装将南还诸友供张祖道都门外徳华酒酣兴发言论蠭起讥评人士商畧世故若鉴应形水赴壑孤翮高抟而迫之以风也虽未可遽窥先生之堂然背而驰去者亦鲜矣至忆旧游话往事俱歴歴在目转盻之间已越二十余禩今昔异时少壮异容怅嵗月之潜移懔余生其如寄不能不以之兴懐古称后之视今亦犹今之视昔悲夫余因歔欷而歌曰昔从君兮平原区花顔肉映云敷腴君今来思西城隅青袍荡漾飘琼琚丹穴郁郁文鹓雏倏然翛翥云霄衢高歌取醉真良图夕阳又下闲庭除歌阕复有倚声而和者曰若有人兮邦之英驾騄駬兮弹青萍驩未洽兮归干旌瞻令仪兮心怦怦去如风澹来云凝愿言两腋清风生相与上下同防于是四座竞起叠和更酬剽者如游回者如留飏者如浮拍者如休方清絶以长讴忽断续而相摎若野鹤之唳高秋孤蝉之咽凉飕也徳华雀跃而起曰有是哉愿彚诸君之作以华吾归乃僭引余説于上而列诸作于别方若岐黄之谈本非徳华素业亦非诸友素期者可不书也
　　代杨师文送董松轩赴吴江簿序
　　吴江江南剧邑纨縠靡丽之风比他邑常什伯去京师数千里狡横且玩税额讼牒未易猝如法故令若簿多败于赇或困于伙绐于诈能官者盖鲜然令患不才耳才则搜抉指顾咄嗟可快意簿动制于令晨起束带诣堂上长揖就坐吏抱牍过前禁莫敢问惟令顔色是候令诚才邪相与同其休不才簿虽蝉蜕鹤举不得不与分其谤矧台郡相沿督责常先僚贰僚贰奔走惶骇令不知也然则吴江之难令居其一而簿居其三亦诚难矣哉甲戌四月夀张董松轩先生得是官余前此屡先生寒暄外他事若不肯出诸口除后再往之则言论纚纚终日不可穷要不出吴江今时最急与簿当最先者岂先生外此固漠然哉非其责可不言亦往日意耳夫处穷巷经世务要非凡士屈伸舒巻惟义惟时求先生于今兹诚未易得以是佐吴江不烦他求自有余地事当在令可以无言事当在簿而令挠之不可无言也士从仕凡以行吾志而已今锋镝供亿之余闾阎糠籺嗷嗷未充者什九幸而吾有具又理当任顾低首饮气自抑以悦人如吾民何即先生之迹渟滀如巨浸激发如风驶中綮要如破的知非若人也然则又何难焉其子汤民侍御常凝神洞视于广众矜语间若有深思至其论事则综覈孤峻逺出人右迩者天子命按畿辅简书甫数日贵势即相戒莫犯盖澄清搏击之术虽深匿未形而锋铦溢发有不可终掩者如蟠蛟螭隠虎豹未及腾踏奋厉而渊然之光蔚然之文泅采者固已望而惊矣英鋭胜于恢容才华韬于浑厚父子志向如出一辙汤民掇贤科跻膴仕而先生矻矻太学晩嵗仅得此是则幸不幸于其间昔人筮仕县簿垂休光照后世如苏长公朱晦庵者史册相望又未可以秩卑任轻専为先生屈也古今宁不相及邪为之上者从臾诱掖之使先生之贤不终汨于氛埃奔走之下则多可称述当复有进于是者濒行羣僚追送国郊且有诗也谓予虽陋年差长强使为之序
　　送如心宗师北还序
　　正徳丙子八月如心玉上人进宗师之号将丐余文以侈其事余心亦既许之矣未几被谪南归匆遽流离心悸气动如心嗫嚅齿頬间欲言之未敢也今年秋裹粮负担从一徒南走千余里抵沙溪别业再索之夫如心之庐缙绅士日接武焉音铿鍧而葩炫熠无难得者顾逺需于蓬茅废罪之余是何忍于终辞余早嵗识如心之师印空上人因与如心游印空博洽持重诸浮屠敬而畏如心谦抑温润诸浮屠慕而爱行孚论定越三十年如心虽无意人知诸浮屠争先延誉惟恐誉之不扬汲引之无阶者无戾情无异词乃共上其名于礼部礼部密诇而公询之始疏以入投名陛见畧如授爵然因念一方外之流欲伟旧观改故步亦甚难焉天下事稍涉荣幸者要不可易致不独浮屠然也每春夏之交宗师登坛宣教村民比丘累千万长跪俛首以听于下暴慢者悟悔悟者泣而所宣者戒杀去贪轮回报应固释氏之粗也犹感人若是若以宗风上乗弥近理者示之又当何如怵奸慝借风示濳驱默相有未可尽非者吾儒懿徳好爵当更倍蓰什伯而学巧于岐仕防于径往往沦为虚谈行不重于时言不信于人功力顾出其下而人人以辟邪卫正自任秪见其诬也已昔恵逺道安辈颇见重于晋宋间今日讲经某寺明日谈空某第王谢诸君子圜坐竦听不少厌亦以其説有可以耸动世俗者不然操笔立论据孔氏以排之岂诸公所短邪如心今日之崇重亦何减于古昔宗伯不逆其情当宁不违其请合千万人羣聚喧呶连旬累日于郊圻近地有司不以为非民俗不以为异岂徒为如心地哉如心之自处要亦无愧古人可也旧见如心谈笑咏歌终日不倦迩来黜聪明谨酬答以求副宗师之实但每与余相对死灰槁木无复畴昔之兴余輙欠伸思睡如心果能忘情于世乎少壮嬉游如在旬日而余頽然亦既老矣抚景遐思识此以为别
