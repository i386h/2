　　●王文端公尺牍叙言 

　　王文端公尺牍凡八卷既成刻矣先是嗣人两中书君裒汇全帙次第授梓奏疏诗文各为题辞而至是以尺牍征叙言爌得以受而卒业焉此于公集中一体也而未始不可以窥公夫尺牍之为用也广矣交知往来缄题训答如第骈联以绚丽藻纤妙以袭清裁施之于庆颂谢请契阃寒暄间即十吏递供百函竞发或可聘才人之致为之而至事关要重机判危安质问疑衷禀仰成画窾言靡得而与焉故夫书记辞命之流也稡兼长以应对而后幸无失辞而欲以操觚之顷得算于心而奏成于手庸几乎公之书牍其启状俪体非不采溢琼玑响谐宫征而情以纬物质有其文其它书奏揆事抒衷端言核论■〈忄款〉悰毕露而文辞烂然而又有异焉者公自官史局蚤负物望四方之士延慕声光书题殷凑每得一赫蹏奉为驷壁□跻揆席迨秉政机诸方隅握重之臣时以军国大计取决庙谟而公晨趋密阁拟涣汗之丝言夕发邮筒授机宜之石画诸如封疆备御之坚瑕夷虏情形之劲弱战款声实之先后以至漕河疏塞之急缓水田开垦之拂顺盐筴鼓铸之壅通莫不烛照几先符合事后默持一是而潜醳纷疑铢分不失也公为文不属稾其启奏报章据案占答斤削不加而辞旨具足此非识略裕于中藏而仓皇以之批导能乎哉昔人论文贵切世用公之制作其鸿篇伟构用以扬谟润猷者固自有在乃至经国之先筹救时之秘略不出尺幅而绪余概见则是编也不必弦次集中即以之孤行于世可也爌以乡邦后进未得望见门墙而于公立朝大节仪世高徽慕向之私不能独后里居每得公文辄肃衣冠读之诸笺答名言亦时得之胪闻如其自叙有云入告退言一心一口又云内不敢求知于宦官宫妾外不敢得罪于贤士大夫又如以虚心观理专而有渐为谏诤之宜以功可相成不必自我为柄事之宜以进身安静门径远迹为始仕之宜甞意斯言定能弃世而兹以参之全牍非经济之绪言即禔修之箴诲不以竿牍视而以典训求无之而非可诵可法者也有志之士旷代得师而况邦之哲人典刑非远象贤竞爽诏我师摹读其文想见其人末赞一辞仰止而巳抑晋之先大夫不有望九原而知所与归者乎睠三立之前修窥一斑而自淑固上愿也托姓字于简端附声以永世则犹其次焉者矣

　　○上晋王贺册封 

　　仰惟殿下温文成性敬慎禔躬养正于蒙夙禀刑家之训承祧惟震式符当壁之祥肆启介藩光膺显册禹宅下土首开晋冀之疆周笃懿亲蚤建叔虞之宇执桐圭而作宝分玉辂以疏荣庆叶三灵欢腾四履某忝依封部幸隶编甿朱芾斯皇久企宜王之度华裾可曳阻陪从史之班敬托鱼椷肃申燕贺伏愿宣昭令德缵服先猷绵胤祚于本支奠邦家于盘石其为欣抃讵克敷陈 

　　○上楚王贺册封 

　　仰惟殿下冑本神明性成仁厚令仪令望誉蚤着于衡湘克长克君化浸行于江汉肆膺宸眷肇启藩封承周室之脤膰允光帝典分汉庭之册玺式备王章凡在司存莫预周旋之力矧惟虚鄙敢当劳赐之仪敬用壁归伏希睿照心之感结词不具宣 

　　○荅周府西亭宗正进书 

　　仰惟门下盘石大宗渊源宿学左图右史聚书不厌旁求往行前言畜德允资多识贮氷壶于腹笥垂金薤于毫芒润饰昭代之典章事详三箧振起中原之文献力劲千钧公族赖以主盟国人于焉矜式官并秩宗之宠齿冑陈仪道符彤伯之尊躬行示范允循循而善诱肆振振其率从讵惟董正之功寔赞修齐之化不佞技惭墨守识愧朱愚卓尔不羣久慕河间之雅猗与有斐未窥淇上之文伏辱德音亶谐素愿顾箱盈琬琰知控谢之非恭乃篚实玄黄本登嘉之莫尽敬抒悰于介绍庶缔好于平生 

　　○上晋王谢慰在告 

　　某用世非才致君无术偶以册储之议谬陈逆耳之言上咈圣心遂千严谴孱夫扛鼎竟罹折足之凶孤犊当辕宜速摧轮之衅硁硁引决但凭小丈夫之心胸啧啧推称敢辱士君子之齿颊罢归田亩幸迩藩封居父母之邦获依桑荫被亲贤之泽窃永椒聊诚假余年可苟安于耕凿庶徼洪庇将厚托于帡幪讵意睿慈特垂温眷劳使臣于绝塞惠音札于衡门珍篚充庭惊睹七襄之烂衮言盈楮宠踰一字之褒登拜知荣省循增愧附谢椷而三沐赐莫报于琼瑶祝庆祉以千秋祚永凝于盘石 

　　○上代王谢慰在告 

　　某奉职无状赐玦以归罪废之人乡闾所耻幸遇周亲启宇明德绍庭屹为盘石之宗茂衍璇源之泽国人共戴讴歌无间于前王倦客新还耕牧尚知具故处卑枝可息即同大厦之帡幪涸辙易盈敢辱西江之灌注猥蒙睿睠厚损贶颁分俎豆之珍业沾异渥盛篚筐之具愈倍恒情登拜为荣省循殊愧肃申谢悃附布末忱伏冀鉴存曷胜瞻感 

　　○上晋王谢贺生 

　　伏念某蓬蒿陋质樗栎冗材自废居畎亩之中适隶在汤沐之邑敬哉有土抚唐叔之遗封卓尔不羣迈献王之令望雍雍在宫之范以御于邦泱泱大国之风丕变乎俗山河表里咸依藩翰之尊岩谷幽阴倍荷容光之照不独纳污而藏垢夫且嘘枯而吹上岁当弧矢之贱辰辄损篚筐之盛贶上仪隆缛玄黄贲采于衡茅睿旨温谆丹雘增华于粪壤宠非其质登嘉无任伛偻恩结于心循省何能报塞肃而下走奏谢牍于属车僣附微芹表积诚于荐盥愿起居之珍护绥纯嘏于缉熙不尽瞻依式廑颂祷 

　　○贺乐昌王太妃诞日 

　　小春应候南极开祥恭喜敞佳宴于瑶池萃欢心于朱邸九秋露湛香浮王母之觞五色云高光捧元君之驾寿直齐乎仙佛庆允洽于家邦某忝托鸿禧阻申燕贺拟效华封之祝聊陈芹藻之仪伏冀麾容曷胜宠藉 

　　○贺张洪阳拜相 

　　尝谓正人在位朝廷自尊君子得舆善类交庆盖表仪系于所树而声气要在相投仰惟门下拔俗贞标格心大学嘉谋告尔后向推启沃之猷闻道在吾先备荷切偲之谊寻更聚散互有去来即嚣然忧畎亩之中巳屹乎负岩石之望祥琴甫御宸册涣颁求忠臣于孝子之门无烦梦卜置良相于明君之侧允叶赓歌仰八柱以承干光赞垂裳之化翊重华而出震会收定策之勋弟某同寅恊恭幸参联于揆席乘时辅运愿遄驾于锋车不尽欢欣但切延伫 

　　○贺沈蛟门拜相 

　　仰惟门下渊涵大学岳峙贞标道裕弥纶劳深启沃向当彩服趋庭之日业廑紫宸侧席之思而皇情徒注于安危至孝直期于终始三年礼室久虚作厉之求一旦枢庭顿慰瞻岩之望泰交孚合讵惟宗庙社稷居歆解泽沦濡将俾草木禽鱼咸若制麻邮布草野驩传藐踽踽之孤踪倍津津其喜色宁忘情于贺□柰绝口于言朝猥廑台使远临珍仪骈锡谊敦故旧觇知吐握之虚怀仁急颠连想见施为之次第欣焉手额感欲魂摇即弹冠之愿永违乃安枕之期可卜敬占谢楮并附庆私物菲心虔形留神往谟谋密勿夙钦补天浴日之猷匡济囏难伫睹旋乾转坤之畧 

　　○贺沈蛟门首揆 

　　恭喜格天业懋巍居首揆之尊取日功高遄定元良之位朝廷一二日万机之政务尽倚决于渊猷祖宗千万年一统之基图倏维安于鼎力纾房乔之谟谋剸以杜克明之果断挈二相而兼总其长养狄公之沉密发以韩稚圭之雄刚匡一主而两收其效此宗庙社稷神灵所共孚歆朝野华夷伦类靡不歌颂者也弟某偾辕孤犊逃笼倦禽息影销声招惊魂而未复操心虑患省积咎以滋危乃幸晋陟上公独提魁□窃谓专精于寅亮伫观运掌于经纶讵图泰阶之柄象方新震器之升华惟亟咫尺而乾坤旋转俄顷而天日熙融紫诏颁恩庆宽条之槩及丹书记过欣宿盩之都蠲若和羹汝作盐梅赖夑调其辛苦人刀俎我为鱼肉荷保护于存亡从此奠枕而安衡泌之□敢不书绅以识帡幪之赐肃裁荒楮走一价以扫门薄荐菲芹企三台而贺厦冀垂鉴照良切瞻驰 

　　○贺陆五台太宰 

　　今朝廷所重者人才郡国所急者吏治品格资于鉴别绩叙视所铨综必权度之素精斯统均之克称恭惟门下道心澄彻涵水鉴以无私德望尊严酌斗枢而独运山涛再居选部夙推启事之名姬公晋陟冢卿益遂进贤之志人才长短若欂栌榱桷之异用巳预度其所宜吏治污隆犹泾渭淄渑之分流胡激扬之弗效世道从兹清淑泰阶赖以登闳喜倍切于弹冠愿莫谐于听履敢藉鱼椷之便附申燕贺之忱冀护八座之启居茂膺九重之眷顾 

　　○贺郝少泉司农 

　　仰惟鼎衡宿望经济宏猷宜进上台不俟今日适疆事萌芽于南北而圣心轸注于安危特简股肱之良托在本根之地导泉货而布之上下国脉端赖以疏通持斗杓而酌其盈虚元气允资于调护盖周礼诏会格心之术居多而大学理财絜矩之方□约允惟耆德克总繁机顾留都之揆席久虚众不胜其徯衮乃执事之封章屡上愚亦讶其费词迭承天语之叮宁兼迫锋车之督促三让而进虽盛德之弥谦一节以趋知吾言之不宿正拟劝驾猥辱赐书感惠贶之骈蕃得无惭悚念宠福之滋至寔切欢欣敬端一介之微代陈谢悃并致寸丝之献用抒饯私冀原谅其不躬邈神驰而未巳
