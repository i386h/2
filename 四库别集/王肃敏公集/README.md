　　●沟断集序

　　韦生有云卧看南山改旧诗余端居无营偶阅旧作意所不满者不忍弃置稍加撺削以成篇章自弘治壬戌至正德己巳共得诗赋文一百八首题日沟断集将以附诸稿之末嗟乎始余之有作也未尝不自善之今也索然乃不适意不欲弃之又从而文之不类于沟中之断复取而饬之以俎豆之乎将不谓神矣乎余不知他日视此又将何如也杇腐而神奇神奇而杇腐将在物乎将在吾心乎二者必有真辩嘉靖元年十月朔日浚川王廷相子衡甫序

　　○悼时赋 【 并序】

　　予谪亳之明年为正德己巳时政愈急抑欝愤闷卧郡斋者数月乃赋以自释其辞曰
　　何予生之弗获于时兮熙和返而凄其固玄化之有变革兮情沉抑而如危苍梧邈而皇逝兮重华逖以何追毕原芜而凤隐兮文胡为在兹繄哲谟将以援世兮众咸难予之施岂弗省同流之为自全兮非素心之所期虎九关而吓众兮鸮群啸而昼翔灵曜曶其冥塞兮两仪黯黮而不彰巉乔岳以路绝兮江河广而无梁帝远而不照予之忠诚兮知进退之不予臧诹同心以求正节兮曰圣哲之所常年鼎鼎不予贷兮恐清修之不杨希凤鸟为友于兮接文鹓为予侣结薜荔以为绅兮带芳杜兴函芷予恃此以自媚兮奚取容于羣子苟终获此皜皜兮虽没龄而靡悔彼饬婪而好进兮予犯众忌而获愆恒申申以忠告兮彼朋訾而不予然嗟时命之不可逭兮矧毒夫之乘权在圣智或靡周兮予孱劣而奚宣强心娱而偷安兮非道轨之所尚览彭咸之茂洁兮纷腼面于俯仰哀日暮而途迫兮胡不高驾而独往苟予踪之可以遁灭兮又何情恋于吾党涉洪河以扬舲兮泛沧海之三洲登崆峒以发轫兮历五岳之崇丘云霳而蔽亏兮潮淼而吞舟丛万崄之不可越兮呌帝阍而无由将东浮于三韩兮南走乎盘瓠之野趍疏勤之龙沙兮及革胥于西夏逓重驿而靡通兮虽吾笑之莫辩俗淫悍而殊中土兮更阴之为沴云阳僻陋之可居兮要情性之舒展吾生尚不自得兮何九夷之为善超三山于鳌极兮栖神洲以为家友羽人以倘徉兮接浮丘之灵槎驾玄虬以远游兮折若木之瑶华扪三辰而载歌兮凌倒景而回车固蒙庄有此寓说兮寔杳茫之无据予岂贪生之常流兮迷异道而诬世诸有求而莫获兮反予聦而大观予苟不畔于道榘兮行穷困而志完矧沉冥为真诠兮终觧脱乎忧患要时命之可俟兮胡悒悒以长叹乱田维河潏潏有泚清兮牝谷之窞矗为陵兮洚水滔天底平成兮理乱相禅靡常行兮詹尹羌?目筴蔑以征兮卓彼高土遗厥荣兮顺命适变圣通明兮殷忧冲冲文伯与兮履危俟时集尔庆兮委顺抱一维土之贞兮

　　○十月之朔剌执权也

　　十月之朔有篲维芒触于斗筐匪辰之常少师亮工司马职方不宁乎位孰执国之臧
　　天降憯祸寺人孔回如火烈烈沦胥以灾贿门恢恢谗口摧昊天有明宁不我哀
　　天降憯乱彼奄卒狂为豺为虎沦胥以亡大夫隐忧庶民遑遑昊天孔昭宁莫我伤
　　彼有相猷汝则遏之彼有虏伐汝则夺之巧言便便汝则达之背图弗底汝则干之
　　亦有喜士为谋式固告尔王度逢尔震怒维彼忍心败及■〈执上目下〉御覆狂以雪靡用后惧
　　嗟尔君子勿作孔吓如彼从禽罟阱是获嗟尔民庶勿作孔厚毒罚罔极弗念尔寿
　　帝听之逖百式之反也尔日衎衎我涕之泫也维斗有枢力不可转也维宿有翼飞不可展也

　　○云汉勖友也

　　昭彼云汉下民企止彼圣之轨孰与迪止
　　维河潝潝会于东极沇别为济终晦其则
　　南山之阳有梓有桐匪圣之修孰遗其风
　　维圣之修翼翼由由彼戾者子我心之忧式诲尔猷与子同仇
　　我用是戒式世之靡我用是大式世之纪匪我是覃式以告子
　　鸿渐于陆载栖于泽我有蓄慝匪子弗克子不以我德式子之惑
　　云汉六章三章章四句三章章六句

　　栢之实离离兮予美兹令仪兮子之丕兮嗟子之绥兮

　　○独漉篇

　　独漉独土蝡泉浊不可溉蠲胡徼尔福彼狂弗率当关秉戈虽有典刑视云则讹中原有菽惟莠芜芜彼农伊何宁莫之图虎则在门蝮则在室兄弟出走则莫我食我有莫邪光则电电谁其侠者不敢以献海澜悠悠中有风舟我骇其危彼不知休

　　○昭君怨

　　古人皆去国何如出塞门苦雪狼山冻惊风沙海昏酪浆气日变金琶调自翻塞涩弦曲中弦忽断流泪比君恩

　　○长门怨

　　凉风吹罗袖凄凄瑶殿深桐叶下金井月明寒素琴扑萤忻淂火梦妆愁断簪看星玉偕里空有抱衾心

　　○铜雀妓

　　分香眷情苦望陵遗恨滋空怜长夜宴无复此生时秘殿深松径高台泠蕙帷虽云歌舞妾揔是断肠词

　　○湘妃怨

　　远游自迷放茫茫望百粤千秋亦有穷恨不亲诀绝愁逐梧云起魂涉湘江没斑斑竹上痕常沾泪血发

　　○临高台

　　临高台天风凄以寒桂槚椅桐飒呇堕琅玕凤翘翘兮骊龙翻飘太虚泛广澜穷历止不四翰四雏孤雌痴而独立谁为援登台怛栗推心肝攀帝都泣帝隅引帝之裾帝赫怒吁勑巨灵六丁行扫除系弱挟广车驱射杀杀台下诸狼狐奉君归有虞曜精垂光天德敷

　　○琵琶引

　　仙娥不作霓裳舞独抱琵琶学凤语四弦嘈嘈催早春雾洞云轩花争吐一弹春宛宛莺语初调娇莫辩再弹春茫茫武陵花泛流泉香三弹四弹秋浩荡风雨交交万潮上湏臾云散江月沉不闻风雨闻龙吟五弹幽怨不可穷千鹤万鹤高唳空仙人百呼不一返羣飞直向东溟东碧梧金井辘轳转逆旅牵愁常不断听曲飘扬星汉低池上鸳鸯双趐敛万里明妃锦??朋马空然抱怨阴山下

　　○赵尔锡病愈

　　一病经时卧文园忆长乡春厨余药裹夜阁癈书檠因悟禅心寂番教世虑轻何年缑岭去与子共吹笙

　　○癸亥元日

　　斗柄春星勲少年玉律回光风轻转蕙晴日暖烘梅彩彩椒花颂深深竹叶杯莱衣又莫展怀抱不成开

　　○五杂组 【 二首】

　　五杂组凤凰毛往复还沧海潮不得已战鸣条
　　五杂组庆云色往复还苏武节不得已受巾帼

　　○宫词 【 二首】

　　花扑珠帘玉殿春翠娥争队唱歌新如今被宠多娇贵领得霓裳不着身
　　芙蓉双衩曳秋裙十二峯头岸云不道昭阳新失宠玉箫犹幸月中闻

　　○春日馆中咏怀 【 二首】

　　鸣鸠文苑静欝木午阴森高步登龙日清华泽豹林孤琴流水曲吾道太玄心谬迹神仙府春风瑶草深
　　华馆宫城近朝回淑景移依稀幽凤萧浩荡碧云期社燕要皆至春花亦不迟虽云宦游客何异在烱时

　　○入直简许启衷

　　寓直清春早彤霞逼曙开文楼切霄汉革盖拟蓬莱事以临轩奏章缘谏猎裁同趍青琐客独羡长乡才

　　○舟泊采石口

　　三日南风送孤舟滞水乡闺人拾苹藻稚子歌沧浪楚嶂褰帘碧江蓠拨棹香浮生靡定止闲处足徜徉

　　○庸之郊墅宴集

　　习素秉超旷养痾希沉冥耕稼果易足盘桓遂初营西郊启欢宴良友展深情垣薜纷暮委庭菊丽秋荣鸣鸟乱将夕飘霞度疏明长啸林墅寂野酌池台清陶令去千祀伊谁契幽贞世网苦未脱翻然愧兹生

　　○张孟复有东园读书之约次韵谢詶

　　抱痾栖闲迥自如野园结构及春初久嫌城府寻幽僻况有高明慰起居草绿沧洲侵杖屦花荣楼阁映图书茅斋窃比匡卢会高论承君下榻余

　　○村居有怀同馆诸友 【 二首】

　　草阁朝光动闻锺想玉除清曹集旧彦白日候金舆已上忧时疏宁无谏猎书却惭多病质幽独赋闲居
　　徐步东林外萧条岁又徂寒螀辞草野疏叶下庭隅地迥书难得人闲病易苏空然慕同侣素志已荒无

　　○黄正郎内子挽歌

　　鹤背鸾车向紫霄洞天七十梦迢迢麻姑行月攀丹桂王母开池宴碧桃俭素平生留白纻徽荣前日上龙绡金堂若取遗音在转觉铅华涴六朝

　　○塞下曲

　　朔气榆关早边声漠鴈来防秋遥度碛游击夜衔放虏帐闻风散胡笳带月哀将军占利涉莫近李陵台

　　○赠张孟复

　　东街张仲尉洼海赤龙驹泉石耽高尚风云屈壮图逢春多艺药闭户独看书何日公交车诏乡人羡硕儒

　　○寄张尹

　　春风宓子国白昼坐鸣琴俗朴陶砦易民贫感慨深涨沙胶宿麦亢日烁青林远忆流亡者应劳惠爱心

　　○忧居答粹夫 【 二首】

　　丹殿辞归抱薜萝佳人一望渺烟波东瀛曾接神仙步南国犹闻太史歌虚拟卧龙栖草泽遥怜仪凤下云和掖垣春漏催金拆正想朝来响玉珂
　　碧水沧洲旧索居忧人情思近何如青春有约攀花懒白日无营见客疏敢谓梦魂忘魏阙已将心迹附江渔不堪药饵殷懃在尘满床头二戴书

　　○秋兴 【 四首】

　　宋玉天南客秋来愁自深清霜数行鴈明月万家砧
　　窈窕青林下荷风醒昼眠苍蝇避尘尾拂响按头弦
　　片月上东方照我水西阁相对可怜影荡漾不能掇
