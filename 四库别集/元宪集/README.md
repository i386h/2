　　元宪集序
　　荆湖之间国朝以来安州为望郡名公钜卿相继而出元宪景文宋公伯仲则其杰也昔太守今右司李公揆绘其像而立之祠逮之强之来请于郡而春秋致祭亦可以夸示后学矣然宋公之典型虽在而文集不传于乡郡谓之阙典可也寓公李令尹之家旧有缮本太守今都运王公允初昔为通守每与之强言欲借而刊之未能逮持节京西于其行以帑藏之余几千缗属之强与之锓本以广其传又分数册以往将以速其就也然攷之二集既富且赡其言八十余万工以字计为钱几四百万米以石计百有二十他费不预焉之强惧其难成而白之今太守陈公芾公一闻之欣然谓之强曰是亦予志也郡当疮痍之后虽赈恤施予日不暇给然亦当辍他费以成之二公政事文章两极其至故能于倥偬艰难之际而为粉饰治具之举其与蓄财而不知予妄费而不知用者岂不大有迳庭耶之强深有感焉因其成也书之篇首以告来者若夫赞元宪景文之钜篇大作则有国史在于前辈之铺张扬厉而为之序则之强晚生不敢措辞观是集者自当知所敬叹今之序姑记其文集之传云尔嘉定二禩三月上澣郡文学陈之强序











　　钦定四库全书　　　　集部三
　　宋元宪集　　　　　　别集类二【宋】
　　提要
　　【臣】等谨案宋元宪集四十巻宋宋庠撰史称庠所着有国语补音三巻纪年通谱十二巻别集四十巻掖垣丛志三巻尊号録一巻今惟国语补音有传本已着録余书与文集并佚
　　国朝厉鹗编宋诗纪事仅采掇西清诗话侯鲭録合璧事类扬州府志所载得诗八首则海内絶无其本已三四百年矣永乐大典修于明初距宋末仅百余年旧刻犹存故得以采録而庠文章淹雅可取者多故所载特为繁富今以类排比仍可得四十巻疑当时全部収入也方回瀛奎律髓载夏竦守安州日庠兄弟以布衣游学席上各赋落花诗竦以为有台辅器赵令畸侯鲭录亦云二宋落花诗为时脍炙今考庠诗所谓汉皋佩冷临江失金谷楼危到地香祈诗所谓将飞更作回风舞已落犹成半面妆者特晚唐浓丽之格实不尽其所长祁集有和庠赴镇圃田游西池作极称其长杨猎近寒罴吼太液歌残瑞鹄飞句叹其惊迈蔡绦西清诗话亦称之又载其许昌西湖诗凿开鱼鸟防情地展尽江湖极目天旷古未有然集中名章隽句络绎纷披固不止是数聨也文章多馆阁之作皆温雅瑰丽沨沨乎治世之音葢文章至五季而极弊北宋诸家各分起振作以追复唐贤之旧穆修栁开以至尹洙欧阳修则沿洄韩栁之波庠兄弟则方驾燕许之轨譬诸贾董枚马体制各殊而同为汉京之极盛固不必论甘而忌辛是丹而非素矣陈振孙称景文清约庄重不逮其兄以此不至公辅今观其集庠有沉博之气而祁多新警之思其气象亦复小殊所谓文章闗乎器识者欤书录解题载是集作四十四巻与史不合然文献通考亦作四十四巻似非讹舛疑别本以掖垣丛志三巻尊号录一巻编入集中共成此数唐宋诸集往往有兼収杂著例也通考于是集之下又附注曰一作湜中集二十巻其名又异然永乐大典实只标宋元宪集则非湜中集明甚故今仍旧目不取通考之名焉乾隆四十九年十月恭校上

　　乾元节赋
　　炎厯四世天子坐法宫宪宗轨深根宁极逺聴髙视其仁如天其道如砥亹亹穆穆为纲为纪方且秋驾轸虑宵衣致治摽甲卧鼓张旃厚篚至和薰而桀骜革鸿泽沦而翾蠢暨龙宅沼牛羊避苇民逺于罪画衣冠而知禁吏久其职保子孙而为氏礼无未补之阙乐有既尽之美日月之所烛霜露之所坠举踵延颈莫不率俾而乃辰居乎蠖濩字泰乎醰粹起真想于黄庭遡庆源于赤水陶钧之所独运声教之所横被遐哉乎越不可载而巳先是有司摅旧典辑元命因诞弥之吉序稽长发之休咏攷干健不息之象含元长体仁之性作为佳节冠于甲令将以合万国之欢啓千龄之圣掩周汉而遐蹈齿尧顼而髙映寖明寖昌惟天下正则是礼也宜我朝而为盛若乃仲吕鸣律恢台御时斗建于已舒将望羲授赤光照室之瑞叶宝轸歌薰之期内外欣踊官师肃只于穆先灵辰备宿设善气前应游氛外彻开阊阖之洪洞廓紫宫之寥泬飞龙抗殿神鼇负阙若夫宣室先帝之正寝大火天王之明堂启稷嗣以辨等率鸿胪而诏方采物纷树衣缨焜煌金匏畜响而待叩籥翟旌勲而启行庭实万品天幕九张哕銮卓马之觐戴斗占云之乡砮丹文越効其质毛车象戴输其祥或布濩于外庭或就望于西厢其内则上宫硕老衮衣绣裳元侯显父珍圭判璋麟定之亲信厚而在列鹭廱之客洁白而来翔莫不委佩垂绅肃穆端庄以竢乎当阳始乃警大庭导初旭纎罗靡动羣隂尽伏辇出于房帝晖有穆正法座于皇极凝邃旒于元服灿乎若瑶极之御羣宿澹焉若天池之朝百谷涂山防兮后万玉华封来兮善三祝岂徒扬甚盛之节盖将储难老之福况乎鷰禖虹渚天启也虽旷代而罕至重瞳采眉君表也惟元圣而后备画堂甲观昭汉嫡之祚金鉴露囊盛唐宫之制盖所以谨帝胄之正统钟天元之粹气谨声身而为律度聪神明而资孝悌作我司牧光有神器若夫定黙之至数重熙之髙致蕴上业而未发候昌朝而来萃则不若是也于是丞相奉觞而进曰臣闻徳之所积者厚则天之所报者隆道之所召者逺则神之所赞者聪今陛下尊祖袭宗寝威盛容退姧士以进道斥雕纂而敦风向学乎金华之殿因心乎长乐之宫程百斤以忘倦慎万几而执中而犹翼翼以惠迪谦谦而养蒙宜乎继万世之典业蔼千祥于圣躬焕至徳而有融等仙劫而无穷者也臣不胜大愿昧死上后天之寿制曰都此髙厚之降康宗祏之流光俾尔炽而昌俾尔夀而臧朕其志哉所不敢忘防举公之觞时左右皆称万岁
　　幽牕赋
　　暗室迷阳开疏就方匪结钱而效饰将修隙以闚光森竖櫺之寸漏限横杙而寻长逼南荣之爱日蔽髙庑之飞霜隠几于下陈篇在旁含阶筠之媚色泄盘蕙之幽香况复霁景初和游氛外灭轻吹袭牖頽阳溢穴映故网而轻明渺方纱而洞彻但见野马羣飞纎虫族悦或蠛蠓之上击或醯鸡而下颉轻若毛举来逾电掣隠见互举如翳如空渺倦目以旁睇监众态之无穷其来也就煦以交舞其处也假照而相攻瞥若浮埃之聚恍如纎霭之蒙驰骛乎蝇头之域游扬乎蚊蜨之宫形不任乎手搏声难为乎耳聪嗟大钧之赋象何至细而兼容忽轻飙之一至兮遂消散而无踪
　　登应州古城赋
　　何兹城之巑岏兮据东北之山巅扼三关之走集兮眺七泽之敞闲乔木矗其雄立兮森防防而刺天翔风薄而长吟兮杂下濑之潺潺南陵渐其岥岮兮尽渺渺之云烟于是太阳启晖六幕澄霁春塍如绣春树如荠啼鸟壊堞苍烟故垒登髙逺望使人心瘁在昔峻宇层构雕枌绣楣车争水度葢逐鸿飞鼓钟沸涌冠劒陆离抗障日之长袂艶从风之罗衣至矣哉合从缔交结俦附党兵略地而鼓行师椎牛而人飨枭鸣牙中梯舞楼上出万死而徇国俟千级而邀赏莫不光沈景灭魂防心折千蒿敛魂野薙凝血同枯骨之死安知誉尧而非桀哉但见灌丛宿楚奥草多露城狐不薰台鹿时聚虎肉委蹊豺牙当路狖失木而号烟鼯投林而啸雨遇发冢之诗书对忧心之禾黍曲池既巳平髙台又以倾古馗时见荒榛自生丛祠隂坞虚闻松声凝睇弭节怅然内惊叹人事之递迁孰长年之能执壮率欢以俱往耄引悲而前集昔牛山之妄叹顾二臣而饮泣使爽鸠之无死非吾君之所及谅迭处而迭去同寒暑之相袭况乃仆本恨人触感増慕岁月逝矣江山非故古既古而尝今今虽今而将古天愁人兮安寄日难挥而催暮命仆夫以効驾吾将遵乎归路
　　夫人城赋【并序】
　　昔晋将朱序守襄阳为苻丕所围序母韩氏自登城按履谓西北角当先受遂领百婢及城外中女丁于其角斜筑二十余丈贼攻西北果溃众固新城丕遂引退谓此为夫人城至今丛祠遗基岿然尚在荆楚岁时乡人祀焉仰其髙风慨然为赋其词曰循汉臯而西望兮何层城之孤峙披南烈之遗堞兮号夫人之故垒谯门堙而中塞兮灌木森其相倚势鬰律以上出兮下坡陁而榛圯渺遗烈而未泯眷髙芬而可纪昔典午之防首属苻丕之虎视区区保乎江汉防一障乎北鄙寇方甚于饿喙地几同于黑子惟韩媪之慈训励朱公之朴忠仗天节以扞敌据江域而临戎于时大羽若日髙旌如虹桀黠同侮跳梁四攻防地中之瓮缶舞楼上之梯冲伊君母之慷慨兮誓丧元而靡悔临大难而克壮兮惧危闉之先溃率民妇而操筑培战陴而相对备之于条之西北兮岂神机之我昧果前陷于厥角率图全于覆篑嗟女子之绵薄兮盖中壸而是修奉苹蘩之常祀承巾栉之余休矧仗节而死难非君子而可求伟此母之挺操亘终古而弗媮赵指括以全宗王勉陵而事刘孟三徙以渐训介俱隠以无尤虽先建于髙躅兮我无忝于前猷若乃宝墉百寻犀兵万旅推毂受命建牙作辅气可盖世威能拉虎或防利而忘义或饱飞而背主输天险于敌国忍吾君之外侮悲丈夫之窃号曽纬嫠而无取傥死者之可作兮非夫人而谁与咏明徳而不已聊盘桓以延竚
