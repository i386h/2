　　衞女琴操一首
　　思归作【有序】
　　琴操曰衞有贤女邵王闻其贤请聘之未至而王薨太子曰吾闻齐桓公得衞姬而霸今衞女贤欲留之大夫曰不可若贤女必不我听若听必不贤不可取也太子留之果不听拘于深宫思归不得援琴作歌曲终缢而死古有弦无歌今弦亦絶矣因补一章
　　恭承母命兮奉先王晐姓后宫兮备酒浆中路王薨兮妾或未亡遂太子之过兮何有三纲淇水沄沄兮菀乎女桑目冷弦絶兮义也难忘
　　席帽山辞
　　席帽山兮岧亭山之人兮朝鉏暮经日月往来兮云下上俯万物兮尘防防尘防防兮江介黒彼山人兮衣苎白带杜蘅兮咀松柏心周流兮影斯息出无节兮入无时猿鸟胥乐兮樵牧我知学茍修兮行着槁死岩壑兮予焉悲
　　寄所思二首
　　七泽连三湘众水相沿洄青青江蓠草缘生洲渚隈苍梧下白日佳人独徘徊容光与香艳幸好未衰摧采采盈襜裳寄子楚王台愿表贞素节路纡难自来

　　虞美人行赠邵倅

　　明皇乐极天宝年拏舟将求海上仙一朝神游跨八极数十万骑屯云烟赤文绿字不复见锦幪黄帕争来献众中一匹目如电膺门沫赭珠流汗龙顔眷爱不肯骑萧萧弄影华清池奚官日给三品料教駣攻驹随所之是时羊车行幸早柳暗花柔忘却晓梦惊鼙鼓遂播迁凝碧空凉遗宿草颠危注意大将贤有马如此无人前至今悲风起毫素濩略欲饮长城泉书生亦自心嗒然
　　危脑帽歌读五代前蜀史有感作

　　天门引
