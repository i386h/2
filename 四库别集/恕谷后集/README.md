　　恕谷后集
　　
　　序
　　宋明之学失古而词章亦变韩昌黎曰上规姚姒浑浑无涯周诰殷盘诘屈聱牙易奇而法诗正而葩宋人则曰文当如菽粟布帛人人可晓遂梵语乡谈羣萃经传易不奇诗不葩文指浑深聱牙者曰不可近而古法几亡矣恕谷李先生躬承颜习斋先生学道之传心性则临深履薄身世则言规行矩而通天察地荷圣苞贤剧旁九达千变亿化巨人智士无以尽其端委也而文即如之天霁日恬雷轰海啸咳唾成风雨鯈忽幻鬼神六花八阵奇正分合鬬乱不乱噫至矣大兴王昆绳曰恕谷之注经超轶汉宋连篇词组皆古文也河南李主事汝懋曰吾徧阅闻人集钱牧斋吴梅村犹是宋明遗习注苕文弱侯朝宗亦涉摩拟方灵皋练或伤气王昆绳主奇变而乃有唐陈若夫渊源圣经旁罗百氏雄洁奥化不名一家其恕谷后集乎知言哉镐从游久颇得闻之乃检录传书序记碑版诸着列为十卷恕谷者先生自名其里也后集者自康熙癸未以前仿欧苏诸大家先生俱置之而惟存其后焉者也雍正四年丙午正阳月吉旦樊舆门人阎镐谨识

　　恕谷后集卷一    蠡县李塨着
　　送黄宗夏南归为其尊翁六十寿序
　　黄子宗夏歙人居于吴游京师闻予友王昆绳称予学因与予交予之学盖得诸颜习斋先生乃举先生之学相示宗夏慨然曰人不作圣非人矣于是悉铲后学浮文求礼乐伦物之实日有所习时有所勘仿予立日谱以自考而其学大进予尝以为汉唐以上气运盛于北其笃生喆人以荷斯道之统者尧舜禹汤文武周公孔子皆行礼奏乐光华宇宙立道垂范以为民极下逮汉唐寖微而董仲舒贾谊郑康成韩愈诸人犹传述礼乐制度遗籍无异说宋明而下天地气移而南张程后有朱晦庵陆象山王阳明各树门帜著书立说鼓动一世于圣经外益以无极主静致良知等名六七百年相从而靡不曰宗朱则曰宗王徒与日盛泛滥于天下且夫古之所为道礼乐文物体诸身而措诸世为天地建实功为民物树实业岂徒讲之口笔之书玩弄心性含咀章句轻礼乐名物使二氏之空幻俗学之浮靡窜入其中人材日萎气运日消虽撊然自附于古圣贤而一如秦鼎之暗移而不觉于戏此天运圣道古今升降出入之大关也癸未宗夏将归为其桩堂复庵翁称六十之觞求言于予予告之曰天地之道极则必返实之极必趋于虚虚之极必归于实当其实之盛而将衰也江淮迤北圣贤接踵而老聃列御寇之流已潜毓其间为空虚之祖今之虚学可谓盛矣盛极将衰则转而返之实者其人不必在北或即在南今子之力学如此使归而振作于南共明斯道以相勉有成将天运圣道自此返虚而实而尧舜孔孟之学复行于天下庶有望乎且吾闻复庵翁仗义好学急友朋不避患难尝自屈其年从刘继庄先生游固非世之尚虚学者所可比宗夏渊源有自从此立德立功立言以不朽其身者寿其亲道孰大于是哉
　　    送都宪石门吴公请假归里序
　　康熙四十有五年都御史石门吴公以疾乞假旋里先公卿引疾者率不许至是知公贫且诚也许之李子五月于草莽闻北入京拜别  国初定鼎大臣尚已二三十年来首推魏敏果公象枢敏果学宋儒自居板庸无赫赫技绩顾以为首也何哉窃闻其乐善下士海内儒流孙征君李中孚张仲诚等虽高隐不出皆  上闻数十年庙堂方面其落落如晨星以搘拄国是澍濡苍黎者如汤荆岘郭华野许酉山于清端陆稼书邵子昆皆敏果所拔茅茹而汇及之者也数十年气运谓非其所斡旋不得矣孔子称及门曰吾得由恶言不入于耳绦也能仁商也不险而独于颜渊曰自吾得回而门人日益亲是七十子三千人皆渊包贯之也为王佐可矣故问邦而孔子以四代礼乐付之萧何邓禹李善长技能不必踰羣英而以荐贤调和诸将遂为汉明元勋盖自古大臣咸如此石门吴公则塨所亲炙者位跻九列无负郭百亩之田尝扈  驾南巡白近侍过几离祸九江谋移关塞外夷人请隟地树公皆力驳之风裁侃侃然而独折节下天下士胖躯伟干见一材一艺輙伛偻笑语接之惟恐后闻某有道术必识其里氏不以学与能穷人寻尺兼茹而待之复称其量人人谓厚已也丁丑年因湖州宋豫葊晤塨见阅史郄视曰治平正鹄也系跋钞存庚辰延予榻其府语其羣从以五礼律吕九章诸学辛巳复下榻予病公日退朝亲检理食药月余乃起偕徐果亭少宰为予梓大学辨业是时冉永光窦敏修二太史尚在馆万季埜王昆绳胡朏明亦寓都门公时聚予五六人论学指曰乾坤赖此柱础也尝乙夜为塨谈寰区政事予因历及古今升降民物安危学术明昧之所以然以及太极河图洛书经史之真伪屯田水利天文地理兵农礼乐之措置公为之踌躇抵掌而起独是敏果值勘定三藩后众政方需人以故荐剡多起用今中外恬熙  朝无轶事公虽仰屋太息慨然欲见一奇而腹果进食无宁胎息而公亦积劳引疾以去我瞻四方而回望魏阙得毋有拳然弗释者乎前辛巳冬宋豫葊瞩予曰吴公吾乡贤者也龙亢返潜讵善始而不善终为先生其辞焉公闻之称是今兹归也可谓进退以道矣
　　    漆天集序
　　东乡乐块然自名其集曰漆天示予览之如夜深风雨怒鬼搏人猿啼鹤唳毛发皆竖漆天风景一何奇也虽然日月履天而光照山河罗地而秀发人物宾正而衣冠礼乐灿列终古常见之弗奇也一思之何以有此奇抑至矣然则不漆之天块然游之又何似耶块然曰吾更有说吾心天也漆者弢之也嗟乎进乎技矣特一以为元牝一以为闇然块然将何从
　　刘氏家谱序
　　刘子来献丐其太翁墓表且出其所著家谱曰此小子祇承父命者也愿弁一言周礼太宰以九两系邦国五曰宗以族得民谓天子诸侯分殊族人不得以其戚戚君乃以君弟称别子者为大宗以收族如周武王封周公于鲁为大宗而卫蔡曹滕诸国皆宗之滕文公公山不狃称鲁为宗国是也鲁庄公立母弟季友为大宗当时称季氏为宗卿是也因之宗立而族收如三桓有季孙孟孙叔孙三族季皆得收之郑之七穆共穆氏而罕氏驷氏国氏良氏游氏丰氏印氏七族子罕皆得收之自封建废而爵如诸侯大夫不世传族人得以戚次子不必贵而大宗不可立适长或愚贱众庶或贵智而宋儒所谓长支主族者亦不可行故朝制亦因时而变设宗人府以司王族之昏丧服爵择贤尸之不别别子庶子然则绅衿者流能稽古立德拳拳敬先收族其贤可为宗而族亦愿宗之是即宗子也如刘子是已刘子之来介可仁言执礼恭言修谱树碑建家祠复谋立公祠祀鼻祖以下诸族先世匔匔恳恳溢于麋宇抵夜分无片辞违孝友仁厚者嗟乎古所谓君子长者非耶平居尝叹南人好虚大家谱追遡瓜瓞牵蔓昔贤虽假冒不计也而北人又过弇陋先世显绩卓行不四五传遂恍惚不复记忆康熙间修明史檄郡县上前朝名臣贤士遗迹郡县挂壁不力行而故家子弟亦闻其先世具功德而茫无以应或惮烦难而止嗟乎不几以苗裔而视其先如路人任其烟清草萎渺无怜念哉予观刘子家谱三世祖昶领乡荐世济其德至高祖鉴以进士为名臣考冲善能持家棅临卒谆嘱其子以继述而刘子承之是岂其祖父之贤而刘子自不能泯耶抑有刘子之贤而祖父乃愈彰耶刘子问礼甚悉携予冠昏祭士相见诸书去钞阅吾知其事四亲构公祠订祭籑仪置助族田立族约凡可以嬗先教睦而登之谱者其未有艾也已
