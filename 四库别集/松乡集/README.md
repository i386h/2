　　提要
　　【臣】等谨案松乡集十巻元任士林撰士林字叔实号松乡奉化人以郝天挺荐授安定书院山长是集所録碑志居多大抵刻意摹韩愈而其力不足以敌愈故句格往往拗涩乃流为刘蜕孙樵之体又间襍偶句为例不纯其自然道士传正一先生传夀光先生传诸篇袭毛頴传而为之亦颇嫌窠臼然南宋季年文章凋敝道学一派以冗沓为详明江湖一派以纎佻为雅隽先民旧法几于荡析无遗士林承极壊之后毅然欲追步于唐人虽明而未融要亦有振衰起废之功所宜过而存之者也赵孟頫尝见其兰芎山寺碑文深相倾挹后士林卒孟頫为志其墓杜夲亦称其谢翺传胡烈妇传能使秉彛好徳之心千载着明固非曲相假借矣乾隆四十六年十月恭校上

　　徽州路重修学记
　　皇帝临御之初诏天下崇庙学徽为郡介在江南山川阻修实维文公其生也乡其服也采式闾而邹鲁存家至而渊骞集庠序之政宜饬具举然而栋宇阙漏祀肄弗共涂轨侵墉臧充宇有司祗旷弗构固壅徳音而贻愧先哲也大德七年夏四月郡博士束从周始舍菜告至顾瞻惕然惧弛祀肄石防版入八百为储士曰不足于养矧能余吏曰不足于需矧能圗从周曰撙削有倡人将来趋于是择时日庀工佣斩技干运陶甓王宫正室葺而新之丹刻有重席仍几衮冕以严公配东坐哲祀攸序下逮羣弟诸儒佩舄簋簠笾豆考礼合宜门庑防卫讲肄之位藏书之室景先之祠以至廪积出内之共鼎镬炊之具罔不饬备凿池思乐被以芹藻植莲其中面池西南师燕有阼峙左屏阜夹右练溪是祠文公乃有别筑于是庙学之制为始崇矣乃相与言曰庙学有政有举无弛束博士之功也博士曰政以郡举教由师作侯郝公之力也夫从周恶贤乃状其事求文于余余惟圣人之教昭在庠序夫子之道着在人心不以言语文字而独存不以礼乐刑政而偏举忠恕之道天地流行修齐之功国家终始是故言政则教举言教则政立夫子之道在是矣文公之学在是矣可不务乎于是乎记
　　奉化州新修学记
　　奉化唐开元中析鄞所置县也今陞为州学宫政化之本是宜广多士而新之有司固未暇也大德三年秋七月肃政亷访副使髙公伯元始易今扁明年夏六月佥事王公焕行郡诸生相与谒事毕坐彛训堂上举凡学之事废宜兴圮宜修唯所画知州事王侯珪唯而退明日学正赵与权録赵进德以佥事公之命请于王侯珪侯曰诺环宫之墙百堵皆作大二百有竒广仞之半髙如广之度而三之雄固杰立覆饰如飞殿堂门庑之壮栋桷之挠折者易治之瓦葢砖级之陊剥者丹白之漫漶不鲜者明饬咸理自夫子像以至四公十哲羣弟诸儒冠冕之饰衣裳之采与夫牺象簠簋爼豆之制严好洁新无遗缺自宋寳元宣和绍兴庆元以来且创且修至于今而始备允亦壮矣于是王侯进诸生而言曰若等知优防歌之所自乎朝廷既右文治尊若道复若家选若人之秀者而官之亦隆且至矣然他日之仕者而泯老者而没壮者而衰故学无常师而文无定业为勉励者不既劳乎吾闻宋庆厯时正议楼公郁一出为乡里师而人輙化之不惟环佩簮笏之盛遂至比屋而舍乡之泽天未忍绝之也古道如新有正议公者不少而江海客食之士贵耶诸生悚而屏遂命之为记
　　新城县重修学记
　　自呉孙氏有地江东谿山小邑皆礼义教化之邦故学官之制盛于中州新城县介桐庐富春间故东安郡况旧为辅邑乎今天下一家夫子之道益大章明宜庙学之设日完且葺弗弛也大德七年今福寜州同知魏侯荣来为尔邑始谒殿既坐进德堂上揖诸生而问故于是钓台山长何庚孙直学徐梦高前曰学宫改筑于邑令陆公演増拓于耿公乗镏公景修今有年矣圮宜修废宜举惟令君实仪图之侯曰诺明日退自公庭致前漕贡进士何谆偕诸生相与谋曰端孙防钱谷曰德麟曰铉度材植曰景星曰应辰庀工役教谕蒋昻奉唯谨乃建直舍凡斋庐庖廪之序始三月讫工夏五月以成时长官若贰数以故挠之而魏侯志弗夺益广堵制遂新王宫栋宇必崇庭楹必度级砖觚尾方宻以丽秋七月既成丹垩辉映衮冕以严八月上丁舍奠升降有容牲币有数既赐胙诸生济济升堂各以序进谢曰维学有政维教有本侯亷以自治仁以淑士其知本欤侯曰匪先孰承匪后孰开尔邑俗淳而士美教洽而政行今燕南河北道亷访副使阿尔思兰哈雅今江南行台监察御史刘公弼旧莅尔邑遗政犹在阿蘓氏治识大体刘公笃厚如之予日兢兢弗承今庙学修举诸君之力而前邑宰之所遗而荣何有焉诸生愿记之而未能也越五年方德麟何庚孙始克请记于余余曰新城有邑在万山中固人物所在而许逺死节睢阳杜棱力拒行宻罗隠耻臣朱梁虽盛衰离合之势天且弗违而志节忠义之气人不得而泯也非礼义教化之所由积乎而况前有邑宰陆耿后有阿　氏刘公其善政其善教耳目濡染固深矣宜乎诸君乐魏侯之令以迄有成也然而修齐治平之学本末精粗之道惟穷通不贰渐积有原诸君行且仕矣具告而子弟饬而学志而道以趋而政教以崇忠厚之风斯其庸有怠乎诗云有斐君子终不可谖兮后之布政教者宜有考云
　　重建文公书院记
　　尚论先王之制党庠术序逺矣唐诏天下置郡县学厥后四书院之名特闻宋庆厯熈寜学校寖盛处士之庐遂废逮徽国朱文公始复白鹿洞之旧祀孔子如开元礼寻创竹林精舍升曽子子思以配孔子以周程张邵司马李延平七先生从祀于是即诸儒讲道之地以立书院者益众文公既没凡所居之乡所任之邦莫不师尊之以求讲其学故书院为尤盛其徒不事月书季考所习皆穷理尽性之蕴其官在郡博士之上其廪官给之田或好义而多赀者请自给世祖皇帝混一区宇郡县学益崇且侈愿以力创书院者有司弗夺其志部使者加察详焉行省设官以主之其选视学正録皇上嗣大厯服播告中外勉励日加明年置各省提举以敦教事永嘉陈君友龙以朝廷首选实来江淛始至起士林于家俾复文公书院于奉化之阳士林载拜承命奉公牒以白诸州州刺史而下翕然曰请如牒昔文公提举浙东常平日循行台温郡泊舟奉化之龙津长吏率诸生请讲书于学宫景定初橘洲姚公希得建海阃敬斋谢公昌元方坐幕府出筹画前进士李君防舒君泌童君防该请立书院于津之左聚徒读书其下榜曰龙津书院寻迁庙学东偏至元十八年改扁文公书院山长李之皓王镃主之既亦废弛先是前朝请大夫赵公崇焘有志改筑卒不就檄下日其犹子必摹图寳鹿山以进其壻前太学博士陈先生着规示之于是州达噜噶齐察罕知州事李侯炳与其贰星侯粲李侯居安议允协其属臧君涓郑君元均賛甚力以工给其役以吏董其事木石之费一不以侵官士又明年祠宇成奉遗像行舍菜事凡受学士林而志于立者咸愿为之徒乡人士亦乐以义廪三之一为之赡州若府状其实于所部于省斯其成矣噫嘻夫子之道散在六经纲领在四书千载之下説益支道益逺子朱子者出诸儒之论始定今天下一家学者无科举之累取朱子之书而读之君臣父子之纲身心家国之目体用兼该本末一致其不为世道深系乎余不得为朱子徒也余私淑诸人也今而后获与二三子周旋矩范之下讵可不知其所自邪或曰州刺史之功刺史不有曰是惟陈君之力陈君力足以任斯文之寄而不屈于贰道足以起学者之慕而不间于欲故创置书院凡九所复升润宜兴学之田万八千畆皆事之著者予不得概书之云
　　绍兴路学重修讲堂记
　　大德九年绍兴郡学重修讲堂成庀是役者山隂赵珆费凢出所捐之田之入初赵氏捐田五百五十畆助郡邑学官廪入且以周里人二百畆郡庠飬二百畆里防嫁百畆邑庠养五十畆里公役郡侯义焉目是堂之敝将压也与博士于天瑞王闵全议曰庀工度材惟赵能费虑所捐为廪养者固不浮且速集赵闻命曰视惟谨凡梁木之朽弗任者柱蠧弗支者瓦葢砖级之陊坏者易新之规不弛前植不虞后郡侯材焉曰是不可不记夫圣人之道在万世言在六经汉诸儒坐重席诸生摄齐升左右屏虚而徃实而归固将为授受地乎今天下学宫一命士坐堂中席鼓五七击部使者郡二千石命诸生东西班坐耳入心着果汉儒授受事乎经世之士明道之儒本非二致也自明道之言支以离而经世之法简以易故有事宦学者徃徃陋所闻以自髙若不相入也圣人之道六经之言所以为修齐治平法者果何事而漠然若此乎去其支且离以示其简且易则精粗兼举本末一致不在所讲乎此堂之不可不修也郡侯贤而知人任事者称义皆宜书侯刘姓名克昌汴人
　　杭州路重建总管府记
　　杭为郡自五代迄今不受兵革之患故生齿日繁庐井螘附城内外居者无虑数十万家旧以两县置城西北隅以聴城以外之治四録事司分置城四隅以聴城以内之治然后受命于郡府郡府拥行省之垣聴外铨者列四道聨外事者郡有司传宣之使日来贡享之夷日接五人聚之地转烦剧郡府方疲于奔命两县四隅之治漫不得梳理固其宜也今大参梁公为杭之明年始上图省府乞以郡治之隙地翼近两县直视四隅使皆在大阃之内将以考成治焉既而郡侯廉公希哲沃呼公好古实来识画规度视梁公为有合郡民吏翕然响从锯斧之工版筑之子执器备用来防庭下始聴事中黄堂幕府旁列吏舍胪分防卫之门休眺之楼栋宇一新左翼仁和右翼钱唐暨四録事中唐隅列东西门入坐各南向缭以周墙揭以表树视聴耸仄有壮且丽经始于大德二年讫工八年六月有弗集者郡侯扎呼岱尔伊喇呼图克与郡贰而下实完成之于是大夫人士佥曰治有剧易操其简则易从聴无比逺笃其近则逺举杭今为既剧且烦郡长贰坐堂上属城之吏日趋于庭一公交孚万虑明白缓急传呼不出晏坐示仪表景速于唯俞顾不在一转移间乎然事大规宏更三政始有成皆宜书
