　　提要
　　【臣】等谨案止堂集宋彭年撰年字子夀清江人乾道五年进士歴官焕章阁待制知江陵府迁湖北安抚使坐事落职寻复官以寳谟阁待制致仕卒諡忠肃事迹具宋史本传年官右史时面折廷诤劘切人主有古直臣之风集中所存奏疏劄子尚五十余篇敷陈明确多闗于国家大计其论光宗不朝重华宫疏凡三四上至于伏地叩额血渍甃甓光宗亦为之感动又尝事宁宗于藩邸有旧学之恩即位后数进谠言拳拳恳到因风雷示变极陈小人之窃权及朱子以论韩侂冑被绌年又上疏请与同斥今诸疏并在集中其严气正性凛然犹可想见史称其学识正大议论简直善恶是非辨析甚严故生平虽不以文章名而恳恻之忱与刚劲之气浩然直达语不求工而自工固非鞶帨为文者所得絜其长短也宋史艺文志载其集四十七卷世久失传今从永乐大典所载益以歴代名臣奏议所録共得文二百二十三首诗二百二十二首依类编次厘为十八卷虽较原目所存只十之四而其一生建白之大者已略具于是矣至若祝语诸篇本非文章正轨今刋本概从删削焉乾隆四十九年十月恭校上

　　乞留侍御史刘光祖疏【绍熙元年四月上时为太学博士】
　　臣等备数学官素餐无补事有职守不敢越思惟念国家开设学校所以涵养天下公议之原而臣等谨以课试文艺苟求塞责诚不足仰称明旨日夕忧惧不知所为适有一事偶闗士气之消长臣等傥顾出位小嫌缄黙自爱揆之初心实所不忍臣等伏见殿中侍御史刘光祖近除太府少卿士论纷纷皆谓因论吴端除授之故事之始末疎逺小臣不能尽知或是或非皆不敢决然臣等窃以为不可者偶留近幸而遽迁台臣其于国体所系甚大臣等尝窃观祖宗借重台谏之意揆之古昔所未见闻刘安世劾胡宗愈至二十余章而不以为渎贵重大臣如文彦博因唐介一言为之亟罢而不以为横盖所以优假直臣奠安国势其为计虑至深逺也今光祖论端其言必不如安世之切陛下偶然进端亦未必宠任如彦博也而祖宗处之如彼陛下处之如此比而观之孰得孰失当不待人言而后喻矣陛下嗣登大宝始初清明隆寛尽下迈迹三五岂容有此瑕玷传播四方宁免疑惑况天下士气方患不振今日之事朝路借借皆以为不可而告陛下者曽几何人平居则仰屋窃叹遇事则缄黙不言陛下既未有以兴起斯敝而敢言者复不得伸臣等深恐言者自此不敢言而不言者亦以言为戒甚非国家之福也臣等私忧犹不止此陛下所以迁端者止念其服役之久而言者适然攻之太亟陛下亦适然拒之太坚小人不知必谓其徒得君如此其专据势如此其固附声托影寖成骄横则他日将有不胜忧者当是之时臣等恐陛下尤费区处不止如今日也陛下天资恢廓以虚受人于天下事初何容心其始本无亲昵近幸之意而遽进端其终亦无厌薄台谏之意而遽疎光祖然疑似之迹未能昭明羣情共忧所当深虑昔韩维因论台官进退有曰自古圣王优待谏诤之臣虽甚狂直必加含忍其势非不能黜也以为黜此一人则伤众多之心遂此一失则败天下之事故不为也老臣忧国其言恳切深中事情臣等欲望陛下平心定气更加审处少抑近幸之恩宠以伸台谏之公论仍留光祖俾复其位使天下晓然知陛下圣德光明本无偏倚自此无复顾虑各罄心腹以不负陛下任使实宗庙无疆之福臣等冒犯宸严罪当万死不胜俯伏俟命之至
　　乞寝罢版行时文疏【绍熙元年四月】
　　右臣待罪太学博士昨准国子监开备准尚书省劄子臣寮上言士子不阅经史子集之文而专意于时文不阅旧来典实之文而专意近日虚浮之文朝廷方以程试取士欲其不习时文固不可得如旧来之典实足矣今欲一洗其敝当自成均始乞令监学官公共精择旧来时文谨严而有法度精粹而有实学者经义词赋论策各若干篇许之版行以为程式奉圣旨依劄付本监本监开臣及监学官详臣寮劄子内事理将新旧时文分官精择去讫臣窃详臣寮申请乃是父兄教子弟之法而非太学教天下士之法虽曰救文之敝而臣恐滋文之敝实自此始不可不虑也请为陛下画一陈之臣闻三代取士皆本德行隋唐而降始尚文词至于本朝循而不改夫以德行取士犹开目取物既能识其大小又可辨其美恶以文词取士犹闭目取物大小美恶无所不收左采右获庶几一中所以忠厚浮薄色色有之盖为是也先正有识之臣率以为病故司马光谓取士之道当以德行为先其次经术其次政事其次艺能近世以来专尚文词文词乃艺能之一耳未足以尽天下士也邹浩尝论科举谓上自辅弼之大臣下至州县之小吏皆佐陛下行道者也非欲其若司马相如枚乘沈佺期宋之问之属以文词供奉而已而乃取之不以道用之不以实谁与成天下之务哉然则取士以文已愧于古况教以时文乎此不可一也自古文士多出东南东南之士不患乏词藻惟患不笃实今居东南之地用东南之人犹病其不文可不深究其所以然哉臣寮所谓不阅经史子集之文而专意于时文是也夫旧日典实之文乃根本乎经史子集今日虚浮之文乃自时文坏之今不教之研穷今古依据义理以涵养根本而复教以时文是恶其湿而使居下流此不可二也夫谓之时文政以与时髙下初无定制也前或以为是后或以为非今或出于此后或出于彼止随一时之去取以为能否今求其义理精深文字浑厚者能有几何纵得一二十篇其格又多不与今同舍之则失简别之本意存之则破文字之近体此不可三也或者必曰朝廷方以时文取士今以时文为不足学则是当并废之而后可乌足为通论臣谓不然天下之材不可一律取也朝廷设科止为中材地尔欲使髙者可使俯而就卑者可使企而及岂谓天下士尽可以科举之文得之唐韩愈谓来京师见有举进士者人多贵之就求其术或出礼部所试诗赋策等以相示以为可无学而能由此观之有司所取之文天下固有以为不足学者也天下虽不能皆韩愈然岂可轻天下无韩愈哉朝廷取士固欲其如愈不欲其不如愈也今乃一切以时文教之编类成帙公私刋行凡应进士者家家售之父诏其子兄语其弟以为朝廷取士不过如此朝读夕思惟此是习经史子集将覆酱瓿臣恐天下文章益见凋敝必又甚于今日也先朝诸臣皆欲尚德行以救文敝而今乃欲择时文以教举子其于本末毋乃甚舛欤此臣所以知其不可也臣闻先朝尝欲更定四场去留指挥时邹浩为颍昌府教授乃沿工执艺事以谏之义抗疏论列今臣为学官职分所在明知不可而以成命既颁忍口不言诡随苟从是不忠也实孤臣志是以不避斧锧之诛肆其狂愚欲望圣慈下臣此章与臣寮之言参订得失如臣言可采将近日精择时文指挥特赐寝罢止令学官于公私试文字精加攷校以义理明正者为上学问淹博者次之文采华赡者为下苟不入格虽是中选不许刋行去取既明趣向自正举子之文将不求典实而自典实矣干冒宸严臣下情无任俯伏俟命之至谨录奏闻伏候敕旨
