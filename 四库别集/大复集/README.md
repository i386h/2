　　【臣】等谨案大复集三十八巻明何景明撰景明字仲黙信阳人治壬戌进士官至陜西提学副使事迹具明史文苑传是集凡赋三巻诗二十六巻文九巻传志行状之属附録于末正嘉之间景明与李梦阳俱倡为复古之学天下翕然从之文体一变然二人天分各殊取径稍异所尚亦复有不同故集中与梦阳论诗诸书反覆诘难龂龂然两不相下自薛蕙有俊逸终怜何大复粗豪不解李空同之句论者往往左何而右李平心而论摹拟蹊径二人之所短畧同至梦阳雄迈之气与景明谐雅之音亦各有所长正不妨离之双美也景明于七言古体深崇四杰转韵之格见所作明月篇叙中王士祯论诗絶句有曰接迹风人明月篇何郎妙悟本从天王杨卢骆当时体莫逐刀圭误后贤乃颇不以景明为然其实七言肇自汉氏率乏长篇魏文帝燕歌行始自为音节鲍照行路难始别成变调继而作者实不多逄至永明以还蝉聫换韵宛转抑扬规模始就故初唐以至长庆多从其格即杜甫诸歌行鱼龙百变不可端倪而洗兵马髙都防骢马行等篇亦不废此一体士祯所论以防浮艶涂饰之则可必以景明之论足误后人则不免于惩羮而吹虀也乾隆四十三年三月恭校上

　　大复集序
　　古今论曰文以代变非也要之存乎人焉耳矣唐虞三代礼乐敷教诗书训义防温雅文质彬彬体之则德植达之则政修寔斯文之防极也汉魏而下较诸古昔殊矣厥辞繁厥道寡厥致辩厥防近日趍于变然尔若所谓代变也及考夫董贾扬马李杜韩栁诸贤各运机衡以追往训当世文轨靡得而拘今综八子视之殆自致羽翮凌驾文囿者矣非存乎其人何哉吾友大复何子蚤岁能文下笔辄数千言不休十有六龄即举于乡至京师学士大夫慕其神藻无不降重造谒以求识靣及登第与北郡李献吉为文社交稽述往古式昭逺模摈弃积俗肇开贤蕴一时修辞之士翕然宗之称曰李何云今详其文侵谟匹雅欱骚俪选遐追周汉俛视六朝温醇典雅丰容色泽靡不备举标而彰之将并日月朗于太虚岂非髙迈独立出世离类不爲时拘者耶古称雄视百代斯文信矣浚川子往与孟望之论大复乃间世精气所钟造化灵秘之久一旦勃勃附化何所不神耶故生而即文即知事即贤即老成即有古人风节非神解之故乃尔能之耶夫人坟籍孰不探道防孰不诠文辞孰不脩风调孰不循德履孰不习终格于不类者天畀之解未神尔正德末余督学四川大复督学闗中一日走书曰孔孟藐矣学渐支离涂岐矣后生汶汶以塞迩得论説数十欲与君约境上之防期越月之讲然竟不果来今集亦不见有此论岂终未成耶抑遗之耶悲哉悲哉其未之究竟者若此天实爲之矣渔石唐子曰以何子之才而得夀考廓而大之纶而不息殆潏乎河汉而无涯也嗟嗟岂不信然哉大复集辞赋三巻四言古诗一巻乐府二巻使集二巻家集五巻京集七巻秦集一巻内篇一巻外篇四巻通二十六巻别论若干巻刻在潞州斯集枣行久矣尚未托有序列其甥王君朝良泣谓余曰公吾舅氏之知已也兹言非公其谁嗟乎造化夺大复去矣斯文出于元精应与元气并存虽造化亦不得而夺之古谓华质长潜而灵迹不灭者其以是夫其以是夫嘉靖十六年六月初伏日仪封王廷相子衡撰













　　大复集序
　　何子仲黙生而敏悟童子时即执简赋诗邈焉寡俦长益肆力焉探防抉竒增华潄润轧晋魏而上之惜乎三十九而卒乃已焉矣康子德涵张子时济皆何子之友也何子既没张子收其逸稿访康子于浒西之野而共扬防之凡得若干首洋洋纚纚一唱三叹而有遗音焉时闗中执事君子咸兴嘉乐之怀乃檄而传之得非以古昔有诗之教邪夫词不雅与雅而阙于行者皆弗得传即传弗逺何子词则雅矣其行履复皭然而有令名则传之也固宜呜呼以何子之才而得夀考廓而大之纶而不息殆潏乎河汉而无涯涘也天顾阨而夭之而俾弗竟其緖乎予闻天之大化理以经之数以纬之以理御数则履其常数角而胜焉变斯出矣是故牛之行羸然而疾也囬之仁陨然而夭也诚格于数即古之大贤犹俛而从之何子其若之何抑诸断鹄之剑照乘之珠天下之至寳也假于涂之人则主人匍匐往索之而不俟越宿焉是故李贺夜梦神人请赋白玉楼觉即属纩殆又一道已乎何子断鹄之劔照乘之珠也则天恒靳其光矣而能不收之邪是亦不可不谓之数也嘉靖三年三月既望兰谿唐龙撰












　　大复集序
　　明兴百六十年其文遐哉盛矣然作者接辙于域中其敦致古昔逖称先王人人能矣而义意繁猥溢于往训摹仿摽敓逺于事实予犹以为过云治时上兴化重文士大夫翕然从之文视昔盛焉是时仲黙为中书舍人而予以次第为翰林修撰一时能文之士凡予所交与者不可胜计予顾独以仲黙爲难能仲黙时方弱冠也予每见仲黙之作叹曰嗟乎文其在兹乎夫叙述以明事要之在实论辨以稽理要之在明文辞以达是二者要之在近厥指意凡仲黙之作三者备焉故予欣慕叹息于一时交逰中独以仲黙为难非私之也武宗皇帝之三年予以忧罢修撰归十有三年仲黙以提学来闗中数能以公事过予出其所论著凡数万言皆当实不修可以上薄屈宋董贾有相如子长之风顾世无知之者独曰何子有诗人之遗意厥觌浅矣仲黙岂独工于诗者故仲黙名徧天下然实无知仲黙者此也汉魏以降讫于开成世代既移音节斯异修辞之士能无恶厥趋斯已优矣复能引而上之不沉于流俗由载籍所覩若仲黙者岂非鲜哉十六年秋仲黙既卒又三年予次苐其文为若干巻首赋次诗次文皆随体区裁因制列巻题曰何仲黙集録存家笥以待后来读其文思考其世可以知予言所指矣仲黙才髙而意宏德纯而识远荣辱毁誉一无所动于中予别有传记兹不载方予定次仲黙集时値张子时济过予所见与予甚同因更与定之如此集云嘉靖三年二月甲子武功友生康海序






　　何大复集序
　　敬皇帝朝化休而融昌异时诸先生业文章显甚重矣学士大夫固欣然称说耳相慕也而独北地李子以非心所好谢去之亡何而又有信阳何子者何子虽稍晩出其材质敏秀瑰丽各以长相当然而李子得何子为益雄也鄙人之言何知仁义向利则德是二君子抉草莽倡防言非有父兄师友之素而夺天下已向之利而自为德于乎难哉去其始可一甲子诗而亡举大厯下者文亡举东京下者即谁力也然二君子之徒不能长縁其师所繇得毛举论难之语以好为胜而它工易者恶津筏者往往左袒何子而龁李子则又似非何子意也夫百羽集于词林而二子雄飞或撇捩逆羊角而横举或顺飚而肆翔其九万里同也李源风何源雅风故长变以明志耳夫覩其沈深莽宕激卭鼓壮喑鸣憯凄忽正而竒正若岳厉竒若海则李子哉是固少孙要之其縁情即象触物比类靡所不遂璧坐玑驰文霞沦漪緖颷摇曵春华徐发骤而如浅复而弥深疑无能隃何子而上者何子为文刻工左史韩非刘向家言大抵于诗鴈行云而闗中康氏乔氏其乡人樊孟氏则盛惜何子志业屈于年未竟世之谈说经纶抵掌事勋者其敖何子以不及如耶令何子不死而称为名公卿已耳所以削涤卑振頽习扶昌运开中兴者何物也功于经纶孰多于是何子之甥袁灿来谓王生若为何子叙其遗言王生曰何子彬彬大家也易言之有亲则可久李得助而久何子之功李子伟矣夫二子之功天下则伟矣夫嘉靖戊午后学吴郡王世贞撰

　　渡泸赋【以下使集】
　　晨瞻崇邱郁乎相袤扄以水峡隐以大洲沙莽寒日江深夕流葢将济于泸水榜人告予以理舟防洪波以直度迎廻颷于上游顾中原而缅邈久西域以滞畱感逆旅之长勤怀古人而增忧想夫汉炎既烬蜀都始家区土未辟士马不加深入五溪横制三巴冐险通塞柔迩来遐收羌髳以带甲率庸卢而习戈挞吴权之坚锐摧魏懿之精华今其断岸遗津寂寥水涯苦雾萦石悲风振沙音尘沦絶古今长嗟叹余风兮莫觏幸故址兮重过西望开创之基形势苍苍襟夔府而控荆门峙巫峡而流瞿塘简书零落阵图纵横烟碛下月隂岸积霜风云惨而犹愤鱼鸟畏而将翔功虽殒而誉逺身既没而国亡南瞻祠庙巍兮惝宏松柏防戸丹青閟宫垣罥林蔓堦巻寒蓬亦徒嘻吁父老涕泪英雄而已呜呼当其长啸之后三顾之余气感云龙恩同水鱼念季叶之否运思中兴之远谟伟雄并乎三分耻偏安乎一隅何遂违其懿志竟弗骋其长驱吾哀斯人遭时不淑宜三代以翺翔胡羣奸而驰逐矧功业兮难成且年岁兮易促嗟哉天道髙不可摩得志者寡失意恒多茍道之不行虽孔孟其如何泛泛吾舟载扬其波舣彼清浔陟彼中阿无人可邀聊独行歌歌曰朔风起兮泸水寒扣檝中流兮怀昔贤遇匪其时兮良独艰嗟嗟遇匪其时兮良独艰
