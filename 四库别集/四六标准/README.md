　　钦定四库全书　　　　集部四
　　四六标准　　　　　　别集类三【宋】
　　提要
　　【臣】等谨按四六标准提要四十巻宋李刘撰明孙云翼笺释刘字公甫崇仁人嘉定七年进士歴官宝章阁待制云翼有橘山四六笺注已著录刘平生无他事可述惟以俪语为専门所着有类藁续类藁梅亭四六今皆未见此本乃其门人罗逄吉所编以刘初年馆何异家及在湖南蜀中所作彚为一集题曰标准盖门弟子尊师之词也凡分七十一目共一千九十六首自六代以来笺啓即多骈偶然其时文体皆然非以是别为一格也至宋而嵗时通仕宦迁除吉凶庆吊无一事不用啓无一人不用啓其啓必以四六遂于四六之内别有専门南渡之始古法犹存孙觌汪藻诸人名篇不乏迨刘晚出惟以流丽穏贴为宗无复前人之典重沿波不返遂变为类书之外编公牍之副本而冗滥极矣然刘之所作颇为隶事亲切措词明畅在彼法之中犹为寸有所长故旧本流传至今犹在录而存之见文章之中有此一体为别派别派之中有此一人为名家亦足以观风会之升降也至云翼之注芜杂特甚然亦有足备考证者旧本所载亦姑附存焉乾隆四十三年六月恭校上

　　四六标准卷一　　　　　　宋　李刘　撰言时政
　　上史丞相【嘉定丙子○宋史宁宗纪嘉定元年以史弥逺为右丞相丙子为嘉定九年宋史本传弥逺字同叔浩之子宁宗崩拥立理宗拜太师左丞相兼枢密使进封防稽郡王卒追封衞王諡忠献初弥逺既诛韩侂胄相宁宗十有七年迨宁宗崩废济王非宁宗意立理宗又独相九年擅权用事专任憸壬理宗德其立己之功不思社稷大计虽台谏言其奸恶弗恤也弥逺死宠渥犹优其子孙厥后为制碑铭以公忠翊运定策元勳题其首济王不得其死识者羣起而论之而弥逺反用李知孝梁成大等以为鹰犬于是一时君子贬窜斥逐不遗余力云○元李有古杭杂记云温陵吕中作国史要略谓南渡之后一壊于绍兴之桧再壊于开禧之韩三壊于嘉定之史愚亦谓理宗四十年在御一壊于嵩之再壊于大全三壊于似道也相之壊国如此哉】
