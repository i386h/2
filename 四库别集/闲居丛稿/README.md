　　提要
　　【臣】等谨案闲居丛稿二十六卷元蒲道源撰道源字得之号顺斋世居眉州之青神徙居兴元初为郡学正罢归皇庆中徴为国史院编修官进国子博士年六十矣越嵗复引疾去后十年召为陜西儒学提举不就迹其生平恬于仕宦大扺间居之日为多故其子机裒辑遗文题曰间居丛稿凡诗赋八卷杂文乐府十八卷诗文俱平实显易不尚华藻黄溍为之序称国家统一海宇士俗醇美一时鸿生硕儒所为文皆雄深浑厚而无靡丽之习承平滋乆风流未坠皇庆延祐间公以性理之学施于台阁之文譬如良金羙玉不假鍜链雕琢而光耀自不掩云云亦言其文之真朴也葢元大徳以后亦如明宣徳正统以后其文大扺雍容不迫浅显不支虽流弊所滋庸沓在所不免而不谓之盛时则不可顾嗣立元诗选引溍此文谓当时风尚如此可以观世运焉斯言允矣乾隆四十一年十二月恭校上

　　闲居丛稿卷一　　　　　元　蒲道源　撰古赋
　　雪轩赋【并序】
　　延祐四年余与牛君楚臣同官于翰林国史院始获相识君京师人宰相知其贤而有文以白衣辟用授国史院编修官未几求去余私谓从君游者检阅宋文郁曰第语牛先生云今司文柄者徒欲得君之贤以为此职非责以朝夕奔走期防姑少留得俸金以给饘粥可乎文郁以告君曰蒲之意甚厚然吾以早衰不能勉强非不乐此职也但食人之食而不事其事吾不为也遂决辞余闻之深有愧其言暨余承乏国学君使人持雪轩卷求余序引余窃思君之取义于雪其必有深意焉今试以君平昔观之则夫雪之皓然一色皎而不汚者庸非君之洁乎其掩秽覆瑕者庸非君之有容乎其倐然而集忽尔而释与时变化者庸非君之出处乎若夫为防麰之瑞则君之孜孜讲学以善及人者可知矣至如天葩缤纷匪雕匪刻作止疎密飞舞之态而其文章之妙天机所到非有意而为之者似焉宜君之以雪名轩而识者亦以为然也向所谓贤而有文与其翻然而去不以穷达累其心者于兹而益信噫君子乐道人之善而欲播于歌咏者观此有以识其大方云既以为之序又从而赋之
　　牛君楚臣以雪名轩太常陈子悉着于篇而又征文于余也余掇取陈子之所弃者以为言方其防用事白昼将夕严风乍止同云坌集天花散坠弥望如一飘然而入吾之户忽尔而缟吾之室余乃掩关退处蛰坏其居萧萧骚骚鸣吾竹牕之纸皎皎洁洁映吾棐几之书怳虚白之渐生俗虑之絶无当此时也或倚蒲团以听其声或据槁梧以观其色神魂洒然而昭苏肝胆以之而莹彻如生乎鸿蒙之世与浇风之悬隔如独立万物之表脱垢氛而超邈此伯夷采薇于西山求仁而有得而顔子躬四勿之训已私之尽克之时也欤噫微斯人而与言啜茗饮以冲黙
　　四言古诗
　　赠宗石泉相士
　　石以为甃泉居其中尘滓莫染渟滀涵空粤有道人了然双瞳览物无遁与泉则同气色显晦骨相穷通我亦何心尔赋斯容挟术见访奈予已翁水落石出一笑相逢
　　五言古诗
　　送邓明仲之察司书吏
　　邓生乡曲英行业雅修整遇事胆轮囷折言骨鲠匹马走京师载笔上台省椟玉正求沽囊锥初脱颖相望五千里寄声问幽屏去秋鲜鞍马过我情味永谁知离别久复此游从幸一朝忽征诗遽以行期请云承宪司檄欲往亲簿领青云有余地白日无停景满意取功名归来娱定省
　　张教授盆池石山种梅
　　曲江安静退闭门但却扫精心砭□余颇复事幽讨巉嵓一拳石爱惜如至寳种梅于其巅硗确费天巧慇懃十二冬要看花实好傍人笑我痴我意真自保南枝氷的皪今果慰怀抱无劳叹春迟似比蟠桃早吾闻橘中乐不减商山皓况此大观中觌面即蓬岛何须穷弱水垫没死枯槁持酒寿先生一醉洗衰老
　　次王仲礼九日感怀韵
　　佳节成虚掷怅如损百金秋菊夕可飡徒为楚骚吟独有渊明家足以供幽寻渊明家所爱兴同无古今眼中人岂乏然各怀其心所以子期没伯牙应辍琴东篱景依旧白衣奚再临无酒须谋賖涤此尘埃襟谁能起废防庻令朋盍簪
　　九月十五日李子文席上佳节若为酬分韵得节若
　　书生时时愁富家日日节乘此闲暇时不饮何怡悦樽前三四友年少文愈杰新诗落笔成句句皆清絶酒来不须辞要使頳吾颊
　　菊节过六日友人邀小酌我时适无事来意已不却宾主情况熟笑谈应自若欣然具肴核不丰亦不约沿溪共徘徊秋景正萧索留连兴未阑野鸦啼日落
　　逸题
　　駃騠千里心奔驰追电掣一朝羁维中不能前跛鼈士生天地间出语比稷契岂知与世谬举足便差跌当途烂羊封空谷白驹洁胡弗事谄谀苦自完气节曲肱陋巷贫炙手权门热行藏道所系去取今已决倘佯复何为歌咏自怡悦惟知诗语工遑恤生涯拙
　　闲居纪事二首
　　客从长安回叙濶访邻里貌悴于去时因而问所以客悲向我言此行幸脱死长安遭饥荒食尽到糠粃升合贵不赀珠金何足恃凌晨出求籴于于如栉比暮归持空囊菜色皆相似老弱困且羸行随墙壁倚邨墟向昏黑剽掠羣凶起咸云田中麦苗枯无雨水今年如弗收炊爨当易子予闻重叹嗟祸及乃至此八政食为先周书寓微防天下苟有饥稷思若由已梁惠战国时民粟知移徙一夫不获所古人心愧耻如何填沟壑遽忍立而视此邦麤偷生唇亡寒及齿嗷嗷食口众身不亲耒耜今朝听此言徒増惊悸耳
　　古人制昏礼所贵在人伦冠笄年既迈受币方交亲后世虽偷薄十中五犹遵如何至今日彛典俱湮沦男痴已合配女幼皆成姻安知虑夭寿奚暇占吉辰媒妁走市井鼓吹喧城闉百年祗仓卒大事为因循问之何遽然诏云选佳人传闻至太甚意失非本真正道难化俗讹言易惑民于初苟不慎其末弊将臻寄声秉钧者风俗何时淳
　　送齐府判北行
　　北风天欲雪欵枢君话别袖出诸公篇意不容我缺蹇余适丁艰离况转悲咽情深讵可辞政美难悉説君才信更綀民瘼亦究彻正论当风霜始怒终见恱侯邦居末僚用未尽施设颇闻求闲局此行计已决谋此行云善许国奚自洁况今访落初求治正急切肯将栋梁具小用为扂楔秉钧方得人此语宜中节
　　送府史窦彦立迁奉元
　　蔼蔼府中英夫谁可屈指窦君长安人众论共推美气和更敏才志锐方壮齿留胷无芥蔕出语有纲纪一朝过我辞试问其所以庙堂新立法为吏互迁徙简书迫程期信宿不容止门当道途旁所幸还乡里闻之心惘然别绪纷莫理一杯送归鞯一言赠吾子人生重名节不义深可耻青云路虽高欲致由践履去去宜勉旃好音行洗耳
　　次德衡弟韵五首
　　古道不可作转觉浇风长人心所去就但与时炎凉燕石藏十袭连城遗路傍汝贤乃迂濶汝直成狂徉世味鼎一脔人争染指尝伊我澹无虑箪瓢身世忘处汚奚害洁视变皆为常彼哉声利徒末路空皇皇
　　羲和破凛冽可爱固其宜晨兴盥漱罢茅檐与相期晴光乱锦绣口有难言诗温然寓天理令人荡邪思苦恨白昼短忽忽光隂移冬者歳之余为学当及时礼乐三千字他年对丹墀
　　万物长养极归根以成冬四时只如此郁郁南山松犹人内主敬外貌严而恭肯随桃李花娇靓为君容涧壑老不悔工师非愿逢
　　用心欲刚明为学贵专静人之不已知于吾奚所病风月自清闲篇章足歌咏钟鼎与山林从来各天性其肯枉尺寻获罪于前圣蓍无庸决去取予能定
　　古人进道心如水朝东势滔滔昼夜间欲已而不已明德与新民颠沛必于是吁嗟朱子没道统谁能嗣年驰志弗立岂不増惊悸学而徒为人固非分内事区区斗筲才恃彼词锋利
　　简德衡弟戒饮
　　人生有秉彛尤物害之极麯蘖本致和为祸亦云剧夏后防其源遂见疎仪狄周公戒淫湎作诰何谆悉大则国政隳小则家业失甚则遭戮辱否则成痼疾书传之所载厯厯记畴昔如何后来者乃循覆车迹吾弟颇涉猎岂不増夕愓忆余自童幼杯杓素所嫉无何人强勉颇亦开涓滴当其下喉时恶药与沙砾方知酩酊中特赖扶持力苟若轻放肆野马恣奔逸闻汝以此戒令我喜折屐虽能勇于始其末乌可必莫学醉伯伦妇言非所恤
　　饯杜仲正经厯美解东归
　　脱屣兰省掾曵裾莲幕宾胷中自水镜笔底皆阳春面无喜愠色口不臧否人余事及文翰出语清絶尘佳篇屡贻我玉唾光犹新寒斋坐移晷倾冩见情真期遽云代归意浩无垠关山路皑皑霜雪晨留之不肯住握手但酸辛后防知何时别离宜自珍
　　题竹雀图
　　昔闻丹穴雏栖梧而食竹要看五色羽辉映森森玉歳晚不一逢黄雀乃见辱此君初无心汝得暂栖宿
　　寿甫教授征余文漫成二首
　　人生天地间分定不可移得亦胡为喜失亦胡为悲我今两俱置顺适了无疑惟余乐善心蚤夜以孳孳我本空疎人虚名谩惊俗外夸实未充内省恒自恧已然知奈何勤学补不足庶用期寡过慕荣非所欲
　　赠雪梅相者
　　天爵人不脩妄意希富贵纵然貌可取其中复何谓道人慎许可更当穷本原子游裴行俭予乃无间言
　　赠月轩日者
　　月轩挟术防言论非剽窃星防相命书一理皆贯彻为人推祸福脗若合符节岂比浮夸徒枯中但饶舌
　　寄太子宾客王山木
　　公昔解郡绂汉中久盘桓江山发令姿草木有荣观逸兴超物表清辞落毫端而予等燕雀何幸随凤鸾闲居侍函丈议论惊层澜或时举玉趾乘兴看药栏优哉聊卒岁俯仰天地寛胡为召檄至去住处尤难义有不可辞促装对金銮储君得正士学问资甘盘固知神明相获遂寝食安恨无飞仙药使我生羽翰一往见顔色晨昏奉清欢
　　题田氏寿母诗卷
　　啜菽胡可乐列鼎胡可悲盖以亲为心贫富安足移田君伟男子母老鹤发垂立身掾冬官禄养谐心期金巵献寿祉防服为儿嬉慈顔自怡悦堂上春熙熙八秩又四稔康宁殊未衰晚福方川増防当庆期頥
