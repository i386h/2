订刻太史升庵文集序 

今海内绅佩之士研精策府撷英词林莫不仰慕杨升庵先生而有不及同时之叹雄文秀句竞相诵传尤龂龂然无繇毕揽全集为憾也余奉命按□□□□□□□乃□□诸君咸称升庵□□□为表章咋种□猥繁今巳多散落恐久而就湮没矣于是谋之抚台澽滨张公檄藩司求之先生令侄大行益所君抄录若干卷凡先生闳言眇词彻于著述比兴者亦略具是爰属稍加厘订删要归正道而论之自一卷至四十卷为赋序记论书志铭祭文跋赞词传与各体诗皆取之文集而以类编纂者自四十一卷至八十一卷皆训释整齐百家杂语取诸丹铅辑录谭苑醍醐巵言等书而以类编纂者□□□□□□□□文集刻成蔡君以□□□余□□盛举也不可辞乃为序曰文者贯道之器儒先共推为不朽盛事云而韩退之以蕲至古之立言者在养其根而竢其实加其赢而希其光则可谓善于论文者也甞观退之樊绍述志从其家得卷籍曰多哉古未甞有也然不袭蹈前言不烦绳削而合遂赞之以道塞复通云升庵先生探六艺之渊而综九家之指其论譔如燕文贞攻比兴而莫能极张文献穷著述而未克备者既以合之双羙乃其罔罗捃摭陶冶大垆卓然自□□家之□骎骎乎追屈马于先路□载□□之论和平之音淡薄愁思之声要妙而欧阳永叔因曰诗穷而工也先生出平津之邸绍玄成之业当是时德流明庭名冠天下而忠猷渊蕴时形咏叹如庆成宴之诗三爵七襄之语其忧国轸民一篇之中三致意焉逮乎获谴投荒行吟泽畔或时绎为贾生之鵩赋或时着为虞氏之春秋类若发愤之所为作然皆蔼然小雅之怨诽不乱者焉是先生超然有曳縰歌商之风而穷通得丧无入于灵府欤至其□羽陵之蠧简稽酉阳之秘藏笼罩靡前指类剽剥亦其涉猎者广慱时有抵捂而其归则纚纚然淑诡可观连犿无伤也兹既总括裒汇采可论而着于篇先生之于斯文殆韩子所称至于是极而成学治古文者彬彬然忻覩先生之集之全信奇事也此外诸所较刻古文杂录拾遗补艺尚多有之以非先生所著书故集中不载然其序所以刻者之意则具编之集序类中矣后之诵读是集者论其世而求其志庶可几君子之前睹哉庶可几君子之前睹哉赐进士第巡按四川监察御史侍经筵平原可泉宋仕书

升庵文集者集杨升庵先生遗文也先生作为文章其书满家纚纚然承学治古文者咸爱而传矣顾简袠错陈统汇靡纪志古者每苦遍观之难尤愳散逸之易也余奉命抚蜀谋之巡詧可泉宋公以文献宜为表章议克恊矣爰檄藩司往从悉取其书得之其家大行以义君藏辑者有先生文集若干卷赋序记论书志碑赞词传与各体诗凡厥抒怀赠述者具焉因就而抡次之复得丹铅辑录谭苑醍醐秇林伐山巵言各种杂着凡厥探赜索隐者具焉因删而汇编之刊削肤引勒成一家之言总之为八十一卷如迁史所称择其言尤雅者着乎篇刻成而卒业焉综之靡所阙而秩之靡所绲矣庶几乎获睹升庵之集之全而无复有荐绅先生之所难言者乎盖余读迁史儒林传而知齐鲁之闲于文学圣人之遗化也蜀自文翁之教行人士彬彬以学显于当世比齐鲁云当其时被服文翁之教者相如之词赋能使天子读之飘飘有凌云之思嗣是若杨雄之奇愽王褒之伟丽伯玉太白之诗追高古于风雅眉山家学之文涣奇思于心得煌煌乎与井络炳灵而汶峨竞爽也爰至我 朝复得之升庵先生盖洞观学海擅能词场于前数子可谓兼之矣其论经学则取衷传记而指末师俗学之非稽往赜则愽综谱谍而斥游谈窽言之谬形之为篇咏则取材六朝遡汉魏之风趣着之为论纂则陶冶两汉会训诰之体裁是于学无所不窥而文则若成诵在心借书于手彬彬然究天人精祲之际镜古今得失之林矣观夫天官注张之觧公侨实沉之愽也岣嵝禹碑之训倚相索丘之闳也即退之自叙上窥姚姒下逮庄骚太史所录先生其曷让焉于乎此宁足以尽升庵哉夫先生以宰相子 武皇时对策擢第一为闺阁之臣唯 天子雅以国器许之士之慕谊问奇争奔走焉独也希志元凯之俦视时俗所趋漠如也逮 肃庙绍统文忠公以定策元勋尊宠于人臣无二先生于是时少肯依违不出长安城门十年即继踵韦玄成苏廷硕功业矣顾乃摅见陈款自比于贾山之愚汲黯之戆已而去国轸怀穷愁著书逌然小雅之怨诽不乱其被发行吟衎衎焉曳縰而歌商颂声满天地者也展如人也所谓立于不贷之圃而为采真之游者非耶是故金马碧鸡之涯长卿子渊之建节视升庵之输校不同其为山川之增辉一也会稽禹穴之巅司马子长之壮游视伯喈之亡命不同其为古今之胜事一也绩学之士得斯集诵读而论其世焉庶灼知先生直谅多闻为古之益友哉藩臬诸君请为先生文集序故附尚论于先生者其说如此云赐进士巡抚四川都察院右副都御史升吏部右侍雍韩澽滨张士佩 书

黄帝晨坐于扈阁降观于荣阿有鸟来巢命音交和乃召天老而问焉天老对曰西申之国丹穴之山爰有神鸟名为凤焉羽族三百六十以应周天天帝命此鸟以为羽族先其为状也鸿前而麟后蛇颈而鱼尾鹳颡而鸳腮龙文而龟身燕颔而鸡咮鹤植而丽化首若青?蒦青戴仁也婴若白垩抱义也斧若赤丹负礼也胷若石墨蕴智也足下赨黄履信也厥像有六厥苞有九六像维何头像天足像地日像在日月像在背翼像曰风尾像曰纬九苞维何口苞命眼合度耳聦达舌屈伸色光彩冠短朱距兑钩音激扬腹文户天为生食爰在积石其树名曰琼枝琦玗琅玗以为实琅玕之膄是曰离珠一人三头逓卧逓起以伺琅玕与琦玗子栖必朝阳高冈之桐啄必紫脱苍筤之粒其饮有仪其食有质其和鸣也小音金大音鼓音如箫自歌翰如于自舞昏鸣曰固常旦鸣曰发明昼鸣曰保章举鸣曰上翔集鸣曰归昌节节足足喈喈邕邕其音英英凄凄锵锵延颈奋翼五彩备明举随八风气应时两徊欝岛翔玄圃过昆仑轶底柱风穴暮宿弱水濯羽五彩晶英九文鲜眀乃先天离宫之畜南方鹑火之精览七德律五音通天祉应地灵翥即文始来即嘉成惟凤为能天下有道得凤象之一则凤过而睨之得凤象之二则凤翔而企之得凤象之三则凤集而翙之得凤象之四则凤春秋会之得凤象之五则凤朝夕比之没身居之矣于是时也凤翩蹮帝竹而不惊宿留帝梧而不去吸昆丘之琅霜吞嵰山之紫露驯于宫邻为帝司晨扈于轸车以节路驱爰率其属朝嘲夜口?夜露翥风翔千百为羣朝于明堂故曰凤忘其翼鸷鸟忘其瓜而可扰神龙丧其角鸷兽忘其距而可御咸若鸟兽而柔缓畴若草木而葳蕤苍麟白驺可系而羁鸟巢鹊梁可俯而窥是以万方来朝四露咸依也乃若刳胎剔女?黾神龙不游覆巢石?为卵凤不翔留四灵之瑞交违则万物之庚不由矣帝于是始作咸池之乐张于洞庭之野有焱氏承而颂之曰凤皇秋秋其音若箫其翼若干有凤有皇乐帝之心于是风后从而??羽?贝?之曰舟张群雍鸧鸧相从八风回回凤皇喈喈力牧载而和之曰敕尔瞽帅尔众工奏尔光诵肃肃邕邕无怠无凶帝曰昭乎哉天老之言也绿图侍侧拜手而赞于帝曰臣闻缥青者曰鸾黄?今黄者曰鹓饶紫者曰鸑多白者曰鹄得凤体而分者也西有鳻雀东有谏珂北有定甲南有锦驼鵕义?鸟飞光而暎天绶鸡曳彩而骈罗窃凤形而似者也且羽嘉生蜚翼蜚翼生凤皇凤皇生鸾鸟鸾鸟生庶鸟庶鸟之变乃产妖鸟是曰羽虫之孽德下衰则得其分德又下衰则得其似德极下衰则得其孽臣恐杄百芯之后淡见而甘闻诬天而矫神必??求凤而弗能将歌舞羽孽而珍贶咎征天无口孰分其然否神无喙孰折其真伪諴孚几何矫枉寔多凤之瑞世仅见孽世则那帝曰远乎哉绿图之见也遂乃命伶伦断嶰谷之竹吹律写凤之音素女调三舌之簧炙笙象凤之翼史皇作斧扆绘凤之羽胡曹薾衣裳画凤之文又采飞龙氏之篆陈帝鸿氏之砚书之右书曰巾几铭图之左图为帝命验
