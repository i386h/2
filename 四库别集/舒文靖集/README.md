舒文靖集     （宋）舒璘


舒文靖集二巻，宋舒璘撰。璘字符质，一字符宾，奉化人，补入太学。张栻官中都，璘往从之，又从陆九渊朱子吕祖谦诸人游，故于性命之旨深有所窥。举干道八年进士，初授徽州教授，擢知平阳县，终宜州通判。淳佑中特谥文靖。事迹具宋史儒林传。璘平生未尝官于朝，故集中不着章疏，然即寻常酬答之文，亦无不原本于学术。本传谓璘教授新安，作诗礼讲解家传人习，自是其学寖盛。今观集中与陈仓札子，论常平义仓茶盐保长之法，深切时弊，皆其教授新安时所作。则璘之成绩，不特着于学校，如宋史所云也。又考本传称璘释褐之后，两除郡教授不就，未着其所除者何郡。今观所作先君承议圹铭，末有曰子璘迪功郎信州州学教授，则是所称两郡之一，而史失之者。又本传称璘为徽州学官，司业汪逵首欲荐璘，或谓璘举员已足，逵曰：吾职当举教官，舎斯人将谁先？卒剡荐之。今集中有申谢傅漕及张守陈仓三人荐举札子，又有谢李提举书，乃独不及汪逵。其与楼大防书云：前张守所惠荐章，蒙示回简方知所自。与王大卿书亦云：荷礼书尤丈漕使傅丈仓使陈丈惠然相举，皆出提奨之赐。亦均不及逵名。岂此集犹有佚篇，抑或所传异词，故史与集多有不符欤？

上淮东总领韩郎中书 

尝谓天下之事初无难易之辨处得其要则大而难者犹将转而为易况于小而易处者乎夫大而难者人必精心竭虑思有以处之故皆不劳而办至于小而易处人必忽而不经意以易处之则难将至矣今日京口之潮闸是也夫京口一闸川陜荆湖江淮之都会漕运赴行阙者之所必趋也舳舻相衔岁不下数千百万贯斛率皆涉重湖冒大江备尝乎险阻艰难幸免乎破碎沦溺而得至于此者莫不以手加额举欣欣而相庆曰是行也已见涯涘吾属无虑矣不谓有司处失其要不量利害轻重以第其出入闸栅一启内外事济强凌弱众暴寡而使舟中之人互为敌国也夫日俟一潮如饥待哺幸而可济莫不手舞足跃执篙拽柂以觊其必入夫何交相持终一潮而获济者不过五六又皆不急之舟江潮一退皆阁乎滩涂之上因之而敧侧破损者日不下三四贯朽之钱尚可复校米粟一破坏不可食矣其利害轻重岂不较然明白也哉又况岁当大有漕运之粟原原未艾其数倍蓰于他年今重纲鳞比待乎外者不啻百数以前日一潮所入如是其少则虽数更朔望不能使之尽入也此事之甚小若不足闗执事之聴然钱谷之问正使台之所司讵可以恝然不经意哉恭惟某官负超卓之竒才藴经纶之素志军储大计暂使提总圣天子姑以试萧相国刘中书之规模措画也某不量愚陋以小人之腹度君子之心则今日之事在使台可以责之有司以次第先后给之木牌重纲先入余舟次之欲出之舟又次之越次者重刑以惩绳之苟如是而处之则一日二日出者入者皆可济矣岂不若难而甚易也哉孟子曰事在易而求之难此之谓也某尝谓布衣之士身居穷约一登王公贵人之门则必为阍人所辞以故非公事未尝一至公门兹因侍亲部纲行在所坐困于此者浃辰矣是敢饰愚陋陈利害修短书为贽见之恭惟阁下垂仁而采择焉
