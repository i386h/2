　　提要
　　【臣】等谨案檀园集十二巻明李流芳撰流芳字长蘅嘉定人万厯丙午举人三上公车不第因魏忠贤乱政遂絶意进取筑檀园读书其中明史文苑传附见唐时升传中是编凡古今体诗六巻杂文四巻题画跋二巻虽才地稍弱不能与其乡归有光等抗衡而当天啓崇祯之时竟陵之盛气方新厯下之余波未絶流芳容与其间独恪守先正之典型歩歩趋趋词归雅洁三百年中斯亦晚秀矣谢三宾刻嘉定四先生集时流芳尚存三賔诣视其疾索所作因尽出平生诗文手自芟纂以成斯集三宾为作序文亦感慨凄动三宾字象三鄞县人天啓乙丑进士后官廵按御史守莱州颇着劳绩掖县毛霦平叛记载之最详云乾隆四十六年三月恭校上

　　檀园集序
　　予为嘉定之三年始谋刻四家文集于时长蘅已病卧檀园予躬致药饵登牀握手长蘅为强起尽出所著作手自芟纂得诗六巻序记杂文四巻画册题跋二巻合十二巻题曰檀园集授其侄宜之以应予之请遂刻自檀园集始明年正月长蘅没予哭其家为经纪其丧欷歔不能去已而刻成因为之序长蘅累世簪缨科名廿载文章书画绚烂海内其徒盗窃名姓及摸勒衒售者犹足以奉父母活妻子而长蘅身没之日园亭水石图书彞鼎之外籯无一金廪无釡粟髙贤静士之风流其大畧亦可覩已为人慷慨遇不平事无问朝野辄义形于色然慈惠乐易其素性也喜接后辈周贫交尤喜成人之美未尝有所怨忌时或发词偏宕或诗文感愤类于骂讥嘲谑者有之然言者无罪闻者足戒正所谓防于风者矣惜其穷老不遇徒放浪于呉山越水盱衡奋袂以自鸣其不平故仅存兹集以传世使得待诏金马延登玉堂拜稽扬厉以上继臯陶史克之作令鄙人小夫帖耳咋舌于文章之有用从此不敢侮易文墨士不亦伟欤而竟优游林壑以没此予之叹息痛闵于长蘅也嗟乎长蘅之所流传未知鸡林等国何如凡我公卿学士下至贾竖野老以及道人剑客无不知敬慕若古人然长蘅亦荣矣然大率珍其画与书耳能得其诗文之意之所在者已不可多得而况其为人之大概乎昔王逸少在东晋时其精识防虑高标伟节识者信为蔡谟温峤之流而为书名所掩至今耳食者但晓宗其翰墨此又予之反覆婉折于兹序也崇祯二年秋七月友人谢三賔序

　　五言古诗【凡七十二首】
　　冬夜书怀
　　怀人不能寐起行眎天末风高夜气爽空庭贮寒月落木何萧疎纵横影交列万籁久逾静中懐耿不灭忆我心所欢生平矢相结嬿婉能几时一朝悲契濶前日送我行擥衣与我诀期我明月夜翩然履我闼将子无愆期指为三四屈筦簟既已安樽罍亦云设期逝子不来音尘望中絶川涂非渺邈江河多舟楫岂不顾前好或以事羁绁一心抱终始怀疑难自决团团天上月光辉有时缺蔼蔼庭中树岂无辞柯叶新欢与故知恐或异凉热引领还入房垂泪空咄咄
　　陇上别
　　青青陇上麦离离江边树此地曾别郎细雨湿归路【一解】郎亦从此去侬亦从此辞感彼路傍人道侬相送时【二解】不忍与郎别又不随郎去只是牵郎衣踌躇复不语【三解】斫枝斫连理折花折竝头今日非昨日侬身那自由【四解】一花持伴侬一花持赠郎不比花颜色但比花参商【五解】侬如车脚泥弃置亦不安郎如失林翼孤栖亦不欢【六解】侬自目送郎请郎莫回顾恐郎为侬辛泪亦为郎茹【七解】譬如不相识相防自有期但愿加餐饭不愿长相思【八解】
　　酒后为郑闲孟画扇戏题
　　吾爱水上亭覆以栁如丝不独水清浅春风故来吹水绿栁鬬青顾眄明须睂荒园二三畆新栽四五枝下有沮洳流上无黄草时亦偕吾友对此斟酌之适兴聊复尔防缀亦何为
　　将赴试白下走笔别荃之
　　忆防婉娈时与子有成说悠悠路傍人使我意不彻龌龊不足论但念子羁绁人生非鹿豕岂难相诀絶不忍区区诚执手即呜咽白门河畔栁句曲邸中月维舟复沽酒与子行相挈车尘十尺深关路百盘折奚囊共结束蹇卫争蹩躠村醪解饥劬松柴憩烦热十年谙客味梦到魂欲裂况复与子辞独行何孑孑水行多风浪陆行畏炎魃车行虞臲卼马行忧蹉跌世路皆如此吾生何屑屑白龙百畆山嘉木防成列朅来偕吾友掉头意已决永怀向子期遂陋张生舌兹游非本情聊以当一吷子如有两意请与子长别
　　题荃之画兰
　　我昔学画时意亦颇浩渺不求工形似但以写怀抱十年弄笔研自顾尚草草子真有夙慧落笔那便好疎疎几叶兰此意亦难了墨肥苦无骨险瘦神亦槁纵笔伤婀娜取态失苍老不独烦位置兼亦贵风藻看子意有余一往何振掉着花花离离着叶叶嫋嫋因风欲翩翻堕雨故天矫开箑飒生气嫣然出物表始知画有真俗工徒潦倒勉旃自珍重成名不足道因忆吾友言惜哉笼此鸟【闲孟遗荃之诗尝有此句】
　　燕中归为闲孟画烟林小景有感而作
　　我行江北路转爱江南趣虽有逺近山而无髙低树山枯石欲死泉涸磵亦痼平生山水欢所遇顿非故朅来扬子边望见江南雾雾中何突兀金焦与北固春山一何青春江一何素得非笔变化毋乃墨吞吐恍如逢故人此意不能喻吾家疁城东小筑溪头住春流正环门夏木将莽互尝耽诗中色兼识画中句游好在所生毋为勤远慕
　　登慈云岭还访严印持忍公无敕邹孟阳闻子将于南山小筑信宿为子将题画作
　　慈云何盘盘崪嵂罗众致白石如叠浪青林若簪髻落日冝远山况与秋爽防忆昨湖上亭解后多意气感君十年心发我千里思江山日待人突兀为我辈从来费梦寐到此烦应对竒懐郁种种笔墨差远寄披襟欲嗒然相与寻冥契
　　苦雨行【纪戊申五月事】
　　冬春苦不雨井竭水值钱入夏雨不休出门水接天咄哉造物功旱涝何其偏贫家少生事俯仰资薄田平陆成江湖一望令心酸二麦既已尽穉苗不得安歳功苟如此何以供粥饘阿母向我言汝忧长未殚且勿计歳功目前亦囏难斗米如斗珠束薪如束缣缾中蓄已罄防下寒无烟小妇谋夕舂大妇愁朝餐前日雨压垣舍北泥盘盘昨夜雨穿屋帐底流潺潺篱落坏不理衣裳湿难干黾勉慰阿母独坐穷忧端海内方嗷嗷常苦征赋繁去年北大水三辅民骚然司农乏远筹往往苛东南连年稍成稔额外不肯寛虽有卒歳储倾囊输上官吾观闾左情岂堪一防年惜哉此长虑谁为叩帝阍传闻四郡间告荒牍如山贤侯轸民瘼步祷良亦防庶几回天心拯此鱼鼈患褰裳出屋头仰视云根蟠水鸟立我傍鸣蛙跳我前础石润欲滴青灯光不炎晴占杳难期雨势殊未厌翻思二三子裹足不到门何时一相呼诉此情纒绵且当冒雨出弄舟村东湾远水正澹荡树色相新鲜浊酒尚可賖聊以开心顔
　　送座师林先生被召北上
　　男儿生世中所贵在心期区区文字间未足定相知蹇余好迂阔赋性不适时落魄十载余一朝生光辉感彼国士恩耿耿懐在兹吾闻昔人言诎伸各有宜有怀不得伸何用相知为末俗文貌繁伛偻效尊卑相对少真味嗫嚅恐见疑先生豁达人容我放言辞况兹分手路长当隔云泥临风效微忱蓄意纷如丝
　　梁溪东南冲自昔称剧邑先生脱儒冠吏治如夙习下车问疾苦豪右争慑息秋霜比皎洁春风让披拂啧啧道路间口碑岂漫灭当今服官者安坐耀朱芾吾闻先生勤六年如一日长官费送迎簿书苦敦迫戴星出视事日旰常不食永言秉此心岂但一称职天子综吏治征书下南国异等推先生循良亦生色但闻朝署闲台谏员半缺去年得俞防至今壅膏泽擢官仍空衔毋乃非名实感时叹多艰增我意抑塞
　　去年西北饥今年东南水极目三呉间洪涛荡千里田穉委阳侯鱼鼈半赤子更闻边储空司农乏经理睠焉财赋区脂膏宁余几嗷嗷沟中瘠当复虞庚癸古人喻获薪斯言亦有防帝阍远难排父母幸孔迩借防已不再攀辕讵能已提携满道傍号呼拥行李恻怆慈母心行行且徙倚徙倚亦何为建白从此始簪笔入承明抗章上殿陛圣主容直言痛哭况时事上言天方难下言灾可弭沛然发徳音寛卹暨东鄙道莩庶以苏菜色行当起先生此猷苍生拭目竢
　　春日过梁溪偕顾虞工秦圆昔呉峦穉刘彦和游寳界信宿陈氏山庄翌日将泛湖至华藏寺阻风不果登鼋头渚而还即事有作
　　梁溪足名胜春风美游遨吾友好事人载酒呼轻舠出郭风日丽山水况相遭新栁正濯濯防缀舒夭桃晴光发野色远近匀轻描言登寳界巅极目湖天遥台榭已零落松风尚飂飂山中贤主人客至爱招邀牀头酒如泉狂呌容吾曹明发恣幽讨相与寻山椒山风忽振荡湖水方怒呺华藏渺何许舟楫不敢操壮哉鼋头渚蜿蜒出惊涛砥柱岂有意神功一何劳下穿怪石藂溅沫声嘈嘈奇状洵不一闪倏心魂摇或如怒兽奔或如劒防交或侧立若屏或嵌空若寮山奴亦不俗济胜矜防趫往往轻险絶引我凌最高吾曹二三子瞠目但呌号落日下骥渚倒见金光漂独山青蒙茸相望转泬寥酾酒酹长风使我情一豪境清难久留归路翻萧骚平生爱奇赏兹游惬久要勿言后期赊但鬬所得饶
　　芙蓉花下独饮戏柬里中兄弟
　　爱此芙蓉花簇簇水之湄上有竹参差下有水涟漪水竹相暎发红白交光辉自我牵物役三年不见兹前日呉门归常恐后花期家人向我言花开尚未齐连朝登楼望风雨何凄其恐复花顦顇徙倚多嗟咨花神好防惜留此迟晴曦今日秋爽佳转觉花多姿过雨着深浅因风故髙低既耀朝阳色更与落日宜一生爱花心输写当此时何以酬花神但有酒满巵自从抱病来甘与麯蘖辞对此不快饮空负看花为但愿十日晴兼谢尘鞅羁客至不出迎闭门客勿疑虽有爱花人不如我情痴我思古人言胜事空自知
　　己酉春日以看梅到弹山信宿山阁读壁间旧题如昨日耳而当时共事者徐孺谷张君实与小史荃之皆死矣退之有言人欲久不死而观居此世者何也怆然兴怀爰作此诗
　　七载阁中人重来死亡半廵览惊孤游数往疑梦幻当年诗酒徒徐生意何悍小史最清发与我情婉娈张生实同调时亦偕汗漫一朝俱灰尘萧条空里闬不见窻中山突兀犹在眼不见门前树森疎好枝干山花向我笑山鸟自相唤壁间旧题字墨迹未漫漶嗟我同游人一逝不复返晓日充山舟夕月铜井院湖波绿欲皱杨梅红始绽曳杖追流云浮杯激飞淀此中乐事多此日欢情变低回阅身世生存腼颜面西州下悲泪黄垆发浩叹予怀不可道松风吹独旦
　　西碛看花宿六浮阁上走笔示闲孟兼呈同游诸子
　　去年梅花新爱杀钱家渚花光与水色映彻乃如许我欲作一诗绪多不能举因之发浩想就此结茆宇西碛亦旷莽数畆足容与朅来寻旧游况复多伴侣信足潭东西入眦浮四五【山中有潭束潭西湖中有五浮】花繁塞远近径折迷处所或疑云浮空复道雪封坞吾徒好奇者遇境爱多取到此徒跌宕耳目不自主翻思去年人重来仅予汝程龚与髯朱何意守环堵清游信难期汝乃以予鼓昨夜怀隂晴江头过风雨入春多佳日造物岂予侮兹行偶得遂予歌汝可舞兴防更秉烛相对寒阁语山风飒然清似劝手中醑买山已得计吾驾不可圉
