　　提要
　　
　　臣等谨案未轩文集十二巻明黄仲昭撰仲昭名潜以字行莆田人成化丙戌进士官至江西提学佥事是集为其门人刘节所编凡文六巻诗五巻词一巻而以诗文墓志铭附之仲昭官编修时与章懋庄■〈日上永下〉并以疏争元宵烟火诗廷杖谪官当时有翰林三君子之目后懋与■〈日上永下〉并以聚徒讲学为事而仲昭独刻意纪述八闽通志延平府志邵武府志南平县志兴化府志皆所编録故枫山定山之名满于天下仲昭几为所掩然三人气节相同居官清介相同文章质实亦略相同未可以仲昭笃志励行不作语録遂分优劣于其间也林瀚作仲昭墓志称其作为文章浑厚典重无艰深聱硗之语郑岳莆阳文献传亦称其有未轩集若干巻文词典雅今观其集虽尚沿当日平实之格而人品既髙自无鄙语颉颃于作者之间正不以坦易为嫌矣

　　谏元宵烟火诗疏（时任翰林院编修同编修章懋检讨庄■〈日上永下〉具）
　　
　　奏为培养圣徳事内阁遣郎中韩定持小揭帖到东阁及史馆分与太常寺卿兼翰林院侍读学士吴节等令各赋烟火等诗以为上元赏玩之具臣等各授一帖内开烟火花火等项面贴诗赞题目仍令照依旧诗格式拟述进呈及观旧式俱是玩好之物鄙亵之词甚非所以养圣心崇圣徳也臣等窃议以为此事必非陛下所为陛下以天纵之圣日新之学游心经术迈迹尧舜凡所举动必欲为法于天下可传于后世方即位之初首下温诏放田租絶贡献蠲逋负停不急之务与民息肩又开言路凡朝廷政治得失军民利病许诸人直言无隠天下忻然以为陛下应天心承祖徳而所以太平万世者在是矣及观去年以来如遣人造楮国家旧制也一闻大臣之言而遂寝节令宴乐每歳常例也一闻廷臣之疏而随罢顷因灾异勑谕群臣同加修省凡此数事皆臣等目击耳闻未尝不拜手稽首称颂以为陛下从善如流改过不吝自禹汤以来未之有也在彼者既皆所不为则烟火之事臣等又决知陛下之不乐于此也今日之举或者两宫皇太后在上陛下欲极孝养奉其欢心非为一身娱乐之计然大孝在乎养志不可徒供耳目之玩好以为养也臣等伏覩两宫母后恭俭慈仁之徳着于天下坤仪贞静举天下之珍竒玩好皆不足以动其心岂以烟火为乐哉况今两广弗靖四川未宁辽东虽云告捷然敌情难测尚费区处不可置之度外北敌毛里孩包蔵启疆之心窥伺间隙犹当深虑江西湖广一旱数千里民不聊生其它灾伤处所尤未易悉举生灵嗷嗷张口待哺虽蒙优诏赈恤然公私匮乏计无所出可为寒心此正陛下宵旰焦劳不遑暇食两宫母后同忧天下之日臣等又知陛下之不暇为此也至如翰林之官以论思代言为职虽曰供奉文字然鄙俚不经之词岂宜进于君上若不取法圣贤而曲引宋祁蘓轼之教坊致语以为比是以三代而下之君望陛下而不以三代而上之君望陛下也臣等遭遇圣明发身黄甲叨与庶吉士之选陛下养之翰林教之诵习六经师法孔孟二年于兹矣近又授以今职感冒国恩夙夜惓惓相与戒饬惟恐曲学阿世无以补报于万一何敢为此鄙词上渎天聴以自取侮慢不敬之罪哉臣等又尝伏读宣宗章皇帝御制翰林院箴有曰启沃之言惟义与仁尧舜之道邹孟以陈若今烟火之举恐非尧舜之道烟火之诗恐非仁义之言臣等知陛下之心即祖宗之心故不敢以是妄陈于陛下之前且知其不可犹顺而为之是不忠也知不可为而不以实闻是不直也不忠不直臣罪大矣古之帝王盘盂有戒几杖有铭目不视非礼之色耳不聴非礼之声兢兢业业惟懐永图虽在纷华靡丽之中幽独得肆之地而所以戒谨恐惧操存省察以致其精一之功者无所不用其极诚以人主一心攻之者众一惑于耳则凡侈靡之声皆乘间而入矣一惑于目则凡侈靡之色皆抵隙而进矣人心愈危则道心愈微天理人欲不容并立若曰上元之乐乃微事耳烟火之举乃细故耳此不足为圣明之累是殆不然书曰不矜细行终累大徳又曰不役耳目百度惟贞若于此一事厌常喜新之念兴则他日之甚于此者将无不至此圣徳修否之所系不可以为微事细故而不之谨也且漆器之作何损于徳而舜则止之旨酒之甘何害于事而禹则絶之露台之费不足为奢而汉文则已之彼圣贤之君何汲汲于是哉正以欲不可纵渐不可长故耳臣等伏愿陛下寛斧钺之诛采刍荛之语将此烟火等事一皆禁止不使接于耳目而移此视聴为文王之视民如伤为大舜之闻善若决江河省此冗费以活流离困苦之民赏征戍劳役之士则干戈可息灾旱可消百姓可以富庶四夷可以宾服亿千万年享太平无疆之休所谓先天下之忧而忧后天下之乐而乐则陛下之所以奉养两宫者其孝岂有大于此哉惟陛下深思而力行之不以臣等为沽矫不以臣言为迂阔使天下后世知大圣人之所作为出于寻常万万岂惟臣等之幸实宗社生灵之大幸也臣等昧死谨具本以闻
　　
　　乞恩养病疏（时任南京大理寺署左寺副事右评事）
　　
　　奏为乞恩养病事臣自歴任前职成化十一年正月内丁母忧回家本年十一月接丁父忧成化十四年二月服满未满之先得患伤寒转成劳疟具告本府节次拨医调治至成化十五年十一月内仅得痊可复恐过违限期随即起复于成化十六年三月到部臣切思禀赋素弱加以连年患病致成虚损五心烦热四肢倦怠目今勉强支持虽能歩履稍遇劳役即觉头目眩晕饮食减少调治累日不得复常臣之微躯多病如此虽图报之念未尝敢忘而蹇劣之姿岂堪鞭策若复叨縻廪禄不无妨废职事如蒙准奏伏乞圣明垂悯勑令该部容臣照例回家养病倘或异时调理平复即当赴部聴用期报圣恩于万一也臣干冒天威无任危惧战栗之至为此具本谨具奏闻
　　
　　乞恩致仕疏（时任江西按察司提学佥事）
　　
　　奏为乞恩休致事臣自歴任今职钦蒙圣旨赐以勑谕俾臣専理学政臣拜命以来夙夜兢惕思以奉宣徳意鼓舞英髦庶其有成以报圣恩万一第愧才学踈謭教导无术抵任迄今已余四载而于士风民俗未见丝毫之益窃伏惟念若不早求退休以避贤路则不惟上孤委任抑且下误后学在臣愚衷实所未安况臣禀赋素弱自少多病稍渉烦劳遂觉疲困自先帝龙飞之初叨窃一第食禄供职纔五六年而疾病纒绵竟无虚歳荷蒙先帝天地大恩俞臣之请放回养病几二十年稍获痊安所以不恤厚颜垂老一出者葢感念先帝厚恩思欲报之于陛下也今年已六十有一须髪颁白齿牙动揺精神气力渐不如昔虽欲勉强策励以求寸进势有不能臣于去歳已兴此念继闻有奏准事例通行天下考选生员此实臣之职业故未敢遽言其私今臣巡歴遵依考选在六七月间可以毕事庸敢冒昧陈请烦渎圣聴如蒙乞勑吏部别选有学行官员前来提督学政仍赦臣委任不效之罪放遣致仕虽余生残喘报称无阶倘犹生存尚当训诲子孙服膺圣化庶有以图报于后日也臣无任瞻天仰圣恳切愿望之至为此具本令义男赍捧谨奏

　　读周易
　　
　　千载之下欲求圣人之言于千古之上此可以理求而不可以辞泥也夫载籍之中阙文误字者多矣求诸理则可以通其所不通泥其辞而于所不通者欲强通之亦何以得其义之所在耶易未济之六三曰未济征凶利涉大川程子传之云三以阴柔不中正之才而居险不足以济征所以凶也然未济有可济之道险终有出险之理上有阳刚之应若能涉险而徃从之则济矣故利涉大川也其言固不为无理吾窃意其终费辞说也朱子疑其有阙文当为不利渉大川可谓深得周公之意矣葢周公之意谓三居险之极而阴柔失位才不足以济险故征则凶而不利涉大川也若曰未济征凶利涉大川则文义有相背者矣或曰他书火于秦故多阙误易以卜筮得免而亦言有阙误何也曰系辞论乾坤为易之门朱子于章内固言其有阙文疑字矣然则以易为未经泰火而无阙误者不亦固哉
　　
　　读尚书
　　
　　文章之髙下系乎气运之升降观于书可见矣夫五经莫古于书所以纪唐虞三代致治之道也然其间已不能无升降焉亦可以观世变矣或曰孔子删书断自唐虞二典三谟唐虞之书也宜其最古矣然观其文平易明白反不若商周多佶屈聱牙者何耶曰不然唐虞之世气运方隆以尧舜为君以禹皋陶稷契为臣相与都俞吁咈于庙堂之上者浑然泰和之治故典谟之文明白正大万世不能及也降及夏商周气运已不及唐虞之隆而其所以致治者视唐虞亦不能无异也故其发于誓命训诰者虽皆一时贤圣之君所述然其语至有佶屈聱牙而不可觧者亦文之变也及夫周室东迁气运陵迟极矣故文侯之命萎薾不振费誓秦誓亦犹黍离降为国风矣其文章何如耶吾故曰文章之髙下系乎气运之升降观于书可见矣或曰若是则平易者果为书之最古而佶屈聱牙者反有所不逮欤曰孔子讲道于洙泗其与羣弟子相答问者皆平易明白至其论文章亦惟曰辞达而已矣然则书之最古者果在于彼耶在于此耶
　　
　　读毛诗
　　
　　自武王有天下诗之作于周者皆谓之雅及平王东迁诗之作于周者则降而为风果其诗之自降耶抑夫子删诗而降之耶谓其自降则平王虽东迁而天子之名犹在其诗固不得谓之风谓夫子降之则夫子修春秋常以尊周为义今而删诗乃降之使下同于列国不亦自相矛盾耶噫是亦足以验世道之升降矣周之世道莫升于武王有天下之后莫降于平王东迁之时世道有升降则诗随之而为升降者理也亦势也是故周之盛时成王为天子周公相之朝廷制作大备故其诗之见于小雅者则为燕飨之乐其见于大雅者则为会朝之乐受厘陈戒之词或欢忻和悦以尽羣下之情或恭敬斋庄以发先王之徳蔼然雍熈泰和之气象焉及周之衰平王东迁朝廷已无制作其诗之见于国中者不过闵宗周之顚覆伤行役之劳苦述其乱离之忧淫奔之情甚至忘申侯杀父之雠反为之戍守致民之怨而形于诗焉果有如小雅之欢忻和悦者欤果有如大雅之恭敬斋庄者欤其诗既无二雅之事则亦民俗歌謡之词与列国等耳夫子葢亦因其诗之自降是以从而降之也虽然平王之诗与二雅之正者固大有间矣然尚为天子之诗以之而附于变雅可也何一降而遂为风耶盖雅之变者其政事虽不及于正雅而其声则有合焉若平王之诗其体固列国之风也夫子亦安得而私之哉
