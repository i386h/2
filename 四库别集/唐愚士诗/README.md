　　提要 

　　臣等谨案唐愚士诗一巻附会稽懐古诗一巻明唐之淳撰之淳字愚士亦以字行山阴人肃之子也建文初诏词臣修鉴戒录方孝孺荐之授翰林院侍读与孝孺同领书局卒于官明史文苑传附载王行传中徐祯卿翦胜野闻载明太祖以布囊贮之淳夜越宫墙入便殿点窜十王册文一事其事荒诞不经殆委巷小人因之淳文思敏捷造是妄语张芹遗忠录称洪武中有荐之者谢不就曹国公李景隆俾其子师焉征行四方皆与俱厯燕蓟秦周览前代遗迹援笔而赋凌轹一时考明史李文忠传景隆以洪武十九年袭封曹国公不载其非征事惟冯胜传载洪武二十年与傅友徳蓝玉赵庸等北征常茂李景隆邓镇皆従是年岁在丁夘与集中寓宁轩记所载洪武丁夘相合当即其时也是集仅其丁夘戊辰二年所作似非完本又诗文相间成编而总题曰诗亦非体例疑当日杂录手稿存此一帙后人因钞传之故编次丛杂如此欤其诗虽未经简汰金砾并存而气格质实无元季纎秾之习其塞外诸作山川物产尤足以资考核会稽懐古诗一巻乃其少作凡五言古诗三十首题下各有小序仿阮阅曾极张尧同之例其中如舜庙不取地志象耕之说禹庙不取禹穴藏书之说皆为有识此巻本于集外别行然篇页寥寥今缀于集后末附长洲戴冠和诗三十首大抵凑泊成篇不及之淳原唱以旧本所有姑亦并存焉

　　出京师述懐（丁夘二月十二日以下皆道中作） 

　　辞亲戒行李祗命出神京念兹行役迟遥遥睇前旌皇仁固无外所怒不来庭授钺必卫霍献谋必良平旗旜耀日色铙吹杂风声饮马河流浑磨刀山石腥野有貔虎势川无蛟鳄惊况当青阳节卉木竞敷荣晴云布鱼丽荑栁啭仓庚士健衣有辉牧肥草青青狼居行可陟祁连待勒铭竖儒际斯盛何以副从征

　　到凤阳奉呈南涧公 

　　行役到淮甸重经感岁年鸡声犹郡郭草色自人烟访古山河在思亲涕泗悬南来愁跨马北去喜乗船便续淮南赋仍歌（阙）篇山前谁杜甫槎上或张骞祇觉心非石徒教髪变玄风依沙寺泊雨借驿楼眠燕社迎沽酒莺春促换绵锦翻花蔼蔼茵细草芊芊饱食王孙饭寒分贵客毡从容酬白昼出处任苍天 

　　奉和南涧公送别之韵 

　　自甘天地一闲人偶得从游南涧滨吟到好风兼细栁锦囊字字合阳春 

　　留别南涧公 

　　几逥投笔厌窓鸡今日云边入马蹄欲赴陈王燕平乐还同李愬入淮西羽书烽火三方静紫塞黄河一路题喜得宠行诗句好船头夜夜爥虹霓 

　　长淮船 

　　长淮船淮流不断船连连二月三月天不雨鹁鸪昼啼杨栁烟淮东土干无麦田淮西鱼贱不论钱但祝好风行十日鱼羮饱饭枕书眠 

　　泊五河县 

　　夜泊淮堤近五河恶风吹面叹蹉跎青山更比吴中少緑浪翻同汉水多去国何须事弓剑思亲有涙落烟波三千里外江南客却倚月明闻楚歌 

　　过盱眙旧县懐南涧公 

　　旧县新城取次过百年世事几风波挂帆客过沽春酒复业人归理钓蓑山水未生淮白少暮云欲尽海青多客懐不及南飞鴈也逐闲鸥到涧河 

　　喜雨 

　　归来鴈影中流见过去钟声两岸闻日没山青星斗黒小龙将雨嫁春云

　　泗州 

　　飘飘帆席去如驰坐倚船窓看晚炊河口驿亭题泗水山头县郭认盱眙当年楚汉兴亡处今日风云际会时已筑祖陵开寝庙黄童白叟望旌旗 

　　项王城 

　　泗州城北项王城千古令人叹未平韩信归刘知有主范増辞楚计无成江淮不洗英雄恨鱼鸟犹疑叱咤声欲唤春醪奠遗迹风帆催客赴修程 

　　韩信城 

　　雉堞平来事已休淡烟芳草一荒丘蒯生不作忠君计吕氏方为少主忧烹犬有时应自喜缚鸡无力岂长谋泗河两岸离离石留与行人系晚舟 

　　二月十五日抵宿迁 

　　长河得天风挂席夜叅半篙师喜利涉卧客生惊叹空明泝流光滉漾入云汉心同双掳争思逐羣波乱春热减衣衾川鸣起鹅鴈已从桃源辞方谋宿迁饭 

　　泗州第一山追和米南宫韵 

　　书画船从此地还云霞草木尽毫间江淮多少名山水此是君家第一山 

　　舟中对花 

　　河畔庄前无数花折来船上饮流霞故园亦有如花树争奈开时不在家 

　　吕梁洪 

　　三月上吕梁春旱水未驶水中半白石错出犬牙鋭其流本闲暇急束若有恃乍疑云汉泻星斗湿流丽又疑沧海涛喷射巨鳌背轰轰万雷撞佶佶千兵势为河作户限匪一乃再至上洪急且纡下洪峻而厉舟航抵其下一叶舞云际跻攀仅分寸失手寻丈外累累腰巨絙牛喘仆夫瘁已涉未足欢欲下心反悸胡然造次顷号呌乞神恵遂令愚无知谄祭及非类方今幅员广南北一衣帯梯航极山海珠玉兼象贝设官使居守人畜数十辈蛟鼍革冥顽行旅释凝滞人心既有依天险不足畏缅思黄河源逺自昆仑致万里不能休朝宗实天意向非神禹功后世亦奚赖到今疏凿处歴歴犹可记河竭功不忘圣人等天地 

　　金龙祠曲 

　　金龙祠前春草緑金龙祠中人簇簇上洪人祝上无滩下洪人祝下平安龙语不传环珓喜牛马轻便舟楫利龙君绣旛赤羽旗龙女朱冠金凤衣愿龙顺人人不怨行人归来赛龙愿 

　　吕梁庙 

　　吕梁庙中何所祠汉有关侯唐尉迟二公生世本英特死亦令人爼豆之当其二主兴家国秦王即是刘玄徳寿亭不遇鄂公遇事在天为岂人力黄河之流声彻天二公庙食千万年不知何年创祠屋临风为取荒碑读 

　　徐州黄楼 

　　尝读子由黄楼赋每恨不作彭城游今来系船览城郭残山剩水令人愁人谁其人我所求楼非其楼名则留烟消云破见斜日水落石出如清秋长公治世才最优敛此大恵施一州不着丝纶玉堂署要洗鞾韈浑河流心存赤子天为助手却河伯人无忧功成事定作宫室酾酒赋诗相献酬一时宁知着千载两赋直可回万牛倚阑击节懐逺谋东风飞来双白鸥台沈戱马登狐兎楼空燕子鸣桑鸠雄图伯业等尘土独有文事岿山丘意公魂魄常夷犹化为千尺之黄虬与波上下浴日月镇遏巨浪同浮沤登临不尽且北去回看钓艇弄沧洲 

　　沛县歌风台 

　　河流西来复东注我方乗舟北河去长涂短服春已深小邑孤城日将暮河傍穹碑倚春树停桡一读知其故不见髙台号大风空余草屋连荒戍当时汉祖居沛中时有云气随飞龙道间未斩白帝子圯下已逢黄石公子婴首落重瞳手百二山河夘金有阴陵失路岂人为鸿门不死真天授残忍者灭仁义王威加四海归故乡箨冠巍峩筑声壮日月照耀旌旗杨身中衮裳朱衣绣酒酣歌舞争为寿父老人人被宠恩租庸世世蒙蠲宥云胡伯心犹未忘意气激烈非虞唐不歌艰难创王业却思壮士守边疆台前风烟起天末台下寒流动毛发黄昏不敢问前途驻目更看芒砀月 

　　清明日 

　　不觉离家月再圆船头揷栁记流年短衣乌帽四千里落花游丝三月天梦里松楸隔吴越眼前行李到幽燕流光倐忽不相惜几时归种山阴田 

　　泊两河口 

　　迭浪龙鳞合双河燕尾分系船欹野树炊饭暖春云鸡唱荒村得沙崩永夜闻明朝逢改火熟食侑芳尊 

　　郓城县

　　寿昌驿 

　　细雨止复落逺山看更无树光连墅迥人影共帆孤县逺先逢驿河穷欲问湖不知齐鲁近祇觉语音殊 

　　安山湖口阻风

　　懐乡中旧游

　　四女树曲

　　夹马营

　　直沽见潮 

　　我家东海头门前潮水鸣朝昏自从江淮来不闻潮声见潮痕别淮入徐沛日日对黄河河流浑浑东到海我不如何愁奈何河中半月余逆流千载折南风喜人北风怨忆着潮声梦吴越下水复上水行行抵直沽我家门前旧潮水忽复见之当海隅忆我去家正月末桃花水生洲渚阔今我忆家春欲归月缺半圆潮水落潮落当更长月缺当更满月色潮声无尽时天地乆长人世短明朝又北去见月不见潮潮头有鱼好寄信欲持尺素托波涛
