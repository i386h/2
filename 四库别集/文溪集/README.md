　　文溪集　　　　　　　　别集类三【宋】
　　提要
　　【臣】等谨案文溪集二十巻宋李昴英撰昴英字俊明畨禺人寳庆三年廷对第三淳祐初官至龙图阁待制吏部侍郎致仕卒諡忠简文溪其退居之地也张端义贵耳集载昴英初任临汀推官陈孝严激军变尽出家资抚定之曾冶鳯帅广激曾忠之变崔菊坡临城借用经畧使印抚谕李缒城入贼晓以祸福五羊城郭得全贼之肇庆就捕朝廷录功名之首除荣王府教授力辞不供职但云素无学问难以移气习士论韪之云云则昴英葢具干济之才而又能介然自守者其后劾史嵩之赵与□直声动天下有自来也是集为元至元间其门人李春叟所辑凡奏稿杂文一百二十二篇诗词一百二十五首明成化中重刻陈献章为之序其文质实简劲如其为人诗间有粗俗之语不离宋格而骨力犹健亦非靡靡之音葢言者心声其刚直之气有自然不掩者矣乾隆四十七年十月恭校上

　　文溪集原序
　　天地之精英人得之以为文可以黼黻朝廷芬芳宇宙存乎胸中必有浩然独立乎万物之表者此天地之正气也故论人之文当先观其人之所养先生昭代伟人年弱冠以雄文魁天下追风絶尘之资刷燕秣越其进也孰御而居官守当言责则宁为国死宁以言去身与家不遑恤何官爵崇卑足计哉则其所养可知矣居官多惠政及民未易缕数姑以立其大者观之初筮汀幕适有军变郡侯束手就执僚属防窜不暇公独奋身直前谕以祸福脱郡侯于虎口不崇朝而大难立觧后数歳家食循阳戍卒倡众直捣广城震撼一道清献崔公判乡郡选幕下士谕贼人人相顾股栗公以廷绅赞阃画毅然请行缒城诣贼垒言未竟聨事者已血刄颈儗刄向公恬不为动从容陈义以调伏其心贼为逡廵退却卒敛兵去城赖以完探虎穴料虎头设不幸则男儿死耳岂全躯保妻子之为者真丈夫哉既而入乌府立朝若少屈意穹班可以一武至公方碎首玉阶与天子争是非辩可否出袖中弹文对仗斥宰相借尚方剑斩佞臣茍利国家虽鼎镬不避一言枘凿拂衣径归视轩冕去来太虚一微尘耳孤忠突兀凛然与氷雪争严三学诸生送行诗有庾岭梅花清似玉一畨香要一畨寒之句闻者壮之刚方正大之气蟠郁胷次泄而为文光芒自不可掩大者中圭瓒小者锵佩环竒峭者如怪石之倚断崖清丽者如明星之炯秋汉进而立朝则论奏丹青言言药石皆足以裨主徳格君心而深衣独乐则嬉笑怒骂字字箴规皆足以植民彛垂世范盖忠义以为之骨学识以为之根故芬郁葩华烂熳宣吐不自知其为文也而文益工天收其声山頽笔絶遗编散落浩不可收仆从先生游旧矣赐墙及肩未覩阃域方将拟集大成以俟识者世运中更衣冠祸烈主家十二楼竟堕昆明刼火中斯文何辜例坐此厄噫尚忍言哉先生有子五人今中峯独存诸孙竞爽追抱往恨皇皇乎赤水元珠之求春叟耄矣于师门无能为役大惧放失永负夙心于是勉收烬余仅得奏藁杂文一百二十二篇诗词一百二十五首编次成集命之曰文溪存藁巻饬而归之羣玉府俾登诸梓以夀其传尝鼎一脔知味者有遗恨焉虽不幸而不得其全也犹幸斯文之未尽丧也杲日行天照映穹壤先生之节卓乎不可尚已先生之文泰华之毫芒耳所可传者仅止此其所不传者又孰得而窥其际哉掩涕濡毫抚巻长叹九原不可作吾谁与归先生名昴英字俊明宋朝赐之諡曰忠简文溪则其自号云峕至元三十一年甲午中元节门人李春叟百拜敬书
　　士之生世当以气节为主文章次之古今文人凡几穷者涵蓄山林逹者显著朝路然文之行世穷者恒多逹者气节难全而易亏少亏则以其文俱废矣文溪先生登丙戌科第三人声名已赫赫初宦于汀州戍卒閧欲害守公以身翼蔽之守获免及里居循戍卒叛回围广城公缒城出谕贼退城获全威武不能屈如此初登言路以直谏去再持从槖亦以直谏去富贵不能淫如此树立高则言辞伟故其文隆崛崔崒渊沦濆瀑千态万状皆气节之所充也晩歳闲适无日不登临无日不赋咏自骑鲸后六丁取将者不少大震初受公训常撰杖屦惜不掇拾以富箧笥五十年间往来懐抱一日其孙宪文以文携存稿见示感怆乆之既喜二文之如姚珽又恨大震不如李汉也公号文溪以郡北诸山泉滙于所居后元有此名盖地灵所钟也噫溪以文显不以愚辱山川亦有所遭云大徳戊戌夏门人陈大震书予尝语李徳孚曰士从事于学功深力到华落实存乃浩然自得则不知天地之为大死生之为变而况于富贵贫贱功利得丧屈伸予夺之间哉今观其先世文溪先生遗藁初渉其流弥汪洋若江河之奔驶而又好为生语险怪百出读者往往惊絶至或不能以句以为文溪直文耳徐考其实则见其重内轻外难进而易退蹈义如弗及畏利若懦夫卓乎有以自立不以物喜不以已悲盖亦庶几乎吾所谓浩然而自得者矣然后置书以叹曰嗟乎此文溪所以为文也亟读而亟思之执巻务尽乃至目倦神疲欠伸欲起辄回顾屑屑焉不忍舎也夫因言以求其心攷迹以观其用故人之深浅毕见愚不敢自谓有得于文溪之蕴顾平昔所以告徳孚者乃区区愿学而未能忽焉亲诸册之中麤若防防虽不尽觧其説要其归与此异者盖寡矣则以之而嗟叹慕悦尚奚疑哉传曰生乎百世之上百世之下闻者莫不兴起此之谓也徳孚念先绪之落落遗稿仅存复多讹缺乃深自愳恧悉访诸族之人得旧所刋本与所誊本叅校阙其所疑刻之家塾命嗣子昭董其事而俾予为序之于乎自予为儿时已闻文溪名而喜少长益向慕而独恨未识其心胷气象为何如比岁京师获交徳孚亦尝一阅其世谱今幸寄目于先生之文而知富贵果不足慕贫贱果不足羞功利得丧屈信予夺一切果皆不足为累天地之为大死生之为变自得者果不可得知而奋乎百世之上兴起百世之下孟轲氏果不予诬其所恃者盖有在也故士必志道然后足以语此徳孚好学老当益壮昭也尚亦有激于予之斯言也乎成化庚寅九月同郡陈献章公甫书

　　肇庆府放生咸若亭记
　　人与物林林然宇宙间气之正偏性之灵防物不得同于人如其肖形然而同一生意天生圣人所以厚羣生使各安其受于天者然后无负于天之付托我者毎岁誔弥日郡国臣子既瓣香祝万夀于佛老氏之宫必笼禽而林之盘鱼而溪之端笏如植视其翚翚而高洋洋而深乃再拜而退筌者罗者争前期生致以供官之须是犹狱无罪之民徐脱释以恩之亦稍顿挫困悴矣终不若此不放彼不捕之适其性也三代而后以好生之德脉其国无如本朝禁翠饰罢蜂贡放洛鲤止庖羔见虫蚁而避却鹑兎之进列圣之心即天之心今天子聪明神武而不杀日月所照霜露所坠皆在圣德包涵中翔走蠢蝡之防孰有出此心天地之外盖无一物不得其所初不待一日之间纵舍百千万亿以示小惠然故典沿袭久谓不如是非所以归美报上也故莫之敢改世以是爲尊君一事则掲防之地崇大爲宜端之放生亭旧不専设始即送迎之驿又迁之僧廜防仅扁之寄州枵然大守臣类窘于支吾姑湫褊简陋之仍霅章侯莅鎭垢刷蠧剔俭其出勤其积再逾年而后有羡力事无如此重且急者亟濒江经营焉七星岩秀郁倔竒爲一郡最胜处若屏幛其后役起重阳而落之腊题曰咸若谓万物无不防被帝泽非止斯亭岁所放者也敞拓华壮簪绅雍容旅进退舆隶堵立其旁猝风雨有所庇潜龙藩体貌始称亭虽三间而闗系大法当得书虽然君子仁民而爱物爱出于仁而民又先乎物者也夫子恐廐焚伤人而马不问非恝然于物者齐宣不忍一牛而百姓不见保则所厚者薄矣人主履大寳位推行天地曰生之大德二千石分土而治所以流布此德也物吾与且恻然全护况民吾同胞乎毋寃系如笼闭毋横敛如竭泽毋暴政逞威怒空其室庐离散其妻子如覆巢毁卵饥溺犹已手拊息嘘使仰事俯育熙熙怡怡安乐之而不自知环千里皆放生亭讵止一林之栖尺水之泳而已哉必如是始无愧于受而爲牧之寄侯于仕学优念此已熟其勉润泽之侯名励爲端平朝士云淳祐九年二月既望
　　广帅方右史行乡饮酒记
　　田复井民必棼战复车卒必奔古之泥今之弊固也若夫乡饮酒昔人岁时讲之常尚贤而人趋善先齿而人兴逊俗以懿世道以升此礼更千万世当如一日人情往往乐简仍陋非有害而惮非小补而忽迂之曰理军市焉用彼苛之曰起伏烦碎将难行力回古风于二千余年之后岂易事耶方公大琮之鎭南服也一年政优二年化周将古饮其州命宾佐日在泮与领袖士讨论之林君公琰出一编曰此某人近岁行于泉者足证也余君震洪君天锡取其本参以仪礼颇不合共质之公公曰吾从周于是议始定宏纲细节必攷必据间一二从时宜而可以谊起人习之咸説未旬而熟谋宾介得顾君梅陈君应辰僎则常平使者王公铎愚何人斯而俾赘僎以德凉齿劣辞不获廼二月十有二日质明凡在位者颙其幅巾粲其深衣大带垂垂方屦几几以次即其席如植羣喙一默惟献者酢者酬者执其事者交际应接翼如襜如莫不舂颂中度罍洗也尊勺也爵觯也豆爼也様上世而新就歌瑟堂亷笙钟磬在下间合铿鍧夫人目盛容耳正音油然心之良驩然情之眞自以爲身苍姬之元而无昌黎不及揖逊其间之恨盛矣哉主人拜凡七十有竒爵于上篚焉取于东荣焉洗于尊前焉实于席前焉授与其他进退升降无算手献宾主人之党二百三十人辩又疑立观二百三十人旅酬辩既彻爼始膝席座以燕漏过三十刻强有力者犹不胜公静专肃庄以至送于庠门曾不少礼仪三百威仪三千所以行之者一积诚定力也礼成诸友合词以请曰吾乡乡饮酒百年几见乾道间龚庄敏尝行之惜纪载脱遗虽宿辈不可得而闻公一循去圣之仪制非碣其大略来者将安稽子与行礼焉此笔宜属之子因吿之曰夫礼者天理之节文学者所以立也平居视听言动一之或非是先失其恭敬之本而物欲得以乘之矣一旦动容周旋于广众中必周章失措防之不可揜于此可以观其人焉原其初心以众人自恕谓若昔圣人非可学而至故自弃至此亦可哀已今师帅搜讲旷典服古器古乐古岂徒崇美名侈美观要使乡人士君子秉古心行古道不睹不闻之境俨若对大宾时已克而礼复自然于其家雍于其里恭于其国也忠无一毫愧于古将见暴慢消鬬争息陶一方之民皆古观乡之效易易盖不诬唐人饮礼稍仿古耳赞皇公在常登歌降饮人犹知劝裴文献在宣歌白华等章且有泣者二子亦直声而公所学过之是举也礼乐犹备人心感发又当何如众曰唯至于度数文爲之详则有编次在此不书淳祐四年三月朔
