　　提要

　　臣等谨按毅斋诗文集八巻明王洪撰洪字希范钱塘人年十八举洪武丁丑进士初授行人寻擢吏科给事中成祖时以荐入翰林由检讨厯官修撰侍讲为永乐大典副总裁官明史文苑传附载林鸿传中称成祖尝命洪为文洪逡廵不应诏为同列所排不复进用而是集有刘公潜所作挽诗序及莫琚后跋乃言洪预修国史会大臣欲载其家瑞异入日厯中洪持不可至闻于成祖前坐谪礼部主事卒于官曾棨挽洪诗所谓玉堂分职见孤忠亦指是事明史葢偶遗也此集即莫琚所编杂文皆朴雅骈体亦工诗尤具有唐格而不为林鸿髙棅之钩摹其序文及序书二篇立论具见根柢其序胡俨诗集谓至元天厯间赵虞范掲各鸣一时之盛及其衰也学者以粗豪为壮以尖新为竒语言纎薄音律怗懘论元末之弊至为切中则洪之所见髙出当日逺矣虽名位不昌要为有明初年屹然一作者明史文苑传称王偁预修永乐大典学博才雄自负无辈行独推让同官王洪则洪之文章槩可见矣

　　瑞象赋

　　瑶光之精至和之珍彩霞之色景星之文兹其所谓瑞象而特应于我圣君者乎尔其始生也协阴阳感星辰集众灵走百神斡玄钧以凝化孕太和而萃真吸沆瀣于夜月耀华彩于朝暾郁律峥嵘魁垒嶙峋巍然而行屹然而蹲突若山峙蓊若云屯其文彩也则璀璨灼烁辉煌烨奕皎累累而含珠郁彬彬而间璧玉辉辉以凝素琼斑斑而点碧丹霞照而紫气腾玉露润而祥光溢逺而望之灿然如繁星之耀苍旻近而观之朗然如晴雪之映朝日也尔乃濯浴清波翱翔琼岛徘徊芳林咀嚼瑶草含和而鸣则嘉气合而清风生临髙而居则灵芝茂而庆云绕蛮夷覩形以奔告蕃王焚香而致祷兹实应我中国圣人之在位特生此为太平之瑞寳也于是茹以素玉之英沐以碧醴之泉縻以五色之绶航以万斛之船越沧溟于万里效职贡于九天舞蹈于金阙之下盘旋于玉阶之前瑞日照耀荣光昭宣龙颜眷而天意悦千官贺而欢声传于是时也玉京启寳殿成三阶平万国宁礼乐明备功崇徳弘四灵咸至诸福毕呈蔼苍麟之来仪雝丹凤之和鸣蔚神岳之玄兎越海陬而贡大廷出凡类而超羣生游灵囿之清髙聴钧天之铿訇荫玉树之芳华饮瑶池之清冷于以承恩顾于以昭文明亦何其幸欤不特此也固将立明堂侍法宫驾龙舆于九天之上导玉辂于五云之中登泰坛而展大礼升清庙而宣孝衷昭万年礼乐之盛应上天福禄之隆俨瑞图而绘质焕灵篇而着功然后知兹象为国家太平之上瑞而千万年之奇逢也岂徒为形质之瓌异与羣兽之雄而已哉为之歌曰干锡瑞兮坤献祯炳其文兮发光英越海外兮献帝廷瑞圣徳兮昭文明永万年兮承天庆

　　瑞应麒麟赋

　　永乐十三年八月麻林国贡麒麟翰林修撰臣王洪为赋以献其辞曰
　　惟圣皇受天命御四海盛徳与天比崇洪业与地侔大幅员六合亘乎无外统和天人茂毓庶类万姓阜业百蛮执贽和气蔼以春畅嘉应蓊而云萃于是至仁之兽粤若麒麟者自海外逺域万里而効瑞观夫麟之为状也瑰玮晔昱醇懿沕穆质振振以翘峙采旼旼而素爥性喣喣以内和气融融而外郁伟哉神乎兹之为兽葢以纪云篇而协瑞图昭文明而申景福也于是徐察其形则足若神骥体肖灵麕珑珑殷殷熿熿彬彬目若景星熠耀精荧上烛苍旻脊若灵龟昭布人文经纬乾坤又象乎神驹出荥河拥瑞云葳蕤纷纶照耀龙鳞奇伟神异不可殚论若乃禀至仁含太和于嗟趾定容与婆娑乐生物之繁茂惜嘉卉之婀娜妥柔角纡灵步回首若頋摩趾若抚昻若仰靡若俯圆中规方中矩时然一鸣应锺协吕和音翕以四达协气盎以弘布獬豸屹立以傍侍鹪鹏缤纷以后翥彼千仭之凤凰翐翩翩其下舞于是或腾崇峻之山或游泱莽之野或饮清冷之泉或荫芳华之树雨师警尘风伯清路太乙含辉以先导众神景从而为驭足之所至身之所处则晔然而矞云冷然而甘雨醴泉流川灵芝芬敷葢以昭迹逺方献祥帝都以彰圣徳周万方而洽九土于是时也羣臣万姓莫不忭跃快覩咸北面稽首曰允哉兹麟昔在黄囿帝徳以明载游周郊王道以成今兹越逺徼超沧溟登青云而献大廷厥应何从惟徳之征不然何其煌煌其神且灵也葢惟我皇神圣文武宪章礼乐率遵圣祖兹武王大孝绳祖武而受天佑也敬祀天地宗庙社稷神锡嘉福兹成汤頋諟明命罔不祇肃也钦若昊天敬授人时好生之徳期于无刑兹尧舜奉天勤民而仁育羣生也于是稽帝典考王度狩都邑抚黎庶时乗六龙肆觐羣后懐柔神祗存问父老湛恩洋溢施于夷夏六合之内八荒之外攘攘藉藉懐生之物莫不浸润膏泽昭朗恺怿同底于寿域百神和而效祉上帝悦而降福是以若兹仁兽郁乎飙至纚乎彚集扶幼示恩归昌表徳来乎遐邦以见神化之敷于逺嶷乎独角以着天下之定于一于惟盛哉且夫白鱼跃舟至微也凤鸣岐山至近也犹燎以告功垂美后胤况赫赫奕奕若是其盛者乎是宜镂黄金之牒纪白玉之册树凌云之碑扬配天之绩宣鸿禧申景锡蜚英声腾茂实使万世永永服我功烈帝谦让而未俞也训羣臣谨庶职求遗贤褒有徳笃干干励翼翼嘉旅獒美无逸放兹麟于灵囿从逍遥而自适以礼乐为华俭朴为质较邃古之淳风趋陶唐之盛迹昭乎永景命于万年而播鸿名于无极也谨作颂曰
　　帝徳天覆民之熙熙甘雨时布万物以滋符瑞众应厥惟昭兹振振仁兽贲其来思游我灵囿穆穆其仪昔闻其名今覩其来仁哉麟哉维徳致之维帝时训维敬以祇咨于臣邻以赞以毗于千万年永兹繁禧

　　观灯赋【有序】

　　伏惟皇上奠安乾坤统御华夷民物阜康率土效职诏于上元之夕特赐灯宴嘉与万国同乐太平圣恩周洽臣民欢庆臣洪幸生圣明获覩盛事而其职业又在赞述功徳谨作赋一篇铺张形容以纪盛美其辞曰
　　惟圣皇之龙飞御天也弘万世之盛业冠百王之神功义威烜赫乎宇宙仁恩洋溢乎华戎六龙时巡万国来同昭人纪凝天工嘉咸宁美时雍于是设太平之伟观与四方万国乐时和而岁丰也于时太昊司晨青阳届节瑞纪书云祯祥应雪和风初扇祥蓂半茁羲和卜日以献吉宗伯考仪而进说虞倕草图以承命鲁班董役而就列攅瓌材之磊砢起崇构之嶻嵲出巨鳌于沧瀛耸神山于金阙其宏壮也则齐云万重凌空千尺横峙霄汉髙迎瑞日蔼翠华之葳蕤蓊祥烟之葱郁居庸北拱太华西揖玉台并峙金茎对植萃和气于九重映层霄于一色于时绘饰五彩灯悬九华奇巧伟丽缤纷交加灿若聫珠烂若蒸霞焕若濯锦芬若扬葩芍药并植以竞秀芙蓉凭虚而吐花尺纾千缬寸眼万罗云母灼其含朝光琉璃莹其澄秋波彩枝婀娜以风转素羽蹁跹而云摩于是寳月澄空金钟告夕有司具奏銮舆乃出开阊阖于中天奏箫韶于空碧举霞觞肆瑶席万户启千门辟忽一蒍之辉煌倐万花之炫赫固有象乎列宿之昭回拱北辰而环紫极虎翼右拒龙角左植玄武呈析津之瑞丹雀表文明之徳帝座中尊蕃卫外翊纡阁道之崔嵬挺华葢之赩■〈艹赤〉严侍卫于勾陈昭徳符于太乙烨文昌之峥嵘焕九霄之奎璧逺而望之朗乎庆云开而耀玄文近而察之焕乎登天庭而仰宸居于咫尺也于是众乐毕陈羣臣咸集按八风调六律子野抚琴瓠巴鼓瑟师襄击磬后夔拊石然后奏巴讴发呉歈舞阳阿歌康衢回阳和于玄谷遏行云于紫虚呈犂轩戏都卢鱼龙凌云以曼衍猿猱縁木而纡徐跃飞骑于帐端舞瑞狮于阶隅或拟贩缯于都市或效采莲于江湖跳蹒跚之龟鳖走■〈负上〈负负〉下〉赑之蟾蜍覩长人之岧嶤讶竫民之侏儒山鼻胡■〈穴八夕，上中下〉拳发蛮雏握通犀手明珠笼凤凰盘珊瑚蹈厉回旋俯伏睢盱又有虞初稗官引诗说书谑陋嘲贪表贤诫愚至若童稚箫鼓以嬉戏农夫蓑笠而耕鉏父老伛偻遨游门闾曳杖步屧峩巾长裾乌■〈牜孛〉前牵黄犊后驱圣心毎悦于民事天颜屡视而怡愉鸿胪光禄奔走先后千盘传炙万瓮行酒珍羞锡海内之所无仙果出人间之未有尔乃下灵祗朝仙真引丹凤于蓬丘来青鸟于昆仑或跨苍虬或驾祥麟或乗云辎或驭飙轮华芝荐芳蟠桃献珍祝圣皇之万寿与万国而皆春鸿恩洽大礼备万国欢群臣醉仰九重呼万岁然后放金花于火树扬紫焰于云骑轰雷鼓警宸卫圣驾兴臣民退于是时也家家有和宁之乐处处享安平之治溢喜气于寰区徧欢声于天地是皆由夫圣皇恭俭勤民故天锡之以丰穰仁慈爱物故神效之以嘉瑞恩周乎覆载之所极明烛乎日月之所至忧以天下初济大业于艰难乐以天下乃与斯民相安于无事是故功髙乎三王徳隆乎五帝雅颂所不能陈简册（阙）

　　（前阙）润威洽乎覆载之所极东夷西戎南蛮北狄越山海以献珍袭冠带而效职仁风之所吹煦神化之所荡涤莫不革犷戾以柔恬服猜暴而顺适是以若兹之神兽越万里而来格涉流沙歴西极登天门羾帝室九重顾瞻以欣豫百僚纵观而抃怿是师也乃敛威弭伏含和容与足跹跹其若蹈首靡靡其若俯口喣喣其若鸣尾缤缤其若舞若翔踊而自怡庆遭逢于圣主仰荣耀于中天脱尘凢于下土于以登灵囿而游琼林饮瑶池而荫玉树聆钧天之韶音乐驺虞之嘉侣亦何其幸欤至若图形于干壮军容而耀国威登像于屏守金门而镇玉户期委质以效忠同师臣之孔武臣载笔于东观幸承恩而快覩仰圣徳之弘敷庆神物之嘉遇拜阊阖而献词播皇风于千古
　　谨按唐太宗贞观中西域贡师子其时侍臣虞世南为赋以献今圣上神圣功徳度越汉唐而臣幸以文字叨列侍从覩兹盛美不敢以不及古人为解谨为赋如右时与士大夫诵而观之以咏歌圣徳于无穷又何其幸欤冬十月九日臣洪谨识
