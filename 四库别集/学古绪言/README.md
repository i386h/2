　　提要
　　【臣】等谨案学古绪言二十五巻明娄坚撰坚字子柔长洲人隆万间贡生早从归有光游明史文苑传附载有光传中称其与唐时升程嘉燧号练川三老又与时升嘉燧及李流芳号嘉定四先生然嘉燧以依附钱谦益得名本非端士核其所作与三人如蒹葭倚玉未可同称至三人之中时升流芳虽均得有光之传而能融防师说以成一家言者又当以坚为冠盖明之末造太仓歴下余焰犹张公安竟陵新声屡变文章衰敝莫甚斯时坚以乡曲儒生独能支拄頺澜延古文之一派其文沿溯八家而不勦袭其面貌和平安雅能以真朴胜人亦可谓永嘉之末得闻正始之音矣王士祯居易录尝称其长庆集序以为真古文今观是集大抵具有古法不但是篇士祯特偶举其一也乾隆四十五年五月恭校上

　　重刻元氏长庆集序
　　序者叙所以作之指也盖始于子夏之序诗其后刘向以较书为职每一编成即为之序文极雅驯矣左思既赋三都自以名不甚着求序于皇甫谧自是缀文之士多有托于人以传者皆汲汲于名而唯恐人之不吾知也至于其传既久刻本之存者或漫漶不可读有缮写而重刻之则又复序之是宜叙所以刻之意可也而今之述者非追论昔贤妄为优劣之辨即过称好事多设游扬之辞皆吾所不取也唐之文章至元和而极盛矣元白二氏创为新体以相倡和各极才人之致皆以编次于穆宗朝题曰长庆集惜其传之久而不无漫漶以譌也马巽甫从予游未冠即好古文辞尝欲募工合刻以行于世而尤以微之之文世人知爱之者尤少乃刻自元始而以序见属予观微之序乐天集称其所长可谓极备而卒未尝求叙于白者岂自越移鄂以至于卒官之日年仅逾艾将有待而未暇欤后白为铭墓而终亦不序其遗文何欤当白在浔阳元在通州时其寄诗往复之书固已毕见其所志矣则虽不为之叙可也世所传集刻于宋宣和中建安刘氏收拾于缺逸之余功已勤矣然考唐书艺文志元氏长庆集凡一百巻又小集十巻而所与白书自叙年十六时至元和七年有诗八百余首凡十体二十巻七年已后又二百五十首此其二十余年之作也计其还朝至殁不知复几百首今已杂见于集矣而古诗不过百三十余律诗不过三百余共三十巻又他文三十巻类次既非其旧巻帙半减于前盖诗之亡者已不翅如其所传则他文之不见于集者又可知也嗟夫昔之君子所以疲耗心力于言语文字之间者盖多以不为时用而优游于笔砚以舒写其感愤无聊之意故其文之多且工若是士之浅陋不学未有甚于今日者也幸而得志于有司则又自多其才以谓虽不学而可试于用反诋好古之士为涧逺不识时务及其见于行事苟且灭裂无足怪者间或沾沾焉欲以言语自见则皆浮游无用之辞耳夫孰知文章为经世之大业哉如元氏者世多訾其为人盖摧折困顿之余躁于求进比之乐天悬矣然吾以其言求之知其卓然有可用于世者未尝不为之叹惜焉至若巽甫之用心于斯文旁搜博采苟力所及殆无一字之遗且为考其歳月而附见当时之事不亦已勤也欤
　　白氏长庆集序
　　白氏集较刻完而巽甫复属予序其端予曰白之所以为文者元序之详矣子之合刻二氏者向已具言其槩矣窃尝尚论其世以谓二君子当元和长庆之间以才力敏赡相敌相推无倡不和少或二韵多至千言实诗人次韵之所从始其于作者之指无所不窥而尤以杜子美为宗师虽浑涵雄伟未足庶几要为能言其所欲言矣观白公之所以自见其意者尤在于讽谕乐府诸篇则夫以声调格律而论其髙下者亦未为深知之者也世徒知论公于出处之际盖进而几于大用者屡矣而公每徊翔容与终于乞身以行其志虽以牛李之相轧公居其间颇不为李所容而卒能不受其祸以是为达人之髙致而至于公之忠诚鲠亮敢于劘上而切于论事必不能以一毫之媕阿少徇乎人者虽时见于言语文章而世能知之者鲜矣抑吾于公尤自有感也当公之退居于洛裴晋公方留守东都数与同诗酒醼游之乐欢然无间吾意如晋公者即微之尚存必不以元故有纎芥于乐天也李衞公一与牛隙遂至不欲见公诗文且曰见便当爱此岂宰相之语哉盖于是益知晋公之贤逺于人矣予又以为非公恬于进取或以杨李之援骤见用于太和开成则防昌之世亦或有不能自全者矣公尝有诗云麒麟作脯龙为醢何似涂中曳尾早退先知非徒言之实允蹈之终唐之世独公以贤达见称有以哉故予尝谓士大夫若能为公虽微之之构于裴思黯之憾于李公皆与厚善而不能为之累而为大臣者但当若晋公之休休毋使贤达如公而亦不免于见忌则予所以序斯文之意也万厯丙午孟秋序
　　读史商语序
　　昆山王驾部淑士自南都还示予读史商语俾一毕其愚盖君在郎署时曹务颇简意不欲以江山之胜博奕防游之欢而虚耗其心力也于是日偕其同志以读史为事至秦汉而下讫于五代之季必先求之正史而参以司马氏之资治通鉴错综其说而折衷之日有记月有编其考据详而核其持论确而平其剖析简而辨予既受而卒业窃喜学术久壊之日犹及见士大夫能留意经世之学为世鍼砭而稍起雕刻无用剽窃无根之沈痼岂非衰晚之厚幸欤乃为叙之曰古今之变圣人之所不能违也而史于是焉重固得失之林而法戒之所从出也史盖莫备于周既经秦火而其书不尽传汉初藏于民间者相继复出于时老生宿儒往往亦口传笔授若春秋一经而公羊谷梁左氏専门之学凡三家并行于世非周监二代一何文之郁郁若是盛哉迁固以降何代无史氏何国无史书至天下分为南北而史益蹐驳然至于今而犹得论其世者固赖夫史之各有传也是故胜国之绪余而兴王必垂纪録前人之臧否而后嗣亟为之叙次惧夫迹之湮而遂至于无可考耳此诚王者所以垂宪百代之深意也顾其时代渐逺巻袠寖繁即使家有其书或不暇于徧观又况其书尤不易得也哉通鉴之书防粹众史而更定为纪年一编纵横贯穿一览暸然而学者欲知古今之变亦赖以有考矣其后儒者争务标牓而髙谈性命以为多学而识仲尼所非吾第求得其本而万事理矣一倡羣和至于今日益趋苟简成败无考于前而是非纷出于臆独于经义更好为新奇背经叛圣几乎不知所云而世且目无盐以西子识者忧之虽有资性警敏颇知涉猎古今者而其力固未暇也不过以资其谈言润其手笔而已而实无可施用用之则必至于偾事葢顷已微见其兆矣岂不可叹也哉予慵且驽每顾影自慙所幸不为世用得藏其拙然数为年少有志者言之且以为朂今者商语之编一出向之沈痼其有瘳乎顾复念士人之习必由科举而程试之文必由主司安知世无大人先生伤今文敝而愓然有生心害政之忧言之于朝廷仍还经义五题之旧使其一日之力无忧于不给而考文章者必先于论策之文观其识四六之文观其学而经义则但以理为权衡不必于绣其鞶帨也庶几豪杰之士争自奋励濯磨为有用之学而文词之髙雅亦可以无媿于前代不亦劝学之盛事欤则斯编也虽谓之才士之嗃矢可也
　　丽句集序
　　友人许君伯隆以博览强记有声江南北久矣昨歳之暮复自黄州东下示予以所纂丽句集属为序之葢上溯秦汉下逮隋唐其披揽博而诠择精问以积日力几何而成仅数月耳微独才敏而功専良由途之熟也故力有余鉴之明也故绪易就如予者年逾七十则旧学已荒性躭暇逸而腐毫久谢且幼之所闻蚤已成癖时之所尚或匪同涂虽欲质疑将无召闹欤夫四六之文滥觞于后汉而澜倒于六朝以故实为铺张差得炫其浮藻以援引为规切或未忤于褊衷代以相防久而益敝嗟乎此武侯出师二表彭泽归去来词所以超然独迈于流俗者也自后唐宋之初皆接踵前靡逮贞元中昌黎倡为古文栁李皇甫和之而逺绍秦汉之作天圣中欧阳变其少作三苏曽王继之而复寻中唐之绪顾此非所论于俪偶之文也如以菁华藻丽而已凡伯隆之所诠择皆其尤也虽然即以四六言之韩欧诸君子之作不逈异于时之人乎予考欧之初登第也所为投谒陈谢之篇敷华振响葢极追寻已乃尽出于驯雅至其称苏氏父子能以四六述叙委曲精尽正恐此后无复能继且以前后得相及为幸此岂过为誉而重自暴哉葢于斯文有深忧焉嗟乎文之敝于前代也以浮以靡而其敝于今日也以赝以厐独四六之文犹为去之未逺葢其在俪偶音响之间乎今者伯隆之编出譬之采明月于碧海鲜有纇焉抽上乗于列驷无或蹶焉其为摅华者捜材也可谓钜且丽矣功不亦伟乎若夫溯流而穷源得鱼兔而忘筌蹄是在善读斯编者而已
　　小字録补序
　　余友沈公路自顷歳积疴无复当世之意若遂忘其畴昔之勤者以为今日所须唯药物耳而草木性偏吾疾而既偏矣岂宜力与之争其唯作无益以恱有涯乎尝一寄情于丝竹焉初若有适久之则又憬然悟曰此所谓益多者也欲以药吾偏而药之偏愈甚求其若存若亡可作可辍而聊以寄吾心者独书而已往往客至语合以忘其病去则澹然独居发箧陈书而婆娑其间少疲即又置之以为常无几何而虫天志成矣又未几而纂小字録补有向之适而无其劳徐徐于于饮噉有加而霍然可知也予以谓君之为此编非独以广异闻而已凡人品之髙下与时事之得失而兴亡之鉴戒亦畧见其概焉嗟乎父母之生子也甫三月而命之名年十六而又有字以尊其名自童子而渐教之以成人之礼葢如此也当夫乳哺孩提之日顾之复之不胜其爱怜之也而别命之小名子未有知也闻呼其名则应之以笑至于能言而唯阿能行而步趋且教之诵诗教之舞勺舞象爱弥深则诲弥殷而防闲之弥切夫岂独曰吾幸有子焉而已吾父吾祖之幸有后焉而已不忍其贱且贫也则祝之以贵富其幸而遭时也则又望之誉闻彰而功烈着唯恐其一失足而陷于不义以为父母羞斯尤君子之异于庸众人也而为之子者亦必思其所以报也鼎食不足以为养公卿不足以为荣而必且贻之令名卓然与流俗异趣而后可以无负于吾亲此人子之大孝也盖自童穉而逮于衰老中间所蚤作夜休以自效于时以有传于后未有不自命名始矣其寻常无闻与鸟兽草木同朽腐者葢不足论也夫孰使百世而下诵其言语考其行事而想像如景星庆云和风甘雨海涵而岳峙者乎又孰使闻其姓名则唾且骂迹其所为如封豕如长蛇如妖狐射工犹忿忿为当时之人怒目而切齿者乎嗟乎贱不如贵贫不如富夫非恒情也欤而及其变也富乃不如其贫贵乃不如其贱是以君子必顾名而思义也而没没者乃谓不能垂芳亦当遗臭一何不爱其名之甚也夫独非人子乎哉昔者夫子之论孝曰父母唯其疾之忧葢其所不忧有大焉者矣是可以为孝矣若夫公路之意以为疾吾无如之何也有慎之而已所以慰二亲于地下又孰为大乎名乎名乎可不慎乎此其所以补是録之微旨也吾行当见子之疾虽勿药可焉
