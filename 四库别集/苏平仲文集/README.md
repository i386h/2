　　提要
　　【臣】等谨案苏平仲集十六巻明苏伯衡撰伯衡有空同子瞽説已著录是集巻首有洪武四年刘基序而集中厚徳庵记云庵成于洪武壬戌十二月则是记乃洪武十五年以后之作基所序者尚未定之初稿也又集末有洪武八年胡翰跋谓伯衡选为太学六年考明史称伯衡以丙午嵗为国子学录伯衡所着国子学同官记称以丁未陞学正其诗又有庚戌七月十日奉命编摩国史口号则伯衡由学正擢编修实在洪武三年上距丙午仅五年翰与伯衡同时所叙不应有悮或史悮移后一年欤集为正统壬戌处州推官黎谅所重刋宋濓序称其不求似古人而未尝不似又明史文苑传称濓以翰林学士承防致仕荐伯衡自代称其文词蔚赡有法殆非虚羙郑瑗井观琐言病其用意太苦遣词太繁缛不可为法则过髙之论矣乾隆四十一年二月恭校上
