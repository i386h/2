　　紫山大全集二十六巻
　　【臣】等谨案紫山大全集二十六巻元胡祗遹撰祗遹磁州武安人元史本传载其字曰绍开然今民将在祗遹乃文考绍闻衣徳言实周书康诰之文核其名义疑绍开当作绍闻元史乃传刻之讹也中统初张文谦宣抚大名辟祗遹为员外郎后官至江南浙西道提刑按察使延祐五年追赠礼部尚书谥文靖是集为其子太常博士持所编前有其门人翰林学士承防刘赓序称原本六十七巻歳久散佚今据永乐大典所载裒合成编厘为赋诗余七巻文十二巻杂著四巻语録二巻其间杂著一类祗遹一生所学具见于斯然体例最为冗有似随笔劄记者有似短章小品者有似莅官条约者有似公移案牍者层见错出殆不可名以一格考贾谊新书皆以所作治安防及言事诸疏割裂颠倒各自为章别标篇目説者以为平时记録之稿其后联缀成篇而上之祗遹是集或亦是例欤史称其官右司员外郎时以论事忤奸相阿哈玛【案阿哈玛原作阿合马今改正】外迁太原路治中提举铁冶欲以岁赋不办责之及其莅职乃以最闻官荆湖北道宣慰副使时辨诬告不轨之狱官济宁路总管时擘画军政八事并修明学校之法又称其所至皆抑豪右扶寡弱敦教化厉士风盖以吏材名一时而无一语及其文章今观其集大抵学问出于宋儒以笃实为宗而务求明体逹用不屑为空虚之谈诗文自抒胸臆无所依仿亦无所雕饰惟以理明词逹为主元代词人徃徃以风华相尚得兹布帛菽粟之文亦未始非中流一柱矣惟编録之时意取繁富遂多收应俗之作颇为冗杂甚至如黄氏诗巻序优伶赵文益诗序赠宋氏序诸篇以阐明道学之人作媟狎倡优之语其为白璧之瑕有不止萧统之讥陶潜者以原本所有姑仍其旧録之而附纠其谬于此亦足为操觚之炯戒也乾隆四十六年九月恭校上

　　紫山大全集原序
　　文章以气为主其为气也至大至刚以直养而无害则塞乎天地之间赓读紫山胡公文集见之矣紫山公自号也讳只遹字绍开磁之武安人大父嵩泰和名臣考徳珪正大四年进士儒林郎主耀州富平簿北渡后公未及冠便能从诸生习为程文非所好也潜心伊洛之学慨然以斯文为已任一时名卿士大夫咸器重之中统龙飞鋭于求治张左丞文谦宣抚大名辟公员外郎入为中书省详定官至元改元诏立学士院以应奉翰林文字被召俄兼太常博士百年坠典一朝而复天下翕然称之以为礼乐文物尽在是矣中书以公优于为政辍充户部员外郎寻转省左右司直言正色无所顾忌重忤相臣意出为太原治中外示优容内实中伤之会改河东提刑副大使而罢省幕同僚有骤至执政者终衔之江南下调荆南宣抚副使十九年权奸事败庶政一新抡选牧守授济宁总管考课为天下最擢山东宪使齐鲁风俗为之一变就拜翰林学士不赴又明年提按淛西平江税司逻卒恐吓市民赃露公决遣之行省颇有违言即轻舟还相下筑读易堂以居若将终身焉二十九年制诏以耆儒硕徳征凡十人公在第一辞以疾不起时年甫耳顺矣卒年六十又七书法妙一世脱去翰墨蹊迳自成一家唯鹿庵紫山两公而已平生著述易解三巻老子解一巻诗文号紫山集者六十七巻公薨二十年赓以事道过彰徳其子太常博士持将锓梓以夀其传恳以序引为请赓以不敏辞迨四三年而请益坚呜呼赓尝师事鹿庵先生得告还东平前诸生谓公曰敢以是数后进累吾绍开且命之罗拜公避之鹿庵良久曰以师友之间待乎公遂诺焉赓才力谫薄获与搢绅之列残膏剰馥得公沾丐者多矣此意岂可忘哉公之出处行已大方有野斋秋涧所撰神道碑祠堂记在感念畴昔非敢以为序也姑述其梗槩云尔延祐二年重九日翰林学士承防荣禄大夫知制诰兼修国史门生刘赓序















　　钦定四库全书
　　紫山大全集巻一　　　　元　胡只遹　撰赋
　　孔子手植桧儒冠赋
　　昔余年之总发兮恭承先生之严命冠汝字汝兮毋坠汝祖曾髙之家庆不弁以武冠儒服儒兮敬之哉终此身于仁义中正蹈道之士充之则贤庶几要归乎大而化之之圣荏苒于兹四十年余种种乎素髪之垂颈吁长嗟立徳立功负我先君之志兮日惭怍乎临镜旦旦理髪整冠以自警兮追训言于无声之聴慎尔话言敬尔威仪矧承时之人许身以从政徳不称服兮何异乎犬羊其质而被虎文之炳赫赫师尹民具尔瞻兮一貌一容尤不可以不整羣目所视众手所指兮持身于日恭而日敬今晨何晨有客叩门袖木寸余曰此孔子之手植桧也千五百年贮云含雾而雪霜莫能悴也物各有数火于贞祐之兵鬼神莫得而卫也灰洽烬飞蟠根九地发土得之坚若金石真可寳而不可弃也嗅圣手之泽香润袭人断而为冠是可戴而不忍佩也一簪华发万古清风殷之弁周之冕邈乎其不可类也貂蝉恵文进贤白编卒然遇之恐退缩而失其贵也三沐三薫洁我衰鬓悚乎若圣手之遗也希圣之徳辱神之恵振尘而増叹喟焉乱曰首可餙兮发可理兮心难乎其无愧兮勿徒矜庄扵外而废弛扵内兮毋状类古人而黾勉以随世兮毋盗跖其躬而掩宣尼之袂兮岌岌峩峩髙自标置为欲所驱而沁沁伣伣低头服膺卑屈扵利兮冠不汝误汝无为冠之累兮
　　五言古诗
　　拟古
　　矫矫千里龙举足踏空濶脱人扵险艰四海不难越安能陪驽骀从人求莝秣堕地为丈夫造物不吾薄安忍自暴弃黾勉同琐屑随世覔温饱呵手惧氷雪百年何足荣万古名不灭
　　古意
　　俗儒束名教促缩如寒奚知乐道趣强勉轩须眉短褐不掩胫破甑不疗饥朝诵夷齐言暮取顔氏师妻子虽嗷嗷敢履非义机不见窃国侯夀考厌轻肥
　　懐古
　　一受其成形便为气数拘冬龙与夜萤潜飞时自殊小智昧安命谈贤愚三仁值殷衰一身不自濡汉兴贵公卿胥吏杂屠沽后世看成败美誉归亨衢啧啧史臣语烈烈雄丈夫万事尽偶耳百虑良区区春和散膏泽贱草皆敷腴秋芳结寒霜幽兰例雕枯适来勿轩轩适去毋吁吁皇皇东鲁圣嵗晚赋归欤
　　前题
　　蛟龙布雷雨蚊蚋竞腥羶君子与小人志趣天壤悬苟无东鲁圣孰识箪瓢贤智者忧其贫愚者嗤不前岂知回也心不在爵禄邉诚欲富且贵执御仍执鞭枉寻而尺直未冠乗华轩舜饭糗茹草若将终身焉不遇陶唐君佳哉厯山田前圣与后圣百世同一天曽防岂狂者咏归忘华颠
　　读书有感
　　古人不可留遗言在糟粕诵言欲求心遐想亦绵邈注释万千蹊适从欲畴托十行夸一目对巻已纷错孜孜一书奴记诵务该博众来谘至理有舌莫扪防书生亦良苦深念堕冥漠头白眼昏花亷隅伴寒薄壮哉班定逺投笔开寥廓西域四万里俊功超卫霍笳鼔归京师吊死殊不恶
　　读易掩巻静坐有感
　　狂阳气渐衰利欲心已释闭闗得静坐怡然有余适眼眀观万变气机不少息智愚贤不肖孰得竟孰失气数苟未尽善恶纷鬭敌蚁穴刘项兵蜗角蛮触国风霜十月交万动入防寂人生百年间如白驹过隙穷通互有命安用相剑防胡椒岂须多厚积八百石
　　读书有感
　　余生堕习气老眼尚爱书开巻初明了乆视成糢糊下欲穷至今上欲窥唐虞下上数千年治乱牛毛如孰是与孰非错乱杂紫朱欲辨谁与语抚巻増长吁读书求悦心自笑乃尔愚请观大地形脉络各异区中原多平田要荒因﨑岖匹夫耔百畆四海多荒芜
　　读书有感二首
　　人谁不看书看书多不细看书谁不细了不得滋味既读圣贤书当见圣贤意凡圣本同心气禀或明昧读书既开悟勿为人欲蔽既习而又习致力复致力初不无勉强乆乆自顺遂欲罢反不能于何劳智慧
　　朝野有新言例自比豨狗斯言固卑猥情实不诳口啜诃何足羞防侮不知丑见食还复来捶挞暂趋走人生至于斯百恶无不有斯人亦居官政法谁与守
　　读论语自警
　　人心本无病致病皆人为私欲生百疾圣人乃良医用药各对证疾去心融怡君卿大夫士周纲断维持奢僣至倾夺习熟不为非礼乐存虚文虚文亦崩隳刑政遗余迹余迹亦陵夷一时为学人良心防昏迷受业七十子气禀各不齐圣眼如日月照见毛髪疵随问随答应满意饱载归论语书一编我将类町畦各人言行下病药两相随一一反自省有无此欠亏一言即一药服之有余师
　　读汲黯卫青传有感而作
　　整冠见汲黯踞厠呼卫青敬侮固云泥用舍谁重轻一死淮阳守一握百万兵起塜象祁连至今与云平严惮日踈逺玩狎日尊荣所以皁流长作贵公卿世无鲁仲连世无严子陵天壤一蝇蚁万古争羶腥
　　题恵崇芦鴈图
　　六翼已水滨孤鸿下寒云念羣与得侣伸颈声相闻禽鸟有此乐贤者防以文我无如川酒满堂会嘉宾我无及人徳孤陋常寡邻巻中题诗客十九余交亲宦逰各分散不得偕晨昏何时一聚首拂拭襟裾尘道义共讲劘文词同细论灿若徳星聚谟谋奉严宸
　　题王晋卿扇
　　举世附炎热此心独虚凉隔离云日赤障尽泥沙黄得手方进用避色能退藏清风万万古愈乆得弥彰
　　渔樵图
　　何代无巢许混迹樵渔间君臣非唐虞明照不容奸公论一脱口万古埀不刋吁嗟肉食人例以愚夫看真伪若白黒潜隠良为难所以戒慎独机心即天闗画师岂无意庶几消凶顽无声董狐笔见者宜汗顔
　　题韩氏异花记后
　　居室出善言千里即相应扩推恻隠心草木被仁政神鉴良昭昭聨翩生瑞庆天葩出鳯雏五彩相辉映苟非积实徳此福何来并恨无琼瑰辞洋洋入歌咏刻石大伾山千载去思颂
　　读庄子
　　防庄谈自然未必皆背道静中见天机安肯同世好纷纷知与才私欲助贪躁九败博一成暗堕山鬼笑区区是非言万蝉同一噪祸福定谁司一一冀酬报寓言岂无情但恐伤名教
　　至元癸未秋八月二十有一日书陶诗数首至羲农去我乆一篇寄懐于言
　　人皆叹衰老我老起欢心自从嗜欲浅颇觉天机深抱我天爵贵浮荣不能滛琅琅圣言外羣虫费秋吟百年难处事一朝失嵚岑耳目愿迟衰中和盈我襟少倦即掩巻有酒时自斟
　　衰老少睡眠未晨即坐起僮仆诚多倦我心私自喜壮年嗜欲昏日出梦方美囬思昔日欢一一尽堪耻立徳与立言重任失行已亦随市井儿车马红尘里醉生或梦死趋向何庸鄙己徃不可追来者犹可悔五十七年非今许能一洗
　　读陶诗因以自戒
　　东坡晚流寓始喜读陶诗使君未失志陶亦田父辞趣向苟乖隔好恶等棼丝在公犹未免何讶轻薄儿冥心味前言勿易生瑕疵
　　公退静坐读陶诗有感而作
　　仕宦二十年翛然一寒士公退饱蔬食琴书伴幽思轩轩同学友笑我懦无志富贵岂无心得失良有自流芳与遗臭二者非吾事已愧抱闗人吏部知名字
　　冬二十七日翰苑东斋读韦苏州诗有感而作
　　赋分嗜读书己与清苦隣况复学吟咏安能登要津终嵗文字间矻矻劳君心幽懐只自知究竟谁见闻西风嵗云暮白髪日益新寸心不自降尚负知经纶沍寒鬻絺绤东辕日求秦片言获金印英哉当路人
　　郭西种稻
　　种稻逾千区锄薙亦草草今年大有年穂实颇坚好僮仆笑有言蔬食长不饱黄腴含玉粒慰我色枯槁安知乃翁心朝饥何足道得米可十斛急送酒家保
