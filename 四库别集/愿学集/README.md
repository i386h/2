　　○三言古

　　简会中同志

　　○四言古

　　别友

　　迢迢觅道与道背驰归而求之自有余师朋友切切兄弟怡怡大心体物敬以自持实有诸已信予匪欺

　　仁文书院

　　父母生我耳聪目明能思能索手持足行至道难闻宁虚此生乗时勉学敢告同盟
　　其二
　　先立其大在识本心时时黙识海里寻针把柄在手点铁为金有鹤在林遗我好音
　　其三
　　大明中天千载一时无叅意见无涉支离卓彼先觉示我良规上帝临汝心实严师
　　其四
　　英贤济济聚首一堂商搉疑义剖晰毫芒凤毛麟趾规圆矩方日新月盛笃实辉光

　　家训

　　诗咏多福易言余庆积善之家罔不繁盛眇予小子厕名士绅愧无实德裨补君民未能治国愿教吾家敷诚布衷寂听无哗凡我宗人无忽予言洗心涤虑培根达源敬奉天地孝养双亲与其浊富宁守清贫勿利货贿嘱托上官小民呌寃尔心何安输赋无讼迹絶公室尊宪守约终鲜差失里闬姻党情谊无涯富贵轮流转眼虚花出入以度惟公惟平夜半叩门尔心不惊永言孝思世德作求我语谆谆我心悠悠毫厘不差神明临汝良心不昧三复斯语

　　黙庵和尚

　　莽莽谭禅谁否谁得有喙三尺横恣胸臆真学道者先须一黙甘苦自尝无徒耳食

　　书刘生卷

　　手开斯卷剖晰玄微芝兰并馥珠玉聮辉愧我不学启口知非冥心冥心尚杜德机
　　圣逺言湮浑浇淳漓意真词繁道德背驰问子痛痒子必自知曰孝曰弟归有余师

　　示昆弟

　　告我诸弟予猎浮名忆昔负遣万死一生圣主恩深归卧溪衡萧萧华发絶知屏营人间毁誉羽毛斯轻惟此簧鼔世比蛙鸣水田之麓可以躬耕岂无好我示我周行

　　勖从子

　　勖哉诸子无汨尔真诜诜轮轮难得者身修身维何蹈道履淳彼佻者子自堕沈沦躭于酒色比于匪人时移运失受此艰辛光阴驹隙黾勉青春纵尔不思当念尔亲

　　○五言古

　　题许给谏广易通

　　归善杨少宰卓哉百世贤一从观化后南望几呜咽挺然赖夫君研易续新编易道通三才谁能得其全明珠并瓦砾随地有真诠达者须识取一勺即深泉矫揉非大道神圣以天然寄语会心人素琴本无弦

　　复真书院覩雪蓬旧隐处

　　不见雪蓬久今见雪蓬屋图书留四壁白驹贲空谷髙风人去逺遗响振林木

　　别曾舜征仪部

　　结发为兄弟恩爱永不离羽翮既差池几度长相思一见一回老悲喜不自持所嗟津途逺问道何多歧前驱日已暮策足可迟迟
　　君处衡之南我处湖之西悠悠一水间辽阔渊与泥言笑方衎衎征马苦长嘶一间若未达犹堕千刼迷莫年戒在得圣训岂无稽

　　涉沅湘歌

　　浩浩湘江水终朝不断流天地有穷处水流无时休东没千顷田西荡百沙洲桑田曾几换厯尽几春秋青青红颜子忽忽尽白头问之何能尔光阴不蹔留试看臂鹰手力能取公侯一朝忽倾否白骨委荒丘俯仰天地间身世如蜉蝣堪怜昏昏者拘拘如楚囚所以洙泗叟耻为寝食谋逝者叹如斯川上坐相求所以碧玉老一生阳春楼背汗虽未收木绵被重裘先哲开我蒙惟有濂溪俦汉唐不知此思之仍含羞学术苦未力终夜犹多愁万物皆备我汲汲当猛修典籍如汗海役役如倒牛吐辞即为经莫把六经抽白云满磵户归袂风飕飕顺理为穷达无心羡勲猷撒手悬崖坐海岱一腔收可生亦可死身外复何忧

　　有儒生思南胡精邵尧夫数学持李同野大叅书谒予赠之以诗

　　圣学本无言言者即不知大道本无思有思属外驰荣华寻常耳今古更共之贫贱信所遭富贵亦有时鹍鹏元万里鹪鹩自一枝万里元非逺一枝亦非卑嗟嗟寰中士扰扰竟何为局蹐一生耳徒令达者嗤置身五行外庶不负男儿君更勿念我久矣玩庖羲

　　叅知蒙山陈先生八十

　　示郑天台山人

　　身在三才中学超象数先有相终为幻无为本自然问予何所事饥餐与倦眠流云闲出岫栖鸟倦飞还万物皆如此谁能悟真诠我欲杜吾口忘言玄更玄

　　独对亭书怀次王阳明先生韵

　　几从江上过危峰坐中见清秋披蒙茸始得陟崇巘诸贤聚一堂图书已识面忆昔迷歧路困衡不善变博文并格物留情经与传于今两寘之深荷圣衷眷古人弃糟粕用为来者劝素琴本无弦了心何足辩

　　雪中偶吟四首

　　人皆笑拙宦我独乐有余浮气日以敛性天日以舒名者不祥器况复窃髙誉洗心力澄汰庶不负居诸
　　其二
　　嶻嶪白云司乃在玄武滨湖中有佳果四时常鲜新昔年产明珠灼烁光四邻今乃伏泥滓一望常悲辛堤上松愈好日暮隐龙鳞
　　其三
　　予性本刚方予度甚汪洋片言切名理服膺讵敢忘泰华难尺度沧溟岂斗量嗟彼夸毘子咄咄徒刚肠
　　其四
　　我钦邓宗伯髙斋絶轮鞅贫难供菽水潜身卧草莽一朝膺特命贤闗振韶响簉迹夔龙间游心羲皇上学问窥天人玄珠得象罔念予性孤贞髙轩时来往拟卧西山曲疑义共欣赏予本澹荡者一见心惝恍

　　杂兴简同志

　　予性本跌宕投闲性所宜济济青衿士设礼强为师忆昔文中子抗颜年少时予学本蹇浅模范无足仪古有青蓝语辞让亦何为斆学元相长矢心共提撕
　　其二
　　大道茫茫尔孝友是根荄利欲戕贼之正路成蒿莱濯足沧浪水轩举泰山隈髙卑皆自取孺子得予怀人生多陷溺平旦良心开良心虽几希达之贯三才
　　其三
　　洙泗微言絶濂洛扬其徽我明诸大儒鲁阳戈正挥世儒纷口耳中心昧是非泰山千仞冈予其欲振衣鸢鱼察上下至理无显微冥心复冥心谁其杜德机
　　其四
　　大镛列东序横击乃有声彼物为妖者不叩而斯鸣孔门称善问竟以大儒名有疑斯有问有问斯乃成教然后知困予任良匪轻
　　其五
　　昔谒庐山师一见相提携嗜利蔽天性【师告予云喜子澹泊予见好利者如下地狱不得出头】斯言予诵之父母同天地因利失孝慈兄弟本同气因利失和怡贫乃士之常颜宪有良规愿言二三子冰霜慎自持
　　其六
　　彼美池中鱼队队何活泼彼美林间鹿一草相呼掇宇宙原同胞八荒皆我闼矧我同心人忍自相秦越迢迢千里道得侣斯不躠为语同心人斯道相濡沫长吟伐木诗慰我中情豁
　　其七
　　涧水何莹彻仁峰何巃嵸弦诵比邹鲁先辈多淳风处则躬孝友出则懋清忠史书扬其烈髙风激予衷年来古道丧随俗而圆通我资禀庸愚耐可守颛蒙藩篱须自固一德以初终
　　其八
　　诸生事占毕朝夕圣贤书六经何浩瀚汗牛与充车古人骨已朽言语皆土苴吾能得其精经史糟粕余斲轮解我意请观无极初年华如流水勿自负居诸
　　其九
　　文章千古事壮夫比雕虫雕虫虽小技斯文岂易工今人竞词藻古人性灵通性灵既以通源泉滋不穷六经文章伯无语不鸿蒙笔可叅造化谁与领春风
　　其十
　　圣贤亦有过此心在自知但能洗心改便是得师时昔为屠沽子撒手证菩提昔日蒙不洁沐浴帝监兹仲由喜闻过卒为百世师人心良炯炯无为蹈自欺

　　闲居杂兴

　　尘途争捧檄皇皇曰为亲苟遂犬马事而坠百年身曾子能养志颜生重归仁岂无负米者扬亲亿万春家有圣善母七十共艰辛容我衡门老朝夕共鲈莼
　　其二

　　其二
