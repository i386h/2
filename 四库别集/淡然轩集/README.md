　　提要

　　臣等谨案淡然轩集八巻明余继登撰继登字世用交河人万厯丁丑进士官至礼部尚书谥文恪是集分奏疏二巻序记三巻志铭及杂文二巻诗一巻继登卒后其友人冯琦选而刻之继登当神宗朝以灾异屡见上疏极陈一切诛求开采之害民者又请神宗躬郊庙册元子停矿税撤中使时将讨播州杨应龙因请罢四川矿税以佐兵食复上言近者天地人皆不和怨毒凝结臣子不能感动君父故天地人皆以非常之变警悟陛下不可恬不为意云云语皆切中时弊其疏具载此集中诗文则应酬之作未免失于刋削然大抵平正淳实无万厯中佻薄之习亦尚不失典型也

　　修史疏一

　　为纂修正史事臣惟正史之体纪表传志悉备纪志以载国家之政典但稽考于朝廷即可备史臣之纂述列传以载一代之人物必博访于四海乃可协众人之公评伏蒙皇上先允辅臣陈于陛之请勑谕儒臣纂修夲朝正史以垂启佑又允辅臣王钖爵等之议行令郡国购求海内典籍以备纂修德意甚盛臣以庸愚滥厕副总裁之末自二十二年八月内开局以来日与诸臣在馆翻阅中秘副録搜求故实将以次辑为志表帝纪等书无容别议惟是自开国以至于今二百余年间祖宗道化之隆国家人才之盛应运而兴光映后先其应载列传者不可胜数矣苐人众则其势必涣时乆则其事易湮若全凭秘録之隠括则志其畧未志其详欲兼采野记之纪闻多传其疑非传其信苟非闻见之博安能纪述之公伏乞勑下礼部移文直隶并各省提学官转行所属各府州县地方其有硕辅名卿德业彰著者首先辑録此外有立朝大节生平卓行表表在人耳目者不拘官秩崇卑一并收采至于名儒良吏孝子贞妇与夫隠逸方伎之流可备纪载者逐一查访或详録其家乗或博询之乡评务要明白开具当时行事实迹编成书册送部备考若有生不齿于时殁无闻于后者不许徇情滥褒致淆公论定为期限载入考成庻臣等得以藉手次第编摩不至躭延岁时虚縻廪饩矣奉圣旨礼部知道

　　修史疏二

　　为俯循职掌敬陈修史一二未定事宜伏候圣裁以昭大信以光大典事臣惟代之有史攟摭故实偹载册书明示将来用垂法戒非一人之书而天下之公也非一时之书而万世之公也故是非虚实之间子不得私诸其父臣不得私诸其君而后可以言公今诸臣奉命纂修首先帝纪行且就草矣臣窃即帝纪而言有不可不自为一纪者有可不必为纪者不可不自为一纪者建文君是已可不必为纪者恭穆献皇帝是已夫建文之号当复诸臣详言之皇上已允行之矣然附载太祖髙皇帝纪之末终觉未妥臣尝考阅实録思成祖所以改建文五年为三十五年者盖縁即位之初欲以子继父为名不欲以叔继侄为名故为是权宜之举耳非有明诏革除之也乃后修史者不逹圣祖之意遂于建文元年以后书其年而削其号并削其行事之迹故此数年间独纪靖难事而不纪所靖之难为何事若有所曲讳者今幸已复其号似当搜罗故牒采集传闻详载当时所用之人所行之事别为少帝之纪是是非非明白无隠则靖难之兵有名圣祖之心益白夫以圣祖所不自讳所不必讳者而臣子乃欲强为之讳非以天奉圣祖也且事须有实直道难枉今野史所记已多失真若不及今明为之纪令后世以乆愤之心信传疑之语则史臣之失职不足惜如圣祖何臣谓不可不自为一纪者此也至献皇帝积德累仁笃生神圣遂垂万世不拔之基然其发祥之长亦不过如德懿熈仁四祖止耳四祖不纪而独为献皇帝立纪岂惟臣子之心不安即皇上之心亦必不安岂惟皇上恐献皇帝在天之灵亦且有踧踖不自安者矣况纪号献皇帝而所书者乃弘治正德之年纪名为帝而所载者乃藩王之事皇上以为于义于例当耶否耶且自古由藩封入继大统者非一人试考史书未见有为其祖父作帝纪者彼非不能纪不欲以私害公耳臣窃以为四祖事只宜附见于太祖髙皇帝纪之前献皇帝事只宜附见于世宗肃皇帝纪之前或别起一例亦宜与列圣帝纪有别庻使名实不紊体裁不淆不然以献皇帝之子孙臣庻欲纪则纪矣如天下后世之公议何臣诚不忍使万世之下有非议圣朝者谓可不必纪者此也凡此皆非臣下所敢擅定者伏乞勑下辅臣会同总裁纂修诸臣商议允当上请圣明裁决行令载笔之臣有所遵守天下后世颂皇上之至公无私如天如地真有髙出寻常万万者而于正史为益光矣

　　捐俸以资赈济疏

　　臣等近接邸报见九卿大臣辞俸助赈荷蒙皇上俯从其奏且明谕五品以下不许辞臣等窃思人臣之立朝也若诏禄计官固分厚薄至同忧共念何间崇卑况以势所不及而言四品以上俸资虽厚岂能遍数州数郡之民以势所可及而言五品以下禄入虽微亦可活匹夫匹妇之命值此甚灾之日又当极匮之时皇上忧民且为旰食宫闱悯旱皆已助赀凡在臣工敢安常禄且臣等职居禁近既未尝厯郡邑睹艰难情状而绘郑监门之图官守闲曹又无能亲职事得黾勉勤劳而效富青州之策执经累岁启沃无由待诏频年优游何补即无事犹当夺禄岂诎时乃可素餐因此不自揣量愿捐俸薪以佐涓滴伏乞圣慈勑下该部淮照在京堂上官一体辞捐则臣等愚心不愧愚分少安矣奉圣旨准捐俸戸部知道

　　捐俸助工疏

　　兹遇皇居鼎新工力浩大阁部诸臣各疏捐俸以济工役臣职列闲曹乆叨岁俸在无事之日尚且深愧素餐岂兴役之时乃敢坐安常禄即较大臣之俸不无厚薄而原臣子之心何分彼此愿照例捐俸一年少资工费宦迹靡定安敢远及岁时禄入有常窃欲少输涓滴伏乞圣慈特勑戸部将该支俸银扣解工部收贮应用以见臣子不敢安常之义不忘趋事之忱奉圣旨戸部知道

　　火灾疏一

　　为遇灾修省恭请圣明临御讲筵并献愚忠以备采择以回天意事兹者干清宫灾臣伏睹皇上明降手札思愆咎之未尽深引过以自责是皇上警惧悔悟之心至真至切数年以来臣下所欲得之于皇上所不能必得之于皇上者而天出灾变以感动之矣夫皇上英明果毅具尧舜之资近乃不乐谠言似未扩其舎已从人之度者特圣心偶未之悟耳一旦翻然猛省惕然觉悟必不敢复以莫违之成心忽难谌之天意是舎旧图新之大机也昔人有言台諌所不敢言者讲臣得言之臣待罪讲幄叨糜廪禄平时既无能启沃主心遇变又不能陈说悃欵不惟负皇上抑且负天故敢少竭其愚臣讲臣也犹忆供事之初皇上时御经筵非寒暑不辍日御讲筵非风雨不免因窃自思以皇上至神至圣岂二三儒绅片时讲诵即能有补于万一而皇上留心问学如此者毋亦谓亲宦官宫妾不如接贤士大夫之雍容处秘殿深宫不如居广厦细旃之敛饬躭醉饱醲鲜不如味圣贤理义之隽永乐声色谀佞不如闻格言法语之砭箴乎是故自古人君讲读日勤则志气日清而处事咸得其当讲读日懈则志气日隳而处事多失其平皇上何不反躬自思以今昔而并观之也昔讲读时昧爽临朝者陛下也今岁不一御矣夫人君象日日一日不照临人且骇异十日而不见日将翘首东望不胜抑郁无聊之思为人臣子而终岁不一睹天颜其瞻天仰日之心当何如者此犹其小者耳下不见上则所为上交者不得不凭章奏而章奏又有时而不行上不接下则所为下交者不得不凭批答而批答又有时而不下苟有炀灶之奸中扃外闭如国事何知皇上在御必无此事傥圣子神孙中刚明稍有不及皇上者而引今日为故事则权不旁落政即下移是皇上作法也岂以明圣而虑不及此也昔讲读时遇旱步祷者陛下也今不一至郊矣夫郊祀不亲已非古人昭事之意岂有目睹天心谴怒至于焚毁宸居震惊圣躬视旱灾为孰切者犹然不一诣郊坛深陈自怨自艾之诚以祈天佑而仅止于遣官恭代岂以代者之恭即能谢天谴而冋天怒乎未可必也此时而不亲郊更无望亲郊时矣昔讲读时时享斋居者陛下也今不一入庙矣臣窃度之方被灾时皇上所望于九庙神灵之保护者岂不甚切而九庙神灵所为黙佑于冥冥之中者岂不甚力幸无事矣乃惜咫尺之劳不躬诣祖考之前一申报谢岂惟皇上之心不能自安即列祖在庙之灵不能不望于皇上也昔讲读时推心以委大臣坦懐以任羣臣而今不然矣中有所主辅臣不能申其说意有所移九卿无能捄其败言疑其私人疑其党大寮既因之而虚位庻官亦因之以另推自古未有堂陛隔絶如此其甚者也夫皇上养士二十四年于兹矣岂生于今者尽懐私植党而无一可信可倚者乎亦薄于视士矣夫国家事如人一身血脉有所不周则必有壅阏痿痹之患今上蓄疑以待下下懐疑以事上至令腹心之臣惶惑而不敢断耳目手足之臣逡巡而莫知所措陛下谁与治天下乎非所以示一体之谊成泰交之风也昔讲读时起用建言之臣优容直谏之士而今不然矣言官而言既怒其言而显斥之不言又怒其不言而槩斥之当事者斥不当事者亦斥用者既斥斥者不许复用故两月来无敢以言进者其心岂真以朝无阙政可遂相安于无言不过畏陛下之积怒不敢言耳夫不言极易事耳然人臣之利非社稷之福也今修省之日正不讳之时臣恐在朝之臣争欲发愤懑効忠诚而言未必悉当词未必悉婉皇上傥复厌其烦怒其激则諌道从此遂塞而陛下之善念阻矣惟陛下辨言之公私以为从违因以辨人之公私以为用舎不可槩以为言耳而槩视之也昔讲读时亦曽惩一二宦寺而罚止逺遣罪不株连今何如矣疑心一生视左右尽若可怒怒心一起视左右尽皆可刑鞭朴之惨满前征逮之使载道夫此辈岂不欲委曲承顺少得皇上之欢心其逆圣意而被严谴者摄于威也夫摄之以威则愈不及而愈见其难堪寛之以情自无不及而自见其可恕惟皇上布大德赦小过治有罪释无辜使人不惧死孰不感恩当死而生感恩必甚由是肘腋■〈执上目下〉御其爱戴皇上当有甚于皇上之自爱者矣陛下亦何惮而不为此也凡此皆行事之偶失而其原皆起于圣志之未清其本皆由于圣学之乆废故臣惓惓以勤讲读为修省之要务也然臣不独言之惟望皇上行之且亟行之夫闻之而能行行之而能亟于行是真能悔悟者也人情初悟初悔皆能奋励而渐乆渐怠不免忽忘故臣愿陛下之亟行之也昔秦穆覇国之君汉武多欲之主而秦誓之书轮台之诏载之经史为后世美谈皇上傥不以臣言为迂而力行之则天下万世颂明天子者为禹为汤而髙出秦穆汉武之上矣
