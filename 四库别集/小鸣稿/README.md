　　提要

　　臣等谨案小鸣稿十巻明秦简王朱诚泳撰诚泳为太祖五世孙愍王樉之元孙成化末袭封自号宾竹道人明史诸王列传称诚泳性孝友恭谨尝铭冠服以自警所著有经进小鸣集即此本也朱彝尊诗话又称王十岁时嫡母陈妃以唐诗教之日记一首嗣位日赋一篇好士礼贤兼留心服食葢能以文雅自见者集为其纪善强晟所编録自巻一至巻八皆诗巻九为杂文巻十为恩赐胜览録乃弘治癸丑诚泳请朝命养疾于凤泉温泉汤泉时所作其诗古体虽未尽合格而杨白花一篇殊有雅调近体俱谐婉可诵而七絶尤为擅场如秋夜诗云霁月满窓明似昼梧桐如雨下空庭又云空庭久坐不成寐明月满阶砧杵声又山行诗云啼鸟无声僧入定半岩风落紫藤花皆风骨戌削不媿作家明代亲藩中以文学著名者要必以诚泳为称首焉

　　圣君曲

　　圣人履天位驭世应昌期寳厯千年运神京万世基梯航来职贡海岳拱神祇常道陈中夏仁恩洽四夷仪文唐制度官属汉威仪气正三辰顺年登百榖滋河清龟出洛日暖凤鸣岐大地歌謡满唐虞今在兹

　　宛转歌

　　夜月明行舟琴更清女郎元是钟情物风前乍聴难为情歌宛转此身今已亡于君有冥契愿作双鸳鸯长太息相看涙沾臆茫茫造物竟谁尸恨不生逢王敬伯歌宛转宛转悲且长红颜易零落古今同一伤

　　明月引

　　青天有月兮照八荒银汉无云兮流素光平分寒暑兮变炎凉几人欢乐兮几悲伤叹明月之弗改念人生之靡常或醉饱于夏屋或困敝于康荘一瓢寒士千乘侯王泥涂囚虏珠翠名倡或安居于甲第或流落于他乡百年能几毕竟亡羊惟达人之知命且对月而衔觞

　　江上曲

　　君为南越儿妾乃东吴女江上一相逢目成心已许与君欲相语奈此舟中人到岸各分散悠悠恨此身风髙波浪险极目孤篷转重晤良独难万里天涯逺

　　春逰曲（五首）

　　车马乗春出莺花触眼新秋千忽飞起故故看逰人
　　走马章台路红尘一骑飞美人招素手楼上唤郎归
　　晓色分杨栁春阴护牡丹王孙驰骏马多是醉中看
　　新栁迎逰骑飞花趂酒瓯年年三二月烂醉曲江头
　　楼外江山丽城边锦绣新春光浓似酒醉杀浪逰人

　　鸡鸣曲

　　沉沉漏箭催残更扶桑日出天鸡鸣羲和叱驭起苍震坐令万国生光明角角再鸣声未已万国祼生都唤起可怜驱入利名场蝇营尽被鸡催死我愿人家莫畜鸡胶胶风雨不湏啼威凤来仪天下乐日日窓前闻喜鹊

　　对酒歌

　　人生埃壒内忽忽如梦中三万六千日所期安得同枝上花正开风前花又落羲驭不我留及时胡不乐君不见三闾大夫夸独醒形容憔悴江潭行回风歌罢塟鱼腹到底只留身后名又不见伯伦荷锸身相随生死有命信所之洁身旷达两何有毕竟得失还归谁荣悴升沈若翻手千古贤愚同一朽昨朝緑鬓映朱颜今日苍髯成皓首秋月春光不等闲何如且罄樽中酒樽中酒岂徒然会须一饮吸百川白日西驰坐成夜休问安期与偓佺

　　古意

　　君好堤边栁妾好岭上松堤栁吐轻絮飞去无定踪不如松有操欝欝傲寒冬
　　君好江上舟妾好岸边树舟行遇风波飘泊无常处何如树不移风波自来去

　　楼上曲

　　东风吹春春欲阑纷纷香雪梨花残美人倚阑娇不语翠袖笼寒懒轻举尘封锦柱暗银筝那忍弹成肠断声掩袂含羞抱幽怨怕见雕梁语双燕蹙损双蛾思转迷粉痕满面伤春啼金莲缓歩下楼去飞絮逰丝荡心绪巫阳望断楚天遥行雨行云在何处

　　逺逰篇

　　至人思逺逰逺逰竟何之聊乘汗漫风直与神仙期晨兴食方丈暮归宿瑶池凭虚弄日月剑佩光陆离渇饮九霄露饥飡千岁芝平头看星斗挥手谢尘覊欢忻朝帝座指麾役灵祇归来驭元气飞云亦追随歩虚振金石天风吹鬓丝两仪与终始三光共盈亏茫茫尘海人蚩蚩讵能知

　　苦热行

　　赤云喷火烧长天蟾宫老桂飞苍烟金乌自恨具毛羽白虎喘杀知谁怜万山烘透销青翠四海波干欲平地苍龙鳞甲隠虫蛆竟日泥蟠嗟殄瘁况是南州瘴疠乡火山炎海连天黄着人毒热若汤火灼我肌肤成炙疮铁甲磨穿藏虮虱万苦千辛死无日蛮人退去幸生还一级微名君尚惜

　　渭城少年行

　　洛下春深栁絮飞渭城少年应忆归马蹄偶入城南陌相逢却是长安客为言前日发秦台而翁传语需君来千门万户燕皆至北苑南园花正开年少寻春湏及早挥鞭疾上长安道一路春光絶可怜懵腾忽过斗城边两两新妆临渭水纷纷逰骑满秦川秦川自昔羡繁华王孙公子争豪夸门开朱户星初出燕设青楼日未斜倚醉聫镳过戚里九衢见者皆兴起玉辔摇风马更骄斗鸡赌得肯相饶闲寻红粉来金市喜聴黄鹂过灞桥桥下倡家新酒熟觧鞍就向倡家宿美人纎手弄琵琶酒边双脸蒸红霞黄金掷与买今夕少年肯负秦川花

　　轻薄篇

　　末世多轻薄轻薄良可嗤昻昻倚门第行行恃先基金多忽贫贱尚肯恤癃疲气豪蔑贤圣讵知慕轩羲一飡烹九鼎下筯犹嗟咨被服极纨绮犹云匪新竒千金买艶妾色衰竟弃之四方致名马一蹶终见遗结交出肺肝盟言日星垂片言茍不合对面生嫌疑仲尼称匹夫造化呼小儿伯阳道徳小孟轲仁义亏孙吴岂谙兵卢扁岂知医陋兹轻薄子宜为君子规薄方惊蝶翅轻于风栁枝愿言执其人为君投四夷

　　公子行

　　白面青瞳照秋水豪华知是诸公子金鞍玊辔紫丝鞭双双笑语东风里越罗披体试春衣鹘似身轻马似飞春色薫人随处好纵逰直过妓楼西美人楼上亲招手公子风流能过否緑杨树底繋花骢指点银瓶来索酒笑将彩胜结同心醉来一掷双南金翠帏锦帐恣欢谑梦入阳台何处寻五更却恨鸡声早马上归来踏芳草弯弓更过曲江头争拾金丸逐飞鸟西城路

　　西城路

　　半是平田半坟墓平田渐少坟渐多酸风苦雨泣铜駞百岁光阴驹过隙营营终日将如何人生少年胡不乐世间那有蘓仙鹤

　　燕燕于飞

　　燕燕何处来营巢寻旧主斜冲碧栁烟低掠青芜雨逐絮绕回塘衔泥过逺浦汉灰无复然莫效当时舞

　　苦哉逺征人

　　苦哉逺征人荷戈西出秦以身自许国举目谁为亲阴云翳落日白草迷黄尘三伏不知暑四月那见春射生充早食卧草当夜茵风髙弓力劲霜重铁衣新主将号令肃部伍催行频提兵誓灭敌僇力敢顾身固知在家乐宁厌行戍贫他日侯王贵绘像登麒麟

　　秋夜长

　　秋夜长明月光露华洗天金气凉寒虫候鴈两何苦声逐西风来枕傍彷徨披衣坐夜深丁东又聴捣衣砧残烛荧荧更漏静无端万事皆萦心

　　遥夜吟

　　秋夜一何遥灯烬炉烟灭低下水精帘含情对明月

　　估客乐

　　郎船下江水乗风疾若飞与侬各分袂掩泣牵郎衣手拔黄金钗临岐赠郎别愿郎心似金坚持莫轻折

　　将军行

　　衔命西来专斧钺逺出萧关探虎穴铁骢夜踏黒河氷金戈冷射闗山月将军令严健儿勇直捣长驱争鼓勇阴山北遁穹庐空惟剰牛羊满丘陇忆昔我军多战没风雨啾啾鬼宵哭于今汉将守边城翻役官军收战骨露布星驰达冕旒将军承诏独封侯争夸一代麒麟阁谁念沙场万髑髅

　　捣衣曲

　　井梧叶坠金风起满庭明月凉扵水为念儿夫戍玉闗尘满征衣谁与洗我有氷缣三两疋胼手殷勤为君织欲制寒衣逺寄将双杵拈来对砧石捣乱西风续又鸣一声声里寓幽情夜深玉腕痛且困不知露下将三更裁缝已就凭谁寄走马朝来有边吏一行书染泪千行寒到君边衣到未

　　夜宴谣

　　良夜沉沉宫漏促两阶庭燎烧银烛上宾入座气如虹帘外佳人面如玉一举飞觥累十觞凤笙龙管声锵锵满堂珠翠照人眼酒边尤物令人狂风动俄然银烛灭暗中求与同心结红颜变色絶冠缨壮士羞颜冷扵铁君王一笑许终欢保全宾客曽无难夜深痛醉各归去感恩直欲倾心肝从来燕礼惟三爵三爵既过多失错不醉无归乃所私饮而不乱真为乐

　　农夫谣

　　我昨过农家农夫于我陈嗟嗟天地间而唯农苦辛春耕土埋足夏耘汗沾巾秋成能几何仅得比比邻老农惟二子输边辛苦均大儿援灵夏性命逐车轮小儿戍甘泉身首犯边尘老妇卖薪去老农空一身荒村絶鸡犬四壁罄仓囷公家不我恤里胥动生嗔鞭笞且不免敢冀周吾贫我农老垂死甘为地下人尚祈孙子辈犹为平世民

　　树中草

　　草木非同气托根犹可生如何兄与弟同气却无情

　　独不见

　　举头望仙山孤坐开水殿芳草自成茵落花复如霰轻云逐翔鸾緑树来新燕光景满目前杳然独不见

　　昔思君

　　昔君与我兮心契意投今君与我兮云散雨收昔君与我兮鱼水相宜今君与我兮风马难追昔君与我兮琴瑟谐和今君与我兮音问蹉跎

　　伤思吟

　　碧云天末寒风起吹乱宾鸿不成字逺道音书那得闻伤心滴尽相思泪泪痕界破腮上红一点灵犀不可通惊破绿窗蝴蝶梦月明砧杵闻丁东玉闗此去三千里离恨又从今日始自怜不及江上莲两两红衣并头死

　　吴钩行

　　吴钩铸鬼工锋锷耀昆铁逰侠购千金羌人犹未灭

　　壮士行

　　寝帐夜闻鸡起坐气如虎怒髪赫冲冠拔剑灯前舞

　　戍妇词

　　奉姑妾尽孝报君郎尽忠忠孝茍无愧死生何必同

　　饮酒乐

　　好鸟窗前宛转春花庭外芳菲任我狂歌醉舞从他兔走乌飞

　　春晓曲

　　隠隠踈钟出林杪杜鹃啼血韶光老西楼残月坠咸池香雾空蒙六宫晓緑槐满院结春阴锦屏银烛髙堂深鸳鸯不暖红窗梦虹壶玉漏声沉沉起来无事偏萧索料峭东风越罗薄笑语丫鬟卷绣帘一庭红雨桃花落

　　羽林郎

　　戮力收城后承恩下紫宸电飞骝马疾日绚锦袍新邃合宵藏妓长途昼杀人赦书终见宥依旧佩麒麟

　　春夜曲

　　皓月明台榭和风度管弦夜深诸女伴犹自竞秋千

　　凉州词

　　黄云漠漠接平沙落日悲风惨暮笳回首长安五千里征人若个不思家
　　三军辛苦觅封侯终岁防邉战不休何事孟佗凭斗酒当时谈笑博凉州

　　相思曲

　　去日春光未老别来秋意方深明月寒更砧杵丁东敲碎侬心

　　炖煌曲

　　何物蕃儿颜渥赭生来个个能骑马公然剽掠恣南行河湟戍卒尸纵横危城穷将呼天泣都护不来谁救急身无羽翼可能飞一纪穷荒甘受围万里长安无一使丐兵回纥何时归懦将既诛绫易麦一饱何辞蹈锋镝朝来羣丑已登城我军四面皆蕃声万人觧甲从蕃俗强把皮裘更我服每直年年寒食时禠裘南向吞声哭汉将黄金空满箧不见征西能报捷

　　龙支行
