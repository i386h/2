守宫砂　　　佚名　　著

序

　　稗史之兴，原用助正史之不足。若忠奸贤否，英雄豪杰，绘色绘声，跳跃纸上者，为其使愚者阅之，知所以劝善而惩恶也。是书以有明武宗任用阉宦，卒至谋乱，河南一幸，几致宗社覆灭。幸李广、楚云辈出，先事预防，拨乱反正，始得转危为安。然其中所谓忠奸贤否，英雄豪杰，无不毕露。阅者深读而玩味之，或亦谅作者之有心惩劝耳。
