　　钦定四库全书　　　　　集部二
　　元氏长庆集　　　　　　别集类一【唐】
　　提要
　　【臣】等谨案元氏长庆集六十巻补遗六巻唐元稹撰稹爵里事迹具唐书本传考稹与白居易书称河东李明府景俭在江陵时僻好仆诗章仆因撰成巻轴其中有防意可观而词近古徃者为古讽意可观而流在乐府者为乐讽词虽近古而止于吟写性情者为古体词寔乐流而止于模象物色者为新题乐府声势沿顺属对稳切者为律诗仍以五七言为两体其中有稍存寄兴与讽为流者为律讽又称有悼亡诗数十首絶诗百余首自十六岁时至元和七年有诗八百余首成二十巻又称昨巴南道中有诗五十一首文书中得七年以后所作二百篇然则稹三十七岁之时已有诗千余首唐书本传称稹卒时五十三其后十六年中又不知所作凡几矣白居易作稹墓志称着文一百巻题曰元氏长庆集唐书艺文志又载有小集十巻然原本已阙佚不传此本为宋宣和甲辰建安刘麟所传明松江马元调重刋而巻帙与旧説不符即标目亦与所自序迥异不知为何人所重编前有麟序称稹文虽盛传一时厥后浸以不显惟嗜书者时时传录其先人尝手自抄写谨募工刻行云云则麟及其父均未尝有所增损盖在北宋即仅有此残本尔乾隆四十四年九月恭校上

　　元氏长庆集原序
　　新唐书艺文志载其当时君臣所撰著文集篇目甚多太宗集四十巻至武后垂拱集一百巻今皆弗传其余名公钜人之文所传盖十一二尔如梁苑文类防昌一品鳯池稾草笠泽丛书经纬穴余遗荣雾居见于集录所称道者毋虑数百家今之所见者仅十数家而已以是知唐人之文亡逸者多矣呜呼樵夫牧叟诡异怪诞之説鬼神幻惑不根之言时时萃为一书以诒好事者观览至于士君子道德仁义之文经国济时之论乃或沉没无闻岂不惜哉元微之有盛名于元和长庆间观其所论奏莫不切当时务诏诰歌词自成一家非大手笔曷臻是哉其文虽盛传一时厥后浸亦不显唯嗜书者时时传录不亦甚可惜乎仆之先子尤爱其文尝手自抄写晓夕玩味称叹不已盖惜其文之工而传之不久且逺也乃者因阅手泽悲不自胜谨募工刻行庶几元氏之文因先子复传于世斯文旧亡其序第冠以新唐书微之本传则微之之于文其所造之浅深可槩见矣宣和甲辰仲夏晦日建安刘麟撰

　　古诗
　　思归乐
　　山中思归乐尽作思归鸣尔是此山鸟安得失乡名应縁此山路自古离人征隂愁感和气俾尔从此生我虽失乡去我无失乡情舒在方寸宠辱将何惊浮生居大块寻丈可寄形身安即形乐岂独乐咸京命者道之本死者天之平安问逺与近何言殇与彭君防赵工部八十支体轻交州二十载一到长安城长安不须叟复作交州行交州又累嵗移镇值江陵归朝新天子济济为上卿肌肤无瘴色饮食康且寜长安如昼夜死者如霣星防车四门出何关炎瘴萦况我三十余百年未半程江陵道途近楚俗云水清遐想玉泉寺久闻岘山亭此去尽绵歴岂无心赏并红餐日充腹碧涧朝析酲酿酒待賔客寄书安弟兄闲穷四声韵闷阅九部经身外皆委顺眼前随所营此意久已定谁防苟求荣所以官甚小不畏权势倾倾心岂不易巧诈神之刑万物有本性况复人至灵金埋无土色玉坠无瓦声劔折有寸利镜破有片眀我可俘为囚我可刃为兵我心终不死金石贯以诚此诚患不立虽困道亦亨微哉满山鸟叫噪何足聴
　　春鸠
　　春鸠与百舌音响讵同年如何一时语俱得春风怜犹知造物意当春不生蝉免教争叫噪沸渭桃花前
　　春蝉
　　我自东归日厌苦春鸠声作诗怜化工不遣春蝉生及来商山道山深气不平春秋两相似虫豸百种鸣风松不成韵蜩螗沸如羮岂无朝阳鳯羞与微物争安得天上雨奔浑河海倾荡涤反时气然后好晴眀
　　兔丝
　　人生莫依倚依倚事不成君看丝蔓依倚榛与荆荆榛易防宻百鸟撩乱鸣下有狐穴奔走亦纵横樵童斫将去柔蔓与之并翳荟生可耻束防死无名桂树月中出瑚石上生俊鹘度海食应龙升天行灵物本特达不复相纒萦纒萦竟何者荆棘与飞茎
　　古社
　　古社基阯在人散社不神唯有空心树妖狐藏魅人狐惑意颠倒臊腥不复闻丘坟变城郭花草仍荆榛良田千万顷占作天荒田主人议芟斫怪见不敢前那言空山烧夜随风马奔壮声鼓鼙震髙熖旗帜飜逡廵荆棘尽狐无子孙狐死魅人灭烟消坛墠存绕坛旧田地给授有等伦农收村落盛社树新团圆社公千万嵗永保村中民
　　松树
　　华山髙幢幢上有髙髙松株株遥各各叶叶相重重槐树夹道植枝叶俱防既无贞直干复有罥【音绢】挂虫何不种松树种之揺清风秦时已曽种顦顇种不供可怜孤松意不与槐树同闲在髙山顶樛盘虬与龙屈为大厦栋庇廕侯与公不肯作行伍俱在尘土中
　　芳树
　　芳树已寥落孤英尤可嘉可怜团圆叶盖覆深深花游蜂竞鑚刺鬬雀亦纷拏天生细碎物不爱好光华非无歼殄法念尔有生涯春雷一声发惊燕亦惊虵清池养神蔡已复长虾蟆雨露贵平施吾其春草芽
　　桐花
　　胧月上山馆紫桐垂好隂可怜暗澹色无人知此心舜没苍梧野鳯归丹穴岑遗落在人世光华那复深年年怨春意不竞桃杏林惟占清眀后牡丹还复侵况此空馆闭云谁恣幽寻徒烦鸟噪集不语山嵚岑满院青苔地一树莲花簮自开还自落暗芳终暗沈尔生不得所我愿裁为琴安置君王侧调和元首音安问宫徴角先辩雅郑淫宫春以君君若春日临商防以臣臣作旱天霖人安角声畅人困鬬不任羽以类万物祅物神不歆征以莭百事奉事罔不钦五者苟不乱天命乃可忱君若问孝理弹作梁山吟君若事宗庙拊以和球琳君若不好諌愿献触防箴君若不罢猎请聴荒于禽君若侈台殿雍门可霑襟君若傲贤隽鹿鸣有食芩君闻祈招什车马勿骎骎君若欲败度中有式如金君闻薫风操志气在愔愔中有阜财语勿受来献賝北里当絶聴祸莫大扵婬南风苟不竞无往遗之擒奸声不入耳巧言寜孔壬枭音亦云革安得沴与祲天子既穆穆羣材亦森森劔士还农野丝人归织絍丹鳯巢阿阁文鱼逰碧浔和气浃寰海易若溉蹄涔改张乃可鼓此语无古今非琴独防尔事有谕因针感尔桐花意闲怨杳难禁待我持斤斧置君为大琛
　　雉媒
　　双雉在野时可怜同嗜欲毛衣前后成一种文章足一雉独先飞冲开芳草緑网罗幽草中暗被潜覊束剪刀摧六翮丝线缝双目啖养防防时依然已驯熟都无旧性灵返与他心腹置在芳草中翻令诱同族前时相失者思君意弥笃朝朝旧处飞往往巢边哭今朝树上啼哀音断还续逺见尔文章知君草中伏和鸣忽相召鼓翅遥相瞩畏我未肯来又啄翳前粟敛翮逺投君飞驰势奔蹙罥挂在君前向君声促促信君决无疑不道君相覆自恨飞太髙踈罗偶然触防看架上鹰拟食无罪肉君意定何如依旧雕笼宿
　　箭镞
　　箭镞本求利淬砺良甚难砺将何所用砺以射凶残不砺射不入不射人不安为盗即当射寜问私与官夜射官中盗中之血阑干带箭君前诉君王悄不欢顷曽为盗者百箭中心攅竞将儿女泪滴沥助辛酸君王责良帅此祸谁为端帅言发硎罪不使刃稍刓君王不忍杀逐之如迸丸仍令后来箭尽可头团团发硎去虽逺砺镞心不阑防射蛟螭尽舟行无恶澜
　　赛神
　　村落事妖神林木大如村事来三十载巫觋传子孙村中四时祭杀尽鸡与豚主人不堪命积燎曽欲燔旋风天地转急雨江河翻采薪持斧者弃斧纵横奔山深多掩映仅免鲸鲵吞主人集隣里各各持酒罇庙中再三拜愿得禾稼存去年大巫死小觋又妖言邑中神眀宰有意効西门焚除计未决伺者迭乗轩庙深荆棘厚但见狐蹲巫言小神变可騐牛马蕃邑吏齐进说幸勿祸乡原逾年计不定县聴良亦烦涉夏祭时至因令修四垣忧虞神愤恨玉帛意弥敦我来神庙下箫鼓正喧喧因言遣妖术灭絶由本根主人中罢舞许我重叠论蜉蝣生湿处鸱鸮集黄昏主人邪心起气焰日夜繁狐狸得蹊径潜穴主人园腥臊袭左右然后托丘樊嵗深树成就曲直可轮辕幽妖尽依倚万怪之所屯主人一心好四靣无篱藩命樵执斤斧怪木寜遽髠主人且倾聴再为谕清浑阿胶在末罔象游上源灵药逡巡尽黑波朝夕喷神龙厌流浊先伐鼍与鼋鼋鼍在龙穴妖气常欝温主人恶淫祀先去邪与惛惛邪中人意蛊祸蚀精魂徳胜妖不作势强威亦尊计穷然后赛后赛复何恩
　　大觜乌
　　阳乌有二类觜白者名慈求食哺慈母因以此名之饮啄颇防俭音响亦柔雌百巢同一树栖宿不复疑得食先返哺一身长苦羸縁知五常性飜被众禽欺其一觜大者攫搏性贪痴有力强如鹘有爪利如锥音声甚防【于交切婬声】嗗【乌或切】潜通妖怪词受日余光庇终天无死期翺翔富人屋栖息屋前枝巫言此乌至财产日丰宜主人一心惑诱引不知疲转见乌来集自言家转孶白鹤门外养花鹰架上维専聴乌喜怒信受若神龟举家同此意弹射不复施往往清池侧却令鹓鹭随羣乌饱梁肉毛羽色泽滋逺近恣所往贪残无不为巢禽攫雏卵廐马啄疮痍渗沥脂膏尽鳯皇那得知主人一朝病争向屋檐窥呦鷕【雉鸣也以绍切】呼羣鵩翩翻集怪鸱主人偏养者啸聚最奔驰夜半仍惊噪鸺【音休】鹠【音留】逐老狸主人病心怯灯火夜深移左右虽无语奄然皆泪垂平眀天出日隂魅走参差乌来屋檐上又惑主人儿儿即富家业翫好方爱竒占募防言鸟置者许髙赀陇树巢鹦鹉言语好光仪美人倾心献雕笼身自持求者临轩坐置在白玉墀先问鸟中苦便言乌若斯众乌齐搏铄翠羽防离披逺掷千余里美人情亦衰举家惩此患事乌逾昔时向言池上鹭啄肉寝其皮夜漏天终晓隂云风定吹况尔乌何者数极不知危防结弥天网尽取一无遗常令阿阁上宛宛宿长离
　　分水岭
　　崔嵬分水岭髙下与云平上有分流水东西随势倾朝同一源出暮隔千里情风雨各自异波澜相背惊势髙竞奔注势曲已回萦偶值当途石蹙缩又纵横有时遭孔穴变作呜咽声褊浅无所用奔波奚所营团团井中水不复东西征上应美人意中涵孤月眀旋风四靣起井深波不生坚氷一时合井深冻不成终年汲引絶不耗复不盈五月金石铄既寒亦既清易时不易性改邑不改名定如拱北极莹若烧玉英君门客如水日夜随势行君看守心者井水为君盟
　　四皓庙
　　巢由昔避世尧舜不得臣伊吕虽急病汤武乃可君四贤胡为者千载名氛氲显晦有遗迹前后疑不伦秦政虐天下黩武穷生民诸侯战必死壮士眉亦嚬张良韩孺子椎碎属车轮遂令英雄意日夜思报秦先生相将去不复婴世尘云巻在孤岫龙潜为小鳞秦王转无道諌者鼎镬亲茅焦脱衣諌先生无一言赵髙杀二世先生如不闻刘项取天下先生逰白云海内八年战先生全一身汉业日已定先生名亦振不得为济世宜哉为隠沦如何一朝起屈作储贰賔安存孝惠帝摧顇戚夫人舍大以谋细虬盘而蠖伸惠帝竟不嗣吕氏祸有因虽懐安刘志未若周与陈皆落子房术先生道何屯出处贵眀白故吾今有云
