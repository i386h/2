　　钦定四库全书　　　　集部二
　　沈下贤集　　　　　　别集类一【唐】
　　提要
　　【臣】等谨案沈下贤集十二巻唐沈亚之撰下贤亚之字也本长安人而原序称曰吴兴人以从其郡望然李贺集有送亚之诗亦曰吴兴才人怨春风又曰家在钱塘东复东则似真吴兴人矣唐人里贯多错互不得其真未之详也亚之登元和十年进士第太和三年柏耆宣慰徳州辟为判官耆罢亚之亦坐贬南康尉是集凡诗赋一巻杂文杂记一巻杂著二巻记二巻书二巻序一巻策文并对一巻碑文墓志表一巻行状祭文一巻杜牧李商隠集均有拟沈下贤诗则亚之固以诗名世而此集所载乃止十有八篇其文则务为险崛在孙樵刘蜕之间观其答学文僧请益书谓陶器速售而易败鍜金难售而经乆送韩静畧序亟述韩愈之言盖亦戛然自异者也其中如秦梦记异梦録湘中怨觧大抵讳其本事托之寓言如唐人后土夫人传之类刘克庄后村诗话诋其名检扫地王士祯池北偶谈亦谓防玉邢鳯等事大抵近小説家言考秦梦记异梦録二篇见太平广记二百八十二巻湘中怨觧一篇见太平广记二百九十八巻均注曰出异闻集不云出亚之本集然则或亚之偶然戯笔为小説家所采后来编亚之集者又从小説摭入之非原本所旧有欤此本前有元祐丙寅重刻序不着姓名钱曽读书敏求记乃称为元祐丙申刻考元祐元年嵗在丙寅至甲戌已改元绍圣中间不应有丙申盖即此本而曽误记寅为申又是集本十二巻曽记为二十巻亦误倒其文也池北偶谈又记末有万厯丙午徐跋此本无之而别有跋曰吴兴文集十二巻义取艰深字多舛脱不可卒读因从秦对岩先生借所藏季沧华抄本校阅一过题曰辛夘仲夏有小印曰邦采不知为谁然则此本校以季氏本季氏本钞自钱氏宋刻其源流固大概可见矣乾隆四十三年六月恭校上

　　沈下贤集原序
　　文章盛衰与世升降唐之文风大振于贞元元和之间韩柳唱其端刘白继其轨当时学者涵濡游泳揽其英华洗濯磨淬辉光日新苟有作者皆足以拔出流俗自成一家之语则吴兴之文是巳公讳亚之字下贤吴兴人元和十年登进士第歴聘藩镇尝游韩愈门李贺许其工为情语有窈窕之思其后杜牧李商隠俱有拟沈下贤诗则当时称声甚盛而存于今者既不尽见世之所有复舛错讹谬脱文漏句十有二三顷得善本再加校覈皆得其正惜其藏于箧笥不得好学之士共有翫绎因命工刋镂以广其传元祐丙寅十月一日题

　　梦游仙赋
　　余昔一梦寓游一方乐态甚适觉而作赋题之梦游仙其词曰
　　杳漠漠兮升絶垠云辔九天兮越崇门星凌晓以淡白澜咽潾于锦砾石榴笑而织娥喜阍导余而就将止袭烈蕙之芳风送丽音于辽耳目恣迈而多适吾超超其乐此银墉兮桂厢差瑶阶兮上玉堂卷红幕以绣户中有人兮结清处语嫣延兮情绰撦命余荫于兰之厦回秾顔以一顾矕娇眸而融冶烟津兮玉盘火桂兮炮鸾鼎娥司味和苦酸嬴吹既调戛湘合我饮食兮乐吾后园乃称诗曰白日低兮春塘满红华芳兮草芽短菱结带兮荇含丝设遨游兮导佳期又诗曰秾光醉兮昏绵绵焉与久乐万年春留连兮其未央吐芳意于荃言忽寤以无覩魂迷念兮情牵既谅人生之皆梦孰云夕非而昼是驰咏想之悠悠兮轴吾情于万里
　　柘枝舞赋【并序】
　　往者某值宴于郑衞之侯坐与客序乐作堂下行舞男女纷杂交贯率以百品而观者盖然既罢升鼓堂上筵吹大奏命为柘枝舞则皆排目矢座客曰今自有土之乐舞堂上者唯胡部与焉而柘枝益肆于态诚足以赋其容也因顾余绪之以洗客念赋曰
　　昔神祖之克戎宾杂舞以混防柘枝信其多妍兮命佳人以继态撼隆冠之繁珂兮披文缨于大带跪闪举以挥獝兮拖旋襟之防曳骛游思于情杳兮注光波于秾睇顾巧度之无穷兮将多变而若云厉唱于鼍鼓兮俨兰露之芳津汨傍俯以袅影兮荡风蕖于横茵愕兮若惊弛兮若嬾歘然逴姹翔然嫣偃婉振修袖以抛拂兮韬纎肱以糅绾差重锦之华衣俟终歌而薄袒既而抑倚昂蹈节振臂驱防蹀以捉碎尽戎仪于弱媚见孙律于武姓入西河之劔气曲响未遒边风袭吹闻代马之清嘶言禽于咏类客曰若此之状也以郑衞而前陈吾固知其将坠
　　古山水障赋【和史馆陈学士作】
　　挹清臣之嘉思兮信疎子以峭孤回补籍之暇笔兮赋他山于遗图掷琨瑶于呐口兮若含采以咀兰惟古工之包化兮啓羣峯于无间势蹙巘以特起牙腾排而上干翠参差以玉立俱疎疎以攒攒于是澒以长澜森以怪木瑟汨栗飗凄烦荡燠浸平缭于楚泽冶妖韶于阳谷低霾横瞑兮陈半腹梦雨神云兮澹而不归怨撦惹乎岩曲缥藓青篱兮事幽澁细涧绵流兮洄喷噎山之人兮夕忘寐而坐忘兴帷绀枝兮带黄葛暖草被崖兮垂緑发翺鸾翔翠相追征兮振锦舒绣错以明兮啼羁酸雌裂眦吻兮风呺穴怒寥而辞兮山中之木何凋荣兮

　　湘中怨解
　　湘中怨者事本怪媚为学者未尝有述然而淫溺之人往往不寤今欲概其所论以着诫而已从生韦敖善撰乐府故牵而广之以应其咏
　　垂拱年中驾在上阳宫太学进士郑生晨铜駞里乗晓月渡洛桥闻桥下有哭甚哀生下马循声索之见有艶女翳然防袖曰我孤养于兄嫂恶常苦我今欲赴水故留哀须臾生曰能逐我归之乎应曰婢御无悔遂载与居号曰汜人所诵楚人九歌招魂九辩之书亦常拟其调赋为怨句其词丽絶世莫有属者因撰光风词曰隆佳秀兮昭盛时播薰緑兮淑华归顾里英与处萼兮潜重房以饰姿见稚态之韶羞兮防长霭以为衣醉融光兮弥弥迷千里兮涵湮湄晨陶陶兮暮熙熙舞婀娜之秾条兮骋盈盈以披迟酡游顔兮倡蔓卉縠流蒨电兮石发髓旎生居贫汜人尝解箧出轻缯一端与卖胡人酬之千金居数嵗生游长安是夕谓生曰我湘中蛟宫之娣也谪而从君今嵗满无以久留君所欲为诀耳即相持啼泣生留之不能竟去后十余年生之兄为岳州刺史防上巳日与家徒登岳阳楼望鄂渚张宴乐酣生愁思吟之曰情无垠兮霭霭洋洋懐佳期兮属三湘声未终有画舻浮漾而来中为防楼髙百余尺其上施纬帐栏笼画饰帷搴有弹鼓吹者皆神仙娥眉被服烟霓裙袖皆广长其中一人起舞含嚬凄怨形类汜人舞而歌曰泝青山兮江之隅拖湘波兮袅緑裾荷拳拳兮情未舒匪同归兮将焉如舞毕敛袖翔然凝望楼中纵观方怡须臾风涛崩怒遂迷所往元和十三年余闻之于朋中因悉补其词题之曰湘中怨盖欲使南昭嗣烟中之志为偶倡也
　　文祝延【有引】
　　文祝延之指其本祷祠闽人歌其质也闽侯居政民荫而安他日侯恙在体巷野之祈祠于神者皆以请侯益忧焉后得闲而词乃舒其俗以为言俚不足自道或谓军副者亚之能变风从律善阐物志因耆耋为请于是与文以通其意且以古之得人者皆祝延之今复用延命为篇目其词二阕
　　闽山之杭杭兮水堋吞荒抱大兮香叠层腾气清浑兮朝昏神生其中兮宅幽凝居如山兮惠如水处端卓兮赴下而忘鄙集人之祈兮从人之所市攀清明兮叩髣髴我民请兮期吉日愿听诚兮陈所当侯临我兮恩如光照导兮天覆惠流吾兮乐且康恭闻侯兮饮食失常民萦忧兮心苦创饱我之饥兮侯由有谷神有泽兮宜荫沃脱侯之恙兮归侯之多福羣卑勤之恭洁兮鉴贞盟乎山竹
　　右一阕为祈
