　　提要
　　余师録　　　　　　　诗文评类
　　【臣】等谨案余师録宋王正徳撰正徳宋史无传其爵里皆未详此书前有自序称绍熈四年则光宗时人也其书辑前代论文之语自北齐下迄乎宋虽习见者较多而当时遗籍今不尽传者亦往往而在宋人论文多区分门户务为溢羙溢恶之辞是録采集众説不参论断而去取之间颇为不茍尤足尚也征引时有小悮盖传冩之讹序称疲于酧答録此以代口述故时代先后畧不诠次此书宋志不着録亦久无传本惟载于永乐大典中首尾虽完具而不分巻数今约畧篇页定为四巻各考其譌缺注于句下序次则仍其旧云乾隆四十六年三月恭校上

　　余师录原序
　　余不肖无所用于世承先人绪业惧弗克负荷故于斯文窃有志焉早从当世大君子游走天下防半穷困来归独坐墙根曝晴景翻故书以娯日或以其虚名来问为文正法余旧学荒落口塞不得对慨念前軰论文章利病甚具散在方册时举一二以告之久輙忘去问者繦属老嬾疲于醻应而仆仆寻检又不可以应猝因记忆平时所诵令儿軰抄录以遗或者使归而求之有余师因题曰余师录若夫嵗月之后先字画之谬误彼此之是非名称之不伦皆不复次第推择观者以意求之余之志也绍熈四年冬至日海陵王正徳引





　　钦定四库全书
　　余师录巻一　　　　　　宋　王正徳　撰陈后山
　　陈后山送邢居实序云始吾得生年十五六识度气志巳如成人有其质也如木之始生玉之始琢顾其所成就何如尔生可不勉乎夫学以明理文以述志思以通其学气以达其文古之人导其聪明广其见闻所以学也正志完气所以言也王氏之学如脱墼耳案其形模而出之不待修饰而成器矣求其为元璧彞鼎其可得乎昔者子孔子谓阙党童子非求益乃欲速者也坐则居位行则先人其志盈矣无以复加矣而阙党童子后无闻焉子韩子谓张童子曰长与少异有成人之礼焉童子之所学不足尚之宜息其已学而勤其未学可也童子以数嵗通二经则竒矣使四五十而不加又何竒而张童子后无传焉吾年如邢生时见子曽子于江汉之间献其説余十万言髙自誉道子曾子不以为狂而报书曰持之以厚吾之不失身子曽子之赐也吾以为三君子之言可法
　　答江端礼书云学始于身而成于性欲善其身而不明于善所谓徒善者也徒善者非善之正也是故学者所以明善也学外也思内也学以佐行思以佐学古之制也若其自得则在子矣言以述志文以成言约之以义行之以信近则致其用逺则致其传文之质也大以为小小以为大简而不约盈而不余文之用也正心完气广之以学斯至矣
　　答秦观书云仆于诗初无师法然少好之而老不厌数以千计及一见黄豫章尽焚其藁而学也豫章以为譬之奕焉弟子髙师一着仅能及之争先则后矣仆之诗豫章之诗也豫章之学博矣而得法于杜少陵其学少陵而不为者也故其诗近之而其进未巳也
　　答晁深之书士之好为师旧矣子问而不以告岂其有所不足于师而莫知所以告耶士能致诚殚敬刳其心以求于师未有不告者也不诚不敬则不足以得之有以得之而心不至焉则又不足以受之也虽然教自外至者也外以导内于是有以自得之则至矣自是而观士何以教子子何待于士耶士方盛时气血动于内容色佻于外得之则感失之则悲气冐其心性乱于习百废一存惟欲之知夫才如水焉窒而挠之不浊牛马汚秽日滋而科斗生焉可不畏乎夫少而好色仆与子同而今悔之以仆之悔故不愿子为之也
　　章善序云名在于善而实不至谓之盗身在于善而意不至谓之伪意在于善而义不至谓之徒善徒善者非古之制也【案此条系送邢居实序中语章善序三字误】
　　诗话云杜之诗法韩之文法也诗文各有体韩以文为诗杜以诗为文故不工尔苏子瞻曰子美之诗退之之文鲁公之书皆集大成者也学诗当以子美为师有规矩故可学退之于诗本无觧处以才髙而好耳渊明不为诗冩其胷中之妙耳学杜不成不失为工无韩之才与陶之妙而学其诗终为乐天耳
　　余以古文为三等周为上七国次之汉为下周之文雅七国之文壮伟其失骋汉之文华赡其失缓东汉而下无取焉
　　诗欲其好则不能好矣王介甫以工苏子瞻以新黄鲁直以竒而杜子美之诗竒常工易新陈莫不好也退之作记记其事尔今之记乃论也少游谓醉翁亭记亦用赋体
　　庄子荀卿皆文士而有学者其説剑成相赋篇与屈骚何异
　　孙莘老喜论文谓退之平淮西碑叙如书铭如诗秦少游谓退之元和圣徳诗于韩文为下与淮西碑如出两手盖其少作也
　　韩退之上尊号表曰析木天街星辰清润北岳医闾神受职曾子固贺赦表曰钩陈太防星纬咸若昆仑渤澥波涛不惊世莫能轻重之也
　　国初士大夫例能四六然用偶语与故事尔杨文公笔力豪赡体亦变而不脱唐末五代之气又喜用古语以巧对为工乃进士赋体耳欧阳公始以文体为对属又善叙事不用故事陈言而文益髙次退之云
　　范文正公为岳阳楼记用对语説时景世以为竒尹师鲁读之曰此传竒体耳传竒唐裴铏所着小説也魏文帝曰文以意为主以气为辅以词为卫子桓不足以及此其有所传乎
　　鲁直与方防书云近世少年多不肯治经术及精读史书乃纵酒以助诗恐致逺则泥想生自追之琢之离此语病也
　　章子厚谓屈氏楚词如骚乃效颂其次效雅最后效风鲍昭诗华而不弱陶潜诗切于事情但不文耳
　　皇甫湜
　　皇甫湜谕业云燕公之文如楩木楠枝缔造大厦上栋下宇孕育气象可变隂阳而阅寒暑坐天子而朝羣后许公之文如应钟鼖鼓笙簧錞磬崇牙树羽考以宫县可以奉神明享宗庙李北海之文如赤羽元甲延亘平野如云如风有防有虎阗然鼓之吁可畏也贾常侍之文如髙冠华簮曵裾鸣玉立于廊庙非法不言可以望为羽仪咨以道义李员外之文则如金轝玉辇雕龙防鳯外虽丹青可掬内亦肌体不充独孤尚书之文如危峯絶壁穿倚霄汉长松怪石倾倒溪壑然而略无和畅雅徳者避之杨崖州之文如长桥新建铁骑夜渡雄震威厉动心骇魄然而鼓作多容君子所戒权文公之文如朱门大第气势宏敞廊庑廪廐户牖悉周然而不能有新规胜槩令人竦观韩吏部之文如长江秋注千里一道冲飇激浪污流不滞然而施诸灌溉或寡于用李襄阳之文如燕市夜鸿华亭晓鹤嘹唳亦足惊聼然而材力偕鲜瞥然髙逺故友沈谘议之文则如隼击鹰厉灭没空碧崇兰繁荣曜光扬蕤虽迅举秀擢而能沛艾絶景
　　作韩文公铭文云先生之作无员无方至是归工扶经之心执圣之权尚友作者敺邪觝异以扶孔氏存皇之极茹古防今无有端涯浑浑灏灏不可窥校及其酣放凌纸怪发鲸铿春丽震燿天下然而栗宻窈章妥句适精能之至出神入天呜呼极矣后人无以加之矣自姬氏以来一人而巳矣
　　功不十倍不可以果志力不兼两不可以角敌号猿贯虱彻札饮羽必非一嵗之决拾仰马出鱼理心顺气必非容易之搏拊浅辟庸种无嘉苖纇絇疎织无良帛夫欲利其获不若优其为获之方若欲显其能不若优其为显之道求诸人不若求诸巳驰其华不若驰其实彼则趦趄于卿士之门我则婆娑于圣贤之域彼则巾车于名利之场我则冠履于文史之囿道寝而后进业勤而后索以其劳于彼曷若勤于此以其背于路曷若赍于家鲔可荐也不虑网罟之不逢橘可贡也不虑包匦之不入务出人之名安得不厉出人之器战横行之阵安得不振横行之略书不千轴不可以语化文不百代不可以知变体无常轨言无常宗物无常用景无常取在覃其理覈其微赋物而穷其致咏歌者极性情之本载述者遵良直之防触类而长不失其要此大略也
　　苏明允
　　苏眀允上欧公书云孟子之丈语约而意尽不为巉刻斩絶之言而其锋不可犯韩子之文如长江大河浑灏流转鱼鼋蛟龙万怪惶惑而抑遏蔽掩不使自露而人望见其渊然之光苍然之色亦自畏避不敢廹视执事之文纡余委备徃复百折而条达疎畅无所间断气尽语极急言竭论而容与闲易无艰难劳苦之态此三者皆断然自成一家之文也惟李翺之文其味黯然而长其光油然而幽俯仰揖逊有执事之态陆贽之文遣言措意切近的当有执事之实而执事之才又自有过人者盖执事之文非孟子韩子之文而欧阳子之文也与孙叔静书云所示文字巳细观必欲求所未至如中正论引舜为正此时文之病凡作论但欲意立而理明不必覔事应副诚未之思尔
　　类要序云晏元献公出东南起童子入秘阁读书遂膺召命入翰林为学士真宗特宠待之每进见劳问及所以任属之者羣臣莫能及皇太子就学公以选入侍太子即皇帝位是为仁宗公遂筦国枢要任政事位宰相其在朝廷五十余年常以文学谋议为任所为赋颂铭碑制诏册命书奏议论之文传天下尤长于诗天下皆吟诵之当真宗之世天下无事方辑福应推功德修封禅及后土山川老子诸祠以报礼上下左右前后之臣非工儒学妙于语言能讨论古今润色太平之业者不能称其位公于是时为学者宗天下慕其声名人皆见公应于外者之不穷而不知公之得于内者何也及得公所为类要上中下帙总七十四篇凡若干门皆公所手抄廼知六艺太史百家之言骚人墨客之文章至于地志族谱佛老方伎之众説旁及九州之外蛮夷荒忽诡变竒迹之序録皆披寻防绎而于三才万物变化情伪是非兴壊之理显隐细钜之委曲莫不究尽公之得于内者在此也公之所以光显于世者有以哉观公之所自致者如此则知士不素学而处从官大臣之列备文儒道徳之任其能不馁且病乎此公之书所以为可传也
　　曾子固
　　曾子固作苏眀允哀词云为文少或百字多或千言其指事析理引物托谕侈能尽之约逺能见之近大能使之微小能使之着烦能不乱肆能不流其雄壮隽伟若决江河而下也其辉光眀白若引星辰而上也
　　逸事云陈后山初携文巻见南丰先生先生览之问曰曾读史记否后山对曰自防年即读之矣南丰曰不然要当且置它书熟读史记三两年尔后山如南丰之言读之后再以文巻见南丰南丰曰如是足也
