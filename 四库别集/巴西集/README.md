　　提要

　　（臣）等谨案巴西集二巻元邓文原撰文原字善之一字匪石绵州人随其父流寓钱塘自称巴西不忘本也生于宋理宗寳佑六年宋末应浙西转运司试中魁选至元间行中书省辟为杭州路儒学正官至集贤直学士兼国子监祭酒致仕致和元年卒于家谥文肃文原学有本原所作皆温醇典雅当大徳延佑之世独以词林耆旧主持风气袁桷贡奎左右之操觚之士响附景从元之文章于是时为极盛文原实有倡导之功所著有内制集履素斋稿今并未见传本此本不知何人所编前后无序跋仅録其碑志记序等文七十余首即顾嗣立元诗选中所録诸诗亦无一首葢后人摘选非其完帙然文原为元代名人而焦竑经籍志中已不载文原集名黄俞邰千顷堂书目载其名而不着巻数葢亦未见传本近时藏书家所有亦皆与此本相同则其全集之存否葢未可知或好事者搜采逸篇以补亡佚亦未可知也然吉光片羽虽少弥珍固当以幸存寳之不当以不完废之矣

　　吴氏义塾记

　　崇徳古御儿地大徳己亥吾尝为其州文学掾吴氏儁卿建门左之塾聘师以训乡之子弟三年矣地故多饶富儁卿非甚雄于赀而志欲敦尚儒风廸成善俗视古称任恤者盖庻几焉未几余入补词垣属又八年州若府疏其事于江浙省而以闻于朝曰吴氏义塾田为畆者二百师生饩廪有度讲肄有业童冠鼓箧而来者逾百员盍举以旌善朝命表其门曰义士会余以提举学事出领江浙西道访义塾之成则已迁于官河之东县庾故址岁输僦直为方七畆有半益以旁近地畆三经度缔构宏丽亢爽中象燕居翼以斋庑其北讲堂寝息有所左右书器庋阁严奥重门辉赫南穴为池直池北东廪舍庖湢各有攸处又増田二百畆以羡岁入中河为桥级石夷平便诸入塾者自造端至今更十有七寒暑而塾始大备凡用钱若干缗米若干石皆约巳撙用以给其规约则岁所敛储必子孙之长且贤者次掌之而师友共稽其出纳有赢亦以周里闾之婚嫁丧葬贫不举者勿侈勿啬勿轧于豪右勿挠于有司以圗惟永久子孙有违约者以不孝论乡得紏正焉儁卿谒余文为记余叹曰古者里门为塾子弟毕学于士师少师而里耆邻长诲之耕出入有时幼长有序其道简而教易行也自秦发闾左之戍而教始大壊今吴氏之为义塾或未能尽合乎古而意则美矣御儿本吴壌始太伯以礼让道民歴十九世而季札乃克弘宣祖训振迈髙风言行焯着夫子亦许为吴之习于礼者再世至阖庐已不承季札之志而自蹶其本识者知吴之不竞不待迁甬东时也夫吴之礼让更数百年封植之不足而隳之一旦易若反掌礼之有闗于斯世有如此者今世之士率视礼为阔迂而莫之讲所谓筋骸之会肌肤之表尚不能自振况望其饬躬澡行以希圣贤之域者哉吾道亦大矣斆学必自蒙养始其勿谓洒扫应对进退为末而思有以涵育性情磨礲气质循序而进以要其成尚无负儁卿所以建塾之意世之观吾言者其亦有所兴起也夫

　　戴祖禹墓志铭

　　剡溪戴氏自安道以清隐着其世绪代有显闻祖禹之九世祖始迁居于山阴之西陵曽祖允能宋廸功郎又自西陵居杭祖安国父应垓皆潜徳不仕比三世皆返塟西陵东西州相距一水祖禹时时省墓朝渉而暮至虽居杭志常在西陵也始余初识祖禹时甫弱冠意气巳颕发倾动流辈所居尘市嚣杂然藏书甚富常闭户读书不妄接人事如穷儒宿学遁迹林谷雠校自乐余见輙愧之而祖禹雅善余及张君仲实言论纚纚商搉今古觞咏间作客至或瞪目耸耳移时不出一语余私问其故则曰吾惟不耐与俗子面而能强为言笑乎余尝规之而祖禹终不欲苟刓其方故于时寡合及其死则莫不哀其才命之畸而思如祖禹者何可复得呜呼自古有志之士龃龉一世殁而公论始定者皆可悲也祖禹喜为诗蚤宗太白渐就深沉用少陵法每论诗至厯代正变是非优劣又如老吏持律明烛幽冥其学自经传诸子百家书靡不研究尤嗜古法书名画及鼎彛器物遇胜友则焚香娱翫殆忘渇饥或勉之仕登名于天府以祖禹会稽人俾正其乡学未几移疾归曰吾不能束带趍走俛事上官也大徳己亥余职教御溪祖禹亦寓传贻精舍■〈尸外立内〉邻事简情好益欵洽踰年而余征诣词林与祖禹别且五稔时得书问无恙最后为其子孟淳来请婚余许诺则以明年春谒告还杭祖禹相见大喜曰四千里寻盟真信士也越秋既醮成礼余复还京师又明年孟淳来讣祖禹卒余得书恸恸非为祖禹私也祖禹尝自谓早通释老书交方外士于死生若有悟然卒秘其传书言疾亟犹规画家事周反靡宻且嘱余铭其墓翛然无怛化此岂其验耶世之知祖禹者既鲜而其毕生狷介不群之学百不一试于时而忽以死天之报施善人其可诘哉祖禹名天锡卒以至大戊申八月四日享年四十有五先是再调淮海书院山长未赴娶王氏先十年卒男三人即孟淳次宜孙实孙尚幼女一人未行塟以是年十一月巳未墓在钱塘县大慈乡艮山之原铭曰早萃厥美又学不止天畀之材中恬以黙有媚其独寕亢无回藴能勿摅阻于中涂君子之哀越墓相望尚安尔藏昌厥后哉

　　皇太子赐大庆寿寺田碑

　　圣上嗣登大宝之初储皇星辉俪极动法祖宗亦旣毗赞元化修明百度阳施春育恩洽黎元又谓金仙之道可以启悟羣迷同归正觉沙门梵刹礼尚优崇矧大庆寿寺宻迩禁庭裕皇祝厘之所頋瞻轮奂若慕羮墙寺有赐田仍谕有司蠲其徭征俾诸比丘勤修佛事导迎祯祥具如先朝明命越二年复赐土田为顷者五十申饬恳至光贲丛林先是尝命词垣具书锡赉颠末以谂来者忝职司文翰不敢以庸陋辞仰惟世祖龙徳渊潜豪俊闻风而云附者靡不虚左以待若方外之士则海云师可庵师皆学契真如辨穷实谛世称宿徳奬遇日臻太保刘文正公尊事海云师以研精内典之余入叅硕画出偹颜行至于息戎衣而混文轨逾三十载帝用休嘉彰其师之道俾寺有恒产得广延来者以畅宗风中统建元之明年有编甿张氏以固安新城两县□□□□村之水陆地来献由是庆寿昭被上赐厥租惟不毛之田岁垦而新之乃益滋殖衍沃廪入丰羡园有树栗陇有来牟环布近郊石煤以薪水轮以磨市区子钱之入皆有赢储钟鱼振响檀施如归祠官秘祝匪颁傍午而庆寿寺遂为京师招提之胜嗣皇提封之内又择蓟州渔阳之膏腴以益之自中统距今余四纪而圣祖文孙垂继后先所以嘉惠于缁流甚厚夫上以禄秩任其下而下以忠信亷耻事其上者公卿大夫士之职也庶民则火耕水耨羮藜饭粟疲筋力出租赋以给县官然而丰凶之系乎天贵贱荣辱之系乎人二者常不可必而忧虞以滋乃若世之学浮图者无圭组之累可以怡神旷志而益闲无耒耜之勤可以栖迟其身而常安层檐夏屋儗于封君而役庸之政不及也羣居而食以禅悦为味而通般若食已而游息则念五藴六尘之幻假而修证菩提超清净之佛乗具圆明之正果以祈皇家亿万斯年治隆化洽羣生乐豫是则如来方便设教济世津梁与上之人尊奉释民之意也今住持西云安公行业粹冲器宇弘大克修前美而提点某监寺某等皆具愿力相兹法会故宜洊膺隆渥以贻永久文原既奉命纪述复系以铭铭曰于皇圣元握符阐珍抚御九有涵煦同仁曰瞿昙教启民廸嚚孰尸余祷像设是因锡□所宅丹雘云矫息繇免征俾安弗挠纶言诞敷极于丕冐睠兹兰若昔帝寅祀臻臻释子天门迩止猗欤承华祖考是式申锡土田原隰广斥仓盈庾亿饭于香积谁云耕凿罔知帝力我观性相如糓之滋■〈艹耨〉以慧器法雨沃之勿希焦芽勿砂而糜每饭食顷常是思惟承佛受记为世药师赞佑皇图永配穹祇

　　熊西父瞿梧集序

　　瞿■〈昔隹〉长梧之论始于是非之相形好恶之偏争祸福利害之轇轕紏纷将欲一之以道枢和之以天钧物我两忘以蹈乎大方其言辨矣肆矣然昔之能言者至圣人而极圣人于是非好恶祸福利害之辨若薫莸玉石不可杂糅非旷尚玄同以为道也则吾又安能舍圣人之教而从其说哉异哉熊君西父以瞿梧名其集也谓君泯然忘情于斯世耶何也乃搯擢胃肾络编言语大篇短韵搜竒抉怪自其壮年角逐于艺文之场至于老不厌则君岂忘情斯世者毋乃翫世滑稽姑托于庄周之寓言耶君以文字掇巍科位朝着垂绅委佩骎骎乎华要矣俄而息驾乎跨鹤之山樵牧之与俱而花竹之与娱峩冠皓首讲道唐虞宜其酣歌啸咏尘蜕天游如草木华落而归根乎太虚鼎俎欲谢曲蘖羞称而从太古之味若是乎洵有取于瞿■〈昔隹〉长梧之论岂以昔之梦今之觉乎梦境之相续也物化之不齐也若阴晴朝暮寒暑晦明之翕忽逓代惟变乃不穷而造化之常者与之为无穷此知道之士所以遯世无闷而推其绪余以立言犹足以不朽也君其起瞿■〈昔隹〉长梧而问之

　　送王明之推官北上序

　　余初识王明之于杭时明之方掾江浙省以才猷英敏见知上官寻任钩校其曹事秩满出宰常之晋陵剧邑君至有惠爱号称治办旄倪遮留不可作去思之碑朝廷亦知君可佐大郡决平矧仕杭久尝谙其俗乃命为杭推官自君官晋陵适余被征为词林属言论不相接者逾十载人能道君不以亟疾苛察为髙而能得事情盖豪猾胁息疲懦旡愁寃也去年冬还杭问其事若何则曰有捍禁黩货弊为怨家□语诬服者有盗率其曹伍剽掠为奸而胁从罣误者有司一切文致于法君皆谳问得寔多所原贷诸如斯类甚众厥今吏部铨选能辨疑狱出死罪若干人者皆视所授秩有加若明之者其进用可量哉吾观造化之于物虽风霜肃杀而生之机常流动旡间为吏者故欲轧其生而毒人于死用法或持巧心析律貮端或务刻深若束湿以要时誉甚则市狱以便私自营此其设心皆为造化戾乃若持禄取容旡所可否者为害适相当士不通经不足与论政刑明之既力于善而复以诗书教子其知本也夫余今将适京师乃序以送之且以谕夫持三尺法者至大辛亥二月望日

　　送黄可玉錬师还龙虎山燕集序

　　世传渊明修静入逺公社盖渊明未之言也或疑好事者为之说夫交道贵心知岂复计形迹之异哉而世徒以形迹分尔汝者此交道之所以薄也王谢支许其出处大不同而当时论交者歆羡以为美盖忘势与忘人之势虽晋宋世而古风尚未冺矧渊明修静髙蹈物表而独何疑于逺公相好哉窃意匡庐之胜岂无若斯人者游息其间而余方宦游北南莫获邂逅与交乃今还杭得交黄君可玉甫其学典丽该洽贯儒名老而同归其文章由古训诚若摐金石而奏韶濩钱塘固多胜士而余居甚迩情义尤欵宻一日语余曰吾将暂还龙虎山□七月复来于是吾党暨方外之士凡十有六人酾酒而与之别雍容谈谑羽觞屡集彷佛莲社故事乃用祥月师咏逺公诗分韵以纪胜集夫古者会盟燕集各赋诗道志义相劘切于风教深有助逺公离情证空于释氏不为异而能为二老破戒过溪此义乃甚髙余既序其事复以发同志者一笑云
