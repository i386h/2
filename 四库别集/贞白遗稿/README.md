　　贞白遗稿 

　　别集类五

　　提要 

　　臣等谨案贞白遗稿十巻附显忠录二巻明程通撰通字彦亨贞白其斋名也绩溪人洪武乙丑贡入太学庚午举应天乡试时方遣诸王将兵因以封建策贡士于廷通所对称旨擢第一授辽府纪善进左长史燕王叛后通上书数千言论战守大计永乐初锦衣卫都督纪纲发其事诏械通诣京师与二子俱论死事迹具明史本传所著述凡百余巻悉毁于官后十年其弟赴荆州辽王以所图通像及遗稿授之嘉靖中党禁渐弛其従孙长等乃搜访佚篇裒为六巻附以辽王并同时诸人赠言及行状小传等篇别为四巻天启中其裔孙枢及子应阶又集前后建祠请谥之文为显忠录二巻附缀于末即此本也初通以祖平乆戍陈情乞赐还乡人称其孝及建文中遭逢国难上防御封事而卒以是死人称其忠今陈情之表具在而封事独有题无文葢嘉靖中刻集时犹有所讳而不敢存也封建二策乃其受知于太祖者持议颇正其它诗文亦俱醇朴有法虽所存无多而大节凛然有不仅以词章论者固宜与方练诸集并传不朽矣

　　●贞白遗稿原序 

　　夫威鳯之翔必有遗羽应龙之升必有遗甲英贤之出必有遗言非其遗者贵也有贵于遗者也吾邑长史贞白先生忠孝天植问学渊深昔在建文若方公孝孺周公是修黄公观卓公敬先生偕之游虑无不意气相期凡有著作辄相酬赏则先生之为先生固自有在而诗若文特其一斑耳原集百有余巻当文皇帝靖难兵南下先生躬冒忌讳遂逮诏狱夷其二子而簿録其家以故巻帙悉毁于官即先生之死难讳而不称者亦夺于国威也迨永乐末年诏释诸死难家属而厥弟彦廸走谒荆藩乃王雅重先生因出所蔵遗稿与其画像授之以归最后从孙长公郡丞伯祥公访诸戚里获其赠言并其题咏裒而集之厪数巻盖虽逸者什九存者什一而手泽在焉至是子姓重加编次属序于予以付剞劂而永其传予夙昔慕先生之为人乃今获披遗稿恍挹前芳大都先生因心为言不假雕刻是故叩阍陈情脱祖戎伍而眷渥髙皇则琅琅乎李令伯之遐标也策对封建详恳剀切而匡辅宗社则纚纚乎贾太傅之遗响也疏上北防忠愤激烈而委身殉国则凛凛乎胡枢宻之卓轨也其它雄篇伟制雅调鸿词则又驾轶欧苏躏歩甫白殆浩乎河渎之?潏而灿乎日星之炳耀者也藉非天佑忠孝复振斯文恶能俾公之遗言绪论重光后禩哉至若藩王睿翰与诸贤华章悉附于集一以彰君臣鱼水之欢一以昭寮寀金兰之契庶千百年后知有先生并知有贤王英哲不亦旷世相感也乎或谓披蔵盈尺是恶覩先生之藩篱哉予曰鳯羽龙甲夫人而珍之矧先生忠节固将与天壤同不朽安在非稿弗传乎故因诸公之请爰为之揭其概令知古人之言果有贵于遗者也胡松拜撰

　　○表 

　　陈情乞祖还郷表 

　　监生臣程通诚惶诚恐稽首顿首上言臣闻为臣思忠为子思孝曰忠曰孝臣子立身之大节也盖孝者忠之本忠者孝之推君亲之分虽殊所以事之则一经云事亲孝故忠可移于君此之谓也洪惟天朝混一寰宇偃武功敷文教内而京师外而府州县咸建学校以育羣才其所以教之之术不越乎忠与孝而已臣本刍荛获游县学蒙君之食师之教亦有年矣若夫圣贤道徳之懿固未探其奥而于臣子忠孝之节亦麄识其方今幸升胄监日近清光竭力効忠正在今日苐念臣父已殁止有祖平见今年七十有四洪武初坐法流陜西逺隔四千里相别二十春音问不通安否何似茕茕只影白首畴依其饿也谁为之食寒也谁为之衣疾病也谁为之奉汤药虽有微臣无由顾盼停飡忘寐徒有所思臣欲养父父已不逮臣祖幸存虽欲致菽水之养又不可得兴言及此实切臣心载惟皇上孝治天下泽被黔黎疲癃残疾者得生鳏寡孤独者得养况臣祖尫羸孤苦尤为可怜且臣无父而有祖祖即父也祖无子而有孙孙犹子也岂有父子骨肉坐视其苦恬然无忧乌在其为孝乎臣窃念汉齐太仓令淳于意有罪当刑其少女缇萦上书赎罪文帝矜其情而免之且缇萦区区女子尚能如是臣虽不才为一介书生不能以一言上闻岂无媿于缇萦乎矧我圣天子在上明同日月无不照临徳配乾坤无不覆帱量若河海靡不涵容祖述乎尧舜宪章乎文武岂汉文之比乎臣是以敢布微忱冒干圣听愿乞臣祖回还以终余年使臣出则事君入则养祖上不负君师教养之恩下得全臣子忠孝之道倘蒙矜愍允臣所言不胜犬马怖惧之至臣通诚惶诚恐稽首顿首谨言洪武二十三年正月十五日国子监监生臣程通谨陈伏候勅旨奉圣旨孝哉若人该部即释平伍籍 

　　○策 

　　封建策一 

　　臣对臣闻圣王之治天下也必聨亲踈之谊而权其重必察安危之故而酌其宜故选贤任能不惟其方而敦睦之恩有独先焉岂为独私亦曰重我本支以示天下不可拔之势安内攘外不专用武而边陲之防有独饬焉岂其为黩亦曰谋贵万全以示天下不可犯之威三代之盛率由此道后世不能推原本始权厥利病而妄以己私姗笑之胡不以往事征焉夫周之封国八百而同姓者居其强半若过于厚然卒能共奬王室而控卫天子徐戎之兴鲁侯一驾而天子不惊犬戎之乱晋文入卫而周室再造固以其臣之习于兵亦以周之所以恵同姓者至渥也东迁之后天下已无王矣然桓文更霸而晋为长秦楚之君俯首听命而莫敢睥睨王室者固以周徳之入人深而亦以慑于周亲之有人也由此观之亲亲之恩可不笃哉李斯不察以封建为树兵而草菅同姓曽不得方面而托足焉卒之陈渉倡乱天下景从而子孙不复振者患在于亲亲耶否耶汉髙戒秦之孤立而封三庶孽割天下半致使僣拟无涯酿成七国之变则以分封无度惩噎者之过也然诸吕为难而左袒一呼六军随灭之者则以内倡有刘章外应有刘肥而髙祖之所以封齐者其势优也由是以谈亲亲之恩可不笃哉自主父偃之说行虽同姓无内变然以踈间亲吹毛求疵洗垢刮瘢诸王莫能保其首领而本支单弱王莽遂敢肆其邪心焉乡使诸侯王强明如故虽逆莽亦胡敢然哉然南阳一倡而故鼎复返者则以汉之子孙其丽不亿而汉初之所以养之者隆也由是以谈亲亲之恩可不笃哉曺冏有言百足之虫至死不僵以其扶之者众旨哉言也其见审矣唐宋以来虽同姓蔓延而事权不假惟习于豢养之中不悉文武之畧故逮其末造不闻有同姓诸王能为天子声大义靖大乱者岂其忠孝之心独冺哉权藉不在也诗曰价人维藩宗子维城又曰凡今之人莫如兄弟岂无谓之言耶故为政而薄于本支是自撤其城藩而割其手足也欲以望国祚之灵长也亦难矣世所称天潢之派盘石之宗者非宗室耶假令天潢不流则利泽不衍盘石不固则地轴不宁虽其法制毕具亦何益也今之理身者必先心腹而肺腑其卫也四体其运也肺腑不安心胡能以独宁而四体又胡能以独运同姓者非天子之肺腑亲耶奈之何欲理身者而薄之肺腑也臣愚以为本支之臣固不可假之太重亦不可制之太轻太重则骄矜而不逊必有非望之想汉之七国是已太轻则懦弱而难立必无振复之几唐宋之末造是已斟酌于轻重之间禄足以周其家而权亦足以监其众傅有董贾之贤弼无贯髙之桀则岂无鋭志文学如河间献王赞我文治者乎奋志武勇如齐王肥捍我王畿者乎而居重驭轻潜消天下奸雄之志又岂无三代之盛者乎昔人有言生子当置之邹鲁礼义之鄊不当置之戎翟之俗是固有然者臣则以为当置之艰阻备尝之中不当置之膏肥美丽之地方今蒙古虽北徙犹时侵扰臣以非诸亲王莫可镇此地者一则时有烽火干戈之警则惧惧则不遑为逸乐之图一则身亲风霜战斗之苦则戒戒则必思为善后之策一则视君亦亲视国亦家而捍卫必固一则戎兵克诘师旅常振而缓急有赖一则豪臣猛将慑懿亲之望重而干城之寄坚一则边陲荒裔惮藩国之势尊而窥伺之心熄此一举而数善备者必能为陛下当北顾之忧埀宗社之宁矣虽埀流之乆或有意料之所不及者要之与其弊之一不若利之百而监古揆今兴衰补弊陛下又必有经制之宜在也臣愿陛下取前之所以得剔前之所以失惩孤立之祸收百足之福絶李斯之刻削酌主父偃之权宜毋以姑息恣恩毋以肺腑生疑俾阃外之寄常畏周亲文武之畧备自帝胤则天潢衍庆盘石永宁维城维藩绵之百世而勿替矣臣草茅之见如斯伏惟陛下采纳焉臣谨对
