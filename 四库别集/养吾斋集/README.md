　　提要
　　养吾斋集三十二卷　　别集类四【元】
　　臣等谨案养吾斋集三十二卷元刘将孙撰将孙字尚友庐陵人辰翁之子尝为延平教官临汀书院山长辰翁已以文名于宋末当文体冗滥之余欲矫以清新幽隽故所著书多标举纎巧而所作亦多以诘屈为奇然蹊迳独开亦遂别自成家不可磨灭将孙濡染家学颇习父风故当日有小须之目吴澄为作集序谓其浩瀚演迤自成为尚友之文如苏洵之有苏轼曾以立序则谓渊源所自淹贯千古今观其感遇诸作效陈子昻张九龄虽音莭不同而寄托深逺时有名理近体亦多佳句序记碑志诸文虽伤于繁富字句亦间涉钩棘然叙事婉曲善言情欵具有其父之所短亦未尝不具有其父之所长又宋元之际故老遗民如胡求鱼聂济之问学赵文刘岳申之文章郭汝介涂世俊之孝行多不见于他书独是集能具其颠末亦颇赖以传至所云欧苏起而常变极于化伊洛兴而讲贯达于粹然尚文者不能畅于理説理者不能推之文其言深中宋人之弊又云时文之精即古文之理韩栁欧苏皆以时文擅名其后为古文如取之固有皇甫湜樊绍述尹洙穆修诸家宁无隽字妙句幽情苦思所以不得与韩欧并时文有不及焉故也其言尤足以砭髙语奇古而不能文从字顺之病虽所作不尽践其言要不能不谓之通论也据曾以立序四十卷而自明以来罕见藏弆惟周南瑞天下同文集首有将孙序一篇中録其文一篇顾嗣立元诗选仅载其诗一首盖亡佚久矣今采永乐大典所载辑为三十二卷以偹文章之一格亦欧阳修偶思螺蛤之意耳乾隆四十六年九月恭校上

　　原序
　　惟天下之公言为不容泯而世之为论至久而后定则人心之始蔽于其私也古文自先秦以来汉推司马班扬唐称韩柳宋数欧苏此天下之公言也而扬子云自谓五百年乃有知吾者韩昌黎集叙云时人始而惊中而笑且排终而翕然随以定苏子瞻叙六一居士集亦言世之不悦者哗而攻之能折困其身而不能屈其言呜呼古今文章议论至子云韩欧犹不能使人言之必无蔽于其私然卒亦莫能以掩其所不容泯者是故頼有君子之论也庐陵自欧阳文忠公倡古文为学者师后百有余年而殿讲巽斋先生太博须溪先生相继以雄文大笔儗于欧尽常苏尽变由是海内之推言文章者必以庐陵为宗先生为须溪先生家督天才宏澹蚤从严侍留古心丞相昭文馆已喧百寮诵佳句为时文輙翘然度越流辈渊源所自淹贯千古故先生之于文浑浩变化长驱直逐涛惊浪骇虎跃龙腾有倾河倒海之势而来者望洋不可涯涘同时宿学知名之士雅相推让必经论著共谓此须溪氏之文也四方慕一言以自异者必相与谋曰非庐陵先生而谁归闲居终日更请交谒未尝经意停笔而原理道性数典考文连引古今证据经史譬之化工随物赋形各极痛快又未有不靡然服也夫攒眉拥鼻血指呕心披拣之而获寸金组织之而成碎锦此非世所乏也至语大制作稽典谟而征雅颂耀金石而播韶钧卓然成一家之言非正音大雅无以与此视钩棘以为艰深蹇拙以为简古数急以为严刻露以为劲峭施之先生真足汗流籍湜仆命骚耳而窘于步者常疾人之速乏于才者毎忘人之多狭于施者輙惮人之广故其得于公论为不少矣而私窃异论亦或有焉抑孰知源泉万斛初不择地而出其浩乎沛然者诚得于天分之本自髙非徒力所能至也方之苏氏父子将追坡翁以翺翔文章固自有定价亦恶可以无君子之论哉西山邵公济之论柳文以为文之行虽系其所载犹有待焉为之纪次非其门人故吏则其亲戚朋友自昔然矣闻礼之在门辱先生爱而敎之顾无所肖似不足纪次惟是先生之文其不容泯者在人心公论四方后来愿见者众乃编彚为集凡四十卷爰锓诸梓以俟君子其散落人间者将广辑而续刋之不揆僭妄诚知无所逃罪抑编次之歳月不可以不之志焉而非私言也先生须溪刘氏讳将孙字尚友学者称养吾先生云门人曽闻礼序
　　西汉之文几三代品其髙下贾太傅司马太史第一汉文歴八代浸敝而唐之二子兴唐文歴五代复敝而宋之五子出宋文人称欧苏盖举先后二人言尔欧而下苏而上老苏曽王未易偏有所取舍如道统之传孔孟而顔曽子思固在其中岂三子不足以绍孔而劣于孟哉叙古文之统其必曰唐韩柳二子宋欧阳苏曽王苏五子也宋迁江南百五十年诸儒孰不欲以文自名可追配五子者谁与国初庐陵刘防孟氏突兀而起一时气焰震耀逺迩乡人尊之比于欧阳其子尚友式克嗣响夫一家二文人由汉迄今仅见眉山二苏而尚友之嗣防孟不忝子瞻之嗣明允呜呼盛矣然欧实宗韩明允乃以为非韩子之文而欧阳子之文刘与欧为同乡而不专宗欧予亦以为非欧阳子之文而刘子之文也明允雄浑竒峭永叔儗以荀卿直跻之周秦间子瞻长江大河一防千里语者曰子瞻之文非明允之文也若防孟之諔诡变化而尚友之浩瀚演迤语者亦曰尚友之文非防孟之文则为知言也已呜呼百世之下有深于文者其亦然予斯论否乎尚友之门人曽闻礼编辑其文自附于韩门李汉予与尚友善素喜其文辞又嘉刘门之有南纪也是以序其卷首云泰定丙寅十二月之望临川吴澂序
　　先兄养吾先生自少时侍先君子于古心江文忠公馆中学为文出语惊人喧众口逹朝路及长习程文屡魁郷校占前名老儒匠手敬服不惟抗衡且避其锐皆以小须称之先君晚年倦踈笔墨凡酬应者悉属之先君弃世之后其文益工四方之求者如先君焉笔力豪赡衮衮莫竭议论援据一出于正而能俯仰随时谐俗不惊挥毫漫兴纸尽词穷旁观唶唶脱手若忘故所著述虽多而存藳亦少其壻曽以立慨然于乐天冩本传两家子孙珍藏之语遂编校刻而传之以立亲炙先兄既久才学日近诗文字法皆已逼真故于斯文不啻两手分也今其所编者多所意授而非效李汉辈徼名者也刻梓已成区区因得附名不朽并述其编次之意如此若夫先兄之文在人心公论非予浅陋者所能赞也泰定乙丑十月季弟参拜书

　　黒猿赋【并序】
　　当庭之前楹丰草隂隂有猿时而防息以为公娯命曰黒猿以嫓赤鳯并为之赋曰
　　参躔之精剑翁之孙潜心子云之易参色道妙之门黒可以鉴黔过于漆染烟化苍裁云守墨体昆仑之童乌面玉毫之团白臂通卢弓之长皮鬬鬒髪之泽龙战野之合气鼋里骨之戴色岂乌孙之异种抑元夫之灵质雾养润而削黳日舒光而藏碧墨卿之翠欲流黒灵之冥逾鐡青玉环之返防醉道士之变石或腾而升或盘而嬉聨络东西之步扶摇下上之枝衣被頴之光发舒老苍之竒团鸦色之飏采淡螺丸之弄姿惟兹之俦自表于羣白敎射于矢人赤骇梦于步兵青表道康之童金记峡山之英毎与人而相即浩变化之留名况夫与天一同其体返无色幽其光固湼之而不可缁亦闇然而有余章门列山房之玩庭止文字之祥勿长啸而懐寥廓勿捷走而志飞扬优游右军砚池之上食息笑啼墨竹之旁幸不为河东先生之所憎是可继玉堂仙人之所谓黒衣郎也

　　寒岩词
　　余来延平首识沙阳张寒岩题其诗去之十八年忽相遇于剑溪之上出旧题一笑且示以寒岩图邀予为之赋溪山然相视白首余固栖迟俛仰无以自异于昔而君亦萧条独徃方欲托之此以老也其能无所感耶因倚风而为之词曰
　　岩之上兮风萧然苍苍之木兮与我齐年东风过之兮碧叶华鲜岁寒兮伴我我不遐兮载笑载言岩不寒兮以我我之寒兮吟肩我不厌其寒兮岩乎岩乎吾与尔兮周旋
　　岩之下兮春草既緑风日岂不暖兮十寒一暴不改吾容兮植杖兮岩之曲岂不懐兮四方之人兮如玉宁此寒兮我独秋风兮卷芳岩不动兮吟客比屋寒不可加兮如此而足笑四方兮春梦斯岩兮宁不我卜
　　山悠悠兮入云荒荒兮无木仙之人兮何之浩东西兮何所于屋我欲从之兮石齧我足白云兮遐瞩毋我逝兮于我乎宿
　　仙之人兮何年昔仙未来兮山与谁兮周旋仙不知其氏名兮山乃以仙而传傥非是山兮仙亦泯然他山无仙兮亦浩荡而蝉聨仙不少留兮山空而云怨欲招仙兮问山山苍苍兮不能言
　　我昔对山兮三霜忽别山兮十五年而强我知我俗兮不敢与仙兮翺翔美人兮招我月露为之旌兮云锦为之囊仙之人兮来思傥分我兮余芳
　　五言古诗一
　　感遇
　　六经之为文其文汪以洋浩然如河汉万古流清光斯文一变史理绌气始张竒字抉幽陈説极焜煌岂不雄千古洪水之汤汤耳目虽可骇意象焉得望况如占毕者呐呐岂文章凄其懐古心宇宙何微茫
　　读书初何为耿耿各有意精神在宇宙沙滓为文字圣贤不自遇存此俟来世悠悠空言载未坠犹既坠徒然三千年竟未见行事谁知此空言亦复有时废来者愈难知伤哉圣贤志
　　道本无一言无言乃为天传心不传言所谓以是传回愚参也唯由赐未足贤多言固破碎不言非空向非载六籍何用行千年沉酣文义间于此求大全赫然废兴统欲夺王伯权何斯黄面老不立言语禅悟入得近似防解驰聨翩尽驱诸夏豪堕落彼中边悄悄八九公一二挽百千书多言自长心苦辨愈坚至今讲糟粕未得离蹄筌
　　王言贵深浑此道何乆荒断从西汉下偶俪为辞章剪截鬬纎巧何异于优倡代言袭一律设科号词场个字夸歇后廋词竞遗忘缀拾蚁注字套类蜂分房谓此台阁体哀哉虞夏商我欲揭古书使识谟洋洋又恐仿大诰句字摹偏旁
　　文章犹小技何况诗云云沛然本情性以是列之经赓歌五字始雅颂谱律声苏李非骚客醻倡流中情噫嘻建安来雅道日以湮晋人善语言其言明且清少许胜多多飘萧欲通灵使其入韵语岂但诸子鸣安得三谢辞逺与陶阮并唐风晚逾陋宋作髙入论遂令后来者末流骋纵横髙者效选体下者唐作程
　　古兴呈签事王朋益
　　鳯凰择朝阳千仞不下视羣狡鳯无能百鸟鳯独贵丹山有梧桐中夜降甘露苟无饥与渴洁身故为瑞猛虎啸隂风百里为惨凄深山复长谷猎者安所施威行岂顾穽变或所不疑养威贵善用慎勿轻出之豺狼煽为羣朝暮据路岐樵苏不敢度吮血为之肥噬吞各饱去逺引如能飞朝朝南山猎独得狐兎归蜘蛛结小网密密覆修竹徐行罥我巾仰视我目局蹐不能言低徊入复出天明属不照风雨曷无力孔雀出炎海毛羽异复驯舒翼为光采娯我主与宾置之雕笼中抚玩诚足珍讵知毛羽异其毒能杀人鹘或好义者鸷搏有异才同类有不平极力为防摧江南多蝮蟒盗抟情可哀萧条呼鹰台安得尔一来一乌延秋啼百乌毕逋应山中倾鹊巢横絶无敢竞飞鸣啄腥腐意得志逾逞哀哀失巢鹊飞鸣不能进巢蜂巢髙髙时出挠行客扑之不能尽弹之愈飞集举火徃攻之飏去不留迹蜂虽蹔得息乔木损可惜蠧生文字间终老无所迁方册穿欲罄滞死不得僊年年七月曝薄福亦莫延不饮复不食性命絶可怜海燕重归来春风漫能语故人何之今人不我处风尘正眯目荆棘动成阻安得画堂成衔泥入君户
