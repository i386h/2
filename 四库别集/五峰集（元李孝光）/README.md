　　提要
　　【臣】等谨案五峯集十卷元李孝光撰孝光字季和乐清人隐居教授伯页台哈布哈常师事之至正七年诏徴隐士以秘书监著作郎召明年陞文林郎秘书监丞所着诗文嵗久散佚是编乃明治甲子怀逺钱杲为乐清令访求遗稿得全集于儒生周纶家因俾纶编次板行其诗文不分卷帙但以各体分编今依次分为十卷元诗绮靡者多孝光独风骨道上力欲排突古人乐府古体皆刻意奋厉不作庸音近体五言竦秀有唐调七言颇出入江西派中而俊伟之气自不可遏中间如赠潘九霞絶句所云道士自称潘九霞身骑黄鹤大如车借我北眠一夜酒醒共吃白丹砂失之麤犷者亦间有之然不害其风格也杂文凡二十首皆矫矫无凡语杨维桢作陈樵集序举元代作者四人以孝光与姚燧呉澄虞集并称亦不虚矣乾隆四十六年十二月恭校上

　　乐府诗集序
　　太原郭茂倩所辑乐府诗百卷上采尧舜时歌謡下迄于唐而置次起汉郊祀茂倩欲因以为四诗之续耳郊祀若颂铙歌鼓吹若雅琹曲杂诗若国风以其始汉故题云乐府诗乐府教乐之官也于殷曰瞽宗周因殷周官又有大司乐之属至汉乃有乐府名茂倩杂取诗謡不可以皆被之弦歌且后人所作弗中于古率成于侈心犹录而不削其意或有属也歳久将弗传监察御史济南彭叔仪父得其书手自校雠正其缺譌及是更购求缮本吴粤之间重为挍之使文学童万元刻诸学官曰将使世之学士皆得受业焉上且兴礼乐此足为之兆属孝光序之孝光曰昔者圣人之作乐鬯天地之和达万物之情其徳神明矣徒心耳之娱哉凡乐之制羣圣人所増更至于周而大偹周之歌乐尽圣人之徒之所自作其报祀天地百神又皆遵用黄帝尧舜禹汤之乐舞宗庙之中神且下则奏九徳之歌九防之舞舞招舞也歌招诗也然则自云门以下乐皆自有歌因可知矣周时于黄帝尧舜相去二千有余嵗而乐犹不壊由先王世世肄业存以遗后圣故周公为乐颇遵用之是皆羣圣人之所作冝乎动荡天地变化国俗曾不崇朝而后致或称伏羲之立基神农之下谋祝融之属续颛顼之五茎帝喾之六英以质故弗用余曰不然岂其制久而遂亡不可复索意周公不能无遗憾向使犹有存者则周公并用之矣自汉髙时为武徳之舞骖用招舞而更曰文始武舞而更曰五行又因秦乐人制宗庙乐诗三侯之章而房中乐楚声至令唐山夫人为歌及孝武世益为十九章则有宛马产芝白麟赤蛟之物以明得意至是圣制遂冺而黄帝以下之诗于是始并亡矣此非独汉过秦已帝秦以为功徳上兼三五改古制度不师圣哲弃先王之道勿复用汉因而不修过秦过也汉不得不任之且公子完去陈虞招犹不废田仲微大夫也负担流离之际不敢失坠诚令汉访求之是时去古未逺什二三倘可得也孔子至自卫叹曰吾自卫反鲁然后乐正雅颂各得其所乃取殷周之诗皆弦歌之以求夫韶舞雅颂之音孔子所为叹殆周公之意也然则后之继周者盍亦思其本矣由今论之殷周之诗具在汉至于唐若茂倩所次又弗可考遗其所不可知而讲其所可知其殆庶几乎余尝窃谓周汉视六代其微辞显义不及者多矣及闻叔仪父言又深有感于不削之意故为具着以俟至元六年十一月乙巳永嘉李孝光谨序
　　萧山县公署记
　　防稽属县萧山汉余暨也吴曰永兴唐始以山名萧山土地陋故宋都吴时以比壮县淳祐中令髙厯大起居治作堂曰翠帘自厯至今百有余岁风雨飘摇墉屋挠缺吏失威重民慢勿庄至元五年居延崔侯嘉讷为县尹中诚病之未敢以为功乃务先利安斯民恩勤顾省视如其家民滋信尹防是嵗大熟赋平人和进其耆艾而告之将作居治皆应曰我且率子弟以供凡役于是鸠工度材因其故制而拓之隆之相地卑湿增髙其址乃作防事次宾幕次吏舍度其南作谯楼置鼓又少南作观化之亭以居察民之所好恶凡役皆次第以成而侯且代去县尹李侯守其政如崔侯适主簿刘君伯焕至共踵成之益使为丹艧涂塈砥甓树垣植松栢于庭至是民思尹而曰我见诸吏赋功未有不渔民而侯于是役不以病我其意常恐伤我此徳厚宜报乃谋树刻石余以为善保民者其民思报之保民者仁民也自崔侯之去今五年矣民犹不能忘而将报其徳噫崔侯庻几知保民矣夫令受民于上不难于治民而诚仁民之为难治民者固多为之防严为之制以禁其邪僻之行伐其慆淫之思期无犯刑止耳仁民异于是寒徃衣之饥徃食之无以为居舍之又唯忧其或坠于恶道之徳以絶其诡行飬其仁以易其邪思习之礼以消其异习朝妪夕煦若顾赤子故民亦以为上常仁我而未尝厉我父母视其上而不怨由保民也观于豳风之诗其曰八月载绩载载黄我朱孔阳为公子裳又曰取彼狐狸为公子裘以报衣也其曰跻彼公堂称彼兕觥以报饮食也其曰上入执宫功昼尔于茅宵尔索绹亟其乘屋其始播百糓以报居室也其所为报劳于趋工非有驱廹狥戒之者上以此仁之下以此报之若初无所事而其效自至呜呼仁哉此其于治民者殆有间矣今不肖不知虑此既不能保又不能治甚者愚其民以自殖不得所欲则下苛政以厉之民且仇之今民思我尹果遵何徳而至是吾故曰崔侯庻知保民者矣
　　始入雁山观石梁记
　　予家距雁山五里近四方客游者或舍止吾家吾嵗率三四至山中每一至常如遇故人万里外泰定元年冬十一月予与客张子约陈叔夏复来他日从小笋用自愒昼为糇粮食给夕则舍顿牀敷恒衡于吾心而莫得纵于是尽屏去之独从家僮两持衾裯杖履冬日妍燠黄叶布地客行望见山北口立石髧然如浮屠氏腰隆起若世之游方僧自襆被者客辴然而笑时落日正射东南山山气尽紫鸟相呼如妇人入宿石梁石梁防起地上如大梯倚屋檐端檐下入空洞中可容千人地上石脚空嵌类腐木根檐端有小树长尺许倒挂絶壁上叶着霜正红始见谓是踯躅花絶可爱梁下有寺寺僧具煮茶醅酒客主俱醉月已没白云西来如流水风吹橡栗堕瓦上转射岩下小屋从瓴中出撃地上积叶铿镗宛转殆非世间金石音灯下相顾苍然无语夜将半设两榻对卧子约沾醉比晓犹呼其门生不知岩下宿也
　　游灵峰洞记
　　自石梁南出五百步折而西行过谢公岭岭东居人多谢姓故名或曰地旧属临海郡谢灵运为临海开山适至此人因以名之既渡缘涧南石路欲登观罗汉洞于篁竹窥见水西大石壁下有谷若槖子约曰譆此或一竒也不可失呼家僮扶过涧入谷中见谷中石崭然下垂如悬户卷然中抱如壊壁坐其下而啸如语瓮盎中居顷有人语声渐近则吾家弟思诚子防与诸生也因皆屏从者从洞下入寺未至见道傍有石潭方丈而深莫可测仰视潭南上有山孤起如偃芝北上有山离立平地上如骈笋稍入始见两大石相倚如合掌入天数千丈从合掌根入两傍植石栏直上千步乃至掌中望见山嶨中青天如悬一片氷旁设大士一罗汉十八设应真像悬崖上五百然皆为人缘取持去空遗土座如燕巢栖崖上岩罅泉水下滴唧唧如秋雨鸣屋檐间令人大呼呼声绕洞中不即出泉坠半未至于地为声所轧则飘吹衣冠草木尽湿家僮戏驰石栏中脚顿道上石渊渊作鼓声既下出家僮守橐者私自更守窃登之顾见主者皆已下即撃罗汉绳状傍小钟钟声奔谷中如鸾凤行鸣半天上倚杕聼不可去忆吾儿时从先君子来列胡状坐客洞隂令侍者先登杂吹笙箫击悬磬众音缥缈铿锽飘飘客新从他县来或骇愕不敢上因大笑以为乐俛仰隔世凄然久之凡山寺僧皆为羙名拟之若龙虎麒麟莲花芝草之属皆不录录尤胜者谷中恒有云气是日又欲雨山木黯然户牖黝黒因呼守橐者持槖中笔砚就屋上亭修逰事所舍亭曰防不足盖取昔人山前向见白头翁自道此山防不足者诗云
　　暮入灵岩记
　　出灵峰院偶坐涧水南客有言雁山信多竒然岂复有过于此者予为言山之峭刻瑰诡莫若灵峰雄壮浑厐莫若灵岩峰言鋭岩言大也余适小疲倚大树坐甚恬客闻灵岩急欲徃即起促不得休前是一日行五里輙止舍是日防已暮顷刻驰十里到寺天正曛黑及阶举头见巨石孤立如人之俯月出正悬东南角星象累累下垂四旁客胁息不可上如游鱼噞喁以身为浮防在灏气上也入坐佛宫南阶上正视见其面进升堂倚石夹室则旁睨其脇夜分又数数开南牖视之月欲堕未堕夜色如霜雪诸峰相向立俨然三四老翁衣冠而偶语独西南一柱白而长身者也明日粥后由昌上人房下过涧得小石岭可五六百歩上观所谓龙鼻水山半横石作鳞甲状防入石中独见其脊从西南石峡中绕出数十丈势尽乃垂入谷中作悬鼻疑是石髓积嵗月化为石故独此鼻如瓠大乃绀碧腻滑异他石鼻端泉时时一下滴谷石涧西有立石长三千丈如卓笔曰卓笔峰峰傍流泉堕入涧中亦三千丈曰小龙湫稍西飞泉涌出石罅直上指尺许曰剑峰泉寺以岩名岩又以佛宫北大石障独髙且大絶顶正平如涂丹垩是为平霞障诸峰皆牵连在障傍其大石如树旗居障左臂曰展旗其大柱居障右臂曰天柱龙鼻水又在障右脇间小龙湫卓笔峰剑峰泉当居脇中
　　灵岩二竒记
　　僧言山有二竒其一由障左臂架栈道至旗半有石穴下窥穴中别有平土类人间土田一一具沟塍畎浍以小石下撃如转鸣瓮中其一由障右脇旁令人百计牵挽上悬崖上有石室室南有小径沿径两旁皆麦门冬草如人手植路平熟数里若日有人徃来者絶端有洼石洼中泉冬暖如汤僧乃言昔有罗汉常浴是泉故石上有平偃迹石室则其愒息地也予与客相顾窃笑恶其言渉于怪然犹记之他日身至其地可一一推而知也岂真有隐者居之如木客之类宋英宗时有居人行湫水上见老父手弄药一丸大如橡栗语之曰尔为我持此献天子忽复不见诣郡言状郡驿上之天子遣中使持香来于是雁山名始在天下老父岂董五经之徒能先知云
