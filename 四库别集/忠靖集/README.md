　　提要 

　　臣等谨案忠靖集六巻明夏原吉撰原吉字维喆湘阴人以乡荐游太学选授户部主事官至户部尚书谥忠靖事迹具明史本传原吉诗文集六巻载于明史艺文志与此集巻数相合盖即旧本后附遗事一巻为其孙廷章所辑刋板久佚此本乃国朝康熙乙酉潘宗洛提督湖广学政时得其裔孙所藏重为校刋前有杨溥序称其诗文平实雅淡不事华靡虽原吉以政事着不以文章着洪永之际作者如林固不能与宋濂王袆诸人齐驱方驾然致用之言疏通畅达以肩随杨士奇黄淮等殆可无愧色矣

　　●忠靖集原序 

　　少保兼太子太师户部尚书长沙夏公诗文遗稿常州太守桂林莫君子朴以其俸余锓梓公之子尚寳司丞瑄请予序公之徳存乎人望隆一时有以服人之心感人之情况所寓诸诗文皆本学问亦不可泯也公儒家子天性眀敏而温厚加以诗书涵濡之以成其器在邑里时即有善誉洪武中由乡荐入太学试地官属各司疑难徃质问之而悉以理折衷不以利害为心遂有声于朝未几自部僚超拜亚卿寻升大司徒经赋之政寛而有制国用以充民不告乏遂驰声于天下立朝四十年余每议大政务从寛厚处同官未尝失色待僚属取其所长略其所短以是人多徳之宣宗皇帝为皇太孙时公受命辅导三朝实録皆预监修公于理有定见故其所行夷险无间人以为难列秩六卿位次三公人不以为过时论朝之大臣有徳量者以公为冠卒之日朝野惜之追赠太师谥忠靖録用其子此其始终大概也诗文平实雅淡不事华靡观者亦可想见其为人也公在永乐中摄冬官政子朴尝为冬官郎中相知为深在常爼豆之事则有司存子朴厯官已四考政和民浃用心于此良厚矣又能使公诗文得传闻于人殆知言者哉予亦湖湘人尝得与公同事用是书之简首云正统八年岁次癸亥春二月榖旦光禄大夫柱国少保礼部尚书兼武英殿大学士国史总裁同知经筵事南郡杨溥序

　　河清（并序） 

　　永乐二年冬十二月十七日至永乐三年春正月十八日黄河水清凡数百里于是秦王及髙平王相继奉表称贺上谦让弗居羣臣固请咸谓不可不承天心以虚灵贶遂俯徇舆情受贺于奉天殿庆协神人欢腾朝野猗欤盛哉诚亘古以来非常之竒遇也臣伏念自古王者政平地出醴泉圣人在位海不扬波祯祥所见必有非偶然者矧今皇上以天纵圣神削平内难简贤命良省刑薄赋恩覃乎逺迩德被乎虫鱼至和氤氲充满无间由是嘉禾生仁兽至野蠒成神龟出然犹未足以擅厥美乃至黄河水清绵亘数百里岂非天意欲以彰皇上之洁白一心澄清九有内体干德外着阳刚故特以此而为休征之应哉是葢不独为皇上一身之庆而且为华夏亿兆民之庆不独为朝廷今日之祥而且为宗社千万世之祥矣臣原吉忝缀六卿获瞻盛事虽匪才浅学莫克称扬然忻忭之余不容自黙谨拜手稽首而献颂曰

　　平安南（并序） 

　　圣天子以皇天眷顾之隆太祖髙皇帝付托之重华夷臣民系属之深不容释乃御大厯服典司神人纲举目张礼明乐备品物咸遂福祥并臻五年于兹矣而安南贼臣黎季牦戕暴其君鱼肉其民侵逼邻邦侮辱朝使皇上以海岛蛮荒不可律以中国宪度逭诛旷罚久矣季牦不知悔祸益肆奸狂于是内而百官外而占城（国名）合辞请讨皇上乃徇舆情命大将帅师征之中军捣其中左军折其翼右军掎其角而占城舟师复扼其奔逸之路季牦自知罪在不赦冒死力抗然而上下解体兵败身縳其胁从将领边逺部落悉皆蒲伏乞哀受命于吏请比内郡乃立都指挥布政按察诸司选用守令以抚安其众累世逋冦一朝剪除报至阙庭内外称庆臣按周宣王平淮夷唐太宗擒颉利当时文人咸形诸赋咏以纪宏休臣职备民曹敢不采摭事实揄扬帝载发为歌颂以显示无极颂曰

　　瑞应驺虞（并序） 

　　恭惟皇帝陛下缵承大统克勤庶政声教诞敷仁恩旁洽天地既位万物自育而瑞应骈臻不可殚记乃宣德四年春滁之来安有驺虞二见于石固山南京守臣襄城伯臣李隆以之献于阙下诏赐羣臣观之葢猊首虎躯白质黒章修尾隅目而其性甚驯真盛世之瑞物也臣原吉退而稽诸载籍曰驺虞仁兽也天下太平人君有至信之德则应窃尝考之往古惟见咏于周之诗人自时厥后千数百年之间旷焉莫之覩我朝自太祖髙皇帝肇兴鸿业至于太宗文皇帝绍承厥绪积德累仁至深至厚海内康宁四方无警然后驺虞迭出臣获见焉及我皇上践祚以来礼乐刑政号令法度之施一遵祖宗之旧与民为信无所更革故和气充周而兹仁兽复见于滁滁固畿内之地且又两瑞并出匹休俪美何其盛哉圣德之所感孚皇天之所眷佑于此端可见矣昔者白雉献于越裳天马来自西域犹且纪诸策书见于歌咏传诵至今矧兹嘉瑞出于中国畿内之近者乎是宜被之声诗以昭圣德垂示无穷臣虽拙于文词然躬覩盛美不敢以黙谨拜手稽首而献颂曰
