　　提要
　　
　　臣等谨案西隠集十巻明宋讷撰讷字仲敏滑县人元至正中举进士任盐山尹弃官归明初征为国子助教升文渊阁大学士迁国子祭酒卒于官正徳中追谥文恪事迹具明史本传刘三吾撰讷墓志称所著西隠集十七巻而明史艺文志黄虞稷千顷堂书目俱作十巻此本有东莱刘师鲁跋称其集初为上海张趋所手録滑人王崇之令上海从其后求得而刻之岁久漫漶师鲁因鸠工重刻葢即十巻之本岂张趋缮録时又有所删并故与墓志不合欤集前四巻为赋诗后六巻为杂文附以明太祖手勅四道及白云茅屋赋二篇记一篇白云茅屋者讷所筑别墅之名也讷领成均胄子之任师道严正为一时典型文章亦浑厚典雅其奉勅制太学碑极为明祖所赏今具载集中又有壬子岁考试秋闱和北平行省照磨叶叔则诗及秋闱即事诸诗壬子乃洪武六年葢讷尝为北平考官而本传墓志均之未载其过元故宫诗十九首纒绵悱恻有黍离麦秀之思读其词尤足悲其志矣

　　勉斋赋（有序）
　　
　　知阳城县李侯文辉之斋以勉名求言于余余二十年流离不知勉者也言将焉取抑不敢终无言也故赋之愿与侯以共勉云其词曰
　　
　　仙李蟠根逺毓清芬有美一斋勉勉徳人踞高明而养性挟敞朗而頥神将自强以进学期为己以修身于是典籍阅诗书亲性命讨道义论有图有画以琴以樽靡华靡陋不俗不尘嘅人欲之易萌叹至道之难闻念千岐之汹汹苦百虑之纷纷究圣愚之曷异由敬肆之是分行易忽而加戒言易诞而防嗔不惰其常朝夕匪懈不惮其艰夙夜惟寅方其处也有过斯改见贤则尊沈潜乎君臣之义渐渍乎父子之仁钦斯承斯无替乎颠沛念兹在兹莫间乎晨昏迨夫出也严以守已庄以莅民澹以整而着绩俨若思以策勲造次克念而居之也则安战兢自持而行之也则纯效古人之进修企前哲之经纶惟实病之是除惟实知而乃言惟实徳而乃邻岂攘掇其土苴岂剽窃其肤皴起居亦学道无时而不进游观亦学徳无日而不新动靡违而靡愆静常励而常勤有操无忘之汲汲有持无怠之循循心内正而不荒事外制而有伦毎原始而求终敢以伪而乱真内外之交正不偏作止之迭资莫冺非诚也其勉曷有非敬也其勉曷存若夫心广体胖光风霁云两瓯茶余一鼎香焚颜四勿而铭心曾三省而书绅雍雍兮愉愉夭夭兮申申威仪凛兮可观衣冠正兮不羣以杜乎忿欲之源以融乎天地之春故能探性命之奥而入圣贤之门也嗟夫胜已者宜友先觉者宜询常登斋而请益预樽俎之留宾酬酢之间兢兢礼乐笑谈之际孜孜典坟瞻风采之颙颙顾文质之彬彬被容接者登龙门而睥睨覩星凤者望少室而逡廵古战场兮文欲降日五色兮赋莫伸北海避词源之万斛义山让笔力之千钧焕乎瑞时之麟皎乎贡国之珍赋勉斋以自讼聊舒意于斯文
　　
　　松云轩赋
　　
　　为前进士王公勉作有客餐松卧云晦迹离羣抱岁寒之高节挹岩壑之清芬过析城山而弭节访松云轩之主人布袖拂缨藜杖叩门主人于是开轩下榻倒屣迎宾会青松以延坐招白云以侑樽清酬静酢吞香饮醇驱俗却尘话素吐真乃问于主人曰松云名轩义可得闻岂取其秦皇之封受以大夫者将自尊乎抑取其丁固之梦为十八公者遗子孙乎又岂取其山中宰相陶弘景之自怡悦者以为珍乎抑取其东篱处士陶靖节之无心出岫者以为邻乎名一轩而取二物而宾主轻重之间岂亦有所分哉主人乃起而言坐而欣幸先生之慨诺爰详敷而细陈夫松耸壑昂霄非羣木之雕悴而云触石润础乃一气之氤氲云有时而舒巻惟松斯植云有时而出入惟松常存盖有取夫四时在目青青不改有子可食有花可采森风雪乎千丈贯氷霜于千载吾之有松励节贞心轩之得松増辉倍彩柱明堂而无慊于时用梁大厦而有功于真宰斯材固不敢以自负斯志亦不敢以自怠若夫盘龙蛇而不屈栖鸾凤而不骇高仙桂于明月寿灵椿于沧海云漠漠兮有时来云郁郁兮有时在恍重楼而叠阁忽点缀而叆叇悠悠横数幅之绮依依张五色之彩懐佳人兮不忘瞻亲舎而靡懈以至资云之变化以为变化察云之动静以为动静是吾之轩又于云而有待也客乃起坐呼童注酒于锺举我潋灔之香沃子磊落之胸主之有轩轩之有松松之有云聚乎轩中当其霁雪晴烟春日秋风衣冠肃雍襟韵和融高致自许清思无穷延骚人留仙翁来菲薄去肥醲浮翠色于几席泛佳气于帘栊彼苍其髯我饰其容彼变其迹我正其躬收济时之泽于掌握纳后雕之操于丹衷斯时也或起而为栋或出而从龙人与轩有名而有实松与云有用而有功则主人名轩之义岂不日美而日隆乎于是开颜一笑玉山西东松暝云寒月侧樽空
　　
　　椰子酒瓢赋（为知滑县事诸君仲仁作君前雷州遂溪县丞）
　　
　　祝融之荒朱崖之疆有木惟椰花实同芳叶毰毸乎凤尾树彷佛乎槟榔融瘴雨之滋养受海风之吹扬凝鲸波之润泽孕蜃气之光芒竹长节兮下生雀五色兮上翔采一殻之贞姿破半瓢之异常不■〈氵〈土上米下〉〉而玄不老而苍讶琢疑雕离圆遯方谓为匏耶而非济水之具谓为螺也而无掩甲之铓制作合自然之度规模岂杞柳之戕薄临池之荷杯轻随波之羽觞尊藤瘿兮土苴巵竹根兮粃糠憎一杯之鹦鹉妬双盏之鸳鸯斟酌乎真一之酝输写乎波若之扬纳麹生之风味友陆醑之壶觞齐列并陈虽翠勺银罂之属无忝先后争操竞执虽嵇康阮籍之徒亦效奔忙注春色于苍颜添风韵于红妆踈水部之醒何亲酒泉之渴羌素过乎陶匏之质贵饶于金玉之相箕山之许由未弃陋巷之颜子不将若夫公子高楼豪客画堂银屛前列绣幙后张荐琼酥之炙鹅封杏酪之蒸羊緑醅涨竹叶之酿紫霞溢葡萄之浆夜月秦若兰春风杜韦娘或妙舞兮埀手或清歌兮遶梁巧笑仙容转宫腰之猗傩懐春翠袖露玉笋之纎长浇彼磊块之胷倾此潋灔之香银凿落兮在侧金叵罗兮在傍瓢于斯时孰低孰昂负清标与雅致任酣颠而醉狂又若幽人草亭高士山房诗礼之庭翰墨之场逍遥兮兰佩葛巾优雅兮野服云装杜瓮半开香脓酾乎蚁緑郫筒初断色浅泛乎鹅黄缕银丝之鲈鲙斫金膏之蠏匡新洞庭之緑橘嘉糟丘之紫姜冠者数人童子两行再献不假于歌喉一倾直润乎诗肠滴残沥于布袍散余馨于银珰瓢于斯时价比琳琅视金罍而不耻顾瓦盆而有光有美一人邦家俊良典六艺于文教肃三语于宪纲蔚赞画之才华近池上之凤凰出贰遂溪来宰灵昌昔访兹瓢得于雷阳封以溪藤之纸韬以蜀锦之囊逮琴堂之公暇时会友以徜徉精曲糵以作醴洁甘旨而起尝味过杜康客邀宋庠幕宾之丰仪俊逸广文之环佩铿锵挂一幅之墨梅设四位之胡床爇水沉兮霏微烧银烛兮荧煌乃命爱子钥启筠箱出清赏于胜集乐良夜之未央蔬如琼芝果剥栗穰肴佳山雉味美河鲂一酢一酬和敬交彰一奠一举宾主相忘延席上之欢伯拓樽前之醉乡引青州之从事啖汤饼于何郎宜贤侯之特达重此器如圭璋呜呼海中之洲琼山效祥实产椰子大小莫量懿为瓢而为榼亦风土之木强来中华兮万里几山海兮梯航物逺见珍爱之恐伤木灵知毒验之允臧杳穷陬之地理恨考索之未详吾将假之过黄公之垆持之唤西家之墙旷席地而幕天脱声锁而名缰属一瓢于海神追秘怪于渺茫酬一瓢于坡仙吊谪居之闻望尽南海于一吸潄岭岛之风霜倒着白接■〈罒离〉翻污云锦裳揖羣彦兮归来拂醉袖于缣缃承主人之索赋聊染翰以成章愿常加乎洗涤示清白以保藏
　　
　　桃溪图赋为知浚县事项侯如英作（侯台州仙居县人）
　　
　　仙居亘兮台山异境迥兮尘寰种桃花兮千树潴溪水兮一湾玉霄孤峯而逺踞琼台双阙而上蟠拥赤城之飞霞洒沧海之奔湍拔呈露之岩壑峭罗列之峰峦深水澄清而莫测怪石嶙峋而可观通逺近兮縁山腰以为径限内外兮束峡口以为关岂武陵之别源秘此景于人间宜画者之作图极巧思而不刋若夫自成一区夐絶市阛云烟昏旦而异状水石峻激而交抟地外宻而中敞山左合而右环石老兮苔封蔓络人行兮萝挽藤攀飞泉喷林外之珠玑丛竹森石上之琅玕佳木齐荣多楩楠而松桧异草旁缀半蕙芷而芝兰春去兮水流红雨风来兮松生翠澜清致过庐山之高颠险非蜀道之难羡溪上之人家咸棋布而星攅至若灼灼夭桃嫩萼破寒妆春色兮孔多孕阳和兮不悭枝剪绡兮旖旎树裁锦兮斓斑花开而游客簮实成而野人餐深红一簇同黄师塔前之爱浅红千树异玄都观里之看又若悠悠山溪非剡非磻清可濯缨深可投竿草邉之色玉染花底之声丝弹嚣尘断而鱼潜在藻幽谷欵而鸿渐于干曲折有趣逾灌水八愚之溪逺近异势陋龙门八节之滩其中则有柴门竹篱屋覆茅菅烟窦云窗台倚画阑扶杖一翁白发苍颜族立羣氓颜如渥丹雅集五宾清酌幽欢船横渡而不动涟漪人过桥而伫聴潺湲又有姝丽仙姿雾鬓云鬟罗袜尘轻绮袖弓弯翘翠揺揺佩玉珊珊埀髫三童娇戏痴顽鸡鸣日午犬吠昼闲恍出尘之风致俨近古之衣冠彼美鸿儒隠处考盘志希颜孟学絶申韩踵锺王之笔阵登李杜之诗坛继先德之潜辉乐桃溪而盘桓自牧黄牛畊青山以食息闲乗画舫盟白鸥而徃还气和神清心广体胖酒美而壶倾书积而床摊门外之落花而呼童轻扫窗前之新句而洗砚重删视浮云之富贵同一梦于邯郸爰筑野以自适保金玉乎五官大明御天召对金銮诚摅抱负策上治安承制牧浚屛弃儒酸诞承流与宣化歴义胆与忠肝常尊贤而奨善时剔蠧而擿奸懿寳轴之珍藏过雾縠而氷纨劳西斋以持图征予文于巻端披图再阅一倡三叹水依稀乎孙位花彷佛乎邉鸾松石仰法乎张藻山林逺师乎范寛藴紫犹深积翠初干嘉山水之竒特玩姿态而莫殚非天造而地设抑鬼凿而神剜予将歴幽胜登■〈山赞〉岏泝溪流而舣舟望桃花而解鞍访主人之高躅殆管窥乎豹斑沾葛屦而松云漠漠滴野服而花露漙漙歌孙绰天台之赋高斟緑蚁吊承祯白云之号细煑龙团询陈襄之正教未逺悟顾欢之栖隠非艰??一笑而倾倒愧文章于二班嗟夫物以画传者有之矣物以文传者有之矣抑又知文有时而在画有时而残乎此东坡之记方日光而玉洁而寳绘之堂已毁瓦而画墁慨予年之衰暮觉才谫而文孱虽不能作金声于掷地亦将续昌黎桃源之圗赋筑野桃溪之圗而濡翰也
