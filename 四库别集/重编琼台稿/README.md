　　钦定四库全书　　　　集部六
　　重编琼台防藁　　　　　别集类五【明】
　　提要
　　【臣】等谨案重编琼台防藁二十四巻明邱濬撰濬有家礼仪节已着録其文集世不一本初其门人蒋冕等刻其诗曰吟稿续又裒其记序表奏曰类稿嘉靖中郑廷鹄合二稿所载益以所得写本厘为十二巻名曰防稿天启初其裔孙尔谷遴类稿十之二増防稿十之三并吟稿合刻曰重编防稿即此本也虽不及类稿防稿之完备而简汰颇严菁华具在足以括濬之著作矣濬相业无可称其立朝与叶盛不相能具载明史盛传中又与庄防相恶具载明史防传中其族御医刘文泰诬防王恕一事虽其妻亦知其非具载明史恕传中讲学家以其力崇朱子曲相囘防迄不能与公论争也其两广平贼之策言之凿凿然韩雍力驳其説竟奏荡平具载明史雍传中则其好论天下事亦不过恃其博辩非有实济然记诵淹洽冠絶一时故其文章尔雅终胜游谈无根者流在有明一代亦不得不置诸作者之列焉乾隆四十六年十月恭校上

　　钦定四库全书
　　重编琼台藁巻一
　　明　邱濬　撰
　　五言古诗
　　古意【三首】
　　千金买寳刀百金买角觹寳刀头有环角觹能解丝慇懃寄逺人用以慰所思所思不见察幽独甘自怡
　　其二
　　客从天上来赠我云锦章五色光夺目上有双鸾翔秘之不自用箧笥谨将藏幸然遇良工为我裁衣裳结以紫霞绶佩以鸣玉珰服之朝九重文采何赫煌盛服今所尚令德古所臧珍重鸊鹈诗三复不敢忘
　　其三
　　逝川无回流落日不再午人生百嵗间役役徒自苦朝为树上花暮作花下土去住无常势奄忽成今古反袂问苍天天乎奈何许
　　感兴【二首】
　　人生防有老老至不自知壮心恒未己外貌忽已移青青巾中髪俄然成素丝灼灼镜中顔忽觉如枯梨盛年不可恃行乐须及时有酒且痛饮不醉将奚为
　　其二
　　天际多刚风沧溟隔弱水相思不可极之子在万里万里各一天恍惚在我前区区倘见察敢不输中悁
　　过友人园居
　　爱此林泉胜载酒时来过芳树发春华方池激清波嘉鱼争出跃好鸟鸣相和天气既已暄景物亦已多主人雅好客相对酣且歌歌竟各叹息人生能几何
　　月下有懐
　　明月空中悬碧云天际合美人渺何许望望转萧索翩翩惊鹊定片片檐花落恻然对孤影下帷闭斋阁
　　夜坐和曲江感遇诗韵【四首】
　　空斋坐幽独夜气澹以清冥心古圣贤悠哉怡我情天机一何深神理亦已精云胡契其妙勉旃惟思诚
　　其二
　　凤凰翔千仞枳棘安足顾一朝览德辉栖止梧桐树飞鸣恒自出罗网岂能惧三灵为我俦百鸟莫予恶笑彼飞鸿犹为弋人慕
　　其三
　　南极有名相风度邈难得鸱枭羣刺天孤凤戢其翼韶石佳山水因之増秀色班班青史间流誉靡终极庄诵感遇时临风三叹息
　　其四
　　深源无浅流髙树无卑枝人生天地间奋发须有为不见东注波逝者恒如斯心中茍自尽意外非所知嗟尔亡羊者纷纷多路岐
　　拟古【四首】
　　江南秋风至草木变焜黄淅淅吹妾衣使妾増悲凉悲凉知为谁良人在沙场暮听胡马嘶朝防胡鴈翔饥餐风中糜渇饮雪下浆羊角冲地起沙砾争飞扬回首望故乡长天但茫茫岂无肥与甘亦有衣与裳妾心空恻恻路逺莫寄将北望长太息涕泪如雨霶几欲徃从之河广无舟梁仰天吁上帝矢心期不忘但愿南风竞吹君来妾旁
　　其二
　　客集髙堂上众味罗珍羞觥筹互交错乐矣忘其忧清醑饮桑落妙声发呉讴寳鳬喷清烟芬芳袭轻裘肥醲正厌饫文锦何温柔独念良人苦逺戍隂山头黄茅连白沙风雪寒飕飕冻雀飞不起依树鸣啁啾马毛缩如猬髀肉胝生周羔裘暖如烘湩酪清如油君身千万艰妾心千万愁梦寐或见之道路阻且脩愿言早成功诸将各封侯良人章章来红日照九州
　　其三
　　依依重依依不忍生别离别离已可悲况值秋风时桞衰不堪折情苦不堪説愿妾为小星君身化明月明月贴天飞小星恒相随月出星随出月归星亦归莫学秋胡妻相逢不相识生者固可惭死者亦何益
　　其四
　　白日日已晩行人日已逺秋风又重来行人犹未返飕飕朔风寒行人衣应单世无杞梁妻千载徒悲酸
　　秋兴【二首】
　　秋来百虫号各自悲其生物情尚有感况乃骨肉情幽幽堂背花嵗晩雪色零忘忧更不忘日夕忧思増嗟尔客游子胡为常逺行
　　其二
　　孟秋再初霁虚馆生晚凉悲风振木末激烈流清商寒蝉抱叶悲哀鸿呌云翔蚯蚓出其窍蛜蝛鸣在房螀蛩鸣我前蟋蟀鸣我旁物各感其时岂必皆不平而我于此际兀然坐欲僵凄然动我心冥思搜枯阳壮哉秋声赋三叹思欧阳兰台一何卑戚戚徒尔伤
　　悼亡【十首】
　　择配得孟光足慰平生心一见如夙昔友之如瑟琴意气两不疑苦口时相箴欣愿自此毕恩爱何其深
　　其二
　　恩爱虽可慕嘉防不可迟同牢未周星而我顾有违辛勤理行装送我登王畿握手一长叹泪下如雨滋收泪向我拜再三频致辞勉我立名节勿作无益悲
　　其三
　　我行逾四载之子苦幽閴登堂侍老母灯下事蚕绩知我为功名不复懐戚戚一见平安书喜气动顔色
　　其四
　　嗟我登文塲局蹐再不遂学陋惭为师恳辞得如志万里忽归来相对如梦寐移灯频近牀悲喜两交至亹亹用甘言慰我不得意
　　其五
　　乆别复良晤欣幸如再生有如练丝弦弦和多妙声自兹拟百年资以辅所行明年还随计挈之趋神京云何未及期一疾不复兴
　　其六
　　临终囓我指与作终天诀双泪注不流恋恋不忍别气促发言迟奄奄殆垂絶勉我赴功名努我立名节事我不尽年命薄将奚说死生皆其天无用过哀切
　　其七
　　朝日忽已沈晨花忽已零居然失良助我也何以生临终永不言此意良不轻矢心铭肺腑讵肯寒兹盟
　　其八
　　结髪六星霜欢防恰嵗半平生止一息而我不及见归来空闻名目中无其面天乎人何尤抚膺坐长叹
　　其九
　　嗟汝止一弟情义深以长汝殁仅阅月汝弟亦继亡一双恩爱刄并割汝父肠毎闻号哭声使我増悲伤平生心爱人相继俄分张如失左右手内外俱皇皇地下倘相逢应念予凄凉
　　其十
　　皇天亦何髙后土亦何深冥鸿失其偶飞飞吐哀音茫茫宇宙间辽邈畴能寻此生甘且休不尽古今心哀伤谅无益暂醉聊自吟
　　述懐【时初入翰林】
　　玉堂在天上瀛洲聚羣僊凡骨昔未蜕无路难夤縁我生幸逢辰滥等厠英贤出入清华地昕夕相周旋经史事幽讨兀兀穷嵗年誓言追往哲絶彼尘累牵立足千仭冈游心万古天长揖谢诸子龌龊谁与怜
　　小景
　　川光涵逺空山色淡平野危楼夕霭间髙树凉云下久客厌尘纷羡杀舟游者
　　母亲太孺人夀旦
　　春三月二日我母初度辰一官系词林称夀嗟无因缅懐劬瘁恩莫能报涓尘稽首望南拜泪落沾衣巾想当髙堂上罗拜方纷纷膝下跪举觞二子少一人慈顔于此时应念未归身母子天一方相思各怆神恨我无羽翼囘首空长嚬人生天地间咸欲显其亲显亲须宦游况我家海濵年年当此日驰心随白云自怜草木姿无以畣阳春仰天祝母夀尽忠希帝仁万一推恩典此志庶少伸
　　分韵得勿字送李应祯因和北征诗与之
　　予友李应祯筮仕未叶吉如金方在镕如女犹处室少小思用世顷刻惟惜日奋志向霄汉栖迹尚圭筚至寳恒自珍深蔵不轻出髙鶱终逺到暂蹶未为失着存心五思克念躬四勿矻矻良独苦搰搰亦何切潜心入混茫沈思超恍惚物理无穷尽志愿罔终毕九转紫府丹三叹清庙瑟妙削鼻污垩快刃缕濡血挥霍迅雷变倐忽浮沤灭葩华绚词林根柢蟠理窟疉出山峩峩一泻水潏潏冥摉鬼为愁悲壮石可裂坚壁仆旗鼓开门合车辙誓将古道敦不必时世悦药笼富芝苓家园乏芋栗空悬万轴籖宁有千树漆剈雕返淳朴敛华就真实禹鼎穷物怪商敦太古拙茫昧鸟迹余浩荡鸥波没片言无欠剩一览备始末游鹍未击水顾兔乆藏穴举世少具眼谁人识竒骨焚舟取胜师即是前日卒际遇防有时造化本无物无人折我脇亦不擢彼髪惟持方寸觹独解千古结纷纷儿女辈得丧辄呜咽慨慷发浩歌一笑为洗雪赠我锦绣叚输君龙缟袜应知同此心未肯屈兹膝从革金百链朝宗水万折岂念欵叚马不揣宽愽褐胸中浩然气需时一发泄见者使心降闻之须股栗朝阳孤凤鸣太虚众星列宛转似抽茧次第同比栉闳肆千万言勿效涂且抹贾生真俊才孟氏岂迂濶直道古所贵正言世方渴工巧由三折顿悟在一喝众心鸡犬放羣言蛙蝉聒就中一转语乃至无有説人才无古今儒道有始卒日月既精明乾坤亦轩豁沛如纵壑鱼奋若独击鹘坐令龌龊者缩首看驰突匪惟当世重要与古人匹功名殊孟浪归志甚勇决北风吹南船顺流去何疾悠悠聊自宽耿耿竟难夺和壁三献有楚城再举拔抚掌拍霜锷酒酣清兴发眼将空海宇气欲吞恒碣男儿负竒志与世相末杀何时乘长风登天问明月我生命在天吾道岂终絶从来君子心迥与世俗别四海皆兄弟奚用忧离折贵爵异赵孟美色匪褒妲勉旃思至理寤寐想前哲前哲虽已矣垂世有谟烈况乃灵台中源头水长活蹇予二三子追趋青琐闼结绶独少君忍送出城阙江头秋草哀天际寒蟾缺去去将奈何临岐愧先达
