　　提要

　　臣等谨案仰节堂集十四巻明曹于汴撰凡文十一巻诗三巻前有髙攀龙冯从吾序于汴尝从二人讲学故也攀龙序谓其文足以定羣嚣明学术诗足以畅天机流性藴从吾序谓其非沾沾以文章名家而撡觚自豪之士无不退避三舍于汴亦尝为从吾作理学文鹄序曰闗中少墟冯先生辑诸大家举子艺百数十首以式多士命曰理学文鹄不命以举业而命曰理学何也见理学举业之非二也云云故于汴之诗文亦在理学举业之间或似语録或似八比盖平生制行髙洁立朝风节凛然震耀一世逺者大者志固有在原不以笔札见长从吾序所谓非沾沾以文章名家者为得其实观是集者谓之文以人重可矣集初刻于首善书院甲申板毁于兵康熙癸卯其外孙景望蘧购得残本其门人吕崇烈鸠乡人醵金重刋即此本也

　　●仰节堂集原序

　　夫学性而已矣夫性善而已矣何以证性善也今人钦钦焉目明耳聪手恭足重心空空而无适于斯时也彻内外非天乎天非性乎性非善乎以其为人之本色无纎毫欠缺无纎毫污染而谓之善也循是而动不违其则之谓道故学莫难于见其本色见本色斯见性矣程子以学者须先识仁而谓不须防检不须穷索夫学岂可废防检穷索欲人识防检穷索之非本色辨其非本色者则知其本色知其本色则防检穷索皆本色也吾见曹真予先生于长安中终日钦钦目明耳聪手恭足重叩其中空空而无适也可以证性矣夫性空言之则无朕也实证则有象也先生非其象乎故先生居乡孚乡立朝孚朝告君者足以定羣嚣明国是告友者足以明学术阐道奥见于咏歌者足以畅天机流性藴所谓循是而动不违其则之道也此之谓性此之谓善知先生者知斯集知斯集者知先生根本枝叶无二物也时天启四年六月二十六日锡山友弟髙攀龙顿首拜撰
　　昔明道先生作字甚敬曰非是要字好即此是学余以为作文亦然非是要文好即此是学若作文甚敬行必顾言吾得之真予曹先生云先生全集梓成余读之喜甚邹鲁嫡传濂洛正脉其在斯乎言言有理言言不苟而又言言有作意它不具论即如题南皋先生教言数语杂之秦汉古文辞中亦不多得寸山起雾勺水兴波赏心哉观止矣蔑以加矣先生以千古絶学自任固非沾沾以文章家名者而作文又甚敬乃尔即世所称操觚自豪之士宁不避三舍退哉先生虽谆谆讲学而非其人不轻发一语即得其人亦不轻发一语易云拟之而后言议之而后动拟议以成其变化先生以之故其著作虽间有应酬而誉必有试独为余文似又轻誉余窃愧之而或谓善信如乐正子孟子进之以美大圣神夫美大圣神而可易言乎哉其期望不得不如此子惟勉之可耳焉用愧犹记前岁少宰缺庙堂误起余而借重先生陪先生特膺简在余方为铨衡得人喜而先生再三力辞竟不税冕行夫铨衡重任也少宰美秩也他人争之若骛而先生弃之若凂此其髙风峻节即古之人宁数数见哉先生之学以躬行二字为宗而辞少宰一节尤为躬行之大者读先生集当因言而求于言之外不然而徒艳羡其文辞浮慕其理致出口入耳忘厥躬行即先生所谓没齿务学终属半涂终日亹亹犹漫道者也岂惟负先生亦且自负或曰薛文清公与先生皆晋产也文清终身学问只是一敬字先生学问渊源葢有所自余曰然青出于蓝而青于蓝氷出于水而寒于水自古记之矣天启乙丑七月既望关中冯从吾撰

　　●重刻仰节堂文集序

　　余师真予曹先生仰节堂文集行世已久余曽得一部朝夕讽诵不能释手且当时家藏户珍者不鲜甲申之后十失八九己丑兵燹运城极厄无论片墨不存并其原板不可问矣奈何一日先生之外孙景君字望蘧者忽遇此集于安邑市上盖几与败絮残楮相为没灭耳景君急购之若惊若喜见余而道其繇来余乃仰天谢曰先生之灵不泯天地鬼神为之呵护也欤于是聚而谋诸弘运书院之友诸友曰硕果之存危哉相与鸠金而为重刻以图不朽刻成诸友属余序余曰比邻夏有司马温公集河津有薛文清公集合仰节堂集可谓三大文章矣自此汾洓重清条霍更翠其关风化人心讵浅微哉然而重刻之友亦与咸休太史公谓苍蝇附骥而行千里余于斯亦云门人吕崇烈顿首题于弘运书院中时康熈二年岁在癸卯六月吉

　　江西乡试录后序

　　臣不敏副江西典试之役既告竣当有言夫兹地习谭圣学而诸以比士来者亦大概以作圣之功三致意焉臣安能更端以告士士不圣人之学而谁学乎学圣人者心其心行其行言其言举子制艺亦言其言者也方其湛重渊之思抉九天之秘鲁邹觌面莘岐倾怀里巷屑越之语乌能有此不谓圣人口吻可乎哉第孔孟伊周启口容声罔非是物而诸士匪临场羔鴈鲜不为里巷屑越之语者要在充其类耳讵宁惟言士亦尝心圣人之心矣行圣人之行矣寻常食息大圣如斯虽在梏亡犹涵至性忽怆臆于颠连忽赧颜于尔汝忽庄容于宾祭忽灼知于着揜其于圣心圣行隙露全露乍合乍离者皆是方其离时一毛万山当其合时百世同室亦在充其类耳充之者言有常也行有常也心有常也夫一脔知五鼎跬步知千里业有其言何不常其言业有其行何不常其行业有其心何不常其心言之实者其言常行之实者其行常心之实者其心常其心实无有不实其心常无有不常无不实者至诚也无不常者不息也充类者致其曲而诚也夫沧浪之咏充之綦隆臻居室之能充之天地察孩提之爱充之仁义达岂以士也而荧荧随熄涓涓自封弗克充满优入圣域哉夫观天者以昭昭察地者以撮土学圣者以一隅故棘内可信棘外一绪可窥全体非过也士诚有此倪也然而由圣倪跻圣极则亦有辨矣臣北产也既壮而渡江北之所闻恪守实蹈尺步绳趋居多焉渡江所闻顿解根宗圆通开广居多焉或相背而各往或相疑而交辩顿解者曰何为琐琐滞滞拘挛格式眼瞳不瞭足趾竟隘恪守者曰何为艶悟忘修驾大弃细以名节不足检以踈纵为无碍臣窃谓二者之言可相济也俱不可废也夫恪守之士斤斤于门外者也顿解之士扬扬于门内者也门内之论终不能胜顾乌得决去藩篱便其情态躬为嚆矢遗人口实道之闇也殆与有责焉夫障缘愈添本真益昧尔先达之重戒者戒之良是贿赂干请任情执见尔先达之明纠者纠之亦良是尔多士两存之一切谨严一切脱化三百三千何思何虑九经九德无臭无声以此为心以此为行以此为言渊源孔孟证可伊周岂不伟与不然胶于师说岐于门户其为辞也诐诐辞不足辱兹录而况生于其心发于其行害且更烈何圣学之足云岂所望于多士哉

　　崇贤录序

　　盱山士绅以宋刘刚烈父子及李博士应列祀典状建议于邑侯许君得诺先是三公显灵于漕直指苏公树坛于直指董公兹议乃兴建祠以祀就祀为会用阐圣学感三公而思济美匪直报之俎豆已尔爰述其经制始卒之详成录问序于余夫宋距今歴年累百英爽犹存追崇不諠今人之性不隔于古人故不諠也古人之性不隔于天地故犹存也然则天人一而已矣东西南北之人古人今人一而已矣而人或自封一局者何欤不达鬼神之情状付之曰乌有亦自封也则何不可自封者门其门户其户名其名利其利歩武几许胡不藩之剖而家之大乎吁嗟此君子之所不为也刘氏父子身其身乎宋与我何有而自糜于锋刃李氏而身其身皇皇于圣学之翼也何为直指邑侯而身其身何有于亡人何有于盱而举之祠祀盱士绅而身其身翕然不忘于昔贤何为而又何会讲之殷殷也是皆根于性也不可得而隔也故不容已也彼自封一局者亦何尝隔而以为若有畛焉若有域焉是幻见也如梦中之画界倐然而醒画界安在诸君子仰止于斯知其无幻见矣幻见于讲互有辩幻见于行互有规幻见于幽隠之衷三公在上不笑且怒几希吁嗟谓三公不笑且怒是以为无神也奚以祀以为有神也尚念旃哉

　　冯少墟先生集序

　　夫道生人失其所以为道则失其所以为人矣谁甘于失其人而每失其道弗思耳道贯于血气之质弗相离也离道而抱空质焉与土梗何殊乎是道也其大无外或狭而小之其宻无间或辍而断之其粹无滓或点而蔑之是故学为急焉学也者恢廓而使之大绵聨而使之宻涤荡而使之粹也道不待学而有而非学无以保其有非学无以复其有非共学无以公其有故孔子蚤岁志学没齿不厌也然学亦难言矣性天之奥本中有本胡以彻之知见之纷岐中有岐胡以析之习情之锢忽醒忽迷胡以觉之是用连朋讲究互参证以求至当相夹持而防堕落故孔子以不讲为忧也夫道需学学需讲有不啻饥之食寒之衣者而讲学顾罹世訾非尽世之尤也不学之士患在不讲讲学之士患在不副或亦艶为美称儋簦聊聚朝朝问路岁岁不越阈譬露背而谈九容挥玉尘而称俭素于我乎何有故孔门之训无行不与夫惟相与以行则学为真讲而万世宗之无斁也少墟冯先生沈潜圣学践履笃至问业之士如云而先生惟有教以阐扬剀实卫道谨严盖亦以行为讲以行为学者也道不在兹哉昔有问楚侗先生以天命之性者先生方欲训解其人曰意公自言其性耳先生为之矍然慕冈先生会友于白下凝然相对或曰冯公何无讲座上曰此人浑身是讲其亦旨于论讲矣于汴不肖仰先生之行有年兹诵其讲道之集若而巻而窥君子之慥慥也敬缀数语志向往

　　理言什一序

　　夫人之所以为人理而已矣里巷庸众对谈之语未尝不谓理云理云然究其精微则峩冠而皓首或不识为何物亦可诧也古先圣贤以理为行亦以理为言言为后人设也而究心者寡安望有窥于理而能齐其行乎知一张先生有味于先正阐理之言摘而录之时时研讨问发以所见遂至成帙命之曰理言什一嗟乎风会之下也士之所趋者流俗污世竞利逐时之习而已即沈涵典籍之中穷年兀兀藉其可禆进取已耳又不然左史英华晋唐膏馥辉润笔毫留连光景已耳闻所谓理言者如奏古乐不终场思睡矣亦有口诵注脚漫谓了了实则锢蔽此又訑訑之见耳先生之志大矣哉其造理之邃制行之卓宜也得其所以为人者矣人之彻内彻外动容启喙孰能外此譬彼鱼然能为顺水能为逆水而不能外于水人亦何为不理之顺也乎虽然什一云者以言言之也理无所谓什者一而已矣亦无所谓一呜呼微哉微哉非究心于理何足以语此
