　　廷试制科 

　　皇帝制曰盖闻上古无为而治不赏而民劝不怒而民威于鈇钺何甚盛也而儒者之论治曰有功不赏有罪不罚虽唐虞不能化天下又谓夏后氏先赏而后罚殷人先罚而后赏周人修而兼用之则二帝三王所繇固与上古殊路欤何同归于治也又有言赏疑从予罚疑从去者有言宁僣无滥者有言仁可过义不可过者以为古昔帝王皆以君子长者之道待天下然则先罚后赏者非欤抑赏罚者帝王致治之具而非其所以治欤我圣祖继天立极垂宪万世恩威莫测其用赏罚务协于中其揭诸祖训首章及载诸圣政纪者同符古治可得而陈其槩欤朕以寡昧托于臣民之上十有四年矣夙夜兢兢惟古训是式成宪是遵不爱爵禄赐予以待功能之士而不法者以三尺重绳之明示好恶以与天下更始然徳泽壅而不究法令尼而不行任老成奬恬退以教让也而浮竞之风益甚革苞苴罪贪墨以训廉也而澄清之效罕闻习俗奢侈示之以俭而人心犹溺于纷华刑狱寃滥示之以寛而吏议多任务于锻炼蠲租赈穷诏尝数下矣胡闾阎之困未苏振旅诘戎令亦屡申矣胡户牖之防未密无乃劝惩之法阙而未备欤抑所谓修职任事者漏赏而欺谩避课者佚罚欤将朕之不敏不明所以风厉之者非其本而督率之者非其实也兹欲赏信罚必以绍明圣祖之法而追古帝王之治何修而可尔多士居则称先王谭当世之务其尚究析古今根极体要详着于篇勿隠朕将亲览焉 
　　臣对臣闻之宋臣苏轼曰天下无事则公卿之言轻于鸿毛天下有事则匹夫之言重于泰山非智有所不能而明有所不及缓急之势异也方其无事也虽齐桓之深信其臣管仲之深得其君以握手丁宁之间将死垂絶之言而不能去其区区之二竖至其有事且急也虽以唐代宗之昏庸程元振之用事栁伉之贱且疏而一言以入之不终朝而去其腹心之疾何则言之于无事之世者易以改为而常患于不及见信言之于有事之世者易以见信而常患于不及改为此忠臣志士之所以深悲天下之所以乱亡相寻而世主之所以不悟也臣诵其言未尝不反复叹息也恭惟皇帝陛下乗干继离居丰抚泰嗣服以来九徳三俊已尽登用奸邪谗佞欺负之徒已尽除去四海已尽敉宁九夷八蛮之在荒服之外者已尽宾贡天下最号无事臣等草莽匹夫岂复有所计议仰佐下风而圣主虚懐若渴采及葑菲进臣等于廷赐之策问不知陛下于臣之所言将重之如泰山乎抑轻之如鸿毛乎异日者陛下凡四策士于兹矣尝以典学立政策士徐而究之陛下之于学于政何如也尝以有为无为策士徐而究之陛下之于有于无何如也尝以建极用徳策士徐而究之陛下之于极于徳何如也尝以仁明武策士徐而究之陛下之于仁于明于武何如也由斯以谈则轻重之效可见于前事已抑臣有言而君不庸非臣之罪也君有求而臣不言实臣之罪也况臣感时发愤有慨于中乆矣今明问及之乃忍缄黙隠讳以欺陛下耶臣请先究陛下之说而后及臣之私惟陛下所轻重焉凡陛下之所以策臣者逖稽上古上嘉唐虞下乐三代祗遹皇祖旁采先儒之说俯咨当世之务无虑数十百言究其指归则赏罚二科而已夫赏者劝天下之法然有不倚于赏者所以劝天下之意也罚者惩天下之法然有不倚于罚者所以惩天下之意也法常有为意常无为有为者以运天下无为者以宰天下故最上修意上古之不赏而劝不怒而威是也其次修意而不废法唐虞之有功而赏有罪而罚是也其次修法而不妨意夏后之先赏殷人之先罚周人修而兼用之是也自意失而始专法法失而始有赏疑从予罚疑从去之论有宁僣无滥之论有仁可过义不可过之论无非恐以其有为者伤其无为者也总之三代而上法与意出于一故其治同三代而下法与意出于二故其治驳伏读圣制有曰赏罚者帝王致治之具而非其所以治至矣尽矣臣不能赞一辞矣洪惟太祖髙皇帝继天立极垂宪万世恩威莫测赏罚协中其祖训首章之所揭大都尚寛其圣政纪之所载大都尚严则法之着于有为者也其尚寛也非以狃世其尚严也非以残民则意之藴于无为者也猗与休哉宜其治与古帝王比隆也曷论晩近世哉肆我皇上绍休承美式古训遵成宪赏罚之道甚具而有法爵禄赐予不吝于功能所为示天下以好者彰彰如是三尺重绳不假于不法所为示天下以恶者彰彰如是然而徳泽不究法令不行此无异故则圣制固言之矣所以风厉之者非其本督率之者非其实也本也实也即臣愚之所谓意也且皇上方鋭然留意于赏罚劝惩而臣妄信其本实之不至非诪张其说以为诳也臣愚窃观当今之势而根极其体要所谓累皇上之意者大几有二皇上明以好示天下而此二者恒阴移其所好皇上明以恶示天下而此二者恒阴移其所恶二者何也曰内宠之将盛也曰羣小之将逞也夫人主席崇髙藉富有无一不足以厌其欲昏其志而惟色为甚色之中人也微而其溺人也最沈锢而不可解圣王之所亟远也昨者皇上以郑妃奉侍勤劳特册封为皇贵妃大小臣工不胜其私忧过计因而请册立皇太子因而请加封王恭妃皇上不温旨报罢则峻旨谴逐夫皇太子国之本也忠言嘉谟国之辅也两者天下之公也郑妃即奉侍勤劳以视天下犹为皇上一己之私也今也以私而掩公以一已而掩天下亦已偏矣偏则皇贵妃或得以爱憎弄威福于内其戚属或得以爱憎弄威福于外不独此耳阉人侍妾又将乗其偏也或得以爱憎弄威福于内外之间若然则赏罚云者将不为皇上之好恶用而为内宠之好恶用欲其信且必未可也夫人主之耳目惟一而天下之耳目人主者且万万虽甚神圣其聪明宜未足以徧也将必有所寄之寄之得其人则安不得其人则危非细故也迩年以来皇上明习政务听览若神盖辨及左髙察及渊鱼几于徧矣窃闻之道路往往二三羣小伺察而得之斯可谓寄得其人耶不得其人耶得不得之间皆不可知而皇上姑寄之其亦有不得已也盖曰向以天下事专付张居正而居正罔上行私一时公卿台省从风而靡外廷之不足信明甚故寄耳目于此軰示天下莫能欺也臣以为不然夫善为治者以全而收其偏不闻以偏而益其偏皇上惩居正之専散而公之于九卿可也若聚而寄之此辈则居正之専尚与皇上为二此辈之専且与皇上为一与皇上为二则救之也尚易与皇上为一则救之也倍难奈之何其弗思也且此辈之始用事也适皇上鋭精求治之初彼方见小信以自结其所称说指陈类多依于公义犹若未害久之则阳公而阴私也又乆之则纯出于私矣若然则赏罚云者将不为皇上之好恶用而为羣小之好恶用欲其信且必未可也徳泽之壅法令之尼有由也所愿皇上清心寡欲去谗远色俾此心炯然如镜之明凝然如衡之平其赏也与天下公其好其罚也与天下公其恶则徳泽之出自皇上者先自无所壅天下孰得而壅焉法令之出自皇上者先自无所尼天下孰得而尼焉由是教之以让而奔竞者息教之以廉而澄清者众习俗奢侈且归于俭刑狱寃滥且归于寛藏富于民不必下蠲租赈穷之诏而其困自苏守在四夷不必申振旅诘戎之令而其防自密将上古逊无为唐虞逊时雍风动禹逊文命汤逊允殖武逊永清而皇祖之大烈于今有光也
