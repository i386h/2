　　原传
　　陆游字务观越州山隂人年十二能诗文防补登仕郎锁防荐送第一秦桧孙埙适居其次桧怒至辠主司明年试礼部主事复置游前列桧显黜之由是为所嫉桧死始赴福州宁德簿以荐者除敕令所删定官时杨存中久掌禁旅游力陈非便上嘉其言遂罢存中中贵人有市北方珍玩以进者游奏陛下以损名斋自经籍翰墨外屏而不御小臣不体圣意辄私买珍玩亏损圣徳乞严行禁絶应诏言非宗室外家虽实有勋劳毋得辄加王爵顷者有以师傅而领殿前都指挥使复有以太尉而领阁门事渎乱名器乞加订正迁大理寺司直兼宗正簿孝宗即位迁枢密院编脩官兼编类圣政所检讨官史浩黄祖舜荐游善词章谙典故召见上曰游力学有闻言论剀切遂赐进士出身入对言陛下初即位乃信诏令以示人之时而官吏将帅一切翫习宜取其尤沮格者与众弃之和议将成游又以书白二府曰江左自吴以来未有舍建康他都者驻跸临安出于权宜形势不固馈饷不便海道逼近凛然意外之忧一和之后盟誓已立动有拘碍今当与之约建康临安皆系驻跸之地北使朝聘或就建康或就临安如此则我得以暇时建都立国彼不我疑时龙大渊曽觌用事游为枢臣张焘言觌大渊招权植党荧惑圣听公及今不言异日将不可去焘遽以闻上诘语所自来焘以游对上怒出通判建康府寻易隆兴府言者论游交结台谏鼓唱是非力说张浚用兵免归久之通判防州王炎宣抚川陜辟为干办公事游为炎陈取进之策以为经略中原必自长安始取长安必陇右始当积粟练兵有衅则攻无则守吴璘子挺代掌兵颇骄恣倾财结士屡以过误杀人炎莫谁何游请以玠子拱代挺炎曰拱怯而寡谋遇敌必败游曰使挺遇敌安保其不败就令有功愈不可驾驭及挺子曦僭叛游言始验范成大帅蜀游为参议官以文字交不拘礼法人讥其頺放因自号放翁后累迁江西常平提举江西水灾奏拨义仓赈济檄诸郡发粟以予民召还给事中赵汝愚驳之遂与祠起知严州过阙陛辞上谕曰严陵山水胜处职事之暇可以赋咏自适再召入见上曰卿笔力回斡是善非他人可及除军器少监绍兴元年迁礼部郎中兼实録院检讨官嘉泰二年以孝宗光宗两朝实録及三朝史未就诏游权同修国史实録院同修撰免奉朝请寻兼秘书监三年书成遂陞寳章阁待制致仕游才气超逸尤长于诗晚年再出为韩侂冑撰南园阅古泉记见讥清议朱熹尝言其能太髙迹太近恐为有力者所牵挽不得全其晚节盖有先见之明焉嘉定二年卒年八十五

　　天申节贺表
　　化国之日舒以长运啓千龄之盛天子有父尊之至心均万防之驩敢即昌期防申寿祝【中贺】恭惟太上皇帝陛下宅心清静受命溥将协气熏为太平华夷衔莫报之徳孙谋以燕翼子宗社侈无疆之休诞敷锡于下民丕灵承于上帝臣方驰使传阻缀朝班望睟表于云霄敢恨微踪之逺被颂声于金石尚希薄技之陈
　　防庆节贺表
　　有王者兴爰啓丕平之运使圣人寿敢忘胥戴之诚【中贺】恭惟皇帝陛下荡乎无名建其有极干戈载戢恩加辽碣之区囹圄一空治格成康之上敛时百福享国万年臣迹滞遐陬心驰魏阙纪虹渚电枢之庆莫厠诸儒演龙宫蕋笈之文徒修故事
　　又
　　帝生商而立子有开必先民戴舜以同心无逺弗届干端肇辟岳贡交修【中贺】臣早以湖海之微生亲见唐虞之盛典蓬转逾二十年之久毎注想于冕旒嵩呼上千万寿之时独阻陪于簪笏兹膺郡寄复在王畿目瞻佳气之郁葱耳聴驩声之洋溢永言疎贱已极光荣恭惟皇帝陛下焕乎其有尧文粲然而兴周道三朝图籍将还荣河温洛之都万里车书已轶碣石榆林之壤以仁政广华夷之徳泽以丰年奉郊庙之牲牷凡曰含齿戴发之俦均被沦肌浃髓之赐光御无疆之厯益培有永之年臣猥以分符莫遑造阙箫韶方奏徒倾就日之心歌颂可陈尚刻齐天之石
　　瑞庆节贺表
　　虹流电绕适当圣作之辰鼇抃嵩呼共效寿祺之祝敢倾丹悃仰叩睿聪【中贺】恭惟皇帝陛下徳当干符躬有圣瑞东渐西被伟声教之混同上际下蟠极仁恩之渗漉帝生商而立子民戴舜以同心歴考前闻孰逾盛际臣生逢千载仕歴四朝岳贡川珍猥预骏奔之末鸢飞鱼跃永依洪造之中
　　光宗册寳贺表
　　食簭从考庙方严于典册仗全乐备都人咸觌于礼容圣教茂昭驩声旁达【中贺】恭惟皇帝陛下道参穹壤徳肖祖宗稽周王小毖之求躬虞帝终身之慕父传归子有光盛举于两朝天定胜人果见太平于今日乃咨元老大臣之参订兼采议郎博士之讨论勒崇垂鸿极髙蟠厚臣在列覩龙飞之旦防书奏麟趾之篇际遇特殊等夷罕及既莫预曲台之议又阻从属车之尘徒有悃诚形于梦想
　　皇帝御正殿贺表
　　肃传清跸端御昕朝徳上际而下蟠泽东渐而西被【中贺】恭惟皇帝陛下体天广覆如日正中率礼无违永叹歳月迁流之速向明而治勉畣臣民爱戴之心诹太史涓吉之辰采奉常緜蕝之议对扬宗社之福爰举公卿之觞两曜清明四方抃舞臣农畴齿髦帝所梦遥华衮光临虽莫望火龙黼黻之盛黄麾备设尚想闻金石丝竹之音
　　皇太子受册贺表
　　明诏建储永为宗社之本正衙发策显畣神祗之心国势奠安舆情闿泽【中贺】恭惟皇帝陛下若稽古训骏惠先猷寿考亿年诞膺不蔽之福本支万世坐拥无疆之休存心养性以事天修身齐家而治国迨举有邦之庆益昭知子之明臣迹逺周行心驰魏阙命太史卜日之吉徒闻播告之传遣上公持节以行莫预观瞻之盛傥未辞于圣代尚自力于声诗
　　贺明堂表
　　农扈屡丰协气方流于緜宇合宫大享旷仪遂举于中天驿置星驰嵩呼雷动伏以圣神在御祈报间行肆严长至之祠一举上辛之典惟兹宗祀实在季秋既得万国之驩心宜罄一精之嘉荐【中贺】恭惟皇帝陛下范围元化斧藻太平采皇祐之旧章茂建中和之极稽绍兴之新制用适古今之宜上帝顾歆殊休丛委臣官縻遐徼心系明廷考汉家汶上之图嗟莫陪于润色继周颂我将之作尚自力于形容
　　谢明堂赦表
　　明堂总章举旷仪于路寝钩陈羽衞敷大号于端闱徳协穹祗春回海县【中谢】恭惟皇帝陛下道绳祖武政酌民言干文仰法于房心肇称钜典解泽黙符于雷雨一洗众愆统和天人空虚囹圄臣适乗使传逺在遐陬奉五百里之驿书徒深鼇抃上千万年之圣寿莫缀鳬趋
　　谢赦表
　　宣布仁恩发扬孝治观人心之鼓舞知天命之协从【中谢】恭惟皇帝陛下躬舜禹之资履曽闵之行损又损而至道老吾老以及人一日三朝虽极宁亲之大养四方万里尚忧庶狱之无辜内广惠心旁流霈泽雨露所被囹圄一空臣适以守藩恭闻孚号虽与民欣戴如瞻咫尺之天然受命祷祈实劳方寸之地
　　谢赐厯日表
　　春秋以王而次春丕显体元之妙闰月定时而成歳适当颁厯之辰治象一新欢声四溢【中谢】恭惟皇帝陛下道兼伦制化被堪舆念王业之艰难毎急农桑之务察天心之仁爱尤深水旱之忧诚意既孚嘉生并应呼嵩髙之万歳幸覩昌期陈泰阶之六符不胜大愿
　　又
　　诏班新厯虽举彝章地近清都独先下拜恩光旁烛小已知荣【中谢】臣闻尧授人时实前民用汉得天统克协帝心方当重熙累洽之盛时宜谨体元居正之大典恭惟皇帝陛下凝图丕赫受命溥将齐七政于璿玑昭示太平之象调四时之玉烛用待来歳之宜爰勑有司以幸天下臣偶叨牧养获与布宣职思其忧劝课誓殚于绵力年运而往功名更感于初心
　　福建到任谢表
　　咸造在廷甫遂朝宗之愿奉使有指遽叨临遣之荣大造难名余生曷报【中谢】伏念臣么然薄命起自穷阎偶以元祐之党家获与绍兴之朝士真人有作景运方开适当宁叹息人才之实难顾一时豪杰号召而未至首防引对面锡殊科遭逢稀濶之知耸动迩遐之聴岂期蹇薄旋困沈绵卒繄全度之恩俾获退藏之分侵寻半世转徙两川三为别乗之行再忝専城之寄五十之年已过非复壮心八千之路来归怳如昨梦敷陈浅拙应对参差惟谴黜之是宜岂超迁之敢望此盖伏遇皇帝陛下道兼伦制泽被堪舆念臣留落有年尚未除于狂态怜臣驰驱无地空窃抱于愚忠顾虽末路之孤踪犹玷外台之髙选臣谨当力思守道深戒瘝官礼乐逺有光华既大逾于素望靖共好是正直庶少畣于鸿私
