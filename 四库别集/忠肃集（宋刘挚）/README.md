　　提要
　　【臣】等谨案忠肃集二十巻宋刘挚撰挚字萃老东光人家于东平登嘉祐四年甲科神宗朝累迁礼部侍郎哲宗即位歴官门下侍郎尚书右仆射以观文殿学士罢知郓州绍圣初坐党籍累贬鼎州团练副使新州安置卒绍兴中追赠少师諡忠肃事迹具宋史本传其文集四十巻见于宋史艺文志久无传本今从永乐大典各韵中裒辑编缀以原书巻目相较尚可存十之六七除青词斋疏等文十七篇谨禀承
　　圣训删削不録各以类排纂厘为二十巻而仍以刘安世原序冠之于首摰忠亮骨鲠于邪正是非之介辨之甚严终以见愠羣小贬死荒裔其为御史时论率钱助役之害至王安石设难相诘而挚反覆条辨侃侃不挠今其疏并在集中他若劾蔡确章惇诸疏见于宋史者亦并存无阙其所谓修严宪法辨别淄渑者言论风采犹可想见固不独文词畅达能曲鬯情事已也至集中有讼韩琦定策功疏论王同老攘功冒赏之罪而道山清话遂谓文彦博再相挚于帘前言王同老劄子皆彦博教之乞下史官改正宣仁不从彦博因力求退今考此事史所不载而集中有请彦博平章重事疏其推重之者甚至尤足以证小说之诬葢当时党论交讧好恶是非率难凭据幸遗集具在得以订正其是非于论世知人之学亦不为无补矣乾隆四十九年十一月恭校上

　　原序
　　宣和六年七月六日宣教郎知开封府临河县丞刘跂寓书于元城刘安世曰先人平生为文方弃诸孤仅存一箧类次之已成编集念当有序引以信于后晩年迁谪事同诸公身后怨家诬谤又防朝廷核实已赐昭雪然而元祐大臣不幸亡殁者类皆不敢纳铭于圹植碑于隧始终大节不应无闻于后也愿因集序并载一二使他日有考焉顾惟衰拙自少受知于先丞相素叨国士之遇中荷荐引寖阶禁从晩嵗迁谪复同忧患而又被谴以来行三十年固穷守道俯仰无愧似不为知己之辱虽懐自顾不足之羞而莫敢辞者葢义之所在不可得而避也公讳摰字莘老永静军东光人也防而敏悟有成人风年未弱冠被荐于渤海声誉籍甚凡四预乡贡礼部奏名为第一复中甲科初始南宫已着风绩英宗诏二府各荐士充阁馆忠献韩公琦以应诏补公馆阁校勘王文公安石初秉政搜择人材擢为中书校正居月余议论多不合防除监察御史欣然就职语家人曰趣装无为安居计即上疏论亳州狱起不止小臣意在倾故相富弼以市进今弼已责愿寛州县之罪神宗皇帝励精求治奬进臣下公既对面赐褒谕且问从学士安石耶安石称卿器识公对曰臣东北人少孤独不识安石也因论人物邪正奏对移时上意向纳公退益感遇思所以称因上疏论率钱助役官自顾人其事有不可胜言者畧陈十害切中时病防御史中丞杨公绘亦论新政并公章下司农司农条件疏校遂劾绘与公险诐中懐向背有防分析公奏曰臣有言责采士民之説敷告于陛下是臣之职也今有司駮奏遂令分析交口相直无乃辱陛下耳目之任哉所谓向背则臣所向者义所背者利所向者君父所背者权臣愿以臣章及司农所奏宣示百官者定当否奏入不报明日复上疏极论时政遂罢御史落馆职拟窜岭外上不暇乃贬衡州哲宗皇帝嗣位宣仁圣烈太皇太后以祖母共政见连年水旱西鄙未宁百姓劳敝而国有大故当务休息遂散遣京城役夫减皇城司觇者废物货场罢戸马等事皆从中出又戒敕内外无敢苛刻扰民已而进退大臣选用台谏擢公为御史公受命之始即具以熙宁告神考之语复陈于哲庙之前两宫聴纳尽行其言曽未期月人情于变使天下有防山之安而无一朝之患者公之力也公既被遇知无不言姧佞刻薄之吏事状显著者公皆正色弹劾多所贬出中外肃然时人以此包希仁吕献可上察其忠义诚信可属事任未几遂大用焉公在中书一日内降画可二状其一裁节宗室宂费其一减定六曹吏额房吏请封送尚书省公曰常时文书録黄过门今封送何也对曰尚书省以吏额事每奏入必径下本名已乆今误至此公曰中书不知其他当如法令遂作録黄初尚书令史任永寿精悍而猾与三省吏不能相数以姧告诸宰执吕丞相大防信任之时户部裁节浮费后省裁定吏额皆论年未就吕丞相专权狠愎尽取其事置吏额房于都省以司空府为局召永寿辈领之未尝谋及同列也永寿见录黄愕曰两省初不与今乃有此即禀丞相命两省各选吏赴局同领其事以是白公公曰中书行録黄法也岂有意与吏为道地今乃使就都省分功何耶他日又持奏稾以丞相防禀公曰吏额事今欲慎宻而速故请径下然未经立法欲三省同奏依致仕官文书法公曰他非其类也更当聚议明日吕相又袖槀厉色示公曰势不可不尔公不欲立异勉应曰诺其后事毕永寿以劳进官时忱苏安静时恽皆迁秩有差于是外议喧然不平台谏交章论列以为事在后者成就已十八九永寿等攘去才两月而都司不用司勲格擅拟优例冒赏徇私章数十上时公已迁门下每于上前开陈吏额本末此皆被省者鼓怨言章风闻适实不足深言吕丞相亦以语客曰使上意晓然者刘门下力也然自此忌公益甚隂谋去之遂引杨畏在言路谏官疏其姧邪反覆章十余上竟不能囬士大夫趋利者汹汹交讧其事于是朋党之论起矣公语丞相曰吾曹心知无他然外议如此非朝廷所宜有愿少引避丞相曰行亦有请是嵗八月一日奏事毕少留奏曰臣入处近列器满必覆愿赐骸骨避贤者路上遣中使召公入对太皇太后谕曰侍郎未可去湏官家亲政然后可去使者数辈趋入视事公不得已受命顷之吕丞相亦求退不许明年公继为丞相不满嵗前日汹汹者在言路诋公竟去位朋党之论遂不可破其本末如此公辅政累年刚明重厚逹于治道朝廷赖之及为相益总大体务守法度辅佐人主于无过之地其于用人先器识后才艺进拟之际必察其人性行厚薄终不轻授以职任故才名之士或多怨公公知之不恤也取人不问识与不识或多南士有以萧望之郑明事諌公曰楚士奈何笑而不答论者谓元祐相以来能以人物为意知所先后而无适莫者公为之首奏事上前言直事核不为縁饰多见聴用与同列语公平不欺未尝以私属人人有所欲多惮公闻之公闻之亦为尽力然终不以语也精力絶人逺甚一见宾客及闻其语终身不防事无剧易临之晓然省吏每以事试公不以乆近区处如一言皆可复故三省事经公所裁定者后皆遵用莫能改云赵彦若子败官下狱彦若奏与监司有嫌乞移狱他路言者论彦若防上不实王巩除知宿州言者论巩前在扬州不法彦若巩皆公姻家也语稍及公公请辞位章七八上遂迁就外舍诏遣近侍宣召公入既对谕曰彦若巩辈事何预也言者皆谓交通邢恕及章惇之子牢笼小人为异日计公心知为言者所中不复自明谢曰臣愚闇招致人言愿就贬责既退固请益坚乃罢相以殿学士守郓给事中駮奏谓刘某忠义自奋力辨邪正有功朝廷擢之大位一旦以疑而罢天下不见其过并给事中罢之于是中外疑骇莫所知谓乆之乃知言者虽多专以章邢事为媒孽初公家子弟与章惇之子相识因入都应举而公家子弟亦游科场尝至府第而言者指为交通之迹邢恕谪官至京师以书抵公公答以手简其末云为国自爱以俟休复茹东济为东排岸官数有请求而公不之许蓄怨甚乆适见公简隂録其语以示言者言者缴上之且解释云休复语出周易以俟休复者俟他日太皇太后复辟也刘某所懐如此葢媒孽之语不一而此最为甚朝野愤之其后杨畏遂升从官搢绅共恶甚于虺蝪东济亦除提举常平谏官复论其素行士论不齿不宜任以监司除命遂寝人情更以为快公天性高明不以己长格物既贵恭俭好礼不改平素淳静嗜书自防至老未尝释巻家藏书多皆自雠校得善本或手抄録孜孜无倦平居不亲妾媵家事有无不以经意虽在相府萧然一室其后南迁不知者谓公不堪其忧亲族门人乃知公谪居自奉简约与在相府无以异也凡有议论惟尚中道不习异説不责苟难务在谨名教而已少好礼学讲防三礼视诸经尤粹晩好春秋考诸儒异同辨其得失通圣人经意为多公文章雅徤清劲如其为人辞达而止不为长语表章书疏未尝假手凡奏议论説记序铭志诗赋诸文章千余篇次第着集为四十巻藏于家公自青社罢职和黄州又分司徒蕲州语诸子曰上用章丞相吾势当得罪若章君顾国事不迁怒百姓但责吾曹死无所恨第恐意在报复法令益峻奈天下何忧形于色初无一言及迁谪也岭表之谪公拜命即日就道惟从一子家人涕泣愿侍皆不聴水陆犇驰见星乃止至贬所屏迹不交人事亦无书自随宴坐静黙家人具馔告之食则食喜怒不形意澹如也公年未五十即屏嗜欲晩嵗南迁气貌安强无衰瘁之色居数月得防疾公自谓将终戒饬后事精神不乱安卧而薨公既殁于岭外所属为公请归塟于朝不许己而诸子坐废家属徙他郡而不着罪于状又无知其故者虽公亦不知也今上登极大赦天下公既归而文及甫蔡渭皆贬湖州然后人稍知其事起于此初及甫持丧在洛阳邢恕谪永州未赴亦以畏在懐州数通书有怨望语及甫又以公在中司尝弹罢其左司郎官怨公尤深以书抵恕其畧曰改月遂除入朝之计未可必当涂猜怨于鹰扬者益深其徒实司马昭之心路人所知也大意谓服除必不得京师官当求外补故深诋当路者绍圣初恕以示蔡硕蔡渭渭数上书讼吕丞相及公而下十余人防害其父确及谋危宗社引见甫书为验朝廷骇之委翰林学士蔡京御史中丞安惇防治焉遂逮及甫就吏而所通初无事证但托以亡父曽説之防治所问司马昭谓谁及甫对意谓公也问其证据事状则曰无有但疑其事势心意如此朝廷照知其妄狱事遂缓防公薨闻犹用蔡京奏以不及考验为辞但坐诸子而已时绍圣五年五月四日也其为诸子叔复防丧还乡里公嗣子跂径伏阙下上疏诉其事又持副封诣都堂叩宰相韩忠彦曽布等皆取实封案牍开视知其谬妄明白其以语跂至建中靖国元年二月二十五日有防文及甫蔡渭所陈显无实状已行贬责绍圣五年五月四日指挥更不施行然复公殁后谗谤所坐皆得解释渭今改名懋云崇宁元年正月癸酉塟公于蕲州湏城县太谷山之原先茔之东曹国夫人任氏袝焉有诏特依前宰臣例又除公壻通直郎蔡蕃知阳谷县应副事送终之礼极其褒荣四方观者莫不叹息呜呼公之厚德髙行追配古人嘉谟伟绩播于天下士民者固不可胜数今特取大节载之其余非公出处所系皆畧而不书八月一日承议郎提举南京鸿庆宫赐紫金鱼袋刘安世序
