　　提要
　　忠惠集十巻附录一巻　别集类三【宋】
　　【臣】等谨案忠惠集十巻附录一巻宋翟汝文撰汝文字公巽润州丹阳人登进士第事徽钦两朝至显谟阁学士出知越州髙宗时厯官参知政事以伉直忤秦桧罢归事迹具宋史本传及孙繁所作志铭中忠惠者其殁后门人所私諡也汝文好古淹博深通篆籀尝从苏轼黄庭坚曾巩防故所为文章尚有熙宁元祐遗风史称其为中书舎人时外制典雅一时称之葢当北宋之季如汪藻孙觌皆以四六著名惟汝文能与之颉颃周必大序觌鸿庆集称中多误收汝文所作亦足见其体格之相近矣杨万里诚斋诗话引汝文左仆射制中古我先王惟图任旧人共政咸有一德克左右厥辟宅师二句以为用成语雅驯妥贴之式又引贺蔡攸除少师启中朝廷无出其右父子同升诸公二句以为截断古语补以一字而读者不觉为巧之至今观其文大都根柢深重措词雄健所谓无一字无来处者庻防足以当之非南宋表启涂餙剽掇之比其为作者所推非徒然也集乃其子耆年所编见于宋史艺文志者三十巻明以来久不复传今从永乐大典各韵中掇拾排比编为十巻以存其梗槩又汝文罢执政后尝提举洞霄宫宋史失载其事近朱彞尊撰洞霄题名记仅以李弥逊筠溪集所作制词为据系之绍兴二年且取靖康原职题曰显谟阁学士今考孙繁志铭则汝文实在绍兴七年冬以郊恩除资政殿学士提举洞霄宫彞尊未见此文故所记俱误今并以繁文附录于末亦足为考证之一助焉乾隆四十六年九月恭校上

　　钦定四库全书
　　忠恵集巻一　　　　　　宋　翟汝文　撰内制
　　戒士大夫不为朋党诏
　　朕惟先王之时在位者有羔羊素丝之节故其诗曰退食自公委蛇委蛇某谓曲折而行可踪迹也其谓出公门入私门而无有外交也盖知诚身以事君为忠而岂以邀夫爵禄呜呼其何修而可以至此乎朕夕寐晨兴以稽若先王之宪以训廸在位罔以侧言乱度罔以逸欲从已躬率是道以先士大夫庶几士大夫咸以名节自励形于四方民有所视效而还观之乆未始云获也朕愍然念之岂朕躬教之辟董正于尔在官者未信欤将风俗未殄而难可遽革欤凡所为爵禄劝勉者不足以警动人欤盖先王所以励世不独以庆赏刑威为国使人于庆赏刑威之外有所愧耻而不为夫秉忠蹈义之士朕之所嘉也比周朋党朕所弗臧也顾所为拂朕之志蹈于耻辱则子大夫其无乃自弃不自好也欤书曰以公灭私民其允懐苟尽公也民懐之矣况朕于尔所欲顾有不察乎继自今其祗若朕训以既乃心尔或不从朕言其于尔躬有罚布告在位明体朕懐
　　赐张孝纯诏
　　勅孝纯卿力保孤城外抗强敌率厉将士乆已逾时朕北顾轸懐无忘终食已遣李纲为宣抚使刘韐为宣抚副使又分命解潜等为制置司同济援师鼓行协力卿宜声势相倚内外合攻乗兹锐兵往御外侮明谕将士各奋忠诚爵赏之行朕无所爱
　　抚问河南制置使解潜诏
　　有勅金人俶扰汾晋宿师命卿抚临式遏乱略徃殄强敌底宁朔方
　　抚问王并一行将佐军兵等诏
　　有勅卿汝等暴露冠盖跋履山川躬提偏师徃定乱略特加抚问以劳旋归
　　赐李纲玉束带战袍金带马甲刀剑细物银绢茶烛等诏
　　有勅卿奋扬天威躬秉武节徃殄敌众抚宁兵师宜有宠颁以光行役
　　赐新除资政殿大学士中山西路安抚使詹度辞免恩命不允诏
　　勅詹度省所再上奏辞免资政殿大学士恩命事具悉卿以纵敌越境缓其归师引愆辞荣可谓知耻方今边陲不靖虔刘朔方命卿总戎于艰难非但洁巳而防谨徃服军事肃将天威戡彊敌之分虞摅中朝之夙愤庶几孟明报殽之役追继曹沫复柯之盟苟能慷慨以立功贤于逡廵而避宠所请宜不允故兹诏示想宜知悉秋热卿比平安好遣书指不多及
　　赐给事中詹乂乞淮浙一小郡或在外宫观不允诏
　　勅詹乂省劄子奏乞除淮浙一小郡或在外宫观差遣一次事具悉贤者在位赖其仪刑嵗计有余岂待勲阀而卿自讼无补志欲懐归与其辞禄而畏讥曷若受宠而思称为朕谋国于尔有辞所请宜不允故兹诏示想宜知悉
　　赐吏部尚书莫俦辞免兼侍讲恩命不允诏
　　勅莫俦省所奏辞免侍讲恩命事具悉朕询求儒学入侍燕间讲明六艺之归允赖多闻之益尔进读方册援证羣书雍容可观诵说有法朕方向意经术追迹古文庶几防言命使折衷尔其信以传信慎所阙疑令朕闻所未闻施于有政所请宜不允故兹诏示想宜知悉赐刑部尚书李弥大乞除便乡小郡或外任宫观不允诏
　　勅弥大省所奏欲望除便乡一小郡或外任宫观任便居住事具悉顷者彊敌内侵嘉尔献防朕惟朔方重寄命尔抚师兵戎乂安疆绥靖事既底定今其遄归顾非宠迁何以逊避所请宜不允疾速朝见供职故兹诏示想宜知悉夏热卿比平安好遣书指不多及
　　赐御史中丞陈过庭乞宫观不允诏
　　勅过庭省所上劄子奏乞除在外宫观差遣任便居住事具悉卿顷任风宪屡贡谠辞肆朕纉承复还旧物督奸为国昔闻臧孙之立言端委在朝今见叔向之遗直而幡然求去殊异所期方今议论毕陈纪纲略定思得卿重徃修厥官所请宜不允仍止来请故兹诏示想宜知悉
　　赐知雄州郭仲荀辞免遂安军承宣使仍更转一官回授白身恩命不允诏
　　勅仲荀省所奏辞免遂安军承宣使仍更转一官回授白身恩命事具悉朕嘉尔扞城宠以使节増崇褒赏之典旌劝忠义之臣辞之非艰思所以称所请宜不允故兹诏示想宜知悉秋热卿比平安好遣书指不多及赐新除给事中杨时辞免恩命乞致仕不允诏
　　勅杨时省所奏伏望追还成命许令致仕事具悉朕起卿燕闲任以言责卿得时遇合欲践所闻擢从谏垣徃莅闼庶几畴昔抗章伏阁之志施于今者论事批勅之间而以疾引年恳辞谢事朕意未可卿其少安所请宜不允故兹诏示想宜知悉
　　赐新除徽猷阁直学士杨时辞免恩命不允诏
　　勅杨时省所奏辞免徽猷阁直学士恩命事具悉朕闻古者崇德先耉老成人之不遗士虽引年皆耄期称道而不乱朕从尔谢事之请嘉尔为臣之终虽皤然旅力之既愆而绰然进退之余裕加命以职宠荣其归使防耻复兴名节滋重士有矜式不亦可乎所请宜不允故兹诏示想宜知悉
　　赐权管干殿前司公事王宗濋辞免殿前都虞候恩命不允诏
　　勅宗濋省所奏辞免殿前都虞候恩命事具悉卿総齐禁旅祗防殿岩守卫京师作固王室命兹环尹以奬尔庸胡为恳辞不承朕意虽褒申伯之元舅欲同懿亲而戒祈父之不聪敢云虚受勉兹职事期于尔成所请宜不允故兹诏示想宜知悉
　　赐武昌军承宣使权管干侍卫歩军司公事闾勍乞宫观不允诏
　　勅闾勍省劄子奏乞一宫观差遣事具悉师律以顺成而为臧军事以无犯而为正朕方将帅乆任欲以纪律而服戎尔其威令必行使知少长之有序惟服朕训无弃尔成所请宜不允故兹诏示想宜知悉
　　赐新除汝州观察使曹曚辞免恩命不允诏
　　勅曹曚省所奏辞免汝州观察使恩命事具悉廼者加命使节属当戎行尔以世禄引嫌抗言避宠朕惟光献临朝之泽烈武佐命之勲及其子孙褒尔阀阅徃服新命无为固辞所请宜不允故兹诏示想宜知悉
　　赐门下侍郎耿南仲辞免男延禧除太常少卿恩命不允诏
　　勅南仲省劄子奏辞免男延禧除太常少卿恩命事具悉朕搜揽贤隽程其器能岂以天官而为私宠奉常之擢谁曰不然而卿以嫌力辞初无祁奚内举之意朕慎用德选宁有尹氏世卿之讥所请宜不允故兹诏示想宜知悉
　　赐朝奉郎试兵部尚书孙傅辞免兼侍读恩命不允诏
　　省所奏辞免恩命事具悉朕复尔旧物思闻嘉言虽殚储宫调防之勤而失露门进读之益故使卒业便于询谋方求胥臣之多闻允赖寛中之笃学其悉尔蕴以廸朕聪所请宜不允故兹诏示想宜知悉
　　赐参政福建宣抚使孟庾诏
　　卿辍从廊庙徃摠戎兵先声所临羣丑自殪虽闽粤巳定方奏凯旋而湖湘未宁犹烦夙驾勉事道路务专精神思见仪刑念山川之悠逺倚成筹防赖樽爼之折冲特谕朕懐益严师律
　　赐福建宣抚副使韩世忠诏
　　卿率先戎行徃遏乱略慨然忠义之奋殄此狂狡之余究观用心造次许国见危授命不以事至而辞难好功与名岂待爵赏而加劝方今底平闽越士巳防勲经略湖南兵未振旅俟卿凯旋献防荡定有期论次前后之庸并加宠数之渥益坚用命其体至懐
　　赐新除端明殿学士江东路安抚大使李光再辞免恩命不允诏
　　勅李光省所奏再辞免恩命事具悉朕观秣陵根本之地有表里河山之固非我文武兼资之臣孰当公侯干城之任乃者辍卿帷幄为国藩翰庶几京师之蒙福将使自叶而流根而卿乃以人言过自为计朕惟谗说殄行能震朕师搜慝服谗本诬盛德岂容小丑歴诋在廷以卿自信不疑之心副朕委任责成之意益务体国无复有辞所请宜不允仍疾速起发兼程前去之任故兹诏示想宜知悉夏热卿比平安好遣书指不多及左中大夫同知枢宻院事沈与求除知枢宻院事制
　　履中微之运日有氷渊之虞念大业之难凛若芒刺之负时方右武事莫急兵乃眷枢筦之臣备宣忠力其正机庭之位増重本朝具官某学洞古今才优经济周旋万事有蓍先见之明夷险一心挺风雨不渝之操任国柱石作朕股肱比从真馆之游起参右府之寄宥夙夜之宻命翼翼秉心賛帷幄之深谋孜孜尽节方重安危之倚式昭翊亮之忠匪进陟于鸿枢其曷副于公论朕将祗承天命以戡定祸乱所赖制胜樽爼以辉耀威灵益修我戎企是机防傥三军五兵之用可以懐徕彼九州四海之人何忧弗格副予至意厉尔壮猷
　　参知政事刘大中除资政殿学士知处州制
　　选用贤能建置辅弼欲以肃清内外圗济艰危然以采之不广则难以得人任之不乆则难以责効至于畏逺权柄以疾为辞恳避宠名以事而解亦不得固留也具官某以劲挺之资守中庸之学沉几敏识足以有为于时同寅协恭足以不愧于昔自擢参于大政实有副于具瞻而称疾病者至于再三上印绶者至于数四朕惟眷倚之重中外则均进退之宜始终以礼是以畀括苍之旧镇加秘殿之隆名以便尔私以从尔请夫官盛任使固古人所以劝大臣之道而易禄难畜亦儒者所以为自处之常惟益究于乃心则有光于朕命
