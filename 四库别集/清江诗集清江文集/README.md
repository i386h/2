　　钦定四库全书　　　　集部六
　　清江诗集　　　　　　别集类五【明】
　　提要
　　臣等谨案清江诗集十巻文集三十一巻明贝琼撰琼字廷居一名阙字廷臣崇徳人元末领乡荐遭乱退居殳山明初徴修元史除国子监助教事迹附见明史宋讷传考程庆珫声文防选以贝阙贝琼为二人然陶宗仪辍耕录载妓女真真事云嘉兴贝阙有诗今真真曲载此集中则琼一名阙审矣两浙名贤录载琼集二十巻明万厯中所刻乃止三巻此本凡诗集十巻文集分海昌集一巻云间集七巻两峯集三巻金陵集十巻中都稿九巻归田稿一巻仅有钞本流传康熙丁亥桐乡金坛购得之始为刋板琼学诗于杨维桢然其论文称立言不在崭絶刻峭而平衍为可观不在荒唐险怪而丰腴为可乐葢虽出于维桢之门而学其所长不学其所短宗防颇不相袭朱彛尊静志居诗话谓其诗爽豁类汪广洋整丽如刘基圆秀胜林鸿清空近袁凯风华亚髙启朗净过张羽繁缛愈孙蕡足以领袖一时乡曲之言未免过实然其诗温厚之中自然髙秀虽不能兼有诸人之胜而驰骤于诸人之间实固无所多让其文亦冲融和雅有一唱三叹之音史称宋濓为司业建议立四学并祀舜禹汤文为先圣琼作释奠觧驳之识者多是琼议则其考证古礼尤有依据不但词采之工矣乾隆四十五年六月恭校上

　　清江诗集巻一　　　　　　明　贝琼　撰赋
　　鹤赋
　　余窃悲此昆阆之灵禽兮胡爲独处而无邻诚仙人之骐骥兮阅寒暑兮三千饥不磔乎腐防兮渴不饮乎盗泉生旣囚而憔悴兮夜长鸣而彻天月朦胧而将曙兮风淅沥而戒寒尚蹁跹而欲舞兮终磊落其如人恋青田之故林兮盼华亭之大泽路中絶而忘归兮媒甚劳而寡匹顶的的以补丹兮毛毵毵而俪雪冀僃君之晓驾兮常结心于紫极欲抗音而复噤兮时顾影而自惜淘河贪而无厌兮鹯又悍而不德彼优游以饱鲜兮亦绚练而矫翼内深省而益悼兮岂以才而见贼希有氏之西来兮忽余眷而踟蹰肉不膏于九鼎兮幸托君之阶除谅一息而万里兮奚孺子之能呼一引歩而三却兮思拔趾于泥涂茍怀恩而报施兮亦岂惮夫陨躯恶其玩而无礼兮义独洁而不汚目四顾而冲冲兮白日奄其西徂访夫以决之兮盍舍旧而改图览中区犹一盎兮曽何足以度世坌千起而万灭兮陋纷纷之蝇蚋寓寥廓之无极兮跨六合而髙厉嗟网罗之虽设兮已超然而独肆将西歴于青城兮何太白之峥嵘杜宇号而血吻兮天懵懵其无情狐九尾而猖狂兮防一足而伶俜望沧海之无垠兮防百川以东入悲精衞之含愤兮力旣微而何及山孰别其有三兮洲孰穷其有十扶桑不可以托巢兮蟠桃不可以爲粒日昼晦而沦精兮雨萧萧而翅湿指神尧之所都兮九门郁其嵯峩载吹笙之王子兮招鼓瑟之女娲栖建章之髙栋兮浴太液之清波夹玉辂以翺翔兮友丹鳯而婆娑尚骋力于广逵兮矢厥心而靡他
　　怀旧赋【并序】
　　余春秋二十有四馆于云间夏景渊氏凡十阅霜暑尤隆主宾之礼后去而觅举武林遂畱西湖中罹兵变东西奔窜邈不相知今年春因访焉则已物故而其子士文克承堂构余私喜其余庆未艾也复觞于漪澜堂者累日髙谈大噱夜继以烛退追旧好不能不爲之慨然乃操翰爲之赋曰
　　览云间之雄秀拾海月于珠宫访野王之荒台俯鼇背之方蓬命羽人以前导召宓妃而后从翳晻晻之宿霭激泠泠之回风悼处士之永逝闯华榱而兴叹或纵饮而宵分或谈而景宴搴渚秀之碧滋擢园英之朱粲歴十载而一来感凋落之过半登髙垅而长思夜漫漫其勿旦乐旣徃而哀袭何遭时之多难已焉哉物忌盈而必隳运实关于否泰豺虎横而麟毙鸱鸮鸣而鸾铩怼霞气之徒餐奚神仙之可企闵余行之未已絶洪流而东迈忍闻笛于山阳泪潸然而独洒呜呼千龄一息兮放旷自娱伊人不可作兮死生异涂聊寄心于翰墨怅旁眺而踟躇
　　大韶赋【并序】
　　传曰尧作大章舜作大韶韶继也言舜能继绍尧之德也周礼曰大韶禹乐名也言禹能大尧舜之德也前乎舜者尧非不继喾也后乎尧者禹非不继舜也特于舜言继者法成乎尧也在喾之时法犹未成尧虽继之而可继之事未备舜协于帝在舜之时功爲已协禹虽继之不足爲难矣则可继之善善继之功惟舜独也是韶爲舜之乐无疑季札观乐见舞韶箾者曰德至矣盛矣如天之无不覆帱如地之无不持载虽甚盛德蔑以加矣后孔子学之于齐三月不知肉味曰不图爲乐之至于斯也葢知乐之尽善尽美莫过于韶宜当时之感召丹朱在位羣后德让祖考来格鸟兽跄跄鳯皇来仪也然非舜之德致和于上防之乐召和于下何以臻此哉故述而爲赋不徒极其声音之美而且本之舜之德云赋曰
　　有东吴公子北走齐鲁之疆观于峄山而见孔林之主人焉主人曰公子之游也亦将有所覩乎曰无也生于震泽三江之上僻陋寡闻窃慕礼乐之事而六律七均之制尝究心久矣愿有请于大人先生焉主人曰嘻吴会东南之天府而天下之善音萃焉然公子犹有所未足者岂将厌澹泊而说铿锵乎必将挟陈娥携赵女擪鹍考鼍鼓若是以爲乐乎公子懑然不恱曰霓裳之曲唐之所以播越也广陵之散晋之所以分裂也固不足言矣若此者又狄之乐无异纣之靡靡以亡其国者岂君子之所乐乎愿闻其他主人曰七德之歌七德之舞太宗之肇王业也亦尝闻之乎曰伯者之事子孙无所法焉曰汉祖蹴嬴龙飞沛中置酒层台悲歌大风造基四百光啓西东若是何如公子曰犹有伯心之存也驳而未纯然亦一世之雄乎请言其上主人曰抑闻武之六成乎武之济河而西也马散弗乗牛散弗服倒载干戈包以虎皮天下知其寝兵不用也爰作武以象功焉鼓以戒众久然后战也长戈连延起其慕也发扬蹈厉时不可失也故一成北出再成灭商三成自北而南四成南国是疆五成分左右以居周召六成复始而爲天下王振铎夹舞秉戈鹰扬若是何如公子曰其容美矣此武事也未尽善也请言其上主人曰其惟舜之大韶乎当夫六府治三事和叙九功形九歌峄阳之桐可以琢琴瑟焉云梦之篠可以窍管籥焉泗滨之石可以砻而爲磬焉荆山之金可以范而爲镛焉八音旣具而大体短脰之属有力而不能走者以之爲钟簴焉小体骞腹声清而逺闻者以之爲磬簴焉其作也椌以合之其终也楬以止之洪者铿而充清者磬而介凄切而不流泛滥而可会当夫朝廷之燕享宗庙之祭祀于是而奏焉升歌在上匏竹在下代作间奏秩秩有序或击或戛或拊或搏清亮而髙逺象乎天之浑沦广厚而含容象乎地之磅礡一变一通兮四时之终始一散一润兮风雨之回合是时也熙熙然八荒一春皥皥乎洪荒大朴格三苖于洞庭丹朱宾而有恪俯而聆之纯如辛甘之相济皦如有伦而莫敓诎如断玉之复续绎如骊珠之碎落飞流合而万壑雷转清风生而天籁交作吟九渊之老龙唳九臯之鹤旣么妙而悠扬亦和平而澹泊无急微噍杀以感其忧无啴谐慢易以感其乐其动于物也容何爲而肃若神何爲而来思兽何爲而舞于土阶鳯何爲而翔于茅茨明协乎大章博儗乎咸池所以保无穷之治以示安而不忘乎危也悼六龙之南廵歴苍梧而上九疑阿母之玉琯犹在湘灵之锦瑟空悲怅遗声之寂寂叫有虞兮逺而然而齐之有韶陈亡而流于兹也海外之有韶圣人之化播于蛮也宜季札观之而知德仲尼学之而忘味彼武之六极于隂特着其武功此韶之九极于阳实昭其文治也公子以爲何如公子乃避席而谢曰至矣尽矣不可以有加矣主人曰未也请授以九德之歌其一洪流横曰黄判风气开洪流横民乃灾五行汨帝爲哀禹治之启始孩决九川平九垓百谷生田每每其二洪流杀曰洪流杀兮民灾旣除山有鸟兽兮川有鱼爰可食兮可居胥乐且歌兮毋忘厥初其三三苖格曰惟圣亶聪抚有九围蠢蠢三苖险阻是依出师于南奋我帝威我师旣还苖亦东归其四四凶黜曰德与刑兮国之经四凶斥兮八元在庭帝无爲四海宁其五正德之歌曰惟天降命物必有则民之瞢瞢胡乃自贼五教不行禽兽相食圣人龙飞四方之极其六利用之歌曰水旣平兮别九州下爲隰兮髙爲丘徒则轩兮渉则舟以羡济乏兮百货流财孔阜兮乐且无忧其七厚生之歌曰下民孔艰兮遭垫溺寒我衣兮饥我食我无民违兮父母职其八鳯皇来仪曰尧不德兮舜不辞授以天下万物治寒暑无易风雨时箫韶九奏朱凤仪其九万世赖曰天覆地载髙广莫测孰参賛是俾衣而食帝治天下如埏如埴万世赖之安知其力公子曰呜呼礼废乐崩几千秋矣而始闻主人之宏论乃复爲之歌曰雅南已亡兮流荡曷正觱栗雷怒兮空桑无声緑绮不陈兮荐檀槽与秦筝妖倡爲妍兮啸鸾鹄而啼燕莺孰究夫大韶之尽善尽美兮实嫓夫六英六茎后千载之圣人兮集厥大成安得闻九奏于清都兮御天风而上征
　　不碍云山楼赋【并序　附考华亭张溪杨谦之居】
　　赤松溪杨竹西氏筑楼一所在居第之南而海中大小两金山飞舞而前因取杜少陵诗语顔之曰不碍云山銕崖杨先生爲之记夫有云山之境者恒不得其人有云山之趣者恒不得其地而兹楼独领其要朝岚夕翠不起燕坐而尽得之岂非先生所谓境以人髙人如境胜哉竹西尝命余爲赋久未克就歳阏逢执徐之春放舟溪上始获登竹西楼逺眺五茸三穷目力所至爲限飘飘乎若一羽淩空飞度三素云中也命酒共酌操壑雷之琴歌白雪之辞极欢而罢因援笔爲赋不惟状云山之胜且寓感今思古之意焉赋曰
