　　提要

　　（臣）等谨按药房樵唱三卷附録一卷元吴景奎撰景奎字文可兰溪人力学有文年三十海道万户刘贞为浙东宪府掾尝辟为从事明年贞去景奎亦归乆之用部使者荐署兴化路儒学録以母老辞不就至正十五年卒于家事迹详具黄溍所作墓志中景奎于书无所不读诗歌尤清丽警抜颇近唐音是集乃其子履与其门人黄琪所编中间五言古体皆源出白居易七言古体间似李贺近体亦音节宏敞豪放自喜故宋濓作序极推重之特编次时失于简汰如偶成诗云挟才胜徳世所薄宁我负人天可欺士之言行苟如此圣经贤传将奚为此等句殊不成格其它应俗之作亦多榛楛勿翦是则履等辑録之过然其菁华自在固不能以此相掩也景奎又尝采取骚选中粹辞奥语勒为一编名曰诸家雅言以为学诗者入门之法其书今已失传云

　　●药房樵唱序

　　序曰药房樵唱者吴公文可所著之诗也夫诗在堪舆间无纎弗囿无巨弗涵太极阴阳之化物则民彛之懿烟风月露之形河山草木之昭气候燠寒之更毛羽鳞介之蕃治乱兴亡之着夭寿死生之变可疑可存可悦可愕可感可慨外触乎物内发乎情情至而形于言言形而比于声声成而诗生焉譬之气至簧鼔神合自然机摇弩发道契冲漠上自王公卿士下逮小夫编氓率藻畅于襟灵一导扬于隠伏大而朝会燕飨被之弦歌小而委巷深闺见诸谲谏虽位号之或殊而情衷之无异商周之隆斯义为盛汉魏以来古意渐削下沿唐宋之间而得之者盖鲜矣于是吴趋楚艶而哇淫之咏汨焉牛鬼蛇神而诞幻之事彰焉霆飞霰掷而粗厉之文布焉胡呗梵吟而忽荒之趣见焉伧言粤语而俚鄙之亵形焉莺支蝶卉而留连之思滞焉诗道亦几乎熄矣公尝与濓剧论至斯为之弹指三叹盖公以雄逸之资济通明之识着于篇翰规仿风雅鼓动江山之气发挥造化之微味玄酒于周廷袭悬黎于梁苑雕龙彩鳯不足为之丽冲飊激浪不足为之豪其凄惋也则孤猿夜号松露初滴其雅驯也则冠冕佩玊俨趋廊庙由其才无不兼所以体无不偹世之读者如入玄圃而揽明月木难之珍如登昆丘而覩天禾肉芝之贵诚可谓擅名制作之林竞爽艺文之场者已如濂不敏年踰四袠学废三余精神遐漂无水宁之梦金石相宣乏荆潭之情景伯之舌徒存文通之笔已失顾念畴昔获陪杖屦濯缨双溪之侧潄齿灵源之上攀萝月以夷犹抚檆云而舒啸公时吞吐群机陶镕庶彚珠玊随风氷雪在口人争传于秀句价欲等于兼金奈何稍歴星霜遽分今古虽瀼西之室遗迹未寒而辽东之鹤一去不返尚忍言哉尚忍言哉公之子履与其门人黄琪编辑遗稿锲之文梓乃縁世契之深逺以首简为属嗟夫玊光剑气直出人间麟角鳯毛终为世瑞萧功曹之新章不冺李奉常之妙什宜传此理之常无足疑者第以疾疢相仍文藻衰落无义山之雅制序漫叟之雄篇姑缀芜辞以信微臆云尔公讳景奎文可字也婺之兰溪人群行存诸别传履博学善属文尤精于诗无忝于公者也至正十八年岁次戊戌三月己亥朔契家侄金华宋濓谨序

　　喻鸠

　　东村五畆园荒秽久不治荷锄种豆苗爱此膏土肥森然已甲拆保之若婴儿群鸠竞拂羽俛啄思无遗傍麾虽稍却投闲复在兹一啄不汝惜再啄苗已稀剜我心上肉充汝膓中饥汝饥诚可怜矰缴那忍施桑林正沃若桑椹方垂垂椹熟可醉饱林深无危机恩怨从此释斯园勿重窥鸠鸣对以臆群然亦高飞

　　采菊

　　爽气浮西山风烟澹原隰悠然出衡门佳节逢九日众草萋以青黄菊正秋色采之不盈把妙意良自得谓能制頺龄可泛忘忧物驩言命壶觞万事付酣适永懐东篱人俛仰已陈迹

　　正月
　　麹尘丝拂晴波暖霁霭空蒙烧痕浅含章宫中万玊妃粉花点额芳霏霏条风东来初解冻蝴蝶吹醒花底梦翠鬟梳罢玊纎寒彩燕衘春上金鳯
　　二月
　　青皇宫中花鸟使沓翠霏红教莺语移春槛小度芳华金犊车轻载歌舞曲房梦断章台月东风不解丁香结钥鱼夜守仓琅根海棠飞落臙脂雪
　　三月
　　紫箫吹月棃花老碧云冉冉迷芳草药栏茧粟怯春寒翠帐流蘓护莺晓丽人一去音尘杳泪洒红绡怨青鸟新蒲细栁自年年曲江流恨波声小
　　四月
　　轻红流烟香雨足新槐影转勾栏曲水晶帘箔度薫风黄吻乌衣语华屋凉簮堕发初破睡粉痕浅护修蛾緑并禽不受雕笼宿背人飞向荷阴浴
　　五月
　　网轩緑艾县飞虎菖蒲花青海榴吐江娥倚竹弄湘弦调笑懐沙怨兰杜南薫生凉纨扇薄雕俎瑶觞劝郎酌彩索光浮系臂纱守宫红映黄金约

　　氷山珑璁间瑶席水拍银盘潄寒碧象床湘簟含风漪腻香粉汗沾凝脂赤帝啾啾火龙老琪树西风转昏晓

　　九月
　　黄金花开香满把烟草荒台谁戏马楚云栉栉雁西流秋色凄人正潇洒泪花蔌蔌啼新愁纒弦五色弹箜篌寳香不暖茱茰帐明月空过翡翠楼
　　十月
　　小春一花西月黄缟衣美人吹暗香锦衾罗荐晓寒薄梦中持赠双明珰霜花莫洒相思树愁杀孤栖金鳯凰
　　十一月
　　八姨手折榑桑枝海天冻合青玻瓈琼楼仙人唤滕六夜入银潢剪瑛琭沈香火暖锦承尘叵罗羔酒生春粼宫沟不寄题红怨日映五纹添弱线
　　十二月

　　闰月
　　若华煌煌萦日驭气朔盈虚积余数低鬟敛黛拜孀娥孤负圑圎十三度生物趋功得岁长山中独厄黄杨树

　　五十自讼示同志

　　嗟我行年今五十鬓毛半染秋霜白方知四十九年非追数生平竟何益少年懒惰废经史明时进取无谋画临财不解竞锥刀饱食安知勤稼穑故园已荒松菊径索居却占渔樵席陆沉井里等樗散宾饯羲娥驶驹隙齿牙摇落筋力疲两目生花眩朱碧母年七袠余四龄饮食康强喜愉怿大儿无成小儿幼有弟相望念离隔弱孙绕膝不识愁弄笔摊书恣狼藉不如意事十八九俯仰兴懐百忧积敢随佻巧人共嗤自信修姱心不易使予浸假嗜酣酗好勇鬬狠从博弈岂如用拙乐贱贫清苦宁辞食氷蘖牛衣不暖泣中夜犊鼻凄凉徒四壁丈夫未遇亦如此顾我何为叹纬繣已将穷达付天公啸歌起视沧溟窄方今忽忽岁遒尽黄落山林见松栢揆余初度安足论母恩未报嗟穷厄醉来聊以写我忧返锁衡门谢宾客

　　中秋对月

　　素娥游宴清虚府手摘蟾花啖寒兔琼楼高处作中秋净洗银盘浥风露此时令我忆去年山中桂影浮觥船今年却来江海上美人千里同婵娟世间忧乐知多少天地无穷月长好明年踪迹何处看此夜阴晴亦难保

　　徐仙卿歌

　　两仪肇分天地位融海结山由至理昆仑之墟西极高天下羣山从此始川流百折皆朝宗山气巃嵸腾苍龙大为京都小郡邑民物搃搃成提封辨方相土传今昔是亦阴阳家一术自从管郭着遗书后世纷纷相祖述胎仙化作徐仙卿黄冠野服芒鞋轻千峰飞步揽空翠佳气欝欝双瞳明玊函玄经妙万化王赖曽杨齐逸驾道傍忽尔得牛眠土与黄金即同价雁鳬银海长冥冥珠襦玊匣非生荣伯伦王孙埋与躶三年石椁何由成我思掩亲必有道十载未成心悄悄卿言只尺有佳城水秀山明宜宅兆徐卿目击而道存涧溪培塿安足论五湖四海浮日月太山乔岳撑乾坤赠君一緉登山屐踏遍江南与江北观光直上燕然山日近蓬莱云五色
