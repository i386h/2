　　提要
　　
　　臣等谨案巽隠集四巻明程本立撰本立字原道巽隠其号也桐乡人洪武九年以明经擢秦王府引礼舎人以母忧去复补周府礼官坐累谪云南马龙他郎甸长官司吏目有平定百夷功征入翰林厯官至右佥都御史调江西按察副使未行值燕王簒位遂自尽以殉事迹具明史本传是集诗二巻文二巻为其曽孙山所编弘治乙丑桐乡知县莆田李廷梧序之嘉靖初南溪吴氏为刋板西虞范氏又重刋之岁久皆散佚此本乃万厯乙丑桐乡知县濮阳棐购得遗稿于其裔孙九泽而属训导李诗校刋者也本立文章典雅诗亦深穏朴健颇近唐音不但节义为足重即以词采而论位置于明初作者之间亦无愧色矣

　　○纪行
　　
　　使淛江发大梁水驿
　　
　　蔡河雪消春水生使者发船将逺行得遇乡里虽慰意郄念妻子难为情草埋金谷楼前路潮打钱塘江上城南北登临总怀古船中有酒且须倾
　　
　　舟中杂兴
　　
　　蔡河百折到沙河沙河入淮风浪多使者官船夜须发奈尔长淮风浪何
　　
　　十日舟行不见山沙河多似蔡河湾舟人忽报安丰近指点数峯烟雨间
　　
　　昨夜雨来风更颠淮河北口且湾船今朝浪静张帆去蓬底看山不碍眠
　　
　　野色空青带薄岚水光深黑入洄潭望到中都三百里好山不断似江南
　　
　　过宝应
　　
　　栁拂津亭春雾开缆牵官舸鼓声催邑中旧尹人能说湖上新堤客再来刼火空城遗鴈塔恶风深浪没龙堆若过甓社看明月切莫探珠老蚌胎
　　
　　过甓社湖
　　
　　平湖一目到天涯明月满空飞浪花半夜珠胎遗老蚌有时金背出妖蟇渔人屋近灯纔见使者舟行鼓乱挝便似吴淞江上路只无红桨唱双娃
　　
　　到京口驿
　　
　　万里长江风浪平羣山犇走入神京船头酾酒晴开港马首张灯夜到城南去北来须白发东渐西被尽苍生明时食禄曾何补只冩声謡记所行
　　
　　许州东屯杂诗六首
　　
　　东屯归欲晚西涧立多时病眼青山怪幽情白鸟知十年无旧业九日有新诗采得篱根菊馨香好遗谁
　　
　　农功将筑圃使者正行田车过闻牛铎诗成堕马鞭凉风苹末起落日树头圆翻忆吴淞岸菱謌处处船
　　
　　出郊观获稻下马立斜晖百顷黄云卷千鎌白月飞莫辞官有赋但使愿无违农器消兵甲时清岁不饥
　　
　　死战开边土生还落野屯时清无盗贼头白有儿孙舂稻霜余粒移桑雨后根百年须饱暖此外复何言
　　
　　丘原莽萧瑟客子暮徘徊山是钧州起河从郑邑来衣冠余古墓歌舞只荒台野哭谁家妇秋声助尔哀
　　
　　径捷车行少溪深马渡迟地临长社邑民指太丘祠所歴多怀古予生岂好诗据鞍吟复啸风雨忽凄其
　　
　　雪中发中牟
　　
　　华轩照白发使节来清霄行人勿畏避丈夫戒矜骄衢路寛逾坦尘沙净不飘髙怀乐清景于以永今朝石田已种玉吾将采琼瑶琼瑶遗所思所思不可招
　　
　　过汉鲁中牟庙
　　
　　鲁公宰中牟千载民弗忘我车式祠下纡郁感中肠公昔为政日异声动四方野桑有驯雉田稼无飞蝗倘非司徒使厥美亦莫彰宰邑几何人孰继芝兰芳催科簿书急应接趋走忙人亡政化息俯仰思循良
　　
　　发郑州
　　
　　驾辕指嵩洛发轫涉京须两厓忽劖削一径中萦纡毡惟命低褰雨雪恐沾濡端居岂不乐顾乃兴叹吁仆夫尽徒跣轨辙多泥涂仁者念同胞饥寒切吾躯行行敢中止戒之以徐驱
　　
　　过虎牢
　　
　　鸿沟楚汉割虎牢晋郑交周衰逮秦亡海内争雄虓生民亦何罪杀戮如刈茅至今战争地鬼哭熊昼号汜水去汨河广武来■〈山孝〉嵺浩歌入深峪悲风起山■〈土幻〉
　　
　　出蓼子峪
　　
　　羣山入巩洛渴骥奔黄河行人山之下窄径縁坡陀单车不方轨峭壁临盘涡上有虎豹嘷下有饥蛟鼍埀堂夙所戒叱驭今重过微忠効驰駈神物烦撝呵
　　
　　过黑石渡
　　
　　邙山障黄河洛水崩黑石方舟载车马烟雨行人立我昔将王命北渡祠河伯兹来复轺传观者笑曾识圣君禹功齐神龟当复出
　　
　　出洛阳城
　　
　　挽车上天津伊阙当我前连峯左右起奔走相后先古来五岳内嵩髙极中天仪形正四表襟带流三川河山固王室岂直金城坚汉业此中兴周都见东迁壮游快一览遗迹悲千年颓垣旧谁筑野蔓凄朝烟
　　
　　重到许州屯
　　
　　士卒欢迎马首罗旧时行处复来过荷殳孰与耕田乐拔剑无烦斫地歌日落两屯烟尽起雨余双泪水偏多强拈薄酒怜人意壶榼频倾奈尔何
　　
　　发中都水驿次韵金应奉李编修纪行倡和诗
　　
　　中都楼观旧登临佳句难酬两翰林白石丹书阴洞古青山王气暖烟深感时自有怀乡恨得禄谁无恋主心庄舄当年名亦显病人不改越人吟
　　
　　过五河县
　　
　　水到五河东北流河边人立看仙舟醒来纸笔有新作乱后衣冠非旧游蜗角地争蛮触国凤凰山是帝王州从今四海无征战野老能为击壤讴
　　
　　过龙窝驿
　　
　　龙窝驿近鼓声催转柁落帆沙岸隈共指深渊是空洞岂无神物待云雷风晴杨栁如看画水緑蒲萄始发醅浩荡好怀春色里诗成南鴈忽飞回
　　
　　中都早行
　　
　　鸡鸣登前途白日未出海堠傍坐之久扶桑发光彩安知云雾起倐忽阴晴改我行正愁雨亟去复奚待此意难重陈无由诉真宰
　　
　　过池河驿
　　
　　萧萧牧鸣马辚辚挽征车人生各有役我何念安居东冈上初日雾暗未光舒依依见亭堠暧暧经村墟我足日已研我仆力已痡自非得神仙岂免劳形躯
　　
　　过太山
　　
　　赤日当中天炎夏势方盛孰知阴已生动物气先应挥汗登太山抱膝坐危磴虫吟与蝉咽满耳入秋听君子识几微于焉发深儆
　　
　　宿大柳树驿
　　
　　柳驿夜寥閴空庭树扶疎逢人两无语独坐临前除明月俄在膝好风又吹裾微吟复谁和幽抱聊自摅入室就衾枕蛩响四壁虚展转不能寐起听鸡鸣初
　　
　　过清流闗
　　
　　昨日太山岭今日清流闗谓非大行坂所歴亦险艰前瞻何突兀后顾犹巑岏下或堕地底上或登云端筋力讵能及十步九嗟叹愿言御冷风安得生羽翰
　　
　　凤阳东闗
　　
　　东闗发船烟霞开城郭山川何壮哉长淮千里出桐柏庆云五色近蓬莱汉廷书生有贾董梁园词客皆邹枚自叹无才日樗散金门退食尚迟回
　　
　　髙邮夜泊
　　
　　城楼月出见觚棱城下官河夜欲冰返照疎林皆野烧残星别浦是船灯腐儒食禄曾无补倦客思家已不胜春雨五湖烟水阔荷蓑归去下鱼罾
　　
　　扬州
　　
　　逺水疎星影动揺西风落木景萧条伤心一片船头月犹照扬州廿四桥
　　
　　扬州城下是官河春雨春风自緑波欲问繁华旧时事太平遗老已无多
　　
　　老翁能说太平年曾见江南进士船打鼓挂帆扬子岸人人尽道是登仙
　　
　　城门疎栁噪寒鸦柳外青旗是酒家无复春风旧桃李行人不必问琼花
　　
　　侍从亲王之国大梁代祀岳渎陵寝与宋季子奉祠偕行至郑州分道
　　
　　天香满袖散氤氲自觉祠官思不羣我望洛中山尚逺子行河北路初分翩翩并马游三日去去孤鸿隔片云久许精诚可相照祝厘万一报吾君
　　
　　京索道中
　　
　　路从京索上陂陀于此劳人感慨多徒割鸿沟分楚汉终归隆准一山河大风沛邑台中筑半夜虞姬帐底歌千载兴亡俱寂寞羣山起伏自嵯峨
　　
　　夜宿郑州
　　
　　昔骑官马来管城闗夫走报将军迎牛车今夜民家宿落莫何曾问姓名姓名碌碌何足问要知蛮貊行忠信槖底宁须陆贾金腰间不必苏秦印我惭二子人所荣谁于宠辱能不惊万里得归贫亦足只消妻子事蚕耕
　　
　　洛阳七里桥命儿还大梁是日风霾大作晚色忽开一诗自遣
　　
　　我老西行几日还归心随尔过崤闗晚晴扫得离愁去稳坐车中饱看山
　　
　　过渑池问路于叟叟指罢兵台（秦赵会盟之地）
　　
　　青山四顾似环滁千载荒城百战余问路罢兵台下去邑人能说蔺相如
　　
　　华阴驻马桥
　　
　　絶谷层闗路屈盘斜冈侧嶂石巑岏今朝驻马桥头立华岳三峯正面看
　　
　　入益门
　　
　　蜀闗秦岭路初分迭嶂层厓日易曛飞栈下临千尺磵行人上出半山云髙堂烟雾曾贪画险道风湍却厌闻万里敢辞筋力尽捐躯未足报吾君
　　
　　过凤县简主簿徐敏
　　
　　城门人迹没蒿莱城上青山四面开土物只看鹦鹉卖邑人空说凤凰来三年簿领心如水十岁儿童瘿佀桮赋得诗成冩厓石欲留姓字刻苍苔
　　
　　过沔县
　　
　　山川良是昔人非马上兴怀驻夕晖汉贼未平诸葛死韩仇已报子房归君臣会合时难得忠义分明事或违千古悠悠何限意诗成不觉雨沾衣
　　
　　过五丁峡
　　
　　涉水复涉水登山复登山行行不计程日日山水间及兹五丁峡蜀道方险艰水多洪涛恶山有乱石顽梁摧乃能渡梯絶无由攀我马毛已焦我仆力已孱岂因筋骨劳而使涕泪澘幽林好鸟鸣听之心自闲西南六诏地此去何时还斯须敢忘君咫尺不违颜愿承风八翼直上天九闗
　　
　　合州
　　
　　合州楼前江水合合州楼外青山匝倚楼西望雨冥冥顺水船来飞两楫阆州山水天下稀不如合州可忘归街头彷佛吴门市人家瓦屋白板扉买鱼沽酒醉今夕相逢谁是吴门客鸡鸣风雨梦初醒邻机轧轧村舂急
　　
　　铜梁县
　　
　　铜梁县庭公事少铜梁县市官桥小上山下山石作街街南街北人家好人家好农不好商生男嫁女不出乡香芹煮羮饭有稻木棉纺丝衣有裳陈郎邀我青樽满情多已觉回鞭懒酒酣欲写铜梁謌百钱买笔紫罗管
　　
　　夜宿走马铺
　　
　　铜梁百里入昌州水竹村村山更幽走马亭中看明月不知今夜是中秋
　　
　　海上白云如白波天涯无奈月明何遥知今夜闺中看比我天涯泪更多
　　
　　大足县
　　
　　稻满平畴水满渠舍南舍北竹疎疎吴侬莫说吴中乐到处青山好卜居方池鹅鸭乱晴波丰草牛羊散暖坡如此山居元不恶我生何地着云窝
　　
　　宿荣昌县田家书所见
　　
　　田家无桑蚕不育寒机不奈蛩声促阳坡种得木棉花雨晴雪点秋云緑小姑大妇不梳妆日日探花如采桑青裙短短赤双脚素手掺掺花满筐归来闭门月昏黑燃薪代烛光照室姑揺纺车妇在机一夜不能成一疋松州茂州道路难乌蒙乌撒雨雪寒丁夫运粮给边饷卖布易米还输官田家不识纨与縠木棉未得裁衣服何日边城罢战输男耕女织万事足
