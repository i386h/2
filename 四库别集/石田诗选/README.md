　　钦定四库全书

　　集部六

　　石田诗选

　　别集类五

　　明

　　提要

　　臣等谨案石田诗选十巻明沈周撰周有石田杂记已着録此集不标体制不谱年月但分天文时令等三十一类葢仿宋人分类杜诗之例据慈溪张鈇跋葢其友光禄寺署丞华汝徳所编也顾元庆夷白斋诗话载都穆学诗于周尝作节妇诗有青灯泪眼枯句周以礼寡妇不夜哭议灯字未穏是周于诗律不为不细然周以画名一代诗非其所留意又晚年画境弥髙颓然天放方圆自造惟意所如诗亦挥洒淋漓自写天趣葢不以字句取工徒以栖心邱壑名利两忘风月徃还烟云供飬其胸次本无尘累故所作亦不琱不琢自然拔俗寄兴于町畦之外可以意会而不可加之以绳削其于诗也亦可谓教外别传矣都穆南濠诗话称其咏钱咏门神咏帘咏混堂咏杨花咏落花诸聮皆未免索之于句下葢穆于诗所得不深故所见止是也集前有呉寛序称其诗余发为图绘妙逼古人核实而论周固以画之余事溢而为诗非以诗之余事溢而为画寛序其诗故主诗而宾画耳又有李东阳后序东阳与周不相识时已为大学士与周势分悬隔以呉寛尝以写本示之重其为人故越三十年后又补为作之然二序皆为全集而作华汝徳刋此选本时仍而録之非序此本者也

　　○天文

　　弘治改元元旦遇雨

　　元旦未及春东风意先苏风来继零雨和润物皆濡助我新天子沛泽弥九区河流亦传清天与圣徳符贤俊日拔庸治道务精图尧舜不异人政化无两途我衰在畎亩感激舞蹈俱告戒各努力勤耕备公租黍穰多充禽渊静无惊鱼上下信影响余生幸于于

　　雪作二首

　　经春十日忽作雪如此纷然门且关瞥见豪华成白屋错加粉饰累青山侵凌广野容无地零乱髙空落自闲兀坐搔头何遣拨梅花寒勒未开颜
　　春雪公然沮春雨莫能为润秪漫漫撒盐差是殊无味执玉徒观不辟寒满地萧条悲乌雀何人徙倚爱阑干冻皴手足须俱折拥被微吟强自寛

　　二月三十日雨中

　　此雨送残月阴云隐有雷草虽宜湿长花亦要晴开沟漫鱼争上泥深马忌来春山只隔眼应待我衔杯

　　三月一日喜晴小步用士弘韵

　　雨春情况闷中谙一日初晴月已三半湿半开花解笑相呼相唤鸟交谈闲行聊乐将谁伴独咏而归也自堪更有余懐成漫酌好风吹面小轩南

　　嘲雨

　　雨泽之所至千里为其方其落万万注河沙数微茫方中髙低田多少落不当髙者少莫润低者多乃殃虽有利物名功过不相偿落之徒劳劳所怨亦穰穰何如一轮月分影不可量一处有一月一月散千光只譬一州城万人住中央一人一觞酒一月照一觞一觞吸一月人醉月无伤月自行中天不惊亦不忙房杜昔为相垂绅坐庙堂天下属黙化熈哉称有唐安石似道辈琐才事更张聚敛以为政戕民隳宋纲君子与小人于焉观否臧我方坐雨厄斐然成此章

　　风雨中戏三郎园樱尽落

　　风风雨雨未开晴忧病忧田白髪生闻道西园尤作恶小僮欢喜拾残樱

　　雨闷

　　乆雨连阴结青天安在哉大由云所子浩及水为灾羣叶无情尽么花借湿开寂然泥淖地苦忆旧人来

　　雨夜自遣

　　需霁寥寥坐夜长呼童转烛更添香澄懐有物嫌诗在聋耳无闻并雨忘头上青天终自出人间白屋暂相当眀朝补漏吾还得手把黄茅作向阳

　　五月十五夜见月映树有作

　　圎月出树东隔看被树罣羣叶相蒙茂光采还透射玉盘虽隐映其魄自整暇脱树渐自髙万里大明奓了了太极圏谁向虚空画坐深已当头其树势斯下小人蔽君子暂尔何足讶树月亦偶当微吟聊譬借

　　中秋湖中翫月

　　爱是中秋月满湖尽贪佳赏亦须臾固知万古有此月但恐百年无老夫鸿鴈长波空眼眦鱼龙清影乱眉须人间乐地因人得莫少诗篇及酒壶

　　钱士弘以东坡船邀泛月病阻因答

　　东坡新船髙似屋两舷开窓三十六昨乘缺月泛昆湖今泛杨湖月方足老夫不及困河鱼到月足时人欠福莫能扶惫踏髙筵马不进兮人累俗翠盘露酒白玉香一饮之縁惜难卜月从水上忆车公还能入户寻幽独枕前只作卧逰人亦有清华烂双目髙吟不寐待回船输却醉乡诗可赎

　　暑中冩雪图

　　六月添衣唤僮子自画雪图茅屋里玉花出笔飞上树惨澹阴山无乃是老生放笔还自笑颠倒炎凉聊戏尔门前有客来借看满眼黄尘汗如雨

　　松庭对雪

　　三白触眼喜欲颠天王继统新改年岂天厌恶下土浊以此雪之同皎然光照眀堂纳祥瑞一平天下知其权疾而蒙藏能过厚卑而令髙如进贤先时上帑宝珠玉今恣四野开琼田何须聚敛自委积过而不拾贪夫悛先时豺虎横食人空山忍饥今可怜威能挟天已阘茸势可炙手还■〈虫连〉蜷老夫忍冻亦惯事草庐安抱牛衣眠庭前孤松有喜色要与老夫相作縁

　　冬日

　　冬日晴可爱温如君子人即之不觉热黙生懐抱春执巻南荣下吾目亦滋神何嫌炙背徒负以献吾君

　　腊月二十八日立春得雨病起漫言

　　小阁棂窓风自推手添香炷拨炉灰残年送病诗中去淑气将春雨里回午案待甞新茁韭阳檐思访欲开梅岁功代谢忙如许似与衰翁逓老来

　　悯日杂言

　　日既去日复来来如赴去如颓来是谁约去是谁推一来一去彼此自禅续无与我与何故使我心惊猜似乎少年有根是汝拔老丑无种是汝栽百年所筭三万六千日自我而数指作枚我今行年已七十歴日二万五千枚所该百而去七大大半又复使我心惊呆虽欲不惊呆猛见霜丝雪缕垂两腮何况人生不满百疾乌捷兎又如此而相催我思天地灵长之气十二万九千六百年然后运穷刼尽荡而为灰吾人亦谓参三才胡乃其气短索不得相追陪凖天地而言人眇尘海之一埃嘅岁月之玩人同古今而一雷我无长绳系日住亦无长戈挥日回亦不知学仙能乆视亦不知托佛能轮回而今而后去之日付一杯来之日付一杯不忧罄其缾耻其罍春暖秋凉山邉水隈访黄菊寻白梅秋月自与吾虑净春云自与吾怀开昼逰之地吾蓬莱夕息之处吾夜台以殇视我吾老大以彭视我吾婴孩信寿夭吾何以外请享此见在不乐胡为哉

　　苦雨寄城中诸友二首

　　一阵接一阵一朝连一朝官仍追旧赋天又没新苗白日不相照浮云那得消君休问饥饱且看沈郎腰
　　新雨似旧雨今年即去年只愁沈垕土或喜梦青天顿顿黄虀瓮家家白鹭田惟应五谷地改纳水衡钱

　　秋雨偶书

　　云暗湖村水漫沙秋堂萧飒类僧家聦明比旧堂堂减老懒随年数数加风扇未愁蕉破叶雨房偏恐稻伤花坐深兀兀多资睡强欲开书候日华

　　中秋赏月与浦汝正诸君同赋

　　少时不辩中秋月视与常时无各别老来偏与月相恋恋月还应恋佳节老人能得几中秋信是流光不可留古今换人不换月旧月新人风马牛壶中有酒且为乐杯廵到手莫推却月圆还似故人圆故人散去如月落眼中渐觉少故人乘月夜游谁我嗔髙歌太白问月句自诧白髪欺青春青春白髪固不及豪卷酒波连月吸老夫老及六十年更问中秋賖四十

　　星堕

　　惟己酉十月五日立冬始鸡后涉黎眀寤者稍稍起有星流西北而从东南止其大如车轮蓬然曳长尾撒星更抛熖遗落相蘂蘂其光白中黄烛地以昼比行旅尽惊仆吠犬鸣鹜雉堕地如炮声引响乆未已床闼皆簸撼约闻三百里七月尝示异不意复有此瞥过视莫谛堕处曷究拟至地当有化其形当何似其堕当何曜其罚当何理窃谓经与纬堕一天缺纪无乃金火余其气互合耳余衍固非正谪罚自有旨菑祥付茫茫不敢扣太史

　　对春雨

　　春雨父母情惠物侔爱子润被发华妍长养助欣喜庭垂存腐蕉土渥芽伊始斋心临散丝清虚及窓几自怆槁飒姿均荣固无理

　　中秋无月歌

　　老年于世百无与待尽光阴而已耳佳时令节或难忘有酒还随大家喜一时过去少一时打筭百年能得几满前童丱倐突弁少者如斯老何恃去年中秋月在城白璧团团酒波里今年中秋月在家买酒邀宾走儿子云漫天黒月失色老子之兴殊败矣老子非月能万古暂借一光何足鄙中秋失月不中秋人固无聊月当耻常时有月我无酒有酒呼月今不起有无无有两难谋要是天公忌全美世途茫茫寓万事顚之倒之亦如此眼中屠贩却封侯意外诗书多饿死便从一月观万变何敢容心必其理感深坐乆欲添衣风急秋髙月如水

　　溧阳道中遇雨

　　浮云接地日无光足力衰迟道路长都是小儿能疾走这场风雨却谁当

　　有竹庄赏月

　　烂拥银盘草屋东白头相赏两三翁青天不老人自老明月正中秋亦中怪底有娥因药误谋诸无妇恐尊空百年各愿身强健此夜年年此燕同

　　八月十七夜待月

　　候月月不出引醪兼话长栁黄延返照苎白应新凉恶璧沉海气待珠开夜光我罍犹未耻愿早贲林堂

　　○时令

　　元日

　　今日之日一年始三百六十年之终云物昭回天杲杲风光澹荡气融融商量栁色緑少许烂漫梅花白大同无事此杯椒栢酒老夫不觉笑颜红

　　四日过有竹荘

　　春早拏舟闻鹊鸣病余勉强竹荘行开年数日匆匆易向老逢人徃徃生阴陇瘦姿松抱冷雪洲芳意草知晴稍通酒■〈角戋〉聊乘兴亦漫诗篇度野情

　　人日喜晴

　　今辰谓人生风日佳可喜七居阳之正宜云造物始占晴协阳吉其兆比君子雾雨忌为阴小人象斯拟君子履于祥岁事无不举小人道用晦顾有菑害倚东皋动春风强健供耒耜持以告吾人毋忧徃时否

　　七日树介

　　千松凛凛清霜戟万竹森森白玉殳无数小儿相怪问老夫搔首但邾娄

　　元夕席上赠冷庵宪副二首

　　石湖乐事漫呉台茅屋今宵也一回上客门前探节到小儿城里买灯来生香竹叶迎春熟活色莲花映火开妆点太平须要此百年常覆掌中杯
　　堂带春星月满台劝君且饮一千回休言美酒消愁得亦被华灯送老来急管繁弦声乍合千门万户夜齐开明年佳节依然在分付春光驻此杯

　　春晴出游

　　天宇及轩霁品物各华研清般约同侪络醑仍击鲜展酌临芳树鸣琴参笑言促羽酬良辰迅光不长延请看刘伶坟酹之莫达泉

　　闲居四时吟

　　白日可形指青春蔑迹寻不应有开花还复有鸣禽何物衰飒翁乐意亦充襟浩然不能遏登临动谣吟谣吟侑杯酌花鸟同山林一畅太和气载鼓康衢音扪心自有会求知非在今
　　赫暑薫永日众皆苦烦酷林庐頼溪次茅茨非厦屋门前荫重栁檐后郁深竹无风气萧爽汗渖免浃肉尚可加白葛喜谢裸裎俗昼静茶瓜余摊书教孙读既倦还偃息梦寐亦清淑
　　秋凉我所爱人情与时宜因念康节言还眷康乐嬉杖屦足济胜出门随所之举睇达遐迩山水陈髙卑洞庭吹白波落木不可持我特覧眀镜因之悲素丝素丝不复玄物变咸有时亟欲学神仙早悯气血衰羡彼南山色青青无老期
