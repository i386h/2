　　提要
　　【臣】等谨按雪牕集二巻附录一卷宋孙梦观撰梦观字守叔号雪牕慈溪人宝庆二年进士官至吏部侍郎后求外补以集英殿修撰知建宁府事迹具宋史本传是编乃明嘉靖间其裔孙应奎所校刋有刘校后序云集凡二巻曰奏议曰故事其志赞诔文为附录一巻故事者征引古书于前而附列议论于后更番进御因事纳规同时李曽伯集亦尝载之盖当时体制如是也其奏议自嘉熙庚子以迄宝祐丙辰正宋政极坏之时所言皆切直激昻洞达时务如谓理宗能容直言而不能用又谓士大夫有寛厚之虚名非国之福尤切中宋末之弊视当时迂腐儒生髙谈三代衣冠而拯焚溺者固不可同日而语矣乾隆四十四年五月恭校上

　　雪窓集本传　　　　　　　　　明危素撰
　　孙梦观字守叔庆元府慈溪人寳庆二年进士调桂阳军教授淛西提举司干办公事差主管吏部架阁文字为武学谕轮对言人主听言不容有所惮尤不容有所玩惮则有言而不能容玩则虽容其言而不能用力请外添差通判严州主管崇道观召为武学博士太常寺丞兼诸王宫大小学教授大宗正丞兼屯田郎官将作少监知嘉兴府仍旧班兼右司郎官将作监轮对极言风宪之地未闻有十疏而攻一竦者封驳之司未闻有三舍人不肯草制者道揆不明法守滋乱天下之权将有所寄而倒持之患作当路者滋不悦出知泉州兼提举市舶改知宁国府蠲逋减赋无筭泛入者尽籍于公帑户部遣官督赋急若星火阖郡皇骇莫知为计梦观曰吾宁委官以去毋宁病民以留力匄祠且将以府印牒所遣官所遣官闻之夜过他日夣观去宁国人言之为之流涕丞相董槐召还帝问江东亷吏槐首以梦观对帝悦乃迁司农少卿兼资善堂赞读轮对谓今内外之臣恃陛下以各遂其私而陛下独一无可恃可为寒心次论郡国当为斯民计朝廷当为郡国计乞命大臣应自前主计之臣夺州县之利而归版曹者复归所属庶几郡国防一分之寛则斯民亦受一分之赐帝善其言迁太府卿宗正少卿兼给事中兼权舍人起居郎八上章辞免以监察御史呉燧论罢直龙图阁予祠授秘阁修撰江淮等路提防铸钱公事甫至官即复召为起居郎兼右侍郎给事中兼賛读兼国子祭酒权吏部侍郎奏事抗论益切以宠赂彰仁贤逝货财偏聚为言且谓未易相之前弊政固不少既易相之后弊政亦自若在廷之士皆危之夣观曰吾以一布衣防上恩至此虽捐躯无以报利钝非所计也力求补外以集英殿修撰知建宁府蠲租税省刑罚郡人徐清叟蔡抗以为有古循吏风民有夣从者甚都迎祠山神出视之则梦观也俄而夣观得疾口授遗表不忘规谏遂卒帝悼惜乆之赙银帛三百夣观退然若不胜衣然义所当为奋徃直前其居败屋数间布衣蔬食而重名节云
　　论曰陆持之学足以承其家而不幸蚤丧徐鹿卿论议眀达克施有政赵逢龙之清操汝腾之不挠孙夣观之平直洪天锡黄师雍徐元杰李伯玉皆悉心直言不避权势孙子秀政绩着见皆当时之杰出云

　　庚子轮对第一劄【聴言】
　　臣一介踈庸滥叨儒级误防恩命分谕右庠兹因轮对敢陈一得之愚惟陛下裁幸臣闻切直之谏人臣之所难言而人君之所难容也盖撄鳞之患从古所戒而受人臣之尽言者惟明主为能恭惟陛下不自神圣乐取诸人以为善自登大寳十有七年炳丹之奏不知其几什百矣臣不暇逺举姑以近事言之旱魃为虐下求言之诏真从谏如流之实德也改元更化下追悔之诏真改过不吝之盛心也士生斯世孰不欲披肝沥胆知无不言言无不尽眀四目达四聪虽尧舜之世不是过也自昔人主讳闻直言于是有谓所为尽善羣臣无得而谏者矣今奏防有曰箴戒之切不能止醉歌之讥溺爱之偏莫能掩道涂之议而陛下未尝罪其言之过于直者自昔大臣多防已短于是有专攻上身傅防大臣者矣今奏疏有曰陨获去就自忤于礼义亷耻之维曲意调娯莫决于隂阳内外之际而陛下未尝恚其言之几于讪也云南防师反以防报昔有以此而滨于危者矣今乃谓淮襄一百余州三分失一而任其自如川蜀五十四郡一掷俱亡而听其自若固未尝不及于境土之日以蹙也山东事起谓不足忧昔有以此而速于亡者矣今乃谓江隂之扰方息而毗陵之哄继作毗陵之哄未巳而青阳之众已聚固未尝不及于事变之日以繁也或有言诸夏虐流天降众异者矣或有言茶塩更法和籴科抑者矣或有言民不安居而至于防民不得食而濒于死者矣不宁惟是言国歩多艰危机交急者有之言事防已极政弊已穷者有之言何等气象见于京城畿辅者有之迹其徃言难徧以举忧治危明槩邻于激昔刘蕡一防当时考官犹以为嫌今陛下不惟赦之又优容之畧无讳忌此非汉唐以来诸君之所可企及者也然危证荐臻坏形层出有如羣臣之所奏几于祸至之无日岂其言果无补于事功欤大抵人君之听言固不容有所惮尤不容有所玩惮则虽有言而不容玩则虽容其言而不能用今日之失毋亦陛下容纳之量虽闳而施行之意实少警悟之机虽生而系吝之私难絶有乃训而未克迈有所闻而未能尊故虽日听盈廷之议而莫有执其咎者不然愤悱之心幡然而作将卧薪尝胆可也将朝不食夕不寐可也将无日不申儆国人于天命之靡常而民生之不易可也尚忍坐视天下之弊之至于此极欤臣虽赘叙夫亦何益伏惟陛下少赐清燕之闲繙阅奏篇择其切于今日之急务上闗君德下闗民命者刚决而力行之庶几天命之将絶者可续人心之将去者可追不惟足以昭吾君听言之实而又足以激臣子敢言之气书曰非知之艰行之惟艰惟陛下留神
　　第二劄【分别试】
　　臣恭惟国家南宫试士典以从橐临以防臣事至重也为士子者穷日夜之力辛酸于笔砚间不可数计也幸而得与计偕盖有数矣又幸而无疾又幸而无故则重趼来京谁不觊夫旅进旅退而角胜于鹄立之塲不幸而葭莩之亲仕于中朝或司文衡或司帘外拘于条令则例皆驱为别试之归夫为之宗族者固所不论甚者母党甚者妻属稍渉于服属之末亦皆不免别试且夫平居未必霑其余沥之润自服而劳自中而选及来省闱荣望遽隔声律之习稍众犹可觊幸其间盖有试一经而四三人者六七人者纵有出类之才局于额而不可得则私相告语以为何苦自诿一试于陆沉之域涕泗逆旅之中怨詈有司之室间有饮气而疾视遂指情亲为仇家者人之常情岂不爱其宗族爱其亲戚于是一遇差官或托他故或祈幸免忽被选差则彷徨而失所措不知朝廷果何惮而不为之处也绍定五年省试令漕举胄举人同避亲人就别所混试所取员数几四十人士子既无孤经之患有司又无避嫌之忧不过拨大院差官之数及取士之额耳于朝廷初无所损而上下悉防其利成规尚在可举而行此去省试不远欲乞圣慈下臣此章送礼部防详臣不胜拳拳
　　丙午轮对第一劄【结人心】
　　臣一介书生比叨睿渥分谕右庠获侍天顔曽殚末议兹又负丞宗寺摄事垦曹次当陛对敢陈一得之愚惟陛下裁察臣闻国之所恃以立者民也民不得其生则国之所存者幸也盗贼公行莫之或止此非昔人之所以挤于亡者欤海濒遐远不霑圣化犹或有之近而畿甸白昼剽金此何景也绣衣之使至烦临遣而犹未革其臂弓腰刄之习设或加之师旅因之饥馑则又将何以处此流离之子襁负相属今且累累然于江之南矣曩者朝廷颇以为虑或遣朝绅或戒郡县必欲究其安宅而后已谋国之人失于区处寻常之费未尝均节犹于此而屯施焉则遗至于今宜也今期年矣玩视习听似已相忘于度外陪都之邑近几激变则其他俟衅而发者直需时耳彼其退无所归进无所倚岂能枵腹而听其自毙天相国家则亦幸其仅为小盗而已且臣闻之淮甸之间兵不足以衞民哨骑突来靡间寒暑春夏之交驱掠我马牛系累我子弟哭声振天闻者惨如也国家平时髙爵重禄以奉邉将美衣丰食以赡邉兵一旦敌至纵之俘掠此固不容于诛矣然臣之所谓大可虑者则又不止是争城争地敌国之常今乃以吾女之闲于纺织者而使之蚕以吾丁壮之业于垦辟者而课之以耕以吾艺事之习于舟车铠甲者而役之工重捐金贝以易之广置田庐以安之生聚教训几欲以吾国之民反害吾国而吾民乃困于盗贼困于流离居则无以衞之生则无以养之是殆类于为渊敺鱼者夫有民斯有土民不之恤何土之能保吾方【阙】　彼方耕桑吾方困守彼方迭出吾之民日以穷约彼之民日以富庶吾之国日有锋镝之忧彼之国日有驱掠之利既墟我襄蜀矣又扰我淮甸矣又声言袭我广右矣我日以弱彼日以强士大夫方且豢于燕安鸩毒之娱略无封疆日蹙之虑徒幸夫祖宗之所以得天得人者尚可凭藉苟焉偷安以觊吾身之不及见不思厄运邉逄危机已露日星雷雪之变今嵗特异殆若与事变邉相符者火已及燃若何而不为之痛哭流涕也先民有言敌国之德黎民懐之三才其舍诸陈亮举以告孝宗皇帝以为今世儒者之所未讲正此谓也呜呼人民吾祖宗之人民也天下吾祖宗之天下也聚防离合之机间不容髪至可畏也吾君相其可不旰食乎然臣窃谓盗贼之变是绥懐之而已耳流民之变是安集之而已耳吾祖宗受命民无以诞保之而诱之来归乃陷于他邦异域之中想其引领南望之心天地鬼神之所痛也为民父母其忍付之无如之何而已哉有殒无贰不为不义屈者宁能有几万一含其畜养习其风土提戈反鄙将有抚我虐我之分事闗宗社未易轻言甚非祖宗在天之灵所望于陛下者也陛下而一念及此则凡上而君德下而民隐内而朝纲外而邉备其得以玩心处之乎臣不敢泛举以凟天听惟陛下毋以为迂愚而忽之则祖宗幸甚天下幸甚
　　贴黄【谨边报】
　　臣闻兵诡道也在彼者我可知在我者彼不可测则攻守可以制胜狡焉强敌扰我邉陲自间谍不精而倐去忽来皆不能以豫为之防吾谋亦已踈矣甚者帅阃之申眀廷绅之奏请所以为固圉之计者则夫人皆知之不幸而有漏师之人则彼得以窥吾动息谂吾坚瑕军志之所谓先声后实者反足以闯吾之机闗矣几事不宻恐非独害成而已也臣愚欲乞自今以后应有言及兵事者付之省府宻为区画特免报行庶几有合于宥宻之义惟陛下裁幸
