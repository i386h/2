　　提要 

　　臣等谨案运甓漫稿七巻明李昌祺撰昌祺名祯以字行庐陵人永乐甲申进士选庶吉士授礼部郎中（按明初六科皆有庶吉士不止翰林有之故得以直授郎中）厯官广西河南左布政使事迹具明史本传是编皆古近体诗并诗余乃天顺三年吉安教授郑纲所编史称昌祺预修永乐大典凡僻书疑事人多就质其诗清新华赡音节自然陈循序称其本之以理充之以气故雅淡清丽宏伟新竒无不该偹不必逺较于古就今而论千百之中不过数辈曹安谰言长语极推其题文丞相研一首朱彛尊静志居诗话亦谓李祯诗务谢朝华力启夕秀取材结体颇与段柯古相似葢由其一变绮靡纎巧之习而以流逸出之故别饶鲜润逈异庸芜郑瑗井观琐言乃曰李布政昌祺人多称其刚毅不挠尝观其运甓诗稿浮艶太逞不类庄人雅士所为所谓枨也欲焉得刚云云是梅花一赋足累宋璟之生平矣执文章正宗一编以进退古今之作者不亦隘乎惟其中驺虞歌汴城阅武诸篇或稍伤俚俗然论一篇之诗当字铢句两而求之论一家之诗则当统观其全局不以一二章定工拙也
