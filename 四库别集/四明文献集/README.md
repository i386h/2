　　提要
　　【臣】等谨按四明文献集五卷宋王应麟撰应麟有周易郑氏注已著录所着深宁集本一百巻然宋志已不著录焦竑国史经籍志亦不载其名则散佚久矣此本乃明鄞县郑真陈朝辅所辑四明文献之一种故一人之作冒总集之名也通一百七十余篇制诰居十之七盖据拾残賸已非其旧矣应麟以词科起家其玉海词学指南诸书賸馥残膏尚多所沾溉故所自作无不典雅温丽有承平馆阁之遗且所载事迹多足与史传相证如宋史应麟本传谓度宗即位应麟草百官表循旧制请聴政四表已上一夕入临宰臣谕防増撰三表考之是集则第一表至第二表乃景定五年十月上第三表至第七表乃十一月上与本传互异又宋史度宗本纪载贾似道罢都督予祠在徳祐元年二月徙居婺州又徙建宁俱在七月而由扬州责归绍兴实在是年五月四日见于是集责贾似道归里制亦足以补本纪之阙则虽零章断简固亦不以残缺弃矣乾隆四十一年十月恭校上

　　周山川图记【词科第一场二月十二日】
　　相古先民有周徳隆泽洽用造区夏章亥所步莫不砥砺建櫜戢武斧藻王度四岳河海咸秩无文名颂曰般裒时之对郑释允犹翕河之语谓按山川图次序祭之孔氏正义谓共为一图虽畿籍方志湮轶亡传谨旉绎旧闻而为记曰太极肇分有浮而清有治而奥巻石之积结为巍巍云蒸雨霈万物育焉勺水之积融为浩浩源迤流衍万物润焉圣人有作乃封乃濬乃刋乃涤肇称明祀为民祈褫若封禅于黄望秩于虞旅祭于禹维见可观然风后受图九则始布山海有经为篇十三图牒犹未详也苍姬开统宪章稽古以地图知山川之数职于司书以地图辨山川之名物职于大司徒以九州图知山川之阻职于司险天下之图职方逌掌其山镇曰会稽衡华曰沂岱岳及医无闾霍常其大川曰三江江汉曰荥雒淮泗曰河泲泾汭及漳虖池呕夷又有山师川师掌其名山虞川衡掌其守艮兼兑丽目击掌运抑犹以为畧也卑髙小大则其位殊次生屛摄则其数异我是以有秩祭之图河先海后辨源委也九河合一同等列也岳视三公渎视诸侯礼文别也天子四望诸侯三望名分严也按图定序因序蒇事于是宗伯为坛典瑞奉璋牧人毛牲校人饰驹乃奏防宾歌函钟舞大夏服毳冕而祀焉上下有章无越望之僭祝史荐信无矫举之愧山泽通气歆馨降嘏百嘉鬯楙四极艾寜盛哉铄乎遹观诗序祀于巡守时迈其邦爰及柴望意者土训夹车道图诏事其在斯时与攷之传记天下名山五千三百七十名川三百支川三千周国其在雍州而立华岳为西岳三代居河洛而四渎咸在山东公羊春秋谓天子有方望之事何休以为望祭凡三十六所若义之修而礼之蒇独于般诗见之夙夜毖祀作周恭先丕天之大律也下逮穆王大朝黄山披图视典殷武疏述郑之言亦参周汉之域騐山川之图孔安国为汉博士具见图籍山川所在学者尚有稽焉春官掌三辰之法犹神示之居注亦训犹为图仰观俯察两尽其挚矣虽然徳懋而山川寜道公而百川理懐柔河岳厥有本元无设官为极之意则经野之图不足以致治无敬徳祈天之忱则卜洛之图不足以永年是图之陈左右也顾岐阳丰水而念王业之孔艰瞻洞庭孟门而监地险之难恃江汉想广及之徳南山思疆理之功清明在躬与山川出云同一运化然后能灵承于旅惟典神天嶪峩蒙鸿洋洋如在玉帛云乎哉兹图外之眇指敢正列其义以诏万世谨记

　　汉百官朝会殿记
　　皇汉稽古建官立三公应台鼎司徒为丞相之任赞元穆縡位冠列辟中兴二叶宪度鸿明号登承平越十五载更作司徒府于开阳门内光观壮丽万国具瞻迪惟询谋佥同虞书垂训谋及卿士禹范陈法时有大政政有大疑必集羣臣议之咨二相决之乃于司徒府建朝会之殿防绎师言审国是焕乎一代闳规也栋宇修广史防无述独于周礼注汉官仪律厯百官志注概见颠末庸识其事弓禅史阙太微中居列宿环拱圣王正表着之位辨亷陛之等法虖天也苍姬制礼天子三朝燕朝列路寝之庭太仆掌之治朝居路门之外宰夫司士职之惟时外朝眡内为重雉门外者小司寇以询万民三公州长暨羣吏叙进臯门内者朝士掌建其法孤卿大夫暨众庶咸列葢建朝于外则尊卑之分肃咨谋于朝则上下之情孚公聴并观訏谟定命聚精会神之地其可忽诸汉监秦辙达聪垂明髙皇肇基下诏长乐相国建议寔始权舆或大议致中或防议温室朝堂百僚之位有萧曹魏邴之谋谟公卿以下朝防坐载于成纪建始元年之注坐而论道流风犹存丞相府有四门随时聴事坐未央廷中单于拜谒体貌匪不严也然问望之计防则遣中朝持单于之语则视中朝具复庙之文则言中朝臣议王嘉之事则下将军中朝者丞相以下为外朝顾不与焉宫府体分与同异矣黄霸问郡国计吏举孝弟为一辈先上殿説者谓丞相所坐外朝之制畧而未详东都重熈卬宪周典徳阳朝羣后象魏旌旧章上仪祲容焜燿千古睠詹公府轮奂未崇经之营之遹彰广殿彤庭殖殖左墄右平王侯更衣于殿之西位宁森严规橅琏敞尧衢室舜总章何以侈兹国有大议车驾亲幸其殿则应劭之书可证百官防府公殿下则蔡邕之言可稽续志通纪熹平四禩臣光臣晃传奏厯元乙夘诏书防司徒府议邕集所载在三月九日其位校尉东面侍中中郎将大夫千石六百石南面议郎博士北面戸曹令使西面中坐读诏臣邕前坐侍中西北近公卿与光问难朝章邦典贲然明着想夫三事在位百辟在庭周爰咨诹知畧辐凑帝联下瞩称制临决衡石公论蓍蔡羣疑匪同防同有绎而已故龙阙对峙榱题翚飞人不以为华铃鼔不设防闼洞闱人不以为简鸣玉峩冕翼翼翔翔吏治否臧民生休戚天聪四辟无隠不敶受计则计吏数百人拜于庭中举謡言则掾属令史防于殿上崇辅弼以尊国体意深逺哉若乃论于白虎朝于云台见于宣明问于崇徳隆寛尽下史不絶书兼聴广谋莫若斯殿康成释礼兼述汉法于朝士曰令司徒府有天子以下大防殿亦古外朝于藁人曰百官朝会殿天子与丞相旧决大事是外朝之存者刘昭谓干寳之説殆未深考小宗伯肆仪为位康成谓若今肄司徒府又非止为大会殿矣蔡邕纳书东观立朝会志未就详莫得闻窃尝考之周中朝聴治汉前殿也周内朝图嘉事汉宣室也东都前殿为路寝嘉徳为燕息见于张衡之赋而景秉传云公卿会议当上殿访邉事若袁安班勇皆于朝堂上书会议又有召百官议朝堂诏百官大防朝堂亦外朝遗意通典曰周有四朝而言三朝者询事非常朝也是殿其询事之朝欤聘礼君与卿图事康成曰谋事者必因朝其位君南面卿西面大夫北面士东面汉制葢如此云谨记
　　唐七学记
　　有唐观人文化天下髙祖太宗戢兵耆武尊右儒术缀礼正乐宪度着明庠声序音锵訇金石稽古建学厥制维六宗缵绪复増其一课试有法训迪有师徳艺兼该揔于胄监誉髦斯士云合雾集丰规茂矩冠冕百王宜祓厥文具详登载谨参合百官选举志儒学传序六典防要而记其畧夫教亦多术矣开数路而翕受皇极之化也萃羣材【阙】
　　上下夏氏设教序列东西左右郊宫
　　商学殊制作人造士匪一而足苍姬之盛教典尤备国学合子弟太学食老更四郊有庠门闱有学或读邦法或习书数达材成徳道化浑融祭义攸述为学有四保傅所纪为学有五学记之注曰内设师保外有庠序六学之制于兹可考治古逾逺彛宪无传汉立三雍晋兴两学亢嘉四学之剏天监五馆之开名实存亡驳虖亡议惟唐文治炳蔚上轧成周鉏类夷荒鸿业甫定生员之置肇于武徳元年五月周孔之祠建于二年六月恢儒乡学敷遗后圣爰暨正观骏惠前猷临幸国庠延吁儒彦肄业之舍广为千二百区着録之生益为三千二百六十员施于六叶増辟学馆袭旧为七若三品以上子孙国子学教之五品以上子孙太学教之曰广文以领国子生之业进士者寔维天寳九载七月乙亥之制曰四门以授七品之子及庶人之俊异者寔维元魏太和二十年之制律学昉于晋梁复于正观六年之二月又复于龙朔二年之五月越明年以详刑而律令格式法例于是习焉书学昉于晋隋复于正观二年之十二月又复于龙朔二年之五月越明年以兰台而石经説文字林于是习焉算学沿隋之旧置于正观二年复于显庆元年越三年九月废之以太史及龙朔二年复之以秘书而九章五曹缀述缉古之属于是习焉训导之职有祭酒司业博士助教直讲孙其业也举选之科有秀才明经俊士进士明法明字明算兴其能也业成则丞莅试不率则簿举法严其训也国子有五经博士广文有知进士助教重其选也助教或十年使迁博士或十九年不易颛其任也生徒之数国子二百人太学五百人四门千三百人书律及算总百十人元和六禩申命更定西京自八十至十人东都自十人至二人而国子学有大成上元二年初置二十人开元二十年复省其半若舍奠齿胄之仪皆行于国子学又本末先后之序也想夫纡袂鼓笥来游来歌鸿生硕师论难衎衎拱绂懐玺濡化染学武飞骑垂衿讲习新罗百济挟防吟诵丰芑燕翼之泽棫朴追琢之章政媺俗淳作周匹休猗欤懿哉铺观唐史立学阐教葢非一端左省文武徳之制东宫崇文政观之规并置生员是为二馆秘书小学以诲皇族内之学馆以训宫掖崇有学以习老庄郡县诸生差列二等若崇化厉贤之原则自七学始以至龙朔改司成馆垂拱更成均监司成宣业官名屡易兹不备载按郑防传广文寓治国子馆李庾西都赋曰左立太学前到广文选举志止言六学何蕃传亦云六馆意者二学合为一与韩愈上表有七馆之称并书以识旧典谨记
