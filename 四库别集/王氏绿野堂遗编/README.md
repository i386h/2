　　●叙王氏绿野堂遗编 

　　不佞舒自垂龆则闻乡长老啧啧王司空公也、已同年友王星父手公遗诗及慈训卷、属序于不佞、曰不腆先人之业、若存若亡、什袭藏之、不啻球璧焉、兹谋所以不朽者、惟吾子图之、舒受而卒业、辄窃叹王氏世德所从来矣、盖王自星父而上、衣冠蝉联者八叶、而司空公最显云、公父为金溪令、母为张夫人、其大父则明白先生、高皇帝以经术征召对称旨、呼为老学士、识者谓先生克继虞贺杜楼之绝学者、公少孤、张夫人故隐君子女也、雅通经史、工诗文、建慈训堂以课诸子、大归乃约之忠孝、廪廪若严师、及公贵而夫人行益着、一时名硕、若杨文定于肃愍诸公、并为声欹纪之、方之陶益、谓夫人善教、公善述也、予惟子舆氏圣人之徒、即弗论经德、頋终以枘凿于时、不获稍见其用、士行虽江左元勋乎、然鞅鞅非纯臣节、至梦思折翼、古今以为口实、而司空公起家广文、为名御史、为名卿、扬历中外者数十年、所至辄贻百世之烈、盖神明钟鼎、先后罕俪焉、今其诗具在、搃之沉雄脱活、即不必屑屑摹画、而适然鼓鬯、便足千古、如登天门关、岳阳楼平蛮班师诸咏、浑涵汪茫、千汇万状、昔人称六经鼓吹、庶几近之、笃哉孝子忠臣之思、于慈训不昭融炜烨哉、嗟乎、今海内荐绅家、为谭公鸿猷骏业、靡不慷慨发□□立、思执鞭而无□□□□□从、至其摛藻抒精、光焰万丈、独使湮没而弗传、则绳武者责焉、予故重王氏之世德、而幸星父之有斯举也、乃为序其意而传之、

　　◆抑斋公仕谱略◆ 

　　○进阶一品荣禄大夫工部尚书都察院右都御史兼大理寺卿王公仕谱畧 

　　公讳来字原之别号抑斋其先汴人也始祖万五府君从宋天子南渡卜居于慈之唐家堰桥历四世而生公大父桓高皇帝征至京师授河南□□雷皿□氏令有道德文章号明白先生桓生五子咸登仕籍公父伯熏居四为金溪令金溪配张氏舜江张天民先生女也金溪生四子公居长为工部尚书次复刑部主事次鼎按察司佥事季则谦也公生于洪武二十八年乙亥之九月六日至永乐十五年丁酉丁外艰自金溪扶柩归葬于九都之风湾时张氏痛抚藐诸孤乃建慈训堂亲教之公与昆季日夜痛父之亡遵母之训惟圣贤之学是淬励至宣德元年丙午公与弟复俱入邑庠补增广生遂赴浙江乡试中第三有经义表刊程序弟复亦中第十九名丁未中乙榜授江西新建学谕作新士气多所造就宣德辛亥以荐起行取拜行在江西道御史壬子奉命按直隶苏松常镇诸郡有遏大珰陈贵禁粮长拷掠细民械贪暴官吏送京师及民有不葬者发其葬婚有失时者毕其婚等事癸丑复命留之内台甲寅公在内台参阅各道刑狱时宣宗出居庸廵边简公扈从陈守边七策乙卯方面员缺有诏举才力堪任者出补之少师杨公士奇举公特拜山西参政不数月尽厘宿弊郡邑肃清有再来包侍制之谣正统元年丙辰公在山西赠其父金溪尹为御史封母张氏为太孺人配钱氏为孺人是年有清宣大二镇粮草布帛等项事至今小民无忧赔偿运头不得包侵皆公力也丁巳公在山西又奉敕督理大同宣府边备戊午提督山西乡试巳未以廵按御史奏特诏公管司事兼理边储军政庚申奉诏偕廵按御史审录山西所属重囚多所平反辛酉奉诏督理预备之政公慨然兴举为仓一百十座为廒二千余间总得粮二百四十余万山西之民无虞荒焉壬戌山西廵抚侍郎于谦奏公廉明刚正练达政体宜升近侍重职以副公论乃命未下而公以前预备事忤当道被劾钦调广东参政癸亥公在广东即按所属稽考仓粮得六百余万石广东至今称便甲子提调乡试乙丑考绩天官以公雄伟之才刚果之操参政大藩克举所职闻于上蒙圣谕慰奖复职戊辰升河南左布政时流民与土民争执田土累岁不决公至勘其户数悉令附籍使流民有定业与土民参编为里长遂以无患先是开封府祥符县黄河水泛冲激民田舍公筑堤四十里以捍之堤傍重植柳木以固其根自是水莫能为害巳巳八月朝廷罹土木之变九月景泰即位以公廉明精敏人望攸归特升都察院左副都御史廵抚河南湖广提督有司赈恤流民操练军士防守寇盗凡官吏臧否军民利病悉许便宜行之公即习武艺籍丁壮设机兵置甲冑刀抢之具十月胡寇越关逼近宫阙公亲率官军驰赴京师胡虏宵遁十二月上遣行人王鉴赍勑谕以公祗事国家清勤公正特赐白金二十两织金文绮二表里景泰元年庚午钦升都察院左都御史时伪苗王韦同烈作乱上特命公自河南往总督军务总兵参将以下悉听节制公受命驰至湖广会总兵官保定伯梁瑶等责以逗遛示之军令分官军九万为五哨期悉行抚捕二年辛未前进贵州至重安江苗王聚众八万据江犯敌公调副总兵毛福寿等大破之苗王率其余党奔香炉山寨其山璧立千仞根盘二十余里公审其乡导密授总兵梁瑶等指画径抵香炉山下围筑排栅榙驾飞楼以飞抢火箭昼夜攻击裂其寨石贼大惧告哀乞生公仍命右参将李襄诱酋长罗义等缚送伪苗王韦同烈械送京师苗贼悉平于是罗义等誓永不敢叛且同緫兵官保定伯梁瑶等请撰刻为文题曰平蛮恩信碑立于贵州之清平通衢云捷报圣上特遣主事孙茂赍勑旌奖赏白金五十两织金文绮五表里三年壬申公在贵州剿捕苗贼散党时土官宣慰使安龙富与其族属白眉毛者争袭官职分占地方互相雠杀公即选委布政使参议应彬宣慰使宋昂平坝卫指挥金桂谕以朝廷威德安龙富等畏威慕义多以牛马献谢九月公乃奏请班师以公兼大理寺卿四年癸酉八月十七日公班师上召公与保定伯梁瑶至便殿慰谕再三并赉羊酒赐宴光禄次日谢恩升南京工部尚书仍兼大理寺卿公条陈十二事上皆嘉纳之陛辞给驿舟抵任先是部中修造等项多无统纪公至疏剔洗涤分目列悉奏而厘正之遇关支则置上下号簿比对务相同程工役则置循环文簿每月至堂督责至十月复奉勑修理太庙社稷该用工料动以万计樽规模各有方法人不告劳而事功集矣十一月诰封祖父母五年甲戌公在南京工部督修大庙工役六年乙亥公在京工部以大庙工完奏闻特降勑书兼赐手酒赏赉其兴毕工日皆勑公祭告后土之神七年丙子先是正阳朝阳神筴金川等门损坏命公条督分理不疾不徐而事功顿竣保障森严都会增胜公之力也十月英宗皇帝即天位天顺元年丁丑公在南京工部闻太夫人张氏丧即赴北京陛辞诏许之寻蒙赐以礼致仕至宪宗即位推恩旧臣凡在先朝以礼致仕者进阶一品公与焉成化六年庚寅四月公弗豫伏枕数日卒于家上闻而震悼之遣官谕祭仍敕有司营葬九月九日夫人钱氏寻卒七年辛卯十月己亥二十二日庚申仲子钥监察御史季子金□英奉公柩与夫人钱氏合葬于县治西耀奕山之原

　　○序 

　　予忝朝士后、三十有余祀、得瞻君子之仪尚矣、仪不可伪为也、非真有所存而不悖于外、其孰能之、监察御史慈溪王公原之、巡行蘓松诸郡、所至肃然、行恶者拔去螫毒、心险者坏其机牙、色温言厉、有大臣体、期月之间、威行化洽、及代还、始以母夫人慈训堂卷、征文于予、且曰、夫人为余姚张天民先生女、嫔于先君金溪令、金溪卒峕、以诸孤嘱夫人、夫人受命悲戚、恐不副先君意、集诸孤于膝下、泣而誓曰、若知先世立门户、与若父之为耶、非勤学无以明理、非谨行无以成德、一理未明、则行或愆违、一事不谨、则德有玷缺、欲其道明德备、必在于缉熙乎、乃揭日新二字于书楼、俾警其目、它日又曰、里中聦慧儿、不籍门弟、苟志于学、皆能自奋、若有父祖旧业、德泽未冺、可舍而弗由乎、又书肯构二字于息游之所、以惕其志时予稍长、诸弟犹稺、虽知慈训之切、不知其心之悲也、退相勉于切磋硺磨之间、期无负于训辞、赖母氏之精诚、格于祖考、俾我兄弟、大启其蔽、相为丽泽、来忝为御史、复中进士、弟鼎谦业皆就绪、皆慈训之力、祈一言以序之、余谓母之于子也、孰不知爱、爱而知教者鲜、而淂其正者尤鲜、夫人之教子也、由勤学以明理、谨行以成德、宜御史公之学、有体有用哉、公尝一至敞邑、廪人莫敢市鱼肉、唯以豆乳和菜作羹进、吏民戢戢、市不敢哗、一时之人、肝胆肺肠、皆得涤去热恼、道路啧啧唴称、争识其面、余甞自谓阅人多、亦为之举手嘉叹、非谓其能食淡也、以其操履中矩度、风采可畏爱、使醉浓饱鲜者、知以为耻、饮氷食蘖者、得以舒气耳、人皆以饮食为细故、殊不知有大欲存焉、于此不能谨、而能谨其余者鲜矣、故并及之以见其眞有所存而不悖于外、着慈训之足征也、行在翰林院致事修撰承务郎同修国史东吴张洪序
