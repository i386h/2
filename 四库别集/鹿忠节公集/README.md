　　●鹿忠节公集序 

　　余自癸丑通籍与定兴鹿伯顺吴门周景文同试政司马署中志意叶洽彼此以名节相砥期为岁寒交嗣余与景文先后出理齐闽而伯顺为计部郎以争留金花饷边左迁去予庚申入吏部引景文为同舍而就家以原官起伯顺辛酉予休沐里居移书长安荐伯顺可入吏部议既定时高阳相公行边伯顺自愿辞铨相随军中五年量移尚玺己巳转奉常浸浸大用矣心愱柄人竟移疾归海内颙颙方以公出处卜世道隆替丙子之变竟殉城以死呜呼伯顺生平大概若此矣其人固无需文以传而予又论定其文梓之序之岂其人之必待文以传哉则固以为伯顺之文也已爱其人者即其人之词组只字宝若天球琳琅况伯顺之文实足以传者乎方今鸡坛错峙鞶帨徒工即使振起作者之林步武先民之轨以是而称词人之雄止耳于世道人心固未有赖也今伯顺文具在用意忠厚摛词恳恻与君父言仁慈臣子言敬孝兄弟言友恭朋友言贞信上自军国朝庙之容下及闺闼帷房之故言必究情语无留旨笔光墨渖剥露血腥剩楮残笺刓剔肠肾使读之者欢爱鼓舞悲愤流连忾然以思卒然相感而中正之心油油乎以生繇是而究伯顺立言之意不模古不规今发乎情止乎义理百世之下闻者犹将兴起焉然此伯顺之文亦即伯顺之人也垂白孺慕四世一庖家无嘻嗃人风静穆其植本良厚矣故帑可以无争也以续三军之命而毅然争铨可以无辞也急君父之难而洒然辞官可以无去也愤柄人之愱而飘然去最后殉城全归计不返顾使素庇一廛咄嗟遇难或牵于难割之家累徼夫幸全之金汤又或怵功令束文法与存与亡计无复之乃死耳伯顺有一于是哉然则伯顺固可以无死也夫可以无然者圣人不以之律人而贤者以之自见至性所存不可强抑善学圣人者孰如伯顺乎躯命不惜浩气常存扶进醇风廉立顽懦伯顺果死也哉因是而覆绎集中忠孝之语伯顺自为写生留照人传文乎文传人乎则是集之有功于天下后世匪浅尠矣宁屑与缀文之士计工拙耶伯顺自题曰认眞眞故人与文两传矣吴桥范景文书于南枢之四友斋 

　　往余落拓长安酒人中时天下新有大故士大夫以及五侯七贵竞向余问兵时杨都尉之子允谐亦时相过从一日诣其家遇一老先生布袍苍髭如穷措大举动朴直如田闲父迫视之精光炯炯非韦布中人也徐讯之知为伯顺后再遇之张太常榻前见其谈边事侃侃无所避遂与心盟又二年奉命参高阳公军事入幕见同舍生则伯顺也自此则天幕地席霜袍雨骑无不与共如此两载余而别伯顺坚卧田园余周遭刀锯又两载余以环召至其家未几受上殊遇旋触谗而罢复为势人罗钳舍其家岁且半伯顺入为尚玺奉常而余以边衅仓皇入辽西仍留家人于其家与之共避兵火出入生死复半载而余罢伯顺亦乞身归子舍及余再被锒铛托身请室半载而事竟伯顺醉劳村舍余轩轩如故而伯顺黯然不胜矣再别六年余以勤王罢归过其家而伯顺已以殉节死狼藉血肉浅埋阛阓闲余哭不能起知其家经兵燹不能为殓乃脱骖赠之归而谋所以永伯顺者伯顺事功节义昭然耳目自足不朽唯有传其言而已先两年伯顺已自刻其言曰认眞草吾师高阳公为序之乃简为二十二卷吾师吴桥公伯顺之执友也慨然为寿传焉余于是喟然而叹曰言之不可已也如是夫朴直如伯顺使无其言则清忠节烈与日月不磨者苟非作者极意摹写不足以传其神今读其言不必知其居身立朝赞军殉死未有不如见其须眉颐颊者也文人之言饰辩士之言伪见之而靡然自失其所挟此有诸衷形诸外精光炯炯如见其人可以知其概不待智者而后察也此高阳吴桥与余欲永其言之意也防风友弟茅元仪篹

　　○微臣待罪逾时疏 

　　为微臣待罪逾时旧疾增剧恳乞圣恩俯允回籍调理事切照臣去年七月闲署广东司事为辽饷将绝借发金花银两权宜即属苦心专擅自知负諐幸蒙圣恩宽其斧钺之诛仅从薄谪吏部钦奉圣谕拟山东都转运盐使司判官添注以请曲贷敢忘洪恩内外皆可自効如得赘衔于盐筴尚期补过于桑榆臣之分亦臣之心也乃待命历时旧疾复作药饵增而饮食减痰火盛而肌骨销困顿旅邸见者相惊臣亦自厌然伏枕展转而不敢辄言者以大造既无弃人微躯或有起色违颜咫尺忍死须臾耳而今岁亦倏更病且益深势难再待矣恋阙之诚虽笃报主之念成虚臣所为踌躇再三而不能已于控吁也臣初拟求堂官代题缘已离旧任乃敢冒昧自陈君父之前伏乞天恩敕下该部允臣回籍调理倘补黥治刖得复视息于人闲而寻壑经邱永愿优容于圣世其为感激没齿难忘臣无任激切待命之至 

　　○微臣病极情廹疏 

　　为微臣病极情廹再申前请恳乞天恩允放回籍事切照臣去年为借发金花银两蒙恩降调候谴逾年病苦交作已于前月具疏恳恩回籍调理拜疏之后日以几曰圣明宥臣罪而未忘使过或者鉴臣病而早为允放乎乃时逾浃旬未蒙简发正静听闲忽接家报臣父闻臣之病亟望臣归泪洒眼穿日以为岁臣切自念留帑金而未果既不得遂其忠萦亲怀而莫慰又不得成其孝因是展转忧思旧患增剧心益苦待益难矣万不得已力疾出城沿途候旨冒罪而行臣岂不知守官之义第旧任已离新命未下无官可守而情亟势迫想亦圣明之所心矜也恳祈天恩俯允回籍倘霍然可冀遂依恋子舍之情而此外何求酬咏歌太平之愿即以径行罪臣所不辞矣臣不胜激切待命之至 

　　○请发帑疏 

　　为辽饷督催愈急新库空虚转甚恳乞圣明速发帑金以济燃眉事切照臣起罪废之余过蒙圣恩使管新库则饷之出入惟臣司之而其有无惟臣知之辽东宁前天津以及蓟密昌霸或以战士之月粮索或以援兵之行粮索或以召买之价银索诸臣情见乎文移委官口穷于叫号而莫不刻画时势难待那借已穷之状而库无以应前件未足后款复开领者尚留催者踵至而库又无以应夫兵急则饷不得不急也月粮行粮折色本色事势相逼同条共贯总为辽事也苟一节之不应将全体之俱伤而索之急如此库之空如此数月以来内帑颁及辽左者非不数数第所颁者赏也非饷也赏无定额特一时暂沛之恩饷有成规实按月常给之数得赏于分外虽不胜喜缺饷于额中殊不胜怨究之赏暂而饷常故怨多而喜少至以百余万之沛发而不能解一时之急者政坐此犒赏之银不能为月粮不能为召买故也委官守候有一月者有两月者有三五月者臣每一出环臣而泣偶获外解聚首而争酌量分给十不得一又复嗸嗸拥诉矣即加派搜括郡县多急公之吏而梯山航海解纳非画一之期彼饥军饿马岂肯有待而不脱巾米价豆值岂肯有待而不腾踊当厄不与事后何追臣纵欲对蹙额之委官强作无情之面孔而能不扼腕于征夫寒心于国事哉臣惧极而思无术以处反复窘迫而恍然悟曰臣得之矣盖此事于臣为职掌于皇上为家事臣之职掌臣能知之而皇上之家事非皇上谁办之况皇上天纵英明深为边饷吃紧李进忠田诏赃银不以入内而以济边也罪人房价不以入内而以济边也迩复悯念征东将士寒苦谕部行赏言言生东海之春字字陨征人之泪皆发自圣心非廷臣之入告也使知索饷之急新库之空岂难涣王居以酬死士乎此事此情臣非管库臣亦不知皇上何从知之今臣知之而不以入告是壅征东将士之苦而不使上闻复郁皇上事事为辽之德意而不使下暨也坐待军心之变立贻庙社之忧臣之肉犹足食乎今臣入告而皇上既知之必有所以处之其则不远有先帝之家法在盖大内金银之气久蚀日月之光先帝不难破一恡以发之天宇重开倒悬立解无穷之志事端赖于皇上之继述及今而出内府之所有余补外解之所不逮使索饷者随叩随与则得饷者士饱马腾此为上着不闻贾人之术乎贾人流通千金以权出入而必于每岁常数之外别操赢余以备不时之需不使有中断之隙从来财用有本则相生易脱节则再续难理固如是今日辽饷何等关系而可听其脱节不为之所也外解之来各有程期即欲强先一日而不能辽饷之索急如星火即欲姑待明日而不可则所以别操赢余为外解之接续而使新库得环中以应无穷非帑金百万曷克有济且新饷原附太仓旧库旧库虽号空虚然外解款目尚多如遇辽饷告急犹可通融那借今凑分新饷另为一库譬如穷人素倚邻里为命一旦孑然独处卒有缓急凭谁奔诉而又谁为匍匐救之者乎臣今日之请非但司辽饷言辽饷实在新库言新库计穷力诎不得不出于此恳乞圣明俯鉴新库转空立发百万以济辽急庶不失特设此库之初旨而罪废小臣亦得竭顶踵以报圣恩于万一辽左幸甚宗社幸甚
