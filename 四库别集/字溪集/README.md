　　字溪集十一巻　　　　别集类三【宋】
　　【臣】等谨案字溪集十一巻附録一巻宋阳枋撰枋字宗骥初名昌朝巴川人居字溪小龙潭之上因以自号端平元年冠乡选淳祐四年以蜀难免入对赐同进士出身阃帅交辟于昌州监酒税于大宁为理椽于绍庆为学官晩以子炎夘贵加朝奉大夫致仕年八十一卒其行履不见于史传惟文渊阁书目载有阳字溪集之名而不着巻数黄虞稷千顷堂书目则称其集为十二巻久无传本今检勘永乐大典所载裒而集之附以其子所作年谱行状仍析为十二巻适符原目之数虽已经断裂未必无所残阙然所佚似亦无多矣枋尝从朱子门人度正防渊游故集中与人往复书简大都讲学之语所言皆明白笃实不涉元虚其易象图説一篇多参以卦气纳甲之法乃不尽与朱子本义合案李性传朱子语録序称诸书答问之际多所异同而易为甚防渊所録一编与本义异者十之三四枋殆述防渊之所授故持论不同欤又有与税与权论启防小传一篇乃暮年所作尤见其孶孶力学至老不衰于紫阳学之中犹不离其宗者云乾隆四十六年九月恭校上

　　字溪集巻一　　　　　　　宋　阳枋　撰表
　　冬至表【代广安军】
　　开于子生于子机潜起于黄钟至于南盛于南运黙扶于紫盖仰一人之有庆宜万邦之作孚中谢恭惟皇帝陛下神化密移天行不息莫能消长听阴终阳始之屈伸弗囿盈虚任月窟天根之来往徳惟纯而不已福茂介以攸宜臣承乏賨城阻趋魏阙氷藏陆北欣逢七日之来春到殿西喜上三呼之祝俯躬承宠拜手扬休
　　谢赐宪书表
　　璇玑齐政仰钦天度之稽玉简颁春俯敬人时之授五纪之章程一正万年之正朔相承中谢恭惟皇帝陛下数显在躬政犹审已念千歳之日可坐致惟省嵗功使百物之产由曲成不遗元化罔愆析因夷隩之序允好财成左右之功臣服采罔庸赐书承宠庆隆熙旦覩斗柄以夀北辰长愿丰年及春事而行南亩
　　正旦表
　　天地交为泰卦成三画之干春秋首书王礼大一王之统叶气庆洪钧之转懽心宜万国之同凡在致身喜均拜手中贺恭惟皇帝陛下首出庶物仁包五常长善长人朝诸侯而图天下新民新物布政事以训四方发生回禹甸之春出治用夏时之正百禄是总万夀无期臣摄守古賨阻趋北阙越在侯服莫称栢叶之觞仰企王朝知献椒花之颂心倾丽日靣逐祥风
　　天基节表
　　飞龙在天周二十八年之星纪流虹明渚绵万有千歳之皇图治盛三登懽腾八表中贺恭惟皇帝陛下乾元用九太极含三思极无邪有诸中者惟精惟一动毋不敬形之外而乃圣乃神长发其祥大徳必夀臣黔巑假守丹陛驰神五叶蓂开虽阻枫宸之拜万重山逺终倾葵日之心
　　又
　　诞弥端月五开尧荚之辰生育后天千卜荘椿之歳御干大造出震昌期中贺恭惟皇帝陛下如日之升乘时之泰赐租寛赋春万象以同仁训亷谨刑雨八荒而被泽矧遇流虹之旦尤深就日之诚干羽两阶衣冠万国臣賨城假守魏阙驰心莫陈千秋鉴録之忠惟致万嵗嵩呼之敬福禄常矣夀命长矣归美益深元首起哉股肱喜哉载歌无已
　　代王敬伯使君谢到任表
　　假熊轼于淮堧畴庸罔效殿龙藩于黔土得宠若惊方辞白水以投闲遽上青天而峻陟具以寛大孚心之防告之穷荒鴂舌之民中谢伏念臣老矣半生猗无他技琐材膴仕江湖浪走于间闗弱歩丰扛唐邓忽膺于赑屭慙非壮烈猥蹈危机溪边钓鱼已织婺川之蓑笠塞上得马遄叨潜邸之丝纶惟绍庆之名藩本酉阳之一县地僻而闗系者重人稀而控扼滋多内接巴庸外环溪洞为秦川之后户实湘楚之隣封冦盗若豺狼困则藏而强则搏吏民如禽鸟弛则玩而急则逃自非文武兼资之材莫称恩威并济之任竦膺涣渥倍切震躬兹盖伏遇皇帝陛下髙拱一堂明见万里去缯弋以安鸿鴈急保障而缓茧丝在知人在安民尘欲清于塞北不泄迩不忘逺俗先抚于黔南臣未即防烘誓矢精白庶矣又富矣言恪守于泗洙安之非胜之效觊收于渤海
　　书
　　上魏鹤山了翁论时政书【礼部尚书】
　　尝谓今日天开斯道兴起之大幸防而亦伏斯道消靡之一大危机有志当世者不可不深长思也一圣天子躬亲万几愤积年佥壬之蠧国一切埽去尽收召名徳夙望儒术老成之士布列朝宁侧席倾心希冀太平此斯道兴起寖昌之一大幸防也然一二年间弊日益甚治日益逺上之心茍厌且疑则小人乘间羣起而拔之岂非斯道消靡薄蚀之一大危机哉吁可畏也常思今日之事不患天下之积弊靡而深而患士大夫之积习専而忌専则任已意而常失于私忌则忽人言而易违乎理所以君臣上下竭思尽谋而弊终如故此无他未能公天下之心也夫天下之见闻弊端之所灼知天下之议论弊事之所审订今士大夫分位隆崇体貎修贵例曰譊譊者肤言适足以乱吾听吾自有卓出众人之谋察察者浅见实足以汩吾聪吾自有洞烛事几之哲进一言唯恐人知行一事唯恐人觉深潜秘密务以相尚必使天下知某防某事専出于某人之功而后已逮夫行之而弗宜施之而弗遂则支离逊避已无与焉是以天下虽有切中事理之言脗合时之计往往愧吾谋之出其下惮其功之成而吾之禄位一旦为其所超越于是寝沮隠遏莫之举行専与忌交痼于胷中而天下之弊日深一日而不可革矣果何贵乎谋人之国家哉明公以大公至正为学道徳性命有以至其极出处行藏无以动其心涵养渊深更厯绵久可谓一而不二天而不人者矣慨念苍生翻然而起非特天子忻然以救之责望之百官亦以是望之天下亦以是望之先生是行诚十手十目之所指视以为国家理乱隆替之决非茍然者然今日之事岂一智能一手足之所可为乎支倾屋者须足用之众材起危证者须足用之众药革弊政者须足用之众贤厥今柱阽榱欹屋倾甚矣阳飞隂沈证危甚矣内阻外讧政弊甚矣而欲以一木支万瓦之頽一药瘳百疾之顿一贤理庶政之弊愚恐般郢失其能岐和失其良伊傅周召之贤亦莫以有为于斯世也髙宗中兴虽得一傅説而旁求俊乂必合天下之材能此所以为傅説成王守成虽得一周公而常徳常人必合天下之智力此所以为周公圣贤之心何一毫専与忌之有哉天下之士皆以傅説周公之事业望明公而明公用心自视与傅説周公何所间特在乎推达大公无我之心而力行之尔圣贤相逢一遇千载岂止李渤星鳯之快覩而已哉愿毋以思不出位而自逊焉天下幸甚
　　上洪中书论时政书【咨防平斋先生】
　　学必贯本末治必该体用二者可全而不可偏正心诚意修身者学之本齐家治国平天下其末也格非辅徳纳诲者治之体修制度明纪纲立条目其用也古之君子未用而修其学自诚意以及平天下之道本末贯通而不遗既用而推其学自正君以及于制度品节之具体用兼该而不废故朝有正心诚意之君臣则天下有正心诚意之善治言之而必行为之而必成如桴鼓影响之捷无他明体兼用揣本齐末不失于一偏焉故也禹臯稷契虽曰君臣都俞吁咈于一堂之上而原其当时平土敷教明刑播穑之事必灿乎有条截乎有制以致雍熙之治者特典谟之辞简略而后世莫能有考焉耳继泰和之盛者莫如成周周公之相成王不独止于无逸告诫之辞而自朝廷以及于乡遂都鄙邦国典章法制所以为天下立心者小大纎悉灿然毕备是以成王垂拱仰成而天下日底于刑措无非以道虽讲之朝廷为甚明治必达之天下而后行也厥今在廷之臣正心有言诚意有言修身有言大学中庸之外洗心以易者又有言朝夕纳诲是宜人君意无不诚而心无不正矣然而求之天下则吏不免于贪鄙士不免于浮薄民不免于困匮军实不免于隳废循至楮币日轻货泉日竭中外岌岌末如之何而推原其故则制度纪纲积习废弛而不饬之故也今日用吏之法不过曰善催科富文学明法令下而履常蹈故无他善状斯谓贤矣亦有急教化而缓簿书奨烦劳而黜退避赏振作而惩委靡之法以砥砺之乎今日取士之法不过曰博记问赡文辞工音律下而趋时媚世迎合有司之意斯谓贤矣亦有教之孝悌忠信劝之徳行道艺课之礼义亷耻之法以陶成之乎民俗之靡敝极矣穷奢于寺观之金碧而莫为之节极靡于斋醮之虚浮而莫为之节僭宫室之竒侈肆口体之珍丽歌舞无厌嬉游罔度沈湎淫泆此政散民流之风不宜见于盛朝也曷不定公卿大夫士庶之等立起居饮食服用之制而一正之乎光武中兴吴冯寇贾之外赳赳者甚众髙宗中兴韩吴刘岳之余洸洸者不少厥今军旅之事朝鳯晨星前无以继其后右无以辅其左中无以援其旁谋单力孤强敌四起今不早为贰矛重弓之图万一敌人分道并进奔逆冲突不知一时谁与御之也夫吏以教化亷勤为先也朝廷徒曰戢贪而奨励之法制弗立焉士以徳行道艺为贵也朝廷徒曰崇雅而教养之法制弗立焉民风以醇厚俭朴为美也朝廷徒曰美俗而品节之法制弗立焉军政以广储将帅为急也朝廷徒曰练军而选防之法制弗立焉宜乎吏日戢而日贪士日养而日薄民日恤而日困军日练而日弱正心诚意之学日讲而其效莫或见于天下也故吏亡善治终无以裕财士乏吉人终无以生财民亡盖藏终无以丰财将亡成绩终无以省财楮由此而日轻镪由此而日匮君臣上下穷思竭谋而终莫能以究其弊盖徒知药疢疾之顔色而不知于腹心根本焉治之耳夫乱生乎小人治生乎君子是理也三尺童子皆知之曩者大往小来天下之不治无足怪者今也端人正士云合辐辏相与聚精防神兴利除害既逾年矣天下举手加额引领东望冀太平之在旦夕也而害日以生弊日以甚天下善类皆窃叹而疑之其间不肖者往往指以为经生学士真无益于人国而小人之徒遂得以幸君子之罔效而肆其诋议之口其机甚可畏也呜呼干蛊弊之极者才无能名理坏证之深者医无竒称君子不幸承蛊壊之頽波治功成否实正道隆替之机不可不反求元气衰微之本根痛加培养使千载而上一洗贡薛韦匡之羞千载而下无重増责备贤者之深叹也或曰执古者悖今易法者矫俗殆于不可嗟夫道有升降政由俗革通其变而民不倦尧舜之盛也天下惟法弊则治弊弊而不革何以得治商鞅变不善之法犹足以强无道之秦况返薄俗以归醇厚之为善变乎今日之法大变则大治小变则小治亦在乎谋国者运量器识如何尔某晩后疎拙嗜学耽书万里来从性善先生而毕四书之业阁下负经纶天下之道得与圣天子上下其议论欣然踊跃条救弊之説以献非以希阁下之切于知己茍合求进而钓誉要名于当世也阁下徳足以厚俗学足以用世智足以烛理才足以用事量足以容物扶危持颠之任舎阁下无望焉言陈而不竒论迂而不切聪听之余少有补焉幸甚
