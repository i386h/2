　　钦定四库全书　　　　集部三
　　景迂生集　　　　　　别集类二【宋】
　　提要
　　臣等谨案景迂生集二十巻宋晁説之撰説之字以道开封人少慕司马光为人故自号景迂元丰五年进士元符中以上书入邪等靖康初召为著作郎试中书舍人兼太子詹事建炎初擢徽猷阁待制髙宗恶其作书非孟子令致仕説之博极羣书尤长于经术年未三十苏轼即以著述科荐之所著书数十种靖康中遭兵燬不存其孙子健访辑遗亡复编为十二巻又续广为二十巻前三巻为奏议四巻至九巻皆诗十巻为昜元星纪谱十一巻易规十一巻尧典中气中星洪范小传各一篇诗序论四篇十二巻中庸传及读史数篇十三巻儒言十四巻杂著十五巻书十六巻记十七巻序十八巻后记十九二十巻传墓表志铭祭文其中辨证经史多极精当星纪谱乃取司马光元厯邵雍元图而合谱之以七十二候六十四卦相配而成盖潜虚之流儒言则力攻荆舒之学党禁以后所作也陈振孙书录解题曰刘跂斯立墓志景迂所撰见易传后此集无之计其佚者多矣此本当即陈氏所见而讹误颇甚洪范小传及十七巻序文内间有佚简又有别本题曰嵩山集所录诗文与此本并同巻帙亦合盖一书而两名今附着于此不复别存其目焉乾隆四十六年四月恭校上
　　总纂官【臣】纪昀【臣】陆锡熊【臣】孙士毅
