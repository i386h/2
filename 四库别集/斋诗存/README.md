　　一诚斋诗存           上虞 许传霈 子醴

　　●序

　　余生而鲁钝，雅不善诗。岁辛巳，由湖返杭，尤绝意不吟咏，复何稿之可存乎。偶理丛残，有不忍遽弃者，念少壮遭际多艰，赖母教辛勤，良朋切磋，得不汨没天性，言情纪事，时见乎词，则又不可以不存。爰按年录之，起咸丰癸丑，终光绪辛巳，得八卷，计古今体若干首。初有无可斋、倦游轩、七二铃馆、春晖室诸编目，兹分注各年下以存旧名。

　　●一诚斋诗存卷一

　　○癸丑（清咸丰三年，一八五三年，十岁）

　　斋中偶占  【以下见无可斋草】
　　桃花复桂花，寒斋不冷落。二月雨红雨，八月雨金粟。【时随侍湖庠。】

　　○丙辰（清咸丰六年，一八五六年，十三岁）

　　九日登岘山
　　重九登高日，同人到岘山。堂堂怀逸老，红树白云闲。

　　○己未（清咸丰九年，一八五九年，十六岁）

　　登飞英塔
　　丹梯缭绕绝尘埃，苕霅风光面面来。画里峰峦天外落，地中钟磬寺前回。飞檐铃语当头答，负郭人家入望该。寻到塔心还有塔，此身恍若住蓬莱。

　　夏夜口占
　　重桂堂前桂荫浓，同侪夜话豁襟胸。二三女弟闲无事，细数天宁寺撞钟。

　　咏庭中白苹花
　　朝开夜合出芳姿，小草居然也准时。吾爱斜阳风动处，闲情多半催新诗。

　　○庚申（清咸丰十年，一八六零年，十七岁）

　　侍母归家
　　布帆无恙挂天涯，未惯行程问日斜。船上老人频指点，曹江西去阿侬家。
　　一峰盘绕势崔巍，老屋三间傍水隈。堂上白头人几个，杖藜言笑出门来。

　　渡钱江
　　浪高船板低，风紧布帆重。四望浑无涯，水天离一缝。

　　落花
　　一庭芳草堕残红，消尽春光满眼中。大半东皇妒艳色，莫教零落怨东风。

　　夏夜雨坐
　　黑云四盖暗山城，未冒峰头雨已行。电逐长空天地昼，雷鸣岩谷树林兵。檐前直泻无穷水，巷里时沈击柝声。夜静高楼风色定，依然星斗一江清。

　　雨中舟行
　　几扇柴扉临水开，稻田湾里舻声来。波纹细皱江如练，雨不催诗诗自催。
　　天空岸远认模糊，江树蒙蒙山有无。一幅丹青浓水墨，教人重绘辋川图。

　　秋夜泛舟
　　秋迎客路十分凉，烟水扁舟引兴长。两岸芦花翻雪白，一江木叶入秋黄。天街空际霜初落，银汉无声月自行。归橹音清闻欸乃，逍遥疑在梦魂乡。

　　家大人集千字文贺胡秀才吉席敬用原韵纪事
　　流离八载及虞城，【六合常氏与母弟避难上虞。】举目无亲孰达情。乃字终能贞素志，相攸合使结良盟。家庭静对陈诗酒，母弟欣谈解甲兵。从此唱随歌永好，百年夫妻凤凰鸣。

　　侍家大人宴金罍观
　　梓里欣情话，荷风残夏凉。山低云树绿，城近稻田黄。煮石仙人井，清心曲水觞。晚来不归去，策杖喜高堂。

　　秋夜半湖棹归
　　暮云秋树郁苍茫，一叶扁舟出野塘。水接天光天似水，霜霏月下月如霜。几家灯火前邨近，满地虫声客梦凉。浩浩御风忘所止，不知漏尽几分长。

　　秋夜独坐
　　木落宵深候，无言独上楼。天空孤月小，窗冷一灯幽。蟋蟀声中老，蜘蛛网里浮。不知今夜望，几处别离愁。

　　同褚子方、刘小墅、范桐轩、高龚甫游卧龙山，登望海亭【在绍兴府城】
　　刘褚与高范，邀我游卧龙。卧龙山色秀，盘郁越城雄。绝顶望海亭，拾级兴何浓。半壁闻松吼，高树罩烟蒙。下览城市小，鉴湖豁双瞳。冻云千里平，落叶万山冬。夕阳射西岩，人影堕长空。飘飘欲飞去，去访海若宫。海若出招我，蜃楼架彩虹。迷离极幻境，凛冽御朔风。风鸣老龙吟，四望不可从。长啸一声去，天黑归路怱。忽闻山下寺，暮鼓打冬冬。回首登临处，寒烟苍霭中。

　　转至蓬莱阁
　　此身已得到瀛洲，镜水稽山面面收。八百鉴湖归眼底，万家烟火起楼头。钟闻山寺天催暮，风吼庭松客报秋。壁上题诗苔迹扫，不愁归远独勾留。

　　消寒夜集
　　余杭褚子耘子方，世交兄弟也。庚申避地来虞，结邻黄氏宇，每夕联消寒诗集，其子侄敦伯叔寅、予叔兄子承兄子乐斋与焉。一名小竹林以集，共七人云
　　咏冬景五律分得酿雪
　　弥望云铺白，高楼雪景催。香浮金谷酒，冷逗玉堂梅。冻雀停檐噪，寒鸦结阵来。须臾飞六出，驴背灞桥回。
　　分得糟蟹
　　新酿开时晚稻红，霜螯买向五湖东。醉乡风味知何似，一笑烦君入瓮中。
　　咏越中名胜分得樵风泾
　　半肩樵采若耶中，暮向朝南一棹通。拾箭有缘酬尔愿，行舟无碍仗神功。挂帆应喜随波稳，归橹何虞扑面冲。堪笑邨人同往者，淹留难趁郑公风。
　　分咏五代史
　　不辱新君洗涤诏，要知劲节本难污。用兵卅载忠先帝，恸哭一朝见老奴。笺刺偶传多俚语，春秋熟读识兵符。直教愧死李崇政，枉自当年号丈夫。【梁敬翔】
　　上下飞来入阵中，一人两骑势如风。铁檛手舞前无敌，重铠身披气自雄。死愿见王伸宿愤，生还无罪建奇功。太原慷慨悲车裂，异日相思泣乃公。【唐李存孝】
　　陛下禁声自有臣，不知大体汉杨邠。直教礼乐为虚器，悉罢朝廷荫出身。晚节广延门下客，敢言谏立耿夫人。也知史传真堪用，课吏多抄亦雅驯。【汉杨邠】
　　咏十国春秋分得蜀王建
　　从行负玺护銮舆，枕膝君王泣下嘘。强据两川图僭窃，恩忘阿父肆诛锄。军儿惯养名多赐，符瑞侈陈史屡书。三十七年前蜀主，少时曾亦盗牛驴。
　　分咏四花
　　江郎本妙笔，绮语散云霞。天心亦爱才，一梦灿生花。【笔花】
　　古艳出芙蓉，手挥锋铦利。秋水照人心，胡霜新淬试。【剑花】
　　今夕独何夕，金粟灿银台。红袖不忍剔，喜卜远人来。【灯花】
　　前日浪花翻，今宵冰花簇。着处皆白描，不待渴鸟啄。【冰花】
　　分咏老渔
　　寒江蓑笠老渔翁，浪迹生涯有小篷。把酒酣眠忘岁月，持竿随意教儿童。偶惊两鬓风霜里，长对一篙烟雨中。为语桑榆须爱护，磻溪知己兆熊飞。
　　咏伯牙琴
　　太息钟期去，哀哉祗破琴。五弦千古感，一曲两人心。流水空留调，高山孰赏音。何如炊爨下，知己尚可寻。
　　咏雪限麻牙瓜叉拏韵
　　满天六出拏，喜瑞手频叉。忽种千亩玉，疑生万树牙。尘无光比月，花散色如麻。戏把琼瑶赠，应不报木瓜。
　　咏梅限纤添尖黏帘韵
　　美人月点纤，数点傍轻帘。陆凯江南寄，寿阳额上添。影疎寻雪里，意冷锁花尖。细摘和香嚼，芬芳满口黏。
　　咏剑限巢郊胶坳抛韵
　　本同冠佩宝书巢，利试昆吾莫易抛。一剑勋名留汉楚，三贤宠赐重庠胶。干将神物藏山谷，河曲金人出水坳。何日销为农父器，田功有事向东郊。
　　咏镜限冬钟龙浓邛韵
　　未闻宫里景阳钟，已启妆台写翠浓。自爱红颜非白髪，还看带马复萦龙。望蟾静对空悬相，化鹊偕谋莫致邛。闲贴花黄添素额，寒梅数点报开冬。
　　咏水车限南三酣韵
　　田家无雨苦耘三，斜架橘槔齐踏酣。知是天工人可代，禾苗兴起亩东南。
　　咏竹夫人
　　不怕蛾眉宠擅房，夫人却好卧牙床。瘦怜一握围腰细，汗渍千行竟体香。高节甘教居下陈，冰心果肯借清凉。祗愁纨扇西风捐，别爱温和煮热汤。
　　咏疆场金鼓声
　　刁斗声寒里，烽烟鼙鼓鸣。阵开边月黑，战劲压风横。将令衔枚肃，胡笳动地惊。忽闻驰露布，饶唱入边城。
　　咏校人烹鱼
　　漫道生还绿水浔，临渊有客羡鱼深。畜池空忆波流逝，得所偏教釜爨寻。出死那知仍入死，仁心终不敌机心。相忘近习知多少，谁勒欺方作座箴。
　　咏脚划船五排限江韵
　　一叶飘然去，生涯老越江。歌来听踏踏，划去响跫跫。篷小惟容膝，舟轻漫立艭。凿开云影碎，踢破浪声淙。赶驿如驰马，奔波异走尨。前程弧箭疾，踞坐语言哤。随水无劳纤，临风不挂籆。船头新设板，柁尾小开窗。港窄行犹阔，桥低过亦降。幻凫盘鸟道，画蠲集鱼矼。略试推移力，还成欸乃腔。青蚨囊换足，晚泊傍寒桩。
　　咏糕老虎七排限先韵
　　万户千门仍旧俗，声声捣米度残年。粘糕入谱时翻样，幻虎新搏貌自全。竟体白描非仅额，两趺蹲坐俨深拳。品分枣栗诸羞外，形出熊罴百兽先。信手拈来真活泼，随心戏弄异雕镌。点臙一样斑斓色，嵌豆双眸闪烁圆。出柙直疑轻启罩，负嵎堪笑满登筵。蓬蓬气上初蒸透，逐逐装成亦宛然。绰有须眉能镇恶，虽无骨格也清坚。承盘共乐乡风古，得句偏疑僻字传。静对悬门同色相，惊看画壁证因缘。优游卒岁添佳制，荐祖祀神礼告虔。
　　咏竹林七贤分得王戎
　　阿戎清赏比浑优，鉴识人伦本隽流。树在道边知苦李，利收水碓执牙筹。论谈元妙留侯契，舒卷随时伯玉俦。太息黄垆稽阮去，山河邈矣竹林游。

　　○辛酉（清咸丰十一年，一八六一年，十八岁）

　　家居集兴
　　一椽聊寄寂寞中，环堵萧然地数弓。扫径到门惟夜月，开窗惠我有清风。占晴课雨庞眉叟，问字抄书总角童。仰见高堂无恙候，贫家虽苦乐融融。
　　芸窗十载我何堪，漫道何堪亦自甘。壁立未空书满架，儒餐不薄菜盈篮。读余时益趋庭训，客到犹欣翦烛谈。闭户浑忘尘世事，青春闲过月将三。
　　满庭芳草绿阴繁，养得生机道自存。鸟解看书常入室，犬知谢客日临门。莫嗟来食泥炉冷，犹喜多年布被温。最好晚来无个事，擘笺记事到黄昏。

　　庭中牡丹
　　第一香开国色夸，壶觞排日赏家家。祗因误入濂溪宅，未识君为富贵花。

　　龙山道中
　　贪看山色歩迟迟，瀑布松涛一路随。最好曹娥江落日，暖风十里晚春时。

　　晓发东山指石峰
　　晓风射篷冷，残月含江碧。欸乃听声声，引入云水窟。船头老人兴，遥遥语指石。突兀蛟龙腾，氤氲风雨积。夜长烟瘴迷，横矗晓天白。山高路崎岖，断崖悬千尺。低压打头来，崔嵬势欲掷。一湾琵琶洲，天然形绣陌。弹自不成声，长指日出没。推篷证所闻，果尔挹秀色。苍茫千百年，独立谁为客。风流谢太傅，高隐适裙屐。东眺复西眺，白云自今昔。石兮不能言，云影望中隔。

　　侍家大人至汤浦
　　曲尽江流路有无，故乡风景自清腴。趁潮数艇挂帆席，傍岸千家隐荻芦。樵斧声稀闻赤壁，【山名。】渔歌唱晚起珠湖。百年桑梓知依旧，珍重汤川忆钓图。【祖父图名。】

　　烟火戏
