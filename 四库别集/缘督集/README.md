　　提要
　　縁督集二十卷　　　　别集类三【宋】
　　臣等谨案縁督集二十卷宋曾丰撰丰字幼度乐安人乾道五年进士官至德庆太守真德秀幼尝受学于丰及执政奏取其集入崇文四部当时尝板行于世岁久不传元元统间丰五世孙德安购其遗集得四十卷翰林学士虞集为之叙谓其气刚而义严辞直而理胜得于易之奇诗之葩其文今见道园学古録中然当时欲授梓不果至明嘉靖间詹事讲始选録十有二卷刻于宣城卷末有万锜后序称摘其尤者存之今观所分诸体如五言排律类中惟有排律一二首余皆五言诗七言排律类中皆七言古诗无一首为律是古今体且不能辨则其去取乖谬大概可知盖事讲从罗洪先游日以讨论心学为事文章一道非所深研遂使丰之菁华反因此选而散佚殊堪惋惜惟永乐大典编自明初尚见丰之原集其所收録较刋本多至数倍今据以増补乃裒然几还旧观佚而复存亦云幸矣丰仕迹不显颇以著述自负集中如六经论之类根柢深邃得马郑诸儒所未发其他诗文间有好竒之癖要皆有物之言非肤浅者所可企及亦南宋一作者也丰晚年筑室自号曰樽斋故是编一名樽斋集见于宋史艺文志今从虞集所广仍取縁督为名集序称凡四十卷而宋志乃作十四卷盖刋刻传冩误倒其文今原目不可复睹谨据所存各体以类区分共厘为二十卷乾隆四十六年九月恭校上

　　原序
　　縁督集者故宋徳庆太守曽侯丰幼度之文也侯抚州乐安人登乾道己丑进士第积官至朝散大夫参知政事真公徳秀幼尝学于侯侯殁真公志其墓石纳竁中不得见而侯之孙焄以防补官歴鴈汊海口两监镇调平江府节度推官则由真公念旧而推引之也二百余年而书亡国朝元统初今监察御史前进士燮理溥化来监其邑询求其乡之先生旧人得其族谱两卷故翰林学士吴公之所叙也又命侯之五世孙徳安购其遗书得今集四十卷将刻之而燮君召拜御史书未及成后三年徳安求予叙之按乐安未置邑时地属庐陵之永丰其先在唐末五代时有异人善相地者为之而子孙果蕃以盛如其占江乡好事者人人能道之徃观者至今车辙马迹不絶于其里其显者则莫盛于徳庆矣侯方未第时自着其族谱叙以为逺不及温陵宣靖公父子福徳功勲之盛近不及南丰子固兄弟文学名位之着慨然有自愤之志不一二年遂防名大廷其志亦宏矣哉予得其所欲刻者而有以见之其气刚而义严辞直而理胜其有得于易之奇诗之葩者乎取譬托兴杰然不溺于风俗山川磅礴雄伟之气盖有以发焉夫物之精华久而不灭则有神明之助者矣一编之书独发于五世诸孙之手殆非偶然也徳安能儒能医则文理之美隂徳之厚尚有以昌其书者乎侯登科之年则我先忠肃公入相之嵗俯仰今昔故为之书虞集伯生序

　　桂岩赋【并序】
　　运使马少卿于聴事之东葺故屋为宴坐所叠石为山植桂其下揭曰桂岩赋为发扬妙致作也
　　阳星之精犇入于月故月窟无凡植橠桂树之婆娑夜影耿兮澄彻月桂之种下堕于粤故粤壤无凡茁蓊桂岭之葱苍秋香扑兮清絶桂岭之别是为罗浮下有岩兮谽谺上有桂兮崒花缟缟兮粲夜叶茸茸兮郁秋小言之广东之蓬岛大言之江南之瀛洲有伟兮异境孰移寘兮公之屏天花开于逡巡石笋拔乎俄顷不墨不素冩出月中之真非水非镜照见天上之影岂公身游月宫手齅桂华心欲得之而植于家决鼻衔其核望舒擢其芽授之使归种为生涯者邪曰不然公有斧修月莫指其痕公有石补天莫扪其罅连根斫桂端乘修月之余平地起山亦出补天之暇元勲尚尔其少稽妙用輙兹其豫借左擘天台之崖右搴合浦之溠斩云骨与烟苗幻风轩与雨榭五丁犹服其敏强六子不辨其真假决鼻望舒公何彼借或曰岱舆之岭羽人所舍厥植有桂其鬻无价子累累兮雌珠而牝玑香馝馝兮仆兰而奴麝公闻之图诣觅观惧成枉驾叱夸娥兮疾趋麾禺彊兮横跨挟彼岱舆而西顿于墙壁之下夫如是则盘古公徒巨灵公亚山川繇我而峙流草木自吾而荣谢凡囿气形率归造化夫修月与补天又特公之细者也虽然至人无我大道同流寓意出机犹未免混沌之凿反身内景得无损汗漫之游盖摸索公之胷次自有一壑与一邱了知假合诚是赘肬聊复当丸弄而竿戯岂真为蜩掇而瓦抠继自今扶天柱于无底絙地维于不周举首戴山之力大试横身负河之志毕醻厌紫府而思凡告玉皇而归休亲戚故旧劳苦咨诹公其倒罗浮于嚢中倾岱舆于笔头以示之谓吾生宦防之所满拾而饱收者如此而已若其笑乎否哉
　　海栢赋【寿益公】
　　南极清老人明税层霄坠重溟化作石质流为栢形躯蕞尔兮傝气浩然兮峥嵘天□兮降材帝饶兮予龄得乔松兮称弟与大椿兮争兄甲子相忘曾何有于寒暑乾坤自大胡为乎厌薄我蓬瀛踽踽然从下士以求介绍谋与上公而为朋哉公眂栢栢眂星兰臭气味歳寒典型体不同而道合貌未接而神迎栢遇主人兮眉黄公获良晤兮眼青契若畴昔懽如平生谒平园兮不独往问讯华隠兮先登方将及身之孤髙乘足之双轻驾翛翛驭泠泠相袭籋而上青防也坐客俱惊起揽公缨非岨轻举有衰老成何当肯留上为朝廷之重下为州里之荣公徇客兮以情客寿公兮以诚有木名榆受气玉衡亶哉彼栢星气蒸也惟钟于栢物中之亭亭故钟于公人中之铮铮有士姓李受气长庚亶哉我公星气凝也惟公于星盖星之精故公于人乃人之英泥丸丹田与造化争却老逃艾还孩反婴华胥之城空同之坰恬寝愉兴娯坐嬉行忘其为长生
　　墨竹赋【中都员光祖所写】
　　我家江南种竹绕屋兹侨于京无竹而俗欣见员郎吐墨为竹不根而萌不笋而茎盘礴之顷天生地成不箨而节不苞而叶逡巡之间神造鬼设谓春滋邪外萎其形谓冬凋邪中益其精潇潇瑟瑟蒙蒙溟溟未雾露兮如湿无风雨兮若声顾壁间数竿之可意与渭上千亩而争清拔我汨汨纳诸泠泠府寺之迹既涴而复洁利名之骨久醉而忽醒惟竹太孤孰同保社鳯驯栖而思凝鹤矫立而神化有固可嘉无奚足讶所少七贤坐于其下七贤面目今莫识者视我形骸随意而写置之竹间以伴潇洒毋谓今人之不古安知真相之非假
　　蠧书鱼赋
　　嗟嗟曾子贫哉其家猗富哉其书粤昔走帝乡而干禄猗爰予书猗作踈弃而去去而归猗匝半期其有余一日发笈出书猗手欲披而莫扶糊偕楮浸腐猗阴湿猗茁菰楮偕缝两不相属猗字画猗予俱诘其蠧吾书谁猗有物焉曰鱼乎鱼而前猗讯问其辜曰微乎女之貌眇乎女之躯爪牙猗匪豹匪防角觜猗匪鼷匪鼩胡为乎敢尔齧吾之楮啗吾之糊寖而蟊贼吾孔孟之遗训尧舜之典谟按三尺与九章猗擢发不足数女之罪顾女蕞然之形猗盖不胜诛余将淬剑以戮女女其何辞以对余若其辞耶则开女以自新之途不则决不贷女之命于须防鱼也哀而乞怜曰公且息一时之怒待吾尽其情以愬倘其自饰以游词然后伏公之刀锯初吾之营营猗非为口腹之故盖闻仁义之胜乎膏粱猗可以供吾之啜哺故潜身入其间猗欲求其餍饫奈何哉其口不识味猗翻为书之蠧虽然吾独小蠧尔不幸为公擒尚有大蠧焉公胡不之虑曾子惊曰今安在哉吾其往捕曰其来也代久歳深其众也云屯螘聚粤自孔墙失护厥徒横骛弥茫其正路出入其异户戕穴吾春秋猗断断乎公谷之据穿窬吾风雅猗拘拘乎毛郑之序书猗胶于秦誓易猗梏于象数又其甚者韩非师老剥天下之肌肤李斯事荀坏先王之法度信夫千里之差初者起于跬步不然胡不见墨子悲丝莫知所措杨朱泣岐傍偟四顾原二子之初心岂欲无君无父也耶由是观之蠧公之书彼盖其尤者也公不锄而去而独归咎于我何欤纵云在我盖出于误于法猗当诛于情猗可恕曽子闻而惊曰吁女之不幸猗至此哉宜其为人之憎恶安得圣人出而鍼其受病之处昔者尝闻原壤之也必至于为晋为梁故孔子为之拔其根之固师商之弊也必至于为杨为墨故孔子为之开其所未喻嗟女蠧书鱼猗惜乎不孔子之遇顾我虽非孔子猗焉忍坐眎其僵仆尔蠧书鱼来吾语女道学有捷法圣门有真趣轲也岂尽信乎书参乎惟一唯之悟继今而尔能糟粕之捐猗醇全之茹则不惟前非之可赎猗自得之学云庶不然岂独为人之憎恶而已哉甚则又重遭其击抪也鱼谢而退曾子归而叹曰悲哉鱼之蠧吾书也不过乎文字之残人之为鱼也并予其道之斁吾为之惧故为之赋
　　楚词
　　祀蚕先【并序】
　　人以蚕兮为衣蚕以桑兮为粮柞樗棘栾岂无叶兮可食之今蚕嗅矣讫不尝是以未即治蚕先期治桑或问蚕祖则曰园客之妻呉县之妇抑尝攷之赤帝之女学道得仙巢于桑树凭桑而灵享蚕者胙夫人生理系桑与农寒衣兮饥食用异兮功同通郡县祀稷为于农兮有功彼蚕先虽未废自后外其孰崇厥功同其祀异防何不与世之所谓受施而忘报者类大事难乎文拘旷礼可以义起祀彼蚕先由吾家始又从而为声诗以相传庶防遍区夏而皆祀其词曰
　　旦谷兮辰刚聿蠲衷兮齐以庄孔伛兮再掺擎厥拳兮三迎将飞防扫兮涴者以清赤松洒兮溽者以凉灵之来嫖嫖兮若翔兮上灵之下婉婉兮若倚兮傍灵之趋躞躞兮若参兮前灵之坐盘盘兮若据兮中央乃陈其太牢侑以兮糜餹餻柔兮膋肉腯鼠脑脆兮糜芗灵其肹只勿虚我佖只世传沙宅兮亥地泥屋兮壬方枣种南舍风来兮东厢夫是谓兮宜蚕征其如我敎兮未详顾蚕兮消长惟灵兮弛张纷再登三熟与四出信吉利而盖故常雅志兮蚕事匪徒制兮已之装丝吾欲系象兮俾不折茧吾欲投火兮俾不伤于以试兮允哉验殆将补兮舜裳灵其烱只勿虚我景只蛹有种蛾有秧圆峤兮色黑寒国兮色苍丝一指兮大茧一尺兮长蚕有兮彼异何当畀兮千斯箱灵其佽只勿虚我企只
　　乞如愿
　　丙午歳首有商叟过余曰如愿叟叟嬖丑也逸不知其姓某或曰喜母之女欢伯之妇得自青洪君之手至自彭泽湖之口妩奉兮叟左婉承兮叟右叟歉兮授以丰叟薄兮授以厚假非女横纵兮坤六倒颠兮干九安能转叟之凡百竒为一切耦乎委以赤心期之白首歘于晨兴潜以形走门有守户有候其故纵之守候是咎守垒候荼眙嗟吁奉职警梱曷尝少踈犹疑未获未出而庐爰搜爰索乃吁乃呼则趦趦趄趄闯于东窳糢糢糊糊霿于紫姑彼紫姑者能文知书叟不识字势难动渠公善形容以乞焉彼宜响应以归欤喜不自禁则节文而为歈曰来紫姑□甘有兮松醽糅以兮胶饧紫姑来下盼余馨□辛有兮椒醁糅以兮葱□紫姑来下盼余馥□淡有兮桃醿糅以兮柏饴紫姑来下盼余馡□三薫兮三酹夫有恳兮其敢昧久求如愿兮顾兹在防为余兮割爱□来紫姑伊嫠且媰【音初】□凡借顔色饶髭须□厥有李赤豫譲范雎□若乃鬼残彼非刑余□彼所有若所无缓急自可相须□讵必如愿而弗以乞余□羌紫姑形不露脰而有影若拇殆余授邪羌如愿声不露噣而有响若吽殆余邪公其愿贵或愿富邪贵岂能玉余骨使莫腐富岂能氷余肉使不臭邪文章可传金石不朽要贵其精须迟以久余心所愿汝假之寿应声□□似曰有有
