　　○古赋

　　石渠阁赋【丙寅江西试题】

　　维大汉甘露三年诏储儒讲五经同异于是名儒自萧望之以下咸集于石渠阁圣上亲称制临决五经学官至是始备谏议大夫臣褒窃睹陛下以戎狄宾服海内又安诏讲经籍丕崇化本甚盛事也愚臣幸陪缙绅末议而学术謭陋不能窥要义折众言至于着之赋颂以铺张鸿休较其劣能亦仅可采窃惟石渠阁者乃藏书之府考论之地宜敷其事以彰徽猷谨拜手稽首而献赋曰
　　猗赫赫乎圣明之中兴也化洽于九区二十有三祀重明掲乎日月茂徳参乎天地慕商宗古训之闻陋周宣大搜之事于是发睿断诏羣儒启金匮临石渠卦爻穷其至赜谟诰订其众疑考风雅之辞义审仪制之等衰圣感麟而有作亦褒贬之宜稽葢非闳硕之士无以核是非辨同异非髙敞之居无以储篇籍列冠带蔚乎盛哉信治朝之伟观人文之巨丽也是阁也据神都之域冠未央之宫拟麒麟之壮抗天禄之雄酇侯设其榘班匠呈其功峻齐山岳巧若鬼神俯压坤轴仰支穹旻凌埃堁而危跱俨飞翚以孤鶱状■〈山上罪下〉嵬而菌层势轇轕而云屯架杏梁之夭矫刻芝栭以丛攒掀觚棱之鳯翅灿雕甍之龙文璇题照日以暠曜珠网蒙雾以纷纶迎朝霞而纳夕照际河汉而迩星辰其中则玲珑绮疏璘■〈王扁〉列钱若摠章之牖户其傍则便娟曲室窈窱间宇若閟宫之广邃环而为渠则端若引绳深若川池微波龙鳞藻荇参差露浟浟之鰋鲤散泛泛之鳬鹥甃以宻石则坚比胶漆莹类圭璧上廉锋削下址铁积象峭壁之将将亘周除之涤涤若乃夷道縰属朱扉洞开车骑络绎于兹往来阅其编简则■〈贝贳〉于秦灰之余侈于孔壁之旧先王之策府莫之尚焉聆其辨说则可否五味之济倡和八音之谐帝廷之吁俞无以异焉于是天子乃备法驾乘次辂树翠华之旗驾苍龙之驷公卿庶僚云滃飙戾济济之臣于于之士下逮羽林之儿带剑之夫靡不森立而骈罗奉简以进协于帝衷臣向臣胜为夔为龙凡在搢绅谘诹是同训必明夫圣之义义必揆夫道之中葢以其一日之论绍乎千载之宗是岂不足以示来裔而耀无穷也哉凡观听之士莫不心醉奥旨辞赞懿徳畅若春熈疑若冰释乃为之颂曰石渠之辟以演圣泽流泱泱兮阿阁与侔祥鳯斯集烜阳光兮日月久照列星咸丽皇文昌兮前圣之启后圣之嗣胠閟藏兮恩被湛露化周仁风覃万方兮典章炳焕鸿业绵邈垂无疆兮群臣于是洋洋欢洽再拜而退咸愿纪述昭于世世

　　蒙山赋

　　蒙山者吾州之峻岳也然郡志仅传名不逺着禹贡言蔡蒙旅平则梁州之境鲁颂言奄有龟蒙则东海之地而史记之注又曰老莱子楚人躬耕蒙山之阳今考周之季世鲁并于楚龟蒙属焉而吾州亦楚域也又安知老莱之隠不在于吾土乎余家于是山之阳瞻神秀之异慕贤哲之风遂为之赋其词曰
　　维堪舆之奠位斯山岳之肇形类波涛之激荡拥泥沙之崚嶒为峤为岑或丘或陵尊若帝王陪以公卿然人也以山而锺秀山也因人而著称在于荒陬虽崇而隠迩乎雄都虽卑而名窃爱夫蒙岭之镇吾地其于终南少室天台会稽何彼重而此轻也哉观其势雄千仞根盘百里嵯峨嵚崟蜿蟺峛崺窍以诸洞界以众水巘崿鳞鳞崖石齿齿疆域之标郡国之纪皂嶂东回春台西峙鳌峰北耸渝流南迤其为寳藏则白金之璀璨碧钿之璘■〈王扁〉表炎方之竒产充天府之上珍其为灵踪则金刹据其阳仙岩在其巅森祗树于翠谷閟小有于洞天若乃瑶草匊芳珠林挺秀风篁球戛烟萝帷覆兰茝缘于径畈芝术产乎嵌窦长阴涧之茯苓绚霜林之橘柚猿猱哀啸熊罴怒咆雾卧文豹湫吟黒蚪扬夜鹤之遥音露秋鹘之悬巢过客眩于妖狐樵子駴乎山魈凡其根荄之品飞走之族时荣时悴以涵以育富于灵囿之骈罗伙于陆海之储蓄博物之张华无以辨其类洽闻之郭璞曷能究其目宜夫絶俗贞士离世潜夫恱云霞而长往与麋鹿以为徒翫冈葩之春媚乐岑木之夏敷吟素秋于月观度玄冬于雪庐坐紫苔兮绿绮奏荫苍松兮芸帙舒或茹芝兮白日晏或扫叶兮凉飙随想夫楚叟之贤栖山之阳躬尚稼穑志轻侯王观山泉之泓渟悟蒙养之端荘惬闲情于浚壑杜妄辔于岩廊嗟轩尧其既逺顾沮溺其可望励雅操之氷玉谢溷世之粃糠疑斯人之或逺复东瞻乎鲁邦既异世以同趣宜随地而安常吾将税俗驾屏尘心企颍阳懐箕阴朝与山兮为俦夕与山兮相寻濯缨松下流振衣云中岑宁抱瓮以自劳赋归来而长吟厯万里于至近返千龄于方今庶永托于兹山嗣髙贤之徽音

　　○五言古体诗

　　归醴溪

　　久厌都市喧俛思山岩静归饮醴溪泉怡我淳朴性神峯杂树蓊石门翠厓并萝悬晨露滋巘秀夕霞映攀陟谐樵牧候谒悔奔竞褰裾荫云松脱屣悦风磴悠然遯客心迭出野人咏方期谷口耕毋诮终南径

　　送辜徳中会试

　　写我山泽言送君云霄举贵人气每骄下士心独苦纷纷竞科第口吐圣贤语朝出原宪室夕与王侯伍我视辜贤良懐珠类良贾乃知淳厚士不独在齐鲁松柏生涧冈出为明堂柱虽言被丹漆赤心若可覩既应纶言征将随玉阶步嵯峨黄金台望君为延竚

　　和杨逸人桃林迁居之作

　　心寂忘尘嚣时清乐岩筑爱此桃华林葺之杜蘅屋葛篱延芳藟石梁照浅渌风翔双白鸟雨卧一黄犊东皋杂妇子中野见樵牧赋诗云霞坞携壶锦绣谷从君以优游于此谢羁束

　　题西山程氏南窗

　　独居南窗静秋木连翠岑鸣琴众叶下把酒孤猿吟宁负朱绂愿莫乖白云心羡彼归田叟髙风留至今

　　为临川章则常题山水圗

　　山拥衡庐青水含潇湘碧两厓蟠蛟树千岁蹲虎石遥遥遡川舟泛泛蹇驴客凭髙者何人闲看楚云白

　　题印土寺罢钓轩

　　曲池照林坞百尺悬萝阴开轩坐埀钓水清如我心小鱼易伤生大鱼或渊沈投竿为之恻于此悔悟深名区縻好爵都市争黄金但恋钩饵馨岂知忧患侵濯足南涧流振衣北山岑既羞结网罟何必溉釡鬵物生免相贼嗜欲常自禁嘉此古人意托之金石音

　　城中女

　　娥娥城中女小楼对门家相骄茜裙新并笑双鬟斜花窗弄鹦鹉月榭弹琵琶不自结衣带笑人说丝麻越女当自羞燕姬何足夸但愁玉颜改眼底生尘沙秋江芙蓉落天寒空怨嗟

　　拟古十二首

　　独处衡门下慨然思九州岛我马苦疲怯山川多阻修丰草被长坂麋鹿或羣逰蔚彼嘉树阴鸣禽自相求彼物各有适而我何寡俦日月双车轮但恐不少留愿与二三友朝夕论王猷毋为自局促穷庐悲白头
　　出门望长道车马何阗阗白日殷轻雷飞尘若为烟周人好商贾锥刀争贸迁赵女夸绮服跕蹝弹鸣弦营营苦不足衣食思华鲜奔涛日东注何由为静渊
　　南州产嘉橘朱实日光炫燕晋夸枣栗连林极丰羡枣栗充糇■〈飠量〉足以饥馁咽橘酸良不如亦可笾豆荐踰淮或成枳性移贵反贱贤哉锺仪子土风慎不变
　　名都少年子金多矜富强连云起甲第峨峨拟侯王外廏骈骐骥侍女罗姬姜豪贵相经过绮席飞琼觞醉言气凌人欢乐殊未央嘲笑东邻士晏食惟糟糠艳艳桃李花随时逞妍芳岂知易零落荣华安可常
　　鳯栖必孤桐鹤集思髙松飞翔择所止羞与鹪鹩同踽踽贱贫士混混常俗中节槩曾闵贤言语班扬工被服常布褐虚室生蒿蓬向非贤哲举何以树勲庸
　　芙蓉在江浦亭亭艳清波虽云出淤泥丽质良可嘉褰裳涉枉渚凌风折芳花愿言赠君子翫之比瑶华于时茍不惜顦顇当奈何
　　时违骨肉弃位髙仇怨亲离合心岂常势利情所因君子羞薄俗古风恐夷泯深知慕管鲍轻怨羞张陈蕙荃不随化球琳岂易磷结交英雄士白首当如新
　　江海处卑下百川皆赴之山木郁萧森上竦无繁枝王侯称孤寡惟恐婴菑危谨虚受众益天道盈必亏周公下白屋吐哺情孜孜孙子相强楚禄丰心愈卑奈何闾阎子往往多矜持
　　昔常好名山五岳期遍歴思见松与乔再拜问仙液微生累妻子常为饥冻役叹此血肉躯何以生羽翼谅非金石同服食竟何益不如安我常百年任所适
　　云门辍清响郑卫音方扬锦衣受垢污不如练布良轩轩青云士鸣玉升庙堂名髙受谗毁宠盛罹愆殃美女恶女仇偏听奸以萌众口能铄金况乃忘周防所以君子心惟用徳自将行行九折阪戒哉衔橜伤
　　驱马出东郭松林见髙坟云是公侯葬华表干层云岁月既已逺朽石生荆榛虺蛇或内蛰狐狸当昼蹲生者歌其上死者宁复闻感此长太息浮生若飙尘惟当朂令徳千载逝犹存
　　我有太古琴千载妙音续七弦何泠泠听之非促遫一弹文王操再弹宣父曲圣人宛见之何由蹑其躅大道日以沦浇诡驰众欲孰思障頺波九州岛返淳俗夔龙为股肱巢许遯岩谷穷通各有志于我奚不足

　　鸡鸣

　　鸡鸣九街曙喧喧多市声马似阴云度尘随红日生自非守贫者何以免营营

　　归息

　　归息蒙岭阳聊比嵩少岑乐此林薮趣遂忘江海心沿涧搴芳藟入谷听幽禽流泉响松下潺湲非俗音猗兰被径傍香气袭衣襟頺景忽西匿浮云夕以阴睠彼羣飞鸟翩翩投北林
　　蔼蔼岩前树翠叶敷鲜滋攅葩杂丹素条蔓相因依丰草覆侧径麋鹿或羣随石潭见逰儵翫之往复来念昔困羁绁山泽心毎违傍岩葺茅茨庶使夙愿谐黄绮各有志澹然何是非
　　春阳既和煦时雨亦沾濡原野绿已遍土润含膏腴兵戈幸少戢良农日垦锄戴胜鸣桑林鶗鴂亦交呼嗟我理编简四体忘勤劬羔裘愧逍遥梁鹈耻安居素餐古所诮偶耕良我徒

　　食蕨

　　薇蕨生固殊类同若兄弟夷齐昔茹薇蕨亦吾所嗜味非驼峰美味非熊掌异但慕夷齐风嗜此心不愧
